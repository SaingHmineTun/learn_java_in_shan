# လွင်ႈၽူႈပူင်သွၼ်: ၸၢႆးမၢဝ်း (Sai Saing Hmine Tun)

## 🚀 ၶေႃႈၼႄႉၼွမ်း
မႂ်ႇသုင်ၶႃႈ ၽူႈလဵပ်ႈႁဵၼ်းၵူႈၵေႃႉ! ၶႃႈပဵၼ် **ၸၢႆးမၢဝ်း** ၶႃႈ။ ၶႃႈၼႆႉပဵၼ် ၽူႈၶူင်ႊသၢင်ႈ Software လႄႈ ၽူႈပူင်သွၼ် ပၢႆးဢၢႆႇထီႇ ဢၼ်ယူႇသၢဝ်းဝႆႉၼႂ်း မိူင်းမျၢၼ်ႇမႃႇ ၵေႃႉၼိုင်ႈၶႃႈ။
ၵၢၼ်တူင်ႉၼိုင်ၶႃႈၼႆႉ မုင်ႈမွင်းတွၼ်ႈတႃႇဢဝ် ပၢႆးပၺ်ႇၺႃႇတႅၵ်ႉၼူဝ်ႇလူဝ်ႇၵျီႇပၢၼ်မႂ်ႇ လႄႈ လွင်ႈႁၵ်ႉသႃလိၵ်ႈလၢႆးၾိင်ႈထုင်းတႆးႁဝ်း မႃးၵွင်ႉသိုပ်ႇၵၼ်ၼႆၶႃႈ။

ၶႃႈပဵၼ်ၽူႈၵေႃႇတင်ႈ **Thung Mao Kham (TMK) Academy** ၶႃႈ။ တီႈၼႆႈ ၶႃႈၶတ်းၸႂ်ႁဵတ်းဢွၵ်ႇပၼ် ၵၢၼ်သွၼ် (Tutorials) လႄႈ ၶေႃႈမုၼ်းလွင်ႈပၢႆးတႅၵ်ႉ တွၼ်ႈတႃႇၸွႆႈထႅမ်ၵူၼ်းၼုမ်ႇ ဢၼ်ၶႂ်ႈပဵၼ် Developer တီႈမိူင်းႁဝ်းၼႆၶႃႈ။

## 📖 လွင်ႈယိူင်းဢၢၼ်းၶႃႈႁဝ်း
ၶႃႈမီး လွင်ႈယုမ်ႇယမ်ဝႆႉဝႃႈ: **ပၢႆးဢၢႆႇထီႇၼႆႉ လူဝ်ႇလႆႈပဵၼ်တွၼ်ႈတႃႇၵူၼ်းၵူႈၵေႃႉ** ၼႆၶႃႈ။ ၵဵဝ်ႇလူၺ်ႈလွင်ႈၼႆႉၼႆ ၶႃးလႆးၶတ်းၸႂ် တူင်ႉၼိုင်မႃးမီးၼင်ႇၼႆၶႃႈ:
- **ၵၢၼ်သွၼ်:** ပိုတ်ႇၽုၺ်ႇ ပၢင်သွၼ်ၵၢၼ်တႅမ်ႈ Program (Java, Python, Flutter, etc.) လူၺ်ႈၽႃႇသႃႇၵႂၢမ်းတႆး။
- **ၶူင်ႊသၢင်ႈ:** ၶူင်ႊသၢင်ႈၶိူင်ႈၸႂ်ႉမိူၼ်ၼင်ႇ TMK Keyboard လႄႈ လွၵ်းလၢႆးပိၼ်ႇလိၵ်ႈ (Transliteration) တွၼ်ႈတႃႇႁၵ်ႉသႃလိၵ်ႈတႆး ၼႂ်းပၢၼ် Digital။
- **Open Source:** ၶဝ်ႈႁူမ်ႈၼႂ်းၵၢၼ်ၶူင်ႊသၢင်ႈ Software ဢၼ်ၵူၼ်းၵူႈၵေႃႉၸႂ်ႉလႆႈလၢႆလၢႆ။

## 🛠 တၢင်းၵတ်ႉၶႅၼ်ႇ လႄႈ ပၢႆးပၺ်ႇၺႃႇ
ပၢႆးဢၼ် ၶႃႈလႆး ၶတ်းၸႂ်လဵပ်ႈႁဵၼ်းမႃး ပီႊယၢဝ်းၶၢဝ်းႁိုင်လႄႈ ဢၼ်သွၼ်ပၼ်မႃးၼႆႉတေႉၵေႃႈ:
- **Mobile:** Android Development (Kotlin/Java), Flutter & Dart.
- **Web:** React, Next.js, Node.js, TypeScript.
- **Languages:** Python, Java, JavaScript, HTML/CSS.
- **Data:** MySQL, PostgreSQL, REST APIs.
- **လွင်ႈၶိုၵ်ႉတွၼ်း:** Shan Language Processing & NLP.

## 🌟 ၵၢၼ်ၶူင်ႊသၢင်ႈဢၼ်လိုဝ်းလင်
- **TMK Keyboard:** ဢႅပ်း တႃႇတႅမ်ႈလိၵ်ႈတႆး လႄႈ လိၵ်ႈမၢၼ်ႈ ၼိူဝ် Android။
- **Pakpi Calendar:** ပၵ်းယဵမ်ႈဝၼ်းတႆး တွၼ်ႈတႃႇၾူၼ်း လႄႈ ၶွမ်ႇပိဝ်ႇတႃႇ။
- **TMK Typing Tutor:** Software တွၼ်ႈတႃႇၽိုၵ်းပေႃႉလိၵ်ႈတႆး လႄႈ မၢၼ်ႈ။

## 🤝 တီႈၵပ်းသိုပ်ႇ
ၵၢၼ်ႁဵၼ်းတႅမ်ႈ Program ၼႆႉ ပဵၼ်ၵၢၼ် ဢၼ်တေလႆႈၸႂ်ႉ ၶၢဝ်းယၢမ်းၶႃႈဢေႃႈ။ ၵၢၼ်သွၼ်ၸိူဝ်းၼႆႉ ၶႃႈႁဵတ်းဝႆႉႁႂ်ႈပဵၼ် PDF တွၼ်ႈတႃႇႁႂ်ႈႁဵၼ်းလႆႈ ၵူႈတီႈၵူႈၶၢဝ်းယၢမ်း (Offline) ၶႃႈ။
သင်မီးတၢင်းႁၼ်ထိုင်သင်၊ ဢမ်ၼၼ်ၵေႃႈ ၶႂ်ႈပၼ်တၢင်းၸွႆႈထႅမ်သင်ၼႆၸိုင် ၵပ်းသိုပ်ႇၶႃးမႃးလႆႈ ၼင်ႇတီႈတႂ်ႈၼႆႉၶႃႈ။

# About the Instructor: Sai Mao (Sai Saing Hmine Tun)

## 🚀 Introduction
Hello, learners! I am **Sai Mao**, a passionate Software Developer and Tech Educator from Myanmar. My work is dedicated to bridging the gap between modern technology and cultural preservation, specifically for the Shan community.

I am the founder of **Thung Mao Kham (TMK) Academy**, where I produce tutorials and resources to empower aspiring developers across Myanmar and beyond.

## 📖 My Mission
My journey is driven by a simple belief: **Technology should be accessible to everyone.** I focus on:
- **Tech Education:** Producing comprehensive programming tutorials (Java, Python, Flutter, etc.) in the Shan language.
- **Language Tech:** Creating tools like the TMK Keyboard and transliteration systems to preserve the Shan language in the digital age.
- **Open Source:** Contributing to projects that make software more accessible and collaborative.

## 🛠 Technologies & Skills
I have spent years mastering and teaching various technologies to help students build real-world applications:
- **Mobile:** Android Development (Kotlin/Java), Flutter & Dart.
- **Web:** React, Next.js, Node.js, TypeScript.
- **Languages:** Python, Java, JavaScript, HTML/CSS.
- **Data:** MySQL, PostgreSQL, REST APIs.
- **Specialty:** Shan Language Processing & NLP.

## 🌟 Featured Projects
- **TMK Keyboard:** Custom Android keyboard for Shan and Myanmar languages.
- **Pakpi Calendar:** Traditional Shan calendar for mobile and desktop.
- **TMK Typing Tutor:** Software to help learners master Shan and Myanmar typing.
