### **Lesson 14: Comparison & Logical Operators (Making Decisions)**

တွၼ်ႈတႃႇပရူဝ်ႇၵရႅမ်ႇႁဝ်း တေတႅပ်းတ်တ်းသၢႆၸႂ်လႆႈၼၼ်ႉ မၼ်းလူဝ်ႇလႆႈ "တႅၵ်ႈၼိူင်း" (Compare) ၶေႃႈမုၼ်းဢွၼ်တၢင်းၶႃႈ။ ၽူၼ်မၼ်း (Result) တေဢွၵ်ႇမႃးပဵၼ် **`boolean`** (`true` / `false`) ၵူၺ်းၶႃႈဢေႃႈ။

---

#### **1. Comparison Operators (ၵၢၼ်တႅၵ်ႈၼိူင်း)**

ၼႆႉပဵၼ်သဵၼ်ႈမၢႆ Operators တႃႇတူၺ်းၵႃႈၶၼ်သွင်ဢၼ်ၶႃႈ:

| Operator | ၼႃႈၵၢၼ် (Description) | တူဝ်ယၢင်ႇ (Example) | ၽူၼ်မၼ်း (Result) |
| :--- | :--- | :--- | :--- |
| **`===`** | Strict Equal (မႅၼ်ႈတႅတ်ႈတေႃး) | `5 === 5` | `true` |
| **`!==`** | Strict Not Equal (ဢမ်ႇမႅၼ်ႈတႅတ်ႈတေႃး) | `5 !== 10` | `true` |
| **`==`** | Loose Equal (တူၺ်းၵႃႈၶၼ်ၵူၺ်း) | `5 == "5"` | `true` |
| **`!=`** | Loose Not Equal (ဢမ်ႇမိူၼ်ၵၼ်လၢႆလၢႆ) | `5 != "10"` | `true` |
| **`>`** | Greater Than (ယႂ်ႇသေ) | `10 > 5` | `true` |
| **`<`** | Less Than (လဵၵ်ႉသေ) | `5 < 10` | `true` |
| **`>=`** | Greater or Equal (ယႂ်ႇသေ ဢမ်ႇၼၼ် မိူၼ်ၵၼ်) | `10 >= 10` | `true` |
| **`<=`** | Less or Equal (လဵၵ်ႉသေ ဢမ်ႇၼၼ် မိူၼ်ၵၼ်) | `5 <= 10` | `true` |



---

#### **2. လွင်ႈပႅၵ်ႇပိူင်ႈ `==` လႄႈ `===` (လွင်ႈယႂ်ႇ!)**

ၼႂ်း TypeScript ႁဝ်းလူဝ်ႇလႆႈယၢင်ႇသတိလွင်ႈၼႆႉတႄႉတႄႉၶႃႈ:

* **`==` (Loose Equality):** မၼ်းတူၺ်းၵူၺ်းဝႃႈ "ၵႃႈၶၼ်" မိူၼ်ၵၼ်ႁႃႉ။ သင်ဝႃႈ Type ဢမ်ႇမိူၼ်ၵၼ် (ၼင်ႇ တူဝ်ၼပ်ႉ တင်း တူဝ်လိၵ်ႈ) မၼ်းတေၸမ်းလႅၵ်ႈ Type ႁင်းၵူၺ်းသေ တႅၵ်ႈၼိူင်းပၼ်ၶႃႈ။ (တူဝ်ယၢင်ႇ: `5 == "5"` ပဵၼ် `true`)
* **`===` (Strict Equality):** မၼ်းတေတူၺ်းတင်း **"ၵႃႈၶၼ်"** လႄႈ **"မဵဝ်းၶေႃႈမုၼ်း (Type)"** ႁႂ်ႈမႅၼ်ႈၵၼ်တင်းသွင်ဢၼ်ၶႃႈ။ (တူဝ်ယၢင်ႇ: `5 === "5"` ပဵၼ် `false` ယွၼ်ႉမဵဝ်းမၼ်းဢမ်ႇမိူၼ်ၵၼ်)
* **`!=` လႄႈ `!==`:** မၼ်းတေႁဵတ်းၵၢၼ်မိူၼ်ၵၼ်ၶႃႈ။ `!==` တေၸႅတ်ႈထတ်းပၼ်တင်း Type ႁႂ်ႈၼႄႇၸႂ်ဝႃႈ မၼ်း "ဢမ်ႇမႅၼ်ႈၵၼ်တေႉတေႉ" ၶႃႈဢေႃႈ။

---

#### **3. Logical Operators (ၵၢၼ်ၸႂ်ႉသၢႆၸႂ်)**

ၸႂ်ႉတႃႇႁူမ်ႈတၢင်းလိူၵ်ႈတင်းၼမ်ၶဝ်ႈၵၼ် ဢမ်ႇၼၼ် တႃႇလႅၵ်ႈလၢႆႈၵႃႈၶၼ် boolean ၶႃႈ။

* **`&&` (Logical AND):** တေပဵၼ် `true` သင်ဝႃႈ **မႅၼ်ႈတင်းသွင်ၽၢႆႇ**။
* **`||` (Logical OR):** တေပဵၼ် `true` သင်ဝႃႈ **မႅၼ်ႈၽၢႆႇလႂ်ၽၢႆႇၼိုင်ႈ**။
* **`!` (Logical NOT):** လႅၵ်ႈလၢႆႈၵႃႈၶၼ်ပဵၼ်ၽၢႆႇၶတ်းၵၼ် (`true` -> `false`)။



**တူဝ်ယၢင်ႇ (Code Example):**
```typescript
let age: number = 20;
let isStudent: boolean = true;

// ၸႅတ်ႈဝႃႈ ဢႃႇယု ၼႃလိူဝ် 18 "လႄႈ" ပဵၼ်လုၵ်ႈႁဵၼ်းႁႃႉ
console.log(age > 18 && isStudent); // true

// ၸႅတ်ႈဝႃႈ ဢႃႇယု လဵၵ်ႉသေ 10 "ဢမ်ႇၼၼ်" ယႂ်ႇသေ 60 ႁႃႉ
console.log(age < 10 || age > 60); // false
```

- [ ] ပွင်ႇၸႂ်ဝႃႈ `!==` လႄႈ `===` လီၸႂ်ႉသေ `!=` လႄႈ `==` ယွၼ်ႉမၼ်းၸႅတ်ႈထတ်းပၼ်တင်း Type။
- [ ] ၸႂ်ႉ Logical Operators (`&&`, `||`) တႃႇႁူမ်ႈတၢင်းလိူၵ်ႈလႆႈ။
- [ ] ၸႂ်ႉ `!` (NOT) တႃႇလႅၵ်ႈလၢႆႈၵႃႈၶၼ် boolean ႁႂ်ႈပဵၼ်ၽၢႆႇၶတ်းၵၼ်။
