### **Lesson 18: Loops (Iterating with For, While, and Do-While)**

မိူဝ်ႈႁဝ်းၶႂ်ႈႁႂ်ႈၶူတ်ႉ (Code) ႁဝ်းႁဵတ်းၵၢၼ် "ပၼ်ႇၵႂႃႇပၼ်ႇမႃး" လၢႆလၢႆပွၵ်ႈ (ၼင်ႇ တႅမ်ႈလိၵ်ႈ 100 ပွၵ်ႈ ဢမ်ႇၼၼ် ဢၢၼ်ႇသဵၼ်ႈမၢႆလုၵ်ႈႁဵၼ်းတင်းသဵၼ်ႈ) ႁဝ်းတေၸႂ်ႉ **Loops** ၶႃႈဢေႃႈ။ မၼ်းၸွႆးႁႂ်ႈႁဝ်းဢမ်ႇလူဝ်ႇတႅမ်ႈၶူတ်ႉသဵၼ်ႈလဵဝ်ၵၼ်ၼၼ်ႉ လၢႆလၢႆပွၵ်ႈၶႃႈ။

---

#### **1. For Loop (ပၼ်ႇၸွမ်းၼင်ႇၼပ်ႉဝႆႉ)**
ပဵၼ် Loop ဢၼ်ႁဝ်းၵႆႉၸႂ်ႉသုတ်းၶႃႈ။ ႁဝ်းတေၸႂ်ႉမၼ်းမိူဝ်ႈႁဝ်း "ႁူႉဝႃႈတေပၼ်ႇလၢႆပွၵ်ႈ" ၼၼ်ႉၶႃႈဢေႃႈ။

**Structure (ပိူင်တႅမ်ႈ):**
```typescript
for (let i = 0; i < 5; i++) {
  console.log("ပွၵ်ႈၵမ်း: " + i);
}
```
* **`let i = 0`**: တႄႇၼပ်ႉတီႈ 0 (Initialization)။
* **`i < 5`**: တေပၼ်ႇယူႇတေႃႇသေ i တေႃႉတႄႉလဵၵ်ႉသေ 5 (Condition)။
* **`i++`**: မိူဝ်ႈယဝ်ႉၼိုင်ႈပွၵ်ႈ ႁႂ်ႈထႅမ်ပၼ် 1 (Increment)။



---

#### **2. While Loop (ပၼ်ႇယူႇတေႃႇသေမႅၼ်ႈ)**
ၸႂ်ႉမိူဝ်ႈႁဝ်း "ဢမ်ႇၼႄႇၸႂ်ဝႃႈတေပၼ်ႇလၢႆပွၵ်ႈ"၊ ၵူၺ်းၵႃႈ ႁဝ်းၶႂ်ႈႁႂ်ႈမၼ်းပၼ်ႇယူႇတေႃႇသေ Condition ၼၼ်ႉပဵၼ် `true` ယူႇၶႃႈ။

```typescript
let count: number = 1;

while (count <= 3) {
  console.log("Count is: " + count);
  count++; // ယႃႇလိုမ်းထႅမ်ၵႃႈၶၼ် မၼ်းၸင်ႇတေၵတ်းလႆႈ
}
```



---

#### **3. Do-While Loop (ႁဵတ်းဢွၼ်တၢင်း ၸႅတ်ႈထတ်းဝၢႆးလင်)**
မၼ်းမိူၼ်ၼင်ႇ `while` ယူႇ၊ ၵူၺ်းၵႃႈ မၼ်းတေ **"ႁဵတ်းၵၢၼ်ဢွၼ်တၢင်း ၼိုင်ႈပွၵ်ႈ"** သေ ၸင်ႇမႃးၸႅတ်ႈထတ်း Condition ဝၢႆးလင်ၶႃႈ။ မၢႆထိုင်ဝႃႈ မၼ်းတေလႅၼ်ႈလႆႈ တီႈၼပ်ႉ 1 ပွၵ်ႈ တႃႇသေႇၶႃႈ။

```typescript
let x: number = 10;

do {
  console.log("ႁဵတ်းၵၢၼ်ယဝ်ႉ!");
  x++;
} while (x < 5); 
// တီႈၼႆႉ x ဢမ်ႇလဵၵ်ႉသေ 5 သေတႃႉ မၼ်းတေလႅၼ်ႈ 1 ပွၵ်ႈယဝ်ႉ ၸင်ႇၵတ်းၶႃႈ။
```



---

#### **4. Break လႄႈ Continue**
* **`break`**: ၸႂ်ႉတႃႇ "ၵတ်း" (Stop) Loop ၼၼ်ႉၵမ်းလဵဝ် (ဢွၵ်ႇၵႂႃႇၶႃႈ)။
* **`continue`**: ၸႂ်ႉတႃႇ "ၶၢမ်ႈ" (Skip) ပွၵ်ႈၼၼ်ႉသေ ၵႂႃႇတႄႇပွၵ်ႈတေႃႇၵႂႃႇၵမ်းလဵဝ်ၶႃႈ။

```typescript
for (let i = 1; i <= 5; i++) {
  if (i === 3) continue; // ၶၢမ်ႈပွၵ်ႈ 3 ၵႂႃႇ
  console.log(i);
}
// ၽူၼ်မၼ်း: 1, 2, 4, 5
```
