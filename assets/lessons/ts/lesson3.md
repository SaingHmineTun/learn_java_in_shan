### **Lesson 3: The Workspace (Installing & Setting up VS Code)**

တွၼ်ႈတႃႇတေတႅမ်ႈ TypeScript ႁႂ်ႈမီးပၺ်ႇၺႃႇ လႄႈ ၽႂ်းၶိုၼ်းၼၼ်ႉ ႁဝ်းလူဝ်ႇမီး **Code Editor** ဢၼ်ၵႅၼ်ႇၶႅင်ၶႃႈ။ **Visual Studio Code (VS Code)** ပဵၼ်ဢၼ်လီသုတ်း တႃႇ TypeScript ယွၼ်ႉမၼ်းမီး "Smart Support" ဝႆႉၼႂ်းတူဝ်မၼ်းၶႃႈဢေႃႈ။

---

#### **1. VS Code ပဵၼ်သင်? (What is VS Code?)**
မၼ်းပဵၼ် "ပပ်ႉတႅမ်ႈလိၵ်ႈဢၼ်မီးပၺ်ႇၺႃႇ (Smart Notebook)" ၶႃႈ။
* **Built with TS:** VS Code ၼႆႉ ၶဝ်ဢဝ် TypeScript တႅမ်ႈဝႆႉ၊ ၵွပ်ႈၼၼ် မၼ်းၸင်ႇႁူႉၸႂ် TS လီသေပိူၼ်ႈၶႃႈ။
* **IntelliSense:** မၼ်းတေ "လၢမ်း" ပၼ်ဝႃႈ ၸဝ်ႈၵဝ်ႇတေတႅမ်ႈသင်တေႃႇ (Auto-complete)။
* **Real-time Error:** သင်တႅမ်ႈၽိတ်း မၼ်းတေမီး **"သဵၼ်ႈလႅင်"** ၶူတ်းဝႆႉတႂ်ႈၶူတ်ႉႁဝ်းၵမ်းလဵဝ်ၶႃႈ။

---

#### **2. ၶၵ်ႉတွၼ်ႈၵၢၼ်သႂ်ႇ (Installation Steps)**
1.  ၵႂႃႇတီႈဝႅပ်ႉသၢႆႉ [code.visualstudio.com](https://code.visualstudio.com/)။
2.  လူတ်ႇဢဝ် (Download) တူဝ်ဢၼ်မႅၼ်ႈၸွမ်းၶွမ်ႇၸဝ်ႈၵဝ်ႇ (Windows / Mac)။
3.  ပိုတ်ႇၾိုင််လ် ဢၼ်လူတ်ႇမႃးၼၼ်ႉသေ **Install** (Next > Next > Finish) ၵႂႃႇလႆႈငၢႆႈငၢႆႈၶႃႈ။



---

#### **3. ၵၢၼ်တင်ႈၵႃႈ (Extensions Setup)**
မိူဝ်ႈပိုတ်ႇ VS Code မႃးယဝ်ႉ ႁႂ်ႈသႂ်ႇ "ၶိူင်ႈၸွႆႈ (Extensions)" ၸိူဝ်းၼႆႉ တႃႇႁဵတ်းႁႂ်ႈၵၢၼ်ႁဵၼ်းငၢႆႈၶိုၼ်းၶႃႈ:

* **Prettier - Code formatter:** ၸွႆးၸတ်းပိူင်ၶူတ်ႉႁဝ်း ႁႂ်ႈႁၢင်ႈလီ လႄႈ ဢၢၼ်ႇငၢႆႈ (Format on Save)။
* **Error Lens:** ဢၼ်ၼႆႉ လီတႃႇ Beginner တႄႉတႄႉ! သင်ႁဝ်းတႅမ်ႈၽိတ်း မၼ်းတေတႅမ်ႈလိၵ်ႈလၢတ်ႈၼႄပၼ် တီႈႁၢင်သဵၼ်ႈၶူတ်ႉၼၼ်ႉၵမ်းလဵဝ်ဝႃႈ ၽိတ်းတီႈလႂ်။



---

#### **4. လၢႆးတႄႇၸႂ်ႉ (The Correct Way to Open)**
တႃႇတေႁႂ်ႈ VS Code ႁဵတ်းၵၢၼ်လႆႈလီသုတ်း ႁႂ်ႈႁဵတ်းၸွမ်းၼင်ႇၼႆၶႃႈ:
1.  သၢင်ႈ **Folder** မႂ်ႇၼိူဝ် Desktop (ၼင်ႇ `ts-learning`)။
2.  လၢၵ်ႈ (Drag & Drop) Folder ၼၼ်ႉ သႂ်ႇၼႂ်း VS Code။
3.  **Start Coding:** ၵမ်းၼႆႉ ၸင်ႇတႄႇသၢင်ႈၾိုင််လ် `.ts` ၼႂ်း VS Code ၶႃႈ။
