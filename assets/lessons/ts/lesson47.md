### **Lesson 47: Debugging in VS Code (Breakpoints and Watches)**

မိူဝ်ႈပရူဝ်ႇၵရႅမ်ႇႁဝ်းမီး Bug ဢမ်ႇၼၼ် ႁဵတ်းၵၢၼ်ၽိတ်းပိူင်၊ ၵၢၼ်ၸႂ်ႉ `console.log()` လၢႆလၢႆမၼ်းဢမ်ႇၵုမ်ႇထူၼ်ႈၶႃႈ။ VS Code မီးၶိူင်ႈၵျၢပ်ႈ **Debugger** ဢၼ်ၸွႆးႁႂ်ႈႁဝ်း "ၵိုတ်း" ၶူတ်ႉဝႆႉသေ ၸႅတ်ႈထတ်းတူၺ်းၶေႃႈမုၼ်းလႆႈၵူႈၸၼ်ႉၶႃႈဢေႃႈ။

---

#### **1. Breakpoints (ၵၢၼ်ၵိုတ်းၶူတ်ႉ)**
**Breakpoint** ပဵၼ်ၵၢၼ်မၢႆဝႆႉဝႃႈ "ႁႂ်ႈပရူဝ်ႇၵရႅမ်ႇၵိုတ်းတီႈ Line ၼႆႉ" ၼႆၶႃႈ။
* **လၢႆးၸႂ်ႉ:** ၵႂႃႇ Click တီႈၼႃႈ "Line Number" (တေမီးၸူတ်ႉသီလႅင်ဢွၵ်ႇမႃး)။
* **ၽွၼ်းလီ:** မိူဝ်ႈၶူတ်ႉလႅၼ်ႈမႃးထိုင်တီႈၼၼ်ႉ မၼ်းတေၵိုတ်းဝႆႉ ႁႂ်ႈႁဝ်းတူၺ်းၵႃႈၶၼ် (Values) ၶွင် Variable တင်းသဵၼ်ႈယၢမ်းၼၼ်ႉၶႃႈ။



---

#### **2. The Debug Toolbar (ၶိူင်ႈၵုမ်းထိင်း)**
မိူဝ်ႈပရူဝ်ႇၵရႅမ်ႇၵိုတ်းယဝ်ႉ ၸဝ်ႈၵဝ်ႇတေႁၼ် Buttons ၸိူဝ်းၼႆႉၶႃႈ:
* **Continue (F5):** လႅၼ်ႈတေႃႇၵႂႃႇထိုင် Breakpoint တူဝ်တေႃႇၵႂႃႇ။
* **Step Over (F10):** ၶၢမ်ႈၵႂႃႇ Line တေႃႇၵႂႃႇ (ဢမ်ႇၶဝ်ႈၵႂႃႇၼႂ်း Function)။
* **Step Into (F11):** "ၶဝ်ႈၵႂႃႇ" ၸႅတ်ႈထတ်းၼႂ်း Function ၼၼ်ႉ။
* **Step Out (Shift+F11):** ဢွၵ်ႇမႃးတီႈ Function ဢၼ်ႁဝ်းၸႅတ်ႈယူႇၼၼ်ႉ။

---

#### **3. Watches (ၵၢၼ်ၸွမ်းတူၺ်း)**
မၢင်ပွၵ်ႈႁဝ်းမီး Variable လၢင်တူဝ် ဢၼ်ႁဝ်းၶႂ်ႈ "ၸွမ်းတူၺ်း" ဝႃႈမၼ်းလႅၵ်ႈလၢႆႈၸိူဝ်ႉႁိုဝ်။
* **လၢႆးၸႂ်ႉ:** ၼႂ်း Debug Side Panel တေမီးတွၼ်ႈ **"WATCH"**။ ၸဝ်ႈၵဝ်ႇထႅမ်ၸိုဝ်ႈ Variable ၶဝ်ႈၵႂႃႇလႆႈၶႃႈ။
* **ၽွၼ်းလီ:** ၸဝ်ႈၵဝ်ႇတေႁၼ်ၵႃႈၶၼ်မၼ်းလႅၵ်ႈလၢႆႈၵႂႃႇ ၸွမ်းၼင်ႇႁဝ်း Step ၵႂႃႇၼၼ်ႉၶႃႈဢေႃႈ။



---

#### **4. Variables Panel**
ၼႂ်းတွၼ်ႈ **"VARIABLES"** ၼၼ်ႉ VS Code တေၼႄပၼ် Variable တင်းသဵၼ်ႈ ဢၼ်မီးၼႂ်း Scope ယၢမ်းၼၼ်ႉ။ ၸဝ်ႈၵဝ်ႇယင်းၸၢင်ႈ "မႄးၵႃႈၶၼ်" (Change Value) မၼ်းလႆႈၵမ်းလဵဝ် တႃႇတူၺ်းဝႃႈ သင်ၵႃႈၶၼ်လႅၵ်ႈၵႂႃႇ ပရူဝ်ႇၵရႅမ်ႇတေႁဵတ်းၵၢၼ်ၸိူဝ်ႉႁိုဝ်ၶႃႈ။

---

#### **5. Summary Checklist**
- [ ] ႁူႉလၢႆးသႂ်ႇ **Breakpoint** (Red dot) တီႈ Line ဢၼ်ၶႂ်ႈၸႅတ်း။
- [ ] ပွင်ႇၸႂ်ဝႃႈ **Step Over** လႄႈ **Step Into** ပႅၵ်ႇၵၼ်ၸိူဝ်ႉႁိုဝ်။
- [ ] ၸႂ်ႉ **Watch** တႃႇၸွမ်းတူၺ်း Variable ဢၼ်လွင်ႈယႂ်ႇ။
- [ ] ႁူႉလၢႆးၸႅတ်ႈထတ်း **Call Stack** တႃႇတူၺ်းဝႃႈ Function လႂ်ႁွင်ႉ Function လႂ်မႃး။
