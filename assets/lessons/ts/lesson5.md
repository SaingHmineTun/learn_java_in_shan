### **Lesson 5: The Project Brain (Deep Dive into tsconfig.json)**

မိူဝ်ႈပရူဝ်ႇၵျႅၵ်ႉႁဝ်းယႂ်ႇမႃး ႁဝ်းဢမ်ႇၸၢင်ႈမႃးၼဵၵ်း `tsc app.ts` ယူႇၵူႈၾိုင််လ်ၶႃႈ။ ႁဝ်းလူဝ်ႇမီးၾိုင််လ်တင်ႈၵႃႈ (Configuration) ဢၼ်ၼိုင်ႈ တႃႇၸီႉသင်ႇ Compiler ႁဝ်း ဢၼ်ႁွင်ႉဝႃႈ **`tsconfig.json`** ၼၼ်ႉၶႃႈဢေႃႈ။

---

#### **1. `tsconfig.json` ပဵၼ်သင်?**
မၼ်းပဵၼ်ၾိုင််လ် ဢၼ်မၢႆဝႆႉဝႃႈ Folder ၼႆႉ ပဵၼ်ပရူဝ်ႇၵျႅၵ်ႉ TypeScript ၶႃႈ။ မၼ်းတေမီးၼႃႈၵၢၼ်:
* ၸီႉသင်ႇဝႃႈ တေပိၼ်ႇ (Compile) ၾိုင််လ်လႂ်ၽေႈ။
* တေပိၼ်ႇပဵၼ် JavaScript Version လႂ် (Target)။
* တေဢဝ်ၾိုင််လ် JS ၵႂႃႇသိမ်းဝႆႉတီႈလႂ် (Output)။

---

#### **2. ၵၢၼ်သၢင်ႈ (How to Create)**
ပိုတ်ႇ Terminal ၼႂ်း Folder ပရူဝ်ႇၵျႅၵ်ႉၸဝ်ႈၵဝ်ႇသေ တႅမ်ႈဝႃႈ:
```bash
tsc --init
```
မၼ်းတေမီးၾိုင််လ် `tsconfig.json` ပေႃႇမႃး ဢၼ်မီးၶေႃႈမုၼ်း (Options) တင်းၼမ်ၶႃႈ။

---

#### **3. ၶေႃႈၵႂၢမ်းပိူင်လူင် ဢၼ်လူဝ်ႇႁူႉ (Core Options)**
ၼႂ်းၾိုင််လ်ၼၼ်ႉ ႁႂ်ႈမႄးတူၺ်း 4 တီႈၼႆႉၶႃႈ:

* **`target` (ပဝ်ႉမၢႆ):** `"target": "ES6"`. မၢႆထိုင်ဝႃႈ တေႁႂ်ႈမၼ်းပိၼ်ႇပဵၼ် JavaScript Version လႂ်။
* **`rootDir` (ငဝ်ႈငႃႇ):** `"rootDir": "./src"`. မၢႆထိုင်ဝႃႈ ၾိုင််လ် `.ts` တင်းၼမ် ဢၼ်ႁဝ်းတႅမ်ႈၼၼ်ႉ တေသိမ်းဝႆႉၼႂ်း Folder `src` ၶႃႈ။
* **`outDir` (တီႈဢွၵ်ႇ):** `"outDir": "./dist"`. မၢႆထိုင်ဝႃႈ မိူဝ်ႈမၼ်းပိၼ်ႇပဵၼ် `.js` ယဝ်ႉ ႁႂ်ႈမၼ်းၵႂႃႇသိမ်းဝႆႉၼႂ်း Folder `dist` ၶႃႈ။
* **`strict` (လွင်ႈၶႅၼ်ႉ):** `"strict": true`. သင်တင်ႈဝႃႈ `true` မၼ်းတေၸႅတ်ႈထတ်းၶူတ်ႉႁဝ်း ၶႅၼ်ႉၶႅၼ်ႉလႅပ်ႈလႅပ်ႈ။



---

#### **4. လၢႆးၸႂ်ႉ (How to Use)**
မိူဝ်ႈမီး `tsconfig.json` ယဝ်ႉ ႁဝ်းဢမ်ႇလူဝ်ႇတႅမ်ႈၸိုဝ်ႈၾိုင််လ်ယဝ်ႉၶႃႈ။ တႅမ်ႈၵမ်းလဵဝ်ဝႃႈ:
```bash
tsc
```
တူဝ် Compiler တေၵႂႃႇလူတ်ႇဢဝ်ၵႃႈ (Settings) ၼႂ်း `tsconfig.json` သေ ပိၼ်ႇပၼ်ၵူႈၾိုင််လ် ဢၼ်မီးၼႂ်း Folder ႁဝ်းၼၼ်ႉ ၵမ်းလဵဝ်ၶႃႈဢေႃႈ။

---

#### **5. Summary Checklist**
- [ ] သၢင်ႈ `tsconfig.json` လူၺ်ႈၶေႃႈၵႂၢမ်း `tsc --init` လႆႈယဝ်ႉ။
- [ ] ႁူႉၸႂ်လွင်ႈ `target`, `rootDir` လႄႈ `outDir` ယဝ်ႉ။
- [ ] ၸႂ်ႉၶေႃႈၵႂၢမ်း `tsc` (ဢမ်ႇသႂ်ႇၸိုဝ်ႈၾိုင််လ်) တႃႇပိၼ်ႇလိၵ်ႈၵူႈဢၼ်ယဝ်ႉ။
- [ ] တင်ႈၵႃႈ `strict: true` တႃႇႁႄႉၵႅတ်ႇလွင်ႈၽိတ်းယဝ်ႉ။
