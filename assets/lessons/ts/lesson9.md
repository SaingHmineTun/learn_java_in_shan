### **Lesson 9: The Danger Zone (any, unknown, and void)**

ၼႂ်း TypeScript ၼႆႉ မၼ်းမီး Type ၸိူဝ်းဢၼ်ႁဝ်းၸႂ်ႉမိူဝ်ႈႁဝ်း "ဢမ်ႇၼႄႇၸႂ်" လွင်ႈၶေႃႈမုၼ်း ဢမ်ႇၼၼ် မိူဝ်ႈႁဝ်းဢမ်ႇၶႂ်ႈႁႂ်ႈမၼ်းမီးၵႃႈၶၼ်သင်ၶႃႈ။ ၵူၺ်းၵႃႈ လူဝ်ႇလႆႈယၢင်ႇသတိလီလီ ယွၼ်ႉမၼ်းၸၢင်ႈႁဵတ်းႁႂ်ႈ "လွင်ႈၵႅတ်ႇၶႄ" (Safety) ႁဝ်းႁၢႆလႆႈၶႃႈဢေႃႈ။

---

#### **1. `any` (တူဝ်လႂ်ၵေႃႈလႆႈ - Danger!)**
`any` ၼႆႉ မိူၼ်ၼင်ႇၵၢၼ်ပိတ်းပႅတ်ႈ TypeScript ၶႃႈ။ သင်ႁဝ်းၸႂ်ႉမၼ်း၊ ႁဝ်းတေၸၢင်ႈသႂ်ႇၶေႃႈမုၼ်းမဵဝ်းလႂ်ၵေႃႈလႆႈ၊ လႅၵ်ႈလၢႆႈပဵၼ်သင်ၵေႃႈလႆႈ၊ တူဝ် Compiler တေဢမ်ႇၸႅတ်ႈထတ်းပၼ်သင်ယဝ်ႉ။

* **ၵွပ်ႈသင်လႄႈလီၵူဝ်:** မၼ်းႁဵတ်းႁႂ်ႈ Bug ၶဝ်ႈမႃးလႆႈငၢႆႈ။ ႁဝ်းတေၸႂ်ႉမၼ်းမိူဝ်ႈ "ဢမ်ႇမီးတၢင်းလိူၵ်ႈယဝ်ႉ" ၼၼ်ႉၵူၺ်းၶႃႈ။
```typescript
let data: any = 10;
data = "Hello"; // ❌ မႅၼ်ႈယူႇ (ၵူၺ်းၵႃႈဢမ်ႇလီၸႂ်ႉ)
data = true;    // ❌ မႅၼ်ႈယူႇ
```

---

#### **2. `unknown` (ဢမ်ႇပႆႇႁူႉ - Safer Version)**
`unknown` ၼႆႉ မိူၼ်ၼင်ႇ `any` ယူႇ၊ ၵူၺ်းၵႃႈ မၼ်းမီးလွင်ႈၵႅတ်ႇၶႄလိူဝ်ၶႃႈ။ ႁဝ်းၸၢင်ႈသႂ်ႇၶေႃႈမုၼ်းသင်ၵေႃႈလႆႈ၊ ၵူၺ်းၵႃႈ မိူဝ်ႈႁဝ်းတေဢဝ်မၼ်းၵႂႃႇၸႂ်ႉၼၼ်ႉ TypeScript တေၸီႉသင်ႇႁႂ်ႈႁဝ်း **"ၸႅတ်ႈထတ်း (Check)"** ဢွၼ်တၢင်းၶႃႈ။



```typescript
let value: unknown = "Sai Mao";

// value.toUpperCase(); ❌ Error: တူဝ် Compiler ဢမ်ႇပႆႇပၼ်ၸႂ်ႉ

if (typeof value === "string") {
    console.log(value.toUpperCase()); // ✅ မႅၼ်ႈယဝ်ႉ ယွၼ်ႉႁဝ်းၸႅတ်ႈယဝ်ႉ
}
```

---

#### **3. `void` (ဢမ်ႇမီးသင် - Empty)**
`void` ၼႆႉ ၵႆႉၸႂ်ႉၸွမ်း **Function** (ၵၢၼ်ႁဵတ်းၵၢၼ်) ၶႃႈ။ မၼ်းမၢႆထိုင်ဝႃႈ Function ၼၼ်ႉ တေႁဵတ်းၵၢၼ်ၵႂႃႇလၢႆလၢႆ၊ ၵူၺ်းၵႃႈ မၼ်းတေဢမ်ႇသူင်ႇၵႃႈၶၼ် (Return Value) သင်ၶိုၼ်းမႃးၶႃႈ။

```typescript
function logMessage(msg: string): void {
    console.log("Message: " + msg);
    // ဢမ်ႇမီး return statement တီႈၼႆႉ
}
```

#### **4. Summary Checklist**
- [ ] ႁူႉဝႃႈ `any` ၼႆႉ ပိတ်းပႅတ်ႈလွင်ႈၵႅတ်ႇၶႄ (Type Safety)။
- [ ] ၸႂ်ႉ `unknown` တႅၼ်း `any` သင်ဝႃႈဢမ်ႇၼႄႇၸႂ်လွင်ႈ Type မၼ်း။
- [ ] ႁူႉဝႃႈ `void` မၢႆထိုင် "ဢမ်ႇမီးၵႃႈၶၼ်သူင်ႇၶိုၼ်း" (No return value)။
