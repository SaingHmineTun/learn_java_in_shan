## Lesson 4: The Declarative Mindset

မိူဝ်ႈပၢၼ်ၵဝ်ႇ (XML) လႄႈ ပၢၼ်မႂ်ႇ (Compose) ၼႆႉ မၼ်းမီးလွင်ႈပႅၵ်ႇပိူင်ႈၵၼ် ၼင်ႇ "ၵၢၼ်သင်ႇ" လႄႈ "ၵၢၼ်လၢတ်ႈၼႄ" ၶႃႈ။

### 1. Imperative (ပၢၼ်ၵဝ်ႇ - ၵၢၼ်သင်ႇ)
မိူၼ်ၼင်ႇ ၸဝ်ႈၵဝ်ႇသင်ႇလုၵ်ႈႁဵၼ်းဝႃႈ:
1. "လုၵ်ႉၶိုၼ်ႈ"
2. "ၵႂႃႇတီႈၽၵ်းတူ"
3. "ပိုတ်ႇၽၵ်းတူ"
* **လွင်ႈယၢပ်ႇ:** ပေႃးမီးလွင်ႈလႅၵ်ႈလၢႆႈသေလွင်ႈလွင်ႈ (မိူၼ်ၼင်ႇ ၽၵ်းတူပိုတ်ႇဝႆႉယဝ်ႉ) ၼႆ၊ Code ႁဝ်းၸၢင်ႈပဵၼ် "Error" ၶႃႈ။ ႁဝ်းလူဝ်ႇလႆႈၸတ်းၵၢၼ်ၵူႈလွင်ႈလွင်ႈ (FindViewByID, SetText, SetVisibility) ႁင်းၶေႃၶႃႈ။

### 2. Declarative (ပၢၼ်မႂ်ႇ - ၵၢၼ်လၢတ်ႈၼႄ)
မိူၼ်ၼင်ႇ ၸဝ်ႈၵဝ်ႇလၢတ်ႈၼႄသၽႃႇဝဝႃႈ:
* "ၽၵ်းတူၼႆႉ မၼ်းတေလႆႈ ပိုတ်ႇ ဝႆႉ"
* **လွင်ႈငၢႆႈ:** ႁဝ်းၵူၺ်းလၢတ်ႈဝႃႈ UI ႁဝ်း "တေလႆႈပဵၼ်ၸိူင်ႉၼႆ" (What it should look like)။ ပေႃးမီးၶေႃႈမုၼ်း (State) လႅၵ်ႈလၢႆႈ၊ Compose တေမႄးတႅမ်ႈ (Recompose) ၼႃႈၸေႃးႁင်းမၼ်း ႁႂ်ႈၵူပ်းၵၼ်တင်း ၶေႃႈမုၼ်းမႂ်ႇၶႃႈ။



### 3. State လႄႈ UI
ၼႂ်း Compose ၼႆႉ **State** (ၶေႃႈမုၼ်း) ပဵၼ် "ၵူၼ်းႁေႃႈလေႃႉ" ၶႃႈ။
* ပေႃး **State** လႅၵ်ႈလၢႆႈ -> **UI** တေလႅၵ်ႈလၢႆႈၸွမ်း။
* ႁဝ်းဢမ်ႇလူဝ်ႇၵႂႃႇသင်ႇဝႃႈ `textView.text = "New Message"`။ ႁဝ်းၵူၺ်းလႅၵ်ႈလၢႆႈ Variable ႁဝ်း၊ ယဝ်ႉၵေႃႈ Compose တေဢဝ်ၵႂႃႇၼႄပၼ်ႁင်းမၼ်းၶႃႈ။

### 4. Anatomy ၶွင် Composable Function
```kotlin
@Composable
fun ProverbText(text: String) {
    // ႁဝ်းလၢတ်ႈဝႃႈ ႁႂ်ႈၼႄလိၵ်ႈ (Text) ဢၼ်မီးသီလိူင်
    Text(
        text = text,
        color = Color.Yellow
    )
}
```
* ၸဝ်ႈၵဝ်ႇတေႁၼ်ဝႃႈ ႁဝ်းဢမ်ႇလႆႈ "Create" Object ဢမ်ႇၼၼ် "Return" သင်။ ႁဝ်းၵူၺ်း "လၢတ်ႈၼႄ" (Declare) ဝႃႈ ႁဝ်းၶႂ်ႈႁၼ်သင်ၼိူဝ်ၼႃႈၸေႃးၾူၼ်းၵူၺ်းၶႃႈ။
