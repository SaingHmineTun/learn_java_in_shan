## Lesson 5: Core Layouts (Column, Row, Box)

ပေႃးႁဝ်းတေတႅမ်ႈ App "Proverb of the Day" ၼၼ်ႉ ႁဝ်းလူဝ်ႇႁူႉဝႃႈ တေဢဝ်လိၵ်ႈလႄႈႁၢင်ႈ ဝႆႉတီႈလႂ်ၶႃႈ။

### 1. Column (ၵၢၼ်ၸတ်း ဝၢင်းလူင်း)
`Column` ၼႆႉ ၸႂ်ႉတႃႇဢဝ် UI Component ၸိူဝ်းၼၼ်ႉ ဝၢင်းသိုပ်ႇၵၼ်လူင်းၵႂႃႇ ၽၢႆႇတႂ်ႈၶႃႈ (Vertical)။
* **မိူၼ်ၼင်ႇ:** ႁဝ်းမီး လိၵ်ႈတႆး (Proverb) ဝႆႉတၢင်းၼိူဝ်၊ ယဝ်ႉၵေႃႈ လိၵ်ႈဢင်းၵိတ်း (Meaning) ဝႆႉၽၢႆႇတႂ်ႈမၼ်းၶႃႈ။



### 2. Row (ၵၢၼ်ၸတ်း ဝၢင်းၶၢင်ႈ)
`Row` ၼႆႉ ၸႂ်ႉတႃႇဢဝ် UI Component ၸိူဝ်းၼၼ်ႉ ဝၢင်းသိုပ်ႇၵၼ်ၵႂႃႇ ၽၢႆႇၶႂႃၶႃႈ (Horizontal)။
* **မိူၼ်ၼင်ႇ:** ႁဝ်းမီး Icon ဢၼ်ၼိုင်ႈ ဝႆႉၽၢႆႇသၢႆႉ၊ ယဝ်ႉၵေႃႈ မီးၸိုဝ်ႈ App ဝႆႉၽၢႆႇၶႂႃမၼ်းၶႃႈ။



### 3. Box (ၵၢၼ်ၸတ်း ဝၢင်းသိုပ်ႇ)
`Box` ၼႆႉ ၸႂ်ႉတႃႇဢဝ် Component ဝၢင်းသွၼ်ႉ (Overlay) ၵၼ်ၶႃႈ။ ဢၼ်တႅမ်ႈဢွၼ်တၢင်း တေယူႇၽၢႆႇတႂ်ႈ၊ ဢၼ်တႅမ်ႈလိုၼ်းလင် တေယူႇတၢင်းၼိူဝ်ၶႃႈ။
* **မိူၼ်ၼင်ႇ:** ႁဝ်းမီး ႁၢင်ႈ (Image) ဝႆႉပဵၼ် Background၊ ယဝ်ႉၵေႃႈ ဢဝ် လိၵ်ႈ (Text) ဝၢင်းသွၼ်ႉဝႆႉတၢင်းၼိူဝ် ႁၢင်ႈၼၼ်ႉၶႃႈ။



---

### Anatomy ၶွင် Layout
တူၺ်း Code တူဝ်ယၢင်ႇဢၼ်ၼႆႉၶႃႈ:

```kotlin
@Composable
fun ProverbLayout() {
    // ၸႂ်ႉ Column တႃႇဝၢင်းလိၵ်ႈသွင်ထႅဝ် သိုပ်ႇၵၼ်လူင်းၵႂႃႇ
    Column {
        Text("ၵူၼ်းႁူႉဢမ်ႇၼမ်")
        Text("ၵူၼ်းၵမ်ဢမ်ႇႁၢႆး")
        
        // ၼႂ်း Column ၼၼ်ႉ ႁဝ်းၸႂ်ႉ Row တႃႇဝၢင်း Icon လႄႈ လိၵ်ႈ သိုပ်ႇၵၼ်ၵႂႃႇၽၢႆႇၶႂႃ
        Row {
            Icon(Icons.Default.Star, contentDescription = null)
            Text("Shan Wisdom")
        }
    }
}
```
