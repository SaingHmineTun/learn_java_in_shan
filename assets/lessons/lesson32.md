
## **Java Basic 32**

#### **CREATING INSTANCE OF A CLASS AND CALLS ITS ATTRIBUTES**

Link - <https://youtu.be/aeJQFNzidK0>

ႁဝ်းမႃးၸႂ်ႉတိုဝ်း class သေ ႁဝ်းတေၶူင်ႈလႆႈ object ဢိူဝ်ႈ။ Class ၼႆႉ မၼ်းၵေႃႈပွင်ႇဝႃႈ ပဵၼ် Reference Data Type ဢၼ်ၼိုင်ႈၵူၺ်းၼင်ႇဢဝ်ႇ။ တႃႇတေၶူင်ႇ variable ၸႂ်ႉတိုဝ်း Reference Data Type ၼႆ ႁဝ်းတေလႆႈၸႂ်ႉတိုဝ်း new keyword ဢိူဝ်ႈ။ ပေႃးၼၼ် ႁဝ်းတေလႆႈမႃး object သေ ႁဝ်းတေၸၢင်ႊၸႂ်ႉတိုဝ်းလႆႈ state လႄႈ behavior တီႈၼႂ်းမၼ်းယဝ်ႉ။

###### လၢႆးၶူင်သၢင်ႈ

```
Nrc nrc = new Nrc(); // Create an object using NRC class and name it nrc
nrc.name = "Sai Mao"; // Assign new value to name state in nrc object
nrc.location = "Mu-se";
String location = nrc.location; // Read state's value of nrc object
```
