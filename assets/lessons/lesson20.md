
## **Java Basic 20**

#### **SWITCH CONDITIONAL STATEMENT**

Link - <https://youtu.be/aUl8zq5knGA>

switch ၵေႃႈ မၼ်းတေႁမ်ႉငၢႆးၵၼ်တင်း if ယူႇ။ ၵူၺ်းၵႃႊ မၼ်းတေဢမ်ႇၸႅတ်ႈလူၺ်ႈ boolean လႄႈ မၼ်းတေၸႅတ်ႈ ၵႃႊၶၼ်မၼ်းၵမ်းသိုဝ်ႊဢိူဝ်ႈ။ လွၵ်းလၢႆးႁဝ်းတႅမ်ႈ switch တီႈ Java ၼႆႉတေႉ မၼ်းတေမီးဝႆႉယူႇ 2 လွင်ႈ။ သွင်လွင်ႈၼႆႉၵေႃႈ ႁဵတ်ႉၵၢၼ်မိူၼ်ၵၼ်ၵူၺ်း သေတႃႉ လၢႆးတႅမ်ႈမၼ်း တေငၢႆးၵၼ်ယူႇၵမ်ႈၽွင်ႈ။

တီႈ switch ၼႆႉ သင်ႁဝ်းၶႆႈၸႅတ်ႈတူၺ်း မိူၼ်ၼင်ႇ ႁႂ်ႈယႂ်ႇမိူၼ် 10 လႄႈ ႁႂ်ႈလဵၵ်ႉမိူၼ် 15 ၼႆၸိုင် ၵွပ်ႈဢမ်ႇၸႂ်ႉလႆႈ relational ၊ logical operators လႄႈ ႁဝ်းတေလႆႈ ၸႅတ်ႈပၼ် ၵမ်းတူဝ်တူဝ်၊ ဢဝ်ၶွမ်ႇမႃႇ (comma) သေၶၢမ်ႈပၼ်ယဝ်ႉ။ သင်ဝႃႈ ဢၼ်ၸႅတ်ႈဝႆႉၼႆႉ ဢမ်ႇထုၵ်ႇမႅၼ်ႈသေဢၼ်ၼႆ တီႈ if ၼႆႉ မၼ်းမီးဝႆႉ else လႄႈတီႈ switch သမ်ႉမီးဝႆႉ default ဢိူဝ်ႈ။

**1\. switch - case - break**

လၢႆးၼႆႉ ႁဝ်းၸႂ်ႉတိုဝ်း keyword 3 ဢၼ်သေ ပဵၼ်လၢႆးၵဝ်ႇၵွၼ်ႇ။ ႁဝ်းတေလႆႈ ၸႂ်ႉၵိုၵ်းၸွမ်း break ဢိူဝ်ႈ။ သင်ဢမ်ႇၼၼ် မၼ်းတေ ဢမ်ႇၵိုတ်ႉသေ တေသိုပ်ႇႁဵတ်ႉၵၢၼ်ၵႂႃႇတိုၵ်ႉတိုၵ်ႉ။ ယဝ်ႉၵေႃႈ လွင်ႈလၢၵ်ႇလၢႆးမၼ်းသမ်ႉ body မၼ်းၼၼ်ႉ ႁဝ်းၸႂ်ႉ : (full column) ယဝ်ႉ။

###### လၢႆးၸႂ်ႉတိုဝ်း

```
switch (value) {
    case 10, 11, 12, 13, 14, 15:
        System.out.println("သူတိုၵ်ႉၵေႃႉလဵၵ်ႉယူႇ");
        break;
    case 20:
        System.out.println("သူဢမ်ႇလဵၵ်ႉယဝ်ႉ");
        break;
    case 30:
        System.out.println("သူၵေႃႉယႂ်ႇယဝ်ႉ။");
        break;
    default:
        System.out.println("ဢမ်ႇႁူႉယဝ်ႉ");
}
```

**switch - arrow - case**

လၢႆးၼႆႉတေႉ ၸႂ်ႉတိုဝ်း keyword 2 ဢၼ်ၵူၺ်းလႄႈ ပဵၼ်လၢႆးမႂ်ႇ ဢၼ်ပႃးၸွမ်း jdk 14 ယဝ်ႉ။ တႃႇၸႅတ်ႈတူၺ်း ဢၼ်ယဝ်ႉဢၼ်ၼၼ်ႉ မၼ်းၸႂ်ႉတိုဝ်း ဢၼ်ႁွင်ႉဝႃႈ arrow လႄႈ ၽၢႆႇလင်မၼ်းသမ်ႉ ႁဝ်းတႅမ်ႈ တူင်ႇၶဝ်ႈသွႆး ၸွမ်းယဝ်ႉ။ ၵွပ်ႈၼႆ body မၼ်းၼႆႉ တေမိူၼ်ၵၼ်တၢင်း if, for ၵႂႃႇယူႇ။ ၵၢင်ၸႂ်ၶႃႈတေႉ ၶႃႈလႆႈၸႂ် လၢႆးၼႆႉတိူဝ်းဢိူဝ်ႈ။ လွင်ႈလီမၼ်းၼႆႉသမ်ႉ ႁဝ်းဢမ်ႇလူဝ်ႇၸႂ်ႉၵိုၵ်ႉ break ယဝ်ႉ။

###### လၢႆးၸႂ်ႉတိုဝ်း

```
switch (value) {
    case 10, 11, 12, 13, 14, 15 -> System.out.println("သူတိုၵ်ႉၵေႃႉလဵၵ်ႉယူႇ");
    case 20 -> System.out.println("သူဢမ်ႇလဵၵ်ႉယဝ်ႉ");
    default -> {
        System.out.println("ဢမ်ႇႁူႉယဝ်ႉ");
    }
}
```

တီႈ if, for, arrow switch ဢၼ်လႆႈၸႂ်ႉ တူင်ႇၶဝ်ႈသွႆးဝႆႉၼၼ်ႉ သင်ၸိူဝ်ႉၼႂ်း body မၼ်းႁဝ်းတႅမ်ႈဝႆႉ code ထႅဝ်လဵဝ်ၵူၺ်းၼႆၸိုင် ႁဝ်းဢမ်ႇတႅမ်ႈ {} မၼ်းၵေႃႈလႆႈယဝ်ႉ။
