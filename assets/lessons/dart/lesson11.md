## 11) The 'dynamic' type: When to use it

သင်ႁဝ်းၶႂ်ႈလႅၵ်ႈလၢႆႈ Type ၶေႃႈမုၼ်း ၼႂ်း Variable ဢၼ်လဵဝ်ၵၼ်လႆႈယူႇတႃႇသေႇၼၼ်ႉ ႁဝ်းၸၢင်ႈၸႂ်ႉ **`dynamic`** ယဝ်ႉ။ မၼ်းပဵၼ် Type ဢၼ် "ယႅၼ်ႇ" (Flexible) သုတ်းၼႂ်း Dart။

### 1. လၢႆးႁဵတ်းၵၢၼ် 'dynamic'

မိူဝ်ႈႁဝ်းပိုၼ်ၽၢဝ်ႇ Variable ပဵၼ် `dynamic` ၼၼ်ႉ Dart တေဢမ်ႇထတ်းသၢင် Type မၼ်းမိူဝ်ႈၽွင်း Compile (Compile-time check)။ ႁဝ်းသႂ်ႇၵႃႈၶၼ်သင်ၵေႃႈလႆႈ၊ လႅၵ်ႈလၢႆႈပဵၼ် Type တၢင်ႇဢၼ်ၵေႃႈလႆႈ တူၵ်းလွၵ်းလႅၵ်ႈယူႇတႃႇသေႇ။

```dart
dynamic x = 100;      // တႄႇလူၺ်ႈ int
x = "Now a String";   // လႅၵ်ႈပဵၼ် String ၵေႃႈလႆႈ (ဢမ်ႇ Error ၼင်ႇ var)
x = true;             // လႅၵ်ႈပဵၼ် bool ၵေႃႈလႆႈထႅင်ႈ

```

### 2. 'var' vs 'dynamic'

* **`var`**: မိူဝ်ႈသႂ်ႇၵႃႈၶၼ်ယဝ်ႉ Type မၼ်းတေထုၵ်ႇ Lock ဝႆႉ။ လႅၵ်ႈပဵၼ် Type တၢင်ႇဢၼ်ဢမ်ႇလႆႈ။
* **`dynamic`**: လႅၵ်ႈလၢႆႈ Type လႆႈၵူႈၶိင်ႇ။

### 3. မိူဝ်ႈလႂ်ထုၵ်ႇၸႂ်ႉ?

* ၸႂ်ႉမိူဝ်ႈႁဝ်းဢမ်ႇႁူႉတႅတ်ႈတေႃးဝႃႈ ၶေႃႈမုၼ်းဢၼ်တေလုၵ်ႉတီႈ Network API ဢမ်ႇၼၼ် JSON မႃးၼၼ်ႉ တေပဵၼ် Type သင်တေႉတေႉ။
* ၸႂ်ႉမိူဝ်ႈႁဝ်းလူဝ်ႇႁဵတ်းၵၢၼ်ၸွမ်း Object ဢၼ်မီး Type လၢႆလၢႆပိူင်ပူၼ်ႇၵၼ်။

### 4. ၶေႃႈသတိ (The Danger)

ယႃႇၸႂ်ႉ `dynamic` ပူၼ်ႉတီႈ ယွၼ်ႉမၼ်းတေႁဵတ်းႁႂ်ႈ App ႁဝ်း "ဢမ်ႇၼိမ်" (Less safe)။

* Dart တေဢမ်ႇၸွႆႈထတ်းသၢင် Error ပၼ်မိူဝ်ႈႁဝ်းတႅမ်ႈ Code (Compile-time)။
* သင်ႁဝ်းႁွင်ႉၸႂ်ႉ Function ဢၼ်ဢမ်ႇမီးၼႂ်း Type ၼၼ်ႉ (တူဝ်ယၢင်ႇ: ဢဝ် Number ၵႂႃႇႁဵတ်း `.toUpperCase()`) မၼ်းတေၵႂႃႇ **Crash** (App တၢႆ) မိူဝ်ႈ App လႅၼ်ႈယူႇ (Runtime) ၼၼ်ႉယဝ်ႉ။
