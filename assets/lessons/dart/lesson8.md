## 8) Numbers: int and double

ၼႂ်း Dart ၼၼ်ႉ ၶေႃႈမုၼ်းဢၼ်ပဵၼ်မၢႆၼပ်ႉ (Numbers) တေမီးယူႇ 2 ပိူင်ပိူင်ယဝ်ႉ။ တင်းသွင်ဢၼ်ၼႆႉ လုၵ်ႉတီႈ Parent Class ဢၼ်ၸိုဝ်ႈဝႃႈ `num` မႃးမိူၼ်ၵၼ်ယဝ်ႉ။

### 1. int (Integer)

ၸႂ်ႉတႃႇသိမ်း မၢႆၼပ်ႉဢၼ်ပဵၼ် "တူဝ်ၵူၼ်ႇ" (Whole numbers)။ မၼ်းဢမ်ႇမီးၸုၵ်း (Decimal point)။

* တူဝ်ယၢင်ႇ: `1`, `500`, `-10`, `0`။

```dart
int bottles = 5;
int year = 2024;

```

### 2. double (Double-precision floating-point)

ၸႂ်ႉတႃႇသိမ်း မၢႆၼပ်ႉဢၼ်မီး "ၸုၵ်း" (Fractional numbers)။ ၼႆႉပဵၼ်မၢႆၼပ်ႉ 64-bit float ယဝ်ႉ။

* တူဝ်ယၢင်ႇ: `3.14`, `10.5`, `-0.99`။

```dart
double price = 1500.50;
double gravity = 9.8;

```

### 3. Number Methods ဢၼ်ၸႂ်ႉၼမ်

မိူဝ်ႈႁဝ်းၸႂ်ႉ Variables ဢၼ်ပဵၼ် Number ၼၼ်ႉ Dart မီး Built-in methods တႃႇၸွႆႈၵိတ်ႇၼပ်ႉလၢႆလၢႆမၢင်:

* **`abs()`**: ပဵၼ်ၵၢၼ်ဢဝ်ၵႃႈၶၼ် Positive (Absolute value)။ တူဝ်ယၢင်ႇ: `-5.abs()` တေလႆႈ `5`။
* **`round()`**: ပဵၼ်ၵၢၼ်ဢဝ်မၢႆၼပ်ႉၸုၵ်း ႁဵတ်းပဵၼ်မၢႆၼပ်ႉတူဝ်ၵူၼ်ႇ ဢၼ်ၸမ်သုတ်း။
* **`ceil()`**: ပဵၼ်ၵၢၼ် "ပတ်ႉၶိုၼ်း" ႁႂ်ႈပဵၼ်မၢႆၼပ်ႉတူဝ်ၵူၼ်ႇဢၼ်ယႂ်ႇလိူဝ် (Ceiling)။ တူဝ်ယၢင်ႇ: `3.1.ceil()` တေလႆႈ `4`။
* **`floor()`**: ပဵၼ်ၵၢၼ် "ပတ်ႉလူင်း" ႁႂ်ႈပဵၼ်မၢႆၼပ်ႉတူဝ်ၵူၼ်ႇဢၼ်လဵၵ်ႉလိူဝ် (Floor)။ တူဝ်ယၢင်ႇ: `3.9.floor()` တေလႆႈ `3`။
* **`remainder()`**: တႃႇႁႃ "ဢၼ်လိူဝ်" (Remainder) မိူဝ်ႈဢဝ်ၵႂႃႇႁၢၼ်။

### 4. ၶေႃႈလမ်ႇလွင်ႈ

တီႈၼႂ်း Dart 2.1 ၶိုၼ်းၼႃႈၼၼ်ႉ သင်ႁဝ်းပိုၼ်ၽၢဝ်ႇ `double` သေ သႂ်ႇၵႃႈၶၼ်ပဵၼ် `int` ၵေႃႈ မၼ်းတေပိၼ်ႇပဵၼ် double ပၼ်ႁင်းၵူၺ်းယဝ်ႉ။

```dart
double distance = 10; // Dart တေမၢႆဝႆႉပဵၼ် 10.0

```
