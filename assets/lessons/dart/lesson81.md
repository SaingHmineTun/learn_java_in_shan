## 81) Event Loop and Single-Threaded Execution

တွၼ်ႈတႃႇတေပွင်ႇၸႂ်ဝႃႈ `async` လႄႈ `await` ႁဵတ်းၵၢၼ်ၸိူဝ်ႉႁိုဝ်ၼၼ်ႉ ႁဝ်းလူဝ်ႇႁူႉဝႃႈ Dart "ပၼ်ႇၵၢၼ်" ၸိူဝ်ႉႁိုဝ်ဢၼ်ဝႃႈၼၼ်ႉဢွၼ်တၢင်းၶႃႈ။

### 1. Single-Threaded (ၵူၼ်းႁဵတ်းၵၢၼ်ၵူၼ်းလဵဝ်)

Dart ၼႆႉပဵၼ် Programming Language ဢၼ်ႁဵတ်းၵၢၼ်လၢႆး **Single-Threaded**။ မၼ်းမၢႆထိုင်ဝႃႈ မၼ်းမီး "ၵူၼ်းႁဵတ်းၵၢၼ်" (Isolate) **ၵူၼ်းလဵဝ်ၵူၺ်း** ဢၼ်တေၸတ်းၵၢၼ် Code ႁဝ်းတင်းသဵင်ႈ။ မၼ်းဢမ်ႇၸၢင်ႈႁဵတ်း Code သွင်ထႅဝ်ၼႂ်းပွၵ်ႈလဵဝ်ၵၼ်လႆႈ။

### 2. The Event Loop (ဝူတ်ႇမူၼ်းၵၢၼ်)

ပေႃးမီးၵူၼ်းႁဵတ်းၵၢၼ်ၵူၼ်းလဵဝ်၊ ႁဵတ်းႁႂ်ႈ App ႁဝ်းတူင်ႉၼိုင်လႆႈလၢႆလၢႆယၢင်ႇ (မိူၼ်ၼင်ႇ လူတ်ႇၶႅပ်းႁၢင်ႈ ၵႂႃႇၸွမ်းတွၼ်ႈ ဢၼ်ႁဝ်းတိုၵ်ႉတဵၵ်း Button ယူႇ) ၼႂ်းပွၵ်ႈလဵဝ်ၵၼ်လႆႈၸိူဝ်ႉႁိုဝ်? ၶေႃႈတွပ်ႇမၼ်းၵေႃႈပဵၼ် **Event Loop** ယဝ်ႉ။

* **The Event Queue (ထႅဝ်ၵၢၼ်):** မိူၼ်ၼင်ႇ "ထႅဝ်ပႂ်ႉၵၢၼ်" ဢၼ်မီးၵၢၼ်လၢႆလၢႆမၢႆ (Events) တႅၼ်းဝႆႉ။ မိူၼ်ၼင်ႇ: User တဵၵ်း Button၊ ၵၢၼ်တွပ်ႇမႃးတီႈ Internet၊ ဢမ်ႇၼၼ် Timer သဵင်ႈၶၢဝ်းယၢမ်း။
* **The Event Loop (ဝူတ်ႇမူၼ်း):** မၼ်းပဵၼ် Loop ဢၼ်ဢမ်ႇမီးဝၼ်းသုတ်း။ မၼ်းတေၵႂႃႇတူၺ်းတီႈ **Event Queue** ဝႃႈ "မီးၵၢၼ်သင်လူဝ်ႇႁဵတ်းထႅင်ႈယူႇႁႃႉ?" ပေႃးမီး မၼ်းတေယိပ်းဢဝ်မႃးႁဵတ်း၊ ပေႃးယဝ်ႉ ၸင်ႇၵႂႃႇယိပ်းဢဝ်ၵၢၼ်ထႅင်ႈဢၼ်ၼိုင်ႈ မႃးႁဵတ်းတေႃႇယူႇတႃႇသေႇ။

### 3. Blocking (လွင်ႈၸၢင်ႈၵၢင်ႉ)

ယွၼ်ႉ Dart မီးၵူၼ်းႁဵတ်းၵၢၼ်ၵူၼ်းလဵဝ်၊ သင်ႁဝ်းပၼ်ၵၢၼ်ဢၼ် "ႁိုင်ၼႃႇ" (မိူၼ်ၼင်ႇ ၼပ်ႉမၢႆ 1 တေႃႇ 10 လၢၼ်ႉ) ၼႂ်းထႅဝ်ၵၢၼ်ယူဝ်းယူဝ်း၊ ၵူၼ်းႁဵတ်းၵၢၼ်ၼၼ်ႉတေ "တိတ်း" (Stuck) ယူႇတီႈၵၢၼ်ၼၼ်ႉၵူၺ်း။

* **ပၼ်ႁႃ:** ယွၼ်ႉမၼ်းတိုၵ်ႉတိတ်းၵၢၼ်ၼပ်ႉမၢႆယူႇ၊ မၼ်းဢမ်ႇၸၢင်ႈၵႂႃႇယိပ်းဢဝ်ၵၢၼ် "Paint UI" (ၵၢၼ်ပၼ်ႇ Screen) မႃးႁဵတ်းလႆႈ။ ၼႆႉယဝ်ႉပဵၼ်လွင်ႈတၢင်းဢၼ်ႁဵတ်းႁႂ်ႈ App ႁဝ်း **Freeze (ၵၢင်ႉ)** ၵႂႃႇၼၼ်ႉၶႃႈ။

---

### **Key Takeaway**

* **Isolate**: ပဵၼ်တူဝ်ၵူၼ်းႁဵတ်းၵၢၼ် (Memory + Thread)။
* **Event Loop**: ပဵၼ်လၢႆးၸတ်းၵၢၼ် ႁႂ်ႈၵူၼ်းႁဵတ်းၵၢၼ်ၵူၼ်းလဵဝ် ႁဵတ်းၵၢၼ်လႆႈလၢႆလၢႆဢၼ်လူၺ်ႈဢမ်ႇၵိုတ်းပႂ်ႉ။
