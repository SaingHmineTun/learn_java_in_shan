## 24) While and Do-While Loops

`while` လႄႈ `do-while` loop ၼႆႉ ၸႂ်ႉတႃႇႁဵတ်းၵၢၼ် "တူဝ်ႈၶိုၼ်း" (Repeat) မိူၼ်ၼင်ႇ `for` loop ၼင်ႇၵဝ်ႇ။ ၵူၺ်းၵႃႈ မၼ်းပႅၵ်ႇပိူင်ႈၵၼ်တီႈဝႃႈ ႁဝ်းတေၸႂ်ႉမၼ်းမိူဝ်ႈႁဝ်း **ဢမ်ႇႁူႉတႅတ်ႈတေႃး** ဝႃႈ တေလႆႈလႅၼ်ႈၵႂႃႇၵေႃႈပွၵ်ႈ (ႁဝ်းႁူႉၵူၺ်းဝႃႈ တေလႆႈလႅၼ်ႈတေႃႇၵႂႃႇ တေႃႇလွင်ႈတၢင်းတေၽိတ်း)။

---

### Loop Comparison Table

| Loop Type | Logic (ပၢႆးဝူၼ်ႉ) | Execution (ၵၢၼ်ႁဵတ်းၵၢၼ်) |
| --- | --- | --- |
| **`while`** | ထတ်းသၢင်လွင်ႈတၢင်း **ဢွၼ်တၢင်း** ႁဵတ်းၵၢၼ် | သင်လွင်ႈတၢင်းၽိတ်းတႄႇတူၼ်း၊ မၼ်းတေဢမ်ႇႁဵတ်းသင်သေဢၼ်။ |
| **`do-while`** | ႁဵတ်းၵၢၼ်ဢွၼ်တၢင်း **သေၸင်ႇ** ထတ်းသၢင် | မၼ်းတေႁဵတ်းၵၢၼ် **ဢယၢတ်ႈမၼ်း 1 ပွၵ်ႈ** တႃႇသေႇ။ |

---

### 1. The 'while' Loop

မၼ်းတေထတ်းသၢင် Condition သင်ပဵၼ် `true` မၼ်းတေလႅၼ်ႈ Code ၼႂ်းၼၼ်ႉ၊ ယဝ်ႉၵေႃႈတေပၼ်ႇၶိုၼ်းမႃးထတ်းသၢင်မႂ်ႇထႅင်ႈ။

**Example: Countdown**

```dart
int count = 3;

while (count > 0) {
  print("Count: $count");
  count--; // လူဝ်ႇမီးလွင်ႈလႅၵ်ႈလၢႆႈ ဢမ်ႇၼၼ် Loop တေဢမ်ႇၵိုတ်း (Infinite Loop)
}
print("Liftoff!");
/* Output:
Count: 3
Count: 2
Count: 1
Liftoff!
*/

```

---

### 2. The 'do-while' Loop

ပိူင်ၼႆႉတႄႉ မၼ်းတေလႅၼ်ႈ Code ၼႂ်း `do { ... }` ဢွၼ်တၢင်း ၼိုင်ႈပွၵ်ႈ၊ သေၸင်ႇၵႂႃႇတူၺ်း `while (...)` ဝႃႈ တေပၼ်ႇထႅင်ႈႁႃႉ။

**Example: Run at least once**

```dart
int i = 5;

do {
  print("Current value: $i");
  i++;
} while (i < 5);

// Output: Current value: 5
// မၼ်းတေ print 5 ဢွၵ်ႇမႃး 1 ပွၵ်ႈ တင်းသေ i < 5 ပဵၼ် false ၵေႃႈယႃႇ

```

---

### 3. Infinite Loops (Loop ဢမ်ႇၵိုတ်း)

သင်ဝႃႈႁဝ်းလိုမ်းထႅမ်/လိုတ်းၵႃႈၶၼ် (Increment/Decrement) ဢမ်ႇၼၼ် သႂ်ႇ Condition ဢၼ်ပဵၼ် `true` တႃႇသေႇ၊ Loop ၼၼ်ႉတေလႅၼ်ႈၵႂႃႇဝႆးဝႆး တေႃႇ App ႁဝ်းတေ Crash ယဝ်ႉ။

**Example (Avoid this!):**

```dart
while (true) {
  print("This will never stop!");
  // လူဝ်ႇမီး 'break' ၼႂ်းၼႆႉ တႃႇတေဢွၵ်ႇလႆႈ
}

```

---

### 4. Summary: Which one to use?

* **`for`**: ၸႂ်ႉမိူဝ်ႈႁူႉဝႃႈတေလႅၼ်ႈ **လၢႆပွၵ်ႈ** (တူဝ်ယၢင်ႇ: ၼပ်ႉ 1-10)။
* **`while`**: ၸႂ်ႉမိူဝ်ႈထတ်းသၢင် **လွင်ႈတၢင်း** (တူဝ်ယၢင်ႇ: တေႃႇ User တေၼိပ်ႉမၢႆ 0 ၸင်ႇၵိုတ်း)။
* **`do-while`**: ၸႂ်ႉမိူဝ်ႈလူဝ်ႇႁႂ်ႈ Code **လႅၼ်ႈဢွၼ်တၢင်း 1 ပွၵ်ႈ** ၼႄႉၼႄႉ (တူဝ်ယၢင်ႇ: ၼႄ Menu ဢွၼ်တၢင်း ၸင်ႇထၢမ် User)။
