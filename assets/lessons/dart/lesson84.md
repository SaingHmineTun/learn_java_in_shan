## 84) The await Mechanism and Function Suspension

တွၼ်ႈတွၼ်ႈၼႆႉ လမ်ႇလွင်ႈၼႃႇၶႃႈ။ ၵူၼ်းတႅမ်ႈ Code တင်းၼမ် ၵႆႉပွင်ႇၸႂ်ၽိတ်းဝႃႈ `await` ၼႆႉ ႁဵတ်းႁႂ်ႈ App ႁဝ်း "ၵိုတ်း" (Freeze) ၵႂႃႇတင်းသဵင်ႈ။ တႄႇတႄႇမၼ်း ဢမ်ႇၸႂ်ႈၼၼ်ႉၶႃႈ။

### 1. လၢႆးႁဵတ်းၵၢၼ် Suspension (ၵၢၼ်ၵိုတ်းၸူဝ်ႈၶၢဝ်း)

မိူဝ်ႈ Code ႁဝ်းလႅၼ်ႈမႃးတူပ်ႉသႂ်ႇ `await`၊ မၼ်းတေမီးၵၢၼ်ႁဵတ်းသၢင်ႈ ၸိူင်ႉၼႆၶႃႈ:

1. **Function Suspension:** Function ဢၼ်မီး `await` ၼၼ်ႉ တေ "ၵိုတ်း" ဝႆႉၸူဝ်ႈၶၢဝ်း။ မၼ်းတေမၢႆဝႆႉဝႃႈ တိုၵ်ႉႁဵတ်းၵၢၼ်ထိုင်ထႅဝ်လႂ်။
2. **Return to Caller:** ၵူၼ်းႁဵတ်းၵၢၼ် (Thread) တေပွတ်ႇပွႆႇ Function ၼၼ်ႉဝႆႉသေ ပွၵ်ႈၵႂႃႇႁဵတ်းၵၢၼ်တၢင်ႇဢၼ် (မိူၼ်ၼင်ႇ UI Animation) ဢၼ်မီးၼႂ်း **Event Queue**။
3. **Resumption:** ပေႃးဝႃႈ Future ဢၼ်ႁဝ်းပႂ်ႉ (Await) ၼၼ်ႉယဝ်ႉတူဝ်ႈ (Completed)၊ မၼ်းတေသူင်ႇသัญญาၵႂႃႇၸူး **Event Loop** ဝႃႈ "ၵဝ်ယဝ်ႉယဝ်ႉ၊ သိုပ်ႇႁဵတ်းၵၢၼ်ထႅဝ်ဢၼ်ၵိုတ်းဝႆႉၼၼ်ႉလႆႈယဝ်ႉ"။

### 2. Suspension ပႅၵ်ႇပိူင်ႈတင်း Blocking ၸိူင်ႉႁိုဝ်?

* **Blocking (Sync):** မိူၼ်ၼင်ႇ ၸဝ်ႈၵဝ်ႇၸုၵ်းပႂ်ႉထႅဝ်သိုဝ်ႉၶဝ်ႈ။ သင်ၵေႃႉၼႃႈၸဝ်ႈၵဝ်ႇ ဢမ်ႇပႆႇလႆႈၶဝ်ႈ၊ ၸဝ်ႈၵဝ်ႇၵေႃႈၵႂႃႇလႂ်ဢမ်ႇလႆႈ၊ တေလႆႈၸုၵ်းပႂ်ႉယူႇၼၼ်ႉ။ (App Freeze)
* **Suspension (Async):** မိူၼ်ၼင်ႇ ၸဝ်ႈၵဝ်ႇသင်ႇၶဝ်ႈယဝ်ႉ လႆႈဝႂ် (Future) မႃး။ ၸဝ်ႈၵဝ်ႇၵႂႃႇၼင်ႈပႂ်ႉတီႈတေႇ၊ တဵၵ်းၾူၼ်း (ႁဵတ်းၵၢၼ်တၢင်ႇဢၼ်) ပႂ်ႉလႆႈ။ ပေႃးၶဝ်ႈယဝ်ႉ ၸင်ႇမႃးဢဝ်။ (App Responsive)

### 3. တူဝ်ယၢင်ႇ လွင်ႈလႅၼ်ႈ Code

```dart
Future<void> testAsync() async {
  print("2: တႄႇႁဵတ်းၵၢၼ်..."); // ႁဵတ်းၵမ်းလဵဝ်
  
  await Future.delayed(Duration(seconds: 1)); // ၵိုတ်း (Suspend) တီႈၼႆႈ 1 sec
  
  print("4: ယဝ်ႉယဝ်ႉ!"); // ႁဵတ်းဝၢႆးလင် 1 sec
}

void main() {
  print("1: တႄႇ Main");
  testAsync();
  print("3: သိုပ်ႇႁဵတ်းၵၢၼ်တၢင်ႇဢၼ်"); // ႁဵတ်းၵမ်းလဵဝ် လူၺ်ႈဢမ်ႇပႂ်ႉ testAsync() ယဝ်ႉ
}

```

**Result:** `1` -> `2` -> `3` -> `4`
သတိတူၺ်းဝႃႈ `3` တေဢွၵ်ႇမႃး ဢွၼ်တၢင်း `4` ယွၼ်ႉ `testAsync` ၵိုတ်းၸူဝ်ႈၶၢဝ်း (Suspended) ဝႆႉၼၼ်ႉယဝ်ႉ။

---

### **Key Takeaway**

`await` ဢမ်ႇလႆႈႁဵတ်းႁႂ်ႈ App ၵၢင်ႉ။ မၼ်းၵူၺ်းႁဵတ်းႁႂ်ႈ **Function ၼၼ်ႉၵူၺ်း** ၵိုတ်းပႂ်ႉ၊ တွၼ်ႈတႃႇႁႂ်ႈ Thread တူဝ်ၼိုင်ႈလဵဝ်ၼၼ်ႉ ၵႂႃႇႁဵတ်းၵၢၼ်တၢင်ႇဢၼ်လႆႈၼၼ်ႉယဝ်ႉ။
