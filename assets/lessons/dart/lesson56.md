## 56) Final vs Private Fields

ၼႂ်းၵၢၼ်တႅမ်ႈ Class, ႁဝ်းၵႆႉလႆႈလိူၵ်ႈၸႂ်ႉ **`final`** လႄႈ **Private (`_`)** တႃႇၵုမ်းၵၢၼ်ၶေႃႈမုၼ်း။ မၢင်ပွၵ်ႈႁဝ်းၸႂ်ႉမၼ်းတင်းသွင်ဢၼ် (တွၼ်ႈတႃႇလွင်ႈပူတ်ႉၽေးသုင်သုတ်း)၊ ၵူၺ်းၵႃႈ မၼ်းမီးယိူင်းဢၢၼ်းပႅၵ်ႇၵၼ်ယူႇယဝ်ႉ။

---

### 1. Final Fields (လႅၵ်ႈလၢႆႈၵႃႈၶၼ်ဢမ်ႇလႆႈ)

Keyword **`final`** ၸႂ်ႉတႃႇမၵ်းမၼ်ႈဝႃႈ Variable ၼၼ်ႉ "ပၼ်ၵႃႈၶၼ်လႆႈပွၵ်ႈလဵဝ်" (Assign once)။ ဝၢႆးသေပၼ်ၵႃႈၶၼ်ယဝ်ႉ၊ ၸဝ်ႈၵဝ်ႇတေမႄးမၼ်းၶိုၼ်းဢမ်ႇလႆႈထႅင်ႈ (Immutable)။

* **ယိူင်းဢၢၼ်း:** တႃႇႁႄႉၵင်ႈၵၢၼ်လႅၵ်ႈလၢႆႈၵႃႈၶၼ် (Value protection)။
* **Visibility:** မၼ်းၸၢင်ႈပဵၼ် Public လႆႈ (ၵူၼ်းၼွၵ်ႈလူလႆႈ ၵူၺ်းၵႃႈမႄးဢမ်ႇလႆႈ)။

**Example:**

```dart
class User {
  final String id; // id တေဢမ်ႇလႅၵ်ႈလၢႆႈလႆႈထႅင်ႈတေႃႇၸူဝ်ႈ
  User(this.id);
}

```

---

### 2. Private Fields (သိပ်းဝႆႉဢမ်ႇႁႂ်ႈႁၼ်)

မၢႆ **Underscore (`_`)** ၸႂ်ႉတႃႇသိပ်း Variable ဢမ်ႇႁႂ်ႈ File တၢင်ႇဢၼ်ႁၼ်။

* **ယိူင်းဢၢၼ်း:** တႃႇၵုမ်းၵၢၼ်လွင်ႈၶဝ်ႈထိုင် (Visibility control)။
* **Mutability:** မၼ်းၸၢင်ႈပဵၼ် Variable ယူဝ်းယူဝ်း ဢၼ်မႄးလႆႈယူႇ (ၵူၺ်းၵႃႈမႄးလႆႈၼႂ်း Class/Library ၼၼ်ႉၵူၺ်း)။

**Example:**

```dart
class Counter {
  int _count = 0; // ၵူၼ်းၼွၵ်ႈဢမ်ႇႁၼ်၊ ၵူၺ်းၵႃႈၼႂ်း Class ၼႆႉမႄးလႆႈယူႇ
}

```

---

### 3. Comparison Table: Final vs Private

| Feature | `final` Field | Private (`_`) Field |
| --- | --- | --- |
| **Main Goal** | ႁႄႉၵင်ႈၵၢၼ်မႄးၵႃႈၶၼ် | ႁႄႉၵင်ႈၵၢၼ်ႁၼ်/ၸႂ်ႉတူဝ် |
| **Modification** | မႄးဢမ်ႇလႆႈ (Immutable) | မႄးလႆႈ (ၼႂ်း File လဵဝ်ၵၼ်) |
| **Access** | ၵူၼ်းၼွၵ်ႈလူလႆႈ (သင်ဢမ်ႇၸႂ်ႉ `_`) | ၵူၼ်းၼွၵ်ႈလူဢမ်ႇလႆႈ (သင်ဢမ်ႇမီး Getter) |
| **Assignment** | လူဝ်ႇပၼ်ၵႃႈၶၼ်ၵမ်းလဵဝ်/ၼႂ်း Constructor | ပၼ်ၵႃႈၶၼ်မိူဝ်ႈလႂ်ၵေႃႈလႆႈ |

---

### 4. Combining Both (`final` + `_`)

လၢႆးတႅမ်ႈဢၼ် **Safe** သုတ်း တွၼ်ႈတႃႇၶေႃႈမုၼ်းဢၼ်လမ်ႇလွင်ႈၼႆႉၵေႃႈပဵၼ် ၵၢၼ်ၸႂ်ႉ **Private Final Field** ၸွမ်းလူၺ်ႈ **Getter** ယဝ်ႉ။

```dart
class ApiConfig {
  // 1. သိပ်းဝႆႉ (Private)
  // 2. ပၼ်မႄးဢမ်ႇလႆႈ (Final)
  final String _apiKey = "ABC-123";

  // 3. ပၼ်လူလႆႈၵူၺ်း (Getter)
  String get apiKey => _apiKey;
}

```

လၢႆးၼႆႉ ၸဝ်ႈၵဝ်ႇတေလႆႈလွင်ႈပူတ်ႉၽေးတင်း 2 ၶၼ်ႉ: ၵူၼ်းၼွၵ်ႈဢမ်ႇၸၢင်ႈမႄး Variable တူဝ်တႄႇမၼ်းလႆႈ (`final`) လႄႈ ၸဝ်ႈၵဝ်ႇၵုမ်းၵၢၼ်လွင်ႈသူင်ႇၶေႃႈမုၼ်းဢွၵ်ႇၵႂႃႇလႆႈ (`_` + `get`)။
