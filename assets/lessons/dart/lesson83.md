## 83) The .then() Callback vs async/await

မိူဝ်ႈႁဝ်းလႆႈ **Future** မႃးယဝ်ႉ၊ ႁဝ်းမီးလၢႆးႁပ်ႉဢဝ်ၶေႃႈမုၼ်း (Data) ၼႂ်းမၼ်းယူႇ 2 လၢႆးလူင်ၶႃႈ။

---

### 1. The .then() Callback (လၢႆးၵဝ်ႇ)

လၢႆးၼႆႉပဵၼ်ၵၢၼ်ၸႂ်ႉ Callback function။ မၼ်းမိူၼ်ၼင်ႇႁဝ်းသင်ႇဝႆႉဝႃႈ "ပေႃး Data မႃးယဝ်ႉ ႁႂ်ႈႁဵတ်းၵၢၼ်ၼႆႉၼႃ"။

* **လွင်ႈလီ:** ဢမ်ႇလူဝ်ႇမၵ်းမၼ်ႈ Function ႁဝ်းပဵၼ် `async`။
* **လွင်ႈဢမ်ႇလီ:** ပေႃးမီး Future လၢႆလၢႆဢၼ်သိုပ်ႇၵၼ်၊ Code တေလူငၢႆႈယၢပ်ႇ (ႁွင်ႉဝႃႈ Callback Hell)။

```dart
void main() {
  print("တႄႇလူတ်ႇ Data...");

  fetchData().then((value) {
    print("လႆႈၶေႃႈမုၼ်းယဝ်ႉ: $value");
  }).catchError((error) {
    print("မီးပၼ်ႁႃ: $error");
  });

  print("ႁဵတ်းၵၢၼ်တၢင်ႇဢၼ်ပႂ်ႉ..."); 
  // ထႅဝ်ၼႆႉတေဢွၵ်ႇမႃး ဢွၼ်တၢင်း "လႆႈၶေႃႈမုၼ်းယဝ်ႉ"
}

```

---

### 2. async/await (လၢႆးမႂ်ႇ - လၢႆးဢၼ်ပိူၼ်ႈၸႂ်ႉၼမ်)

လၢႆးၼႆႉႁဵတ်းႁႂ်ႈ Code Asynchronous ႁဝ်း လူတူၺ်းငၢႆႈမိူၼ် Code ယူဝ်းယူဝ်း (Synchronous)။

* **`async`**: သႂ်ႇဝႆႉၼႃႈ Body `{}` တွၼ်ႈတႃႇမၵ်းမၼ်ႈဝႃႈ Function ၼႆႉတေမီးၵၢၼ်ပႂ်ႉ (Await)။
* **`await`**: သႂ်ႇဝႆႉၼႃႈ Future တွၼ်ႈတႃႇ "ၵိုတ်းပႂ်ႉ" ႁႂ်ႈ Data မႃးယဝ်ႉ ၸင်ႇသိုပ်ႇၵႂႃႇထႅဝ်ၼႃႈ။

```dart
Future<void> main() async {
  print("တႄႇလူတ်ႇ Data...");

  try {
    String value = await fetchData();
    print("လႆႈၶေႃႈမုၼ်းယဝ်ႉ: $value");
  } catch (error) {
    print("မီးပၼ်ႁႃ: $error");
  }

  print("ယဝ်ႉတူဝ်ႈတင်းသဵင်ႈယဝ်ႉ။");
  // ထႅဝ်ၼႆႉတေဢွၵ်ႇမႃး ဝၢႆးလင် "လႆႈၶေႃႈမုၼ်းယဝ်ႉ"
}

```

---

### **Table Comparison (လွင်ႈပႅၵ်ႇပိူင်ႈ)**

| ႁူဝ်ၶေႃႈ | .then() | async / await |
| --- | --- | --- |
| **လၢႆးတႅမ်ႈ** | ၸႂ်ႉ Callback (သႂ်ႇၼႂ်းဝူတ်ႇ) | တႅမ်ႈလူင်းၵႂႃႇၸွမ်းထႅဝ် |
| **လွင်ႈလူငၢႆႈ** | ယၢပ်ႇပေႃးမီးလၢႆထႅဝ် | ငၢႆႈ လႄႈ သႅၼ်ႈသႂ် |
| **ၵၢၼ်ၸတ်း Error** | ၸႂ်ႉ `.catchError()` | ၸႂ်ႉ `try-catch` |

---

**Summary:** ၼႂ်းၵၢၼ်တႅမ်ႈ Flutter App တႄႇတႄႇၼၼ်ႉ ပိူၼ်ႈသူင်ၸႂ်ႉ **async/await** လိူဝ်၊ ယွၼ်ႉမၼ်းႁဵတ်းႁႂ်ႈ Code ႁဝ်းသႅၼ်ႈသႂ် လႄႈ ၸတ်းၵၢၼ်ငၢႆႈလိူဝ်ၼၼ်ႉယဝ်ႉ။
