## 7) Variable Declaration: Type vs Name

ၼႂ်းလိၵ်ႈ Dart ၼၼ်ႉ တႃႇတေသိမ်းၵႃႈၶၼ် (Value) သင်ဝႆႉသေဢၼ်ဢၼ် တႃႇတေၸႂ်ႉတိုင်ႈၼႂ်း Program ႁဝ်းၼၼ်ႉ ႁဝ်းလူဝ်ႇၸႂ်ႉ **Variable** (တူဝ်လႅၵ်ႈလၢႆႈ) ယဝ်ႉ။

### 1. ပိူင်ၽွတ်ႈဝႆႉ Variable

မိူဝ်ႈႁဝ်းတေပိုၼ်ၽၢဝ်ႇ (Declare) Variable ၼိုင်ႈဢၼ်ၼၼ်ႉ မၼ်းမီးပိူင်သၢင်ႈဝႆႉ 3 တူဝ်:

1. **Data Type**: လၢတ်ႈၼႄဝႃႈ ၶေႃႈမုၼ်းဢၼ်ႁဝ်းတေသိမ်းၼၼ်ႉ ပဵၼ်သင် (လိၵ်ႈ၊ မၢႆၼပ်ႉ၊ ဝၼ်းထီႉ)။
2. **Variable Name**: ပဵၼ်ၸိုဝ်ႈဢၼ်ႁဝ်းပၼ်ဝႆႉ တႃႇတေႁွင်ႉၸႂ်ႉၶိုၼ်း။
3. **Value**: ပဵၼ်ၵႃႈၶၼ် ဢၼ်ႁဝ်းတေသိမ်းဝႆႉၼႂ်းမၼ်း (သႂ်ႇလႆႈလူၺ်ႈၸႂ်ႉပုမ်ႇ `=` )။

တူဝ်ယၢင်ႇ:

```dart
String name = "Sai Sai";
int age = 20;

```

### 2. ပိူင်ပၼ်ၸိုဝ်ႈ Variable (Naming Rules)

တႃႇတေပၼ်ၸိုဝ်ႈ Variable ၼၼ်ႉ ႁဝ်းလူဝ်ႇႁဵတ်းၸွမ်းပိူင်တႂ်ႈၼႆႉယဝ်ႉ:

* **Lower Camel Case**: တူဝ်လိၵ်ႈတႄႇမၼ်း တေလႆႈပဵၼ်တူဝ်လဵၵ်ႉ၊ သင်မီးသွင်ၶေႃႈ ၶေႃႈထူၼ်ႈသွင်ၼၼ်ႉ တေလႆႈတႄႇလူၺ်ႈတူဝ်လူင် (တူဝ်ယၢင်ႇ: `userName`, `totalPrice`)။
* **ဢမ်ႇလႆႈတႄႇလူၺ်ႈမၢႆၼပ်ႉ**: တူဝ်ယၢင်ႇ `1user` ၼႆႉ ဢမ်ႇလႆႈ။ ၵူၺ်းၵႃႈ `user1` ၼႆႉလႆႈယူႇ။
* **ဢမ်ႇလႆႈမီးသုၼ်ႇဝၢင်ႈ (Space)**: တူဝ်ယၢင်ႇ `user name` ၼႆႉ ဢမ်ႇလႆႈ။
* **ဢမ်ႇလႆႈၸႂ်ႉ Reserved Words**: ၸိုဝ်ႈဢၼ် Dart ၸႂ်ႉဝႆႉယဝ်ႉ မိူၼ်ၼင်ႇ `void`, `main`, `class`, `if` ၸိူဝ်းၼႆႉ ဢဝ်မႃးပၼ်ပဵၼ်ၸိုဝ်ႈ Variable ဢမ်ႇလႆႈ။

### 3. Declaration vs Initialization

* **Declaration**: ၼႆႉပဵၼ်ၵၢၼ် "ပိုၼ်ၽၢဝ်ႇ" ၸိုဝ်ႈလႄႈ Type မၼ်းၵူၺ်း (တူဝ်ယၢင်ႇ: `int score;`)။ ၼႂ်း Dart ၼၼ်ႉ သင်ႁဝ်းဢမ်ႇပႆႇသႂ်ႇၵႃႈၶၼ်၊ ႁဝ်းလူဝ်ႇလႆႈၾၢင်ႉလွင်ႈ Null Safety (ဢၼ်တေလဵပ်ႈႁဵၼ်းဝၼ်းၼႃႈ)။
* **Initialization**: ၼႆႉသမ်ႉ ပဵၼ်ၵၢၼ် "သႂ်ႇၵႃႈၶၼ်" ပၼ်မၼ်းပွၵ်ႈၵမ်းဢွၼ်တၢင်းသုတ်း (တူဝ်ယၢင်ႇ: `score = 100;`)။ ႁဝ်းၸၢင်ႈႁဵတ်းတင်းသွင်ယၢင်ႇ ၼႂ်းထႅဝ်လဵဝ်ၵၼ်လႆႈယူႇ (`int score = 100;`)။
