
## 6) Comments & Documentation

Comments ၼႆႉ ပဵၼ်လိၵ်ႈဢၼ်ႁဝ်းတႅမ်ႈဝႆႉၼႂ်း Code တႃႇႁႂ်ႈၵူၼ်း (ဢမ်ႇၼၼ် ႁဝ်းၶိုၼ်း) ဢၢၼ်ႇသေ ပွင်ႇၸႂ်ဝႃႈ Code ၼႆႉႁဵတ်းသင်။ Dart Compiler တေဢမ်ႇလူဢၢၼ်ႇ လႄႈ တေဢမ်ႇလႅၼ်ႈ Comments ၸိူဝ်းၼႆႉ။

### 1. Single-line Comment

ၸႂ်ႉတႃႇတႅမ်ႈလိၵ်ႈဢၼ်မီးထႅဝ်လဵဝ်။

```dart
// ၼႆႉပဵၼ် လၢႆးတႅမ်ႈလိၵ်ႈထႅဝ်လဵဝ်
print('Hello'); // ၼႆႉၵေႃႈပဵၼ် Comment

```

### 2. Multi-line Comment

ၸႂ်ႉတႃႇတႅမ်ႈလိၵ်ႈလၢႆလၢႆထႅဝ်။

```dart
/* ၼႆႉပဵၼ် လၢႆးတႅမ်ႈလိၵ်ႈ
ဢၼ်ယၢဝ်းလၢႆထႅဝ်
တႃႇသပ်းလႅင်း Code */

```

### 3. Documentation Comment

ၼႆႉပဵၼ်လၢႆးတႅမ်ႈ Comment ဢၼ်ၶူးဝႆႉ (Professional)။ မိူဝ်ႈႁဝ်းဢဝ် Mouse ၵႂႃႇၸီႉဝႆႉတီႈၼိူဝ် Function ၼၼ်ႉ IDE တေၼႄလိၵ်ႈၸိူဝ်းၼႆႉပၼ်ႁဝ်းယဝ်ႉ။

```dart
/// ၼႆႉပဵၼ် Function တႃႇတႄႇလႅၼ်ႈ App
void main() { 
  // ...
}

```
