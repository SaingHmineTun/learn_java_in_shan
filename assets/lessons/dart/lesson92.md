## 92) Stream Transformations: map, where, and expand

မိူဝ်ႈၶေႃႈမုၼ်း (Data) လႆမႃးၸွမ်း Pipe ၼၼ်ႉ၊ မၢင်ပွၵ်ႈႁဝ်းဢမ်ႇၶႂ်ႈလႆႈၶေႃႈမုၼ်း "တူဝ်ၵဝ်ႇ" မၼ်း။ ႁဝ်းၸၢင်ႈ "ပိၼ်ႇလႅၵ်ႈ" (Transform) မၼ်းဢွၼ်တၢင်း ၸင်ႇၵႂႃႇထိုင်ၵူၼ်းထွမ်ႇၶႃႈ။

ၼႆႉမိူၼ်ၼင်ႇ ႁဝ်းသႂ်ႇ "တူဝ်ထွင်ၼမ်ႉ" (Filter) ဝႆႉၼႂ်း Pipe ၼၼ်ႉယဝ်ႉ။

---

### 1. where (ၵၢၼ်ထွင်ဢဝ်)

ၸႂ်ႉတွၼ်ႈတႃႇ "လိူၵ်ႈ" ဢဝ်ၶေႃႈမုၼ်း ဢၼ်ၵူတ်ႇထတ်းယဝ်ႉဝႃႈ မၼ်းမၢၼ်ႇမႅၼ်ႈၸွမ်းပိူင်ႁဝ်း။ ၶေႃႈမုၼ်းဢၼ်ဢမ်ႇၶဝ်ႈပိူင် တေဢမ်ႇလႆႈပူၼ်ႉၵႂႃႇၸူးၵူၼ်းထွမ်ႇ။

```dart
// ၵူၺ်းလိူၵ်ႈဢဝ် မၢႆၵူပ်ႉ (Even numbers) ၵူၺ်း
stream.where((num) => num % 2 == 0).listen(print); 

```

### 2. map (ၵၢၼ်ပိၼ်ႇလႅၵ်ႈ)

ၸႂ်ႉတွၼ်ႈတႃႇ "ပိၼ်ႇ" ၶေႃႈမုၼ်းၼၼ်ႉ ႁႂ်ႈပဵၼ်မဵဝ်းမႂ်ႇ။ တူဝ်ယၢင်ႇ မိူၼ်ၼင်ႇလႆႈမၢႆမႃး သမ်ႉၶႂ်ႈပိၼ်ႇပဵၼ် String။

```dart
// ပိၼ်ႇမၢႆ ၼပ်ႉ ႁႂ်ႈပဵၼ် ၶေႃႈၵႂၢမ်း
stream.map((num) => "မၢႆထိုင်ယဝ်ႉ: $num").listen(print);

```

### 3. expand (ၵၢၼ်ၽႄဢွၵ်ႇ)

ၸႂ်ႉမိူဝ်ႈ Data ၼိုင်ႈဢၼ်လႆမႃး သမ်ႉၶႂ်ႈၽႄမၼ်းဢွၵ်ႇပဵၼ် လၢႆလၢႆဢၼ်။

```dart
// ပေႃးလႆႈ 1 မႃး ႁႂ်ႈမၼ်းသူင်ႇ [1, 1] ဢွၵ်ႇၵႂႃႇ
stream.expand((num) => [num, num]).listen(print);

```

---

### 4. တူဝ်ယၢင်ႇၵၢၼ်ၸႂ်ႉႁူမ်ႈၵၼ် (Method Chaining)

ႁဝ်းၸၢင်ႈဢဝ်တူဝ်ထွင် (Filters) ၸိူဝ်းၼႆႉ သိုပ်ႇၵၼ်လႆႈလၢႆလၢႆၸၼ်ႉၶႃႈ:

```dart
import 'dart:async';

void main() {
  final controller = StreamController<int>();

  controller.stream
      .where((n) => n > 10)            // 1. လိူၵ်ႈဢဝ်ဢၼ်ယႂ်ႇလိူဝ် 10
      .map((n) => "Value is: $n")     // 2. ပိၼ်ႇပဵၼ်လိၵ်ႈ
      .listen((data) => print(data)); // 3. ၸင်ႇထွမ်ႇဢဝ်

  controller.sink.add(5);  // တေဢမ်ႇဢွၵ်ႇ (ယွၼ်ႉဝႃႈ ဢမ်ႇလိူဝ် 10)
  controller.sink.add(20); // တေဢွၵ်ႇဝႃႈ: Value is: 20
  
  controller.close();
}

```

---

### **Summary**

* **where**: တႃႇ "လိူၵ်ႈ" (Filter)။
* **map**: တႃႇ "ပိၼ်ႇ" (Convert)။
* **expand**: တႃႇ "ၽႄ" (Multiply)။
