
## 88) Stream Theory: The Continuous Data Pipe

တွၼ်ႈတႃႇတေပွင်ႇၸႂ် **Stream** ၼၼ်ႉ ႁဝ်းလူဝ်ႇတူၺ်း လွင်ႈပႅၵ်ႇပိူင်ႈမၼ်းတင်း **Future** ဢၼ်ႁဝ်းႁဵၼ်းမႃးၼၼ်ႉၶႃႈ။

### 1. Future vs Stream (ပႅၵ်ႇပိူင်ႈၵၼ်ၸိူဝ်ႉႁိုဝ်?)

* **Future (One-time):** မိူၼ်ၼင်ႇႁဝ်းသင်ႇ Pizza။ ႁဝ်းပႂ်ႉ (Await)၊ ပေႃးမၼ်းမႃးထိုင်ယဝ်ႉၵေႃႈ ၵၢၼ်ယဝ်ႉတူဝ်ႈၵႂႃႇ။ ၸဝ်ႈၵဝ်ႇလႆႈ Data ပွၵ်ႈလဵဝ်ၵူၺ်း။
* **Stream (Continuous):** မိူၼ်ၼင်ႇႁဝ်းတူၺ်း **YouTube Live** ဢမ်ႇၼၼ် **Spotify**။ ၶေႃႈမုၼ်း (Data) တေလႆႈမႃးလၢႆလၢႆတွၼ်ႈ၊ ပဵၼ်ၸုမ်းပဵၼ်ၸုမ်းသေ လႆႈမႃးယူႇတႃႇသေႇ တေႃႇပေႃးႁဝ်း ပိတ်း (Close) မၼ်းပႅတ်ႈ။

### 2. ပိူင်လူင် 4 ယၢင်ႇ ၼႂ်း Stream

တွၼ်ႈတႃႇ Stream ၼိုင်ႈဢၼ်တေႁဵတ်းၵၢၼ်လႆႈၼၼ်ႉ လူဝ်ႇမီး:

1. **The Source (တီႈမႃး):** တီႈဢၼ်သူင်ႇၶေႃႈမုၼ်းဢွၵ်ႇမႃး (မိူၼ်ၼင်ႇ Firebase ဢမ်ႇၼၼ် Sensor)။
2. **The Data (Event):** ၶေႃႈမုၼ်းဢၼ်လႆႈမႃး။ မၼ်းၸၢင်ႈပဵၼ်လႆႈ 3 ပိူင်:
* **Data:** ၶေႃႈမုၼ်းဢၼ်ႁဝ်းၶႂ်ႈလႆႈ။
* **Error:** ပေႃးမီးပၼ်ႁႃ (မိူၼ်ၼင်ႇ Internet တတ်း)။
* **Done:** ပေႃးၶေႃႈမုၼ်းသုတ်းယဝ်ႉ။


3. **The Controller (တူဝ်ၵုမ်း):** တူဝ်ဢၼ်ၸတ်းၵၢၼ် ပိုတ်ႇ/ပိတ်း ၼမ်ႉလႆ။
4. **The Listener (ၵူၼ်းထွမ်ႇ):** Code ႁဝ်း ဢၼ်ပႂ်ႉႁပ်ႉဢဝ်ၶေႃႈမုၼ်း တွၼ်ႈတႃႇတေဢဝ်ၵႂႃႇၼႄၼိူဝ် Screen (UI)။

### 3. တူဝ်ယၢင်ႇ ၼႂ်း Flutter App

* **Chat App:** မိူဝ်ႈမီးၵူၼ်းသူင်ႇလိၵ်ႈမႃး၊ Stream တေသူင်ႇ Data မႂ်ႇမႃးပၼ်ႁဝ်းၵမ်းလဵဝ် လူၺ်ႈဢမ်ႇလူဝ်ႇ Refresh Screen။
* **Downloading File:** ႁဝ်းၸၢင်ႈၸႂ်ႉ Stream တႃႇၼႄ % ဢၼ်လူတ်ႇယဝ်ႉ (10%, 20%, 30%...)။
* **User Auth:** မိူဝ်ႈ User Login ဢမ်ႇၼၼ် Logout၊ Stream တေသူင်ႇသัญญาမႃး ႁႂ်ႈ App ႁဝ်းပိၼ်ႇၼႃႈ Screen ႁင်းၵူၺ်း။

---

### **Key Takeaway**

**Stream** ၵေႃႈပဵၼ် Asynchronous ၼင်ႇၵဝ်ႇ၊ ၵူၺ်းၵႃႈမၼ်းပႅၵ်ႇပိူင်ႈတီႈဝႃႈ မၼ်းသူင်ႇ Result ဢွၵ်ႇမႃးလႆႈ **"လၢႆလၢႆပွၵ်ႈ"** ၼႂ်း Pipe ဢၼ်လဵဝ်ၵၼ်ၼၼ်ႉယဝ်ႉၶႃႈ။
