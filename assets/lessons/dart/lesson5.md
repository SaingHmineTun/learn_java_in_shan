## 5) The main() function & Basic Syntax

ၼႂ်းလိၵ်ႈ Dart ၼၼ်ႉ **`main()` function** ၼႆႉလမ်ႇလွင်ႈသုတ်းယဝ်ႉ။ မၼ်းပဵၼ် "တူၵ်းတႄႇ" (Entry point) ၶေႃႈ Application ႁဝ်း။ ဢမ်ႇဝႃႈ App ႁဝ်းတေယႂ်ႇၵၢၼ်ႉတေႃႇႁိုဝ်ၵေႃႈယဝ်ႉ၊ မၼ်းတေတႄႇလႅၼ်ႈတီႈ `main()` ၼႆႉၵူၺ်း။

### 1. Anatomy ၶေႃႈ main() function

တူၺ်းတူဝ်ယၢင်ႇ Code တႂ်ႈၼႆႉ:

```dart
void main() {
  print('မႂ်ႇသုင် ဝၼ်းမိူင်းတႆး!');
}

```

* **void**: ၼႆႉပဵၼ် "Return Type"။ မၼ်းလၢတ်ႈၼႄဝႃႈ Function ၼႆႉ ႁဵတ်းၵၢၼ်ယဝ်ႉၵေႃႈ တေဢမ်ႇသူင်ႇၵႃႈၶၼ် (Value) သင်ပၼ်ၶိုၼ်း။
* **main**: ၼႆႉပဵၼ်ၸိုဝ်ႈ Function ဢၼ် Dart Runtime တူတ်ႉႁႃ။
* **()**: ၼႆႉပဵၼ်တီႈဝႆႉ "Parameters" (သင်မီး)။
* **{}**: ၼႆႉပဵၼ် "Function Body"။ Code ဢၼ်ႁဝ်းၶႂ်ႈႁႂ်ႈလႅၼ်ႈၼၼ်ႉ တေလႆႈတႅမ်ႈဝႆႉၼႂ်းၵႄႈ ပိုတ်ႇလႄႈပိတ်ႇ ၶွပ်ႇ (Braces) ၼႆႉယဝ်ႉ။

### 2. ပိူင်တႅမ်ႈပိုၼ်ႉထၢၼ် (Basic Syntax Rules)

တႃႇတေတႅမ်ႈ Code ႁႂ်ႈထုၵ်ႇမႅၼ်ႈၸွမ်းပိူင် (Syntax) ၼၼ်ႉ ႁဝ်းလူဝ်ႇႁူႉဝႆႉ:

* **Semicolon ( ; )**: တီႈသုတ်းထႅဝ် Code (Statement) ၵူႈထႅဝ် တေလႆႈသႂ်ႇ Semicolon ဝႆႉတေႃႇ။ သင်ဢမ်ႇသႂ်ႇ ၶွမ်းတေဢမ်ႇပွင်ႇၸႂ်သေ တေၼႄ Error ယဝ်ႉ။
* **Case Sensitivity**: Dart ၼႆႉ မၼ်းထတ်းတူၺ်း တူဝ်လိၵ်ႈ (လူင်/လဵၵ်ႉ)။ တူဝ်ယၢင်ႇ: `void` လႄႈ `Void` ၼႆႉ ဢမ်ႇမိူၼ်ၵၼ်။
* **Print Statement**: ၸႂ်ႉ `print()` တႃႇထုတ်ႇၼႄ (Output) လိၵ်ႈ ဢမ်ႇၼၼ် ၵႃႈၶၼ်ၵမ်းဢၼ်ဢၼ် တီႈၼႂ်း Console/Terminal။

### 3. လွင်ႈၸႂ်ႉ String (လိၵ်ႈ) ၼႂ်း Print

မိူဝ်ႈႁဝ်းတေ print လိၵ်ႈဢွၵ်ႇမႃးၼၼ်ႉ ႁဝ်းၸၢင်ႈၸႂ်ႉ:

* Single Quotes: `'Hello'`
* Double Quotes: `"Hello"`
  တင်းသွင်ပိူင်ၼႆႉ ႁဵတ်းၵၢၼ်မိူၼ်ၵၼ်ယဝ်ႉ။
