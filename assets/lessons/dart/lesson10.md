## 10) Type Inference with 'var'

Dart ၼႆႉ ပဵၼ် "Strongly Typed Language" (လူဝ်ႇမီး Type တႅတ်ႈတေႃး) ၵေႃႈယဝ်ႉ၊ ၵူၺ်းၵႃႈ ႁဝ်းဢမ်ႇလူဝ်ႇလၢတ်ႈၼႄ Type မၼ်းၵူႈပွၵ်ႈၵေႃႈလႆႈ။ ႁဝ်းၸၢင်ႈၸႂ်ႉ Keyword **`var`** ယဝ်ႉ။

### 1. လၢႆးႁဵတ်းၵၢၼ် 'var'

မိူဝ်ႈႁဝ်းၸႂ်ႉ `var` သေ သႂ်ႇၵႃႈၶၼ်ပၼ်မၼ်းၵမ်းလဵဝ်ၼၼ်ႉ Dart Compiler တေ "လႅပ်ႈႁူႉ" (Infer) ႁင်းၵူၺ်းဝႃႈ မၼ်းလူဝ်ႇပဵၼ် Type သင် လူၺ်ႈတူၺ်းၵႃႈၶၼ် (Value) ဢၼ်ႁဝ်းသႂ်ႇပၼ်ၼၼ်ႉယဝ်ႉ။

```dart
var username = "saisai_dev"; // Dart တေႁူႉဝႃႈ username ပဵၼ် String
var loginAttempts = 3;       // Dart တေႁူႉဝႃႈ loginAttempts ပဵၼ် int
var isOpen = true;           // Dart တေႁူႉဝႃႈ isOpen ပဵၼ် bool

```

### 2. ပိူင်မၵ်းမၼ်ႈ (Type Locking)

ၶေႃႈလမ်ႇလွင်ႈသုတ်းတေႃႇ `var` ၼၼ်ႉပဵၼ် - မိူဝ်ႈ Dart ႁူႉယဝ်ႉဝႃႈ Variable ၼၼ်ႉ ပဵၼ် Type သင် (တူဝ်ယၢင်ႇ: String) ၼႆ **Type မၼ်းတေထုၵ်ႇ Lock ဝႆႉၵမ်းလဵဝ်**။ ႁဝ်းတေဢမ်ႇၸၢင်ႈဢဝ် Type တၢင်ႇဢၼ် (တူဝ်ယၢင်ႇ: int) မႃးလႅၵ်ႈလၢႆႈသႂ်ႇၼႂ်းမၼ်းထႅင်ႈလႆႈယဝ်ႉ။

```dart
var x = 10; // x ထုၵ်ႇမၵ်းမၼ်ႈပဵၼ် int
x = 20;     // လႆႈယူႇ ယွၼ်ႉပဵၼ် int မိူၼ်ၵၼ်
x = "Hi";   // Error! ယွၼ်ႉ x ပဵၼ် int ဝႆႉယဝ်ႉ၊ သႂ်ႇ String ဢမ်ႇလႆႈ။

```

### 3. မိူဝ်ႈလႂ်ထုၵ်ႇၸႂ်ႉ 'var'?

* ၸႂ်ႉမိူဝ်ႈႁဝ်းသႂ်ႇၵႃႈၶၼ် (Initialization) ပၼ်မၼ်းၵမ်းလဵဝ် တီႈထႅဝ်ပိုၼ်ၽၢဝ်ႇၼၼ်ႉ။
* ၸႂ်ႉတႃႇႁႂ်ႈ Code ႁဝ်းလူဢၢၼ်ႇငၢႆႈ (Clean code) မိူဝ်ႈ Type မၼ်းယၢဝ်းပူၼ်ႉတီႈ။
* သင်ဝႃႈႁဝ်းပိုၼ်ၽၢဝ်ႇ `var x;` သေ ဢမ်ႇပႆႇသႂ်ႇၵႃႈၶၼ်၊ Dart တေမၢႆဝႆႉပဵၼ် `dynamic` ယဝ်ႉ (ဢၼ်ၼႆႉလူဝ်ႇလႆႈၾၢင်ႉ)။
