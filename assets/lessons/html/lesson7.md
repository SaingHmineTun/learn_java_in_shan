## Lesson 7: Comments: Leaving Notes for the Scribe

မိူဝ်ႈႁဝ်းတႅမ်ႈ Code ၼမ်မႃးၼၼ်ႉ ၵမ်းလၢႆးႁဝ်းၸၢင်ႈလိုမ်းဝႃႈ "တွၼ်ႈၼႆႉ ႁဝ်းတႅမ်ႈဝႆႉတႃႇသင်?" ဢမ်ႇၼၼ် ႁဝ်းၶႂ်ႈၾၢၵ်ႇၶေႃႈၵႂၢမ်းဝႆႉပၼ် ၵူၼ်းတႅမ်ႈ Code တၢင်ႇၵေႃႉ။ တီႈၼႆႉႁဝ်းၸႂ်ႉ **Comments** ၶႃႈဢေႃႈ။

---

#### 1. What are Comments? (ၶေႃႈမုၼ်းၾၢၵ်ႇ)

**Comments** ပဵၼ်တူဝ်လိၵ်ႈ ဢၼ်ႁဝ်းတႅမ်ႈဝႆႉၼႂ်း Code ၵူၺ်း၊ Browser တေဢမ်ႇလူ (Ignore) မၼ်း လႄႈ တေဢမ်ႇၼႄႁၢင်ႈသင်ၼိူဝ်ၼႃႈဝႅပ်ႉသၢႆႉ။ မၼ်းမီးဝႆႉတႃႇ "ၵူၼ်း" လူၵူၺ်း၊ ဢမ်ႇၸႂ်ႈတႃႇ "ၶွမ်ႇပိဝ်ႇတႃႇ" လူၶႃႈ။

* **တူဝ်ယၢင်ႇ:**

```html
<h1>မႂ်ႇသုင်ၶႃႈ</h1>

```

---

#### 2. Why use them? (ပဵၼ်သင်လႄႈလူဝ်ႇၸႂ်ႉ?)

* **Organization:** ၸွႆႈမၢႆဝႆႉဝႃႈ Code တွၼ်ႈလႂ် တႄႇတီႈလႂ် ယဝ်ႉတီႈလႂ်။
* **Collaboration:** မိူဝ်ႈႁဵတ်းၵၢၼ်ၸွမ်းပိူၼ်ႈ၊ မၼ်းၸွႆႈလၢတ်ႈၼႄဝႃႈ Code ၼႆႉႁဵတ်းၵၢၼ်ၼင်ႇႁိုဝ်။
* **Testing:** သင်ႁဝ်းၶႂ်ႈ "ပိၵ်ႉ" Code တွၼ်ႈၼိုင်ႈဝႆႉၸူဝ်ႈၵြႃႉ (ဢမ်ႇၶႂ်ႈတႅမ်ႈမွတ်ႇပႅတ်ႈတင်းသဵင်ႈ)၊ ႁဝ်းၸၢင်ႈဢဝ် Comment ၵုမ်းဝႆႉလႆႈၶႃႈ။

---

#### 3. Syntax (လွၵ်းလၢႆးတႅမ်ႈ)

HTML comment တေလႆႈတႄႇလူၺ်ႈ ``။

* **Important:** တွၼ်ႈၼႂ်းၵၢင်မၼ်းၼၼ်ႉ ၸဝ်ႈၵဝ်ႇတေတႅမ်ႈလိၵ်ႈသင်ၵေႃႈလႆႈ၊ တႅမ်ႈလၢႆလၢႆထႅဝ်ၵေႃႈလႆႈၶႃႈ။
* **Note:** ဢမ်ႇလီသႂ်ႇ "Password" ဢမ်ႇၼၼ် ၶေႃႈမုၼ်းလပ်ႉၼႂ်း Comment၊ ယေႃႉဝႃႈ ၵူၼ်းၸႂ်ႉတိုဝ်း (User) တေၸၢင်ႈ "View Source" သေ လူႁၼ်လႆႈယူႇၶႃႈ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉတႅမ်ႈပၵ်းယဵမ်ႈဝၼ်း (Journal):

1. **Code:** ပဵၼ်လွင်ႈတႄႇတႅမ်ႈဝၼ်းၼိုင်ႈယဝ်ႉဝၼ်းၼိုင်ႈ။
2. **Comments:** ပဵၼ် "Sticky Note" (ၸေႈတိတ်း) ဢၼ်ၸဝ်ႈၵဝ်ႇတိတ်းဝႆႉၼိူဝ်ၼႃႈလိၵ်ႈ တွၼ်ႈတႃႇတือนၸႂ်ၸဝ်ႈၵဝ်ႇဝႃႈ "ယႃႇလိုမ်းမႃးၵႄႇၶိုၼ်းတွၼ်ႈၼႆႉ"၊ ၵူၺ်းၵႃႈ မၼ်းဢမ်ႇၸႂ်ႈၼိူဝ်ႉလိၵ်ႈတႄႉတႄႉၼႂ်းပပ်ႉၼၼ်ႉယဝ်ႉ။
