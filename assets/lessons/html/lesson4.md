## Lesson 4: The Meta Charset: Unlocking Shan Script (UTF-8)

တွၼ်ႈၼႆႉ ပဵၼ်တွၼ်ႈဢၼ်လမ်ႇလွင်ႈသုတ်း တႃႇၵူၼ်းတႅမ်ႈလိၵ်ႈတႆး (Shan Script) ၶႃႈဢေႃႈ။ သင်ဝႃႈႁဝ်းဢမ်ႇသႂ်ႇ Code တွၼ်ႈၼႆႉ၊ လိၵ်ႈတႆးႁဝ်းတေၼႄႁၢင်ႈပဵၼ် "တူဝ်လိၵ်ႈလူႉ" ဢမ်ႇၼၼ် ပဵၼ်လွၵ်းသီႇၸဵင်ႇၼၼ်ႉယဝ်ႉ။

---

#### 1. Character Encoding (ၵၢၼ်လႅၵ်ႈလၢႆႈတူဝ်လိၵ်ႈ)

ၶွမ်ႇပိဝ်ႇတႃႇၼႆႉ မၼ်းဢမ်ႇႁူႉၸၵ်း "တူဝ်လိၵ်ႈ"၊ မၼ်းႁူႉၸၵ်း "မၢႆလိၵ်ႈ" (Numbers) ၵူၺ်း။ **Character Encoding** ပဵၼ်ပိူင်ဢၼ်လၢတ်ႈပၼ်ၶွမ်ႇပိဝ်ႇတႃႇဝႃႈ "မၢႆၼႆႉ ပဵၼ်တူဝ်လိၵ်ႈတႆး **ၵ** ဢေႃႈ" ၼႆၼၼ်ႉယဝ်ႉ။

* **တူဝ်ယၢင်ႇ:**

```html
<meta charset="UTF-8">

```

---

#### 2. Why UTF-8? (ပဵၼ်သင်လူဝ်ႇၸႂ်ႉ UTF-8?)

မိူဝ်ႈၵွၼ်ႇ မီးပိူင်လၢႆမဵဝ်း (ၼင်ႇ ASCII ဢမ်ႇၼၼ် ISO)၊ ၵူၺ်းၵႃႈ ပိူင်ၸိူဝ်းၼၼ်ႉ ဢမ်ႇမီး "တီႈယူႇ" တႃႇလိၵ်ႈတႆးႁဝ်း။

* **Universal:** **UTF-8** ပဵၼ် Standard ဢၼ်ႁပ်ႉဢဝ်လႆႈၵူႈၽႃႇသႃႇၼႂ်းၵမ္ဘာႇ (Unicode)။
* **Shan Script Ready:** မၼ်းၸွႆႈပၼ် Browser ႁူႉဝႃႈ တေလႆႈလူ (Read) တူဝ်လိၵ်ႈတႆး၊ တူဝ်လိၵ်ႈမၢၼ်ႈ၊ လႄႈ တူဝ်လိၵ်ႈထႆး ႁႂ်ႈဢွၵ်ႇမႃးထုၵ်ႇမႅၼ်ႈ။

---

#### 3. Where to put it? (တေလႆႈဝႆႉတီႈလႂ်?)

Tag ၼႆႉ တေလႆႈဝႆႉၼႂ်းၵၢင် `<head>` သေ တေလႆႈဝႆႉ "ၼိူဝ်သုတ်း" ပိူၼ်ႈ ၵမ်းလဵဝ် ဝၢႆးသေပိုၼ်ႇ head tag ယဝ်ႉ။

* **တွၼ်ႈတႃႇ Programmer:** သင်ၸဝ်ႈၵဝ်ႇတႅမ်ႈလိၵ်ႈတႆးယဝ်ႉ ၼႃႈၸေႃးၼႄပဵၼ် `????` ဢမ်ႇၼၼ် `` ၼႆ၊ မၢႆပွင်ႇဝႃႈ ၸဝ်ႈၵဝ်ႇလိုမ်းသႂ်ႇ `meta charset="UTF-8"` ၼၼ်ႉယဝ်ႉ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉလူလိၵ်ႈဢၼ်ပဵၼ် "Code" (Secret Message):

1. **Without UTF-8:** မၼ်းတေမိူၼ်ၼင်ႇ ၸဝ်ႈၵဝ်ႇမီးလိၵ်ႈလပ်ႉ ၵူၺ်းၵႃႈ ဢမ်ႇမီး "ၵေႃး" (Key) တွၼ်ႈတႃႇၶေႇဢွၵ်ႇ။
2. **With UTF-8:** မၼ်းပဵၼ် "ၵေႃး" ဢၼ်လၢတ်ႈပၼ် Browser ဝႃႈ "ၶေႇလိၵ်ႈၼႆႉလူၺ်ႈပိူင် Unicode တႃႉ" ၼႆယဝ်ႉ။
