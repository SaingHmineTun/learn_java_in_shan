## Lesson 39: Unordered Lists: Creating Bullet Points

မိူဝ်ႈႁဝ်းမီးသဵၼ်ႈမၢႆ (List) ဢၼ်ဢမ်ႇလူဝ်ႇလႆႈၸႅၵ်ႇ "လမ်ႇတပ်ႉ" (Order) ဝႃႈ ဢၼ်လႂ်တေမႃးၵွၼ်ႇ ဢၼ်လႂ်တေမႃးၼႃႈၼၼ်ႉ ႁဝ်းၸႂ်ႉ **Unordered List** ၶႃႈဢေႃႈ။

#### 1. The Tag Structure

တွၼ်ႈတႃႇသၢင်ႈသဵၼ်ႈမၢႆၼႆႉ ႁဝ်းလူဝ်ႇၸႂ်ႉ Tag 2 မဵဝ်းႁူမ်ႈၵၼ်:

* **`<ul>` (Unordered List):** ပဵၼ် Tag ဢၼ်ႁေႃႇၵုမ်းသဵၼ်ႈမၢႆတင်းသဵင်ႈဝႆႉ။
* **`<li>` (List Item):** ပဵၼ် Tag ဢၼ်ၸႂ်ႉတွၼ်ႈတႃႇသႂ်ႇ "ႁူဝ်ၶေႃႈ" ၼိုင်ႈဢၼ်လႄႈ ၼိုင်ႈဢၼ် ၼႂ်းသဵၼ်ႈမၢႆၼၼ်ႉ။

---

#### 2. Structure Example (တူဝ်ယၢင်ႇၵၢၼ်တႅမ်ႈ)

```html
<h3>ၶူဝ်းၵိၼ်တႆး ဢၼ်ႁဝ်းလုၵ်ႉၸႂ်:</h3>
<ul>
  <li>ၶဝ်ႈသွႆးတႆး</li>
  <li>ထူဝ်ႇၼဝ်ႈၶႅပ်း</li>
  <li>ၼိူဝ်ႉသူမ်ႈ</li>
  <li>ၽၵ်းၵၢတ်ႇသူမ်ႈ</li>
</ul>

```

#### 3. Appearance (ႁၢင်ႈၾၢင်)

Browser တေၼႄပၼ် "မၢႆတွင်းတူင်ႇလမ်" (Bullet points) ဝႆႉၽၢႆႇၼႃႈၵူႈထႅဝ်။ ၸဝ်ႈၵဝ်ႇၸၢင်ႈလႅၵ်ႈလၢႆႈႁၢင်ႈ Bullet ၼႆႉလႆႈလူၺ်ႈ CSS ၼႂ်းဝၼ်းၼႃႈၶႃႈ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉတႅမ်ႈ **သဵၼ်ႈမၢႆသိုဝ်ႉၶူဝ်း** (Grocery List):

* ၸဝ်ႈၵဝ်ႇတေတႅမ်ႈဝႃႈ "သိုဝ်ႉၵိူဝ်၊ သိုဝ်ႉၼမ်ႉမၼ်း၊ သိုဝ်ႉၽၵ်းမီႇ"။ ၸဝ်ႈၵဝ်ႇတေလိူၵ်ႈသိုဝ်ႉဢၼ်လႂ်ၵွၼ်ႇၵေႃႈလႆႈ၊ မၼ်းဢမ်ႇမီးၶေႃႈမၵ်းမၼ်ႈဝႃႈ လူဝ်ႇလႆႈသိုဝ်ႉၵိူဝ်ၵွၼ်ႇၸင်ႇသိုဝ်ႉၼမ်ႉမၼ်းလႆႈ။ ၼႆႉယဝ်ႉပဵၼ်ပိူင် Unordered List ၶႃႈဢေႃႈ။
