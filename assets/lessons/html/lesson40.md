## Lesson 40: Ordered Lists: Step-by-Step Instructions

မိူဝ်ႈႁဝ်းမီးသဵၼ်ႈမၢႆဢၼ် "လမ်ႇတပ်ႉ" (Sequence) မၼ်းလမ်ႇလွင်ႈ၊ ၼင်ႇၵၢၼ်ၼႄလၢႆးႁဵတ်းသၢင်ႈပဵၼ်ၶၼ်ႉပဵၼ်တွၼ်ႈၼၼ်ႉ ႁဝ်းတေလႆႈၸႂ်ႉ **Ordered List** ၶႃႈဢေႃႈ။

---

#### 1. The Tag Structure

မိူၼ်ၵၼ်တင်း Unordered List ၼၼ်ႉၶႃႈယဝ်ႉ၊ မၼ်းၸႂ်ႉ Tag 2 မဵဝ်းႁူမ်ႈၵၼ်:

* **`<ol>` (Ordered List):** ပဵၼ် Tag ႁေႃႇၵုမ်းသဵၼ်ႈမၢႆဢၼ်မီးမၢႆၼပ်ႉ။
* **`<li>` (List Item):** ပဵၼ် Tag သႂ်ႇၼိူဝ်ႉလိၵ်ႈသဵၼ်ႈမၢႆ။

---

#### 2. Structure Example (တူဝ်ယၢင်ႇၵၢၼ်တႅမ်ႈ)

```html
<h3>လၢႆးႁဵတ်းၼမ်ႉၼိုင်ႈ (Step-by-Step):</h3>
<ol>
  <li>သုၵ်ႈၶဝ်ႈႁႂ်ႈသႅၼ်ႈ။</li>
  <li>သႂ်ႇၼမ်ႉၼႂ်းမေႃႈၶဝ်ႈ ႁႂ်ႈပေႃးသၢင်ႇထုၵ်ႇ။</li>
  <li>ၼဵၵ်းပုမ်ႇႁုင်သေ မုင်ႈမွင်းႁႂ်ႈမၼ်းသုၵ်း။</li>
</ol>

```

#### 3. The `type` Attribute

ၸဝ်ႈၵဝ်ႇၸၢင်ႈလိူၵ်ႈလႆႈဝႃႈ ၶႂ်ႈႁႂ်ႈမၼ်းၼႄမၢႆၼပ်ႉပဵၼ်ပိူင်လႂ်:

* `type="1"`: မၢႆၼပ်ႉ (1, 2, 3) - **Default**
* `type="A"`: တူဝ်လိၵ်ႈဢိင်းၵလဵတ်ႈတူဝ်လူင် (A, B, C)
* `type="a"`: တူဝ်လိၵ်ႈဢိင်းၵလဵတ်ႈတူဝ်ဢွၼ်ႇ (a, b, c)
* `type="I"`: မၢႆၼပ်ႉ Roman တူဝ်လူင် (I, II, III)
* `type="i"`: မၢႆၼပ်ႉ Roman တူဝ်ဢွၼ်ႇ (i, ii, iii)

---

#### 4. The `start` Attribute

သင်ၸဝ်ႈၵဝ်ႇဢမ်ႇၶႂ်ႈတႄႇလူၺ်ႈမၢႆ (1)၊ ၸဝ်ႈၵဝ်ႇၸၢင်ႈၸႂ်ႉ `start` တွၼ်ႈတႃႇမၵ်းမၼ်ႈမၢႆတႄႇမၼ်းလႆႈၶႃႈ။

```html
<ol start="5">
  <li>တွၼ်ႈလိၵ်ႈမၢႆ 5</li>
  <li>တွၼ်ႈလိၵ်ႈမၢႆ 6</li>
</ol>

```

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉတႅမ်ႈ **လၢႆးႁဵတ်းၵိၼ်** (Recipe):

* ၸဝ်ႈၵဝ်ႇတေလႆႈလၢတ်ႈဝႃႈ "1. ပိုတ်ႇၾႆး၊ 2. သႂ်ႇၼမ်ႉမၼ်း၊ 3. သႂ်ႇၽၵ်း"။ သင်ၵူၼ်းႁဵတ်းၸွမ်းၼၼ်ႉ ၵႂႃႇသႂ်ႇၽၵ်းၵွၼ်ႇပိုတ်ႇၾႆး (ဢမ်ႇၸွမ်းလမ်ႇတပ်ႉ)၊ ၽၵ်းၼၼ်ႉတေဢမ်ႇဝၢၼ်လႆႈ။ ၼႆႉယဝ်ႉပဵၼ်ပိူင် Ordered List ဢၼ်လမ်ႇတပ်ႉမၼ်းလမ်ႇလွင်ႈတႄႉတႄႉၼၼ်ႉယဝ်ႉ။
