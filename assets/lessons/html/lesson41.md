## Lesson 41: Description Lists: Terms and Definitions

ၵမ်းလၢႆး ႁဝ်းမီးသဵၼ်ႈမၢႆဢၼ်ဢမ်ႇၸႂ်ႈၵၢၼ်ၼပ်ႉမၢႆ (1, 2, 3) ဢမ်ႇၼၼ် Bullet points၊ ၵူၺ်းၵႃႈ ပဵၼ်ၵၢၼ်ပၼ် "ၶေႃႈၵႂၢမ်း" (Terms) လႄႈ "ၶေႃႈသပ်းလႅင်း" (Definitions) မၼ်း။ တွၼ်ႈတႃႇႁဵတ်းၼႆ ႁဝ်းၸႂ်ႉ **Description List** ၶႃႈဢေႃႈ။

---

#### 1. The Tag Structure

Description List ၼႆႉ ၸႂ်ႉ Tag 3 မဵဝ်းႁူမ်ႈၵၼ် တွၼ်ႈတႃႇသၢင်ႈပိူင်မၼ်း:

* **`<dl>` (Description List):** ပဵၼ် Tag ဢၼ်ႁေႃႇၵုမ်းသဵၼ်ႈမၢႆတင်းသဵင်ႈဝႆႉ။
* **`<dt>` (Description Term):** ပဵၼ် "ၸိုဝ်ႈ" ဢမ်ႇၼၼ် "ၶေႃႈၵႂၢမ်း" ဢၼ်ႁဝ်းၶႂ်ႈသပ်းလႅင်း (ၵႆႉၼႄဝႆႉၽၢႆႇသၢႆႉ ဢမ်ႇၼၼ် ပဵၼ်တူဝ်ၼႃ)။
* **`<dd>` (Description Details):** ပဵၼ် "ၶေႃႈသပ်းလႅင်း" ၶွင်ၶေႃႈၵႂၢမ်းၼၼ်ႉ (Browser ၵႆႉတေႁဵတ်းဝၢင်ႈလွၵ်း Indent ဝႆႉပၼ်ၶႃႈ)။

---

#### 2. Structure Example (တူဝ်ယၢင်ႇၵၢၼ်တႅမ်ႈ)

```html
<h3>ပတ်းၸေႃလိၵ်ႈတႆး (Shan Glossary):</h3>
<dl>
  <dt>မႂ်ႇသုင်</dt>
  <dd>ပဵၼ်ၶေႃႈၵႂၢမ်းၸႂ်ႉတွၼ်ႈတႃႇ တူၵ်ႉယွၼ်းၵၼ် မိူဝ်ႈထူပ်းၵၼ် ဢမ်ႇၼၼ် လၢတ်ႈၶွပ်ႈၸႂ်။</dd>

  <dt>ၶဝ်ႈသွႆး</dt>
  <dd>ပဵၼ်ဢႁႃႇရတႆး ဢၼ်ႁဵတ်းလူၺ်ႈမူၵ်ႉၶဝ်ႈၸဝ်ႈ သေ မီးၼမ်ႉသူပ်းဝၢၼ်။</dd>

  <dt>ၵႃတႆး</dt>
  <dd>ပဵၼ်ၵၢၼ်ၵႃပိူင်ၼိုင်ႈ ဢၼ်ၼႄပႃးၾိင်ႈထုင်းတႆး လႄႈ လွင်ႈဢူၼ်ႈဢွၼ်ႇ။</dd>
</dl>

```

---

#### 3. When to use Description Lists?

* **Glossaries:** မိူဝ်ႈသၢင်ႈပပ်ႉသပ်း (Dictionary) လဵၵ်ႉလဵၵ်ႉၼႂ်းဝႅပ်ႉသၢႆႉ။
* **Metadata:** ၼင်ႇၵၢၼ်ၼႄ "ၸိုဝ်ႈၽူႈတႅမ်ႈ: ၸၢႆးၶိူဝ်း"၊ "ဝၼ်းထီႉ: 2026"။
* **FAQ Sections:** တွၼ်ႈတႃႇသၢင်ႈ "ၶေႃႈထၢမ်" (dt) လႄႈ "ၶေႃႈတွပ်ႇ" (dd)။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉတႅမ်ႈ **ပပ်ႉသပ်း** (Dictionary):

* ၸဝ်ႈၵဝ်ႇတေမီး "တူဝ်လိၵ်ႈ" ဢၼ်ၶႂ်ႈႁႃ (Term) သေ မီး "တီႈပွင်ႇ" (Definition) ယူႇၽၢႆႇတႂ်ႈမၼ်း။ ၼႆႉယဝ်ႉပဵၼ်ပိူင် Description List ဢၼ်ၸွႆႈႁႂ်ႈၶေႃႈမုၼ်းၸဝ်ႈၵဝ်ႇ တူၺ်းငၢႆႈ လႄႈ မီးပိူင်မီႈ (Semantic) တႄႉတႄႉၼၼ်ႉယဝ်ႉ။
