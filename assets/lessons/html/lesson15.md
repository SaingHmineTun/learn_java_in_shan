## Lesson 15: The Span Tag: Targeting Small Text Segments

မိူဝ်ႈႁဝ်းႁဵၼ်းလွင်ႈ `<div>` ၼၼ်ႉ ႁဝ်းႁူႉဝႃႈမၼ်းပဵၼ်ၵွၵ်းလူင် (Block element) ဢၼ်တႄႇထႅဝ်မႂ်ႇၵူႈပွၵ်ႈ။ ၵူၺ်းၵႃႈ သင်ႁဝ်းၶႂ်ႈလိူၵ်ႈမၵ်းမၼ်ႈ "တူဝ်လိၵ်ႈၵမ်းၼိုင်ႈ" ဢမ်ႇၼၼ် "ၶေႃႈၵႂၢမ်းၶေႃႈၼိုင်ႈ" ဢၼ်ယူႇၼႂ်းၵၢင်ထႅဝ်လိၵ်ႈ တွၼ်ႈတႃႇမႄးသီ ဢမ်ႇၼၼ် မႄးႁၢင်ႈမၼ်းၼၼ်ႉ ႁဝ်းတေလႆႈၸႂ်ႉ `<span>` ၶႃႈဢေႃႈ။

---

#### 1. What is a Span? (Span ပဵၼ်သင်?)

`<span>` ပဵၼ် **Inline Container** ဢၼ်ဢမ်ႇမီး "တီႈပွင်ႇ" (Non-semantic) ၼင်ႇၵဝ်ႇ။ မၼ်းဢမ်ႇတႄႇထႅဝ်မႂ်ႇ၊ မၼ်းတေယူႇဝႆႉၸွမ်းထႅဝ်လိၵ်ႈ (Inline) ၼၼ်ႉၵမ်းသိုဝ်ႈယဝ်ႉ။

* **Inline behavior:** မၼ်းတေၵုမ်းၵမ်ဢဝ် တၢင်းၵႂၢင်ႈ (Width) တေႃႉတႄႉၵိုင်ႇမႅၼ်ႈၸွမ်း တူဝ်လိၵ်ႈဢၼ်ယူႇၼႂ်းမၼ်းၵူၺ်း။

---

#### 2. When to use it? (တေၸႂ်ႉမိူဝ်ႈလႂ်?)

ႁဝ်းလူဝ်ႇၸႂ်ႉ `<span>` မိူဝ်ႈ:

* ၶႂ်ႈပၼ်သီ (Color) တူဝ်လိၵ်ႈမၢင်တူဝ် ၼႂ်းတွၼ်ႈလိၵ်ႈ (`<p>`)။
* ၶႂ်ႈမႄး Font style တူဝ်လိၵ်ႈမၢင်ဢၼ် ႁႂ်ႈပႅၵ်ႇပိူၼ်ႈ။
* ၶႂ်ႈမၵ်းမၼ်ႈ (Mark) ၶေႃႈၵႂၢမ်းမၢင်မဵဝ်း တွၼ်ႈတႃႇၸႂ်ႉ JavaScript (ၼင်ႇ - ႁႂ်ႈမၼ်းတူင်ႉလႆႈ မိူဝ်ႈဢဝ် Mouse ၵႂႃႇၸီႉ)။

#### 3. Structure Example (တူဝ်ယၢင်ႇၵၢၼ်တႅမ်ႈ)

```html
<p>
  ပပ်ႉႁဵၼ်းၼႆႉပဵၼ် <span style="color: blue; font-weight: bold;">HTML</span> 
  တွၼ်ႈတႃႇၵူၼ်းတႅမ်ႈလိၵ်ႈတႆး။
</p>

```

*(ၼႂ်းတူဝ်ယၢင်ႇၼႆႉ တူဝ်လိၵ်ႈ "HTML" တေပဵၼ်သီသွမ်ႇ လႄႈ တူဝ်ၼႃ၊ ၵူၺ်းၵႃႈ လိၵ်ႈထႅဝ်တၢင်ႇၸိူဝ်းတေပဵၼ်တူဝ်လိၵ်ႈ Standard ယူႇၼင်ႇၵဝ်ႇၶႃႈ။)*

---

#### 4. Div vs Span (လွင်ႈပႅၵ်ႇပိူင်ႈ)

| Feature | `<div>` | `<span>` |
| --- | --- | --- |
| **Type** | Block-level (တႄႇထႅဝ်မႂ်ႇ) | Inline (ယူႇၼႂ်းထႅဝ်) |
| **Width** | တၢင်းၵႂၢင်ႈတင်းၼႃႈၸေႃး | တၢင်းၵႂၢင်ႈတေႃႉတႄႉတူဝ်လိၵ်ႈ |
| **Usage** | ၸတ်းဝၢင်း Layout (Sections) | မႄးတူဝ်လိၵ်ႈ (Specific words) |

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉတႅမ်ႈလိၵ်ႈၼိူဝ်ၸေႈ:

1. **Div:** မိူၼ်ၼင်ႇ ၸဝ်ႈၵဝ်ႇဢဝ်ပႅၼ်းသတီး (Highlighter) သေ ၶีดႇပၼ်ႇႁွပ်ႈ (Box) တွၼ်ႈလိၵ်ႈၼၼ်ႉတင်းသဵင်ႈ။
2. **Span:** မိူၼ်ၼင်ႇ ၸဝ်ႈၵဝ်ႇဢဝ်ပႅၼ်းသတီး ၶีดႇတူပ်ႉ (Highlight) ပၼ်ၼိူဝ်တူဝ်လိၵ်ႈၶေႃႈၼိုင်ႈၵူၺ်း တွၼ်ႈတႃႇႁႂ်ႈမၼ်းပႅၵ်ႇပိူၼ်ႈၼၼ်ႉယဝ်ႉ။
