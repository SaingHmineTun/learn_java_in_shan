## Lesson 2: The HTML Tag: The Root and Lang Attributes

မိူဝ်ႈႁဝ်းပိုၼ်ၽၢဝ်ႇ Doctype ယဝ်ႉယဝ်ႉၼၼ်ႉ တွၼ်ႈတေႃႇၵႂႃႇၵေႃႈ ပဵၼ်ၵၢၼ်တႄႇသၢင်ႈ "တူဝ်ပပ်ႉ" ႁဝ်းယဝ်ႉ။ တူဝ်လိၵ်ႈ HTML တင်းသဵင်ႈ တေလႆႈယူႇဝႆႉၼႂ်းၵၢင် `<html>` tag ၼႆႉၶႃႈဢေႃႈ။

---

#### 1. The Root Element (ငဝ်ႈမၼ်း)

`<html>` ၼႆႉ ႁဝ်းႁွင်ႉဝႃႈ **Root Element**။ မၼ်းပဵၼ် "ၵၢပ်ႇ" (Container) ဢၼ်ယႂ်ႇသုတ်း တွၼ်ႈတႃႇၵုမ်းၵမ်ဢဝ် tags တင်းၼမ် ႁႂ်ႈယူႇဝႆႉတီႈလဵဝ်ၵၼ်။ သင်ဢမ်ႇမီး tag ၼႆႉ Browser တေဢမ်ႇႁူႉဝႃႈ Code ႁဝ်းတႄႇတီႈလႂ် ယဝ်ႉတီႈလႂ်ၶႃႈ။

* **တူဝ်ယၢင်ႇ:**

```html
<html>
  </html>

```

---

#### 2. The Lang Attribute (ၵၢၼ်မၵ်းမၼ်ႈၽႃႇသႃႇ)

တွၼ်ႈၼႆႉ လၢတ်ႈလွင်ႈ **Attribute** ဢၼ်ပဵၼ်ၶေႃႈမုၼ်းလိူဝ် (Extra information) ပၼ် tag ၼၼ်ႉယဝ်ႉ။ `lang` ၼႆႉ ၸိူၼႄဝႃႈ ၼႃႈလိၵ်ႈဝႅပ်ႉသၢႆႉႁဝ်းၼႆႉ တႅမ်ႈဝႆႉလူၺ်ႈ "ၽႃႇသႃႇ" သင်။

* **တူဝ်ယၢင်ႇ:**

```html
<html lang="shn">

```

*(တီႈၼႆႉ `shn` ပွင်ႇဝႃႈ ၽႃႇသႃႇတႆး - Shan Language)*

---

#### 3. Why it matters (ပဵၼ်သင်လႄႈလူဝ်ႇလႆႈသႂ်ႇ)

* **Search Engines:** Google တေႁူႉဝႃႈ ဝႅပ်ႉသၢႆႉၼႆႉပဵၼ်ၽႃႇသႃႇသင် သေ တေၼႄပၼ် ၵူၼ်းဢၼ်ႁႃလိၵ်ႈၽႃႇသႃႇၼၼ်ႉ။
* **Screen Readers:** တွၼ်ႈတႃႇၵူၼ်းတႃမွတ်ႇ ဢၼ်ၸႂ်ႉ Software လူလိၵ်ႈပၼ်ၼၼ်ႉ မၼ်းတေလူဢွၵ်ႇသဵင်လႆႈ ထုၵ်ႇမႅၼ်ႈၸွမ်းသဵင်ၽႃႇသႃႇမၼ်း။
* **Browser Translation:** Browser တေၸၢင်ႈထၢမ်ဝႃႈ "တေပိၼ်ႇၽႃႇသႃႇ (Translate) ပၼ်ၼႃႈလိၵ်ႈၼႆႉႁိုဝ်?" လူၺ်ႈဢိင်ၼိူဝ် `lang` ဢၼ်ၼႆႉယဝ်ႉ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉသူင်ႇလိၵ်ႈ (Letter) ၵႂႃႇတီႈၵူၼ်းၵေႃႉၼိုင်ႈ:

1. **HTML Tag:** ပဵၼ် "သောင်းလိၵ်ႈ" (Envelope) ဢၼ်ၵုမ်းၵမ်လိၵ်ႈဝႆႉ။
2. **Lang Attribute:** ပဵၼ်ၵၢၼ်တႅမ်ႈဝႆႉၼိူဝ်သောင်းလိၵ်ႈဝႃႈ "လိၵ်ႈတႆး" (This is a Shan letter) တွၼ်ႈတႃႇႁႂ်ႈၵူၼ်းႁပ်ႉႁူႉဝႃႈ တေလႆႈလူလူၺ်ႈၽႃႇသႃႇသင်။
