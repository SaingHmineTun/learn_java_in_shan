## Lesson 26: Relative vs. Absolute Paths

မိူဝ်ႈႁဝ်းသၢင်ႈ Link ၼၼ်ႉ ႁဝ်းလူဝ်ႇလႆႈပၼ် "သဵၼ်ႈတၢင်း" (Path) တွၼ်ႈတႃႇ Browser တေၵႂႃႇႁႃၾၢႆႇ (File) ၼၼ်ႉလႆႈမႅၼ်ႈ။ လၢႆးပၼ်သဵၼ်ႈတၢင်းမီးယူႇ 2 မဵဝ်း ဢၼ်ပႅၵ်ႇပိူင်ႈၵၼ်ၶႃႈ။

---

#### 1. Absolute Paths (သဵၼ်ႈတၢင်းတႅတ်ႈၼႅတ်ႈ)

Absolute Path ပဵၼ်ၵၢၼ်ပၼ် "တီႈယူႇတင်းသဵင်ႈ" (Full Address)။ မၼ်းၵႆႉတႄႇလူၺ်ႈ `https://`။

* **မိူဝ်ႈလႂ်ၸႂ်ႉ:** ၸႂ်ႉမိူဝ်ႈႁဝ်းၶႂ်ႈၵွင်ႉၵၢႆႇ (Link) ၵႂႃႇၸူးဝႅပ်ႉသၢႆႉ **ၽၢႆႇၼွၵ်ႈ** (External sites) ၼင်ႇ Google, Facebook ဢမ်ႇၼၼ် ဝႅပ်ႉသၢႆႉတၢင်ႇဢၼ်။

**တူဝ်ယၢင်ႇ:**

```html
<a href="https://www.wikipedia.org">ၵႂႃႇၸူး Wikipedia</a>

```

---

#### 2. Relative Paths (သဵၼ်ႈတၢင်းၵွင်ႉၵၢႆႇ)

Relative Path ပဵၼ်ၵၢၼ်ပၼ်သဵၼ်ႈတၢင်း "လူၺ်ႈတူၺ်းတီႈယူႇၾၢႆႇဢၼ်တိုၵ်ႉၸႂ်ႉယူႇ"။ မၼ်းဢမ်ႇလူဝ်ႇပႃး Domain name (URL) တင်းသဵင်ႈ။

* **မိူဝ်ႈလႂ်ၸႂ်ႉ:** ၸႂ်ႉမိူဝ်ႈႁဝ်းၶႂ်ႈ Link ၵႂႃႇၸူးၼႃႈလိၵ်ႈတၢင်ႇဢၼ် **ၼႂ်းဝႅပ်ႉသၢႆႉႁဝ်းၵမ်းလဵဝ်** (Internal pages)။

**တူဝ်ယၢင်ႇ:**

* **ယူႇၼႂ်း Folder လဵဝ်ၵၼ်:** `href="about.html"`
* **ယူႇၼႂ်း Folder ယွႆႈ:** `href="lessons/lesson1.html"`
* **လုၵ်ႉၼႂ်း Folder ယွႆႈ ထွၼ်ဢွၵ်ႇမႃးၽၢႆႇၼွၵ်ႈ:** `href="../index.html"`

---

#### 3. Why use Relative Paths? (ပဵၼ်သင်လူဝ်ႇၸႂ်ႉ?)

1. **Portable:** သင်ၸဝ်ႈၵဝ်ႇလႅၵ်ႈလၢႆႈၸိုဝ်ႈဝႅပ်ႉသၢႆႉ (Domain) ဢမ်ႇၼၼ် ယၢႆႉ Code ၵႂႃႇဝႆႉတီႈ Server တၢင်ႇဢၼ်၊ Link ၸိူဝ်းၼၼ်ႉတေတိုၵ်ႉႁဵတ်းၵၢၼ်လႆႈယူႇ (ပေႃးၾၢႆႇၸိူဝ်းၼၼ်ႉယူႇၸွမ်းၵၼ်ဝႆႉ)။
2. **Speed:** တႅမ်ႈငၢႆႈ လႄႈ ၸတ်းၵၢၼ်ငၢႆႈလိူဝ် Absolute path တွၼ်ႈတႃႇၾၢႆႇၼႂ်းႁိူၼ်းၵမ်းလဵဝ်။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉလၢတ်ႈတၢင်းပၼ်ပိူၼ်ႈ:

1. **Absolute Path:** မိူၼ်ၼင်ႇပၼ်မၢႆႁိူၼ်းတႅတ်ႈၼႅတ်ႈ "ႁိူၼ်းမၢႆ (၁၀)၊ သဵၼ်ႈတၢင်း (၅)၊ ဝဵင်းတူင်ႇၵျီး"။ ၽႂ်ယူႇတီႈလႂ်ၵေႃႈ ၸွမ်းမႃးလႆႈမႅၼ်ႈ။
2. **Relative Path:** မိူၼ်ၼင်ႇပၼ်တၢင်းဝႃႈ "ၵႂႃႇတီႈႁွင်ႈၼွၼ်းၽၢႆႇသၢႆႉ"။ ၶေႃႈၵႂၢမ်းၼႆႉတေမႅၼ်ႈၵေႃႈ တေႃႉတႄႉမိူဝ်ႈၵူၼ်းၼၼ်ႉ **ယူႇဝႆႉၼႂ်းႁိူၼ်းၸဝ်ႈၵဝ်ႇ** ယဝ်ႉၵူၺ်း။
