## Lesson 46: Input Types: Text, Password, and Email

ၼႂ်း Lesson 44 ႁဝ်းလႆႈႁၼ်ယဝ်ႉဝႃႈ `type="text"` ၸႂ်ႉတွၼ်ႈတႃႇတႅမ်ႈလိၵ်ႈ။ ၵူၺ်းၵႃႈ HTML5 မီး `type` ထႅင်ႈတင်းၼမ် တွၼ်ႈတႃႇႁပ်ႉဢဝ်ၶေႃႈမုၼ်းလၢႆမဵဝ်း လႄႈ ၸွႆႈၵုမ်းၵမ်လွင်ႈထုၵ်ႇမႅၼ်ႈ (Validation) ၶႃႈ။

---

#### 1. Input Type: Text (`type="text"`)

ပဵၼ်ပိူင်ပိုၼ်ႉထၢၼ်သုတ်း တွၼ်ႈတႃႇတႅမ်ႈလိၵ်ႈထႅဝ်လဵဝ် ၼင်ႇ ၸိုဝ်ႈ၊ ၸိုဝ်ႈဝဵင်း၊ ဢမ်ႇၼၼ် ႁူဝ်လိၵ်ႈ။

```html
<label for="username">ၸိုဝ်ႈၽူႈၸႂ်ႉတိုဝ်း:</label>
<input type="text" id="username" name="username">

```

---

#### 2. Input Type: Password (`type="password"`)

မဵဝ်းၼႆႉလမ်ႇလွင်ႈတွၼ်ႈတႃႇ လွင်ႈၵၢင်းသႂ်ႇ (Security)။ မိူဝ်ႈၵူၼ်းၸႂ်ႉတိုဝ်းတႅမ်ႈလိၵ်ႈၼႂ်းလွၵ်းၼႆႉ၊ Browser တေဢမ်ႇၼႄတူဝ်လိၵ်ႈ၊ မၼ်းတေၼႄပဵၼ် "ၸုၵ်းလမ်" (`●●●●`) ဢမ်ႇၼၼ် "လၢဝ်" (`****`) တႅၼ်းၶႃႈ။

```html
<label for="pwd">ၵုၼ်းသေႃ (Password):</label>
<input type="password" id="pwd" name="pwd">

```

---

#### 3. Input Type: Email (`type="email"`)

မဵဝ်းၼႆႉၸွႆႈထတ်းတူၺ်းဝႃႈ လိၵ်ႈဢၼ်တႅမ်ႈသႂ်ႇၼၼ်ႉ မႅၼ်ႈၸွမ်းပိူင် Email (ၼင်ႇ မီး `@` လႄႈ `.com`) ၼၼ်ႉႁႃ။ သင်တႅမ်ႈၽိတ်း၊ မိူဝ်ႈၼဵၵ်း Submit Browser တေလၢတ်ႈၼႄဝႃႈ "တႅမ်ႈ Email ၽိတ်းဝႆႉၶႃႈ"။

* **💡 Pro Tip:** မိူဝ်ႈပိုတ်ႇၼႂ်းၾူၼ်း (Mobile), Browser တေပိုတ်ႇ Keyboard ဢၼ်မီးတူဝ် `@` ဝႆႉပၼ်ၵမ်းလဵဝ် ႁႂ်ႈတႅမ်ႈငၢႆႈၶႃႈ။

```html
<label for="email">Email ၸဝ်ႈၵဝ်ႇ:</label>
<input type="email" id="email" name="email">

```

---

#### 4. Additional Attributes

ႁဝ်းၸၢင်ႈသႂ်ႇ Attribute ၸိူဝ်းၼႆႉ တွၼ်ႈတႃႇၵုမ်းၵမ်လိူဝ်ႁႅင်း:

* **`required`**: တေလႆႈတႅမ်ႈသႂ်ႇၵူၺ်းၸင်ႇသူင်ႇလႆႈ (တေဝၢင်ႈလွၵ်းဝႆႉဢမ်ႇလႆႈ)။
* **`minlength` & `maxlength**`: မၵ်းမၼ်ႈတူဝ်လိၵ်ႈ ႁႂ်ႈဢမ်ႇဢေႇပူၼ်ႉ ဢမ်ႇၼၼ် ဢမ်ႇၼမ်ပူၼ်ႉ။

```html
<input type="password" name="pwd" required minlength="8">

```

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉ **ႁဵတ်းဝတ်းမၢႆ** (ID Card):

1. **`text`**: မိူၼ်ၼင်ႇ လွၵ်းတႅမ်ႈၸိုဝ်ႈ လႄႈ ပူႇမွၼ်ႇၼၢႆးယႃႈ။
2. **`password`**: မိူၼ်ၼင်ႇ လွၵ်းတႅမ်ႈ "မၢႆလပ်ႉ" ဢၼ်ၸဝ်ႈၵဝ်ႇတေဢဝ်မိုဝ်းႁူမ်ႇဝႆႉ ဢမ်ႇၶႂ်ႈႁႂ်ႈၵူၼ်းႁိမ်းႁၢင်ႈႁၼ်။
3. **`email`**: မိူၼ်ၼင်ႇ ၵူၼ်းႁဵတ်းဝတ်းမၢႆၼၼ်ႉ ထတ်းတူၺ်းဝႃႈ "မီး @ ဝႆႉယူႇႁႃ?" ၵွၼ်ႇတေႁပ်ႉဢဝ်ဝတ်းမၢႆၸဝ်ႈၵဝ်ႇၼၼ်ႉယဝ်ႉ။
