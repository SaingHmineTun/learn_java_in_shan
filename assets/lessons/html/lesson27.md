## Lesson 27: The Target Attribute: Control How Links Open

မိူဝ်ႈႁဝ်းၼဵၵ်း Link ၼိုင်ႈဢၼ်ၼၼ်ႉ၊ ပိူင်ပူၵ်ႉၵတိ (Default) မၼ်းတေပိုတ်ႇၼႃႈလိၵ်ႈမႂ်ႇၼၼ်ႉ တႅၼ်းတီႈၼႃႈလိၵ်ႈဢၼ်ႁဝ်းတိုၵ်ႉလူယူႇယၢမ်းလဵဝ်။ ၵူၺ်းၵႃႈ ၵမ်းလၢႆးႁဝ်းၶႂ်ႈႁႂ်ႈမၼ်း "ပိုတ်ႇၼႃႈမႂ်ႇ" (New Tab) ဝႆႉ ႁႂ်ႈဝႅပ်ႉသၢႆႉႁဝ်းဢမ်ႇႁၢႆၵႂႃႇ။ တီႈၼႆႉႁဝ်းၸႂ်ႉ attribute `target` ၶႃႈဢေႃႈ။

---

#### 1. The `target` Attribute

Attribute ၼႆႉ ၸႂ်ႉတွၼ်ႈတႃႇလၢတ်ႈပၼ် Browser ဝႃႈ "တေလႆႈပိုတ်ႇ Link ၼႆႉတီႈလႂ်?"။ တူဝ်ဢၼ်ႁဝ်းၵႆႉၸႂ်ႉသုတ်းမီး 2 ဢၼ်ၶႃႈ:

* **`_self` (Default):** ပိုတ်ႇၼႃႈလိၵ်ႈၼႂ်း Tab ဢၼ်ၵဝ်ႇ။ (သင်ဢမ်ႇသႂ်ႇ target သင်၊ မၼ်းတေပဵၼ်ဢၼ်ၼႆႉၵမ်းလဵဝ်ယဝ်ႉ)။
* **`_blank`:** ပိုတ်ႇၼႃႈလိၵ်ႈမႂ်ႇၼႂ်း **New Tab** ဢမ်ႇၼၼ် Window မႂ်ႇ။

---

#### 2. Structure Example (တူဝ်ယၢင်ႇၵၢၼ်တႅမ်ႈ)

```html
<a href="https://www.google.com" target="_blank">ၵႂႃႇၸူး Google (Tab မႂ်ႇ)</a>

<a href="about.html" target="_self">လွင်ႈႁဝ်း</a>

```

---

#### 3. Security Tip (လွင်ႈႁႅင်းၵၢင်ႇ) 🔒

မိူဝ်ႈၸဝ်ႈၵဝ်ႇၸႂ်ႉ `target="_blank"` တွၼ်ႈတႃႇၵႂႃႇၸူးဝႅပ်ႉသၢႆႉၽၢႆႇၼွၵ်ႈ (External Links)၊ မၼ်းမီးလွင်ႈၵၢင်းသႂ်ႇ (Security risk) လဵၵ်ႉလဵၵ်ႉဢၼ်ၼိုင်ႈ။ လူဝ်ႇလႆႈသႂ်ႇပႃး `rel="noopener"` ၸွမ်းၵူႈပွၵ်ႈၶႃႈ။

* **တူဝ်ယၢင်ႇဢၼ်လီသုတ်း:**

```html
<a href="https://example.com" target="_blank" rel="noopener">Link ဢၼ်ၵၢင်းသႂ်ႇ</a>

```

---

#### 4. When to use which? (တေၸႂ်ႉမိူဝ်ႈလႂ်?)

* **ၸႂ်ႉ `_blank`:** မိူဝ်ႈ Link ၼၼ်ႉ ၵႂႃႇၸူးဝႅပ်ႉသၢႆႉတၢင်ႇဢၼ် (External)။ ႁဝ်းဢမ်ႇၶႂ်ႈႁႂ်ႈၵူၼ်းလူလိၵ်ႈ ဢွၵ်ႇၵႂႃႇတီႈဝႅပ်ႉသၢႆႉႁဝ်း။
* **ၸႂ်ႉ `_self`:** မိူဝ်ႈ Link ၼၼ်ႉ ၵႂႃႇၸူးၼႃႈလိၵ်ႈတၢင်ႇဢၼ်ၼႂ်းဝႅပ်ႉသၢႆႉႁဝ်းၵမ်းလဵဝ် (Internal)။ မၼ်းႁဵတ်းႁႂ်ႈၵူၼ်းၸႂ်ႉတိုဝ်း ပွင်ႇၸႂ်ငၢႆႈလိူဝ်။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉလူ **ပပ်ႉလိၵ်ႈ**:

1. **Target `_self`:** မိူၼ်ၼင်ႇၸဝ်ႈၵဝ်ႇ "ပိုတ်ႇၼႃႈလိၵ်ႈထတ်းၵႂႃႇ"။ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉလူပပ်ႉဢၼ်ၵဝ်ႇယူႇ။
2. **Target `_blank`:** မိူၼ်ၼင်ႇ ၸဝ်ႈၵဝ်ႇ "ယုၵ်ႉဢဝ်ပပ်ႉထႅင်ႈပပ်ႉၼိုင်ႈမႃးပိုတ်ႇ" ဝႆႉၽၢႆႇၶၢင်ႈ။ ၸဝ်ႈၵဝ်ႇၸၢင်ႈတူၺ်းလႆႈတင်းသွင်ပပ်ႉ ၼႂ်းၶၢဝ်းယၢမ်းလဵဝ်ၵၼ်ယဝ်ႉ။
