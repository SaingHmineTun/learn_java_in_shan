## Lesson 28: ID Attributes: Linking to Sections on One Page

ၵမ်းလၢႆး ဝႅပ်ႉသၢႆႉႁဝ်းၸၢင်ႈမီးၼႃႈလိၵ်ႈဢၼ်ယၢဝ်းၼႃႇ (ၼင်ႇ Single Page Website)။ တႅၼ်းတီႈတေႁႂ်ႈၵူၼ်းၸႂ်ႉတိုဝ်း Scroll လူင်းၵႂႃႇႁင်းၶဝ်၊ ႁဝ်းၸၢင်ႈသၢင်ႈ Link ဢၼ်ၼဵၵ်းယဝ်ႉ "ၵျွၵ်း" (Jump) ၵႂႃႇၸူးတွၼ်ႈလႂ်တွၼ်ႈၼၼ်ႉ ၼႂ်းၼႃႈလိၵ်ႈလဵဝ်ၵၼ်လႆႈၶႃႈ။ တီႈၼႆႉႁဝ်းလူဝ်ႇၸႂ်ႉ **ID Attribute** ၶႃႈဢေႃႈ။

---

#### 1. What is an ID? (ID ပဵၼ်သင်?)

`id` ပဵၼ် "ၸိုဝ်ႈတႅတ်ႈၼႅတ်ႈ" ဢၼ်ႁဝ်းပၼ်ဝႆႉတီႈ Tag တူဝ်လႂ်တူဝ်ၼၼ်ႉ။

* **ပိူင်ထူး:** ၼႂ်းၼိုင်ႈၼႃႈလိၵ်ႈ (HTML File)၊ `id` ၸိုဝ်ႈၼိုင်ႈဢၼ် လူဝ်ႇမီးလႆႈတီႈ Tag ၼိုင်ႈတူဝ်ၵူၺ်း (Unique)။ မၼ်းမိူၼ်ၼင်ႇ မၢႆၾူၼ်း ဢမ်ႇၼၼ် မၢႆမတ်ႉတိုင်ႇၵူၼ်းမိူင်း ဢၼ်ဢမ်ႇလႆႈမိူၼ်ၵၼ်ၼၼ်ႉယဝ်ႉ။

#### 2. How to Jump to an ID? (လၢႆးၵျွၵ်းၵႂႃႇၸူး)

တွၼ်ႈတႃႇ Link ၵႂႃႇၸူး ID ၼၼ်ႉ၊ ၼႂ်း `href` ႁဝ်းတေလႆႈသႂ်ႇမၢႆ **#** (Hashtag) ၽၢႆႇၼႃႈၸိုဝ်ႈ ID ၼၼ်ႉၶႃႈ။

---

#### 3. Structure Example (တူဝ်ယၢင်ႇၵၢၼ်တႅမ်ႈ)

```html
<nav>
  <a href="#history">ပိုၼ်းတႆး</a> | 
  <a href="#culture">ၾိင်ႈထုင်း</a>
</nav>

<div style="height: 1000px;">
  </div>

<h2 id="history">ပိုၼ်းတႆး</h2>
<p>ၼိူဝ်ႉလိၵ်ႈလွင်ႈပိုၼ်း...</p>

<div style="height: 1000px;"></div>

<h2 id="culture">ၾိင်ႈထုင်း</h2>
<p>ၼိူဝ်ႉလိၵ်ႈလွင်ႈၾိင်ႈထုင်း...</p>

<a href="#">မိူဝ်းၶိုၼ်းၼိူဝ်သုတ်း</a>

```

---

#### 4. Top of the Page Shortcut

သင်ၸဝ်ႈၵဝ်ႇသႂ်ႇ `href="#"` လၢႆလၢႆ (ဢမ်ႇပႃးၸိုဝ်ႈ ID)၊ Browser တေဢဝ်ၸဝ်ႈၵဝ်ႇမိူဝ်းဝႆႉတီႈ "ၼိူဝ်သုတ်း" (Top) ၶွင်ၼႃႈလိၵ်ႈၼၼ်ႉၵမ်းလဵဝ်ယဝ်ႉ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉလူ **ပပ်ႉလိၵ်ႈလူင်** ဢၼ်မီးၼႃႈလိၵ်ႈ ၅၀၀:

1. **ID Attribute:** မိူၼ်ၼင်ႇၸဝ်ႈၵဝ်ႇဢဝ် "ၸေႈသီႇ" (Bookmark) တိတ်းဝႆႉတီႈၼႃႈလိၵ်ႈ ၁၀၀ သေ တႅမ်ႈဝႆႉဝႃႈ "တွၼ်ႈ ၃"။
2. **Anchor with #ID:** မိူၼ်ၼင်ႇ ၸဝ်ႈၵဝ်ႇတူၺ်းတီႈ "သဵၼ်ႈမၢႆႁူဝ်လိၵ်ႈ" (Table of Contents) သေ ၼဵၵ်းပိုတ်ႇၵႂႃႇၸူး ၼႃႈလိၵ်ႈ ၁၀၀ ၼၼ်ႉၵမ်းလဵဝ် လူၺ်ႈဢမ်ႇလူဝ်ႇပိုတ်ႇၵႂႃႇၸွမ်းလႅၼ်မၼ်းယဝ်ႉ။
