## Lesson 17: The Paragraph: Writing in Shan Blocks

ဝၢႆးသေႁဝ်းမီးႁူဝ်လိၵ်ႈ (Headings) ယဝ်ႉ၊ တွၼ်ႈတေႃႇၵႂႃႇပဵၼ်ၵၢၼ်တႅမ်ႈ "ၼိူဝ်ႉလိၵ်ႈ" တႄႉတႄႉယဝ်ႉၶႃႈ။ ၼႂ်း HTML ႁဝ်းၸႂ်ႉ tag `<p>` တွၼ်ႈတႃႇသၢင်ႈတွၼ်ႈလိၵ်ႈ (Paragraphs)။

---

#### 1. The Paragraph Tag (တွၼ်ႈလိၵ်ႈ)

`<p>` ပဵၼ်တူဝ်ယေႃႈၶေႃႈၵႂၢမ်း **Paragraph**။ မိူဝ်ႈႁဝ်းသႂ်ႇလိၵ်ႈၼႂ်း tag ၼႆႉ၊ Browser တေၸတ်းဝၢင်းပၼ် ႁႂ်ႈမၼ်းမီးလွၵ်းဝၢင်ႈ (Margin) ၼိူဝ်လႄႈတႂ်ႈ ႁႂ်ႈတူဝ်လိၵ်ႈဢမ်ႇသိၵ်းၵၼ် လႄႈ ႁႂ်ႈလူငၢႆႈၶႃႈ။

* **တူဝ်ယၢင်ႇ:**

```html
<p>မႂ်ႇသုင်ၶႃႈ၊ ၼႆႉပဵၼ်လိၵ်ႈတႆး ဢၼ်တႅမ်ႈဝႆႉၼႂ်း tag paragraph ဢေႃႈ။</p>
<p>သင်ႁဝ်းတႅမ်ႈထႅင်ႈတွၼ်ႈၼိုင်ႈ၊ မၼ်းတေတႄႇထႅဝ်မႂ်ႇပၼ်ၵမ်းလဵဝ်ယဝ်ႉ။</p>

```

---

#### 2. Writing in Shan Script (ၵၢၼ်တႅမ်ႈလိၵ်ႈတႆး)

မိူဝ်ႈႁဝ်းတႅမ်ႈလိၵ်ႈတႆးၼႂ်း `<p>` tag ၼၼ်ႉ၊ မီးလွင်ႈလမ်ႇလွင်ႈဢၼ်လူဝ်ႇတွင်းယူႇၶႃႈ:

* **No Auto-Breaking:** လိၵ်ႈတႆးႁဝ်းၼႆႉ ၵမ်ႉၼမ်ဢမ်ႇၸႂ်ႉ "လွၵ်းဝၢင်ႈ" (Space) ၼႂ်းၵၢင်ၶေႃႈၵႂၢမ်း။ Browser မၢင်ၸိူဝ်းၸၢင်ႈတေဢမ်ႇႁူႉဝႃႈ တေလႆႈတတ်းထႅဝ် (Wrap) တီႈလႂ်။
* **Solution:** ႁဝ်းၸၢင်ႈၸႂ်ႉ CSS (`word-break: keep-all;` ဢမ်ႇၼၼ် `overflow-wrap: break-word;`) တွၼ်ႈတႃႇၸွႆႈပၼ် ႁႂ်ႈမၼ်းတတ်းထႅဝ်လႆႈလီလီၼႂ်းၾူၼ်းၶႃႈ။

---

#### 3. White Space in HTML (လွၵ်းဝၢင်ႈၼႂ်း Code)

ၼႂ်း HTML ၼၼ်ႉ သင်ၸဝ်ႈၵဝ်ႇၼဵၵ်း "Enter" တႅမ်ႈလိၵ်ႈလၢႆလၢႆထႅဝ်ၼႂ်း Code ၵေႃႈ၊ Browser တေဢဝ်မႃးၸပ်းၵၼ်တင်းသဵင်ႈယူႇၶႃႈ။ သင်ၶႂ်ႈလႆႈထႅဝ်မႂ်ႇ (New block)၊ တေလႆႈၸႂ်ႉ `<p>` tag မႂ်ႇထႅင်ႈဢၼ်ၼိုင်ႈၵူၺ်း။

* **တူဝ်ယၢင်ႇ (ဢၼ်ၽိတ်း):**

```html
<p>
  လိၵ်ႈထႅဝ် ၁
  လိၵ်ႈထႅဝ် ၂
</p>

```

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉၸႂ်ႉ Microsoft Word:

1. **Paragraph Tag:** မိူဝ်ႈၸဝ်ႈၵဝ်ႇတႅမ်ႈလိၵ်ႈယဝ်ႉသေ ၼဵၵ်း "Enter" ၼၼ်ႉယဝ်ႉ။ မၼ်းတေတႄႇတွၼ်ႈလိၵ်ႈမႂ်ႇ ဢၼ်မီးလွၵ်းဝၢင်ႈပၼ် ႁႂ်ႈတူဝ်လိၵ်ႈဢမ်ႇသိၵ်းၵၼ်ၼၼ်ႉယဝ်ႉ။
