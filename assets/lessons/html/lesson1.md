## Lesson 1: The Doctype: Declaring the Version

ၼႃႈလိၵ်ႈ HTML ၵူႈဢၼ် တေလႆႈတႄႇလူၺ်ႈ **Doctype declaration** ၶႃႈဢေႃႈ။ မၼ်းဢမ်ႇၸႂ်ႈ HTML tag၊ မၼ်းပဵၼ် "ၵႂၢမ်းပိုၼ်ၽၢဝ်ႇ" တႃႇပၼ် Browser ႁူႉဝႃႈ ႁဝ်းတေၸႂ်ႉ HTML ဝႃးသျိၼ်းဢၼ်လႂ်ၼၼ်ႉယဝ်ႉ။

---

#### 1. Doctype (ၵၢၼ်ပိုၼ်ၽၢဝ်ႇ)
**Doctype** ပဵၼ်တွၼ်ႈတႃႇပၼ်သัญญาလူင်ႈၼႃႈ တီႈ Browser (ၼင်ႇ Chrome ဢမ်ႇၼၼ် Firefox) ဝႃႈ "ၼႆႉပဵၼ်လိၵ်ႈ HTML5 ဢေႃႈ"။ သင်ဝႃႈႁဝ်းဢမ်ႇသႂ်ႇ Doctype၊ Browser တေထင်ဝႃႈပဵၼ်ဝႅပ်ႉသၢႆႉၵဝ်ႇ လႄႈ ႁၢင်ႈၾၢင်မၼ်းတေၸၢင်ႈလူႉလႅဝ်လႆႈ။

* **တူဝ်ယၢင်ႇ:** 
```html
<!DOCTYPE html>
```
---

#### 2. Why it is needed (ပဵၼ်သင်လႄႈလူဝ်ႇလႆႈၸႂ်ႉ)

မိူဝ်ႈၶွမ်ႇပိဝ်ႇတႃႇလူလိၵ်ႈ Code ႁဝ်းၼၼ်ႉ မၼ်းလူဝ်ႇလႆႈမီး "Standard" တွၼ်ႈတႃႇၼႄႁၢင်ႈ။

* **Standard Mode:** ႁဵတ်းႁႂ်ႈ HTML လႄႈ CSS ႁဝ်းၼႄႁၢင်ႈလႆႈထုၵ်ႇမႅၼ်ႈၸွမ်းပိူင်။
* **Quirks Mode:** သင်ဢမ်ႇသႂ်ႇ Doctype, Browser တေၼႄႁၢင်ႈၸွမ်းပိူင်ၵဝ်ႇ ဢၼ်ဢမ်ႇမီး Standard လႄႈ တေႁဵတ်းႁႂ်ႈ Programmer တုၵ်ႉယၢၵ်ႈ။

---

#### 3. Structure (လွၵ်းလၢႆးတႅမ်ႈ)

Doctype ၼႆႉ တေလႆႈဝႆႉတီႈ "ၼိူဝ်သုတ်း" ၵူၺ်း။

* **Case Insensitive:** တူဝ်လိၵ်ႈ `doctype` ၼၼ်ႉ တေတႅမ်ႈတူဝ်လူင် `<!DOCTYPE html>` ဢမ်ႇၼၼ် တူဝ်ဢွၼ်ႇ `<!doctype html>` ၵေႃႈ လႆႈၼင်ႇၵၼ်ၶႃႈ။
* **တွၼ်ႈတႃႇ Programmer:** ၼႆႉပဵၼ် "ၵၢပ်ႇ" မၼ်းဢၼ်လူဝ်ႇလႆႈသႂ်ႇၵူႈပွၵ်ႈ မိူဝ်ႈတႄႇသၢင်ႈ file HTML မႂ်ႇ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉတႄႇတႅမ်ႈလိၵ်ႈၼႂ်းပပ်ႉ (Book):

1. **Doctype:** ပဵၼ်ၼႃႈ "Title Page" ဢၼ်လၢတ်ႈဝႃႈ ပပ်ႉၼႆႉပဵၼ်ပပ်ႉသင်။
2. **HTML Tag:** ပဵၼ် "ၵၢပ်ႇပပ်ႉ" (The Cover).
3. **Content:** ပဵၼ် "တူဝ်လိၵ်ႈ" ဢၼ်ယူႇဝႆႉၼႂ်းပပ်ႉ။
