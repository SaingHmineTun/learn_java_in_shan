## Lesson 43: Table Caption, Thead, Tbody, and Tfoot

မိူဝ်ႈ Table ႁဝ်းမီးၶေႃႈမုၼ်းၼမ်မႃး၊ ႁဝ်းလူဝ်ႇၸႂ်ႉ Tag ဢၼ်ၸႅၵ်ႇ "တွၼ်ႈ" မၼ်းႁႂ်ႈပဵၼ်ပိူင်မီႈ (Semantic)။ ၼႆႉတေၸွႆႈႁႂ်ႈ Browser လႄႈ Software ၵူၼ်းတႃမွတ်ႇ ပွင်ႇၸႂ်ဝႃႈ တွၼ်ႈလႂ်ပဵၼ်ႁူဝ်၊ တွၼ်ႈလႂ်ပဵၼ်ၼိူဝ်ႉ၊ လႄႈ တွၼ်ႈလႂ်ပဵၼ်တွၼ်ႈပိတ်းၶႃႈ။

---

#### 1. The Semantic Table Tags

* **`<caption>`**: ၸႂ်ႉတွၼ်ႈတႃႇပၼ် "ၸိုဝ်ႈ Table"။ မၼ်းတေလူဝ်ႇယူႇဝႆႉပႃႈတႂ်ႈ tag `<table>` ၵမ်းလဵဝ်။
* **`<thead>`**: ႁေႃႇၵုမ်း "ထႅဝ်ႁူဝ်ၶေႃႈ" (Header rows) ဝႆႉ။
* **`<tbody>`**: ႁေႃႇၵုမ်း "ၼိူဝ်ႉလိၵ်ႈၶေႃႈမုၼ်း" (Body data) ပိူင်လူင်။
* **`<tfoot>`**: ႁေႃႇၵုမ်း "တွၼ်ႈပိတ်း" (Footer) ၼင်ႇ ၽွၼ်းလပ်ႉႁူမ်ႈ (Total)။

---

#### 2. Structure Example (တူဝ်ယၢင်ႇၵၢၼ်တႅမ်ႈ)

```html
<table border="1">
  <caption>သဵၼ်ႈမၢႆၵႃႈၶၼ်ၽၵ်း - ဝၼ်းၼႆႉ</caption>
  
  <thead>
    <tr>
      <th>ၸိုဝ်ႈၽၵ်း</th>
      <th>ၵႃႈၶၼ် (ၵျပ်ႉ)</th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td>ၽၵ်းၵၢတ်ႇသူမ်ႈ</td>
      <td>၁၅၀၀</td>
    </tr>
    <tr>
      <td>ထူဝ်ႇၼဝ်ႈၶႅပ်း</td>
      <td>၁၀၀၀</td>
    </tr>
  </tbody>

  <tfoot>
    <tr>
      <td>ႁူမ်ႈတင်းသဵင်ႈ</td>
      <td>၂၅၀၀</td>
    </tr>
  </tfoot>
</table>

```

---

#### 3. Why use these tags? (ပဵၼ်သင်လူဝ်ႇၸႂ်ႉ?)

1. **Print Friendly**: မိူဝ်ႈပိုတ်ႇ Print Table ယႂ်ႇယႂ်ႇဢၼ်မီးလၢႆၼႃႈၸေႃ၊ Browser မၢင်ၸိူဝ်းတေဢဝ် `<thead>` မႃးၼႄပၼ်သိုပ်ႇၵၼ်ၵူႈၼႃႈ ႁႂ်ႈလူငၢႆႈၶႃႈ။
2. **Styling**: ၸဝ်ႈၵဝ်ႇၸၢင်ႈမႄးသီ (Color) ႁႂ်ႈၼႃႈလိၵ်ႈ `<thead>` ပဵၼ်သီၼိုင်ႈ၊ `<tbody>` ပဵၼ်သီၼိုင်ႈ လႆႈငၢႆႈငၢႆႈလူၺ်ႈ CSS။
3. **Accessibility**: ၸွႆႈႁႂ်ႈ Screen readers ဢၢၼ်ႇၶေႃႈမုၼ်းပၼ် ၸွမ်းၼင်ႇတွၼ်ႈမၼ်းမႅၼ်ႈ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉတူၺ်း **ဝႂ်ပၼ်ႇငိုၼ်း** (Receipt):

* **`<caption>`**: ပဵၼ်ၸိုဝ်ႈလၢၼ်ႉဢၼ်ယူႇပႃႈၼိူဝ်သုတ်း။
* **`<thead>`**: ပဵၼ်ထႅဝ်ဢၼ်တႅမ်ႈဝႃႈ "ၸိုဝ်ႈၶူဝ်း" လႄႈ "ၵႃႈၶၼ်"။
* **`<tbody>`**: ပဵၼ်သဵၼ်ႈမၢႆၶူဝ်းၸိူဝ်းဢၼ်ၸဝ်ႈၵဝ်ႇသိုဝ်ႉၵႂႃႇတင်းသဵင်ႈ။
* **`<tfoot>`**: ပဵၼ်တွၼ်ႈတႂ်ႈသုတ်း ဢၼ်ၼႄဝႃႈ "ငိုၼ်းႁူမ်ႈတင်းသဵင်ႈ" ၼၼ်ႉယဝ်ႉ။
