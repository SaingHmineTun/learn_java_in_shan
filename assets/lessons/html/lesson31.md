## Lesson 31: Building a Navigation Menu in Pure HTML

ယၢမ်းလဵဝ် ႁဝ်းႁဵၼ်းႁူႉလွင်ႈ Link မဵဝ်းပႅၵ်ႇပိူင်ႈၵၼ်တင်းၼမ်ယဝ်ႉ။ တွၼ်ႈသုတ်းသေပိူၼ်ႈၼႂ်း Module ၼႆႉ ႁဝ်းတေဢဝ်တၢင်းႁူႉၸိူဝ်းၼၼ်ႉ မႃးႁူမ်ႈၵၼ်သၢင်ႈ **Navigation Menu** (သဵၼ်ႈတၢင်းပၼ်ႇၼႂ်းဝႅပ်ႉ) ဢၼ်ပဵၼ်ႁူဝ်ใจဝႅပ်ႉသၢႆႉၶႃႈဢေႃႈ။

---

#### 1. The Structure (ပိူင်သၢင်ႈ)

တွၼ်ႈတႃႇသၢင်ႈ Menu ဢၼ်မီးပိူင်မီႈ (Semantic HTML) ၼၼ်ႉ ႁဝ်းၵႆႉၸႂ်ႉ Tag 3 မဵဝ်းႁူမ်ႈၵၼ်:

* **`<nav>`**: တွၼ်ႈတႃႇမၵ်းမၼ်ႈဝႃႈ ၼႆႉပဵၼ်တွၼ်ႈ "ၵၢၼ်ပႆတၢင်း" (Navigation)။
* **`<ul>`**: တွၼ်ႈတႃႇသၢင်ႈ "သဵၼ်ႈမၢႆဢၼ်ဢမ်ႇမီးမၢႆၼပ်ႉ" (Unordered List)။
* **`<a>`**: တွၼ်ႈတႃႇသၢင်ႈ "တူဝ်ၵွင်ႉ" (Links)။

---

#### 2. Building the Menu (ၵၢၼ်သၢင်ႈ)

ၼႆႉပဵၼ် Code တူဝ်ယၢင်ႇ တွၼ်ႈတႃႇဝႅပ်ႉသၢႆႉဢၼ်မီးလၢႆၼႃႈလိၵ်ႈၶႃႈ:

```html
<nav>
  <ul>
    <li><a href="index.html">ၼႃႈႁႅၵ်ႈ (Home)</a></li>
    <li><a href="about.html">လွင်ႈႁဝ်း (About)</a></li>
    <li><a href="services.html">ၵၢၼ်ၸွႆႈထႅမ် (Services)</a></li>
    <li><a href="contact.html">ၵွင်ႉသၢၼ် (Contact)</a></li>
  </ul>
</nav>

```

---

#### 3. Why use a List (`<ul>`)?

ပဵၼ်သင်ႁဝ်းဢမ်ႇတႅမ်ႈ `<a>` ၸပ်းၵၼ်ၵႂႃႇလၢႆလၢႆ?

1. **Screen Readers:** တွၼ်ႈတႃႇၵူၼ်းတႃမွတ်ႇ ဢၼ်ၸႂ်ႉ Software လူလိၵ်ႈပၼ်ၼၼ်ႉ မၼ်းတေလၢတ်ႈပၼ်ဝႃႈ "ၼႆႉပဵၼ်သဵၼ်ႈမၢႆ Link မီး 4 ဢၼ်" ဢၼ်ၼႆႉၸွႆႈႁႂ်ႈၶဝ်ပွင်ႇၸႂ်ငၢႆႈၶႃႈ။
2. **Styling:** မိူဝ်ႈႁဝ်းႁဵၼ်း CSS ထႅင်ႈၼႃႈၼၼ်ႉ ၵၢၼ်မႄးႁၢင်ႈ Menu ဢၼ်မီး `<ul>` လႄ `<li>` ၼၼ်ႉ မၼ်းငၢႆႈလိူဝ် လႄႈ ၸတ်းဝၢင်းႁၢင်ႈလီလိူဝ်ၶႃႈ။

---

#### 4. Navigation within the Same Page

သင်ဝႅပ်ႉသၢႆႉၸဝ်ႈၵဝ်ႇမီးၼႃႈလိၵ်ႈလဵဝ် (One Page Website)၊ ၸဝ်ႈၵဝ်ႇၸၢင်ႈၸႂ်ႉ **ID Link** (တွၼ်ႈ ၂၈) မႃးသၢင်ႈ Menu လႆႈၶႃႈ:

```html
<nav>
  <ul>
    <li><a href="#home">ၼႃႈႁႅၵ်ႈ</a></li>
    <li><a href="#history">ပိုၼ်းတႆး</a></li>
    <li><a href="#gallery">ႁွင်ႈၼႄႁၢင်ႈ</a></li>
  </ul>
</nav>

```

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉယူႇၼႂ်း **ႁူင်းႁႅမ်းလူင်** (Hotel):

1. **`<nav>`:** မိူၼ်ၼင်ႇ "ပၢႆႉပိုၼ်ၽၢဝ်ႇလူင်" ဢၼ်တိတ်းဝႆႉတီႈၼႃႈႁွင်ႈႁပ်ႉၶႅၵ်ႇ။
2. **`<ul>` လႄႈ `<li>`:** မိူၼ်ၼင်ႇ "သဵၼ်ႈမၢႆၸိုဝ်ႈႁွင်ႈ" ဢၼ်လႅၵ်ႈၸႅၵ်ႇဝႆႉပဵၼ်ထႅဝ်ပဵၼ်ထႅဝ်။
3. **`<a>`:** မိူၼ်ၼင်ႇ "ၵူၼ်းဢွၼ်တၢင်း" ဢၼ်တေဢွၼ်ၸဝ်ႈၵဝ်ႇၵႂႃႇၸူး ႁွင်ႈဢၼ်ၸဝ်ႈၵဝ်ႇၶႂ်ႈၵႂႃႇၼၼ်ႉယဝ်ႉ။
