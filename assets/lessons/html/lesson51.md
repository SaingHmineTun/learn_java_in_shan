## Lesson 51: GitHub: Uploading the Manuscript Folder

ယၢမ်းလဵဝ် ဝႅပ်ႉသၢႆႉၸဝ်ႈၵဝ်ႇၵေႃႈယဝ်ႉတူဝ်ႈ၊ Code ၸဝ်ႈၵဝ်ႇၵေႃႈသႅၼ်ႈသႂ်ယဝ်ႉ။ ၵူၺ်းၵႃႈ မၼ်းတိုၵ်ႉမီးဝႆႉၼႂ်း Computer ၸဝ်ႈၵဝ်ႇၵူၺ်း။ တွၼ်ႈတႃႇတေဢဝ်ဝႅပ်ႉသၢႆႉၼႆႉ ပိုၼ်ၽၢဝ်ႇၼိူဝ် Internet လႆႈၼၼ်ႉ ႁဝ်းလူဝ်ႇဢဝ်ၾၢႆႇ (Files) ႁဝ်းၵႂႃႇဝႆႉတီႈ **GitHub** ၶႃႈဢေႃႈ။

---

#### 1. What is GitHub?

**GitHub** မိူၼ်ၼင်ႇ "Cloud Storage" တွၼ်ႈတႃႇ Programmer ၶဝ်။ မၼ်းပဵၼ်တီႈၵဵပ်းသိမ်း Code ဢၼ်မီးပိူင်မီႈ လႄႈ ၸွႆႈၵုမ်းၵမ် "Version" ၶွင် Code (ၼင်ႇ: Code ဝၼ်းမိူဝ်ႈၼႆႉ လႄႈ Code ဢၼ်မႄးဝႆႉမိူဝ်ႈဝႃး)။

---

#### 2. How to Create a GitHub Account (လၢႆးသၢင်ႈ Account)

ဢွၼ်တၢင်းသုတ်း ၸဝ်ႈၵဝ်ႇလူဝ်ႇမီးႁူိၼ်းတွၼ်ႈတႃႇၵဵပ်း Code ၵွၼ်ႇၶႃႈ။

1. ၵႂႃႇတီႈ [github.com](https://github.com) သေ ၼဵၵ်းပုမ်ႇ **Sign Up**။
2. သႂ်ႇ **Email** ၸဝ်ႈၵဝ်ႇ သေ သၢင်ႈ **Password** ဢၼ်ၶႅင်ႁႅင်း။
3. လိူၵ်ႈ **Username** (ၸိုဝ်ႈဢၼ်တေၼႄၼႂ်း GitHub ၼင်ႇ `sai-dev-2026`)။
4. Verify တူၺ်းဝႃႈ ၸဝ်ႈၵဝ်ႇပဵၼ်ၵူၼ်းတႄႉႁႃ (ၼဵၵ်း Puzzle ဢွၼ်ႇ)။
5. ဢဝ်မၢႆလပ်ႉ 6 တူဝ် ဢၼ်ၶဝ်သူင်ႇၸူး Email ၸဝ်ႈၵဝ်ႇၼၼ်ႉ မႃးသႂ်ႇသေ Confirm။

---

#### 3. How to Upload (လၢႆးဢဝ် Code ၶိုၼ်ႈ)

မိူဝ်ႈမီး Account ယဝ်ႉ၊ ႁဝ်းတေဢဝ် Folder ဝႅပ်ႉသၢႆႉႁဝ်း ၶိုၼ်ႈၵႂႃႇဝႆႉၶႃႈ:

1. **Create a Repository:** သၢင်ႈ "ဢႅင်ႇၵဵပ်းၶေႃႈမုၼ်း" (Repository) မႂ်ႇၼိုင်ႈဢၼ် သေ ပၼ်ၸိုဝ်ႈဝႃႈ `my-first-website`။
2. **Upload Files:** ၼဵၵ်းပုမ်ႇ "uploading an existing file" တီႈၼႃႈလိၵ်ႈ Repository ၼၼ်ႉ။
3. **Drag and Drop:** တရွတ်ႈ (Drag) ဢဝ် Folder ဝႅပ်ႉသၢႆႉၸဝ်ႈၵဝ်ႇ (index.html, images, etc.) သေ ဝၢင်းသႂ်ႇ။
4. **Commit Changes:** တႅမ်ႈလိၵ်ႈမၢႆတွင်းဢွၼ်ႇဝႃႈ "First upload" သေ ၼဵၵ်းပုမ်ႇ **Commit**။

---

#### 4. Why GitHub is Essential?

* **Safety:** သင် Computer ၸဝ်ႈၵဝ်ႇလူႉၵေႃႈ Code ၸဝ်ႈၵဝ်ႇတိုၵ်ႉမီးဝႆႉတီႈ GitHub လွတ်ႈၽေးယူႇ။
* **Collaboration:** ၸဝ်ႈၵဝ်ႇၸၢင်ႈၸွႆႈၵၼ်တႅမ်ႈ Code တင်းပိူၼ်ႈလႆႈ ၼႂ်း Project လဵဝ်ၵၼ်။
* **Bridge to Deployment:** ဝႅပ်ႉသၢႆႉ Hosting ၼင်ႇ Vercel တေဢဝ် Code ၸဝ်ႈၵဝ်ႇ လုၵ်ႉတီႈ GitHub သေ ၵႂႃႇပိုၼ်ၽၢဝ်ႇပၼ်ၼိူဝ် Internet ၵမ်းလဵဝ်ၶႃႈ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉတႅမ်ႈ **ပပ်ႉပိုၼ်း** (Manuscript) လႅမ်ႈၼိုင်ႈ:

* **Computer:** မိူၼ်ၼင်ႇ ၸေႈဢၼ်ၸဝ်ႈၵဝ်ႇတႅမ်ႈဝႆႉ တီႈႁိူၼ်း။ သင်ႁိူၼ်းၾႆးမႆႈ ဢမ်ႇၼၼ် ၸေႈႁၢႆ ၵေႃႈယဝ်ႉယဝ်ႉ။
* **GitHub:** မိူၼ်ၼင်ႇ ၸဝ်ႈၵဝ်ႇဢဝ်ပပ်ႉၼၼ်ႉ ၵႂႃႇဢၢပ်ႈဝႆႉတီႈ **ႁွင်ႈၵျောင်းလူင်** (Library)။ ၶဝ်တေၵဵပ်းသိမ်းပၼ် ႁႂ်ႈမၼ်းလွတ်ႈၽေး လႄႈ ပိုတ်ႇပၼ်တၢင်ႇၵူၼ်းမႃးလူတူၺ်းလႆႈ ၵူႈၶၢဝ်းယၢမ်းၼၼ်ႉယဝ်ႉ။
