## Lesson 25: The Anchor Tag: Creating Hyperlinks

တွၼ်ႈတႃႇသၢင်ႈ Link ၼၼ်ႉ ႁဝ်းၸႂ်ႉ tag `<a>` ဢၼ်ပဵၼ်တူဝ်ယေႃႈ **Anchor** (သဝ်လွၵ်း/သဝ်ႁိၼ်) ၶႃႈဢေႃႈ။

#### 1. The Href Attribute

Tag `<a>` ၼႆႉ သင်ဢမ်ႇပႃး Attribute ဢၼ်ၸိုဝ်ႈဝႃႈ `href` ၼၼ်ႉ မၼ်းတေဢမ်ႇႁဵတ်းၵၢၼ်လႆႈၶႃႈ။ `href` (Hypertext Reference) ပဵၼ်တီႈဢၼ်ႁဝ်းတေလႆႈသႂ်ႇ "တီႈယူႇ" (URL) ဢၼ်ႁဝ်းၶႂ်ႈႁႂ်ႈၵူၼ်းၼဵၵ်းသေ ၵႂႃႇၸူးၼၼ်ႉယဝ်ႉ။

#### 2. Structure Example (တူဝ်ယၢင်ႇၵၢၼ်တႅမ်ႈ)

```html
<a href="https://www.google.com">ၵႂႃႇၸူး Google</a>

```

* **`<a>`**: တူဝ်တႄႇ Link။
* **`href="..."`**: တီႈယူႇဝႅပ်ႉသၢႆႉ။
* **ၵႂႃႇၸူး Google**: လိၵ်ႈဢၼ်တေၼႄႁၢင်ႈ (Link Text) တွၼ်ႈတႃႇႁႂ်ႈၵူၼ်းၼဵၵ်း။
* **`</a>`**: တူဝ်ပိၵ်ႉ Link။

#### 3. Best Practices (လၢႆးၸႂ်ႉဢၼ်လီ)

* **Descriptive Text:** ယႃႇတႅမ်ႈဝႃႈ "ၼဵၵ်းတီႈၼႆႉ" (Click here) လၢႆလၢႆ။ လူဝ်ႇတႅမ်ႈႁႂ်ႈပိူၼ်ႈႁူႉဝႃႈ ၼဵၵ်းယဝ်ႉတေၵႂႃႇတီႈလႂ် ၼင်ႇ "လူလိၵ်ႈပိုၼ်းတႆး"။
* **Underline:** Browser ၵမ်ႉၼမ် တေၼႄ Link ပဵၼ် **သီသွမ်ႇ** လႄႈ **မီးသဵၼ်ႈၶีดႇတႂ်ႈ** ႁင်းမၼ်းၵမ်းလဵဝ်ၶႃႈ။

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉယူႇၼႂ်း **ႁူင်းႁေႃလိၵ်ႈ** (Library):

1. **Anchor Tag:** မိူၼ်ၼင်ႇ "ၸေႈပၼ်မၢႆ" ဢၼ်တႅမ်ႈဝႆႉဝႃႈ "သင်ၶႂ်ႈလူလွင်ႈပၢႆးသၢႆႈ ၵႂႃႇတူၺ်းတီႈ တူၵ်ႉမၢႆ ၅။ မိူဝ်ႈၸဝ်ႈၵဝ်ႇလူယဝ်ႉ ၸဝ်ႈၵဝ်ႇၸွမ်းၵႂႃႇၸူးတီႈယူႇၼၼ်ႉယဝ်ႉ။
