## Lesson 14: The Div Tag: The Generic Container

မိူဝ်ႈႁဝ်းႁဵၼ်းမႃးၼၼ်ႉ ႁဝ်းႁၼ်ဝႃႈ HTML Tags ၵမ်ႉၼမ် မီး "တီႈပွင်ႇ" (Meaning) ၼင်ႇ `<header>`, `<main>`, ဢမ်ႇၼၼ် `<footer>`။ ၵူၺ်းၵႃႈ ၵမ်းလၢႆးႁဝ်းလူဝ်ႇလႆႈမီး "ၵွၵ်း" (Box) ဢၼ်ဢမ်ႇမီးတီႈပွင်ႇသင် တွၼ်ႈတႃႇၸႅၵ်ႇ Code ႁဝ်းဢွၵ်ႇပဵၼ်တွၼ်ႈ တွၼ်ႈတႃႇမႄးႁၢင်ႈ (Styling) ၵူၺ်း။ တီႈၼႆႉႁဝ်းၸႂ်ႉ `<div>` ၶႃႈဢေႃႈ။

---

#### 1. What is a Div? (Div ပဵၼ်သင်?)

`<div>` ပဵၼ်တူဝ်ယေႃႈၶေႃႈၵႂၢမ်း **Division** (ၵၢၼ်ၸႅၵ်ႇတွၼ်ႈ)။ မၼ်းပဵၼ် **Generic Container** (ၵၢပ်ႇၵူႈလွင်ႈလွင်ႈ) ဢၼ်ဢမ်ႇလၢတ်ႈပၼ် Browser ဝႃႈ ၼိူဝ်ႉလိၵ်ႈၼႂ်းမၼ်းပဵၼ်သင်။

* **Block-level:** `<div>` ပဵၼ် Block element၊ မၢႆပွင်ႇဝႃႈ မၼ်းတေတႄႇထႅဝ်မႂ်ႇၵူႈပွၵ်ႈ လႄႈ ၵုမ်းၵမ်ဢဝ်တၢင်းၵႂၢင်ႈ (Width) ၼႃႈၸေႃးတင်းသဵင်ႈ။

---

#### 2. When to use it? (တေၸႂ်ႉမိူဝ်ႈလႂ်?)

ႁဝ်းလူဝ်ႇၸႂ်ႉ `<div>` မိူဝ်ႈ:

* ဢမ်ႇမီး Semantic tag (ၼင်ႇ `<section>` ဢမ်ႇၼၼ် `<article>`) ဢၼ်ၵိုင်ႇမႅၼ်ႈၸွမ်းၼိူဝ်ႉလိၵ်ႈ။
* လူဝ်ႇလႆႈ "ၵွၵ်း" တွၼ်ႈတႃႇသႂ်ႇ CSS Styles (ၼင်ႇ သႂ်ႇသီလင်၊ သႂ်ႇၶွပ်ႇ၊ ဢမ်ႇၼၼ် ၸတ်းဝၢင်း Layout)။
* လူဝ်ႇၸႅၵ်ႇ Code ဢွၵ်ႇပဵၼ်ၸုမ်း တွၼ်ႈတႃႇၸႂ်ႉ JavaScript။

#### 3. Structure Example (တူဝ်ယၢင်ႇၵၢၼ်တႅမ်ႈ)

```html
<div class="card-container">
  <h2>ၸိုဝ်ႈၵုၼ်ႇသိၼ်</h2>
  <p>ၼႆႉပဵၼ်ၶေႃႈမုၼ်းလွင်ႈၵုၼ်ႇသိၼ်...</p>
  <button>သိုဝ်ႉယေႃႈ</button>
</div>

```

---

#### ⚠️ Warning (ၶေႃႈသတိ)

ယႃႇၸႂ်ႉ `<div>` ၵူႈတီႈ (Divitis)။ သင်ဝႃႈတွၼ်ႈလိၵ်ႈၼၼ်ႉ မီးတီႈပွင်ႇ (ၼင်ႇ ပဵၼ် Navigation)၊ လူဝ်ႇလႆႈၸႂ်ႉ `<nav>` တႅၼ်း `<div>` ၶႃႈ။ Semantic tags တေၸွႆႈပၼ် SEO လႄႈ Accessibility လီၵႂႃႇလိူဝ် `<div>` တႄႉတႄႉၶႃႈဢေႃႈ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉၸတ်းၶူဝ်းၼႂ်းတူဝ်း (Closet):

1. **Semantic Tags:** မိူၼ်ၼင်ႇ "ၵွၵ်းသႂ်ႇၶႅပ်းတိၼ်" ဢမ်ႇၼၼ် "ပဵၼ်းႁၢႆႉသိူဝ်ႈၶူဝ်း" ဢၼ်မီးၸိုဝ်ႈလႄႈမီးတီႈပွင်ႇတႅတ်ႈၼႅတ်ႈ။
2. **Div Tag:** မိူၼ်ၼင်ႇ "ၵွၵ်းပလတ်ႉသတိၵ်ႉ" (Plain plastic box) ဢၼ်ဢမ်ႇမီးၸိုဝ်ႈသင်။ ၸဝ်ႈၵဝ်ႇၸၢင်ႈသႂ်ႇသင်ၵေႃႈလႆႈၼႂ်းမၼ်း၊ မၼ်းၸွႆႈႁႂ်ႈၶူဝ်းၶွင်ယူႇဝႆႉပဵၼ်တီႈပဵၼ်တၢင်း ၵူၺ်းၵႃႈ တူၺ်းၽၢႆႇၼွၵ်ႈတေဢမ်ႇႁူႉဝႃႈၼႂ်းမၼ်းမီးသင်။
