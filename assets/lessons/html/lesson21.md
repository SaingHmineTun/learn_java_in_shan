## Lesson 21: Line Breaks and Horizontal Rules

မိူဝ်ႈႁဝ်းတႅမ်ႈလိၵ်ႈၼႂ်း HTML ၼၼ်ႉ ၵမ်းလၢႆးႁဝ်းဢမ်ႇၶႂ်ႈတႄႇတွၼ်ႈလိၵ်ႈမႂ်ႇ (`<p>`)၊ ၵူၺ်းၵႃႈ ႁဝ်းၶႂ်ႈႁႂ်ႈမၼ်း "တတ်းထႅဝ်" လုၵ်ႉတီႈၼိူဝ်လူင်းတႂ်ႈ ဢမ်ႇၼၼ် ၶႂ်ႈၶีดႇသဵၼ်ႈၶႅၼ်ႈတွၼ်ႈလိၵ်ႈၶႃႈဢေႃႈ။

---

#### 1. The Line Break Tag (`<br>`)

`<br>` ပဵၼ်တူဝ်ယေႃႈ **Break**။ ၸႂ်ႉတွၼ်ႈတႃႇတတ်းထႅဝ်လိၵ်ႈ ႁႂ်ႈမၼ်းလူင်းၵႂႃႇတႄႇထႅဝ်မႂ်ႇ တီႈတႂ်ႈၵမ်းလဵဝ် တီႈၼႂ်းတွၼ်ႈလိၵ်ႈ (Paragraph) လဵဝ်ၵၼ်။

* **ပိူင်ထူး:** Tag ၼႆႉပဵၼ် **Empty Tag** (ဢမ်ႇမီးတူဝ်ပိၵ်ႉ)။ ၸဝ်ႈၵဝ်ႇတႅမ်ႈ `<br>` ၵူၺ်းၵေႃႈ လႆႈယဝ်ႉၶႃႈ။
* **မိူဝ်ႈလႂ်လီၸႂ်ႉ:** ၸႂ်ႉမိူဝ်ႈတႅမ်ႈ "ၵႂၢမ်းၵပ်းၵႂၢမ်းလေႃး" (Poem) ဢမ်ႇၼၼ် "တီႈယူႇ" (Address) ဢၼ်လူဝ်ႇလႆႈလူင်းထႅဝ်ဝႆးဝႆး။

#### 2. The Horizontal Rule Tag (`<hr>`)

`<hr>` ပဵၼ်တူဝ်ယေႃႈ **Horizontal Rule**။ ၸႂ်ႉတွၼ်ႈတႃႇၶีดႇ "သဵၼ်ႈသိုဝ်ႈ" ၼွၼ်းၶႅၼ်ႈၼႃႈလိၵ်ႈ။

* **Meaning:** မၼ်းၼႄဝႃႈ "ၼိူဝ်ႉလိၵ်ႈတွၼ်ႈၼႆႉ ယဝ်ႉယဝ်ႉ၊ တွၼ်ႈတေႃႇၵႂႃႇတေပဵၼ် ႁူဝ်ၶေႃႈမႂ်ႇယဝ်ႉ" (Thematic break)။
* **Appearance:** Browser တေၶีดႇသဵၼ်ႈပၼ်သိုဝ်ႈသိုဝ်ႈ ၶႅၼ်ႈၼႃႈလိၵ်ႈဝႆႉၶႃႈ။

---

#### 3. Structure Example (တူဝ်ယၢင်ႇၵၢၼ်တႅမ်ႈ)

```html
<p>
  ၸိုဝ်ႈ: ၸၢႆးသႅင် <br>
  ဝဵင်း: တူင်ႇၵျီး <br>
  မိူင်း: မိူင်းတႆး
</p>

<hr>

<p>ၼႆႉပဵၼ်တွၼ်ႈလိၵ်ႈမႂ်ႇ ဢၼ်ၶႅၼ်ႈဝႆႉလူၺ်ႈသဵၼ်ႈ hr ၶႃႈဢေႃႈ။</p>

```

---

#### ⚠️ Warning (ၶေႃႈသတိ)

* ယႃႇၸႂ်ႉ `<br>` တႅၼ်းၵၢၼ်ၸႂ်ႉ `<p>` တွၼ်ႈတႃႇႁဵတ်းႁႂ်ႈမီးလွၵ်းဝၢင်ႈ (Margin)။ သင်ၶႂ်ႈလႆႈလွၵ်းဝၢင်ႈၼႃႈလိၵ်ႈ ႁႂ်ႈႁၢင်ႈလီ၊ လူဝ်ႇၸႂ်ႉ CSS ၶႃႈ။
* ယႃႇၸႂ်ႉ `<br>` လၢႆလၢႆဢၼ်ၸပ်းၵၼ် တွၼ်ႈတႃႇယၢင်ႇထႅဝ်၊ မၼ်းတေႁဵတ်းႁႂ်ႈ Code ၸဝ်ႈၵဝ်ႇဢမ်ႇလီၶႃႈ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉတႅမ်ႈလိၵ်ႈ:

1. **Line Break (`<br>`):** မိူၼ်ၼင်ႇ ၸဝ်ႈၵဝ်ႇတႅမ်ႈလိၵ်ႈၸပ်းၸပ်းၵၼ်သေ ၼဵၵ်း **Shift + Enter** ၼိူဝ် Keyboard တွၼ်ႈတႃႇလူင်းထႅဝ်ၵူၺ်း။
2. **Horizontal Rule (`<hr>`):** မိူၼ်ၼင်ႇ ၸဝ်ႈၵဝ်ႇဢဝ်ပႅၼ်းသတ်းသေ ၶีดႇသဵၼ်ႈလူင်ၶႅၼ်ႈဝႆႉ တွၼ်ႈတႃႇမၵ်းမၼ်ႈဝႃႈ "တွၼ်ႈၼႆႉယဝ်ႉယဝ်ႉ" ၼၼ်ႉယဝ်ႉ။
