## Lesson 44: Forms: Creating the Input Field

ယၢမ်းလဵဝ် ႁဝ်းတေမႃးႁဵၼ်းလွင်ႈဢၼ်လမ်ႇလွင်ႈသုတ်း တွၼ်ႈတႃႇႁဵတ်းႁႂ်ႈဝႅပ်ႉသၢႆႉႁဝ်း "တွပ်ႇလၢတ်ႈ" (Interact) တင်းၵူၼ်းၸႂ်ႉတိုဝ်းလႆႈ၊ ၼၼ်ႉၵေႃႈပဵၼ် **Form** (ပႅင်ႇလိၵ်ႈ) ၶႃႈဢေႃႈ။ ၸဝ်ႈၵဝ်ႇၸၢင်ႈၸႂ်ႉ Form တွၼ်ႈတႃႇႁပ်ႉဢဝ် ၸိုဝ်ႈ၊ Email၊ ဢမ်ႇၼၼ် ၶေႃႈၵႂၢမ်းတွပ်ႇလၢတ်ႈတၢင်ႇၸိူဝ်း။

---

#### 1. The Form Tag (`<form>`)

Tag ၼႆႉ ပဵၼ် "ထူင်" ဢၼ်ႁေႃႇၵုမ်း Input ၵူႈမဵဝ်းမဵဝ်းဝႆႉၸွမ်းၵၼ်။ မၼ်းမီး Attributes လမ်ႇလွင်ႈ 2 ဢၼ်:

* **`action`**: ပဵၼ်တီႈယူႇ (URL) ဢၼ်ႁဝ်းတေသူင်ႇၶေႃႈမုၼ်းၵႂႃႇ (ၵႆႉပဵၼ်ၾၢႆႇ PHP, Python ဢမ်ႇၼၼ် Server-side script)။
* **`method`**: ပဵၼ်လၢႆးသူင်ႇ (ၵႆႉၸႂ်ႉ `GET` တွၼ်ႈတႃႇတူၺ်းၶေႃႈမုၼ်း လႄႈ `POST` တွၼ်ႈတႃႇသူင်ႇၶေႃႈမုၼ်းဢၼ်လမ်ႇလွင်ႈ)။

---

#### 2. The Input Tag (`<input>`)

ၼႆႉပဵၼ် Tag ဢၼ်ႁဝ်းၸႂ်ႉၼမ်သုတ်း တွၼ်ႈတႃႇသၢင်ႈလွၵ်းတႅမ်ႈလိၵ်ႈ။ မၼ်းပဵၼ် **Empty Tag** (ဢမ်ႇမီးတူဝ်ပိၵ်ႉ)။

* **`type`**: မၵ်းမၼ်ႈဝႃႈ ပဵၼ် Input မဵဝ်းလႂ် (ၼင်ႇ `text`, `password`, `submit`)။
* **`name`**: ပဵၼ် "ၸိုဝ်ႈ" ၶွင်ၶေႃႈမုၼ်းၼၼ်ႉ တွၼ်ႈတႃႇ Server တေႁူႉဝႃႈ ၶေႃႈမုၼ်းၼႆႉ မႃးတီႈလွၵ်းလႂ်။

---

#### 3. Structure Example (တူဝ်ယၢင်ႇၵၢၼ်တႅမ်ႈ)

```html
<form action="/submit-form" method="POST">
  ၸိုဝ်ႈ: <input type="text" name="username">
  
  <br><br>

  <input type="submit" value="သူင်ႇလိၵ်ႈ">
</form>

```

---

#### 4. The Placeholder Attribute

ၸဝ်ႈၵဝ်ႇၸၢင်ႈသႂ်ႇလိၵ်ႈ "သပ်းလႅင်းဢွၼ်ႇ" ဝႆႉၼႂ်းလွၵ်း Input ၼၼ်ႉ တွၼ်ႈတႃႇႁႂ်ႈၵူၼ်းၸႂ်ႉတိုဝ်းႁူႉဝႃႈ လူဝ်ႇတႅမ်ႈသင်။

```html
<input type="text" name="city" placeholder="တႅမ်ႈၸိုဝ်ႈဝဵင်းၸဝ်ႈၵဝ်ႇ...">

```

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉတႅမ်ႈ **ဝႂ်လွၵ်းၵၢတ်ႇ** (Order Form):

1. **`<form>`**: မိူၼ်ၼင်ႇ "ၸေႈဝႂ်" တင်းသဵင်ႈ ဢၼ်မီးႁူဝ်လိၵ်ႈဝႃႈ တေလႆႈသူင်ႇၵႂႃႇတီႈလႂ်။
2. **`<input>`**: မိူၼ်ၼင်ႇ "လွၵ်းဝၢင်ႈ" ဢၼ်ၸဝ်ႈၵဝ်ႇလူဝ်ႇဢဝ် ပႅၼ်းသႄ်း (Pencil) သေတႅမ်ႈသႂ်ႇ။
3. **`type="submit"`**: မိူၼ်ၼင်ႇ "ၵၢၼ်ယိုၼ်ႈဝႂ်" ၼၼ်ႉပၼ်ၵူၼ်းၶၢႆ ႁႂ်ႈၶဝ်ႁူႉဝႃႈ ၸဝ်ႈၵဝ်ႇၶႂ်ႈလႆႈသင်ၼၼ်ႉယဝ်ႉ။
