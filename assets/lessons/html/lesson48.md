## Lesson 48: W3C Validation: Checking for Broken Ink

မိူဝ်ႈႁဝ်းတႅမ်ႈ Code HTML ယဝ်ႉၼၼ်ႉ၊ ၵမ်းလၢႆးႁဝ်းၸၢင်ႈ "လိုမ်းပိၵ်ႉ Tag" ဢမ်ႇၼၼ် "တႅမ်ႈ Attribute ၽိတ်း"။ Browser မၢင်ၸိူဝ်းၸၢင်ႈတေၼႄႁၢင်ႈလီယူႇ၊ ၵူၺ်းၵႃႈ မၼ်းတေႁဵတ်းႁႂ်ႈဝႅပ်ႉသၢႆႉႁဝ်းဢမ်ႇၼိမ် (Unstable)။ **W3C Validation** ပဵၼ်ၵၢၼ်ၵူတ်ႇထတ်းဝႃႈ Code ႁဝ်းမႅၼ်ႈၸွမ်းပိူင်မီႈလုမ်ႈၾႃႉႁႃ။

#### 1. What is W3C?

**W3C (World Wide Web Consortium)** ပဵၼ်ၸုမ်းဢၼ်မၵ်းမၼ်ႈ "ပိူင်မီႈ" (Standards) ၶွင် HTML။ ၶဝ်မီးၶိူင်ႈမိုဝ်း [W3C Markup Validation Service](https://validator.w3.org/) တွၼ်ႈတႃႇထတ်း Code ႁဝ်း။

#### 2. Why Validate?

* **Consistency:** ႁႂ်ႈဝႅပ်ႉသၢႆႉၼႄႁၢင်ႈမိူၼ်ၵၼ်ၼႂ်း Chrome, Safari, လႄႈ Firefox။
* **SEO:** Google တေပၼ်ၶႅတ်ႉ (Rank) ဝႅပ်ႉသၢႆႉဢၼ် Code သႅၼ်ႈသႂ် လႄႈ ဢမ်ႇမီးၶေႃႈၽိတ်းၼၼ်ႉ လီလိူဝ်။
* **Debugging:** ၸွႆႈႁႃၶေႃႈၽိတ်းဢၼ်ႁဝ်းမုင်ႈမွင်းဢမ်ႇႁၼ်။

---

#### 3. How to use it?

1. ၵႂႃႇတီႈဝႅပ်ႉသၢႆႉ **validator.w3.org**။
2. လိူၵ်ႈလၢႆး "Validate by Direct Input"။
3. Copy ဢဝ် Code HTML ၸဝ်ႈၵဝ်ႇတင်းသဵင်ႈသေ ဝၢင်း (Paste) သႂ်ႇ။
4. ၼဵၵ်းပုမ်ႇ **Check**။

---

#### 4. Understanding the Results

* **🔴 Errors:** ၶေႃႈၽိတ်းဢၼ်လမ်ႇလွင်ႈ (ၼင်ႇ လိုမ်းပိၵ်ႉ tag ဢမ်ႇၼၼ် သႂ်ႇ tag ၽိတ်းတီႈ)။ တေလႆႈမႄးၵမ်းလဵဝ်။
* **🟡 Warnings:** ၶေႃႈၼႄၼမ်း (ၼင်ႇ လိုမ်းသႂ်ႇ attribute `lang` ၼႂ်း `<html>`)။ မႄးၵေႃႈလီ ဢမ်ႇမႄးၵေႃႈဝႅပ်ႉသၢႆႉတိုၵ်ႉႁဵတ်းၵၢၼ်လႆႈ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉတႅမ်ႈ **လိၵ်ႈတွပ်ႇၵၢၼ်ၸႅတ်ႈ** (Exam Paper):

* **W3C Validation:** မိူၼ်ၼင်ႇ "ၶိူင်ႈၵူတ်ႇလိၵ်ႈ" (Spell Checker)။ မၼ်းတေထတ်းတူၺ်းဝႃႈ ၸဝ်ႈၵဝ်ႇတႅမ်ႈတူဝ်မႄႈလိၵ်ႈမႅၼ်ႈႁႃ၊ ဝၢင်းပိူင်ထႅဝ်လိၵ်ႈမႅၼ်ႈၸွမ်းပိူင်ႁႃ။ သင်ၸဝ်ႈၵဝ်ႇတႅမ်ႈၽိတ်းၼမ်၊ ၵူၼ်းလူၵေႃႈတေဢမ်ႇပွင်ႇၸႂ်၊ ၶူးၵေႃႈတေဢမ်ႇပၼ်မၢႆ (Marks) ၼၼ်ႉယဝ်ႉ။
