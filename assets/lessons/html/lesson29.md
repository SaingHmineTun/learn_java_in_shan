## Lesson 29: The Download Attribute: Sharing Files

ၵမ်းလၢႆး ၸဝ်ႈၵဝ်ႇၶႂ်ႈႁႂ်ႈၵူၼ်းၸႂ်ႉတိုဝ်း Link ၼၼ်ႉ "လူတ်ႇဢဝ်" (Download) ၾၢႆႇၼိုင်ႈဢၼ် (ၼင်ႇ PDF, ပုံႁၢင်ႈ, ဢမ်ႇၼၼ် Program) တႅၼ်းတီႈတေပိုတ်ႇတူၺ်းၼႂ်း Browser။ တီႈၼႆႉႁဝ်းၸႂ်ႉ attribute `download` ၶႃႈဢေႃႈ။

---

#### 1. The `download` Attribute

မိူဝ်ႈႁဝ်းသႂ်ႇ attribute `download` ဝႆႉၼႂ်း tag `<a>` ၼၼ်ႉ၊ Browser တေႁူႉဝႃႈ "ယႃႇပိုတ်ႇၾၢႆႇၼႆႉ၊ ႁႂ်ႈၵူၼ်းၸႂ်ႉတိုဝ်း သိမ်းဝႆႉ (Save) ၼႂ်းၶွမ်ႇပိဝ်ႇတႃႇၶဝ်လႄႈ"။

* **လွင်ႈထူး:** ၸဝ်ႈၵဝ်ႇၸၢင်ႈ "ပႅတ်ႈၸိုဝ်ႈမႂ်ႇ" ပၼ်ၾၢႆႇၼၼ်ႉလႆႈ မိူဝ်ႈၵူၼ်းလူတ်ႇဢဝ် (Download) ယဝ်ႉ။

---

#### 2. Structure Example (တူဝ်ယၢင်ႇၵၢၼ်တႅမ်ႈ)

```html
<a href="files/shan-history.pdf" download>လူတ်ႇဢဝ်ပပ်ႉပိုၼ်းတႆး (PDF)</a>

<a href="images/huge-map-001.jpg" download="shan-state-map.jpg">လူတ်ႇဢဝ်ၽႅၼ်ႇတီႈမိူင်းတႆး</a>

```

---

#### 3. Important Notes (ၶေႃႈလမ်ႇလွင်ႈ)

* **Same Origin:** Attribute ၼႆႉ ၵမ်ႉၼမ်တေႁဵတ်းၵၢၼ်လႆႈ တွၼ်ႈတႃႇၾၢႆႇဢၼ်မီးဝႆႉၼႂ်း **Server ႁဝ်းၵမ်းလဵဝ်** (Same domain)။ သင်ပဵၼ်ၾၢႆႇတီႈ Server တၢင်ႇဢၼ် (ၼင်ႇ Google Drive)၊ မၼ်းၸၢင်ႈတေပိုတ်ႇတူၺ်း (View) တႅၼ်းၵၢၼ်လူတ်ႇဢဝ် (Download) ၶႃႈ။
* **File Types:** ၸဝ်ႈၵဝ်ႇၸၢင်ႈၸႂ်ႉလႆႈတင်းၾၢႆႇၵူႈမဵဝ်းမဵဝ်း (PDF, ZIP, JPG, MP3)။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇယူႇၼႂ်း **လၢၼ်ႉၶၢႆၶူဝ်း**:

1. **Standard Link:** မိူၼ်ၼင်ႇၸဝ်ႈၵဝ်ႇ "ယုၵ်ႉတူၺ်းသိူဝ်ႈ" ၼႂ်းလၢၼ်ႉ။ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉယူႇၼႂ်းလၢၼ်ႉယူႇ။
2. **Download Attribute:** မိူၼ်ၼင်ႇၸဝ်ႈၵဝ်ႇ "သိုဝ်ႉသိူဝ်ႈသေ သႂ်ႇထူင်ဢဝ်မိူဝ်းႁိူၼ်း"။ ယၢမ်းလဵဝ် သိူဝ်ႈၼၼ်ႉယူႇဝႆႉတီႈၸဝ်ႈၵဝ်ႇ၊ ၸဝ်ႈၵဝ်ႇတေပိုတ်ႇတူၺ်းမိူဝ်ႈလႂ်ၵေႃႈလႆႈ တီႈႁိူၼ်းၸဝ်ႈၵဝ်ႇယဝ်ႉ။
