## Lesson 50: SEO Basics: Meta Descriptions and Open Graph

မိူဝ်ႈၸဝ်ႈၵဝ်ႇသၢင်ႈဝႅပ်ႉသၢႆႉယဝ်ႉ၊ ၸဝ်ႈၵဝ်ႇတေၶႂ်ႈႁႂ်ႈၵူၼ်းႁႃႁၼ်လႆႈၼႂ်း **Google** ဢမ်ႇၼၼ် မိူဝ်ႈသူင်ႇ Link ၼႂ်း **Facebook** ႁႂ်ႈမၼ်းဢွၵ်ႇႁၢင်ႈလီလီ။ လွင်ႈၼႆႉႁဝ်းႁွင်ႉဝႃႈ **SEO (Search Engine Optimization)** ၶႃႈဢေႃႈ။

---

#### 1. Meta Description

ပဵၼ်လိၵ်ႈသပ်းလႅင်းဢွၼ်ႇ ဢၼ်တေဢွၵ်ႇမႃးတီႈတႂ်ႈၸိုဝ်ႈဝႅပ်ႉသၢႆႉ မိူဝ်ႈႁဝ်းႁႃၼႂ်း Google။ သင်ဢမ်ႇသႂ်ႇဝႆႉ၊ Google တေလိူၵ်ႈဢဝ်လိၵ်ႈၼႂ်းဝႅပ်ႉႁဝ်းသေမဵဝ်းမႃးၼႄ၊ ဢၼ်ၼၼ်ႉၵမ်းလၢႆးမၼ်းတေဢမ်ႇႁၢင်ႈလီၶႃႈ။

**Code:** (သႂ်ႇၼႂ်း `<head>`)

```html
<meta name="description" content="လဵၼ်ႈသွၼ် HTML ပဵၼ်ၽႃႇသႃႇတႆး တႄႇတီႈပိုၼ်ႉထၢၼ် တေႃႇပေႃးသၢင်ႈဝႅပ်ႉသၢႆႉလႆႈ။">

```

---

#### 2. Open Graph (OG) Tags

Tags ၸိူဝ်းၼႆႉ ၸႂ်ႉတွၼ်ႈတႃႇမၵ်းမၼ်ႈဝႃႈ မိူဝ်ႈပိူၼ်ႈ Copy Link ဝႅပ်ႉသၢႆႉၸဝ်ႈၵဝ်ႇသေ ၵႂႃႇ Paste ၼႂ်း **Facebook, Messenger, ဢမ်ႇၼၼ် WhatsApp** ၼၼ်ႉ ႁႂ်ႈမၼ်းၼႄႁၢင်ႈသင်၊ ႁူဝ်လိၵ်ႈသင်။

* **`og:title`**: ႁူဝ်လိၵ်ႈမိူဝ်ႈသူင်ႇ Link။
* **`og:image`**: ႁၢင်ႈ Thumbnail ဢၼ်တေဢွၵ်ႇမႃးၸွမ်း Link။
* **`og:description`**: လိၵ်ႈသပ်းလႅင်းဢွၼ်ႇ ၽၢႆႇတႂ်ႈႁၢင်ႈ။

```html
<meta property="og:title" content="သွၼ် HTML ၽႃႇသႃႇတႆး">
<meta property="og:description" content="ႁဵၼ်း HTML ငၢႆႈငၢႆႈလူၺ်ႈၽႃႇသႃႇတႆးတီႈၼႆႉ။">
<meta property="og:image" content="https://example.com/images/og-thumbnail.jpg">

```

---

#### 3. Why it Matters?

1. **Professionalism:** ဝႅပ်ႉသၢႆႉဢၼ်သူင်ႇ Link သေ မီးႁၢင်ႈမီးလိၵ်ႈသပ်းလႅင်းလီလီၼၼ်ႉ ၵူၼ်းတေၶႂ်ႈၼဵၵ်းတူၺ်း (Click) လိူဝ်သေ Link ပဝ်ႇပဝ်ႇ။
2. **Search Ranking:** Google တေထတ်းတူၺ်း Meta Description တွၼ်ႈတႃႇႁူႉဝႃႈ ဝႅပ်ႉသၢႆႉၸဝ်ႈၵဝ်ႇ လၢတ်ႈလွင်ႈသင်တႄႉတႄႉ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉ **သူင်ႇၶွင်ၾၢၵ်ႇ** (Gift) ပၼ်ပိူၼ်ႈ:

* **Meta & OG Tags:** မိူၼ်ၼင်ႇ "ၵွပ်ႇႁေႃႇၶွင်ၾၢၵ်ႇ" ဢၼ်မီးႁၢင်ႈႁၢင်ႈလီလီ လႄႈ မီးလိၵ်ႈတႅမ်ႈဝႆႉပႃႈၼိူဝ်ဝႃႈ "ၼႆႉပဵၼ်ၶဝ်ႈမုၼ်းဝၢၼ်ဝၢၼ်ဢေႃႈ"။ သင်ၸဝ်ႈၵဝ်ႇသူင်ႇၵႂႃႇပဝ်ႇပဝ်ႇ လူၺ်ႈဢမ်ႇႁေႃႇသင် (ဢမ်ႇသႂ်ႇ Tags)၊ ၵူၼ်းႁပ်ႉၶွင်ၼၼ်ႉ ၸၢင်ႈတေဢမ်ႇႁတ်းပိုတ်ႇတူၺ်း ၵွပ်ႈဢမ်ႇႁူႉဝႃႈမၼ်းပဵၼ်သင်ၼၼ်ႉယဝ်ႉ။
