## Lesson 45: Labels, Fieldsets, and Legends

မိူဝ်ႈႁဝ်းသၢင်ႈ Form ၼၼ်ႉ၊ လွင်ႈၸတ်းဝၢင်းႁႂ်ႈမၼ်း "လူငၢႆႈ" လႄႈ "ပွင်ႇၸႂ်ငၢႆႈ" ပဵၼ်လွင်ႈလမ်ႇလွင်ႈတႄႉတႄႉၶႃႈ။ တီႈၼႆႉႁဝ်းတေမႃးႁဵၼ်းလွင်ႈ Tag ဢၼ်ၸွႆႈၸတ်းပိူင် Form ႁဝ်းၶႃႈ။

---

#### 1. The Label Tag (`<label>`)

Tag ၼႆႉ ၸႂ်ႉတွၼ်ႈတႃႇပၼ် "ၸိုဝ်ႈ" တီႈလွၵ်း Input။ မၼ်းၸွႆႈႁႂ်ႈၵူၼ်းၸႂ်ႉတိုဝ်းႁူႉဝႃႈ လွၵ်းၼႆႉပဵၼ်လွင်ႈသင်။

* **လွင်ႈလီ:** သင်ႁဝ်းၸႂ်ႉ `<label>`, မိူဝ်ႈၵူၼ်းၸႂ်ႉတိုဝ်း "ၼဵၵ်း" တီႈတူဝ်လိၵ်ႈ Label ၼၼ်ႉ၊ Cursor တေၵျွၵ်းၵႂႃႇၼႂ်းလွၵ်း Input ၼၼ်ႉၵမ်းလဵဝ်။
* **လၢႆးၵွင်ႉ:** ႁဝ်းၸႂ်ႉ attribute `for` ၼႂ်း label ႁႂ်ႈမိူၼ်ၵၼ်တင်း `id` ၼႂ်း input။

```html
<label for="fname">ၸိုဝ်ႈ:</label>
<input type="text" id="fname" name="firstname">

```

---

#### 2. Fieldset and Legend

မိူဝ်ႈ Form ႁဝ်းယႂ်ႇၼႃႇ၊ ႁဝ်းလူဝ်ႇ "ၸႅၵ်ႇတွၼ်ႈ" မၼ်း။

* **`<fieldset>`**: ၸႂ်ႉတွၼ်ႈတႃႇ "ႁေႃႇၵုမ်း" Input ၸိူဝ်းဢၼ်မီးလွင်ႈၵွင်ႉၵၢႆႇၵၼ် ဝႆႉၼႂ်းလွၵ်းလဵဝ်ၵၼ် (Browser တေတႅမ်ႈသဵၼ်ႈၶွပ်ႇႁေႃႇဝႆႉပၼ်ၶႃႈ)။
* **`<legend>`**: ၸႂ်ႉတွၼ်ႈတႃႇပၼ် "ႁူဝ်လိၵ်ႈ" တီႈသဵၼ်ႈၶွပ်ႇ fieldset ၼၼ်ႉ။

---

#### 3. Structure Example (တူဝ်ယၢင်ႇၵၢၼ်တႅမ်ႈ)

```html
<form>
  <fieldset>
    <legend>ၶေႃႈမုၼ်းသုၼ်ႇတူဝ်</legend>
    
    <label for="name">ၸိုဝ်ႈ:</label>
    <input type="text" id="name" name="user_name"><br><br>
    
    <label for="email">Email:</label>
    <input type="email" id="email" name="user_email">
  </fieldset>

  <br>

  <fieldset>
    <legend>တီႈယူႇ</legend>
    <label for="city">ဝဵင်း:</label>
    <input type="text" id="city" name="user_city">
  </fieldset>
</form>

```

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉတႅမ်ႈ **ဝႂ်သႅၼ်းၸိုဝ်ႈၶဝ်ႈႁူင်းႁဵၼ်း**:

1. **`<label>`**: မိူၼ်ၼင်ႇ လိၵ်ႈဢၼ်တႅမ်ႈဝႆႉၽၢႆႇၼႃႈလွၵ်းဝႃႈ "ၸိုဝ်ႈဢူႈ"၊ "ၸိုဝ်ႈမႄႈ"။
2. **`<fieldset>`**: မိူၼ်ၼင်ႇ ၵၢၼ်ဢဝ်ပႅၼ်းသႄ်း "ၶีดလွၵ်း" ႁေႃႇဝႆႉ တွၼ်ႈတႃႇၶေႃႈမုၼ်းပေႃႈမႄႈ ဝႆႉတွၼ်ႈၼိုင်ႈ၊ ၶေႃႈမုၼ်းလုၵ်ႈႁဵၼ်း ဝႆႉတွၼ်ႈၼိုင်ႈ။
3. **`<legend>`**: မိူၼ်ၼင်ႇ လိၵ်ႈဢၼ်တႅမ်ႈဝႆႉၼိူဝ်သဵၼ်ႈၶွပ်ႇၼၼ်ႉဝႃႈ "တွၼ်ႈ ၁ - ၶေႃႈမုၼ်းလုၵ်ႈႁဵၼ်း" ၼၼ်ႉယဝ်ႉ။
