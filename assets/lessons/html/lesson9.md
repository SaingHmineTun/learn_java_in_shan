## Lesson 9: Semantic Layout: Using Header and Nav

မိူဝ်ႈႁဝ်းတႄႇသၢင်ႈဝႅပ်ႉသၢႆႉ၊ တွၼ်ႈၼိူဝ်သုတ်းၼၼ်ႉ ပဵၼ်တီႈဢၼ်ႁဝ်းတေလႆႈသႂ်ႇၸိုဝ်ႈဝႅပ်ႉ လႄႈ တီႈၼဵၵ်းၵႂႃႇမႃး (Menu) ၼၼ်ႉယဝ်ႉ။ ၵၢၼ်ၸႂ်ႉ Semantic tags ၸိူဝ်းၼႆႉ ၸွႆႈပၼ် Browser လႄႈ Google ပွင်ႇၸႂ်ဝႃႈ တွၼ်ႈလႂ်ပဵၼ်တွၼ်ႈ "ႁူဝ်" ဝႅပ်ႉသၢႆႉႁဝ်းၶႃႈ။

---

#### 1. The Header Tag (ႁူဝ်ၼႃႈလိၵ်ႈ)

`<header>` ၸႂ်ႉတွၼ်ႈတႃႇၵုမ်းၵမ်ဢဝ် ႁူဝ်လိၵ်ႈ (Logo)၊ ၸိုဝ်ႈဝႅပ်ႉသၢႆႉ ဢမ်ႇၼၼ် ၶေႃႈမုၼ်းဢၼ်ၼႄဝႆႉတီႈၼိူဝ်သုတ်း။ မၼ်းမိူၼ်ၼင်ႇ "ပၢႆႉၸိုဝ်ႈ" ႁၢၼ်ႉဢၼ်ယူႇၼိူဝ်သုတ်းၼၼ်ႉယဝ်ႉ။

#### 2. The Nav Tag (ၵၢၼ်ၵႂႃႇမႃး)

`<nav>` ပဵၼ်တူဝ်ယေႃႈ **Navigation**။ ႁဝ်းၸႂ်ႉမၼ်းတွၼ်ႈတႃႇသႂ်ႇ Menu Links ၸိူဝ်းဢၼ်တေပၼ်ၵူၼ်းၸႂ်ႉတိုဝ်း ၼဵၵ်းၵႂႃႇမႃးၼႂ်းၼႃႈလိၵ်ႈတၢင်ႇဢၼ်။ ၵမ်ႉၼမ် ႁဝ်းတေၸႂ်ႉမၼ်းႁူမ်ႈၵၼ်တင်း Lists (`<ul>`, `<li>`) ၶႃႈဢေႃႈ။

---

#### 3. Structure Example (တူဝ်ယၢင်ႇၵၢၼ်တႅမ်ႈ)

```html
<header>
  <h1>ဝႅပ်ႉသၢႆႉတႆးႁဝ်း</h1>
  <nav>
    <ul>
      <li><a href="index.html">ၼႃႈႁႅၵ်ႈ</a></li>
      <li><a href="news.html">ၶၢဝ်ႇ</a></li>
      <li><a href="about.html">လွင်ႈႁဝ်း</a></li>
    </ul>
  </nav>
</header>

```

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉလူ **ပပ်ႉၶၢဝ်ႇ** (Newspaper):

1. **Header:** ပဵၼ်ၸိုဝ်ႈပပ်ႉၶၢဝ်ႇ (ၼင်ႇ - "ၶၢဝ်ႇတႆးလွတ်ႈလႅဝ်း") ဢၼ်ယူႇၼိူဝ်သုတ်း ႁႂ်ႈပိူၼ်ႈႁူႉဝႃႈ ပဵၼ်ပပ်ႉသင်။
2. **Nav:** ပဵၼ် "သဵၼ်ႈမၢႆၼႃႈလိၵ်ႈ" (Table of Contents) ဢၼ်လၢတ်ႈဝႃႈ ၶၢဝ်ႇၵၢၼ်မိူင်းယူႇၼႃႈလႂ်၊ ၶၢဝ်ႇလဵၼ်ႈႁႅင်းယူႇၼႃႈလႂ် တွၼ်ႈတႃႇႁႂ်ႈႁဝ်းပိုၼ်ႇၵႂႃႇလႆႈငၢႆႈငၢႆႈၼၼ်ႉယဝ်ႉ။
