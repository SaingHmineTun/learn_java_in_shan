## Lesson 54: Performance Check: Optimizing Assets for Speed

ယၢမ်းလဵဝ် ဝႅပ်ႉသၢႆႉၸဝ်ႈၵဝ်ႇၵေႃႈယူႇၼိူဝ် Internet ယဝ်ႉ။ ၵူၺ်းၵႃႈ လွင်ႈလမ်ႇလွင်ႈထႅင်ႈပိူင်ၼိုင်ႈၵေႃႈပဵၼ် "တၢင်းဝႆး" (Speed) ၶႃႈဢေႃႈ။ သင်ဝႅပ်ႉသၢႆႉၸဝ်ႈၵဝ်ႇပိုတ်ႇထိူင်း (Slow)၊ ၵူၼ်းတေဢမ်ႇမုင်ႈမွင်းသေ တေပိၵ်ႉပႅတ်ႈၵမ်းလဵဝ်။ Lesson ၼႆႉ ႁဝ်းတေမႃးႁဵၼ်းလၢႆးႁဵတ်းႁႂ်ႈဝႅပ်ႉႁဝ်း "ဝႆး" မိူၼ်ၼင်ႇၾႃႉၽႅပ်ႉၶႃႈ။

---

#### 1. Image Optimization (မႄးႁၢင်ႈႁႂ်ႈဝႆး)

ႁၢင်ႈ (Images) ပဵၼ်ဢၼ်ၵိၼ် "ၼမ်ႉၼၵ်း" (File Size) ၼမ်သုတ်းၼႂ်းဝႅပ်ႉသၢႆႉ။

* **Resize:** ဢမ်ႇလူဝ်ႇဢဝ်ႁၢင်ႈလူင် `4000px` မႃးသႂ်ႇၼႂ်းဝၢင်ႈဢွၼ်ႇ `400px`။ လူဝ်ႇမႄးသႅၼ်းႁၢင်ႈႁႂ်ႈပေႃးၵိုင်ႇတူဝ်မၼ်းၵူၺ်း။
* **Format:** ၸႂ်ႉ Format မႂ်ႇၼင်ႇ **.webp** တႅၼ်း .jpg ဢမ်ႇၼၼ် .png။ မၼ်းတေႁဵတ်းႁႂ်ႈၾၢႆႇႁၢင်ႈလဵၵ်ႉလူင်း 50% တေႃႇ 80% ၵူၺ်းၵႃႈ ႁၢင်ႈတိုၵ်ႉလႅင်းလီယူႇ။
* **Lazy Loading:** သႂ်ႇ attribute `loading="lazy"` တီႈ tag `<img>` တွၼ်ႈတႃႇႁႂ်ႈ Browser ပိုတ်ႇႁၢင်ႈမိူဝ်ႈၵူၼ်းလူ တိုၵ်ႉၸၼ် (Scroll) ထိုင်တီႈမၼ်းၵူၺ်း။

---

#### 2. Minification (ၵၢၼ်တတ်းဝၢင်ႈပဝ်ႇ)

မိူဝ်ႈႁဝ်းတႅမ်ႈ Code, ႁဝ်းၵႆႉဝၢင်ႈ "ထႅဝ်ပဝ်ႇ" (Whitespace) လႄႈ "လိၵ်ႈမၢႆတွင်း" (Comments) ၼမ်ၼမ် တွၼ်ႈတႃႇႁႂ်ႈႁဝ်းလူငၢႆႈ။ ၵူၺ်းၵႃႈ Browser ဢမ်ႇလူဝ်ႇၸိူဝ်းၼၼ်ႉၶႃႈ။ **Minification** ပဵၼ်ၵၢၼ်တတ်းပႅတ်ႈဝၢင်ႈပဝ်ႇၸိူဝ်းၼၼ်ႉ ႁႂ်ႈၾၢႆႇ Code ႁဝ်းလဵၵ်ႉသုတ်း။

---

#### 3. Lighthouse: The Speed Doctor

Google မီးၶိူင်ႈမိုဝ်းၼိုင်ႈဢၼ် ႁွင်ႉဝႃႈ **Lighthouse** ဝႆႉၼႂ်း Chrome DevTools တွၼ်ႈတႃႇထတ်းတူၺ်း တၢင်းဝႆးဝႅပ်ႉသၢႆႉ။

1. ပိုတ်ႇဝႅပ်ႉသၢႆႉၸဝ်ႈၵဝ်ႇၼႂ်း Chrome။
2. Right-click သေလိူၵ်ႈ **Inspect** > ၵႂႃႇတီႈ Tab ဢၼ်ၸိုဝ်ႈဝႃႈ **Lighthouse**။
3. ၼဵၵ်းပုမ်ႇ **Analyze page load**။
4. မၼ်းတေပၼ် "မၢႆ (Score)" ၸဝ်ႈၵဝ်ႇ (0-100) လႄႈ လၢတ်ႈၼႄဝႃႈ လူဝ်ႇမႄးတီႈလႂ် ႁႂ်ႈဝႆးလိူဝ်ၵဝ်ႇ။

---

#### 4. Clean Code Habits

* **Avoid Broken Links:** ၸႅတ်ႈတူၺ်းဝႃႈ Link ၵူႈဢၼ်၊ ႁၢင်ႈၵူႈႁၢင်ႈ တိုၵ်ႉၸႂ်ႉလႆႈယူႇႁႃ။
* **Semantic HTML:** ၸႂ်ႉ Tag ႁႂ်ႈထုၵ်ႇမႅၼ်ႈၸွမ်းပိူင် (ၼင်ႇႁဝ်းႁဵၼ်းမႃး) ၸွႆႈႁႂ်ႈ Browser ႁဵတ်းၵၢၼ်လႆႈငၢႆႈလိူဝ်။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉ **ၵႂႃႇၶၢဝ်းတၢင်း** (Traveling):

* **Non-Optimized:** မိူၼ်ၼင်ႇ ၸဝ်ႈၵဝ်ႇတႅပ်း (Pack) ၶူဝ်းသႂ်ႇၼႂ်းထူင်ယႂ်ႇလူင် 10 ထူင်။ ၸဝ်ႈၵဝ်ႇတေၵႂႃႇလႆႈထိူင်းတႄႉတႄႉ။
* **Optimized:** မိူၼ်ၼင်ႇ ၸဝ်ႈၵဝ်ႇလိူၵ်ႈဢဝ်ၶူဝ်းဢၼ်လူဝ်ႇၸႂ်ႉတႄႉတႄႉ (Image Resizing)၊ မူၼ်းၶူဝ်းႁႂ်ႈမၼ်းၵႅပ်ႈ (Minification) သေ သႂ်ႇၼႂ်းထူင်ပႃႇလႅင်ႇ (Backpack) လႅမ်ႈလဵဝ်။ ၸဝ်ႈၵဝ်ႇတေပႆၵႂႃႇလႆႈဝႆး လႄႈ ၸဵဝ်းထိုင်တီႈယိူင်းဢၢၼ်းၼၼ်ႉယဝ်ႉ။
