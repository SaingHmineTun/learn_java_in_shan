## Lesson 42: Tables: Rows, Headers, and Data Cells

မိူဝ်ႈႁဝ်းမီးၶေႃႈမုၼ်း (Data) ဢၼ်လူဝ်ႇလႆႈတူၺ်းၼိူဝ် "ထႅဝ်ၼွၼ်း" (Rows) လႄႈ "ထႅဝ်တင်ႈ" (Columns) ၼင်ႇ တารางၶၢဝ်းယၢမ်း ဢမ်ႇၼၼ် သဵၼ်ႈမၢႆၵႃႈၶၼ်ၶူဝ်းၼၼ်ႉ ႁဝ်းၸႂ်ႉ **Table** ၶႃႈဢေႃႈ။

---

#### 1. The Core Table Tags

တွၼ်ႈတႃႇသၢင်ႈ Table ၼိုင်ႈဢၼ် ႁဝ်းလူဝ်ႇၸႂ်ႉ Tag ပိူင်လူင် 4 မဵဝ်း:

* **`<table>`**: Tag ဢၼ်ႁေႃႇၵုမ်း Tableတင်းသဵင်ႈဝႆႉ။
* **`<tr>` (Table Row)**: ၸႂ်ႉတွၼ်ႈတႃႇသၢင်ႈ "ထႅဝ်ၼွၼ်း" ၼိုင်ႈထႅဝ်။
* **`<th>` (Table Header)**: ၸႂ်ႉတွၼ်ႈတႃႇ "ႁူဝ်ၶေႃႈ" ၼႂ်းထႅဝ်တင်ႈ (Browser ၵႆႉၼႄပဵၼ်တူဝ်ၼႃ လႄႈ ဝႆႉဝၢင်ႈၵၢင်ပၼ်)။
* **`<td>` (Table Data)**: ၸႂ်ႉတွၼ်ႈတႃႇ "လွၵ်းၶေႃႈမုၼ်း" ပႅတ်ႉလႅင်းလၢႆလၢႆ။

---

#### 2. Structure Example (တူဝ်ယၢင်ႇၵၢၼ်တႅမ်ႈ)

```html
<table border="1">
  <tr>
    <th>ၸိုဝ်ႈၶူဝ်းၵိၼ်</th>
    <th>ၵႃႈၶၼ် (ၵျပ်ႉ)</th>
  </tr>

  <tr>
    <td>ၶဝ်ႈသွႆးတႆး</td>
    <td>၂၅၀၀</td>
  </tr>

  <tr>
    <td>ၼမ်ႉၼိုင်ႈ</td>
    <td>၅၀၀</td>
  </tr>
</table>

```

> **Note:** Attribute `border="1"` ၼၼ်ႉ ၸႂ်ႉတွၼ်ႈတႃႇႁႂ်ႈႁၼ် "သဵၼ်ႈၶွပ်ႇ" Table လၢႆလၢႆၵူၺ်း။ ပိူင်ပူၵ်ႉၵတိတႄႉ ႁဝ်းတေၸႂ်ႉ CSS တွၼ်ႈတႃႇမႄးႁၢင်ႈၶွပ်ႇမၼ်း ႁႂ်ႈႁၢင်ႈလီၶႃႈ။

---

#### 3. How to Think About Tables

မိူဝ်ႈၸဝ်ႈၵဝ်ႇတႅမ်ႈ Code Table ၼၼ်ႉ ႁႂ်ႈဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉသၢင်ႈ "ထႅဝ်ၼွၼ်း" (Row) ၵႂႃႇၼိုင်ႈထႅဝ်ယဝ်ႉ ၼိုင်ႈထႅဝ်။ ၼႂ်းၼိုင်ႈထႅဝ်ၼၼ်ႉ ၸင်ႇၵွႆႈၸႅၵ်ႇပဵၼ် "လွၵ်း" (Cell) ၵႂႃႇၸွမ်းလႅၼ်မၼ်းၶႃႈ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉတႅမ်ႈ **တารางသွၼ်လိၵ်ႈ**:

* **`<tr>`**: မိူၼ်ၼင်ႇ "ဝၼ်းၸၼ်" ၼိုင်ႈထႅဝ်တင်းသဵင်ႈ။
* **`<th>`**: မိူၼ်ၼင်ႇ "ၶၢဝ်းယၢမ်း" ဢမ်ႇၼၼ် "ဝိသမ်ႇႇမ" ဢၼ်ယူႇပႃႈၼိူဝ်သုတ်း။
* **`<td>`**: မိူၼ်ၼင်ႇ "လိၵ်ႈတႆး", "ထႅမ်ႇပိုင်ႇ", "Computer" ဢၼ်ယူႇၼႂ်းလွၵ်းၵူႈဢၼ်ဢၼ်ၼၼ်ႉယဝ်ႉ။
