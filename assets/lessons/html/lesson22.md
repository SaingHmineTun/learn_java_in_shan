## Lesson 22: Preformatted Text and Code Snippets

မိူဝ်ႈႁဝ်းတႅမ်ႈလိၵ်ႈၼႂ်း HTML ၼၼ်ႉ ႁဝ်းႁူႉယဝ်ႉဝႃႈ Browser တေဢမ်ႇၼႄပၼ် "လွၵ်းဝၢင်ႈ" (Spaces) လႄႈ "ၵၢၼ်လူင်းထႅဝ်" (Enter) ၸိူဝ်းဢၼ်ႁဝ်းၼဵၵ်းဝႆႉၼႂ်း Code ႁဝ်း။ ၵူၺ်းၵႃႈ သင်ႁဝ်းၶႂ်ႈၼႄလိၵ်ႈ ႁႂ်ႈမၼ်းဢွၵ်ႇမႃး "မိူၼ်ၼင်ႇဢၼ်ႁဝ်းတႅမ်ႈဝႆႉတႄႉတႄႉ" ၼၼ်ႉ ႁဝ်းတေလႆႈၸႂ်ႉ tag ၸိူဝ်းၼႆႉၶႃႈဢေႃႈ။

---

#### 1. The Preformatted Tag (`<pre>`)

`<pre>` ပဵၼ်တူဝ်ယေႃႈ **Preformatted Text**။ မၼ်းတေၼႄလိၵ်ႈ ႁႂ်ႈမိူၼ်ၼင်ႇဢၼ်ၸဝ်ႈၵဝ်ႇတႅမ်ႈဝႆႉၼႂ်း Code ၵမ်းသိုဝ်ႈ။

* **Spaces and Enters:** သင်ၸဝ်ႈၵဝ်ႇၼဵၵ်း Space 5 ဢၼ် ဢမ်ႇၼၼ် ၼဵၵ်း Enter 3 တွၼ်ႈ၊ Browser တေၼႄပၼ်မိူၼ်ၼၼ်ၵူၺ်းၶႃႈ။
* **Font:** ၵမ်ႉၼမ် Browser တေၼႄပဵၼ်တူဝ်လိၵ်ႈ "Monospace" (တူဝ်လိၵ်ႈဢၼ်တၢင်းၵႂၢင်ႈမိူၼ်ၵၼ်ၵူႈတူဝ်)။

#### 2. The Code Tag (`<code>`)

`<code>` ၸႂ်ႉတွၼ်ႈတႃႇၼႄ "Code ၶွမ်ႇပိဝ်ႇတႃႇ" ၼႂ်းၵၢင်ထႅဝ်လိၵ်ႈ (Inline)။ မၼ်းတေႁဵတ်းႁႂ်ႈတူဝ်လိၵ်ႈပဵၼ် Monospace ႁႂ်ႈတူၺ်းႁူႉဝႃႈ မၼ်းပဵၼ် Code။

---

#### 3. Combining Pre and Code (ၵၢၼ်ၸႂ်ႉႁူမ်ႈၵၼ်)

မိူဝ်ႈႁဝ်းၶႂ်ႈၼႄ Code လၢႆလၢႆထႅဝ် (Code Block) ႁႂ်ႈမၼ်းမီးလွၵ်းဝၢင်ႈလီလီၼၼ်ႉ ႁဝ်းၵႆႉၸႂ်ႉ `<pre>` ၵုမ်းၵမ် `<code >` ဝႆႉၶႃႈ။

* **တူဝ်ယၢင်ႇၵၢၼ်တႅမ်ႈ:**

```html
<p>တွၼ်ႈတႃႇၼႄလိၵ်ႈၼႃ လူဝ်ႇၸႂ်ႉ tag <code>&lt;strong&gt;</code> ဢေႃႈ။</p>

<pre>
<code>
  function helloShan() {
    console.log("မႂ်ႇသုင်ၶႃႈ");
  }
</code>
</pre>

```

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉတႅမ်ႈလိၵ်ႈ:

1. **Standard Tag (`<p>`):** မိူၼ်ၼင်ႇ ၸဝ်ႈၵဝ်ႇတႅမ်ႈလိၵ်ႈသေ Browser ၶိုၼ်းၸတ်းဝၢင်းပၼ် ႁႂ်ႈမၼ်းတႅတ်ႈၼႅတ်ႈ။
2. **Pre Tag (`<pre>`):** မိူၼ်ၼင်ႇ ၸဝ်ႈၵဝ်ႇဢဝ်ၶႅပ်းႁၢင်ႈ (Screenshot) တီႈလိၵ်ႈၸဝ်ႈၵဝ်ႇသေ ၵႂႃႇတိတ်းဝႆႉ။ မၼ်းတေဢမ်ႇလႅၵ်ႈလၢႆႈသင်၊ ဝၢင်ႈဝႆႉၼင်ႇႁိုဝ်ၵေႃႈ တေယူႇဝႆႉၼင်ႇၼၼ်ယဝ်ႉ။
