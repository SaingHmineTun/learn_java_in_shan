## Lesson 52: Vercel: Connecting and Zero-Config Deploy

မိူဝ်ႈ Code ၸဝ်ႈၵဝ်ႇၵႂႃႇမီးၼႂ်း GitHub ယဝ်ႉ၊ ၵူၼ်းတၢင်ႇၵူၼ်းတိုၵ်ႉလူလႆႈ Code ၸဝ်ႈၵဝ်ႇၵူၺ်း၊ ၶဝ်တိုၵ်ႉဢမ်ႇႁၼ် "ဝႅပ်ႉသၢႆႉ" ၸဝ်ႈၵဝ်ႇတႄႉတႄႉ။ **Vercel** ပဵၼ်ၶိူင်ႈမိုဝ်းဢၼ်တေဢဝ် Code ၼႂ်း GitHub ၼၼ်ႉသေ မႃး "ပိုၼ်ၽၢဝ်ႇ" (Deploy) ႁႂ်ႈပဵၼ် Link ဝႅပ်ႉသၢႆႉတႄႉတႄႉ ၼႂ်းၶၢဝ်းယၢမ်း 1 မိၼိတ်ႉၶႃႈ။

---

#### 1. Why Vercel?

* **Zero-Config:** ၸဝ်ႈၵဝ်ႇဢမ်ႇလူဝ်ႇမႄးသင်ၼမ်၊ မၼ်းတေႁူႉၵမ်းလဵဝ်ဝႃႈ လူဝ်ႇပိုတ်ႇၾၢႆႇလႂ်။
* **Fast:** ဝႅပ်ႉသၢႆႉၸဝ်ႈၵဝ်ႇတေဝႆးတႄႉတႄႉ ၵွပ်ႈမၼ်းမီး Server ယူႇၵူႈတီႈၼႂ်းလုမ်ႈၾႃႉ။
* **Auto-Update:** မိူဝ်ႈၸဝ်ႈၵဝ်ႇမႄး Code ၼႂ်း GitHub သေ Upload ၶိုၼ်ႈၵႂႃႇမႂ်ႇ၊ ဝႅပ်ႉသၢႆႉၸဝ်ႈၵဝ်ႇၼႂ်း Vercel တေလႅၵ်ႈလၢႆႈၸွမ်း (Update) ၵမ်းလဵဝ်လူၺ်ႈဢမ်ႇလူဝ်ႇႁဵတ်းသင်ထႅင်ႈ။

---

#### 2. How to Deploy (လၢႆးႁဵတ်း)

1. **Log in:** ၵႂႃႇတီႈ [vercel.com](https://vercel.com) သေ ၼဵၵ်းပုမ်ႇ **Continue with GitHub** (မၼ်းတေၵွင်ႉဢဝ် Account GitHub ၸဝ်ႈၵဝ်ႇၵမ်းလဵဝ်)။
2. **Add New:** ၼဵၵ်းပုမ်ႇ **Add New** > **Project**။
3. **Import:** ႁႃၸိုဝ်ႈ Repository ဢၼ်ၸဝ်ႈၵဝ်ႇသၢင်ႈဝႆႉ (ၼင်ႇ `my-first-website`) သေ ၼဵၵ်းပုမ်ႇ **Import**။
4. **Deploy:** ဢမ်ႇလူဝ်ႇလႅၵ်ႈလၢႆႈ Settings သင်၊ ၼဵၵ်းပုမ်ႇ **Deploy**။
5. **Success!:** မုင်ႈမွင်းဢိတ်းၼိုင်ႈ၊ ၸဝ်ႈၵဝ်ႇတေလႆႈ Link ဝႅပ်ႉသၢႆႉ (ၼင်ႇ `my-site.vercel.app`) ဢၼ်ၸၢင်ႈသူင်ႇပၼ်ၵူၼ်းတင်းလုမ်ႈၾႃႉတူၺ်းလႆႈယဝ်ႉၶႃႈ။

---

#### 3. Understanding the Link

မိူဝ်ႈၸဝ်ႈၵဝ်ႇ Deploy ယဝ်ႉ၊ Vercel တေပၼ် Link ပဵၼ်:
`https://your-project-name.vercel.app`

ၼႆႉပဵၼ် Link ပလၢႆ (Free) ဢၼ်ၸဝ်ႈၵဝ်ႇၸႂ်ႉလႆႈတေႃႇၸူဝ်ႈ။ သင်ၶႂ်ႈလႅၵ်ႈပဵၼ်ၸိုဝ်ႈတူဝ်ၵဝ်ႇတႄႉ (ၼင်ႇ `.com` ဢမ်ႇၼၼ် `.org`)၊ ႁဝ်းတေလႆႈႁဵၼ်းထႅင်ႈၼႂ်း Lesson ၼႃႈၶႃႈ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉ **ႁဵတ်းၶႅပ်းႁၢင်ႈ** (Photo Printing):

* **GitHub:** မိူၼ်ၼင်ႇ "ၾၢႆႇႁၢင်ႈ" (Digital File) ဢၼ်မီးၼႂ်း Memory Card။ ၸဝ်ႈၵဝ်ႇႁၼ်မၼ်းလႆႈ ၵူၺ်းၵႃႈ တိုၵ်ႉဢမ်ႇၸၢင်ႈဢဝ်တိတ်းၽႃ။
* **Vercel:** မိူၼ်ၼင်ႇ "လၢၼ်ႉထုတ်ႇႁၢင်ႈ" (Print Shop)။ ၸဝ်ႈၵဝ်ႇဢဝ် Memory Card (GitHub) ပၼ်ၶဝ်၊ ၶဝ်တေထုတ်ႇဢွၵ်ႇမႃးပဵၼ်ၶႅပ်းႁၢင်ႈတႄႉတႄႉ (Website) ဢၼ်ၸဝ်ႈၵဝ်ႇၸၢင်ႈဢဝ်ၼႄပိူၼ်ႈတင်းလုမ်ႈၾႃႉလႆႈၼၼ်ႉယဝ်ႉ။
