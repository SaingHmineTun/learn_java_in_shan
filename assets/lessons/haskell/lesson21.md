## **Lesson 21: Thinking Recursively: The Mindset Shift**

### **1. Recursion ဢၼ်ဝႃႈၼၼ်ႉ ပဵၼ်သင်?**
**Recursion** ၼႆႉ ပဵၼ်လၢႆးပိုတ်ႇတီႈပွင်ႇ Function ဢၼ် **"ႁွင်ႉၸႂ်ႉတူဝ်မၼ်းၶိုၼ်း"** ၼႂ်း Equation ၶႃႈဢေႃႈ။
* **လၢႆးဝူၼ်ႉ:** ပၼ်ႁႃဢၼ်ယႂ်ႇၼၼ်ႉ ႁဝ်းတတ်းမၼ်းပဵၼ် "တွၼ်ႈဢွၼ်ႇ" သေ ႁဵတ်းၵႂႃႇတိူၵ်းတိူၵ်း တေႃႇမၼ်းတေ "သုတ်း" ၵႂႃႇၶႃႈ။



### **2. ၵၢၼ်ပႅတ်ႈ Loops (Why No Loops?)**
ၼႂ်း Haskell ႁဝ်းဢမ်ႇမီး `for (int i=0; i<10; i++)` ၵွပ်ႈဝႃႈ:
1.  **Immutability:** ႁဝ်းဢမ်ႇၸၢင်ႈလႅၵ်ႈလၢႆႈၵႃႈၶၼ် `i = i + 1` လႆႈ။
2.  **Declarative:** ႁဝ်းတႅမ်ႈဝႃႈ "မၼ်းပဵၼ်သင်" (What it is) တႅၼ်း "ႁဵတ်းၼင်ႇႁိုဝ်" (How to do)။

### **3. လၢႆးဝူၼ်ႉပဵၼ် Recursion (The Logic)**
ႁႂ်ႈဝူၼ်ႉတူၺ်းၵၢၼ်ၼပ်ႉတူဝ်ၼပ်ႉၼႂ်း List `[1, 2, 3, 4]` ၶႃႈ။
* **Imperative (Java):** "တႄႇတီႈ 0၊ ဢဝ် 1 မႃးထႅမ်၊ ယဝ်ႉဢဝ် 2 မႃးထႅမ်... တေႃႇပေႃးသုတ်း List"။
* **Recursive (Haskell):** "ၵႃႈၶၼ်တင်းหมด ပဵၼ် **တူဝ်ၼႃႈသုတ်း (1)** ပวก တင်း **ၵႃႈၶၼ်ဢၼ်ၵိုတ်းဝႆႉ [2, 3, 4]**"။
    * ယဝ်ႉၵေႃႈ ၼပ်ႉ [2, 3, 4] ၼၼ်ႉ လူၺ်ႈၸႂ်ႉလၢႆးဝူၼ်ႉဢၼ်ၵဝ်ႇ ၶိုၼ်းၶႃႈ။



---

### **4. တူဝ်ယၢင်ႇ: ၵၢၼ်ၼပ်ႉ Factorial ($n!$)**
Factorial ၼႆႉ ပဵၼ်တူဝ်ယၢင်ႇ ဢၼ်လူပွင်ႇၸႂ်ငၢႆႈသုတ်းၶႃႈ။
* $5! = 5 \times 4 \times 3 \times 2 \times 1$
* လၢႆးဝူၼ်ႉ Recursive: $5! = 5 \times 4!$

```haskell
factorial :: Integer -> Integer
factorial 0 = 1                -- Base Case (တီႈၵိုတ်း)
factorial n = n * factorial (n - 1) -- Recursive Step (ႁွင်ႉတူဝ်မၼ်းၶိုၼ်း)
```

**မၼ်းပၼ်ႇၵၢၼ်ၼင်ႇႁိုဝ်?**
1. `factorial 3`
2. `3 * factorial 2`
3. `3 * (2 * factorial 1)`
4. `3 * (2 * (1 * factorial 0))`
5. `3 * (2 * (1 * 1))` $\rightarrow$ ဢွၵ်ႇမႃး **6**

### **5. ပၵ်းပိူင် 2 ယိူင်ႈ ဢၼ်တိုၼ်းတေလႆႈမီး**
တွၼ်ႈတႃႇတႅမ်ႈ Recursion ႁႂ်ႈမၢၼ်ႇမႅၼ်ႈၼၼ်ႉ လုၵ်ႈႁဵၼ်းလူဝ်ႇမီး:
1.  **Base Case (ဢွင်ႈတီႈၵိုတ်း):** မိူၼ်ၼင်ႇ "ၽၵ်းတူဢွၵ်ႇ"။ သင်ဢမ်ႇမီးဢၼ်ၼႆႉ Program ႁဝ်းတေလႅၼ်ႈဢမ်ႇၵိုတ်း (Infinite Loop) သေ ႁဵတ်းႁႂ်ႈ Computer ၶាំង (Stack Overflow) ၶႃႈ။
2.  **Recursive Step:** ၵၢၼ်ႁွင်ႉၸႂ်ႉ Function ဢၼ်ၵဝ်ႇၶိုၼ်း လူၺ်ႈပၼ် Parameters ဢၼ် "ဢွၼ်ႇလူင်း" ၵႂႃႇတိူၵ်းတိူၵ်း တွၼ်ႈတႃႇႁႂ်ႈမၼ်းၵႂႃႇၸူး Base Case ၶႃႈ။

### **Summary for TMK Students:**
* **Recursion** ပဵၼ်ၵၢၼ် "တတ်းပၼ်ႁႃယႂ်ႇ ႁႂ်ႈပဵၼ်တွၼ်ႈဢွၼ်ႇ"။
* **Base Case** ပဵၼ် "ၽၵ်းတူဢွၵ်ႇ" တွၼ်ႈတႃႇၵိုတ်းၵၢၼ်ႁဵတ်းၵၢၼ်။
* ႁဝ်းတေၸႂ်ႉလၢႆးၼႆႉ တႅၼ်းတူဝ် Loops တင်းหมด ၼႂ်း Haskell ၶႃႈ။
