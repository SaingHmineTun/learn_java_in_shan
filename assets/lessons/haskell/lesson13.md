## **Lesson 13: The Anatomy of a List (The Cons Operator : and Empty List [])**

ၼႂ်း Haskell ၼၼ်ႉ List တိုၼ်းတေလႆႈပဵၼ် **Homogeneous** (မဵဝ်းၶေႃႈမုၼ်း ဢၼ်မိူၼ်ၵၼ်) ၶႃႈဢေႃႈ။ (တူဝ်ယၢင်ႇ: သင်ပဵၼ် Int ၵေႃႈ တေလႆႈပဵၼ် Int တင်းမူတ်း)။

### **1. The Empty List (List ဢၼ်လၢႆ) `[]`**
ပဵၼ်ပိုၼ်ႉထၢၼ်သုတ်းတီႈ List တင်းမူတ်းၶႃႈ။
* ၼႂ်း Haskell ႁဝ်းႁွင်ႉဝႃႈ **"Nil"**။
* List တူဝ်လႂ်ၵေႃႈယဝ်ႉ တိုၼ်းတေလႆႈၵႂႃႇယဝ်ႉတူဝ်ႈတီႈ `[]` ယူႇသေႇသေႇၶႃႈ။

### **2. The Cons Operator (ၵၢၼ်သိုပ်ႇႁူဝ်) `:`**
မၢႆ **`:`** (Colon) ၼႆႉ ႁဝ်းႁွင်ႉဝႃႈ **Cons Operator** (လုၵ်ႉတီႈၶေႃႈၵႂၢမ်း "Construct")။ 
မၼ်းၸႂ်ႉတႃႇဢဝ် "တူဝ်လိၵ်ႈတူဝ်ၼိုင်ႈ" မႃးၸပ်းသႂ်ႇၽၢႆႇၼႃႈ "List" ၶႃႈဢေႃႈ။

**တူဝ်ယၢင်ႇ:**
```haskell
ghci> 1 : [2, 3]
[1, 2, 3]

ghci> 'S' : "han"
"Shan"
```

### **3. How Lists are Built (လၢႆးသၢင်ႈ List တႄႉတႄႉ)**
မိူဝ်ႈႁဝ်းတႅမ်ႈ `[1, 2, 3]` ၼၼ်ႉ တီႈတႄႉမၼ်း Haskell ပွင်ႇၸႂ်ဝႃႈမၼ်းပဵၼ်ၵၢၼ်သိုပ်ႇၵၼ်ၼင်ႇၼႆၶႃႈ:
`1 : (2 : (3 : []))`



* **1** ပဵၼ်တူဝ်ၼႃႈသုတ်း။
* **:** သိုပ်ႇၵႂႃႇၸူး List ဢၼ်မီး 2။
* **[]** ပဵၼ်တီႈယဝ်ႉတူဝ်ႈ (End of list)။

### **4. Syntactic Sugar (လၢႆးတႅမ်ႈဢၼ်ငၢႆႈ)**
ၵွပ်ႈဝႃႈၵၢၼ်တႅမ်ႈ `1:2:3:[]` မၼ်းယၢပ်ႇလႄႈ Haskell ၸင်ႇပၼ်ႁဝ်းၸႂ်ႉ **Square Brackets `[]`** တႃႇတႅမ်ႈႁႂ်ႈငၢႆႈၶႃႈ။
* `[1, 2, 3]` တင်း `1:2:3:[]` ၼႆႉ **ပဵၼ်ဢၼ်မိူၼ်ၵၼ်** ၶႃႈဢေႃႈ။

---

### **Teacher's Lab: "Building Lists Manually"**
ႁႂ်ႈလုၵ်ႈႁဵၼ်းထတ်းတူၺ်းၼႂ်း GHCi လူၺ်ႈၸႂ်ႉ `:t` (Type) တူၺ်းၶႃႈ:

```haskell
ghci> :t []
[] :: [a]  -- (List ဢၼ်လၢႆ ၸၢင်ႈပဵၼ် Type သင်ၵေႃႈလႆႈ)

ghci> :t 5 : []
5 : [] :: Num a => [a] -- (List ဢၼ်မီး 5 တူဝ်လဵဝ်)

ghci> 1 : 2 : 3 : []
[1, 2, 3]
```

### **သတိ (Common Mistakes):**
1. **Wrong:** `[1, 2] : 3` (Cons operator တိုၼ်းတေလႆႈဢဝ် "တူဝ်လဵဝ်" ဝႆႉၽၢႆႇၼႃႈ "List" ယူႇသေႇသေႇ)။
2. **Wrong:** `1 : 2` (ၽၢႆႇၶႂႃတိုၼ်းတေလႆႈပဵၼ် List)။
3. **Correct:** `1 : [2]` (ၽၢႆႇၶႂႃပဵၼ် List ဢၼ်မီး 2 ဝႆႉတူဝ်လဵဝ်)။

---

**Summary for TMK Students:**
* **`[]`** ပဵၼ် List ဢၼ်လၢႆ (The End)။
* **`:`** ပဵၼ်ၵၢၼ်ဢဝ် တူဝ်လဵဝ် မႃးၸပ်းသႂ်ႇ ၽၢႆႇၼႃႈ List။
* String `"ABC"` တီႈတႄႉမၼ်းပဵၼ် `'A':'B':'C':[]`။
