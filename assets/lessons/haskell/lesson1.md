## **Lesson 1: What is Functional Programming? (Concepts vs. Instructions)**

ၼႂ်းၵၢၼ်တႅမ်ႈ Program ၼၼ်ႉ မၼ်းမီးလၢႆးဝူၼ်ႉယူႇ 2 မဵဝ်း ဢၼ်ပႅၵ်ႇၵၼ်ဝႆႉၶႃႈ။

### **1. Imperative Programming (လၢႆးၸိသင်ႇ)**
မိူၼ်ၼင်ႇႁဝ်းတႅမ်ႈ **"လိၵ်ႈၸိသင်ႇ" (Instruction Manual)** ပၼ်ၶွမ်ႇပိဝ်ႇတႃႇၼၼ်ႉၶႃႈ။ ႁဝ်းတေလႆႈလၢတ်ႈၼႄမၼ်းဝႃႈ "ႁဵတ်းဢၼ်ၼႆႉယဝ်ႉ၊ ၵႂႃႇႁဵတ်းဢၼ်ၼၼ်ႉ"။
* **တူဝ်ယၢင်ႇ:** Java, Python, JavaScript.
* **လၢႆးဝူၼ်ႉ:** မၼ်းပဵၼ်ၵၢၼ် **"လႅၵ်ႈလၢႆႈၵႃႈၶၼ် (Change State)"** ယူႇတႃႇသေႇ။

### **2. Functional Programming (လၢႆးပိုတ်ႇတီႈပွင်ႇ)**
မိူၼ်ၼင်ႇႁဝ်းတႅမ်ႈ **"လိၵ်ႈပိုတ်ႇတီႈပွင်ႇ" (Mathematical Definition)** ၼၼ်ႉၶႃႈ။ ႁဝ်းတေလၢတ်ႈဝႃႈ "တူဝ်ၼႆႉ မၼ်းပဵၼ်သင် (What it is)"။
* **တူဝ်ယၢင်ႇ:** Haskell, Elixir, Lisp.
* **လၢႆးဝူၼ်ႉ:** မၼ်းပဵၼ်ၵၢၼ် **"လႅၵ်ႈလၢႆႈၶေႃႈမုၼ်း (Transform Data)"** လူၺ်ႈဢမ်ႇလႆႈယႃႉၵုၼ်ႇ (Side Effects) တၢင်ႇတီႈၶႃႈ။

---

### **လၵ်းၵၢၼ်လူင် 3 ယိူင်ႈ ၼႂ်း Haskell**

#### **ၵ. Immutability (လႅၵ်ႈလၢႆႈဢမ်ႇလႆႈ)**
ၼႂ်း Haskell ၼၼ်ႉ Variable တိုၼ်းလႅၵ်ႈလၢႆႈဢမ်ႇလႆႈၶႃႈ။ သင်ႁဝ်းဝႃႈ `x = 5` ၼႆ `x` တေပဵၼ် `5` ဝႆႉယူႇၸူဝ်ႈၵပ်းၸူဝ်ႈပၢၼ်ၶႃႈယဝ်ႉ။
> **Benefit:** ႁဵတ်းႁႂ်ႈ Program ႁဝ်းၼိမ် (Stable) လႄႈ ႁႃတီႈၽိတ်း (Debug) လႆႈငၢႆႈၶႃႈဢေႃႈ။

#### **ၶ. Pure Functions (Functions ဢၼ်သႅၼ်ႈသႂ်ႇ)**
Function ၼႂ်း Haskell ၼႆႉ တေဢမ်ႇၵႂႃႇယႃႉၵုၼ်ႇ (Side Effects) တီႈတၢင်ႇတီႈ ၼႂ်း Computer ႁဝ်းၶႃႈ။
* သင်ႁဝ်းသႂ်ႇ `Input` ဢၼ်ၵဝ်ႇၶဝ်ႈၵႂႃႇ၊ မၼ်းတေဢွၵ်ႇမႃး `Output` ဢၼ်ၵဝ်ႇယူႇသေႇသေႇ။

#### **င. Declarative Logic (ဝူၼ်ႉတီႈၵၢၼ်လႅၵ်ႈလၢႆႈ)**
ႁဝ်းတေဢမ်ႇၸႂ်ႉ `for` loop ဢမ်ႇၼၼ် `while` loop ၶႃႈ။ ႁဝ်းတေၸႂ်ႉလၢႆး "လႅၵ်ႈလၢႆႈၶေႃႈမုၼ်း" (Transformation) ဢဝ်ၶႃႈဢေႃႈ။
