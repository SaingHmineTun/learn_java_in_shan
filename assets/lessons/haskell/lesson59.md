## **Lesson 59: The Monad Laws**

### **1. Left Identity (ၵႃႈၶၼ်ၽၢႆႇသၢႆႉ)**
ပၵ်းပိူင်ၼႆႉလၢတ်ႈဝႃႈ: သင်ႁဝ်းဢဝ်ၵႃႈၶၼ်ထမ်ႇမတႃႇ `x` သႂ်ႇၼႂ်း `return` ယဝ်ႉ ၸင်ႇဢဝ်မႃး Bind (`>>=`) တင်း Function `f` ၼႆ၊ ၽွၼ်းလႅၼ်ႈမၼ်းတိုၼ်းတေလႆႈမိူၼ်ၵၼ်တင်း ၵၢၼ်ဢဝ် `x` သႂ်ႇၼႂ်း `f` ၵမ်းလဵဝ်ၶႃႈ။

$$return\ x\ >>= f\ ==\ f\ x$$

> **တီႈပွင်ႇ:** `return` တိုၼ်းတေလႆႈပဵၼ်ၵၢၼ်ဢဝ် Data သႂ်ႇၼႂ်း Context "လၢႆလၢႆ" လူၺ်ႈဢမ်ႇလႅၵ်ႈလၢႆႈ Logic သင်ၶႃႈ။

---

### **2. Right Identity (ၵႃႈၶၼ်ၽၢႆႇၶႂႃ)**
ပၵ်းပိူင်ၼႆႉလၢတ်ႈဝႃႈ: သင်ႁဝ်းမီး Monadic Value `m` သေ ဢဝ်မႃး Bind တင်း `return` ၼႆ၊ ၽွၼ်းလႅၼ်ႈမၼ်းတိုၼ်းတေလႆႈမိူၼ် `m` တူဝ်ၵဝ်ႇၶႃႈ။

$$m\ >>=\ return\ ==\ m$$

> **တီႈပွင်ႇ:** ၵၢၼ်ဢဝ်ၶေႃႈမုၼ်းဢွၵ်ႇမႃးသေ သႂ်ႇၶဝ်ႈၵႂႃႇၼႂ်း `return` ၶိုၼ်းၼၼ်ႉ တေလႆႈဢမ်ႇမီးသင်လႅၵ်ႈလၢႆႈၶႃႈ။



---

### **3. Associativity (ၵၢၼ်ႁူမ်ႈၸုမ်း)**
ၼႆႉပဵၼ်ပၵ်းပိူင်ဢၼ် "လႅၵ်ႈလၢႆႈၵမ်ႇၽႃႇ" ၶႃႈ။ မၼ်းလၢတ်ႈဝႃႈ မိူဝ်ႈႁဝ်းသိုပ်ႇ Bind (`>>=`) ၵၼ်လၢႆလၢႆၸၼ်ႉ၊ ႁဝ်းတေႁူမ်ႈၵၼ်လုၵ်ႉၽၢႆႇလႂ်ၵေႃႈယဝ်ႉ ၽွၼ်းလႅၼ်ႈတိုၼ်းတေမိူၼ်ၵၼ်ၶႃႈ။

$$(m\ >>= f)\ >>= g\ ==\ m\ >>=\ (\lambda x \rightarrow f\ x\ >>= g)$$

> **တီႈပွင်ႇ:** ၼႆႉယဝ်ႉၶႃႈ ဢၼ်ႁဵတ်းႁႂ်ႈ `do` block ႁဝ်းလႅၼ်ႈလႆႈ။ ႁဝ်းဢမ်ႇလူဝ်ႇသူၼ်ၸႂ်ဝႃႈ Function ၸိူဝ်းၼၼ်ႉ တေႁေႃႇ (Nest) ၵၼ်ဝႆႉၼင်ႇႁိုဝ်၊ ႁဝ်းၵူၺ်းတႅမ်ႈသိုပ်ႇၵၼ်လူင်းမႃး ၸွမ်းလႅၼ်းမၼ်းၵေႃႈယဝ်ႉတူဝ်ႈၶႃႈယဝ်ႉ။



---

### **ၵွပ်ႈသင်ႁဝ်းလူဝ်ႇ Laws ၸိူဝ်းၼႆႉ?**
ၸဝ်ႈၶူး ၸၢင်ႈသပ်းလႅင်းၼႄလုၵ်ႈႁဵၼ်း တီႈ **TMK Academy** ဝႃႈ:
1. **Refactoring Safety**: ႁဝ်းၸၢင်ႈလႅၵ်ႈလၢႆးတႅမ်ႈ Code ႁဝ်း (မိူၼ်ၼင်ႇ လုၵ်ႉတီႈ Bind ယၢဝ်းယၢဝ်း ၵႂႃႇပဵၼ် `do` block) လူၺ်ႈဝႆႉၸႂ်လႆႈဝႃႈ ၽွၼ်းလႅၼ်ႈတေမိူၼ်ၵဝ်ႇ။
2. **Predictable Code**: ႁဝ်းႁူႉဝႃႈ `return` တေဢမ်ႇႁဵတ်းႁႂ်ႈ Program ႁဝ်း Error ဢမ်ႇၼၼ် လႅၵ်ႈလၢႆႈ Data ႁဝ်းလူၺ်ႈဢမ်ႇမီးသႃႁေတုၶႃႈ။
3. **Algebra of Programming**: မၼ်းႁဵတ်းႁႂ်ႈ Programming ပဵၼ်ပၢႆးၼပ်ႉ ဢၼ်ႁဝ်း "ၼပ်ႉ" (Prove) လႆႈဝႃႈမၢၼ်ႇမႅၼ်ႈႁႃႉ ၼႆၶႃႈ။

---

### **Teacher's Lab: "The Law Check"**
ႁႂ်ႈလုၵ်ႈႁဵၼ်း ထတ်းတူၺ်း `Left Identity` တင်း `Maybe` ၶႃႈ:

```haskell
-- return x >>= f == f x
-- return တွၼ်ႈတႃႇ Maybe ပဵၼ် Just

f x = Just (x + 10)

-- Check:
ghci> return 5 >>= f
Just 15

ghci> f 5
Just 15
-- Result: မၢၼ်ႇမႅၼ်ႈ (Match) ၶႃႈ!
```

---

### **Summary for TMK Students:**
* **Laws**: ပဵၼ်ပၵ်းပိူင်ဢၼ် "ၵႅတ်ႇၶႄ" ႁႂ်ႈ Monad ႁဵတ်းၵၢၼ်ထုၵ်ႇၵိုင်ႇ။
* **Return**: တိုၼ်းတေလႆႈပဵၼ် "Neutral" (ဢမ်ႇမီး Side Effect)။
* **Bind**: တိုၼ်းတေလႆႈသိုပ်ႇၵၼ်လႆႈ ဢမ်ႇဝႃႈတေႁူမ်ႈၸုမ်းလၢႆးလႂ်။
