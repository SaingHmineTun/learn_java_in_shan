## **Lesson 42: The '$' Operator: Cleaning Up Parentheses**

**`$`** ပဵၼ် Operator ဢၼ်မီးၸိုဝ်ႈဝႃႈ **Function Application Operator** ၶႃႈ။

### **1. ပၵ်းပိူင် (How it works)**
ၼႂ်း Haskell ၵၢၼ်ႁွင်ႉ Function ပိုၼ်ႉထၢၼ် (Space) တိုၼ်းတေမီး "ဢဵၼ်ႁႅင်းသုင်သုတ်း" (Highest precedence) လႄႈ ႁဵတ်းၵၢၼ်လုၵ်ႉၽၢႆႇသၢႆႉၵႂႃႇၶႂႃ။ ၵူၺ်းၵႃႈ **`$`** သမ်ႉ:
* မီး "ဢဵၼ်ႁႅင်းတႅမ်ႇသုတ်း" (Lowest precedence)။
* ႁဵတ်းၵၢၼ်လုၵ်ႉၽၢႆႇ **ၶႂႃမႃးသၢႆႉ** ၶႃႈ။



---

### **2. တူဝ်ယၢင်ႇ: ၵၢၼ်လူတ်းယွမ်း ဝူင်ႇၵွတ်ႇ**

ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းၶႂ်ႈၼပ်ႉ `sqrt (3 + 4 + 5)` ၶႃႈ:

* **လၢႆးတႅမ်ႈၵဝ်ႇ:** `sqrt (3 + 4 + 5)`
* **လၢႆးၸႂ်ႉ `$`:** `sqrt $ 3 + 4 + 5`

> **လၢႆးမၼ်းဝူၼ်ႉ:** ၵွပ်ႈဝႃႈ `$` မီးဢဵၼ်ႁႅင်းတႅမ်ႇသုတ်း၊ မၼ်းတေ "ၵင်ႈ" ဝႆႉသေ ပၼ်ႁႂ်ႈ `3 + 4 + 5` (ၽၢႆႇၶႂႃ) ၼပ်ႉယဝ်ႉတူဝ်ႈဢွၼ်ႇတၢင်း ၸင်ႇဢဝ်ၽွၼ်းလႅၼ်ႈ `12` သူင်ႇပၼ် `sqrt` ၶႃႈဢေႃႈ။

---

### **3. ၵၢၼ်ၸႂ်ႉတင်း Function ဢၼ်ႁေႃႇၵၼ်ဝႆႉ (Nested Functions)**

ႁႂ်ႈလုၵ်ႈႁဵၼ်း တီႈ **TMK Academy** ထတ်းတူၺ်းလၢႆးလႅၵ်ႈလၢႆႈၼႆႉၶႃႈ:

* **လၢႆးယုင်ႈယၢင်ႈ:** `sum (filter (>5) (map (*2) [1..10]))`
* **လၢႆးသႅၼ်ႈသႂ်ႇ:** `sum $filter (>5)$ map (*2) [1..10]`



---

### **4. Difference between `.` and `$`**
ၼႆႉပဵၼ်ဢၼ်လုၵ်ႈႁဵၼ်း ၵႆႉထၢမ်လိူဝ်သုတ်းၶႃႈ:
* **`.` (Composition)**: ၸႂ်ႉတွၼ်ႈတႃႇ "သိုပ်ႇ" Function သွင်ဢၼ် ႁႂ်ႈပဵၼ် Function မႂ်ႇ (ယူႇၵႄႈ Function လႄႈ Function)။
* **`$` (Application)**: ၸႂ်ႉတွၼ်ႈတႃႇ "သူင်ႇ" ၶေႃႈမုၼ်းၶဝ်ႈၼႂ်း Function (ယူႇၵႄႈ Function လႄႈ Data)။

---

### **Teacher's Lab: "Cleaning the Pipeline"**
ႁႂ်ႈလုၵ်ႈႁဵၼ်း တႅမ်ႈ Code တွၼ်ႈတႃႇ "ႁႃတူဝ်ၼပ်ႉယႂ်ႇသုတ်း" (Maximum) ၼႂ်း List ဢၼ် "ထႅမ်ဝႆႉ 10" ၵူႈတူဝ်ၶႃႈ:

```haskell
-- လၢႆးတႅမ်ႈဢၼ်ၸႂ်ႉ $
maxPlusTen :: [Int] -> Int
maxPlusTen xs = maximum $ map (+10) xs

-- ghci> maxPlusTen [1, 5, 2] --> 15
```

---

### **Summary of Module 5 (Higher-Order Functional Power)**
1. **Currying**: ပွင်ႇၸႂ်ဝႃႈ Function တိုၼ်းႁပ်ႉ Parameter ၵူၺ်းတူဝ်လဵဝ်။
2. **Map/Filter**: မေႃလႅၵ်ႈလၢႆႈ လႄႈ လိူၵ်ႈၶေႃႈမုၼ်း ၼႂ်း List။
3. **Lambdas**: မေႃတႅမ်ႈ Function ဢမ်ႇမီးၸိုဝ်ႈ တွၼ်ႈတႃႇၸႂ်ႉဝႆးဝႆး။
4. **Folds**: မေႃႁူပ်ႇ List ႁႂ်ႈပဵၼ်ၵႃႈၶၼ်တူဝ်လဵဝ်။
5. **Composition & $**: မေႃသိုပ်ႇ Function လႄႈ လၢင်ႉ Code ႁႂ်ႈသႅၼ်ႈသႂ်ႇ။
