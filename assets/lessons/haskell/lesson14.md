## **Lesson 14: Basic List Functions (Head, Tail, Last, and Init)**

ႁႂ်ႈလုၵ်ႈႁဵၼ်း တီႈ **TMK Academy** ဝူၼ်ႉတူၺ်းဝႃႈ List ၼႆႉ မိူၼ်ၼင်ႇ "တူဝ်သတ်း" (Animal) ဢၼ်ၼိုင်ႈ ဢၼ်မီးတင်း **ႁူဝ် (Head)** လႄႈ **ႁၢင် (Tail)** ၶႃႈ။

### **1. `head` (ႁူဝ်)**
ဢဝ်တူဝ်လိၵ်ႈ **တူဝ်ၼႃႈသုတ်း** (First element) ဢွၵ်ႇမႃးတူဝ်လဵဝ်ၵူၺ်း။
* **Input:** `[1, 2, 3, 4]`
* **Output:** `1`

### **2. `tail` (ႁၢင်)**
ဢဝ်တူဝ်လိၵ်ႈတင်းလင် ဢွၵ်ႇမႃး **လိူဝ်သေတူဝ်ၼႃႈသုတ်း**။ (ဢွၵ်ႇမႃးပဵၼ် List ၶိုၼ်း)။
* **Input:** `[1, 2, 3, 4]`
* **Output:** `[2, 3, 4]`



### **3. `last` (တူဝ်လိုၼ်း)**
ဢဝ်တူဝ်လိၵ်ႈ **တူဝ်လိုၼ်းသုတ်း** ဢွၵ်ႇမႃးတူဝ်လဵဝ်ၵူၺ်း။
* **Input:** `[1, 2, 3, 4]`
* **Output:** `4`

### **4. `init` (တူဝ်ဢွၼ်ၼႃႈ)**
ဢဝ်တူဝ်လိၵ်ႈဢွၼ်ၼႃႈ ဢွၵ်ႇမႃး **လိူဝ်သေတူဝ်လိုၼ်းသုတ်း**။
* **Input:** `[1, 2, 3, 4]`
* **Output:** `[1, 2, 3]`



---

### **ၵၢၼ်ထတ်းသိၼ်ၼႂ်း GHCi**
ႁႂ်ႈလုၵ်ႈႁဵၼ်းတႅမ်ႈထတ်းတူၺ်းၼင်ႇၼႆၶႃႈ:

```haskell
ghci> let numbers = [10, 20, 30, 40, 50]
ghci> head numbers
10
ghci> tail numbers
[20, 30, 40, 50]
ghci> last numbers
50
ghci> init numbers
[10, 20, 30, 40]
```

---

### **⚠️ ၶေႃႈသတိလူင် (The Empty List Danger)**
ဢၼ်ၼႆႉ ပဵၼ်လွင်ႈဢၼ်လုၵ်ႈႁဵၼ်းတေ Error လႆႈငၢႆႈသုတ်းၶႃႈ။ သင်ႁဝ်းၸႂ်ႉ Function ၸိူဝ်းၼႆႉ ၼိူဝ် **List ဢၼ်ပဝ်ႇ (`[]`)** ၼႆ Haskell တေ **Exception** ၵမ်းလဵဝ်။

* **Error:** `head []` $\rightarrow$ `*** Exception: Prelude.head: empty list`

**ၵွပ်ႈသင်?** ၵွပ်ႈဝႃႈ List ဢၼ်လၢႆ မၼ်းဢမ်ႇမီး "ႁူဝ်" လႄႈ "ႁၢင်" တွၼ်ႈတႃႇတေပၼ်ဢွၵ်ႇမႃးၼၼ်ႉၶႃႈ။

---

### **Summary for TMK Students:**
* **`head`** vs **`tail`**: ၸႅၵ်ႇဢဝ်တူဝ်ၼႃႈသုတ်း လႄႈ တူဝ်ဢၼ်ၵိုတ်း။
* **`last`** vs **`init`**: ၸႅၵ်ႇဢဝ်တူဝ်လိုၼ်းသုတ်း လႄႈ တူဝ်ဢၼ်ၵိုတ်းၽၢႆႇၼႃႈ။
* တွၼ်ႈတႃႇ "ၵႅတ်ႇၶႄ" (Safety) ၼၼ်ႉ ႁဝ်းတေလႆႈထတ်းတူၺ်းဝႃႈ List ႁဝ်းလၢႆႁႃႉ? ဢွၼ်တၢင်းတေၸႂ်ႉ Function ၸိူဝ်းၼႆႉၶႃႈ။
