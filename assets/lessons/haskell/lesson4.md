## **Lesson 4: Numbers and Math (Int, Integer, and Floating Point)**

ၼႂ်း Haskell ၼႆႉ တူဝ်ၼပ်ႉ (Numbers) မၼ်းမီးလၢႆမဵဝ်း။ 
ဢၼ်လုၵ်ႈႁဵၼ်း တီႈ **TMK Academy** လူဝ်ႇႁူႉဝႆႉမီး 3 မဵဝ်းလူင်ၶႃႈ။

### **1. Int (Fixed-precision Integer)**
**Int** ၼႆႉ ပဵၼ်တူဝ်ၼပ်ႉထူၼ်ႈ (Whole numbers) ဢၼ်မီးၶွပ်ႇၵႅတ်းမၼ်းဝႆႉ။
* မၼ်းမိူၼ်ၼင်ႇ `int` ၼႂ်း Java ၶႃႈ။
* မၼ်းၼပ်ႉလႆႈတေႃႇထိုင် -2,147,483,648 တေႃႇ 2,147,483,647 (ပုၼ်ႈတႃႇ 32-bit)။
* **ၸႂ်ႉမိူဝ်ႈလႂ်?** ၸႂ်ႉမိူဝ်ႈႁဝ်းႁူႉဝႃႈ တူဝ်ၼပ်ႉႁဝ်းတေဢမ်ႇယႂ်ႇပူၼ်ႉတီႈ။

### **2. Integer (Arbitrary-precision Integer)**
ဢၼ်ၼႆႉပဵၼ် "Superpower" ၶေႃႈၼိုင်ႈၶေႃႈ Haskell ၶႃႈ!
* **Integer** ၼႆႉ ပဵၼ်တူဝ်ၼပ်ႉထူၼ်ႈ ဢၼ် **ဢမ်ႇမီးၶွပ်ႇၵႅတ်း**။
* မၼ်းၸၢင်ႈၼပ်ႉလႆႈတူဝ်ၼပ်ႉ ဢၼ်မီးလၵ်းပဵၼ်ႁူဝ်ႁဵၼ်း ႁူဝ်မိုၼ်ႇၵေႃႈလႆႈ။ Memory Computer ႁဝ်းၵုမ်ႇၵေႃႈ မၼ်းၼပ်ႉလႆႈၵႂႃႇတိူၵ်းတိူၵ်းၶႃႈယဝ်ႉ။
* **တူဝ်ယၢင်ႇ:** တွၼ်ႈတႃႇၼပ်ႉ $2^{100}$ ၼၼ်ႉ `Int` တေၼပ်ႉဢမ်ႇလႆႈ (Overflow)၊ တေလႆႈၸႂ်ႉ `Integer` ဢဝ်ၶႃႈ။



### **3. Float & Double (Floating Point)**
တွၼ်ႈတႃႇတူဝ်ၼပ်ႉ ဢၼ်မီး "တူဝ်ၼပ်ႉလႅတ်း" (Decimals) ၼၼ်ႉ ႁဝ်းတေၸႂ်ႉ:
* **Float:** ပဵၼ် Single precision (ၼပ်ႉလႅတ်းဢေႇ)။
* **Double:** ပဵၼ် Double precision (ၼပ်ႉလႅတ်းလႆႈတႅတ်ႈတေႃးလိူဝ်)။ ၼႂ်း Haskell ႁဝ်းတေၸႂ်ႉ `Double` ပဵၼ်ပိူင်လူင်ၶႃႈ။

### **ၵၢၼ်ၼပ်ႉဢၢၼ်ႇ (Basic Arithmetic Operators)**

ၼႂ်း GHCi ႁဝ်းၸၢင်ႈၸႂ်ႉ Operators ၸိူဝ်းၼႆႉလႆႈၵမ်းလဵဝ်ၶႃႈ:

* **`+`** (တိုင်ၵၼ် / Addition)
* **`-`** (ထွၼ်ၵၼ် / Subtraction)
* **`*`** (ၼပ်ႉၵၼ် / Multiplication)
* **`/`** (ၸိူၵ်ႈၵၼ် / Division - ၸႂ်ႉတႃႇ Floating point)
* **`^`** (ယုၵ်ႉၵမ်ႉ / Exponentiation - ၸႂ်ႉတႃႇတူဝ်ၼပ်ႉထူၼ်ႈ)

**ၶေႃႈၾၢင်ႉ (Important Note):**
သင်ႁဝ်းတေၸိူၵ်ႈ (Divide) တူဝ်ၼပ်ႉထူၼ်ႈ ႁဝ်းတေၸႂ်ႉ `/` ဢမ်ႇလႆႈၶႃႈ။ ႁဝ်းလူဝ်ႇၸႂ်ႉ Function ဢၼ်ႁွင်ႉဝႃႈ **`div`** (တႃႇဢဝ်ၵႃႈၶၼ်ထူၼ်ႈ) လႄႈ **`mod`** (တႃႇဢဝ်ၵႃႈၶၼ်လိူဝ်)။

```haskell
ghci> 10 / 3      -- ဢွၵ်ႇမႃး 3.3333333333333335
ghci> div 10 3    -- ဢွၵ်ႇမႃး 3
ghci> mod 10 3    -- ဢွၵ်ႇမႃး 1
```

### **ၵၢၼ်ထတ်းသိၼ် Type (:t)**
လုၵ်ႈႁဵၼ်းၸၢင်ႈၸႂ်ႉ `:t` ၼႂ်း GHCi တႃႇတူၺ်းဝႃႈ တူဝ်ၼပ်ႉၼၼ်ႉ ပဵၼ် Type သင်:
```haskell
ghci> :t 5
5 :: Num a => a  -- (မၼ်းပဵၼ်တူဝ်ၼပ်ႉ Generic)
ghci> :t 5.5
5.5 :: Fractional a => a
```

**Summary for TMK Students:**
* **Int:** တူဝ်ၼပ်ႉထူၼ်ႈ (မီးၶွပ်ႇၵႅတ်း)။
* **Integer:** တူဝ်ၼပ်ႉထူၼ်ႈ (ယႂ်ႇၵေႃႈလႆႈ ဢမ်ႇမီးၶွပ်ႇၵႅတ်း)။
* **Double:** တူဝ်ၼပ်ႉမီးၸုၼ် (Decimals)။
* **div/mod:** ၸႂ်ႉတႃႇၸိူၵ်ႈတူဝ်ၼပ်ႉထူၼ်ႈ။
