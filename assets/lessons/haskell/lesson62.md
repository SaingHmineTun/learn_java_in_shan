## **Lesson 62: The List Monad: Non-determinism**

မိူဝ်ႈႁဝ်းၸႂ်ႉ List ပဵၼ် Monad၊ ႁဝ်းတေဝူၼ်ႉဝႃႈ Action ၼိုင်ႈဢၼ် ၸၢင်ႈပၼ်ၽွၼ်းလႅၼ်ႈဢွၵ်ႇမႃး **လၢႆလၢႆဢၼ်** ၼႂ်းပွၵ်ႈလဵဝ်ၶႃႈ။

### **1. ပၵ်းပိူင် Bind (`>>=`) ၶေႃႈ List**
ၼႆႉပဵၼ်လၢႆးဢၼ်မၼ်းႁဵတ်းၵၢၼ်ၶႃႈ:
* မၼ်းတေဢဝ် Function ၵႂႃႇၸႂ်ႉတင်း **ၵူႈတူဝ်** ၼႂ်း List။
* ယဝ်ႉၵေႃႈ ဢဝ်ၽွၼ်းလႅၼ်ႈတင်းหมดမႃး "ႁူပ်ႇ" (Flatten) ၶဝ်ႈၵၼ်ပဵၼ် List လဵဝ်ၶႃႈ။



---

### **2. တူဝ်ယၢင်ႇ: ၵၢၼ်ၼပ်ႉလၢႆလၢႆၸၼ်ႉ**
ဝူၼ်ႉတူၺ်းဝႃႈ ၸဝ်ႈၶူး **Sai Mao** ၶႂ်ႈဢဝ်တူဝ်ၼပ်ႉၼႂ်း List မႃးၼပ်ႉတင်း `10` ယဝ်ႉၵေႃႈ ဢဝ်မႃး "လົບ" (-x) ၶႃႈ:

```haskell
-- လၢႆးၸႂ်ႉ Bind (>>=)
ghci> [1, 2, 3] >>= \x -> [x + 10, -x]
[11, -1, 12, -2, 13, -3]
```
*(လၢႆးဝူၼ်ႉ: 1 ပဵၼ် [11, -1]၊ 2 ပဵၼ် [12, -2]... ယဝ်ႉဢဝ်မႃးႁူမ်ႈၵၼ်ၶႃႈ)*

---

### **3. List Comprehension လႄႈ 'do' Notation**
ၸဝ်ႈၶူး ၸၢင်ႈၼႄလုၵ်ႈႁဵၼ်းဝႃႈ **List Comprehension** ဢၼ်ႁဝ်းႁဵၼ်းမႃးၼၼ်ႉ တီႈတႄႉမၼ်းၵေႃႈပဵၼ် **List Monad** ၶႃႈယဝ်ႉ!

* **လၢႆးတႅမ်ႈ List Comprehension:**
  ```haskell
  [(x, y) | x <- [1,2], y <- ['a','b']]
  -- Result: [(1,'a'),(1,'b'),(2,'a'),(2,'b')]
  ```

* **လၢႆးတႅမ်ႈ 'do' block (List Monad):**
  ```haskell
  pairs = do
      x <- [1, 2]
      y <- ['a', 'b']
      return (x, y)
  -- Result: [(1,'a'),(1,'b'),(2,'a'),(2,'b')]
  ```



---

### **4. ၵၢၼ်ၸႂ်ႉ 'guard' (ၵၢၼ်လိူၵ်ႈၶေႃႈမုၼ်း)**
ၼႂ်း List Monad ႁဝ်းၸၢင်ႈၸႂ်ႉ Function **`guard`** (လုၵ်ႉတီႈ `Control.Monad`) တႃႇလိူၵ်ႈဢဝ်ၵူၺ်းၶေႃႈမုၼ်း ဢၼ်ႁဝ်းၶႂ်ႈလႆႈၶႃႈ:

```haskell
import Control.Monad (guard)

evenPairs = do
    x <- [1..10]
    guard (even x) -- သင်ဢမ်ႇပဵၼ်တူဝ်ၵူပ်ႉ မၼ်းတေ "ၵိုတ်း" (ပဵၼ် List လၢႆ [])
    return x

-- ghci> evenPairs --> [2, 4, 6, 8, 10]
```

### **Teacher's Lab: "The Multiplier"**
ႁႂ်ႈလုၵ်ႈႁဵၼ်း တႅမ်ႈ Function ဢၼ်တေဢဝ် List ၶေႃႈတူဝ်ၼပ်ႉမႃး သေပၼ်ဢွၵ်ႇမႃးပဵၼ် Pair `(x, x*x)` ၵူၺ်းတွၼ်ႈတႃႇတူဝ်ၼပ်ႉဢၼ်ယႂ်ႇလိူဝ် `5` ၶႃႈ:

```haskell
squareBigOnes :: [Int] -> [(Int, Int)]
squareBigOnes nums = do
    n <- nums
    guard (n > 5)
    return (n, n * n)

-- ghci> squareBigOnes [3, 6, 4, 8] --> [(6, 36), (8, 64)]
```

---

### **Summary for TMK Students:**
* **Non-determinism**: List Monad ၸွႆႈႁႂ်ႈႁဝ်းႁႃၽွၼ်းလႅၼ်ႈလႆႈ "ၵူႈလၢႆးလၢႆး" ၼႂ်းပွၵ်ႈလဵဝ်။
* **Bind**: ႁဵတ်းၵၢၼ်တင်းၵူႈတူဝ် ယဝ်ႉ "ႁူပ်ႇ" (Flatten) ၶဝ်ႈၵၼ်။
* **Guard**: ၸႂ်ႉတွၼ်ႈတႃႇ "တတ်းပႅတ်ႈ" လႅၼ်းတၢင်း ဢၼ်ႁဝ်းဢမ်ႇၶႂ်ႈလႆႈ။
