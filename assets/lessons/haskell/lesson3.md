## **Lesson 3: The GHCi Playground (REPL Basics and Commands)**

**GHCi** ၼႆႉ ပဵၼ် "ၵူၼ်းတွပ်ႇၶေႃႈထၢမ်" ၶႃႈဢေႃႈ။ ႁဝ်းတႅမ်ႈ Code ၶဝ်ႈၵႂႃႇသေ မၼ်းတေတွပ်ႇဢွၵ်ႇမႃးပၼ်ၵမ်းလဵဝ်။ မၼ်းႁွင်ႉဝႃႈ **REPL** (Read-Eval-Print Loop) ၶႃႈ။

### **1. လၢႆးပိုတ်ႇ GHCi**
ႁႂ်ႈပိုတ်ႇ Terminal (PowerShell ဢမ်ႇၼၼ် CMD) သေ တႅမ်ႈဝႃႈ:
```bash
ghci
```
သင်မၼ်းဢွၵ်ႇမႃးဝႃႈ `λ>` ဢမ်ႇၼၼ် `ghci>` ၼႆၵေႃႈ မၼ်းတမ်းဝၢင်းဝႆႉ ပုၼ်ႈတႃႇႁဝ်းတႅမ်ႈ Code ယဝ်ႉၶႃႈ။

### **2. ၶေႃႈၸိသင်ႇ (Commands) ဢၼ်လူဝ်ႇႁူႉဝႆႉ**
ၼႂ်း GHCi ၼၼ်ႉ မၼ်းမီးၶေႃႈၸိသင်ႇ ဢၼ်တႄႇလူၺ်ႈ **`:`** (Colon) ဢၼ်ၸွႆႈႁႂ်ႈႁဝ်းႁဵတ်းၵၢၼ်လႆႈငၢႆႈၶႃႈဢေႃႈ။

| Command | တီႈပွင်ႇ (Description) |
| :--- | :--- |
| **`:load`** (ဢမ်ႇၼၼ် **`:l`**) | တႃႇလူတ်ႇ (Load) File ဢၼ်ႁဝ်းတႅမ်ႈဝႆႉ (တူဝ်ယၢင်ႇ: `:l main.hs`) |
| **`:reload`** (ဢမ်ႇၼၼ် **`:r`**) | တႃႇလူတ်ႇၶိုၼ်း (Reload) ဝၢႆးသေႁဝ်းမူၼ်ႉမႄး Code ယဝ်ႉ |
| **`:type`** (ဢမ်ႇၼၼ် **`:t`**) | တႃႇထတ်းတူၺ်း "မဵဝ်းၶေႃႈမုၼ်း" (Data Type) ၶေႃႈမုၼ်းၼၼ်ႉ |
| **`:info`** (ဢမ်ႇၼၼ် **`:i`**) | တႃႇတူၺ်း "တီႈပွင်ႇ" (Details) ၶေႃႈမုၼ်း ဢမ်ႇၼၼ် Function ၼၼ်ႉ |
| **`:quit`** (ဢမ်ႇၼၼ် **`:q`**) | တႃႇဢွၵ်ႇ (Exit) ၵႂႃႇတီႈ GHCi |
| **`:clear`** | တႃႇလၢင်ႉ (Clear) ၼႃႈလိၵ်ႈ Terminal ႁဝ်း |

---

### **3. လၢႆးတႅမ်ႈ Code ၼႂ်း GHCi**
ႁဝ်းၸၢင်ႈတႅမ်ႈ Expression ငၢႆႈငၢႆႈ ၼႂ်းၼႆႉလႆႈၶႃႈဢေႃႈ။

**တူဝ်ယၢင်ႇ ၵၢၼ်ၼပ်ႉ (Math):**
```haskell
ghci> 5 + 5
10
ghci> 10 * 2
20
```

**တူဝ်ယၢင်ႇ ၵၢၼ်တႅမ်ႈ Function ၼႂ်း GHCi:**
သင်ႁဝ်းၶႂ်ႈတႅမ်ႈ Function ၼႂ်း GHCi ၵမ်းလဵဝ် ႁဝ်းၸၢင်ႈတႅမ်ႈလႆႈၼင်ႇၼႆၶႃႈ:
```haskell
ghci> let double x = x + x
ghci> double 10
20
```
*(မတ်ႉၸႂ်: တီႈၼႂ်း GHCi ၼၼ်ႉ လူဝ်ႇသႂ်ႇ `let` ဢွၼ်ၼႃႈ ပုၼ်ႈတႃႇပိုတ်ႇ Variable ဢမ်ႇၼၼ် Function ၶႃႈဢေႃႈ။)*



---

### **4. လၢႆးလူတ်ႇ File (.hs)**
ပုၼ်ႈတႃႇတႅမ်ႈ Code ဢၼ်ယၢဝ်းၼၼ်ႉ ႁဝ်းတေလႆႈတႅမ်ႈၼႂ်း VS Code သေ သိမ်းဝႆႉပဵၼ် `.hs` File (တူဝ်ယၢင်ႇ: `tmk.hs`)။
1. တႅမ်ႈ Code သေ သိမ်းဝႆႉ (Save)။
2. ၼႂ်း GHCi တႅမ်ႈဝႃႈ `:l tmk.hs`။
3. သင်ႁဝ်းမူၼ်ႉမႄး Code ၼႂ်း VS Code ယဝ်ႉ ႁႂ်ႈၼဵၵ်း Save သေ ၵႂႃႇတႅမ်ႈ `:r` ၼႂ်း GHCi ၶိုၼ်းၶႃႈ။

---

**Summary for TMK Students:**
* **`:l`** တႃႇပိုတ်ႇ File။
* **`:r`** တႃႇ Update Code မႂ်ႇ။
* **`:t`** တႃႇတူၺ်း Data Type။
