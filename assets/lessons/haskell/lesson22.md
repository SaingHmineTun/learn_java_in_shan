## **Lesson 22: The Base Case vs. The Recursive Step**

ၵၢၼ်တႅမ်ႈ Recursion ၼႆႉ မိူၼ်ၼင်ႇႁဝ်း "လူင်းၸၼ်ႉၵႆး" (Stairs) ၶႃႈဢေႃႈ။

### **1. The Base Case (တီႈၵိုတ်း / ၽၵ်းတူဢွၵ်ႇ)**
**Base Case** ပဵၼ်တွၼ်ႈဢၼ် "ငၢႆႈသုတ်း" ဢၼ်ႁဝ်းႁူႉၵႃႈၶၼ်မၼ်းယဝ်ႉ သေ ဢမ်ႇလူဝ်ႇၼပ်ႉသင်ထႅင်ႈယဝ်ႉ။
* မၼ်းပဵၼ် "တီႈၵိုတ်း" တွၼ်ႈတႃႇဢမ်ႇႁႂ်ႈ Function ႁွင်ႉတူဝ်မၼ်းၶိုၼ်း။
* **တူဝ်ယၢင်ႇ:** သင်ႁဝ်းၼပ်ႉ Factorial, Base Case ၵေႃႈပဵၼ် `0! = 1`။ သင်ႁဝ်းၼပ်ႉ List, Base Case ၵေႃႈပဵၼ် `[]` (List ဢၼ်လၢႆ)။



### **2. The Recursive Step (ၵၢၼ်သိုပ်ႇႁဵတ်း / ႁွင်ႉတူဝ်မၼ်းၶိုၼ်း)**
**Recursive Step** ပဵၼ်တွၼ်ႈဢၼ်ႁဝ်း "တတ်း" ပၼ်ႁႃ ႁႂ်ႈမၼ်းဢွၼ်ႇလူင်း သေ ႁွင်ႉၸႂ်ႉ Function ဢၼ်ၵဝ်ႇၼၼ်ႉၶိုၼ်း။
* မၼ်းတေလႆႈပဵၼ်ၵၢၼ် "ၶိုတ်းၸူး" (Moving towards) Base Case ယူႇသေႇသေႇ။
* **တူဝ်ယၢင်ႇ:** သင်ႁဝ်းမီး `n` ႁဝ်းတေလႆႈႁွင်ႉ `n - 1`။ သင်ႁဝ်းမီး `(x:xs)` ႁဝ်းတေလႆႈႁွင်ႉ `xs` (Tail)။

---

### **3. တူဝ်ယၢင်ႇ: ၵၢၼ်ၼပ်ႉတူဝ်လိၵ်ႈ (Summing a List)**
ႁႂ်ႈတူၺ်းလၢႆးတႅမ်ႈ Function `sum'` တွၼ်ႈတႃႇဢဝ်တူဝ်ၼပ်ႉၼႂ်း List မႃး ပวก ၵၼ်တင်းหมดၶႃႈ:

```haskell
sum' :: [Int] -> Int
-- 1. Base Case: သင် List လၢႆ ၵႃႈၶၼ်မၼ်းပဵၼ် 0
sum' [] = 0

-- 2. Recursive Step: ဢဝ် Head (x) ပวก တင်း sum ၶေႃႈ Tail (xs)
sum' (x:xs) = x + sum' xs
```

**လၢႆးမၼ်းႁဵတ်းၵၢၼ်:**
သင်ႁဝ်းသႂ်ႇ `sum' [1, 2, 3]`
1. `1 + sum' [2, 3]`
2. `1 + (2 + sum' [3])`
3. `1 + (2 + (3 + sum' []))`
4. `1 + (2 + (3 + 0))`  <-- ၵိုတ်းတီႈ Base Case
5. **Result:** `6`



---

### **4. ၵွပ်ႈသင်ႁဝ်းလူဝ်ႇမီး Base Case?**
သင်ဝႃႈၸဝ်ႈၶူးလိုမ်းတႅမ်ႈထႅဝ် `sum' [] = 0` ၼႆ:
* Haskell တေၼပ်ႉၵႂႃႇတိူၵ်းတိူၵ်း ဢမ်ႇမီးတီႈသုတ်း။
* Computer တေၸႂ်ႉ Memory (Stack) တေႃႇပေႃးသဵင်ႈ သေ တွပ်ႇဢွၵ်ႇမႃးဝႃႈ **"Stack space overflow"** ၶႃႈယဝ်ႉ။

---

### **Teacher's Lab: "Thinking about the Exit Door"**
ႁႂ်ႈလုၵ်ႈႁဵၼ်းဝူၼ်ႉတူၺ်း လွင်ႈၵၢၼ် "ၼပ်ႉထွႆလင်" (Countdown) 10 တေႃႇ 0 ၶႃႈ:
* **Base Case:** မိူဝ်ႈတူဝ်ၼပ်ႉပဵၼ် `0` ႁႂ်ႈၵိုတ်း။
* **Recursive Step:** ဢဝ်တူဝ်ၼပ်ႉ `n` ဢွၵ်ႇမႃး သေ ႁွင်ႉ `countdown (n - 1)`။

```haskell
countdown :: Int -> [Int]
countdown 0 = [0]
countdown n = n : countdown (n - 1)
```

---

**Summary for TMK Students:**
* **Base Case**: မိူၼ်ၼင်ႇ "Break" ၼႂ်းၵႃး၊ တႃႇၵိုတ်း Program။
* **Recursive Step**: မိူၼ်ၼင်ႇ "ၵၢၼ်ပၼ်ႇလေႃႉ"၊ တႃႇသိုပ်ႇႁဵတ်းၵၢၼ်လူၺ်ႈပၼ်ႁႃဢွၼ်ႇလူင်း။
* တိုၼ်းတေလႆႈမီး **Pattern Matching** တႃႇၸႅၵ်ႇသွင်တွၼ်ႈၼႆႉၶႃႈ။
