## **Lesson 17: Introduction to Tuples (Storing Different Data Types Together)**

### **1. Tuples ပႅၵ်ႇပိူင်ႈတင်း Lists ႁိုဝ်?**

| Feature | **List `[]`**                             | **Tuple `()`**                                         |
| :--- |:------------------------------------------|:-------------------------------------------------------|
| **Types** | တိုၼ်းတေလႆႈမိူၼ်ၵၼ်တင်းမူတ်း              | ၸၢင်ႈပႅၵ်ႇၵၼ်လႆႈ (Int, String, etc.)                   |
| **Size** | လႅၵ်ႈလၢႆႈလႆႈ (သႂ်ႇထႅမ် ဢမ်ႇၼၼ် ထွၼ်ဢွၵ်ႇ) | **Fixed Size** (ပိုတ်ႇဝႆႉၵႃႈႁိုဝ် တိုၼ်းၵိုတ်းၵႃႈၼၼ်ႉ) |
| **Usage** | ၸႂ်ႉတႃႇသိမ်းၶူဝ်းမဵဝ်းလဵဝ်ၵၼ်လၢႆလၢႆတူဝ်   | ၸႂ်ႉတႃႇသိမ်း "ၶေႃႈမုၼ်း" ဢၼ်ၵဵဝ်ႇၵၼ်                   |



### **2. လၢႆးတႅမ်ႈ (Syntax)**
ႁဝ်းၸႂ်ႉ **ဝူင်ႇၵွတ်ႇ `()`** လႄႈ **ၸုၼ် `,`** တႃႇသၢင်ႈ Tuple ၶႃႈ။

* **တူဝ်ယၢင်ႇ:** `("Sai Mao", 25)`
* ၼႂ်း Tuple ၼႆႉ တွၼ်ႈၼႃႈပဵၼ် **String** သေ တွၼ်ႈလင်ပဵၼ် **Int** ၶႃႈ။

### **3. Pair Functions (တွၼ်ႈတႃႇ Tuple ဢၼ်မီး 2 တူဝ်)**
ၼႂ်း Haskell မီး Function ဢၼ်ၸွႆႈထတ်းဢဝ်ၶေႃႈမုၼ်း ၼႂ်း Tuple ဢၼ်မီး 2 တူဝ် (Pair) ၶႃႈ:

* **`fst` (First):** ဢဝ်ၶေႃႈမုၼ်းတူဝ်ၼႃႈသုတ်း။
    * `fst ("TMK", 100)` `->` `"TMK"`
* **`snd` (Second):** ဢဝ်ၶေႃႈမုၼ်းတူဝ်ထူၼ်ႈသွင်။
    * `snd ("TMK", 100)` `->` `100`



### **4. Zip: လၢႆးသိုပ်ႇ List 2 ဢၼ်**
Function **`zip`** ၼႆႉ မၼ်းတေဢဝ် List 2 ဢၼ် မႃး "ၸပ်းၵူပ်ႉ" ၵၼ် ႁႂ်ႈပဵၼ် List ၶေႃႈ Tuples ၶႃႈ။

```haskell
ghci> zip [1, 2, 3] ["One", "Two", "Three"]
[(1,"One"), (2,"Two"), (3,"Three")]
```
> **Tip:** သင်ဝႃႈ List ၼိုင်ႈဢၼ် ယၢဝ်းသေထႅင်ႈဢၼ်ၼိုင်ႈၼႆ `zip` တေတတ်းဢဝ်ၵူၺ်းတွၼ်ႈဢၼ်ၵိုင်ႇၵၼ်သေ ဢၼ်လိူဝ်ၼၼ်ႉမၼ်းတေပႅတ်ႈၶႃႈဢေႃႈ။

### **Teacher's Lab: "Finding Right Triangles"**
ႁဝ်းၸၢင်ႈၸႂ်ႉ **List Comprehension** တင်း **Tuples** မႃးႁဵတ်းပၢႆးၼပ်ႉယၢပ်ႇယၢပ်ႇလႆႈၶႃႈ။ တူဝ်ယၢင်ႇ: သၢင်ႈ List ၶေႃႈတူဝ်ၼပ်ႉ 3 တူဝ် `(a, b, c)` ဢၼ်ပဵၼ်သၢမ်ၸဵင်ႇမၢၼ်ႇ (Right Triangle) ဢၼ်မီးလၵ်းၵႅတ်း $a^2 + b^2 = c^2$ လႄႈ တင်းหมดပဵၼ် 24။

```haskell
-- တႅမ်ႈၼႂ်း GHCi
let triangles = [ (a,b,c) | c <- [1..10], b <- [1..c], a <- [1..b], a^2 + b^2 == c^2, a+b+c == 24]
```

**Summary for TMK Students:**
* **Tuple `()`** ၸၢင်ႈသိမ်း Type ဢၼ်ပႅၵ်ႇၵၼ်ဝႆႉၸွမ်းၵၼ်။
* **Fixed Size:** သင်ပိုတ်ႇဝႆႉ 2 တူဝ် မၼ်းၵေႃႈတေပဵၼ် 2 တူဝ်ယူႇသေႇသေႇ (ဢမ်ႇၸၢင်ႈထႅမ်ပဵၼ် 3 တူဝ်လႆႈ)။
* **`fst` / `snd`**: ၸႂ်ႉလႆႈၵူၺ်းတင်း Pairs (Tuple ဢၼ်မီး 2 တူဝ်)။
* **`zip`**: ၸႂ်ႉတႃႇဢဝ် List 2 ဢၼ် မႃးၵူပ်ႉၵၼ်။
