# Lesson 13: "The Basic Syntax of C#" (ပိူင်သၢင်ႈပိုၼ်ႉထၢၼ် C#)

ၼႂ်းၵၢၼ်တႅမ်ႈ C# ၼၼ်ႉ မၼ်းမီးပိူင်လၢႆးတႅမ်ႈ ဢၼ်ႁဝ်းလူဝ်ႇတိုဝ်းၵမ်ၸွမ်း ႁႂ်ႈ Compiler ပွင်ႇၸႂ်လႆႈမၢၼ်ႇမႅၼ်ႈဢေႃႈ။

### 1. Statements (ၶေႃႈသင်ႇ)
**Statement** ၼႆႉပဵၼ်ၶေႃႈသင်ႇၼိုင်ႈဢၼ် ဢၼ်ႁဝ်းလၢတ်ႈတေႃႇၶွမ်းပိဝ်ႇတႃႇ ႁႂ်ႈမၼ်းႁဵတ်းၵၢၼ်သင်သၵ်းပိူင်။ ၼႂ်း C# ၼၼ်ႉ ၵူႈ Statement တေလႆႈသုတ်းတူဝ်တင်း **Semicolon ( ; )** ဢေႃႈ။
* **တူဝ်ယင်ႇ:**
  ```csharp
  Console.WriteLine("Mao App Academy"); // ၼႆႉပဵၼ် Statement ၼိုင်ႈဢၼ်
  int age = 20; // ၼႆႉၵေႃႈပဵၼ် Statement ၼိုင်ႈဢၼ်
  ```
> **သတိ:** ပေႃးလืมသႂ်ႇ `;` မၼ်းတေပဵၼ် Compilation Error (CS1002) ၵမ်းလဵဝ်ၶႃႈ။

### 2. Code Blocks (ၸုမ်းၶူတ်ႉ { })
မိူဝ်ႈႁဝ်းမီး Statement လၢႆလၢႆဢၼ် ဢၼ်ႁဝ်းၶႂ်ႈႁူမ်ႈမၼ်းဝႆႉၸွမ်းၵၼ်၊ ႁဝ်းတေၸႂ်ႉ **Curly Braces `{ }`** တႃႇႁဵတ်းပဵၼ် **Code Block** ဢေႃႈ။
* Code Block ၼႆႉၵႆႉၸႂ်ႉၼႂ်း `Class`, `Method`, `If-Else`, လႄႈ `Loops`။
* Variable ဢၼ်သၢင်ႈဝႆႉၼႂ်း Block ၼိုင်ႈဢၼ်ၼၼ်ႉ တေၸႂ်ႉလႆႈၼႂ်း Block ၼၼ်ႉၵူၺ်း (Scope)။

### 3. Whitespace (လွၵ်းပဝ်ႇ)
C# ၼႆႉမၼ်းဢမ်ႇသူၼ်ၸႂ်လွင်ႈ **Whitespace** (လွၵ်းပဝ်ႇ/Space) ၶႃႈ။ 
ႁဝ်းတေတႅမ်ႈ Code ႁႂ်ႈႁၢင်ႇၵၼ် ဢမ်ႇၼၼ် တႅမ်ႈၸပ်းၵၼ်ၵေႃႈ ၶွမ်းပွင်ႇၸႂ်မိူၼ်ၵၼ်။ 
ၵူၺ်းၵႃႈ ႁဝ်းလူဝ်ႇၸတ်းႁႂ်ႈမၼ်းႁၢင်ႈလီ တႃႇႁႂ်ႈ "ၵူၼ်း" ဢၢၼ်ႇငၢႆႈၶႃႈ။



---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Statement:** ၶေႃႈသင်ႇၼိုင်ႈထႅဝ် ဢၼ်သင်ႇႁႂ်ႈ Computer ႁဵတ်းၵၢၼ်။
* **Semicolon (;):** ၶိူင်ႈမၢႆဢၼ်ၸႂ်ႉတႃႇ "ပၼ်သိုၼ်" ဝႃႈ ၶေႃႈသင်ႇၼၼ်ႉ သုတ်းတူဝ်ယဝ်ႉ။
* **Code Block:** ၸုမ်းၶူတ်ႉဢၼ်ယူႇၼႂ်း `{ }` ဢၼ်ႁဵတ်းၵၢၼ်ၸွမ်းၵၼ် ပဵၼ်ပိူင်ၼိုင်ႈ။
* **Case-Sensitivity:** ၼမ်ႉၵတ်ႉဢၼ်ၸႅၵ်ႇတူဝ်လိၵ်ႈ "တူဝ်လူင်" (A) လႄႈ "တူဝ်လဵၵ်း" (a) ဝႃႈမၼ်းဢမ်ႇမိူၼ်ၵၼ် (ၼႂ်း C# ၼႆႉ `Console` လႄႈ `console` ဢမ်ႇမိူၼ်ၵၼ်ၶႃႈ)။
* **Indentation:** ၵၢၼ်တႅမ်ႈ Code ႁႂ်ႈယဵၼ်ႇၶဝ်ႈၵႂႃႇၽၢႆႇၼႂ်း (Tab) ၼႂ်း Code Block တႃႇႁႂ်ႈဢၢၼ်းငၢႆႈ။
