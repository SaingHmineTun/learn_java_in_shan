# Lesson 253: "WinForms Basics: The Designer, Toolbox, and Properties"

မိူဝ်ႈႁဝ်းသၢင်ႈ Project ပဵၼ် **Windows Forms App (.NET)** ယဝ်ႉၼၼ်ႉ ႁဝ်းတေႁၼ်ၼႃႈတၢင်ႇ Visual Studio ပႅၵ်ႇၵႂႃႇတီႈ Console App ၶႃႈ။

### 1. The Designer (ၼႃႈၸေႃးတႅမ်ႈႁၢင်ႈ)
ၼႆႉပဵၼ် "တီႈလႅၼ်ႈၵၢၼ်" ပိူင်လူင်ၶွင်ႁဝ်းၶႃႈ။
* မၼ်းတေၼႄႁၢင်ႈ **Form** (Window) ဢၼ်ပဝ်ႇလၢႆဝႆႉ။
* ႁဝ်းၸၢင်ႈတူၺ်းလႆႈဝႃႈ App ႁဝ်းတေမီးႁၢင်ႈၽၢင်ၸိူင်ႉႁိုဝ် မိူဝ်ႈ User ၸႂ်ႉတႄႉတႄႉ။
* **Shortcut:** ၼဵၵ်း `F7` တႃႇတူၺ်း Code သေ ၼဵၵ်း `Shift + F7` တႃႇၶိုၼ်းမႃးၼႃႈ Designer ၶႃႈ။

### 2. The Toolbox (ၵွင်ႇၶိူင်ႈမိုဝ်း)
ၵႆႉမီးယူႇၽၢႆႇသၢႆႉၶွင် Visual Studio ၶႃႈ။
* မၼ်းမီး **Controls** (ၶိူင်ႈၸႂ်ႉ) ၵူႈပိူင် မိူၼ်ၼင်ႇ `Button`, `Label`, `TextBox`, `CheckBox`။
* **လၢႆးၸႂ်ႉ:** ႁဝ်းၵူၺ်းလႆႈ "လၢၵ်ႈ" (Drag) ၶဝ်လုၵ်ႉတီႈ Toolbox သေမႃး "ဝၢင်း" (Drop) ၼိူဝ် Form ၶႃႈ။

### 3. Properties Window (လွၵ်းမၵ်းမၼ်ႈၵႃႈၶၼ်)
ၵႆႉမီးယူႇၽၢႆႇၶႂႃ ၽၢႆႇတႂ်ႈ Solution Explorer ၶႃႈ။ ၼႆႉပဵၼ်တီႈဢၼ်ႁဝ်းတေမႄးႁၢင်ႈၽၢင် Controls ၶႃႈ။
* **Text:** တႃႇလႅၵ်ႈလိၵ်ႈဢၼ်ၼႄၼိူဝ် Button (မိူၼ်ၼင်ႇ "ၼဵၵ်းတီႈၼႆႉ")။
* **Name:** ၸိုဝ်ႈဢၼ်ႁဝ်းတေၸႂ်ႉတႅမ်ႈ Code (မိူၼ်ၼင်ႇ `btnSave`)။
* **BackColor / ForeColor:** တႃႇမႄးသီပိုၼ်ႉလႄႈသီတူဝ်လိၵ်ႈ။
* **Font:** တႃႇမႄးသႅၼ်းတူဝ်လိၵ်ႈ လႄႈ ပိူင်လိၵ်ႈတႆး။



---

### လွင်ႈလမ်ႇလွင်ႈ: ၸိုဝ်ႈ (Name) vs လိၵ်ႈ (Text)
လုၵ်ႈႁဵၼ်းမႂ်ႇၵႆႉယုင်ႈၵႄႈ 2 ဢၼ်ၼႆႉၶႃႈ:
* **Text:** ပဵၼ်ဢၼ် **User ႁၼ်** (မိူၼ်ၼင်ႇ "Login")။
* **Name:** ပဵၼ်ဢၼ် **Developer ၸႂ်ႉ** ၼႂ်း Code (မိူၼ်ၼင်ႇ `btnLogin`)။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Control:** ၶိူင်ႈၸႂ်ႉၵူႈယိူင်ႈ ဢၼ်ဝႆႉၼိူဝ် Form (Button, Label, etc.)။
* **Form:** ၼႃႈတၢင်ႇ (Window) ပိူင်လူင်ၶွင် App။
* **Visual Studio:** ၶိူင်ႈမိုဝ်း (IDE) ဢၼ်ႁဝ်းၸႂ်ႉတႅမ်ႈ C#။
* **Drag and Drop:** လၢႆးၵၢၼ်လၢၵ်ႈၶဝ်မႃးဝၢင်း တႅၼ်းၵၢၼ်တႅမ်ႈ Code ႁၢင်ႈၽၢင်။
