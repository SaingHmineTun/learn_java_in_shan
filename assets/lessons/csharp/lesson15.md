# Lesson 15: "Integral Numeric Types" (ပိူင်ၶေႃႈမုၼ်း မၢႆၼပ်ႉတဵမ်)

မၢႆၼပ်ႉတဵမ် (Integral Types) ၼႆႉပဵၼ် မၢႆၼပ်ႉဢၼ်ဢမ်ႇပႃး "ၸုၵ်း" (Decimal) မိူၼ်ၼင်ႇ 1, 50, -100 ၼၼ်ႉယဝ်ႉ။ ႁဝ်းလိူၵ်ႈၸႂ်ႉမၼ်းၸွမ်းၼင်ႇ "တၢင်းယႂ်ႇ" ၶေႃႈမုၼ်းႁဝ်းၶႃႈ။

### 1. ပိူင်လူင် 4 သႅၼ်း (Common Types)

| Type | Size (Bytes) | Range (တၢင်းၵႂၢင်ႈမၼ်း) | မိူဝ်ႈလႂ်လူဝ်ႇၸႂ်ႉ                            |
| :--- | :--- | :--- |:----------------------------------------------|
| **byte** | 1 Byte | 0 to 255 | တႃႇသိမ်းၶေႃႈမုၼ်းဢွၼ်ႇ                       |
| **short** | 2 Bytes | -32,768 to 32,767 | တႃႇသိမ်းမၢႆၼပ်ႉဢၼ်ဢမ်ႇပေႃးယႂ်ႇ                |
| **int** | 4 Bytes | ±2.1 Billion | **(Standard)** ၸႂ်ႉၼမ်သေပိူၼ်ႈ တႃႇမၢႆၼပ်ႉတဵမ် |
| **long** | 8 Bytes | ယႂ်ႇၼႃႇ (±9 Quintillion) | တႃႇသိမ်းမၢႆၼပ်ႉယႂ်ႇ (World Population)        |



### 2. လၢႆးတႅမ်ႈ Code
မိူဝ်ႈႁဝ်းတႅမ်ႈ `long` ၼၼ်ႉ ႁဝ်းၵႆႉသႂ်ႇတူဝ်လိၵ်ႈ `L` ဝႆႉၽၢႆႇလင် တႃႇလၢတ်ႈၼႄ Compiler ဝႃႈ ၼႆႉပဵၼ် Long တႄႉတႄႉဢေႃႈ။

```csharp
byte myAge = 25;
short score = 30000;
int population = 2000000;
long worldDebt = 300000000000L; // သႂ်ႇ L ဝႆႉၽၢႆႇလင်
```

### 3. Digit Separator (တူဝ်ၸႅၵ်ႇမၢႆၼပ်ႉ)
ၼႂ်း C# သႅၼ်းမႂ်ႇ ႁဝ်းၸႂ်ႉ Underscore (`_`) တႃႇၸႅၵ်ႇမၢႆၼပ်ႉယႂ်ႇယႂ်ႇ ႁႂ်ႈဢၢၼ်းငၢႆႈလႆႈၶႃႈ (မၼ်းဢမ်ႇမီးၽိူၼ်သင် တေႃႇၵၢၼ်ႁဵတ်းၵၢၼ်ၶွမ်း)။
* **တူဝ်ယင်ႇ:** `int money = 1_000_000;` (မိူၼ်တင်း 1,000,000 ယဝ်ႉ)

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Integral:** မၢႆၼပ်ႉတဵမ် ဢၼ်ဢမ်ႇပႃး ဝူတ်ႇ (Fraction) ဢမ်ႇၼၼ် ၸုၵ်း (Decimal)။
* **Bit / Byte:** ႁူဝ်ၼပ်ႉသိမ်းၶေႃႈမုၼ်းၼႂ်းၶွမ်း (8 Bits = 1 Byte)။
* **Overflow:** ပၼ်ႁႃဢၼ်ပဵၼ်မိူဝ်ႈ ႁဝ်းဢဝ်မၢႆၼပ်ႉယႂ်ႇယႂ်ႇ ၵႂႃႇသႂ်ႇၼႂ်း "ၵွင်" ဢၼ်လဵၵ်းလဵၵ်း (မိူၼ်ၼင်ႇ ဢဝ် 500 သႂ်ႇၼႂ်း `byte`)။
* **Signed vs Unsigned:** `Signed` (မီးမၢႆ - လႄႈ +)၊ `Unsigned` (မီးမၢႆ + ၵူၺ်း မိူၼ်ၼင်ႇ `uint`, `ushort`)။
