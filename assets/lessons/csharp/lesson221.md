# Lesson 221: "Quantifiers: Any, All, and Contains"

### 1. Any (မီးမၢင်တူဝ်ႁႃႉ?)
**`.Any()`** ၸွႆးၵူတ်ႇထတ်းဝႃႈ "ၸွင်ႇမီးၶေႃႈမုၼ်းသေတူဝ်တူဝ် ဢၼ်မႅၼ်ႈၸွမ်းပၵ်းၵၢၼ်ၶႂ်ႈလႆႈႁႃႉ?"။ 
သင်ႁႃႁၼ် ၼိုင်ႈတူဝ်ၵေႃႈ မၼ်းတေသူင်ႇ `True` ၵမ်းလဵဝ်ၶႃႈ။

* **တူဝ်ယင်ႇ:** "ၸွင်ႇမီးလုၵ်ႈႁဵၼ်း ဢႃႇယုလိူဝ် 30 ယူႇၼႂ်း Class ၼႆႉႁႃႉ?"
```csharp
bool hasAdults = students.Any(s => s.Age > 30);
```

### 2. All (မႅၼ်ႈၵူႈတူဝ်ႁႃႉ?)
**`.All()`** ၸွႆးၵူတ်ႇထတ်းဝႃႈ "ၸွင်ႇၶေႃႈမုၼ်း **တင်းမူတ်း** မႅၼ်ႈၸွမ်းပၵ်းၵၢၼ်ႁႃႉ?"။ 
သင်မီးတူဝ်ၼိုင်ႈ ဢၼ်ဢမ်ႇမႅၼ်ႈ... မၼ်းတေသူင်ႇ `False` ၶႃႈ။

* **တူဝ်ယင်ႇ:** "ၸွင်ႇလုၵ်ႈႁဵၼ်း **ၵူႈၵေႃႉ** ဢွင်ႇပူၼ်ႉ 50 မၢႆႁႃႉ?"
```csharp
bool everyonePassed = scores.All(s => s >= 50);
```



### 3. Contains (မီးတူဝ်ၼႆႉႁႃႉ?)
**`.Contains()`** ၸႂ်ႉတႃႇႁႃ "ၵႃႈၶၼ်တႅတ်ႈတေႃး" (Specific Value) ဢၼ်မီးၼႂ်း Collection ၶႃႈ။ 
မၼ်းဢမ်ႇၸႂ်ႉ Lambda၊ မၼ်းၵူၺ်းႁႃတူဝ်ဢၼ်မိူၼ်ၵၼ်ၵူၺ်းၶႃႈ။

* **တူဝ်ယင်ႇ:** "ၸွင်ႇၼႂ်း List ဝဵင်းႁဝ်း မီးဝဵင်း 'Muse' ယူႇႁႃႉ?"
```csharp
List<string> cities = new List<string> { "Muse", "Lashio", "Taunggyi" };
bool isMuseInList = cities.Contains("Muse"); // True
```

### 4. ပၵ်းၵၢၼ်တႅၵ်ႈၵၼ် (Comparison Table)

| Method | လွင်ႈတၢင်းၵူတ်ႇထတ်း                  | ၽွၼ်းလႆႈ (Return) |
| :--- |:-------------------------------------| :--- |
| **Any** | မီးမၢင်တူဝ် (At least one)           | `bool` |
| **All** | ၵူႈတူဝ်တေလႆႈမႅၼ်ႈ (Every single one) | `bool` |
| **Contains** | မီးတူဝ်ဢၼ်မိူၼ်ၵၼ်တႅတ်ႈတေႃးယူႇႁႃႉ    | `bool` |

### ႁဵတ်းသင်ႁဝ်းၸင်ႇလူဝ်ႇၸႂ်ႉ?
* **Validation:** ၸွႆးၵူတ်ႇထတ်းၶေႃႈမုၼ်း ဢွၼ်တၢင်းတေ Save (တူဝ်ယင်ႇ: `if (users.Any(u => u.Email == newEmail))`).
* **Performance:** `.Any()` ဝႆးလိူဝ် `.Count() > 0` ယွၼ်ႉမၼ်းႁႃႁၼ် 1 တူဝ်ၵေႃႈ မၼ်းၵိုတ်းၵမ်းလဵဝ်၊ ဢမ်ႇလူဝ်ႇပၼ်ႇ Loop တေႃႇသုတ်းၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Quantifier:** Method ဢၼ်ၸွႆႈ မၵ်းမၼ်ႈ "တၢင်းၼမ်" ဢမ်ႇၼၼ် "ငဝ်းလၢႆးႁူမ်ႈ" ၶွင်ၶေႃႈမုၼ်း။
* **Boolean Result:** ၽွၼ်းလႆႈဢၼ်မီး True (မႅၼ်ႈ) ဢမ်ႇၼၼ် False (ၽိတ်း) ၵူၺ်း။
* **Early Exit:** ၵၢၼ်ဢၼ် Method ၵိုတ်းၵၢၼ်ႁဵတ်းၵမ်းလဵဝ် မိူဝ်ႈလႆႈၽွၼ်းလႆႈယဝ်ႉ (မိူၼ်ၼင်ႇ Any)။
