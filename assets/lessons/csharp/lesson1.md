# Lesson 1: "The World of C# and .NET" (လေႃးၵ C# လႄႈ .NET)

မႂ်ႇသုင်လုၵ်ႈႁဵၼ်းၵူႈၵေႃႉၶႃႈ! မိူဝ်ႈၼႆႉႁဝ်းတေမႃးတႄႇလဵပ်ႈႁဵၼ်း လွင်ႈၽႃႇသႃႇ Programmimg ဢၼ်ၸိုဝ်ႈဝႃႈ **C# (C-Sharp)** လႄႈ **.NET Ecosystem** ဢေႃႈ။ ၼႆႉပဵၼ်ၵၢၼ်တႄႇပိုတ်ႇၽၵ်းတူ တႃႇတေၵႂႃႇပဵၼ် Software Developer ၵေႃႉၼိုင်ႈယဝ်ႉ။

### 1. C# ၼႆႉပဵၼ်သင်? (What is C#?)
C# ၼႆႉပဵၼ်ၽႃႇသႃႇ Modern Programming ဢၼ် Microsoft သၢင်ႈဝႆႉ။ မၼ်းပဵၼ်ၽႃႇသႃႇဢၼ်:
* **Object-Oriented (OOP):** ၸတ်းၵၢၼ်ၶူတ်ႉ (Code) ႁဝ်းႁႂ်ႈမိူၼ်ၼင်ႇ ၶူဝ်းၶေႃႈၼႂ်းလေႃးၵတႄႉ။
* **Type-Safe:** မၼ်းတေၸွႆးၵူတ်ႇထတ်း Error ႁႂ်ႈႁဝ်း ၸဵမ်မိူဝ်ႈတႄႇတႅမ်ႈ ႁႂ်ႈဢမ်ႇလူႉငၢႆႈ။
* **Versatile:** တႅမ်ႈလႆႈတင်း Desktop (Windows), Mobile (Android/iOS), Web လႄႈ Game (Unity) ဢေႃႈ။

### 2. .NET ၼႆႉပဵၼ်သင်? (What is .NET?)
လုၵ်ႈႁဵၼ်းၵုမ်ႇမၢႆတွင်းဝႃႈ **C# ပဵၼ်ၵႂၢမ်းလၢတ်ႈ (Language)** တူဝ်ၼိုင်ႈ၊ လုၵ်ႉ **.NET ၼႆႉသမ်ႉ ပဵၼ်ႁွင်ႈၵၢၼ် (Platform)** ဢၼ်ႁဵတ်းႁႂ်ႈၵႂၢမ်းလၢတ်ႈၼၼ်ႉ ႁဵတ်းၵၢၼ်လႆႈၼၼ်ႉယဝ်ႉ။
* မိူဝ်ႈၵွၼ်ႇ .NET ၸႂ်ႉလႆႈၼိူဝ် Windows ၵူၺ်း။
* တေႃႈလဵဝ် .NET 7/8/9 ၼႆႉ ပဵၼ် **Cross-platform** ယဝ်ႉ၊ မၼ်းၸႂ်ႉလႆႈတင်း Windows, Linux, လႄႈ macOS ဢေႃႈ။

### 3. ပိူင်လူင်မၼ်း (Core Benefits)
တႃႇလုၵ်ႈႁဵၼ်းႁဝ်း ဢၼ်တေတႅမ်ႈ App တႃႇ **Shan Community** ၼၼ်ႉ C# ၼႆႉလီတီႈ:
1. မၼ်းၵမ်ႉထႅမ် **Unicode (Shan Script)** လႆႈလီၼႃႇ။
2. မၼ်းမီး Library ၼမ် တႃႇၸွႆးႁဝ်းတႅမ်ႈ App ႁႂ်ႈယဝ်ႉၽႂ်း။
3. မၼ်းမိူၼ်တင်း Java လႄႈ Dart (Flutter) တႄႉတႄႉ၊ ပေႃးႁဵၼ်း C# ယဝ်ႉ တေၵႂႃႇတႅမ်ႈ Flutter ၵေႃႈငၢႆႈ။

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Programming Language:** ၽႃႇသႃႇၵႂၢမ်းလၢတ်ႈ ဢၼ်ၸႂ်ႉတႃႇသင်ႇၵၢၼ် ၶွမ်းပိဝ်ႇတႃႇ (Computer)။
* **Framework / Platform:** ၸုမ်းၶိူင်ႈၵမ်ႉထႅမ် (Tools) လႄႈ ႁွင်ႈၵၢၼ် ဢၼ်ၸွႆးႁႂ်ႈၵၢၼ်တႅမ်ႈ App ငၢႆႈၶိုၼ်ႈ။
* **Cross-platform:** ၼမ်ႉၵတ်ႉဢၼ် ၶူတ်ႉ (Code) ပိူင်ၼိုင်ႈ ၸႂ်ႉလႆႈၼိူဝ် ၶိူင်ႈၸၢၵ်ႈလၢႆလၢႆပိူင် (မိူၼ်ၼင်ႇ Android လႄႈ Windows)။
* **High-level Language:** ၽႃႇသႃႇ Programming ဢၼ်ၵူၼ်းႁဝ်း လူဢၢၼ်းပွင်ႇၸႂ်လႆႈငၢႆႈ (မိူၼ်ၵႂၢမ်းဢင်းၵိတ်း)။
