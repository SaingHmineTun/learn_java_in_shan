# Lesson 20: "The var Keyword" (ၵၢၼ်ၸႂ်ႉ Variable ဢၼ်ဢမ်ႇပိုၼ်ၽၢဝ်ႇ Type တႅတ်ႉတႅတ်ႉ)

**var** ၼႆႉပဵၼ်ၵၢၼ်လၢတ်ႈၼႄ Compiler ဝႃႈ "ႁႂ်ႈၸဝ်ႈၵဝ်ႇတူၺ်း ၵႃႈၶၼ် (Value) မၼ်းသေ တႅတ်ႉတတ်းပၼ်ဝႃႈ မၼ်းပဵၼ် Type သင်" ၼႆယဝ်ႉ။ 
ႁဝ်းႁွင်ႉလၢႆးၼႆႉဝႃႈ **Implicit Typing** ဢေႃႈ။

### 1. လၢႆးတႅမ်ႈ Code
တႅၼ်းတၢင်ဢၼ်ႁဝ်းတေတႅမ်ႈ `string` ဢမ်ႇၼၼ် `int` ၼၼ်ႉ ႁဝ်းၸႂ်ႉ `var` လႆႈယူႇၶႃႈ:

```csharp
var studentName = "Sai Mao"; // Compiler တေႁူႉဝႃႈပဵၼ် string
var studentAge = 25;         // Compiler တေႁူႉဝႃႈပဵၼ် int
var isLearning = true;       // Compiler တေႁူႉဝႃႈပဵၼ် bool
```



### 2. ပိူင်တႃႇၸႂ်ႉတိုဝ်းမၼ်း (Rules of var)
မၼ်းမီးလွင်ႈလူဝ်ႇၾၢင်ႉ 3 ပိူင် မိူဝ်ႈတေၸႂ်ႉ `var` ၶႃႈ:
1. **လူဝ်ႇ Initialization ၵမ်းလဵဝ်:** ႁဝ်းတေလႆႈသႂ်ႇၵႃႈၶၼ် (Value) ႁူမ်ႈၵၼ်တင်း `var` ၼႂ်းထႅဝ်လဵဝ်ၵၼ် (တူဝ်ယင်ႇ: `var x;` ၼႆႉၽိတ်းၶႃႈ၊ ယွၼ်ႉ Compiler ဢမ်ႇႁူႉဝႃႈ x ပဵၼ်သင်)။
2. **လႅၵ်ႈလၢႆႈ Type ဢမ်ႇလႆႈ:** ပေႃးႁဝ်းသႂ်ႇ `"Hello"` သႂ်ႇၼႂ်း `var` ယဝ်ႉ၊ မၼ်းတေပဵၼ် `string` တႃႇသေႇ။ ႁဝ်းတေဢဝ်မၢႆၼပ်ႉ `10` မႃးသႂ်ႇၼႂ်းမၼ်း ထႅင်ႈပွၵ်ႈဢမ်ႇလႆႈယဝ်ႉ။
3. **ၸႂ်ႉလႆႈၼႂ်း Local ၵူၺ်း:** `var` ၼႆႉၸႂ်ႉလႆႈၼႂ်း Method ဢမ်ႇၼၼ် Code Block ၵူၺ်း၊ ၸႂ်ႉပဵၼ် Class Level Variable ဢမ်ႇလႆႈၶႃႈ။

### 3. မိူဝ်ႈလႂ်လူဝ်ႇၸႂ်ႉ `var`?
Developer ၵႆႉၸႂ်ႉ `var` မိူဝ်ႈ:
* ၸိုဝ်ႈ Data Type မၼ်းယၢဝ်းၼႃႇ (တူဝ်ယင်ႇ: `Dictionary<int, string> list = ...`)။
* မိူဝ်ႈႁဝ်းႁူႉၸႅင်ႈလႅင်းယူႇယဝ်ႉဝႃႈ ၶေႃႈမုၼ်းၽၢႆႇၶႂႃပဵၼ်သင် (တူဝ်ယင်ႇ: `var result = new Student();`)။

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Implicitly Typed:** ၵၢၼ်ပိုၼ်ၽၢဝ်ႇ Variable ဢၼ်ပႂ်ႉႁႂ်ႈ Compiler မၵ်ႉမၼ်ႈ Type ပၼ်ႁင်းမၼ်း။
* **Explicitly Typed:** ၵၢၼ်ပိုၼ်ၽၢဝ်ႇ Variable ဢၼ်တႅမ်ႈၸိုဝ်ႈ Type မၼ်းတႅတ်ႉတႅတ်ႉတေႃးတေႃး (တူဝ်ယင်ႇ: `int x = 5;`)။
* **Local Variable:** Variable ဢၼ်သၢင်ႈဝႆႉၼႂ်း Method သေ ၸႂ်ႉလႆႈၼႂ်း Method ၼၼ်ႉၵူၺ်း။
* **Strongly Typed:** ၼမ်ႉၵတ်ႉ C# ဢၼ် Variable ၼိုင်ႈဢၼ် တေလႆႈမီး Type ၼိုင်ႈပိူင်သေ ဝၢႆးသေပိုၼ်ၽၢဝ်ႇယဝ်ႉ တေလႅၵ်ႈလၢႆႈ ႁႂ်ႈပဵၼ်ထႅင်းမဵဝ်းၼိုင်ႈ ဢမ်ႇလႆႈ။
