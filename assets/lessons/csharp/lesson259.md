# Lesson 259: "WPF Layouts: Grids, StackPanels, and WrapPanels"

ၼႂ်း WinForms ႁဝ်းၸႂ်ႉ "မၢႆ Pixel" တႅတ်ႈတေႃး တႃႇဝၢင်း Controls၊ ၵူၺ်းၵႃႈၼႂ်း WPF ႁဝ်းၸႂ်ႉ **"Layout Panels"** တႃႇၸတ်းဝၢင်းၶဝ်ႁင်းၵူၺ်း (Automatic Layout) ၶႃႈ။

### 1. The Grid (ၵူၼ်ႁူဝ်ၸႂ်ၶွင် WPF)
**Grid** မိူၼ်တင်း Table ၶႃႈ။ မၼ်းၸႅၵ်ႇၼႃႈၸေႃးပဵၼ် ထႅဝ်ၼွၼ်း (Rows) လႄႈ ထႅဝ်တင်ႈ (Columns)။
* **လွင်ႈလီ:** ႁဝ်းၸၢင်ႈမၵ်းမၼ်ႈသႅၼ်းမၼ်းပဵၼ် `*` (Star Sizing) တႃႇႁႂ်ႈမၼ်း "ယိုတ်းတူဝ်" ၸွမ်းၼႃႈၸေႃးလႆႈငၢႆႈငၢႆႈၶႃႈ။
* **တူဝ်ယင်ႇ:** ၸႅၵ်ႇၼႃႈၸေႃးပဵၼ် 2 ၽၢႆႇ (သၢႆႉ/ၶႂႃ) တႃႇၼႄ Menu လႄႈ Content။



### 2. StackPanel (ၵၢၼ်ၸတ်းဝၢင်းၸွမ်းထႅဝ်)
ၼႆႉပဵၼ်တူဝ်ဢၼ်ငၢႆႈသုတ်းၶႃႈ။ မၼ်းတေဢဝ် Controls မႃး "ဝၢင်းသိုပ်ႇၵၼ်" ၸိူင်ႉၼႆ:
* **Vertical (ထႅဝ်တင်ႈ):** ဝၢင်းတေႃႇၵၼ် လူင်းတႂ်ႈ (Default)။
* **Horizontal (ထႅဝ်ၼွၼ်း):** ဝၢင်းတေႃႇၵၼ် ၵႂႃႇၽၢႆႇၶႂႃ။
* **မိူၼ်သင်:** မိူၼ်တင်း `Column` ဢမ်ႇၼၼ် `Row` ၼႂ်း Flutter ၼၼ်ႉယဝ်ႉၶႃႈ (ၼႆႉပဵၼ်လွင်ႈၵွင်ႉၵၢႆႇမၼ်းၶႃႈဢေႃႈ)။



### 3. WrapPanel (ၵၢၼ်ဝၢင်းၸွတ်ႉၼႃႈၸေႃး)
မိူၼ်တင်း StackPanel ၵူၺ်းၵႃႈ သင်ပိုၼ်ႉတီႈၼႃႈၸေႃး "တဵမ်" ယဝ်ႉၼႆ မၼ်းတေ **"ၵျွၵ်းလူင်းထႅဝ်မႂ်ႇ"** ပၼ်ႁင်းၵူၺ်းၶႃႈ။
* **တူဝ်ယင်ႇ:** ၵၢၼ်ၼႄၶႅပ်းႁၢင်ႈၼႂ်း Gallery၊ မိူဝ်ႈႁဝ်းယိုတ်း Window ႁႂ်ႈၵႅပ်ႈၼႆ ၶႅပ်းႁၢင်ႈတေလူင်းၵႂႃႇယူႇထႅဝ်တႂ်ႈႁင်းၵူၺ်းၶႃႈ။



---

### ႁဵတ်းသင် WPF Layout ၸင်ႇလီလိူဝ် WinForms?
* **Flexible:** ဢမ်ႇဝႃႈ User တေၸႂ်ႉၼႃႈၸေႃးလဵၵ်ႉ ဢမ်ႇၼၼ် ယႂ်ႇ Layout ႁဝ်းတေ "တူင်ႉၼိုင်" (Responsive) ၸွမ်းလႆႈလီငၢႆႈၶႃႈ။
* **No Pixel Counting:** ႁဝ်းဢမ်ႇလူဝ်ႇမႃးၼပ်ႉ Pixel ဝႃႈ တုမ်ႇၼႆႉယူႇတီႈ 100, 150 ႁႃႉ၊ ႁဝ်းၵူၺ်းလၢတ်ႈဝႃႈ "ဝၢင်းဝႆႉၼႂ်း Grid ထႅဝ် 1" ၵေႃႈယဝ်ႉယဝ်ႉၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Panel:** Control ဢၼ်မီးၼႃႈၵၢၼ် "ႁပ်ႉသိမ်း" လႄႈ "ၸတ်းဝၢင်း" Control တၢင်ႇဢၼ်ဝႆႉၼႂ်းတူဝ်။
* **Alignment:** ၵၢၼ်ၸတ်းႁႂ်ႈယူႇ ၵၢင်, သၢႆႉ, ဢမ်ႇၼၼ် ၶႂ်ႈ။
* **Star Sizing (*):** ၵၢၼ်မၵ်းမၼ်ႈသႅၼ်း ႁႂ်ႈၵိၼ်ပိုၼ်ႉတီႈ "ဢၼ်ၵိုတ်းဝႆႉတင်းမူတ်း"။
* **Orientation:** တၢင်းႁၼ်ၶွင်ၵၢၼ်ဝၢင်း (Horizontal ဢမ်ႇၼၼ် Vertical)။
