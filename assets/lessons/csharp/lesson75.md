# Lesson 75: "Array Slicing and Indices"

ၼႂ်း C# 8.0 ၶိုၼ်ႈၼိူဝ် မၼ်းမီး Operator မႂ်ႇသွင်ဢၼ် ဢၼ်ႁဵတ်းႁႂ်ႈၵၢၼ်ၸတ်းၵၢၼ် Array လီၵျေႃႇလိူဝ်ၵဝ်ႇၶႃႈ။

### 1. The Index Operator (`^`) - တေႇၼပ်ႉတီႈၽၢႆႇလင်
မၢႆ **`^`** (Hat) ၼႆႉ ၸွႆးႁႂ်ႈႁဝ်းၼပ်ႉ Index တေႇတီႈၽၢႆႇလင် မႃးတင်းၼႃႈၶႃႈ။ 

* **`^1`** : ပဵၼ်တူဝ်သုတ်းပိူၼ်ႈ (မိူၼ်တင်း `Length - 1`)
* **`^2`** : ပဵၼ်တူဝ်ထီႉသွင် လုၵ်ႉၽၢႆႇလင်ၼပ်ႉမႃး



```csharp
string[] names = { "Sai Mao", "Nang Kham", "Sai Tun", "Nang Aye" };

string last = names[^1];   // ၽွၼ်းလႆႈ: Nang Aye
string secondLast = names[^2]; // ၽွၼ်းလႆႈ: Sai Tun
```

### 2. The Range Operator (`..`) - ၵၢၼ်တတ်း (Slicing)
မၢႆ **`..`** (Range) ၼႆႉ ၸႂ်ႉတႃႇ "တတ်းဢဝ်" (Slice) တွၼ်ႈၼိုင်ႈ ၼႂ်း Array သေ သၢင်ႈပဵၼ် Array ဢၼ်မႂ်ႇၶႃႈ။

* **ပိူင်သၢင်ႈ:** `[start..end]`
* *မၢႆတွင်း:* တူဝ် `start` ၼၼ်ႉမၼ်းတေဢဝ်ပႃး၊ ၵူၺ်းၵႃႈ တူဝ် `end` ၼၼ်ႉ မၼ်းတေ **ဢမ်ႇဢဝ်ပႃး** ၶႃႈ (Exclusive)။

```csharp
string[] students = { "A", "B", "C", "D", "E" };

// တတ်းဢဝ် Index 1, 2, 3 (ဢမ်ႇဢဝ် 4)
var middle = students[1..4]; // ၽွၼ်းလႆႈ: { "B", "C", "D" }

// တတ်းဢဝ် 3 တူဝ်သုတ်းပၢႆ
var lastThree = students[^3..^0]; // ၽွၼ်းလႆႈ: { "C", "D", "E" }
```

### 3. Shortcut လၢႆးတႅမ်ႈပွတ်း
* **`[..3]`** : တႄႇတီႈ 0 တေႃႇပေႃးထိုင် Index 3 (ဢမ်ႇဢဝ် 3)။
* **`[2..]`** : တႄႇတီႈ Index 2 တေႃႇပေႃးထိုင်သုတ်းပၢႆ။
* **`[..]`** : ဢဝ်တင်းမူတ်း (Copy Array)။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Index Operator (^):** မၢႆၼပ်ႉ ဢၼ်တႄႇၼပ်ႉၽၢႆႇလင် (From End)။
* **Range Operator (..):** မၢႆဢၼ်ၸႂ်ႉမၵ်းမၼ်ႈ တွၼ်ႈတႄႇ လႄႈ တွၼ်ႈသုတ်း ၶွင်ၶေႃႈမုၼ်း။
* **Slicing:** ၵၢၼ်တတ်းဢဝ်ၶေႃႈမုၼ်း တွၼ်ႈၼိုင်ႈ ၼႂ်း Collection။
* **Exclusive End:** တူဝ်သုတ်းပၢႆၶွင် Range ဢၼ်ဢမ်ႇလႆႈဢဝ်မႃးၼပ်ႉပႃး။
