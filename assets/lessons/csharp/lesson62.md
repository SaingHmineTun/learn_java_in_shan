# Lesson 62: "Performance: Choosing the Right Loop for the Task"

ၼႂ်း **Lesson 62** ၼႆႉ ႁဝ်းတေမႃးဢုပ်ႇၵၼ်လွင်ႈ **Performance** (ၼမ်ႉၵတ်ႉၵၢၼ်ႁဵတ်းၵၢၼ်) ဢေႃႈ။ 
မိူဝ်ႈႁဝ်းမီး Loop လၢႆပိူင်၊ ႁဝ်းတေလိူၵ်ႈၸႂ်ႉဢၼ်လႂ် ႁႂ်ႈမၼ်းသႅၼ်ႈသႂ် လႄႈ ႁႂ်ႈ App ႁဝ်းလႅၼ်ႈၽႂ်းသေပိူၼ်ႈ?

### 1. လွင်ႈလိူၵ်ႈၸႂ်ႉ (Which Loop to Use?)

| Loop Type | မိူဝ်ႈလႂ်လူဝ်ႇၸႂ်ႉ?                                                 | တၢင်းလီ (Pros) |
| :--- |:--------------------------------------------------------------------| :--- |
| **for** | မိူဝ်ႈႁူႉႁူဝ်ၼပ်ႉသေ ၶႂ်ႈမႄး (Modify) ၶေႃႈမုၼ်းၸွမ်းမၢႆၼပ်ႉ (Index)။ | ၽႂ်းသေပိူၼ်ႈ လႄႈ ၵုမ်းထိင်းမၢႆၼပ်ႉလႆႈငၢႆႈ။ |
| **foreach** | မိူဝ်ႈၶႂ်ႈ **ဢၢၼ်ႇ/လူ** ၶေႃႈမုၼ်းတင်းမူတ်း ၼႂ်း Array ဢမ်ႇၼၼ် List။ | ဢၢၼ်ႇလူငၢႆႈ၊ ပွတ်းသေ သႅၼ်ႈသႂ် (Clean)။ |
| **while** | မိူဝ်ႈဢမ်ႇႁူႉႁူဝ်ၼပ်ႉသေ ၶႂ်ႈပႂ်ႉတေႃႇ "ၶေႃႈထၢမ်" မႅၼ်ႈ။              | လႅပ်ႈတွၼ်ႈတႃႇပႂ်ႉ User Input ဢမ်ႇၼၼ် Network။ |
| **do-while** | မိူဝ်ႈၶႂ်ႈႁႂ်ႈ Code ႁဵတ်းၵၢၼ် တီႈဢေႇသုတ်ႉ ၼိုင်ႈပွၵ်ႈ။              | လီတွၼ်ႈတႃႇ Menu ဢမ်ႇၼၼ် Password Check။ |


### 2. လွင်ႈလႅတ်းၽႂ်း (Speed)
ၼႂ်း C# မိူဝ်ႈလဵဝ်...
* **`for` loop** ၼႆႉ ၵႆႉလႅၼ်ႈၽႂ်းလိူဝ် `foreach` ဢိတ်းၼိုင်ႈ ယွၼ်ႉမၼ်းဢမ်ႇလူဝ်ႇသၢင်ႈ Object တႃႇပႆၶၢမ်ႈ (Enumerator)။
* ၵူၺ်းၵႃႈ သင်ၶေႃႈမုၼ်းဢမ်ႇပေႃးၼမ် (မိူၼ်ၼင်ႇ 1,000 လူင်းတႂ်ႈ)၊ ႁဝ်းတေဢမ်ႇႁၼ် လွင်ႈပႅၵ်ႇၵၼ်သင်ၶႃႈ။



### 3. လွင်ႈလူဝ်ႇၾၢင်ႉ (Performance Pitfalls)
* **Nested Loops:** ပေႃးသွၼ်ႉၵၼ် 3-4 ၸၼ်ႉ၊ App တေ "ႁဵတ်းၵၢၼ်ထိူင်း" ၵႂႃႇၵမ်းလဵဝ်။ ႁဝ်းႁွင်ႉဝႃႈ $O(n^2)$ ဢမ်ႇၼၼ် $O(n^3)$ Complexity ဢေႃႈ။
* **Infinite Loops:** ၼႆႉပဵၼ်တူဝ်ယႃႉ Performance ဢၼ်ႁၢႆႉသေပိူၼ်ႈ ယွၼ်ႉမၼ်းဢဝ် CPU ၵႂႃႇၸႂ်ႉ 100% သေႁဵတ်းႁႂ်ႈ "ၶွမ်းၶမ်"။
* **Break Early:** သင်ႁႃၶေႃႈမုၼ်းၺႃးယဝ်ႉ ႁႂ်ႈၸႂ်ႉ `break` ၵမ်းလဵဝ်။ ယႃႇပေႁႂ်ႈ Computer ပၼ်ႇတူင်ႇလူၺ်ႈ ဢမ်ႇမီးတင်းယိူင်းဢၢၼ်းသင် ၶႃႈ။

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Performance:** ၼမ်ႉၵတ်ႉ လႄႈ လွင်ႈလႅတ်းၽႂ်း လွင်ႈၵၢၼ်ႁဵတ်းၵၢၼ်ၶွင် App။
* **Readability:** ၼမ်ႉၵတ်ႉၶွင် Code ဢၼ်ႁဵတ်းႁႂ်ႈ Developer ၵေႃႉတူၺ်းၸွမ်းၼၼ်ႉ ပွင်ႇၸႂ်လႆႈငၢႆႈ။
* **Complexity:** လွင်ႈယုင်ႈယၢင်ႈၶွင် Logic ဢၼ်လၢတ်ႈၼႄဝႃႈ Computer တေလႆႈႁဵတ်းၵၢၼ်ၼမ်ၵႃႈႁိုဝ်။
* **Optimization:** ၵၢၼ်မႄး Code ႁႂ်ႈမၼ်းႁဵတ်းၵၢၼ်လႆႈလီလိူဝ်ၵဝ်ႇ။
