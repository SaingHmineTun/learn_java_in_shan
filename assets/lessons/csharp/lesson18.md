# Lesson 18: "The Char and String Types" (ပိူင်ၶေႃႈမုၼ်း တူဝ်လိၵ်ႈ လႄႈ ၸုမ်းတူဝ်လိၵ်ႈ)

ၼႂ်း C# မၼ်းမီး လၢႆးသိမ်းတူဝ်လိၵ်ႈ 2 ပိူင်။ 2 ပိူင်ၼႆႉ တေပႅၵ်ႇၵၼ်တီႈ "တၢင်းၼမ်" ၶေႃႈမုၼ်းၶႃႈ။

### 1. The Char Type (`char`)
**char** (Character) ၼႆႉၸႂ်ႉတႃႇသိမ်း **တူဝ်လိၵ်ႈတူဝ်လဵဝ်** ၵူၺ်း။
* **မၢႆတွင်း:** လူဝ်ႇၸႂ်ႉ **Single Quote (`'`)** တႃႇႁူမ်ႈမၼ်းဝႆႉ။
* **Unicode:** `char` ၼိုင်ႈတူဝ် မီးတၢင်းယႂ်ႇ 2 Bytes (16-bit) ဢၼ်ၸွႆးႁႂ်ႈမၼ်းသိမ်းတူဝ်လိၵ်ႈ Unicode (မိူၼ်ၼင်ႇ လိၵ်ႈတႆး 'မ') လႆႈၼိုင်ႈတူဝ်ဢေႃႈ။

```csharp
char grade = 'A';
char shanLetter = 'ၵ'; // Unicode Character
```

### 2. The String Type (`string`)
**string** ၼႆႉပဵၼ် **ၸုမ်းတူဝ်လိၵ်ႈ** ဢၼ်မီးလၢႆလၢႆတူဝ်သိုပ်ႇၵၼ်။ မၼ်းပဵၼ် "Reference Type" (ဢၼ်ႁဝ်းတေလဵပ်ႈႁဵၼ်းၼႂ်း Lesson 22)။
* **မၢႆတွင်း:** လူဝ်ႇၸႂ်ႉ **Double Quotes (`"`)** တႃႇႁူမ်ႈမၼ်းဝႆႉ။
* **Length:** ႁဝ်းၸၢင်ႈတူၺ်းလႆႈဝႃႈ ၼႂ်း string ၼၼ်ႉမီးလိၵ်ႈလၢႆတူဝ် ၸႂ်ႉ `.Length`။

```csharp
string schoolName = "Mao App Academy";
string greeting = "မႂ်ႇသုင်ၶႃႈ";

Console.WriteLine(greeting.Length); // တေၼႄတၢင်းၼမ်တူဝ်လိၵ်ႈ
```



### 3. Escape Sequences (တူဝ်လိၵ်ႈလၢၵ်ႇလၢႆး)
မၢင်ပွၵ်ႈႁဝ်းလူဝ်ႇသႂ်ႇ "တူဝ်လိၵ်ႈဢၼ်တႅမ်ႈဢမ်ႇလႆႈ" မိူၼ်ၼင်ႇ ၵၢၼ်လူင်းထႅဝ်မႂ်ႇ ဢမ်ႇၼၼ် မၢႆၵမ် (`"`) ၼႂ်း string။ သင်ၼၼ် ႁဝ်းတေလႆႈၸႂ်ႉ Backslash (`\`) ၸွႆႈၶႃႈ။

| Escape Sequence | Meaning |
| :--- | :--- |
| `\n` | New Line (လူင်းထႅဝ်မႂ်ႇ) |
| `\t` | Tab (ႁၢင်ႇၵၼ်ၼိုင်ႈလွၵ်း) |
| `\"` | Double Quote (သႂ်ႇမၢႆၵမ် ၼႂ်း string) |
| `\\` | Backslash (သႂ်ႇမၢႆၶတ်း) |

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Char:** ပိူင်ၶေႃႈမုၼ်း တႃႇသိမ်းတူဝ်လိၵ်ႈတူဝ်လဵဝ် (Single Character)။
* **String:** ပိူင်ၶေႃႈမုၼ်း တႃႇသိမ်းၸုမ်းတူဝ်လိၵ်ႈ (Sequence of characters)။
* **Unicode:** ပိူင်မၢႆတွင်း တူဝ်လိၵ်ႈလၢႆၽႃႇသႃႇ ဢၼ် C# ၸႂ်ႉဝႆႉပဵၼ် Standard (UTF-16)။
* **Escape Sequence:** တူဝ်လိၵ်ႈ ဢၼ်တႄႇတင်း `\` တႃႇသင်ႇၵၢၼ်လၢၵ်ႇလၢႆး ၼႂ်း string။
* **Literal:** ၵႃႈၶၼ်ဢၼ်ႁဝ်းတႅမ်ႈသႂ်ႇၼႂ်း Code တႅတ်ႉတႅတ်ႉတေႃးတေႃး (တူဝ်ယင်ႇ: `"Hello"` ၼႆႉပဵၼ် string literal)။
