# Lesson 197: "Rethrowing Exceptions: 'throw' vs. 'throw ex'"

ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းမီး Method သွင်ဢၼ်ႁွင်ႉသိုပ်ႇၵၼ်။ 
သင်ပၼ်ႁႃပဵၼ်ၼႂ်း Method ၽၢႆႇတႂ်ႈ... ႁဝ်းၶႂ်ႈႁႂ်ႈမၼ်း "တူၵ်းမႃး" ၸူး Method ၽၢႆႇၼိူဝ် ႁႂ်ႈႁူႉလွင်ႈတၢင်းတင်းမူတ်းၶႃႈ။

### 1. ၵၢၼ်ၸႂ်ႉ `throw;` (လၢႆးဢၼ်လီသုတ်း)
မိူဝ်ႈႁဝ်းတႅမ်ႈ `throw;` လၢႆလၢႆ... C# တေသူင်ႇပၼ်ႁႃၼၼ်ႉသိုပ်ႇၵႂႃႇ လူၺ်ႈဢမ်ႇယႃႉပႅတ်ႈ "ပိုၼ်း" (History) ၶွင်မၼ်းၶႃႈ။ 
ႁဝ်းတေတိုၵ်ႉႁူႉယူႇဝႃႈ ပၼ်ႁႃၼၼ်ႉ "တႄႇပဵၼ်" တီႈလႂ်ၼႆၶႃႈ။

* **StackTrace:** တိုၵ်ႉမီးၶေႃႈမုၼ်း လၢတ်ႈၼႄလွင်ႈတီႈတႄႇပဵၼ် ဢွင်ႈတီႈၽိတ်းတႄႉတႄႉ (Original Location)။

```csharp
try {
    // ၸၢမ်းႁဵတ်းၵၢၼ်
} catch (Exception ex) {
    // 1. မၢႆဝႆႉၼႂ်း Log
    Console.WriteLine("Logging exception...");
    
    // 2. သူင်ႇသိုပ်ႇၵႂႃႇ (Rethrow)
    throw; // ✅ လၢႆးၼႆႉလီသုတ်း ယွၼ်ႉမၼ်းၵဵပ်း StackTrace ဝႆႉတင်းမူတ်း
}
```

### 2. ၵၢၼ်ၸႂ်ႉ `throw ex;` (လၢႆးဢၼ်ႁဝ်းဢမ်ႇထုၵ်ႇလီၸႂ်ႉ)
သင်ႁဝ်းသူင်ႇ Variable `ex` ၵႂႃႇၸွမ်း... C# တေဝူၼ်ႉဝႃႈ "ၼႆႉပဵၼ်ပၼ်ႁႃမႂ်ႇ" သေ မၼ်းတေ "လူတ်ႇပႅတ်ႈ" ပိုၼ်းၵဝ်ႇတင်းမူတ်းၶႃႈ။

* **StackTrace:** မၼ်းတေမၢႆတွင်းဝႃႈ ပၼ်ႁႃ "တႄႇပဵၼ်" တီႈထႅဝ်လိၵ်ႈဢၼ်မီး `throw ex;` ၼၼ်ႉၵူၺ်း၊ သေႁဵတ်းႁႂ်ႈႁဝ်း ႁႃတီႈၽိတ်းတႄႉတႄႉ လႆႈယၢပ်ႇၶႃႈ။



### 3. ပၵ်းၵၢၼ်တႅၵ်ႈၵၼ်: `throw` vs `throw ex`

| လၢႆးတႅမ်ႈ | ၶေႃႈမုၼ်း StackTrace | လွင်ႈပဵၼ်တႄႉ |
| :--- | :--- | :--- |
| **`throw;`** | တိုၵ်ႉမီး ပိုၼ်းတီႈၽိတ်းတႄႉတႄႉ (Preserves History) | **လီသုတ်း (Recommended)** |
| **`throw ex;`** | ပိုၼ်းတီႈၽိတ်းတႄႉတႄႉ ထုၵ်ႇလူတ်ႇပႅတ်ႈ (Resets History) | **ယႃႇပေၸႂ်ႉ (Avoid)** |

### 4. ႁဵတ်းသင်ႁဝ်းၸင်ႇလူဝ်ႇ Rethrow?
1.  **Partial Handling:** ႁဝ်းၶႂ်ႈႁဵတ်းၵၢၼ် "ၵမ်းၼိုင်ႈ" (မိူၼ်ၼင်ႇ ပိၵ်ႉ Connection ဢမ်ႇၼၼ် Log) သေ ပွႆႇႁႂ်ႈ Method ၽၢႆႇၼိူဝ် တႅပ်းတတ်းၵၢၼ်သိုပ်ႇၵႂႃႇ။
2.  **Centralized Logging:** တႃႇႁႂ်ႈမၼ်းမၢႆဝႆႉၵူႈတီႈ ဢၼ်ပၼ်ႁႃလႅၼ်ႈၽၢၼ်ႇၵႂႃႇ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Rethrow:** ၵၢၼ်သူင်ႇပၼ်ႁႃ ဢၼ်ႁပ်ႉလႆႈယဝ်ႉၼၼ်ႉ သိုပ်ႇၵႂႃႇထႅင်ႈၸၼ်ႉၼိုင်ႈ။
* **Trace Preservation:** ၵၢၼ်ၵဵပ်းသိမ်း "သဵၼ်ႈတၢင်း" ဢၼ်ပၼ်ႁႃလႅၼ်ႈၽၢၼ်ႇမႃး။
* **Log:** ပပ်ႉမၢႆတွင်း ၵၢၼ်ႁဵတ်းၵၢၼ်ၶွင် Program။
* **Call Stack:** ၸၼ်ႉၵၢၼ်ႁွင်ႉ Method ဢၼ်သိမ်းဝႆႉဝႃႈ တူဝ်လႂ်ႁွင်ႉတူဝ်လႂ်။
