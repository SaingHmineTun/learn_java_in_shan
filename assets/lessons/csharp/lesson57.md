# Lesson 57: "Nested Loops: Working with Grids and Tables"

ၼႂ်း **Lesson 57** ၼႆႉ ႁဝ်းတေမႃးလဵပ်ႈႁဵၼ်းလွင်ႈ **"Loop သွၼ်ႉ Loop"** ဢေႃႈ။ 
မၼ်းပဵၼ်ၵၢၼ်ဢဝ် Loop ဢၼ်ၼိုင်ႈ ၵႂႃႇဝႆႉၼႂ်း Loop ထႅင်ႈဢၼ်ၼိုင်ႈ တႃႇႁဵတ်းၵၢၼ်ဢၼ်မီး "သွင်ၼႃႈ" (2D) မိူၼ်ၼင်ႇ Table ဢမ်ႇၼၼ် Grid ၶႃႈ။

### 1. ႁဵတ်းႁိုဝ်ႁဵတ်းၵၢၼ်? (How it Works)
ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းမီး **Outer Loop** (Loop ၽၢႆႇၼွၵ်ႈ) လႄႈ **Inner Loop** (Loop ၽၢႆႇၼႂ်း)။
* ပေႃး Outer Loop ပၼ်ႇတူင်ႇ **ၼိုင်ႈႁွပ်ႈ**...
* Inner Loop တေလႆႈပၼ်ႇတူင်ႇ **တင်းမူတ်း** (တေႃႇပေႃးယဝ်ႉ)။



### 2. တူဝ်ယင်ႇ Code: ႁဵတ်းလွၵ်းသီႇၸဵင်ႇ (Square Grid)
ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းၶႂ်ႈၼႄမၢႆ "*" ႁႂ်ႈပဵၼ် 3 ထႅဝ် (Rows) လႄႈ 3 လႅၼ်း (Columns) ၶႃႈ။

```csharp
// Outer Loop (တႅၼ်းတၢင်း ထႅဝ် - Rows)
for (int i = 1; i <= 3; i++) 
{
    // Inner Loop (တႅၼ်းတၢင်း လႅၼ်း - Columns)
    for (int j = 1; j <= 3; j++) 
    {
        Console.Write("* "); // ၼႄမၢႆ * ဝႆႉၼႂ်းထႅဝ်လဵဝ်ၵၼ်
    }
    
    Console.WriteLine(); // ပေႃး Inner Loop ယဝ်ႉ ႁႂ်ႈမၼ်းလူင်းထႅဝ်မႂ်ႇ
}
```

**ၽွၼ်းလႆႈ (Output):**
```text
* * * * * * * * * ```

### 3. မိူဝ်ႈလႂ်လူဝ်ႇၸႂ်ႉ?
* **Game Grids:** တႃႇၼႄႁၢင်ႈၽႅၼ်ႇလိၼ် (Map) ၼႂ်းၵဵမ်း။
* **Processing Pixels:** ၵၢၼ်မႄးႁၢင်ႈၽၢင် (Image Processing) ဢၼ်မီးမၢႆသီ (Pixel) လၢႆထႅဝ် လၢႆလႅၼ်း။

### 4. လွင်ႈလူဝ်ႇၾၢင်ႉ (Performance)
သင်ဝႃႈ Outer Loop မီး 1,000 ႁွပ်ႈ လႄႈ Inner Loop မီး 1,000 ႁွပ်ႈ၊ Computer တေလႆႈႁဵတ်းၵၢၼ်တင်းမူတ်း **1,000,000** (ၼိုင်ႈလၢၼ်ႉ) ပွၵ်ႈၶႃႈ! 
ပေႃးသွၼ်ႉၵၼ်ၼမ်ၼႃႇ (မိူၼ်ၼင်ႇ 3-4 ၸၼ်ႉ) မၼ်းတေႁဵတ်းႁႂ်ႈ App ႁဝ်းႁဵတ်းၵၢၼ်ထိူင်း (Slow) ၶႃႈ။

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Nested Loop:** Loop ဢၼ်ယူႇၼႂ်း Loop ထႅင်ႈဢၼ်ၼိုင်ႈ။
* **Outer Loop:** Loop ၽၢႆႇၼွၵ်ႈ ဢၼ်ၵုမ်းထိင်းႁွပ်ႈလူင်။
* **Inner Loop:** Loop ၽၢႆႇၼႂ်း ဢၼ်ႁဵတ်းၵၢၼ်ယဝ်ႉတူဝ်ႈ ၼႂ်းၵူႈႁွပ်ႈၶွင် Outer Loop။
* **Row:** ထႅဝ် (ၼႃႈၼွၼ်း)။
* **Column:** လႅၼ်း (ၼႃႈတင်ႈ)။
* **2D Grid:** ႁၢင်ႈၾၢင်လွၵ်း ဢၼ်မီးတင်းၵႂၢင်ႈ လႄႈ တင်းသုင်။
