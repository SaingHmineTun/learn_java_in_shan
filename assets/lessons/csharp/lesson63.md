# Lesson 63: "Iterating through Strings: Processing Shan Script"

ၼႂ်း C# ၼၼ်ႉ `string` (တူဝ်လိၵ်ႈ) မၼ်းမိူၼ်ၼင်ႇ "သဵၼ်ႈၸိုဝ်ႈ" (Array) ၶွင်တူဝ်လိၵ်ႈဢွၼ်ႇ (char) ဢၼ်မႃးသိုပ်ႇၵၼ်ဝႆႉၶႃႈ။

### 1. ၵၢၼ်ၸႂ်ႉ foreach တႃႇဢၢၼ်ႇလူတူဝ်လိၵ်ႈ
လၢႆးၼႆႉငၢႆႈသေပိူၼ်ႈ တႃႇတူၺ်းဝႃႈ ၼႂ်းလိၵ်ႈၼၼ်ႉ မီးတူဝ်သင်ဝႆႉလၢႆလၢႆၶႃႈ။

```csharp
string text = "တႆး";

foreach (char c in text)
{
    Console.WriteLine("Character: " + c);
}
```

**ၽွၼ်းလႆႈ (Output):**
```text
Character: တ
Character: ႆ
Character: း
```

### 2. ၵၢၼ်ၸႂ်ႉ for loop တႃႇမၵ်းမၼ်ႈမၢႆၼပ်ႉ (Index)
မိူဝ်ႈႁဝ်းၶႂ်ႈႁူႉဝႃႈ တူဝ်လိၵ်ႈၼၼ်ႉယူႇတီႈမၢႆၼပ်ႉၵႃႈႁိုဝ် ႁဝ်းၸႂ်ႉ `for` loop လႆႈၶႃႈ။

```csharp
string city = "မူႇၸေႊ";

for (int i = 0; i < city.Length; i++)
{
    Console.WriteLine($"Position {i}: {city[i]}");
}
```

### 3. လွင်ႈလူဝ်ႇၾၢင်ႉ (Unicode & Shan Script)
လိၵ်ႈတႆးႁဝ်းၼႆႉ မီး "မၢႆသဵင်" လႄႈ "တူဝ်သွၼ်ႉ" (Combining Marks)။
* **Length:** ၼႂ်း C# မၼ်းတေၼပ်ႉ `တ` ပဵၼ် 1 တူဝ်၊ `ႆ` ပဵၼ် 1 တူဝ်။
* **Shan Character:** ဝူၼ်ႉတူၺ်းဝႃႈ တူဝ်လိၵ်ႈ `တႆး` ၼႆႉ ၵူၼ်းႁဝ်းဝႃႈပဵၼ် "ၼိုင်ႈတူဝ်" ၵူၺ်းၵႃႈ Computer တေဝႃႈမီး 3 Characters (တ + ႆ + း) ၶႃႈ။



### 4. တူဝ်ယင်ႇၵၢၼ်ၸႂ်ႉတႄႉ: ၼပ်ႉတူဝ်မႄႈလိၵ်ႈ `ၵ`
ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းၶႂ်ႈၼပ်ႉဝႃႈ ၼႂ်းလိၵ်ႈႁဝ်းမီး `ၵ` လၢႆတူဝ်ၶႃႈ။

```csharp
string message = "ၵႂႃႇၵိၼ်ၶဝ်ႈၵမ်းလဵဝ်";
int count = 0;

foreach (char c in message)
{
    if (c == 'ၵ')
    {
        count++;
    }
}
Console.WriteLine("မီးတူဝ် 'ၵ' တင်းမူတ်း: " + count);
```

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Char:** တူဝ်လိၵ်ႈဢွၼ်ႇၼိုင်ႈတူဝ် (မိူၼ်ၼင်ႇ 'ၵ', 'ၶ')။
* **String:** ၸုမ်းတူဝ်လိၵ်ႈဢွၼ်ႇ ဢၼ်မႃးသိုပ်ႇၵၼ်ပဵၼ်ထႅဝ် (မိူၼ်ၼင်ႇ "မိူင်းတႆး")။
* **Length:** ႁူဝ်ၼပ်ႉ Character တင်းမူတ်း ဢၼ်မီးၼႂ်း String။
* **Index:** မၢႆၼပ်ႉတီႈယူႇၶွင်တူဝ်လိၵ်ႈ (တႄႇၼပ်ႉတီႈ 0 တႃႇသေႇ)။
* **Unicode:** ပိူင်မၵ်းမၼ်ႈမၢႆၼပ်ႉတူဝ်လိၵ်ႈ တႃႇၵူႈၽႃႇသႃႇ ဢၼ်မီးဝႆႉတီႈၼႂ်းလုမ်ႈၾႃႉ။
