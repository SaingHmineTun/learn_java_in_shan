# Lesson 242: "The Task and Task<T> Classes"

ၼႂ်း C# ႁဝ်းဢမ်ႇသူႈၸတ်းၵၢၼ် Threads ႁင်းၵူၺ်းယဝ်ႉၶႃႈ။ ႁဝ်းၸႂ်ႉ Class **`Task`** ဢၼ်ယူႇၼႂ်း Namespace `System.Threading.Tasks` တႅၼ်းၶႃႈ။

### 1. Task ပဵၼ်သင်? (The Promise)

**Task** ပဵၼ် Object ဢၼ်တႅၼ်းထိုင် "ၵၢၼ်ဢၼ်တိုၵ်ႉႁဵတ်းယူႇ" ၶႃႈ။

* မိူဝ်ႈႁဝ်းတႄႇ Task ၼႆ မၼ်းတေသူင်ႇ Object ၼၼ်ႉမႃးပၼ်ႁဝ်းၵမ်းလဵဝ်။
* ႁဝ်းၸၢင်ႈၸႂ်ႉ Object ၼၼ်ႉ တႃႇၵူတ်ႇထတ်းဝႃႈ "ၵၢၼ်ယဝ်ႉယဝ်ႉႁႃႉ?" ဢမ်ႇၼၼ် "မီး Error သင်ႁႃႉ?" ၼႆၶႃႈ။

### 2. Task vs. Task<T>

မၼ်းမီးသွင်ပိူင် ဢၼ်ႁဝ်းတေလႆႈလိူၵ်ႈၸႂ်ႉ ၸွမ်းၼင်ႇၽွၼ်းလႆႈၶႃႈ:

* **`Task`**: ၸႂ်ႉတွၼ်ႈတႃႇၵၢၼ်ဢၼ် **ဢမ်ႇသူင်ႇၵႃႈၶၼ်** ၶိုၼ်း (မိူၼ်ၼင်ႇ `void`)။
    * *တူဝ်ယင်ႇ:* ၵၢၼ်သိမ်း File, ၵၢၼ်ပိုတ်ႇသဵင်။
* **`Task<T>`**: ၸႂ်ႉတွၼ်ႈတႃႇၵၢၼ်ဢၼ် **သူင်ႇၵႃႈၶၼ်** (Result) ၶိုၼ်းမႃး။
    * *တူဝ်ယင်ႇ:* `Task<string>` (ၵၢၼ် Download လိၵ်ႈ), `Task<int>` (ၵၢၼ်ၼပ်ႉသွၼ်ႇမၢႆတူဝ်ၼပ်ႉ)။

### 3. State ၶွင် Task

Task ၼိုင်ႈဢၼ် တေလႅၵ်ႈ လႅၼ်ႈပိၼ်ႇႁၢင်ႈ (Status) ၵႂႃႇၸိူင်ႉၼႆၶႃႈ:

1.  **Running**: တိုၵ်ႉႁဵတ်းၵၢၼ်ယူႇ။
2.  **RanToCompletion**: ယဝ်ႉတူဝ်ႈလီငၢႆႈ။
3.  **Faulted**: ယဝ်ႉတူဝ်ႈလူၺ်ႈၵၢၼ်မီး Exception (Error)။
4.  **Canceled**: ထုၵ်ႇယႃႉပႅတ်ႈ ဢွၼ်တၢင်းတေယဝ်ႉ။

### 4. တူဝ်ယင်ႇ Code ပွတ်းပွတ်း

ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းမီး Method ဢၼ်သူင်ႇ "ၵွင်ႇလိၵ်ႈ" ၵၢၼ် Download ၶဝ်ႈသွႆး မႃးပၼ်ႁဝ်းၶႃႈ။

```csharp
// ၵၢၼ်ဢၼ်တေသူင်ႇ string မႃးပၼ်ၼႂ်းမိူဝ်းၼႃႈ
Task<string> orderTask = DownloadNoodleAsync(); 

// ႁဝ်းၸၢင်ႈႁဵတ်းၵၢၼ်တၢင်ႇပိူင်ၵႂႃႇ...
DoOtherWork();

// မိူဝ်ႈလူဝ်ႇၵႃႈၶၼ်တႄႉတႄႉ ႁဝ်းၸင်ႇၵွႆႈဢဝ် (ၵူၺ်းၵႃႈလၢႆးၼႆႉတေ Block UI)
string result = orderTask.Result; 
```

*(သတိ: ၵၢၼ်ၸႂ်ႉ `.Result` ၵမ်းလဵဝ်ၼၼ်ႉ မၼ်းတေႁဵတ်းႁႂ်ႈ UI Thread ပႂ်ႉ လႄႈ Freeze လႆႈၶႃႈ။ ႁဝ်းတေမႃးႁဵၼ်းလၢႆးၵႄႈၼႂ်းတွၼ်ႈ `await` ၶႃႈ။)*

-----

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Task**: Object ဢၼ်တႅၼ်းတၢင် ၵၢၼ်ဢၼ်တိုၵ်ႉႁဵတ်းယူႇ ၽၢႆႇလင် (Background)။
* **Task<T>**: Task ဢၼ်မီးၵႃႈၶၼ် (Result) သူင်ႇၶိုၼ်းမႃး။
* **Async Operation:** ၵၢၼ်ႁဵတ်းၵၢၼ် ဢၼ်ဢမ်ႇပႂ်ႉၵၼ် ၼႂ်းသဵၼ်ႈတၢင်းလႅၼ်ႈ Code တူဝ်လဵဝ်ၵၼ်။
* **Promise:** ၶေႃႈၵႂၢမ်းၵၢင် ၼႂ်း Programming ဢၼ်ပွင်ႇဝႃႈ "တေသူင်ႇ Data မႃးပၼ်"။
