# Lesson 280: "Understanding HTTP"

ဝူၼ်ႉတူၺ်းဝႃႈ HTTP မိူၼ်တင်း "ၵူၼ်းသူင်ႇလိၵ်ႈ" ၶႃႈ။ မၼ်းမီးလွၵ်းလၢႆး ၵၢၼ်သူင်ႇၶေႃႈၵႂၢမ်းၵႂႃႇမႃး ဢၼ်ၸႅၵ်ႇပဵၼ် 2 ပိူင်လူင် ၼင်ႇတီႈတႂ်ႈၼႆႉၶႃႈ:

### 1. The Request (ၶေႃႈထၢမ်)
မိူဝ်ႈ User တႅမ်ႈ URL သေၼဵၵ်း Enter ၼၼ်ႉ Browser တေသူင်ႇ **Request** ၵႂႃႇၸူး Server။ ၼႂ်း Request ၼၼ်ႉတေပႃး:
* **URL:** တီႈယူႇ Website (မိူၼ်ၼင်ႇ `shan-news.com/today`).
* **Method (Verb):** ၵၢၼ်ဢၼ်ၶႂ်ႈႁႂ်ႈ Server ႁဵတ်း (တူၺ်းတီႈမၢႆ 3).
* **Headers:** ၶေႃႈမုၼ်းထႅမ် (မိူၼ်ၼင်ႇ ၸႂ်ႉ Browser သင်?).

### 2. The Response (ၶေႃႈတွပ်ႇ)
မိူဝ်ႈ Server ႁပ်ႉလႆႈ Request ယဝ်ႉၼၼ်ႉ မၼ်းတေသူင်ႇ **Response** ၶိုၼ်းမႃးၸူး Browser။ ၼႂ်း Response ၼၼ်ႉတေပႃး:
* **Status Code:** မၢႆတူဝ်ဢၼ်လၢတ်ႈဝႃႈ ၵၢၼ်ယဝ်ႉတူဝ်ႈလီႁႃႉ? (မိူၼ်ၼင်ႇ **200 OK** ဢမ်ႇၼၼ် **404 Not Found**).
* **Body:** ၶေႃႈမုၼ်းတႄႉတႄႉ (မိူၼ်ၼင်ႇ HTML, CSS ဢမ်ႇၼၼ် JSON Data).



---

### 3. HTTP Verbs (ၶေႃႈသင်ႇၵၢၼ်)
တွၼ်ႈတႃႇႁႂ်ႈ Server ႁူႉဝႃႈႁဝ်းၶႂ်ႈႁဵတ်းသင်တႄႉတႄႉၼၼ်ႉ ႁဝ်းတေလႆႈၸႂ်ႉ "ၶေႃႈၵႂၢမ်းၵမ်" (Verbs) ၸိူင်ႉၼႆၶႃႈ:

| Verb | ၼႃႈၵၢၼ် (Action) | တူဝ်ယင်ႇ (Example)                          |
| :--- | :--- |:--------------------------------------------|
| **GET** | **လူ / ဢဝ်မႃး** | လူတူၺ်းသဵၼ်ႈမၢႆပွႆးတႆးတင်းမူတ်း             |
| **POST** | **သၢင်ႈ / သူင်ႇမႂ်ႇ** | တႅမ်ႈသဵၼ်ႈမၢႆပွႆးမႂ်ႇၶဝ်ႈၵႂႃႇ (Submit Form) |
| **PUT** | **မႄးတင်းမူတ်း** | လႅၵ်ႈပိၼ်ႇၶေႃႈမုၼ်းပွႆးၼၼ်ႉတင်းမူတ်း        |
| **DELETE** | **ယႃႉပႅတ်ႈ** | ယႃႉသဵၼ်ႈမၢႆပွႆးဢၼ်ယဝ်ႉၵႂႃႇယဝ်ႉ              |



---

### 4. Status Codes (မၢႆတူဝ်ပိုၼ်တ်း)
မိူဝ်ႈ Server တွပ်ႇမႃးၼၼ်ႉ မၼ်းတေပႃးမၢႆတူဝ် 3 တူဝ် တႃႇလၢတ်ႈငဝ်းလၢႆးၶႃႈ:
* **200 OK:** ၵၢၼ်ယဝ်ႉတူဝ်ႈလီ။
* **201 Created:** သိမ်းၶေႃႈမုၼ်းမႂ်ႇယဝ်ႉတူဝ်ႈလီ (ၸႂ်ႉၸွမ်း POST).
* **400 Bad Request:** Browser သူင်ႇၶေႃႈမုၼ်းမႃးၽိတ်းပိူင်။
* **404 Not Found:** ႁႃၼႃႈ Website ဢၼ်ယွၼ်းမႃးၼၼ်ႉဢမ်ႇႁၼ်။
* **500 Internal Server Error:** Server မီးပၼ်ႁႃ (Code ႁဝ်း Bug).

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Stateless:** HTTP ဢမ်ႇမၢႆတွင်းဝႃႈ ၽႂ်ယွၼ်းသင်ဝႆႉမိူဝ်ႈၵႆႈ (Request ၵူႈဢၼ် ပဵၼ်ဢၼ်မႂ်ႇတႃႇသေႇ).
* **Payload:** ၶေႃႈမုၼ်း (Data) ဢၼ်သူင်ႇပႃးၵႂႃႇၼႂ်း Request Body (ၵႆႉၸႂ်ႉၸွမ်း POST).
* **Endpoint:** URL တႅတ်ႈတေႃး ဢၼ်ႁဝ်းၵွင်ႉၸူး တႃႇႁဵတ်းၵၢၼ် (မိူၼ်ၼင်ႇ `/api/students`).
* **Protocol:** ပၵ်းၵၢၼ်ဢၼ်မၵ်းမၼ်ႈဝႃႈ ၵၢၼ်သူင်ႇၶေႃႈမုၼ်းတေလႆႈႁဵတ်းၸိူင်ႉႁိုဝ်။
