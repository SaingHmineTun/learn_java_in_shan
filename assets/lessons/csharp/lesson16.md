# Lesson 16: "Floating-Point Types" (ပိူင်ၶေႃႈမုၼ်း မၢႆၼပ်ႉမီးၸုၵ်း)

မိူဝ်ႈႁဝ်းတေသိမ်းၶေႃႈမုၼ်း မိူၼ်ၼင်ႇ ရႃႇၶႃႇၶူဝ်း (Price), တၢင်းႁူႉၼမ်ႉၼၵ်း (Weight), ဢမ်ႇၼၼ် မၢႆၼပ်ႉပၢႆးသၢႆႊ (Scientific Data) ဢၼ်မီးၸုၵ်းၼၼ်ႉ ႁဝ်းလူဝ်ႇၸႂ်ႉ Floating-point types ဢေႃႈ။

### 1. ပိူင်လူင် 3 သႅၼ်း (Floating-Point Comparison)

| Type | Size | Precision (Digits) | Suffix (တူဝ်သိုပ်ႇလင်) | မိူဝ်ႈလႂ်လူဝ်ႇၸႂ်ႉ |
| :--- | :--- | :--- | :--- | :--- |
| **float** | 4 Bytes | ~6-9 digits | `f` | တႃႇ Graphics လႄႈ Games (Unity) |
| **double** | 8 Bytes | ~15-17 digits | `d` (Optional) | **(Standard)** တႃႇ Math လႄႈ သိမ်းၵႃႈၶၼ်ၵူႈလွင်ႈ |
| **decimal** | 16 Bytes | ~28-29 digits | `m` | **တႃႇလွင်ႈငိုၼ်းတွင်း (Financial)** |



### 2. လၢႆးတႅမ်ႈ Code လႄႈ Suffix
ၼႂ်း C# ၼၼ်ႉ ပေႃးႁဝ်းတႅမ်ႈမၢႆၼပ်ႉမီးၸုၵ်း (တူဝ်ယင်ႇ: `10.5`) မၼ်းတေႁၼ်ပဵၼ် `double` ႁင်းမၼ်း။ ပေႃးႁဝ်းၶႂ်ႈၸႂ်ႉ `float` ဢမ်ႇၼၼ် `decimal` ႁဝ်းတေလႆႈသႂ်ႇ တူဝ်လိၵ်ႈလင်မၼ်း (Suffix) ဢေႃႈ။

```csharp
float distance = 10.5f;     // လူဝ်ႇသႂ်ႇ f
double pi = 3.14159;        // ဢမ်ႇသႂ်ႇၵေႃႈလႆႈ (Default)
decimal price = 99.99m;     // လူဝ်ႇသႂ်ႇ m (Money)
```

### 3. Precision (လွင်ႈတႅတ်ႉတေႃး)
* **float/double:** ပဵၼ် Binary floating-point။ မၼ်းႁဵတ်းၵၢၼ်ၽႂ်း ၵူၺ်းၵႃႈ မၢင်ပွၵ်ႈမၼ်းမီး Error ဢေတ်းလဵဝ် ၼႂ်းၵၢၼ်ၼပ်ႉသၢင်ႈ (မိူၼ်ၼင်ႇ `0.1 + 0.2` တေဢွၵ်ႇမႃး `0.30000000000000004`)။
* **decimal:** ပဵၼ် Decimal floating-point။ မၼ်းတႅတ်ႉတေႃးၼႃႇ (High Precision)၊ ၵူၺ်းၵႃႈ မၼ်းႁဵတ်းၵၢၼ်ဢိူၵ်ႇ (Slow) သေပိူၼ်ႈ။ လႅပ်ႈတႃႇၵၢၼ်ငိုၼ်းတွင်း (Banking) တႄႉတႄႉ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Precision:** လွင်ႈတႅတ်ႉတေႃး ၶွင်တူဝ်လိၵ်ႈပၢႆၸုၵ်း (ဝႃႈမၼ်းမၢႆတွင်းလႆႈလၢႆတူဝ်)။
* **Suffix:** တူဝ်လိၵ်ႈဢၼ်သႂ်ႇဝႆႉ ၽၢႆႇလင်မၢႆၼပ်ႉ တႃႇလၢတ်ႈၼႄ Compiler ဝႃႈပဵၼ် Type သင် (f, d, m)။
* **Scientific Notation:** လၢႆးတႅမ်ႈမၢႆၼပ်ႉယႂ်ႇယႂ်ႇ (တူဝ်ယင်ႇ: `1.2e-3` ၼႂ်း C#)။
* **Rounding Error:** ၶေႃႈၽိတ်းပိူင်ႇဢေတ်းလဵဝ် ဢၼ်ပဵၼ်မိူဝ်ႈၶွမ်းပိဝ်ႇတႃႇ ဢမ်ႇၸၢင်ႈသိမ်းမၢႆၼပ်ႉၸုၵ်း ႁႂ်ႈတဵမ်ထူၼ်ႈ 100% လႆႈ။
