# Lesson 34: "Advanced Output" (ၵၢၼ်ၸတ်းႁၢင်ႈ လႄႈ ၵၢၼ်တႅၼ်းထႅဝ်)

မိူဝ်ႈႁဝ်းၸႂ်ႉ `Console.WriteLine` ႁဝ်းၸၢင်ႈၸႂ်ႉ **Format Specifiers** ၼႂ်း String Interpolation (`$`) 
တႃႇၵုမ်းထိင်း "တၢင်းႁၢင်ႈ" ၶေႃႈမုၼ်းႁဝ်းၶႃႈ။

### 1. Alignment (ၵၢၼ်ၸတ်းထႅဝ် သၢႆႉ/ၶႂႃ)
ႁဝ်းၸၢင်ႈမၵ်းမၼ်ႈ "တၢင်းၵႂၢင်ႈ" (Width) ၶွင်တူဝ်လိၵ်ႈ ၸႂ်ႉမၢႆ **Comma (,)** ၼႂ်း `{ }` ဢေႃႈ။
* **Positive Number (10):** ၸတ်းဝႆႉၽၢႆႇ **"ၶႂႃ"** (Right Align)။
* **Negative Number (-10):** ၸတ်းဝႆႉၽၢႆႇ **"သၢႆႉ"** (Left Align)။

```csharp
string name = "Sai Mao";
int score = 95;

// ၸတ်းၸိုဝ်ႈဝႆႉသၢႆႉ (20 တူဝ်) လႄႈ ၸတ်း Score ဝႆႉၶႂႃ (5 တူဝ်)
Console.WriteLine($"|{name,-20}|{score,5}|");
```


### 2. Number Formatting (ၵၢၼ်ၸတ်းပိူင်မၢႆၼပ်ႉ)
ႁဝ်းၸၢင်ႈၸႂ်ႉ **Colon (:)** တႃႇမၵ်းမၼ်ႈဝႃႈ ၶႂ်ႈႁႂ်ႈမၢႆၼပ်ႉၼၼ်ႉ ဢွၵ်ႇမႃးၼင်ႇႁိုဝ်ၶႃႈ။

| Format | Name | Example (`123.456`) | Result |
| :--- | :--- | :--- | :--- |
| **`:C`** | **Currency** | `{123.45:C}` | $123.45 (ၸွမ်းၼင်ႇ Region ၶွမ်း) |
| **`:N2`** | **Number** | `{123.456:N2}` | 123.46 (မီး Comma လႄႈ ၸုၵ်း 2 တူဝ်) |
| **`:P0`** | **Percent** | `{0.75:P0}` | 75% |
| **`:000`** | **Padding** | `{5:000}` | 005 (သႂ်ႇ 0 ၽၢႆႇၼႃႈႁႂ်ႈတဵမ်) |

### 3. တူဝ်ယင်ႇ Code တႃႇႁဵတ်း Table
```csharp
Console.WriteLine($"{"Name",-15} {"Price",10}");
Console.WriteLine(new string('-', 26));
Console.WriteLine($"{"Shan Noodles",-15} {2500,10:N0} MMK");
Console.WriteLine($"{"Papaya Salad",-15} {1500,10:N0} MMK");
```

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Formatting:** ၵၢၼ်ၸတ်း "တၢင်းႁၢင်ႈ" ၶေႃႈမုၼ်း ႁႂ်ႈဢၢၼ်ႇငၢႆႈ လႄႈ ႁၢင်ႈလီ။
* **Alignment:** ၵၢၼ်တႅၼ်းတူဝ်လိၵ်ႈ ႁႂ်ႈၸပ်းၽၢႆႇသၢႆႉ ဢမ်ႇၼၼ် ၽၢႆႇၶႂႃ။
* **Specifier:** တူဝ်လိၵ်ႈဢွၼ်ႇ (မိူၼ်ၼင်ႇ C, N, P) ဢၼ်ၸႂ်ႉတႃႇသင်ႇၵၢၼ် Formatting။
* **Precision Specifier:** မၢႆၼပ်ႉဢၼ်ၸွမ်းလင် Specifier (တူဝ်ယင်ႇ: `N2`) တႃႇမၵ်းမၼ်ႈဝႃႈ ၶႂ်ႈလႆႈပၢႆၸုၵ်းလၢႆတူဝ်။
* **Padding:** ၵၢၼ်သႂ်ႇ "လွၵ်းပဝ်ႇ" ဢမ်ႇၼၼ် "မၢႆ 0" တႃႇႁႂ်ႈတၢင်းယၢဝ်းလိၵ်ႈ မၼ်းတဵမ်ထူၼ်ႈၸွမ်းပိူင်။
