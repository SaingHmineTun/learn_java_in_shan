# Lesson 125: "Best Practices: Why Fields Should Almost Always Be Private"

ၼႂ်းလေႃးၵတႅမ်ႈ Code ၽူႈမီးၸၼ်ႉ (Professional) ၼၼ်ႉ... ႁဝ်းမီးပၵ်းပိူင်ဝႆႉဝႃႈ **"Fields တေလႆႈပဵၼ် `private` တႃႇသေႇ"** ၶႃႈ။ 
သင်ၶႂ်ႈပၼ်ပိူၼ်ႈၸႂ်ႉ ႁဝ်းတေလႆႈၸႂ်ႉ **Properties** (Module ၼႃႈ) ဢမ်ႇၼၼ် **Methods** တႅၼ်းၶႃႈ။

### 1. ႁႄႉၵၢင်ႈ "ၶေႃႈမုၼ်းယုင်ႈ" (Preventing Corruption)
သင် `public int Age` ၼႆ... ၽႂ်ၵေႃႈမႃးတႅမ်ႈ `student.Age = -500;` လႆႈၶႃႈ။ ၵူၺ်းၵႃႈ သင်မၼ်းပဵၼ် `private` ႁဝ်းၸၢင်ႈၵုမ်းထိင်းလႆႈ:

```csharp
private int _age;

public void SetAge(int age) {
    if (age > 0 && age < 100) {
        _age = age; // ပၼ်မႄးၵူၺ်း မိူဝ်ႈၶေႃႈမုၼ်းမႅၼ်ႈယံ
    }
}
```

### 2. "လွင်ႈလွတ်ႈလႅဝ်း" ၼႂ်းၵၢၼ်မႄး Code (Flexibility)
ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းတႅမ်ႈ Class `Product` သေ ပၼ်ပိူၼ်ႈၸႂ်ႉ `public double Price` ၵႂႃႇၼႂ်း Project 100 ဢၼ်ယဝ်ႉ။ ဝၢႆးၼႆႉမႃး ႁဝ်းၶႂ်ႈလႅၵ်ႈလၢႆႈထႅမ်ႈထႅင်ႈ Tax တီႈၼႂ်း Price ၼႆ... ႁဝ်းတေလႆႈၸွမ်းမႄး Code တႃႇ Project တင်း 100 ဢၼ်ၼၼ်ႉၶႃႈ။

သင်ပဵၼ် `private` ဝႆႉၼႆ ႁဝ်းမႄးၼႂ်း Class `Product` တီႈလဵဝ်ၵေႃႈယဝ်ႉယဝ်ႉၶႃႈ။



### 3. "ႁူႉၵူၺ်း ဢၼ်လူဝ်ႇႁူႉ" (Abstraction)
ၵူၼ်းၸႂ်ႉ Code ႁဝ်း ဢမ်ႇလူဝ်ႇႁူႉဝႃႈ ႁဝ်းသိမ်းၶေႃႈမုၼ်းၼၼ်ႉၸိူင်ႉႁိုဝ် (သိမ်းၼႂ်း variable, သိမ်းၼႂ်း file, ဢမ်ႇၼၼ် ၼပ်ႉဢၢၼ်ႇဢဝ်ၵမ်းလဵဝ်)။ 
ၵၢၼ်ႁဵတ်း Fields ႁႂ်ႈပဵၼ် `private` ၸွႆးသိူင်ႇလွင်ႈသုၵ်ႉယုင်ႈ (Complexity) ၸိူဝ်းၼႆႉဝႆႉၶႃႈ။


### 4. Naming Convention (ပိူင်တႅမ်ႈၸိုဝ်ႈ)
ၼႂ်း C#... Fields ဢၼ်ပဵၼ် **`private`** ၼၼ်ႉ ႁဝ်းၵႆႉတႅမ်ႈမၢႆၸွမ်း **Underscore (`_`)** ဝႆႉၽၢႆႇၼႃႈ သေၸႂ်ႉ **camelCase** ၶႃႈ။
* တူဝ်ယင်ႇ: `private string _studentName;`
* လွင်ႈၼႆႉ ၸွႆႈႁႂ်ႈႁဝ်း ႁူႉလႆႈၵမ်းလဵဝ်ဝႃႈ တူဝ်ၼႆႉပဵၼ် Field (ၼႂ်း class) ဢမ်ႇၸႂ်ႈ Local Variable (ၼႂ်း method)။

---

### ၶေႃႈၵႂမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Best Practice:** လွၵ်းလၢႆးတႅမ်ႈ Code ဢၼ်လီသေပိူၼ်ႈ ဢၼ်ပိူၼ်ႈယွမ်းၸႂ်ႉၵူႈတီႈ။
* **Data Corruption:** ၵၢၼ်ဢၼ်ၶေႃႈမုၼ်းၽိတ်းပိူင် ယွၼ်ႉမီးၵၢၼ်မႄးဢၼ် ဢမ်ႇထုၵ်ႇမႅၼ်ႈ။
* **Validation:** ၵၢၼ်ၵူတ်ႇထတ်းၶေႃႈမုၼ်း ႁႂ်ႈမႅၼ်ႈၸွမ်းပိူင်မီႈ။
* **Maintenance:** ၵၢၼ်ၵုမ်းထိင်း လႄႈ မႄး Code ႁႂ်ႈမၼ်းၸႂ်ႉလႆႈႁိုင်ๆ။
