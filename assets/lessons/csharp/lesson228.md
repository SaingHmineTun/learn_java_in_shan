# Lesson 228: "The System.IO Namespace: Understanding Streams and Files"

ၼႂ်း C# ၼႆႉ ၵၢၼ်ၵွင်ႉၵၢႆႇတင်း Hard Drive တႃႇလူ ဢမ်ႇၼၼ် တႅမ်ႈ File ၼၼ်ႉ 
ယူႇၼႂ်း Namespace ဢၼ်ၸိုဝ်ႈဝႃႈ **`System.IO`** (IO ယေႃႈမႃးတီႈ **Input / Output**) ၶႃႈ။

### 1. File vs. Directory
* **File:** ပဵၼ်တူဝ်သိမ်းၶေႃႈမုၼ်းတႄႉတႄႉ (မိူၼ်ၼင်ႇ `note.txt`, `data.json`)။
* **Directory (Folder):** ပဵၼ်တီႈယူႇ ဢၼ်ၸတ်းၸုမ်း File ဝႆႉႁူမ်ႈၵၼ်။

### 2. Stream ပဵၼ်သင်?
ဝူၼ်ႉတူၺ်းဝႃႈ **Stream** မိူၼ်ၼင်ႇ "ၼမ်ႉလႆ" ၶႃႈ။ မိူဝ်ႈႁဝ်းလူ File ဢၼ်ယႂ်ႇၼႃႇၼႆ ႁဝ်းဢမ်ႇၸၢင်ႈဢဝ်မၼ်းမႃးဝႆႉၼႂ်း RAM ၵမ်းလဵဝ်တင်းမူတ်း။
ႁဝ်းၸင်ႇလူဝ်ႇ "ပွႆႇႁႂ်ႈၶေႃႈမုၼ်းလႆမႃး" ၵမ်းဢိတ်းၵမ်းဢိတ်း (Byte by Byte) ၶႃႈ။



* **Read Stream:** ၶေႃႈမုၼ်းလႆ လုၵ်ႉတီႈ File ၶဝ်ႈမႃးၼႂ်း App ႁဝ်း (Input)။
* **Write Stream:** ၶေႃႈမုၼ်းလႆ လုၵ်ႉတီႈ App ႁဝ်း ၵႂႃႇသိမ်းဝႆႉၼႂ်း File (Output)။

### 3. Class ဢၼ်လမ်ႇလွင်ႈၼႂ်း System.IO
ၼႂ်း Namespace ၼႆႉ မီး Class ၸိူဝ်းဢၼ်ႁဝ်းတေလႆႈၸႂ်ႉၼမ်သုတ်းယဝ်ႉ:

| Class | ၼႃႈၵၢၼ် (Purpose) |
| :--- | :--- |
| **File** | တႃႇသၢင်ႈ, လူတ်ႇ, ၶၢႆႉ ဢမ်ႇၼၼ် လူ File (Static methods) |
| **Directory** | တႃႇၸတ်းၵၢၼ် Folder (သၢင်ႈ, ႁႃ File ၼႂ်း Folder) |
| **Path** | တႃႇၸတ်းၵၢၼ် "သဵၼ်ႈတၢင်း" File (မိူၼ်ၼင်ႇ `C:\MyData\test.txt`) |
| **FileStream** | တႃႇလူ လႄႈ တႅမ်ႈ Byte ၼႂ်း File လၢႆး Stream |

### 4. ႁဵတ်းသင်ႁဝ်းၸင်ႇလူဝ်ႇ Streams?
1.  **Efficiency:** ဢမ်ႇၵိၼ် Memory ၼမ် မိူဝ်ႈလူ File ဢၼ်ယႂ်ႇ (မိူၼ်ၼင်ႇ Video ဢမ်ႇၼၼ် Database ယႂ်ႇယႂ်ႇ)။
2.  **Real-time:** ႁဝ်းၸၢင်ႈတႄႇႁဵတ်းၵၢၼ်ၸွမ်းၶေႃႈမုၼ်း တွၼ်ႈတႄႇမၼ်းလႆႈ ၵမ်းလဵဝ် လူၺ်ႈဢမ်ႇလူဝ်ႇပႂ်ႉႁႂ်ႈ လူယဝ်ႉတင်းမူတ်း။
3.  **Versatility:** Stream ၼႆႉ ဢမ်ႇၸႂ်ဝႃႈ ၸႂ်ႉလႆႈတင်း File ၵူၺ်းၶႃႈ။ ၸႂ်ႉမၼ်းၸွမ်း Network (သူင်ႇ Data ၶိုၼ်ႈၼိူဝ် Internet) ၵေႃႈလႆႈၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **I/O (Input/Output):** ၵၢၼ်သူင်ႇၶေႃႈမုၼ်းၶဝ်ႈ လႄႈ ဢွၵ်ႇ ၼႂ်း App။
* **Stream:** သဵၼ်ႈတၢင်းလႆ ၶွင်ၶေႃႈမုၼ်း (Sequence of bytes)။
* **Persistence:** ၵၢၼ်သိမ်းၶေႃႈမုၼ်းဝႆႉႁႂ်ႈမၼ်း "ယူႇမၼ်ႈ" တႃႇသေႇ ဢမ်ႇႁၢႆၵႂႃႇမိူဝ်ႈပိၵ်ႉ App။
* **Buffer:** တီႈၵဵပ်းၶေႃႈမုၼ်းဝႆႉၸူဝ်ႈၶၢဝ်ႇ ၼႂ်းၵၢၼ်သူင်ႇ Stream တႃႇႁႂ်ႈမၼ်းလႅၼ်ႈလီ။
