# Lesson 241: "The Threading Model: Why the UI Freezes"

တွၼ်ႈတႃႇပွင်ႇၸႂ်လွင်ႈ Async တႄႉတႄႉၼၼ်ႉ ႁဝ်းတေလႆႈႁူႉဝႃႈ App ၼိုင်ႈဢၼ်ၼႆႉ မၼ်းႁဵတ်းၵၢၼ်ၸိူင်ႉႁိုဝ်ၶႃႈ။

### 1. UI Thread ပဵၼ်သင်?

ၼႂ်း App ၵူႈဢၼ် (Android, Flutter, Windows) တေမီး Thread တူဝ်ၼိုင်ႈ ဢၼ်ႁွင်ႉဝႃႈ **Main Thread** ဢမ်ႇၼၼ် **UI Thread** ၶႃႈ။ 
မၼ်းမီးၼႃႈၵၢၼ် 2 ဢၼ်:

1.  **Drawing:** တႅမ်ႈႁၢင်ႈ၊ ၼႄလိၵ်ႈ၊ ပၼ်ႇ Animation ၼိူဝ် Screen။
2.  **Interaction:** ႁပ်ႉၵၢၼ်ၼဵၵ်း (Click) လႄႈ ၵၢၼ်တႅမ်ႈလိၵ်ႈတီႈ User။

### 2. ႁဵတ်းသင် UI ၸင်ႇ Freeze?

ဝူၼ်ႉတူၺ်းဝႃႈ UI Thread ၼႆႉ မိူၼ်ၼင်ႇ "ၵူၼ်းႁဵတ်းၵၢၼ် ၵေႃႉလဵဝ်" ၶႃႈ။ သင်ႁဝ်းၸႂ်ႉမၼ်းၵႂႃႇ Download File ဢၼ်ယႂ်ႇယႂ်ႇ (လၢႆး Synchronous) ၼႆၸိုင် မၼ်းတေၵႂႃႇ **"ပႂ်ႉ"** ယူႇတီႈၼၼ်ႈ သေဢမ်ႇမီးၶၢဝ်းယၢမ်းမႃး "တႅမ်ႈႁၢင်ႈ" ၼိူဝ် Screen ယဝ်ႉ။

* **ၽွၼ်းလႆႈ:** App ႁဝ်းတေၶႅင်ၵႂႃႇ၊ ၼဵၵ်းသင်ၵေႃႈဢမ်ႇလႆႈ၊ Animation ၵေႃႈၵိုတ်းၵႂႃႇၶႃႈ။

### 3. လၢႆးၵႄႈ (The Fix: Background Threads)

တႃႇႁႂ်ႈ UI ႁဝ်းတိုၵ်ႉ "တူင်ႉၼိုင်" လႆႈယူႇၼၼ်ႉ ႁဝ်းတေလႆႈသူင်ႇၵၢၼ်ဢၼ်ၼၵ်းၼၵ်း ၵႂႃႇႁဵတ်းဝႆႉတီႈ **Background Thread** ၶႃႈ။

* **UI Thread:** တိုၵ်ႉတႅမ်ႈႁၢင်ႈ လႄႈ ႁပ်ႉၵၢၼ်ၼဵၵ်းယူႇ။
* **Background Thread:** တိုၵ်ႉလူ File ဢမ်ႇၼၼ် Download Data ယူႇ။

### 4. Async/Await ၸွႆးၸိူင်ႉႁိုဝ်?

မိူဝ်ႈၵွၼ်ႇ ၵၢၼ်ၸႅၵ်ႇ Thread ၼႆႉ ယုင်ႈယၢင်ႈၼႃႇ။ ၵူၺ်းၵႃႈ C\# ၸႂ်ႉပိူင် **Task-based Asynchronous Pattern (TAP)** ဢၼ်ႁဵတ်းႁႂ်ႈႁဝ်းၸၢင်ႈ:

1.  ပွႆႇႁႂ်ႈၵၢၼ်ၼၵ်းၼၵ်း ႁဵတ်းၵၢၼ်ၽၢႆႇလင်ဝႆႉ။
2.  ပွႆႇ UI Thread မိူဝ်းႁဵတ်းၵၢၼ် တၢင်ႇပိူင်။
3.  မိူဝ်ႈၵၢၼ်ၽၢႆႇလင်ယဝ်ႉယဝ်ႉ ၸင်ႇၵွႆႈပိၼ်ႇမႃး Update UI ၶိုၼ်းၶႃႈ။

-----

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Thread:** သဵၼ်ႈတၢင်း ၵၢၼ်ႁဵတ်းလဵၵ်ႉလဵၵ်ႉ ၼႂ်း App။
* **UI Thread:** Thread တူဝ်ပိူင်လူင် ဢၼ်ၸတ်းၵၢၼ် တႃႇၼႄႁၢင်ႈတီႈ App။
* **Background Thread:** Thread ဢၼ်ႁဵတ်းၵၢၼ်ယူႇၽၢႆႇလင်။
* **Deadlock:** ပဵၼ်လွင်ႈဢၼ် Thread သွင်တူဝ် ပႂ်ႉၵၼ်ၵႂႃႇမႃး တေႃႇပေႃး App ၶႅင်တင်းမူတ်း။
