# Lesson 67: "The Zero-Based Indexing System"

မိူဝ်ႈႁဝ်းသၢင်ႈ Array ယဝ်ႉယဝ်ႉၼၼ်ႉ ၶေႃႈမုၼ်းၵူႈဢၼ် (Element) တေမီး "မၢႆၼပ်ႉတီႈယူႇ" ဢၼ်ႁဝ်းႁွင်ႉဝႃႈ **Index** ၶႃႈ။

### 1. ပိူင်ၵၢၼ်ၼပ်ႉ (Zero-Based)
တီႈယူႇတီႈၼႂ်း Array ၼႆႉ တေႇတီႈ 0 ယဝ်ႉ။ ပွင်ႇဝႃႈ ၶေႃႈမုၼ်း ဢွၼ်တၢင်းသုတ်ႉ ဢၼ်မီးဝႆႉ တီႈၼႂ်းမၼ်းၼၼ်ႉ မီးဝႆႉတီႈ မၢႆ 0 ယဝ်ႉ။
ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းမီး Array ဢၼ်သိမ်းၸိုဝ်ႈ ၽၵ်းဝႆႉ 3 ဢၼ် ၼႆလႄႈ။
`string[] veg = { "Kailan", "Cabbage", "Mustard" };`

* **Index 0:** Kailan (တူဝ်ထီႉၼိုင်ႈ)
* **Index 1:** Cabbage (တူဝ်ထီႉသွင်)
* **Index 2:** Mustard (တူဝ်ထီႉသၢမ်)



### 2. လၢႆးဢၢၼ်ႇလူ လႄႈ လၢႆးမႄးၶေႃႈမုၼ်း
ႁဝ်းၸႂ်ႉမၢႆ **`[]`** ႁူမ်ႈတင်းမၢႆၼပ်ႉ Index တႃႇဢဝ်ၶေႃႈမုၼ်းၼၼ်ႉမႃးၸႂ်ႉၶႃႈ။

* **ၵၢၼ်ဢၢၼ်ႇလူ (Access):**
```csharp
Console.WriteLine(veg[0]); // ၽွၼ်းလႆႈ: Kailan
```

* **ၵၢၼ်မႄးလႅၵ်ႈလၢႆႈ (Modify):**
```csharp
veg[1] = "Broccoli"; // လႅၵ်ႈလၢႆႈ Cabbage ႁႂ်ႈပဵၼ် Broccoli
```

### 3. လွင်ႈလူဝ်ႇၾၢင်ႉ (The Index Error)
သင်ဝႃႈ Array ႁဝ်းမီး 3 ဢၼ် (Index 0, 1, 2) သေ ႁဝ်းၵႂႃႇႁွင်ႉၸႂ်ႉ `veg[3]` ၼႆ... Program တေ **Crash** ၵမ်းလဵဝ်ၶႃႈ!
မၼ်းတေၼႄ Error ဝႃႈ **`IndexOutOfRangeException`** (ပွင်ႇဝႃႈ: ႁွင်ႉၸႂ်ႉမၢႆၼပ်ႉဢၼ်ဢမ်ႇမီးဝႆႉ)။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Index:** မၢႆၼပ်ႉတီႈယူႇၶွင်ၶေႃႈမုၼ်း ၼႂ်း Array (တႄႇတီႈ 0)။
* **Zero-Based:** ပိူင်ၵၢၼ်ၼပ်ႉဢၼ်တႄႇတီႈ သုၼ် (Zero)။
* **Element:** ၶေႃႈမုၼ်းၼိုင်ႈဢၼ် ဢၼ်သိမ်းဝႆႉၼႂ်းလွၵ်း Array။
* **Exception:** ၶေႃႈၽိတ်း (Error) ဢၼ်ပဵၼ်မိူဝ်ႈ Program တိုၵ်ႉလႅၼ်ႈယူႇ။
* **OutOfRange:** ၵၢၼ်ႁွင်ႉၸႂ်ႉမၢႆၼပ်ႉဢၼ်ပူၼ်ႉၶွပ်ႇၵင်ႈ (Boundary) ၶွင် Array။
