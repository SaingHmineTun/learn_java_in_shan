# Lesson 263: "Shan Fonts and Unicode in Desktop UI"

ၵၢၼ်ၼႄလိၵ်ႈတႆးၼႂ်း Windows ၼၼ်ႉ မၼ်းမီးလွင်ႈယၢပ်ႇဢိတ်းၼိုင်ႈ ယွၼ်ႉပိူင် Windows ၵႆႉၸႂ်ႉ Font Standard ဢၼ်ဢမ်ႇမီး "တူဝ်လိၵ်ႈတႆး" ပႃးဝႆႉၶႃႈ။

### 1. Unicode vs. Non-Unicode (Zawgyi-Tai)
* **Unicode (Standard):** ပဵၼ်ပိူင်သၢင်ႈဢၼ်ၵူႈမိူင်းမိူင်းယွမ်းႁပ်ႉ။ ၼႂ်း C# လႄႈ .NET ၵေႃႈ Code ႁဝ်းၸႂ်ႉ Unicode တႃႇသေႇၶႃႈ။
* **Zawgyi-Tai:** ပဵၼ် Font ပိူင်ၵဝ်ႇ ဢၼ်ၵႆႉမီးပၼ်ႁႃလွင်ႈၵၢၼ် "သွၵ်ႈႁႃ" (Search) လႄႈ ၵၢၼ်ၸတ်း "ၸႅၼ်ႇထႅဝ်" (Sorting)။
* **Best Practice:** ႁဝ်းထုၵ်ႇလီၸႂ်ႉ **Unicode** (မိူၼ်ၼင်ႇ NamKhone, PangLong) တႃႇသေႇ ၼႂ်းၵၢၼ်တႅမ်ႈ App ပၢၼ်မႂ်ႇၶႃႈ။

### 2. Embedded Fonts (ၵၢၼ်သႂ်ႇ Font ၵႂႃႇၸွမ် App)
ပၼ်ႁႃဢၼ်ယႂ်ႇသုတ်းပဵၼ် မိူဝ်ႈႁဝ်းသူင်ႇ App ႁဝ်းပၼ် User သေ ၼႂ်းၶွမ်းၶဝ် **ဢမ်ႇမီး** Font တႆးဝႆႉၶႃႈ။
* **Solution:** ႁဝ်းၸၢင်ႈဢဝ် Font ၾၢႆႇ `.ttf` သႂ်ႇဝႆႉၼႂ်း Project (Resources) သေသင်ႇႁႂ်ႈ App ၸႂ်ႉ Font ၼၼ်ႉၵမ်းလဵဝ် လူၺ်ႈ User ဢမ်ႇလူဝ်ႇ Install Font ၶဝ်ႈၼႂ်း Windows ၶႃႈ။

### 3. လၢႆးတႅမ်ႈ XAML တႃႇၼႄလိၵ်ႈတႆး
ၼႂ်း WPF ႁဝ်းၸၢင်ႈမၵ်းမၼ်ႈ `FontFamily` ႁႂ်ႈပဵၼ် Font တႆးလႆႈငၢႆႈငၢႆႈၶႃႈ:

```xml
<TextBlock Text="မႂ်ႇသုင်ၶႃႈ" 
           FontFamily="PangLong" 
           FontSize="24" />

<TextBlock Text="တႆးႁဝ်းၶိူဝ်းယႂ်ႇ" 
           FontFamily="./Fonts/#NamKhone Unicode" 
           FontSize="24" />
```



### 4. Handling Line Spacing (လွင်ႈႁၢင်ႇၵႄႈထႅဝ်)
တူဝ်လိၵ်ႈတႆးမီး "မၢႆ" (Tone marks) ၽၢႆႇၼိူဝ် လႄႈ ၽၢႆႇတႂ်ႈၼမ်။ မၢင်ပွၵ်ႈၼႂ်း TextBox ၼၼ်ႉ လိၵ်ႈၸၢင်ႈ "ထုၵ်ႇတတ်း" (Clipped) ၵႂႃႇၶႃႈ။
* **Solution:** ႁဝ်းတေလႆႈမႄး Property **`LineHeight`** ဢမ်ႇၼၼ်ထႅမ် **`Padding`** ၽၢႆႇၼိူဝ်/တႂ်ႈ ႁႂ်ႈမၼ်းမီးပိုၼ်ႉတီႈတႃႇၼႄမၢႆတႆးလႆႈတဵမ်ထႅၼ်ႈၶႃႈ။

---

### ႁဵတ်းသင်လွင်ႈၼႆႉၸင်ႇလမ်ႇလွင်ႈ?
1. **Accessibility:** ႁႂ်ႈၵူၼ်းတႆးႁဝ်း ၸႂ်ႉ App လႆႈငၢႆႈငၢႆႈ လူၺ်ႈဢမ်ႇလူဝ်ႇမီးတၢင်းႁူႉၶွမ်းၼမ်။
2. **Standardization:** ၸွႆးႁႂ်ႈလိၵ်ႈတႆးၼိူဝ် Internet လႄႈ App မီး Standard ဢၼ်မႅၼ်ႈၸွမ်း Unicode တင်းမူတ်း။
3. **Professionalism:** App ဢၼ်ၼႄလိၵ်ႈတႆးလႆႈႁၢင်ႈလီ လႄႈ တႅတ်ႉတေႃး တေႁဵတ်းႁႂ်ႈ User ယုမ်ႇယမ်လိူဝ်ၵဝ်ႇၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Unicode:** ၸိူဝ်ႉၾၼ်ႇ (Standard) တွၼ်ႈတႃႇတူဝ်လိၵ်ႈၵူႈမိူင်းမိူင်း။
* **Embedded Font:** ၵၢၼ်ဢဝ် Font သႂ်ႇဝႆႉၼႂ်းတူဝ် App တႃႇသူင်ႇၵႂႃႇၸွမ်ၵၼ်။
* **Fallback Font:** Font ဢၼ် Windows တေလိူၵ်ႈၸႂ်ႉ သင်မၼ်းႁႃ Font ဢၼ်ႁဝ်းမၵ်းမၼ်ႈဝႆႉဢမ်ႇႁၼ်။
* **Glyph:** ႁၢင်ႈၽၢင်တူဝ်လိၵ်ႈ ဢၼ်ၼႄၼိူဝ်ၼႃႈၸေႃး။
