# Lesson 281: "The Model-View-Controller (MVC) Pattern"

**MVC** ပဵၼ်လၢႆးၸႅၵ်ႇ Code ဢွၵ်ႇပဵၼ် 3 ပိူင်လူင် တွၼ်ႈတႃႇႁႂ်ႈ App ႁဝ်း မႄးငၢႆႈ လႄႈ ဢမ်ႇယုင်ႈယၢင်ႈၵၼ်ၶႃႈ။ 
ၼႆႉႁဝ်းႁွင်ႉဝႃႈ **Separation of Concerns** (ၵၢၼ်ၸႅၵ်ႇၼႃႈၵၢၼ်ၽႂ်မၼ်း) ၶႃႈဢေႃႈ။

### 1. Model (ၶေႃႈမုၼ်း)
* **ၼႃႈၵၢၼ်:** မီးၼႃႈၵၢၼ် တႃႇၸတ်းၵၢၼ် "ၶေႃႈမုၼ်း" (Data) လႄႈ "ပၵ်းၵၢၼ်" (Business Logic)။
* **တူဝ်ယင်ႇ:** Class `Student` ဢၼ်ႁဝ်းႁဵၼ်းမႃးၼႂ်း Module 22 ၼၼ်ႉယဝ်ႉၶႃႈ။ မၼ်းတေလၢတ်ႈတေႃႇတင်း Database တႃႇလူ ဢမ်ႇၼၼ် သိမ်း Data ၶႃႈ။

### 2. View (ၼႃႈတႃ UI)
* **ၼႃႈၵၢၼ်:** မီးၼႃႈၵၢၼ် "ၼႄႁၢင်ႈ" ၵူၺ်း။ မၼ်းတေဢဝ်ၶေႃႈမုၼ်းမႃးၼႄပဵၼ် HTML ႁၢင်ႈလီလီ ႁႂ်ႈ User ႁၼ်ၶႃႈ။
* **တူဝ်ယင်ႇ:** ၼႃႈလိၵ်ႈ Website ဢၼ်မီးသဵၼ်ႈမၢႆလုၵ်ႈႁဵၼ်း ဢၼ်ႁဝ်းႁၼ်ၼိူဝ် Browser ၼၼ်ႉၶႃႈ။

### 3. Controller (တူဝ်ၸတ်းၵၢၼ်)
* **ၼႃႈၵၢၼ်:** ပဵၼ် "ၵူၼ်းၵႄႈၵၢင်" ဢၼ်ပႂ်ႉႁပ်ႉ Request လုၵ်ႉတီႈ Browser သေ သင်ႇၵၢၼ်ၸူး Model လႄႈ View။
* **တူဝ်ယင်ႇ:** မိူဝ်ႈ User ၼဵၵ်းတူၺ်း "သဵၼ်ႈမၢႆလုၵ်ႈႁဵၼ်း" ၼၼ်ႉ Controller တေၵႂႃႇထၢမ် Model ဝႃႈ "ဢဝ် Data လုၵ်ႈႁဵၼ်းမႃးပၼ်ၽွင်ႈ" သေသူင်ႇ Data ၼၼ်ႉၵႂႃႇပၼ် View တႃႇၼႄႁၢင်ႈၶႃႈ။



---

### ႁဵတ်းသင်ႁဝ်းၸင်ႇလူဝ်ႇ MVC?
1. **Easy to Maintain:** သင်ႁဝ်းၶႂ်ႈမႄး "ၼႃႈတႃ" (View) ႁႂ်ႈႁၢင်ႈလီလိူဝ်ၵဝ်ႇၼႆ ႁဝ်းဢမ်ႇလူဝ်ႇၵႂႃႇမႄး Code ဢၼ်ၵွင်ႉ Database (Model) ၶႃႈ။
2. **Parallel Development:** ၵူၼ်းၵေႃႉၼိုင်ႈတႅမ်ႈ HTML (View) သေ ထႅင်ႈၵေႃႉၼိုင်ႈတႅမ်ႈ C# (Controller) ၸွမ်းၵၼ်လႆႈၶႃႈ။
3. **Reusability:** Model တူဝ်လဵဝ်ၵၼ် ၸၢင်ႈဢဝ်ၵႂႃႇၼႄလႆႈၼႂ်းလၢႆလၢႆ View (မိူၼ်ၼင်ႇ ၼႄၼိူဝ် Web တင်း ၼႄၼိူဝ် Mobile App) ၶႃႈဢေႃႈ။

---

### လၢႆးႁဵတ်းၵၢၼ် (Workflow)
1. **User** သူင်ႇ Request (GET) မႃးၸူး **Controller**။
2. **Controller** ၵႂႃႇဢဝ် Data တီႈ **Model**။
3. **Model** သူင်ႇ Data ၶိုၼ်းပၼ် **Controller**။
4. **Controller** သူင်ႇ Data ၼၼ်ႉၵႂႃႇပၼ် **View**။
5. **View** ႁၢင်ႈႁႅၼ်း HTML သေသူင်ႇၶိုၼ်းပၼ် **User** ၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Separation of Concerns:** ၵၢၼ်ၸႅၵ်ႇ Code ဢွၵ်ႇၸွမ်းၼင်ႇ ၼႃႈၵၢၼ်ၽႂ်မၼ်း။
* **Business Logic:** ပၵ်းၵၢၼ် ၵၢၼ်ႁဵတ်းၵၢၼ် ၶွင် App။
* **Routing:** ပိူင်သၢင်ႈဢၼ်တႅပ်းတတ်းၸႂ်ဝႃႈ URL လႂ် တေၵႂႃႇၸူး Controller တူဝ်လႂ်။
* **Payload:** ၶေႃႈမုၼ်းဢၼ်သူင်ႇပႃးၵႂႃႇၵႄႈ Controller လႄႈ View။
