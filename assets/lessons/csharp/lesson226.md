# Lesson 226: "Deferred Execution"

မိူဝ်ႈႁဝ်းတႅမ်ႈ LINQ Query (မိူၼ်ၼင်ႇ `.Where()`), C# တေဢမ်ႇပၼ်ႇ Loop တႃႇႁႃၶေႃႈမုၼ်း **ၵမ်းလဵဝ်** ၶႃႈ။ 
မၼ်းၵူၺ်း "တွင်းဝႆႉ" ပၵ်းၵၢၼ် (Plan) ဢၼ်ႁဝ်းပၼ်မၼ်းၼၼ်ႉၵူၺ်း။

### 1. မၼ်းႁဵတ်းၵၢၼ်မိူဝ်ႈလႂ်? (When does it run?)
LINQ တေတႄႇလႅၼ်ႈ (Execute) ၵေႃႈတေႃႇမိူဝ်ႈ:
1.  ႁဝ်းပၼ်ႇ Loop လူမၼ်း (ၸႂ်ႉ **`foreach`**)။
2.  ႁဝ်းပိၼ်ႇမၼ်းပဵၼ် List ဢမ်ႇၼၼ် Array (ၸႂ်ႉ **`.ToList()`**, **`.ToArray()`**)။
3.  ႁဝ်းၸႂ်ႉ Method ဢၼ်သူင်ႇၵႃႈၶၼ်တူဝ်လဵဝ် (မိူၼ်ၼင်ႇ **`.Count()`**, **`.First()`**, **`.Average()`**)။



### 2. တူဝ်ယင်ႇ Code: ၵၢၼ်ယၼ်ႇၶၢဝ်းယၢမ်း
ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းမီး List တူဝ်ၼပ်ႉ သေႁဝ်းသၢင်ႈ Query ဝႆႉၶႃႈ။

```csharp
var numbers = new List<int> { 1, 2, 3 };

// 1. သၢင်ႈ Query ဝႆႉ (မၼ်းပႆႇလႅၼ်ႈၵၢၼ်!)
var query = numbers.Where(n => n > 2);

// 2. ထႅမ်ၶေႃႈမုၼ်းမႂ်ႇ ၶဝ်ႈၵႂႃႇၼႂ်း List
numbers.Add(4);

// 3. ၸင်ႇၵွႆႈပၼ်ႇ Loop လူ (မၼ်းတေလႅၼ်ႈၵၢၼ်ယၢမ်းၼႆႉ!)
foreach (var n in query) {
    Console.WriteLine(n); // ၼႄဢွၵ်ႇ: 3, 4
}
```
> **Notice:** မၼ်းၼႄမၢႆ **4** ဢွၵ်ႇမႃးၸွမ်း ယွၼ်ႉမၼ်းတႄႇႁႃၶေႃႈမုၼ်း မိူဝ်ႈႁဝ်းႁွင်ႉၸႂ်ႉ `foreach` ၼၼ်ႉၶႃႈ။

### 3. Immediate Execution (ၵၢၼ်ႁဵတ်းၵၢၼ်ၵမ်းလဵဝ်)
သင်ႁဝ်းၶႂ်ႈႁႂ်ႈမၼ်း "လႅၼ်ႈၵၢၼ်ၵမ်းလဵဝ်" သေသိမ်းၽွၼ်းလႆႈဝႆႉတႅတ်ႈတေႃး... ႁဝ်းတေလႆႈၸႂ်ႉ **`.ToList()`** ၶႃႈ။

```csharp
// ႁဵတ်းၵၢၼ်ၵမ်းလဵဝ် သေသိမ်းဝႆႉၼႂ်း Memory
var resultList = numbers.Where(n => n > 2).ToList();

numbers.Add(5); // ထႅမ်မၢႆ 5 ဝၢႆးလင်
// resultList တေဢမ်ႇမီးမၢႆ 5 ယွၼ်ႉမၼ်းယဝ်ႉၵၢၼ်ၵႂႃႇယဝ်ႉ
```

### 4. ႁဵတ်းသင်ႁဝ်းၸင်ႇလူဝ်ႇႁူႉ?
* **Performance:** ႁဝ်းၸၢင်ႈသၢင်ႈ Query ဢၼ်ယုင်ႈယၢင်ႈဝႆႉလၢႆလၢႆၸၼ်ႉ သေပွႆႇႁႂ်ႈမၼ်းလႅၼ်ႈ "ၵမ်းလဵဝ်" မိူဝ်ႈလူဝ်ႇၼႄၽွၼ်းလႆႈ ႁဵတ်းႁႂ်ႈ App ဢမ်ႇၼၵ်းၶႃႈ။
* **Fresh Data:** ႁဝ်းတေလႆႈၶေႃႈမုၼ်းဢၼ် "မႂ်ႇသုတ်း" တႃႇသေႇ မိူဝ်ႈႁဝ်းပၼ်ႇ Loop လူမၼ်းၶႃႈ။
* **Avoid Re-calculation:** သင်ႁဝ်းပၼ်ႇ Loop လူ Query တူဝ်လဵဝ်ၵၼ် 2 ပွၵ်ႈ... မၼ်းတေ "လႅၼ်ႈၵၢၼ်" 2 ပွၵ်ႈၶႃႈ။ သင်ႁဝ်းဢမ်ႇၶႂ်ႈႁႂ်ႈမၼ်းလႅၼ်ႈလၢႆပွၵ်ႈ ႁႂ်ႈၸႂ်ႉ `.ToList()` သိမ်းဝႆႉၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Deferred Execution:** ၵၢၼ်ဢၼ် LINQ Query တေဢမ်ႇလႅၼ်ႈၵၢၼ် တေႃႇမိူဝ်ႈမီးၵၢၼ်ႁွင်ႉၸႂ်ႉ (Iteration)။
* **Immediate Execution:** ၵၢၼ်ႁဵတ်းၵၢၼ်ၵမ်းလဵဝ် ၽၢၼ်ႇ Method မိူၼ်ၼင်ႇ ToList() ဢမ်ႇၼၼ် Count()။
* **Lazy Loading:** ၵၢၼ်လူၶေႃႈမုၼ်း မိူဝ်ႈလူဝ်ႇၸႂ်ႉတႄႉတႄႉ (တႅၼ်းဢၼ်တေလူဝႆႉတင်းမူတ်းၵမ်းလဵဝ်)။
* **Streaming:** ၵၢၼ်လူၶေႃႈမုၼ်းၵမ်းတူဝ် ၼႂ်းယၢမ်းဢၼ်လႅၼ်ႈၵၢၼ် (မိူၼ်ၼင်ႇ foreach)။
