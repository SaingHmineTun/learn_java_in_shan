# Lesson 91: "The OOP Revolution: Thinking in Objects vs. Procedures"

မိူဝ်ႈၵွၼ်ႇ ႁဝ်းတႅမ်ႈ Code ၸွမ်းၼင်ႇ **Procedural Programming** (လၢႆးပဵၼ်တွၼ်ႈ)။ 
မၼ်းမိူၼ်ၼင်ႇ "ပပ်ႉႁုင်ၽတ်း" ဢၼ်လၢတ်ႈဝႃႈ: ႁဵတ်းဢၼ်ၼႆႉယဝ်ႉ ၵႂႃႇႁဵတ်းဢၼ်ၼၼ်ႉ ယဝ်ႉၵေႃႈ ႁဵတ်းထႅင်ႈဢၼ်ၼႆႉ။ 
ၵူၺ်းၵႃႈ ပေႃး App ႁဝ်းယႂ်ႇမႃး လၢႆးၼႆႉမၼ်းယုင်ႈၼႃႇၶႃႈ။

### 1. OOP ပဵၼ်သင်?
**Object-Oriented Programming (OOP)** ၼႆႉ ပဵၼ်လွၵ်းလၢႆးဢၼ်ႁဝ်းတူၺ်း Code ႁဝ်း ႁႂ်ႈမိူၼ်တင်း "ဝတ်ႉထု" (Objects) ၼႂ်းလေႃးၵတႄႉၶႃႈ။

* **Procedural:** ဝူၼ်ႉထိုင် "ၵၢၼ်ႁဵတ်း" (Functions/Verbs) ပဵၼ်လူင်။
* **OOP:** ဝူၼ်ႉထိုင် "တူဝ်ဝတ်ႉထု" (Objects/Nouns) ပဵၼ်လူင်။



### 2. တူဝ်ယင်ႇ: ၵၢၼ်သၢင်ႈ App ႁူင်းႁဵၼ်း
သင်ႁဝ်းတႅမ်ႈ App တႃႇႁူင်းႁဵၼ်း "ထုင်ႉမၢဝ်းၶမ်း" ႁဝ်း:
* **Procedural Style:** ႁဝ်းတေမီး Function `RegisterStudent()`, `CalculateGrade()`, `PrintIDCard()`. (Code မၼ်းတေယုင်ႈၵၼ်ဝႆႉ ၵူႈတီႈၶႃႈ)။
* **OOP Style:** ႁဝ်းတေမီး Object ဢၼ်ၸိုဝ်ႈဝႃႈ **`Student`**။ ၼႂ်းတူဝ် Student ၼၼ်ႉ မၼ်းတေမီးဝႆႉ ၶေႃႈမုၼ်း (Name, Age) လႄႈ ၵၢၼ်ႁဵတ်း (Register, Study) ယူႇႁူမ်ႈၵၼ်ဝႆႉၶႃႈ။

### 3. ပိူင်လူင် 4 ယၢင်ႇ ၶွင် OOP (The 4 Pillars)
တႃႇတေပဵၼ် OOP တေလႆႈမီး:
1.  **Encapsulation (ၵၢၼ်ႁေႃႇဝႆႉ):** သိမ်းၶေႃႈမုၼ်း လႄႈ Code ဝႆႉၼႂ်း Class လဵဝ်ၵၼ်။
2.  **Inheritance (ၵၢၼ်သိုပ်ႇၶိူဝ်း):** ဢဝ် Class ၵဝ်ႇ မႃးသၢင်ႈ Class မႂ်ႇ ႁႂ်ႈမၼ်းၵတ်ႉလိူဝ်ၵဝ်ႇ။
3.  **Polymorphism (ၵၢၼ်မီးလၢႆႁၢင်ႈ):** ၸိုဝ်ႈလဵဝ်ၵၼ် ၵူၺ်းၵႃႈ ႁဵတ်းၵၢၼ်ပႅၵ်ႇၵၼ် ယူႇတီႈတူဝ်ဢၼ်ၸႂ်ႉမၼ်း။
4.  **Abstraction (ၵၢၼ်လိူၵ်ႈၼႄ):** ၼႄတွၼ်ႈဢၼ်လူဝ်ႇၸႂ်ႉၵူၺ်းသေ တွၼ်ႈဢၼ်သုၵ်ႉယုင်ႈၼႂ်းမၼ်းၼၼ်ႉ ဢဝ်လပ်ႉဝႆႉၶႃႈ။

### 4. ႁဵတ်းသင်ၸင်ႇလႅၵ်ႈမႃးၸႂ်ႉ OOP?
* **Real-world Mapping:** မၼ်းငၢႆႈတႃႇဝူၼ်ႉ ယွၼ်ႉမၼ်းမိူၼ်လေႃးၵတႄႉ။
* **Scalability:** ပေႃး App ယႂ်ႇမႃး ၵုမ်းထိင်း (Manage) ငၢႆႈ။
* **Reusability:** ၸႂ်ႉ Class ဢၼ်တႅမ်ႈဝႆႉယဝ်ႉၼၼ်ႉ ၵႂႃႇၸႂ်ႉၼႂ်း Project တၢင်ႇဢၼ်ၵေႃႈ လႆႈၵမ်းလဵဝ်။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **OOP:** Object-Oriented Programming (လွၵ်းလၢႆးတႅမ်ႈ Code ဢၼ်ၸွမ်းပိူင် ဝတ်ႉထု)။
* **Procedure:** ပိူင်ၵၢၼ် ဢမ်ႇၼၼ် တွၼ်ႈ Code ဢၼ်လႅၼ်ႈၸွမ်းလွၵ်းလၢႆး။
* **Object:** တူဝ်ဝတ်ႉထု ဢၼ်မီးဝႆႉ ၶေႃႈမုၼ်း လႄႈ ၵၢၼ်ႁဵတ်း။
* **Class:** ၽႅၼ်ႇႁၢင်ႈ (Blueprint) ဢၼ်ၸႂ်ႉသၢင်ႈ Object။
