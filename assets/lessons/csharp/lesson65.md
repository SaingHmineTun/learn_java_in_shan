# Lesson 65: "Introduction to Data Structures: Why Use Arrays?"

မိူဝ်ႈပူၼ်ႉမႃး ႁဝ်းသွၼ်လွင်ႈ Variable ဢၼ်သိမ်းလႆႈ "ၵႃႈၶၼ်လဵဝ်" (မိူၼ်ၼင်ႇ `int score = 90;`)။ 
ၵူၺ်းၵႃႈ ဝူၼ်ႉတူၺ်းဝႃႈ သင်ႁဝ်းမီးလုၵ်ႈႁဵၼ်း 50 ၵေႃႉ၊ ႁဝ်းတေလႆႈသၢင်ႈ variable 50 တူဝ်ႁႃႉ? 
ၼၼ်ႉယဝ်ႉ ႁဝ်းၸင်ႇလူဝ်ႇ **Array** ၶႃႈ။

### 1. Array ပဵၼ်သင်? (What is an Array?)
**Array** ၼႆႉ မိူၼ်ၼင်ႇ "ၵွၵ်း" ဢမ်ႇၼၼ် "လွၵ်းသိမ်းၶူဝ်း" ဢၼ်မီးလၢႆလၢႆႁွင်ႈ သိုပ်ႇၵၼ်ဝႆႉၼႂ်း Variable ဢၼ်လဵဝ်။
* မၼ်းၸၢင်ႈသိမ်းၶေႃႈမုၼ်း **"ပိူင်လဵဝ်ၵၼ်"** (Same Data Type) လႆႈလၢႆလၢႆဢၼ်။
* ၶေႃႈမုၼ်းၵူႈဢၼ် တေမီး **"မၢႆၼပ်ႉတီႈယူႇ"** (Index) ႁင်းမၼ်း။



### 2. ႁဵတ်းသင်ႁဝ်းလူဝ်ႇၸႂ်ႉ Arrays?
* **Organization:** ႁဵတ်းႁႂ်ႈၶေႃႈမုၼ်းဢၼ်ၵဵဝ်ႇၵၼ် (မိူၼ်ၼင်ႇ ၸိုဝ်ႈလုၵ်ႈႁဵတ်းတင်းမူတ်း) ယူႇႁူမ်ႈၵၼ်လႆႈငၢႆႈ။
* **Efficiency:** ႁဝ်းၸၢင်ႈၸႂ်ႉ **Loops** (ၵၢၼ်ႁဵတ်းပၼ်ႇတူင်ႇ) တႃႇၸတ်းၵၢၼ်ၶေႃႈမုၼ်းတင်းမူတ်း လႆႈၵမ်းလဵဝ်။
* **Memory Management:** Computer တေၸတ်းပၼ်တီႈယူႇ (Memory) ဝႆႉပဵၼ်ၵွၼ်ႇပဵၼ်တႅဝ်း ႁဵတ်းႁႂ်ႈ App လႅၼ်ႈၽႂ်းၶိုၼ်ႈ။

### 3. တူဝ်ယင်ႇၼႂ်းၸူဝ်ႈပၢၼ်တႄႉ
* **Attendance List:** သဵၼ်ႈၸိုဝ်ႈလုၵ်ႈႁဵၼ်း ဢၼ်မႃးၶိုၼ်ႈႁဵၼ်း ၼႂ်းႁူင်းႁဵၼ်း "ထုင်ႉမၢဝ်းၶမ်း"။
* **Playlist:** သဵၼ်ႈၸိုဝ်ႈၵႂၢမ်း ၼႂ်း App လဵၼ်ႈၵႂၢမ်း။
* **Price List:** သဵၼ်ႈၸိုဝ်ႈၵႃႈၶၼ်ၵုၼ်ႇ ၼႂ်းသႅင်ႇၶၢႆၶူဝ်း။

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Data Structure:** လွၵ်းလၢႆးၸတ်းပိူင် လႄႈ သိမ်းၶေႃႈမုၼ်း ၼႂ်း Computer ႁႂ်ႈၸႂ်ႉလႆႈငၢႆႈ။
* **Array:** ၸူႉတွမ် Variable ပိူင်လဵဝ်ၵၼ် ဢၼ်သိမ်းဝႆႉၼႂ်းၸိုဝ်ႈလဵဝ်။
* **Element:** ၶေႃႈမုၼ်း "ၼိုင်ႈဢၼ်" ဢၼ်ယူႇၼႂ်း Array။
* **Collection:** ၸိုဝ်ႈႁွင်ႉႁူမ်ႈ ၸုမ်းၶေႃႈမုၼ်းလၢႆလၢႆဢၼ်။
