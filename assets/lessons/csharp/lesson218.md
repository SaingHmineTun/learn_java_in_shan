# Lesson 218: "Filtering Data with 'Where'"

Method **`.Where()`** တေပၼ်ႇတူၺ်း (Iterate) ၵူႈတူဝ်ၼႂ်း Collection သေ ၵူတ်ႇထတ်း "ငဝ်းလၢႆး" (Condition) ဢၼ်ႁဝ်းပၼ်ဝႆႉၼၼ်ႉၶႃႈ။ 
သင်ငဝ်းလၢႆးပဵၼ် **True** မၼ်းတေသိမ်းဝႆႉ၊ သင်ပဵၼ် **False** မၼ်းတေၶၢမ်းပူၼ်ႉပႅတ်ႈၶႃႈ။

### 1. ပိူင်သၢင်ႈ (Syntax)
ႁဝ်းၸႂ်ႉ **Lambda Expression** တႃႇမၵ်းမၼ်ႈ ၼႃႈၵၢၼ်ၵူတ်ႇထတ်းၶႃႈ။

```csharp
// ႁႃတူဝ်ၼပ်ႉဢၼ်ယႂ်ႇလိူဝ် 50
var bigNumbers = numbers.Where(n => n > 50);
```

### 2. ၵၢၼ်ၸႂ်ႉ Logical Operators
ႁဝ်းၸၢင်ႈၸႂ်ႉ **`&&` (And)**, **`||` (Or)**, လႄႈ **`!` (Not)** တႃႇသၢင်ႈ ငဝ်းလၢႆးဢၼ်ယုင်ႈယၢင်ႈ လိူဝ်ၵဝ်ႇၶႃႈ။

* **And (`&&`):** တေလႆႈမႅၼ်ႈၸွမ်းပိူင် "တင်းသွင်ၽၢႆႇ"။
* **Or (`||`):** မႅၼ်ႈၸွမ်း "သေပိူင်ႈၼိုင်ႈ" ၵေႃႈလႆႈယဝ်ႉ။



### 3. တူဝ်ယင်ႇ Code: ႁႃၶေႃႈမုၼ်းလုၵ်ႈႁဵၼ်း
ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းမီး List ၶွင် Class `Student` သေၶႂ်ႈႁႃၵူၼ်း ဢၼ်ယူႇၼႂ်း ဝဵင်းမူႇၸေႊလႄႈ ဢႃႇယူႇ တေလႆႈယႂ်ႇလိူဝ် 20 ၼႆၶႃႈ။

```csharp
var students = new List<Student> {
    new Student { Name = "ၸၢႆးသႅင်", Age = 19, City = "Muse" },
    new Student { Name = "ၼၢင်းမိူင်း", Age = 22, City = "Muse" },
    new Student { Name = "ၸၢႆးလႅင်း", Age = 25, City = "Lashio" }
};

// ႁႃလုၵ်ႈႁဵၼ်း ဢၼ်ယူႇဝဵင်း "Muse" လႄႈ ဢႃႇယုယႂ်ႇလိူဝ် 20
var result = students.Where(s => s.City == "Muse" && s.Age > 20);

foreach (var s in result) {
    Console.WriteLine($"ၵူၼ်းဢၼ်ႁႃႁၼ်: {s.Name}"); // ၼႄဢွၵ်ႇ: ၼၢင်းမိူင်း
}
```

### 4. လွင်ႈပႅၵ်ႇပိူင် ဢၼ်လူဝ်ႇတွင်း
* **`.Where()`** တေသူင်ႇၽွၼ်းလႆႈဢွၵ်ႇမႃးပဵၼ် **Collection** (IEnumerable) တႃႇသေႇ၊ ဢမ်ႇဝႃႈမၼ်းတေႁႃႁၼ် 1 တူဝ် ဢမ်ႇၼၼ် 1,000 တူဝ်ၵေႃႈယႃႇၶႃႈ။
* သင်မၼ်းႁႃဢမ်ႇႁၼ်သေတူဝ်... မၼ်းတေသူင်ႇ **Empty Collection** ဢွၵ်ႇမႃး (ဢမ်ႇပဵၼ် null ၶႃႈ)။
* မၼ်းၸၢင်ႈၸႂ်ႉ Method ၼႂ်း string ၸွမ်းၵေႃႈလႆႈ မိူၼ်ၼင်ႇ `.Where(s => s.Name.StartsWith("ၸ"))`။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Predicate:** Lambda Expression ဢၼ်သူင်ႇၵႃႈၶၼ် bool (true/false) ၶဝ်ႈၵႂႃႇၼႂ်း `.Where()`။
* **Logical Operators:** တူဝ်ၸွႆႈၵွင်ႉငဝ်းလၢႆး (&&, ||, !)။
* **Filtering:** ၵၢၼ်လိူၵ်ႈဢဝ်ၶေႃႈမုၼ်း ဢၼ်တႅတ်ႈတေႃးၸွမ်းၵၢင်ၸႂ်။
* **Collection:** ၵွင်ၶေႃႈမုၼ်း ဢၼ်သိမ်းဝႆႉလၢႆလၢႆတူဝ်။
