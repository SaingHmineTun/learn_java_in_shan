# Lesson 3: "Preparing the Development Environment" (ၵၢၼ်တင်ႈၶိူင်ႈမိုဝ်းတႃႇတႅမ်ႈ Code)

တႃႇတေပဵၼ် Developer ၼၼ်ႉ ႁဝ်းလူဝ်ႇမီး **IDE (Integrated Development Environment)** ဢၼ်မိူၼ်ၼင်ႇ "ႁွင်ႈၵၢၼ်" တႅမ်ႈ Code ဢေႃႈ။ ၼႂ်းလေႃးၵ C# ၼႆႉ မီးၶိူင်ႈမိုဝ်းလူင် 2 ပိူင် ဢၼ်ၵူၼ်းၸႂ်ႉၼမ်သေပိူၼ်ႈ။

### 1. Visual Studio 2022 (The Giant)
ၼႆႉပဵၼ် IDE ဢၼ်တဵမ်ထူၼ်ႈသေပိူၼ်ႈ။ မၼ်းလူဝ်ႇသႂ်ႇ Memory ၼမ်ဢိတ်းၼိုင်ႈ တေႃႉတႄႉမၼ်းမီးၶိူင်ႈၸွႆး (Features) ၼမ်ၼႃႇ။
* **သႅၼ်း (Edition):** ႁဝ်းတေၸႂ်ႉ **Community Edition** (မၼ်းလၢႆ တႃႇလုၵ်ႈႁဵၼ်း)။
* **Workload:** မိူဝ်ႈလူင်း (Install) ၼၼ်ႉ လူဝ်ႇလိူၵ်ႈ **".NET desktop development"** တေႃႇလဵဝ်တေၸႂ်ႉတႅမ်ႈ Console App ဢေႃႈ။

### 2. Visual Studio Code (The Lightweight)
ပေႃးၶွမ်းလုၵ်ႈႁဵၼ်း ဢမ်ႇပေႃးႁႅင်း (RAM ဢေတ်း)၊ **VS Code** ၼႆႉၶႅမ်ႉသေပိူၼ်ႈ။ မၼ်းၽႂ်း လႄႈ ၸႂ်ႉလႆႈၵူႈၽႃႇသႃႇ။
* **Extension:** လူဝ်ႇလူင်း Extension ဢၼ်ၸိုဝ်ႈဝႃႈ **"C Dev Kit"** ဢမ်ႇၼၼ် **"C#"** ၼႆႉမၼ်းၸင်ႇတေပွင်ႇၸႂ် Code ႁဝ်းဢေႃႈ။



### 3. ၵၢၼ်ၵူတ်ႇထတ်း .NET SDK
မိူဝ်ႈလူင်း IDE ယဝ်ႉယဝ်ႉၼၼ်ႉ ႁဝ်းလူဝ်ႇၵူတ်ႇတူၺ်းဝႃႈ ၼႂ်းၶွမ်းႁဝ်းမီး "ၸၢၵ်ႈ" .NET ယဝ်ႉႁိုဝ်။
1. ပိုတ်ႇ **Terminal** ဢမ်ႇၼၼ် **CMD**။
2. တႅမ်ႈဝႃႈ: `dotnet --version`။
3. ပေႃးမၼ်းဢွၵ်ႇမႃးဝႃႈ `8.0.xxx` ဢမ်ႇၼၼ် `9.0.xxx` ၼႆ ၼၼ်ႉမၼ်းၸႂ်ႉလႆႈယဝ်ႉၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **IDE (Integrated Development Environment):** Program ဢၼ်ၸွႆးတႅမ်ႈ Code, ၵူတ်ႇထတ်း Error လႄႈ ႁဵတ်းႁႂ်ႈ App ႁဵတ်းၵၢၼ်လႆႈ ၼႂ်းတီႈလဵဝ်ၵၼ်။
* **Extension / Plugin:** Program ဢၼ်မႃးသႂ်ႇထႅမ် ႁႂ်ႈ IDE ႁဝ်းမီးၼမ်ႉၵတ်ႉၼမ်ၶိုၼ်ႈ။
* **Terminal / Console:** ၼႃႈလိၵ်ႈသီလမ် ဢၼ်ႁဝ်းၸႂ်ႉတႅမ်ႈ "ၶေႃႈသင်ႇ" (Commands) တႃႇလၢတ်ႈတေႃႇၶိူင်ႈၸၢၵ်ႈ။
* **Community Edition:** သႅၼ်း Program ဢၼ်ပိုတ်ႇပၼ်ၸႂ်ႉလၢႆ တႃႇလုၵ်ႈႁဵၼ်း လႄႈ ၵူၼ်းတႅမ်ႈ App သုၼ်ႇတူဝ်။
