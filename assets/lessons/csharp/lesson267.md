# Lesson 267: "Relational Database Concepts"

တွၼ်ႈတႃႇတေၵဵပ်း ၶေႃႈမုၼ်းႁႂ်ႈမီးပိူင်သၢင်ႈ (Structure) တႅတ်ႈတေႃးၼၼ်ႉ ႁဝ်းတေလႆႈပွင်ႇၸႂ်ၶေႃႈၵႂၢမ်း 4 ဢၼ်ၼႆႉၶႃႈ:

### 1. Table (ပိူင်သၢင်ႈၵဵပ်းၶေႃႈမုၼ်း)
**Table** ပဵၼ်တီႈၵဵပ်းၶေႃႈမုၼ်းၸုမ်းလဵဝ်ၵၼ်။ တူဝ်ယင်ႇမိူၼ်ၼင်ႇ Table `Students` (လုၵ်ႈႁဵၼ်း), `Teachers` (ၶူးသွၼ်), ဢမ်ႇၼၼ် `Courses` (ပပ်ႉႁဵၼ်း) ၶႃႈ။

### 2. Columns (ထႅဝ်တင်ႈ / Fields)
**Columns** မၵ်းမၼ်ႈဝႃႈ "ႁဝ်းတေသိမ်းၶေႃႈမုၼ်းသင်ပႃး?" ၼႂ်း Table ၼၼ်ႉ။
* တူဝ်ယင်ႇ: ၼႂ်း Table `Students` တေမီး Column `FirstName`, `LastName`, `Age`, လႄႈ `VillageName` ၶႃႈ။

### 3. Rows (ထႅဝ်ၼွၼ်း / Records)
**Rows** ပဵၼ် "ၶေႃႈမုၼ်းတႄႉတႄႉ" ၶွင်ၵူၼ်းၼိုင်ႈၵေႃႈ။
* တူဝ်ယင်ႇ: ထႅဝ်ၼွၼ်း (Row) ၼိုင်ႈဢၼ် တေမီးၶေႃႈမုၼ်းဝႃႈ `Sai Mao, 25, Muse` ၶႃႈဢေႃႈ။



---

### 4. Keys (တူဝ်မၢႆၵွင်ႉၵၢႆႇ) - လွင်ႈလမ်ႇလွင်ႈသုတ်း
ၼႂ်း Relational Database ႁဝ်းၸႂ်ႉ **Keys** တႃႇၸႅၵ်ႇလႄႈ ၵွင်ႉၶေႃႈမုၼ်းၶႃႈ:

* **Primary Key (PK):** ပဵၼ် "တူဝ်မၢႆၾၢင်" ဢၼ်ဢမ်ႇမိူၼ်ၽႂ် (Unique) ၼႂ်း Table ၼၼ်ႉ။ မၼ်းၸွႆးႁႂ်ႈႁဝ်းႁႃ Row ၼၼ်ႉလႆႈတႅတ်ႈတေႃး။ (တူဝ်ယင်ႇ: StudentID `101`)။
* **Foreign Key (FK):** ပဵၼ် Primary Key ၶွင် Table ဢၼ်ၼိုင်ႈ ဢၼ်ၵႂႃႇသႂ်ႇဝႆႉၼႂ်း Table ထႅင်ႈဢၼ်ၼိုင်ႈ တႃႇ "ၵွင်ႉ" (Link) ၶဝ်ၸွမ်းၵၼ်ၶႃႈ။

> **တူဝ်ယင်ႇၵၢၼ်ၵွင်ႉၵၢႆႇ (Relationship):**
> * Table `Villages` မီး PK ပဵၼ် **VillageID**.
> * Table `Students` မီး FK ပဵၼ် **VillageID**.
> * ၼႆႉပွင်ႇဝႃႈ လုၵ်ႈႁဵၼ်းၵေႃႉၼႆႉ လုၵ်ႉမႃးတီႈဝၢၼ်ႈ (Village) မၢႆလႂ်ၼၼ်ႉၶႃႈ။



---

### ႁဵတ်းသင်ႁဝ်းၸင်ႇလူဝ်ႇၸႅၵ်ႇပဵၼ် Tables လၢႆလၢႆဢၼ်?
1.  **Reduce Redundancy:** ဢမ်ႇလူဝ်ႇတႅမ်ႈၸိုဝ်ႈဝၢၼ်ႈ "Muse" သွၼ်ႉၵၼ် 1,000 ပွၵ်ႈ၊ ႁဝ်းၵူၺ်းတႅမ်ႈမၢႆ ID `1` သေၵွင်ႉၸူး Table ဝၢၼ်ႈၵေႃႈယဝ်ႉယဝ်ႉၶႃႈ။
2.  **Data Integrity:** သင်ဝၢၼ်ႈ "Muse" လႅၵ်ႈၸိုဝ်ႈမႂ်ႇၼႆၸိုင် ႁဝ်းမႄးတီႈလဵဝ် (Table Villages) ၵေႃႈ ယဝ်ႉတင်း App ၶႃႈ။
3.  **Speed:** ၵၢၼ်သွၵ်ႈႁႃတူဝ်ၼပ်ႉ ID (Keys) မၼ်းဝႆးလိူဝ်ၵၢၼ်သွၵ်ႈႁႃတူဝ်လိၵ်ႈ (String) တင်းၼမ်ၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Schema:** ပိူင်သၢင်ႈတင်းမူတ်းၶွင် Database (Tables, Columns, Relationships)။
* **Unique Constaint:** ပၵ်းၵၢၼ်ဢၼ်ဢမ်ႇပၼ်ၶေႃႈမုၼ်းသွၼ်ႉၵၼ် (မိူၼ် Primary Key)။
* **Relational Model:** ပိူင်သၢင်ႈၵဵပ်း Data ဢၼ်ၵွင်ႉၵၢႆႇၵၼ်လူၺ်ႈ Keys။
* **Normalization:** လၢႆးၸတ်း Tables ယႃႇႁႂ်ႈမီးၶေႃႈမုၼ်းသွၼ်ႉၵၼ်ၼမ်။
