# Lesson 56: "The 'foreach' Loop: Iterating Over Collections and Strings"

ၼႂ်း **Lesson 56** ၼႆႉ ႁဝ်းတေမႃးတူၺ်း **'foreach' loop** ဢေႃႈ။ 
မၼ်းပိူင်ႈၵၼ်တင်း `for` loop တီႈဝႃႈ ႁဝ်းဢမ်ႇလူဝ်ႇမၵ်းမၼ်ႈ "တီႈတႄႇ" ဢမ်ႇၼၼ် "မၢႆၼပ်ႉ" (Counter) ႁင်းမၼ်း။ 
မၼ်းတေ "ဢဝ်" (Fetch) ၶေႃႈမုၼ်း ဢၼ်မီးဝႆႉၼႂ်း တူဝ်လိၵ်ႈ ဢမ်ႇၼၼ် collection တူဝ်ၼိုင်ႈယဝ်ႉ တူဝ်ၼိုင်ႈ ထိုင်တီႈပေႃးသုတ်ႉယဝ်ႉ။

### 1. ပိူင်သၢင်ႈ (Syntax)

```csharp
foreach (type variableName in collection)
{
    // Code တွၼ်ႈၼႆႉ တေႁဵတ်းပၼ်ႇတူင်ႇ ၸွမ်းၼင်ႇႁူဝ်ၼပ်ႉၶေႃႈမုၼ်း
}
```

### 2. တူဝ်ယင်ႇ Code တႄႉတႄႉ
ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းမီးသဵၼ်ႈၸိုဝ်ႈ (Array) ၶွင်လုၵ်ႈႁဵၼ်း 3 ၵေႃႉၶႃႈ။

```csharp
string[] students = { "Sai Mao", "Nang Kham", "Sai Tun" };

foreach (string name in students)
{
    Console.WriteLine("Student: " + name);
}
```

**ပိူင်ၵၢၼ်လႅၼ်ႈမၼ်း:**
* ပွၵ်ႈၵမ်း 1: ၼပ်ႉဢဝ် "Sai Mao" မႃးသႂ်ႇၼႂ်း `name` -> ၼႄလိၵ်ႈ။
* ပွၵ်ႈၵမ်း 2: ၼပ်ႉဢဝ် "Nang Kham" မႃးသႂ်ႇၼႂ်း `name` -> ၼႄလိၵ်ႈ။
* ပွၵ်ႈၵမ်း 3: ၼပ်ႉဢဝ် "Sai Tun" မႃးသႂ်ႇၼႂ်း `name` -> ၼႄလိၵ်ႈ။
* ဝၢႆးၼၼ်ႉ ဢမ်ႇမီးၶေႃႈမုၼ်းယဝ်ႉ -> **ဢွၵ်ႇ Loop ၵမ်းလဵဝ်**။


### 3. ၵၢၼ်ၸႂ်ႉတင်း String (တူဝ်လိၵ်ႈ)
ႁဝ်းၸၢင်ႈၸႂ်ႉ `foreach` တႃႇဢၢၼ်ႇတူဝ်လိၵ်ႈ ၼိုင်ႈတူဝ်ယဝ်ႉၼိုင်ႈတူဝ် ၼႂ်း String လႆႈၶႃႈ။

```csharp
string city = "MUSE";

foreach (char c in city)
{
    Console.WriteLine("Character: " + c);
}
```

### 4. လွင်ႈလူဝ်ႇၾၢင်ႉ (Read-Only)
ၼႂ်း `foreach` ၼၼ်ႉ ႁဝ်း **"ဢမ်ႇၸၢင်ႈမႄး"** (Modify) ၵႃႈၶၼ်ၶေႃႈမုၼ်းၼႂ်း Array ၼၼ်ႉလႆႈၶႃႈ။ မၼ်းမီးဝႆႉတႃႇ "ဢၢၼ်ႇ" (Read) ၵူၺ်း။ သင်ၶႂ်ႈမႄးၶေႃႈမုၼ်း တေလႆႈၸႂ်ႉ `for` loop တႅၼ်းတၢင်ၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Foreach Loop:** ၵၢၼ်ႁဵတ်းပၼ်ႇတူင်ႇ တႃႇဢၢၼ်းၶေႃႈမုၼ်းၼႂ်း Collection (မိူၼ်ၼင်ႇ Array, List)။
* **Collection:** ၸုမ်းၶေႃႈမုၼ်း ဢၼ်သိမ်းဝႆႉလၢႆလၢႆဢၼ် ၼႂ်း Variable ဢၼ်လဵဝ်။
* **Iterating:** ၵၢၼ်ပႆၶၢမ်ႈ ၶေႃႈမုၼ်းၼိုင်ႈဢၼ်ယဝ်ႉ ၼိုင်ႈဢၼ်။
* **Read-Only:** ၵၢၼ်ဢၼ်ဢၢၼ်ႇလႆႈၵူၺ်း ဢမ်ႇပၼ်သုၼ်ႇမႄးလႅၵ်ႈလၢႆႈ။
