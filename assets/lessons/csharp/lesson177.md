# Lesson 177: "Introduction to Generics"

မိူဝ်ႈၵဝ်ႇၼၼ်ႉ (C\# 1.0) ႁဝ်းလူဝ်ႇတႅမ်ႈ Code ၸႅၵ်ႇၵၼ် တႃႇ `int`, `string`, ဢမ်ႇၼၼ် `double`။ ၵူၺ်းၵႃႈ **Generics** မႃးၸွႆးႁႂ်ႈႁဝ်းတႅမ်ႈ Code **"ၵမ်းလဵဝ်ၵူၺ်း"** သေ ၸႂ်ႉလႆႈတင်း **"ၵူႈ Data Type"** ၶႃႈ။

### 1\. တူဝ် **`<T>`** ၼႆႉပဵၼ်သင်?

တူဝ် **`T`** ပွင်ႇဝႃႈ **Type** ၶႃႈ။ မၼ်းပဵၼ် "တူဝ်တႅၼ်း" (Placeholder) ဢၼ်ႁဝ်းပႆႇမၵ်းမၼ်ႈဝႃႈ မၼ်းပဵၼ် Data Type သင်တႄႉတႄႉ။

* မိူဝ်ႈၶူးသၢႆမၢဝ်းၸႂ်ႉ `List<int>` -\> `T` တေပဵၼ် `int`။
* မိူဝ်ႈၶူးၸႂ်ႉ `List<string>` -\> `T` တေပဵၼ် `string`။

### 2\. ႁဵတ်းသင်ၸင်ႇဝႃႈမၼ်းပဵၼ် "Game Changer"?

ဢွၼ်တၢင်းတေမီး Generics ၼၼ်ႉ... ႁဝ်းၵႆႉလႆႈၸႂ်ႉ `object` တႃႇၵဵပ်းၶေႃႈမုၼ်း လၢႆလၢႆပိူင်။ ၵူၺ်းၵႃႈ မၼ်းမီးပၼ်ႁႃ 2 ၶေႃႈ:

1.  **Type Safety (ဢမ်ႇလွတ်ႈၽေး):** ႁဝ်းၸၢင်ႈသူင်ႇ `string` ၶဝ်ႈၵႂႃႇၼႂ်း List ဢၼ်ႁဝ်းၶႂ်ႈသိမ်းၵူၺ်း `int` ၼၼ်ႉလႆႈ၊ သေမၼ်းတေႁဵတ်းႁႂ်ႈ App ၶျႃး (Crash) မိူဝ်ႈလႅၼ်းၵၢၼ်ၶႃႈ။
2.  **Performance (ၸႃႉ):** ၵၢၼ်ပိၼ်ႇ `int` ၵႂႃႇပဵၼ် `object` (Boxing) လႄႈ ပိၼ်ႇၶိုၼ်း (Unboxing) ၼၼ်ႉ ၵိၼ်ႁႅင်း CPU ၼမ်ၶႃႈ။

### 3\. ၽွၼ်းလီၶွင် Generics

* **Code Reusability:** တႅမ်ႈ Logic ၵမ်းလဵဝ်၊ ၸႂ်ႉလႆႈၵူႈ Type။
* **Type Safety:** Compiler တေၵူတ်ႇထတ်းပၼ်ဝႃႈ "ၶေႃႈမုၼ်းမႅၼ်ႈႁႃႉ?" တင်ႈတႄႇမိူဝ်ႈတႅမ်ႈ Code ၶႃႈ။
* **Performance:** ဢမ်ႇလူဝ်ႇႁဵတ်း Boxing/Unboxing ႁဵတ်းႁႂ်ႈ Code လႅၼ်းဝႆးၶႃႈ။

### 4\. တူဝ်ယင်ႇ ၵၢၼ်လူလိၵ်ႈ

မိူဝ်ႈၶူးႁၼ် `<T>` ၼႂ်း Class ဢမ်ႇၼၼ် Method... ႁႂ်ႈၶူးဝူၼ်ႉဝႃႈ: **"ၼႆႉပဵၼ် ပိူင်ႁၢင်ႈ (Template) ဢၼ်တေပိၼ်ႇပဵၼ် Type တႄႉတႄႉ မိူဝ်ႈႁဝ်းၸႂ်ႉမၼ်းၼၼ်ႉယဝ်ႉ"** ၼႆၶႃႈ။

```csharp
// တူဝ်ယင်ႇ: List ဢၼ်သိမ်းလႆႈၵူၺ်း string (Type-Safe)
List<string> shanNames = new List<string>();
shanNames.Add("Sai Mao"); 
// shanNames.Add(2026); // ERROR! (Compiler တေႁႄႉဝႆႉပၼ်ၵမ်းလဵဝ်)
```

-----

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Generics:** ပိူင်တႅမ်ႈ Code ဢၼ်ၸႂ်ႉ Placeholder `<T>` တႅၼ်း Data Type တႄႉတႄႉ။
* **Type Parameter:** တူဝ် `T` ဢၼ်ယူႇၼႂ်း `< >`။
* **Type Safety:** ၵၢၼ်မၼ်ႈၸႂ်ဝႃႈ Data Type မႅၼ်ႈၸွမ်းၼင်ႇဢၼ်မၵ်းမၼ်ႈဝႆႉ။
* **Placeholder:** တီႈယူႇတႅၼ်း ဢၼ်တေထုၵ်ႇလႅၵ်ႈလၢႆႈ မိူဝ်ႈၸႂ်ႉတႄႉ။
