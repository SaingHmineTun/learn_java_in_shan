# Lesson 69: "Iterating through Arrays" (ၵၢၼ်ပၼ်ႇတူင်ႇဢၢၼ်ႇလူ Array)

ၵၢၼ်ဢၢၼ်ႇလူ (Iterate) ၶေႃႈမုၼ်းၼႂ်း Array ၼၼ်ႉ မီးသွင်လၢႆးလူင်ၶႃႈ:

### 1. ၸႂ်ႉ `for` loop (Index-Based)
လၢႆးၼႆႉ ႁဝ်းၸႂ်ႉမၢႆၼပ်ႉ `i` တႃႇၸိၼႄလွၵ်းမၼ်း။ 
မၼ်းမီးဢၢမ်းၼၢတ်ႈၼမ် ယွၼ်ႉႁဝ်းၸၢင်ႈ "မႄး" ၶေႃႈမုၼ်းလႆႈၸွမ်းမၢႆၼပ်ႉၶႃႈ။



```csharp
int[] grades = { 85, 90, 78 };

for (int i = 0; i < grades.Length; i++) 
{
    // ႁဝ်းၸၢင်ႈမႄးၵႃႈၶၼ်လႆႈ
    grades[i] += 5; // ထႅမ်ပၼ် 5 မၢႆ ၵူႈၵေႃႉ
    Console.WriteLine($"Student {i}: {grades[i]}");
}
```

### 2. ၸႂ်ႉ `foreach` loop (Value-Based)
လၢႆးၼႆႉ ငၢႆႈသေ သႅၼ်ႈသႂ် (Clean) သေပိူၼ်ႈ။ မၼ်းတေဢၢၼ်ႇလူပၼ်ၵမ်းလဵဝ် ဢမ်ႇလူဝ်ႇမၢႆၼပ်ႉ `i`။



```csharp
string[] shanFood = { "Khao Soi", "Papaya Salad", "Shan Tofu" };

foreach (string food in shanFood) 
{
    // food တေပဵၼ်ၵႃႈၶၼ် (Value) ၵမ်းလဵဝ်
    Console.WriteLine("Today's menu: " + food);
    
    // မၢႆတွင်း: ၼႂ်း foreach ႁဝ်း "ဢမ်ႇၸၢင်ႈမႄး" ၵႃႈၶၼ်ၼႂ်း Array လႆႈၶႃႈ
}
```

### 3. ပႅၵ်ႇၵၼ်ၼင်ႇႁိုဝ်? (Which one to use?)

| Comparison | **for** loop | **foreach** loop                |
| :--- | :--- |:--------------------------------|
| **Index** | မီးမၢႆၼပ်ႉ `i` (0, 1, 2...) | ဢမ်ႇမီးမၢႆၼပ်ႉ                  |
| **Modification** | မႄးလႅၵ်ႈလၢႆႈၶေႃႈမုၼ်းလႆႈ | **ဢၢၼ်ႇလူလႆႈၵူၺ်း** (Read-Only) |
| **Readability** | တႅမ်ႈယၢဝ်းလိူဝ် | တႅမ်ႈပွတ်း လႄႈ ဢၢၼ်ႇလူငၢႆႈ      |
| **Direction** | ၸၢမ်းပႆထွႆလင်လႆႈ (`i--`) | ပႆၵႂႃႇၽၢႆႇၼႃႈလႆႈတင်းလဵဝ်ၵူၺ်း   |

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Iterate:** ၵၢၼ်ပႆၶၢမ်ႈၶေႃႈမုၼ်းၼိုင်ႈဢၼ်ယဝ်ႉ ၼိုင်ႈဢၼ်။
* **Index-Based:** ၵၢၼ်ၸတ်းၵၢၼ်ၶေႃႈမုၼ်းလူၺ်းမၢႆၼပ်ႉတီႈယူႇ။
* **Value-Based:** ၵၢၼ်ဢဝ်ၵႃႈၶၼ် (Value) မၼ်းမႃးၸႂ်ႉၵမ်းလဵဝ်။
* **Read-Only:** ၵၢၼ်ဢၼ်ပၼ်ဢၢၼ်ႇလူၵူၺ်း ဢမ်ႇပၼ်မႄးလႅၵ်ႈလၢႆႈ။
