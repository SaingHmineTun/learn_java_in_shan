# Lesson 308: "App Lifecycle Management"

ၼႂ်း .NET MAUI မီး Events ပိူင်လူင် 3 ဢၼ် ဢၼ်လၢတ်ႈၼႄငဝ်းလၢႆး App ၶႃႈ:

### 1. OnStart (မိူဝ်ႈတႄႇပိုတ်ႇ App)
ၼႆႉပဵၼ်ယၢမ်းဢၼ် App တႄႇ "ၵိူတ်ႇ" မႃးၶႃႈ။
* **ၵၢၼ်ႁဵတ်း:** ႁဝ်းၸၢင်ႈၸႂ်ႉတွၼ်ႈတႃႇလူ (Load) ၶေႃႈမုၼ်းလုၵ်ႉၼႂ်း Database ဢမ်ႇၼၼ် ၵူတ်ႇထတ်းဝႃႈ User လႆႈ Login ဝႆႉယဝ်ႉႁႃႉ။

### 2. OnSleep (မိူဝ်ႈ App ၵႂႃႇယူႇၽၢႆႇလင် - Background)
မိူဝ်ႈ User ၼဵၵ်းဢွၵ်ႇ App ၵႂႃႇ ၵူၺ်းၵႃႈပႆႇလႆႈပိၵ်ႉ (Close) မၼ်းတဵမ်ထူၼ်ႈၼႆ App တေၶဝ်ႈၵႂႃႇယူႇၼႂ်း "Sleep Mode" ၶႃႈ။
* **ၵၢၼ်ႁဵတ်း:** ၼႆႉပဵၼ်ယၢမ်းဢၼ်လီသုတ်း တွၼ်ႈတႃႇ **သိမ်းၶေႃႈမုၼ်း** (Save Data) ဢၼ် User တိုၵ်ႉတႅမ်ႈဝႆႉ ဢမ်ႇၼၼ် ပိၵ်ႉၵၢၼ်ၸႂ်ႉ GPS တွၼ်ႈတႃႇလူတ်းၵၢၼ်ၸႂ်ႉ Battery ၶႃႈ။



### 3. OnResume (မိူဝ်ႈ User ဝၢႆႇမႃးၸႂ်ႉ App ၶိုၼ်း)
မိူဝ်ႈ User ၼဵၵ်းၶဝ်ႈ App ႁဝ်းၶိုၼ်းလုၵ်ႉတီႈၼႃႈၸေႃး Recent Apps ၼႆ App တေ "တိုၼ်ႇ" မႃးၶႃႈ။
* **ၵၢၼ်ႁဵတ်း:** ႁဝ်းၸၢင်ႈၸႂ်ႉတွၼ်ႈတႃႇ "Refresh Data" (မိူၼ်ၼင်ႇ ၵူတ်ႇထတ်းၶၢဝ်ႇမႂ်ႇ) ႁႂ်ႈ User ႁၼ်ၶေႃႈမုၼ်းဢၼ်မႂ်ႇသုတ်းယူႇတႃႇသေႇၶႃႈ။

---

### 4. ႁဵတ်းသင်လွင်ႈၼႆႉၸင်ႇလမ်ႇလွင်ႈ?
1.  **Data Loss Prevention:** သင် User တိုၵ်ႉတႅမ်ႈလိၵ်ႈဝႆႉသေ App တူၵ်း (Crash) ဢမ်ႇၼၼ် OS ပိၵ်ႉ App ႁဝ်းပႅတ်ႈၼႆ သင်ႁဝ်းသိမ်းဝႆႉၼႂ်း `OnSleep` ၶေႃႈမုၼ်းၶဝ်တေဢမ်ႇႁၢႆၶႃႈ။
2.  **Resource Management:** ၸွႆးႁႂ်ႈ App ႁဝ်းဢမ်ႇၵိၼ် RAM လႄႈ Battery ၼမ်လိူဝ်ပူၼ်ႉတီႈ မိူဝ်ႈႁဝ်းဢမ်ႇလႆႈၸႂ်ႉမၼ်းၶႃႈ။
3.  **User Experience:** ၸွႆးႁႂ်ႈ User ႁူႉဝႃႈ App ႁဝ်း "ဝႆး" ယွၼ်ႉမၼ်းပႂ်ႉဝႆႉသေ ႁၢင်ႈႁႅၼ်းတႃႇႁဵတ်းၵၢၼ်ယူႇ တႃႇသေႇၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Lifecycle:** ၸၼ်ႉၵၢၼ်လႅၵ်ႈပိၼ်ႇငဝ်းလၢႆးၶွင် App တႄႇတီႈၵိူတ်ႇ တေႃႇပေႃးတၢႆ (ပိၵ်ႉ)။
* **Background:** ငဝ်းလၢႆးဢၼ် App တိုၵ်ႉလႅၼ်ႈယူႇ ၵူၺ်းၵႃႈ User ဢမ်ႇႁၼ်ၼိူဝ်ၼႃႈၸေႃး။
* **Foreground:** ငဝ်းလၢႆးဢၼ် App တိုၵ်ႉၼႄဝႆႉၼိူဝ်ၼႃႈၸေႃး လႄႈ User တိုၵ်ႉၸႂ်ႉယူႇ။
* **State Persistence:** ၵၢၼ်သိမ်း "ငဝ်းလၢႆး" ၶွင် App ႁႂ်ႈမိူၼ်ၵဝ်ႇ မိူဝ်ႈဝၢႆႇမႃးပိုတ်ႇၶိုၼ်း။
