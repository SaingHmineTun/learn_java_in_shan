# Lesson 55: "The 'for' Loop" (ၵၢၼ်ႁဵတ်းပၼ်ႇတူင်ႇ ၸႂ်ႉမၢႆၼပ်ႉ)

**'for' loop** ၼႆႉ လႅပ်ႈတွၼ်ႈတႃႇၵၢၼ်ႁဵတ်းပၼ်ႇတူင်ႇ ဢၼ်ႁဝ်းႁူႉဝႆႉယဝ်ႉဝႃႈ "တေႁဵတ်းလၢႆပွၵ်ႈ" (မိူၼ်ၼင်ႇ ၼပ်ႉ 1 ထိုင် 10) ၶႃႈ။

### 1. ပိူင်သၢင်ႈ (Anatomy of For Loop)



ၼႂ်းၶွပ်ႇမူၼ်း `( )` ၶွင် for loop မီး 3 တွၼ်ႈ ဢၼ်ၸႅၵ်ႇၵၼ်တင်း Semicolon (`;`) ၶႃႈ:
1.  **Initialization (တီႈတႄႇ):** သၢင်ႈ Variable ၼပ်ႉမၢႆ (Counter)။
2.  **Condition (ၶေႃႈထၢမ်):** ၵူတ်ႇထတ်းဝႃႈ ထုၵ်ႇလီသိုပ်ႇပၼ်ႇတူင်ႇႁႃႉ?
3.  **Iterator (ၵၢၼ်လႅၵ်ႈလၢႆႈ):** ထႅမ် ဢမ်ႇၼၼ် ယွမ်း ၵႃႈၶၼ် Counter ဝၢႆးသေမၼ်း ႁဵတ်းယဝ်ႉၼိုင်ႈႁွပ်ႈ။

### 2. တူဝ်ယင်ႇ Code တႄႉတႄႉ

```csharp
// ၼပ်ႉ 1 ထိုင် 5
for (int i = 1; i <= 5; i++)
{
    Console.WriteLine("Iteration: " + i);
}
```

**ပိူင်ၵၢၼ်လႅၼ်ႈမၼ်း:**
* **Step 1:** `int i = 1` (ႁဵတ်းၵၢၼ်ပွၵ်ႈလဵဝ် မိူဝ်ႈတႄႇ)။
* **Step 2:** ၵူတ်ႇထတ်း `i <= 5` (True)။
* **Step 3:** ႁဵတ်းၵၢၼ်ၼႂ်း `{ }` (ၼႄလိၵ်ႈ Iteration: 1)။
* **Step 4:** ႁဵတ်း `i++` (i ပဵၼ် 2)။
* **Step 5:** ပၼ်ႇတူင်ႇၵႂႃႇတီႈ Step 2 ၶိုၼ်း တေႃႇထိုင် `i` ပဵၼ် 6 ၸင်ႇၵိုတ်း (Exit Condition)။

### 3. 'for' vs 'while'
* ၸႂ်ႉ **'while'**: မိူဝ်ႈႁဝ်း **"ဢမ်ႇႁူႉ"** ဝႃႈ တေလႆႈပၼ်ႇတူင်ႇလၢႆပွၵ်ႈ (မိူၼ်ၼင်ႇ ပႂ်ႉ User တႅမ်ႈ Password ထိုင်ပေႃးမႅၼ်ႈ)။
* ၸႂ်ႉ **'for'**: မိူဝ်ႈႁဝ်း **"ႁူႉၸႅင်ႈ"** ဝႃႈတေလႆႈႁဵတ်းလၢႆပွၵ်ႈ (မိူၼ်ၼင်ႇ ၼႄသဵၼ်ႈၸိုဝ်ႈလုၵ်ႈႁဵၼ်း 20 ၵေႃႉ)။

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **For Loop:** ၵၢၼ်ႁဵတ်းပၼ်ႇတူင်ႇ ဢၼ်မၵ်းမၼ်ႈႁူဝ်ၼပ်ႉဝႆႉလီငၢမ်း။
* **Initialization:** ၵၢၼ်သၢင်ႈ variable တႃႇတႄႇၼပ်ႉ။
* **Iterator:** တွၼ်ႈဢၼ်လႅၵ်ႈလၢႆႈၵႃႈၶၼ် Counter (မိူၼ်ၼင်ႇ `i++`, `i--`, `i += 2`)။
* **Counter-Based:** ၵၢၼ်ၸႂ်ႉမၢႆၼပ်ႉ (Variable i) တႃႇၵုမ်းထိင်း Loop။
* **Condition:** ၶေႃႈထၢမ်ဢၼ်မၵ်းမၼ်ႈဝႃႈ Loop တေၵိုတ်းတီႈလႂ်။
