# Lesson 286: "Layouts and Partial Views"

### 1. Layouts (ပိူင်သၢင်ႈပိုၼ်ႉထၢၼ်)
**Layout** ပဵၼ်ၾၢႆႇ Razor (`_Layout.cshtml`) ဢၼ်သိမ်း Code HTML ဢၼ်သွၼ်ႉၵၼ် (မိူၼ်ၼင်ႇ Navigation Bar, CSS Links) ဝႆႉတီႈလဵဝ်ၶႃႈ။

* **`@RenderBody()`**: ၼႆႉပဵၼ် Code ဢၼ်လမ်ႇလွင်ႈသုတ်းၼႂ်း Layout ၶႃႈ။ မၼ်းပဵၼ် "တီႈဝၢင်း" (Placeholder) ဢၼ်တေဢဝ်ၶေႃႈမုၼ်းလုၵ်ႉတီႈ View တၢင်ႇဢၼ် (မိူၼ်ၼင်ႇ Index, About) မႃးသႂ်ႇဝႆႉၼႂ်းၵႄႈၵၢင် Layout ၶႃႈ။



### 2. Partial Views (ၼႃႈတႃသုၼ်ႇလဵၵ်ႉ)
**Partial View** ပဵၼ်ၾၢႆႇ Razor လဵၵ်ႉလဵၵ်ႉ ဢၼ်ႁဝ်းၸႅၵ်ႇဢွၵ်ႇမႃး တႃႇႁႂ်း Code ႁဝ်းလူငၢႆႈ။ မိူၼ်ၼင်ႇ "တုမ်ႇသွၵ်ႈႁႃ" (Search Bar) ဢမ်ႇၼၼ် "ၶၢတ်ႉၼႄၶၢဝ်ႇ" (News Card) ဢၼ်ႁဝ်းၶႂ်ႈဢဝ်ၵႂႃႇၸႂ်ႉလၢႆလၢႆတီႈၶႃႈ။

* **လၢႆးႁွင်ႉၸႂ်ႉ:**
  ```razor
  <partial name="_ShanLanguageSwitcher" />
  ```
  *(ၵႆႉတႄႇၸိုဝ်ႈၾၢႆႇလူၺ်ႈတူဝ် `_` တွၼ်ႈတႃႇႁႂ်ႈႁူႉဝႃႈမၼ်းပဵၼ် Partial ၶႃႈ)*

### 3. ViewStart လႄႈ ViewImports
* **`_ViewStart.cshtml`**: ၾၢႆႇၼႆႉတေသင်ႇႁႂ်ႈ View တင်းမူတ်းၸႂ်ႉ Layout တူဝ်လဵဝ်ၵၼ် ႁင်းၵူၺ်းမၼ်းၶႃႈ။
* **`_ViewImports.cshtml`**: တီႈဝၢင်း Code ဢၼ်သွၼ်ႉၵၼ် မိူၼ်ၼင်ႇ `@using MyShanApp.Models` တႃႇႁႂ်း View ၵူႈဢၼ် ႁူႉၸၵ်း Class ႁဝ်းၶႃႈ။

---

### ႁဵတ်းသင်လွင်ႈၼႆႉၸင်ႇလမ်ႇလွင်ႈတွၼ်ႈတႃႇ Shan UI?
1. **Consistency:** ႁဵတ်းႁႂ်ႈ User ၸႂ်ႉ App ႁဝ်းငၢႆႈ ယွၼ်ႉ Menu လႄႈ သႅၼ်းလိၵ်ႈတႆး (Font) မီးႁၢင်ႈမိူၼ်ၵၼ်ၵူႈၼႃႈလိၵ်ႈၶႃႈ။
2. **Easy Updates:** သင်ႁဝ်းၶႂ်ႈလႅၵ်ႈပိၼ်ႇ "Logo ႁူင်းႁဵၼ်း" ၼိူဝ် Header ၼႆ ႁဝ်းၵူၺ်းလႆႈမႄးတီႈလဵဝ် (ၼႂ်း _Layout) သေ ၼႃႈလိၵ်ႈၵူႈဢၼ် တေလႅၵ်ႈပိၼ်ႇၸွမ်းၵမ်းလဵဝ်ၶႃႈ။
3. **Reusable Components:** ႁဝ်းၸၢင်ႈသၢင်ႈ Partial View တွၼ်ႈတႃႇ "Keyboard တႆး" သေ ဢဝ်ၵႂႃႇသႂ်ႇဝႆႉၼႂ်းၼႃႈလိၵ်ႈၵူႈဢၼ် ဢၼ်လူဝ်ႇတႅမ်ႈလိၵ်ႈတႆးလႆႈငၢႆႈငၢႆႈၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Layout:** ၾၢႆႇငဝ်ႈႁၢၵ်ႈ ဢၼ်မၵ်းမၼ်ႈႁၢင်ႈၽၢင်တင်းမူတ်းၶွင် Website။
* **Partial View:** UI ဢွၼ်ႇဢွၼ်ႇ ဢၼ်ၸၢင်ႈဢဝ်ၵႂႃႇၸႂ်ႉ လႆႈလၢႆလၢႆတီႈ သွၼ်ႉၵၼ် (Reusable UI component)။
* **RenderBody:** Method ၼႂ်း Layout ဢၼ်မႃးၼႄၶေႃႈမုၼ်းလုၵ်ႉတီႈ View။
* **View Hierachy:** ပိူင်သၢင်ႈၸၼ်ႉၶွင် View လုၵ်ႉတီႈ Layout လူင်းၵႂႃႇထိုင် Partial။
