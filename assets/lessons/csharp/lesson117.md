# Lesson 117: "The Principle of Encapsulation: Data Hiding and Security"

**Encapsulation** (ဢႅၼ်ႇ-ၶႅပ်ႉ-သူႇ-လေး-သျိၼ်း) ပွင်ႇဝႃႈ "ၵၢၼ်ႁေႃႇ" ၶေႃႈမုၼ်း (Fields) လႄႈ လၢႆးႁဵတ်းသၢင်ႈ (Methods) ဝႆႉၼႂ်း "ႁွင်ႈ" (Unit/Class) လဵဝ်ၵၼ် သေယဝ်ႉ **ပိၵ်ႉလပ်ႉ** တွၼ်ႈဢၼ်သုၵ်ႉယုင်ႈ ဝႆႉၶႃႈ။



### 1. ႁဵတ်းသင်ႁဝ်းလႆႈ "လပ်ႉ" ၶေႃႈမုၼ်း?
ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းတႅမ်ႈ App "ငိုၼ်းၾၢၵ်ႇယေးငိုၼ်း" (Bank Account)။ သင်ဝႃႈ Field `balance` (ငိုၼ်းၵိုၵ်းပၼ်ႇ) ၼၼ်ႉပဵၼ် `public`... ၽႂ်ၵေႃႈယႃႇ ၸၢင်ႈမႃးမႄးလႆႈၸိူင်ႉၼႆၶႃႈ:
* `account.balance = 100000000;` (မႄးထႅမ်ငိုၼ်းၸွမ်းၸႂ်)
* `account.balance = -500;` (ငိုၼ်းဢေႇလိူဝ်သုၼ် မၼ်းဢမ်ႇထုၵ်ႇပိူင်)

ၵၢၼ်ၸႂ်ႉ **Encapsulation** တေၸွႆးႁႄႉၵၢင်ႈ ပၼ်ႁႃၸိူဝ်းၼႆႉၶႃႈ။

### 2. လွၼ်ႉမၼ်း 2 ယၢင်ႇ
1.  **Data Hiding (ၵၢၼ်သိူင်ႇၶေႃႈမုၼ်း):** ႁဝ်းႁဵတ်းပဵၼ် `private` တႃႇဢမ်ႇပၼ်ၵေႃႉၼွၵ်ႈမႄးလႆႈၵမ်းလဵဝ်။
2.  **Controlled Access (ၵၢၼ်ပၼ်သုၼ်ႇၶဝ်ႈၸႂ်ႉ):** ႁဝ်းပၼ်ၵေႃႉၼွၵ်ႈ ၸႂ်ႉလႆႈ "သဵၼ်ႈတၢင်း" (Methods) ဢၼ်ႁဝ်းမၵ်းမၼ်ႈဝႆႉ (မိူၼ်ၼင်ႇ `Deposit()` ဢမ်ႇၼၼ် `Withdraw()`) ၼၼ်ႉၵူၺ်းၶႃႈ။

### 3. တၢင်းလီၶွင် Encapsulation
* **Security (လွင်ႈၵႅတ်ႇၶႄ):** ၶေႃႈမုၼ်းဢမ်ႇၸၢင်ႈထုၵ်ႇမႄးလူၺ်း လွင်ႈဢမ်ႇထုၵ်ႇမႅၼ်ႈ (Accidental damage)။
* **Flexibility (လွင်ႈလႅၵ်ႈလၢႆႈငၢႆႈ):** ႁဝ်းၸၢင်ႈမႄး Logic ၼႂ်း Class ႁဝ်းလႆႈ တင်ႈတႄႇမိူဝ်ႈလႂ်ၵေႃႈယႃႇ လူၺ်းဢမ်ႇႁဵတ်းႁႂ်ႈ Code ၽၢႆႇၼွၵ်ႈ "လူႉလႅဝ်" (Break) ၶႃႈ။
* **Complexity Management (ၵုမ်းထိင်းလွင်ႈယုင်ႈယၢင်ႈ):** ၵေႃႉၸႂ်ႉ Code ႁဝ်း ဢမ်ႇလူဝ်ႇႁူႉဝႃႈ ၼႂ်းမၼ်း မၵ်းမၼ်ႈဝႆႉၸိူင်ႉႁိုဝ်၊ ၶဝ်ၵူၺ်းလူဝ်ႇႁူႉဝႃႈ "ႁွင်ႉၸႂ်ႉၸိူင်ႉၼႆ ယဝ်ႉယဝ်ႉ" ၼႆၵူၺ်းၶႃႈ။

### 4. တူဝ်ယင်ႇၼႂ်းၸူဝ်ႈပၢၼ်တႄႉ
မိူၼ်တင်း **"ၸၢၵ်ႈၵႃႇၾေး"** (Coffee Machine)။
* **Interface:** ႁဝ်းႁၼ် "တုမ်ႇၼဵၵ်း" (Buttons)။ ႁဝ်းၼဵၵ်းတုမ်ႇယဝ်ႉ ၵႃႇၾေးၵေႃႈဢွၵ်ႇမႃး။
* **Encapsulation:** ပိူင်သႂ်ႇၼမ်ႉ၊ ပိူင်သၢင်ႈတၢင်းႁူႉ၊ ပိူင်ၵူတ်ႇထတ်း တၢင်းမႆႈ... ၸိူဝ်းၼၼ်ႉထုၵ်ႇ "ႁေႃႇ" ဝႆႉၼႂ်း ၸၢၵ်ႈၼၼ်ႉယဝ်ႉၶႃႈ။ သင်ႁဝ်းၶႂ်ႈၵိၼ်ၼႆ ႁဝ်းဢမ်ႇလူဝ်ႇၵႂႃႇမႄးသင်လႄႈ မၼ်းတေဢဝ် ၵႃႇၾေးၼၼ်ႉ ပၼ်ႁဝ်းလႆႈၶႃႈ။

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Encapsulation:** ၵၢၼ်ႁေႃႇ ၶေႃႈမုၼ်း လႄႈ တၢင်းႁဵတ်းသၢင်ႈ ဝႆႉႁူမ်ႈၵၼ်။
* **Data Hiding:** ၵၢၼ်ပိၵ်ႉလပ်ႉ Implementation Details တႃႇလွင်ႈၵႅတ်ႇၶႄ။
* **Black Box:** ၶေႃႈၵႂၢမ်းမၢႆထိုင် Object ဢၼ်ႁဝ်းႁူႉ "ၵၢၼ်ႁဵတ်း" မၼ်း ၵူၺ်းၵႃႈ ဢမ်ႇလူဝ်ႇႁူႉ "လၢႆးႁဵတ်း" ၼႂ်းမၼ်း။
* **Access Modifiers:** ၶေႃႈၵႂၢမ်း (public, private) ဢၼ်ၸႂ်ႉတႃႇႁဵတ်း Encapsulation။
