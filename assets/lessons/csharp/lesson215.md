# Lesson 215: "Introduction to LINQ: Why Query Data Inside C#?"

**LINQ** ယေႃႇမႃးလုၵ်ႉတီႈ **Language Integrated Query** ၶႃႈ။ 
မၼ်းပဵၼ်ၶိူင်ႈမိုဝ်း ဢၼ် Microsoft သၢင်ႈမႃး တႃႇႁႂ်ႈႁဝ်း "ထၢမ်ႁႃၶေႃႈမုၼ်း" လုၵ်ႉတီႈ Array, List, XML, ဢမ်ႇၼၼ် Database လႆႈလူၺ်ႈၸႂ်ႉ "လိၵ်ႈ C#" ၵမ်းလဵဝ်ၶႃႈ။

### 1. ပၼ်ႁႃမိူဝ်ႈၵဝ်ႇ (The Old Way)
မိူဝ်ႈဢမ်ႇပႆႇမီး LINQ ၼၼ်ႉ သင်ႁဝ်းမီး List လုၵ်ႈႁဵၼ်း 100 ၵေႃႉ သေၶႂ်ႈႁႃ "ၵေႃႉဢၼ် ဢႃႇယုယႂ်ႇလိူဝ် 20" ႁဝ်းတေလႆႈတႅမ်ႈ:
1.  သၢင်ႈ List မႂ်ႇတႃႇသိမ်းၽွၼ်းလႆႈ။
2.  တႅမ်ႈ `foreach` loop တႃႇပၼ်ႇတူၺ်းၵူႈၵေႃႉ။
3.  တႅမ်ႈ `if` တႃႇၵူတ်ႇထတ်းဢႃႇယု။
4.  ဢဝ်ၵူၼ်းဢၼ်မႅၼ်ႈၸွမ်းပၵ်းၵၢၼ်သေ ထႅမ်သႂ်ႇၼႂ်း List မႂ်ႇ။



### 2. LINQ ပဵၼ်သင်? (The LINQ Way)
LINQ ႁဵတ်းႁႂ်ႈ Code 4-5 ထႅဝ် ၸိူဝ်းၼၼ်ႉ ႁႂ်ႈပဵၼ် **ၼိုင်ႈထႅဝ်** ၶႃႈ! မၼ်းၸွႆးႁႂ်ႈႁဝ်းလၢတ်ႈဝႃႈ "ႁဝ်းၶႂ်ႈလႆႈသင်" (What we want) တႅၼ်းဢၼ်တေလႆႈလၢတ်ႈဝႃႈ "ႁဵတ်းၸိူင်ႉႁိုဝ်ၸင်ႇတေလႆႈ" (How to do it)။

```csharp
// လၢႆးတႅမ်ႈ LINQ ပွတ်းๆ
var adults = students.Where(s => s.Age > 20);
```

### 3. ႁဵတ်းသင်ႁဝ်းၸင်ႇလူဝ်ႇၸႂ်ႉ LINQ?
1.  **Readability:** Code ႁဝ်းလူငၢႆႈလိူဝ်ၵဝ်ႇၼမ်တႄႉတႄႉ။ ၵူၼ်းတႅမ်ႈ Code တၢင်ႇၵေႃႉမႃးလူၵေႃႉ ပွင်ႇၸႂ်ၵမ်းလဵဝ်ဝႃႈ ႁဝ်းတိုၵ်ႉႁႃသင်ယူႇ။
2.  **Type Safety:** LINQ ႁူႉၸၵ်း Type ၶေႃႈမုၼ်းႁဝ်းလီ (Int, String, Student)၊ သင်ႁဝ်းတႅမ်ႈၽိတ်း Compiler တေၼႄပၼ် Error ၵမ်းလဵဝ်။
3.  **Unified Syntax:** ဢမ်ႇဝႃႈႁဝ်းတေႁႃၶေႃႈမုၼ်းၼႂ်း **List**, **XML File**, ဢမ်ႇၼၼ် **SQL Database**... ႁဝ်းၸႂ်ႉလၢႆးတႅမ်ႈ LINQ မိူၼ်ၵၼ်တင်းမူတ်းၶႃႈ!
4.  **IntelliSense:** ၼႂ်း Visual Studio မၼ်းတေၸွႆးၼႄ Method ၸိူဝ်းဢၼ်ႁဝ်းၸႂ်ႉလႆႈ ႁဵတ်းႁႂ်ႈတႅမ်ႈ Code ဝႆးၶႃႈ။

### 4. LINQ ၸၢင်ႈၸႂ်ႉတင်းသင်လႆႈလၢႆလၢႆ?
* **LINQ to Objects:** ၸႂ်ႉတင်း Arrays, Lists (ဢၼ်ႁဝ်းတေႁဵၼ်းၼႂ်းတွၼ်ႈၼႆႉ)။
* **LINQ to SQL / Entities:** ၸႂ်ႉတင်း Database။
* **LINQ to XML:** ၸႂ်ႉတင်း File XML။
* **LINQ to JSON:** ၸႂ်ႉတင်း Data ဢၼ်လုၵ်ႉတီႈ Internet မႃး။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Query:** ၵၢၼ်ထၢမ်ႁႃၶေႃႈမုၼ်း လုၵ်ႉတီႈၵွင်ၶေႃႈမုၼ်း (Data Source)။
* **Data Source:** တီႈၵဵပ်းၶေႃႈမုၼ်း (မိူၼ်ၼင်ႇ List ဢမ်ႇၼၼ် Array)။
* **Declarative Programming:** လၢႆးတႅမ်ႈ Code ဢၼ်ၼႄ "ၽွၼ်းလႆႈဢၼ်ၶႂ်ႈလႆႈ" တႅၼ်းဢၼ်တေၼႄ "ၶၼ်တွၼ်ႈၵၢၼ်ႁဵတ်း"။
* **Filtering:** ၵၢၼ်လိူၵ်ႈဢဝ်ၶေႃႈမုၼ်း ဢၼ်မႅၼ်ႈၸွမ်းပၵ်းၵၢၼ် ဢၼ်ႁဝ်းမၵ်းမၼ်ႈဝႆႉ။
