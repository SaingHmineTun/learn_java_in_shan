# Lesson 33: "Operator Precedence and Associativity"

ၼႂ်းပၢႆးၼပ်ႉ ႁဝ်းမီးပိူင် **PEMDAS** (Parentheses, Exponents, Multiplication, Division, Addition, Subtraction)။ 
ၼႂ်း C# ၵေႃႈမီးပိူင်မိူၼ်ၵၼ် ဢၼ်ႁဝ်းႁွင်ႉဝႃႈ **Precedence** (လွင်ႈပၼ်သုၼ်ႇဢွၼ်ႇတၢင်း) ဢေႃႈ။

### 1. တူဝ်ယင်ႇလွင်ႈယုင်ႈယၢပ်ႈ
တူၺ်း Code ထႅဝ်ၼႆႉၶႃႈ:
```csharp
int result = 5 + 2 * 10; 
```
* ပေႃးႁဝ်းဢဝ် `5 + 2` ဢွၼ်ႇတၢင်း -> `7 * 10 = 70`
* ပေႃးႁဝ်းဢဝ် `2 * 10` ဢွၼ်ႇတၢင်း -> `5 + 20 = 25`
  **ၽွၼ်းလႆႈတႄႉတႄႉၼႂ်း C# ပဵၼ်: 25** (ယွၼ်ႉ `*` မီး Precedence သုင်သေ `+`)။



### 2. ပိူင်လွၼ်ႉမၼ်း (Order of Operations)

| Order | Operators            | Meaning |
| :--- |:---------------------| :--- |
| **1** | `( )`                | **Parentheses** (ႁဵတ်းၼႂ်းဝူတ်ႇဢွၼ်ႇတၢင်းသုတ်း) |
| **2** | `++`, `--`, `!`, `~` | Unary Operators (ထႅမ်/ယွမ်း/ပိၼ်ႇ ဢွၼ်ႇတၢင်း) |
| **3** | `*`, `/`, `%`        | Multiplicative (ၼပ်ႉ/ႁၢၼ်) |
| **4** | `+`, `-`             | Additive (ထႅမ်/ယွမ်း) |
| **5** | `<<`, `>>`           | Shift (ယွၼ်ႇ Bit) |
| **6** | `<`, `>`, `==`, `!=` | Comparison (တဵၵ်းၵၼ်) |
| **7** | `&&`, `\|\|`         | Logical (ပၢႆးဝူၼ်ႉ) |
| **8** | `=`, `+=`, `-=`      | Assignment (ပၼ်ၵႃႈၶၼ် - **ႁဵတ်းဝၢႆးလင်သေပိူၼ်ႈ**) |

### 3. Associativity (ၵၢၼ်ၼပ်ႉသၢႆႉၵႂႃႇၶႂႃ)
ပေႃးၶိူင်ႈမၢႆမီး Precedence မိူၼ်ၵၼ် (မိူၼ်ၼင်ႇ `*` လႄႈ `/`)၊ C# တေၼပ်ႉလုၵ်ႉတီႈ **ၽၢႆႇသၢႆႉ ၵႂႃႇ ၽၢႆႇၶႂႃ** (Left-to-Right) ဢေႃႈ။
* **တူဝ်ယင်ႇ:** `10 / 2 * 5`
    * `10 / 2 = 5`
    * `5 * 5 = 25` (ၼပ်ႉလုၵ်ႉၽၢႆႇသၢႆႉမႃး)

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Precedence:** လွင်ႈပၼ်သုၼ်ႇ (Priority) ဝႃႈၶိူင်ႈမၢႆလႂ် တေလႆႈႁဵတ်းၵၢၼ်ဢွၼ်ႇတၢင်း။
* **Associativity:** လွၵ်းလၢႆးဢၼ် C# ၸႂ်ႉ မိူဝ်ႈမီးၶိူင်ႈမၢႆ Level မိူၼ်ၵၼ် (သၢႆႉၵႂႃႇၶႂႃ ဢမ်ႇၼၼ် ၶႂႃမႃးသၢႆႉ)။
* **Parentheses ():** ၶိူင်ႈမိုဝ်းဢၼ်မီးဢၢမ်းၼၢတ်ႈ သုင်သေပိူၼ်ႈ တႃႇ "တဵၵ်း" ႁႂ်ႈ C# ၼပ်ႉသၢင်ႈတွၼ်ႈၼၼ်ႉဢွၼ်ႇတၢင်း။
* **Expression:** ထႅဝ် Code ဢၼ်မီးၵၢၼ်ၼပ်ႉသၢင်ႈသေ သူင်ႇၽွၼ်းလႆႈမႃးပဵၼ်ၵႃႈၶၼ် (Value)။
