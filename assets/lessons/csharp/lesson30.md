# Lesson 30: "Comparison Operators" (ၶိူင်ႈမၢႆတဵၵ်းၵၼ်)

မိူဝ်ႈႁဝ်းၸႂ်ႉ Comparison Operators ၼပ်ႉသၢင်ႈယဝ်ႉ၊ 
ၽွၼ်းလႆႈ (Result) မၼ်းတေပဵၼ် **Boolean** (`true` ဢမ်ႇၼၼ် `false`) တႃႇသေႇၶႃႈ။

### 1. ၶိူင်ႈမၢႆတႅၵ်ႈၵၼ် 6 ပိူင်

| Operator | Name | Meaning                  | Example (x=10, y=5) | Result |
| :--- | :--- |:-------------------------| :--- | :--- |
| **`==`** | Equal to | တႅတ်ႉတႅတ်ႉ/မိူၼ်ၵၼ်      | `x == y` | **false** |
| **`!=`** | Not equal to | ဢမ်ႇတႅတ်ႉ/ဢမ်ႇမိူၼ်ၵၼ်   | `x != y` | **true** |
| **`>`** | Greater than | ယႂ်ႇသေ                   | `x > y` | **true** |
| **`<`** | Less than | လဵၵ်းသေ                  | `x < y` | **false** |
| **`>=`** | Greater or equal | ယႂ်ႇသေ ဢမ်ႇၼၼ် မိူၼ်ၵၼ်  | `10 >= 10` | **true** |
| **`<=`** | Less or equal | လဵၵ်းသေ ဢမ်ႇၼၼ် မိူၼ်ၵၼ် | `5 <= 10` | **true** |



### 2. လွင်ႈလူဝ်ႇၾၢင်ႉ: `=` vs `==`
ၼႆႉပဵၼ်လွင်ႈဢၼ်လုၵ်ႈႁဵၼ်းမႂ်ႇ ၵႆႉတႅမ်ႈၽိတ်းသေပိူၼ်ႈၶႃႈ:
* **`=` (Assignment):** ၸႂ်ႉတႃႇ "သႂ်ႇ" ၵႃႈၶၼ် (တူဝ်ယင်ႇ: `x = 5;`)။
* **`==` (Comparison):** ၸႂ်ႉတႃႇ "ထၢမ်" ဝႃႈမၼ်းမိူၼ်ၵၼ်ႁိုဝ် (တူဝ်ယင်ႇ: `if (x == 5)`)။

### 3. ၵၢၼ်တႅၵ်ႈ String (တူဝ်လိၵ်ႈ)
ႁဝ်းၸၢင်ႈၸႂ်ႉ `==` လႄႈ `!=` တႃႇတႅၵ်ႈ String လႆႈၸွမ်းၼင်ႇမၢႆၼပ်ႉယဝ်ႉ။ 
ၵူၺ်းၵႃႈ မၼ်းတေတူၺ်း "တူဝ်လူင်/တူဝ်လဵၵ်း" (Case-sensitive) ၸွမ်းဢေႃႈ။
* `"Mao" == "Mao"` -> **true**
* `"Mao" == "mao"` -> **false**

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Comparison:** ၵၢၼ်တႅၵ်ႈၵႃႈၶၼ် 2 ဢၼ် တႃႇႁႃလွင်ႈပႅၵ်ႇပိူင်ႈ။
* **Equality:** လွင်ႈတႅတ်ႉတႅတ်ႉ/မိူၼ်ၵၼ် (==)။
* **Inequality:** လွင်ႈဢမ်ႇတႅတ်ႉ/ဢမ်ႇမိူၼ်ၵၼ် (!=)။
* **Boolean Result:** ၽွၼ်းလႆႈ ဢၼ်မီးလႆႈ ၵႃႈၶၼ် 2 လွင်ႈ (True/False)။
* **Relational Operators:** ၸိုဝ်ႈထႅင်ႈပိူင်ၼိုင်ႈ ၶွင် Comparison Operators။
