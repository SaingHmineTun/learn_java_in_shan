# Lesson 154: "The Need for Abstraction"

**Abstraction** (ၵၢၼ်ၸႅၵ်ႇလႅၼ်းလွၼ်ႉမၼ်း) ပွင်ႇဝႃႈ **"ၵၢၼ်ၼႄၵူၺ်း တွၼ်ႈဢၼ်လမ်ႇလွင်ႈ သေ လပ်ႉဝႆႉ တွၼ်ႈဢၼ်ယုင်ႈယၢင်ႈ"** ၼၼ်ႉယဝ်ႉၶႃႈ။

### 1. တူဝ်ယင်ႇၼႂ်းၸူဝ်ႈပၢၼ်တႄႉ (The Coffee Machine)
ဝူၼ်ႉတူၺ်းဝႃႈ ၶူးသၢႆမၢဝ်း ၵႂႃႇၸႂ်ႉၶိူင်ႈၸုၵ်းၵေႃႇၾီႇ (Coffee Machine)။
* **တွၼ်ႈဢၼ်ၶူးႁၼ်:** မီးပုမ်ႇ "Espresso" လႄႈ "Latte"။ ၶူးၵူၺ်းလူဝ်ႇၼိပ်ႇပုမ်ႇၵူၺ်း။
* **တွၼ်ႈဢၼ်ၶူးဢမ်ႇႁၼ် (Hidden Implementation):** လၢႆးဢၼ်မၼ်းတုမ်ႉၼမ်ႉ၊ လၢႆးဢၼ်မၼ်းလႅၼ်းၵေႃႇၾီႇ၊ လၢႆးဢၼ်မၼ်းၸႅၵ်ႇၼမ်ႉၼမ်း။
* **လွၼ်ႉမၼ်း:** ၶူးဢမ်ႇလူဝ်ႇႁူႉဝႃႈ ၶိူင်ႈၸၢၵ်ႈၽၢႆႇၼႂ်းမၼ်းႁဵတ်းၵၢၼ်ၸိင်ႉႁိုဝ်၊ ၶူးၵူၺ်းလူဝ်ႇႁူႉဝႃႈ "တေၼိပ်ႇပုမ်ႇလႂ်" တႃႇလႆႈၵေႃႇၾီႇမႃးၵူၺ်းၶႃႈ။



### 2. Abstraction ၼႂ်းၵၢၼ်တႅမ်ႈ Code
ၼႂ်း C#... Abstraction ၸွႆးႁႂ်ႈႁဝ်းမၵ်းမၼ်ႈဝႃႈ **"Object တူဝ်ၼႆႉ ႁဵတ်းသင်လႆႈၽတ်ႉ?" (What it does)** သေ လပ်ႉဝႆႉ **"မၼ်းႁဵတ်းၸိင်ႉႁိုဝ်?" (How it does it)** ၼၼ်ႉယဝ်ႉ။

### 3. ႁဵတ်းသင်ၸင်ႇလူဝ်ႇ Abstraction?
1.  **Reduce Complexity:** ၸွႆးႁႂ်ႈ Code ႁဝ်းဢမ်ႇယုင်ႈ။ ၵူၼ်းၸႂ်ႉ Class ႁဝ်း ဢမ်ႇလူဝ်ႇႁူႉ Logic ဢၼ်ယုင်ႈယၢင်ႈၼႂ်းမၼ်း။
2.  **Focus on the Goal:** ႁဵတ်းႁႂ်ႈႁဝ်းဝူၼ်ႉၵူၺ်း "ၼႃႈတီႈ" (Contract) ဢၼ် Object လူဝ်ႇမီး။
3.  **Flexibility:** ႁဝ်းၸၢင်ႈလႅၵ်ႈလၢႆႈ Logic ၽၢႆႇၼႂ်း (Implementation) လႆႈၵူႈၶၢဝ်းယၢမ်း လူၺ်းဢမ်ႇတုမ်ႉတိူဝ်ႉ ၵူၼ်းဢၼ်ႁွင်ႉၸႂ်ႉ Class ႁဝ်းယူႇၶႃႈ။

### 4. တူဝ်ယင်ႇ ၼႂ်း Shan Community Project
ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းမီး Class **`PaymentGateway`** (လၢႆးပၼ်ငိုၼ်း)။
* ၶူးမၵ်းမၼ်ႈဝႃႈ မၼ်းတေလႆႈမီး Method `Pay()`။
* `KPay` ၵေႃႈမီး `Pay()` (လႅၼ်း Code မၼ်းတေၵွင်ႉၸူး KBZ Bank)။
* `WaveMoney` ၵေႃႈမီး `Pay()` (လႅၼ်း Code မၼ်းတေၵွင်ႉၸူး Telenor/Yoma)။
* **Abstraction:** တွၼ်ႈ App ႁဝ်း ၵူၺ်းလူဝ်ႇႁွင်ႉ `Pay()` ၵူၺ်း၊ မၼ်းဢမ်ႇလူဝ်ႇႁူႉဝႃႈ ၽၢႆႇၼႂ်းမၼ်းၵွင်ႉၸူး Bank လႂ်ၸိင်ႉႁိုဝ်ၶႃႈ။

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Abstraction:** ၵၢၼ်လပ်ႉ Implementation သေ ၼႄၵူၺ်း Essential features။
* **Implementation Details:** တွၼ်ႈ Code ၽၢႆႇၼႂ်း ဢၼ်ႁဵတ်းၵၢၼ်တႄႉတႄႉ။
* **Contract (ၵတိ):** လွင်ႈမၵ်းမၼ်ႈဝႃႈ Class ၼိုင်ႈတူဝ် တေလႆႈႁဵတ်းသင်လႆႈၽတ်ႉ။
* **Hiding Complexity:** ၵၢၼ်ႁဵတ်းႁႂ်ႈလွင်ႈယုင်ႈယၢင်ႈ လွၵ်ႇလၢႆၵႂႃႇ။
