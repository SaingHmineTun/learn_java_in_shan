# Lesson 70: "Modifying Arrays and Memory"

မိူဝ်ႈႁဝ်းမႄးၵႃႈၶၼ် (Value) ၼႂ်း Array ၼၼ်ႉ ႁဝ်းတိုၵ်ႉၸႂ်ႉ "လွၵ်းၵဝ်ႇ" ဢၼ်မီးဝႆႉၼႂ်း Memory ၼၼ်ႉယူႇၶႃႈ။

### 1. ၵၢၼ်မႄးၶေႃႈမုၼ်း (Modifying Elements)
ႁဝ်းၸၢင်ႈလႅၵ်ႈလၢႆႈၵႃႈၶၼ်လႆႈ ၵမ်းလဵဝ်လူၺ်းၵၢၼ်ၸႂ်ႉ **Index** ၶႃႈ။

```csharp
int[] luckyNumbers = { 7, 14, 21 };
luckyNumbers[1] = 99; // လႅၵ်ႈလၢႆႈ 14 ႁႂ်ႈပဵၼ် 99 ၵမ်းလဵဝ်
```

### 2. Array ပဵၼ် Reference Type
ၼႆႉပဵၼ်လွင်ႈတၢင်းဢၼ်လွင်ႈယႂ်ႇ (Important) သေပိူၼ်ႈၶႃႈ။ Array ၼႆႉ မၼ်းဢမ်ႇသိမ်းၶေႃႈမုၼ်းဝႆႉၼႂ်း Variable ၵမ်းလဵဝ်၊ မၼ်းသိမ်းဝႆႉ **"မၢႆၼပ်ႉတီႈယူႇ" (Address)** ဢၼ်ၸိၼႄၵႂႃႇၸူး Memory တွၼ်ႈဢၼ်ႁွင်ႉဝႃႈ **Heap** ၼၼ်ႉၶႃႈ။



* **Stack:** သိမ်းၸိုဝ်ႈ Variable လႄႈ မၢႆ Address (Pointer)။
* **Heap:** သိမ်းၶေႃႈမုၼ်း Array တႄႉတႄႉ ဢၼ်မီးလွၵ်းသိုပ်ႇၵၼ်ဝႆႉ။

### 3. Memory Implications (လွင်ႈတၢင်း Memory)
* **Fixed Size (ပဵၼ်ဝႆႉပိူင်ႈတၢႆ):** မိူဝ်ႈႁဝ်းသၢင်ႈ `new int[5]` ယဝ်ႉ၊ Computer တေၸွၵ်းဝႆႉလွၵ်း Memory 5 လွၵ်းသိုပ်ႇၵၼ်။ ႁဝ်းတေထႅမ်လွၵ်းထီႉ 6 ၶဝ်ႈၵႂႃႇၼႂ်း Array ၵဝ်ႇၼၼ်ႉ **ဢမ်ႇလႆႈ** ၶႃႈ။
* **Performance:** ယွၼ်ႉမၼ်းဝႆႉလွၵ်းသိုပ်ႇၵၼ် (Contiguous Memory)၊ Computer ၸင်ႇၸၢင်ႈဢၢၼ်ႇလူၶေႃႈမုၼ်းလႆႈ **ၽႂ်းၼႃႇ**။ မၼ်းဢမ်ႇလူဝ်ႇလႅၼ်ႈႁႃလၢႆလၢႆတီႈၶႃႈ။
* **Reference Sharing:** သင်ဝႃႈႁဝ်းတႅမ်ႈ `int[] b = a;` ၼႆ... `b` တင်း `a` တေၸိၼႄၸူး Array **တူဝ်လဵဝ်ၵၼ်** ၼႂ်း Memory ၶႃႈ။ ပေႃးမႄး `b`၊ ၵႃႈၶၼ်ၼႂ်း `a` ၵေႃႈတေလႅၵ်ႈလၢႆႈၸွမ်းၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Reference Type:** ပိူင် Variable ဢၼ်သိမ်းမၢႆ Address (တီႈယူႇ) တႅၼ်းတၢင်းၵႃႈၶၼ်တႄႉ။
* **Stack:** Memory တွၼ်ႈဢၼ်ၸႂ်ႉသိမ်း Variable ဢွၼ်ႇ လႄႈ Address (ၽႂ်းၼႃႇ)။
* **Heap:** Memory တွၼ်ႈလူင် ဢၼ်ၸႂ်ႉသိမ်း Object ယႂ်ႇယႂ်ႇ မိူၼ်ၼင်ႇ Array။
* **Contiguous Memory:** လွၵ်း Memory ဢၼ်ယူႇသိုပ်ႇၵၼ် ဢမ်ႇမီးတွၼ်ႈၶၢတ်ႇ။
* **Pointer/Address:** မၢႆၼပ်ႉဢၼ်လၢတ်ႈၼႄဝႃႈ ၶေႃႈမုၼ်းယူႇတီႈလႂ်ၼႂ်း Memory။
