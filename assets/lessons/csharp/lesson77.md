# Lesson 77: "The Power of Modular Programming: Why Use Methods?"

ဝူၼ်ႉတူၺ်းဝႃႈ သင်ႁဝ်းတႅမ်ႈ App ဢၼ်ၼိုင်ႈသေ မီး Code တွၼ်ႈ "ၵူတ်ႇထတ်း Password" ယူႇ 10 ထႅဝ်။ 
သင် App ၼၼ်ႉ လူဝ်ႇၵူတ်ႇ Password တီႈၼႃႈ Login ၼိုင်ႈၵမ်း၊ တီႈၼႃႈ Register ထႅင်ႈပွၵ်ႈၼိုင်ႈ၊ တီႈၼႃႈ Change Password ထႅင်ႈၵမ်းၼိုင်ႈ ၼႆၸိုင်... 
ႁဝ်းတေလႆႈ Copy Code 10 ထႅဝ်ၼၼ်ႉၵႂႃႇဝႆႉၵူႈတီႈႁႃႉ? 
ၼၼ်ႉယဝ်ႉ ႁဝ်းၸင်ႇလူဝ်ႇ **Method** ၶႃႈ။

### 1. Method ပဵၼ်သင်? (What is a Method?)
**Method** ၼႆႉ ပဵၼ် "ၵွၼ်ႈ Code" (Block of Code) ဢၼ်ႁဝ်းပၼ်ၸိုဝ်ႈမၼ်းဝႆႉ ႁႂ်ႈႁဵတ်းၵၢၼ်သေလွင်ႈလွင်ႈယဝ်ႉ။ 
ပေႃးႁဝ်းၶႂ်ႈၸႂ်ႉ မၼ်းၵေႃႈ ႁွင်ႉ "ၸိုဝ်ႈ" မၼ်းမႃးသေ ၸႂ်ႉလႆႈၵမ်းလဵဝ်ၶႃႈ။



### 2. တၢင်းလီၶွင် Modular Programming
* **Reusability (ၸႂ်ႉၶိုၼ်းလႆႈ):** တႅမ်ႈပွၵ်ႈလဵဝ် ၸႂ်ႉလႆႈလၢႆလၢႆတီႈ။
* **Readability (ဢၢၼ်ႇလူငၢႆႈ):** Code ႁဝ်းတေဢမ်ႇယၢဝ်းပူၼ်ႉတီႈ။ ႁဝ်းတေႁူႉၵမ်းလဵဝ်ဝႃႈ `CheckPassword()` ၼႆႉ ႁဵတ်းၵၢၼ်သင်။
* **Easy to Maintain (မႄးငၢႆႈ):** သင်မီး Error ၼႂ်းၵၢၼ်ၵူတ်ႇ Password၊ ႁဝ်းၵူၺ်းၵႂႃႇမႄးတီႈ Method ၼၼ်ႉတီႈလဵဝ်၊ မၼ်းတေမႄးပၼ်တင်း App ၵမ်းလဵဝ်ၶႃႈ။
* **Testing:** ႁဝ်းၸၢင်ႈၸၢမ်း (Test) Code တွၼ်ႈၼိုင်ႈ ႁႂ်ႈမၼ်းမႅၼ်ႈလီ မိူင်ႇၼင်ႇ ႁဝ်းၶႂ်ႈပဵၼ်ယဝ်ႉ။

### 3. ပိူင်ယင်ႇၼႂ်းၸူဝ်ႈပၢၼ်တႄႉ
မိူၼ်ၼင်ႇ "ၵၢၼ်ႁုင်ၶဝ်ႈ" ၶႃႈ။ ႁဝ်းဢမ်ႇလူဝ်ႇလၢတ်ႈဝႃႈ "ၵႂႃႇဢဝ်မေႃႈ၊ သႂ်ႇၶဝ်ႈ၊ သႂ်ႇၼမ်ႉ၊ ၼဵၵ်းၾႆး" ၵူႈပွၵ်ႈ။ ႁဝ်းၵူၺ်းႁွင်ႉဝႃႈ **`CookRice()`** ၼႆၵေႃႈ ၵူႈၵေႃႉပွင်ႇၸႂ်ပိူင်ၵၢၼ်မၼ်းတင်းမူတ်းယဝ်ႉၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Method / Function:** တွၼ်ႈ Code ဢၼ်မီးၸိုဝ်ႈသေ ႁဵတ်းၵၢၼ်လႆႈလီငၢမ်း။
* **Modular Programming:** လွၵ်းလၢႆးတႅမ်ႈ Code လူၺ်းၵၢၼ်ၸႅၵ်ႇပဵၼ်တွၼ်ႈပဵၼ်တွၼ်ႈ (Modules)။
* **Code Reuse:** ၵၢၼ်ဢဝ် Code ၵဝ်ႇမႃးၸႂ်ႉၶိုၼ်း။
* **Maintenance:** ၵၢၼ်ၵုမ်းထိင်း လႄႈ မႄးလႅၵ်ႈလၢႆႈ Code ႁႂ်ႈမၼ်းမႅၼ်ႈယူႇတႃႇသေႇ။
