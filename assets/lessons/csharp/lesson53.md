# Lesson 53: "The 'while' Loop: Condition-First Repetition"

ၼႂ်း **Lesson 53** ၼႆႉ ႁဝ်းတေမႃးလဵပ်ႈႁဵၼ်းလွင်ႈ **'while' loop** ဢေႃႈ။ 
ၼႆႉပဵၼ် Loop ဢၼ်ငၢႆႈသေပိူၼ်ႈ။ 
မၼ်းတေတူၺ်း **"ၶေႃႈထၢမ်"** ဢွၼ်တၢင်း၊ သင်မႅၼ်ႈ (True) ၸင်ႇတေၶဝ်ႈႁဵတ်းၵၢၼ်။ 
သင်ဝႃႈၶေႃႈထၢမ်ၽိတ်း (False) ၸဵမ်မိူဝ်ႈတေႇၼၼ်ႉ မၼ်းတေဢမ်ႇႁဵတ်းၵၢၼ် သေပွၵ်ႈၶႃႈ။



### 1. ပိူင်သၢင်ႈ (Syntax)
```csharp
while (ၶေႃႈထၢမ်)
{
    // Code တွၼ်ႈၼႆႉ တေႁဵတ်းပၼ်ႇတူင်ႇတိၵ်ႉတိၵ်ႉ 
    // ပေႃး ၶေႃႈထၢမ် တိုၵ်ႉပဵၼ် true ယူႇ
}
```

### 2. တူဝ်ယင်ႇ Code တႄႉတႄႉ
ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းၶႂ်ႈၼပ်ႉမၢႆၼပ်ႉ 1 ထိုင် 5 ၶႃႈ။

```csharp
int count = 1; // 1. Start (တီႈတႄႇ)

while (count <= 5) // 2. Condition (ၶေႃႈထၢမ်)
{
    Console.WriteLine("Number: " + count);
    
    count++; // 3. Update (ၵၢၼ်လႅၵ်ႈလၢႆႈ - ထႅမ်ၵႃႈၶၼ်)
}

Console.WriteLine("Loop finished!");
```

**ပိူင်ၵၢၼ်လႅၼ်ႈမၼ်း:**
1. count = 1 -> `1 <= 5` (True) -> ၼႄ "Number: 1" -> count ပဵၼ် 2
2. count = 2 -> `2 <= 5` (True) -> ၼႄ "Number: 2" -> count ပဵၼ် 3
   ...
3. count = 6 -> `6 <= 5` (**False**) -> **ဢွၵ်ႇ Loop ၵမ်းလဵဝ် (Exit Condition)**

### 3. လွင်ႈလူဝ်ႇၾၢင်ႉ (The Infinite Loop)
သင်ဝႃႈႁဝ်း "လိုမ်း" တႅမ်ႈ `count++` ၼၼ်ႉ၊ `count` တေပဵၼ် 1 တႃႇသေႇ၊ ၶေႃႈထၢမ် `count <= 5` ၵေႃႈတေပဵၼ် **True** တႃႇသေႇ။ 
App ႁဝ်းတေႁဵတ်းပၼ်ႇတူင်ႇဢမ်ႇၵိုတ်းသေ ႁဵတ်းႁႂ်ႈၶွမ်း "ၶမ်" (Freeze) ၵႂႃႇလႆႈၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **While Loop:** ၵၢၼ်ႁဵတ်းပၼ်ႇတူင်ႇ ဢၼ်ၵူတ်ႇထတ်းၶေႃႈထၢမ်ဢွၼ်တၢင်း (Pre-test loop)။
* **Condition-First:** လွၵ်းလၢႆးဢၼ်တေဢမ်ႇႁဵတ်းၵၢၼ်သေပွၵ်ႈ သင်ဝႃႈၶေႃႈထၢမ်ၽိတ်း (False) ၸဵမ်မိူဝ်ႈတေႇ။
* **Body of Loop:** Code ဢၼ်ယူႇၼႂ်း `{ }` ဢၼ်တေလႆႈႁဵတ်းပၼ်ႇတူင်ႇ။
* **Increment:** ၵၢၼ်ထႅမ်ၵႃႈၶၼ် (မိူၼ်ၼင်ႇ `++`) တႃႇႁႂ်ႈ Loop ပႆၵႂႃႇၽၢႆႇၼႃႈ။
