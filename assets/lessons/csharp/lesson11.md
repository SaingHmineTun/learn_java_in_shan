# Lesson 11: "Debugging Basics" (ပိုၼ်ႉထၢၼ်ၵၢၼ်ၵူတ်ႇထတ်းႁႃပၼ်ႁႃ)

မိူဝ်ႈႁဝ်းတႅမ်ႈ Code ယဝ်ႉ၊ မၢင်ပွၵ်ႈမၼ်းဢမ်ႇဢွၵ်ႇ (Result) မိူၼ်ၼင်ႇႁဝ်းၶႂ်ႈလႆႈ။ 
ႁဝ်းလူဝ်ႇၸႂ်ႉ **Debugger** တႃႇတူၺ်းဝႃႈ ၼႂ်းတူဝ် App ႁဝ်းၼၼ်ႉ မၼ်းမီးသင်ပဵၼ်ယူႇတႄႉတႄႉ။

### 1. Breakpoint (မၢႆၵိုတ်း)
**Breakpoint** ၼႆႉမိူၼ်ၼင်ႇ "ပၢႆႉၵိုတ်းလူတ်ႉ" ယဝ်ႉ။ ပေႃးႁဝ်းသႂ်ႇဝႆႉတီႈထႅဝ် Code လႂ်၊ မိူဝ်ႈႁဝ်း Run App ၼၼ်ႉ မၼ်းတေမႃးၵိုတ်းဝႆႉတီႈထႅဝ်ၼၼ်ႉ ႁႂ်ႈႁဝ်းၸွပ်ႇတူၺ်းလႆႈ။
* **လၢႆးၸႂ်ႉ:** တဵၵ်းတီႈ "လွၵ်းသီသွမ်ႇ" ၽၢႆႇၼႃႈထႅဝ် Code ၼႂ်း Visual Studio ႁႂ်ႈမၼ်းပဵၼ် **တူတ်ႇသီလႅင်**။


### 2. Step Over & Step Into
မိူဝ်ႈ App ၵိုတ်းတီႈ Breakpoint ယဝ်ႉ၊ ႁဝ်းတေလႆႈၸႂ်ႉၶိူင်ႈမိုဝ်းတႃႇ "ယၢင်ႈ" ၵႂႃႇၸွမ်း Code:
* **Step Over (F10):** ယၢင်ႈၵႂႃႇထႅဝ်ၼိုင်ႈသေ ၶၢမ်ႈၵႂႃႇထႅဝ်ၼိုင်ႈ (ဢမ်ႇၶဝ်ႈၵႂႃႇၼႂ်း Method ဢွၼ်ႇ)။
* **Step Into (F11):** ယၢင်ႈၶဝ်ႈၵႂႃႇတူၺ်း တႅတ်ႉတႅတ်ႉတေႃးတေႃး ၼႂ်း Method ဢၼ်ႁဝ်းႁွင်ႉၸႂ်ႉဝႆႉၼၼ်ႉ။

### 3. Call Stack (သဵၼ်ႈတၢင်းၵၢၼ်ႁွင်ႉၸႂ်ႉ)
**Call Stack** ၼႆႉပဵၼ်ၼႃႈလိၵ်ႈဢၼ်ၼႄဝႃႈ "Code ႁဝ်းလုၵ်ႉတီႈလႂ် မႃးတီႈလႂ်"။ မၼ်းမိူၼ်ၼင်ႇ ႁွႆးတိၼ် (Footprints) ဢၼ်ၼႄဝႃႈ Method လႂ် ႁွင်ႉ Method လႂ် ၸင်ႇမႃးထိုင် တီႈဢၼ်မၼ်း Error ယူႇတေႃႈလဵဝ်ၼၼ်ႉယဝ်ႉ။


---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Bug:** ၶေႃႈၽိတ်းပိူင်ႇ (Error) ၼႂ်း Code ဢၼ်ႁဵတ်းႁႂ်ႈ App ႁဵတ်းၵၢၼ်ဢမ်ႇမႅၼ်ႈ။
* **Debugging:** လၢႆးၵၢၼ်ႁႃ လႄႈ ၵႄႈလႅတ်ႈပၼ်ႁႃ (Bugs) ၼႂ်း Code။
* **Breakpoint:** တီႈဢၼ်ႁဝ်းသင်ႇႁႂ်ႈ App ၵိုတ်းၸူဝ်ႈၶၢဝ်ႇ တႃႇၸွပ်ႇတူၺ်းၶေႃႈမုၼ်း (Variables)။
* **Call Stack:** သဵၼ်ႈမၢႆၵၢၼ်ႁွင်ႉၸႂ်ႉ Method ၸွမ်းလၢႆးမၼ်း တေႃႇပေႃးထိုင်တီႈၵိုတ်းယူႇၼၼ်ႉ။
* **Watch Window:** ၼႃႈလိၵ်ႈဢၼ်ႁဝ်းၸႂ်ႉတူၺ်း "ၵႃႈၶၼ်" (Value) ၶေႃႈမုၼ်းၼႂ်း Variable မိူဝ်ႈႁဝ်း Debugging ယူႇ။
