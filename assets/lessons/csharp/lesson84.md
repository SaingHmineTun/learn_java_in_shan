# Lesson 84: "Passing by Value" (ၵၢၼ်သူင်ႇၵႃႈၶၼ်)

မိူဝ်ႈႁဝ်းသူင်ႇ Variable ၶဝ်ႈၵႂႃႇၼႂ်း Method လူၺ်ႈလၢႆး **Pass by Value** ၼႆႉ... 
Method ၼၼ်ႉ တေဢမ်ႇလႆႈ Variable တူဝ်တႄႉမၼ်းၵႂႃႇၶႃႈ။ 
မၼ်းတေလႆႈ **"ၵေႃႇပီႇ" (Copy)** ၶွင်ၵႃႈၶၼ်ၼၼ်ႉၵႂႃႇၵူၺ်းၶႃႈ။

### 1. မၼ်းႁဵတ်းၵၢၼ်ၸိူင်ႉႁိုဝ်?
ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းမီး "ပပ်ႉလိၵ်ႈ" (Variable) တူဝ်တႄႉ။ မိူဝ်ႈသူင်ႇပၼ်လုၵ်ႈႁဵၼ်း ၸႂ်ႉလၢႆး Pass by Value ၼႆႉ မိူၼ်ၼင်ႇ ႁဝ်းထႆႇၶႅပ်းႁၢင်ႈ (Photocopy) ပပ်ႉၼၼ်ႉသေ သူင်ႇပၼ်ၶဝ်ၶႃႈ။
* သင်လုၵ်ႈႁဵၼ်း တႅမ်ႈလိၵ်ႈသႂ်ႇၼႂ်း ပပ်ႉထႆႇၶႅပ်း (Copy) ၼၼ်ႉ...
* ပပ်ႉတူဝ်တႄႉ (Original) ၶွင်ႁဝ်း တေ **ဢမ်ႇလႅၵ်ႈလၢႆႈ** သင်ၶႃႈ။



### 2. တူဝ်ယင်ႇ Code
```csharp
class Program {
    static void Main() {
        int myNumber = 10;
        
        // သူင်ႇ myNumber ၶဝ်ႈၵႂႃႇ (သူင်ႇၵေႃႇပီႇၵႂႃႇ)
        ChangeValue(myNumber);

        // ၵႃႈၶၼ်တူဝ်တႄႉ တေတိုၵ်ႉပဵၼ် 10 ယူႇမိူၼ်ၵဝ်ႇ
        Console.WriteLine("Original Value: " + myNumber); 
    }

    static void ChangeValue(int x) {
        x = 500; // မႄးလႅၵ်ႈလၢႆႈၼႂ်းၵေႃႇပီႇ (x) ၵူၺ်း
        Console.WriteLine("Value inside Method: " + x);
    }
}
```

### 3. ႁဵတ်းသင်ၸင်ႇၸႂ်ႉ?
* **Safety:** ႁဵတ်းႁႂ်ႈႁဝ်းတမ်းၸႂ်လႆႈဝႃႈ Method ဢၼ်ႁဝ်းႁွင်ႉၸႂ်ႉၼၼ်ႉ တေဢမ်ႇၵႂႃႇမႄး (Change) ၶေႃႈမုၼ်းတူဝ်တႄႉ ႁဝ်းလူၺ်ႈဢမ်ႇတင်ႈၸႂ်။
* **Isolation:** ၸႅၵ်ႇၵၢၼ်ႁဵတ်းသၢင်ႈၼႂ်း Method ႁႂ်ႈယူႇႁင်းၵူၺ်းမၼ်း။

### 4. တႃႇ Reference Types (Array, Class)
မၢႆတွင်းဝႆႉဢိတ်းၼိုင်ႈၶႃႈ: သင်ပဵၼ် **Array** ၼႆ... တူဝ် "မၢႆၼပ်ႉတီႈယူႇ" (Address) မၼ်းတေသူင်ႇပဵၼ် Value ၵႂႃႇ။ ၵွပ်ႈၼၼ် ပေႃးမႄးၶေႃႈမုၼ်းၼႂ်း Array ၼၼ်ႉ မၼ်းတေလႅၵ်ႈလၢႆႈၸွမ်း (မိူၼ် Lesson 70)။ ၵူၺ်းၵႃႈ သင်ပဵၼ် Simple Types (int, double, string, bool) တႄႉ မၼ်းတေဢမ်ႇလႅၵ်ႈလၢႆႈၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Pass by Value:** ၵၢၼ်သူင်ႇၵေႃႇပီႇ (Copy) ၶွင်ၶေႃႈမုၼ်းၶဝ်ႈၵႂႃႇၼႂ်း Method။
* **Local Scope:** ၶွပ်ႇၵင်ႈၵၢၼ်ၸႂ်ႉတူဝ် Parameter (x) ဢၼ်ယူႇၼႂ်း Method ၼၼ်ႉၵူၺ်း။
* **Original Variable:** တူဝ် Variable တူဝ်တႄႉ ဢၼ်ယူႇၽၢႆႇၼွၵ်ႈ Method။
* **Immutable:** ပိူင်ၶေႃႈမုၼ်းဢၼ်လႅၵ်ႈလၢႆႈဢမ်ႇလႆႈ (မိူၼ်ၼင်ႇ String ၼႂ်း C#)။
