# Lesson 266: "Introduction to Databases: Why We Need SQL Server"

မိူဝ်ႈႁဝ်းတႅမ်ႈ App ၼႂ်း Module ဢၼ်ယဝ်ႉမႃးၸိူဝ်းၼၼ်ႉ ၶေႃႈမုၼ်း (Data) ႁဝ်းယူႇၼႂ်း **RAM** (Memory) ၵူၺ်းၶႃႈ။ 
မိူဝ်ႈပိၵ်ႉ App ၵႂႃႇၼႆ Data ၸိူဝ်းၼၼ်ႉၵေႃႈ တေႁၢႆၵႂႃႇၸွမ်းၶႃႈဢေႃႈ။ 
တွၼ်ႈတႃႇႁႂ်ႈ App ႁဝ်းၸႂ်ႉလႆႈတႄႉတႄႉ (မိူၼ်ၼင်ႇ ၵဵပ်းသဵၼ်ႈမၢႆၵူၼ်းၼႂ်းဝဵင်း Muse) ၼႆၸိုင် ႁဝ်းတေလႆႈၸႂ်ႉ **Database** ၶႃႈ။

### 1. Database ပဵၼ်သင်?
**Database** ပဵၼ် "ႁွင်ႈၵဵပ်းၶေႃႈမုၼ်း" ဢၼ်မီးပိူင်သၢင်ႈ (Organized) တႅတ်းတေႃးၶႃႈ။
* မၼ်းဢမ်ႇၸႂ်ဝႃႈ သိမ်းလႆႈ File Text ၵူၺ်းၶႃႈ။ မၼ်းပဵၼ် System ဢၼ်ၸတ်းၵၢၼ်လွင်ႈ **"သွၵ်ႈႁႃ"** (Search), **"သိမ်း"** (Store), လႄႈ **"ၵုမ်းၵၢၼ်"** (Manage) Data လႄႈ မၼ်းႁဵတ်းလႆႈဝႆးဝႆးၶႃႈ။

### 2. ႁဵတ်းသင်ႁဝ်းၸင်ႇလူဝ်ႇ SQL Server?
**SQL Server** ပဵၼ် **RDBMS** (Relational Database Management System) ၶွင် Microsoft ၶႃႈ။ ႁဵတ်းသင်ႁဝ်းၸင်ႇလိူၵ်ႈၸႂ်ႉမၼ်း?
1.  **Security (လွင်ႈလွတ်ႈၽေး):** မၼ်းမီးပိူင်ႁႄႉၵၢင်ႈ ဢမ်ႇပၼ်ၽႂ်ၶဝ်ႈလူ Data ႁဝ်းလႆႈငၢႆႈငၢႆႈၶႃႈ။
2.  **Scalability (ၵၢၼ်ယႂ်ႇထူၼ်ႈ):** မၼ်းသိမ်း Data လႆႈၼမ်ၼႃႇ (ပဵၼ် Terabytes) ၵေႃႈတိုၵ်ႉလႅၼ်ႈဝႆးယူႇၶႃႈ။
3.  **Concurrency (ၵၢၼ်ၸႂ်ႉၸွမ်းၵၼ်):** ပၼ်သုၼ်ႇႁႂ်းၵူၼ်း 100 ၵေႃႉ တႅမ်ႈ Data ၶဝ်ႈၵႂႃႇၼႂ်းယၢမ်းလဵဝ်ၵၼ် လူၺ်ႈ Data ဢမ်ႇလႅဝ်ၶႃႈ။
4.  **C# Integration:** ယွၼ်ႉမၼ်းပဵၼ်ၶွင် Microsoft မိူၼ်ၵၼ်လႄႈ မၼ်းၸင်ႇၵွင်ႉၵၢႆႇတင်း C# လႄႈ .NET လႆႈလီသုတ်းၶႃႈ။



### 3. ပႅၵ်ႇပိူင်သင်တင်း Excel?
လုၵ်ႈႁဵၼ်းမၢင်ၵေႃႉၵႆႉထၢမ်ဝႃႈ "ၸႂ်ႉ Excel သိမ်း Data ဢမ်ႇလႆႈႁႃႉ?"
* **Excel:** လီတွၼ်ႈတႃႇၵူၼ်း 1 ၵေႃႉ လူ/မႄး Data လဵၵ်ႉလဵၵ်ႉ ဢၼ်ဢမ်ႇယုင်ႈယၢင်ႈၶႃႈ။
* **SQL Server:** လီတွၼ်ႈတႃႇ "Application" ဢၼ်လူဝ်ႇၸတ်းၵၢၼ် Data ယႂ်ႇယႂ်ႇ လႄႈ ၵွင်ႉၵၢႆႇၵႄႈ Tables လၢႆလၢႆဢၼ် (မိူၼ်ၼင်ႇ Table 'လုၵ်ႈႁဵၼ်း' ၵွင်ႉၸူး Table 'ၶူးသွၼ်') ၶႃႈဢေႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Persistent Data:** ၶေႃႈမုၼ်းဢၼ် မီးယူႇတႃႇသေႇလႄႈ ဢမ်ႇႁၢႆၵႂႃႇ မိူဝ်ႈပိၵ်ႉ App။
* **RDBMS:** System ဢၼ်ၸတ်းၵၢၼ် Database လၢႆးဢၼ်မီး "လွင်ႈၵွင်ႉၵၢႆႇ" (Relational)။
* **Query:** ၶေႃႈၵႂၢမ်းသင်ႇၵၢၼ် တႃႇသွၵ်ႈႁႃ ဢမ်ႇၼၼ် မႄး Data ၼႂ်း Database။
* **SQL (Structured Query Language):** လိၵ်ႈဢၼ်ၸႂ်ႉတႅမ်ႈသင်ႇၵၢၼ် Database ၶႃႈ။
