# Lesson 195: "Throwing Exceptions: The 'throw' Keyword"

မၢင်ပွၵ်ႈ Code ႁဝ်းဢမ်ႇလႆႈၽိတ်းၸွမ်းပၵ်းပိူင် C# (မိူၼ်ၼင်ႇ ဢဝ် 0 မႃးၸႅၵ်ႇ)၊ 
ၵူၺ်းၵႃႈ မၼ်းၽိတ်းၸွမ်း "ပၵ်းပိူင်ၼႃႈၵၢၼ်" (Business Logic) ႁဝ်းၶႃႈ။ 
တူဝ်ယင်ႇ မိူၼ်ၼင်ႇ: ၵူၼ်းၸႂ်ႉသႂ်ႇ "ဢႃႇယု" ပဵၼ် -5 ၼႆ... 
ၼႂ်းပၢႆးၼပ်ႉမၼ်းဢမ်ႇၽိတ်း၊ ၵူၺ်းၵႃႈ ၼႂ်းလွင်ႈတႄႉမၼ်းပဵၼ်ၵႂႃႇဢမ်ႇလႆႈၶႃႈ။ 
ၼႂ်းငဝ်းလၢႆးၼင်ႇၼႆ ႁဝ်းတေၸႂ်ႉ Keyword **`throw`** တႃႇတႅပ်းတတ်းဝႃႈ "ၼႆႉပဵၼ်ပၼ်ႁႃယဝ်ႉ" ၼႆၶႃႈ။

### 1. ႁဵတ်းသင်ႁဝ်းၸင်ႇလူဝ်ႇ `throw`?
* **Validation:** တႃႇၵူတ်ႇထတ်းၶေႃႈမုၼ်း ႁႂ်ႈမႅၼ်ႈၸွမ်းပၵ်းပိူင် ဢၼ်ႁဝ်းမၵ်းမၼ်ႈဝႆႉ (မိူၼ်ၼင်ႇ Password တေလႆႈမီး 8 တူဝ်)။
* **Early Failure:** တႃႇႁႂ်ႈ Program ၵိုတ်းၵမ်းလဵဝ် မိူဝ်ႈမီးၶေႃႈမုၼ်း ဢၼ်ဢမ်ႇထုၵ်ႇလီပဵၼ်။

### 2. ပိူင်သၢင်ႈ (Syntax)
ႁဝ်းၸၢင်ႈ `throw` Object ဢၼ်လုၵ်ႉတီႈ Class `Exception` မႃးၶႃႈ။

```csharp
if (age < 0) {
    throw new ArgumentOutOfRangeException("ဢႃႇယု တေလႆႈယႂ်ႇလိူဝ် သုၼ် ၶႃႈဢေႃႈ။");
}
```



### 3. တူဝ်ယင်ႇ Code: ၵၢၼ်ၵူတ်ႇထတ်း (Validation)
ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းတႅမ်ႈ App တႃႇသႂ်ႇၸိုဝ်ႈလုၵ်ႈႁဵၼ်း။ သင်ၸိုဝ်ႈၼၼ်ႉပဵၼ် "လိၵ်ႈပဝ်ႇ" (Empty string) ႁဝ်းတေဢမ်ႇႁပ်ႉၶႃႈ။

```csharp
public void RegisterStudent(string name) {
    if (string.IsNullOrWhiteSpace(name)) {
        // သၢင်ႈပၼ်ႁႃဢွၵ်ႇမႃး ႁင်းၵူၺ်း
        throw new ArgumentException("ၸိုဝ်ႈလုၵ်ႈႁဵၼ်း တေလႆႈတႅမ်ႈသႂ်ႇ၊ ယႃႇဝႆႉပဝ်ႇၶႃႈ။");
    }
    Console.WriteLine($"Register: {name} success!");
}

// ၵၢၼ်ႁွင်ႉၸႂ်ႉ (Usage)
try {
    RegisterStudent(""); // ❌ ၼႆႉတေပဵၼ်ပၼ်ႁႃၵမ်းလဵဝ်
} 
catch (Exception ex) {
    Console.WriteLine($"ၽိတ်း: {ex.Message}");
}
```

### 4. ၶေႃႈမၢႆတွင်းလမ်ႇလွင်ႈ
* မိူဝ်ႈႁဝ်း `throw` ယဝ်ႉ... Code ဢၼ်ယူႇၽၢႆႇတႂ်ႈမၼ်း တေဢမ်ႇလႅၼ်ႈထႅင်ႈၶႃႈ။
* ႁဝ်းထုၵ်ႇလီလိူၵ်ႈၸႂ်ႉ Exception Type ဢၼ်မႅၼ်ႈတင်းလွင်ႈတၢင်း (မိူၼ်ၼင်ႇ `ArgumentException` မိူဝ်ႈၽိတ်းတီႈ Parameter)။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Validation:** ၵၢၼ်ၵူတ်ႇထတ်းၶေႃႈမုၼ်း ႁႂ်ႈမၼ်ႈၸႂ်ဝႃႈမႅၼ်ႈၸွမ်းပၵ်းပိူင်။
* **Argument Exception:** ပၼ်ႁႃဢၼ်ပဵၼ် ယွၼ်ႉ Parameter ဢၼ်သူင်ႇၶဝ်ႈမႃးၼၼ်ႉ ဢမ်ႇမႅၼ်ႈၸွမ်းပိူင်။
* **Explicit Throw:** ၵၢၼ်ဢၼ်ႁဝ်းတႅပ်းတတ်း သၢင်ႈပၼ်ႁႃဢွၵ်ႇမႃးႁင်းၵူၺ်း လူၺ်ႈတႅမ်ႈ Code ဝႆႉ။
* **Logic Error:** လွင်ႈၽိတ်းပိူင် ဢၼ်ပဵၼ် ၸွမ်းလၢႆးဝူၼ်ႉၼႃႈၵၢၼ်။
