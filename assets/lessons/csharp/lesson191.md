# Lesson 191: "The 'try-catch' Block"

လၢႆးႁဵတ်းၵၢၼ်မၼ်း တေမီးၼင်ႇၼႆၶႃႈ: 
**"ၸၢမ်းႁဵတ်းတူၺ်း (Try)၊ သင်မီးပၼ်ႁႃ... ႁပ်ႉမၼ်းဝႆႉ (Catch)"**။

### 1. ပိူင်သၢင်ႈ (Syntax)

```csharp
try {
    // Code ဢၼ်ႁဝ်းဝူၼ်ႉဝႃႈ "ၸၢင်ႈမီးပၼ်ႁႃ"
} catch (Exception ex) {
    // Code ဢၼ်တေလႅၼ်ႈ မိူဝ်ႈမီးပၼ်ႁႃ (တႃႇႁႄႉဢမ်ႇႁႂ်ႈ App လူႉ)
}
```



### 2. တူဝ်ယင်ႇ: ၵၢၼ်ႁၢၼ်လူၺ်ႈ သုၼ် (0)
ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းတႅမ်ႈ Code တႃႇၸႅၵ်ႇတူဝ်ၼပ်ႉ။ သင်ၵူၼ်းၸႂ်ႉ သႂ်ႇတူဝ်ၸႅၵ်ႇပဵၼ် `0`... App တေ Crash ၵမ်းလဵဝ်။

**Code ဢၼ်ဢမ်ႇမီး try-catch:**
```csharp
int x = 10;
int y = 0;
int result = x / y; // ❌ App Crash ၵမ်းလဵဝ်! (DivideByZeroException)
Console.WriteLine(result);
```

**Code ဢၼ်မီး try-catch:**
```csharp
try {
    int x = 10;
    int y = 0;
    int result = x / y; 
    Console.WriteLine(result);
} catch (Exception ex) {
    // သင်မီးပၼ်ႁႃ... မၼ်းတေမႃးလႅၼ်ႈထႅဝ်ၼႆႉ တႅၼ်းၵၢၼ် Crash
    Console.WriteLine("မီးပၼ်ႁႃ: ဢဝ်တူဝ်ၼပ်ႉမႃးၸႅၵ်ႇလူၺ်ႈ သုၼ် ဢမ်ႇလႆႈၶႃႈ!");
}

Console.WriteLine("App သိုပ်ႇလႅၼ်ႈၵႂႃႇလႆႈယူႇ..."); // ✅ App ဢမ်ႇလူႉ
```

### 3. ၶၼ်တွၼ်ႈၵၢၼ်ႁဵတ်းၵၢၼ်
1.  **Try Block:** Code လႅၼ်ႈၵႂႃႇၸွမ်းပၵ်းၵၢၼ်။
2.  **Exception occurs:** သင်မီးပၼ်ႁႃၵဵတ်းၶိုၼ်း... Code ၼႂ်း `try` ဢၼ်ၵိုတ်းဝႆႉ တေထုၵ်ႇၶၢမ်ႇ (Skip) ပႅတ်ႈၵမ်းလဵဝ်။
3.  **Catch Block:** Compiler တေၵျွၵ်းမႃးႁဵတ်းၵၢၼ်ၼႂ်း `catch` သေ ႁပ်ႉဢဝ် Object ပၼ်ႁႃၼၼ်ႉဝႆႉ။
4.  **Normal Flow:** ဝၢႆးလင် `catch` ယဝ်ႉ... Program တေသိုပ်ႇလႅၼ်ႈၵႂႃႇၸွမ်းထႅဝ်လိၵ်ႈ ၽၢႆႇတႂ်ႈ `try-catch` ၼၼ်ႉၶႃႈ။

### 4. ႁဵတ်းသင်ၸင်ႇလီလိူဝ်?
* **User Experience:** ၵူၼ်းၸႂ်ႉတေလႆႈႁၼ် "ၶေႃႈၵႂၢမ်းတိူၼ်း" တႅၼ်းဢၼ်တေႁၼ် App ပိတ်းၵႂႃႇလၢႆๆ။
* **Data Safety:** ႁႄႉၵၢင်ႈဢမ်ႇပၼ်ၶေႃႈမုၼ်းလူႉယႅတ်း မိူဝ်ႈ App ၵျွၵ်းဢွၵ်ႇၵမ်းလဵဝ်။
* **Control:** ႁဝ်းပဵၼ်ၵူၼ်းၵုမ်းၵၢၼ် (Control) ပၼ်ႁႃလႆႈ လူၺ်ႈဢမ်ႇပၼ် Runtime ပဵၼ်ၵူၼ်းတႅပ်းတတ်း။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Handled Exception:** ပၼ်ႁႃဢၼ်ႁဝ်းတႅမ်ႈ Code ႁႄႉဝႆႉယဝ်ႉ။
* **Unhandled Exception:** ပၼ်ႁႃဢၼ်ႁဝ်းဢမ်ႇလႆႈႁပ်ႉဝႆႉ သေႁဵတ်းႁႂ်ႈ App Crash။
* **Block:** တွၼ်ႈ Code ဢၼ်ယူႇၼႂ်း `{ }`။
* **Exception Variable (`ex`):** တူဝ်တႅၼ်းဢၼ်သိမ်းၶေႃႈမုၼ်း လွင်ႈတၢင်းဢၼ်ၽိတ်းဝႆႉ။
