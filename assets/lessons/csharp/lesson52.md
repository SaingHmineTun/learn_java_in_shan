# Lesson 52: "The Concept of Iteration" (လွင်ႈတၢင်းၵၢၼ်ႁဵတ်းပၼ်ႇတူင်ႇ)

**Iteration** ၼႆႉ ပွင်ႇဝႃႈ "ၵၢၼ်ႁဵတ်းပၼ်ႇတူင်ႇ" (Repetition) ဢေႃႈ။ 
ၼႂ်းၵၢၼ်တႅမ်ႈ Program ၼၼ်ႉ ႁဝ်းၸႂ်ႉ **Loops** တႃႇသင်ႇႁႂ်ႈ Code တွၼ်ႈၼိုင်ႈ ႁဵတ်းၵၢၼ်လၢႆလၢႆပွၵ်ႈ ၸွမ်းၼင်ႇႁဝ်းလူဝ်ႇၶႃႈ။

### 1. ႁဵတ်းသင်ႁဝ်းလူဝ်ႇ Loops?
ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းၶႂ်ႈႁႂ်ႈ App ၼႄလိၵ်ႈဝႃႈ `"Hello Shan Community!"` ၼိုင်ႈသႅၼ် (100,000) ပွၵ်ႈၶႃႈ။
* **သင်ဢမ်ႇမီး Loop:** ႁဝ်းတေလႆႈ Copy-Paste ထႅဝ်လိၵ်ႈ `Console.WriteLine` ၼၼ်ႉ ၼိုင်ႈသႅၼ်ထႅဝ် (Code တေယၢဝ်းၼႃႇ လႄႈ မႄးယၢပ်ႇ)။
* **သင်မီး Loop:** ႁဝ်းတႅမ်ႈ Code 3-4 ထႅဝ်ၵူၺ်းၵေႃႈ ႁဵတ်းလႆႈယဝ်ႉၶႃႈ။



### 2. ပိူင်ၵၢၼ် Loops ၵူႈပိူင်
Loop ၵူႈဢၼ် ၼႂ်း C# မီးလွင်ႈ 3 ယၢင်ႇ ဢၼ်မိူၼ်ၵၼ်ၶႃႈ:
1.  **Start (တီႈတႄႇ):** မၵ်းမၼ်ႈဝႃႈ တေတႄႇၼပ်ႉတီႈလႂ်။
2.  **Condition (ၶေႃႈထၢမ်):** တေႁဵတ်းပၼ်ႇတူင်ႇတိၵ်ႉတိၵ်ႉ မိူဝ်ႈၽွင်း ၶေႃႈထၢမ် တိုၵ်ႉပဵၼ် **True** ယူႇ။
3.  **Update (ၵၢၼ်လႅၵ်ႈလၢႆႈ):** ဝၢႆးႁဵတ်းပၼ်ႇတူင်ႇယဝ်ႉ ၼိုင်ႈႁွပ်ႈယဝ်ႉ တေလႆႈလႅၵ်ႈလၢႆႈၵႃႈၶၼ် (မိူၼ်ၼင်ႇ ထႅမ် 1) တႃႇႁႂ်ႈမၼ်းပူၼ်ႉၵႂႃႇပွၵ်ႈၼႃႈ။


### 3. Loop ၼႂ်းၸူဝ်ႈပၢၼ်တႄႉ (Real-world Examples)
* **Music Player:** ႁဵတ်းပၼ်ႇတူင် (လဵၼ်ႈယဝ်ႉၵေႃႈ လဵၼ်ႈထႅင်ႈ - Repeat) ၵႂၢမ်းၼိုင်ႈႁူဝ် တေႃႇၵူၼ်းၸႂ်ႉ ၼဵၵ်း Stop။
* **ATM:** ပႂ်ႉႁပ်ႉ Password တေႃႇထိုင်တႅမ်ႈမႅၼ်ႈ (ၵူၺ်းၵႃႈ ပၼ်ၸၢမ်းလႆႈ 3 ပွၵ်ႈ)။
* **Facebook Feed:** ၼႄ Post ၵႂႃႇတိၵ်းတိၵ်း တေႃႇမိူဝ်ႈၵူၼ်းၸႂ်ႉ Scroll လူင်းတၢင်းတႂ်ႈ။

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Iteration:** ၵၢၼ်ႁဵတ်းပၼ်ႇတူင်ႇ တွၼ်ႈ Code ၼိုင်ႈႁွပ်ႈ။
* **Loop:** ၶိူင်ႈမိုဝ်းၼႂ်း Program ဢၼ်ၸႂ်ႉတႃႇႁဵတ်း Iteration။
* **Infinite Loop:** ၵၢၼ်ႁဵတ်းပၼ်ႇတူင်ႇ ဢၼ်ဢမ်ႇမေႃၵိုတ်း (ယွၼ်ႉၶေႃႈထၢမ် ပဵၼ် True တႃႇသေႇ)။
* **Counter:** Variable ဢၼ်ၸႂ်ႉတႃႇ "ၼပ်ႉ" ဝႃႈႁဵတ်းပၼ်ႇတူင်ႇ ယဝ်ႉလၢႆပွၵ်ႈယဝ်ႉ။
* **Exit Condition:** ၶေႃႈသင်ႇ ဢၼ်တေႁဵတ်းႁႂ်ႈ Loop ၵိုတ်း (ပဵၼ် False)။
