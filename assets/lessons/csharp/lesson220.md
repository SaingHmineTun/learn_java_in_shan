
# Lesson 220: "Ordering Data"

LINQ မီး Method တႃႇၸတ်းထႅဝ် ဢၼ်လူငၢႆႈ လႄႈ ၸႅင်ႈလႅင်းၼႃႇၶႃႈ။

### 1. OrderBy (ၸတ်းထႅဝ်ၶိုၼ်ႈ)
ၸႂ်ႉတႃႇၸတ်းၶေႃႈမုၼ်း တႄႇတီႈဢေႇ ၵႂႃႇၸူးၼမ် (Ascending)။ သင်ပဵၼ်လိၵ်ႈၵေႃႈ တႄႇတီႈ A ၵႂႃႇၸူး Z ၶႃႈ။

```csharp
// ၸတ်းၸိုဝ်ႈလုၵ်ႈႁဵၼ်း ၸွမ်းတူဝ်လိၵ်ႈ A-Z
var sortedNames = names.OrderBy(n => n);
```

### 2. OrderByDescending (ၸတ်းထႅဝ်လူင်း)
ၸႂ်ႉတႃႇၸတ်းၶေႃႈမုၼ်း တႄႇတီႈၼမ် ၵႂႃႇၸူးဢေႇ (Descending)။ သင်ပဵၼ် "ၵႃႈၶၼ်" ၵေႃႈ တေဢဝ်ဢၼ်ၵႃႈယႂ်ႇ ဝႆႉၼိူဝ်သုတ်းၶႃႈ။

```csharp
// ၸတ်းဢႃႇယု လုၵ်ႉတီႈၵူၼ်းထဝ်ႈ ၵႂႃႇၸူးၵူၼ်းၼုမ်ႇ
var oldestFirst = students.OrderByDescending(s => s.Age);
```



### 3. ThenBy (ၵၢၼ်ၸတ်းထႅဝ်ၸၼ်ႉသွင်)
မၢင်ပွၵ်ႈ ၶေႃႈမုၼ်းႁဝ်းမီးၵႃႈၶၼ် "မိူၼ်ၵၼ်"။ တူဝ်ယင်ႇ: သင်လုၵ်ႈႁဵၼ်း 2 ၵေႃႉ ဢႃႇယု 20 မိူၼ်ၵၼ်... ႁဝ်းတေလိူၵ်ႈဢဝ်ၽူႈလႂ်ဝႆႉၼိူဝ်? 
ႁဝ်းၸင်ႇလူဝ်ႇၸႂ်ႉ **`.ThenBy()`** တႃႇမၵ်းမၼ်ႈ ပၵ်းၵၢၼ်ထႅင်ႈၸၼ်ႉၼိုင်ႈၶႃႈ။

```csharp
// 1. ၸတ်းၸွမ်း ဝဵင်း (City) ဢွၼ်တၢင်း
// 2. သင်ဝဵင်းမိူၼ်ၵၼ်... ၸတ်းၸွမ်း ၸိုဝ်ႈ (Name)
var complexSort = students.OrderBy(s => s.City)
                          .ThenBy(s => s.Name);
```

### 4. ပၵ်းၵၢၼ်ၸတ်းထႅဝ် ဢၼ်လူဝ်ႇတွင်း
* **`.OrderBy()`**: တေလႆႈၸႂ်ႉပဵၼ် "တူဝ်တႄႇ" (Primary Sort) တႃႇသေႇၶႃႈ။
* **`.ThenBy()`**: တေၸႂ်ႉလႆႈၵေႃႈတေႃႇမိူဝ်ႈ မီး `OrderBy` ယူႇၽၢႆႇၼႃႈမၼ်းယဝ်ႉၶႃႈ။
* ႁဝ်းၸၢင်ႈၸႂ်ႉ `.ThenByDescending()` ၵေႃႈလႆႈ ၸွမ်းၼင်ႇႁဝ်းလူဝ်ႇၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Sorting:** ၵၢၼ်ၸတ်းထႅဝ်ၶေႃႈမုၼ်း ႁႂ်ႈမီးပိူင်သၢင်ႈ။
* **Ascending:** ၵၢၼ်ၸတ်းလုၵ်ႉတီႈ ဢေႇ ၵႂႃႇ ၼမ် (A -> Z, 1 -> 10)။
* **Descending:** ၵၢၼ်ၸတ်းလုၵ်ႉတီႈ ၼမ် ၵႂႃႇ ဢေႇ (Z -> A, 10 -> 1)။
* **Primary vs Secondary Sort:** ၵၢၼ်ၸတ်းထႅဝ်ၸၼ်ႉၼိုင်ႈ လႄႈ ၸၼ်ႉသွင် ဢၼ်သိုပ်ႇၵၼ်။
