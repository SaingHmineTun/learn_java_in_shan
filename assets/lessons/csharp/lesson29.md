# Lesson 29: "Increment and Decrement" (ၵၢၼ်ထႅမ်ၼိုင်ႈ လႄႈ ယွမ်းၼိုင်ႈ)

မိူဝ်ႈႁဝ်းၶႂ်ႈထႅမ်ၵႃႈၶၼ်သႂ်ႇၼႂ်း Variable ထႅင်ႈ **1** တူဝ်ၼၼ်ႉ၊ 
တႅၼ်းတၢင်တေတႅမ်ႈ `x = x + 1;` ဢမ်ႇၼၼ် `x += 1;` ၼၼ်ႉ 
ႁဝ်းၸႂ်ႉၶိူင်ႈမၢႆပွတ်းဢွၼ်ႇၼႆႉလႆႈၶႃႈ။

### 1. ၶိူင်ႈမၢႆပိုၼ်ႉထၢၼ်
* **Increment (`++`):** ထႅမ် 1 သႂ်ႇၼႂ်း Variable။
* **Decrement (`--`):** ယွမ်း 1 တီႈၼႂ်း Variable။

### 2. Pre-increment vs. Post-increment
ၼႆႉပဵၼ်တွၼ်ႈ ဢၼ်လုၵ်ႈႁဵၼ်း ၵႆႉယုင်ႈသေပိူၼ်ႈၶႃႈ။ မၼ်းပႅၵ်ႇၵၼ်တီႈ **"ၶၢဝ်းယၢမ်း"** ဢၼ်မၼ်းထႅမ်ၼၼ်ႉယဝ်ႉ။

#### **A. Post-increment (`x++`)**
* **ၵၢၼ်ႁဵတ်း:** ဢဝ်ၵႃႈၶၼ် "ၵဝ်ႇ" မႃးၸႂ်ႉဢွၼ်တၢင်း၊ ယဝ်ႉၸင်ႇ "ထႅမ် 1" သႂ်ႇဝၢႆးလင်။
```csharp
int x = 5;
int y = x++; // y တေလႆႈ 5 (ၵႃႈၶၼ်ၵဝ်ႇ)၊ ဝၢႆးၼၼ်ႉ x ၸင်ႇပဵၼ် 6
```

#### **B. Pre-increment (`++x`)**
* **ၵၢၼ်ႁဵတ်း:** "ထႅမ် 1" ဢွၼ်တၢင်း၊ ယဝ်ႉၸင်ႇဢဝ်ၵႃႈၶၼ် "မႂ်ႇ" မႃးၸႂ်ႉ။
```csharp
int a = 5;
int b = ++a; // a ပဵၼ် 6 ဢွၼ်တၢင်း၊ ယဝ်ႉ b ၵေႃႈလႆႈ 6 ၸွမ်း
```



### 3. တူဝ်ယင်ႇတႃႇယွမ်းၼိုင်ႈ (Decrement)
မၼ်းၵေႃႈႁဵတ်းၵၢၼ်မိူၼ်ၵၼ်တင်း Increment ၼၼ်ႉယဝ်ႉၶႃႈ။
* `x--`: ၸႂ်ႉၵဝ်ႇ ယဝ်ႉၸင်ႇယွမ်း။
* `--x`: ယွမ်းဢွၼ်တၢင်း ယဝ်ႉၸင်ႇၸႂ်ႉ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Increment:** ၵၢၼ်ထႅမ်ၵႃႈၶၼ်ၶိုၼ်ႈ 1။
* **Decrement:** ၵၢၼ်ယွမ်းၵႃႈၶၼ်လူင်း 1။
* **Prefix:** ၶိူင်ႈမၢႆဢၼ်ဝႆႉၽၢႆႇၼႃႈ Variable (တူဝ်ယင်ႇ: `++x`)။
* **Postfix:** ၶိူင်ႈမၢႆဢၼ်ဝႆႉၽၢႆႇလင် Variable (တူဝ်ယင်ႇ: `x++`)။
* **Side Effect:** ၵၢၼ်လႅၵ်ႈလၢႆႈၵႃႈၶၼ် Variable ဢၼ်ပဵၼ်ၸွမ်း ဝၢႆးသေၼပ်ႉသၢင်ႈယဝ်ႉ။
