### 2. SVG (Vector) Images - တူဝ်လိူၵ်ႈဢၼ်လီသုတ်း
တွၼ်ႈတႃႇႁႂ်ႈ Logo ဢမ်ႇၼၼ် Icon App ႁဝ်း "ၸႅင်ႈလီ" (Sharp) တႃႇသေႇၼၼ်ႉ ႁဝ်းထုၵ်ႇလီၸႂ်ႉၾၢႆႇ **SVG** ၶႃႈ။
* **SVG** ဢမ်ႇၸႂ်ႉ Pixel တွၼ်ႈတႃႇသၢင်ႈႁၢင်ႈ၊ မၼ်းၸႂ်ႉ "မၢႆသွၼ်ႇ" (Mathematical equations)။
* ဢမ်ႇဝႃႈၶႅပ်းႁၢင်ႈႁဝ်း ယႂ်ႇတၢၼ်ႇႁိုဝ်ၵေႃႈယဝ်ႉ မၼ်းတေၸႅင်ႈလီယူႇတႃႇသေႇ ဢမ်ႇဝၢဝ်းၶႃႈ။



### 3. Font Icons (ၵၢၼ်ၸႂ်ႉ Font တႅၼ်းႁၢင်ႈ)
ၼႂ်း App ၶိုတ်းၵၢပ်ႈ ႁဝ်းၵႆႉၸႂ်ႉ Font မိူၼ်ၼင်ႇ **Font Awesome** ဢမ်ႇၼၼ် **Material Icons** တႅၼ်းၵၢၼ်ၸႂ်ႉၶႅပ်းႁၢင်ႈ တွၼ်ႈတႃႇတုမ်ႇ Buttons ၶႃႈ။
* **လွင်ႈလီ:** ႁဝ်းၸၢင်ႈလႅၵ်ႈပိၼ်ႇ "သီ" (Color) လႄႈ "တၢင်းယႂ်ႇ" (Size) ၶွင် Icon ၼၼ်ႉ လူၺ်ႈတႅမ်ႈ Code C# ဢမ်ႇၼၼ် XAML ငၢႆႈငၢႆႈ ၶႃႈ။

### 4. တူဝ်ယင်ႇၵၢၼ်ၸႂ်ႉ Image ၼႂ်း XAML

```xml
<Image Source="shan_logo.png" 
       WidthRequest="200" 
       HeightRequest="200"
       Aspect="AspectFit" />

```

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Screen Density (DPI/PPI):** တၢင်းၼမ်ၶွင် "မၢႆသီ" (Pixels) ၼႂ်းၼႃႈၸေႃး 1 ၼိဝ်ႉ။
* **Vector (SVG):** ၾၢႆႇႁၢင်ႈဢၼ်ယႂ်ႇလႆႈလူၺ်ႈဢမ်ႇလူႉ (Resolution independent)။
* **Raster (PNG/JPG):** ၾၢႆႇႁၢင်ႈဢၼ်ၸႂ်ႉ Pixel တွၼ်ႈတႃႇသၢင်ႈ (သင်ယိုတ်ႈယႂ်ႇပူၼ်ႉတေဝၢဝ်း)။
* **Aspect:** ပိူင်မၵ်းမၼ်ႈဝႃႈ ၶႅပ်းႁၢင်ႈတေလႆႈဝၢင်းတူဝ်ၸိူင်ႉႁိုဝ် (မိူၼ်ၼင်ႇ AspectFit, AspectFill)။
