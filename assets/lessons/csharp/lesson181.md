# Lesson 181: "The Dictionary<K, V>"

**`Dictionary<K, V>`** သိမ်းၶေႃႈမုၼ်းပဵၼ်ၵူပ်း (Pairs)။ ႁဝ်းၸႂ်ႉ **Key** တႃႇႁႃ **Value** ဢၼ်ၵွင်ႉၵၢႆႇၵၼ်ဝႆႉၼၼ်ႉၶႃႈ။

### 1. တူဝ် `<K, V>` ပွင်ႇဝႃႈသင်?
* **`K` (Key):** ပဵၼ် "တူဝ်ၸိၼႄ" ဢမ်ႇၼၼ် "ၸိုဝ်ႈႁႃ"။ မၼ်းတေလႆႈ **ပႅၵ်ႇပိူင်ၵၼ် (Unique)** တႃႇသေႇ၊ တေမိူၼ်ၵၼ်ဢမ်ႇလႆႈၶႃႈ။
* **`V` (Value):** ပဵၼ် "ၶေႃႈမုၼ်း" ဢၼ်သိမ်းဝႆႉ။ ၼႆႉတႄႉ မိူၼ်ၵၼ်ၵေႃႈလႆႈၶႃႈ။



### 2. ၵၢၼ်သၢင်ႈ (Declaration)
ဝူၼ်ႉတူၺ်းဝႃႈ ၶူးသၢႆမၢဝ်းၶႂ်ႈသိမ်း "မၢႆၾူၼ်း" ၸွမ်တင်း "ၸိုဝ်ႈၵူၼ်း"။
* Key = `string` (ၸိုဝ်ႈၵူၼ်း)
* Value = `string` (မၢႆၾူၼ်း)

```csharp
Dictionary<string, string> shanPhoneBook = new Dictionary<string, string>();
```

### 3. ၵၢၼ်ထႅမ်ၶေႃႈမုၼ်း (Adding Data)
ႁဝ်းၸၢင်ႈၸႂ်ႉ Method **`.Add()`** ဢမ်ႇၼၼ် ၸႂ်ႉ **`[]`** တႃႇသႂ်ႇၵႃႈၶၼ်ၵမ်းလဵဝ်ၶႃႈ။

```csharp
// လၢႆးထႅမ် 1
shanPhoneBook.Add("Sai Mao", "09-123456");

// လၢႆးထႅမ် 2 (သင်မီး Key ၼႆႉဝႆႉယဝ်ႉ မၼ်းတေ Update ပၼ်ၵမ်းလဵဝ်)
shanPhoneBook["Nang Mo"] = "09-777888";
```

### 4. ၵၢၼ်ႁွင်ႉၸႂ်ႉ (Accessing Data)
ႁဝ်းဢမ်ႇၸႂ်ႉ Number Index (0, 1, 2) ၼင်ႇ List ယဝ်ႉ၊ ႁဝ်းၸႂ်ႉ **Key** မၼ်းတႃႇႁွင်ႉဢဝ် Value ၶႃႈ။

```csharp
string maoNumber = shanPhoneBook["Sai Mao"]; // ဢွၵ်ႇ: "09-123456"
```

### 5. ႁဵတ်းသင်ၸင်ႇလူဝ်ႇၸႂ်ႉ?
1.  **Fast Lookup:** ႁႃၶေႃႈမုၼ်းလႆႈဝႆးၼႃႇ (ဢမ်ႇဝႃႈတေမီးၶေႃႈမုၼ်း 1,000,000 တူဝ်ၵေႃႈ ႁႃၸွမ်း Key လႆႈၵမ်းလဵဝ်)။
2.  **Meaningful Index:** တႅၼ်းဢၼ်တေမၢႆတွင်းဝႃႈ `List[5]` ပဵၼ်သင်... ႁဝ်းမၢႆတွင်း `Dictionary["SaiMao"]` ၼၼ်ႉငၢႆႈလိူဝ်ၶႃႈ။
3.  **Unique Keys:** ၸွႆးႁႄႉၵၢင်ႈဢမ်ႇပၼ်မီးၶေႃႈမုၼ်းတူဝ်ၸိမိူၼ်ၵၼ် (မိူၼ်ၼင်ႇ Student ID ဢမ်ႇၼၼ် Username)။

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Key-Value Pair (KVP):** ၶေႃႈမုၼ်း ဢၼ်ၵွင်ႉၵၢႆႇၵၼ်ပဵၼ်ၵူပ်း။
* **Unique Constraint:** ပၵ်းၵၢၼ်ဢၼ်မၵ်းမၼ်ႈဝႃႈ Key တေလႆႈပႅၵ်ႇပိူင်ၵၼ်တႃႇသေႇ။
* **Hashing:** လၢႆးႁဵတ်းၵၢၼ်ၽၢႆႇၼႂ်း ဢၼ်ႁဵတ်းႁႂ်ႈ Dictionary ႁႃၶေႃႈမုၼ်းလႆႈဝႆး။
* **Lookup:** ၵၢၼ်ႁႃၶေႃႈမုၼ်း လုၵ်ႉတီႈ Key ၵႂႃႇၸူး Value။
