# Lesson 252: "The History of Windows Desktop Development: WinForms vs. WPF"

မိူဝ်ႈႁဝ်းတေတႅမ်ႈ App ၼိူဝ် Desktop ၼၼ်ႉ ၼႂ်း C# .NET မီးတၢင်းလိူၵ်ႈလူင်ယူႇ 2 ပိူင်ၶႃႈ။ ႁဝ်းမႃးတူၺ်းပိုၼ်း လႄႈ လွင်ႈပႅၵ်ႇပိူင်မၼ်းၶႃႈ။

### 1. Windows Forms (WinForms) - ပိူင်ၵဝ်ႇဢၼ်ငၢႆႈ
WinForms မီးမႃးၸဵမ်မိူဝ်ႈပီ **2002** ၶႃႈ။ မၼ်းပဵၼ်ပိူင် **"Drag and Drop"** (လၢၵ်ႈမႃးဝၢင်း) ဢၼ်ငၢႆႈသုတ်း။
* **လၢႆးႁဵတ်းၵၢၼ်:** မၼ်းၸႂ်ႉ Windows API ၶွင်တူဝ် Windows ၵမ်းလဵဝ် တႃႇတႅမ်ႈ Button ဢမ်ႇၼၼ် TextBox။
* **ၽွၼ်းလီ:** သင်ႁဝ်းၶႂ်ႈလႆႈ App ဢၼ်ၸႂ်ႉၼႂ်း Office ငၢႆႈငၢႆႈ၊ ဢမ်ႇလူဝ်ႇလွင်ႈႁၢင်ႈလီၼမ်၊ WinForms ဝႆးလိူဝ်ပိူၼ်ႈၶႃႈ။
* **ၽွၼ်းၸႃႉ:** မၼ်းယၢပ်ႇ တႃႇႁဵတ်းႁႂ်ႈ App ႁၢင်ႈလီ (Custom UI) လႄႈ မၼ်းဢမ်ႇသူႈ Support ၼႃႈၸေႃး ဢၼ်ယႂ်ႇယႂ်ႇၶႃႈ။

### 2. Windows Presentation Foundation (WPF) - ပိူင်မႂ်ႇဢၼ်ႁႅင်း
WPF မႃးၼႂ်းပီ **2006** သေ ပိၼ်ႇလၢႆးတႅမ်ႈ App ၵႂႃႇတင်းမူတ်းၶႃႈ။
* **လၢႆးႁဵတ်းၵၢၼ်:** မၼ်းၸႂ်ႉ **DirectX** (ပိူင်လဵၼ်ႈ Game) တႃႇတႅမ်ႈ UI။ မၼ်းၸႅၵ်ႇလွင်ႈ "တႅမ်ႈႁၢင်ႈ" (Design) လႄႈ "တႅမ်ႈ Logic" ဢွၵ်ႇၵၼ်ၶႃႈ။
* **XAML:** WPF ၸႂ်ႉလိၵ်ႈ **XAML** (မိူၼ် HTML) တႃႇတႅမ်ႈၼႃႈၸေႃးၶႃႈ။
* **ၽွၼ်းလီ:** ႁဵတ်းႁႂ်ႈ App ႁၢင်ႈလီ၊ လဵၼ်ႈ Animation လႆႈလႄႈ Support ၼႃႈၸေႃးတႃႇ Resolution ၵူႈဢၼ်ၶႃႈ။
* **ၽွၼ်းၸႃႉ:** တေလႆႈႁဵၼ်းလိၵ်ႈ XAML ထႅင်ႈဢၼ်ၼိုင်ႈ၊ လႅၼ်ႈယၢပ်ႇလိူဝ် WinForms ဢိတ်းၼိုင်ႈၶႃႈ။

### လွင်ႈပႅၵ်ႇပိူင် ၶႃ 2 ဢၼ် (Comparison Table)

| တၢင်းလိူၵ်ႈ | WinForms                  | WPF |
| :--- |:--------------------------| :--- |
| **တႄႇမီးမိူဝ်ႈ** | 2002                      | 2006 |
| **လၢႆး Design** | Drag & Drop (Pixel-based) | XAML (Vector-based) |
| **Graphics** | GDI+ (ထိူင်းလိူဝ်)        | DirectX (ဝႆးလႄႈႁၢင်ႈလီ) |
| **Data Binding** | ဢေႇ/ယၢပ်ႇ                 | ႁႅင်း/ငၢႆႈ (Standard) |
| **လွင်ႈယၢပ်ႇ** | တႅမ်ႈငၢႆႈ (Easy)          | တေလႆႈႁဵၼ်းၼမ် (Learning Curve) |

### ႁဵတ်းသင်ႁဝ်းၸင်ႇတေလႆႈႁဵၼ်းတင်းသွင်ပိူင်?
ၼႂ်းၵၢၼ်ႁဵၼ်း **Module 21** ၼႆႉ ႁဝ်းတေတႄႇလူၺ်ႈ **WinForms** ဢွၼ်တၢင်း တႃႇႁႂ်ႈပွင်ႇၸႂ်လွင်ႈ **Event-Driven Programming** (ၵၢၼ်တႅမ်ႈ Code ၸွမ်းၼင်ႇၵၢၼ်ၼဵၵ်း တုမ်ႇ)။ 
ဝၢႆးၼၼ်ႉ ႁဝ်းတေသိုပ်ႇၵႂႃႇ **WPF** ဢၼ်ပဵၼ် Standard တွၼ်ႈတႃႇၵူၼ်းတႅမ်ႈ App ပၢၼ်မႂ်ႇၶႃႈဢေႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Legacy:** ပိူင်ၵဝ်ႇ ဢၼ်တိုၵ်ႉၸႂ်ႉလႆႈယူႇ (WinForms ၵႆႉထုၵ်ႇမၢႆတွင်းၸိူင်ႉၼႆ)။
* **Vector-based:** ႁၢင်ႈၾၢင်ဢၼ်ၸၢင်ႈယိုတ်းလႆႈ လူၺ်ႈဢမ်ႇလူႉ (မိူၼ်ၼႂ်း WPF)။
* **Pixel-based:** ႁၢင်ႈၾၢင်ဢၼ်မၵ်းမၼ်ႈတႅတ်ႈတေႃး (သင်ယိုတ်းၼႃႈၸေႃး ႁၢင်ႈတေလႅဝ် - မိူၼ်ၼႂ်း WinForms)။
* **DirectX:** ပိူင်ၸတ်းၵၢၼ် Graphics ၼႂ်း Windows ဢၼ်ဝႆးၼႃႇ။
