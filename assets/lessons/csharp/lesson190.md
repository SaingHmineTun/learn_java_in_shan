# Lesson 190: "Introduction to Exceptions: Errors vs. Exceptions"

ၼႂ်းၵၢၼ်တႅမ်ႈပိူင်သၢင်ႈ (Programming) ၼၼ်ႉ ပၼ်ႁႃဢၼ်ႁဵတ်းႁႂ်ႈ Code ႁဝ်းဢမ်ႇလႅၼ်ႈလႆႈၼၼ်ႉ ႁဝ်းၸၢင်ႈၸႅၵ်ႇဢွၵ်ႇပဵၼ် 2 ပိူင်လူင်ၶႃႈ။

### 1. Error (ပၼ်ႁႃဢၼ်ၵႄႈဢမ်ႇလႆႈ)
**Errors** ၼႆႉ ၵႆႉပဵၼ်ပၼ်ႁႃဢၼ် "ႁႅင်း" သေ တူဝ် Program ႁဝ်းဢမ်ႇၸၢင်ႈၵႄႈလႆႈ (Unrecoverable)။ မၼ်းၵႆႉပဵၼ်ပၼ်ႁႃလုၵ်ႉတီႈ ပိူင်သၢင်ႈၶွမ်း (System) ဢမ်ႇၼၼ် ၶိူင်ႈ (Hardware) ၶႃႈ။
* **တူဝ်ယင်ႇ:** Memory တဵမ် (OutOfMemoryError), ပိူင်ပၼ်ႇႁႃဢမ်ႇသုတ်းပေႃး Memory မူတ်း (StackOverflowError)။
* **လွၼ်ႉမၼ်း:** သင်ပဵၼ် Error... App တေပိတ်းၵႂႃႇၵမ်းလဵဝ် သေႁဝ်းၵႄႈၼႂ်း Code ႁဝ်းဢမ်ႇလႆႈၶႃႈ။

### 2. Exception (ပၼ်ႁႃဢၼ်ႁဝ်း "ႁပ်ႉမိုဝ်း" လႆႈ)
**Exceptions** ၼႆႉ ပဵၼ်လွင်ႈတၢင်းဢၼ်ပႅၵ်ႇပိူင်ဢွၵ်ႇၵႂႃႇ မိူဝ်ႈ App တိုၵ်ႉလႅၼ်ႈယူႇ (Runtime)။ ၼႆႉပဵၼ်ပၼ်ႁႃဢၼ်ႁဝ်းၸၢင်ႈ "မၢႆတွင်းဝႆႉဢွၼ်တၢင်း" လႆႈ သေ ႁဝ်းၸၢင်ႈတႅမ်ႈ Code တႃႇ "ႁပ်ႉမိုဝ်း" (Handle) မၼ်း ႁႂ်ႈ App ႁဝ်းသိုပ်ႇလႅၼ်ႈၵႂႃႇလႆႈယူႇၶႃႈ။
* **တူဝ်ယင်ႇ:** ႁႃ File ဢမ်ႇႁၼ်, ၵူၼ်းၸႂ်ႉတႅမ်ႈလိၵ်ႈတႅၼ်းတူဝ်ၼပ်ႉ, ဢမ်ႇၼၼ် ဢဝ်တူဝ်ၼပ်ႉမႃးႁၢၼ်လူၺ်ႈ သုၼ် (0)။



### 3. ပၵ်းၵၢၼ်တႅၵ်ႈၵၼ်: Error vs Exception

| လွင်ႈပႅၵ်ႇပိူင် | Error (ဢၼ်ၽိတ်းႁႅင်း) | Exception (ဢၼ်ပဵၼ်လႆႈ) |
| :--- | :--- | :--- |
| **ၸၼ်ႉၵၢၼ်ၵႄႈ** | ၵႄႈဢမ်ႇလႆႈ | ၵႄႈလႆႈ / ႁပ်ႉမိုဝ်းလႆႈ |
| **တၢင်းပဵၼ်** | လုၵ်ႉတီႈၶိူင်ႈ / ပိူင်သၢင်ႈလူင် | လုၵ်ႉတီႈ Code / ၵူၼ်းၸႂ်ႉ |
| **လၢႆးႁပ်ႉမိုဝ်း** | ဢမ်ႇလူဝ်ႇ Try-Catch | လူဝ်ႇၸႂ်ႉ **Try-Catch** |
| **တူဝ်ယင်ႇ** | Memory မူတ်း | ႁႃ File ဢမ်ႇႁၼ် |

### 4. ႁဵတ်းသင်ၸင်ႇလူဝ်ႇၸႂ်ႉ Exception Handling?
ဝူၼ်ႉတူၺ်းဝႃႈ ၶူးတႅမ်ႈ App **"Shan Dictionary"**။ မိူဝ်ႈလုၵ်ႈႁဵၼ်းႁႃၶေႃႈၵႂၢမ်း... သင်မၼ်းဢမ်ႇမီး Internet သေ App ၶျႃး (Crash) ပႅတ်ႈၵမ်းလဵဝ်ၼႆ... ၵူၼ်းၸႂ်ႉတေဢမ်ႇထုၵ်ႇၸႂ်ၶႃႈ။
* သင်ႁဝ်းၸႂ်ႉ **Exception Handling**... ႁဝ်းၸၢင်ႈလၢတ်ႈၼႄလုၵ်ႈႁဵၼ်းဝႃႈ "Internet ၶူးလူႉယူႇၶႃႈ၊ ၵိုတ်းၵမ်းၼိုင်ႈၼႃ" တႅၼ်းဢၼ် App တေလူႉၵႂႃႇၼၼ်ႉယဝ်ႉၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Runtime Error:** ပၼ်ႁႃဢၼ်ၵဵတ်းၶိုၼ်းမိူဝ်ႈ App တိုၵ်ႉလႅၼ်ႈယူႇ။
* **Robustness:** လွင်ႈၶႅင်ၵႅၼ်ႇၶွင် Code ဢၼ်ၸၢင်ႈႁပ်ႉမိုဝ်းတင်းပၼ်ႁႃလႆႈ။
* **Exception Object:** မိူဝ်ႈပၼ်ႁႃၵဵတ်းၶိုၼ်း... C# တေသူင်ႇ "ၶေႃႈမုၼ်း" ဢၼ်ၼိုင်ႈဢွၵ်ႇမႃး ဢၼ်မီးလွင်ႈပၼ်ႁႃၼၼ်ႉ။
* **Crash:** ၵၢၼ်ဢၼ် Program ပိတ်းၵႂႃႇၵမ်းလဵဝ် ယွၼ်ႉမီးပၼ်ႁႃဢၼ်ႁပ်ႉမိုဝ်းဢမ်ႇလႆႈ။
