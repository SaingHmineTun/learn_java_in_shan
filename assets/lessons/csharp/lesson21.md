# Lesson 21: "Constants and Read-Only Fields"

မၢင်ပွၵ်ႈ ႁဝ်းမီးၶေႃႈမုၼ်း ဢၼ်ႁဝ်းဢမ်ႇၶႂ်ႈႁႂ်ႈမၼ်းလႅၵ်ႈလၢႆႈ (Immutable) တေႃႇ Program ပေႃးသုတ်းယဝ်ႉ။ မိူၼ်ၼင်ႇ ၵႃႈၶၼ် Pi (3.14) ဢမ်ႇၼၼ် ၸိုဝ်ႈႁူင်းႁဵၼ်းႁဝ်းၶႃႈ။

### 1. Constants (`const`)
**const** ၼႆႉပဵၼ်ၶေႃႈမုၼ်း ဢၼ်တေလႆႈသႂ်ႇၵႃႈၶၼ် (Value) တႅတ်ႉတႅတ်ႉတေႃးတေႃး ၸဵမ်မိူဝ်ႈတႅမ်ႈ Code (Compile-time)။
* **လၢၵ်ႇလၢႆး:** ဝၢႆးသေတႅမ်ႈယဝ်ႉ ႁဝ်းတေဢဝ် Code ထႅဝ်တၢင်ႇဢၼ် မႃးလႅၵ်ႈလၢႆႈမၼ်းဢမ်ႇလႆႈယဝ်ႉ။
* **တူဝ်ယင်ႇ:**
  ```csharp
  const double Pi = 3.14159;
  const string SchoolName = "Thung Mao Kham";
  
  // Pi = 3.15; // ၼႆႉတေပဵၼ် Error ၵမ်းလဵဝ်ၶႃႈ
  ```

### 2. Read-Only Fields (`readonly`)
**readonly** ၼႆႉသမ်ႉ ပဵၼ် Variable ဢၼ်ႁဝ်းၸၢင်ႈ "ပၼ်ၵႃႈၶၼ်" မၼ်းလႆႈ မိူဝ်ႈ App တႄႇႁဵတ်းၵၢၼ် (Runtime) ၵူၺ်း။
* **လၢၵ်ႇလၢႆး:** ၵႆႉၸႂ်ႉၼႂ်း Class တႃႇပိုတ်ႇ App မႃးယဝ်ႉ ၸင်ႇဢဝ်ၶေႃႈမုၼ်း မႃးတီႈ Database ဢမ်ႇၼၼ် User သႂ်ႇဝႆႉသေ ဢမ်ႇပၼ်လႅၵ်ႈလၢႆႈထႅင်ႈယဝ်ႉ။
* **တူဝ်ယင်ႇ:**
  ```csharp
  readonly DateTime appStartTime = DateTime.Now; 
  // မၼ်းတေမၢႆတွင်းၶၢဝ်းယၢမ်းပိုတ်ႇ App ဝႆႉသေ လႅၵ်ႈလၢႆႈဢမ်ႇလႆႈယဝ်ႉ
  ```



### 3. မိူဝ်ႈလႂ်လူဝ်ႇၸႂ်ႉသင်?
* **ၸႂ်ႉ const:** မိူဝ်ႈႁဝ်းႁူႉၵႃႈၶၼ်မၼ်းတႅတ်ႉတႅတ်ႉ (Fixed) ၸဵမ်မိူဝ်ႈတႅမ်ႈ Code။ (Compile-time constant)
* **ၸႂ်ႉ readonly:** မိူဝ်ႈႁဝ်းလူဝ်ႇလုၵ်ႉတီႈ "ၵၢၼ်ၼပ်ႉသၢင်ႈ" (Calculation) ဢမ်ႇၼၼ် "လုၵ်ႉတီႈၵူၼ်းၸႂ်ႉ" ၸင်ႇလႆႈၶေႃႈမုၼ်းမႃး သေၶႂ်ႈမၢႆတွင်းဝႆႉႁႂ်ႈမၼ်းၼႅၼ်ႈၼႃ။ (Runtime constant)

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Immutable:** ၶေႃႈမုၼ်းဢၼ်လႅၵ်ႈလၢႆႈဢမ်ႇလႆႈ (ပဵၼ်သင်ၵေႃႈ တေပဵၼ်ၼၼ်ယူႇတႃႇသေႇ)။
* **Compile-time:** ၶၢဝ်းယၢမ်းမိူဝ်ႈႁဝ်းတႅမ်ႈ Code လႄႈ Build Project (const ႁဵတ်းၵၢၼ်တီႈၼႆႉ)။
* **Runtime:** ၶၢဝ်းယၢမ်းမိူဝ်ႈၵူၼ်းၸႂ်ႉ ပိုတ်ႇ App ႁဝ်းႁဵတ်းၵၢၼ်ယူႇ (readonly ၸၢင်ႈလႆႈၵႃႈၶၼ်တီႈၼႆႉ)။
* **Field:** Variable ဢၼ်သၢင်ႈဝႆႉတီႈၼႂ်း Class (ဢမ်ႇၸႂ်ႈၼႂ်း Method ၵူၺ်း)။
