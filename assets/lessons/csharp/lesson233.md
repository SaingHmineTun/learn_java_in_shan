# Lesson 233: "Introduction to Serialization"

ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းမီး Object `Student` ၼႂ်း C# ဢၼ်မီးဝႆႉ Name, Age, လႄႈ Score ၼႆၶႃႈ။
ႁဝ်းဢမ်ႇၸၢင်ႈဢဝ် Object တူဝ်ၼၼ်ႉ "တႅမ်ႈ" ၶဝ်ႈၵႂႃႇၼႂ်း File ၵမ်းလဵဝ်လႆႈ။ 
ႁဝ်းတေလႆႈ "ပိၼ်ႇ" မၼ်းႁႂ်ႈပဵၼ်ထႅဝ်လိၵ်ႈ (String) ဢွၼ်တၢင်းၶႃႈ။

### 1. Serialization ပဵၼ်သင်?
**Serialization** ပဵၼ်လွင်ႈ ဢၼ်ႁဝ်းဢဝ် **Object** (ဢၼ်ယူႇၼႂ်း RAM) မႃးပိၼ်ႇႁၢင်ႈႁႂ်ႈပဵၼ် **Byte Stream** ဢမ်ႇၼၼ် **Text** (မိူၼ်ၼင်ႇ JSON/XML) တႃႇသိမ်းဝႆႉ ဢမ်ႇၼၼ် သူင်ႇၵႂႃႇတၢင်ႇတီႈၶႃႈ။

### 2. Deserialization ပဵၼ်သင်?
မၼ်းပဵၼ်ၵၢၼ်ဢဝ် လိၵ်ႈၼၼ်ႉ ပိၼ်ႇပဵၼ် Object ၶိုၼ်းၶႃႈ။ 
ႁဝ်းလူ File ဢၼ်ပဵၼ် Text ၶဝ်ႈမႃးၼၼ်ႉသေ ဢဝ်မၼ်းမႃးပိၼ်ႇၶိုၼ်းႁႂ်ႈပဵၼ် **C# Object** တူဝ်ၵဝ်ႇၼၼ်ႉၶႃႈ။



### 3. ႁဵတ်းသင်ႁဝ်းၸင်ႇလူဝ်ႇမၼ်း?
* **Data Persistence:** တႃႇသိမ်း List ၶွင်လုၵ်ႈႁဵၼ်းတင်းမူတ်း ဝႆႉၼႂ်း File သေ ပိုတ်ႇလူၶိုၼ်းလႆႈ မိူဝ်ႈပိုတ်ႇ App မႂ်ႇ။
* **Deep Copy:** တႃႇသၢင်ႈ Object တူဝ်မႂ်ႇ ဢၼ်မိူၼ်တူဝ်ၵဝ်ႇၵူႈပိူင်။
* **Communication:** တႃႇသူင်ႇၶေႃႈမုၼ်းလုၵ်ႉတီႈ App C# ၵႂႃႇၸူး App Flutter (Android)။

### 4. ပိူင်သၢင်ႈ (Formats) ဢၼ်ၵႆႉၸႂ်ႉ
ၼႂ်းလုမ်ႈၾႃႉ Programming ယၢမ်းလဵဝ် မီး 2 ပိူင်ဢၼ်ၵူၼ်းၸႂ်ႉၼမ်:
1.  **JSON (JavaScript Object Notation):** လူငၢႆႈ၊ လဵၵ်ႉ၊ လႅၼ်ႈဝႆး။ ပဵၼ်ဢၼ် Standard သုတ်းယၢမ်းလဵဝ် (ႁဝ်းတေႁဵၼ်းဢၼ်ၼႆႉၶႃႈ)။
2.  **XML:** မိူၼ်လိၵ်ႈ HTML၊ ယႂ်ႇလိူဝ် JSON ဢိတ်းၼိုင်ႈ။
3.  **Binary:** ပဵၼ်တူဝ်ၼပ်ႉ 0 1 ၵူၺ်း၊ ၵူၼ်းလူဢမ်ႇပွင်ႇၸႂ် ၵူၺ်းၵႃႈၶွမ်းလူဝႆးသုတ်း။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Serialization:** ၵၢၼ်ပိၼ်ႇ Object ႁႂ်ႈပဵၼ် Data ဢၼ်သူင်ႇလႆႈ (JSON, Byte).
* **Deserialization:** ၵၢၼ်ပိၼ်ႇ Data ၶိုၼ်းႁႂ်ႈပဵၼ် Object ၼႂ်း RAM.
* **State:** ၵႃႈၶၼ် (Data) တင်းမူတ်း ဢၼ်ယူႇၼႂ်း Object
* **POCO (Plain Old CLR Object):** Class လၢႆလၢႆ ဢၼ်မီး Property (မိူၼ်ၼင်ႇ Student class ႁဝ်းၼၼ်ႉၶႃႈ) ၵူၺ်း.
