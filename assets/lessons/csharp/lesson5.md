# Lesson 5: "Project File Structure" (ပိူင်သၢင်ႈ File ၼႂ်း Project ႁဝ်း)

မိူဝ်ႈႁဝ်းသၢင်ႈ Project ယဝ်ႉၼၼ်ႉ ၼႂ်း Folder ႁဝ်းတေမီး File လၢႆလၢႆပိူင်။ ႁဝ်းလူဝ်ႇမႃးႁူႉၸတ်း File လူင် 3 ပိူင် ဢၼ်မီးၼႃႈတီႈပႅၵ်ႇၵၼ်ၶႃႈ။

### 1. .sln (Solution File)
File ၼႆႉမိူၼ်ၼင်ႇ "ၵူႈမိုဝ်းလူင်" (Container) ယဝ်ႉ။ ၼႂ်း Solution ၼိုင်ႈဢၼ် မၼ်းမီးလႆႈလၢႆလၢႆ Project။ ပေႃးႁဝ်းတေပိုတ်ႇၵၢၼ် (Work) ႁဝ်းၶိုၼ်ႈမႃးထႅင်ႈပွၵ်ႈၼိုင်ႈၼၼ်ႉ ႁဝ်းတေလႆႈတဵၵ်းပိုတ်ႇတီႈ File `.sln` ၼႆႉဢေႃႈ။

### 2. .csproj (C# Project File)
ၼႆႉပဵၼ် File ဢၼ်မၢႆတွင်း "ၶေႃႈမုၼ်း Project" ႁဝ်း။ ၼႂ်းမၼ်းတေမီး:
* ႁဝ်းၸႂ်ႉ .NET Version လႂ်? (တူဝ်ယင်ႇ: `net8.0`)
* ႁဝ်းလူင်း Library (NuGet Packages) သင်ဝႆႉၽွင်ႈ?
* Project ႁဝ်းပဵၼ်သႅၼ်းလႂ်? (Console ႁႃႉ၊ Web ႁႃႉ?)

### 3. Folder bin လႄႈ obj
Folder ၸိူဝ်းၼႆႉ မၼ်းတေဢွၵ်ႇမႃးႁင်းမၼ်း မိူဝ်ႈႁဝ်း "Build" ဢမ်ႇၼၼ် "Run" Code ႁဝ်း။
* **obj (Object):** ဝႆႉတႃႇသိမ်း File ၸူဝ်ႈၶၢဝ်ႇ မိူဝ်ႈ Compiler တူၵ်းတႅင်ႈ Code ယူႇၼၼ်ႉ။
* **bin (Binary):** ၼႆႉပဵၼ်တီႈဢၼ် "App တႄႉ" (File .exe) ႁဝ်းၵႂႃႇယူႇဝႆႉယဝ်ႉ။ ပေႃးႁဝ်းတေဢဝ် App ၵႂႃႇပၼ်ပိူၼ်ႈ ႁဝ်းတေလႆႈၵႂႃႇဢဝ်ၼႂ်း Folder `bin/Debug` ဢမ်ႇၼၼ် `bin/Release` ၼၼ်ႉဢေႃႈ။

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Solution:** ၸုမ်းႁွင်ႈၵၢၼ်လူင် ဢၼ်ႁူမ်ႈဢဝ် Project လၢႆလၢႆဢၼ် ဝႆႉၸွမ်းၵၼ်။
* **Metadata:** ၶေႃႈမုၼ်းဢၼ်လၢတ်ႈၼႄလွင်ႈ "ပိူင်သၢင်ႈ" (မိူၼ်ၼင်ႇ version) ဢမ်ႇၸႂ်ႈ Code ဢၼ်ႁဵတ်းၵၢၼ်တႄႉ။
* **Binary File:** File ဢၼ်ပဵၼ်ၽႃႇသႃႇၶိူင်ႈၸၢၵ်ႈ (0, 1) ဢၼ်ၶွမ်းပိဝ်ႇတႃႇပိုတ်ႇၸႂ်ႉလႆႈ (တူဝ်ယင်ႇ: .exe)။
* **Clean & Rebuild:** ၵၢၼ်လူႉယႃႉ File ၼႂ်း `bin` လႄႈ `obj` ယဝ်ႉ သၢင်ႈမႂ်ႇထႅင်ႈပွၵ်ႈၼိုင်ႈ (ၸႂ်ႉမိူဝ်ႈ App မီး Error ဢၼ်ႁဝ်းႁႃဢမ်ႇႁၼ်)။
