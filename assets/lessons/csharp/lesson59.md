# Lesson 59: "The 'break' Statement: Exiting a Loop Early"

ၼႂ်း **Lesson 59** ၼႆႉ ႁဝ်းတေမႃးတူၺ်း **'break'** ဢေႃႈ။ 
မၼ်းပဵၼ်ၶေႃႈသင်ႇဢၼ်ၸႂ်ႉတႃႇ "တတ်းၶၢတ်ႇ" ၵၢၼ်ႁဵတ်းပၼ်ႇတူင်ႇၵမ်းလဵဝ်၊ 
ဢမ်ႇပႂ်ႉတေႃႇပေႃးထိုင် **Exit Condition** ၶွင် Loop ၼၼ်ႉၶႃႈ။

### 1. ပိူင်ၵၢၼ် (How it Works)
မိူဝ်ႈ Program လႅၼ်ႈမႃးၺႃးၶေႃႈၵႂၢမ်း `break;`...
* မၼ်းတေ "ၵိုတ်း" Loop ၼၼ်ႉၵမ်းလဵဝ်။
* ယဝ်ႉသမ်ႉ "ဝႅၼ်" (Jump) ဢွၵ်ႇၵႂႃႇႁဵတ်း Code တွၼ်ႈဢၼ်ယူႇလင် Loop ၼၼ်ႉၶႃႈ။



### 2. တူဝ်ယင်ႇ Code တႄႉတႄႉ
ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းၼပ်ႉ 1 ထိုင် 10၊ ၵူၺ်းၵႃႈ ပေႃးထိုင် 5 ႁဝ်းၶႂ်ႈၵိုတ်းယဝ်ႉၶႃႈ။

```csharp
for (int i = 1; i <= 10; i++)
{
    if (i == 5)
    {
        break; // တတ်းၶၢတ်ႇၵၢၼ်ႁဵတ်းပၼ်ႇတူင်ႇၵမ်းလဵဝ်
    }
    Console.WriteLine("Number: " + i);
}

Console.WriteLine("Out of the loop!");
```

**ၽိူၼ်ဢွၵ်ႇ (Output):**
```text
Number: 1
Number: 2
Number: 3
Number: 4
Out of the loop!
```
*(မၢႆတွင်း: 5 တေဢမ်ႇၼႄယဝ်ႉ ယွၼ်ႉမၼ်းၺႃး break သေဢွၵ်ႇၵႂႃႇဢွၼ်တၢင်းၶႃႈ)*

### 3. မိူဝ်ႈလႂ်လူဝ်ႇၸႂ်ႉ?
* **Searching:** မိူဝ်ႈႁဝ်းႁႃ "ၶေႃႈမုၼ်း" ၼႂ်းသဵၼ်ႈၸိုဝ်ႈ (Array)။ ပေႃးႁၼ်ယဝ်ႉၵေႃႈ ဢမ်ႇလူဝ်ႇသိုပ်ႇပၼ်ႇတူင်ႇယဝ်ႉၶႃႈ (ၼႆႉတေၸွႆးႁႂ်ႈ App ၽႂ်းၶိုၼ်ႈ)။
* **Infinite Loop Control:** ၸႂ်ႉၼႂ်း `while (true)` တႃႇဢွၵ်ႇမိူဝ်ႈ User ၼဵၵ်း Stop ဢမ်ႇၼၼ် တႅမ်ႈၶေႃႈမုၼ်းမႅၼ်ႈ။


### 4. လွင်ႈလူဝ်ႇၾၢင်ႉ (Nested Loops)
သင်ဝႃႈႁဝ်းၸႂ်ႉ `break` ၼႂ်း **Inner Loop** (Loop ၽၢႆႇၼႂ်း)...
* မၼ်းတေဢွၵ်ႇပၼ်တီႈ **Loop ဢၼ်မၼ်းယူႇဝႆႉ** ၼၼ်ႉၵူၺ်းၶႃႈ။
* **Outer Loop** (Loop ၽၢႆႇၼွၵ်ႈ) တိုၵ်ႉတေသိုပ်ႇႁဵတ်းၵၢၼ်ယူႇၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Break Statement:** ၶေႃႈသင်ႇတတ်းၶၢတ်ႇ ၵၢၼ်ႁဵတ်းပၼ်ႇတူင်ႇ။
* **Jump Statement:** ၸုမ်းၶေႃႈသင်ႇဢၼ်လႅၵ်ႈလၢႆႈတၢင်းပႆ Program ႁႂ်ႈဝႅၼ်ၶၢမ်ႈ Code။
* **Execution Flow:** တၢင်းပႆၶွင် Program ဢၼ်လုၵ်ႉတၢင်းၼိူဝ်လူင်းတၢင်းတႂ်ႈ။
