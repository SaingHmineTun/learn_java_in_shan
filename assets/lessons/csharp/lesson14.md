# Lesson 14: "Variables: Declaration, Initialization, and Naming Conventions"

ၼႂ်းၵၢၼ်တႅမ်ႈ App ၼၼ်ႉ **Variable** ၼႆႉမိူၼ်ၼင်ႇ "ၵွၵ်း" ဢၼ်ႁဝ်းပၼ်ၸိုဝ်ႈဝႆႉသေ ဢဝ်ၶေႃႈမုၼ်းသႂ်ႇဝႆႉၼႂ်းမၼ်းယဝ်ႉ။ 
တႃႇတေၸႂ်ႉ Variable ၼိုင်ႈဢၼ် ႁဝ်းလူဝ်ႇႁဵတ်းဝႆႉ 2 ၸၼ်ႉၼႆႉ။

### 1. Declaration (ၵၢၼ်ပိုၼ်ၽၢဝ်ႇသၢင်ႈ)
ၼႆႉပဵၼ်ၵၢၼ်လၢတ်ႈၼႄ Compiler ဝႃႈ ႁဝ်းၶႂ်ႈလႆႈ "ၵွၵ်း" မဵဝ်းလႂ် (Data Type) လႄႈ ၸိုဝ်ႈသင် (Name)။
* **တူဝ်ယင်ႇ:** `string studentName;` (ႁဝ်းသၢင်ႈၵွၵ်း တႃႇသိမ်းတူဝ်လိၵ်ႈလႄႈ ပၼ်ၸိုဝ်ႈမၼ်းဝႃႈ studentName)။

### 2. Initialization (ၵၢၼ်သႂ်ႇၶေႃႈမုၼ်းဢွၼ်တၢင်းသုတ်း)
ၼႆႉပဵၼ်ၵၢၼ်ဢဝ် "ၵႃႈၶၼ်" (Value) သႂ်ႇၼႂ်းၵွၵ်းၼၼ်ႉ ပွၵ်ႈၵမ်းဢွၼ်တၢင်းသုတ်း။
* **တူဝ်ယင်ႇ:** `studentName = "Sai Mao";`

> ႁဝ်းႁဵတ်း တင်း2လွင်ႈ ၼႂ်းထႅဝ်လဵဝ်ၵေႃႈလႆႈၶႃႈ:
> `int studentAge = 25;`

### 3. Naming Conventions (camelCase)
တႃႇႁႂ်ႈ Developer တင်းၵမ်ႇၽႃႇ ဢၢၼ်ႇ Code ၵၼ်ပွင်ႇၸႂ်ငၢႆႈၼၼ်ႉ ၼႂ်း C# ႁဝ်းၸႂ်ႉပိူင် **camelCase** တႃႇတင်ႈၸိုဝ်ႈ Variable:
* တူဝ်လိၵ်ႈတႄႇႁူဝ် ၶေႃႈၵႂၢမ်းထႅဝ်ဢွၼ်တၢင်းသုတ်း တေလႆႈပဵၼ် **တူဝ်လဵၵ်း** (lowercase)။
* ၶေႃႈၵႂၢမ်းထႅဝ်ထႅင်ႈပိူင်ၼိုင်ႈ တေလႆႈတႄႇတင်း **တူဝ်လူင်** (Uppercase)။
* **တူဝ်ယင်ႇ:** `totalScoreCount`, `isMemberActive`, `shanWordTranslation`။



### 4. ပိူင်တႃႇၶူင်သၢင်ႈ (Rules)
* တေလႆႈတႄႇတင်း တူဝ်လိၵ်ႈ (Letter) ဢမ်ႇၼၼ် Underscore (`_`) ၵူၺ်း။
* တေဢဝ် မၢႆၼပ်ႉ (Number) မႃးတႄႇၼႃႈဢမ်ႇလႆႈ (တူဝ်ယင်ႇ: `1stPlace` ၼႆႉၽိတ်း)။
* တေဢဝ် "ၶေႃႈၵႂၢမ်းၵိုတ်းဝႆႉ" (Reserved Keywords) မိူၼ်ၼင်ႇ `int`, `class`, `string` မႃးတင်ႈပဵၼ်ၸိုဝ်ႈဢမ်ႇလႆႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Declaration:** ၵၢၼ်ပိုၼ်ၽၢဝ်ႇ တႃႇသၢင်ႈ Variable မႂ်ႇ (မၢႆတွင်းဝႃႈ ပဵၼ် Type သင် လႄႈ ၸိုဝ်ႈသင်)။
* **Initialization:** ၵၢၼ်သႂ်ႇၶေႃႈမုၼ်း ဢွၼ်တၢင်းသုတ်း သႂ်ႇၼႂ်း Variable။
* **camelCase:** ပိူင်လၢႆးတင်ႈၸိုဝ်ႈ ဢၼ်တႄႇတင်းတူဝ်လဵၵ်း ယဝ်ႉၸွမ်းတင်းတူဝ်လူင် ၼႂ်းၶေႃႈၵႂၢမ်းထႅပ်ႈၼၼ်ႉ။
* **Assignment Operator:** ၶိူင်ႈမၢႆ `=` ဢၼ်ၸႂ်ႉတႃႇ "ပၼ်" ၵႃႈၶၼ်သႂ်ႇၼႂ်း Variable (ဢမ်ႇၸႂ်ႈမၢႆ "တႅတ်ႉတႅတ်ႉ" မိူၼ်ၼႂ်း Math)။
