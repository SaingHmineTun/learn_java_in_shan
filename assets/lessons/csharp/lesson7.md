# Lesson 7: "Basic Output" (လၢႆးၼႄၶေႃႈမုၼ်း ၼိူဝ်ၼႃႈလိၵ်ႈ)

ၼႂ်း C# ၼၼ်ႉ ႁဝ်းၸႂ်ႉ `Console` တႃႇလၢတ်ႈတေႃႇၵူၼ်းၸႂ်ႉ (User)။ ၵူၺ်းၵႃႈ လၢႆးၼႄလိၵ်ႈမၼ်းမီး 2 ပိူင် ဢၼ်လူဝ်ႇလိူၵ်ႈၸႂ်ႉႁႂ်ႈမႅၼ်ႈတီႈမၼ်းၶႃႈ။

### 1. Console.WriteLine()
ၼႆႉပဵၼ်ဢၼ်ႁဝ်းၸႂ်ႉၼမ်သေပိူၼ်ႈ။ မိူဝ်ႈမၼ်းတႅမ်ႈလိၵ်ႈယဝ်ႉယဝ်ႉၼၼ်ႉ မၼ်းတေ **"လူင်းထႅဝ်မႂ်ႇ"** (New Line) ပၼ်ၵမ်းလဵဝ်ဢေႃႈ။
* **တူဝ်ယင်ႇ:**
  ```csharp
  Console.WriteLine("မႂ်ႇသုင်");
  Console.WriteLine("ၶူးၸၢႆးမၢဝ်း");
  ```
* **ၽိူၼ် (Result):**
  ```text
  မႂ်ႇသုင်
  ၶူးၸၢႆးမၢဝ်း
  ```

### 2. Console.Write()
ဢၼ်ၼႆႉသမ်ႉ မိူဝ်ႈမၼ်းတႅမ်ႈလိၵ်ႈယဝ်ႉယဝ်ႉၼၼ်ႉ မၼ်းတေ **"ဢမ်ႇလူင်းထႅဝ်မႂ်ႇ"**။ မၼ်းတေၵိုတ်းဝႆႉတီႈသုတ်းထႅဝ်ၼၼ်ႉသေ ပေႃးမီးလိၵ်ႈမႃးထႅင်ႈ မၼ်းတေတႅမ်ႈတေႃႇၵၼ်ၵႂႃႇၵမ်းလဵဝ်ယဝ်ႉ။
* **တူဝ်ယင်ႇ:**
  ```csharp
  Console.Write("ဝဵင်း");
  Console.Write("မူႇၸေႊ");
  ```
* **ၽိူၼ် (Result):**
  ```text
  ဝဵင်းမူႇၸေႊ
  ```



### 3. မိူဝ်ႈလႂ်လူဝ်ႇၸႂ်ႉသင်?
* **WriteLine:** ၸႂ်ႉမိူဝ်ႈႁဝ်းတေလၢတ်ႈၶေႃႈၵႂၢမ်း ဢၼ်ပဵၼ်ထႅဝ်ပဵၼ်တွၼ်ႈ (မိူၼ်ၼင်ႇ ႁူဝ်ၶေႃႈ လႄႈ ၶေႃႈလၢတ်ႈၵူႈလွင်ႈ)။
* **Write:** ၸႂ်ႉမိူဝ်ႈႁဝ်းတေႁဵတ်း "ၶေႃႈထၢမ်" (Prompt) ဢၼ်ၶႂ်ႈႁႂ်ႈၵူၼ်းၸႂ်ႉ တႅမ်ႈလိၵ်ႈတေႃႇတီႈသုတ်းထႅဝ်ၼၼ်ႉ (တူဝ်ယင်ႇ: `Enter your name: _`)။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Output:** ၶေႃႈမုၼ်း ဢမ်ႇၼၼ် ၽိူၼ်ဢွၵ်ႇ ဢၼ် Program ၼႄပၼ်ၵူၼ်းၸႂ်ႉ။
* **New Line / Line Break:** ၵၢၼ်လူင်းထႅဝ်မႂ်ႇ (မိူၼ်ၼင်ႇႁဝ်းတဵၵ်း Enter ၼိူဝ် Keyboard)။
* **Parameter / Argument:** ၶေႃႈမုၼ်း (လိၵ်ႈ) ဢၼ်ႁဝ်းသႂ်ႇဝႆႉၼႂ်း ဝူတ်ႇ `( )` တႃႇႁႂ်ႈ Method ၼၼ်ႉဢဝ်ၵႂႃႇႁဵတ်းၵၢၼ်။
* **String:** ၸုမ်းတူဝ်လိၵ်ႈ ဢၼ်ႁဝ်းသႂ်ႇဝႆႉၼႂ်း မၢႆၵမ် (Double Quotes) `" "`။
