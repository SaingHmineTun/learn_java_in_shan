# Lesson 10: "Comments and Documentation" (ၵၢၼ်တႅမ်ႈမၢႆတွင်း လႄႈ လိၵ်ႈၵိုၵ်း Code)

**Comments** ၼႆႉပဵၼ်တူဝ်လိၵ်ႈ ဢၼ်ႁဝ်းတႅမ်ႈဝႆႉၼႂ်း File Code ႁဝ်း၊ 
ၵူၺ်းၵႃႈ Compiler တေဢမ်ႇဢၢၼ်ႇမၼ်း လႄႈ မၼ်းတေဢမ်ႇႁဵတ်းၵၢၼ်သင်။ 
မၼ်းမီးဝႆႉတႃႇ "ၵူၼ်း" ဢၢၼ်ႇၵူၺ်းဢေႃႈ။ ၼႂ်း C# မၼ်းမီး comment ဝႆႉ 3 ပိူင်ၶႃႈ။

### 1. Single-line Comment (မၢႆတွင်းထႅဝ်လဵဝ်)
ၸႂ်ႉၶိူင်ႈမၢႆ `//` (Double Slash) တႃႇတႅမ်ႈမၢႆတွင်းထႅဝ်လဵဝ်။
```csharp
// ၼႆႉပဵၼ်ၵၢၼ်တင်ႈၵၢၼ်တႃႇလိၵ်ႈတႆး
Console.OutputEncoding = Encoding.UTF8; 
```

### 2. Multi-line Comment (မၢႆတွင်းလၢႆထႅဝ်)
ၸႂ်ႉၶိူင်ႈမၢႆ `/* ... */` မိူဝ်ႈႁဝ်းလူဝ်ႇတႅမ်ႈၶေႃႈပွင်ႇၸႂ်ယၢဝ်းယၢဝ်း ဢၼ်မီးလၢႆလၢႆထႅဝ်။
```csharp
/* Project: Shan Dictionary
   Author: Sai Mao
   Date: 2026
*/
```

### 3. XML Documentation Comment (မၢႆတွင်းတႃႇၵၢၼ်ၸႂ်ႉတႅင်ႈ)
ၸႂ်ႉၶိူင်ႈမၢႆ `///` (Triple Slash) ဝႆႉတီႈၼိူဝ် Method ဢမ်ႇၼၼ် Class။ 
ၼႆႉမၼ်းၶႅမ်ႉၼႃႇ ယွၼ်ႉမၼ်းတေၼႄပၼ် "ၶေႃႈၸွႆးထႅမ်" (IntelliSense) မိူဝ်ႈႁဝ်းဢဝ် Mouse ၵႂႃႇၸီႉဝႆႉတီႈ Code ၼၼ်ႉယဝ်ႉ။
```csharp
/// <summary>
/// Method ၼႆႉၸႂ်ႉတႃႇလၢတ်ႈမႂ်ႇသုင်ပဵၼ်လိၵ်ႈတႆး
/// </summary>
void SayHello() { ... }
```


---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Comment:** တွၼ်ႈလိၵ်ႈၼႂ်း Code ဢၼ် Compiler ဝူတ်ႇဝႆႉ (Ignore) သေ ဢမ်ႇဢဝ်ၵႂႃႇႁဵတ်းၵၢၼ်။
* **Documentation:** လိၵ်ႈဢธิบาย လွင်ႈႁဵတ်းၵၢၼ် Software ႁႂ်ႈပိူၼ်ႈၸႂ်ႉတႅင်ႈလႆႈမၢၼ်ႇမႅၼ်ႈ။
* **Summary Tag:** ၶေႃႈၵႂၢမ်း XML ဢၼ်ၸႂ်ႉမၢႆတွင်း ႁူဝ်ႁုပ်ႈၵၢၼ်ႁဵတ်း (Summary) ၼႂ်း C#။
* **Clean Code:** လၢႆးတႅမ်ႈ Code ဢၼ်ၸတ်းၵၢၼ်ဝႆႉလီလီ၊ ဢၢၼ်းပွင်ႇၸႂ်လႆႈငၢႆႈ (Comment ဢၼ်ၼမ်လွၼ်ႉၵႂႃႇၵေႃႈ ဢမ်ႇၸႂ်ႈ Clean Code)။
