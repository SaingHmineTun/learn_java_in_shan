# Lesson 192: "Catching Specific Exceptions"

ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းပဵၼ် မေႃယႃၼႆၶႃႈ။ သင်မီးၵူၼ်းပဵၼ်မႃးႁႃ... ႁဝ်းတေ ဢမ်ႇဢဝ် ယႃႈယႃဢၼ်လဵဝ် ယႃပၼ် တၢင်းပဵၼ်ၵူႈလွင်ႈၶႃႈ။ 
ပၼ်ႁႃၼႂ်း Code ၵေႃႈမိူၼ်ၵၼ်၊ ပၼ်ႁႃ "တူဝ်ၼပ်ႉ" တေလႆႈၵႄႈလၢႆးၼိုင်ႈ၊ ပၼ်ႁႃ "လိၵ်ႈၽိတ်း" သမ်ႉတေလႆႈၵႄႈထႅင်ႈလၢႆးၼိုင်ႈၶႃႈ။

### 1. ႁဵတ်းသင်ၸင်ႇလူဝ်ႇၸႅၵ်ႇတႅတ်ႈတေႃး? (Why Specific?)
1.  **Precise Feedback:** ႁႂ်ႈႁဝ်းလၢတ်ႈၼႄၵူၼ်းၸႂ်ႉ (User) လႆႈတႅတ်ႈတေႃးဝႃႈ မၼ်းၽိတ်းတီႈလႂ်။
2.  **Specific Recovery:** ႁဝ်းၸၢင်ႈတႅပ်းတတ်းလႆႈဝႃႈ သင်ပဵၼ်ပၼ်ႁႃ A ႁႂ်ႈႁဵတ်းၼႆ၊ သင်ပဵၼ် B ႁႂ်ႈႁဵတ်းထႅင်ႈလၢႆးၼိုင်ႈ။

### 2. ပိူင်သၢင်ႈ (Multiple Catch Blocks)
ႁဝ်းၸၢင်ႈမီး `catch` လၢႆလၢႆဢၼ် ၸွမ်း `try` တူဝ်လဵဝ်ၶႃႈ။ C# တေလိူၵ်ႈဢဝ်ဢၼ် "မႅၼ်ႈသုတ်း" မႃးၸႂ်ႉၶႃႈ။

```csharp
try {
    Console.Write("သႂ်ႇတူဝ်ၼပ်ႉ: ");
    int num = int.Parse(Console.ReadLine());
    int result = 100 / num;
    Console.WriteLine($"Result: {result}");
} 
catch (FormatException) {
    // ပၼ်ႁႃ: ၵူၼ်းၸႂ်ႉတႅမ်ႈလိၵ်ႈ တႅၼ်းဢၼ်တေလႆႈတႅမ်ႈတူဝ်ၼပ်ႉ
    Console.WriteLine("ၽိတ်း: သႂ်ႇတူဝ်ၼပ်ႉၵူၺ်းၶႃႈ၊ ယႃႇသႂ်ႇလိၵ်ႈ!");
}
catch (DivideByZeroException) {
    // ပၼ်ႁႃ: ဢဝ်တူဝ်ၼပ်ႉမႃးၸႅၵ်ႇလူၺ်ႈ သုၼ် (0)
    Console.WriteLine("ၽိတ်း: ဢဝ် 0 မႃးၸႅၵ်ႇဢမ်ႇလႆႈၶႃႈ!");
}
catch (Exception ex) {
    // ပၼ်ႁႃတၢင်ႇၸိူဝ်း: တူဝ်ၼႆႉတေႁပ်ႉဢဝ် ဢၼ်ၵိုတ်းဝႆႉတင်းမူတ်း
    Console.WriteLine($"မီးပၼ်ႁႃတၢင်ႇပိူင်: {ex.Message}");
}
```



### 3. Exceptions ဢၼ်ၵႆႉႁၼ်ၼမ် (Common Exceptions)

* **`FormatException`**: ပဵၼ်မိူဝ်ႈ Data Type ဢမ်ႇမႅၼ်ႈၵၼ် (မိူၼ်ၼင်ႇ ဢဝ် "Shan" ၵႂႃႇပိၼ်ႇပဵၼ် `int`)။
* **`DivideByZeroException`**: ပဵၼ်မိူဝ်ႈဢဝ်တူဝ်ၼပ်ႉမႃးၸႅၵ်ႇလူၺ်ႈ 0။
* **`NullReferenceException`**: ပဵၼ်မိူဝ်ႈႁဝ်းႁွင်ႉၸႂ်ႉ Object ဢၼ်တိုၵ်ႉပဵၼ် `null` ယူႇ။
* **`IndexOutOfRangeException`**: ပဵၼ်မိူဝ်ႈႁဝ်းႁွင်ႉ Index ၶွင် Array/List ဢၼ်ဢမ်ႇမီးဝႆႉ (မိူၼ်ၼင်ႇ Array မီး 3 တူဝ် သေၵႂႃႇႁွင်ႉတူဝ်မၢႆ 10)။

### 4. ပၵ်းၵၢၼ်လမ်ႇလွင်ႈ (The Order Rule)
ႁဝ်းတေလႆႈဝႆႉ **Specific Exception** (တူဝ်ဢၼ်တႅတ်ႈတေႃး) ဝႆႉၽၢႆႇၼိူဝ်သေ ဝႆႉ **General Exception** (`Exception` တူဝ်လူင်) ဝႆႉၽၢႆႇတႂ်ႈသုတ်းတႃႇသေႇၶႃႈ။
> **သင်ဝႆႉ `Exception` တူဝ်လူင်ဝႆႉၽၢႆႇၼိူဝ်... မၼ်းတေႁပ်ႉဢဝ်ၵူႈပိူင် သေတူဝ်တႂ်ႈမၼ်းတေဢမ်ႇလႆႈႁဵတ်းၵၢၼ်သင်ၶႃႈ။**

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Catch Hierarchy**: ၸၼ်ႉၵၢၼ်ဝၢင်း Catch လုၵ်ႉတီႈတူဝ်ဢွၼ်ႇ ၵႂႃႇၸူးတူဝ်လူင်။
* **Input Validation**: ၵၢၼ်ၵူတ်ႇထတ်းၶေႃႈမုၼ်း ဢၼ်လုၵ်ႉတီႈၵူၼ်းၸႂ်ႉမႃး။
* **Data Type Mismatch**: လွင်ႈဢၼ်ၶေႃႈမုၼ်း သွင်ပိူင်ဢမ်ႇၶဝ်ႈၵၼ်လႆႈ။
