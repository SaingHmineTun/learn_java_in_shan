# Lesson 139: "The Concept of Inheritance: Is-A Relationships"

**Inheritance** (ၵၢၼ်သိုပ်ႇၸိူဝ်ႉ) ပဵၼ်ပိူင်ၵၢၼ်ဢၼ် Class ၼိုင်ႈဢၼ် ၸၢင်ႈ **"သိုပ်ႇဢဝ်"** (Inherit) တၢင်းႁူႉ (Fields) လႄႈ လၢႆးႁဵတ်းသၢင်ႈ (Methods) တီႈ Class တၢင်ႇဢၼ်မႃးၶႃႈ။

### 1. "Is-A" Relationship ပဵၼ်သင်?
ၼႂ်းၵၢၼ်တႅမ်ႈ Code... မိူဝ်ႈႁဝ်းဝႃႈ Class B သိုပ်ႇၸိူဝ်းၽၼ်းတီႈ Class A ၼၼ်ႉ၊ မၼ်းပွင်ႇဝႃႈ **"B Is-A A"** (B ပဵၼ် A ပိူင်ၼိုင်ႈ) ၶႃႈ။



**တူဝ်ယင်ႇ ၼႂ်းၸူဝ်ႈပၢၼ်တႄႉ:**
* **Car** is a **Vehicle** (လူတ်ႉၵႃး ပဵၼ် လူတ်ႉၶိူင်ႈၸၢၵ်ႈ ပိူင်ၼိုင်ႈ)
* **Teacher** is a **Staff** (ၶူးသွၼ် ပဵၼ် ၵူၼ်းႁဵတ်းၵၢၼ် ၵေႃႉၼိုင်ႈ)
* **Dog** is an **Animal** (မႃ ပဵၼ် တူဝ်သတ်း တူဝ်ၼိုင်ႈ)

### 2. ႁဵတ်းသင်ၸင်ႇလူဝ်ႇၸႂ်ႉ Inheritance?
1.  **Code Reusability:** ဢမ်ႇလူဝ်ႇတႅမ်ႈ Code ၸိူဝ်ႉမိူၼ်ၵၼ်ၶိုၼ်းလၢႆပွၵ်ႈ။ သင် Class တူဝ်ပေႃႈ (Parent) မီး `Run()` ယဝ်ႉ... Class တူဝ်လုၵ်ႈ (Child) ဢမ်ႇလူဝ်ႇတႅမ်ႈ `Run()` ၶိုၼ်း၊ မၼ်းၸၢင်ႈၸႂ်ႉလႆႈၵမ်းလဵဝ်ၶႃႈ။
2.  **Hierarchical Classification:** ၸွႆးၸႅၵ်ႇၽႄ Code ႁဝ်းႁႂ်ႈမီးၸၼ်ႉ မိူၼ်ၼင်ႇ တူဝ်သတ်း -> သတ်းလဵင်ႉလုၵ်ႈလူၺ်ႈၼမ်ႉၼူမ်း -> မႃ။
3.  **Extensibility:** ႁဝ်းၸၢင်ႈထႅမ် "လွင်ႈလႅတ်းသႅဝ်းမႂ်ႇမႂ်ႇ" သႂ်ႇၼႂ်း Class လုၵ်ႈ လူၺ်းဢမ်ႇတုမ်ႉတိူဝ်ႉ Class ပေႃးၶႃႈ။

### 3. ၶေႃႈၵႂမ်းႁွင်ႉ (Terminology)
ၼႂ်း C# ႁဝ်းမီးၸိုဝ်ႈႁွင်ႉလၢႆပိူင် ဢၼ်မီးတီႈပွင်ႇမိူၼ်ၵၼ်ၶႃႈ:
* **Base Class** (ဢမ်ႇၼၼ် Parent Class): တူဝ်ဢၼ်ပဵၼ် "တူဝ်တႄႇ" ဢၼ်သူင်ႇၸိူဝ်ႉၽၼ်းပၼ်ပိူၼ်ႈ။
* **Derived Class** (ဢမ်ႇၼၼ် Child Class): တူဝ်ဢၼ်ပဵၼ် "တူဝ်သိုပ်ႇ" ဢၼ်သိုပ်ႇၸိူဝ်ႉၽၼ်းမႃး။

### 4. တူဝ်ယင်ႇ ဝူၼ်ႉတူၺ်း Code
ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းမီး Class **`Vehicle`** (လူတ်ႉ)။
* မၼ်းမီး `Speed` လႄႈ `Move()`။
* မိူဝ်ႈႁဝ်းသၢင်ႈ Class **`Car`** သေ သိုပ်ႇၸိူဝ်ႉမၼ်းမႃးတီႈ Vehicle ၼႆၸိုင်... `Car` တေမီး `Speed` လႄႈ `Move()` ႁင်းၵူၺ်းမၼ်းၵမ်းလဵဝ် လူၺ်းဢမ်ႇလူဝ်ႇတႅမ်ႈသေထႅဝ်ၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Inheritance:** ၵၢၼ်သိုပ်ႇဢဝ် တၢင်းႁူႉ လႄႈ လၢႆးႁဵတ်းသၢင်ႈ လုၵ်ႉတီႈ Class ၼိုင်ႈ ၵႂႃႇၸူးထႅင်ႈ Class ၼိုင်ႈ။
* **Is-A Relationship:** ပိူင်ၵၢၼ်ၵွင်ႉၵၢႆႇ ဢၼ်ၼႄဝႃႈ Object တူဝ်ၼိုင်ႈ ပဵၼ်တွၼ်ႈၼိုင်ႈ ၶွင် Class ဢၼ်ယႂ်ႇလိူဝ်မၼ်း။
* **Base Class:** Class ပိုၼ်ႉထၢၼ် ဢၼ်သူင်ႇၸိူဝ်ႉၽၼ်း။
* **Derived Class:** Class ဢၼ်လႆႈဝႆႉ ၸိူဝ်းၽၼ်းမၼ်းမႃး သေမီးလွင်ႈလႅတ်းသႅဝ်းလိူဝ်ၵဝ်ႇ။
