# Lesson 27: "Arithmetic Operators" (ၶိူင်ႈမၢႆၼပ်ႉသၢင်ႈပိုၼ်ႉထၢၼ်)

ၼႂ်း C# ၼၼ်ႉ ႁဝ်းၸႂ်ႉ Arithmetic Operators တႃႇႁဵတ်းၵၢၼ်ပၢႆးၼပ်ႉ (Mathematics) ၼိူဝ် Variable ဢၼ်ပဵၼ်မၢႆၼပ်ႉ (int, double, etc.) ၶႃႈ။

### 1. ၶိူင်ႈမၢႆပိုၼ်ႉထၢၼ် 5 ပိူင်

| Operator | Name | Meaning                             | Example (int a = 10, b = 3) | Result |
| :--- | :--- |:------------------------------------| :--- | :--- |
| **`+`** | Addition | တႃႇဢဝ်မႃး **လေႃး/ထႅမ်/ႁူမ်ႈ** ၵၼ်   | `a + b` | 13 |
| **`-`** | Subtraction | တႃႇဢဝ်မႃး **ထွၼ်/လိူၵ်ႈ/ယွမ်း** ၵၼ် | `a - b` | 7 |
| **`*`** | Multiplication | တႃႇဢဝ်မႃး **ၵုၼ်း** ၵၼ်             | `a * b` | 30 |
| **`/`** | Division | တႃႇဢဝ်မႃး **ၸႅၵ်ႇ** ၵၼ်             | `a / b` | 3 |
| **`%`** | Modulus | တႃႇႁႃ **တူဝ်လိူဝ်** ဝၢႆးၸႅၵ်ႇယဝ်ႉ   | `a % b` | 1 |



### 2. လွင်ႈလူဝ်ႇၾၢင်ႉမိူဝ်ႈ ၸႅၵ်ႇၵၼ် (Division)
ၼႂ်း C# ၼၼ်ႉ ပေႃးႁဝ်းဢဝ် Integer (မၢႆၼပ်ႉတဵမ်) မႃးၸႅၵ်ႇၵၼ် မၼ်းတေဢွၵ်ႇပဵၼ် Integer တႃႇသေႇ။ 
တွၼ်ႈဢၼ်ပဵၼ် "ၸုၵ်း" ၼၼ်ႉ မၼ်းတေထွၼ်ပႅတ်ႇ (Truncate) ၵမ်းလဵဝ်။
* **တူဝ်ယင်ႇ:** `10 / 3` တေလႆး `3` (ဢမ်ႇၸႂ်ႈ 3.33)။
* **လၢႆးၵႄႈ:** လူဝ်ႇၸႂ်ႉ `double` ဢမ်ႇၼၼ် `float` တႃႇလႆႈပၢႆၸုၵ်းမၼ်း။
    * `10.0 / 3.0` -> `3.3333333333333335`

### 3. Modulus (%) - တူဝ်ႁႃပၢႆ
ၶိူင်ႈမၢႆ `%` ၼႆႉမီးၶဝ်ႈတၢင်းတိုဝ်းၼႃႇ။ တီႈတႂ်ႈၼၼ်ႉ ႁဝ်းၸႂ်ႉမၼ်း တႃႇၵူတ်ႇထတ်းဝႃႈ မၢႆၼပ်ႉၼၼ်ႉ ပဵၼ် **မၢႆၵူပ်ႉ** (Even) ႁႃႉ **မၢႆၵိၵ်ႉ** (Odd)။
* ပေႃး `number % 2 == 0` -> ပဵၼ်မၢႆၵူပ်ႉ (Even)။
* ပေႃး `number % 2 != 0` -> ပဵၼ်မၢႆၵိၵ်ႉ (Odd)။

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Operator:** ၶိူင်ႈမၢႆဢၼ်ၸႂ်ႉသင်ႇၵၢၼ်ၼပ်ႉသၢင်ႈ (မိူၼ်ၼင်ႇ +, -)။
* **Operand:** ၵႃႈၶၼ် ဢမ်ႇၼၼ် Variable ဢၼ်ထုၵ်ႇၼပ်ႉသၢင်ႈ (ၼႂ်း `5 + 10` ၼၼ်ႉ 5 လႄႈ 10 ပဵၼ် Operand)။
* **Integer Division:** ၵၢၼ်ၸႅၵ်ႇမၢႆၼပ်ႉတဵမ် ဢၼ်တေဢမ်ႇသိမ်းဝႆႉတွၼ်ႈပၢႆၸုၵ်း။
* **Remainder:** ပၢႆဢၼ်လိူဝ်ဝႆႉ ဝၢႆးသေၸႅၵ်ႇယဝ်ႉ (Modulus ႁႃပၼ်တွၼ်ႈၼႆႉ)။
