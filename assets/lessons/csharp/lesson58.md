# Lesson 58: "Infinite Loops: Causes, Dangers, and How to Stop Them"

**Infinite Loop** ၼႆႉ ပဵၼ်ၵၢၼ်ႁဵတ်းပၼ်ႇတူင်ႇ ဢၼ် **"ၶေႃႈထၢမ်"** (Condition) မၼ်းပဵၼ် **True** တႃႇသေႇ ယဝ်ႉ။

### 1. လွင်ႈတၢင်းဢၼ်ႁဵတ်းႁႂ်ႈပဵၼ် (Causes)
ၵႆႉပဵၼ်ယွၼ်ႉလွင်ႈတၢင်း 3 ယၢင်ႇၼႆႉၶႃႈ:
* **လိုမ်း Update Variable:** မိူၼ်ၼင်ႇ ႁဝ်းလိုမ်းတႅမ်ႈ `i++` သေ `i` တေပဵၼ် 0 တႃႇသေႇ။
* **ၶေႃႈထၢမ်ၽိတ်း (Wrong Condition):** မိူၼ်ၼင်ႇ တႅမ်ႈဝႃႈ `while (i >= 0)` သေ `i` သမ်ႉထႅမ်ၶိုၼ်ႈတိၵ်းတိၵ်း (မၼ်းၵေႃႈတေပူၼ်ႉ 0 တႃႇသေႇ)။
* **တႅမ်ႈႁႂ်ႈမၼ်းပဵၼ် True တႃႇသေႇ:** တႅမ်ႈဝႃႈ `while (true)` သေ သမ်ႉလိုမ်းၸႂ်ႉ `break` တႃႇဢွၵ်ႇၶႃႈ။



### 2. ၽေးၶဵၼ် (Dangers)
* **CPU Usage:** CPU တေလႆႈႁဵတ်းၵၢၼ် 100% တႃႇသေႇလႄႈ သေႁဵတ်းႁႂ်ႈၶွမ်းမႆႈ ယဝ်ႉ ၸႂ်ႉၾႆးၼမ်။
* **App Freezes (ၶွမ်းၶမ်):** User တေဢမ်ႇၸၢင်ႈၼဵၵ်း (Click) သင်လႆႈသေပွၵ်ႈ ယွၼ်ႉ App ၵိုတ်းဝႆႉတီႈ Loop ၼၼ်ႉၵူၺ်း။
* **Memory Leak:** သင်ဝႃႈၼႂ်း Loop ၼၼ်ႉ မီးၵၢၼ်သၢင်ႈ Variable မႂ်ႇတိၵ်းတိၵ်း၊ Memory (RAM) တေတဵမ်သေ App တေ Crash ၵႂႃႇၶႃႈ။

### 3. လၢႆးၵိုတ်းမၼ်း (How to Stop)
* **မိူဝ်ႈတိုၵ်ႉတႅမ်ႈ Code:** ၼဵၵ်း **`Ctrl + C`** ၼႂ်း Console (Terminal) တႃႇသင်ႇၵိုတ်း Program ၵမ်းလဵဝ်။
* **ၼႂ်း Visual Studio:** ၼဵၵ်းမၢႆသီႇၸဵင်ႇလႅင် **Stop Debugging** (Shift + F5)။
* **လၢႆးၵႅတ်ႇၶႄ (Prevention):** မၵ်ႉမၼ်ႈ **Exit Condition** ႁႂ်ႈလီယဝ်ႉ ၸင်ႇလႅၼ်ႈ (Run) Code ၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Infinite Loop:** ၵၢၼ်ႁဵတ်းပၼ်ႇတူင်ႇ ဢၼ်ဢမ်ႇမေႃယဝ်ႉတူဝ်ႈ။
* **Freeze (ၶွမ်းၶမ်):** ၵၢၼ်ဢၼ် Program ဢမ်ႇတွပ်ႇသင်မႃးၶိုၼ်း (Not Responding)။
* **Terminating:** ၵၢၼ်သင်ႇၵိုတ်း Program ဢၼ်လႅၼ်ႈယူႇ။
* **Logic Error:** ၶေႃႈၽိတ်းၼႂ်းပၢႆးဝူၼ်ႉ ဢၼ် Compiler ဢမ်ႇႁၼ် ၵူၺ်းၵႃႈ App ႁဵတ်းၵၢၼ်ၽိတ်း။
