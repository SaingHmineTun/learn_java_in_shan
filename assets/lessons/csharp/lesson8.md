# Lesson 8: "Basic Input" (လၢႆးႁပ်ႉဢဝ်ၶေႃႈမုၼ်း လုၵ်ႉတီႈၵူၼ်းၸႂ်ႉ)

မိူဝ်ႈႁဝ်းၶႂ်ႈႁႂ်ႈၵူၼ်းၸႂ်ႉ တႅမ်ႈလိၵ်ႈသႂ်ႇၼႂ်း Program ႁဝ်း (မိူၼ်ၼင်ႇ တႅမ်ႈၸိုဝ်ႈ ဢမ်ႇၼၼ် တႅမ်ႈၶေႃႈၵႂၢမ်းတႃႇပိၼ်ႇၽႃႇသႃႇ) ႁဝ်းတေလႆႈၸႂ်ႉ Method ဢၼ်ၸိုဝ်ႈဝႃႈ `Console.ReadLine()` ဢေႃႈ။

### 1. Console.ReadLine()
Method ၼႆႉမၼ်းတေ "ၵိုတ်း" (Pause) ဝႆႉသေ ပႂ်ႉႁႂ်ႈၵူၼ်းၸႂ်ႉ တႅမ်ႈလိၵ်ႈယဝ်ႉ တဵၵ်း **Enter**။ မိူဝ်ႈတဵၵ်း Enter ယဝ်ႉယဝ်ႉၼၼ်ႉ မၼ်းတေဢဝ်လိၵ်ႈၼၼ်ႉ သူင်ႇပၼ် Program ႁဝ်းၶႃႈ။

### 2. ၵၢၼ်သိမ်းၶေႃႈမုၼ်းသႂ်ႇၼႂ်း Variable
တႃႇတေဢဝ်လိၵ်ႈဢၼ်ၵူၼ်းၸႂ်ႉတႅမ်ႈမႃးၼၼ်ႉ ၵႂႃႇၸႂ်ႉၵၢၼ်လႆႈ ႁဝ်းလူဝ်ႇမီး "ၵွင်" (Variable) တႃႇသိမ်းမၼ်းဝႆႉ။ ၼႂ်း C# ႁဝ်းၸႂ်ႉ `string` တႃႇသိမ်းတူဝ်လိၵ်ႈဢေႃႈ။

* **တူဝ်ယင်ႇ Code:**
  ```csharp
  Console.Write("Enter your name: "); // ၼႄၶေႃႈထၢမ်
  string userName = Console.ReadLine(); // ႁပ်ႉဢဝ်ၸိုဝ်ႈမႃးသိမ်းဝႆႉ
  
  Console.WriteLine("မႂ်ႇသုင်ၶႃႈ, " + userName + "!"); // ဢဝ်ၸိုဝ်ႈမႃးၼႄၶိုၼ်ႈ
  ```



### 3. လွင်ႈလူဝ်ႇၾၢင်ႉ (Null Handling)
မိူဝ်ႈႁဝ်းၸႂ်ႉ `ReadLine()` ၼၼ်ႉ မၼ်းမီးလွင်ႈၼိုင်ႈ ဢၼ် C# တေၵူတ်ႇထတ်း—ၼၼ်ႉၵေႃႈပဵၼ် "ပေႃးၵူၼ်းၸႂ်ႉဢမ်ႇတႅမ်ႈသင်သေ တဵၵ်း Enter ၵႂႃႇလၢႆလၢႆလႄႇ?" ၼႆႉယဝ်ႉ။
* ၼႂ်း C# သႅၼ်းမႂ်ႇ မၼ်းတေၼႄသီလိူင်ဝႆႉ (Warning) ဝႃႈ ၶေႃႈမုၼ်းဢၼ်လႆႈမႃးၼၼ်ႉ မၼ်းၸၢင်ႈပဵၼ် `null` (ဢမ်ႇမီးသင်) ဢေႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Input:** ၶေႃႈမုၼ်းဢၼ် Program ႁပ်ႉဢဝ်မႃးလုၵ်ႉတီႈ ၵူၼ်းၸႂ်ႉ (Keyboard) ဢမ်ႇၼၼ် တီႈတၢင်ႇပိူင်။
* **Variable:** တီႈသိမ်းၶေႃႈမုၼ်း ၼႂ်း Memory ၶွမ်းပိဝ်ႇတႃႇ (မိူၼ်ၼင်ႇ "ၵွင်" ဢၼ်ႁဝ်းပၼ်ၸိုဝ်ႈဝႆႉ)။
* **String:** ပိူင်ၶေႃႈမုၼ်း ဢၼ်ပဵၼ်ၸုမ်းတူဝ်လိၵ်ႈ (Text)။
* **Concatenation:** ၵၢၼ်ဢဝ် string 2 ဢၼ် မႃးသိုပ်ႇၵၼ် ႁႂ်ႈပဵၼ်ထႅဝ်လဵဝ် (ၸႂ်ႉၶိူင်ႈမၢႆ `+`)။
* **Null:** ၶေႃႈမုၼ်းဢၼ်ပဝ်ႇပၢင်ႇ ဢမ်ႇၼၼ် "ဢမ်ႇမီးသင်" (Empty value)။
