# Lesson 255: "Common WinForms Controls"

ၼႂ်း WinForms မီး Controls တင်းၼမ်၊ ၵူၺ်းၵႃႈပိုၼ်ႉထၢၼ်ဢၼ်လမ်ႇလွင်ႈသုတ်းမီး 3 ဢၼ်ၼႆႉၶႃႈ:

### 1. Label (တူဝ်ၼႄလိၵ်ႈ)
ၸႂ်ႉတွၼ်ႈတႃႇၼႄလိၵ်ႈဢၼ် User **ဢမ်ႇၸၢင်ႈမႄးလႆႈ** (Read-only)။
* **လၢႆးၸႂ်ႉ:** ၸႂ်ႉတႅမ်ႈၸိုဝ်ႈ "Name:", "Age:", ဢမ်ႇၼၼ် ၶေႃႈၵႂၢမ်းၼႄတၢင်းၼိူဝ် Form။
* **Property ဢၼ်လမ်ႇလွင်ႈ:** `Text` (လိၵ်ႈဢၼ်တေၼႄ)။
* **Prefix ဢၼ်ၵႆႉၸႂ်ႉ:** `lbl` (တူဝ်ယင်ႇ: `lblName`)။

### 2. TextBox (လွၵ်းတႅမ်ႈလိၵ်ႈ)
ၸႂ်ႉတွၼ်ႈတႃႇ **ႁပ်ႉၶေႃႈမုၼ်း** လုၵ်ႉတီႈ User ၼဵၵ်းတႅမ်ႈၶဝ်ႈမႃးၶႃႈ။
* **လၢႆးၸႂ်ႉ:** ႁပ်ႉၸိုဝ်ႈ၊ ႁပ်ႉ Password၊ ႁပ်ႉမၢႆၾူၼ်း။
* **Property ဢၼ်လမ်ႇလွင်ႈ:**
    * `Text`: လိၵ်ႈဢၼ် User တႅမ်ႈၶဝ်ႈမႃး (ပဵၼ် String တႃႇသေႇ)။
    * `PasswordChar`: သင်သႂ်ႇ `*` ၼႆ မၼ်းတေလပ်ႉ (Hide) တူဝ်လိၵ်ႈဝႆႉ တႃႇလွင်ႈလွတ်ႈၽေး။
    * `Multiline`: သင်ပဵၼ် `true` ၼႆ မၼ်းတေတႅမ်ႈလႆႈလၢႆလၢႆထႅဝ်။
* **Prefix ဢၼ်ၵႆႉၸႂ်ႉ:** `txt` (တူဝ်ယင်ႇ: `txtUserName`)။

### 3. CheckBox (တူဝ်လိူၵ်ႈမၢႆၶီတ်း)
ၸႂ်ႉတွၼ်ႈတႃႇပၼ် User လိူၵ်ႈ "ၸႂ်ႈ" (Yes) ဢမ်ႇၼၼ် "ဢမ်ႇၸႂ်ႈ" (No) ၶႃႈ။
* **လၢႆးၸႂ်ႉ:** တူဝ်ယင်ႇ "ႁပ်ႉပၵ်းၵၢၼ်ႁဝ်းႁႃႉ?", "တွၼ်ႈတႃႇ Save Password ႁႃႉ?"။
* **Property ဢၼ်လမ်ႇလွင်ႈ:** `Checked` (ပဵၼ် `true` မိူဝ်ႈမီးမၢႆၶိတ်း၊ ပဵၼ် `false` မိူဝ်ႈဢမ်ႇမီး)။
* **Event ဢၼ်လမ်ႇလွင်ႈ:** `CheckedChanged` (လႅၼ်ႈၵၢၼ်မိူဝ်ႈ User ၼဵၵ်းမၢႆၶိတ်းၶဝ်ႈ/ဢွၵ်ႇ)။
* **Prefix ဢၼ်ၵႆႉၸႂ်ႉ:** `chk` (တူဝ်ယင်ႇ: `chkAgree`)။



---

### တူဝ်ယင်ႇ Code: ၵၢၼ်ၸွမ်ၵၼ်
ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းမီး TextBox တႃႇႁပ်ႉၸိုဝ်ႈ လႄႈ CheckBox တႃႇၼဵၵ်း "Agree" ၶႃႈ။

```csharp
private void btnSubmit_Click(object sender, EventArgs e)
{
    // 1. ဢဝ်လိၵ်ႈလုၵ်ႉတီႈ TextBox
    string userName = txtName.Text;

    // 2. ၵူတ်ႇထတ်း CheckBox
    if (chkAgree.Checked)
    {
        MessageBox.Show($"မႂ်ႇသုင်ၶႃႈ {userName}! ႁဝ်းႁပ်ႉၶေႃႈမုၼ်းၶူးယဝ်ႉ။");
    }
    else
    {
        MessageBox.Show("ၶေႃႈၵႂၢမ်းၶႂ်ႈလၢတ်ႈ: ၶႅၼ်းတေႃႈ ၼဵၵ်းမၢႆၶိတ်း Agree ဢွၼ်တၢင်းၶႃႈ။");
    }
}
```

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Prefix:** တူဝ်လိၵ်ႈဢွၼ်တၢင်းၸိုဝ်ႈ (lblName, txtName) ဢၼ်ၸွႆးႁႂ်ႈႁဝ်းႁူႉဝႃႈမၼ်းပဵၼ် Control ပိူင်လႂ် ၼႂ်းယၢမ်းတႅမ်ႈ Code ၶႃႈ။
* **Read-only:** ၵၢၼ်ဢၼ်လူလႆႈ ၵူၺ်းၵႃႈမႄးဢမ်ႇလႆႈ။
* **Boolean (Checked):** ၵႃႈၶၼ်ဢၼ်မီး True (မီးမၢႆၶိတ်း) ဢမ်ႇၼၼ် False (ဢမ်ႇမီး)။
