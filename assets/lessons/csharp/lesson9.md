# Lesson 9: "Enabling Unicode/UTF-8 for Shan Script" (ၵၢၼ်ပိုတ်ႇၸႂ်ႉ Unicode တႃႇလိၵ်ႈတႆး)

ၼႂ်းၼႃႈလိၵ်ႈ Console သီလမ်ၼၼ်ႉ မၼ်းဢမ်ႇပွင်ႇၸႂ်လိၵ်ႈတႆး (Unicode) ႁင်းမၼ်း။ ႁဝ်းလူဝ်ႇသင်ႇႁႂ်ႈ Program ႁဝ်းပိၼ်ႇပဵၼ် **UTF-8 Encoding** ဢွၼ်တၢင်း ၸင်ႇတေတႅမ်ႈလႄႈ ဢၢၼ်းလိၵ်ႈတႆးလႆႈ မၢၼ်ႇမႅၼ်ႈဢေႃႈ။

### 1. ႁဵတ်းသင်လႄႈလူဝ်ႇ UTF-8?
* **ASCII:** ပဵၼ်လၢႆးၵဝ်ႇ ဢၼ်ပွင်ႇၸႂ်တူဝ်လိၵ်ႈဢင်းၵိတ်းၵူၺ်း။
* **UTF-8 (Unicode):** ပဵၼ်လၢႆးမႂ်ႇ ဢၼ်ႁူမ်ႈဢဝ်တူဝ်လိၵ်ႈၵူႈၽႃႇသႃႇၼႂ်းလေႃးၵ (ပႃးတင်းလိၵ်ႈတႆး) ဝႆႉၸွမ်းၵၼ်။

### 2. Code တႃႇပိုတ်ႇၸႂ်ႉ Unicode ၼႂ်း C#
ႁဝ်းလူဝ်ႇတႅမ်ႈ Code 2 ထႅဝ်ၼႆႉ ဝႆႉတီႈၼိူဝ်သုတ်း (Top) ၼႂ်း File `Program.cs` ႁဝ်းၶႃႈ:

```csharp
using System.Text;

// 1. တင်ႈၵၢၼ်တႃႇၼႄၽိူၼ် (Output) ႁႂ်ႈပဵၼ် Unicode
Console.OutputEncoding = Encoding.UTF8;

// 2. တင်ႈၵၢၼ်တႃႇႁပ်ႉၶေႃႈမုၼ်း (Input) ႁႂ်ႈပဵၼ် Unicode
Console.InputEncoding = Encoding.UTF8;

Console.WriteLine("မႂ်ႇသုင်ၶႃႈ! ၼႆႉပဵၼ်လိၵ်ႈတႆးၼႂ်း C#။");
```

### 3. ၵၢၼ်လိူၵ်ႈ Font ၼႂ်း Console
ပေႃးတႅမ်ႈ Code ယဝ်ႉယဝ်ႉ ၵူၺ်းၵႃႈလိၵ်ႈတႆးပႆႇဢွၵ်ႇ (ပဵၼ်လွၵ်းသီႇၸဵင်ႇ)၊ ၼၼ်ႉယွၼ်ႉ Font ၼႃႈလိၵ်ႈသီလမ်ၼၼ်ႉ ဢမ်ႇပႃးလိၵ်ႈတႆးဢေႃႈ။
1. တဵၵ်း Right-click တီႈႁူဝ်ၼႃႈလိၵ်ႈ Console (သီလမ်)။
2. လိူၵ်ႈ **Properties** > **Font**။
3. လိူၵ်ႈ Font ဢၼ်ၵမ်ႉထႅမ် Unicode မိူၼ်ၼင်ႇ **"Consolas"**, **"Cascadia Code"**, ဢမ်ႇၼၼ် Font တႆး **"NamKhone Unicode"** ၶႃႈ။

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Unicode:** ပိူင်မၢႆတွင်းတူဝ်လိၵ်ႈ သႅၼ်းၵူႈမိူင်းမိူင်း ဢၼ်ႁဵတ်းႁႂ်ႈၶွမ်းပိဝ်ႇတႃႇ ဢၢၼ်းလိၵ်ႈလႆႈၵူႈၽႃႇသႃႇ။
* **Encoding:** လၢႆးပိၼ်ႇတူဝ်လိၵ်ႈ ႁႂ်ႈပဵၼ်မၢႆ (Numbers) ဢၼ်ၶွမ်းပိဝ်ႇတႃႇသိမ်းဝႆႉလႆႈ။
* **UTF-8:** လၢႆး Encoding ဢၼ်ၵူၼ်းၸႂ်ႉၼမ်သေပိူၼ်ႈ ၼိူဝ် Internet လႄႈ ၼႂ်း Programming။
* **Glyph:** ႁူပ်ႈႁၢင်ႈတူဝ်လိၵ်ႈ ဢၼ်ၼႄပၼ်ၼိူဝ်ၼႃႈလိၵ်ႈ (ပေႃးဢမ်ႇမီး Glyph မၼ်းတေဢွၵ်ႇပဵၼ် လွၵ်းပဝ်ႇ)။
