# Lesson 28: "Assignment Operators" (ၶိူင်ႈမၢႆပၼ်ၵႃႈၶၼ်)

မိူဝ်ႈႁဝ်းၶႂ်ႈဢဝ် Variable ၼိုင်ႈဢၼ် မႃးၼပ်ႉသၢင်ႈယဝ်ႉ သႂ်ႇၶိုၼ်းၼႂ်းတူဝ်မၼ်းၼၼ်ႉ ႁဝ်းမီးလၢႆးတႅမ်ႈ 2 ပိူင်ၶႃႈ။

### 1. Simple Assignment (=)
ၼႆႉပဵၼ်ဢၼ်ႁဝ်းၸႂ်ႉယူႇၵူႈပွၵ်ႈ တႃႇဢဝ်ၵႃႈၶၼ်ၽၢႆႇၶႂႃ ၵႂႃႇသႂ်ႇၼႂ်း Variable ၽၢႆႇသၢႆႉ။
* **တူဝ်ယင်ႇ:** `int x = 10;`

### 2. Compound Assignment (+=, -=, *=, /=, %=)
ၼႆႉပဵၼ်လၢႆး "ႁူမ်ႈ" ၶိူင်ႈမၢႆၼပ်ႉသၢင်ႈ (Arithmetic Operators) လႄႈ ၶိူင်ႈမၢႆသႂ်ႇၵႃႈၶၼ် (Assignment Operator) ၸွမ်းၵၼ်ဢေႃႈ။

| Operator | Full Form (လၢႆးတႅမ်ႈတဵမ်) | Shorthand (လၢႆးတႅမ်ႈပွတ်း) | Meaning                              |
| :--- | :--- | :--- |:-------------------------------------|
| **`+=`** | `x = x + 5;` | **`x += 5;`** | ထႅမ် 5 သႂ်ႇၼႂ်း x                    |
| **`-=`** | `x = x - 2;` | **`x -= 2;`** | ထွၼ် 2 တီႈၼႂ်း x                     |
| **`*=`** | `x = x * 3;` | **`x *= 3;`** | ၵုၼ်း 3 ပၼ်ႇၼႂ်း x                   |
| **`/=`** | `x = x / 4;` | **`x /= 4;`** | ၸႅၵ်ႇ 4 ၼႂ်း x                       |
| **`%=`** | `x = x % 2;` | **`x %= 2;`** | ဢဝ်တူဝ်လိူဝ် ဝၢႆးသေၸႅၵ်ႇ 2 တီႈၼႂ်း x |



### 3. တူဝ်ယင်ႇ Code တႄႉတႄႉ
လွင်ႈၸႂ်ႉတိုဝ်း Compound Assignment ၼၼ်ႉ တေႁဵတ်းႁႂ်ႈၶူတ်ႉႁဝ်း လူငၢႆလႄႈ ပွတ်းလိူဝ်ၵဝ်ႇယဝ်ႉ။
ၶူတ်ႉတႂ်ႈၼႆႉ လႆႈၸႂ်ႉဝႆႉ မိူဝ်ႈႁဝ်းႁဵတ်း Game သေ ၶႂ်ႈထႅမ် "Score" ပၼ်လုၵ်ႈႁဵၼ်းယဝ်ႉ။

```csharp
int myScore = 100;

// ထႅမ် Score 50 ၵမ်းလဵဝ်
myScore += 50;  # မိူၼ်ၵၼ်တင်း myScore = myScore + 50
Console.WriteLine(myScore); // Output: 150

// ယွမ်း Score 20
myScore -= 20;  # မိူၼ်ၵၼ်တင်း myScore = myScore - 20
Console.WriteLine(myScore); // Output: 130
```

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Assignment:** ၵၢၼ်သႂ်ႇၵႃႈၶၼ် (Value) ၼႂ်း Variable။
* **Compound:** လွင်ႈဢဝ်ၶေႃႈၵႂၢမ်း ဢမ်ႇၼၼ် ၶိူင်ႈမၢႆ လၢႆဢၼ်မႃးႁူမ်ႈၵၼ်။
* **Shorthand:** လၢႆးတႅမ်ႈ Code ႁႂ်ႈပွတ်း လႄႈ ၽႂ်းၶိုၼ်ႈ။
* **Readability:** လွင်ႈဢၢၼ်ႇ Code လႆႈငၢႆႈ (Compound assignment ၸွႆးႁႂ်ႈ Code ႁဝ်းသႅၼ်ႈသႂ် ဢမ်ႇယုင်ႈၼမ်)။
