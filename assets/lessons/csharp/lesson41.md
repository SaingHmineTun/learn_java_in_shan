# Lesson 41: "The 'if' Statement" (ၵၢၼ်တႅတ်ႉသႅတ်းငၢႆႈငၢႆႈ)

**'if' statement** ၼႆႉ ၸႂ်ႉတႃႇသင်ႇႁႂ်ႈ Program ႁဵတ်းၵၢၼ်ၼႂ်း Code Block `{ }` ဢၼ်လႆႈတႅမ်ႈဝႆႉၼႂ်းတူဝ်မၼ်း
မိူဝ်ႈ "ၶေႃႈထၢမ်" (Condition) ၼၼ်ႉပဵၼ် **True** ယဝ်ႉ။ ပေႃးပဵၼ် False မၼ်းတေ "ၶၢမ်ႈ" (Skip) ၵႂႃႇၵမ်းလဵဝ်ယဝ်ႉ။

### 1. ပိူင်သၢင်ႈ (Syntax)
```csharp
if (condition) 
{
    // Code တွၼ်ႈၼႆႉ တေႁဵတ်းၵၢၼ် ပေႃး condition ပဵၼ် true
}
```



### 2. တူဝ်ယင်ႇ Code တႄႉတႄႉ
ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းတႅမ်ႈ App တႃႇၵူတ်ႇထတ်းမၢႆၼပ်ႉ Score လုၵ်ႈႁဵၼ်းၶႃႈ။

```csharp
int score = 85;

if (score >= 50) 
{
    Console.WriteLine("Congratulations! You passed the exam.");
}

// ထႅဝ်ၼႆႉ တေႁဵတ်းၵၢၼ်တႃႇသေႇ ဢမ်ႇၵဵဝ်ႇသင်ၵၼ်တင်း if
Console.WriteLine("End of program.");
```

* **ပေႃး score ပဵၼ် 85:** မၼ်းတေၼႄ "Congratulations!..." ယဝ်ႉၸင်ႇၼႄ "End of program."
* **ပေႃး score ပဵၼ် 40:** မၼ်းတေ "ၶၢမ်ႈ" တွၼ်ႈၼႂ်း `{ }` သေ ၼႄ "End of program." ၵမ်းလဵဝ်။


### 3. လွင်ႈလူဝ်ႇၾၢင်ႉ (Common Mistakes)
* **Semicolon (;):** ႁဝ်းဢမ်ႇသႂ်ႇ `;` ဝႆႉလင် `if (...)` ၶႃႈ။ ပေႃးသႂ်ႇ `;` ဝႆႉလင်မၼ်း၊ C# တေဝူၼ်ႉဝႃႈ `if` ယဝ်ႉတူဝ်ႈၵႂႃႇယဝ်ႉသေ တေႁဵတ်းႁႂ်ႈ Code ၼႂ်း `{ }` မၼ်းၼၼ်ႉ ႁဵတ်းၵၢၼ်တႃႇသေႇ (Logic Error)။
* **Curly Braces { }:** ပေႃး Code ၼႂ်း `if` မီးထႅဝ်လဵဝ်ၵူၺ်း ႁဝ်းဢမ်ႇသႂ်ႇ `{ }` ၵေႃႈလႆႈ၊ ၵူၺ်းၵႃႈ တႃႇလုၵ်ႈႁဵၼ်းမႂ်ႇတႄႉ **သႂ်ႇဝႆႉတႃႇသေႇ** တေလီသေပိူၼ်ႈၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Branching:** ၵၢၼ်ဢၼ် Program ၸႅၵ်ႇတၢင်းပႆ ပဵၼ်လၢႆလၢႆသဵၼ်ႈ (မိူၼ်ၼင်ႇ ၵိင်ႇမႆႉ)။
* **Statement:** ထႅဝ် Code ၼိုင်ႈထႅဝ် ဢၼ်သင်ႇၵၢၼ် Program။
* **Predicate:** ၸိုဝ်ႈထႅင်ႈပိူင်ၼိုင်ႈ ၶွင် Boolean Expression ဢၼ်ယူႇၼႂ်း `if`။
* **Indentation:** ၵၢၼ်ႁူၼ်ထႅဝ် Code ၶဝ်ႈၵႂႃႇၽၢႆႇၼႂ်း (Tab) တႃႇႁႂ်ႈႁူႉဝႃႈ Code တွၼ်ႈၼၼ်ႉ ၵိုၵ်းတင်း `if` ဢၼ်လႂ်။
