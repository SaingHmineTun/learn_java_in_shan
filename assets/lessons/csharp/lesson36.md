# Lesson 36: "The Math Class" (ၸုမ်းၶိူင်ႈမိုဝ်းပၢႆးၼပ်ႉ Level သုင်)

**Math Class** ၼႆႉပဵၼ် Static Class (ႁဝ်းၸႂ်ႉၸိုဝ်ႈမၼ်း ႁွင်ႉလႆႈၵမ်းလဵဝ်) ဢၼ်မီး Method လၢႆလၢႆဢၼ် တႃႇၸွႆႈ ၵၢၼ်ၼပ်ႉသၢင်ႈၶႃႈ။

### 1. Common Math Methods (Method ဢၼ်ၸႂ်ႉၼမ်)

| Method | Purpose (တၢင်းၸႂ်ႉတိုဝ်း) | Example | Result |
| :--- | :--- | :--- | :--- |
| **`Math.Pow(x, y)`** | **Power** (x ၼပ်ႉပၼ်ႇ y ပွၵ်ႈ) | `Math.Pow(2, 3)` | 8.0 ($2^3$) |
| **`Math.Sqrt(x)`** | **Square Root** (ႁႃႁၢၵ်ႈၵူပ်ႉ) | `Math.Sqrt(16)` | 4.0 |
| **`Math.Abs(x)`** | **Absolute Value** (ဢဝ်ၵႃႈၶၼ်ပဵၼ် +) | `Math.Abs(-50)` | 50 |
| **`Math.Max(a, b)`** | **Maximum** (လိူၵ်ႈဢဝ်တူဝ်ယႂ်ႇ) | `Math.Max(10, 20)` | 20 |
| **`Math.Min(a, b)`** | **Minimum** (လိူၵ်ႈဢဝ်တူဝ်လဵၵ်း) | `Math.Min(10, 20)` | 10 |



### 2. Rounding Functions (ၵၢၼ်ပိၼ်ႇမၢႆၼပ်ႉ)
မိူဝ်ႈႁဝ်းမီးမၢႆၼပ်ႉၸုၵ်း (Double) ယဝ်ႉၶႂ်ႈပိၼ်ႇႁႂ်ႈမၼ်းပဵၼ်မၢႆၼပ်ႉတဵမ်ၼၼ် ႁဝ်းမီး 3 လၢႆးၼႆႉၶႃႈ:

* **`Math.Round(x)`**: ပိၼ်ႇၸူးတူဝ်ဢၼ်ၸမ်သေပိူၼ်ႈ (1.2 ပဵၼ် 1၊ 2.9 ပဵၼ် 3)။
* **`Math.Ceiling(x)`**: ပိၼ်ႇ **"ၶိုၼ်ႈ"** တႃႇသေႇ (မိူၼ်ၼင်ႇ ၸပ်းၽၢၵ်ႈ / Ceiling)။
    * `Math.Ceiling(4.1)` -> **5**
* **`Math.Floor(x)`**: ပိၼ်ႇ **"လူင်း"** တႃႇသေႇ (မိူၼ်ၼင်ႇ ၸပ်းလိၼ် / Floor)။
    * `Math.Floor(4.9)` -> **4**

### 3. Math Constants (ၵႃႈၶၼ်တႅတ်ႉတေႃး)
ႁဝ်းဢမ်ႇလူဝ်ႇတႅမ်ႈ 3.14 ႁင်းၵဝ်ႇယဝ်ႉ၊ ႁဝ်းၸႂ်ႉၵႃႈၶၼ် ဢၼ်မီးဝႆႉၼႂ်း Math လႆႈၵမ်းလဵဝ်:
* `Math.PI` -> 3.14159265358979...
* `Math.E` -> 2.71828182845904...

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Static Class:** Class ဢၼ်ႁဝ်းဢမ်ႇလူဝ်ႇသၢင်ႈ `new` (Object) ၊ ၸႂ်ႉၸိုဝ်ႈမၼ်းႁွင်ႉ Method လႆႈၵမ်းလဵဝ်။
* **Argument:** ၵႃႈၶၼ် ဢမ်ႇၼၼ် Variable ဢၼ်ႁဝ်းသူင်ႇၶဝ်ႈၵႂႃႇၼႂ်း Method (ၼႂ်း `Math.Sqrt(16)` ၼၼ်ႉ 16 ပဵၼ် Argument)။
* **Return Value:** ၽွၼ်းလႆႈ ဢၼ် Method သူင်ႇမႃးပၼ်ႁဝ်းၶိုၼ်း။
* **Absolute:** ၵႃႈၶၼ်ဢၼ်ဢမ်ႇသူၼ်ၸႂ်မၢႆ "လႅပ်ႉ/လူပ်း" (-)၊ တေဢွၵ်ႇမႃးပဵၼ် "ၵႃႈၶၼ်တႄႉ" (+) တႃႇသေႇ။
