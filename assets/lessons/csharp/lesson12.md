# Lesson 12: "Common Compilation Errors" (ၶေႃႈၽိတ်းပိူင်ႇ ဢၼ်ၵႆႉပဵၼ် လႄႈ လၢႆးၵႄႈလႅတ်ႈ)

မိူဝ်ႈႁဝ်းတႅမ်ႈ Code ၽိတ်းပိူင် "သတ်ႉတႃႇ" (Syntax) မၼ်းၼၼ်ႉ **Compiler** တေဢမ်ႇยอมပိၼ်ႇ Code ႁဝ်းပဵၼ် App လႆႈ။ မၼ်းတေၼႄသီလႅင်ဝႆႉ ဢၼ်ႁဝ်းႁွင်ႉဝႃႈ **Compilation Error** ၼၼ်ႉယဝ်ႉ။

### 1. တူဝ်ယင်ႇ Error ဢၼ်ၵႆႉပဵၼ်ၼမ်သေပိူၼ်ႈ
* **Semicolon Expected (CS1002):** လืมသႂ်ႇမၢႆ `;` တီႈသုတ်းထႅဝ်။
* **The name '...' does not exist in the current context (CS0103):** တႅမ်ႈၸိုဝ်ႈ Variable ၽိတ်း ဢမ်ႇၼၼ် ပႆႇလႆႈပိုတ်ႇၸႂ်ႉ (Declare) မၼ်း။
* **Cannot implicitly convert type 'string' to 'int' (CS0029):** ဢဝ်ၶေႃႈမုၼ်းလၢႆပိူင် (လိၵ်ႈ လႄႈ မၢႆၼပ်ႉ) မႃးပၼ်ၵၼ် ၼႂ်းလၢႆးဢၼ်ဢမ်ႇမႅၼ်ႈ။

### 2. လၢႆးဢၢၼ်း Error List
မိူဝ်ႈမီး Error၊ ၼႂ်း Visual Studio တေမီးၼႃႈလိၵ်ႈ **Error List** ယူႇၽၢႆႇတႂ်ႈ။
1. **Description:** မၼ်းတေလၢတ်ႈၼႄဝႃႈ မၼ်းၽိတ်းသင်။
2. **Project/File:** မၼ်းတေၼႄဝႃႈ ၽိတ်းဝႆႉတီႈ File လႂ်။
3. **Line:** မၼ်းတေၼႄ "မၢႆထႅဝ်" ဢၼ်ၽိတ်းၼၼ်ႉတႅတ်ႉတႅတ်ႉ။


### 3. Quick Actions (လၢႆးၵႄႈလႅတ်ႈၽႂ်း)
Visual Studio မီး "ၵွၼ်မႆႉၵၢင်" (Magic Wand) ဢၼ်ၸိုဝ်ႈဝႃႈ **Quick Actions** (မၢႆႁူပ်ႈ ငၢဝ်းၾႆးသီလိူင်)။ ပေႃးႁဝ်းတဵၵ်း **Ctrl + .** (မၢႆၸုၵ်း) တီႈထႅဝ်ဢၼ်ၽိတ်းၼၼ်ႉ မၼ်းတေၼႄလၢႆးၵႄႈလႅတ်ႈပၼ်ႁဝ်းၵမ်းလဵဝ်။

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Compilation Error:** ၶေႃႈၽိတ်းပိူင်ႇ ဢၼ်ပဵၼ်ယွၼ်ႉတႅမ်ႈ Code ဢမ်ႇမႅၼ်ႈၸွမ်းပိူင် (Syntax) ႁဵတ်းႁႂ်ႈ App သၢင်ႈ (Build) ဢမ်ႇလႆႈ။
* **Runtime Error:** ၶေႃႈၽိတ်းပိူင်ႇ ဢၼ်ပဵၼ်မိူဝ်ႈ App ႁဵတ်းၵၢၼ်ယူႇ (မိူၼ်ၼင်ႇ ဢဝ်မၢႆၼပ်ႉမႃးႁၢၼ်တင်း 0)။
* **Error Code:** မၢႆ (ID) ဢၼ် Microsoft တင်ႈဝႆႉ တႃႇၸႅၵ်ႇလွင်ႈၽိတ်းပႅၵ်ႇၵၼ် (တူဝ်ယင်ႇ: CS1002)။
* **Syntax:** ပိူင်လၢႆးတႅမ်ႈ Code ဢၼ်မၢၼ်ႇမႅၼ်ႈ ၸွမ်းၼင်ႇၽႃႇသႃႇ Programming တင်ႈဝႆႉ။
