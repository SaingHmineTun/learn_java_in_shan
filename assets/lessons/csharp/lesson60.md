# Lesson 60: "The 'continue' Statement: Skipping to the Next Iteration"

**'continue'** ၼႆႉ ၸႂ်ႉတႃႇသင်ႇႁႂ်ႈ Program ၵိုတ်းၵၢၼ်ႁဵတ်းသၢင်ႈ ၼိုင်ႈႁွပ်ႈ လႄႈ ႁႂ်ႈမၼ်းၵႂႃႇၵူတ်ႇထတ်း **"ၶေႃႈထၢမ်"** (Condition) တႃႇတေပၼ်ႇတူင်ႇထႅင်ႈယဝ်ႉ။

### 1. ပိူင်ၵၢၼ် (How it Works)
မိူဝ်ႈ Program လႅၼ်ႈမႃးၺႃးၶေႃႈၵႂၢမ်း `continue;`...
* Code ဢၼ်မီးဝႆႉၽၢႆႇတႂ်ႈ `continue` တင်းမူတ်း (ၼႂ်း Loop ၼၼ်ႉ) တေထုၵ်ႇ **"ၶၢမ်ႈ"** (Skip) ၵႂႃႇ။
* Loop တေဝႅၼ်ၶိုၼ်းၵႂႃႇတီႈ **"ၵၢၼ်လႅၵ်ႈလၢႆႈ"** (Update/Iterator) သေ ၵႂႃႇၵူတ်ႇထတ်းၶေႃႈထၢမ်တႃႇႁွပ်ႈၼႃႈၶႃႈ။



### 2. တူဝ်ယင်ႇ Code
ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းၶႂ်ႈၼပ်ႉ 1 ထိုင် 5၊ ၵူၺ်းၵႃႈ ႁဝ်းဢမ်ႇၶႂ်ႈၼႄမၢႆ 3 ၶႃႈ။

```csharp
for (int i = 1; i <= 5; i++)
{
    if (i == 3)
    {
        continue; // ဝျႅတ်ႉၶၢမ်ႈမၢႆ 3 သေ ၵႂႃႇတႄႇ i = 4 ၵမ်းလဵဝ်
    }
    Console.WriteLine("Number: " + i);
}
```

**ၽွၼ်းလႆႈ (Output):**
```text
Number: 1
Number: 2
Number: 4
Number: 5
```
*(မၢႆတွင်း: မၢႆ 3 တေဢမ်ႇၼႄ ယွၼ်ႉမၼ်းၺႃး continue သေဝႅၼ်ၶၢမ်ႈထႅဝ် WriteLine ၵႂႃႇၶႃႈ)*

### 3. မိူဝ်ႈလႂ်လူဝ်ႇၸႂ်ႉ?
* **Filtering:** မိူဝ်ႈႁဝ်းၶႂ်ႈၶၢမ်ႈ (Skip) ၶေႃႈမုၼ်းဢၼ်ႁဝ်းဢမ်ႇၶႂ်ႈလႆႈ (မိူၼ်ၼင်ႇ ၶၢမ်ႈမၢႆၵူပ်ႉ ဢမ်ႇၼၼ် ၶၢမ်ႈၸိုဝ်ႈဢၼ်ပဝ်ႇဝႆႉ)။
* **Cleaning Data:** ၸႂ်ႉတႃႇၵူတ်ႇထတ်းၶေႃႈမုၼ်း ပေႃးၺႃးဢၼ်ၽိတ်း ႁႂ်ႈမၼ်းၶၢမ်ႈၵႂႃႇႁဵတ်းတူဝ်ၼႃႈ။

### 4. လွင်ႈလူဝ်ႇၾၢင်ႉ (Infinite Loop with While)
သင်ၸႂ်ႉ `continue` ၼႂ်း `while` loop၊ လူဝ်ႇၾၢင်ႉဝႃႈ ႁဝ်းလႆႈ **Update Variable** (မိူၼ်ၼင်ႇ `i++`) ဢွၼ်တၢင်းတေ `continue` ႁႃႉ? သင်ဢမ်ႇ Update ဢွၼ်တၢင်း၊ မၼ်းတေဝႅၼ်ၶိုၼ်းၵႂႃႇထၢမ်ၶေႃႈထၢမ် တီႈၵႃႈၶၼ်ၵဝ်ႇ သေႁဵတ်းႁႂ်ႈပဵၼ် **Infinite Loop** ၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Continue Statement:** ၶေႃႈသင်ႇဝႅၼ်ၶၢမ်ႈၵၢၼ်ႁဵတ်းသၢင်ႈၼႂ်းႁွပ်ႈၼၼ်ႉ သေၵႂႃႇတႄႇႁွပ်ႈၼႃႈ။
* **Next Iteration:** ၵၢၼ်ႁဵတ်းပၼ်ႇတူင်ႇ ထႅင်ႈၼိုင်ႈႁွပ်ႈ။
* **Skipping:** ၵၢၼ်ဢၼ်ဢမ်ႇႁဵတ်းၵၢၼ် Code တွၼ်ႈဢၼ်မၵ်းမၼ်ႈဝႆႉသေ ၶၢမ်ႈပူၼ်ႉၵႂႃႇ။
* **Flow Control:** ၵၢၼ်ၵုမ်းထိင်းတၢင်းပႆၶွင် Program ႁႂ်ႈပဵၼ်ၸွမ်းၼင်ႇႁဝ်းလူဝ်ႇ။
