# Lesson 73: "The Array Class: Sorting, Reversing, and Searching Data"

**Array Class** ၼႆႉ ပဵၼ် Static Class ဢၼ်မီးဝႆႉၼႂ်း `System` Namespace ဢေႃႈ။ 
ႁဝ်းဢမ်ႇလူဝ်ႇသၢင်ႈမၼ်းမႂ်ႇ၊ ႁဝ်းႁွင်ႉၸႂ်ႉလႆႈၵမ်းလဵဝ်ၶႃႈ။

### 1. ၵၢၼ်ၶပ်ႉထႅဝ် (Sorting)
သင်ႁဝ်းမီးမၢႆၼပ်ႉ ဢမ်ႇၼၼ် ၸိုဝ်ႈဝႆႉသေ ၶႂ်ႈႁႂ်ႈမၼ်း "ၶပ်ႉထႅဝ်" (Sort) ၸွမ်းၼင်ႇ တူဝ်လိၵ်ႈ (A-Z) ဢမ်ႇၼၼ် မၢႆၼပ်ႉ (လဵၵ်းၵႂႃႇယႂ်ႇ)။

```csharp
string[] students = { "Sai Tun", "Nang Kham", "Sai Mao" };

Array.Sort(students); 
// ၽွၼ်းလႆႈတေပဵၼ်: { "Nang Kham", "Sai Mao", "Sai Tun" }
```



### 2. ၵၢၼ်ပိၼ်ႈၽၢႆႇ (Reversing)
လၢႆးၼႆႉ တေဢဝ်တူဝ်သုတ်း မႃးဝႆႉတူဝ်ၼႃႈသေ ပိၼ်ႈၶိုၼ်းတင်းမူတ်းၶႃႈ။

```csharp
int[] numbers = { 1, 2, 3, 4, 5 };

Array.Reverse(numbers);
// ၽွၼ်းလႆႈတေပဵၼ်: { 5, 4, 3, 2, 1 }
```

### 3. ၵၢၼ်ႁႃၶေႃႈမုၼ်း (Searching)
ႁဝ်းၸၢင်ႈၸႂ်ႉ **`Array.IndexOf()`** တႃႇႁႃဝႃႈ ၶေႃႈမုၼ်းဢၼ်ႁဝ်းႁႃၼၼ်ႉ ယူႇတီႈ **မၢႆၼပ်ႉ (Index)** ၵႃႈႁိုဝ်ၶႃႈ။

```csharp
string[] fruits = { "Apple", "Banana", "Orange" };

int position = Array.IndexOf(fruits, "Banana");
// ၽွၼ်းလႆႈတေပဵၼ်: 1
// (သင်ႁႃဢမ်ႇၺႃး မၼ်းတေသူင်ႇၵႃႈၶၼ် -1 မႃးပၼ်ၶႃႈ)
```

### 4. ၵၢၼ်ၽႃႇ (Binary Search)
သင်ဝႃႈ Array ႁဝ်း **Sort ဝႆႉယဝ်ႉယဝ်ႉ** (မီးလွၵ်းလၢႆး A-Z)၊ ႁဝ်းၸၢင်ႈၸႂ်ႉ **`Array.BinarySearch()`** တႃႇႁႃၶေႃႈမုၼ်းလႆႈ ၽႂ်းလိူဝ် `IndexOf` ၼမ်ၼႃႇၶႃႈ။ 
(လႅပ်ႈတွၼ်ႈတႃႇ ၶေႃႈမုၼ်းဢၼ်မီးပဵၼ်လၢၼ်ႉဢၼ်)။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Static Method:** Method ဢၼ်လုၵ်ႉတီႈၸိုဝ်ႈ Class သေ ႁွင်ႉၸႂ်ႉလႆႈၵမ်းလဵဝ်  (မိူၼ်ၼင်ႇ Array.Sort)။
* **Sorting:** ၵၢၼ်ၶပ်ႉထႅဝ်ၶေႃႈမုၼ်း ႁႂ်ႈပဵၼ်ထႅဝ်ၸွမ်းၼင်ႇပိူင်မၵ်းမၼ်ႈ။
* **Reversing:** ၵၢၼ်ပိၼ်ႈၽၢႆႇၶေႃႈမုၼ်း ဢဝ်လင်ပဵၼ်ၼႃႈ/ၼႃႈပဵၼ်လင်။
* **IndexOf:** ၵၢၼ်ႁႃမၢႆၼပ်ႉ Index ၶွင်ၶေႃႈမုၼ်း ဢၼ်ႁဝ်းလူဝ်ႇ။
* **Binary Search:** လၢႆးႁႃၶေႃႈမုၼ်းဢၼ်ၽႂ်းၼႃႇ (ၵူၺ်းၵႃႈ လူဝ်ႇ Sort ဝႆႉဢွၼ်တၢင်း)။
