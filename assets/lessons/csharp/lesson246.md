# Lesson 246: "The 'Async' Naming Convention"

မိူဝ်ႈႁဝ်းတႅမ်ႈ Code ဢၼ်ပဵၼ် Asynchronous ၼၼ်ႉ ႁဝ်းမီးပၵ်းၵၢၼ် (Convention) ဢၼ်ၵူၼ်းတႅမ်ႈ App ၵူႈၵေႃႉ တိုဝ်းၵမ်ၸွမ်းၵၼ် တႃႇႁႂ်ႈ Code ႁဝ်းလူငၢႆႈ လႄႈ ဢမ်ႇယုင်ႈယၢင်ႈၶႃႈ။

### 1. တေလႆႈသုတ်းတွၼ်ႈလူၺ်ႈ "Async"
ၵူႈပွၵ်ႈဢၼ် **ႁဝ်း** သၢင်ႈ Method ဢၼ်သူင်ႇၽွၼ်းလႆႈပဵၼ် `Task` ဢမ်ႇၼၼ် `Task<T>` ၼၼ်ႉ ႁဝ်းတေလႆႈထႅမ်ၶေႃႈၵႂၢမ်း **"Async"** ဝႆႉတီႈပၢႆၸိုဝ်ႈ Method တႃႇသေႇၶႃႈ။

* **လၢႆးတႅမ်ႈဢၼ်ၽိတ်း:** `public Task GetData()`
* **လၢႆးတႅမ်ႈဢၼ်မႅၼ်ႈ:** `public Task GetDataAsync()`



### 2. ႁဵတ်းသင်ႁဝ်းၸင်ႇလူဝ်ႇထႅမ် "Async"?
1.  **ႁႂ်ႈႁူႉၵမ်းလဵဝ်:** မိူဝ်ႈၵူၼ်းတၢင်ႇၵေႃႉမႃးႁွင်ႉၸႂ်ႉ Method ႁဝ်းၼႆ ၶဝ်တေႁူႉလႆႈၵမ်းလဵဝ်ဝႃႈ "ဢေႃႈ... ဢၼ်ၼႆႉတေလႆႈၸႂ်ႉ `await` ပႂ်ႉမၼ်းၼႃ" ၼႆၶႃႈ။
2.  **ႁႄႉၵၢင်ႈၵၢၼ်လူင်တူဝ်:** သင်ဢမ်ႇမီးၸိုဝ်ႈ Async ဝႆႉၼႆ ႁဝ်းၸၢင်ႈလူင်သေႁွင်ႉၸႂ်ႉမၼ်းလၢႆး "ထမ်ႇမတႃႇ" (Synchronous) သေ ႁဵတ်းႁႂ်ႈ UI ႁဝ်း Freeze ၵႂႃႇလႆႈၶႃႈ။
3.  **Overloading:** မၢင်ပွၵ်ႈႁဝ်းလူဝ်ႇမီး တင်းလၢႆး Sync လႄႈ Async ၸွမ်းၵၼ် ႁဝ်းၸင်ႇၸႅၵ်ႇၸိုဝ်ႈမၼ်းလႆႈငၢႆႈၶႃႈ:
    * `ReadData()` (Sync - ပႂ်ႉတေႃႇယဝ်ႉ)
    * `ReadDataAsync()` (Async - ဢမ်ႇပႂ်ႉ)

### 3. ၶေႃႈယႅၵ်ႈ (Exceptions)
ႁဝ်းတေ **ဢမ်ႇထႅမ်** "Async" ဝႆႉတီႈပၢႆၸိုဝ်ႈ မိူဝ်ႈပဵၼ်:
* **Event Handlers:** မိူၼ်ၼင်ႇ `btnSave_Click` (ႁဝ်းဢမ်ႇတႅမ်ႈ `btnSave_ClickAsync`)။
* **Controllers ၼႂ်း Web API:** မၢင်ပွၵ်ႈၼႂ်း ASP.NET ၶဝ်ၵေႃႈဢမ်ႇသူႈထႅမ် ၸွမ်းၼင်ႇပိူင် Routing မၼ်းၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Naming Convention:** ပၵ်းၵၢၼ်ပၼ်ၸိုဝ်ႈ ဢၼ်ၵူၼ်းတႅမ်ႈ Code တိုဝ်းၵမ်ၸွမ်းၵၼ်။
* **Suffix:** ၶေႃႈၵႂၢမ်းဢၼ်ထႅမ်သႂ်ႇတီႈပၢႆ (မိူၼ်ၼင်ႇ ...Async)။
* **Consistency:** လွင်ႈမိူၼ်ၵၼ်တႃႇသေႇ ၼႂ်းၵၢၼ်တႅမ်ႈ Code တင်း Project။
* **API Consumer:** ၵူၼ်းဢၼ်မႃးႁွင်ႉၸႂ်ႉ Code ဢၼ်ႁဝ်းတႅမ်ႈဝႆႉ။
