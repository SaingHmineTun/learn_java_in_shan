# Lesson 80: "The Return Statement: Getting Data Back"

မိူဝ်ႈ Method ႁဵတ်းၵၢၼ်ယဝ်ႉတူဝ်ႈယဝ်ႉၼၼ်ႉ မၼ်းၸၢင်ႈ "သူင်ႇၵႃႈၶၼ်" (Value) မႃးပၼ် တီႈဢၼ်ႁဝ်းႁွင်ႉၸႂ်ႉ (Caller) ၼၼ်ႉၶိုၼ်းယဝ်ႉ။

### 1. ၶေႃႈၵႂၢမ်း `return`
ႁဝ်းၸႂ်ႉ `return` တႃႇသူင်ႇၶေႃႈမုၼ်းဢွၵ်ႇမႃး မိူဝ်ႈ Program လႅၼ်ႈမႃးၺႃး `return` ယဝ်ႉ...
သင်ၼၼ် 
* Method တေၵိုတ်းၵၢၼ်ႁဵတ်းသၢင်ႈၵမ်းလဵဝ်။
* ၵႃႈၶၼ်ဢၼ်ယူႇၽၢႆႇလင် `return` တေလႆႈသူင်ႇၶိုၼ်းၵႂႃႇၸူး Caller။



### 2. တူဝ်ယင်ႇ Code (Math Calculation)
ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းမီး Method တႃႇလေႃးမၢႆၼပ်ႉသွင်တူဝ် သေၶႂ်ႈလႆႈၶေႃႈတွပ်ႇမၼ်းမႃးၸႂ်ႉထႅင်ႈၶႃႈ။

```csharp
// Return Type ပဵၼ် 'int' (မၢႆၼပ်ႉ)
public int Add(int a, int b) 
{
    int result = a + b;
    return result; // သူင်ႇမၢႆၼပ်ႉဢွၵ်ႇၵႂႃႇ
}

// ၵၢၼ်ႁွင်ႉၸႂ်ႉ
int sum = Add(10, 20); // sum တေမီးၵႃႈၶၼ်ပဵၼ် 30
Console.WriteLine("Total: " + sum);
```


### 3. လွင်ႈလူဝ်ႇၾၢင်ႉ (Return Rules)
* **Type Matching:** သင်ႁဝ်းမၵ်းမၼ်ႈဝႆႉဝႃႈ `public string`... ႁဝ်းတေလႆႈ `return` ပဵၼ်တူဝ်လိၵ်ႈ (String) တႃႇသေႇၶႃႈ။
* **One Value:** Method ၼိုင်ႈဢၼ် ၸၢင်ႈ `return` လႆႈ **ၵႃႈၶၼ်ဢၼ်လဵဝ်** ၵူၺ်းၶႃႈ (ၵူၺ်းၵႃႈ ႁဝ်းၸၢင်ႈသူင်ႇ Array ဢမ်ႇၼၼ် Object ဢွၵ်ႇမႃးလႆႈၶႃႈ)။
* **Void Method:** သင်ပဵၼ် `void` ႁဝ်းဢမ်ႇလူဝ်ႇ `return` ၵႃႈၶၼ်သင်၊ ၵူၺ်းၵႃႈ ႁဝ်းၸၢင်ႈၸႂ်ႉ `return;` တႃႇဢွၵ်ႇမႃးၼႂ်း Method ၼၼ်ႉဝႆးဝႆးလႆႈၶႃႈ။



### 4. ႁဵတ်းသင်ၸင်ႇၸႂ်ႉ?
* **Calculations:** တႃႇလႆႈၶေႃႈတွပ်ႇမၢႆၼပ်ႉ။
* **Validation:** တႃႇၵူတ်ႇထတ်းဝႃႈ "မႅၼ်ႈ" (`true`) ဢမ်ႇၼၼ် "ၽိတ်း" (`false`)။
* **Data Processing:** တႃႇလႆႈတူဝ်လိၵ်ႈဢၼ်မႄးယဝ်ႉယဝ်ႉ (မိူၼ်ၼင်ႇ ၸိုဝ်ႈတဵမ်)။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Return Type:** မဵဝ်းၶေႃႈမုၼ်း ဢၼ် Method မၵ်းမၼ်ႈဝႃႈ တေသူင်ႇဢွၵ်ႇမႃး။
* **Return Value:** ၶေႃႈမုၼ်းတႄႉတႄႉ ဢၼ်သူင်ႇၶိုၼ်းမႃးၸူး Caller။
* **Caller:** တွၼ်ႈ Code တီႈဢၼ်သင်ႇႁွင်ႉၸႂ်ႉ Method။
* **Expression:** တွၼ်ႈ Code ဢၼ်မီးၵႃႈၶၼ် (မိူၼ်ၼင်ႇ a + b) ဢၼ်ၸၢင်ႈ return လႆႈ။
