# Lesson 118: "Access Modifiers Overview"

**Access Modifiers** ပွင်ႇဝႃႈ "ၶေႃႈၵႂၢမ်းမၵ်းမၼ်ႈသုၼ်ႇၶဝ်ႈၸႂ်ႉ"။ မၼ်းတေလၢတ်ႈၼႄ Compiler ဝႃႈ "Code တွၼ်ႈၼႆႉ ၽႂ်ၸၢင်ႈႁၼ်လႆႈ? ၽႂ်ၸၢင်ႈၸႂ်ႉလႆႈ?" ၼၼ်ႉယဝ်ႉၶႃႈ။

### 1. ႁဵတ်းသင်ႁဝ်းလူဝ်ႇမီးလၢႆပိူင်?
ဝူၼ်ႉတူၺ်းဝႃႈ ၼႂ်းႁိူၼ်းႁဝ်း:
* **Public:** မိူၼ် "သူၼ်မွၵ်ႇၼႃႈႁိူၼ်း" (ၽႂ်ပူၼ်ႉၵႂႃႇမႃးၵေႃႈႁၼ် လႄႈ ၶဝ်ႈလႆႈ)။
* **Private:** မိူၼ် "တီႈသိမ်းငိုၼ်း" (ၸဝ်ႈႁိူၼ်းၵူၺ်း ဢၼ်ႁူႉ လႄႈ ပိုတ်ႇလႆႈ)။
* **Internal:** မိူၼ် "ႁူင်းၵိၼ်ၶဝ်ႈၼႂ်းႁိူၼ်း" (ၵူၼ်းၼႂ်းၼႃႈႁိူၼ်းလဵဝ်ၵၼ်ၵူၺ်း ဢၼ်ၶဝ်ႈလႆႈ)။

### 2. ပိူင်လူင် 4 ယၢင်ႇ ၼႂ်း C#



| Modifier | လႅၼ်းႁၼ် (Scope)                        | ၵၢၼ်ၸႂ်ႉတႄႉ |
| :--- |:----------------------------------------| :--- |
| **`public`** | **ၵူႈတီႈ** (Anywhere)                   | တႃႇ Methods ဢၼ်ပိူၼ်ႈလူဝ်ႇႁွင်ႉၸႂ်ႉ |
| **`private`** | **ၼႂ်း Class လဵဝ်ၵၼ်**                  | တႃႇ Fields (Data) ဢၼ်လူဝ်ႇလပ်ႉဝႆႉ |
| **`protected`** | **ၼႂ်း Class မၼ်း လႄႈ Class လုၵ်ႈလၢၼ်** | တႃႇလွင်ႈသိုပ်ႇၸိူဝ်ႉၽၼ်း (Inheritance) |
| **`internal`** | **ၼႂ်း Project (Assembly) လဵဝ်ၵၼ်**     | တႃႇ Component ၼႂ်း Tool ဢၼ်ႁဝ်းသၢင်ႈ |

### 3. ပိူင်ၵၢၼ် (Default Rules)
* သင်ႁဝ်း **ဢမ်ႇတႅမ်ႈ** Access Modifier ဝႆႉၼႂ်း Class Member (Field/Method) ၸိုင်... C# တေမၵ်းမၼ်ႈပဵၼ် **`private`** ႁင်းၵူၺ်းမၼ်း တႃႇၵႅတ်ႇၶႄၶေႃႈမုၼ်းၶႃႈ။
* သင်ပဵၼ် Class ၼၼ်ႉသမ်ႉ တေပဵၼ် **`internal`** ႁင်းၵူၺ်းမၼ်းၶႃႈ။

### 4. ႁဵတ်းသင်ၸင်ႇလမ်ႇလွင်ႈ?
1. **Security:** ႁႄႉၵၢင်ႈဢမ်ႇပၼ် Code ၽၢႆႇၼွၵ်ႈ မႃးမႄးၶေႃႈမုၼ်းသုၵ်ႉယုင်ႈ။
2. **Clean Interface:** ပၼ်ၵူၼ်းၸႂ်ႉ Code ႁဝ်းႁၼ် "တွၼ်ႈဢၼ်ၶဝ်လူဝ်ႇၸႂ်ႉ" ၵူၺ်း၊ ဢမ်ႇလူဝ်ႇႁႂ်ႈၶဝ် ႁၼ်တွၼ်ႈဢၼ်သုၵ်ႉယုင်ႈၼႂ်းမၼ်း။
3. **Decoupling:** ႁဵတ်းႁႂ်ႈ Code ႁဝ်းၸႅၵ်ႇၵၼ်ပဵၼ်တွၼ်ႈသေ မႄးတွၼ်ႈၼိုင်ႈ ဢမ်ႇတုမ်ႉတိူဝ်ႉတၢင်ႇတွၼ်ႈလႆႈ။

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Access Modifier:** ၶေႃႈၵႂၢမ်းၵုမ်းထိင်းသုၼ်ႇၶဝ်ႈၸႂ်ႉ (Visibility)။
* **Assembly:** Project ဢၼ် Compile ယဝ်ႉ (မိူၼ်ၼင်ႇ .dll ဢမ်ႇၼၼ် .exe)။
* **Member:** Field, Method, ဢမ်ႇၼၼ် Property ဢၼ်ယူႇၼႂ်း Class။
* **Scope:** ၼႃႈတီႈဢၼ် Code ၼၼ်ႉ "မီးတူဝ်ယူႇ" လႄႈ ႁွင်ႉၸႂ်ႉလႆႈ။
