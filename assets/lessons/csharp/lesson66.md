# Lesson 66: "Declaring and Initializing Arrays"

ၵၢၼ်သၢင်ႈ Array ၼႂ်း C# မီးယူႇသွင်တွၼ်ႈၶႃႈ: **Declaration** (ၵၢၼ်ပိုၼ်ၽၢဝ်ႇၸိုဝ်ႈ) လႄႈ **Initialization** (ၵၢၼ်ပၼ်ၵႃႈၶၼ်တႄႇမၼ်း)။

### 1. ပိူင်သၢင်ႈ (Syntax)
ႁဝ်းၸႂ်ႉမၢႆ **`[]`** တႃႇလၢတ်ႈဝႃႈ ၼႆႉပဵၼ် Array ဢေႃႈ။



```csharp
// 1. ပိုၼ်ၽၢဝ်ႇ Array ဢၼ်တေသိမ်းမၢႆၼပ်ႉ (Integer)
int[] scores;

// 2. မၵ်းမၼ်ႈဝႃႈ တေသိမ်းလႆႈ 5 ဢၼ် (Size)
scores = new int[5];
```

### 2. လၢႆးတႅမ်ႈႁူမ်ႈၵၼ် (Initialization)
ႁဝ်းၸၢင်ႈတႅမ်ႈႁူမ်ႈၵၼ်ၼႂ်းထႅဝ်လဵဝ် လႄႈ ပၼ်ၵႃႈၶၼ်ၵမ်းလဵဝ်လႆႈၶႃႈ:

* **လၢႆးမၵ်းမၼ်ႈႁူဝ်ၼပ်ႉဝႆႉဢွၼ်တၢင်း:**
```csharp
string[] students = new string[3]; // သၢင်ႈလွၵ်းပဝ်ႇဝႆႉ 3 လွၵ်း
```

* **လၢႆးပၼ်ၵႃႈၶၼ်ၵမ်းလဵဝ် (Array Initializer):**
```csharp
int[] grades = { 90, 85, 77, 95 }; // Computer တေၼပ်ႉပၼ်ဝႃႈမီး 4 ဢၼ်
```

* **လၢႆးတႅမ်ႈတဵမ်ထူၼ်ႈ:**
```csharp
string[] names = new string[] { "Sai Mao", "Nang Kham", "Sai Tun" };
```

### 3. လွင်ႈလူဝ်ႇၾၢင်ႉ (Fixed Size)
မိူဝ်ႈႁဝ်းသၢင်ႈ Array ယဝ်ႉယဝ်ႉၼၼ်ႉ **"ႁူဝ်ၼပ်ႉ (Size)" မၼ်းတေလႅၵ်ႈလၢႆႈဢမ်ႇလႆႈယဝ်ႉ**။ 
သင်ဝႃႈသၢင်ႈဝႆႉ 5 လွၵ်း၊ တေဢဝ်ၶေႃႈမုၼ်းသႂ်ႇ 6 ဢၼ်ဢမ်ႇလႆႈၶႃႈ။ 
(သင်ၶႂ်ႈလႆႈဢၼ်လႅၵ်ႈလၢႆႈလႆႈ တေလႆႈၸႂ်ႉ `List` ဢၼ်ႁဝ်းတေသွၼ်ၼႂ်းတွၼ်ႈၼႃႈၶႃႈ)။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Declaration:** ၵၢၼ်ပိုၼ်ၽၢဝ်ႇပိူင် (Type) လႄႈ ၸိုဝ်ႈ Array။
* **Initialization:** ၵၢၼ်သၢင်ႈတူဝ် Array တႄႉတႄႉၼႂ်း Memory လူၺ်းၶေႃႈၵႂၢမ်း `new`။
* **Size/Length:** ႁူဝ်ၼပ်ႉ "လွၵ်း" တင်းမူတ်း ဢၼ်မီးၼႂ်း Array။
* **Data Type:** မဵဝ်းၶေႃႈမုၼ်း (မိူၼ်ၼင်ႇ string, int, double) ဢၼ် Array တေသိမ်းပၼ်။
* **New Keyword:** ၶေႃႈၵႂၢမ်းဢၼ်ၸႂ်ႉတႃႇသၢင်ႈ Object မႂ်ႇၼႂ်း Memory။
