# Lesson 54: "The 'do-while' Loop: Guaranteed Single Execution"

ၼႂ်း **Lesson 54** ၼႆႉ ႁဝ်းတေမႃးတူၺ်း **'do-while' loop** ဢေႃႈ။ 
မၼ်းပိူင်ႈၵၼ်တင်း `while` loop တီႈဝႃႈ မၼ်းတေဢမ်ႇထၢမ်သင်သေ **"ႁဵတ်းဢွၼ်တၢင်း"** ၼိုင်ႈပွၵ်ႈ ၸင်ႇတေမႃးၵူတ်ႇထတ်း **"ၶေႃႈထၢမ်"** ဝၢႆးလင်ၶႃႈ။

### 1. ပိူင်သၢင်ႈ (Syntax)

```csharp
do 
{
    // Code တွၼ်ႈၼႆႉ တေႁဵတ်းပၼ်ႇတူင်ႇဢွၼ်တၢင်း 1 ပွၵ်ႈ
    // ယဝ်ႉၸင်ႇတေမႃးၵူတ်ႇထတ်း ၶေႃႈထၢမ်
} 
while (ၶေႃႈထၢမ်); 
```
*(မၢႆတွင်း: တေလႆႈသႂ်ႇ Semicolon `;` ဝႆႉလင် while ၼႂ်းပိူင်ၼႆႉတႃႇသေႇၶႃႈ)*

### 2. တူဝ်ယင်ႇ Code တႄႉတႄႉ
ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းမီးမၢႆၼပ်ႉ `count = 10` သေတႅမ်ႈၶေႃႈထၢမ်ဝႃႈ `count < 5`။

**သင်ၸႂ်ႉ `while` loop:** မၼ်းတေဢမ်ႇၼႄသင်သေပွၵ်ႈ (ယွၼ်ႉ 10 ဢမ်ႇလဵၵ်းသေ 5)။
**သင်ၸႂ်ႉ `do-while` loop:**

```csharp
int count = 10;

do 
{
    Console.WriteLine("Executing at least once!");
} 
while (count < 5); // ၶေႃႈထၢမ်ပဵၼ် False

// ၽိူၼ်ဢွၵ်ႇ: Executing at least once!
```

### 3. မိူဝ်ႈလႂ်လူဝ်ႇၸႂ်ႉ (Real-world Use Case)
ႁဝ်းၵႆႉၸႂ်ႉ `do-while` မိူဝ်ႈႁဝ်းၶႂ်ႈႁပ်ႉၶေႃႈမုၼ်းတီႈ Keyboard ၶႃႈ။ တူဝ်ယင်ႇမိူၼ်ၼင်ႇ ၵၢၼ်ထၢမ် Password:
* တေလႆႈပၼ် User တႅမ်ႈ Password ဢွၼ်တၢင်း (do)
* ၸင်ႇမႃးၵူတ်ႇထတ်းဝႃႈ မႅၼ်ႈႁႃႉ ဢမ်ႇမႅၼ်ႈ (while)
* သင်ဢမ်ႇမႅၼ်ႈ ႁႂ်ႈမၼ်းပၼ်ႇတူင်ႇၵႂႃႇထၢမ်ထႅင်ႈပွၵ်ႈၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Do-While Loop:** ၵၢၼ်ႁဵတ်းပၼ်ႇတူင်ႇ ဢၼ်ၵူတ်ႇထတ်းၶေႃႈထၢမ်ဝၢႆးလင် (Post-test loop)။
* **Guaranteed Execution:** လွင်ႈမၼ်ႈၸႂ်ဝႃႈ Code ၼႂ်း Loop တေႁဵတ်းၵၢၼ် တီႈဢေႇသုတ်ႉ **"ၼိုင်ႈပွၵ်ႈ"**။
* **Post-test:** ၵၢၼ်ၵူတ်ႇထတ်းၶေႃႈထၢမ် ဝၢႆးသေ Code ႁဵတ်းၵၢၼ်ယဝ်ႉ။
* **Semicolon Termination:** ၵၢၼ်လူဝ်ႇသႂ်ႇ `;` ဝႆႉလင် `while(...)` ၼႂ်း do-while။
