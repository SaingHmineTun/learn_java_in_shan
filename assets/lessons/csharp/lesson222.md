# Lesson 222: "Element Operators"

Method ၸိူဝ်းၼႆႉ တေသူင်ႇ "Object" ဢွၵ်ႇမႃးတူဝ်လဵဝ်ၵူၺ်းၶႃႈ (ဢမ်ႇသူင်ႇပဵၼ် List မိူၼ် `.Where()`)။

### 1. First လႄႈ FirstOrDefault (တူဝ်တႄႇသုတ်း)
* **`.First()`**: ဢဝ်တူဝ်ဢွၼ်တၢင်းသုတ်း ဢၼ်ႁႃႁၼ်။ သင်ႁႃဢမ်ႇႁၼ်သေတူဝ်... မၼ်းတေပဵၼ် **Exception (Crash)** ၶႃႈ။
* **`.FirstOrDefault()`**: မိူၼ်ၵၼ်တင်း First ၵူၺ်းၵႃႈ သင်ႁႃဢမ်ႇႁၼ်... မၼ်းတေသူင်ႇ **`null`** (ဢမ်ႇၼၼ် ၵႃႈၶၼ် Default) ဢွၵ်ႇမႃး ႁဵတ်းႁႂ်ႈ App ႁဝ်းလွတ်ႈၽေးၶႃႈ။

```csharp
// ႁႃလုၵ်ႈႁဵၼ်းၵေႃႉဢွၼ်တၢင်းသုတ်း ဢၼ်ဢႃႇယု 20
var firstStudent = students.FirstOrDefault(s => s.Age == 20);
```

### 2. Last လႄႈ LastOrDefault (တူဝ်လင်သုတ်း)
ၸႂ်ႉတႃႇဢဝ်ၶေႃႈမုၼ်း "တူဝ်လင်သုတ်း" ၼႂ်း Collection ၶႃႈ။

* **`.Last()`**: ဢဝ်တူဝ်လင်သုတ်း (သင်ဢမ်ႇမီး တေ Crash)။
* **`.LastOrDefault()`**: ဢဝ်တူဝ်လင်သုတ်း (သင်ဢမ်ႇမီး သူင်ႇ null)။



### 3. Single လႄႈ SingleOrDefault (တူဝ်လဵဝ်ၵူၺ်း)
ၼႆႉပဵၼ်လၢႆးဢၼ် "ၵႅမ်း" (Strict) လိူဝ်ပိူၼ်ႈၶႃႈ။
* **`.Single()`**: တေလႆႈမီး **တူဝ်လဵဝ်ၵူၺ်း** ဢၼ်မႅၼ်ႈၸွမ်းပၵ်းၵၢၼ်။ သင်ႁႃႁၼ် 2 တူဝ် ဢမ်ႇၼၼ် ဢမ်ႇႁၼ်သေတူဝ်... မၼ်းတေ **Crash** ၶႃႈ။
* **`.SingleOrDefault()`**: သင်ဢမ်ႇႁၼ်သေတူဝ် မၼ်းတေသူင်ႇ null၊ ၵူၺ်းၵႃႈ သင်ႁႃႁၼ် 2 တူဝ်... မၼ်းတေ **Crash** ယူႇမိူၼ်ၵဝ်ႇၶႃႈ။

> **Note:** ၵႆႉၸႂ်ႉတႃႇႁႃၶေႃႈမုၼ်းဢၼ်မီး "ID" တႅတ်ႉတေႃး ဢၼ်ဢမ်ႇၸၢင်ႈမိူၼ်ၵၼ်လႆႈ (Unique ID)။

### 4. ပၵ်းၵၢၼ်လိူၵ်ႈၸႂ်ႉ (Summary Table)

| Method | သင်ႁႃႁၼ် 0 တူဝ် | သင်ႁႃႁၼ် ၼမ်လိူဝ် 1 တူဝ် | Usage (ၵၢၼ်ၸႂ်ႉ) |
| :--- | :--- | :--- | :--- |
| **First** | Crash | ဢဝ်တူဝ်တႄႇသုတ်း | ႁႃတူဝ်လႂ်ၵေႃႈလႆႈ ဢၼ်မႃးဢွၼ်တၢင်း |
| **FirstOrDefault** | **Return null** | ဢဝ်တူဝ်တႄႇသုတ်း | **လွတ်ႈၽေးသုတ်း (Safe)** |
| **Single** | Crash | Crash | တွၼ်ႈတႃႇၶေႃႈမုၼ်း ဢၼ်တေလႆႈမီးတူဝ်လဵဝ်ၵူၺ်း (ID) |
| **Last** | Crash | ဢဝ်တူဝ်လင်သုတ်း | ဢဝ်တူဝ်လင်သုတ်း ၼႂ်း List |

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Element Operator:** Method ဢၼ်သူင်ႇၶေႃႈမုၼ်းဢွၵ်ႇမႃးတူဝ်လဵဝ် (Single Item)။
* **Default Value:** ၵႃႈၶၼ်ပိုၼ်ႉထၢၼ် (မိူၼ်ၼင်ႇ null တႃႇ Object, 0 တႃႇ Int)။
* **Unique Identification:** ၵၢၼ်မၵ်းမၼ်ႈၶေႃႈမုၼ်း ဢၼ်ဢမ်ႇၸၢင်ႈမီးတူဝ်မိူၼ်။
* **Exception Handling:** ၵၢၼ်ႁပ်ႉပၼ်ႁႃ မိူဝ်ႈ Method တူဝ်ၼၼ်ႉ Crash (မိူၼ်ၼင်ႇ First() ၼႂ်း Empty List)။
