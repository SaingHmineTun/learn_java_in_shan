# Lesson 74: "Implicitly Typed Arrays and the 'new[]' Syntax"

မိူဝ်ႈႁဝ်းသၢင်ႈ Array ၼၼ်ႉ ႁဝ်းၵႆႉတႅမ်ႈ `int[] numbers = new int[] { 1, 2, 3 };`။ 
ၵူၺ်းၵႃႈ သင်ႁဝ်းၸႂ်ႉ **`var`** ဢမ်ႇၼၼ် **`new[]`** ၼႆ ႁဝ်းတေတႅမ်ႈပွတ်းလႆႈလိူဝ်ၼၼ်ႉၶႃႈ။

### 1. ၵၢၼ်ၸႂ်ႉ `var` တႃႇသၢင်ႈ Array
မိူဝ်ႈႁဝ်းၸႂ်ႉ `var`... Compiler တေတူၺ်းၶေႃႈမုၼ်းၼႂ်း `{ }` သေ ၼပ်ႉသၢင်ႈပၼ်ဝႃႈ မၼ်းလူဝ်ႇပဵၼ် `int[]` ႁႃႉ ဢမ်ႇၼၼ် `string[]` ၼႆၶႃႈ။

```csharp
// Compiler တေႁူႉၵမ်းလဵဝ်ဝႃႈ ၼႆႉပဵၼ် int[]
var scores = new[] { 90, 85, 77, 95 };

// Compiler တေႁူႉၵမ်းလဵဝ်ဝႃႈ ၼႆႉပဵၼ် string[]
var students = new[] { "Sai Mao", "Nang Kham" };
```



### 2. ပိူင်ၵၢၼ် (Rules) ဢၼ်လူဝ်ႇၾၢင်ႉ
* **လူဝ်ႇမီးၵႃႈၶၼ်တႄႇ (Initializer):** သင်ၸႂ်ႉ `new[]` ႁဝ်းတေလႆႈသႂ်ႇ `{ }` ၵမ်းလဵဝ်။ ႁဝ်းတႅမ်ႈ `var a = new[] [5];` ၼႆ **ဢမ်ႇလႆႈ** ၶႃႈ။
* **Type လူဝ်ႇမိူၼ်ၵၼ်:** ၶေႃႈမုၼ်းၼႂ်း `{ }` တေလႆႈပဵၼ် Type လဵဝ်ၵၼ်မူတ်းၶႃႈ။
    * `new[] { 1, "Sai" }` -> **Error!** (ယွၼ်ႉမၼ်းမီးဝႆႉ int လႄႈ string ဢၼ်ဢမ်ႇမိူၼ်ၵၼ် ႁူမ်ႈၵၼ်)။


### 3. ႁဵတ်းသင်ၸင်ႇၸႂ်ႉ?
* **Clean Code:** ႁဵတ်းႁႂ်ႈ Code ႁဝ်းဢၢၼ်ႇလူငၢႆႈ ဢမ်ႇယုင်ႈယၢင်ႈ။
* **Anonymous Types:** ၵႆႉၸႂ်ႉမိူဝ်ႈႁဝ်းသၢင်ႈ Array ဢၼ်သိမ်း Object ဢၼ်ဢမ်ႇမီးၸိုဝ်ႈ Class တႅတ်ႉတေႃး (တွၼ်ႈၼႆႉ တေႁဵၼ်းၼႂ်း LINQ ၶႃႈ)။

### 4. တူဝ်ယင်ႇၵၢၼ်ပႅၵ်ႇၵၼ်

| လၢႆးတႅမ်ႈ (Syntax Style) | တူဝ်ယင်ႇ (Example) |
| :--- | :--- |
| **Explicit (တႅမ်ႈတဵမ်)** | `int[] nums = new int[] { 1, 2, 3 };` |
| **Implicit (တႅမ်ႈပွတ်း)** | `var nums = new[] { 1, 2, 3 };` |
| **Shortest (ပွတ်းသေပိူၼ်ႈ)** | `int[] nums = { 1, 2, 3 };` |

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Implicitly Typed:** ၵၢၼ်ဢၼ် Compiler ၼပ်ႉသၢင်ႈပၼ် Type ႁင်းၵူၺ်း လုၵ်ႉတီႈၶေႃႈမုၼ်း။
* **var Keyword:** ၶေႃႈၵႂမ်းဢၼ်ၸႂ်ႉတႅၼ်းတၢင် Type။ ၵူၺ်းၵႃႈ လူဝ်ႇပၼ်ၵႃႈၶၼ်ဝႆႉၵမ်းလဵဝ်။
* **Type Inference:** ပိူင်သၢင်ႈၶွင် C# ဢၼ်ၸွႆးႁူႉဝႃႈ Variable ၼၼ်ႉ ပဵၼ် Type သင်။
* **Initializer List:** ၸုမ်းၶေႃႈမုၼ်းၼႂ်း `{ }` ဢၼ်ပၼ်ၵႃႈၶၼ်တႄႇ။
