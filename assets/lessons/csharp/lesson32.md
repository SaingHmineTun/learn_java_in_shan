# Lesson 32: "Bitwise Operators" (ၵၢၼ်ၸတ်းၵၢၼ်မၢႆ Binary)

Computer ၼႆႉမၼ်းႁူႉမၢႆ `0` လႄႈ `1` ၵူၺ်း။ 
မိူဝ်ႈႁဝ်းၸႂ်ႉ Bitwise Operators, C# တေပိၼ်ႇမၢႆၼပ်ႉ (Decimal) ႁႂ်ႈပဵၼ် Binary သေ ၸင်ႇၼပ်ႉသၢင်ႈၶႃႈ။

### 1. ၶိူင်ႈမၢႆ Bitwise 6 ပိူင်

| Operator | Name | Meaning | Example (Binary)           |
|:---------| :--- | :--- |:---------------------------|
| **`&`**  | **AND** | ပဵၼ် 1 ပေႃးတင်းသွင်ပဵၼ် 1 | `1 & 1 = 1`, `1 & 0 = 0`   |
| **`\|`** | **OR** | ပဵၼ် 1 ပေႃးမီး 1 ဢၼ်ၼိုင်ႈၵေႃႈလႆႈ | `1 \| 0 = 1`, `0 \| 0 = 0` |
| **`^`**  | **XOR** | ပဵၼ် 1 ပေႃးမၼ်း **"ပႅၵ်ႇၵၼ်"** | `1 ^ 0 = 1`, `1 ^ 1 = 0`   |
| **`~`**  | **NOT** | ပိၼ်ႇ 0 ပဵၼ် 1 / 1 ပဵၼ် 0 | `~1 = 0`, `~0 = 1`         |
| **`<<`** | **Left Shift** | ယွၼ်ႇ Bit ၵႂႃႇၽၢႆႇသၢႆႉ (ၼပ်ႉ 2 ပၼ်ႇ) | `5 << 1` (ပဵၼ် 10)         |
| **`>>`** | **Right Shift** | ယွၼ်ႇ Bit ၵႂႃႇၽၢႆႇၶႂႃ (ႁၢၼ် 2) | `10 >> 1` (ပဵၼ် 5)         |



### 2. တူဝ်ယင်ႇ Code
မိူဝ်ႈႁဝ်းတႅမ်ႈ Code, C# တေၸတ်းၵၢၼ်ပၼ်ႁင်းမၼ်းၶႃႈ:

```csharp
int a = 5;  // Binary: 0101
int b = 3;  // Binary: 0011

int andResult = a & b; // 0101 & 0011 = 0001 (Decimal: 1)
int orResult  = a | b; // 0101 | 0011 = 0111 (Decimal: 7)
int xorResult = a ^ b; // 0101 ^ 0011 = 0110 (Decimal: 6)

Console.WriteLine(andResult); // Output: 1
```

### 3. မိူဝ်ႈလႂ်လူဝ်ႇၸႂ်ႉ Bitwise?
* **Hardware Interfacing:** တႃႇတႅမ်ႈ Code ၸတ်းၵၢၼ်ၶိူင်ႈ Hardware (မိူၼ်ၼင်ႇ Arduino/IoT)။
* **Optimization:** တႃႇႁႂ်ႈ App ႁဵတ်းၵၢၼ်ၽႂ်း (ယွၼ်ႉ Bitwise ၼႆႉ ၽႂ်းသေ Arithmetic)။
* **Flags:** တႃႇသိမ်းၶေႃႈမုၼ်းလၢႆလၢႆယၢင်ႇ ဝႆႉၼႂ်း Variable ဢၼ်လဵဝ် (Bit Flags)။

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Bit:** ႁူဝ်ၼပ်ႉၶေႃႈမုၼ်းဢၼ်လဵၵ်းသေပိူၼ်ႈ (0 ဢမ်ႇၼၼ် 1)။
* **Binary:** ပိူင်မၢႆၼပ်ႉ ပိုၼ်ႉထၢၼ် 2 (Base-2)။
* **XOR (Exclusive OR):** ၶိူင်ႈမၢႆဢၼ်တေပဵၼ် 1 မိူဝ်ႈ Bits တင်းသွင်ၽၢႆႇ ဢမ်ႇမိူၼ်ၵၼ်။
* **Left Shift (<<):** ၵၢၼ်တႅၼ်းတီႈ Bits ၵႂႃႇၽၢႆႇသၢႆႉ၊ မၼ်းမိူၼ်တင်းၵၢၼ်ၼပ်ႉ 2 (Multiply by 2)။
* **Right Shift (>>):** ၵၢၼ်တႅၼ်းတီႈ Bits ၵႂႃႇၽၢႆႇၶႂႃ၊ မၼ်းမိူၼ်တင်းၵၢၼ်ႁၢၼ် 2 (Divide by 2)။
