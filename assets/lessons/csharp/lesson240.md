# Lesson 240: "Introduction to Asynchrony: Synchronous vs. Asynchronous"

ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းသင်ႇၵိၼ်ၶဝ်ႈသွႆး (Noodles) ၼႂ်းသႅင်ႇၶႃႈ။

### 1. Synchronous (ႁဵတ်းၵမ်းထႅဝ်ထႅဝ်)
မိူဝ်ႈႁဝ်းသင်ႇၶဝ်ႈယဝ်ႉၼၼ်ႉ ႁဝ်းတေလႆႈ **ၸုၵ်းပႂ်ႉ** ယူႇတီႈၼႃႈၵွင်ႇတႃႇၼၼ်ႉ တေႃႇပေႃးလႆႈၶဝ်ႈသွႆး။ ႁဝ်းဢမ်ႇၸၢင်ႈၵႂႃႇႁဵတ်းတၢင်ႇပိူင်၊ ဢမ်ႇၸၢင်ႈလဵၼ်ႈၾူၼ်း၊ ဢမ်ႇၸၢင်ႈလၢတ်ႈၵႂၢမ်းတင်းၽႂ်။ ၼႆႉႁွင်ႉဝႃႈ **Blocking** ၶႃႈ။
* **ၼႂ်း Code:** App တေပႂ်ႉႁႂ်ႈ Line 1 ႁဵတ်းယဝ်ႉၵွၼ်ႇ ၸင်ႇတေသိုပ်ႇႁဵတ်း Line 2 ၶႃႈ။

### 2. Asynchronous (ႁဵတ်းၵၢၼ်လၢႆလၢႆဢၼ်ၸွမ်ၵၼ်)
မိူဝ်ႈႁဝ်းသင်ႇယဝ်ႉၼၼ်ႉ ႁဝ်းလႆႈ "မၢႆထႅဝ်" (Ticket) မႃးဢၼ်ၼိုင်ႈ။ ႁဝ်းၵႂႃႇၼင်ႈလူလိၵ်ႈ၊ တွပ်ႇ Message ယူႇၸူဝ်ႈၶၢဝ်ႇ။ မိူဝ်ႈၶဝ်ႈသွႆးယဝ်ႉယဝ်ႉ ၵူၼ်းၶၢႆၸင်ႇႁွင်ႉႁဝ်းၵႂႃႇဢဝ်။ ႁဝ်းဢမ်ႇလူဝ်ႇၸုၵ်းပႂ်ႉလၢႆလၢႆၶႃႈ။ ၼႆႉႁွင်ႉဝႃႈ **Non-blocking** ၶႃႈ။
* **ၼႂ်း Code:** App တေသင်ႇႁႂ်ႈၵၢၼ်ဢၼ်ယၢဝ်းၼၼ်ႉ (မိူၼ်ၼင်ႇလူ File) ႁဵတ်းၵႂႃႇ ၽၢႆႇလင် (Background) သေ App တေသိုပ်ႇႁဵတ်းၵၢၼ်တၢင်ႇပိူင် (မိူၼ်ၼင်ႇႁပ်ႉ User Input) ပၼ်ၵမ်းလဵဝ်ၶႃႈ။



### 3. ႁဵတ်းသင်ႁဝ်းၸင်ႇလူဝ်ႇ Async?
1.  **Responsiveness:** တႃႇႁႂ်ႈ App ႁဝ်း (UI) တိုၵ်ႉၼဵၵ်းလႆႈ၊ ပၼ်ႇလႆႈ မိူဝ်ၽွင်း မၼ်းတိုၵ်ႉလူ Data ယႂ်ႇယႂ်ႇယူႇ။
2.  **Efficiency:** တႃႇၸႂ်ႉ CPU ႁႂ်ႈၵုမ်ႇထူၼ်ႈ။ တႅၼ်းဢၼ်တေပႂ်ႉလၢႆလၢႆ ႁႂ်ႈမၼ်းၵႂႃႇႁဵတ်းၵၢၼ်တၢင်ႇပိူင်ၸူဝ်ႈၶၢဝ်း။
3.  **User Experience:** User တေဢမ်ႇထၢင်ႇဝႃႈ App ႁဝ်း "လူႉ" (Hang/Freeze) ၵႂႃႇၶႃႈ။

### 4. ယၢမ်းလႂ်ထုၵ်ႇလီၸႂ်ႉ Async?
ႁဝ်းၵႆႉၸႂ်ႉမိူဝ်ႈ App ႁဝ်းလူဝ်ႇႁဵတ်းၵၢၼ်တင်း "ၽၢႆႇၼွၵ်ႈ" (I/O Bound) ဢၼ်ႁဝ်းမၵ်းမၼ်ႈ ၶၢဝ်းယၢမ်းမၼ်းဢမ်ႇလႆႈ:
* ၵၢၼ်လူ/တႅမ်ႈ File ဢၼ်ယႂ်ႇ (System.IO)။
* ၵၢၼ်သူင်ႇ/ႁပ်ႉ Data ၼိူဝ် Internet (API/Database)။
* ၵၢၼ်ၼပ်ႉသွၼ်ႇဢၼ်ယုင်ႈယၢင်ႈၼႃႇ (CPU Bound)။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Synchronous:** ၵၢၼ်ႁဵတ်းၸွမ်းလၢႆး ဢၼ်တေလႆႈပႂ်ႉ ႁႂ်ႈဢၼ်ၼိုင်ႈယဝ်ႉ ၸင်ႇတေသိုပ်ႇထႅင်ႈဢၼ်ၼိုင်ႈ။
* **Asynchronous:** ၵၢၼ်ႁဵတ်းၵၢၼ်လၢႆဢၼ် ဢၼ်ဢမ်ႇလူဝ်ႇပႂ်ႉၵၼ်။ ႁဵတ်းၸွမ်းၵၼ်လႆႈ ၼႂ်းၶၢဝ်းယၢမ်းလဵဝ်ၵၼ်။
* **Blocking:** ၵၢၼ်ဢၼ် Code ၵိုတ်းယတ်းဝႆႉ တႃႇပႂ်ႉၽွၼ်းလႆႈ။
* **Non-blocking:** ၵၢၼ်ဢၼ် Code သိုပ်ႇလႅၼ်ႈၵႂႃႇလႆႈ တႅၼ်းၵၢၼ်ပႂ်ႉ။
