# Lesson 297: "The .NET MAUI Shell"

**Shell** မိူၼ်တင်း "လုပ်ႇ" (Skeleton) ၶွင် App ၶႃႈ။ မၼ်းမၵ်းမၼ်ႈဝႃႈ App ႁဝ်းတေမီး Menu ၸိူင်ႉႁိုဝ် လႄႈ တေၵႂႃႇၼႃႈလႂ်လႆႈၽွင်ႈၶႃႈ။

### 1. Flyout Menu
**Flyout** ပဵၼ် Menu ဢၼ်လႅၼ်ႈဢွၵ်ႇမႃးၽၢႆႇသၢႆႉ မိူဝ်ႈႁဝ်းၼဵၵ်းတုမ်ႇ "3 ထႅဝ်" (Hamburger Icon) ၼၼ်ႉၶႃႈ။
* ႁဝ်းၸၢင်ႈသႂ်ႇ Header ႁၢင်ႈလီလီ (မိူၼ်ၼင်ႇ ႁၢင်ႈ Logo ႁူင်းႁဵၼ်း)။
* ႁဝ်းၸၢင်ႈမၵ်းမၼ်ႈ သဵၼ်ႈမၢႆၼႃႈလိၵ်ႈ (MenuItems) ၼႂ်း XAML လႆႈၵမ်းလဵဝ်။



### 2. TabBar (Menu ၽၢႆႇတႂ်ႈ)
သင်ႁဝ်းၶႂ်ႈႁဵတ်း App မိူၼ် Facebook ဢမ်ႇၼၼ် YouTube ဢၼ်မီး Menu ယူႇၽၢႆႇတႂ်ႈၼႆၸိုင် Shell ၸွႆးႁဝ်းလႆႈငၢႆႈၼႃႇၶႃႈ။
* ႁဝ်းၵူၺ်းလူဝ်ႇၸႂ်ႉ Tag `<Tab>` သေမၵ်းမၼ်ႈ Icon လႄႈ ၼႃႈလိၵ်ႈၶႃႈ။



### 3. Route-based Navigation (ၵၢၼ်ၵႂႃႇၼႃႈလိၵ်ႈလူၺ်ႈ URL)
Shell ၸႂ်ႉပိူင်သၢင်ႈ Navigation ဢၼ်မိူၼ်တင်း Website (Module 23) ၶႃႈ။ ႁဝ်းၸၢင်ႈၸႂ်ႉ "သဵၼ်ႈတၢင်း" (Routes) တႃႇၵႂႃႇၼႃႈလိၵ်ႈတၢင်ႇဢၼ်:
* **လၢႆးတႅမ်ႈ Code:**
  ```csharp
  // ၵႂႃႇၸူးၼႃႈ Details ၸွမ်တင်းသူင်ႇ ID ၵႂႃႇၸွမ်
  await Shell.Current.GoToAsync($"studentdetails?id=101");
  ```

---

### 4. ႁဵတ်းသင်ႁဝ်းၸင်ႇၸႂ်ႉ Shell?
1. **Simplified Code:** ႁဝ်းဢမ်ႇလူဝ်ႇတႅမ်ႈ Code ၸတ်းၵၢၼ် Menu ၼမ်လိူဝ်ပူၼ်ႉတီႈ။
2. **Visual Hierarchy:** ၸွႆးႁႂ်ႈ User ႁူႉဝႃႈ ယၢမ်းလဵဝ်ၶဝ်ယူႇတီႈလႂ် ၼႂ်း App။
3. **Common UI Patterns:** မၼ်းပႃးပိူင်သၢင်ႈ Navigation ဢၼ်ပဵၼ် Standard ၶွင် Android လႄႈ iOS ဝႆႉယဝ်ႉၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Flyout:** Menu ဢၼ်သိမ်းဝႆႉၽၢႆႇၶၢင်ႈသေ တေပိုတ်ႇဢွၵ်ႇမႃးမိူဝ်ႈၼဵၵ်းတုမ်ႇ။
* **TabBar:** ၸုမ်း Menu ဢၼ်ယူႇၽၢႆႇတႂ်ႈ ၼႃႈၸေႃး။
* **Route:** ၸိုဝ်ႈ "သဵၼ်ႈတၢင်း" ဢၼ်မၵ်းမၼ်ႈဝႃႈ ၼႃႈလိၵ်ႈၼၼ်ႉၸိုဝ်ႈသင် (မိူၼ်ၼင်ႇ `"//mainpage"`)။
* **Backstack:** ပိူင်မၢႆတွင်းဝႃႈ User ၵႂႃႇၼႃႈလိၵ်ႈလႂ်မႃးၽွင်ႈ (တွၼ်ႈတႃႇၼဵၵ်း "Back" သေဝၢႆႇၶိုၼ်း)။
