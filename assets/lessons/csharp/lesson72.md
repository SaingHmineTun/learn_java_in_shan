# Lesson 72: Jagged Arrays (Array ဢၼ်တႅဝ်းမၼ်းဢမ်ႇၽဵင်ႇၵၼ်)

ၼႂ်း **Lesson 72** ၼႆႉ ႁဝ်းတေမႃးတူၺ်း **Jagged Array** ဢေႃႈ။ 
သင်ဝႃႈ 2D Array (Rectangular) ၼၼ်ႉ မီးထႅဝ် လႄႈ တႅဝ်း ဢၼ်ၽဵင်ႇၵၼ်ပဵၼ်သီႇၸဵင်ႇၼႆ... 
Jagged Array သမ်ႉပဵၼ် **"Array ၼႂ်း Array"** ဢၼ်ၵူႈထႅဝ် မီးႁူဝ်ၼပ်ႉတႅဝ်းဢမ်ႇမိူၼ်ၵၼ်လႆႈၶႃႈ။

### 1. ပိူင်သၢင်ႈ (Syntax)
ႁဝ်းၸႂ်ႉမၢႆ **`[][]`** (သွင်ၵူပ်ႉ) တႃႇလၢတ်ႈဝႃႈ ၼႆႉပဵၼ် Array ၶွင် Array ၶႃႈ။



```csharp
// 1. ပိုၼ်ၽၢဝ်ႇ Jagged Array ဢၼ်မီး 3 ထႅဝ်
int[][] jagged = new int[3][];

// 2. မၵ်းမၼ်ႈႁူဝ်ၼပ်ႉတႅဝ်း ၼႂ်းၵူႈထႅဝ် (ဢမ်ႇတူဝ်ႈမိူၼ်ၵၼ်)
jagged[0] = new int[2]; // ထႅဝ် 0 မီး 2 တႅဝ်း
jagged[1] = new int[4]; // ထႅဝ် 1 မီး 4 တႅဝ်း
jagged[2] = new int[3]; // ထႅဝ် 2 မီး 3 တႅဝ်း
```

### 2. ၵၢၼ်ပၼ်ၵႃႈၶၼ်ၵမ်းလဵဝ်
```csharp
int[][] scores = {
    new int[] { 90, 80 },
    new int[] { 70, 85, 95, 100 },
    new int[] { 60, 75, 80 }
};
```

### 3. ၵၢၼ်ဢဝ်ၶေႃႈမုၼ်းဢွၵ်ႇမႃးၸႂ်ႉ (Accessing)
လၢႆးၸႂ်ႉမၼ်းတေမိူၼ်တင်း 2D Array ယူႇၶႃႈ၊ ၵူၺ်းၵႃႈ တေလႆႈသႂ်ႇၵူပ်ႉ `[]` ၸႅၵ်ႇၵၼ်။

```csharp
Console.WriteLine(scores[1][2]); // ၽွၼ်းလႆႈ: 95 (ထႅဝ် 1, တႅဝ်း 2)
```

### 4. '2D Array' vs 'Jagged Array' ပႅၵ်ႇၵၼ်ၼင်ႇႁိုဝ်?

| ပိူင် (Feature) | 2D Array (Rectangular) | Jagged Array                                |
| :--- | :--- |:--------------------------------------------|
| **Syntax** | `int[,]` | `int[][]`                                   |
| **ႁၢင်ႈၾၢင်** | ၵူႈထႅဝ်မီးတႅဝ်းၽဵင်ႇၵၼ် (သီႇၸဵင်ႇ) | ၵူႈထႅဝ်မီးတႅဝ်းဢမ်ႇၽဵင်ႇၵၼ်                 |
| **Memory** | သိမ်းဝႆႉပဵၼ်ၵွၼ်ႇလဵဝ် (Contiguous) | သိမ်းဝႆႉလၢႆလၢႆၵွၼ်ႇ (Array ၼႂ်း Array)      |
| **Performance** | ၽႂ်းလိူဝ်ဢိတ်းၼိုင်ႈ | ထိူင်းလိူဝ်ဢိတ်းၼိုင်ႈ (ယွၼ်ႉလူဝ်ႇႁႃလၢႆတီႈ) |

### 5. မိူဝ်ႈလႂ်လူဝ်ႇၸႂ်ႉ?
ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းမီးလုၵ်ႈႁဵၼ်း 3 ႁွင်ႈ၊ ၵူၺ်းၵႃႈ ၵူႈႁွင်ႈမီးႁူဝ်ၼပ်ႉလုၵ်ႈႁဵၼ်းဢမ်ႇမိူၼ်ၵၼ်။ 
ၼႆႉယဝ်ႉ ႁဝ်းၸင်ႇလူဝ်ႇၸႂ်ႉ Jagged Array ၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Jagged Array:** Array ဢၼ်သိမ်း Array တူဝ်တၢင်ႇၸိူဝ်းဝႆႉၼႂ်းမၼ်း။
* **Array of Arrays:** ၸိုဝ်ႈႁွင်ႉထႅင်ႈပိူင်ၼိုင်ႈၶွင် Jagged Array။
* **Dimension:** ၼႃႈ ဢမ်ႇၼၼ် ၸၼ်ႉၶွင် Array။
* **Sub-array:** Array ဢွၼ်ႇဢၼ်ယူႇၼႂ်း Array လူင်ထႅင်ႈၸၼ်ႉၼိုင်ႈ။
