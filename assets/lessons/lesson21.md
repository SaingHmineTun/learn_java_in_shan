
## **Java Basic 21**

#### **LOOPING**

Link - <https://youtu.be/eoQ9gHLANVE>

Looping ၼႆႉ ပွင်ႇဝႃႈ ႁဝ်းႁဵတ်ႉၵၢၼ် သေဢၼ်ဢၼ်ၼၼ်ႉ ႁဵတ်ႉပၼ်ႇတူင်ႇ ႁဵတ်ႇဢမ်ႉၵိုတ်းဢမ်ႇလိုဝ်ႈ

မိူဝ်ႈၽွင်းမၼ်းမႅၼ်ႈၸႂ် တေႃႇထိုင် မၼ်းပေႃးဢမ်ႇမႅၼ်ႈၸႂ် ၼႆယဝ်ႉ။

တီႈ Java ၼႆႉတေႉ တႃႇတေၸႂ်ႉတိုဝ်း looping ၼႆ မၼ်းတေမီးဝႆႉ 4 လၢႆးလႄႈ ပဵၼ်

- _while loop_
- _do while loop_
- _for loop_
- _enhanced for loop_

ၼႆယဝ်ႉ။ while, do while လႄႈ for ၼႆႉတေႉ ႁဝ်းၸႂ်ႉၸွမ်း variable သေ enhanced for ၼႆႉသမ်ႉ ႁဝ်းၸႂ်ႉၸွမ်း array လႄႈ collection ၵူၺ်း။
