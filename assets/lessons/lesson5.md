**Floating data types**

Float လႄႈ Double ၼႆႉတေႉ ပဵၼ်တႃႇ ႁဝ်းတႅမ်ႈ တူဝ်ၼပ်ႉ ဢၼ်ပႃးဝႆႉ decimal point ယဝ်ႉ။ ႁဝ်းၶိုၼ်း ႁွင်ႉမၼ်းဝႃႈ Floating data type ၼႆၵေႃႈ လႆႈယူႇ။ တၢင်းသွင်ဢၼ်ၼႆႉၵေႃႈ ပိူင်ႈၵၼ်တီႈ တၢင်းယႂ်ႇ (capacity) မၼ်းၼင်ႇၵဝ်ႇ။ float တေ သိမ်းပၼ်လႆႈ 4 bytes သေ double သမ်ႉ တေသိမ်းပၼ်လႆႈ 8 bytes ယဝ်ႉ။

လၢႆးၸႂ်ႉတိုဝ်း

```
float floatVar = 2_147_483_647.0F; // float သိမ်းလႆႈ 4 bytes
double doubleValue = 29345.1935; // double သိမ်းလႆႈ 8 bytes
```
သင်ႁဝ်းၶႆႈတႅမ်ႈ float ၼႆၸိုင် တေလႆႈသႂ်ႇပၼ် F ဢမ်ႇၼၼ် f ၸင်ႇႁဵတ်းၵၢၼ်။ သင်ဢမ်ႇၼၼ် မၼ်းတေၼႄဝႆႉ ၽိတ်းၼႆယူႇ။

**Character data type**

တီႈ Java ၼႆႉ ႁဝ်းၸႂ်ႉတိုဝ်း char တႃႇတေသိမ်းဝႆႉ တူဝ်လိၵ်ႈယဝ်ႉ။ ၵူၺ်းၵႃႊ char ၼႆႉ မၼ်းတေသိမ်းဝႆႉ ပၼ်လႆႈ တူဝ်လိၵ်ႈ တူဝ်လဵဝ်ၵူၺ်း။ ပေႃးတေၶႆႈၸႂ်ႉတိုဝ်း char ၼႆ ႁဝ်းတေလႆႈၸႂ်ႉ single quote ('ၵ') ယဝ်ႉ။ char ၼႆႉ မၼ်းႁပ်ႉၶၢမ်ႇလႆႈ int ဢမ်ႇၼၼ် hexadecimal သေတႃႉ တီႈတေႉ မၼ်းၵူၺ်းၼႄပၼ် ၵႃႊၶၼ် ယူႇၼီႇၶူတ်ႉ ဢၼ်မီးဝႆႉ တီႈမၢႆၼၼ်ႉၵူၺ်း။ char ၼႆႉ မၼ်းသိမ်းပၼ်လႆႈ 2 bytes လႄႈ ၵွပ်ႈတီႈယူႇၼီႇၶူတ်ႉၼၼ်ႉ ၶဝ်ဢမ်ႇၸႂ်ႉ Negative value ၼႆလႄႈ တၢင်းယႂ်ႇမၼ်း ၸင်ႇပဵၼ်ၵႂႃႇ 65535 ယဝ်ႉ။

လၢႆးၸႂ်ႉတိုဝ်း


```
// Character သိမ်းလႆႈ 2 bytes
char ka = 4213; // သိမ်းလူၺ်ႈ decimal
ka = 'ၵ'; // သိမ်းလူၺ်ႈ character
ka = 0x1075; // သိမ်းလူၺ်ႈ hexadecimal
ka = '\\u1075'; // သိမ်းလူၺ်ႈ unicode character
```

**Boolean data type**

Boolean ၼႆႉသမ်ႉ မၼ်းတေသိမ်းပၼ်လႆႈ ၵႃႊၶၼ် 2 ဢၼ်ၵူၺ်း။ မၼ်းပဵၼ် true လႄႈ false ယဝ်ႉ။ ၵူၺ်းၵႃႊ မၼ်းၵေႃႈပဵၼ် data type ဢၼ်လမ်ႇလွင်ႈတေႉတေႉ။ မိူၼ်ၼင်ႇ သင်ၸိူဝ်ႉ ႁဝ်းၶႆႈတႅၵ်ႈၼိူင်း ၵႃႊၶၼ်သွင်ဢၼ် မိူၼ်ၸိူင်ႉ ( 5 > 10 ) ၼႆၸိုင် ၵွပ်ႈ 5 ၼႆႉ ဢမ်ႇယႂ်ႇသေ 10 လႄႈ မၼ်းပဵၼ် false ယဝ်ႉ။

လၢႆးၸႂ်ႉတိုဝ်း

```
boolean bool = true;
boolean bool2 = false;
```
