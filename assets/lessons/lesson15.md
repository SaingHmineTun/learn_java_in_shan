
## **Java Basic 15**

#### **CREATE ARRAY FROM STRING**

Link - <https://youtu.be/f3KQ_-QO4iA>

တီႈတေႉ ၼႂ်း String ၼႆႉ မၼ်းမီး method တၢင်းၼမ်သေ ၼႂ်းၼၼ်ႉ မၼ်းၵေႃႈ မီးဝႆႉထႅင်ႈ method ၵမ်ႈၽွင်ႈ တႃႇဢဝ် String ၼႆႉပိၼ်ႇပဵၼ် Array ယဝ်ႉ။

**ပိၼ်ႇပဵၼ် char array**

သင်ႁဝ်း မီးဝႆႉ String ၼိုင်ႈဢၼ်ၼႆ ပေႃးႁွင်ႉၸႂ်ႉ toCharArray() ၼႆၸိုင် မၼ်းတေဢဝ် String သေပိၼ်ႇပဵၼ် char\[\] ပၼ်ယဝ်ႉ။

###### လၢႆးၸႂ်ႉတိုဝ်း

```
String name = "Sai";
char[] chars = name.toCharArray(); // {'S', 'a', 'i'}
```

**ပိၼ်ႇပဵၼ် String array**

ၼႆႉတေႉ ႁဝ်းတေလႆႈ ႁွင်ႉၸႂ်ႉ split() method လႄႈ တေလႆႈသူင်ႇပၼ် parameter မၼ်းပၼ်ဢိူဝ်း။ ဢိင်ၼိူဝ် parameter ဢၼ်ႁဝ်းသူင်ႇပၼ်ၼၼ်ႉသေ မၼ်းၸင်ႇၸႅၵ်ႇ String ႁႂ်ႈပဵၼ် String Array ယဝ်ႉ။ parameter ၼႆႉပဵၼ် ဢၼ်ႁဝ်းသႂ်ႇၵႃႊၶၼ်ပၼ် တီႈၼႂ်း တူင်ႇမူၼ်း ဢၼ်မီးဝႆႉတီႈ ၽၢႆႇလင် method ဢိူဝ်ႈ

###### လၢႆးၸႂ်ႉတိုဝ်း

```
String nakama = "Luffy, Zoro, Sanji";
String[] nakamas = nakama.split(","); // {"Luffy", "Zoro", "Sanji"}
```

**ၸွပ်ႇၼႂ်း Array မီးဝႆႉလၢႆဢၼ်**

ၼႂ်း array ႁဝ်းၼႆႉ မီးဝႆႉ ၵႃႊၶၼ်လၢႆဢၼ်ၼႆၸိုင် ၸႂ်ႉတိုဝ်း length variable သေ ၸွပ်ႇလႆႈဢိူဝ်ႈ။

###### လၢႆးၸႂ်ႉတိုဝ်း

```
int length = chars.length; // 3
```

**ဢွၵ်ႇၼႄ ၵႃႊၶၼ်လိုၼ်းသုတ်ႉ ဢၼ်မီးဝႆႉတီႈၼႂ်း Array**

တႃႇၼႄပၼ် ၵႃႊၶၼ်ဢွၼ်တၢင်းသုတ်ႉ ဢၼ်မီးဝႆႉ တီႈၼႂ်း array ၼႆၸိုင် ႁဝ်းၸႂ်ႉလႆႈ index zero သေတႃႉ တႃႇၼႄပၼ် ၵႃႊၶၼ်လိုၼ်းသုတ်ႉမၼ်းတေႉ ႁဝ်းတေလႆႈႁူႉဝႃႈ ၼႂ်း array ႁဝ်းၼႆႉ မီးဝႆႉ လၢႆဢၼ်ၼႆယဝ်ႉ။ လၢႆးငၢႆႊမၼ်းတေႉ ႁဝ်းႁွင်ႉၸႂ်ႉ length သေ ထွၼ်ပၼ် 1 ယဝ်ႉ။ ၵွပ်ႈပိူဝ်ႈ indexing ၼႆႉ တေႇတီႈ 0 ၼႆလႄႈ ပေႃးဢမ်ႇထွၼ်ပၼ် 1 ၼႆၸိုင် တေပဵၼ် Error ယဝ်ႉ။ ၵွပ်ႈဝႃး ၼႂ်း Array ၼႆႉ သင်မီးဝႆႉ ၵႃႊၶၼ် 5 ဢၼ်ၼႆၸိုင် ၵႃႊၶၼ်လိုၼ်းသုတ်ႉမၼ်းၼႆႉ ႁဝ်းတေလႆႈႁွင်ႉၸႂ်ႉ \[4\] ဢမ်ႇၸႂ် \[5\] ။

###### လၢႆးၸႂ်ႉတိုဝ်း

```
char lastChar = chars[length - 1];
```
