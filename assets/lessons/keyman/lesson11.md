## Lesson 11: The any() and index() Power Combo

မိူဝ်ႈႁဝ်းတႅမ်ႈ Keyboard တႆးၼၼ်ႉ ႁဝ်းမီးတူဝ်လိၵ်ႈတင်းၼမ်။ 
သင်ႁဝ်းတႅမ်ႈ Rule ၼိုင်ႈတူဝ် ၼိုင်ႈထႅဝ် Code ႁဝ်းတေယၢဝ်းၼမ်။ 
ၵၢၼ်ၸႂ်ႉ `any()` လႄႈ `index()` တေၸွႆးႁႂ်ႈႁဝ်းတႅမ်ႈ Rule ၼိုင်ႈထႅဝ်ၵူၺ်းၵေႃႈ ၸတ်းၵၢၼ်လႆႈတူဝ်လိၵ်ႈပဵၼ်သိပ်းတူဝ်ယဝ်ႉ။

### 1. ၵၢၼ်ႁဵတ်းၵၢၼ်ၶွင် `any()`
`any()` ပဵၼ်ၶေႃႈသင်ႇတႃႇထတ်းတူၺ်းဝႃႈ "မီးတူဝ်လႂ်တူဝ်ၼိုင်ႈၼႂ်း store ႁႃႉ?"
* မၼ်းတေတူၺ်း တူဝ်လိၵ်ႈတင်းမူတ်း ဢၼ်ႁဝ်းသႂ်ႇဝႆႉၼႂ်း `store()` ယဝ်ႉ။

### 2. ၵၢၼ်ႁဵတ်းၵၢၼ်ၶွင် `index()`
`index()` ပဵၼ်ၶေႃႈသင်ႇတႃႇဢဝ်တူဝ်လိၵ်ႈဢွၵ်ႇမႃး ၸွမ်းၼင်ႇ Index ဢၼ်မႅၼ်ႈၵၼ်တင်းတူဝ်ဢၼ်ႁဝ်းၼဵၵ်ႉၼၼ်ႉယဝ်ႉ။



### 3. ၵၢၼ်တႅမ်ႈ Code (The Power Combo)
ႁဝ်းတေၸၢင်ႈဢဝ် `any()` လႄႈ `index()` မႃးၸႂ်ႉၵိုၵ်းၵၼ် တွၼ်ႈတႃႇမႄး Unicode ႁႂ်ႈမၼ်းပဵၼ်ပိူင် (Normalization/Reordering) လႆႈငၢႆႈငၢႆႈၼင်ႇၼႆၶႃႈ:

```keyman
c --- 1. တင်ႈၵႃႈ Store ---
store(vowels)   U+1083 U+1084 U+107D c ု ူ ွ
store(tones)    U+1087 U+1088 U+1089 c ႇ ႈ ႉ

c --- 2. Smart Reordering Rule ---
c သင်ၼဵၵ်ႉ Tone ဢွၼ်ၼႃႈ Vowel ႁႂ်ႈမၼ်း Swap (လႅၵ်ႈလၢႆႈ) ပၼ်ၵမ်းလဵဝ်
any(tones) + any(vowels) > index(vowels, 2) index(tones, 1)
```

**လွင်ႈႁဵတ်းၵၢၼ်မၼ်း:**
1.  Keyboard တူၺ်း Context ဝႃႈ မီး `any(tones)` ဝႆႉယူႇႁႃႉ။
2.  သင်ႁဝ်းၼဵၵ်ႉ `any(vowels)` ထႅင်ႈ။
3.  မၼ်းတေဢဝ်တူဝ် `vowels` (တူဝ်ထူၼ်ႈ 2 ၼႂ်း Rule) မႃးဝႆႉၽၢႆႇၼႃႈ။
4.  ယဝ်ႉၵေႃႈ ဢဝ်တူဝ် `tones` (တူဝ်ထူၼ်ႈ 1 ၼႂ်း Rule) မႃးဝႆႉၽၢႆႇလႅင်ယဝ်ႉ။

### 4. လွင်ႈလီၶွင်ၵၢၼ်ၸႂ်ႉ Power Combo
* **Reduced Code:** ၸဝ်ႈၵဝ်ႇဢမ်ႇလူဝ်ႇတႅမ်ႈ Rule ယၢဝ်းယႅၼ်းတႃႇ Vowels လႄႈ Tones ၵူႈတူဝ်။
* **Precision:** မၼ်းၸွႆးႁႂ်ႈ Unicode ဢွၵ်ႇမႃးမႅၼ်ႈၸွမ်းပိူင် Logical Order တႃႇသေႇ။
* **Scalability:** သင်ၸဝ်ႈၵဝ်ႇၶႂ်ႈသႂ်ႇတူဝ်လိၵ်ႈထႅင်ႈၼႆ ၵႂႃႇမႄးတီႈ `store()` တီႈလဵဝ်ၵေႃႈယဝ်ႉယဝ်ႉ။

### Summary (တွၼ်ႈၵႅပ်ႈ Lesson 11)

| IT Terms | ၶေႃႈၵႂၢမ်းတႆး / Definition |
| :--- | :--- |
| **any()** | ၶေႃႈသင်ႇထတ်းတူၺ်း တူဝ်လိၵ်ႈၼႂ်း Array (Pattern Matching) |
| **index()** | ၶေႃႈသင်ႇဢဝ် Output ဢွၵ်ႇမႃး ၸွမ်းၼင်ႇလၢႆးဝၢင်း (Index) |
| **Pattern Matching** | ၵၢၼ်ထတ်းတူၺ်း ႁၢင်ႈၽၢင်တူဝ်လိၵ်ႈ ဢၼ်မႅၼ်ႈၸွမ်းပိူင် |
| **Dynamic Swap** | ၵၢၼ်လႅၵ်ႈလၢႆႈတူဝ်လိၵ်ႈၵမ်းလဵဝ် လူၺ်ႈၸႂ်ႉ Logic |

**Summary:** ၼႂ်းတွၼ်ႈသွၼ်ၼႆႉ ႁဝ်းလႆႈႁဵၼ်းႁူႉလွင်ႈၵၢၼ်ၸႂ်ႉ `any()` လႄႈ `index()` ၵိုၵ်းၵၼ် တႃႇႁဵတ်းႁႂ်ႈ Keyboard **TMK Shan** ႁဝ်းမီး "ဢွၵ်းဢေႃ" တႅတ်ႈတေႃးယဝ်ႉ။ 
ၵၢၼ်ၸႂ်ႉ Power Combo ၼႆႉ တေႁဵတ်းႁႂ်ႈႁဝ်းၸတ်းၵၢၼ် Unicode Reordering လႆႈငၢႆႈ လႄႈ Code ႁဝ်းသႅၼ်ႈလူင်းၼမ်ယဝ်ႉ။
