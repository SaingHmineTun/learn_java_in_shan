## Lesson 16: Smart Backspacing

**Smart Backspacing** ပဵၼ်ၵၢၼ်ၸတ်းၵၢၼ် Key Backspace ႁႂ်ႈမၼ်းလူတ်ႈဢွၵ်ႇ (Delete) တူဝ်လိၵ်ႈၸွမ်းၼင်ႇ "ပိူင်ႁၢင်ႈ" (Logic) ဢၼ်ႁဝ်းတင်ႈဝႆႉ။ 
တႅၼ်းတူဝ်ၵၢၼ်လူတ်ႈဢွၵ်ႇပႅတ်ႈ ၼိုင်ႈ Codepoint တေႃႇၼိုင်ႈပွၵ်ႈ ၵူၺ်းၼၼ်ႉယဝ်ႉ။

### 1. ပၼ်ႁႃၶွင် Backspace
ၼႂ်းလိၵ်ႈတႆးၼႆႉ တူဝ်လိၵ်ႈမၢင်တူဝ် မိူၼ်ၼင်ႇ **"ၵေႃ"** မၼ်းမီး Unicode ၼမ် (ၵ + ေ + ႃ)။ သင်ႁဝ်းၼဵၵ်ႉ Backspace ၼႆ:
* ၼဵၵ်ႉပွၵ်ႈ 1: လူတ်ႈ `ႃ` (ၵိုတ်း `ၵေ`)
* ၼဵၵ်ႉပွၵ်ႈ 2: လုမ်း `ေ` (ၵိုတ်း `ၵ`)
* **Smart Logic:** ႁဝ်းၸၢင်ႈသင်ႇႁႂ်ႈမၼ်းလုမ်း `ေႃ` (Vowel Complex) ပႅတ်ႈၵမ်းလဵဝ် ႁႂ်ႈၵူၼ်းၸႂ်ႉဢမ်ႇလူဝ်ႇၼဵၵ်ႉလၢႆပွၵ်ႈယဝ်ႉ။

### 2. ၵၢၼ်တႅမ်ႈ Code တႃႇ Smart Backspacing
ႁဝ်းတေၸႂ်ႉ Virtual Key `[K_BKSP]` ၵိုၵ်းတင်း Context တႃႇၵုမ်းထိင်းၵၢၼ်လူတ်ႈၶႃႈ။

**တူဝ်ယၢင်ႇ Code:**
```keyman
c --- သင်မီး Vowel 'ေႃ' ဝႆႉယူႇ သေၼဵၵ်ႉ Backspace ---
'ေ' 'ႃ' + [K_BKSP] > nul  c လုမ်းပႅတ်ႈတင်းသွင်တူဝ်ၵမ်းလဵဝ်

c --- သင်မီး Consonant + Medial + Vowel သေၶႂ်ႈလုမ်းပႅတ်ႈပဵၼ် Unit ---
'ၵ' 'ႂ' '်' + [K_BKSP] > 'ၵ'  c လုမ်း 'ႂ်' ဢွၵ်ႇၵမ်းလဵဝ် ၵိုတ်းဝႆႉ 'ၵ'
```



### 3. ၵၢၼ်ၵႅတ်ႇၶႄ "တူဝ်လိၵ်ႈၵိုတ်းၶၢႆး" (Dotted Circles)
မိူဝ်ႈႁဝ်းလူတ်ႉတူဝ် Base Consonant ဢွၵ်ႇပႅတ်ႈၼၼ်ႉ ၵူၺ်းၵႃႈ သမ်ႉၵိုတ်းဝႆႉ "မၢႆသဵင်" (Tone) ဢမ်ႇၼၼ် "သရ" (Vowel) ၼၼ်ႉ Font တေၼႄႁၢင်ႈပဵၼ် "တူင်ႇမူၼ်း" (Dotted circle) ဢၼ်ႁၢင်ႈၸႃႉၼၼ်ႉယဝ်ႉ။ 
**Smart Backspacing** ၸၢင်ႈၸွႆးလူတ်းပႅတ်ႈတင်း "Logical Unit" ႁႂ်ႈမၼ်းၵိုတ်းဝႆႉ တူဝ်လိၵ်ႈဢၼ်လူလႆႈၵူၺ်းယဝ်ႉ။

### 4. လွင်ႈလီၶွင် Smart Backspacing
* **Efficiency:** ၵူၼ်းၸႂ်ႉဢမ်ႇလူဝ်ႇၼဵၵ်ႉ Backspace လၢႆလၢႆပွၵ်ႈ တႃႇလူတ်းတူဝ်လိၵ်ႈ 1 တူဝ်။
* **User Experience:** ႁဵတ်းႁႂ်ႈ Keyboard ႁဵတ်းၵၢၼ် "မႅၼ်ႈၸွမ်းၸႂ်" ၵူၼ်းတႅမ်ႈလိၵ်ႈတႆး။
* **Visual Consistency:** ၵႅတ်ႇၶႄပၼ်ႁႃ တူဝ်လိၵ်ႈဢၼ်ၼႄႁၢင်ႈၽိတ်းပိူင် ဝၢႆးသေလူတ်းတူဝ်လိၵ်ႈယဝ်ႉ။
