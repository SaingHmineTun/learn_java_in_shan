## Lesson 9: Understanding the Context Pipe

ၼႂ်း Keyman Language ၼၼ်ႉ **Context Pipe** ပဵၼ်လွင်ႈႁဵတ်းၵၢၼ် ဢၼ် Keyboard "တူၺ်းၶိုၼ်း" ဝႃႈ ၼႃႈၸေႃး (Screen) ႁဝ်းၼၼ်ႉ 
မီးတူဝ်လိၵ်ႈသင်ဝႆႉယူႇ ဢွၼ်ၼႃႈတေၼဵၵ်ႉ Key ထႅင်ႈတူဝ်ၼိုင်ႈၼၼ်ႉယဝ်ႉ။

### 1. Context ဢၼ်ဝႃႈၼၼ်ႉ ပဵၼ်သင်?
Context ပဵၼ်တူဝ်လိၵ်ႈ (Characters) ဢၼ်မီးဝႆႉယူႇၽၢႆႇသၢႆႉၶွင် Cursor ႁဝ်းၼၼ်ႉယဝ်ႉ။
* **ပိူင်တႅမ်ႈမၼ်း:** `Context + Key > Output`
* **တူဝ်ယၢင်ႇ:** သင်ဝႃႈၼိူဝ် Screen မီးတူဝ် `ၵ` ဝႆႉယူႇၼႆ ၵမ်းၼႆႉႁဝ်းၼဵၵ်ႉ `ႇ` (Tone-1) ထႅင်ႈၼႆ Keyboard တေလူ Context သေ တႅတ်ႈတေႃးပၼ်ဝႃႈ တေဢွၵ်ႇ `ၵႇ` ၼႆယဝ်ႉ။



### 2. ၵၢၼ်ၸႂ်ႉ `context` Keyword
တွၼ်ႈတႃႇႁႂ်ႈ Keyboard ၵိုတ်းသိမ်း (Preserve) တူဝ်လိၵ်ႈဢၼ်မီးဝႆႉယူႇၼၼ်ႉ ႁဝ်းလူဝ်ႇၸႂ်ႉၶေႃႈသင်ႇ `context` ယဝ်ႉ။

**တူဝ်ယၢင်ႇ Code:**
```keyman
c --- သင်မီး 'ၵ' ဝႆႉယူႇသေ ၼဵၵ်ႉ 'ႇ' ---
'ၵ' + [K_J] > context 'ႇ'
```
* သင်ႁဝ်းဢမ်ႇသႂ်ႇ `context` ဝႆႉၽၢႆႇၶႂႃ၊ တူဝ် `ၵ` တေႁၢႆၵႂႃႇသေ ဢွၵ်ႇမႃးတူဝ် `ႇ` တူဝ်လဵဝ်ၵူၺ်းယဝ်ႉ။

### 3. ၵၢၼ်တူၺ်း Context လၢႆလၢႆတူဝ်
ႁဝ်းၸၢင်ႈၸႂ်ႉ `context` တႃႇတူၺ်းၶိုၼ်းလၢႆလၢႆ Characters ပႃးယဝ်ႉ။
* **Rule:** `'ၵ' 'ႃ' + [K_J] > context 'ႇ'`
* **Result:** `ၵႃ` + `ႇ` = `ၵႃႇ`

### 4. လွင်ႈလၢၵ်ႇလၢႆးၶွင် Keyman Pipe
Keyman မၼ်းဢမ်ႇမိူၼ် Keyboard Driver။ မၼ်းတေ "လူ" တူဝ်လိၵ်ႈတူဝ်ၵဝ်ႇတႃႇသေႇ။ 
သင်ဝႃႈႁဝ်းတႅမ်ႈ Rule ဢၼ်တႅတ်ႈတေႃးဝႆႉၼႆ မၼ်းတေမႄး (Modify) တူဝ်လိၵ်ႈဢၼ်မီးဝႆႉယူႇၼၼ်ႉသေ
လႅၵ်ႈလၢႆႈပဵၼ်တူဝ်မႂ်ႇလႆႈၵမ်းလဵဝ် (Dynamic Character Processing) ယဝ်ႉ။

| IT Terms | ၶေႃႈၵႂၢမ်းတႆး / Definition |
| :--- | :--- |
| **Context** | တူဝ်လိၵ်ႈဢၼ်မီးဝႆႉယူႇ ၽၢႆႇၼႃႈ (ၽၢႆႇသၢႆႉ) ၶွင် Cursor |

| **Pipe** | ၸၼ်ႉၵၢၼ်သူင်ႇၶေႃႈမုၼ်း တႄႇတီႈ Key တေႃႇပေႃးဢွၵ်ႇပဵၼ် Output |

**Summary:** ၼႂ်းတွၼ်ႈသွၼ်ၼႆႉ ႁဝ်းလႆႈႁဵၼ်းႁူႉလွင်ႈ **Context Pipe** ယဝ်ႉ။ ႁဝ်းလႆႈပွင်ႇၸႂ်ဝႃႈ Keyboard ၸၢင်ႈ "တူၺ်းၶိုၼ်း" တူဝ်လိၵ်ႈဢၼ်မီးဝႆႉယူႇၼိူဝ် Screen သေ ၸတ်းၵၢၼ်လႅၵ်ႈလၢႆႈ Output ႁႂ်ႈမၼ်းမႅၼ်ႈၸွမ်းပိူင်လူၺ်ႈၵၢၼ်ၸႂ်ႉၶေႃႈသင်ႇ `context` ယဝ်ႉ။
