## Lesson 12: Preventing "Illegal" Sequences

ၵၢၼ်တႅမ်ႈလိၵ်ႈတႆးၼၼ်ႉ မၼ်းမီးပိူင်ဝၢင်းတူဝ်လိၵ်ႈဢၼ်တႅတ်ႈတေႃး။ သင်ႁဝ်းပၼ်ၵူၼ်းၸႂ်ႉတႅမ်ႈ "မၢႆသဵင်" (Tone) ဝႆႉၽၢႆႇၼႃႈ "တူဝ်မႄႈလိၵ်ႈ" (Consonant) ၼႆ Unicode မၼ်းတေၽိတ်း သေ Font တေၼႄႁၢင်ႈဢမ်ႇလႆႈယဝ်ႉ။

### 1. ၵၢၼ်ၵႅတ်ႇၶႄ Tone Stacking (မၢႆသဵင်သႂ်ႇသွၼ်ႉၵၼ်)
ၼႂ်းလိၵ်ႈတႆး ႁဝ်းဢမ်ႇၸၢင်ႈသႂ်ႇမၢႆသဵင် 2 တူဝ်ၵိုၵ်းၵၼ် (မိူၼ်ၼင်ႇ `ႇ` + `ႈ`)။ ႁဝ်းလူဝ်ႇတႅမ်ႈ Rule တႃႇ "ႁၢမ်ႈ" (Block) တူဝ်ထူၼ်ႈသွင်ယဝ်ႉ။

**တူဝ်ယၢင်ႇ Code:**
```keyman
store(tones) U+1087 U+1088 U+1089 U+108A U+109C U+109D c ႇ ႈ ႉ း ႜ ႝ

c --- သင်မီး Tone ဝႆႉယူႇယဝ်ႉ သမ်ႉၼဵၵ်ႉ Tone ထႅင်ႈ ---
any(tones) + any(tones) > context  c ႁႂ်ႈမၼ်းၵိုတ်းဝႆႉတီႈ Context တူဝ်ၵဝ်ႇ၊ ဢမ်ႇပၼ်ဢွၵ်ႇတူဝ်မႂ်ႇ
```

### 2. ၵၢၼ်ၵႅတ်ႇၶႄ Vowels Standalone (သရယူႇတူဝ်လဵဝ်)
သရ (Vowels) ၼႂ်းလိၵ်ႈတႆး တေလႆႈမီး Consonant ဝႆႉၽၢႆႇၼႃႈတႃႇသေႇ။ သင်ဝႃႈၼႃႈ Cursor ပဵၼ် "တီႈပဝ်ႇ" (Empty) ဢမ်ႇၼၼ် ပဵၼ် "ၵူႈသိုဝ်ႈ" (Space)၊ ႁဝ်းဢမ်ႇလူဝ်ႇပၼ်တႅမ်ႈ Vowel ယဝ်ႉ။

**တူဝ်ယၢင်ႇ Code:**
```keyman
store(vowels) U+1083 U+1084 U+107D c ု ူ ွ
store(valid_base) [K_SPACE] [K_ENTER]  c ၸိူဝ်းဢၼ်ဢမ်ႇၸၢင်ႈတႅၼ်းတူဝ် Base လႆႈ

c --- သင်ပဵၼ် Start of Line ဢမ်ႇၼၼ် Space သေ သမ်ႉၼဵၵ်ႉ Vowel ---
nul + any(vowels) > beep  c ႁႂ်ႈမၼ်းမီးသဵင် Beep သေ ဢမ်ႇပၼ်ဢွၵ်ႇတူဝ်လိၵ်ႈ
```

### 3. ၵၢၼ်ၸႂ်ႉ `beep` Command
`beep` ပဵၼ်ၶေႃႈသင်ႇ ဢၼ်တေႁဵတ်းႁႂ်ႈ Computer/Phone မီးသဵင် "တွၵ်ႉ" (ဢမ်ႇၼၼ် သၼ်ႇ) တႃႇလၢတ်ႈၼႄဝႃႈ "ၼဵၵ်ႉၽိတ်းယဝ်ႉ" ၼႆယဝ်ႉ။



### 4. လွင်ႈလီၶွင်ၵၢၼ် Preventing Illegal Sequences
* **Data Integrity:** ႁဵတ်းႁႂ်ႈ Document ဢၼ်တႅမ်ႈလူၺ်ႈ Keyboard ၼႆႉ မီး Unicode ဢၼ်မႅၼ်ႈၸွမ်းပိူင် 100%။
* **User Education:** ၸွႆႈသွၼ်ပၼ်ၵူၼ်းၸႂ်ႉ ႁႂ်ႈႁူႉလၢႆးတႅမ်ႈလိၵ်ႈတႆး ဢၼ်မႅၼ်ႈမၼ်း။
* **Rendering Safety:** ၵႅတ်ႇၶႄပၼ်ႁႃ တူဝ်လိၵ်ႈပဵၼ် "လႅၵ်းၶွပ်ႇ" (Dotted circles/boxes) ဢၼ်ၵိူတ်ႇယွၼ်ႉ Unicode ၽိတ်းယဝ်ႉ။

ဢမ်ႇပၼ်သႂ်ႇ Tone သွၼ်ႉၵၼ် လႄႈ ဢမ်ႇပၼ် Vowel ယူႇတူဝ်လဵဝ်။ 
ၵၢၼ်ၸႂ်ႉ `context` လႄႈ `beep` တေႁဵတ်းႁႂ်ႈ Keyboard **TMK Shan** ႁဝ်း ပဵၼ် Keyboard ဢၼ်မီး Quality သုင် လႄႈ ၸွႆးၵုမ်းထိင်း Unicode ႁႂ်ႈမႅၼ်ႈၸွမ်း Standard တႃႇသေႇယဝ်ႉ။
