## Lesson 10: Unicode Normalization & Reordering

မိူဝ်ႈႁဝ်းတႅမ်ႈလိၵ်ႈတႆးၼၼ်ႉ တူဝ်လိၵ်ႈမၢင်တူဝ် ၸၢင်ႈတႅမ်ႈလႆႈလၢႆလၢႆလၢႆး (Input Order) ယဝ်ႉ။
ၵူၺ်းၵႃႈ ၼႂ်း Unicode မၼ်းလူဝ်ႇမီးလၢႆးဝၢင်းဢၼ်မႅၼ်ႈ (Logical Order) တူဝ်လဵဝ်ၵူၺ်းယဝ်ႉ။

### 1. Unicode Normalization (ၵၢၼ်ႁဵတ်းႁႂ်ႈပဵၼ်ပိူင် Standard)
Normalization ပဵၼ်ၵၢၼ်မႄးတူဝ်လိၵ်ႈ ဢၼ်ႁၢင်ႈမိူၼ်ၵၼ် ၵူၺ်းၵႃႈ Code ဢမ်ႇမိူၼ်ၵၼ် ႁႂ်ႈမၼ်းပဵၼ် Code တူဝ်လဵဝ်ၵၼ် တူဝ်ဢၼ်မႅၼ်ႈ Standard ၼၼ်ႉယဝ်ႉ。
* **တူဝ်ယၢင်ႇ:** တူဝ် **"ၷ"** (Pali Ga) လႄႈ တူဝ် **"ၷ"** (ဢၼ်ဢဝ် 'ၵ' + 'ႂ' မႃးလေႃးၵၼ်) ၼၼ်ႉ တူၺ်းမိူၼ်ၵၼ်ဝႆႉ။ ၵူၺ်းၵႃႈ ၼႂ်း Computer မၼ်းပဵၼ်ၵူၼ်း ၶူတ်ႉၽႂ်မၼ်းဝႆႉ။
* **Professional Tip:** ၼႂ်း Keyboard ႁဝ်း ႁဝ်းလူဝ်ႇတႅမ်ႈ Rule ႁႂ်ႈမၼ်း Output ဢွၵ်ႇမႃးပဵၼ် Unicode Codepoint ဢၼ်မႅၼ်ႈၸွမ်း Standard ၵူၺ်း။ ႁႂ်ႈမၼ်းငၢႆႈတီႈၵၢၼ် Search လႄႈ ၵၢၼ်ပိၼ်ႇ Font ယဝ်ႉ。

### 2. Unicode Reordering (ၵၢၼ်လႅၵ်ႈလၢႆႈတႅၼ်ႈတီႈ)
ပိူင်ဝၢင်း Unicode လိၵ်ႈတႆးၼၼ်ႉ မီးပိူင်ဝႆႉၼင်ႇၼႆ: **မႄႈလိၵ်ႈ (Consonant) + တူဝ်ႁွႆႈ (Medial) + သရ (Vowel) + မၢႆသဵင် (Tone)**。

သင်ဝႃႈၵူၼ်းၸႂ်ႉ ၼဵၵ်ႉမၢႆသဵင် (Tone) ဢွၼ်ၼႃႈ သရ (Vowel) ၼႆ Keyboard ႁဝ်းလူဝ်ႇမီး "ဢွၵ်းဢေႃ" တႃႇလႅၵ်ႈလၢႆႈ (Swap) ႁႂ်ႈမၼ်းမႅၼ်ႈၸွမ်းပိူင် Unicode ယဝ်ႉ。



**တူဝ်ယၢင်ႇ Code တႃႇ Reordering:**
```keyman
c --- သင်ၼဵၵ်ႉ Tone (ႇ) ဢွၼ်ၼႃႈ Vowel (ု) ---
U+1087 + [K_K] > U+1083 U+1087  c လႅၵ်ႈလၢႆႈပဵၼ် ု + ႇ
```

### 3. ၵၢၼ်ၸတ်းၵၢၼ် Double Symbols
မၢင်ပွၵ်ႈၵူၼ်းၸႂ်ႉ ၸၢင်ႈၼဵၵ်ႉမၢႆသဵင် သွင်ပွၵ်ႈ (Double Tones)၊ Keyboard ႁဝ်းလူဝ်ႇထတ်းသေ လုမ်းဢွၵ်ႇပၼ်တူဝ်ၼိုင်ႈ ႁႂ်ႈမၼ်းဢမ်ႇပဵၼ် Unicode ဢၼ်ၽိတ်းယဝ်ႉ。

```keyman
c --- သင်မီး Tone ဝႆႉယူႇသေ ၼဵၵ်ႉ Tone ထႅင်ႈတူဝ်ၼိုင်ႈ ---
U+1087 + [K_J] > context  c ဢမ်ႇႁႂ်ႈမၼ်းဢွၵ်ႇမႃးထႅင်ႈ (Ignore)
```

### Summary (တွၼ်ႈၵႅပ်ႈ Lesson 10)

| IT Terms | ၶေႃႈၵႂၢမ်းတႆး / Definition                                  |
| :--- |:------------------------------------------------------------|
| **Normalization** | ၵၢၼ်မႄး Character ႁႂ်ႈပဵၼ် Unicode Standard တူဝ်လဵဝ်ၵၼ်     |
| **Reordering** | ၵၢၼ်လႅၵ်ႈလၢႆႈတူဝ်လိၵ်ႈ ႁႂ်ႈမၼ်းမႅၼ်ႈၸွမ်း Logical Order     |
| **Logical Order** | လၢႆးဝၢင်းတူဝ်လိၵ်ႈ ဢၼ် Unicode Standard မၢႆဝႆႉ              |
| **Input Swap** | ၵၢၼ်လႅၵ်ႈလၢႆႈတူဝ်လိၵ်ႈ ဢၼ်ၵူၼ်းၸႂ်ႉတႅမ်ႈၽိတ်း ႁႂ်ႈမၼ်းမႅၼ်ႈ |

**Summary:** ၼႂ်းတွၼ်ႈသွၼ်ၼႆႉ ႁဝ်းလႆႈႁဵၼ်းႁူႉလွင်ႈၵၢၼ်ၸတ်းၵၢၼ် Unicode ႁႂ်ႈမၼ်းပဵၼ်ပိူင် Standard (Normalization) လႄႈ ၵၢၼ်မႄးလၢႆးဝၢင်းတူဝ်လိၵ်ႈ (Reordering) ႁႂ်ႈမႅၼ်ႈၸွမ်းပိူင် Consonant + Medial + Vowel + Tone ယဝ်ႉ。 ၵၢၼ်ႁဵတ်းၼင်ႇၼႆ တေႁဵတ်းႁႂ်ႈလိၵ်ႈတႆး ဢၼ်တႅမ်ႈလူၺ်ႈ **TMK Keyboard** ၼၼ်ႉ ၸႂ်ႉၵၢၼ်လႆႈၵူႈတီႈ ဢမ်ႇမီးပၼ်ႁႃလွင်ႈ Unicode ၽိတ်းယဝ်ႉ。
