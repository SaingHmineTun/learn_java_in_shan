## Lesson 13: Deadkeys (State Management)

**Deadkey** ဢၼ်ဝႃႈၼၼ်ႉ မၼ်းမိူၼ်တင်း Key ဢၼ် "တၢႆ" ဝႆႉယူႇၸူဝ်ႈၶၢဝ်းယဝ်ႉ။
မိူဝ်ႈႁဝ်းၼဵၵ်ႉမၼ်းၼၼ်ႉ ၼႃႈၸေႃး (Screen) တေဢမ်ႇၼႄသင်။
ၵူၺ်းၵႃႈ ၼႂ်း "ဢွၵ်းဢေႃ" Keyboard မၼ်းတေမၢႆဝႆႉဝႃႈ "Key ၼႆႉ ၼဵၵ်ႉဝႆႉယဝ်ႉယဝ်ႉ" ၼႆသေ 
မၼ်းတေၸွႆးလႅၵ်ႈလၢႆႈ Output ၶွင် Key တူဝ်ထူၼ်ႈသွင်ၼၼ်ႉယဝ်ႉ။

### 1. ၵၢၼ်ႁဵတ်းၵၢၼ်ၶွင် Deadkey
* **Step 1:** ၼဵၵ်ႉ Deadkey (မိူၼ်ၼင်ႇ Key `/`) -> ဢမ်ႇမီးသင်ဢွၵ်ႇၼိူဝ် Screen။
* **Step 2:** ၼဵၵ်ႉ Key `k` -> Keyboard လူထတ်းဝႃႈ "မီး Deadkey ဝႆႉယူႇ"။
* **Step 3:** Keyboard ပိၼ်ႇဢွၵ်ႇပဵၼ်တူဝ်လိၵ်ႈလၢၵ်ႇလၢႆး (မိူၼ်ၼင်ႇ `ၵၵ` ဢမ်ႇၼၼ် တူဝ် Pali)။



### ### 2. ၵၢၼ်တႅမ်ႈ Code တွၼ်ႈတႃႇ Deadkey
ၼႂ်း Keyman ႁဝ်းတေၸႂ်ႉၶေႃႈသင်ႇ `dk(n)` (n ပဵၼ်မၢႆလၢပ်ႈ 1, 2, 3...) ယဝ်ႉ။

**တူဝ်ယၢင်ႇ:** ႁဝ်းၶႂ်ႈၸႂ်ႉ Key `/` ပဵၼ် Deadkey တႃႇတႅမ်ႈတူဝ်လိၵ်ႈ Pali ႁႂ်ႈငၢႆႈ။
```keyman
c --- 1. တင်ႈၵႃႈ Deadkey ---
+ [K_SLASH] > dk(1)

c --- 2. ၵၢၼ်ၸႂ်ႉ Deadkey ၵိုၵ်းတင်း Key တၢင်ႇဢၼ် ---
dk(1) + [K_K] > U+1075 U+1075  c ၵၵ (Stacking)
dk(1) + [K_G] > U+107B         c ၷ (Pali Ga)
```

### 3. လၢႆးၸႂ်ႉ Deadkeys ၼႂ်း TMK Keyboard
* **Secondary Layer:** တွၼ်ႈတႃႇၵူၼ်းၸႂ်ႉ ဢၼ်ဢမ်ႇၶႂ်ႈၼဵၵ်ႉ Shift ၵမ်းၵမ်းၼၼ်ႉ ႁဝ်းၸၢင်ႈၸႂ်ႉ Deadkey သေ ႁဵတ်းပဵၼ် Shortcut တႃႇတႅမ်ႈတူဝ်လိၵ်ႈၸၼ်ႉ Shift လႆႈယဝ်ႉ။
* **Character Variants:** ၸႂ်ႉတႃႇတႅမ်ႈတူဝ်လိၵ်ႈဢၼ်မီးႁၢင်ႈလၢႆလၢႆပိူင် (မိူၼ်ၼင်ႇ `ၼ` လႄႈ `ꧣ`)။
* **Smart Shortcuts:** ၼဵၵ်ႉ Deadkey သေ ၸၢင်ႈဢွၵ်ႇမႃးပဵၼ် ၶေႃႈၵႂၢမ်းၵိုၵ်း (Phrases) ဢမ်ႇၼၼ် တူဝ်လိၵ်ႈဢၼ်ၸႂ်ႉၼမ်။

### 4. လွင်ႈပႅၵ်ႇပိူင်ႈတင်း Shift
* **Shift:** တေလႆႈၼဵၵ်ႉ "ၵိုၵ်းၵၼ်" (Press and hold)။
* **Deadkey:** တေလႆႈၼဵၵ်ႉ "ၵမ်းတူဝ်တူဝ်" (Press and release, then press next key)။ မၼ်းငၢႆႈတွၼ်ႈတႃႇၵူၼ်းၸႂ်ႉ ဢၼ်ဢမ်ႇမေႃၼဵၵ်ႉ Key လၢႆလၢႆတူဝ်ၵိုၵ်းၵၼ်ယဝ်ႉ။

ၵၢၼ်ၸႂ်ႉ Deadkeys တေၸွႆးႁႂ်ႈ Keyboard **TMK Shan** ႁဝ်း မီးလၢႆးတႅမ်ႈတူဝ်လိၵ်ႈဢၼ်ငၢႆႈမႃး (Shortcut) လႄႈ ၸတ်းၵၢၼ်တူဝ်လိၵ်ႈဢၼ်ယၢပ်ႇၼၼ်ႉ လႆႈၶႅမ်ႉလႅပ်ႈယဝ်ႉ။
