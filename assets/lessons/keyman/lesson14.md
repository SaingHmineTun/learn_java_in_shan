## Lesson 14: The notany() and nul Commands

### 1. ၵၢၼ်ႁဵတ်းၵၢၼ်ၶွင် `notany()`
`notany()` ပဵၼ်ၶေႃႈသင်ႇဢၼ်လၢတ်ႈဝႃႈ "သင်ဝႃႈ Context ဢမ်ႇၸႂ်ႈတူဝ်လိၵ်ႈဢၼ်မီးၼႂ်း store ၼႆႉ" ၼႆယဝ်ႉ။
မၼ်းပဵၼ်လွင်ႈၵၢႆႇၶွတ်ႇ (Inverse) ၶွင် `any()` ၼၼ်ႉယဝ်ႉ။

* **လွင်ႈလီမၼ်း:** ၸွႆးၵႅတ်ႇၶႄ ပၼ်ႁႃတႅမ်ႈလိၵ်ႈၽိတ်းပိူင် ၸွမ်းၼင်ႇပိူင် Unicode Logical Order။

**တူဝ်ယၢင်ႇ Code:**
ႁဝ်းၶႂ်ႈလၢတ်ႈဝႃႈ "သင်ၽၢႆႇၼႃႈဢမ်ႇၸႂ်ႈမႄႈလိၵ်ႈ (Consonant) ၼႆ ဢမ်ႇပၼ်သႂ်ႇမၢႆသဵင် (Tone)"။
```keyman
store(consonants) U+1075 .. U+1082  c ၵ - ႁ

notany(consonants) + [K_J] > beep
```

### 2. ၵၢၼ်ႁဵတ်းၵၢၼ်ၶွင် `nul`
`nul` ပဵၼ်ၶေႃႈသင်ႇတႃႇယႃႉပႅတ်ႈ (Nullify) Output ယဝ်ႉ။ မၼ်းပွင်ႇဝႃႈ "ဢမ်ႇႁႂ်ႈမီးသင်ဢွၵ်ႇမႃး" သေဢမ်ႇၵႃး မၼ်းယႃႉပႅတ်ႈ Rule တၢင်ႇဢၼ် ဢၼ်ၸၢင်ႈမႃးမႅၼ်ႈ (Match) ပႃးယဝ်ႉ။

* **Usage 1: Start of Line** - ၸႂ်ႉ `nul` တီႈၽၢႆႇသၢႆႉ တႃႇမၢႆထတ်းတူၺ်း "တႄႇထႅဝ်မႂ်ႇ" (Start of context)။
* **Usage 2: Blocking Input** - ၸႂ်ႉတီႈၽၢႆႇၶႂႃ တႃႇဢမ်ႇပၼ် Key ၼၼ်ႉဢွၵ်ႇ Output သင်။



### 3. ၵၢၼ်ၸႂ်ႉ `notany()` လႄႈ `nul` ၵိုၵ်းၵၼ်
ၼႆႉပဵၼ် Logic ၸၼ်ႉသုင် တႃႇၵႅတ်ႇၶႄ Illegal Sequences ဢၼ်ႁဝ်းလႆႈႁဵၼ်းမႃးၼႂ်း Lesson 12 ယဝ်ႉ။

```keyman
c --- ဢမ်ႇပၼ် Vowel ယူႇတူဝ်လဵဝ် တီႈတႄႇထႅဝ် ---
nul + any(vowels) > beep

c --- သင်မီးတူဝ်လိၵ်ႈဢၼ်ဢမ်ႇၸႂ်ႈ Consonant ဝႆႉယူႇ သမ်ႉၼဵၵ်ႉ Medial Wa ---
notany(consonants) + [K_G] > nul
```

### 4. လွင်ႈလီၶွင်ၵၢၼ်ၸႂ်ႉ Lessons ၼႆႉ
* **Robust Validation:** ႁဵတ်းႁႂ်ႈ Keyboard မီးဢွၵ်းဢေႃ ၵုမ်းထိင်း Unicode ႁႂ်ႈမႅၼ်ႈၸွမ်းပိူင် 100%။
* **Silent Error Handling:** ၸၢင်ႈၸႂ်ႉ `nul` တႃႇထွႆ (Ignore) ပႅတ်ႈ Key ဢၼ်ၼဵၵ်ႉၽိတ်း လူၺ်ႈဢမ်ႇလူဝ်ႇမီးသဵင် Beep။
* **Contextual Control:** ၸွႆးၸတ်းၵၢၼ် Rule ႁႂ်ႈမၼ်းႁဵတ်းၵၢၼ် တႅတ်ႈတေႃးၸွမ်းၼင်ႇ Context ဢၼ်မႅၼ်ႈမၼ်းၵူၺ်း။
