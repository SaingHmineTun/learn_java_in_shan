## ## Lesson 8: Number Rows & Shan Symbols

### 1. Shan Number Row (တူဝ်ၼပ်ႉတႆး)
ၸွမ်းၼင်ႇ XML Layout ဢၼ်ၸဝ်ႈၵဝ်ႇလႆႈဝၢင်းဝႆႉၼၼ်ႉ ႁဝ်းတေမႃးလႅၵ်ႈလၢႆႈတူဝ်ၼပ်ႉ 1-0 ႁႂ်ႈပဵၼ် Unicode Hex ၼင်ႇၼႆၶႃႈ။

| Key | Shan Number | Unicode Hex |
| :--- | :---: | :--- |
| `[K_1]` | **႑** | `U+1091` |
| `[K_2]` | **႒** | `U+1092` |
| `[K_3]` | **႓** | `U+1093` |
| `[K_4]` | **႔** | `U+1094` |
| `[K_5]` | **႕** | `U+1095` |
| `[K_6]` | **႖** | `U+1096` |
| `[K_7]` | **႗** | `U+1097` |
| `[K_8]` | **႘** | `U+1098` |
| `[K_9]` | **႙** | `U+1099` |
| `[K_0]` | **႐** | `U+1090` |



### 2. Shan Punctuation & Symbols (ၶိူင်ႈမၢႆတႆး)
ၼႂ်းလိၵ်ႈတႆးႁဝ်း မၼ်းမီးၶိူင်ႈမၢႆ ဢၼ်ပႅၵ်ႇပိူင်ႈတင်းလိၵ်ႈဢင်းၵိတ်ႉ။ ႁဝ်းလူဝ်ႇမၢႆဝႆႉ (Map) ႁႂ်ႈမၼ်းမႅၼ်ႈၸွမ်း Unicode Standard ယဝ်ႉ။

* **Shan Comma (၊):** ၸႂ်ႉ `U+104A` (တႅၼ်းတူဝ် `,`)
* **Shan Full Stop (။):** ၸႂ်ႉ `U+104B` (တႅၼ်းတူဝ် `.`)
* **Visible Tone (း):** ၸႂ်ႉ `U+108A` (Visarga)

### 3. ၵၢၼ်တႅမ်ႈ Code တွၼ်ႈတႃႇ Lesson 8
ႁဝ်းတေၸၢင်ႈဢဝ် `store()` လႄႈ `Unicode Hex` မႃးၸႂ်ႉၵိုၵ်းၵၼ် ႁႂ်ႈ Code ႁဝ်းသၼ်းၼင်ႇၼႆၶႃႈ။

```keyman
c --- Shan Numbers Store ---
store(num_key) '1234567890'
store(num_val) U+1091 U+1092 U+1093 U+1094 U+1095 U+1096 U+1097 U+1098 U+1099 U+1090

group(main) using keys

c --- Number Row Logic ---
+ any(num_key) > index(num_val, 1)

c --- Punctuation Logic ---
+ [K_PERIOD] > U+104B  c Shan Full Stop (။)
+ [K_COMMA]  > U+104A  c Shan Comma (၊)
```

### Summary (တွၼ်ႈၵႅပ်ႈ Lesson 8)

| IT Terms | ၶေႃႈၵႂၢမ်းတႆး / Definition |
| :--- | :--- |
| **Number Row** | ၸၼ်ႉ Key တူဝ်ၼပ်ႉ ဢၼ်မီးၼိူဝ်သုတ်းၶွင် Keyboard |
| **Punctuation** | ၶိူင်ႈမၢႆလုမ်း (မိူၼ်ၼင်ႇ ၊ ။ ?) ဢၼ်ၸႂ်ႉၼႂ်းၵၢၼ်တႅမ်ႈလိၵ်ႈ |
| **Hex String** | ၵၢၼ်တႅမ်ႈ Unicode Hex လၢႆလၢႆတူဝ် တူၵ်းၵၼ်ၼႂ်း store ၼိုင်ႈဢၼ် |
| **Mapping Order** | ၵၢၼ်ၸတ်းလၢႆး ဝၢင်းတူဝ်လိၵ်ႈ ႁႂ်ႈမၼ်းမႅၼ်ႈၸွမ်း Key ဢၼ်ၼဵၵ်ႉ |

**Summary:** ၼႂ်းတွၼ်ႈသွၼ်ၼႆႉ ႁဝ်းလႆႈႁဵၼ်းႁူႉလွင်ႈၵၢၼ်တႅမ်ႈ Code တႃႇတူဝ်ၼပ်ႉတႆး လႄႈ ၶိူင်ႈမၢႆတႆး (။ ၊) လူၺ်ႈၵၢၼ်ၸႂ်ႉ **Unicode Hex**။ ၵၢၼ်ၸႂ်ႉ `store()` ၵိုၵ်းတင်း `any()` လႄႈ `index()` တေႁဵတ်းႁႂ်ႈ Code ႁဝ်းသၼ်းလူင်းၼမ်သေ ၸတ်းၵၢၼ်လႆႈငၢႆႈယဝ်ႉ။
