
## **Java Basic 6**

#### **ARITHMETIC OPERATORS**

Link - <https://youtu.be/CSduNT-meME>

Operator ၼႆႉ ပဵၼ်မၢႆ Symbol ဢၼ်ႁဝ်းတေ ၸႂ်ႉတိုဝ်းလႆႈ တီႈ program ႁဝ်းဢိူဝ်ႈ။ Arithmetic Operators ၼႆႉသမ်ႉ ပဵၼ်မၢႆ ဢၼ်ႁဝ်းတေ ၸႂ်ႉတိုဝ်းလႆႈ မိူဝ်းႁဝ်းတေႁဵတ်ႉ လွင်ႈၼပ်ႉသွၼ်ႇ ၼၼ်ႉယဝ်ႉ။ တီႈ Java ၼႆႉ Arithmetic operator မီးယူႇ 5 မဵဝ်းၼႆႉၵူၺ်း။

- _\+ ( Addition ) - မၢႆလေႃး_
- _\- ( Subtraction ) - မၢႆထွၼ်_
- _\* ( Multiplication ) - မၢႆၵုၼ်း_
- _/ ( Division ) - မၢႆၸႅၵ်ႇ_
- _% ( Modulus ) - မၢႆၸႅၵ်ႇလိူဝ်_

မၢႆ Arithmetic ၼႆႉသမ်ႉ မၼ်းတေ ႁဵတ်ႉၵၢၼ်ၸွမ်း int ဢမ်ႇၼၼ် float သေ မၼ်းတေ ပၼ်ႁဝ်းၶိုၼ်းၵေႃႈ int ဢမ်ႇၼၼ် float ယဝ်ႉ။

လၢႆးႁဵတ်ႉၵၢၼ်

- **Integer + Integer = Integer**
- **Integer + Floating = Floating**
- **Floating + Integer = Floating**
- **Floating + Floating = Floating**

ၵွပ်ႈပိူဝ်ႈ int / int ၼႆႉ မၼ်းတေပၼ်ႁဝ်းၶိုၼ်းလႆႈ int ၵူၺ်းလႄႈ ပေႃးဝႃႈ ½ ၼႆ မၼ်းတေပၼ်ႁဝ်းၶိုၼ်း 0 ၵူၺ်း။ သင်ၶႆႈလႆႈ ၵႃႊၶၼ်ဢၼ်ပႃး decimal point ၼႆၸိုင် operand သေဢၼ်ဢၼ်ၼၼ်ႉ လၢႆႊပဵၼ် floating သေလႄႈ။

###### လၢႆးၸႂ်ႉတိုဝ်း

```
int sum = 5 + 3; // မၢႆလေႃး
int remain = 5 - 3; // မၢႆထွၼ်
int multiplication = 5 \* 3; // မၢႆၵုၼ်း
double division = 5 / 3.0; // မၢႆၸႅၵ်ႇ
int modulus = 4 % 3; // မၢႆၸႅၵ်ႇလိူဝ်
```
