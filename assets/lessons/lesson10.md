
## **Java Basic 10**

#### **LOGICAL OPERATORS**

Link - <https://youtu.be/WYkfUf5-I-4>

တီႈ Java ၼႆႉတေႉ မၼ်းတေမီးဝႆႉ Logical Operator 3 ဢၼ်ၶႃႈ

- _&& ( Logical AND )_
- _|| ( Logical OR )_
- _! ( Logical NOT )_

Logical operator ၼႆႉသမ်ႉ မၼ်းတေ ႁဵတ်ႉၵၢၼ်ၸွမ်း boolean ၵူၺ်းလႄႈ မၼ်းပၼ်ၶိုၼ်း boolean ႁဝ်းၶိုၼ်းၵူၺ်း။

လၢႆးႁဵတ်ႉၵၢၼ်

###### && ( Logical AND operator)

**_true && true => true_**

**_false && true => false_**

**_false && false => false_**

**_true && false => false_**

###### || ( Logical OR )

**_true || true => true_**

**_false || true => true_**

**_true || false => true_**

**_false || false => false_**

###### ! ( Logical NOT )

**_!true = false_**

**_!false = true_**

###### လၢႆးၸႂ်ႉတိုဝ်း

```
boolean and = (5 < 8) && (10 <= 11);

boolean or = false || false;

boolean not = !(6 > 5);
```
