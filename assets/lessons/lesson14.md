
## **Java Basic 14**

#### **ARRAY**

Link - <https://youtu.be/2aIZgzyumU4>

Array ၼႆႉ ႁဝ်းၸႂ်ႉတိုဝ်း မိူဝ်ႈႁဝ်း ၶႆႈသိမ်းဝႆႉ ၵႃႊၶၼ် တၢင်းၼမ်တၢင်းလၢႆယဝ်ႉ။ ၵူၺ်းၵႃႊ ၵႃႊၶၼ်တၢင်းမူတ်း ဢၼ်မီးဝႆႉ တီႈၼႂ်းမၼ်းၼႆႉသမ်ႉ တေလႆႈပဵၼ် Data type ဢၼ်လဵဝ်ၵူၺ်း။

**ႁဵတ်ႉ Array ၼိုင်ႈဢၼ်**

တႃႇတေႁဵတ်ႉ Array ၼႆႉတေႉ ဢွၼ်တၢင်း ႁဝ်းသႂ်ႇပၼ် data type မၼ်းယဝ်ႉသေ ၸင်ႇဢဝ် တူင်ႇသီႇၸဵင်ႇ \[\] ၼႆႉတႅမ်ႈၵိုၵ်ႉယဝ်ႉ။ မိူၼ်ၼင်ႇ ပေႃးႁဝ်းၶႆႈႁဵတ်ႉ int array ၼႆၸိုင် ႁဝ်းတႅမ်ႈ int\[\] ယဝ်ႉ။ တႃႇတေ ၶူင်သၢင်ႈ object ၼႆႉတေႉ လၢႆးမၼ်း မီးယူႇ သၢမ်လၢႆး။ ၵွပ်ႈ Array ပဵၼ် non-primitive လႄႈ ႁဝ်းႁွင်ႉၸႂ်ႉ new ၵေႃႈလႆႈယူႇ။ ၵူၺ်းၵႃႊ မၼ်းၼႆႉၵေႃႈပဵၼ် data type ဢၼ်ၸႂ်ႉတိုဝ်းၼမ် မိူၼ်ၼင်ႇ String ဝႆႉၼႆလႄႈ ႁဝ်းၸႂ်ႉလႆႈ Array literal ၵေႃႈလႆႈယဝ်ႉ။

###### လၢႆးၸႂ်ႉတိုဝ်း

```
int[] numbers0 = new int[5]; // ၵႃႊၶၼ်ပဵၼ် 0 မူတ်းမူတ်း_
int[] numbers1 = new int[]{1, 2, 3, 4, 5};
int[] numbers2 = {2, 4, 6, 8, 10};
```

**ဢဝ်တူၺ်းၵႃႊၶၼ်ၼႂ်း Array**

တႃႇတေ ဢဝ်ၵႃႊၶၼ် ဢၼ်ႁဝ်းသိမ်းဝႆႉၼႂ်း Array ၵမ်းဢၼ်ဢၼ်ၼႆၸိုင် ႁဝ်းၸႂ်ႉတိုဝ်း squared bracket \[\] လႄႈ indexing ယဝ်ႉ။ index ၼႆႉ ၵႃႊၶၼ်မၼ်း တေႇတီႈသုၼ် (0) ဢိူဝ်ႈ။ ၵွပ်ႈၼႆ သင်ႁဝ်းၶႆႈလႆႈ ၵႃႊၶၼ်ဢွၼ်တၢင်းသုတ်ႉ ဢၼ်မီးဝႆႉၼႂ်း Array ၼႆႁဝ်းတေလႆႈၸႂ်ႉ \[0\] ယဝ်ႉ။

###### လၢႆးၸႂ်ႉတိုဝ်း

```
int num = numbers2[3] + numbers1[4];
System.out.println(num);
```

**ပိၼ်ႇလၢႆႊၵႃႊၶၼ် တီႈၼႂ်း Array**

တႃႇပိၼ်ႇလၢႆႊၵႃႊၶၼ်မၼ်းၼႆၵေႃႈ မိူၼ်ၵၼ်တၢင်း မိူဝ်ႈႁွင်ႉၸႂ်ႉ ၵႃႊၶၼ်မၼ်းၼင်ႇၵဝ်ႇ။ ႁဝ်းတေလႆႈၸႂ်ႉ index သေၵေႃႈ ႁဵတ်ႉ update ပၼ်ယဝ်ႉ။

###### လၢႆးၸႂ်ႉတိုဝ်း

```
System.out.println("Before : " + numbers1[0]);
numbers1[0] = 10;
System.out.println("After : " + numbers1[0]); // 10
```

**Array ၼႆႉပေႃးႁဵတ်ႉယဝ်ႉ ထႅမ်မႂ်ႇ၊ ထွၼ်ၵဝ်ႇ ဢမ်ႇလႆႈ**

Array ၼႆႉမီး fixed length ဝႆႉဢိူဝ်ႈ။ ဝၢႆးသေ ႁဝ်းၶူင်သၢင်ႈ object မၼ်းယဝ်ႉၼႆၸိုင် ႁဝ်းတေထႅမ်မႂ်ႇ၊ ထွၼ်ၵဝ်ႇ ဢမ်ႇလႆႈ။ သင်ႁဵတ်ႉတူၺ်းၼႆတေႉ မၼ်းတေၼႄပၼ် ERROR ယဝ်ႉ။
