# Lesson 6: The Compilation Workflow

### ၶေႃႈတၢင်ႇလဝ်ႈ (Introduction)
ၵၢၼ်ပိၼ်ႇၶူတ်ႉ (Compilation) ၼႂ်း C++ ၼႆႉ မၼ်းဢမ်ႇၸႂ်ႈဝႃႈ ပိၼ်ႇၵမ်းလဵဝ်ယဝ်ႉၶႃႈ။ 
မၼ်းမီးၶၵ်ႉတွၼ်ႈလူင် 4 ဢၼ် ဢၼ်ႁဵတ်းႁႂ်ႈ Source Code ႁဝ်း ပဵၼ်ၾၢႆႇ Binary (.exe) ဢၼ်ႁဵတ်းၵၢၼ်လႆႈၼၼ်ႉယဝ်ႉ။

---

### 1. Preprocessing (ၵၢၼ်ႁႅၼ်းဝႆႉဢွၼ်တၢင်း)
ၶၵ်ႉတွၼ်ႈၼႆႉ မၼ်းတေလူတူၺ်းထႅဝ် ဢၼ်တႄႇလူၺ်ႈ `#` (မိူၼ်ၼင်ႇ `#include`)။
* မၼ်းတေ "ၵႂႃႇၵျွၵ်ႉဢဝ်" (Copy-Paste) ၶူတ်ႉၼႂ်း Library မႃးသႂ်ႇဝႆႉ။
* မၼ်းတေထွၼ်ပႅတ်ႈ ၶေႃႈမၢႆတွင်း (Comments) ၸိူဝ်းဢၼ်ႁဝ်းတႅမ်ႈဝႆႉၼၼ်ႉၶႃႈ။

### 2. Compilation (ၵၢၼ်ပိၼ်ႇပဵၼ် Assembly)
Compiler တေဢဝ်ၶူတ်ႉ ဢၼ်ႁႅၼ်းဝႆႉယဝ်ႉၼၼ်ႉ ပိၼ်ႇပဵၼ် **Assembly Code**။ 
ၼႆႉပဵၼ်ၽႃႇသႃႇ ဢၼ်ၸမ်တင်း Hardware လိူဝ်ပိူၼ်ႈယဝ်ႉ။ 
ၼႂ်းၶၵ်ႉတွၼ်ႈၼႆႉ မၼ်းတေၵူတ်ႇထတ်းတူၺ်း လွင်ႈၽိတ်း (Syntax Error) ႁဝ်းၶႃႈ။

### 3. Assembly (ၵၢၼ်ပိၼ်ႇပဵၼ် Object Code)
Assembler တေဢဝ် Assembly Code ၼၼ်ႉ ပိၼ်ႇပဵၼ် **Object Code (Machine Code)** ဢၼ်ပဵၼ် 0 လႄႈ 1 ၶႃႈ။ 
ၾၢႆႇဢွၼ်ႇၼႆႉ တေမီးတူဝ်သုတ်းဝႃႈ `.o` ဢမ်ႇၼၼ် `.obj`။ ၵူၺ်းၵႃႈ မၼ်းယင်းဢမ်ႇပႆႇႁဵတ်းၵၢၼ် (Run) လႆႈၶႃႈ။

### 4. Linking (ၵၢၼ်သိုပ်ႇၸူးၵၼ်)
ၼႆႉပဵၼ်ၶၵ်ႉတွၼ်ႈသုတ်း။ Linker တေဢဝ် Object Code ႁဝ်း ၵႂႃႇၸပ်းၸူးတင်း Library ဢၼ်လူဝ်ႇၸႂ်ႉ (မိူၼ်ၼင်ႇ `iostream`) သေယဝ်ႉ ႁဵတ်းပဵၼ်ၾၢႆႇ **Executable (.exe)** ဢၼ်ႁဝ်းပိုတ်ႇလႆႈၼၼ်ႉယဝ်ႉ။



---

### ၵွပ်ႈသင်ႁဝ်းတေလႆႈႁူႉ?
* ပေႃးႁဝ်းမီး Error မိူဝ်ႈ Linker ႁဵတ်းၵၢၼ် (Linker Error)၊ ႁဝ်းတေႁူႉဝႃႈ ႁဝ်းလူဝ်ႇၵႂႃႇၵူတ်ႇထတ်း လွင်ႈၸႂ်ႉတိုဝ်း Library ႁဝ်းၶႃႈ။
* မၼ်းၸွႆႈႁႂ်ႈႁဝ်းပွင်ႇၸႂ်ဝႃႈ ၵွပ်ႈသင် C++ တေလႆႈသၢင်ႈၾၢႆႇ `.exe` ဢွၵ်ႇမႃး။

---

### ၶေႃႈမၼ်ႈၸႂ် (Summary)
1. **Source Code**: ၶူတ်ႉဢၼ်ႁဝ်းတႅမ်ႈ (.cpp)
2. **Preprocessed**: ႁွင်ႉ Library မႃးသႂ်ႇ
3. **Assembly**: ပိၼ်ႇပဵၼ်ၶေႃႈၵႂၢမ်းၸမ် Hardware
4. **Object Code**: ပိၼ်ႇပဵၼ် 0101 (.o)
5. **Executable**: ၾၢႆႇသိုပ်ႇယဝ်ႉတူဝ်ႈ (.exe)
