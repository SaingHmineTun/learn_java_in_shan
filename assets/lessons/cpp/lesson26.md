# Lesson 4.26: The While Loop: Condition-based Repetition

### ၶေႃႈတၢင်ႇလဝ်ႈ (Introduction)
မၢင်ပွၵ်ႈမၢင်လႂ် ႁဝ်းလူဝ်ႇႁႂ်ႈပရူဝ်ႇၶႅမ်ႇ ႁဵတ်းၵၢၼ်သင်မဵဝ်းမဵဝ်း တင်းၼမ် (မိူၼ်ၼင်ႇ ၼေလိၵ်ႈ 100 ထႅဝ်)။ 
ၽိူဝ်သေ တေတႅမ်ႈၶူတ်ႉ 100 ဢၼ်၊ ႁဝ်းၸၢင်ႈၸႂ်ႉ **Loop** တႃႇသင်ႇႁႂ်ႈမၼ်း "ပၼ်ႇ" (Rotate) ႁဵတ်းၵၢၼ်ၵႂႃႇ တေႃႇပေႃးထိုင် တီႈယဝ်ႉမၼ်းၶႃႈ။ 
**`while` loop** ၼႆႉ ပဵၼ် Loop ဢၼ်ငၢႆႈသေပိူၼ်ႈ ၵွပ်ႈမၼ်းတေပၼ်ႇ ႁဵတ်းၵၢၼ်ၵူႈပွၵ်ႈ သင်လွင်ႈတၢင်းမၼ်း တိုၵ်ႉပဵၼ် **`true`** ယူႇၼၼ်ႉယဝ်ႉ။

---

### 1. ပိူင်သၢင်ႈ (Syntax) ၶွင် `while`
မၼ်းတေ "ၵူတ်ႇထတ်း" (Check) လွင်ႈတၢင်းဢွၼ်တၢင်း၊ ပေႃးမႅၼ်ႈ (True) ၸိုင် ၸင်ႇႁဵတ်းၵၢၼ်ၼႂ်း `{ }` ၶႃႈ။

```cpp
while (condition) {
    // ႁဵတ်းၵၢၼ်ၼႂ်းၼႆႈ
    // တေလႆႈမီးၵၢၼ်လႅၵ်ႈလၢႆႈ condition တီႈၼႆႈ (Update)
}
```


---

### 2. ၶၵ်ႉတွၼ်ႈ 3 ပိူင် ဢၼ်လူဝ်ႇလႆႈမီး (Essentials)
တႃႇႁႂ်ႈ Loop ႁဵတ်းၵၢၼ်လႆႈလီ လႄႈ ယႃႇႁႂ်ႈပဵၼ် ပၼ်ႇဢမ်ႇယဝ်ႉ (Infinite Loop)၊ ႁဝ်းလူဝ်ႇလႆႈမီး:
1.  **Initialization**: တင်ႈၵႃႈ Variable တႄႇမၼ်း (မိူၼ်ၼင်ႇ `int i = 1;`)။
2.  **Condition**: လွင်ႈတၢင်းတႃႇတႅပ်းတတ်း (မိူၼ်ၼင်ႇ `i <= 5;`)။
3.  **Update**: ၵၢၼ်ထႅမ်/ယွမ်း ၵႃႈၶၼ် (မိူၼ်ၼင်ႇ `i++;`)။

---

### 3. တူဝ်ယၢင်ႇၶူတ်ႉ (Example Code)

```cpp
#include <iostream>

int main() {
    int count = 1; // 1. Initialization

    // 2. Condition
    while (count <= 5) {
        std::cout << "Count is: " << count << std::endl;
        
        count++; // 3. Update (ပေႃးဢမ်ႇပႃးဢၼ်ၼႆႉ Loop တေဢမ်ႇယဝ်ႉ)
    }

    std::cout << "Loop finished!" << std::endl;
    return 0;
}
```

---

### 4. Infinite Loop (Loop ဢမ်ႇယဝ်ႉ)
ပေႃးႁဝ်းလိုမ်း Update Variable ဢမ်ႇၼၼ် ပၼ် Condition ဢၼ်ပဵၼ် `true` ဝႆႉယူႇသေႇသေႇ၊ 
ပရူဝ်ႇၶႅမ်ႇႁဝ်းတေပၼ်ႇၵႂႃႇ ဢမ်ႇယဝ်ႉဢမ်ႇဝၢႆး။ ၼႆႉၸၢင်ႈႁဵတ်းႁႂ်ႈၶမ်ႇပိဝ်ႇတႃႇ "ၶၢင်း" (Hang) ၶႃႈ။

* **တူဝ်ယၢင်ႇ:** `while (true) { ... }` (ၸႂ်ႉတႃႇပရူဝ်ႇၶႅမ်ႇ ဢၼ်ႁဝ်းၶႂ်ႈႁႂ်ႈႁဵတ်းၵၢၼ်ဝႆႉ ယူႇတႃႇသေႇ)။

---

### ၶေႃႈမၼ်ႈၸႂ် (Summary)
* **`while`**: ၵူတ်ႇထတ်းလွင်ႈတၢင်း **"ဢွၼ်တၢင်း"** (Pre-test loop)။
* ပေႃးလွင်ႈတၢင်းပဵၼ် `false` ၵမ်းလဵဝ်၊ မၼ်းတေဢမ်ႇႁဵတ်းၵၢၼ်သေဢိတ်းၶႃႈ။
* ယႃႇလိုမ်း **Update** Variable ၼႂ်း Loop တႃႇႁႂ်ႈမၼ်းမီးဝၼ်းယဝ်ႉတူဝ်ႈ။
