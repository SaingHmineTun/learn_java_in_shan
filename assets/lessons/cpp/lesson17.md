# Lesson 17: Relational Operators: Comparing Values

### ၶေႃႈတၢင်ႇလဝ်ႈ (Introduction)

တႃႇတေႁဵတ်းႁႂ်ႈ ပရူဝ်ႇၶႅမ်ႇႁဝ်း "တႅပ်းတတ်းၸႂ်" လႆႈၼၼ်ႉ မၼ်းလူဝ်ႇလႆႈမီးၵၢၼ်ၼိူင်းၶေႃႈမုၼ်းၶႃႈ။
မိူၼ်ၼင်ႇ "ၵႃႈငိုၼ်းၼႂ်း ATM မီးၼမ်လိူဝ် ၵႃႈငိုၼ်း ဢၼ်တေထွၼ်ႁႃႉ?" ဢမ်ႇၼၼ် "Password ဢၼ်တႅမ်ႈသႂ်ႇၼၼ်ႉ
မႅၼ်ႈၸွမ်းဢၼ်သိမ်းဝႆႉႁႃႉ?" ၼႆယဝ်ႉ။
ၵၢၼ်ၼိူင်းၸိူဝ်းၼႆႉ တေဢွၵ်ႇၽွၼ်းမႃးပဵၼ် **`bool`** (`true` ဢမ်ႇၼၼ် `false`) သေႇသေႇၶႃႈ။

---

### 1. တူဝ်ၼိူင်းပိုၼ်ႉထၢၼ် (Relational Operators)

ၼႂ်း C++ မီးတူဝ်ၼိူင်းယူႇ 6 မဵဝ်းလူင်ၶႃႈ:

| Operator | တီႈပွင်ႇ (Meaning)         | တူဝ်ယၢင်ႇ (a=10, b=5) | ၽွၼ်းဢွၵ်ႇ |
|:---------|:---------------------------|:----------------------|:-----------|
| **`==`** | ထဝ်ႇၵၼ် (Equal to)         | `a == b`              | false      |
| **`!=`** | ဢမ်ႇထဝ်ႇၵၼ် (Not equal to) | `a != b`              | true       |
| **`>`**  | ယႂ်ႇလိူဝ် (Greater than)   | `a > b`               | true       |
| **`<`**  | လဵၵ်ႉလိူဝ် (Less than)     | `a < b`               | false      |
| **`>=`** | ယႂ်ႇလိူဝ် ဢမ်ႇၼၼ် ထဝ်ႇၵၼ်  | `a >= 10`             | true       |
| **`<=`** | လဵၵ်ႉလိူဝ် ဢမ်ႇၼၼ် ထဝ်ႇၵၼ် | `a <= 5`              | false      |

---

### 2. လွင်ႈလုမ်းၼႃး (Common Mistakes)

* **`==` vs `=`**: ၼႆႉပဵၼ်တီႈ ၽိတ်းၼမ်သေပိူၼ်ႈၶႃႈ!
    * **`=`** (Single equal): ၸႂ်ႉတႃႇ "ပၼ်ၵႃႈၶၼ်" (Assignment).
    * **`==`** (Double equal): ၸႂ်ႉတႃႇ "ၼိူင်းတူၺ်း" (Comparison).
* **Floating-point Comparison**: ၵၢၼ်ဢဝ် `double` မႃးၼိူင်းၸွမ်း `==` ၼႆႉ မၼ်းၸၢင်ႈ မီးတီႈၽိတ်းၶႃႈ။ ၵွပ်ႈဝႃႈ
  တူဝ်ၸုတ်ႇ (တူဝ်ၼပ်ႉဢၼ်မီးၽၢႆႇလင်မၢႆၸုတ်ႇ) မၼ်းၸၢင်ႈမီး လွင်ႈပႅၵ်ႇပိူင်ႈဢွၼ်ႇဢွၼ်ႇ။

---

### 3. တူဝ်ယၢင်ႇၶူတ်ႉ (Example Code)

```cpp
#include <iostream>

int main() {
    int myAge = 25;
    int votingAge = 18;

    // ၸၢမ်းၼိူင်းတူၺ်း
    bool canVote = (myAge >= votingAge);
    bool isExactly18 = (myAge == 18);
    bool isNot18 = (myAge != 18);

    std::cout << std::boolalpha; // တႃႇႁႂ်ႈၼေ true/false
    
    std::cout << "Can I vote? " << canVote << std::endl;
    std::cout << "Am I exactly 18? " << isExactly18 << std::endl;
    std::cout << "Am I not 18? " << isNot18 << std::endl;

    return 0;
}
```

---

### ၶေႃႈမၼ်ႈၸႂ် (Summary)

* ၽွၼ်းဢွၵ်ႇ ၵၢၼ်ၼိူင်း တေပဵၼ် **`true` (1)** ဢမ်ႇၼၼ် **`false` (0)** သေႇသေႇ။
* ၸႂ်ႉ **`==`** တႃႇၵူတ်ႇထတ်း လွင်ႈမိူၼ်ၵၼ်။
* ၸႂ်ႉ **`!=`** တႃႇၵူတ်ႇထတ်း လွင်ႈဢမ်ႇမိူၼ်ၵၼ်။
