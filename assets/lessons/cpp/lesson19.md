# Lesson 3.19: Compound Assignment Operators

### ၶေႃႈတၢင်ႇလဝ်ႈ (Introduction)
မၢင်ပွၵ်ႈမၢင်လႂ် ႁဝ်းလူဝ်ႇၼပ်ႉသွၼ်ႇ Variable ဢၼ်မီးဝႆႉယဝ်ႉၼၼ်ႉသေ သႂ်ႇၶိုၼ်းၼႂ်းတူဝ်မၼ်း (မိူၼ်ၼင်ႇ ၵၢၼ်ထႅမ်တူဝ်ၼပ်ႉ Score ၼႂ်းၵဵမ်း)။ 
ၽိူဝ်ႇတႃႉတေတႅမ်ႈယၢဝ်းယၢဝ်း၊ C++ မီးလၢႆးတႅမ်ႈဢၼ်ပွတ်းသေ ႁွင်ႉဝႃႈ **Compound Assignment** ၶႃႈ။

---

### 1. တူဝ်ၼပ်ႉသွၼ်ႇႁူမ်ႈ (Compound Operators)
ၼႆႉပဵၼ်ၵၢၼ်ဢဝ် Arithmetic Operator (`+`, `-`, `*`, `/`) မႃးႁူမ်ႈၸွမ်း `=` ၶႃႈ:

| Operator | တူဝ်ယၢင်ႇ (Short) | တီႈပွင်ႇ (Long form) | တူဝ်ယၢင်ႇ (x=10) | ၽွၼ်းဢွၵ်ႇ |
| :--- | :--- | :--- | :--- | :--- |
| **`+=`** | `x += 5` | `x = x + 5` | `10 + 5` | 15 |
| **`-=`** | `x -= 3` | `x = x - 3` | `10 - 3` | 7 |
| **`*=`** | `x *= 2` | `x = x * 2` | `10 * 2` | 20 |
| **`/=`** | `x /= 2` | `x = x / 2` | `10 / 2` | 5 |
| **`%=`** | `x %= 3` | `x = x % 3` | `10 % 3` | 1 |



---

### 2. ပဵၼ်သင်ႁဝ်းတေလႆႈၸႂ်ႉ?
1.  **Readability (လူငၢႆႈ)**: ၶူတ်ႉႁဝ်းတေပွတ်းသေ လူငၢႆႈလိူဝ်ၵဝ်ႇ။
2.  **Less Errors**: ႁဝ်းဢမ်ႇလူဝ်ႇတႅမ်ႈၸိုဝ်ႈ Variable သွင်ပွၵ်ႈ၊ ႁႄႉၵင်ႈလွင်ႈတႅမ်ႈၸိုဝ်ႈၽိတ်းၶႃႈ။

---

### 3. တူဝ်ယၢင်ႇၶူတ်ႉ (Example Code)

```cpp
#include <iostream>

int main() {
    int score = 100;
    
    // ထႅမ် Score ၶိုၼ်ႈ 50
    score += 50;  // score = 150
    
    // ယွမ်း Score လူင်း 20
    score -= 20;  // score = 130
    
    // ပၼ်ႇ (Multiplication) သွင်ပၼ်ႇ
    score *= 2;   // score = 260
    
    std::cout << "Final Score: " << score << std::endl;

    return 0;
}
```

### ၶေႃႈမၼ်ႈၸႂ် (Summary)
* ၸႂ်ႉ **`+=`**, **`-=`**, **`*=`** တႃႇၼပ်ႉသွၼ်ႇသေ သိမ်းၶိုၼ်းၼႂ်း Variable တူဝ်ၵဝ်ႇ။
