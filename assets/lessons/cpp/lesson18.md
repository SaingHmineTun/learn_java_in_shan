# Lesson 18: Logical Operators: &&, ||, and !

### ၶေႃႈတၢင်ႇလဝ်ႈ (Introduction)
ၼႂ်း Lesson ပူၼ်ႉမႃး ႁဝ်းၼိူင်းၶေႃႈမုၼ်းလႆႈ ပွတ်းၼိုင်ႈ (မိူၼ်ၼင်ႇ `age > 18`)။ 
ၵူၺ်းၵႃႈ ပေႃးႁဝ်းလူဝ်ႇၵူတ်ႇထတ်း လွင်ႈတၢင်းလၢႆလၢႆပိူင် ၸိူဝ်ႉၼင်ႇ "ပေႃး **User** ပဵၼ် **Admin** **လႄႈ** **Password** မႅၼ်ႈ" ၸိုင် 
ႁဝ်းလူဝ်ႇလႆႈၸႂ်ႉ **Logical Operators** တႃႇဢဝ်ၶေႃႈၼိူင်းၸိူဝ်းၼၼ်ႉ မႃးမတ်ႉၸပ်းၵၼ်ၶႃႈ။

---

### 1. တူဝ်ၼပ်ႉသွၼ်ႇ Logic (Logical Operators)
ၼႂ်း C++ မီးယူႇ 3 မဵဝ်းလူင်ၶႃႈ:

| Operator   | ၸိုဝ်ႈႁွင်ႉ | တီႈပွင်ႇ (Meaning) | တူဝ်ယၢင်ႇ                       |
|:-----------| :--- | :--- |:--------------------------------|
| **`&&`**   | **AND** | မႅၼ်ႈ (True) ပေႃး **တင်းသွင်ၽၢႆႇ** မႅၼ်ႈ | `(5 > 2) && (10 > 5)` -> true   |
| **`\|\|`** | **OR** | မႅၼ်ႈ (True) ပေႃး **မီးၽၢႆႇၼိုင်ႈ** မႅၼ်ႈ | `(5 > 10) \|\| (2 > 1)` -> true |
| **`!`**    | **NOT** | **ပိၼ်ႇၽၢႆႇ** (မႅၼ်ႈ ပဵၼ် ၽိတ်း / ၽိတ်း ပဵၼ် မႅၼ်ႈ) | `!(5 > 2)` -> false             |

---

### 2. Short-Circuit Evaluation (ၵၢၼ်တတ်းသဵၼ်ႈတၢင်း)
ၼႆႉပဵၼ် လွင်ႈလၢၵ်ႇလၢႆး C++ ဢၼ်ႁဵတ်းႁႂ်ႈပရူဝ်ႇၶႅမ်ႇ ဝႆးၶႃႈ:
* **`&&` (AND)**: ပေႃးၽၢႆႇသၢႆႉ **ၽိတ်း (False)** ယဝ်ႉ၊ C++ တေဢမ်ႇၵႂႃႇလူ ၽၢႆႇၶႂႃထႅင်ႈ၊ ၵွပ်ႈဝႃႈ ၽွၼ်းမၼ်းတေလႆႈ ၽိတ်း (False) သေႇသေႇယဝ်ႉ။
* **`||` (OR)**: ပေႃးၽၢႆႇသၢႆႉ **မႅၼ်ႈ (True)** ယဝ်ႉ၊ မၼ်းတေဢမ်ႇၵႂႃႇလူ ၽၢႆႇၶႂႃထႅင်ႈ၊ ၵွပ်ႈဝႃႈ ၽွၼ်းမၼ်းတေလႆႈ မႅၼ်ႈ (True) သေႇသေႇယဝ်ႉ။

---

### 3. တူဝ်ယၢင်ႇၶူတ်ႉ (Example Code)

```cpp
#include <iostream>

int main() {
    bool isStudent = true;
    int age = 20;

    // ၸၢမ်းၸႂ်ႉ AND (&&) - တေလႆႈမႅၼ်ႈ တင်းသွင်ဢၼ်
    bool canGetDiscount = (isStudent && age < 25); 

    // ၸၢမ်းၸႂ်ႉ OR (||) - မႅၼ်ႈဢၼ်ၼိုင်ႈၵေႃႈလႆႈ
    bool isSpecialCase = (age < 10 || age > 60);

    // ၸၢမ်းၸႂ်ႉ NOT (!) - ပိၼ်ႇၽၢႆႇ
    bool isNotStudent = !isStudent;

    std::cout << std::boolalpha;
    std::cout << "Can get discount: " << canGetDiscount << std::endl;
    std::cout << "Is special case: " << isSpecialCase << std::endl;
    std::cout << "Is NOT a student: " << isNotStudent << std::endl;

    return 0;
}
```

---

### ၶေႃႈမၼ်ႈၸႂ် (Summary)
* **`&&`**: လူဝ်ႇလႆႈ "မႅၼ်ႈ" တင်းမူတ်း။
* **`||`**: လူဝ်ႇလႆႈ "မႅၼ်ႈ" ဢၼ်ၼိုင်ႈၵေႃႈလႆႈ။
* **`!`**: ၸႂ်ႉတႃႇပိၼ်ႇၵႃႈၶၼ် Boolean။
