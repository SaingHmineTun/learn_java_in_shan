# Lesson 20: The Mystery of Increment and Decrement (++i vs i++)

### ၶေႃႈတၢင်ႇလဝ်ႈ (Introduction)
ၵၢၼ်ထႅမ်တူဝ်ၼပ်ႉ 1 တူဝ် (`x = x + 1`) ၼၼ်ႉ ႁဝ်းၸၢင်ႈတႅမ်ႈလႆႈငၢႆႈငၢႆႈလူၺ်ႈၸႂ်ႉ **Increment Operator (`++`)** ၶႃႈ။ 
ၵူၺ်းၵႃႈ ယူႇတီႈႁဝ်းဢဝ် ဝႆႉၽၢႆႇၼႃႈ (`++i`) ဢမ်ႇၼၼ် ဝႆႉၽၢႆႇလင် (`i++`) ၼၼ်ႉ မၼ်းမီးလွင်ႈပႅၵ်ႇပိူင်ႈၵၼ်တႄႉတႄႉ တီႈၵၢၼ်ၼပ်ႉသွၼ်ႇၶႃႈ။

---

### 1. Prefix Increment (`++i`)
**Prefix** တေဝႆႉတူဝ် `++` ဝႆႉ **ၽၢႆႇၼႃႈ** Variable ၶႃႈ။
* **ၵၢၼ်ႁဵတ်း**: မၼ်းတေ "ထႅမ်တူဝ်ၼပ်ႉဢွၼ်တၢင်း" (Increment first) သေယဝ်ႉ ၸင်ႇဢဝ်ၵႃႈၶၼ်မၼ်းၵႂႃႇၸႂ်ႉၶႃႈ။

```cpp
int x = 5;
int y = ++x; // x တေပဵၼ် 6 ဢွၼ်တၢင်း၊ သေယဝ်ႉ y ၸင်ႇပဵၼ် 6 ၸွမ်း
```

---

### 2. Postfix Increment (`i++`)
**Postfix** တေဝႆႉတူဝ် `++` ဝႆႉ **ၽၢႆႇလင်** Variable ၶႃႈ။
* **ၵၢၼ်ႁဵတ်း**: မၼ်းတေ "ဢဝ်ၵႃႈၶၼ်ၵဝ်ႇၵႂႃႇၸႂ်ႉဢွၼ်တၢင်း" (Use old value first) သေယဝ်ႉ ၸင်ႇၵႂႃႇထႅမ်တူဝ်ၼပ်ႉမၼ်း ဝၢႆးလင်ၶႃႈ။

```cpp
int x = 5;
int y = x++; // y တေဢဝ် 5 ၵႂႃႇဢွၼ်တၢင်း၊ သေယဝ်ႉ x ၸင်ႇပဵၼ် 6 ဝၢႆးလင်
```



---

### 3. Comparison Table ( Increment & Decrement )

| Operator | ၸိုဝ်ႈႁွင်ႉ | ၵၢၼ်ႁဵတ်း (Step-by-step) |
| :--- | :--- | :--- |
| **`++i`** | **Prefix** | ထႅမ် 1 ဢွၼ်တၢင်း -> ၸင်ႇဢဝ်ၵႃႈၶၼ်မႂ်ႇၵႂႃႇၸႂ်ႉ |
| **`i++`** | **Postfix** | ဢဝ်ၵႃႈၶၼ်ၵဝ်ႇၵႂႃႇၸႂ်ႉဢွၼ်တၢင်း -> ၸင်ႇထႅမ် 1 ဝၢႆးလင် |
| **`--i`** | **Prefix Dec** | ယွမ်း 1 ဢွၼ်တၢင်း -> ၸင်ႇဢဝ်ၵႃႈၶၼ်မႂ်ႇၵႂႃႇၸႂ်ႉ |
| **`i--`** | **Postfix Dec** | ဢဝ်ၵႃႈၶၼ်ၵဝ်ႇၵႂႃႇၸႂ်ႉဢွၼ်တၢင်း -> ၸင်ႇယွမ်း 1 ဝၢႆးလင် |

---

### 4. တူဝ်ယၢင်ႇၶူတ်ႉ (Example Code)

```cpp
#include <iostream>

int main() {
    int a = 10;
    int b = 10;

    // ၸၢမ်း Prefix
    std::cout << "Prefix (++a): " << ++a << std::endl; // တေၼေ 11
    std::cout << "After Prefix, a is: " << a << std::endl;

    // ၸၢမ်း Postfix
    std::cout << "Postfix (b++): " << b++ << std::endl; // တေၼေ 10 (ၵႃႈၵဝ်ႇ)
    std::cout << "After Postfix, b is: " << b << std::endl; // တေပဵၼ် 11 ဝၢႆးလင်

    return 0;
}
```

---

### ၶေႃႈမၼ်ႈၸႂ် (Summary)
* ၸႂ်ႉ **`++i`** ပေႃးလူဝ်ႇထႅမ်ၵႃႈၶၼ် ဢွၼ်တၢင်းတေဢဝ်ၵႂႃႇၸႂ်ႉ။
* ၸႂ်ႉ **`i++`** ပေႃးလူဝ်ႇဢဝ်ၵႃႈၶၼ်ၵဝ်ႇၵႂႃႇၸႂ်ႉ ဢွၼ်တၢင်းမၼ်းတေလႅၵ်ႈလၢႆႈ။
* ၼႂ်း **Loops** (ဢၼ်ႁဝ်းတေသွၼ်ၼႂ်း Module 4)၊ Programmer ၵမ်ႉၼမ်ၵေႃႈၸႂ်ႉ `++i` ၵွပ်ႈမၼ်းႁဵတ်းၵၢၼ်ဝႆးလိူဝ်ဢိတ်းၼိုင်ႈ ၼႂ်းပိူင်သၢင်ႈ C++ ၶႃႈ။
