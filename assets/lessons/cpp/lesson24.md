# Lesson 24: Making Decisions: The If, Else If, and Else ladder

### ၶေႃႈတၢင်ႇလဝ်ႈ (Introduction)
မိူဝ်ႈႁဝ်းတႅမ်ႈပရူဝ်ႇၶႅမ်ႇ၊ ႁဝ်းဢမ်ႇၶႂ်ႈႁႂ်ႈမၼ်းႁဵတ်းၵၢၼ် ထႅဝ်ၼိုင်ႈယဝ်ႉထႅဝ်ၼိုင်ႈ ၼိူဝ်လူင်းတႂ်ႈၵူၺ်းၶႃႈ။ 
ႁဝ်းၶႂ်ႈႁႂ်ႈမၼ်း "လိူၵ်ႈႁဵတ်း" ၸွမ်းၼင်ႇ လွင်ႈတၢင်း (Condition) ဢၼ်ၵိူတ်ႇပဵၼ်မႃး။ 
တူဝ်ယၢင်ႇ - "ပေႃး Password မႅၼ်ႈၸိုင် ႁႂ်ႈၶဝ်ႈလႆႈ၊ ပေႃးဢမ်ႇမႅၼ်ႈၸိုင် ႁႂ်ႈၼေ Error" ၼႆယဝ်ႉ။

---

### 1. The `if` Statement (ၵၢၼ်ၵူတ်ႇထတ်းပိုၼ်ႉထၢၼ်)
`if` ၸႂ်ႉတႃႇၵူတ်ႇထတ်းဝႃႈ လွင်ႈတၢင်းၼၼ်ႉ "မႅၼ်ႈ" (True) ႁႃႉ? ပေႃးမႅၼ်ႈ ႁဵတ်းၵၢၼ်ၼႂ်း `{ }` ၼၼ်ႉၶႃႈ။

```cpp
if (condition) {
    // ႁဵတ်းၵၢၼ်တီႈၼႆႈ ပေႃး condition ပဵၼ် true
}
```

---

### 2. The `if-else` (လိူၵ်ႈပႆသွင်တၢင်း)
ပေႃးလွင်ႈတၢင်းၼၼ်ႉ "ၽိတ်း" (False) ၸိုင် ႁဝ်းၸၢင်ႈၸႂ်ႉ `else` တႃႇႁပ်ႉႁဵတ်းၵၢၼ် ထႅင်ႈၽၢႆႇၼိုင်ႈၶႃႈ။

```cpp
if (age >= 18) {
    std::cout << "You can vote!";
} else {
    std::cout << "You are too young.";
}
```



---

### 3. The `if - else if - else` Ladder (လိူၵ်ႈပႆလၢႆလၢႆတၢင်း)
မိူဝ်ႈႁဝ်းမီး လွင်ႈတၢင်းတင်းၼမ် ဢၼ်တေလႆႈလိူၵ်ႈ၊ ႁဝ်းၸၢင်ႈၸႂ်ႉ `else if` မႃးသွၼ်ႉၵၼ် ပဵၼ်ၶၵ်ႉပဵၼ်ၸၼ်ႉ (Ladder) ၶႃႈ။

```cpp
int score = 85;

if (score >= 90) {
    std::cout << "Grade: A";
} else if (score >= 80) {
    std::cout << "Grade: B"; // တေႁဵတ်းၵၢၼ်တီႈၼႆႈ
} else if (score >= 70) {
    std::cout << "Grade: C";
} else {
    std::cout << "Grade: F";
}
```

---

### 4. ၵၢၼ်ၸႂ်ႉ Logic တီႈၼႂ်း `if`
ႁဝ်းၸၢင်ႈဢဝ် **Logical Operators (`&&`, `||`)** မႃးၸႂ်ႉၼႂ်း Condition လႆႈၶႃႈ:
```cpp
if (age >= 18 && hasID == true) {
    std::cout << "Entry Allowed";
}
```

---

### ၶေႃႈမၼ်ႈၸႂ် (Summary)
* **`if`**: ၵူတ်ႇထတ်းလွင်ႈတၢင်းဢွၼ်တၢင်းသုတ်း။
* **`else if`**: ၵူတ်ႇထတ်းလွင်ႈတၢင်းတၢင်ႇဢၼ် ပေႃးဢၼ်ၼိူဝ်မၼ်း ၽိတ်း (False)။
* **`else`**: ႁဵတ်းၵၢၼ်တွၼ်ႈသုတ်း ပေႃးလွင်ႈတၢင်း တင်းမူတ်းၼၼ်ႉ ၽိတ်း (False)။
* ၶမ်ႇပိဝ်ႇတႃႇတေႁဵတ်းၵၢၼ် ၵူၺ်း **"တွၼ်ႈလဵဝ်"** (Block) ဢၼ်မႅၼ်ႈဢွၼ်တၢင်းသုတ်းၼၼ်ႉၶႃႈ။
