# Lesson 5: The Anatomy of C++ Code

### ၶေႃႈတၢင်ႇလဝ်ႈ (Introduction)
ပရူဝ်ႇၶႅမ်ႇ C++ ၵူႈဢၼ် တေလႆႈမီး ပိူင်သၢင်ႈ (Structure) ဢၼ်ၵႅၼ်ႇၶႅင်။ 
ပေႃးႁဝ်းပွင်ႇၸႂ် ၵမ်းထႅဝ် ၵမ်းထႅဝ်ယဝ်ႉ ႁဝ်းတေတႅမ်ႈၶူတ်ႉလႆႈ လီလိူဝ်ၵဝ်ႇၶႃႈ။

---

### 1. Preprocessor Directive
```cpp
#include <iostream>
```
* **သင်ႇၵေတ်ႉ:** မၼ်းတႄႇလူၺ်ႈ `#` (Hashtag)။
* **တီႈပွင်ႇ:** ၼႆႉပဵၼ်ၵၢၼ် "ႁွင်ႉဢဝ်" ၶိူင်ႈမိုဝ်းဢၼ်မီးဝႆႉယဝ်ႉၼၼ်ႉမႃးၸႂ်ႉ။
* **`<iostream>`**: ပဵၼ် Input/Output Stream တႃႇတေၼေလိၵ်ႈဢွၵ်ႇ (Output) လႄႈ ႁပ်ႉဢဝ်လိၵ်ႈၶဝ်ႈ (Input) ၼၼ်ႉယဝ်ႉ။

---

### 2. The Main Function
```cpp
int main() {
    // ... code ...
}
```
* **Entry Point:** ၼႆႉပဵၼ် "ပၢၵ်ႇတူ" ဢွၼ်ႇတၢင်းသုတ်း ဢၼ်ပရူဝ်ႇၶႅမ်ႇတေတႄႇႁဵတ်းၵၢၼ်။
* **`int`**: တီႈပွင်ႇဝႃႈ မိူဝ်ႈပရူဝ်ႇၶႅမ်ႇယဝ်ႉတူဝ်ႈၼၼ်ႉ မၼ်းတေသူင်ႇပၼ် "တူဝ်ၼပ်ႉ" (Integer) ၶိုၼ်းပၼ် OS။
* **`{ }`**: လိၵ်ႈဢၼ်မီးၼႂ်း ပႅတ်ႇၵႅတ်ႉ (Curly braces) ၼႆႉ ႁွင်ႉဝႃႈ **Function Body** ၶႃႈ။

---

### 3. Statements & Namespaces
```cpp
std::cout << "Hello, World!";
```
* **`std`**: ပဵၼ် Standard Namespace။
* **`cout`**: (Character Output) တႃႇၼေလိၵ်ႈဢွၵ်ႇၼႃႈၸေႃး။
* **`<<`**: ႁွင်ႉဝႃႈ Insertion Operator။ မၼ်းတေ "သူင်ႇ" လိၵ်ႈၵႂႃႇၸူး `cout` ၶႃႈ။
* **`;`**: (Semicolon) ၼႆႉလၢၵ်ႇလၢႆးတႄႉတႄႉ။ မၼ်းပဵၼ် တူဝ်သုတ်းထႅဝ်။ ပေႃးဢမ်ႇပႃးမၼ်း Compiler တေဢမ်ႇပွင်ႇၸႂ်ၶႃႈ။



---

### 4. Ending the Program
```cpp
return 0;
```
* **တီႈပွင်ႇ:** ပေႃးပရူဝ်ႇၶႅမ်ႇႁဵတ်းၵၢၼ် ထိုင်ထႅဝ်ၼႆႉယဝ်ႉ တီႈပွင်ႇဝႃႈ "ၵၢၼ်ယဝ်ႉတူဝ်ႈလီငၢမ်း (Success)" ၼႆၶႃႈ။

---

### ၶေႃႈမၼ်ႈၸႂ် (Summary)
* **`#include`**: ႁွင်ႉၸႂ်ႉ Library။
* **`int main()`**: တီႈတႄႇႁဵတ်းၵၢၼ်။
* **`std::cout`**: ၼေလိၵ်ႈဢွၵ်ႇ။
* **`;`**: တူဝ်သုတ်းထႅဝ်ၶေႃႈၵႂၢမ်း။
