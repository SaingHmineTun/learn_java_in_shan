# Lesson 22: Expressions and Statement Evaluation

### ၶေႃႈတၢင်ႇလဝ်ႈ (Introduction)
ၵူႈထႅဝ်လိၵ်ႈ ဢၼ်ႁဝ်းတႅမ်ႈၼႂ်း C++ ၼၼ်ႉ မၼ်းမီးလွင်ႈပႅၵ်ႇပိူင်ႈၵၼ်ယူႇၶႃႈ။ 
ပေႃးႁဝ်းပွင်ႇၸႂ်ဝႃႈ ထႅဝ်လႂ်ပဵၼ် "ၵၢၼ်ၼပ်ႉသွၼ်ႇ" (Expression) လႄႈ ထႅဝ်လႂ်ပဵၼ် "ၶေႃႈၵႂၢမ်းသင်ႇ" (Statement) ၸိုင် 
ႁဝ်းတေတႅမ်ႈၶူတ်ႉလႆႈ ထႅတ်းလႅတ်း (Precise) လိူဝ်ၵဝ်ႇၶႃႈယဝ်ႉ။

---

### 1. Expression (ၵၢၼ်ၼပ်ႉသွၼ်ႇ)
**Expression** ၼႆႉ ပဵၼ်တွၼ်ႈၶူတ်ႉ ဢၼ်ၼပ်ႉသွၼ်ႇယဝ်ႉသေ **"သူင်ႇၵႃႈၶၼ်" (Returns a value)** ဢွၵ်ႇမႃးပၼ်ၶႃႈ။

* **တူဝ်ယၢင်ႇ:**
    * `5 + 2` (သူင်ႇၵႃႈၶၼ် 7)
    * `x * y` (သူင်ႇၵႃႈၶၼ် ၽွၼ်းၵုၼ်း)
    * `a == b` (သူင်ႇၵႃႈၶၼ် `true` ဢမ်ႇၼၼ် `false`)



---

### 2. Statement (ၶေႃႈၵႂၢမ်းသင်ႇ)
**Statement** ၼႆႉ ပဵၼ် "ၶၵ်ႉတွၼ်ႈၵၢၼ်ႁဵတ်း" ဢၼ်ယဝ်ႉတူဝ်ႈၼႂ်းတူဝ်မၼ်း။ 
မၼ်းဢမ်ႇသူင်ႇၵႃႈၶၼ်ပၼ်ၽႂ်၊ ၵူၺ်းၵႃႈ မၼ်းသင်ႇႁႂ်ႈၶမ်ႇပိဝ်ႇတႃႇ "ႁဵတ်းၵၢၼ်သင်မဵဝ်းမဵဝ်း" ၶႃႈ။ 
ၼႂ်း C++၊ Statement ၵမ်ႉၼမ် တေလႆႈသုတ်းလူၺ်ႈ **Semicolon (`;`)** သေႇသေႇၶႃႈ။

* **တူဝ်ယၢင်ႇ:**
    * `int x = 10;` (Declaration Statement)
    * `x = 5 + 2;` (Assignment Statement - ဢဝ် Expression `5+2` သႂ်ႇၼႂ်း Variable)
    * `std::cout << x;` (Output Statement)

---

### 3. Expression Statement
ၼႆႉပဵၼ် Expression ဢၼ်ႁဝ်းသႂ်ႇ `;` ဝႆႉတီႈသုတ်းမၼ်း တႃႇႁႂ်ႈမၼ်းပဵၼ် Statement တူဝ်ၼိုင်ႈၶႃႈ။
* `x = 10;` -> ၼႆႉပဵၼ် Expression `x = 10` ဢၼ်သူင်ႇၵႃႈၶၼ် 10၊ ပေႃးသႂ်ႇ `;` မၼ်းၵေႃႈပဵၼ် Statement ၵမ်းသိုဝ်ႈၶႃႈ။

---

### 4. Evaluation Sequence (လွင်ႈၼပ်ႉသွၼ်ႇၸွမ်းလำတပ်ႉ)
မိူဝ်ႈ C++ ႁူပ်ႉထူပ်း Expression ဢၼ်ယၢဝ်းယၢဝ်း၊ မၼ်းတေၸႂ်ႉပၵ်းယၢမ်ႇ ဢၼ်ႁဝ်းသွၼ်မႃးၼႂ်း **Lesson 16** (Precedence & Associativity) တႃႇတႅပ်းတတ်းဝႃႈ တေၼပ်ႉသွၼ်ႇၵႃႈၶၼ် (Value) ဢၼ်လႂ်ဢွၼ်တၢင်းၼၼ်ႉၶႃႈ။

**တူဝ်ယၢင်ႇၵၢၼ်ၼပ်ႉသွၼ်ႇ (Evaluation):**
`int result = 5 + 2 * 10;`
1. Evaluate `2 * 10` -> လႆႈ 20
2. Evaluate `5 + 20` -> လႆႈ 25
3. Assign `25` ၵႂႃႇၼႂ်း Variable `result`။

---

### ၶေႃႈမၼ်ႈၸႂ် (Summary)
* **Expression**: ၼပ်ႉသွၼ်ႇသေ **သူင်ႇၵႃႈၶၼ်** (မိူၼ်ၼင်ႇ `a + b`)။
* **Statement**: ပဵၼ် **ၶေႃႈၵႂၢမ်းသင်ႇ** ဢၼ်ယဝ်ႉတူဝ်ႈ (မိူၼ်ၼင်ႇ `int x = 5;`)။
* **Semicolon (`;`)**: ၸႂ်ႉတႃႇတူဝ်သုတ်း Statement ၵူႈဢၼ်။
