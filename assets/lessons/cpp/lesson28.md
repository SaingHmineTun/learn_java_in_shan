# Lesson 28: The Do-While Loop: Guaranteed Execution

### ၶေႃႈတၢင်ႇလဝ်ႈ (Introduction)
ၼႂ်း `while` လႄႈ `for` loop ၼၼ်ႉ၊ ပေႃးလွင်ႈတၢင်း (Condition) ၽိတ်း (False) တီႈတေႇမၼ်း၊ 
မၼ်းတေဢမ်ႇႁဵတ်းၵၢၼ်သင် သေဢိတ်းၶႃႈ။ ၵူၺ်းၵႃႈ မၢင်ပွၵ်ႈလႂ် ႁဝ်းလူဝ်ႇႁႂ်ႈပရူဝ်ႇၶႅမ်ႇ **"ႁဵတ်းၵၢၼ်ဢွၼ်တၢင်း 1 ပွၵ်ႈ"** သေႇသေႇ (မိူၼ်ၼင်ႇ ၵၢၼ်ၼေ Menu ႁႂ်ႈၵူၼ်းလိူၵ်ႈ)၊ 
ဝၢႆးၼၼ်ႉၸင်ႇၵူတ်ႇထတ်းဝႃႈ တေပၼ်ႇႁဵတ်းထႅင်ႈႁႃႉ? ၼႆယဝ်ႉ။ ၼႆႉပဵၼ်ၼႃႈၵၢၼ်ၶွင် **`do-while`** ၶႃႈ။

---

### 1. ပိူင်သၢင်ႈ (Syntax) ၶွင် `do-while`
တီႈၼႆႈ Condition မၼ်းၵႂႃႇမီးဝႆႉ **ၽၢႆႇတႂ်ႈသုတ်း** ၶႃႈ:

```cpp
do {
    // ႁဵတ်းၵၢၼ်ၼႂ်းၼႆႈဢွၼ်တၢင်း 1 ပွၵ်ႈ
} while (condition); // *** လူဝ်ႇလႆႈမီး Semicolon (;) တီႈၼႆႈသေႇသေႇ
```



---

### 2. လွင်ႈပႅၵ်ႇပိူင်ႈ (Key Difference)
* **`while` loop**: ၵူတ်ႇထတ်းဢွၼ်တၢင်း (Entry-controlled)။ ပေႃးၽိတ်း တေဢမ်ႇႁဵတ်းသင်။
* **`do-while` loop**: ၵူတ်ႇထတ်းဝၢႆးလင် (Exit-controlled)။ မၼ်းတေႁဵတ်းၵၢၼ် **"တီႈဢေႇသုတ်ႉ 1 ပွၵ်ႈ"** သေႇသေႇ၊ ဢမ်ႇဝႃႈလွင်ႈတၢင်းတေႇမၼ်း တေၽိတ်းယူႇၵေႃႈယဝ်ႉ။

---

### 3. တူဝ်ယၢင်ႇၶူတ်ႉ (Example Code)
ၸၢမ်းတူၺ်းပရူဝ်ႇၶႅမ်ႇ ဢၼ်ထၢမ် Password တေႃႇပေႃးမႅၼ်ႈၼၼ်ႉၶႃႈ:

```cpp
#include <iostream>

int main() {
    int secretCode = 1234;
    int userInput;

    do {
        std::cout << "Enter the secret code: ";
        std::cin >> userInput; // ႁဵတ်းၵၢၼ်တွၼ်ႈၼႆႉဢွၼ်တၢင်း

    } while (userInput != secretCode); // ပေႃး Password ဢမ်ႇမႅၼ်ႈ ၸိုင်ပၼ်ႇၶိုၼ်း

    std::cout << "Access Granted!" << std::endl;
    return 0;
}
```

---

### 4. ပဵၼ်သင်ႁဝ်းတေလႆႈၸႂ်ႉ?
1.  **Menu Selection**: မိူဝ်ႈပိုတ်ႇပရူဝ်ႇၶႅမ်ႇ ႁဝ်းၶႂ်ႈၼေ Menu ဢွၼ်တၢင်း ၸင်ႇႁႂ်ႈၵူၼ်းလိူၵ်ႈ။
2.  **Input Validation**: ႁပ်ႉဢဝ်ၶေႃႈမုၼ်းဢွၼ်တၢင်း သေယဝ်ႉၸင်ႇၵူတ်ႇထတ်းဝႃႈ ၶေႃႈမုၼ်းၼၼ်ႉ မႅၼ်ႈၸွမ်းပၵ်းယၢမ်ႇႁႃႉ?

---

### ၶေႃႈမၼ်ႈၸႂ် (Summary)
* **`do-while`**: ႁဵတ်းၵၢၼ်ဢွၼ်တၢင်း ၸင်ႇၵူတ်ႇထတ်း (Guaranteed execution).
* တေလႆႈမီး **`;`** တီႈသုတ်းထႅဝ် `while (condition);`။
* ၸႂ်ႉတႃႇၵၢၼ်ႁပ်ႉဢဝ် Input ဢၼ်လူဝ်ႇလႆႈမီးၵၢၼ်ၸၢမ်းႁဵတ်း ဢွၼ်တၢင်းတေၵူတ်ႇထတ်းၶႃႈ။
