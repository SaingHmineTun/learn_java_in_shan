# Lesson 2: Compilers vs. Interpreters

### ၶေႃႈတၢင်ႇလဝ်ႈ (Introduction)
မိူဝ်ႈႁဝ်းတႅမ်ႈၶူတ်ႉ (Source Code) ယဝ်ႉၼၼ်ႉ ၶမ်ႇပိဝ်ႇတႃႇ (CPU) မၼ်းဢမ်ႇပွင်ႇၸႂ် ၽႃႇသႃႇဢၼ်ႁဝ်းတႅမ်ႈၵမ်းသိုဝ်ႈၶႃႈ။ 
မၼ်းလူဝ်ႇလႆႈမီး **"ၽူႈပိၼ်ႇၽႃႇသႃႇ"** (Translator) တႃႇတေပိၼ်ႇ ၶူတ်ႉႁဝ်း ႁႂ်ႈပဵၼ် **Machine Code (0101)** ဢၼ်ၶိူင်ႈပွင်ႇၸႂ်ၼၼ်ႉယဝ်ႉ။

ၼႂ်းလေႃးၵ Programming ၼႆႉ မီးပိူင်ပိၼ်ႇၽႃႇသႃႇယူႇ 2 မဵဝ်းလူင်: **Compiler** လႄႈ **Interpreter** ၶႃႈ။

---

### 1. Compiler (ၸႂ်ႉၼႂ်း C++, Java, C#)
Compiler ၼႆႉ မၼ်းတေလူ (Read) ၶူတ်ႉႁဝ်း **တင်းမူတ်း (Whole Program)** သေယဝ်ႉ ပိၼ်ႇမၼ်းပဵၼ်ၾၢႆႇ (File) ဢၼ်မႂ်ႇ ဢၼ်ႁွင်ႉဝႃႈ **Executable File (.exe)** ၼၼ်ႉၶႃႈ။

* **လွင်ႈလီ:** ပရူဝ်ႇၶႅမ်ႇႁဵတ်းၵၢၼ်ဝႆး (Fast Performance) ယွၼ်ႉမၼ်းပိၼ်ႇဝႆႉယဝ်ႉတူဝ်ႈယဝ်ႉ။
* **လွင်ႈဢမ်ႇလီ:** ပေႃးမီးတီႈၽိတ်း (Error) ၼႂ်းၶူတ်ႉ တီႈလႂ်တီႈၼိုင်ႈ မၼ်းတေဢမ်ႇပိၼ်ႇပၼ်သေဢိတ်းၶႃႈ။



### 2. Interpreter (ၸႂ်ႉၼႂ်း Python, JavaScript, Dart)
Interpreter ၼႆႉ မၼ်းတေလူ ၶူတ်ႉႁဝ်း **ပၼ်ႇထႅဝ် (Line by line)** သေယဝ်ႉ ႁဵတ်းၵၢၼ်ၵႂႃႇၵမ်းလဵဝ်ၶႃႈ။ မၼ်းဢမ်ႇသၢင်ႈၾၢႆႇ .exe ဝႆႉ။

* **လွင်ႈလီ:** တႃႇတေၸၢမ်း (Debug) လႄႈ ၵႄႈလွင်ႈၽိတ်းၼၼ်ႉ ငၢႆႈ (Easy to debug)။
* **လွင်ႈဢမ်ႇလီ:** ႁဵတ်းၵၢၼ်ဢိူၼ် (Slower) ၵွပ်ႈမၼ်းလႆႈပိၼ်ႇၵႂႃႇ ႁဵတ်းၵၢၼ်ၵႂႃႇ ၼႂ်းၶိင်ႇၼၼ်ႉၵမ်းလဵဝ်။

---

### ပဵၼ်သင် C++ သမ်ႉၸႂ်ႉ Compiler?
ၵွပ်ႈဝႃႈ C++ ၼႆႉ မုင်ႈမွင်းလွင်ႈ **Speed (လွင်ႈဝႆး)** လႄႈ **Efficiency (လွင်ႈၶႅမ်ႉ)** ၶႃႈ။ မိူဝ်ႈႁဝ်း Compile ယဝ်ႉၼၼ်ႉ ႁဝ်းလႆႈၾၢႆႇဢၼ်ဝႆးသေပိူၼ်ႈ တႃႇတေၵႂႃႇၸႂ်ႉၼႂ်း ၵဵမ်း (Games) ဢမ်ႇၼၼ် Software ယႂ်ႇယႂ်ႇၶႃႈ။

### တူဝ်ယၢင်ႇ လွင်ႈပႅၵ်ႇပိူင်ႈ (Comparison Table)

| တၢင်ႇဢၼ် (Feature) | Compiler (C++) | Interpreter (Python) |
| :--- | :--- | :--- |
| **ၵၢၼ်ပိၼ်ႇ** | ပိၼ်ႇတင်းမူတ်း (All at once) | ပိၼ်ႇပၼ်ႇထႅဝ် (Line by line) |
| **လွင်ႈဝႆး** | ဝႆးတႄႉတႄႉ | ဢိူၼ်သေ Compiler |
| **ၾၢႆႇဢွၵ်ႇ** | မီးၾၢႆႇ .exe | ဢမ်ႇမီးၾၢႆႇ .exe |
| **ၵၢၼ်ၵႄႈ Error** | ထူပ်း Error ဝၢႆးပိၼ်ႇယဝ်ႉ | ထူပ်း Error မိူဝ်ႈတိုၵ်ႉႁဵတ်းၵၢၼ် |

---

### ၶေႃႈမၼ်ႈၸႂ် (Summary)
* **C++** ပဵၼ် **Compiled Language**။
* မၼ်းၸႂ်ႉ Compiler တႃႇတေပိၼ်ႇ Source Code ႁႂ်ႈပဵၼ် Machine Code ဢၼ်ဝႆးလိူဝ်ပိူၼ်ႈ။
* ၼႆႉပဵၼ်လွင်ႈတၢင်း ဢၼ်ႁဵတ်းႁႂ်ႈ C++ ၶႅၼ်ႇလိူဝ် ၽႃႇသႃႇတၢင်ႇၸိူဝ်းၼၼ်ႉယဝ်ႉ။
