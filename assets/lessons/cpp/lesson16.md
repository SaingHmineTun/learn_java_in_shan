
# Lesson 16: Arithmetic Operators and Operator Precedence

### ၶေႃႈတၢင်ႇလဝ်ႈ (Introduction)
တႃႇတေႁဵတ်းပရူဝ်ႇၶႅမ်ႇ ဢၼ်ၶိုၵ်ႉတွၼ်းၼၼ်ႉ ႁဝ်းလူဝ်ႇလႆႈၼပ်ႉသွၼ်ႇ (Calculate) ၶေႃႈမုၼ်းၶႃႈ။ 
ၼႂ်း C++ မၼ်းမီးတူဝ်ၼပ်ႉသွၼ်ႇ (Operators) မိူၼ်ၼင်ႇ ၼႂ်းပၢႆးၼပ်ႉ (Math) ဢၼ်ႁဝ်းယိၼ်းႁူႉၸၵ်းဝႆႉယဝ်ႉၼၼ်ႉၶႃႈ။ 
ၵူၺ်းၵႃႈ မၼ်းမီးလွင်ႈလၢၵ်ႇလၢႆးဢိတ်းၼိုင်ႈ တီႈၵၢၼ်ၸႅၵ်ႇ (Division) လႄႈ ၵၢၼ်ဢဝ်တူဝ်ၵိုတ်း (Modulo) ၶႃႈ။

---

### 1. တူဝ်ၼပ်ႉသွၼ်ႇပိုၼ်ႉထၢၼ် (Basic Operators)
| Operator | ၵၢၼ်ႁဵတ်း (Operation)      | တူဝ်ယၢင်ႇ (5, 2) | ၽွၼ်းဢွၵ်ႇ (Result) |
| :--- |:---------------------------| :--- | :--- |
| **`+`** | ၵၢၼ်လေႃး (Addition)        | `5 + 2` | 7 |
| **`-`** | ၵၢၼ်ထွၼ် (Subtraction)     | `5 - 2` | 3 |
| **`*`** | ၵၢၼ်ၵုၼ်း (Multiplication) | `5 * 2` | 10 |
| **`/`** | ၵၢၼ်ၸႅၵ်ႇ (Division)       | `5 / 2` | 2 (ပေႃးပဵၼ် int) |
| **`%`** | ၵၢၼ်ၸႅၵ်ႇလိူဝ် (Modulo)    | `5 % 2` | 1 (ၵိုတ်း 1) |

**မၢႆတွင်း:** `5 / 2` တေဢွၵ်ႇ `2` ၵွပ်ႈဝႃႈ C++ တေတတ်းတူဝ်ၸုတ်ႇပႅတ်ႈ ပေႃးႁဝ်းၸႂ်ႉ `int`။ 
တူဝ် `%` (Modulo) သမ်ႉ ၸႂ်ႉတႃႇႁႃ "တူဝ်ၵိုတ်း" ၵူၺ်းၶႃႈ။

---

### 2. လွင်ႈၼပ်ႉသွၼ်ႇ ဢွၼ်တၢင်း (Operator Precedence)
ပေႃးၼႂ်းထႅဝ်လိၵ်ႈၼိုင်ႈထႅဝ် မီးတူဝ်ၼပ်ႉသွၼ်ႇတင်းၼမ် (မိူၼ်ၼင်ႇ `5 + 2 * 10`)၊ C++ တေလိူၵ်ႈၼပ်ႉသွၼ်ႇၸွမ်းၼင်ႇ **Precedence Rules** (ပၵ်းယၢမ်ႇ ဢၼ်မီးဢဵၼ်ႁႅင်းလိူဝ်) ၶႃႈ။

**ပၵ်းယၢမ်ႇပိုၼ်ႉထၢၼ် (High to Low):**
1.  **Parentheses `( )`**: လိၵ်ႈၼႂ်းၶွပ်ႇမူၼ်း တေလႆႈၼပ်ႉသွၼ်ႇ **ဢွၼ်တၢင်းသုတ်း**။
2.  **Multiplication/Division `* / %`**: တေၼပ်ႉသွၼ်ႇ ပဵၼ်တွၼ်ႈ 2။
3.  **Addition/Subtraction `+ -`**: တေၼပ်ႉသွၼ်ႇ ပဵၼ်တွၼ်ႈသုတ်း။



**တူဝ်ယၢင်ႇ:**
* `5 + 2 * 10`  => `5 + 20` = **25**
* `(5 + 2) * 10` => `7 * 10`  = **70**

---

### 3. Associativity (ၵၢၼ်ၼပ်ႉသွၼ်ႇ ၶၢႆႉၽၢႆႇ)
ပေႃးတူဝ်ၼပ်ႉသွၼ်ႇ မီးဢဵၼ်ႁႅင်း (Precedence) မိူၼ်ၵၼ် (မိူၼ်ၼင်ႇ `*` လႄႈ `/`)၊ 
C++ တေၼပ်ႉသွၼ်ႇပၼ် တႄႇဢဝ် **ၽၢႆႇသၢႆႉ ၵႂႃႇ ၽၢႆႇၶႂႃ** (Left-to-Right) ၶႃႈ။

* `10 / 2 * 5` => `5 * 5` = **25** (ဢမ်ႇၸႂ်ႈ 10 / 10 = 1)

---

### 4. တူဝ်ယၢင်ႇၶူတ်ႉ (Example Code)

```cpp
#include <iostream>

int main() {
    int a = 20;
    int b = 10;
    int c = 5;
    int result;

    result = a + b * c;     // 20 + (10 * 5) = 70
    std::cout << "Result 1: " << result << std::endl;

    result = (a + b) * c;   // (20 + 10) * 5 = 150
    std::cout << "Result 2: " << result << std::endl;

    // ၸၢမ်းၸႂ်ႉ Modulo (%)
    int remainder = 10 % 3; // 10 ႁၢၼ် 3 ၵိုတ်း 1
    std::cout << "Remainder of 10 % 3: " << remainder << std::endl;

    return 0;
}
```

---

### ၶေႃႈမၼ်ႈၸႂ် (Summary)
* ၸႂ်ႉ **`%`** တႃႇႁႃတူဝ်ၵိုတ်း။
* ၸႂ်ႉ **`( )`** တႃႇတႅပ်းတတ်း လွင်ႈၼပ်ႉသွၼ်ႇ ဢၼ်ႁဝ်းလူဝ်ႇႁႂ်ႈမၼ်းႁဵတ်းဢွၼ်တၢင်း။
* C++ တေၼပ်ႉသွၼ်ႇ `*` လႄႈ `/` ဢွၼ်တၢင်း `+` လႄႈ `-`။
