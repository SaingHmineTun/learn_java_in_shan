# Lesson 1: The World of C++

### ၶေႃႈတၢင်ႇလဝ်ႈ (Introduction)
C++ ၼႆႉ ပဵၼ်ၽႃႇသႃႇၶမ်ႇပိဝ်ႇတႃႇ (Programming Language) ဢၼ်မီးဢဵၼ်ႁႅင်းလူင် တင်းၵမ်ႇၽႃႇ လႄႈ ပဵၼ်ပိုၼ်ႉထၢၼ်ဢၼ်လုမ်းၼႃးတႄႉတႄႉၼၼ်ႉယဝ်ႉ။

### C++ ၼႆႉ ပဵၼ်သင်? (What is C++)
C++ ၼႆႉ ထုၵ်ႇၵေႃႇသၢင်ႈၶိုၼ်ႈမႃးမိူဝ်ႈပီ 1979 ၼၼ်ႉယဝ်ႉ။ မၼ်းပဵၼ်ၽႃႇသႃႇဢၼ်ႁွင်ႉဝႃႈ **"High-level Language"** (ငၢႆႈတႃႇၵူၼ်းႁဝ်းလူ) 
ၵူၺ်းၵႃႈ မၼ်းၵေႃႈယင်းမီးလွင်ႈႁဵတ်းၵၢၼ် ဢၼ်ၸမ်တင်း Hardware (လူင်ႉလႅင်းၼႂ်းၶိူင်ႈ) ၼၼ်ႉၶႃႈ။

### ပိူင်ၽၢင်လွင်ႈလၢၵ်ႇလၢႆး C++ (Key Features)
* **Performance:** မၼ်းႁဵတ်းၵၢၼ်လႆႈ ဝႆးတႄႉတႄႉ။
* **Memory Management:** ႁဝ်းၸၢင်ႈၵုမ်းထိင်း လွင်ႈၸႂ်ႉတိုဝ်း Memory (RAM) လႆႈၸွမ်းၼင်ႇၸႂ်ႁဝ်းမီး။
* **Object-Oriented (OOP):** ၸၢင်ႈၵေႃႇသၢင်ႈပရူဝ်ႇၶႅမ်ႇ ဢၼ်ယႂ်ႇလူင် လႄႈ ၸတ်းၵၢၼ်လႆႈငၢႆႈ။

### ၵွပ်ႈသင်ႁဝ်းထုၵ်ႇလီသွၼ် C++?
ပေႃးၸဝ်ႈၵဝ်ႇလႆႈပိုၼ်ႉထၢၼ် C++ ယဝ်ႉ ၸဝ်ႈၵဝ်ႇတေပွင်ႇၸႂ်လွင်ႈ **Computer Architecture** (ပိူင်သၢင်ႈၶမ်ႇပိဝ်ႇတႃႇ) လႆႈလီသေ ၽႃႇသႃႇတၢင်ႇဢၼ်ၶႃႈ။ 
ၼႂ်းၵမ်ႇၽႃႇယၢမ်းလဵဝ် C++ ထုၵ်ႇဢဝ်ၸႂ်ႉဝႆႉၼႂ်း:
1.  **Game Development:** မိူၼ်ၼင်ႇ Unreal Engine ဢၼ်ၸႂ်ႉတႅမ်ႈၵဵမ်း AAA ယႂ်ႇယႂ်ႇၶဝ်။
2.  **Operating Systems:** Windows, macOS လႄႈ Linux ၵေႃႈၸႂ်ႉ C++ တႅမ်ႈဝႆႉ။
3.  **Browsers:** Google Chrome လႄႈ Firefox။
4.  **Embedded Systems:** ၸႂ်ႉၼႂ်းၶိူင်ႈၸၢၵ်ႈ၊ လူတ်ႉယိူၼ်း လႄႈ ၶိူင်ႈႁိူၼ်း (IoT)။

---

### တူဝ်ယၢင်ႇၶူတ်ႉ (Example Code)
တူၺ်းတူဝ်ယၢင်ႇၶူတ်ႉ C++ ဢၼ်ငၢႆႈသေပိူၼ်ႈ ဢၼ်တေၼေဢွၵ်ႇလိၵ်ႈဝႃႈ "Mai Sung Kha!" (မႂ်ႇသုင်ၶႃႈ!) ၼႆႉၶႃႈ:

```cpp
#include <iostream>

int main() {
    // ၼႆႉပဵၼ်ၵၢၼ်ၼေလိၵ်ႈဢွၵ်ႇၼႃႈၸေႃး
    std::cout << "Mai Sung Kha! Welcome to C++ Course." << std::endl;
    return 0;
}
```

### ၶေႃႈမၼ်ႈၸႂ် (Summary)
ၵၢၼ်သွၼ် C++ ၼႆႉ မၼ်းတေယၢပ်ႇၵွၼ်ႇ Java ဢမ်ႇၼၼ် Python ပႅတ်ႈဢိတ်းၼိုင်ႈ၊ ၵူၺ်းၵႃႈ မၼ်းတေႁဵတ်းႁႂ်ႈၸဝ်ႈၵဝ်ႇ ပဵၼ် **Expert Developer** ဢၼ်မီးပိုၼ်ႉထၢၼ်ၶႅၼ်ႇတႄႉတႄႉၼၼ်ႉယဝ်ႉ။
