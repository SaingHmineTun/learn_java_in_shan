# Lesson 9: Integral Types: short, int, long, and long long

### ၶေႃႈတၢင်ႇလဝ်ႈ (Introduction)
ၼႂ်း C++ ၼၼ်ႉ ႁဝ်းဢမ်ႇၸႂ်ႉ "ၵွၵ်း" သႅၼ်းလဵဝ်ၵၼ် တႃႇသိမ်းတူဝ်ၼပ်ႉၵူႈမဵဝ်းၶႃႈ။ 
ၵွပ်ႈဝႃႈ ၶမ်ႇပိဝ်ႇတႃႇမီး Memory ဢၼ်မီးၵႃႈၶၼ်၊ ႁဝ်းလူဝ်ႇလိူၵ်ႈၸႂ်ႉ **Data Type** ဢၼ်ၵိုင်ႇတင်း သႅၼ်းတူဝ်ၼပ်ႉ (Number Size) ဢၼ်ႁဝ်းတေသိမ်းၼၼ်ႉယဝ်ႉ။

---

### 1. တူဝ်ၼပ်ႉမဵဝ်းပႅၵ်ႇပိူင်ႈၵၼ် (Integer Types)
ၼႆႉပဵၼ် တူဝ်ၼပ်ႉ 4 မဵဝ်းလူင် ဢၼ်ႁဝ်းၸႂ်ႉၼႂ်း C++ ၶႃႈ:

| Data Type | Memory Size (Bytes) | Range (ၵႃႈၶၼ်ဢၼ်သိမ်းလႆႈ) |
| :--- | :--- | :--- |
| **`short`** | 2 Bytes | -32,768 ၵႂႃႇၸူး 32,767 |
| **`int`** | 4 Bytes | -2 Billion ၵႂႃႇၸူး 2 Billion |
| **`long`** | 4 ဢမ်ႇၼၼ် 8 Bytes | မိူၼ် `int` ဢမ်ႇၼၼ် ယႂ်ႇလိူဝ် |
| **`long long`** | 8 Bytes | ယႂ်ႇတႄႉတႄႉ (တႃႇတူဝ်ၼပ်ႉသႅၼ်း Billion+) |



---

### 2. Signed vs. Unsigned
ႁဝ်းယင်းၸၢင်ႈထႅမ်သႂ်ႇ **Modifiers** တႃႇၵုမ်းထိင်း လွင်ႈသိမ်းတူဝ်ၼပ်ႉ လႆႈထႅင်ႈၶႃႈ:
* **`signed`**: (Default) သိမ်းလႆႈတင်း တူဝ်ၼပ်ႉ **မီးၶေႃႈ (-) လႄႈ (+)**။
* **`unsigned`**: သိမ်းလႆႈ **တူဝ်ၼပ်ႉ (+)** ဢၼ်လဵဝ်ၵူၺ်း။ ပေႃးႁဝ်းၸႂ်ႉ `unsigned` ၼႆ ႁဝ်းတေသိမ်းတူဝ်ၼပ်ႉလႆႈ ယႂ်ႇလိူဝ်ၵဝ်ႇ ၼိုင်ႈပုၼ်ႈ (2x) ၶႃႈ။

**တူဝ်ယၢင်ႇ:**
```cpp
unsigned int students = 50; // တူဝ်ၼပ်ႉၵူၼ်း ဢမ်ႇမီး (-) ၶႃႈ
```

---

### 3. လၢႆးၵူတ်ႇထတ်း Memory (The sizeof Operator)
ၸဝ်ႈၵဝ်ႇၸၢင်ႈၸႂ်ႉ `sizeof()` တႃႇတူၺ်းဝႃႈ ၼႂ်းၶမ်ႇပိဝ်ႇတႃႇၸဝ်ႈၵဝ်ႇၼၼ်ႉ Data Type ၵူႈမဵဝ်းမၼ်းၸႂ်ႉ Memory တၢၼ်ႇႁိူဝ် ၼႆၶႃႈ:

```cpp
#include <iostream>

int main() {
    std::cout << "Size of short: " << sizeof(short) << " bytes" << std::endl;
    std::cout << "Size of int: " << sizeof(int) << " bytes" << std::endl;
    std::cout << "Size of long long: " << sizeof(long long) << " bytes" << std::endl;
    
    return 0;
}
```

---

### 4. Overflow (လွင်ႈၶေႃႈမုၼ်းလွၼ်ႉ)
ပေႃးႁဝ်းဢဝ်တူဝ်ၼပ်ႉ ဢၼ်ယႂ်ႇလိူဝ် သႅၼ်းၵွၵ်းမၼ်း ၵႂႃႇသႂ်ႇၼၼ်ႉ မၼ်းတေၵိူတ်ႇပဵၼ် **Overflow** ၶႃႈ။ 
မိူၼ်ၼင်ႇ ဢဝ်ၼမ်ႉ 2 လီႇတႃႇ သႂ်ႇၼႂ်းၵွၵ်း ဢၼ်ႁပ်ႉလႆႈ 1 လီႇတႃႇ၊ ၼမ်ႉတေလူၼ်ႉဢွၵ်ႇပႅတ်ႈ (ၶေႃႈမုၼ်းတေၽိတ်းၵႂႃႇ) ၶႃႈ။

---

### ၶေႃႈမၼ်ႈၸႂ် (Summary)
* လိူၵ်ႈၸႂ်ႉ **`short`** တႃႇတူဝ်ၼပ်ႉလဵၵ်ႉ။
* လိူၵ်ႈၸႂ်ႉ **`int`** ပဵၼ်ပိုၼ်ႉထၢၼ် တႃႇတူဝ်ၼပ်ႉၵမ်ႉၼမ်။
* လိူၵ်ႈၸႂ်ႉ **`long long`** တႃႇတူဝ်ၼပ်ႉဢၼ်ယႂ်ႇတႄႉတႄႉ။
* ၸႂ်ႉ **`unsigned`** ပေႃးႁူႉဝႃႈ တူဝ်ၼပ်ႉၼၼ်ႉ တေဢမ်ႇပဵၼ် (-) သေႇသေႇ။
