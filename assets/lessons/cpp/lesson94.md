# Lesson 94: Sequence Containers: Vector, List, and Deque

### ၶေႃႈတၢင်ႇလဝ်ႈ (Introduction)
Sequence Containers သိမ်းၶေႃႈမုၼ်းၸွမ်းၼင်ႇ "ၶပ်ႉတွၼ်ႈ" (Order) ဢၼ်ႁဝ်းသႂ်ႇၶဝ်ႈၵႂႃႇ။ 
ႁဝ်းတေမႃးတူၺ်း 3 တူဝ်ပိူင်လူင်ၼႆႉ ဢၼ် Programmer ၵူႈၵေႃႉ လူဝ်ႇႁူႉဝႆႉၶႃႈ:

---

### 1. `std::vector` (Dynamic Array)
ၼႆႉပဵၼ် "Container ၵူႈလွင်ႈလွင်ႈ" ဢၼ် C++ ၸႂ်ႉပဵၼ် လၵ်းၸဵင် (Default) ၶႃႈ။
* **လွင်ႈႁဵတ်းၵၢၼ်**: သိမ်းၶေႃႈမုၼ်းဝႆႉၸပ်းၵၼ် (Contiguous memory)။
* **ၽွၼ်းလီ**: ႁွင်ႉၸႂ်ႉၶေႃႈမုၼ်း (Access) လႆႈဝႆးသုတ်း လူၺ်ႈၸႂ်ႉ Index `[ ]`။
* **ၽွၼ်းၸႃႉ**: သင်ထႅမ်/လူပ်း ၶေႃႈမုၼ်းတီႈ "တၢင်းၼႃႈ" ဢမ်ႇၼၼ် "ဝူင်ၵၢင်" မၼ်းတေထိူင်း (Slow) ၵွပ်ႈမၼ်းလူဝ်ႇ "ၶၢႆႉ" (Shift) Data တူဝ်တၢင်ႇဢၼ် တင်းမူတ်းၶႃႈ။



---

### 2. `std::list` (Doubly Linked List)
သိမ်းၶေႃႈမုၼ်းပဵၼ် Nodes ဢၼ်မီး Pointer ၸီႉၸူးၵၼ်။
* **လွင်ႈႁဵတ်းၵၢၼ်**: ၶေႃႈမုၼ်းဢမ်ႇလႆႈယူႇၸပ်းၵၼ်ၼႂ်း Memory။
* **ၽွၼ်းလီ**: ထႅမ်/လူပ်း ၶေႃႈမုၼ်းတီႈ "လႂ်ၵေႃႈယဝ်ႉ" လႆႈဝႆးတႄႉတႄႉ (Fast insertion/deletion)။
* **ၽွၼ်းၸႃႉ**: ႁွင်ႉၸႂ်ႉ Index `[ ]` ဢမ်ႇလႆႈ။ သင်ၶႂ်ႈလႆႈတူဝ်ထူၼ်ႈ 5 ၸဝ်ႈၵဝ်ႇတေလႆႈ "လႅၼ်း" တႄႇဢဝ်တူဝ် 1 ၵႂႃႇႁႃမၼ်း (Slow access)။



---

### 3. `std::deque` (Double-Ended Queue)
မၼ်းမိူၼ်ၼင်ႇ Vector ၵူၺ်းၵႃႈ မၼ်း "ပိုတ်ႇ" တင်းသွင်ၽၢႆႇၶႃႈ။
* **လွင်ႈႁဵတ်းၵၢၼ်**: သိမ်းၶေႃႈမုၼ်းပဵၼ် Blocks လၢႆလၢႆဢၼ်။
* **ၽွၼ်းလီ**: ထႅမ်/လူပ်း ၶေႃႈမုၼ်းတီႈ "တၢင်းၼႃႈ" (Front) လႄႈ "တၢင်းလင်" (Back) လႆႈဝႆးတင်းသွင်ၽၢႆႇ။
* **ၽွၼ်းၸႃႉ**: ၵၢၼ်ႁွင်ႉၸႂ်ႉ Index ၸၢင်ႈထိူင်းလိူဝ် Vector ဢိတ်းၼိုင်ႈ ၵွပ်ႈ Memory မၼ်းဢမ်ႇၸပ်းၵၼ်ပဵၼ်သဵၼ်ႈလဵဝ်။

---

### 4. ၵၢၼ်တႅမ်ႈ Code ၼိူင်းၵၼ် (Comparison)

```cpp
#include <iostream>
#include <vector>
#include <list>
#include <deque>

int main() {
    // Vector: ၵႆႉၸႂ်ႉ တႃႇ Access ဝႆး
    std::vector<int> v = {1, 2, 3};
    v.push_back(4); // ထႅမ်တၢင်းလင်

    // Deque: တႅမ်ႈထႅမ်တၢင်းၼႃႈလႆႈဝႆး
    std::deque<int> d = {10, 20};
    d.push_front(5); // ထႅမ်တၢင်းၼႃႈ (Vector ႁဵတ်းဢမ်ႇလႆႈငၢႆႈ)

    // List: Insert တီႈလႂ်ၵေႃႈဝႆး (ၵူၺ်းၵႃႈလူဝ်ႇ Iterator)
    std::list<int> l = {100, 200};
    l.push_back(300);

    return 0;
}
```

### 5. လၢႆးလိူၵ်ႈၸႂ်ႉ (How to Choose?)
| Container | ႁွင်ႉၸႂ်ႉ (Access) | ထႅမ်/လူပ်း (Insert/Delete) | လၢႆးၸႂ်ႉ |
| :--- | :--- | :--- | :--- |
| **Vector** | ဝႆးသုတ်း ($O(1)$) | တၢင်းလင်ဝႆး၊ ဝူင်ႇၵၢင်ဢိူၼ် | ၸႂ်ႉပဵၼ်ပၵ်းယၢမ်ႇ (Default) |
| **Deque** | ဝႆး ($O(1)$) | တၢင်းၼႃႈ လႄႈ တၢင်းလင် ဝႆး | ၸႂ်ႉမိူဝ်ႈလူဝ်ႇထႅမ်ၶေႃႈမုၼ်းသွင်ၽၢႆႇ |
| **List** | ဢိူၼ် ($O(n)$) | ဝႆးၵူႈတီႈ ($O(1)$) | ၸႂ်ႉမိူဝ်ႈလူဝ်ႇထႅမ်/လူပ်းၶေႃႈမုၼ်းၼမ်ၼမ် |

---

### ၶေႃႈမၼ်ႈၸႂ် (Summary)
* **Vector**: ၸိူင်ႉၼင်ႇ Array ဢၼ်ယႂ်ႇလႆႈ၊ ၵႆႉၸႂ်ႉသုတ်း။
* **Deque**: မိူၼ် Vector ၵူၺ်းၵႃႈ ထႅမ်တၢင်းၼႃႈလႆႈဝႆး။
* **List**: ၸိူင်ႉၼင်ႇ သၢႆၶဵင်း (Chain)၊ ထႅမ်/လူပ်း ဝႆး ၵူၺ်းၵႃႈ ႁႃၶေႃႈမုၼ်းထိူင်း။
