# Lesson 27: The For Loop: Count-based Iteration

### ၶေႃႈတၢင်ႇလဝ်ႈ (Introduction)
မိူဝ်ႈႁဝ်းႁူႉဝႃႈ ႁဝ်းၶႂ်ႈႁႂ်ႈ Loop ပၼ်ႇႁဵတ်းၵၢၼ် **"လၢႆပွၵ်ႈလၢႆၵမ်း" (Exact Number of Times)** ၼၼ်ႉ၊ 
ၵၢၼ်ၸႂ်ႉ `for` loop မၼ်းၶႅၼ်ႇလိူဝ် `while` ၶႃႈ။ 
မၼ်းတေဢဝ် Initialization, Condition, လႄႈ Update မႃးဝႆႉၼႂ်း **"ထႅဝ်လဵဝ်ၵၼ်"** တီႈၼိူဝ်သုတ်းမၼ်းၼၼ်ႉယဝ်ႉ။

---

### 1. ပိူင်သၢင်ႈ (Syntax) ၶွင် `for`
မၼ်းမီး 3 တွၼ်ႈ ဢၼ်ၸႅၵ်ႇၵၼ်လူၺ်ႈ **Semicolon (`;`)** ၶႃႈ:

```cpp
for (initialization; condition; update) {
    // ႁဵတ်းၵၢၼ်ၼႂ်းၼႆႈ
}
```



---

### 2. ၶၵ်ႉတွၼ်ႈၵၢၼ်ႁဵတ်း (How it Works)
1.  **Initialization**: ႁဵတ်းၵၢၼ် **ပွၵ်ႈလဵဝ်** မိူဝ်ႈတႄႇ Loop (မိူၼ်ၼင်ႇ `int i = 0`) ၵူၺ်း။
2.  **Condition**: ၵူတ်ႇထတ်းလွင်ႈတၢင်း၊ ပေႃး `true` ၸိုင် ႁဵတ်းၵၢၼ်ၼႂ်း Loop။
3.  **Update**: မိူဝ်ႈႁဵတ်းၵၢၼ်ၼႂ်း Loop ယဝ်ႉၼၼ်ႉ မၼ်းတေမႃး Update ၵႃႈၶၼ် (မိူၼ်ၼင်ႇ `i++`) သေယဝ်ႉ ၵႂႃႇၵူတ်ႇထတ်း Condition ထႅင်ႈပွၵ်ႈၼိုင်ႈ။

---

### 3. တူဝ်ယၢင်ႇၶူတ်ႉ (Example Code)

```cpp
#include <iostream>

int main() {
    // ပၼ်ႇႁဵတ်းၵၢၼ် 5 ပွၵ်ႈ (တႄႇ 1 တေႃႇ 5)
    for (int i = 1; i <= 5; i++) {
        std::cout << "Iteration number: " << i << std::endl;
    }

    std::cout << "---------------------------" << std::endl;

    // ပၼ်ႇယွမ်းလူင်း (Count down)
    for (int j = 10; j > 0; j -= 2) {
        std::cout << "Counting down: " << j << std::endl;
    }

    return 0;
}
```

---

### 4. ပဵၼ်သင်ႁဝ်းတေလႆႈၸႂ်ႉ `for` တႅၼ်း `while`?
* **Clean Code**: ႁဝ်းဢမ်ႇလိုမ်းသႂ်ႇ Update (i++) ၵွပ်ႈမၼ်းဝႆႉၼႃႈတႃၸွမ်းၵၼ်ဝႆႉယဝ်ႉ။
* **Scope**: Variable `i` ဢၼ်ႁဝ်းသၢင်ႈၼႂ်း `for` ၼၼ်ႉ တေႁၢႆၵႂႃႇမိူဝ်ႈ Loop ယဝ်ႉတူဝ်ႈ (ၸွႆႈႁႂ်ႈ Memory လႅတ်း)။
* **Readability**: လူယဝ်ႉႁူႉၵမ်းလဵဝ်ဝႃႈ Loop ၼႆႉတႄႇတီႈလႂ်၊ ယဝ်ႉတီႈလႂ် လႄႈ ၶၢႆႉၵႂႃႇၸိူဝ်ႉႁိူဝ်။

---

### ၶေႃႈမၼ်ႈၸႂ် (Summary)
* **`for`**: ၸႂ်ႉတႃႇပၼ်ႇႁဵတ်းၵၢၼ် ၸွမ်းၼင်ႇတူဝ်ၼပ်ႉဢၼ်ႁဝ်းႁူႉဝႆႉ။
* တွၼ်ႈ 3 တွၼ်ႈ ၼႂ်း `( )` ၼၼ်ႉ တေလႆႈၸႅၵ်ႇလူၺ်ႈ **`;`**။
* မၼ်းပဵၼ် Loop ဢၼ်ၸႂ်ႉၼမ်လိူဝ်ပိူၼ်ႈ ၼႂ်းၵၢၼ်တႅမ်ႈပရူဝ်ႇၶႅမ်ႇ။
