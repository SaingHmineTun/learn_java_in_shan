# Lesson 3.21: Introduction to Bitwise Operators (The CPU Level)

### ၶေႃႈတၢင်ႇလဝ်ႈ (Introduction)
ၼႂ်း C++ ႁဝ်းဢမ်ႇၸႂ်ႉၵူၺ်းတူဝ်ၼပ်ႉ `10`, `20` ၶႃႈ။ ႁဝ်းယင်းၸၢင်ႈၵႂႃႇ "ၵဝ်း" (Manipulate) ၶေႃႈမုၼ်း တီႈ **Binary Level** (0 လႄႈ 1) လႆႈထႅင်ႈၶႃႈ။ ၵၢၼ်ၸႂ်ႉ **Bitwise Operators** ၼႆႉ မၼ်းဝႆးတႄႉတႄႉ လႄႈ ၸႂ်ႉ Memory လဵၵ်ႉလိူဝ်ပိူၼ်ႈ ၵွပ်ႈမၼ်းႁဵတ်းၵၢၼ်ၸူး CPU ၵမ်းသိုဝ်ႈၶႃႈ။

---

### 1. တူဝ်ၼပ်ႉသွၼ်ႇ Bitwise (Bitwise Operators)
ၼႆႉပဵၼ်တူဝ်ၼပ်ႉသွၼ်ႇ ဢၼ်ၼပ်ႉသွၼ်ႇၵမ်းထႅဝ် (Bit-by-bit) ၶႃႈ:

| Operator | ၸိုဝ်ႈႁွင်ႉ | ၵၢၼ်ႁဵတ်း (Operation) |
|:---------| :--- | :--- |
| **`&`**  | **Bitwise AND** | ပဵၼ် 1 ပေႃး **တင်းသွင် Bit** ပဵၼ် 1 |
| **`\|`** | **Bitwise OR** | ပဵၼ် 1 ပေႃး **မီး Bit ၼိုင်ႈ** ပဵၼ် 1 |
| **`^`**  | **Bitwise XOR** | ပဵၼ် 1 ပေႃး **Bit သွင်ဢၼ် ဢမ်ႇမိူၼ်ၵၼ်** |
| **`~`**  | **Bitwise NOT** | **ပိၼ်ႇၽၢႆႇ** (0 ပဵၼ် 1, 1 ပဵၼ် 0) |
| **`<<`** | **Left Shift** | ယွၼ်ႇ Bit ၵႂႃႇၽၢႆႇသၢႆႉ (မိူၼ်ၼင်ႇ **ၵၢၼ်ၵုၼ်း** လူၺ်ႈ 2) |
| **`>>`** | **Right Shift** | ယွၼ်ႇ Bit ၵႂႃႇၽၢႆႇၶႂႃ (မိူၼ်ၼင်ႇ **ၵၢၼ်ၸႅၵ်ႇ** လူၺ်ႈ 2) |



---

### 2. တူဝ်ယၢင်ႇၵၢၼ်ၼပ်ႉ (Example Calculation)
ႁႂ်ႈတူၺ်းတူဝ်ၼပ်ႉ **5** လႄႈ **3** ၼႂ်း Binary:
* 5 = `0101`
* 3 = `0011`

**5 & 3 (Bitwise AND):**
```text
  0101 (5)
& 0011 (3)
----------
  0001 (Result = 1)
```

**5 | 3 (Bitwise OR):**
```text
  0101 (5)
| 0011 (3)
----------
  0111 (Result = 7)
```

---

### 3. တူဝ်ယၢင်ႇၶူတ်ႉ (Example Code)

```cpp
#include <iostream>

int main() {
    int a = 5;  // Binary: 0101
    int b = 3;  // Binary: 0011

    std::cout << "a & b : " << (a & b) << std::endl; // Output: 1
    std::cout << "a | b : " << (a | b) << std::endl; // Output: 7
    std::cout << "a ^ b : " << (a ^ b) << std::endl; // Output: 6
    
    // Left Shift (5 << 1) => 0101 ပဵၼ် 1010 (10)
    // မိူၼ်ၼင်ႇ ဢဝ် 5 ၵုၼ်းလူၺ်ႈ 2
    std::cout << "5 << 1 : " << (5 << 1) << std::endl; 

    return 0;
}
```

### 4. ပဵၼ်သင်ႁဝ်းတေလႆႈၸႂ်ႉ?
1.  **Speed**: မၼ်းပဵၼ်ၵၢၼ်ၼပ်ႉသွၼ်ႇ ဢၼ်ဝႆးသေပိူၼ်ႈ ၼႂ်းၶမ်ႇပိဝ်ႇတႃႇ။
2.  **Embedded Systems**: ၸႂ်ႉတႃႇၵုမ်းထိင်း Hardware (မိူၼ်ၼင်ႇ ပိုတ်ႇ/ပိတ်း ၾႆး LED)။
3.  **Optimization**: ၸႂ်ႉၼႂ်းၵၢၼ်ႁဵတ်း Encryption (လွင်ႈလွတ်ႈၽေး) လႄႈ Graphics ၶႃႈ။

### ၶေႃႈမၼ်ႈၸႂ် (Summary)
* Bitwise Operators ႁဵတ်းၵၢၼ်ၸူး တူဝ် **0 လႄႈ 1**။
* ၸႂ်ႉ **`&`**, **`|`**, **`^`** တႃႇၼပ်ႉသွၼ်ႇ Logic တီႈ Bit Level။
* **Left Shift (`<<`)** မိူၼ်ၼင်ႇၵၢၼ်ၵုၼ်း၊ **Right Shift (`>>`)** မိူၼ်ၼင်ႇၵၢၼ်ၸႅၵ်ႇ။
