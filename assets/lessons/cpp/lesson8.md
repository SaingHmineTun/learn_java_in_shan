# Lesson 8: Introduction to Memory & Variables

### ၶေႃႈတၢင်ႇလဝ်ႈ (Introduction)
ၼႂ်းလေႃးၵ ပရူဝ်ႇၶႅမ်ႇၼႆႉ ႁဝ်းလူဝ်ႇလႆႈ "သိမ်း" (Store) ၶေႃႈမုၼ်း (Data) ဝႆႉ တႃႇတေဢဝ်မႃးၸႂ်ႉၶိုၼ်း။ 
တီႈဢၼ်ႁဝ်းသိမ်းၼၼ်ႉ ႁွင်ႉဝႃႈ **RAM (Random Access Memory)** ၶႃႈ။ 
**Variable** ၼႆႉ ပဵၼ်ၵၢၼ်ပၼ် "ၸိုဝ်ႈ" တီႈယူႇၼႂ်း Memory ၼၼ်ႉ တႃႇႁႂ်ႈႁဝ်းႁွင်ႉၸႂ်ႉလႆႈငၢႆႈငၢႆႈၼၼ်ႉယဝ်ႉ။

---

### 1. Memory မၼ်းမိူၼ်သင်? (How Memory Works)
ႁႂ်ႈဝူၼ်ႉတူၺ်းဝႃႈ Memory ၼႆႉ မိူၼ်ၼင်ႇ "ႁွင်ႈၵၢၼ်ဢၼ်မီးၵွၵ်း (Boxes)" တင်းၼမ်။ ၵွၵ်းၵူႈဢၼ် တေမီး:
1.  **Address**: တီႈယူႇ (Address) ဢၼ်ပဵၼ်တူဝ်ၼပ်ႉ (Hexadecimal)။
2.  **Value**: ၶေႃႈမုၼ်း ဢၼ်သိမ်းဝႆႉၼႂ်းၵွၵ်း။



---

### 2. Variable ၼႆႉ ပဵၼ်သင်? (What is a Variable)
Variable ၼႆႉ ပဵၼ်ၵၢၼ်ဢဝ် "ၸိုဝ်ႈ" (Name) ၵႂႃႇၸပ်းဝႆႉတီႈ Address ၼၼ်ႉၶႃႈ။ 
ၽိူဝ်ႇႁဝ်း တေလႆႈတွင်း Address ဢၼ်ယၢပ်ႇၼၼ်ႉ ႁဝ်းၵေႃႈႁွင်ႉၸႂ်ႉၸိုဝ်ႈမၼ်းလႆႈ ၵမ်းလဵဝ်ၶႃႈ။

**ပိူင်သၢင်ႈ (Syntax) တႃႇသၢင်ႈ Variable:**
```cpp
data_type variable_name = value;
```

တူဝ်ယၢင်ႇ:
```cpp
int age = 25;
```
* `int`: ပဵၼ်မဵဝ်းၶေႃႈမုၼ်း (Data Type) ဢၼ်ပဵၼ်တူဝ်ၼပ်ႉ။
* `age`: ပဵၼ်ၸိုဝ်ႈ Variable (Identifier)။
* `25`: ပဵၼ်ၶေႃႈမုၼ်း ဢၼ်သိမ်းဝႆႉ။

---

### 3. Declaration vs. Initialization
ၼႂ်း C++ မၼ်းမီး 2 ၶၵ်ႉတွၼ်ႈ:
1.  **Declaration**: လၢတ်ႈၼေ Compiler ဝႃႈ "ၵဝ်လူဝ်ႇၸႂ်ႉၵွၵ်းဢၼ်ၼိုင်ႈ ဢၼ်တေသိမ်းတူဝ်ၼပ်ႉ (int) သေ ပၼ်ၸိုဝ်ႈဝႃႈ `x` ၼႆ"။
    ```cpp
    int x; 
    ```
2.  **Initialization**: ဢဝ်ၶေႃႈမုၼ်း သႂ်ႇၼႂ်းၵွၵ်းၼၼ်ႉ။
    ```cpp
    x = 10;
    ```
*ၸဝ်ႈၵဝ်ႇ ၸၢင်ႈႁဵတ်းတင်း 2 ဢၼ် ၼႂ်းထႅဝ်လဵဝ်ၵၼ်လႆႈၶႃႈ:* `int x = 10;`

---

### 4. ၵၢၼ်ပၼ်ၸိုဝ်ႈ (Naming Rules)
* တေလႆႈတႄႇလူၺ်ႈ **လိၵ်ႈ (Letter)** ဢမ်ႇၼၼ် **Underscore (_)**။
* တေလႆႈဢမ်ႇမီး **ပိုၼ်ႉတီႈလွတ်ႈ (Space)**။
* တေလႆႈဢမ်ႇၸႂ်ႉ **Keywords** ဢၼ် C++ ၸႂ်ႉဝႆႉယဝ်ႉ (မိူၼ်ၼင်ႇ `int`, `main`, `return`)။
* **Case Sensitive**: `age` လႄႈ `Age` ၼႆႉ ပဵၼ် Variable ၵူၼ်းမဵဝ်းၵၼ်ၶႃႈ။

---

### ၶေႃႈမၼ်ႈၸႂ် (Summary)
* **Memory** ပဵၼ်တီႈသိမ်းၶေႃႈမုၼ်း။
* **Variable** ပဵၼ်ၸိုဝ်ႈဢၼ်ႁဝ်းပၼ် တီႈယူႇၼႂ်း Memory။
* ႁဝ်းလူဝ်ႇလႆႈလၢတ်ႈ **Data Type** မၼ်းသေႇသေႇ မိူဝ်ႈသၢင်ႈ Variable။
