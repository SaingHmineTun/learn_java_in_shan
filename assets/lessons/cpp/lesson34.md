# Lesson 34: Array Indexing and Memory Layout

### ၶေႃႈတၢင်ႇလဝ်ႈ (Introduction)
ဝၢႆးႁဝ်းသၢင်ႈ Array ယဝ်ႉ၊ ႁဝ်းလူဝ်ႇလႆႈႁူႉလၢႆး "ၵႂႃႇဢဝ်" (Access) ၶေႃႈမုၼ်းၼႂ်းၼၼ်ႉၶႃႈ။ 
ၼႂ်း C++၊ ႁဝ်းၸႂ်ႉတူဝ်ၼပ်ႉ ဢၼ်ႁွင်ႉဝႃႈ **Index**။ 
လွင်ႈလၢၵ်ႇလၢႆးမၼ်းတႄႉ ၶမ်ႇပိဝ်ႇတႃႇဢမ်ႇတႄႇၼပ်ႉ 1 ၶႃႈ၊ မၼ်းတႄႇၼပ်ႉ **0** ဢွၼ်တၢင်းသုတ်းၶႃႈ။

---

### 1. Array Indexing (ၵၢၼ်ၼပ်ႉမၢႆႁွင်ႈ)
Index ပဵၼ်တူဝ်ၼပ်ႉ ဢၼ်လၢတ်ႈၼေဝႃႈ ၶေႃႈမုၼ်းၼၼ်ႉ ယူႇဝႆႉ "ႁွင်ႈထီႉလႂ်" ၼႂ်း Array ၶႃႈ။

* **Zero-based Indexing**: ႁွင်ႈဢွၼ်တၢင်းသုတ်း ပဵၼ် Index **`[0]`** သေႇသေႇ။
* **Last Index**: ႁွင်ႈသုတ်းသုတ်း တေပဵၼ် **`[Size - 1]`** ၶႃႈ။



**တူဝ်ယၢင်ႇ:** ပေႃး Array မီး Size 5၊ Index မၼ်းတေမီး `0, 1, 2, 3, 4`။

---

### 2. Memory Layout (ပိူင်သၢင်ႈ Memory)
Array သိမ်းၶေႃႈမုၼ်းဝႆႉ **"ၸပ်းၵၼ်" (Contiguous Memory)** ၶႃႈ။
* ပေႃးႁဝ်းမီး `int arr[3]`၊ ၶေႃႈမုၼ်းတင်း 3 တူဝ်ၼၼ်ႉ တေယူႇဝႆႉထႅဝ်လဵဝ်ၵၼ်ၼႂ်း RAM။
* ၼႆႉႁဵတ်းႁႂ်ႈ CPU ၵႂႃႇဢဝ်ၶေႃႈမုၼ်းလႆႈ **ဝႆးတႄႉတႄႉ** ၵွပ်ႈမၼ်းႁူႉ တီႈယူႇ (Address) ဢၼ်တေႃႇၵၼ်ၵႂႃႇၼၼ်ႉၶႃႈယဝ်ႉ။



---

### 3. တူဝ်ယၢင်ႇၶူတ်ႉ (Accessing & Modifying)

```cpp
#include <iostream>

int main() {
    int marks[3] = {85, 92, 78};

    // 1. လူဢဝ်ၵႃႈၶၼ် (Accessing)
    std::cout << "First student: " << marks[0] << std::endl; // 85
    std::cout << "Second student: " << marks[1] << std::endl; // 92

    // 2. လႅၵ်ႈလၢႆႈၵႃႈၶၼ် (Modifying)
    marks[2] = 80; // လႅၵ်ႈ 78 ႁႂ်ႈပဵၼ် 80
    std::cout << "Third student updated: " << marks[2] << std::endl;

    return 0;
}
```

---

### 4. ၶေႃႈလုမ်းၼႃး: Out of Bounds (ၼပ်ႉမၢႆပူၼ်ႉၶွပ်ႇ)
ၼႆႉပဵၼ် လွင်ႈၽိတ်း (Bug) ဢၼ်ယႂ်ႇသေပိူၼ်ႈၶႃႈ။ ပေႃး Array မီး Size 5၊ ၵူၺ်းၵႃႈ ၸဝ်ႈၵဝ်ႇၵႂႃႇႁွင်ႉၸႂ်ႉ `marks[5]` (ဢၼ်ပဵၼ်ႁွင်ႈထီႉ 6)၊ C++ တေဢမ်ႇႁႄႉၵင်ႈၸဝ်ႈၵဝ်ႇၶႃႈ။
* မၼ်းတေၵႂႃႇလူဢဝ်ၶေႃႈမုၼ်း "ယုင်ႈယၢင်ႈ" (Garbage value) မႃးၼေ။
* ဢမ်ႇၼၼ် ပရူဝ်ႇၶႅမ်ႇၸၢင်ႈ "တၢႆ" (Crash) ၵႂႃႇၵမ်းလဵဝ်ၶႃႈ။

---

### ၶေႃႈမၼ်ႈၸႂ် (Summary)
* **Index** တႄႇၼပ်ႉတီႈ **0** သေႇသေႇ။
* ၶေႃႈမုၼ်း Array ယူႇဝႆႉ **ၸပ်းၵၼ်** ၼႂ်း Memory။
* တေလႆႈလုမ်းၼႃး ယႃႇႁႂ်ႈၸႂ်ႉ Index ဢၼ် **ပူၼ်ႉ Size** ၶွင်မၼ်း။
