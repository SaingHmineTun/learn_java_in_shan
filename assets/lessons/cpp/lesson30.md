# Lesson 30: Loop Control: Break and Continue

### ၶေႃႈတၢင်ႇလဝ်ႈ (Introduction)
မၢင်လႂ် ႁဝ်းဢမ်ႇၶႂ်ႈႁႂ်ႈ Loop ပၼ်ႇႁဵတ်းၵၢၼ် တေႃႇပေႃးသုတ်းၶႃႈ။ မိူၼ်ၼင်ႇ "ၵၢၼ်ႁႃၶေႃႈမုၼ်း"၊ ပေႃးႁဝ်းထူပ်းယဝ်ႉ ႁဝ်းၵေႃႈၶႂ်ႈႁႂ်ႈမၼ်း "ၵိုတ်း" ၵမ်းလဵဝ်။ 
ဢမ်ႇၼၼ် ႁဝ်းၶႂ်ႈႁႂ်ႈမၼ်း "ၶၢမ်ႈပူၼ်ႉ" တူဝ်ၼပ်ႉဢၼ်ၼိုင်ႈပႅတ်ႈ။ ၼႆႉပဵၼ်ၼႃႈၵၢၼ်ၶွင် **`break`** လႄႈ **`continue`** ၶႃႈ။

---

### 1. The `break` Statement (ၵၢၼ်ၵိုတ်းၵမ်းလဵဝ်)
`break` ၸႂ်ႉတႃႇ "တႅပ်းတတ်း" လႄႈ "ဢွၵ်ႇ" (Exit) ၵႂႃႇၽၢႆႇၼွၵ်ႈ Loop ၵမ်းလဵဝ်၊ ဢမ်ႇဝႃႈ Loop ၼၼ်ႉ ယင်းပႆႇပၼ်ႇယဝ်ႉၵေႃႈယဝ်ႉ။

* **ၵၢၼ်ၸႂ်ႉတိုဝ်း**: ၸႂ်ႉမိူဝ်ႈႁဝ်းထူပ်း ၶေႃႈမုၼ်းဢၼ်ႁဝ်းလူဝ်ႇယဝ်ႉ ဢမ်ႇၼၼ် မိူဝ်ႈမီးလွင်ႈၽိတ်း (Error) ၶႃႈ။



---

### 2. The `continue` Statement (ၵၢၼ်ၵျွၵ်ႉၶၢမ်ႇ)
`continue` ၸႂ်ႉတႃႇ "ၶၢမ်ႈပူၼ်ႉ" (Skip) ၵၢၼ်ႁဵတ်းၵၢၼ် တွၼ်ႈဢၼ်ၵိုတ်းဝႆႉၼႂ်း Loop ၼၼ်ႉ သေယဝ်ႉ ႁႂ်ႈမၼ်း "ၵျွၵ်ႉ" ၵႂႃႇပၼ်ႇ ထႅင်ႈပွၵ်ႈၼိုင်ႈ (Next iteration) ၵမ်းလဵဝ်ၶႃႈ။

* **ၵၢၼ်ၸႂ်ႉတိုဝ်း**: ၸႂ်ႉမိူဝ်ႈႁဝ်းဢမ်ႇၶႂ်ႈၼပ်ႉသွၼ်ႇ ၶေႃႈမုၼ်း သေမဵဝ်းမဵဝ်း (မိူၼ်ၼင်ႇ ၶၢမ်ႈတူဝ်ၼပ်ႉၵူပ်ႉ) ၶႃႈ။



---

### 3. တူဝ်ယၢင်ႇၶူတ်ႉ (Example Code)

```cpp
#include <iostream>

int main() {
    std::cout << "--- Testing Break ---" << std::endl;
    for (int i = 1; i <= 5; i++) {
        if (i == 3) {
            break; // တေၵိုတ်းတီႈ 3 သေ ဢွၵ်ႇၵႂႃႇၵမ်းလဵဝ်
        }
        std::cout << i << " ";
    }
    // ၽွၼ်းဢွၵ်ႇ: 1 2

    std::cout << "\n\n--- Testing Continue ---" << std::endl;
    for (int j = 1; j <= 5; j++) {
        if (j == 3) {
            continue; // တေၶၢမ်ႈပူၼ်ႉ 3 သေ ၵႂႃႇသိုပ်ႇႁဵတ်း 4
        }
        std::cout << j << " ";
    }
    // ၽွၼ်းဢွၵ်ႇ: 1 2 4 5

    return 0;
}
```

---

### 4. ၶေႃႈလုမ်းၼႃး (Things to Remember)
* **Nested Loops**: ပေႃးၸဝ်ႈၵဝ်ႇၸႂ်ႉ `break` ၼႂ်း Loop သွၼ်ႉ Loop (Nested)၊ မၼ်းတေဢွၵ်ႇၼႂ်း Loop **ၽၢႆႇၼႂ်း (Inner)** ဢၼ်မၼ်းယူႇဝႆႉၼၼ်ႉၵူၺ်းၶႃႈ၊ မၼ်းဢမ်ႇဢွၵ်ႇ Outer Loop ၸွမ်းၶႃႈ။
* **Infinite Loops**: ႁဝ်းၸၢင်ႈၸႂ်ႉ `while(true)` သေယဝ်ႉ ၸႂ်ႉ `break` တႃႇၵုမ်းထိင်းၵၢၼ်ဢွၵ်ႇ Loop ၸွမ်းၼင်ႇ Logic ႁဝ်းၶႃႈ။

---

### ၶေႃႈမၼ်ႈၸႂ် (Summary)
* **`break`**: ၵိုတ်း Loop တင်းမူတ်း (Stop everything).
* **`continue`**: ၶၢမ်ႇပွၵ်ႈၼႆႉသေ ၵႂႃႇတႄႇပွၵ်ႈမႂ်ႇ (Skip this turn).
* ၸွႆႈႁႂ်ႈပရူဝ်ႇၶႅမ်ႇႁဝ်း ႁဵတ်းၵၢၼ်လႅတ်း (Efficient) လိူဝ်ၵဝ်ႇ ၵွပ်ႈဢမ်ႇလႆႈပၼ်ႇႁဵတ်းၵၢၼ် ဢၼ်ဢမ်ႇလူဝ်ႇၼၼ်ႉၶႃႈ။
