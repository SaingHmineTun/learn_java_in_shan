## Lesson 30: Strings in C: Character Arrays & The Null Terminator

### 1. String ၼႂ်း C ပဵၼ်သင်?
ၼႂ်း C ႁဝ်းဢမ်ႇမီး String Type ၶႃႈ။
String ၼႂ်း C ပဵၼ် **One-dimensional Array ၶွင် Characters** ဢၼ်ယဝ်ႉတူဝ်ႈလူၺ်ႈ မၢႆ **`\0`** (Null character) ၵူၺ်းၶႃႈ။
* **Character Array:** ၵဵပ်းတူဝ်လိၵ်ႈၼိုင်ႈတူဝ်ယဝ်ႉၼိုင်ႈတူဝ်။
* **Null Terminator (`\0`):** ပဵၼ်မၢႆဢၼ်သင်ႇႁႂ်ႈ `printf` ဢမ်ႇၼၼ် `strlen` ႁူႉဝႃႈ "လိၵ်ႈယဝ်ႉတူဝ်ႈတီႈၼႆႈယဝ်ႉ" ၼၼ်ႉၶႃႈဢေႃႈ။



### 2. ၵၢၼ်ပိုၼ်ၽၢဝ်ႇ (Declaration)
ႁဝ်းတႅမ်ႈလႆႈ 2 လၢႆးၶႃႈ:
1. **Array Style:** `char name[] = {'S', 'a', 'i', '\0'};` (လူဝ်ႇသႂ်ႇ `\0` ႁင်းၵူၺ်း)
2. **String Literal Style:** `char name[] = "Sai";` (C တေထႅမ် `\0` ပၼ်ႁင်းမၼ်း တီႈပၢႆမၼ်းၶႃႈ)

**သတိ:** သင်ၸဝ်ႈၵဝ်ႇတႅမ်ႈ `char name[3] = "Sai";` ၼႆ တေ Error ၶႃႈ! ၵွပ်ႈဝႃႈ "Sai" မီး 3 တူဝ်၊ ၵူၺ်းၵႃႈ လူဝ်ႇထႅင်ႈ 1 ႁွင်ႉ တွၼ်ႈတႃႇၵဵပ်း `\0`။ ၵွပ်ႈၼၼ် လူဝ်ႇၸႂ်ႉ `char name[4]` ၶႃႈဢေႃႈ။

### 3. Memory Architecture ၶွင် String
ႁႂ်ႈဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းမီး `char word[] = "Shan";`
| Index | 0 | 1 | 2 | 3 | 4 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **Value** | 'S' | 'h' | 'a' | 'n' | **`\0`** |
| **ASCII** | 83 | 104 | 97 | 110 | **0** |



### 4. ၵၢၼ်ႁပ်ႉဢဝ် လိၵ်ႈတီႈ User
* **`scanf("%s", name);`**: ႁပ်ႉလႆႈလိၵ်ႈ ဢၼ်ဢမ်ႇမီး "ဢွင်ႈဝၢင်ႇ" (Space) ၵူၺ်း။ (သင်သႂ်ႇ "Sai Mao" မၼ်းတေဢဝ်ၵူၺ်း "Sai" ၶႃႈ)။
* **`fgets(name, sizeof(name), stdin);`**: ၼႆႉပဵၼ်လၢႆးဢၼ် "လႅတ်ႈသုတ်း" ၶႃႈ။ မၼ်းႁပ်ႉလႆႈတင်း Space လႄႈ ႁႄႉၵင်ႈဢမ်ႇႁႂ်ႈလိၵ်ႈ ယႂ်ႇလိူဝ်သႅၼ်း Array (Buffer Overflow) ၶႃႈဢေႃႈ။

### 5. ၵွပ်ႈသင် `\0` သမ်ႉလမ်ႇလွင်ႈ?
သင်ဢမ်ႇမီး `\0` တီႈပၢႆ String၊ ၶွမ်ႇပိဝ်ႇတႃႇတေဢမ်ႇႁူႉဝႃႈ လိၵ်ႈယဝ်ႉတီႈလႂ်။ မၼ်းတေ "ဢၢၼ်ႇလိၵ်ႈၵႂႃႇတိၵ်းတိၵ်း" ၼႂ်း RAM တေႃႇပေႃးထိုင်တီႈဢၼ်မၼ်းႁၼ် မၢႆ 0 (Garbage values) သေ ႁဵတ်းႁႂ်ႈပူဝ်ႇၵရႅမ်ႇ Crash ၵႂႃႇလႆႈၶႃႈ။
