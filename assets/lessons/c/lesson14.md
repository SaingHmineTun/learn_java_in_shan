## Lesson 14: Arithmetic & Assignment: The Basics

### 1. တူဝ်ၼပ်ႉပိုၼ်ႉထၢၼ် (Arithmetic Operators)
ၼႂ်းလိၵ်ႈ C ႁဝ်းၸႂ်ႉတူဝ်မၢႆၸိူဝ်းၼႆႉ တႃႇၼပ်ႉဢၢၼ်ႇတူဝ်ၼပ်ႉၶႃႈ။

| Operator | ၼႃႈၵၢၼ် (Operation) | တူဝ်ယၢင်ႇ (Example) |
| :--- | :--- | :--- |
| **`+`** | တူဝ်ထႅမ် (Addition) | `10 + 5 = 15` |
| **`-`** | တူဝ်ယုတ်း (Subtraction) | `10 - 5 = 5` |
| **`*`** | တူဝ်ပုၼ်း (Multiplication) | `10 * 5 = 50` |
| **`/`** | တူဝ်ၽႄ (Division) | `10 / 4 = 2` (*သင်ပဵၼ် int*) |
| **`%`** | တူဝ်လိူဝ် (Modulus) | `10 % 3 = 1` |



### 2. လွင်ႈပႅၵ်ႇပိူင်ႈၶွင် ၵၢၼ်ၽႄ (Integer vs Float Division)
ၼႆႉပဵၼ်လွင်ႈဢၼ် လုၵ်ႈႁဵၼ်းၸၢင်ႈၽိတ်းၼမ်သုတ်းၶႃႈ။
* **Integer Division:** သင်ဢဝ် `int` ၽႄတင်း `int` ၼႆ တွပ်ႇမၼ်းတေဢွၵ်ႇပဵၼ် `int` တူဝ်တဵမ်ၵူၺ်း။ (မိူၼ်ၼင်ႇ `5 / 2` တေလႆႈ `2`၊ တူဝ်ႁွႆႈ `.5` ၼၼ်ႉတေႁၢႆၵႂႃႇၶႃႈ)။
* **Float Division:** သင်ၶႂ်ႈလႆႈတူဝ်ႁွႆႈ လူဝ်ႇမီးတူဝ်ၼိုင်ႈပဵၼ် `float` (မိူၼ်ၼင်ႇ `5.0 / 2` တေလႆႈ `2.5` ၶႃႈ)။

### 3. Modulus Operator (`%`)
တူဝ်မၢႆ `%` ၼႆႉ ၸႂ်ႉတႃႇႁႃ "တူဝ်လိူဝ်" ၶွင်ၵၢၼ်ၽႄၶႃႈ။
* `10 % 3` = `1` (ၵွပ်ႈဝႃႈ 3 ၵႂႃႇ 3 ပၼ်ႇလႆႈ 9၊ လိူဝ် 1)။
* **သတိ:** တူဝ်မၢႆ `%` ၼႆႉ ၸႂ်ႉလႆႈတင်း **`int`** ၵူၺ်းၶႃႈ၊ ၸႂ်ႉတင်း `float` ဢမ်ႇလႆႈ။

### 4. တူဝ်မၢႆပၼ်ၵၢၼ် (Assignment Operators)
တူဝ်မၢႆ **`=`** ၼႆႉ ဢမ်ႇၸႂ်ႉ "ထၢမ်ဝႃႈတူႇၵၼ်ႁိုဝ်"၊ မၼ်းၸႂ်ႉတႃႇ "ဢဝ်ၶေႃႈမုၼ်းၽၢႆႇၶႂႃ သႂ်ႇဝႆႉၼႂ်း Variable ၽၢႆႇသၢႆႉ" ၶႃႈ။

**Compound Assignment (လၢႆးတႅမ်ႈပွတ်း):**
တွၼ်ႈတႃႇႁႂ်ႈ Code ႁဝ်းသႅၼ်ႈသႂ် ႁဝ်းၸႂ်ႉလၢႆးၼႆႉလႆႈၶႃႈ -
* `x += 5;` မိူၼ်ၵၼ်တင်း `x = x + 5;`
* `x -= 2;` မိူၼ်ၵၼ်တင်း `x = x - 2;`
* `x *= 3;` မိူၼ်ၵၼ်တင်း `x = x * 3;`



### 5. Increment & Decrement (`++` and `--`)
ၼႆႉပဵၼ်ၵၢၼ် "ထႅမ် 1" ဢမ်ႇၼၼ် "ယုတ်း 1" ဢၼ်ၸႂ်ႉၼမ်သုတ်းၼႂ်း Loops (ၵၢၼ်ႁဵတ်းၶိုၼ်း) ၶႃႈ။
* **`x++`**: ထႅမ် 1 သႂ်ႇ x။
* **`x--`**: ယုတ်း 1 ဢွၵ်ႇတီႈ x။
