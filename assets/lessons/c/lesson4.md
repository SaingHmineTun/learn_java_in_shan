## Lesson 4: The Compilation Pipeline: From Code to Binary

### 1. ၶၵ်ႉတွၼ်ႈၵၢၼ်ပိၼ်ႇလိၵ်ႈ (What is Compilation?)
မိူဝ်ႈႁဝ်းတႅမ်ႈ Code ယဝ်ႉသေ တဵၵ်း "Run" ၼၼ်ႉ Compiler ႁဝ်း (မိူၼ်ၼင်ႇ GCC) မၼ်းဢမ်ႇပိၼ်ႇၵမ်းလဵဝ်ၶႃႈ။ မၼ်းမီးၶၵ်ႉတွၼ်ႈ 4 ယိူင်ႈ ဢၼ်လုၵ်ႈႁဵၼ်းလူဝ်ႇႁူႉဝႆႉ တွၼ်ႈတႃႇၵႄႈလႅတ်ႈလွင်ႈၽိတ်း (Debugging) ၶႃႈ။



### 2. Step 1: Preprocessing (ၵၢၼ်ၸတ်းၵၢၼ်လူင်ႈၼႃႈ)
ၼႆႉပဵၼ်ၶၵ်ႉတွၼ်ႈဢွၼ်တၢင်းသုတ်း။ Compiler တေတူၺ်းထႅဝ်လိၵ်ႈ ဢၼ်တႄႇတင်းမၢႆ **`#`** ၼၼ်ႉၶႃႈ။
* **Header Files:** ဢဝ် Code တီႈ `#include <stdio.h>` ၼၼ်ႉ မႃးသႂ်ႇဝႆႉၼႂ်း File ႁဝ်း။
* **Macros:** လႅၵ်ႈလၢႆႈ `#define` ႁႂ်ႈပဵၼ်တူဝ်ၼပ်ႉတူဝ်တေႉမၼ်း။
* **Comments:** ထွၼ်ပႅတ်ႈ Comment (လွင်ႈတႅမ်ႈမၢႆ) တင်းသဵင်ႈ ၵွပ်ႈဝႃႈၶွမ်ႇပိဝ်ႇတႃႇ ဢမ်ႇလူဝ်ႇၸႂ်ႉၶႃႈ။
* **Output:** လႆႈမႃး File ဢၼ်မီးတူဝ်မၢႆ `.i` ၶႃႈ။

### 3. Step 2: Compiling (ၵၢၼ်ပိၼ်ႇပဵၼ် Assembly)
ၼႂ်းၶၵ်ႉတွၼ်ႈၼႆႉ Compiler တေၵူတ်ႇထတ်း မၢႆမီႈ (Syntax) ႁဝ်းဝႃႈ မီးတီႈၽိတ်းႁိုဝ်ဢမ်ႇမီး?
* သင်မၢႆမီႈၸဝ်ႈၵဝ်ႇမၢၼ်ႇယဝ်ႉၼႆ မၼ်းတေပိၼ်ႇ C Code ႁဝ်း ႁႂ်ႈပဵၼ် **Assembly Language** (လိၵ်ႈၸၢၵ်ႈ ဢၼ်ၵူၼ်းတေႃႉတိုၵ်ႉလူလႆႈယူႇ)။
* **Output:** လႆႈမႃး File ဢၼ်မီးတူဝ်မၢႆ `.s` ၶႃႈ။



### 4. Step 3: Assembling (ၵၢၼ်ပိၼ်ႇပဵၼ် Object Code)
ၼႆႉပဵၼ်ၵၢၼ်ပိၼ်ႇ Assembly ႁႂ်ႈပဵၼ် Binary (လိၵ်ႈၸၢၵ်ႈ 0 လႄႈ 1) တေႉတေႉယဝ်ႉၶႃႈ။
* ၶွမ်ႇပိဝ်ႇတႃႇ တေတႄႇပွင်ႇၸႂ် Code ႁဝ်း တီႈၶၵ်ႉတွၼ်ႈၼႆႉၶႃႈဢေႃႈ။
* **Output:** လႆႈမႃး **Object File** ဢၼ်မီးတူဝ်မၢႆ `.o` (ဢမ်ႇၼၼ် `.obj`)။ ၵူၺ်းၵႃႈ ဢၼ်ၼႆႉၵေႃႈ တေႃႉတိုၵ်ႉ Run ဢမ်ႇလႆႈၵမ်းလဵဝ်ၶႃႈ။

### 5. Step 4: Linking (ၵၢၼ်ႁူမ်ႈ File တႃႇ Run)
ၼႆႉပဵၼ်ၶၵ်ႉတွၼ်ႈလိုၼ်းသုတ်း။ ပူဝ်ႇၵရႅမ်ႇႁဝ်း ၸၢင်ႈတေမီးလၢႆလၢႆ File ဢမ်ႇၼၼ် လူဝ်ႇၸႂ်ႉ Library တၢင်ႇၸိူဝ်း။
* **Linker** တေဢဝ် Object Files တင်းသဵင်ႈ လႄႈ Libraries (မိူၼ်ၼင်ႇ `stdio`) မႃးၸပ်းၵၼ် ႁႂ်ႈပဵၼ် File လဵဝ်ၵူၺ်း။
* **Output:** လႆႈမႃး **Executable File** ဢၼ်မီးတူဝ်မၢႆ `.exe` (ၼႂ်း Windows) ဢမ်ႇၼၼ် `.out` (ၼႂ်း Linux) ဢၼ်ႁဝ်းတဵၵ်း Run လႆႈၼၼ်ႉၶႃႈဢေႃႈ။

## 💡 တွၼ်ႈတွၼ်းမၢႆ (Key Takeaway)
ၵွပ်ႈသင် လုၵ်ႈႁဵၼ်းလူဝ်ႇႁူႉ?
သင်ပူဝ်ႇၵရႅမ်ႇၸဝ်ႈၵဝ်ႇ Error တီႈ **Compiling** ၼႆ ပွင်ႇဝႃႈၸဝ်ႈၵဝ်ႇတႅမ်ႈ Syntax ၽိတ်း (မိူၼ်ၼင်ႇ လိုမ်းသႂ်ႇ `;`)။ ၵူၺ်းၵႃႈ သင်မၼ်း Error တီႈ **Linking** ၼႆ ပွင်ႇဝႃႈၸဝ်ႈၵဝ်ႇ ႁွင်ႉၸိုဝ်ႈ Function ၽိတ်း ဢမ်ႇၼၼ် ႁွင်ႉဢဝ် Library မႃးဢမ်ႇလႆႈၼၼ်ႉၶႃႈဢေႃႈ။
