## Lesson 24: for Loops: The Powerhouse

### 1. `for` Loop ၼႆႉ ပဵၼ်သင်?
`for` loop ၼႆႉ ၸႂ်ႉတွၼ်ႈတႃႇႁဵတ်းၵၢၼ်ၶိုၼ်း မိူဝ်ႈႁဝ်း **ႁူႉႁူဝ်ၼပ်ႉတႅတ်ႈတေႃး** ဝႃႈ တေႁႂ်ႈမၼ်းဝၼ်ႇပၼ်ႇလၢႆၵမ်း (မိူၼ်ၼင်ႇ ၼပ်ႉ 1 တေႃႇ 100) ၶႃႈဢေႃႈ။



### 2. ပိူင်တႅမ်ႈ (Syntax)
မၼ်းမီး 3 တွၼ်ႈ ၼႂ်းဝူင်းလဵပ်ႈၶႃႈ:
```c
for (initialization; condition; increment/decrement) {
    // Code ဢၼ်ၶႂ်ႈႁႂ်ႈႁဵတ်းၵၢၼ်ၶိုၼ်း
}
```

* **Initialization:** တႄႇတင်ႈတူဝ်ၼပ်ႉ (မိူၼ်ၼင်ႇ `int i = 0;`) - ႁဵတ်းၵၢၼ် **1 ၵမ်း** ဢွၼ်တၢင်းသုတ်း။
* **Condition:** ၵူတ်ႇထတ်းဝႃႈ တေႁဵတ်းသိုပ်ႇႁိုဝ်? (မိူၼ်ၼင်ႇ `i < 10;`) - သင်ပဵၼ် True ၸင်ႇၶဝ်ႈႁဵတ်းၵၢၼ်။
* **Increment/Decrement:** ၵၢၼ်ထႅမ် ဢမ်ႇၼၼ် ယုတ်းတူဝ်ၼပ်ႉ (မိူၼ်ၼင်ႇ `i++`) - ႁဵတ်းၵၢၼ် **ဝၢႆးသေ** Code ၼႂ်း `{ }` ယဝ်ႉတူဝ်ႈၵူႈပွၵ်ႈ။

### 3. တူဝ်ယၢင်ႇ: ၵၢၼ်ၼပ်ႉထႅဝ် (Multiplication Table)
```c
int num = 5;
for (int i = 1; i <= 10; i++) {
    printf("%d x %d = %d\n", num, i, num * i);
}
```
*(Code ၼႆႉ တေၼႄမၢႆပုၼ်း 5 တေႃႇပေႃးထိုင် 5 x 10 ၶႃႈဢေႃႈ)*



### 4. ၵွပ်ႈသင် `for` သမ်ႉလီလိူဝ် `while`?
* **Compact:** Code မၼ်းသႅၼ်ႈသႂ်လိူဝ် (Clean)။
* **Safety:** ႁဝ်းဢမ်ႇလိုမ်း "ထႅမ်တူဝ်ၼပ်ႉ" (`i++`) ၵွပ်ႈမၼ်းမီးဝႆႉၼႂ်းႁူဝ်မၼ်းတူဝ်တူဝ်ၶႃႈယဝ်ႉ။
* **Scope:** Variable `i` ဢၼ်တႄႇဝႆႉၼႂ်း `for` ၼၼ်ႉ တေႁၢႆၵႂႃႇမိူဝ်ႈ Loop ယဝ်ႉတူဝ်ႈ (သႅတ်ႉ Memory လႆႈလီၶႃႈ)။
