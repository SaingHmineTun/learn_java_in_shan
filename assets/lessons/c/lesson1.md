## Lesson 1: The World of C: History & Philosophy

### 1. C ၼႆႉ ပဵၼ်သင်? (What is C?)
C ၼႆႉ ပဵၼ်လိၵ်ႈပူဝ်ႇၵရႅမ်ႇမိင်း ဢၼ်မီးဢဵၼ်ႁႅင်းယႂ်ႇ လႄႈ ၸႂ်ႉတႅမ်ႈလႆႈၸမ်ၵၼ်တင်း ၶိူင်ႈၸၢၵ်ႈ (Hardware) ၶႃႈဢေႃႈ။ ၼႂ်းလုမ်ႈၾႃႉ IT ၼႆႉ ၶဝ်ႁွင်ႉလိၵ်ႈ C ဝႃႈ **"မႄႈလိၵ်ႈပူဝ်ႇၵရႅမ်ႇမိင်း တင်းသဵင်ႈ"** (The Mother of all Languages) ၼႆၶႃႈ။ ၵွပ်ႈဝႃႈ လိၵ်ႈပၢၼ်မႂ်ႇမိူၼ်ၼင်ႇ Java, Python ဢမ်ႇၼၼ် Dart ဢၼ်ၸႂ်ႉတႅမ်ႈ Flutter ၼၼ်ႉၵေႃႈ လႆႈဢဝ်ပိုၼ်ႉฐานမၼ်းမႃးတီႈ C တင်းၼမ်ၶႃႈဢေႃႈ။



### 2. ပိုၼ်းတၢင်းလင် (A Brief History)
* **ၵေႃႉၵေႃႇတင်ႈ:** Dennis Ritchie (တႅၼ်းၼိတ်ႉ ရိတ်ႉၶျီႇ) တီႈႁွင်ႈၵၢၼ် Bell Labs။
* **ပီႇဢၼ်ဢွၵ်ႇမႃး:** ပီ 1972။
* **ယိူင်းဢၢၼ်း:** မၼ်းလႆႈတႅမ်ႈဢွၵ်ႇမႃး တႃႇတေၸႂ်ႉႁဵတ်းသၢင်ႈ Operating System (OS) ဢၼ်ၸိုဝ်ႈဝႃႈ **Unix** ၼၼ်ႉၶႃႈ။ တေႃႈလဵဝ် ပိူင်ၾႆးပိူင်လူင် (Kernel) ၶွင် Windows, macOS လႄႈ Linux ၸိူဝ်းၼႆႉၵေႃႈ တႅမ်ႈဝႆႉတင်း C ပဵၼ်ပိူင်လူင်ၶႃႈဢေႃႈ။

### 3. ပၢႆးဝူၼ်ႉၶွင် C (The Philosophy)
ပၢႆးဝူၼ်ႉၶွင် C ၼႆႉ ပႅၵ်ႇပိူင်ႈတင်း လိၵ်ႈပၢၼ်မႂ်ႇၶႃႈ။ မၼ်းမီးလွင်ႈယုမ်ႇယမ်ဝႃႈ **"Trust the Programmer"** (ၵေႃႉတႅမ်ႈ Code ၼၼ်ႉ ႁူႉလီသေပိူၼ်ႈ) ၼႆၶႃႈဢေႃႈ။
* **လွင်ႈၽႂ်း (Speed):** ၵွပ်ႈဝႃႈ မၼ်းဢမ်ႇမီး "ၵူၼ်းၵၢင်" တႃႇတေပိၼ်ႇၽႃႇသႃႇၼမ်ၼၼ်ႉ မၼ်းႁဵတ်းၵၢၼ်လႆႈၽႂ်းမိူၼ်ၸၢၵ်ႈတေႉတေႉၶႃႈ။
* **လွင်ႈၵုမ်းၵမ် (Control):** ၸဝ်ႈၵဝ်ႇ ၵုမ်းၵမ်လႆႈ Memory (RAM) ၶွင်ၶွမ်ႇပိဝ်ႇတႃႇလႆႈ တူဝ်တူဝ် လႄႈ ၸတ်းၵၢၼ်လႆႈၸွမ်းၼင်ႇၸႂ်ၶႂ်ႈပဵၼ်ၶႃႈ။
* **ၵႂႃႇလႆႈၵူႈတီႈ (Portability):** Code ဢၼ်တႅမ်ႈဝႆႉတင်း C ၼၼ်ႉ ဢဝ်ၵႂႃႇ Run တီႈၶိူင်ႈၸၢၵ်ႈသင်ၵေႃႈလႆႈ—တႄႇဢဝ်ၸၢၵ်ႈသၵ်းၶူဝ်း တေႃႇပေႃးထိုင် ၶွမ်ႇပိဝ်ႇတႃႇလူင် ၼၼ်ႉၶႃႈဢေႃႈ။

### 4. ၵွပ်ႈသင် လုၵ်ႈႁဵၼ်းတႆး ထိုင်တေလႆႈႁဵၼ်း C?
သင်ၸဝ်ႈၵဝ်ႇ ၶႂ်ႈပဵၼ်ၵေႃႉၵေႃႇသၢင်ႈ Software ဢၼ်ၶႅမ်ႉလႅပ်ႈ (Architect) တွၼ်ႈတႃႇၶိူဝ်းၵူၼ်းတႆးႁဝ်း—မိူၼ်ၼင်ႇ ၶိူင်ႈမိုဝ်းတႅမ်ႈလိၵ်ႈတႆး (Typing Tutor) ဢမ်ႇၼၼ် ပိူင်ၸတ်းၵၢၼ်လုၵ်ႈႁဵၼ်းၼၼ်ႉ—လိၵ်ႈ C တေၸွႆႈပၼ်ႁႂ်ႈၸဝ်ႈၵဝ်ႇ ပွင်ႇၸႂ်လွင်ႈ "ပိူင်ၾႆးၶွမ်ႇပိဝ်ႇတႃႇ" လႆႈလိုၵ်ႉလိုၵ်ႉသေ လိၵ်ႈတၢင်ႇၸိူဝ်းၶႃႈဢေႃႈ။ မၼ်းတေသွၼ်ပၼ်ႁဝ်း ႁႂ်ႈပဵၼ်ၵေႃႉ "ၵေႃႇသၢင်ႈ" (Creator) မဟုတ်ဘဲ "ၵေႃႉၸႂ်ႉ" (User) ၵူၺ်းၶႃႈ။
