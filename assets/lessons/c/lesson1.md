## Lesson 1: The World of C: History & Philosophy

### 1. C ၼႆႉ ပဵၼ်သင်? (What is C?)
C ၼႆႉ ပဵၼ်လိၵ်ႈပူဝ်ႇၵရႅမ်ႇမိင်း ဢၼ်မီးဢဵၼ်ႁႅင်းယႂ်ႇ လႄႈ ၸႂ်ႉတႅမ်ႈလႆႈၸမ်ၵၼ်တင်း ၶိူင်ႈၸၢၵ်ႈ (Hardware) ၶႃႈဢေႃႈ။ 
ၼႂ်းလုမ်ႈၾႃႉ IT ၼႆႉ ၶဝ်ႁွင်ႉလိၵ်ႈ C ဝႃႈ **"မႄႈလိၵ်ႈပူဝ်ႇၵရႅမ်ႇမိင်း တင်းသဵင်ႈ"** (The Mother of all Languages) ၼႆၶႃႈ။ 
ၵွပ်ႈဝႃႈ လိၵ်ႈပၢၼ်မႂ်ႇမိူၼ်ၼင်ႇ Java, Python ဢမ်ႇၼၼ် Dart ဢၼ်ၸႂ်ႉတႅမ်ႈ Flutter ၼၼ်ႉၵေႃႈ လႆႈဢဝ်ပိုၼ်ႉထၢၼ်မၼ်းမႃးတီႈ C တင်းၼမ်ၶႃႈဢေႃႈ။



### 2. ပိုၼ်းတၢင်းလင် (A Brief History)
* **ၵေႃႉၵေႃႇတင်ႈ:** Dennis Ritchie (တႅၼ်းၼိတ်ႉ ရိတ်ႉၶျီႇ) တီႈႁွင်ႈၵၢၼ် Bell Labs။
* **ပီႇဢၼ်ဢွၵ်ႇမႃး:** ပီ 1972။
* **ယိူင်းဢၢၼ်း:** မၼ်းလႆႈတႅမ်ႈဢွၵ်ႇမႃး တႃႇတေၸႂ်ႉႁဵတ်းသၢင်ႈ Operating System (OS) ဢၼ်ၸိုဝ်ႈဝႃႈ **Unix** ၼၼ်ႉၶႃႈ။ တေႃႈလဵဝ် ပိူင်ၾႆးပိူင်လူင် (Kernel) ၶွင် Windows, macOS လႄႈ Linux ၸိူဝ်းၼႆႉၵေႃႈ တႅမ်ႈဝႆႉတင်း C ပဵၼ်ပိူင်လူင်ၶႃႈဢေႃႈ။

### 3. ပၢႆးဝူၼ်ႉၶွင် C (The Philosophy)
ပၢႆးဝူၼ်ႉၶွင် C ၼႆႉ ပႅၵ်ႇပိူင်ႈတင်း လိၵ်ႈပၢၼ်မႂ်ႇၶႃႈ။ 
မၼ်းမီးလွင်ႈယုမ်ႇယမ်ဝႃႈ **"Trust the Programmer"** (ၵေႃႉတႅမ်ႈ Code ၼၼ်ႉ ႁူႉလီသေပိူၼ်ႈ) ၼႆၶႃႈဢေႃႈ။
* **လွင်ႈၽႂ်း (Speed):** ၵွပ်ႈဝႃႈ မၼ်းဢမ်ႇမီး "ၵူၼ်းၵၢင်" တႃႇတေပိၼ်ႇၽႃႇသႃႇၼမ်ၼၼ်ႉ မၼ်းႁဵတ်းၵၢၼ်လႆႈၽႂ်းမိူၼ်ၸၢၵ်ႈတေႉတေႉၶႃႈ။
* **လွင်ႈၵုမ်းၵမ် (Control):** ၸဝ်ႈၵဝ်ႇ ၵုမ်းၵမ်လႆႈ Memory (RAM) ၶွင်ၶွမ်ႇပိဝ်ႇတႃႇလႆႈ တူဝ်တူဝ် လႄႈ ၸတ်းၵၢၼ်လႆႈၸွမ်းၼင်ႇၸႂ်ၶႂ်ႈပဵၼ်ၶႃႈ။
* **ၵႂႃႇလႆႈၵူႈတီႈ (Portability):** Code ဢၼ်တႅမ်ႈဝႆႉတင်း C ၼၼ်ႉ ဢဝ်ၵႂႃႇ Run တီႈၶိူင်ႈၸၢၵ်ႈသင်ၵေႃႈလႆႈ—တႄႇဢဝ်ၸၢၵ်ႈသၵ်းၶူဝ်း တေႃႇပေႃးထိုင် ၶွမ်ႇပိဝ်ႇတႃႇလူင် ၼၼ်ႉၶႃႈဢေႃႈ။

### 4. C ပဵၼ် Procedural Language (ပိူင်တႅမ်ႈၸွမ်းၶၵ်ႉတွၼ်ႈ)
C ၼႆႉ မၼ်းပဵၼ် **Procedural Programming Language** ၶႃႈဢေႃႈ။
* **မၼ်းပွင်ႇဝႃႈသင်?** မၼ်းပွင်ႇဝႃႈ ၵၢၼ်တႅမ်ႈ Code ၼၼ်ႉ မိူၼ်တင်းႁဝ်းတႅမ်ႈ "သဵၼ်ႈမၢႆၵၢၼ်ႁဵတ်း" (A list of instructions) ႁႂ်ႈၶွမ်ႇပိဝ်ႇတႃႇႁဵတ်းၸွမ်း တႄႇၼိူဝ်လူင်းတႂ်ႈ ၼိုင်ႈထႅဝ်ယဝ်ႉ ၼိုင်ႈထႅဝ်ၶႃႈ။
* **လွင်ႈၸႅၵ်ႇတွၼ်ႈ (Functions):** ၼႂ်း Procedural Programming ၼႆႉ ႁဝ်းတေၸႅၵ်ႇပၼ်ႁႃယႂ်ႇယႂ်ႇ ႁႂ်ႈပဵၼ်တွၼ်ႈဢွၼ်ႇ ဢၼ်ႁွင်ႉဝႃႈ **Functions** (ဢမ်ႇၼၼ် Procedures) ၶႃႈ။ မၼ်းမိူၼ်တင်း ႁဝ်းမီးၵူၼ်းၸွႆႈၵၢၼ် လၢႆလၢႆၵေႃႉ၊ ၵေႃႉၼိုင်ႈႁဵတ်းၵၢၼ်ၼိုင်ႈမဵဝ်း သေႁွင်ႉၸႂ်ႉၵၼ်ၵႂႃႇၸွမ်းၶၵ်ႉမၼ်းၶႃႈ။
* **ၵွပ်ႈသင်ႁဝ်းထုၵ်ႇလီႁဵၼ်း?** ၵၢၼ်တႅမ်ႈၸွမ်းၶၵ်ႉတွၼ်ႈၼႆႉ သွၼ်ပၼ်ႁဝ်းႁႂ်ႈမီး "ပၢႆးဝူၼ်ႉဢၼ်မီးလွၵ်းလၢႆး" (Logical Thinking)။ သင်ၸဝ်ႈၵဝ်ႇပွင်ႇၸႂ်လွင်ႈ Procedural ၼႂ်း C ယဝ်ႉ၊ မိူဝ်ႈၵႂႃႇႁဵၼ်း Object-Oriented (မိူၼ်ၼင်ႇ Java/Dart) ၼၼ်ႉ ၸဝ်ႈၵဝ်ႇတေႁၼ်လွင်ႈပႅၵ်ႇပိူင်ႈမၼ်းလႆႈၸႅၼ်ႇလႅင်းသေ ႁဵတ်းႁႂ်ႈပဵၼ် Programmer ဢၼ်ၵႅၼ်ႇၶႅင်ၶႃႈဢေႃႈ။

### 💡 ၵွပ်ႈသင် Windows တင်း Linux တိုၵ်ႉၸႂ်ႉ C ယူႇ?
> ၵွပ်ႈဝႃႈ C မၼ်းပဵၼ် **Procedural** ဢၼ်ၸမ်တင်း Hardware သုတ်း။ ၵၢၼ်ၵုမ်းၵမ် Memory လႄႈ CPU ၼၼ်ႉ လူဝ်ႇလွင်ႈ "တႅတ်ႈတေႃး" (Precision) လႄႈ "လွင်ႈၽႂ်း" (Efficiency) ဢၼ်လိၵ်ႈပၢၼ်မႂ်ႇ (High-level) တိုၵ်ႉႁဵတ်းဢမ်ႇလႆႈၵႅၼ်ႇမိူၼ် C ၼၼ်ႉၶႃႈဢေႃႈ။
