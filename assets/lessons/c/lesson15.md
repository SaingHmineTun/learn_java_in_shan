## Lesson 15: Relational & Logical: Making Decisions

### 1. တူဝ်မၢႆၵၢၼ်တႅၵ်ႈၼိူင်း (Relational Operators)
ႁဝ်းၸႂ်ႉတူဝ်မၢႆၸိူဝ်းၼႆႉ တႃႇတူၺ်း "လွင်ႈၵပ်းသိုပ်ႇ" ၼႂ်းၵႄႈတူဝ်ၼပ်ႉသွင်တူဝ်ၶႃႈ။ တွပ်ႇမၼ်းတေဢွၵ်ႇမႃးပဵၼ် **1 (True)** ဢမ်ႇၼၼ် **0 (False)** ၶႃႈဢေႃႈ။

| Operator | တီႈပွင်ႇ (Meaning)          | တူဝ်ယၢင်ႇ (Example) |
| :--- |:----------------------------| :--- |
| **`==`** | မိူၼ်ၵၼ် (Equal to)         | `5 == 5` (1) |
| **`!=`** | ဢမ်ႇမိူၼ်ၵၼ် (Not equal to) | `5 != 3` (1) |
| **`>`** | ယႂ်ႇသေ (Greater than)       | `10 > 5` (1) |
| **`<`** | ဢွၼ်ႇသေ (Less than)         | `2 < 8` (1) |
| **`>=`** | ယႂ်ႇသေ ဢမ်ႇၼၼ် မိူၼ်ၵၼ်     | `10 >= 10` (1) |
| **`<=`** | ဢွၼ်ႇသေ ဢမ်ႇၼၼ် မိူၼ်ၵၼ်    | `5 <= 4` (0) |



**ၾၢင်ႉ:** လူဝ်ႇတွၼ်းမၢႆဝႆႉဝႃႈ `==` (တူဝ်မၢႆမိူၼ်ၵၼ်) ပႅၵ်ႇပိူင်ႈတင်း `=` (ၵၢၼ်ပၼ်ၵၢၼ်) ၶႃႈဢေႃႈ။

### 2. တူဝ်မၢႆၵၢၼ်ဝူၼ်ႉ (Logical Operators)
မိူဝ်ႈႁဝ်းၶႂ်ႈတူၺ်း "လွင်ႈတႅပ်းတတ်း" လၢႆလၢႆယိူင်ႈၸွမ်းၵၼ်၊ ႁဝ်းလူဝ်ႇၸႂ်ႉ Logical Operators ၶႃႈ။

* **`&&` (AND)**: တေပဵၼ် True (1) မိူဝ်ႈ **တင်းသွင်ၽၢႆႇ** ပဵၼ် True။
    * *တူဝ်ယၢင်ႇ:* `(5 > 3) && (10 < 20)` -> 1
* **`||` (OR)**: တေပဵၼ် True (1) မိူဝ်ႈ **ၽၢႆႇလႂ်ၽၢႆႇၼိုင်ႈ** ပဵၼ် True။
    * *တူဝ်ယၢင်ႇ:* `(5 > 10) || (10 < 20)` -> 1
* **`!` (NOT)**: ပိၼ်ႇ True ႁႂ်ႈပဵၼ် False (ဢမ်ႇၼၼ် ပိၼ်ႇ False ႁႂ်ႈပဵၼ် True)။
    * *တူဝ်ယၢင်ႇ:* `!(5 == 5)` -> 0



### 3. Boolean ၼႂ်း C (The 0 and 1 Rule)
မိူၼ်ၼင်ႇႁဝ်းႁဵၼ်းမႃးၼႂ်း Module 2 ၼၼ်ႉၶႃႈ၊ ၼႂ်း C ဢမ်ႇမီး `true` ဢမ်ႇၼၼ် `false` တူဝ်တေႉမၼ်း။
* **`0`** = **False**
* **တူဝ်ၼပ်ႉတၢင်ႇၸိူဝ်း (1, 2, -100)** = **True**

**တူဝ်ယၢင်ႇ Code:**
```c
int age = 20;
int isAdult = (age >= 18); // isAdult တေပဵၼ် 1 ၶႃႈ
```

### 4. Short-Circuit Evaluation
ၼႆႉပဵၼ်လွင်ႈ "လႅတ်ႈ" ၶွင် Compiler ၶႃႈ။
* ၼႂ်းၵၢၼ်ၸႂ်ႉ **`&&`**: သင်ၽၢႆႇသၢႆႉပဵၼ် False ယဝ်ႉ၊ မၼ်းတေဢမ်ႇၵႂႃႇတူၺ်းၽၢႆႇၶႂႃထႅင်ႈ ၵွပ်ႈမၼ်းႁူႉဝႃႈ တွပ်ႇမၼ်းတေပဵၼ် False တူဝ်ႈတူဝ်ႈၶႃႈယဝ်ႉ။
* ၼႂ်းၵၢၼ်ၸႂ်ႉ **`||`**: သင်ၽၢႆႇသၢႆႉပဵၼ် True ယဝ်ႉ၊ မၼ်းၵေႃႈတေဢမ်ႇၵႂႃႇတူၺ်းၽၢႆႇၶႂႃထႅင်ႈၶႃႈ။
