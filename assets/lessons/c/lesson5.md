## Lesson 5: Errors & Debugging: Finding the Bugs

### 1. လွင်ႈၽိတ်း 3 မဵဝ်း ၼႂ်း C (3 Types of Errors)
မိူဝ်ႈႁဝ်းတႅမ်ႈ Code ၼၼ်ႉ ႁဝ်းတေလႆႈထူပ်းလွင်ႈၽိတ်း 3 ပိူင်လူင်ၶႃႈ။

#### ၵ) Syntax Errors (လွင်ႈတႅမ်ႈမၢႆမီႈၽိတ်း)
ၼႆႉပဵၼ်လွင်ႈၽိတ်း ဢၼ် Compiler (GCC) ၵူတ်ႇထတ်းႁၼ်လႆႈ ဢွၼ်တၢင်းသုတ်း။
* **လွင်ႈတၢင်း:** လိုမ်းသႂ်ႇ `;`, လိုမ်းပိၵ်ႉ `{ }`, ဢမ်ႇၼၼ် တႅမ်ႈၸိုဝ်ႈ Function ၽိတ်း (မိူၼ်ၼင်ႇ `prinf` တႅၼ်း `printf`)။
* **လၢႆးၵႄႈ:** တူၺ်းတီႈ Terminal ဢၼ် Compiler ၸီႉၼႄဝႃႈ မၼ်းၽိတ်းတီႈ "Line Number" ၵေႃႉလႂ်။

#### ၶ) Runtime Errors (လွင်ႈၽိတ်းမိူဝ်ႈ Run ယူႇ)
Code ႁဝ်းမၢၼ်ႇယဝ်ႉ၊ Compile လႆႈယဝ်ႉ၊ ၵူၺ်းၵႃႈ မိူဝ်ႈတိုၵ်ႉ Run ယူႇၼၼ်ႉ ပူဝ်ႇၵရႅမ်ႇသမ်ႉ "ၵိုတ်းၵႂႃႇ" (Crash)။
* **လွင်ႈတၢင်း:** ဢဝ်တူဝ်ၼပ်ႉၽႄတင်း `0`, ဢမ်ႇၼၼ် ၸႂ်ႉ Memory ဢၼ်ဢမ်ႇမီးသုၼ်ႇၸႂ်ႉ (Segmentation Fault)။
* **လၢႆးၵႄႈ:** လူဝ်ႇၵူတ်ႇထတ်း Logic ၼႂ်း Code ႁဝ်းၶိုၼ်းလီလီ။

#### င) Logical Errors (လွင်ႈဝူၼ်ႉၽိတ်း)
ၼႆႉပဵၼ်လွင်ႈဢၼ် "ယၢပ်ႇသုတ်း"။ ပူဝ်ႇၵရႅမ်ႇဢမ်ႇ Crash၊ Compile ၵေႃႈလႆႈ၊ ၵူၺ်းၵႃႈ "တွပ်ႇမၼ်းဢွၵ်ႇမႃးၽိတ်း"။
* **တူဝ်ယၢင်ႇ:** ၸဝ်ႈၵဝ်ႇၶႂ်ႈဢဝ် `5 + 5` ၵူၺ်းၵႃႈ ၼႂ်း Code သမ်ႉတႅမ်ႈဝႆႉ `5 * 5`။ တွပ်ႇမၼ်းတေဢွၵ်ႇ `25` တႅၼ်း `10` ၼၼ်ႉၶႃႈဢေႃႈ။
* **လၢႆးၵႄႈ:** လူဝ်ႇၸႂ်ႉလၢႆး Debugging တႃႇတူၺ်းၶေႃႈမုၼ်း ထႅဝ်ၼိုင်ႈယဝ်ႉထႅဝ်ၼိုင်ႈ။



### 2. Debugging ၼႆႉ ပဵၼ်သင်? (What is Debugging?)
**Debugging** ၼႆႉ ပဵၼ်ၶၵ်ႉတွၼ်ႈၵၢၼ် "ႁႃ" လႄႈ "ၵႄႈလႅတ်ႈ" လွင်ႈၽိတ်း (Bugs) ၼၼ်ႉၶႃႈ။ ၸဝ်ႈၵဝ်ႇပဵၼ် "ၵူၼ်းႁႃမႅင်း" ၼႂ်း Code ၸဝ်ႈၵဝ်ႇၼၼ်ႉၶႃႈဢေႃႈ။

### 3. လၢႆး Debug ငၢႆႈငၢႆႈ (Simple Debugging Techniques)

1.  **Print Debugging:** ၸႂ်ႉ `printf()` တႃႇၼႄၶေႃႈမုၼ်း (Variable Values) ၼႂ်းၵႄႈထႅဝ် Code ႁဝ်း။ တွၼ်ႈတႃႇတူၺ်းဝႃႈ ပူဝ်ႇၵရႅမ်ႇႁဝ်း ႁဵတ်းၵၢၼ်ထိုင်တီႈလႂ်ယဝ်ႉ။
    * *Example:* `printf("Debug: value of x is %d\n", x);`
2.  **Using a Debugger (GDB):** ၸႂ်ႉၶိူင်ႈမိုဝ်းမိူၼ်ၼင်ႇ **GDB** (GNU Debugger) ဢၼ်မီးၼႂ်း VS Code။ မၼ်းတေၸွႆႈႁႂ်ႈႁဝ်း:
    * **Breakpoints:** သင်ႇႁႂ်ႈပူဝ်ႇၵရႅမ်ႇၵိုတ်းဝႆႉ တီႈထႅဝ်ဢၼ်ႁဝ်းၶႂ်ႈတူၺ်း။
    * **Step Over/Into:** ႁႂ်ႈပူဝ်ႇၵရႅမ်ႇ ႁဵတ်းၵၢၼ်ၵႂႃႇၼႃႈ ထႅဝ်ၼိုင်ႈယဝ်ႉထႅဝ်ၼိုင်ႈ။
    * **Watch Variables:** တူၺ်း Variable ႁဝ်းဝႃႈ မၼ်းလႅၵ်ႈလၢႆႈၵႂႃႇၸိူဝ်ႉႁိူဝ်။
