## Lesson 28: Introduction to Arrays: Homogeneous Collections

### 1. Array ၼႆႉ ပဵၼ်သင်? (The Concept)
Array ၼႆႉ ပဵၼ်ၵွၵ်းၵဵပ်းၶေႃႈမုၼ်း (Variable) ဢၼ်မီး **Data Type မိူၼ်ၵၼ်** (Homogeneous) လၢႆလၢႆတူဝ် ႁူမ်ႈဝႆႉၼႂ်းၸိုဝ်ႈလဵဝ်ၵၼ်။
* **Homogeneous:** သင်ပဵၼ် `int` ၵေႃႈ တေလႆႈပဵၼ် `int` တင်းသဵင်ႈ၊ သင်ပဵၼ် `float` ၵေႃႈ တေလႆႈပဵၼ် `float` တင်းသဵင်ႈၶႃႈ။



### 2. ၵၢၼ်ပိုၼ်ၽၢဝ်ႇ (Declaration & Initialization)
မိူဝ်ႈႁဝ်းတေၸႂ်ႉ Array၊ ႁဝ်းလူဝ်ႇလၢတ်ႈၼႄဝႃႈ မၼ်းတေၵဵပ်းလႆႈ "လၢႆတူဝ်" (Size) ၶႃႈ။

**Syntax:** `type arrayName[size];`

```c
int marks[5]; // ၵဵပ်းတူဝ်ၼပ်ႉတဵမ်လႆႈ 5 တူဝ်
int scores[3] = {90, 85, 70}; // တင်ႈတူဝ်ၼပ်ႉဝႆႉၵမ်းလဵဝ်
```

### 3. Array Index (မၢႆႁွင်ႉ)
ၼႆႉပဵၼ်တွၼ်ႈဢၼ်လမ်ႇလွင်ႈသုတ်းၶႃႈ။ ၼႂ်းလိၵ်ႈ C (လႄႈ Java/Dart)၊ ႁဝ်းတႄႇၼပ်ႉဢၢၼ်ႇ "ႁွင်ႉ" တီႈမၢႆ **0** တႃႇသေႇ။
* တူဝ်ထီႉ 1 ယူႇတီႈ `marks[0]`
* တူဝ်ထီႉ 2 ယူႇတီႈ `marks[1]`
* တူဝ်ထီႉ n ယူႇတီႈ `marks[n-1]`



### 4. Arrays ၼႂ်း Memory
Arrays ၼႂ်း C မၼ်းၵဵပ်းဝႆႉပဵၼ်ထႅဝ်သိုဝ်ႈၵၼ် (Contiguous Memory) ၼႂ်း RAM ၶႃႈ။
* သင် `marks[0]` ယူႇတီႈ Address `1000` ၼႆ
* `marks[1]` တေယူႇတီႈ Address `1004` (သင်ပဵၼ် `int` ဢၼ်မီး 4 Bytes) ၶႃႈဢေႃႈ။
* ၼႆႉၸင်ႇႁဵတ်းႁႂ်ႈ ၶွမ်ႇပိဝ်ႇတႃႇ ၶဝ်ႈၸူးၶေႃႈမုၼ်းလႆႈ "ၽႂ်း" တေႉတေႉၶႃႈ။

### 5. ၵၢၼ်ၸႂ်ႉ `for` loop တင်း Array
Array လႄႈ Loop ၼႆႉ ပဵၼ် "ၵူပ်ႉၵၼ်" (Best friends) ၶႃႈ။ ႁဝ်းၸႂ်ႉ Loop တႃႇၶဝ်ႈၸူး ဢမ်ႇၼၼ် ၼႄၶေႃႈမုၼ်းၼႂ်း Array တင်းသဵင်ႈလႆႈငၢႆႈငၢႆႈၶႃႈ။

```c
int prices[4] = {100, 200, 300, 400};

for (int i = 0; i < 4; i++) {
    printf("Item %d: %d\n", i, prices[i]);
}
```
