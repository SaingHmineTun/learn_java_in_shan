## Lesson 23: while & do-while: Condition-first Loops

### 1. `while` Loop (ၵူတ်ႇထတ်းဢွၼ်တၢင်း ၸင်ႇႁဵတ်း)
`while` loop ၼႆႉ မိူၼ်ၼင်ႇ "ၽၵ်းတူ" ဢၼ်မီးၵူၼ်းယၢမ်းၶႃႈ။ မၼ်းတေၵူတ်ႇထတ်း Condition ဢွၼ်တၢင်းသုတ်း၊ သင်မၢၼ်ႇ (True) ၸင်ႇတေပၼ်ၶဝ်ႈၵႂႃႇႁဵတ်းၵၢၼ်ၼႂ်း `{ }` ၶႃႈ။

**Syntax:**
```c
while (condition) {
    // Code ႁဵတ်းၵၢၼ်တီႈၼႆႈ တေႃႇပေႃး Condition ပဵၼ် False
}
```



**တူဝ်ယၢင်ႇ:** ၼပ်ႉ 1 တေႃႇ 5
```c
int i = 1;
while (i <= 5) {
    printf("%d ", i);
    i++; // လူဝ်ႇထႅမ်တူဝ်ၼပ်ႉ တႃႇႁႂ်ႈ Loop ယဝ်ႉတူဝ်ႈ
}
```

### 2. `do-while` Loop (ႁဵတ်းဢွၼ်တၢင်း ၸင်ႇၵူတ်ႇထတ်း)
`do-while` ၼႆႉ သမ်ႉပႅၵ်ႇပိူင်ႈၵၼ်ဢိတ်းၼိုင်ႈၶႃႈ။ မၼ်းတေ **ႁဵတ်းၵၢၼ်ဢွၼ်တၢင်းသုတ်း 1 ၵမ်း**၊ ယဝ်ႉၸင်ႇၵူတ်ႇထတ်း Condition ဝႃႈ တေႁဵတ်းၶိုၼ်းထႅင်ႈပွၵ်ႈႁိုဝ်?

**Syntax:**
```c
do {
    // Code တေႁဵတ်းၵၢၼ်ဢွၼ်တၢင်း 1 ၵမ်းတႃႇသေႇ
} while (condition); 
```
*(သတိ: လူဝ်ႇသႂ်ႇ `;` တီႈလင် `while` ၶွင် do-while ၶႃႈ)*



### 3. လွင်ႈပႅၵ်ႇပိူင်ႈ (Key Difference)
* **`while`**: သင် Condition ၽိတ်းတႄႇဢဝ်ႁူဝ်ที၊ မၼ်းတေ **ဢမ်ႇႁဵတ်းၵၢၼ်သင်သေဢိတ်း** ၶႃႈ။
* **`do-while`**: မၼ်းတေ **ႁဵတ်းၵၢၼ် 1 ၵမ်းတႃႇသေႇ**၊ ဢမ်ႇဝႃႈ Condition တေမၢၼ်ႇ ႁိုဝ် ၽိတ်းၵေႃႈယဝ်ႉၶႃႈ။

### 4. Infinite Loop (Loop ဢၼ်ဢမ်ႇမီးဝၼ်းၵိုတ်း)
သင်ၸဝ်ႈၵဝ်ႇလိုမ်း "ထႅမ်တူဝ်ၼပ်ႉ" (i++) ဢမ်ႇၼၼ် Condition ၸဝ်ႈၵဝ်ႇပဵၼ် True တႃႇသေႇၼႆ၊ ပူဝ်ႇၵရႅမ်ႇတေႁဵတ်းၵၢၼ်ဢမ်ႇၵိုတ်းသေ ႁဵတ်းႁႂ်ႈၶွမ်ႇပိဝ်ႇတႃႇၼၵ်းၶႃႈ။
* *တူဝ်ယၢင်ႇ:* `while(1) { ... }` ၼႆႉ တေပဵၼ် Loop ဢၼ်ႁဵတ်းၵၢၼ်တႃႇသေႇၶႃႈ။
