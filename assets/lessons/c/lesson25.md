## Lesson 25: Break & Continue: The Flow Controllers

### 1. တူဝ်မၢႆ `break` (ၵၢၼ်ထွၼ်တူဝ်ၵမ်းလဵဝ်)
`break` ၼႆႉ ၸႂ်ႉတွၼ်ႈတႃႇ "ပိတ်း" Loop တင်းသဵင်ႈသေ ဢွၵ်ႇမႃးၽၢႆႇၼွၵ်ႈၵမ်းလဵဝ်၊ ဢမ်ႇဝႃႈ Condition တေမၢၼ်ႇယူႇၵေႃႈယဝ်ႉၶႃႈ။

* **ၸႂ်ႉမိူဝ်ႈလႂ်?**: မိူဝ်ႈႁဝ်းႁႃၶေႃႈမုၼ်း (Search) ႁၼ်ယဝ်ႉ၊ ဢမ်ႇၼၼ် မိူဝ်ႈမီးလွင်ႈၽိတ်းပိူင်ႈ (Error) သေ လူဝ်ႇၵိုတ်းပူဝ်ႇၵရႅမ်ႇၶႃႈ။

**တူဝ်ယၢင်ႇ:**
```c
for (int i = 1; i <= 10; i++) {
    if (i == 5) {
        break; // တေၵိုတ်းမိူဝ်ႈထိုင် 5
    }
    printf("%d ", i);
}
// တွပ်ႇ: 1 2 3 4
```



### 2. တူဝ်မၢႆ `continue` (ၵၢၼ်ၶၢမ်ႈၵႂႃႇ Step တၢင်ႇၼိုင်ႈ)
`continue` သမ်ႉ ပႅၵ်ႇပိူင်ႈတင်း `break` ၶႃႈ။ မၼ်းတေဢမ်ႇပိတ်း Loop၊ မၼ်းတေတေႃႉတိုၵ်ႉ "ၶၢမ်ႈ" Code ဢၼ်ၵိုတ်းဝႆႉၼႂ်းတွၼ်ႈၼၼ်ႉသေ ၵႂႃႇတႄႇ Step တၢင်ႇၼိုင်ႈ (Iteration) ၵမ်းလဵဝ်ၶႃႈ။

* **ၸႂ်ႉမိူဝ်ႈလႂ်?**: မိူဝ်ႈႁဝ်းၶႂ်ႈ "ဝႄႉ" ဢမ်ႇၼၼ် ဢမ်ႇၶႂ်ႈၼပ်ႉဢၢၼ်ႇ ၶေႃႈမုၼ်းမၢင်ၸိူဝ်းၶႃႈ။

**တူဝ်ယၢင်ႇ:**
```c
for (int i = 1; i <= 5; i++) {
    if (i == 3) {
        continue; // တေၶၢမ်ႈမၢႆ 3 ၵႂႃႇ
    }
    printf("%d ", i);
}
// တွပ်ႇ: 1 2 4 5
```



### 3. လွင်ႈပႅၵ်ႇပိူင်ႈ (Key Difference)
| Feature | **`break`** | **`continue`** |
| :--- | :--- | :--- |
| **Loop Status** | ပိတ်း Loop တင်းသဵင်ႈ (Stop) | သိုပ်ႇႁဵတ်း Loop ထႅင်ႈ (Keep going) |
| **Code Next** | ၵျွၵ်းၵႂႃႇ Code တႂ်ႈ Loop | ၵျွၵ်းၵႂႃႇ Step တၢင်ႇၼိုင်ႈ (Next i) |

### 4. သတိလွင်ႈၸႂ်ႉ `continue` ၼႂ်း `while` Loop
မိူဝ်ႈၸႂ်ႉ `continue` ၼႂ်း `while` loop ၼၼ်ႉ လူဝ်ႇသတိဝႆႉဝႃႈ ၸဝ်ႈၵဝ်ႇလႆႈထႅမ်တူဝ်ၼပ်ႉ (`i++`) ဝႆႉ **တီႈၼႃႈ** `continue` ႁိုဝ်ဢမ်ႇၸႂ်? သင်ဢမ်ႇသႂ်ႇဝႆႉၼႆ ပူဝ်ႇၵရႅမ်ႇတေပဵၼ် **Infinite Loop** ၵွပ်ႈမၼ်းဝၼ်ႇပၼ်ႇယူႇတီႈမၢႆၵဝ်ႇၼၼ်ႉတႃႇသေႇၶႃႈ။
