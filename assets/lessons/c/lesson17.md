## Lesson 17: Bit Manipulation: Flags & Masks

### 1. Bit Flags ၼႆႉ ပဵၼ်သင်?
ႁႂ်ႈဝူၼ်ႉတူၺ်းဝႃႈ ၸဝ်ႈၵဝ်ႇမီး Status 8 မဵဝ်း ဢၼ်ပဵၼ် "ပိုတ်ႇ/ပိတ်း" (ON/OFF)။
* သင်ၸႂ်ႉ `bool` 8 တူဝ် = ၸႂ်ႉ Memory 8 Bytes။
* သင်ၸႂ်ႉ **Bit Flags** = ၸႂ်ႉ Memory ၵူၺ်း **1 Byte** (`char`) ၶႃႈဢေႃႈ။
  ၼႂ်း 1 Byte ၼၼ်ႉ မီး 8 Bits၊ ႁဝ်းၵဵပ်း Status ဝႆႉလႆႈ 8 မဵဝ်း ၼႂ်း Bit ၼိုင်ႈတူဝ်လႂ်ၶႃႈ။



### 2. Bit Masking (ၵၢၼ်ၸႂ်ႉတူဝ်တူပ်ႉ)
**Mask** ၼႆႉ ပဵၼ်တူဝ်ၼပ်ႉ Binary ဢၼ်ႁဝ်းသၢင်ႇႁႄႉဝႆႉ တွၼ်ႈတႃႇ "ပိုတ်ႇ၊ ပိတ်း ဢမ်ႇၼၼ် ၵူတ်ႇထတ်း" Bit တူဝ်ဢၼ်ႁဝ်းၶႂ်ႈလႆႈၼၼ်ႉၶႃႈ။

#### ၵ) Setting a Bit (ၵၢၼ်ပိုတ်ႇ Bit - ၸႂ်ႉ `|`)
သင်ၶႂ်ႈပိုတ်ႇ Bit တူဝ်ထီႉ 3 (တႄႇၼပ်ႉတီႈ 0) ႁႂ်ႈပဵၼ် `1` တႃႇသေႇ:
```c
status = status | (1 << 3); 
```
*(ၸႂ်ႉ OR ၵွပ်ႈဝႃႈ 1 | သင်ၵေႃႈယဝ်ႉ တေပဵၼ် 1 တႃႇသေႇၶႃႈ)*

#### ၶ) Clearing a Bit (ၵၢၼ်ပိတ်း Bit - ၸႂ်ႉ `&` လႄႈ `~`)
သင်ၶႂ်ႈပိတ်း Bit တူဝ်ထီႉ 3 ႁႂ်ႈပဵၼ် `0` တႃႇသေႇ:
```c
status = status & ~(1 << 3);
```
*(ၸႂ်ႉ AND တင်း NOT တွၼ်ႈတႃႇ "တူပ်ႉ" တီႈ Bit ၼၼ်ႉႁႂ်ႈပဵၼ် 0 သေ တၢင်ႇၸိူဝ်းဝႆႉပဵၼ် 1 ၶႃႈ)*

#### င) Checking a Bit (ၵၢၼ်ၵူတ်ႇထတ်း Bit - ၸႂ်ႉ `&`)
သင်ၶႂ်ႈႁူႉဝႃႈ Bit တူဝ်ထီႉ 3 ပဵၼ် `1` ႁိုဝ် `0`:
```c
if (status & (1 << 3)) {
    printf("Bit is ON");
}
```



### 3. Bit Masking ၼႂ်းလုမ်ႈၾႃႉတေႉ
* **File Permissions:** ၼႂ်း Linux၊ သုၼ်ႇ "လူ၊ တႅမ်ႈ၊ ႁဵတ်းသၢင်ႈ" (Read, Write, Execute) ၵဵပ်းဝႆႉပဵၼ် Bit Flags ၶႃႈ။
* **Graphics:** ၵၢၼ်ၸတ်းၵၢၼ်သီ (RGB) ၼႂ်းၶွမ်ႇပိဝ်ႇတႃႇ ၵေႃႈၸႂ်ႉ Bit Manipulation ၶႃႈဢေႃႈ။

### 4. တူဝ်ယၢင်ႇ Code တူဝ်တေႉ
```c
#include <stdio.h>

#define READ_BIT    (1 << 0) // 0001
#define WRITE_BIT   (1 << 1) // 0010
#define EXECUTE_BIT (1 << 2) // 0100

int main() {
    unsigned char my_perm = 0; // 0000

    // ပိုတ်ႇသုၼ်ႇ Read လႄႈ Write
    my_perm |= READ_BIT;  // 0001
    my_perm |= WRITE_BIT; // 0011

    // ၵူတ်ႇထတ်းဝႃႈ မီးသုၼ်ႇ Write ႁိုဝ်?
    if (my_perm & WRITE_BIT) {
        printf("You have Write Permission!\n");
    }

    return 0;
}
```
