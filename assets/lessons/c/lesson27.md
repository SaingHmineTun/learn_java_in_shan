## Lesson 27: The goto Statement: Proceed with Caution

### 1. `goto` ၼႆႉ ပဵၼ်သင်?
`goto` ပဵၼ် Keyword ဢၼ်သင်ႇႁႂ်ႈပူဝ်ႇၵရႅမ်ႇ "ၵျွၵ်း" (Jump) တႄႇဢဝ်ထႅဝ်ၼိုင်ႈ ၵႂႃႇၸူးထႅင်ႈထႅဝ်ၼိုင်ႈ ဢၼ်ႁဝ်းမၢႆဝႆႉ (Label) ၼၼ်ႉၶႃႈဢေႃႈ။ မၼ်းဢမ်ႇတူၺ်း Condition ၼႂ်း Loop၊ မၼ်းၵျွၵ်းၵႂႃႇၵမ်းလဵဝ်ၶႃႈ။

### 2. ပိူင်တႅမ်ႈ (Syntax)
မၼ်းလူဝ်ႇမီးသွင်တွၼ်ႈၶႃႈ:
1.  **Label:** ၸိုဝ်ႈဢွင်ႈတီႈ ဢၼ်ႁဝ်းၶႂ်ႈၵျွၵ်းၵႂႃႇ (မၢႆဝႆႉလူၺ်ႈ `:` )။
2.  **goto command:** ၶေႃႈသင်ႇႁႂ်ႈၵျွၵ်းၵႂႃႇၸူး Label ၼၼ်ႉ။

```c
goto error_handler; // ၵျွၵ်းၵႂႃႇ

error_handler:     // Label
    printf("An error occurred!");
```



### 3. တူဝ်ယၢင်ႇ: ၵၢၼ်ဢွၵ်ႇ Nested Loop (Multi-level Break)
ၼႆႉပဵၼ်ဢွင်ႈတီႈ ဢၼ် `goto` မီးၽွၼ်းလီသုတ်းၶႃႈ။ မိူဝ်ႈႁဝ်းမီး Loop ၼမ်ၼမ်သေ ၶႂ်ႈဢွၵ်ႇမႃးတင်းသဵင်ႈၵမ်းလဵဝ် (ဢၼ် `break` ယူဝ်းယူဝ်းႁဵတ်းဢမ်ႇလႆႈ)။

```c
for (int i = 0; i < 10; i++) {
    for (int j = 0; j < 10; j++) {
        if (some_error) {
            goto end_loop; // ဢွၵ်ႇတင်းသွင် Loop ၵမ်းလဵဝ်
        }
    }
}
end_loop:
printf("Exited all loops.");
```

### 4. ၵွပ်ႈသင်ႁဝ်း "ဢမ်ႇထုၵ်ႇလီၸႂ်ႉ"? (The Spaghetti Code)
ၼႂ်းပၢႆး Software Engineering ၶဝ်သင်ႇဝႆႉဝႃႈ ယႃႇၸႂ်ႉ `goto` သင်ဢမ်ႇလူဝ်ႇတေႉတေႉ။
* **Unreadable:** Code တေၵျွၵ်းၵႂႃႇၵျွၵ်းမႃး တေႃႇပေႃးပွင်ႇၸႂ်ယၢပ်ႇ။
* **Hard to Debug:** မိူဝ်ႈပူဝ်ႇၵရႅမ်ႇမီး Bug၊ ႁဝ်းတေႁႃႁွႆးမၼ်းယၢပ်ႇ ၵွပ်ႈမၼ်းဢမ်ႇလႆၸွမ်းပိူင် (Flow) မၼ်းၶႃႈ။
* **Spaghetti Code:** ႁွင်ႉဝႃႈ Code ဢၼ်သုၵ်ႉယုင်ႈ မိူၼ်ၼင်ႇ "သဵၼ်ႈၶဝ်ႈသွႆး" ဢၼ်ႁဝ်းၵႄႈဢမ်ႇဢွၵ်ႇၼၼ်ႉၶႃႈဢေႃႈ။



### 5. မၢႆမီႈၵၢၼ်ၸႂ်ႉ (Rules of Thumb)
1. ၸႂ်ႉတွၼ်ႈတႃႇ **Error Handling** ၵူၺ်း (မိူၼ်ၼင်ႇ တီႈပၢႆ Function)။
2. ၸႂ်ႉတွၼ်ႈတႃႇဢွၵ်ႇ **Deeply Nested Loops**။
3. ယႃႇၸႂ်ႉၵျွၵ်း "ၶိုၼ်ႈၼိူဝ်" (Backward jump) ၵွပ်ႈမၼ်းၸၢင်ႈပဵၼ် Infinite Loop ဢၼ်လူယၢပ်ႇၶႃႈ။
