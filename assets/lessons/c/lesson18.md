## Lesson 18: Operator Precedence: The Order of Power

### 1. Operator Precedence ၼႆႉ ပဵၼ်သင်?
မိူဝ်ႈၸဝ်ႈၵဝ်ႇတႅမ်ႈ Code ဢၼ်မီးတူဝ်မၢႆလၢႆလၢႆမဵဝ်း ၼႂ်းထႅဝ်လဵဝ်ၵၼ် (တူဝ်ယၢင်ႇ: `x = 5 + 10 * 2;`)၊ ၶွမ်ႇပိဝ်ႇတႃႇတေလႆႈတႅပ်းတတ်းဝႃႈ တေဢဝ် 5 ထႅမ် 10 ဢွၼ်တၢင်း ႁိုဝ် တေဢဝ် 10 ပုၼ်း 2 ဢွၼ်တၢင်း ၼၼ်ႉၶႃႈ။

* **Precedence:** တွၼ်ႈတႃႇတူၺ်း "ဢဵၼ်ႁႅင်း" (Priority)။
* **Associativity:** တွၼ်ႈတႃႇတူၺ်း "တၢင်းႁဵတ်းသၢင်ႈ" (Direction) မိူဝ်ႈတူဝ်မၢႆမီးဢဵၼ်ႁႅင်းတူႇၵၼ် (မိူၼ်ၼင်ႇ ၽၢႆႇသၢႆႉ ၵႂႃႇ ၽၢႆႇၶႂႃ)။



### 2. ပိူင်ထႅဝ်ဢဵၼ်ႁႅင်း (The Hierarchy)
ၼႆႉပဵၼ် ပိူင်ထႅဝ်ဢၼ်ၸႂ်ႉၼမ်သုတ်း တႄႇဢဝ် "ဢဵၼ်ႁႅင်းယႂ်ႇ" တေႃႇပေႃးထိုင် "ဢဵၼ်ႁႅင်းဢွၼ်ႇ" ၶႃႈ:

1.  **Parentheses `( )`**: ၼႆႉမီးဢဵၼ်ႁႅင်းယႂ်ႇသုတ်း။ သင်ၶႂ်ႈႁႂ်ႈၼပ်ႉဢၼ်လႂ်ဢွၼ်တၢင်း လူဝ်ႇသႂ်ႇဝူင်းလဵပ်ႈဝႆႉၶႃႈ။
2.  **Increment/Decrement `++`, `--`**: တူဝ်ထႅမ်/ယုတ်း ၽႂ်း။
3.  **Multiplicative `*`, `/`, `%`**: တူဝ်ပုၼ်း လႄႈ တူဝ်ၽႄ။
4.  **Additive `+`, `-`**: တူဝ်ထႅမ် လႄႈ တူဝ်ယုတ်း။
5.  **Relational `<`, `>`, `<=`, `>=`**: တူဝ်တႅၵ်ႈၼိူင်း။
6.  **Equality `==`, `!=`**: တူဝ်ထၢမ်လွင်ႈတူႇၵၼ်။
7.  **Logical AND `&&`**: ၵၢၼ်ဝူၼ်ႉ AND။
8.  **Logical OR `||`**: ၵၢၼ်ဝူၼ်ႉ OR။
9.  **Assignment `=`, `+=`, `-=`**: တူဝ်ပၼ်ၵၢၼ် (မီးဢဵၼ်ႁႅင်းဢွၼ်ႇသုတ်း)။

### 3. Associativity (ၵၢၼ်ၼပ်ႉ ၽၢႆႇသၢႆႉ ႁိုဝ် ၽၢႆႇၶႂႃ)
မိူဝ်ႈတူဝ်မၢႆမီးဢဵၼ်ႁႅင်းတူႇၵၼ် (မိူၼ်ၼင်ႇ `*` လႄႈ `/`)၊ C တေၼပ်ႉ **တႄႇၽၢႆႇသၢႆႉ ၵႂႃႇ ၽၢႆႇၶႂႃ** (Left-to-Right) ၶႃႈ။

**တူဝ်ယၢင်ႇ:** `10 / 2 * 5`
* တေၼပ်ႉ `10 / 2` ဢွၼ်တၢင်း (လႆႈ 5)။
* ယဝ်ႉၸင်ႇဢဝ် `5 * 5` (တွပ်ႇမၼ်းပဵၼ် **25**) ၶႃႈ။

**သတိ:** တူဝ်မၢႆ **Assignment (`=`)** ၼၼ်ႉသမ်ႉ ၼပ်ႉ **တႄႇၽၢႆႇၶႂႃ ၵႂႃႇ ၽၢႆႇသၢႆႉ** (Right-to-Left) ၶႃႈဢေႃႈ။
* `x = y = 10;` (မၼ်းတေဢဝ် 10 သႂ်ႇၼႂ်း y ဢွၼ်တၢင်း၊ ယဝ်ႉၸင်ႇဢဝ် y သႂ်ႇၼႂ်း x ၶႃႈ)။

### 4. တူဝ်ယၢင်ႇ ဢၼ်ၸၢင်ႈငိူဝ်ႇ (Tricky Example)
```c
int result = 5 + 3 * 2; 
// တွပ်ႇမၼ်းပဵၼ် 11 (ၵွပ်ႈ * ယႂ်ႇသေ +)

int result2 = (5 + 3) * 2; 
// တွပ်ႇမၼ်းပဵၼ် 16 (ၵွပ်ႈ ( ) ယႂ်ႇသုတ်း)
```
