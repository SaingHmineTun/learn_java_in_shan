## Lesson 26: Nested Loops: Pattern Printing

### 1. Nested Loop ၼႆႉ ပဵၼ်သင်?
**Nested Loop** ပွင်ႇဝႃႈ "ၵၢၼ်သႂ်ႇ Loop ဝႆႉၼႂ်း Loop ထႅင်ႈတူဝ်ၼိုင်ႈ" ၶႃႈ။
* **Outer Loop (Loop ၼွၵ်ႈ):** ၵုမ်းၵမ် "ထႅဝ်ၼွၼ်း" (Rows)။
* **Inner Loop (Loop ၼႂ်း):** ၵုမ်းၵမ် "ထႅဝ်သိုဝ်ႈ" (Columns) ၼႂ်းၼိုင်ႈထႅဝ်ၼွၼ်းၼၼ်ႉ။

မိူဝ်ႈ Outer Loop ပၼ်ႇ 1 ၵမ်း၊ Inner Loop တေလႆႈပၼ်ႇတေႃႇပေႃးသုတ်း ၸင်ႇတေပၼ်ႇ Outer Loop ၵမ်းထႅင်ႈၼိုင်ႈၶႃႈ။



### 2. ပိူင်တႅမ်ႈ (Syntax)
```c
for (int i = 1; i <= 3; i++) {      // Outer Loop
    for (int j = 1; j <= 3; j++) {  // Inner Loop
        printf("* ");
    }
    printf("\n"); // ယဵပ်ႇလူင်းထႅဝ်မႂ်ႇ မိူဝ်ႈ Inner Loop ယဝ်ႉၼိုင်ႈပၼ်ႇ
}
```

### 3. တူဝ်ယၢင်ႇ: Pattern Printing (ႁၢင်ႈလၢႆး)
ၵၢၼ်တႅမ်ႈ "ႁၢင်ႈလၢႆး" ၼႆႉ ပဵၼ်လၢႆးပိုၵ်ႉႁတ်း Logic ဢၼ်လီသုတ်းၶႃႈ။

#### ၵ) Square Pattern (ႁၢင်ႈသီႇၸဵင်ႇသိုဝ်ႈ)
```c
// Output:
// * * *
// * * *
// * * *
```

#### ၶ) Right Triangle Pattern (ႁၢင်ႈသၢမ်ၸဵင်ႇ)
ၼႂ်းတွၼ်ႈၼႆႉ၊ Inner Loop တေပၼ်ႇၸွမ်းၼင်ႇမၢႆထႅဝ်ၶွင် Outer Loop (j <= i) ၶႃႈ။
```c
for (int i = 1; i <= 5; i++) {
    for (int j = 1; j <= i; j++) {
        printf("* ");
    }
    printf("\n");
}
// Output:
// *
// * *
// * * *
```



### 4. ၵွပ်ႈသင်ႁဝ်းလူဝ်ႇၸႂ်ႉ? (Real World Applications)
1. **Tables:** ၸႂ်ႉတႃႇၼႄၶေႃႈမုၼ်း ဢၼ်မီး Rows လႄႈ Columns (မိူၼ်ၼင်ႇ Excel ဢမ်ႇၼၼ် ပပ်ႉသဵၼ်ႈမၢႆလုၵ်ႈႁဵၼ်း)။
2. **Matrices:** ၼႂ်းပၢႆးၼပ်ႉ (Mathematics) တွၼ်ႈတႃႇၼပ်ႉဢၢၼ်ႇ Matrix ၶႃႈ။
3. **Images:** ႁၢင်ႈၽၢပ်ႈ (Digital Image) ၵူႈဢၼ် ပဵၼ် Nested Loop ၶွင် Pixels ၶႃႈဢေႃႈ။
