## Lesson 10: Constants & Literals: Fixed Data in Memory

### 1. Constants ၼႆႉ ပဵၼ်သင်? (What are Constants?)
**Constants** ပွင်ႇဝႃႈ "တူဝ်ၵိုမ်း" ၶႃႈ။ မၼ်းပဵၼ် Variable ဢၼ်ႁဝ်းပၼ်တူဝ်ၼပ်ႉဝႆႉသေ **ဢမ်ႇပၼ်လႅၵ်ႈလၢႆႈ** တေႃႇပေႃးယဝ်ႉ ပူဝ်ႇၵရႅမ်ႇၶႃႈ။ သင်ႁဝ်းၸၢမ်းမႄးတူဝ်ၼပ်ႉမၼ်းၼႆ Compiler တေၼႄ Error ၵမ်းလဵဝ်ၶႃႈဢေႃႈ။

### 2. လၢႆးတႅမ်ႈ Constants (Two Ways)
ၼႂ်းလိၵ်ႈ C ႁဝ်းတႅမ်ႈလႆႈ 2 လၢႆးၶႃႈ:

#### ၵ) ၸႂ်ႉ Keyword `const`
ႁဝ်းသႂ်ႇ `const` ဝႆႉတီႈၼႃႈ Data Type ၶႃႈ။
```c
const float PI = 3.14;
// PI = 3.15; // သင်တႅမ်ႈၼႆ Compiler တေ Error ၶႃႈ
```

#### ၶ) ၸႂ်ႉ `#define` (Preprocessor)
ဢၼ်ၼႆႉ ႁဝ်းတႅမ်ႈဝႆႉတီႈ "ႁူဝ်" File (Header) ၶႃႈ။ မၼ်းဢမ်ႇၸႂ်ႉ Memory တူဝ်တူဝ် မိူၼ် Variable ၵူၺ်းၵႃႈ မၼ်းတေလႅၵ်ႈလၢႆႈၸိုဝ်ႈ ႁႂ်ႈပဵၼ်တူဝ်ၼပ်ႉ မိူဝ်ႈတိုၵ်ႉ Compile ယူႇၼၼ်ႉၶႃႈ။
```c
#define SHAN_YEAR 2118
```



### 3. Literals ၼႆႉ ပဵၼ်သင်? (What are Literals?)
**Literals** ပွင်ႇဝႃႈ "ၶေႃႈမုၼ်းတေႉ" ဢၼ်ႁဝ်းတႅမ်ႈသႂ်ႇဝႆႉၼႂ်း Code ႁဝ်းၼၼ်ႉၶႃႈ။ မၼ်းဢမ်ႇမီးၸိုဝ်ႈ Variable မိူၼ်ၼင်ႇ `10`, `3.14`, `'A'`.

**မဵဝ်းၶွင် Literals:**
* **Integer Literals:** တူဝ်ၼပ်ႉတဵမ် (မိူၼ်ၼင်ႇ `100`, `-50`)။
* **Floating-point Literals:** တူဝ်ၼပ်ႉမီးၸုၵ်း (မိူၼ်ၼင်ႇ `3.14`, `0.5`)။
* **Character Literals:** တူဝ်လိၵ်ႈၼိုင်ႈတူဝ် (မိူၼ်ၼင်ႇ `'K'`, `'!'`)။
* **String Literals:** လိၵ်ႈပဵၼ်ထႅဝ် ဢၼ်မီးၼႂ်းမၢႆ `" "` (မိူၼ်ၼင်ႇ `"Shan State"`)။

### 4. ၵၢၼ်ၵဵပ်းသိမ်းၼႂ်း Memory (Fixed Data)
မိူဝ်ႈႁဝ်းၸႂ်ႉ `const` ဢမ်ႇၼၼ် String Literals ၼၼ်ႉ၊ ၶွမ်ႇပိဝ်ႇတႃႇတေၵဵပ်းၶဝ်ဝႆႉၼႂ်းတွၼ်ႈ Memory ဢၼ်ႁွင်ႉဝႃႈ **Read-Only Data Segment** ၶႃႈ။
* မၼ်းပွင်ႇဝႃႈ "လူလႆႈၵူၺ်း၊ တႅမ်ႈသႂ်ႇ (မႄး) ဢမ်ႇလႆႈ" ၼၼ်ႉၶႃႈဢေႃႈ။



### 5. ၵွပ်ႈသင်ႁဝ်းလူဝ်ႇၸႂ်ႉ? (Why use them?)
1. **Safety:** တွၼ်ႈတႃႇႁႄႉၵင်ႈ ဢမ်ႇႁႂ်ႈပိူၼ်ႈမႃးမႄး ၶေႃႈမုၼ်းဢၼ်လမ်ႇလွင်ႈ (မိူၼ်ၼင်ႇ မၢႆ Tax, မၢႆ Pi)။
2. **Readability:** တႅမ်ႈ `PI` ၼႆ မၼ်းပွင်ႇၸႂ်ငၢႆႈလိူဝ်သေ တႅမ်ႈ `3.14159...` ၵူႈတီႈၶႃႈ။
3. **Performance:** Compiler ၸွႆႈႁဵတ်းႁႂ်ႈ ပူဝ်ႇၵရႅမ်ႇႁဝ်း ႁဵတ်းၵၢၼ်လႆႈၽႂ်းလိူဝ် ၵွပ်ႈမၼ်းႁူႉဝႃႈ တူဝ်ၼပ်ႉၼႆႉ တေဢမ်ႇလႅၵ်ႈလၢႆႈၶႃႈ။
