## Lesson 58: Introduction to File Streams: FILE Pointer

### 1. File Stream ပဵၼ်သင်?
**Stream** ၼႆႉ ႁႂ်ႈဝူၼ်ႉတူၺ်းဝႃႈ မၼ်းပဵၼ် "ၵူင်ႇ" (Tunnel) ဢၼ်ၵွင်ႉဝႆႉၼႂ်းၵႄႈ **RAM** (တီႈပူဝ်ႇၵရႅမ်ႇႁဝ်းႁဵတ်းၵၢၼ်) လႄႈ **Hard Disk** (တီႈၵဵပ်း File) ၶႃႈ။
* **Input Stream:** ဢဝ်ၶေႃႈမုၼ်းတႄႇဢဝ် File ၼႂ်း Disk မႃးသႂ်ႇၼႂ်း RAM။
* **Output Stream:** ဢဝ်ၶေႃႈမုၼ်းတႄႇဢဝ် RAM ၵႂႃႇတႅမ်ႈသႂ်ႇၼႂ်း File ၼႂ်း Disk။



### 2. The `FILE` Pointer
တွၼ်ႈတႃႇတေၵုမ်းၵမ် "ၵူင်ႇ" (Stream) ၼႆႉလႆႈ၊ C မီး Structure ၶူးဝႆႉဢၼ်ၼိုင်ႈ ဢၼ်ၸိုဝ်ႈဝႃႈ **`FILE`** (ယူႇၼႂ်း `<stdio.h>`)။ ႁဝ်းတေလႆႈၸႂ်ႉ **Pointer** တႃႇၸี้ၵႂႃႇၸူး File ၼၼ်ႉၶႃႈ။

**Syntax:**
```c
FILE *fp; // 'fp' ပွင်ႇဝႃႈ File Pointer ၶႃႈဢေႃႈ
```
`fp` ၼႆႉ မၼ်းတေၵဵပ်းၶေႃႈမုၼ်းဝႆႉဝႃႈ:
* File ၼၼ်ႉၸိုဝ်ႈသင်?
* ယၢမ်းလဵဝ် ႁဝ်းလူလိၵ်ႈထိုင်ထႅဝ်လႂ်ယဝ်ႉ?
* File ၼၼ်ႉ ပိုတ်ႇဝႆႉတွၼ်ႈတႃႇ "လူ" (Read) ႁိုဝ် "တႅမ်ႈ" (Write)?

### 3. Buffer (ဢွင်ႈတီႈၵိုတ်းယတ်ႉ)
မိူဝ်ႈႁဝ်းတႅမ်ႈလိၵ်ႈသႂ်ႇ File၊ C တေဢမ်ႇၵႂႃႇတႅမ်ႈသႂ်ႇ Disk ၵမ်းလဵဝ်ၵူႈတူဝ်လိၵ်ႈ (ၵွပ်ႈမၼ်းတေႁဵတ်းႁႂ်ႈၶွမ်ႇပိဝ်ႇတႃႇဢืดႇ)။ မၼ်းတေၵဵပ်းဝႆႉၼႂ်း **Buffer** (Memory ဢွၼ်ႇၼႂ်း RAM) သေ မိူဝ်ႈတဵမ်ယဝ်ႉ ၸင်ႇတေသူင်ႇၵႂႃႇတႅမ်ႈသႂ်ႇ Disk ၼႂ်းၵမ်းလဵဝ်ၶႃႈ။ ၼႆႉႁွင်ႉဝႃႈ **Buffered I/O** ၶႃႈဢေႃႈ။



### 4. Why Pointer?
ၵွပ်ႈသင်ႁဝ်းသမ်ႉၸႂ်ႉ `FILE *fp` သေ ဢမ်ႇၸႂ်ႉ `FILE fp` ယူဝ်းယူဝ်း?
* ၵွပ်ႈဝႃႈ **`FILE`** ၼႆႉပဵၼ် Structure ဢၼ်ယႂ်ႇ လႄႈ ၵုမ်းၵမ်လူၺ်ႈ **Operating System (OS)**။
* ႁဝ်းၵူၺ်းလူဝ်ႇ "မၢႆလိၵ်ႈ Address" (Pointer) တွၼ်ႈတႃႇသင်ႇၵၢၼ်ၸူး OS ဝႃႈ "ၸွႆႈပိုတ်ႇ File ၼႆႉပၼ်ၶႃႈ" ၼၼ်ႉၵူၺ်းၶႃႈ။
