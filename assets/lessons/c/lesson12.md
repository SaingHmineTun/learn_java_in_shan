## Lesson 12: Interacting with the User: I/O & Buffers

### 1. Standard Input & Output (I/O)
ၼႂ်းလိၵ်ႈ C၊ ႁဝ်းၸႂ်ႉ Library `<stdio.h>` တွၼ်ႈတႃႇၵပ်းသိုပ်ႇတင်း User ၶႃႈ။
* **`printf()`**: ၸႂ်ႉတႃႇသူင်ႇၶေႃႈမုၼ်းဢွၵ်ႇ Screen (Standard Output)။
* **`scanf()`**: ၸႂ်ႉတႃႇႁပ်ႉဢဝ်ၶေႃႈမုၼ်းတီႈ Keyboard (Standard Input)။

### 2. Formatted Input/Output (Format Specifiers)
မိူဝ်ႈႁဝ်းႁပ်ႉ ဢမ်ႇၼၼ် ၼႄၶေႃႈမုၼ်း၊ ႁဝ်းလူဝ်ႇလၢတ်ႈၼႄ "မဵဝ်း" (Type) ၶေႃႈမုၼ်းမၼ်းတေႃႈလဵဝ်ၶႃႈ။

| Specifier | မဵဝ်းၶေႃႈမုၼ်း (Data Type) |
| :--- | :--- |
| **`%d`** | တူဝ်ၼပ်ႉတဵမ် (int) |
| **`%f`** | တူဝ်ၼပ်ႉမီးၸုၵ်း (float) |
| **`%c`** | တူဝ်လိၵ်ႈ (char) |
| **`%s`** | လိၵ်ႈပဵၼ်ထႅဝ် (string) |



### 3. Buffer ၼႆႉ ပဵၼ်သင်? (Understanding Input Buffers)
**Buffer** ၼႆႉ မိူၼ်ၼင်ႇ "ႁွင်ႉၵဵပ်းၶေႃႈမုၼ်းၸူဝ်ႉၵေႃႉ" (Temporary Storage) ၶႃႈ။ မိူဝ်ႈၸဝ်ႈၵဝ်ႇတဵၵ်း Keyboard ၼၼ်ႉ၊ တူဝ်လိၵ်ႈၸိူဝ်းၼၼ်ႉ တေဢမ်ႇၵႂႃႇၸူး Variable ၵမ်းလဵဝ်ၶႃႈ။ မၼ်းတေၵႂႃႇၵိုတ်းဝႆႉၼႂ်း **Input Buffer** တေႃႇပေႃးၸဝ်ႈၵဝ်ႇတဵၵ်း **Enter** ၶႃႈဢေႃႈ။

**လွင်ႈယၢပ်ႇ:** မိူဝ်ႈၸဝ်ႈၵဝ်ႇၸႂ်ႉ `scanf("%d", &age);` ယဝ်ႉ၊ မၢႆ "Enter" (`\n`) ၼၼ်ႉ မၼ်းတေႃႉတိုၵ်ႉၵိုတ်းဝႆႉၼႂ်း Buffer ၶႃႈ။ သင်ဝႃႈထႅဝ်တႂ်ႈမၼ်း ၸဝ်ႈၵဝ်ႇႁပ်ႉဢဝ် `char` ၼႆ `scanf` တေၵႂႃႇၵဵပ်းဢဝ်မၢႆ Enter ၼၼ်ႉသေ **"ၶၢမ်ႈ"** လွင်ႈႁပ်ႉၶေႃႈမုၼ်းၵႂႃႇၵမ်းလဵဝ်ၶႃႈဢေႃႈ။



### 4. လၢႆးၵႄႈလႅတ်ႈ Buffer (Clearing the Buffer)
တွၼ်ႈတႃႇႁႄႉၵင်ႈ ဢမ်ႇႁႂ်ႈမၼ်း "ၶၢမ်ႈ" ၼၼ်ႉ၊ ႁဝ်းလူဝ်ႇလၢတ်ႈၼႄ `scanf` ႁႂ်ႈမၼ်းဢၢၼ်ႇၶၢမ်ႈ ဢွင်ႈဝၢင်ႇ (Whitespace) တင်းသဵင်ႈၶႃႈ။
```c
scanf(" %c", &myChar); // သႂ်ႇ Space ဝႆႉတီႈၼႃႈ %c
```

### 5. ၵၢၼ်ႁပ်ႉလိၵ်ႈတႆး (Handling Strings with Spaces)
`scanf("%s", ...)` ၼႆႉ မၼ်းတေႁပ်ႉဢဝ်လႆႈ တေႃႇပေႃးထိုင် "ဢွင်ႈဝၢင်ႇ" (Space) ၵူၺ်းၶႃႈ။ သင်ၶႂ်ႈႁပ်ႉလိၵ်ႈယၢဝ်းယၢဝ်း (မိူၼ်ၼင်ႇ ၸိုဝ်ႈတဵမ် "Sai Mao Muse") ႁဝ်းလူဝ်ႇၸႂ်ႉ **`fgets()`** ၶႃႈ။

```c
char name[50];
printf("Enter your name: ");
fgets(name, sizeof(name), stdin); 
```
