## Lesson 37: Pointer Arithmetic: Moving in Memory

### 1. ၵၢၼ်ယဵပ်ႇ Pointer (Scaling Factor)
မိူဝ်ႈႁဝ်းၸႂ်ႉ `ptr++` ဢမ်ႇၼၼ် `ptr + 1`၊ C တေၼပ်ႉဢၢၼ်ႇလွင်ႈယဵပ်ႇ Address ၸိူင်ႉၼႆၶႃႈ:
* **New Address = Current Address + (n * sizeof(Data Type))**

**တူဝ်ယၢင်ႇ:** သင်ဝႃႈ `int` မီး 4 Bytes သေ `ptr` ယူႇတီႈ Address `1000` ၼႆ:
* `ptr + 1` တေပဵၼ် **1004** (1000 + 1 * 4)
* `ptr + 2` တေပဵၼ် **1008** (1000 + 2 * 4)



### 2. တူဝ်မၢႆဢၼ်ၸႂ်ႉလႆႈ (Allowed Operators)
ႁဝ်းၸႂ်ႉ Pointer Arithmetic လႆႈၵူၺ်း 4 မဵဝ်းၶႃႈ:
1. **`++` (Increment):** ယဵပ်ႇၵႂႃႇၸူး Address ထႅင်ႈဢၼ်ၼိုင်ႈ ၽၢႆႇၼႃႈ။
2. **`--` (Decrement):** ယဵပ်ႇၶိုၼ်းၵႂႃႇၸူး Address ၽၢႆႇလင်။
3. **`+` (Addition):** ယဵပ်ႇၵႂႃႇလၢႆလၢႆ "ႁွင်ႉ" (Steps) ၸွမ်းၵၼ်။
4. **`-` (Subtraction):** ယဵပ်ႇၶိုၼ်းလၢႆလၢႆ "ႁွင်ႉ" ဢမ်ႇၼၼ် တႅၵ်ႈၼိူင်းဝႃႈ Pointer သွင်ဢၼ် ၵႆၵၼ်ၵႃႈႁိုဝ်။

### 3. Pointer တႅၵ်ႈၼိူင်း (Comparison)
ႁဝ်းၸၢင်ႈဢဝ် Pointer သွင်ဢၼ်မႃးတႅၵ်ႈၼိူင်းၵၼ်လႆႈ လူၺ်ႈၸႂ်ႉတူဝ်မၢႆ `==`, `!=`, `<`, `>` ၶႃႈ။
* ၼႆႉၸႂ်ႉၼမ် မိူဝ်ႈႁဝ်းဝၼ်ႇပၼ်ႇ (Loop) ၼႂ်း Array သေ ၶႂ်ႈႁူႉဝႃႈ မၼ်းထိုင်တီႈပၢႆ Array ယဝ်ႉႁိုဝ်? ၼၼ်ႉၶႃႈဢေႃႈ။

### 4. ၵၢၼ်ၸႂ်ႉ `*ptr++` vs `(*ptr)++`
ၼႆႉပဵၼ်လွင်ႈဢၼ်လုၵ်ႈႁဵၼ်း ၸၢင်ႈ "ငိူဝ်ႇ" သုတ်းၶႃႈ:
* **`*ptr++`**: ယဵပ်ႇ **Pointer** ၵႂႃႇၸူး Address ထႅင်ႈဢၼ်ၼိုင်ႈ (ယဝ်ႉၸင်ႇဢဝ် Value)။
* **`(*ptr)++`**: ထႅမ်မၢႆ **Value** ဢၼ်ယူႇၼႂ်း Address ၼၼ်ႉ ႁႂ်ႈယႂ်ႇတုၺ်း (Address ဢမ်ႇလႅၵ်ႈလၢႆႈ)။



### 5. ၵွပ်ႈသင်ႁဝ်းလူဝ်ႇၸႂ်ႉ?
1. **Array Navigation:** ၸွႆႈႁႂ်ႈႁဝ်း "လႆ" ၵႂႃႇၼႂ်း Array လႆႈငၢႆႈငၢႆႈ လူၺ်ႈဢမ်ႇလူဝ်ႇၸႂ်ႉ Index `[i]` ၶႃႈ။
2. **Performance:** ၵၢၼ်ၸႂ်ႉ Pointer Arithmetic မၼ်းၸၢင်ႈႁဵတ်းႁႂ်ႈ Code ႁဝ်းႁဵတ်းၵၢၼ်ၽႂ်းလိူဝ် (Optimization) ၼႂ်း Hardware ၸၼ်ႉတႂ်ႈၶႃႈ။
