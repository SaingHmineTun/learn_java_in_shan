## Lesson 11: The Unicode Challenge: Storing Shan Script

### 1. ASCII vs Unicode (လွင်ႈပႅၵ်ႇပိူင်ႈ)
မိူဝ်ႈပူဝ်ႇၵရႅမ်ႇမိင်း တႄႇဢွၵ်ႇမႃးၼၼ်ႉ ၶဝ်ၸႂ်ႉ **ASCII** ဢၼ်မီးၵူၺ်း 128 တူဝ် (A-Z, 0-9)။
* **ASCII:** ၸႂ်ႉ **1 Byte** (8 Bits) တႃႇတူဝ်လိၵ်ႈၼိုင်ႈတူဝ်။
* **Unicode (UTF-8):** ပဵၼ်ပိူင်ဢၼ်ၵဵပ်းလႆႈ လိၵ်ႈၵူႈၽႃႇသႃႇၼႂ်းလုမ်ႈၾႃႉ၊ ပႃးတင်း **လိၵ်ႈတႆး** ႁဝ်းၶႃႈ။

### 2. Unicode Challenge: လွင်ႈၼၵ်းၶွင်လိၵ်ႈတႆး
ၼႂ်းလိၵ်ႈ C၊ `char` ၼႆႉ မၼ်းၵဵပ်းလႆႈၵူၺ်း **1 Byte** ၶႃႈ။ ၵူၺ်းၵႃႈ လိၵ်ႈတႆးၼိုင်ႈတူဝ် (မိူၼ်ၼင်ႇ **"ၵ"**) ၼၼ်ႉ ၼႂ်းပိူင် UTF-8 မၼ်းၸႂ်ႉထိုင် **3 Bytes** ၶႃႈဢေႃႈ။

**တူဝ်ယၢင်ႇ:**
* 'A' = `01000001` (1 Byte)
* 'ၵ' = `11101000 10100000 10100000` (3 Bytes - *တူဝ်ယၢင်ႇ*)



### 3. လွင်ႈၽိတ်းဢၼ်ၸၢင်ႈပဵၼ် (Common Mistakes)
သင်လုၵ်ႈႁဵၼ်းတႅမ်ႈ Code ၼင်ႇၼႆ တေၽိတ်းၶႃႈ:
```c
char c = 'ၵ'; // ၽိတ်း! (Error: Character too large)
```
**ၵွပ်ႈသင်?:** ၵွပ်ႈဝႃႈ `char` မၼ်းမိူၼ်ၼင်ႇ "ၵွၵ်းဢွၼ်ႇ" ဢၼ်သႂ်ႇလႆႈ 1 Byte၊ ၵူၺ်းၵႃႈ "ၵ" မၼ်းယႂ်ႇထိုင် 3 Bytes လႄႈ မၼ်းသႂ်ႇဢမ်ႇသဵင်ႈၶႃႈ။

### 4. လၢႆးၵဵပ်းလိၵ်ႈတႆး ၼႂ်း C (The Solution)
တွၼ်ႈတႃႇၵဵပ်းလိၵ်ႈတႆး၊ ႁဝ်းလူဝ်ႇၸႂ်ႉ **Character Array** (ဢမ်ႇၼၼ် String) တႃႇသေႇၶႃႈ။
```c
char shan_text[] = "ၵ"; 
// ၼႂ်း Memory မၼ်းတေၸႂ်ႉ 3 Bytes + 1 Byte (Null terminator) = 4 Bytes
```



### 5. ၵၢၼ်ၼပ်ႉဢၢၼ်ႇ (String Length vs Bytes)
ၼႆႉပဵၼ်လွင်ႈဢၼ် လုၵ်ႈႁဵၼ်းတႆး ၸၢင်ႈငိူဝ်ႇ (Confused) သုတ်းၶႃႈ။
* **`strlen()`**: မၼ်းတေၼပ်ႉဢၢၼ်ႇ "Bytes" ၵူၺ်း၊ မၼ်းဢမ်ႇၼပ်ႉဢၢၼ်ႇ "တူဝ်လိၵ်ႈ" ၶႃႈ။

**တူဝ်ယၢင်ႇ Code:**
```c
#include <stdio.h>
#include <string.h>

int main() {
    char text[] = "တႆး"; 
    printf("Bytes used: %zu\n", strlen(text)); 
    return 0;
}
```
*(သင် Run Code ၼႆႉ၊ မၼ်းတေၼႄဝႃႈ **9 Bytes** ၶႃႈ! ၵွပ်ႈဝႃႈ "တ", "ႆ", "း" ၼိုင်ႈတူဝ်မီး 3 Bytes ၶႃႈဢေႃႈ)*
