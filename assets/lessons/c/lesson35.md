## Lesson 35: Memory Addresses: The & Operator

### 1. Memory Address ပဵၼ်သင်?
ၼႂ်းၶွမ်ႇပိဝ်ႇတႃႇ၊ RAM ၼႆႉ မိူၼ်ၼင်ႇ "ဝဵင်းလူင်" ဢၼ်မီးႁိူၼ်းလၢႆလၢႆလင်။ ႁိူၼ်းၼိုင်ႈလင် (1 Byte) တေမီး **မၢႆလိၵ်ႈႁိူၼ်း** (Address) ႁင်းၵူၺ်းမၼ်းတႃႇသေႇ။
* မိူဝ်ႈႁဝ်းပိုၼ်ၽၢဝ်ႇ `int x = 10;` ၼႆ၊ ၶွမ်ႇပိဝ်ႇတႃႇတေဢဝ်မၢႆ 10 ၵႂႃႇၵဵပ်းဝႆႉၼႂ်း "ႁိူၼ်း" လင်ၼိုင်ႈ ဢၼ်မီး Address ၶိုၵ်ႉတွၼ်း (မိူၼ်ၼင်ႇ `0x7ff7`) ၶႃႈ။



### 2. The Address-of Operator (`&`)
ၼႂ်းလိၵ်ႈ C၊ ႁဝ်းၸႂ်ႉတူဝ်မၢႆ **`&`** (Ampersand) တွၼ်ႈတႃႇ "ထၢမ်" ၶွမ်ႇပိဝ်ႇတႃႇဝႃႈ Variable တူဝ်ၼၼ်ႉ ၵဵပ်းဝႆႉတီႈလႂ်ၼႂ်း Memory ၶႃႈဢေႃႈ။

```c
#include <stdio.h>

int main() {
    int age = 25;
    
    printf("Value of age: %d\n", age);      // ၼႄမၢႆၶေႃႈမုၼ်း (25)
    printf("Address of age: %p\n", &age);   // ၼႄမၢႆလိၵ်ႈ Memory (Hexadecimal)
    
    return 0;
}
```
*(သတိ: ၸႂ်ႉ `%p` တွၼ်ႈတႃႇၼႄမၢႆလိၵ်ႈ Address ၶႃႈ)*



### 3. Hexadecimal Format
မၢႆလိၵ်ႈ Address တေဢွၵ်ႇမႃးပဵၼ် **Hexadecimal** (Base-16) မိူၼ်ၼင်ႇ `0x7ffee45b8`။ ၼႆႉပဵၼ်လၢႆးဢၼ်ၶွမ်ႇပိဝ်ႇတႃႇ ၸတ်းၵၢၼ် Memory ႁႂ်ႈမၼ်းလူငၢႆႈလိူဝ် တူဝ်ၼပ်ႉယႂ်ႇယႂ်ႇၶႃႈ။

### 4. ၵွပ်ႈသင်ႁဝ်းလူဝ်ႇႁူႉ?
သင်ႁဝ်းႁူႉ "မၢႆလိၵ်ႈ" (Address) ၼႆ:
1. ႁဝ်းၸၢင်ႈ "ၸี้" (Point) ၵႂႃႇၸူးၶေႃႈမုၼ်းၼၼ်ႉ လႆႈၵမ်းလဵဝ်။
2. ႁဝ်းၸၢင်ႈလႅၵ်ႈလၢႆႈၶေႃႈမုၼ်း ၼႂ်း Variable တူဝ်ၼၼ်ႉ တႄႇဢဝ်ၼႂ်း Function တၢင်ႇဢၼ် (Call by Reference)။
3. ၼႆႉပဵၼ်ပိုၼ်ႉထၢၼ် ဢၼ်တေႁဵတ်းႁႂ်ႈႁဝ်းသၢင်ႈ **Pointer** လႆႈၼၼ်ႉၶႃႈဢေႃႈ။

## 🏗️ Logic for Expert Programming
မိူဝ်ႈၸဝ်ႈၵဝ်ႇၸႂ်ႉ `scanf("%d", &age);` ၼၼ်ႉ၊ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉ "ပၼ်မၢႆလိၵ်ႈ" (Address) ၶွင် Variable `age` ၵႂႃႇၸူး Function `scanf` ၶႃႈဢေႃႈ။ မၼ်းၸင်ႇႁူႉဝႃႈ လူဝ်ႇဢဝ်တူဝ်ၼပ်ႉဢၼ် User တဵၵ်းမႃးၼၼ်ႉ ၵႂႃႇ "သႂ်ႇ" ဝႆႉၼႂ်းႁွင်ႉ Memory လင်လႂ် ၼၼ်ႉၶႃႈယဝ်ႉ။
