## Lesson 29: Multidimensional Arrays: Matrices & Grids

### 1. 2D Array ၼႆႉ ပဵၼ်သင်? (The Grid Concept)
သင်ဝႃႈ 1D Array (Lesson 28) ပဵၼ် "ႁိူၼ်းတႅဝ်း" ထႅဝ်လဵဝ်ၼႆ၊ **2D Array** သမ်ႉမိူၼ်ၼင်ႇ "ႁိူင်းၸၼ်ႉ" (Apartment Building) ဢၼ်မီးတင်း **ၸၼ်ႉ** (Rows) လႄႈ **မၢႆႁွင်ႉ** (Columns) ၶႃႈယဝ်ႉ။

* **Rows (i):** ၵုမ်းၵမ်ထႅဝ်ၼွၼ်း (တႄႇၼပ်ႉတီႈ 0)။
* **Columns (j):** ၵုမ်းၵမ်ထႅဝ်သိုဝ်ႈ (တႄႇၼပ်ႉတီႈ 0)။



### 2. ၵၢၼ်ပိုၼ်ၽၢဝ်ႇ (Declaration)
ႁဝ်းလူဝ်ႇသႂ်ႇဝူင်းလဵပ်ႈသီႇၸဵင်ႇ `[ ]` သွင်ဢၼ်ၶႃႈ:
**Syntax:** `type arrayName[rows][columns];`

```c
int matrix[3][4]; // ၵဵပ်းတူဝ်ၼပ်ႉလႆႈ 3 ထႅဝ်ၼွၼ်း, 4 ထႅဝ်သိုဝ်ႈ (တင်းသဵင်ႈ 12 တူဝ်)

// တင်ႈတူဝ်ၼပ်ႉဝႆႉၵမ်းလဵဝ် (Initialization)
int grid[2][3] = {
    {1, 2, 3}, // Row 0
    {4, 5, 6}  // Row 1
};
```

### 3. ၵၢၼ်ၶဝ်ႈၸူးၶေႃႈမုၼ်း (Accessing Elements)
တွၼ်ႈတႃႇတေဢဝ်ၶေႃႈမုၼ်းၼိုင်ႈတူဝ်ဢွၵ်ႇမႃး၊ ႁဝ်းလူဝ်ႇလၢတ်ႈၼႄတင်း **မၢႆထႅဝ်** လႄႈ **မၢႆၸၼ်ႉ** ၶႃႈ။
* `grid[0][1]` = တေလႆႈမၢႆ **2** (Row 0, Column 1)
* `grid[1][2]` = တေလႆႈမၢႆ **6** (Row 1, Column 2)



### 4. ၵၢၼ်ၸႂ်ႉ Nested Loops (ၵူပ်ႉၵၼ်တင်း 2D Array)
တွၼ်ႈတႃႇတေ "ၼႄ" ဢမ်ႇၼၼ် "ႁပ်ႉ" ၶေႃႈမုၼ်းၼႂ်း 2D Array၊ ႁဝ်းတေလႆႈၸႂ်ႉ **Nested Loop** (Lesson 26) တႃႇသေႇၶႃႈ။

```c
int matrix[2][2] = {{10, 20}, {30, 40}};

for (int i = 0; i < 2; i++) {       // Loop တွၼ်ႈတႃႇ Rows
    for (int j = 0; j < 2; j++) {   // Loop တွၼ်ႈတႃႇ Columns
        printf("%d ", matrix[i][j]);
    }
    printf("\n"); // ယဵပ်ႇလူင်းထႅဝ်မႂ်ႇ မိူဝ်ႈယဝ်ႉၼိုင်ႈ Row
}
```

### 5. Memory Architecture (Row-Major Order)
ၼႂ်း RAM တေႉတေႉမၼ်း၊ ၶွမ်ႇပိဝ်ႇတႃႇဢမ်ႇမီး "ထႅဝ်ၼွၼ်း/ထႅဝ်သိုဝ်ႈ" ၶႃႈ။ မၼ်းၵဵပ်းၶေႃႈမုၼ်းတင်းသဵင်ႈဝႆႉပဵၼ် "ထႅဝ်ယၢဝ်းသိုဝ်ႈ" ၵၼ်ၵူၺ်း။
* မၼ်းတေၵဵပ်း Row 0 တင်းသဵင်ႈဢွၼ်တၢင်း၊ ယဝ်ႉၸင်ႇၵဵပ်း Row 1 သိုပ်ႇၵၼ်ၵႂႃႇၶႃႈဢေႃႈ။ (ႁွင်ႉဝႃႈ **Row-Major Order**)။
