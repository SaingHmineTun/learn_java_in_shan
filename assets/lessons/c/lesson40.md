## Lesson 40: Dynamic Memory Allocation: malloc, calloc & free

### 1. ၵွပ်ႈသင်ႁဝ်းလူဝ်ႇ DMA?
မိူဝ်ႈႁဝ်းပိုၼ်ၽၢဝ်ႇ Array မိူၼ်ၼင်ႇ `int arr[100];` ၼၼ်ႉ:
* မၼ်းၵဵပ်းဝႆႉၼႂ်း **Stack Memory**။
* တင်းယႂ်ႇ (Size) မၼ်းတေလႆႈမၵ်းမၼ်ႈဝႆႉ တႄႇဢဝ်မိူဝ်ႈတႅမ်ႈ Code (Static)။
* သင်လုၵ်ႈႁဵၼ်းမီး 101 ၵေႃႉ ၼႆ Array ၼၼ်ႉတေတဵမ်၊ ဢမ်ႇၼၼ် သင်ၵူၺ်းမီး 2 ၵေႃႉ ၼႆ Memory ဢၼ်လိူဝ်ၼၼ်ႉ ပႅတ်ႈလၢႆၶႃႈ။

**DMA** သမ်ႉတေၸွႆႈႁႂ်ႈႁဝ်းဢဝ် Memory ၼႂ်း **Heap** ၸွမ်းၼင်ႇႁူဝ်ၼပ်ႉ ဢၼ်ႁဝ်းလူဝ်ႇတေႉတေႉၼၼ်ႉၶႃႈဢေႃႈ။



### 2. `malloc()` (Memory Allocation)
`malloc()` ၸႂ်ႉတႃႇဢဝ် Memory ပွတ်းၼိုင်ႈ ဢၼ်မီးတင်းယႂ်ႇ ၸွမ်းၼင်ႇ Bytes ဢၼ်ႁဝ်းသင်ႇဝႆႉ။
* မၼ်းတေ `return` ပၼ် **Void Pointer** ဢၼ်ၸီႉၵႂႃႇၸူး Address တႄႇမၼ်း။
* ႁဝ်းတေလႆႈၸႂ်ႉ **Typecasting** တႃႇလႅၵ်ႈပဵၼ် Pointer Type ဢၼ်ႁဝ်းၶႂ်ႈလႆႈၶႃႈ။

```c
int *ptr;
int n = 5;
ptr = (int*) malloc(n * sizeof(int)); // ၶေႃ Memory တႃႇ int 5 တူဝ်
```

### 3. `calloc()` (Contiguous Allocation)
မိူၼ်တင်း `malloc` ၵူၺ်းၵႃႈ မၼ်းမီးလွင်ႈပႅၵ်ႇပိူင်ႈ 2 မဵဝ်း:
1. မၼ်းႁပ်ႉ Parameter 2 ဢၼ် (ႁူဝ်ၼပ်ႉတူဝ် လႄႈ သႅၼ်း Bytes)။
2. မၼ်းတေ **Clear** ၶေႃႈမုၼ်းတင်းသဵင်ႈ ႁႂ်ႈပဵၼ် **0** (Zero-initialize) တႃႇသေႇ။

```c
ptr = (int*) calloc(n, sizeof(int)); // ၵူႈႁွင်ႉ Memory တေပဵၼ် 0 တင်းသဵင်ႈ
```



### 4. `free()` (Memory Deallocation)
ၼႆႉလမ်ႇလွင်ႈသုတ်းၶႃႈ! ၵွပ်ႈဝႃႈ C ဢမ်ႇမီး "Garbage Collector" မိူၼ်ၼင်ႇ Java ဢမ်ႇၼၼ် Dart၊ မိူဝ်ႈႁဝ်းၸႂ်ႉ Memory ယဝ်ႉ ႁဝ်းတေလႆႈ **"ၶိုၼ်းပၼ်"** (Release) ၸူး OS ၶႃႈ။
* သင်ဢမ်ႇ `free()` ၼႆ RAM တေတဵမ်ၵႂႃႇတိၵ်းတိၵ်း (ႁွင်ႉဝႃႈ **Memory Leak**)။

```c
free(ptr); // ၶိုၼ်းပၼ် Memory ႁႂ်ႈပိူၼ်ႈၸႂ်ႉလႆႈတၢင်ႇဢၼ်
```

### 5. `realloc()` (Re-allocation)
သင်ႁဝ်းၶေႃဝႆႉ 5 တူဝ် ယဝ်ႉသမ်ႉလူဝ်ႇထႅမ်ပဵၼ် 10 တူဝ်ၼႆ၊ ႁဝ်းၸႂ်ႉ `realloc()` တႃႇလႅၵ်ႈလၢႆႈသႅၼ်း Memory ဢၼ်မီးဝႆႉယူႇယဝ်ႉၼၼ်ႉၶႃႈ။
