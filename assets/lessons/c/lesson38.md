## Lesson 38: Pointers & Arrays: The Secret Relationship

### 1. Array Name ပဵၼ် Pointer တူဝ်ၼိုင်ႈ
မိူဝ်ႈၸဝ်ႈၵဝ်ႇပိုၼ်ၽၢဝ်ႇ `int arr[5];` ၼႆ၊ ၸိုဝ်ႈ **`arr`** ၼၼ်ႉ မၼ်းဢမ်ႇၸႂ် ၶေႃႈမုၼ်းတင်းသဵင်ႈၶႃႈ။ မၼ်းပဵၼ် **Constant Pointer** ဢၼ်ၸี้ၵႂႃႇၸူး **Address ၶွင်တူဝ်ထီႉ ၼိုင်ႈ (`arr[0]`)** တႃႇသေႇၶႃႈယဝ်ႉ။

* `arr` မိူၼ်ၵၼ်တင်း `&arr[0]`
* ၼႆႉႁွင်ႉဝႃႈ **Array Decay** (ၵၢၼ်လႅၵ်ႈပဵၼ် Pointer ႁင်းၵူၺ်း)။



### 2. Pointer Notation vs Array Notation
ႁဝ်းၸၢင်ႈၶဝ်ႈၸူး ၶေႃႈမုၼ်းၼႂ်း Array လႆႈ 2 လၢႆးၶႃႈ:

| Array Style (Index) | Pointer Style (Arithmetic) | တီႈပွင်ႇ |
| :--- |:---------------------------| :--- |
| `arr[0]` | `*arr`                     | တူဝ်ထီႉ 1 |
| `arr[1]` | `*(arr + 1)`               | တူဝ်ထီႉ 2 |
| `arr[i]` | **`*(arr + i)`**           | တူဝ်ထီႉ i |

**သတိ:** `*(arr + 1)` တေဢဝ် Value ၶွင် `arr[1]`၊ ၵူၺ်းၵႃႈ `*arr + 1` သမ်ႉတေဢဝ် Value `arr[0]` မႃးထႅမ်မၢႆ 1 ၶႃႈ။ (လူဝ်ႇသႂ်ႇဝူင်းလဵပ်ႈႁႂ်ႈမႅၼ်ႈၶႃႈ)။



### 3. ၵွပ်ႈသင်ႁဝ်းလူဝ်ႇ "Pointer Style"?
1. **Speed:** ၵၢၼ်ၸႂ်ႉ Pointer Arithmetic မၼ်းၸၢင်ႈႁဵတ်းႁႂ်ႈ Compiler ႁဵတ်းၵၢၼ်ၽႂ်းလိူဝ် ၵၢၼ်ၼပ်ႉဢၢၼ်ႇ Index ၶႃႈ။
2. **Strings:** ၵွပ်ႈ String ပဵၼ် Character Array၊ ႁဝ်းၸႂ်ႉ Pointer တႃႇၸတ်းၵၢၼ်လိၵ်ႈ လႆႈငၢႆႈလိူဝ်ၼမ်ၶႃႈ။
3. **Dynamic Memory:** မိူဝ်ႈႁဝ်းၶေႃ Memory မႂ်ႇ (Lesson 40)၊ ႁဝ်းတေလႆႈ Address မႃးသေ ၸႂ်ႉမၼ်းမိူၼ် Array ၼၼ်ႉၶႃႈဢေႃႈ။

### 4. လွင်ႈပႅၵ်ႇပိူင်ႈဢွၼ်ႇဢွၼ်ႇ
* **Array name** ပဵၼ် **Constant Pointer** (ၸဝ်ႈၵဝ်ႇတႅမ်ႈ `arr++` ဢမ်ႇလႆႈ ၵွပ်ႈမၢႆလိၵ်ႈႁိူၼ်းမၼ်းၵိုတ်းဝႆႉတီႈၵဝ်ႇ)။
* **Pointer Variable** သမ်ႉပဵၼ် **Non-constant** (ၸဝ်ႈၵဝ်ႇတႅမ်ႈ `ptr++` လႆႈ တွၼ်ႈတႃႇယဵပ်ႇၵႂႃႇတူၺ်းတူဝ်တၢင်ႇဢၼ်ၶႃႈ)။

  ၼႆႉၸင်ႇႁဵတ်းႁႂ်ႈ C ပဵၼ်လိၵ်ႈဢၼ် "ၽႂ်း" သေပိူၼ်ႈ ၵွပ်ႈမၼ်းဢမ်ႇသဵင်ႈ Memory တႃႇၵေႃႇပီႇၶေႃႈမုၼ်းၵႂႃႇၵျွၵ်းၵႂႃႇမႃးၼၼ်ႉၶႃႈယဝ်ႉ။
