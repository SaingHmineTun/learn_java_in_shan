## Lesson 8: Primitive Data Types: The Java vs. C Gap

### 1. Primitive Data Types ၼႂ်း C မီးသင်ပႃး?
ၼႂ်း C ၼႆႉ မၼ်းဢမ်ႇၸႅၵ်ႇလၢႆမဵဝ်းမိူၼ် Java ၶႃႈ။ မၼ်းမီးပိုၼ်ႉထၢၼ် 4 မဵဝ်းပိူင်လူင်ၵူၺ်း:
1.  **`char`**: ၵဵပ်းတူဝ်လိၵ်ႈ (ဢမ်ႇၼၼ် တူဝ်ၼပ်ႉဢွၼ်ႇ 1 Byte)။
2.  **`int`**: ၵဵပ်းတူဝ်ၼပ်ႉတဵမ်။
3.  **`float`**: ၵဵပ်းတူဝ်ၼပ်ႉမီးၸုၵ်း (Single precision)။
4.  **`double`**: ၵဵပ်းတူဝ်ၼပ်ႉမီးၸုၵ်း (Double precision)။

---

### 2. လွင်ႈပႅၵ်ႇပိူင်ႈ (The Big Differences)

| Feature | **Java** | **C** |
| :--- | :--- | :--- |
| **Boolean** | မီး `boolean` (true/false) | ဢမ်ႇမီး (ၸႂ်ႉ `0` တႃႇ False, `1` တႃႇ True) |
| **Byte Size** | တႅတ်ႈတေႃး (int = 4 bytes တႃႇသေႇ) | ဢမ်ႇတႅတ်ႈတေႃး (ၸွမ်းၼင်ႇ OS/CPU) |
| **Default Value** | မီး (int = 0 တႃႇသေႇ) | ဢမ်ႇမီး (ပဵၼ် "Garbage Value") |
| **String** | ပဵၼ် Object (မီး Method ၼမ်) | ဢမ်ႇမီး (ပဵၼ် `char` array ၵူၺ်း) |



---

### 3. "Garbage Value" ၼႆႉ ပဵၼ်သင်? (The Danger)
ၼႂ်း Java မိူဝ်ႈၸဝ်ႈၵဝ်ႇတႅမ်ႈ `int x;` သေ ဢမ်ႇပၼ်တူဝ်ၼပ်ႉမၼ်းၼၼ်ႉ Java တေထႅမ် `0` ပၼ်ႁင်းမၼ်းၶႃႈ။ ၵူၺ်းၵႃႈ ၼႂ်း **C** မၼ်းဢမ်ႇႁဵတ်းၼၼ်ၶႃႈ!
* **C** တေဢဝ် "ၶေႃႈမုၼ်းၵဝ်ႇ" ဢၼ်မီးဝႆႉၼႂ်း RAMAddress ၼၼ်ႉမႃးၼႄၵမ်းလဵဝ်။ ႁဝ်းႁွင်ႉဝႃႈ **Garbage Value** (ၶႅတ်းလႅတ်း)။
* **သတိ:** လူဝ်ႇပၼ်တူဝ်ၼပ်ႉ (Initialize) Variable တႃႇသေႇ မိူၼ်ၼင်ႇ `int x = 0;` ၶႃႈ။

---

### 4. တူဝ်ၼပ်ႉ တႅၼ်း တူဝ်လိၵ်ႈ (Char is an Integer)
ၼႂ်း C၊ `char` ၼႆႉ တေႃႉတိုၵ်ႉပဵၼ်တူဝ်ၼပ်ႉ (Integer) ဢွၼ်ႇ ဢၼ်မီးသႅၼ်း 1 Byte ၵူၺ်းၶႃႈ။ ႁဝ်းဢဝ်တူဝ်ၼပ်ႉမၼ်း ၼပ်ႉဢၢၼ်ႇလႆႈတူဝ်တူဝ်ၶႃႈဢေႃႈ။

```c
#include <stdio.h>

int main() {
    char letter = 'A';
    printf("Character: %c\n", letter); // ၼႄ 'A'
    printf("ASCII Value: %d\n", letter); // ၼႄ 65
    return 0;
}
```



---

### 5. ၵွပ်ႈသင် C သမ်ႉဢမ်ႇမီး `boolean`?
မိူဝ်ႈပီ 1972 ၼၼ်ႉ Dennis Ritchie ဝူၼ်ႉဝႃႈ လွင်ႈၸႂ်ႉ `0` လႄႈ `1` ၼၼ်ႉ မၼ်းသႅၼ်ႈသႂ် လႄႈ ၸႂ်ႉ RAM ဢေႇလိူဝ်ၶႃႈ။
* **0** = **False**
* **တူဝ်ၼပ်ႉတၢင်ႇၸိူဝ်း (1, 2, -5)** = **True**
  *(ၼႂ်း C99 ၸင်ႇမီး `<stdbool.h>` တွၼ်ႈတႃႇၸႂ်ႉ `bool` လႆႈမိူၼ် Java ၶႃႈ)*
