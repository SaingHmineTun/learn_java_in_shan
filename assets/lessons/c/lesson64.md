## Lesson 64: File Positioning: fseek, ftell & rewind

မိူဝ်ႈႁဝ်းပိုတ်ႇ File ၼိုင်ႈဢၼ်၊ C တေမီး **"File Pointer"** ဢွၼ်ႇတူဝ်ၼိုင်ႈ ဢၼ်ၸี้ဝႆႉတီႈတႄႇမၼ်း။ မိူဝ်ႈႁဝ်းလူလိၵ်ႈ၊ Pointer ၼၼ်ႉၵေႃႈတေၶယိူၼ်ႉၵႂႃႇၽၢႆႇၼႃႈတိၵ်းတိၵ်းၶႃႈ။

### 1. `ftell()` (ႁႃတီႈယူႇ)
ၸႂ်ႉတွၼ်ႈတႃႇထၢမ်ဝႃႈ "ယၢမ်းလဵဝ် ႁဝ်းလူလိၵ်ႈထိုင် Byte တူဝ်ထီႉၵႃႈႁိုဝ်ယဝ်ႉ?"
* **Return:** တူဝ်ၼပ်ႉ `long` (Byte position)။
```c
long pos = ftell(fp);
printf("Current position: %ld\n", pos);
```

### 2. `rewind()` (ပွၵ်ႈမိူဝ်းတီႈတႄႇ)
ၸႂ်ႉတွၼ်ႈတႃႇႁႂ်ႈ File Pointer ပွၵ်ႈမိူဝ်းၸี้တီႈ **"တႄႇသုတ်း"** ၶွင် File ၶိုၼ်းၵမ်းလဵဝ်ၶႃႈ။
```c
rewind(fp); // မိူၼ်တင်း fseek(fp, 0, SEEK_SET) ၶႃႈ
```

### 3. `fseek()` (ၵျွၵ်းၵႂႃႇၸူးတီႈဢၼ်လူဝ်ႇ)
ၼႆႉပဵၼ် Function ဢၼ်မီးဢဵၼ်ႁႅင်းသုတ်း ၵွပ်ႈမၼ်းၸၢင်ႈ "သင်ႇ" ႁႂ်ႈ Pointer ၵျွၵ်းၵႂႃႇလႆႈ ၵူႈတီႈၶႃႈ။



**Syntax:**
```c
fseek(fp, offset, origin);
```
* **offset:** တေၵျွၵ်းၵႂႃႇလၢႆ Byte? (ၸၢင်ႈပဵၼ် + ႁိုဝ် -)။
* **origin:** တေတႄႇၼပ်ႉၵႃႈတီႈလႂ်?
    * **`SEEK_SET`**: တႄႇၼပ်ႉတႄႇဢဝ် **"ႁူဝ်သုတ်း"** File။
    * **`SEEK_CUR`**: တႄႇၼပ်ႉတႄႇဢဝ် **"တီႈယူႇယၢမ်းလဵဝ်"**။
    * **`SEEK_END`**: တႄႇၼပ်ႉတႄႇဢဝ် **"ပၢႆသုတ်း"** File။

### 4. တူဝ်ယၢင်ႇ: Updating a Specific Record
သင်ၸဝ်ႈၵဝ်ႇမီးလုၵ်ႈႁဵၼ်း 100 ၵေႃႉ၊ သေၶႂ်ႈမႄးၶေႃႈမုၼ်း **ၵေႃႉထီႉ 5** ၵမ်းလဵဝ် လူၺ်ႈဢမ်ႇလူဝ်ႇလူတႄႇဢဝ်ၵေႃႉထီႉ 1 ၼႆ:

```c
struct Student s;
FILE *fp = fopen("students.dat", "rb+"); // rb+ = ပိုတ်ႇတႃႇလူလႄႈတႅမ်ႈ Binary

// ၵျွၵ်းၵႂႃႇၸူး ၵေႃႉထီႉ 5 (ၼပ်ႉတႄႇဢဝ် 0, 1, 2, 3, 4)
fseek(fp, 4 * sizeof(struct Student), SEEK_SET);

// တႅမ်ႈၶေႃႈမုၼ်းမႂ်ႇ သႂ်ႇတီႈၼၼ်ႉၵမ်းလဵဝ်
fwrite(&newStudent, sizeof(struct Student), 1, fp);

fclose(fp);
```
