## Lesson 3: Anatomy of a C Program: Structure & Syntax

### 1. တူဝ်ယၢင်ႇ Code ႁူဝ်ทีပထမ (Hello World Example)
ၼႆႉပဵၼ် ပူဝ်ႇၵရႅမ်ႇဢၼ်ငၢႆႈသုတ်း ဢၼ်ႁဝ်းတေၸႂ်ႉတႅမ်ႈၼႄ လုမ်ႈၾႃႉၶႃႈ -

```c
#include <stdio.h>

int main() {
    printf("Hello, Shan Learners!");
    return 0;
}
```

### 2. ၸႅၵ်ႇၽႄဢင်ႇၵႃႇ (Breaking Down the Anatomy)
ႁဝ်းတေမႃးတူၺ်း ထႅဝ်ၼိုင်ႈလႂ် မၼ်းမီးတီႈပွင်ႇ လႄႈ ၼႃႈၵၢၼ်ၸိူဝ်ႉႁိူဝ်?

* **`#include <stdio.h>` (Preprocessor Directive):** ဢၼ်ၼႆႉ ပဵၼ်ၵၢၼ်ႁွင်ႉဢဝ် "ပပ်ႉၶိူင်ႈမိုဝ်း" (Library) မႃးၸႂ်ႉ။ `stdio.h` ပွင်ႇဝႃႈ **Standard Input Output**။ မၼ်းၸွႆႈႁႂ်ႈပူဝ်ႇၵရႅမ်ႇႁဝ်း ၼႄလိၵ်ႈဢွၵ်ႇ Screen လႆႈၶႃႈ။
* **`int main()` (The Entry Point):** ၼႆႉပဵၼ် "ႁူဝ်ၸႂ်" ၶွင်ပူဝ်ႇၵရႅမ်ႇ။ ၶွမ်ႇပိဝ်ႇတႃႇတေတႄႇဢၢၼ်ႇ Code တီႈၼႆႈပဵၼ် လၢတ်ႈဢွၼ်တၢင်းသုတ်းၶႃႈ။
* **`{ ... }` (Curly Braces):** ပဵၼ်ၵၢၼ်မၢႆဝႆႉ "ၶွပ်ႇပိူင်" (Block)။ Code ဢၼ်မီးၼႂ်းၵႄႈမၼ်းၼႆႉ ပဵၼ်ၵၢၼ်ႁဵတ်းသၢင်ႈၶွင် `main` တူဝ်ႈတူဝ်ႈၶႃႈ။
* **`printf("...");` (Statement):** ပဵၼ်ၶေႃႈၵႂၢမ်း သင်ႇႁႂ်ႈၶွမ်ႇပိဝ်ႇတႃႇ "ၼႄလိၵ်ႈ" ဢွၵ်ႇမႃး။ ၼႂ်းလိၵ်ႈ C ႁဝ်းႁွင်ႉထႅဝ်ၼိုင်ႈလႂ်ဝႃႈ **Statement** ၶႃႈ။
* **`;` (Semicolon):** ၼႆႉပဵၼ် "မၢႆထႅဝ်ယဝ်ႉ" (Statement Terminator)။ ၼႂ်းလိၵ်ႈ C ၼႆႉ ပေႃးယဝ်ႉထႅဝ်ၼိုင်ႈ လူဝ်ႇသႂ်ႇ `;` တႃႇသေႇ၊ သင်လိုမ်းၼႆ ပူဝ်ႇၵရႅမ်ႇတေဢမ်ႇ Run ၶႃႈ။
* **`return 0;`:** ပဵၼ်လွင်ႈလၢတ်ႈၼႄ OS ဝႃႈ "ပူဝ်ႇၵရႅမ်ႇၼႆႉ ႁဵတ်းၵၢၼ်ယဝ်ႉတူဝ်ႈလီယဝ်ႉ၊ ဢမ်ႇမီးလွင်ႈၽိတ်းသင်" ၼႆၶႃႈဢေႃႈ။



### 3. မၢႆမီႈ Syntax ဢၼ်လမ်ႇလွင်ႈ (Key Syntax Rules)
တႃႇတေတႅမ်ႈ Code ႁႂ်ႈမၼ်းသၢင်ႇထုၵ်ႇၼၼ်ႉ လူဝ်ႇတွၼ်းမၢႆဝႆႉ 3 ယိူင်ႈၶႃႈ -
1.  **Case Sensitive:** တူဝ်လိၵ်ႈဢွၼ်ႇ လႄႈ တူဝ်လိၵ်ႈလူင် ပႅၵ်ႇပိူင်ႈၵၼ်ၶႃႈ (မိူၼ်ၼင်ႇ `Main` လႄႈ `main` ၼၼ်ႉ မၼ်းဢမ်ႇမိူၼ်ၵၼ်၊ ၶွမ်ႇပိဝ်ႇတႃႇတေႁၼ်ပဵၼ် ၵူၼ်းၵေႃႉပႅၵ်ႇၵၼ်ၶႃႈ)။
2.  **White Space:** ဢွင်ႈဝၢင်ႇ (Space) ၼႂ်းၵႄႈထႅဝ်ၼၼ်ႉ C ဢမ်ႇသူၼ်ၸႂ်ၶႃႈ၊ ၵူၺ်းၵႃႈ ႁဝ်းလူဝ်ႇၸႂ်ႉမၼ်း တႃႇႁႂ်ႈ "ၵူၼ်း" လူလိၵ်ႈလႆႈငၢႆႈငၢႆႈ (Indentation)။
3.  **Comments (လွင်ႈတႅမ်ႈမၢႆဝႆႉ):**
    * `//` : တႃႇတႅမ်ႈမၢႆဝႆႉ ၼႂ်းထႅဝ်လဵဝ်။
    * `/* ... */` : တႃႇတႅမ်ႈမၢႆဝႆႉ လၢႆလၢႆထႅဝ်။
      *(ၶွမ်ႇပိဝ်ႇတႃႇ တေဢမ်ႇဢၢၼ်ႇ Code ၼႂ်း Comment ၶႃႈ)*



### 4. လွင်ႈတႅမ်ႈ Code ႁႂ်ႈႁၢင်ႈလီ (Indentation)
ၵၢၼ်ယဵပ်ႇထႅဝ် (Indentation) ၼၼ်ႉ လမ်ႇလွင်ႈတေႉတေႉ တွၼ်ႈတႃႇၵေႃႉတႅမ်ႈ Code (Programmer)။ မိူဝ်ႈႁဝ်းတႅမ်ႈ Code ၼႂ်း `{ }` ၼၼ်ႉ ႁဝ်းလူဝ်ႇ "ယဵပ်ႇ" (Tab) ၶဝ်ႈၵႂႃႇ 4 ဢွင်ႈဝၢင်ႇ တွၼ်ႈတႃႇႁႂ်ႈႁူႉဝႃႈ Code ၸိူဝ်းၼၼ်ႉ ပဵၼ်လုၵ်ႈၼွင်ႉၶွင် Function ၼၼ်ႉၶႃႈဢေႃႈ။
