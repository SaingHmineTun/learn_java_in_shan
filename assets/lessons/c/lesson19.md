## Lesson 19: Type Casting & Promotion: Mixing Data Types

### 1. Type Promotion (ၵၢၼ်လႅၵ်ႈလၢႆႈႁင်းၵူၺ်း)
မၢင်ပွၵ်ႈ C တေပိၼ်ႇမဵဝ်းၶေႃႈမုၼ်းပၼ်ႁင်းမၼ်း (Implicit Conversion) တွၼ်ႈတႃႇႁႄႉၵင်ႈဢမ်ႇႁႂ်ႈၶေႃႈမုၼ်းႁၢႆၶႃႈ။
* **Rule:** တူဝ်ၼပ်ႉဢၼ် "ဢွၼ်ႇ" (Memory ဢေႇ) တေထုၵ်ႇပိၼ်ႇပဵၼ်တူဝ်ၼပ်ႉဢၼ် "ယႂ်ႇ" (Memory ၼမ်) တႃႇသေႇၶႃႈ။
* **Hierarchy:** `char` -> `int` -> `float` -> `double`

**တူဝ်ယၢင်ႇ:**
```c
int a = 5;
float b = 2.0;
float res = a + b; // 'a' တေထုၵ်ႇပိၼ်ႇပဵၼ် float (5.0) ဢွၼ်တၢင်း ၸင်ႇၼပ်ႉ
```



### 2. Type Casting (ၵၢၼ်ၸိသင်ႇပိၼ်ႇမဵဝ်း)
ၼႆႉပဵၼ်ၵၢၼ်ဢၼ်ႁဝ်း "သင်ႇ" (Explicit Conversion) ႁႂ်ႈ Compiler ပိၼ်ႇမဵဝ်းၶေႃႈမုၼ်းၸွမ်းၼင်ႇႁဝ်းၶႂ်ႈလႆႈၶႃႈ။
* **Syntax:** `(type) value;`

**တူဝ်ယၢင်ႇ ဢၼ်ၸႂ်ႉၼမ်သုတ်း (Division):**
```c
int x = 5, y = 2;
float res = (float)x / y; // တွပ်ႇမၼ်းတေဢွၵ်ႇ 2.5 (ဢမ်ႇၸႂ် 2)
```
*(သင်ဢမ်ႇသႂ်ႇ `(float)` ၼၼ်ႉ တွပ်ႇမၼ်းတေဢွၵ်ႇ `2.0` ၵူၺ်း ၵွပ်ႈဝႃႈမၼ်းပဵၼ် Integer Division ၶႃႈ)*

### 3. Data Loss (လွင်ႈၶေႃႈမုၼ်းႁၢႆ)
မိူဝ်ႈႁဝ်းပိၼ်ႇၶေႃႈမုၼ်း "ယႂ်ႇ" ႁႂ်ႈပဵၼ် "ဢွၼ်ႇ" (Narrowing) ၼၼ်ႉ ၶေႃႈမုၼ်းၸၢင်ႈႁၢႆၵႂႃႇၶႃႈ။
* **Double -> Int:** `(int) 3.99` တွပ်ႇမၼ်းတေဢွၵ်ႇ **3** ၵူၺ်း (မၼ်းတေဢမ်ႇၼပ်ႉတူဝ်ႁွႆႈ၊ မၼ်းတေ "တတ်း" ပႅတ်ႈတူဝ်တၢင်းလင်တင်းသဵင်ႈၶႃႈ)။



### 4. Integer Promotion ၶွင် `char`
မိူၼ်ၼင်ႇႁဝ်းႁဵၼ်းမႃး၊ `char` ၼႆႉ ပဵၼ်တူဝ်ၼပ်ႉဢွၼ်ႇ (1 Byte)။ မိူဝ်ႈႁဝ်းဢဝ်မၼ်းမႃးၼပ်ႉၸွမ်း `int` ၼၼ်ႉ C တေဢဝ်မၢႆ **ASCII** မၼ်းမႃးၸႂ်ႉၶႃႈ။
```c
char c = 'A'; // ASCII 65
int res = c + 5; // 65 + 5 = 70
```
