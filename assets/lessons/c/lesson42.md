## Lesson 42: Dangling Pointers & Memory Leaks

### 1. Dangling Pointers (Pointer ဢၼ်ၸี้ၵႂႃႇတီႈလႅင်)
**Dangling Pointer** ပွင်ႇဝႃႈ ပဵၼ် Pointer ဢၼ်တိုၵ်ႉၵဵပ်း Address ၶွင် Memory ဢၼ်ထုၵ်ႇ "ယႃႉပႅတ်ႈ" (Deleted/Freed) ယဝ်ႉၼၼ်ႉၶႃႈ။

* **လွင်ႈတၢင်း:** မိူဝ်ႈႁဝ်းၸႂ်ႉ `free(ptr)` ယဝ်ႉ၊ ၵူၺ်းၵႃႈ `ptr` သမ်ႉတိုၵ်ႉမီး Address ႁွႆးၵဝ်ႇဝႆႉယူႇ။
* **ၽေး:** သင်ႁဝ်းၸၢမ်း Dereference (`*ptr`) ၼႆ ပူဝ်ႇၵရႅမ်ႇတေ Crash ၵမ်းလဵဝ် ၵွပ်ႈမၼ်းၵႂႃႇၵဝ်း Memory ဢၼ်ဢမ်ႇၸႂ်ႇၶွင်မၼ်းယဝ်ႉ။



**လၢႆးၵႄႈ:** ဝၢႆးသေ `free(ptr)` ယဝ်ႉတူဝ်ႈ၊ ႁႂ်ႈပၼ်မၼ်းပဵၼ် **`NULL`** တႃႇသေႇၶႃႈ။
```c
free(ptr);
ptr = NULL; // ယဝ်ႉယဝ်ႉ ဢမ်ႇမီးၽေး
```

### 2. Memory Leaks (Memory ယွႆႉ)
**Memory Leak** ပဵၼ်လွင်ႈတၢင်း ဢၼ်ႁဝ်း "ၶေႃ" Memory (`malloc`) မႃးၸႂ်ႉယဝ်ႉ၊ ၵူၺ်းၵႃႈ "လိုမ်းၶိုၼ်းပၼ်" (`free`) မိူဝ်ႈယဝ်ႉတူဝ်ႈၼၼ်ႉၶႃႈ။

* **ၽွၼ်းလႆႈ:** RAM တေထုၵ်ႇၸႂ်ႉၵႂႃႇတိၵ်းတိၵ်း တေႃႇပေႃးတဵမ်။ သင်ပဵၼ် Software ဢၼ်လူဝ်ႇပိုတ်ႇဝႆႉႁိုင်ႁိုင် (မိူၼ်ၼင်ႇ Server) ၼႆ မၼ်းတေႁဵတ်းႁႂ်ႈ System တင်းသဵင်ႈ ၵိုတ်းၵႂႃႇလႆႈၶႃႈဢေႃႈ။



**တူဝ်ယၢင်ႇ ဢၼ်ၽိတ်း:**
```c
void my_func() {
    int *data = (int*) malloc(100 * sizeof(int));
    // ႁဵတ်းၵၢၼ်ယဝ်ႉ... ၵူၺ်းၵႃႈ ဢမ်ႇ free(data)
} 
// မိူဝ်ႈဢွၵ်ႇ Function၊ Pointer 'data' တေႁၢႆၵႂႃႇ ၵူၺ်းၵႃႈ Memory 100 ႁွင်ႉၼၼ်ႉ တေၵိုတ်းၶႂ်ဝႆႉၼႂ်း RAM တႃႇသေႇ
```

### 3. Wild Pointers (Pointer ဢၼ်ဢမ်ႇမီးပေႃႈမႄး)
ပဵၼ် Pointer ဢၼ်ႁဝ်း "ပိုၼ်ၽၢဝ်ႇ" (Declare) ဝႆႉ၊ ၵူၺ်းၵႃႈ "ဢမ်ႇပၼ် Address" မၼ်းသေဢိတ်း။ မၼ်းတေၸี้ၵႂႃႇတီႈ Memory ဢၼ်သုၵ်ႉယုင်ႈ (Garbage Address) သေ ႁဵတ်းႁႂ်ႈပူဝ်ႇၵရႅမ်ႇလူလႆႈၵမ်းလဵဝ်ၶႃႈ။
