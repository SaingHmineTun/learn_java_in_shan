## Lesson 16: Bitwise Operators: Thinking in Binary

### 1. Bitwise Operators ၼႆႉ ပဵၼ်သင်?
ၼႂ်းလိၵ်ႈ C၊ **Bitwise Operators** ၼႆႉ ၸႂ်ႉတႃႇၼပ်ႉဢၢၼ်ႇၶေႃႈမုၼ်း ၼႂ်းၸၼ်ႉ **Binary (0, 1)** ၶႃႈ။ မၼ်းဢမ်ႇတူၺ်းတူဝ်ၼပ်ႉတင်းတူဝ် (မိူၼ်ၼင်ႇ 5)၊ မၼ်းတေတူၺ်းၵႂႃႇတီႈ Bit ၼိုင်ႈတူဝ်လႂ် ၼႂ်း Memory ၶႃႈဢေႃႈ။



### 2. တူဝ်မၢႆ Bitwise ပိူင်လူင်
ႁႂ်ႈဝူၼ်ႉတူၺ်းဝႃႈ တူဝ်ၼပ်ႉ 5 ပဵၼ် `0101` လႄႈ တူဝ်ၼပ်ႉ 3 ပဵၼ် `0011` ၼႆၶႃႈလႃႆ။

| Operator | ၸိုဝ်ႈ (Name) | တီႈပွင်ႇ (Meaning)                                     |
|:---------| :--- |:-------------------------------------------------------|
| **`&`**  | **AND** | တေပဵၼ် 1 မိူဝ်ႈ **Bit တင်းသွင်** ပဵၼ် 1                |
| **`\|`** | **OR** | တေပဵၼ် 1 မိူဝ်ႈ **Bit သေဢၼ်ဢၼ်ပဵၼ်ၼိုင်ႈ** ပဵၼ် 1 |
| **`^`**  | **XOR** | တေပဵၼ် 1 မိူဝ်ႈ **Bit သွင်တူဝ် ဢမ်ႇမိူၼ်ၵၼ်**          |
| **`~`**  | **NOT** | ပိၼ်ႇ 0 ႁႂ်ႈပဵၼ် 1 (လႄႈ ပိၼ်ႇ 1 ပဵၼ် 0)                |
| **`<<`** | **Left Shift** | ယဵပ်ႇ Bit ၵႂႃႇၽၢႆႇသၢႆႉ (ထႅမ် 0 တီႈပၢႆ)                 |
| **`>>`** | **Right Shift** | ယဵပ်ႇ Bit ၵႂႃႇၽၢႆႇၶႂႃ (ထွၼ် 0 တီႈပၢႆ)                  |

### 3. Left Shift (`<<`) လႄႈ Right Shift (`>>`)
ၼႆႉပဵၼ်လၢႆး "ၼပ်ႉဢၢၼ်ႇၽႂ်း" ၼႂ်းၶွမ်ႇပိဝ်ႇတႃႇၶႃႈ။
* **`x << n`**: မိူၼ်ၵၼ်တင်း **`x * (2^n)`** (ပုၼ်းတင်း 2 ၼမ်ႉၵေႃႉလႂ် n ပၼ်ႇ)။
* **`x >> n`**: မိူၼ်ၵၼ်တင်း **`x / (2^n)`** (ၽႄတင်း 2 ၼမ်ႉၵေႃႉလႂ် n ပၼ်ႇ)။

**တူဝ်ယၢင်ႇ:** `5 << 1` (Binary `0101` ပဵၼ် `1010`) တွပ်ႇမၼ်းတေပဵၼ် **10** ၶႃႈ။

### 4. ၵွပ်ႈသင်ႁဝ်းလူဝ်ႇၸႂ်ႉ?
1. **Speed:** ၵၢၼ်ၸႂ်ႉ Bitwise မၼ်းၽႂ်းသေ ၵၢၼ်ၸႂ်ႉ `*` ဢမ်ႇၼၼ် `/` တေႉတေႉၶႃႈ။
2. **Memory Saving:** ႁဝ်းၵဵပ်းၶေႃႈမုၼ်း (True/False) လႆႈ 8 မဵဝ်း ၼႂ်း `char` (1 Byte) တူဝ်လဵဝ်ၵူၺ်း။
3. **Encoding:** ၸႂ်ႉတႃႇၸတ်းၵၢၼ်လိၵ်ႈတႆး (Unicode) ၼႂ်းၸၼ်ႉ Bytes ၶႃႈဢေႃႈ။

### 5. တူဝ်ယၢင်ႇ Code
```c
#include <stdio.h>

int main() {
    int a = 5; // 0101
    int b = 3; // 0011
    
    printf("a & b: %d\n", a & b); // 0001 = 1
    printf("a | b: %d\n", a | b); // 0111 = 7
    printf("a ^ b: %d\n", a ^ b); // 0110 = 6
    printf("5 << 1: %d\n", a << 1); // 1010 = 10
    
    return 0;
}
```
