## Lesson 7: Memory Architecture: How Variables Work

### 1. Memory (RAM) ၼႆႉ ပဵၼ်သင်?
မိူဝ်ႈႁဝ်း Run ပူဝ်ႇၵရႅမ်ႇ C ၼၼ်ႉ၊ ၶေႃႈမုၼ်း (Data) တင်းသဵင်ႈ တေၵႂႃႇၵဵပ်းဝႆႉၼႂ်း **RAM** (Random Access Memory)။ ႁႂ်ႈဝူၼ်ႉတူၺ်းဝႃႈ RAM ၼႆႉ မိူၼ်ၼင်ႇ "ႁူင်းႁႅမ်းလူင်" ဢၼ်မီးႁွင်ႉၼမ်ၼမ်ၶႃႈ။
* ႁွင်ႉၼိုင်ႈႁွင်ႉ = **1 Byte** (မီး 8 Bits)။
* ႁွင်ႉၵူႈႁွင်ႉ မီး **"မၢႆႁိူၼ်း"** (Address) ၶွင်မၼ်းႁင်းၵူၺ်း။



### 2. Variable ပဵၼ် "ၸိုဝ်ႈတႅၼ်း" မၢႆႁိူၼ်း
ၼႂ်းလိၵ်ႈ C၊ **Variable** ၼႆႉ ဢမ်ႇၸႂ်ႉၵူၺ်းပဵၼ်ၵွၵ်းၵဵပ်းၶေႃႈမုၼ်း၊ မၼ်းပဵၼ် "ၸိုဝ်ႈ" ဢၼ်ႁဝ်းတင်ႈပၼ် တွၼ်ႈတႃႇမၢႆႁိူၼ်း (Memory Address) ၼၼ်ႉၶႃႈဢေႃႈ။
* တူဝ်ယၢင်ႇ: `int age = 20;`
* ၶွမ်ႇပိဝ်ႇတႃႇတေၵႂႃႇၸွၵ်းဢဝ် ဢွင်ႈဝၢင်ႇၼႂ်း RAM (တူဝ်ယၢင်ႇ: Address `0x100`) သေ သႂ်ႇတူဝ်ၼပ်ႉ `20` ဝႆႉၼႂ်းၼၼ်ႉၶႃႈ။

### 3. လွင်ႈၸႂ်ႉမၢႆ `&` (Address-of Operator)
ၼႆႉပဵၼ်လွင်ႈၶိုၵ်ႉတွၼ်းၶွင်လိၵ်ႈ C ၶႃႈ။ ႁဝ်းၸၢမ်းတူၺ်း "မၢႆႁိူၼ်း" တေႉတေႉမၼ်းလႆႈ လူၺ်ႈၸႂ်ႉမၢႆ **`&`** ဝႆႉတီႈၼႃႈၸိုဝ်ႈ Variable ၶႃႈ။

```c
#include <stdio.h>

int main() {
    int age = 20;
    // %p ၸႂ်ႉတႃႇၼႄမၢႆ Address (Pointer)
    printf("Value of age: %d\n", age);
    printf("Memory Address of age: %p\n", &age); 
    return 0;
}
```
*(မၢႆ Address တေဢွၵ်ႇမႃးပဵၼ် Hexadecimal မိူၼ်ၼင်ႇ `0x7ffee...` ၼၼ်ႉၶႃႈဢေႃႈ)*

### 4. Memory Layout ၶွင် Variable
Variable ၼိုင်ႈမဵဝ်း လႄႈ ၼိုင်ႈမဵဝ်း ၸႂ်ႉတူဝ်ၼပ်ႉ "ႁွင်ႉႁႅမ်း" (Bytes) ဢမ်ႇတူဝ်ႇၵၼ်ၶႃႈ:
* **`char`**: ၸႂ်ႉ 1 ႁွင်ႉ (1 Byte)
* **`int`**: ၸႂ်ႉ 4 ႁွင်ႉ (4 Bytes)
* **`double`**: ၸႂ်ႉ 8 ႁွင်ႉ (8 Bytes)



### 5. ၵွပ်ႈသင်ႁဝ်းလူဝ်ႇပွင်ႇၸႂ် Memory?
1. **Efficiency:** တႃႇတႅမ်ႈ Code ႁႂ်ႈၸႂ်ႉ RAM ဢေႇသုတ်း (လမ်ႇလွင်ႈတႃႇတႅမ်ႈ App ၼိူဝ် Mobile)။
2. **Pointers:** သင်ၸဝ်ႈၵဝ်ႇ ပွင်ႇၸႂ်လွင်ႈ Memory Address ယဝ်ႉၼႆ၊ လွင်ႈ Pointers ဢၼ်ပိူၼ်ႈဝႃႈယၢပ်ႇၼၼ်ႉ တေငၢႆႈၵမ်းလဵဝ်ၶႃႈ။
3. **Debugging:** ႁူႉဝႃႈၵွပ်ႈသင် ပူဝ်ႇၵရႅမ်ႇသမ်ႉ Crash (မိူၼ်ၼင်ႇ Segmentation Fault)။

* တူၺ်းမၢႆ Address ၸိူဝ်းၼၼ်ႉ၊ မၼ်းမီးလွင်ႈလႅၵ်ႈလၢႆႈၸိူဝ်ႉႁိူဝ် မိူဝ်ႈၸဝ်ႈၵဝ်ႇ ပိတ်းသေ ပိုတ်ႇ Run ပူဝ်ႇၵရႅမ်ႇမႂ်ႇထႅင်ႈပွၵ်ႈၼိုင်ႈ? (ၼႆႉတေသွၼ်ပၼ်လွင်ႈ Dynamic Memory ၶႃႈ)။
