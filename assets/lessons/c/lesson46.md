## Lesson 46: Function Basics: Defining Your Own Tools

### 1. Function ပဵၼ်သင်?
**Function** ၼႆႉ ပဵၼ်တွၼ်ႈ Code ဢၼ်ႁဝ်း "ၸႅၵ်ႇဝႆႉႁင်းၵူၺ်း" တွၼ်ႈတႃႇႁဵတ်းၵၢၼ်သင်မဵဝ်းမဵဝ်း။ ႁႂ်ႈဝူၼ်ႉတူၺ်းဝႃႈ မၼ်းပဵၼ် **"Recipe"** (လၢႆးႁဵတ်းၵိၼ်) ဢၼ်ႁဝ်းတႅမ်ႈဝႆႉသေ မိူဝ်ႈလႂ်ႁဝ်းၶႂ်ႈၵိၼ် ႁဝ်းၵေႃႈၵူၺ်း "ႁွင်ႉၸႂ်ႉ" (Call) မၼ်းၵူၺ်းၶႃႈ။



### 2. ပိူင်ၵေႃႇသၢင်ႈ (Function Anatomy)
တွၼ်ႈတႃႇတေပိုၼ်ၽၢဝ်ႇ Function ၼိုင်ႈဢၼ် ႁဝ်းလူဝ်ႇမီး 4 ပွတ်းၶႃႈ:
1.  **Return Type:** ၽွၼ်းလႆႈဢၼ်မၼ်းတေသူင်ႇၶိုၼ်း (မိူၼ်ၼင်ႇ `int`, `void`, `float`)။
2.  **Function Name:** ၸိုဝ်ႈၶိူင်ႈမိုဝ်းႁဝ်း (မိူၼ်ၼင်ႇ `displayMenu`)။
3.  **Parameters:** ၶေႃႈမုၼ်းဢၼ်မၼ်းလူဝ်ႇၸႂ်ႉ (Input)။
4.  **Body:** Code ဢၼ်ယူႇၼႂ်း `{ }` ဢၼ်တေႁဵတ်းၵၢၼ်တေႉတေႉ။

### 3. တူဝ်ယၢင်ႇတူဝ်တေႉ
မႃးႁဝ်းမႃးသၢင်ႈ Function ဢၼ်တေၼႄၸိုဝ်ႈသႅၼ်း (Header) တွၼ်ႈတႃႇ **TMK Student Manager** ၸဝ်ႈၵဝ်ႇၶႃႈ။

```c
#include <stdio.h>

// 1. Defining the Function (သၢင်ႈၶိူင်ႈမိုဝ်း)
void printHeader() {
    printf("==============================\n");
    printf("   TMK STUDENT MANAGER v1.0   \n");
    printf("==============================\n");
}

int main() {
    // 2. Calling the Function (ႁွင်ႉၸႂ်ႉ)
    printHeader(); 
    
    printf("Welcome, Sai Mao!\n");
    
    // ႁွင်ႉၸႂ်ႉထႅင်ႈၵမ်းၵေႃႈလႆႈ
    printHeader(); 
    
    return 0;
}
```

### 4. ၵွပ်ႈသင်ႁဝ်းလူဝ်ႇၸႂ်ႉ? (The "Why")
* **Don't Repeat Yourself (DRY):** သင်ၸဝ်ႈၵဝ်ႇလူဝ်ႇၼႄ Header ၼႆႉ 10 တီႈ၊ ၸဝ်ႈၵဝ်ႇဢမ်ႇလူဝ်ႇ Copy & Paste တႅမ်ႈ `printf` 30 ထႅဝ်ၶႃႈ။ ၵူၺ်းၵႃႈ တႅမ်ႈ `printHeader();` 10 ၵမ်းၵေႃႈယဝ်ႉယဝ်ႉ။
* **Modularization:** ၸွႆႈႁႂ်ႈ `main()` ၸဝ်ႈၵဝ်ႇသႅၼ်ႈသႂ်။ ၸဝ်ႈၵဝ်ႇတေႁၼ်ၵူၺ်းၸိုဝ်ႈ Function ဢၼ်လူငၢႆႈငၢႆႈ။
* **Maintenance:** သင်ၸဝ်ႈၵဝ်ႇၶႂ်ႈလႅၵ်ႈလၢႆႈၸိုဝ်ႈ App ပဵၼ် "Shan Student Pro" ၼႆ၊ ၸဝ်ႈၵဝ်ႇလႅၵ်ႈလၢႆႈတီႈၼႂ်း Function တီႈလဵဝ်ၵူၺ်း၊ တင်းပူဝ်ႇၵရႅမ်ႇၵေႃႈတေလႅၵ်ႈလၢႆႈၸွမ်းၵမ်းလဵဝ်ၶႃႈ။

## 🏗️ Logic for Expert Programming
ၼႂ်းၵၢၼ်တႅမ်ႈ Software ၸၼ်ႉသုင်၊ `main()` ၼႆႉ ထုၵ်ႇလီမီး Code ဢွၼ်ႇဢွၼ်ႇၵူၺ်းၶႃႈ။ မၼ်းလူဝ်ႇမီးဝႆႉၵူၺ်းတႃႇ "သင်ႇၵၢၼ်" ၸူး Functions တၢင်ႇဢၼ် ၼင်ႇၼႆ:
1. `initializeDatabase();`
2. `showLoginScreen();`
3. `runMainMenu();`
4. `closeProgram();`

ၵၢၼ်ႁဵတ်းၼႆ တေႁဵတ်းႁႂ်ႈ Code ၸဝ်ႈၵဝ်ႇ "ယုင်ႈ" (Complex) ၵႃႈႁိုဝ်ၵေႃႈ ၸဝ်ႈၵဝ်ႇတေတူၺ်းပွင်ႇၸႂ်လႆႈငၢႆႈငၢႆႈၶႃႈယဝ်ႉ။
