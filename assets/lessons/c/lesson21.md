## Lesson 21: The if-else Empire: Decision Making

### 1. Simple `if` (ၵၢၼ်တႅပ်းတတ်း လၢႆးငၢႆႈ)
ႁဝ်းၸႂ်ႉ `if` မိူဝ်ႈႁဝ်းၶႂ်ႈႁႂ်ႈ Code ႁဵတ်းၵၢၼ် **ၵေႃႉမိူဝ်ႈ Condition မၢၼ်ႇမႅၼ်ႈ (True)** ၵူၺ်းၶႃႈ။

**Syntax:**
```c
if (condition) {
    // Code တီႈၼႆႈ တေႁဵတ်းၵၢၼ် သင် Condition ပဵၼ် True (1)
}
```



### 2. `if-else` (သဵၼ်ႈတၢင်းသွင်ၽၢႆႇ)
သင် Condition ၽိတ်း (False) သမ်ႉ ႁဝ်းၶႂ်ႈႁႂ်ႈမၼ်းႁဵတ်းသင်? ႁဝ်းၸႂ်ႉ `else` တွၼ်ႈတႃႇမၢႆဝႆႉ သဵၼ်ႈတၢင်းထႅင်ႈၽၢႆႇၼိုင်ႈၶႃႈ။

**တူဝ်ယၢင်ႇ:**
```c
int marks = 35;
if (marks >= 40) {
    printf("You Passed!");
} else {
    printf("You Failed!");
}
```



### 3. `else-if` Ladder (ၵၢၼ်တႅပ်းတတ်း လၢႆးၸၼ်ႉ)
မိူဝ်ႈႁဝ်းမီး Condition ၼမ်ၼမ် (မိူၼ်ၼင်ႇ ၵၢၼ်ၼပ်ႉ Grade) ႁဝ်းၸႂ်ႉ `else if` တွၼ်ႈတႃႇၵူတ်ႇထတ်းၵႂႃႇ ၸၼ်ႉၼိုင်ႈယဝ်ႉၸၼ်ႉၼိုင်ႈၶႃႈ။

**တူဝ်ယၢင်ႇ:**
```c
if (marks >= 80) {
    printf("Grade A");
} else if (marks >= 60) {
    printf("Grade B");
} else if (marks >= 40) {
    printf("Grade C");
} else {
    printf("Grade F");
}
```



### 4. Nested `if` (If ၼႂ်း If)
ႁဝ်းၸၢင်ႈသႂ်ႇ `if` ဝႆႉၼႂ်း `if` ထႅင်ႈၸၼ်ႉၼိုင်ႈ တွၼ်ႈတႃႇၵူတ်ႇထတ်း လွင်ႈတႅပ်းတတ်းဢၼ်လိုၵ်ႉသေၵဝ်ႇၶႃႈ။

**တူဝ်ယၢင်ႇ:**
```c
if (isMember == 1) {
    if (age >= 60) {
        printf("Senior Discount: 50%%");
    } else {
        printf("Member Discount: 10%%");
    }
}
```

### 5. Common Pitfalls (လွင်ႈၸၢင်ႈၽိတ်း)
* **`=` vs `==`**: ၼႂ်း `if` လူဝ်ႇၸႂ်ႉ `==` တႃႇတႅၵ်ႈၼိူင်းတႃႇသေႇ။ သင်ၸဝ်ႈၵဝ်ႇတႅမ်ႈ `if (x = 5)` ၼႆ မၼ်းတေပဵၼ် **True** တႃႇသေႇ ၵွပ်ႈဝႃႈ 5 ဢမ်ႇၸႂ် 0 ၶႃႈ။
* **Semicolon `;`**: ယႃႇသႂ်ႇ `;` တီႈလင် `if (condition)` ၶႃႈ။ သင်သႂ်ႇၼႆ `if` တေဢမ်ႇၵုမ်းၵမ် Code ၼႂ်း `{ }` ၼၼ်ႉၶႃႈယဝ်ႉ။
