## Lesson 9: Modifiers: Signed, Unsigned, Short & Long

### 1. Modifiers ၼႆႉ ပဵၼ်သင်?
Modifiers ၼႆႉ ပဵၼ်ၶေႃႈၵႂမ်း ဢၼ်ႁဝ်းသႂ်ႇဝႆႉ တီႈၼႃႈ Data Type (မိူၼ်ၼင်ႇ `int`) တွၼ်ႈတႃႇလႅၵ်ႈလၢႆႈ:
1.  **Range:** တူဝ်ၼပ်ႉယႂ်ႇၵေႃႉလႂ် ဢၼ်မၼ်းၵဵပ်းလႆႈ?
2.  **Memory:** မၼ်းတေၸႂ်ႉ Bytes ၼမ်ၵေႃႉလႂ် ၼႂ်း RAM?

### 2. Signed vs Unsigned (မၢႆ + လႄႈ -)
ၼႂ်း C၊ တူဝ်ၼပ်ႉတဵမ် (Integers) တေမီးသွင်မဵဝ်းၶႃႈ:
* **`signed` (Default):** ၵဵပ်းလႆႈတင်းတူဝ်ၼပ်ႉ "လပ်း" (-) လႄႈ "တဵမ်" (+). တူဝ်ယၢင်ႇ: `-10`, `0`, `50`။
* **`unsigned`:** ၵဵပ်းလႆႈတူဝ်ၼပ်ႉ "တဵမ်" (+) ၵူၺ်း။ ၵူၺ်းၵႃႈ မၼ်းၵဵပ်းလႆႈ တူဝ်ၼပ်ႉယႂ်ႇလိူဝ် `signed` ထိုင် **2 ပၼ်ႇ** (Double the positive range) ၶႃႈဢေႃႈ။



**တူဝ်ယၢင်ႇ:**
```c
unsigned int age = 25; // age တေဢမ်ႇမီးဝၼ်းပဵၼ် -25 ၶႃႈ
```

### 3. Short vs Long (သႅၼ်း Memory)
ႁဝ်းလိူၵ်ႈလႆႈဝႃႈ တေၸႂ်ႉ Memory ဢေႇ ဢမ်ႇၼၼ် ၼမ် ၸွမ်းၼင်ႇတူဝ်ၼပ်ႉႁဝ်းၶႃႈ:
* **`short`**: ၸႂ်ႉ Memory ဢေႇ (2 Bytes)။ ၸႂ်ႉတႃႇတူဝ်ၼပ်ႉဢွၼ်ႇ (တေႃႇပေႃးထိုင် 32,767)။
* **`long`**: ၸႂ်ႉ Memory ၼမ် (4 ဢမ်ႇၼၼ် 8 Bytes)။ ၸႂ်ႉတႃႇတူဝ်ၼပ်ႉယႂ်ႇ (တေႃႇပေႃးထိုင် 2 Billion+)။
* **`long long`**: (8 Bytes) ၸႂ်ႉတႃႇတူဝ်ၼပ်ႉ ဢၼ်ယႂ်ႇတေႉတေႉၶႃႈ။



### 4. ၵၢၼ်ႁူမ်ႈၸႂ်ႉ (Combining Modifiers)
ႁဝ်းဢဝ်ၶဝ်မႃးၸပ်းၵၼ်သေ တႅမ်ႈလႆႈၼင်ႇၼႆၶႃႈ:
* `unsigned short int`
* `signed long long int`

### 5. တူၺ်းသႅၼ်း Memory တေႉတေႉ (sizeof)
တွၼ်ႈတႃႇတေႁူႉဝႃႈ ၼႂ်းၶွမ်ႇပိဝ်ႇတႃႇၸဝ်ႈၵဝ်ႇၼၼ်ႉ `long` မၼ်းၸႂ်ႉလၢႆ Bytes ၼၼ်ႉ လူဝ်ႇၸႂ်ႉ Function **`sizeof()`** ၶႃႈ။

```c
#include <stdio.h>

int main() {
    printf("short int: %zu bytes\n", sizeof(short int));
    printf("long int: %zu bytes\n", sizeof(long int));
    printf("unsigned int: %zu bytes\n", sizeof(unsigned int));
    return 0;
}
```

## 💡 ၵွပ်ႈသင်ႁဝ်းလူဝ်ႇၸႂ်ႉ? (Why use them?)
သင်ၸဝ်ႈၵဝ်ႇတႅမ်ႈ App တွၼ်ႈတႃႇ "ၼပ်ႉဢႃႇယုၵူၼ်း" (Human Age)၊ ဢႃႇယုၵူၼ်းၼႆႉ မၼ်းဢမ်ႇပူၼ်ႉ 150 ၶႃႈ။
* သင်ၸႂ်ႉ `int` (4 Bytes) = ၸႂ်ႉ RAM ၼမ်ၵႂႃႇလၢႆၶႃႈ။
* သင်ၸႂ်ႉ `unsigned char` (1 Byte) = မၼ်းၵဵပ်းလႆႈ 0-255၊ သၢင်ႇထုၵ်ႇသေပိူၼ်ႈ လႄႈ သႅတ်ႉ RAM လႆႈထိုင် 4 ပၼ်ႇၶႃႈဢေႃႈ။
