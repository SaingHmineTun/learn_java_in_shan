## Lesson 20: The Ternary Operator: Short-hand Logic

### 1. Ternary Operator ၼႆႉ ပဵၼ်သင်?
**Ternary Operator** ၼႆႉ ပဵၼ်တူဝ်မၢႆဢၼ်မီး 3 တွၼ်ႈ (Operand) ၶႃႈ။ မၼ်းၸႂ်ႉတႃႇတႅပ်းတတ်းသိၼ်ၸႂ် တီႈၼႂ်းထႅဝ်လဵဝ်ၵၼ် (Inline)။ ႁဝ်းၸၢင်ႈႁွင်ႉမၼ်းဝႃႈ **Conditional Operator** ၵေႃႈလႆႈၶႃႈ။

**တူဝ်မၢႆ:** `? :`

### 2. ပိူင်တႅမ်ႈ (Syntax)
ပိူင်တႅမ်ႈမၼ်းမီးၼင်ႇၼႆၶႃႈ:
`condition ? expression_if_true : expression_if_false;`



### 3. တႅၵ်ႈၼိူင်းတင်း `if-else`
ႁႂ်ႈတူၺ်းလွင်ႈပႅၵ်ႇပိူင်ႈၵၼ် ၼႂ်းၵႄႈၵၢၼ်တႅမ်ႈလၢႆးၵဝ်ႇ လႄႈ လၢႆးပွတ်းၶႃႈ:

**လၢႆးၸႂ်ႉ `if-else` (ယၢဝ်း):**
```c
int age = 20;
char* status;

if (age >= 18) {
    status = "Adult";
} else {
    status = "Minor";
}
```

**လၢႆးၸႂ်ႉ `Ternary` (ပွတ်း):**
```c
int age = 20;
char* status = (age >= 18) ? "Adult" : "Minor";
```



### 4. ၵၢၼ်ၸႂ်ႉတူပ်ႉၵၼ် (Nested Ternary)
ႁဝ်းၸၢင်ႈဢဝ် Ternary မႃးၸပ်းၵၼ်လၢႆလၢႆၸၼ်ႉလႆႈယူႇ၊ ၵူၺ်းၵႃႈ လူဝ်ႇသတိဝႃႈ မၼ်းၸၢင်ႈႁဵတ်းႁႂ်ႈ Code လူယၢပ်ႇၵႂႃႇၶႃႈ။
```c
int marks = 85;
char grade = (marks >= 80) ? 'A' : (marks >= 60) ? 'B' : 'C';
```

### 5. ၵွပ်ႈသင်ႁဝ်းလူဝ်ႇၸႂ်ႉ?
1. **Conciseness:** ႁဵတ်းႁႂ်ႈ Code ပွတ်းလူင်း လႄႈ သႅၼ်ႈသႂ် (Clean)။
2. **Assignment:** ၸွႆႈႁႂ်ႈပၼ်ၵၢၼ် Variable လႆႈငၢႆႈငၢႆႈ ၼႂ်းထႅဝ်လဵဝ်ၵၼ်။
3. **Readability:** တွၼ်ႈတႃႇ Logic ဢွၼ်ႇဢွၼ်ႇ (မိူၼ်ၼင်ႇ ၵၢၼ်လိူၵ်ႈသီ ဢမ်ႇၼၼ် ၵၢၼ်ၼပ်ႉတူဝ်ၼပ်ႉယႂ်ႇသေပိူၼ်ႈ) မၼ်းတူၺ်းငၢႆႈလိူဝ်ၶႃႈ။
