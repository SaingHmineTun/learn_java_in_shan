## Lesson 13: Advanced Formatting: Professional CLI Output

### 1. ၵၢၼ်ၵုမ်းၵမ် တူဝ်ၼပ်ႉမီးၸုၵ်း (Width & Precision)
မိူဝ်ႈႁဝ်းၼႄတူဝ်ၼပ်ႉ `float` ဢမ်ႇၼၼ် `double` ၼၼ်ႉ၊ မၢင်ပွၵ်ႈမၼ်းတေၼႄတူဝ်ႁွႆႈမႃးၼမ်ၼမ် (မိူၼ်ၼင်ႇ `3.141593`)။ ႁဝ်းၵုမ်းၵမ်မၼ်းလႆႈၼင်ႇၼႆၶႃႈ -

* **`%.2f`**: ၼႄတူဝ်ႁွႆႈ 2 တူဝ်ၵူၺ်း (မိူၼ်ၼင်ႇ `3.14`)။
* **`%10d`**: ၼပ်ႉဢဝ်ဢွင်ႈဝၢင်ႇ 10 တီႈ တွၼ်ႈတႃႇၼႄတူဝ်ၼပ်ႉ (ၸွႆႈႁႂ်ႈတူဝ်ၼပ်ႉႁဝ်း "တင်ႈထႅဝ်" ၵၼ်လီလီ)။



### 2. ၵၢၼ်ၸတ်းထႅဝ် (Alignment)
မိူဝ်ႈၸဝ်ႈၵဝ်ႇတႅမ်ႈ "ပပ်ႉသဵၼ်ႈမၢႆ" (Table) ၼႂ်း Terminal၊ ႁဝ်းလူဝ်ႇႁႂ်ႈလိၵ်ႈမၼ်း တင်ႈထႅဝ်ၵၼ်သိုဝ်ႈသိုဝ်ႈၶႃႈ။
* **`%10s`**: ၼႄလိၵ်ႈ ႁႂ်ႈမၼ်းယူႇ "ၽၢႆႇၶႂႃ" (Right Aligned) ၼႂ်းဢွင်ႈဝၢင်ႇ 10 တီႈ။
* **`%-10s`**: ၼႄလိၵ်ႈ ႁႂ်ႈမၼ်းယူႇ "ၽၢႆႇသၢႆႉ" (Left Aligned) ၼႂ်းဢွင်ႈဝၢင်ႇ 10 တီႈ။

```c
printf("%-10s %-5s\n", "Name", "Age");
printf("%-10s %-5d\n", "Sai Mao", 25);
```

### 3. Escape Sequences တွၼ်ႈတႃႇၸတ်းပိူင်
ႁဝ်းၸႂ်ႉတူဝ်မၢႆၶိုၵ်ႉတွၼ်း ၸိူဝ်းၼႆႉ တႃႇၸတ်းပိူင်လိၵ်ႈၶႃႈ -
* **`\t` (Tab)**: ယဵပ်ႇလိၵ်ႈၵႂႃႇၼႃႈ ႁႂ်ႈမၼ်းပဵၼ်ထႅဝ်သိုဝ်ႈၵၼ်။
* **`\v` (Vertical Tab)**: ယဵပ်ႇလိၵ်ႈလူင်းတႂ်ႈ ပဵၼ်ၸၼ်ႉ။
* **`\a` (Alert)**: ႁႂ်ႈၶွမ်ႇပိဝ်ႇတႃႇ "ပွင်ႇသဵင်" (Beep) ၼိုင်ႈၵမ်း။

### 4. ၵၢၼ်သႂ်ႇ "သီ" (Adding Colors with ANSI Codes)
ၼႂ်း Terminal (CLI) ႁဝ်းသႂ်ႇ "သီလိၵ်ႈ" လႆႈပႃးၶႃႈဢေႃႈ။ ၼႆႉတေၸွႆႈႁႂ်ႈ Error Message ၸဝ်ႈၵဝ်ႇ ပဵၼ် "သီလႅင်" ႁိုဝ် "သီၶဵဝ်" ၶႃႈ။

| ANSI Code | သီ (Color) |
| :--- | :--- |
| **`\033[0;31m`** | သီလႅင် (Red) |
| **`\033[0;32m`** | သီၶဵဝ် (Green) |
| **`\033[0;33m`** | သီလိူင် (Yellow) |
| **`\033[0m`** | ၶိုၼ်းပဵၼ်သီၵဝ်ႇ (Reset) |

**တူဝ်ယၢင်ႇ Code:**
```c
printf("\033[0;32mSuccess!\033[0m Operation complete.\n");
```



### 5. ၵၢၼ်ၼႄလိၵ်ႈတႆး ႁႂ်ႈသိုဝ်ႈထႅဝ်ၵၼ်
ၵွပ်ႈဝႃႈလိၵ်ႈတႆးၼိုင်ႈတူဝ် မီး 3 Bytes၊ မၢင်ပွၵ်ႈၵၢၼ်ၸႂ်ႉ `%10s` ၸၢမ်းတေမီးလွင်ႈၽိတ်းပိူင်ႈဢိတ်းၼိုင်ႈ ၵွပ်ႈ `printf` မၼ်းၼပ်ႉဢၢၼ်ႇ "Bytes" တႅၼ်း "တူဝ်လိၵ်ႈ" ၶႃႈ။ တွၼ်ႈတႃႇၵႄႈလႅတ်ႈၼၼ်ႉ ႁဝ်းလူဝ်ႇၸႂ်ႉ **`\t` (Tab)** တေလီသေပိူၼ်ႈ တွၼ်ႈတႃႇလိၵ်ႈတႆးၶႃႈ။
