## Lesson 2: Setting Up Your Forge: Compilers & IDEs

### 1. ၶိူင်ႈမိုဝ်းသွင်ယိူင်ႈ ဢၼ်လူဝ်ႇမီး (Two Essential Tools)
တွၼ်ႈတႃႇတေတႅမ်ႈလိၵ်ႈ C ၼၼ်ႉ ႁဝ်းလူဝ်ႇမီးၶိူင်ႈမိုဝ်းပိူင်လူင် သွင်ယိူင်ႈ ဢၼ်ႁဵတ်းၵၢၼ်ပႅၵ်ႇပိူင်ႈၵၼ်ၶႃႈ။
* **Text Editor (ၵေႃႉတႅမ်ႈလိၵ်ႈ):** တီႈဢၼ်ႁဝ်းတေတႅမ်ႈ Code ၶွင်ႁဝ်းသႂ်ႇ (မိူၼ်ၼင်ႇ VS Code)။ မၼ်းမိူၼ်ၼင်ႇ ပပ်ႉလိၵ်ႈ ဢၼ်ႁဝ်းတႅမ်ႈမၢႆဝႆႉၼၼ်ႉၶႃႈ။
* **Compiler (ၵေႃႉပိၼ်ႇၽႃႇသႃႇ):** ၼႆႉပဵၼ်ၶိူင်ႈမိုဝ်း ဢၼ်လမ်ႇလွင်ႈသုတ်း။ မၼ်းတေပိၼ်ႇ Code ဢၼ်ႁဝ်းတႅမ်ႈၼၼ်ႉ ႁႂ်ႈပဵၼ် File ဢၼ်ၶွမ်ႇပိဝ်ႇတႃႇ ႁဵတ်းသၢင်ႈၵၢၼ်လႆႈ (Executable File)။

### 2. Compiler ၼႆႉ ပဵၼ်သင်? (What is a Compiler?)
ၶွမ်ႇပိဝ်ႇတႃႇၼၼ်ႉ မၼ်းဢမ်ႇပွင်ႇၸႂ် လိၵ်ႈဢၼ်ႁဝ်းတႅမ်ႈမိူၼ်ၼင်ႇ `printf("Hello")` ၼၼ်ႉၶႃႈ။ မၼ်းပွင်ႇၸႂ်ၵူၺ်း လိၵ်ႈၸၢၵ်ႈ (Machine Code) ဢၼ်ပဵၼ် `0` လႄႈ `1` ၼၼ်ႉၵူၺ်း။
**Compiler** ၼႆႉ ႁဵတ်းၵၢၼ်မိူၼ်ၼင်ႇ "ၵူၼ်းပိၼ်ႇလိၵ်ႈ" (Translator) ဢၼ်တေပိၼ်ႇ C Code ႁဝ်း ႁႂ်ႈပဵၼ် `0` လႄႈ `1` တွၼ်ႈတႃႇ CPU တေႁဵတ်းသၢင်ႈၵၢၼ်လႆႈၼၼ်ႉၶႃႈဢေႃႈ။



### 3. ပၵ်းသဝ်း Compiler (Installing GCC)
တွၼ်ႈတႃႇလုၵ်ႈႁဵၼ်း Windows ၼၼ်ႉ ႁဝ်းတေၸႂ်ႉ **MinGW-w64** ဢၼ်မီး Compiler ၸိုဝ်ႈဝႃႈ **GCC** (GNU Compiler Collection) ၼၼ်ႉၶႃႈ။ ၼႆႉပဵၼ် Compiler ဢၼ်ၸိုဝ်ႈယႂ်ႇ လႄႈ ၸႂ်ႉလႆႈလၢႆးၵူႈတီႈၶႃႈ။
* **Windows:** တေလႆႈပၵ်းသဝ်း MinGW သေ သႂ်ႇဝႆႉၼႂ်း System Path။
* **Linux:** ၸႂ်ႉၶေႃႈၵႂၢမ်း `sudo apt install build-essential`။
* **macOS:** ၸႂ်ႉ `xcode-select --install`။

### 4. တီႈတႅမ်ႈ Code (Choosing an IDE/Editor)
ႁဝ်းတေၸႂ်ႉ **Visual Studio Code (VS Code)** ပဵၼ်ပိူင်လူင် ၵွပ်ႈဝႃႈမၼ်းငၢႆႈ၊ ၽႂ်း လႄႈ မီးၶိူင်ႈၸွႆႈ (Extensions) ၼမ်ၶႃႈဢေႃႈ။
* **IDE (Integrated Development Environment):** ပဵၼ်ၶိူင်ႈမိုဝ်း ဢၼ်ႁူမ်ႈဢဝ် Editor လႄႈ Compiler ဝႆႉၸွမ်းၵၼ် (မိူၼ်ၼင်ႇ Code::Blocks ဢမ်ႇၼၼ် CLion)။
* **Editor + Compiler:** ပဵၼ်လၢႆးဢၼ်ႁဝ်းၸႂ်ႉ VS Code + GCC ၼႆႉၶႃႈ။ မၼ်းတေသွၼ်ပၼ်ႁဝ်း ႁႂ်ႈပွင်ႇၸႂ် ၶၵ်ႉတွၼ်ႈၵၢၼ်တႅမ်ႈ Code လႆႈလီလိူဝ်ၶႃႈ။



### 5. ၸၢမ်းတူၺ်း ၶိူင်ႈမိုဝ်းႁဝ်း (Verification)
မိူဝ်ႈၸဝ်ႈၵဝ်ႇ ပၵ်းသဝ်းယဝ်ႉယဝ်ႉၼၼ်ႉ လူဝ်ႇၸၢမ်းတူၺ်းၼႂ်း **Terminal/Command Prompt** ဝႃႈမၼ်းႁဵတ်းၵၢၼ်ႁိုဝ်?
* တႅမ်ႈၶေႃႈၵႂၢမ်း: `gcc --version`
* သင်မၼ်းၼႄမႃးဝႃႈ `gcc (MinGW-W64) 13.x.x` ၼႆ ပွင်ႇဝႃႈ "Forge" ၸဝ်ႈၵဝ်ႇ ႁၢင်ႈႁႄယဝ်ႉၶႃႈ!
