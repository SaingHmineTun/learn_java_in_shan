## Lesson 59: Opening & Closing Files: fopen & fclose

### 1. `fopen()` (ၵၢၼ်ပိုတ်ႇ File)
တွၼ်ႈတႃႇပိုတ်ႇ File၊ ႁဝ်းၸႂ်ႉ Function **`fopen()`** ၶႃႈ။ မၼ်းလူဝ်ႇ Parameters 2 ဢၼ်:
1.  **File Name:** ၸိုဝ်ႈ File ဢၼ်ႁဝ်းၶႂ်ႈပိုတ်ႇ (မိူၼ်ၼင်ႇ `"students.txt"`)။
2.  **Mode:** လၢႆးဢၼ်ႁဝ်းၶႂ်ႈၸႂ်ႉ (လူ၊ တႅမ်ႈ ႁိုဝ် ထႅမ်)။

**Syntax:**
```c
FILE *fp;
fp = fopen("filename.txt", "mode");
```

### 2. File Modes (လၢႆးပိုတ်ႇ File)
ၼႆႉလမ်ႇလွင်ႈသုတ်းၶႃႈ၊ ၵွပ်ႈဝႃႈမၼ်းတေတႅပ်းတတ်းဝႃႈ ၶေႃႈမုၼ်းၵဝ်ႇတေႁၢႆ ႁိုဝ် တေၵိုတ်းဝႆႉၶႃႈ။

| Mode | Meaning | လွင်ႈႁဵတ်းသၢင်ႈ |
| :--- | :--- | :--- |
| **"r"** | **Read** | ပိုတ်ႇတႃႇ "လူ" ၵူၺ်း။ (သင်ဢမ်ႇမီး File ၼႆ မၼ်းတေပဵၼ် `NULL`) |
| **"w"** | **Write** | ပိုတ်ႇတႃႇ "တႅမ်ႈ"။ (**သင်မီး File ဝႆႉယူႇ ၶေႃႈမုၼ်းၵဝ်ႇတေႁၢႆသဵင်ႈ**) |
| **"a"** | **Append** | ပိုတ်ႇတႃႇ "ထႅမ်"။ (တေတႅမ်ႈသိုပ်ႇၽၢႆႇလင် ၶေႃႈမုၼ်းၵဝ်ႇတေဢမ်ႇႁၢႆ) |
| **"r+"** | Read/Write | ပိုတ်ႇတႃႇလူၵေႃႈလႆႈ တႅမ်ႈၵေႃႈလႆႈ။ |



### 3. NULL Check (ၵူတ်ႇထတ်းလွင်ႈလွတ်ႈၽေး)
သင်ၸဝ်ႈၵဝ်ႇသင်ႇပိုတ်ႇ File ဢၼ်ဢမ်ႇမီးဝႆႉ (ၼႂ်း Mode "r")၊ `fopen()` တေ `return` ပၼ် **`NULL`** ၶႃႈ။ ႁဝ်းလူဝ်ႇၵူတ်ႇထတ်းတႃႇသေႇ တႃႇဝႄႉပူဝ်ႇၵရႅမ်ႇ Crash ၶႃႈ။

```c
FILE *fp = fopen("data.txt", "r");
if (fp == NULL) {
    printf("Error: Could not open file!\n");
    return 1; // ဢွၵ်ႇပူဝ်ႇၵရႅမ်ႇ
}
```

### 4. `fclose()` (ၵၢၼ်ပိတ်း File)
မိူဝ်ႈၸႂ်ႉ File ယဝ်ႉတူဝ်ႈ၊ ၸဝ်ႈၵဝ်ႇတေလႆႈ **ပိတ်း (Close)** မၼ်းတႃႇသေႇၶႃႈ။
* တွၼ်ႈတႃႇ "ၶိုၼ်းပၼ်" (Release) Memory ၸူး OS။
* တွၼ်ႈတႃႇႁႂ်ႈၶေႃႈမုၼ်းၼႂ်း **Buffer** ထုၵ်ႇသူင်ႇၵႂႃႇတႅမ်ႈၼႂ်း Disk တႅတ်ႈတေႃး (Save)။

```c
fclose(fp); // ပိတ်းၵူင်ႇ File
```
