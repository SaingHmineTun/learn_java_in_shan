
## **Java Basic 9**

#### **RELATIONAL OPERATORS**

Link - <https://youtu.be/EMDes1PFveg>

Operator ၼႆႉသမ်ႉ ႁဝ်းၸႂ်ႉတိုဝ်း မိူဝ်ႈႁဝ်းတေၶႆႈ တႅၵ်ႈလိူင်း ၵႃႊၶၼ်ၵၼ်ယဝ်ႉ။ တီႈ Java ၼႆႉတေႉ မၼ်းတေမီးဝႆႉ Relational Operator 6 ဢၼ်ဢိူဝ်ႈ။

- _\== ( Equal )_
- _!= ( Not Equal )_
- _\> ( Greater than )_
- _\>= ( Greater than and Equal )_
- _< ( Less than )_
- _<= ( Less than and Equal )_

Relational operator ၼႆႉ မၼ်းတေႁဵတ်းၵၢၼ်ၸွမ်း int ဢမ်ႇၼၼ် float သေတႃႉ မၼ်းတေ ပၼ်ႁဝ်းၶိုၼ်း ၵႃႊၶၼ် boolean ၵူၺ်း။

###### လၢႆးၸႂ်ႉတိုဝ်း

```
int x = 5;
int y = 6;

// == (Equal)
boolean equal = x == y;

// != (Not Equal)
boolean notEqual = x != y;

// < (Less Than)
boolean lessThan = x < y;

// > (Greater Than)
boolean greaterThan = x > y;

// <= (Less than and equal)
boolean lessThanAndEqual = x <= y;

// >= (Greater than and equal)
boolean greaterThanAndEqual = x >= y;
```
