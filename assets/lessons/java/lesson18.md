
## **Java Basic 18**

#### **IF CONDITIONAL STATEMENT**

Link - <https://youtu.be/skVI_6mmIwQ>

If statement ၼႆႉတေႉ ၼႂ်းတူင်ႇမူၼ်းမၼ်းၼၼ်ႉ ႁဝ်းသႂ်ႇပၼ်ၵႃႊၶၼ် boolean ယဝ်ႉ။ ပေႃးပဵၼ် true ၶူတ်ႉဢၼ်တႅမ်ႈဝႆႉတီႈၼႂ်း if block တေႁဵတ်းၵၢၼ်၊ ပေႃးပဵၼ် false ၼႆသမ်ႉ သင်ဢမ်ႇၸႂ်ႉၸွမ်ႈ else ၼႆ မၼ်းတေဢမ်ႇႁဵတ်းၵၢၼ်သင်၊ သင်ၸႂ်ႉၸွမ်း else ၼႆသမ်ႉ ၶူတ်ႉဢၼ်မီးဝႆႉ ၼႂ်း else ၼၼ်ႉ တေႁဵတ်းၵၢၼ်ယဝ်ႉ။

###### လၢႆးၸႂ်ႉတိုဝ်း

```
int a = 5;
int b = 8;
int c = 11;

if (a == b || b > c) {
    System.out.println("IF");
} else {
    System.out.println("ELSE");
}
```
