
## **Java Basic 3**

#### **VARIABLE & DATA TYPE**

Link - <https://youtu.be/\_osMwaaivws>

Variable ၼႆႉ ပဵၼ်ၵွၵ်းဢၼ်ၼိုင်ႈ တႃႇႁဝ်းတေသိမ်းဝႆႉ ၵႃႊၶၼ်ဢိူဝ်ႈ။ ၼႂ်းၵွၵ်းၼႆႉ ႁဝ်းသႂ်ႇလႆႈ ၵေႃႇၽီႇ၊ ၼမ်ႉလဵင်ႈ၊ ၼမ်ႉၼူမ်း ၸိူဝ်းၼၼ်ႉသေ ၼႂ်း variable ၼႆႉသမ်ႉ ႁဝ်းတေသႂ်ႇဝႆႉလႆႈ Data type ဢိူဝ်ႈ။ တီႈ Java ၼႆႉမီးဝႆႉ Data type (2) လွင်ႈယဝ်ႉ။

- **Primitive Data Type**
- **Non-primitive or Reference Data Type**

**1) Primitive Data Type**

တီႈ **Primitive** ၼႆႉတေႉ တေမီးယူႇ 8 လွင်ႈၼႆႉၵူၺ်းလႄႈ လိူဝ်သေ ၸိူဝ်းၼၼ်ႉ ပဵꧣ် Non-primitive မူတ်းယဝ်ႉ။ Primitive data type 8 လွင်ႈၼႆႉပဵၼ်

- byte
- short
- int
- long
- float
- double
- char
- boolean

ၼႆယဝ်ႉ။ တီႈၼႆႉ byte, short, int, long ၼႆႉတေႉ ပဵၼ်တႃႇ ႁဝ်းတေသိမ်းဝႆႉ ၵႃႊၶၼ်တူဝ်ၼပ်ႉ ဢၼ်ဢမ်ႇပႃး decimal point လႄႈ float လႄႈ double သမ်ႉ ပဵၼ်တႃႇ ႁဝ်းသိမ်း ၵႃႊၶၼ်တူဝ်ၼပ်ႉ ဢၼ်ပႃး decimal point ယဝ်ႉ။

လၢႆးၸႂ်ႉတိုဝ်း

```
var hello = "Hello World";
var version = 20;
version = 25;
```
