
## **Java Basic 4**

#### **INTEGER DATA TYPES (BYTE, SHORT, INT, LONG)**

Link - <https://youtu.be/ZzF708Le5xk>

byte, short, int, long ၸိူဝ်းၼႆႉတၢင်းသဵင်ႈ ပဵၼ် Integer data type ဝႆႉသေတႃႉ တၢင်းယႂ်ႇ တႃႇၶဝ်တေ သိမ်းဝႆႉပၼ်လႆႈၼၼ်ႉ ဢမ်ႇမိူၼ်ၵၼ်။

- **Byte** ၼႆႉ မၼ်းတေသိမ်းပၼ်လႆႈ 1 byte ၵူၺ်း။
- **Short** သမ်ႉ 2 bytes
- **Int** သမ်ႉ 4 bytes လႄႈ
- **Long** သမ်ႉ 8 bytes ယဝ်ႉ။

1 byte ၼႆႉပဵၼ် 8 bits ။ 1 bit ၼႆႉ တေသိမ်းလႆႈ 0 လႄႈ 1 ၵူၺ်း။ မၼ်းပဵၼ် binary (2 based) ဢိူဝ်ႈ။ 1 byte ၼႆႉပေႃးၼပ်ႉ တူၺ်းၸွမ်း ပိူင် decimal (10 based) ၼႆၸိုင် မၼ်းတေသိမ်းဝႆႉ ပၼ်လႆႈထိုင် 255 ယဝ်ႉ။ ၵူၺ်းၵႃႊ integer တၢင်းမူတ်း ၶဝ်တေသိမ်းပၼ်ၸွမ်း Negative value လႄႈ ၸင်ႇပဵၼ်ၵႂႃႇ -128 တေႃႇထိုင် 127 ယဝ်ႉ။

လၢႆးၸႂ်ႉတိုဝ်း

```
byte byteVar = 127; // byte သိမ်းလႆႈ 1 byte
short shortVar = 32_767; // short သိမ်းလႆႈ 2 bytes
int intVar = 2_147_483_647; // int သိမ်းလႆႈ 4 bytes
long longVar = 2_147_483_648L; // long သိမ်းလႆႈ 8 bytes
```
မိူဝ်ႈတႅမ်ႈ long ၼႆႉ ပေႃးပူၼ်ႉပႅင် int သိမ်းလႆႈၼႆၸိုင် တေလႆႈသႂ်ႇပဵၼ် တူဝ်လိၵ်ႈ L ဢမ်ၼၼ် l မၼ်းၸင်ႇမၢၼ်ႇ။ ပေႃးဢမ်ႇၼၼ် တေၼႄဝႆႉၽိတ်းၼႆယဝ်ႉ။
