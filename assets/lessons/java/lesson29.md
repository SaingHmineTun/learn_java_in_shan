
## **Java Basic 29**

#### **STRING REPLACE METHOD**

Link - <https://youtu.be/GsQAO-JXe7s>

ပဵၼ် method ဢၼ်ၶႃႈ လႆႈၸႂ်ႉတိုဝ်းၼမ်သေပိူၼ်ႈယဝ်ႉ။ မၢင်ၸိူဝ်းၵေႃႈ တေႁူႉၸၵ်းၶႃႈ ၵွပ်ႈပိူဝ်ႈၶႃႈ ပဵၼ်ၵူၼ်းႁဵတ်း လိၵ်ႈတႆးမၢဝ်း &lt;=&gt; လိၵ်ႈတူဝ်မူၼ်း Converter ယူႇ။ လွင်ႈၼႆႉ ၶႃႈမႃးၸႂ်ႉသင်ၼႆ ၶႃႈမႃးၸႂ်ႉ replace ဝႆႉၵူၺ်း။ ၵူၺ်းၵႃႈ မၼ်းလႆႈၸႂ်ႉၸွမ်း Regular Expression (Regex) လႄႈ လွင်ႈၼႆႉတေႉ တီႈၵၢၼ်သွၼ်ၼႆႉ ၶႃႈတေပႆႇလၢတ်ႈ။

လၢႆးႁဵတ်းၵၢၼ် မၼ်းၼႆႉၵေႃႈ ၼႂ်း method ၼႆႉ မၼ်းတေမီးဝႆႉယူႇ parameter သွင်ဢၼ်။ ဢၼ်ၼိုင်ႈသမ်ႉ တေလႆႈပဵၼ် ဢၼ်မီးဝႆႉ တီႈၼႂ်း String လႄႈ ထႅင်ႈဢၼ်သမ်ႉ ပဵၼ်တူဝ်တႅၼ်းမၼ်ယဝ်ႉ။ တီႈ Code ႁဝ်းၼႆႉ လႆႈၼႄဝႆႉ လွင်ႈဢၼ်ဢဝ် ၵၶင ၼၼ်ႉပိၼ်ႇပဵၼ် လိၵ်ႈတူဝ်ယၢဝ်း ᥐᥑᥒ ဝႆႉယဝ်ႉ။

###### လၢႆးၸႂ်ႉတိုဝ်း  

```
String consonants = "ၵၶင";  
String taiLe = consonants.replace('ၵ', 'ᥐ'); // ᥐၶင  
taiLe = taiLe.replace('ၶ', '\\u1951'); // ᥐᥑင  
taiLe = taiLe.replace('င', '\\u1952'); // ᥐᥑᥒ
```
