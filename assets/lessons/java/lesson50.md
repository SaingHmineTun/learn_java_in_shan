
## **Java Basic 50**

#### **MATH CLASS**

Link - <https://youtu.be/X_cLm0oXMqs>

ၼႆႉပဵၼ် class ဢၼ်ၵိုၵ်းပႃးဝႆ့ static members မူတ်းလႄႈ ႁဝ်းႁွင်ႉမၼ်းဝႃႈ static class ၼႆယဝ်ႉ။

ၽိတ်း - class ဢၼ်ပႃးဝႆႉ static members ၼႆႉ ႁဝ်းဢမ်ႇႁွင်ႉ static class ။ တီႈ Java ၼႆႉ မၼ်းမီးဝႆႉ static inner class ႁင်းၶေႃယူႇ။

ၼႆႉပဵၼ် static method ၵမ်ႈၽွင်ႈ ဢၼ်မီးဝႆႉၼႂ်း Math class လႄႈ ပဵၼ်ဢၼ်ၸႂ်ႉတိုဝ်းၵၼ် ၼမ်ဝႆႉယဝ်ႉ။

- _abs_ \- ပိၼ်ႇလၢႆႊၵႃႊၶၼ် ႁႂ်ႈပဵၼ် positive
- _min_ \- ၼႂ်းၵႄႈ ၵႃႊၶၼ် သွင်ဢၼ်ၼၼ်ႉ ၼႄပၼ်ၶိုၼ်း ဢၼ်ၵႃႊၶၼ်ဢေႇသေပိူၼ်ႈ
- _max_ \- ၼႂ်းၵႄႈ ၵႃႊၶၼ် သွင်ဢၼ်ၼၼ်ႉ ၼႄပၼ်ၶိုၼ်း ဢၼ်ၵႃႊၶၼ် ၼမ်သေပိူၼ်ႈ
- _sqrt_ \- Square Root
- _pow_ \- power
- _ceil_ \- ၸႂ်ႉၸွမ်း decimal value သေ ပၼ်ၶိုၼ်းၵႃႊၶၼ် ဢၼ်ယူႇသုင် (ceiling = တိူမ်း)
- _floor_ \- ၸႂ်ႉၸွမ်း decimal value သေ ပၼ်ၶိုၼ်းၵႃႊၶၼ် ဢၼ်ယူႇတႅမ်ႇ (floor = ၽၢၵ်ႈ)
- _round_ \- ၸႂ်ႉၸွမ်း decimal value သေ ပၼ်ၶိုၼ်းၵႃႊၶၼ် ဢၼ်ၸမ်သေပိူၼ်ႈ

###### လၢႆးၸႂ်ႉတိုဝ်း

```
Math.abs(-5); // 5
Math.min(100, 50); // 50
Math.max(100, 50); // 100
Math.sqrt(25); // 5
Math.pow(5, 5); // 25
Math.ceil(1.9); // 1.0
Math.floor(1.1); // 2.0
Math.round(1.4); // 1.0
Math.round(1.6); // 2.0
```
