
## **Java Basic 31**

#### **CREATE A CLASS IN JAVA**

Link - <https://youtu.be/drRSofShJzE>

Java ၼႆႉ ပဵၼ် Pure OOP ( Object-Oriented Programming ) ဢိူဝ်ႈ။ ပွင်ႇဝႃႈ တီႈၼႂ်းမၼ်း ႁဝ်းႁဵတ်း File သင်သေဢမ်ႇဝႃႈ မၼ်းတေလႆႈပဵၼ် class file ယဝ်ႉ။ တႃႇတေႁဵတ်း class file ၼိုင်ႈဢၼ်ၼႆ

- တေလႆႈမီး .java extension
- တေလႆႈဢဝ် class keyword သေ ၶူင်ဝႆႉ။

Class ၼႆႉ ပေႃးတေဝႃႈႁႂ်ႈပွင်ႇၸႂ်ငၢႆၼႆ ပဵၼ် File ဢၼ်ၼိုင်ႈ ဢၼ်ႁဝ်းၸၢင်ႊၶူင်ဢွၵ်ႇလႆႈ Reference Data Type ဢိူဝ်ႈ။ မိူၼ်ၼင်ႇ String class ၼၼ်ႉယဝ်ႉ။ ယဝ်ႉၵေႃႈ သင်ႁဝ်းၶူင်သၢင်ႈ class ၼႆၸိုဝ်ႈ class လႄႈ ၸိုဝ်ႈ file မၼ်းၼၼ်ႉ တေလႆႈမိူၼ်ၵၼ်ဝႆႉ။

**ၼႂ်း Class တေတႅမ်ႈလႆႈသင်**

သင်ၶူင်သၢင်ႈ class ၼိုင်ႈဢၼ်ၼႆၸိုင် ၼႂ်းမၼ်းၼၼ်ႉ ႁဝ်းတေတႅမ်ႈလႆႈ state လႄႈ behavior ယဝ်ႉ။

**State** ၼႆႉပဵၼ် variable ဢၼ်ႁဝ်း ၶူင်ဝႆႉ တီႈၼႂ်း Class ၼႆႉဢိူဝ်ႈ။ မၼ်းတေၸၢင်ႊ မၵ်းမၼ်ႈလႆႈ ၵႃႊၶၼ်လႄႈ ၸၢင်ႊၼႄပၼ် ၵႃႊၶၼ်မၼ်းဢိူဝ်ႈ။

**Behaviors** ၼႆႉသမ်ႉ ပဵၼ် Methods ဢၼ်ႁဝ်းၶူင်ဝႆႉၼႂ်း Class ၼၼ်ႉဢိူဝ်ႈ။ လိူဝ်သေ မၵ်းမၼ်ႈ ၊ ၼႄပၼ် ၵႃႊၶၼ်ယဝ်ႉ မၼ်းတေၸၢင်ႊႁဵတ်းလႆႈ ၵၢၼ်မၢင်ၸိူဝ်ႉမၢင်လွင်ႈ မိူၼ်ၼင်ႇ print ၸိူဝ်းၼၼ်ႉဢိူဝ်ႈ။

###### လၢႆးပၼ်ၸိုဝ်ႈ

ၵွပ်ႈပိူဝ်ႈဢမ်ႇၸႂ်ႈ Primitive လႄႈ ႁဝ်းထုၵ်ႇလီဢဝ် တူဝ်လိၵ်ႈယႂ်ႇသေပၼ် မိူၼ်ၼင်ႇ String, Scanner ဢိူဝ်ႈ။ သင်သမ်ႉ ၸိုဝ်ႈဢၼ်ႁဝ်းၶႆႈပၼ်ၼႆႉ မီးဝႆႉယူႇ 2, 3 တူဝ်ၼႆၸိုင် ႁဝ်းတေလႆႈၸွမ်းပိူင် Camel Case မိူၼ်ၼင်ႇ ArrayList, HashMap ဢိူဝ်ႈ။
