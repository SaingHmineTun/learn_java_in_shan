
## **Java Basic 7**

#### **ASSIGNMENT OPERATORS**

Link - <https://youtu.be/g8YpkS57E2I>

Operator ၼႆႉသမ်ႉ ႁဝ်းၸႂ်ႉတိုဝ်း မိူဝ်ႈၽွင်ႈ ႁဝ်းၶႆႈဢဝ် ၵႃႊၶၼ်သေဢၼ်ဢၼ် သႂ်ႇဝႆႉၼႂ်း variable ၼၼ်ႉယဝ်ႉ။ လၢႆးႁဵတ်းၵၢၼ်မၼ်းၼႆႉၵေႃႈ ဢဝ်ၵႃႊၶၼ် ဢၼ်ယူႇတီႈ ၽၢႆႇၶႂႃႇ operator ၼၼ်ႉသေ ၶိုၼ်းသႂ်ႇဝႆႉ တီႈၼႂ်း variable ဢၼ်ယူႇဝႆႉတီႈ ၽၢႆႇသၢႆႉ operator ၼၼ်ႉယဝ်ႉ။ = ၼႆႉ ႁဝ်းၸႂ်ႉတႃႇ Assignment ယဝ်ႉ။ မၼ်းလႄႈ Arithmetic Operator ႁူမ်ႈၵၼ်သေ ၶိုၼ်းဢွၵ်ႇမႃးထႅင်ႈ 5 ဢၼ်လႄႈ လေႃးၵၼ်တၢင်းမူတ်း ၸင်ႇမီးဝႆႉ 6 ဢၼ်ယဝ်ႉ။

Assignment Operators မီး 6 မဵဝ်းၶႃႈ

- _\= ( Assignment )_
- _+= ( Addition Assignment )_
- _\-= ( Subtraction Assignment )_
- _\*= ( Multiplication Assignment )_
- _/= ( Division Assignment )_
- _%= ( Modulus Assignment )_

လၢႆးၸႂ်ႉတိုဝ်း

```
int x = 5;
int a = 10;
a += 5; // a = a + 5;

int b = 10;
b -= 8; // b = b - 8;

int c = 2;
c *= 2; // c = c * 2;

int d = 9;
d = 3; // d = d / 3;

int e = 5;
e %= 3; // e = e % 3;
```
