## 34) Functions

Function ၼႆႉပဵၼ်တီႈ ဢၼ်ႁဝ်းတႅမ်ႈၶူတ်ႉသေ သင်ဢမ်ႇႁွင်ႉမၼ်းၼႆ မၼ်းတေဢမ်ႇႁဵတ်ႉၵၢၼ်။ တီႈ python ၼႆႉ ႁဝ်းၸႂ်ႉ def keyword တႃႇၶူင်သၢင်ႈ function ယဝ်ႉ။ Function ၼႆႉ မၼ်းတေ ႁပ်ႉၶၢမ်ႇလႆႈ parameter သေ၊ မၼ်းတေ ပၼ်ႁဝ်းၶိုၼ်း return value ယဝ်ႉ။

#### 1. Creating a function

တႃႇတေ ၶူင်သၢင်ႈ function ဢၼ်ၼိုင်ႈၼႆႉ တီႈ python ႁဝ်းၸႂ်ႉ def keyword ယဝ်ႉ။ ဝၢႆးသေတႅမ်ႈ def ယဝ်ႉ၊ ၽၢႆႇလင်မၼ်း ၶိုၼ်းသႂ်ႇပၼ် ၸိုဝ်ႈ function သေ ႁဝ်းၶိုၼ်းဢဝ် တူင်ႇမူၼ်းသေ ဢိူတ်ႉပၼ်ယဝ်ႉ။ လိုၼ်းသုတ်ႉ ႁဝ်းၶိုၼ်းသႂ်ႇပၼ် full column (:) သေၼႂ်းမၼ်းၼၼ်ႉ ႁဝ်းၶိုၼ်းတႅမ်ႈဝႆႉၶူတ်ႉ ဢၼ်ႁဝ်းၶႆႈႁႂ်ႈမၼ်း ႁဵတ်ႉၵၢၼ်ၼၼ်ႉယဝ်ႉ။

```
def my_function():  
    print("Hello from a function")
```

#### 2) Calling a function

တႃႇတေ ႁွင်ႉၸႂ်ႉ function ဢၼ်ႁဝ်းလႆႈ ၶူင်ဝႆႉၼႆႉ ႁဝ်းၶိုၼ်းတႅမ်ႈပၼ် ၸိုဝ်ႈ function မၼ်း ၼင်ႇတီႈတႂ်ႈၼႆႉၵေႃႈ လႆႈယဝ်ႉ။ တီႈတေႉ လွင်ႈဢၼ်ႁဝ်းၸႂ်ႉတိုဝ်း print() ၼႆႉ ပဵၼ်လွင်ႈဢၼ်ႁဝ်း ၸႂ်ႉတိုဝ်း function ယဝ်ႉ။ မၼ်းပဵၼ် function ဢၼ် python ႁဵတ်ႉပၼ်ႁဝ်းဢိူဝ်ႈ။

```
my_function()
```
