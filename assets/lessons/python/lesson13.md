
## 13) Python Casting

Python ၼႆႉ မၼ်းၵေႃႈပဵၼ် OOP လႄႈ data type မၼ်းၵမ်ႈၼမ်ၵေႃႈ လႆႈဢဝ် class သေၶူင်ဝႆႉၵူၺ်းၼင်ႇၵဝ်ႇ။ ၵွပ်ႈၼႆ သင်ႁဝ်း ၶႆႈဢဝ် data type ဢၼ်ၼိုင်ႈ ပိၼ်ႇပဵၼ်ဢၼ်ၼိုင်ႈၼႆ ႁဝ်းၸႂ်ႉတိုဝ်း constructor ဢၼ်မီးဝႆႉ တီႈ data class ၼၼ်ႉယဝ်ႉ။

**int()**

ၼႆႉ ဢဝ် int, float (ဢဝ် decimal မၼ်းပႅတ်ႈ), str (သင်ၼႂ်း str မီးဝႆႉ number) ၸိူဝ်းၼႆႉသေ ပိၼ်ႇႁႂ်ႈပဵၼ် int ယဝ်ႉ။

```
y = int(2.8) # y will be 2
```

**float()**

ၼႆႉ ဢဝ် int, float, str သေ ပိၼ်ႇပဵၼ် float ယဝ်ႉ

```
w = float("4.2") # w will be 4.2
```

**str()**

ၼႆႉၵေႃႈ ဢဝ် int, float, str သေ ပိၼ်ႇပဵၼ် string ယဝ်ႉ။

```
z = str(3.0)  # z will be '3.0'
```
