
## 36) Return values

သင်ႁဝ်း ၶႆးႁႂ်ႈ function ႁဝ်းပၼ်ၶိုၼ်း ၵႃႊၶၼ်သေ ဢၼ်ဢၼ်ၼႆၸိုင် ႁဝ်းၸႂ်ႉ return keyword သေ သူင်ႇပၼ်လႆႈ ၵႃႊၶၼ်မၼ်းယဝ်ႉ။ တွင်းဝႆႉပၼ်ဝႃႈ function ဢၼ်ဢမ်ႇပၼ်ၶိုၼ်း ၵႃႊၶၼ်သင်ၼႆႉ ပၼ်တေၶိုၼ်းပၼ်ႁဝ်း None type ၼႆလႄႈ။ ၶူတ်ႉတီႈတႂ်ႈၼႆႉ ၵွပ်ႈမၼ်းပၼ်ႁဝ်းၶိုၼ်း ၵႃႊၶၼ် return value လႄႈ ၸင်ႇသိမ်းဝႆႉၼႂ်း val variable ယဝ်ႉ။ သင်ဢဝ်တူၺ်းၵႃႊၶၼ်မၼ်းၼႆၸိုင် ႁဝ်းတေလႆႈႁၼ် **15** ယဝ်ႉ။

```
def multiply(a, b=5):  
    return a * b  

val = multiply(3)
print(val) # 15
```
