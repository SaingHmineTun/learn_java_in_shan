## 10) Boolean Type

ၼႂ်း boolean ၼႆႉ ႁဝ်းတေသိမ်းလႆႈ ၵႃႊၶၼ် သွင်ဢၼ်ၵူၺ်း။ ပဵၼ် **True** လႄႈ **False** ယဝ်ႉ။ မၼ်းတူဝ်လဵဝ် ဢမ်ႇၸႂ်ႉတိုဝ်းၵၼ်သေတႃႉ မၼ်းၵေႃႈပဵၼ် Type ဢၼ်လွင်ႈယႂ်ႇဝႆႉဢၼ်ၼိုင်ႈ တႃႇတေသိုပ်ႇလဵပ်ႈႁဵၼ်းလွင်ႈ conditional, looping ၸိူဝ်းၼႆႉယဝ်ႉ။ ယဝ်ႉၵေႃႈ operator မိူၼ်ၼင်ႇ comparison, logical ၸိူဝ်းၼႆႉၵေႃႈ ၶဝ်ထုၵ်ႈဢွၵ်ႇပၼ် ၵႃႊၶၼ် Boolean ယဝ်ႉ။ မိူၼ်ၼင်ႇ ၶူတ်ႉတီႈတႂ်ႈၼႆႉ ၶဝ်ထုၵ်ႈဢွၵ်ႇပဵၼ် Boolean မူတ်းယဝ်ႉ။

```
print(True)  
print(False)  
is_10_greater_than_20 = 10 > 20 
print(is_10_greater_than_20) # False
```
