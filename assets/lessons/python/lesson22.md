
## 22) Assignment operators

ၼႆႉသမ်ႉ ႁဝ်းၸႂ်ႉတိုဝ်း တႃႇတေဢဝ် ၵႃႊၶၼ်သေ သိမ်းဝႆႉၼႂ်း variable ယဝ်ႉ။ = ဢၼ်လဵဝ်ၼႆႉ မၼ်းပဵၼ် assignment ယဝ်ႉ။ ၵူၺ်းၵႃႊ မၼ်းမႃး ႁူမ်ႈၵၼ်တင်း arithmetic ၼႆၵေႃႈ ၸင်ႇပေႃႇဢွၵ်ႇမႃးထႅင်ႈ assignment operators ထႅင်ႈ 8 ဢၼ်ယဝ်ႉ။

| Operator | Example | Same As |
| --- | --- | --- |
| \=  | x = 5 | x = 5 |
| +=  | x += 3 | x = x + 3 |
| \-= | x -= 3 | x = x - 3 |
| \*= | x \*= 3 | x = x \* 3 |
| /=  | x /= 3 | x = x / 3 |
| %=  | x %= 3 | x = x % 3 |
| //= | x //= 3 | x = x // 3 |
| \*\*= | x \*\*= 3 | x = x \*\* 3 |

###### လၢႆးၸႂ်ႉတိုဝ်း 
```
x = 5
x += 3 # x = x + 3 (8)
x -= 2 # x = x - 2 (6)
x *= 2 # x = x * 2 (12)
x /= 2 # x = x / 2 (3)
x %= 2 # x = x % 2 (1)
x //= 2 # x = x // 2 (1)
x **= 2 # x = x ** 2 (4)
```
