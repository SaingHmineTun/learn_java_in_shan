
## 27) Operator Precedence

သင်ၼႂ်း ၶူတ်ႉႁဝ်း ထႅဝ်လဵဝ်ၼႆႉ မီးဝႆႉ operators တၢင်းၼမ်ၼႆၸိုင် ၶဝ်ဢမ်ႇႁဵတ်ႉၵၢၼ်လူၺ်ႈ သၢႆႉတေႃႇၶႂႃ၊ ၶဝ်တေႁဵတ်ႉၵၢၼ်ၸွမ်း operator precedence ယဝ်ႉ။ ပွင်ႇဝႃႈ operator လႂ် precedence မၼ်းမီးသုင်ၵေႃႈ operator ၼၼ်ႉ တေႁဵတ်ႉၵၢၼ် ဢွၼ်တၢင်းသုတ်ႉယဝ်ႉ။

| Operator | Description |
| --- | --- |
| ()  | Parentheses |
| \*\* | Exponentiation |
| +x  -x | Unary plus and unary minus |
| \*  /  //  % | Multiplication, division, floor division, and modulus |
| +  - | Addition and subtraction |
| \==  !=  >  >=  <  <=  is  is not  in  not in | Comparisons, identity, and membership operators |
| not | Logical NOT |
| and | AND |
| or  | OR  |

###### တူဝ်ယၢင်ႇ (example)
```
x = 5 + 5 - 5 * 5
print(x) # -15

x = (5 + 5 - 5) * 5
print(x) # 25
```
