
## User Input

တီႈ python ၼႆႉ သင်ႁဝ်းၶႆႈႁႂ်ႈ user သႂ်ႇပၼ် ၶေႃႈမုၼ်းတီႈ console ၼႆၸိုင် ႁဝ်းတေၸႂ်ႉလႆႈ input() function ယဝ်ႉ။ ၶူတ်ႉတီႈတႂ်ႈၼႆႉ ယွၼ်းဝႆႉ ၸိုဝ်ႈ user သေ သင်သႂ်ႇပၼ်ယဝ်ႉၼႆ မၼ်းတေ ဢွၵ်ႇၼႄပၼ်ႁဝ်းၶိုၼ်းတီႈ console လူၺ်ႈ print() function ယဝ်ႉ။ input() function ၼႆႉမၼ်းတေပၼ်ႁဝ်းၶိုၼ်း string ၵူၺ်းလႄႈ သင်ၶႆႈယွၼ်းဝႆႉ int, float ၸိူဝ်းၼႆႉၼႆၸိုင် ၶိုၼ်းႁဵတ်ႉပဵၼ် type casting သေလႄႈ။

```
username = input("Enter username:")  
print("Username is: " + username)
```

သင်သမ်ႉ ၶႂ်ႈလႆႈ input ပဵၼ် int, double ၼႆၸိုင် ႁဝ်းတေလႆႈၸႂ်ႉတိုဝ်းၸွမ်း casting ယဝ်ႉ

```
first_number = int(input('First number: '))
second_number = double(input('Second number: '))
```
