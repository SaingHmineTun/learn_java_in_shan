## 1) What is Python

Python ၼႆႉပဵၼ် High level programming language ဝႆႉ ဢၼ်ၼိုင်ႈဢိူဝ်ႈ။ ၵူၼ်းတေႇၶူင်ဢွၵ်ႇမၼ်းပဵၼ် Guido van Rossum လႄႈ မၼ်းတေႇဢွၵ်ႇမႃးတီႈ ပီ1991 ၼႆယဝ်ႉ။ ႁဝ်းၸႂ်ႉ python သေ

- ၶူင်သၢင်ႈ ဝႅပ့်သၢႆႉၵေႃႈလႆႈ
- ၶူင်သၢင်ႈ သွပ်ႉဝႄး တႃႇၸႂ်ႉတိုဝ်း ၼႂ်းၽူၼ်း၊ ၼႂ်းၶွမ်း ၵေႃႈလႆႈ
- ၶူင်သၢင်ႈ Artificial Intelligence ႁိုဝ် Machine Learning ၸိူဝ်းၼႆႉၵေႃႈလႆႈ ယဝ်ႉ။

ၵွပ်ႈၼႆလႄႈ မၼ်းပဵၼ်ၽႃႇသႃႇ ဢၼ်ဢွၵ်ႇမႃးၵေႃႈ မီးၵႂႃႇ 30 ပီႊပၢႆသေတႃႉ ထိုင်ယၢမ်းလဵဝ်ၵေႃႈ ၵူၼ်းၸႂ်ႉတိုဝ်းမၼ်း တိုၵ်ႉမီးၼမ်ဝႆႉယူႇ။

Python ၼႆႉမၼ်းတေမီးဝႆႉ version 2 လႄႈ version 3 လႄႈ သွင်ဢၼ်ၼႆႉ ဢမ်ႇမိူၼ်ၵၼ်။ ၼႂ်းပပ်ႉၼႆႉတေႉ ၶႃႈတေၸႂ်ႉတိုဝ်း Python 3 သေသွၼ်ပၼ်ၵႂႃႇ။
