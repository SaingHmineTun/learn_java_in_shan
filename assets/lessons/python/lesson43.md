
## from import module

သင်သမ်ႉ ၼႂ်း module ႁဝ်းမီး function ဢမ်ႇၼၼ် variables တၢင်းၼမ်လႄႈ ႁဝ်းသမ်ႉၵူၺ်း ၶႆးႁွင်ႉၸႂ်ႉတိုဝ်း သေဢၼ်ဢၼ်ၵူၺ်းၼႆၸိုင် ၵၢၼ်ဢၼ်ႁဵတ်ႉ import module တၢင်းမူတ်းၼႆႉ မၼ်းတေႁႂ်ႈ program ႁဝ်းႁဵတ်ႉၵၢၼ် ၼၵ်ႉတိူဝ်းၵူၺ်း။ တီႈၼႆႈ ႁဝ်းႁႂ်ႉ from လႆႈယဝ်ႉ။ ဝႆႉဝႃႈ ၼႂ်း mymodule.py ၶႃႈၼႆႉ မီးဝႆႉ greeting() function လႄႈ person1 variable သေတႃႉ ၶႃႈသမ်ႉ ၶႆႈႁွင်ႉၸႂ်ႉ person1 ၵူၺ်းၼႆလႄႈ ၶႃႈတႅမ်ႈလႆႈ မိူၼ်ၼင်ႇ တီႈတႂ်ႈမၼ်းယဝ်ႉ။ လွင်ႈလီမၼ်း ထႅင်ႈလွင်ႈၼိုင်ႈၵေႃႈ သင်ႁဝ်းၸႂ်ႉၸွမ်း from ၼႆၸိုင် ႁဝ်းဢမ်ႇလူဝ်ႇႁွင်ႉ ၸိုဝ်ႈ module မၼ်းသေ ႁွင်ႉၸႂ်ႉလႆႈ function ဢမ်ႇၼၼ် variables ဢၼ်မၼ်းႁဵတ်ႉ import ဝႆႉၵမ်းလဵဝ်ယဝ်ႉ။

```
from mymodule import person1  
print (person1["age"])
```
