
## 30) While Loop Statement

Looping ၼႆႉပွင်ႇဝႃႈ ႁဝ်းႁဵတ်ႉၵၢၼ်သေဢၼ်ဢၼ်ၼႆႉ ႁဵတ်ႉဢမ်ႇၵိုတ်းဢမ်ႇလိုဝ်ႈ၊ ႁဵတ်ႉတေႃႇပေႃး **မႅၼ်ႈၸႂ်** ၼႆယဝ်ႉ။ တီႈ python ၼႆႉ မၼ်းတေမီးဝႆႉ loop (2) လွင်ႈ။

**1) while loop**

**2) for loop**

ၼႆယဝ်ႉ။

while ၼႆႉႁဝ်းၸႂ်ႉၸွမ်း boolean expression မိူၼ်ၼင်ႇ "if" ၵူၺ်း။ သင် expression မၼ်းပဵၼ် **True** ၼႆ မၼ်းတေသိုပ်ႇႁဵတ်ႉၵၢၼ်ၵႂႃႇတိၵ်ႉတိၵ်ႉယဝ်ႉ။ ၵွပ်ႈၼႆ သင်ႁဝ်း ဢမ်ႇၸႂ်ႉမၼ်းလီလီၼႆ ႁဝ်းၸၢင်ႊၶူင်ဢွၵ်ႇပဵၼ် infinite loop ဢိူဝ်ႈ။ မၼ်းပွင်ႇဝႃႈ loop ႁဝ်းၼႆႉ ဢမ်ႇမီးဝၼ်းတေၵိုတ်း ၼႆယဝ်ႉ။

တႃႇတေၸႂ်ႉ while loop ႁႂ်ႈမီးၽွၼ်းလီ၊ ယႃႇႁႂ်ႈပဵၼ် infinite loop ၼႆၸိုင် ႁဝ်းတေလႆႈၸွမ်း ပိူင် (3) ပိူင်ၼႆႉယဝ်ႉ။

1) Initialize variables

2) Boolean expression

3) Update value of variables

ၶူတ်ႉတႂ်ႈၼႆႉ မၼ်းတေႁဵတ်ႉၵၢၼ်ပၼ် 5 ပွၵ်ႈၵူၺ်း။ ပွၵ်ႈလႄႈပွၵ်ႈ ၵႃႊၶၼ် I ၼႆႉ မၼ်းတေတိူဝ်းမႃးပၼ် 1 တေႃႇထိုင်ၽႅဝ်ႁွတ်ႈ 5 ယဝ်ႉ။ ၵွပ်ႈပိူဝ် 6 မၼ်းဢမ်ႇလဵၵ်ႉသေ 6 ၼႆလႄႈ boolean expression ႁဝ်းၸင်ႇပဵၼ် False ၵႂႃႇယဝ်ႉ။ သင်ၼႆၵေႃႈ loop ႁဝ်းမၼ်းတေၵိုတ်းလွင်ႈ ႁဵတ်ႉၵၢၼ်ၵႂႃႇယဝ်ႉ။

```
x = 1  
while x < 6:  
    print(x)  
    x += 1
```

**The "break" statement**

"break" ၼႆႉ ပဵၼ် keyword ဢၼ်ႁဝ်းၸႂ်ႉတိုဝ်း တႃႇၵိုတ်း looping ၼႆယဝ်ႉ။ ၵမ်ႈၼမ် ႁဝ်းတေၸႂ်ႉတိုဝ်းၸွမ်း if statement ယဝ်ႉ။ ၶူတ့်တီႈတႂ်ႈၼႆႉ ႁဝ်းသင်ႇႁႂ်ႈမၼ်းပၼ်ႇတူင်ႇ 5 ၵမ်း (1 - 5) သေတႃႉ သင် x ႁွတ်ႈထိုင် 3 ၼႆ ၵွပ်ႈလႆး break ဝႆႉလႄႈ ပရူဝ်ႇၵရမ်ႇႁဝ်း မၼ်းတေႁဵတ်ႉၵၢၼ်ၵႂႃႇ (1 - 3) ၵူၺ်းယဝ်ႉ။

```
x = 1  
while x < 6:  
    print(x)  
    if x == 3: break  
    x += 1
```

**The "continue" statement**

ၼႆႉသမ်ႉ မၼ်းတေၵိုတ်းလွင်ႈ ႁဵတ်ႉၵၢၼ်ပၼ် iteration ပွၵ်ႈလဵဝ်ၵူၺ်း။ မၼ်းတေႁမ်ႉ မိူၼ်ၵၼ်တၢင်း skip (ၶၢမ်ႈပူၼ်ႉ) ၼႆႉယူႇ။ ၶူတ်ႉတႂ်ႈၼႆႉ မိူဝ်ႈတေႇ လႆႈပၼ်ႇဝႆႉ 1-5 သေတႃႉ ၵွပ်ႈလႆးႁဵတ်ႉ continue ဝႆႉတီႈ 3 ၼႆလႄႈ 3 ၼႆႉႁဝ်းတေဢမ်ႇႁၼ်လႆႈ၊ ၵွပ်ႈဝၢႆးသေမၼ်းႁၼ်မႅၼ်ႈ continue ၼႆ မၼ်းတေဢမ်ႇသိုပ်ႇလူင်းၵႂႃႇလႆႈ တီႈတႂ်ႈ ဢၼ်မီးဝႆႉ print ၼၼ်ႉလႄႈ ႁဝ်းတေလႆႈၶိုၼ်း (1,2,4,5) ၼႆၵူၺ်းယဝ်ႉ။

```
x = 0  
while x < 6:  
    x += 1  
    if x == 3: continue  
    print(x)
```

**The "else" statement**

သင်ဝႃႈ while statement ပဵၼ် True မၼ်းတေ ၶဝ်ႈႁဵတ်ႉၵၢၼ်ၼႂ်း while block သေ သင်ဝႃႈ မၼ်းပဵၼ် False ၵႂႃႇယဝ်ႉၼႆၸိုင် မၼ်းၶိုၼ်းတေ ၶဝ်ႈႁဵတ်ႉၵၢၼ်ၼႂ်း else block ယဝ်ႉ။ ၶူတ်ႉတႂ်ႈၼႆႉ ဝၢႆးသေ ဢွၵ်ႇၼႄပၼ် 1-5 ယဝ်ႉၼႆ မၼ်းတေဢွၵ်ႇပၼ်ထႅင်ႈ "x is no longer less than 6" ၼႆယူႇ။

```
x = 1  
while x < 6:  
    print(x)  
    x += 1  
else:  
    print("i is no longer less than 6")
```
