
## 53) File Input & Output

တီႈ python ၼႆႉ သင်ၸိူဝ်ႉဝႃႈ ႁဝ်းၶႆႈႁဵတ်ႉၵၢၼ်ၸွမ်း File မိူၼ်ၼင်ႇ လူလိၵ်ႈ၊ တႅမ်ႈလိၵ်ႈ၊ မႄးတႅမ်ႈ၊ ထွၼ်ႇမွတ်ႇၼႆ ႁဝ်းၸႂ်ႉလႆႈ open() function ယဝ်ႉ။ ၼႂ်းမၼ်းၼၼ်ႉ မၼ်းတေယွၼ်းဝႆႉ parameter (2) ဢၼ်လႄႈ ဢၼ်တေႇ ပဵၼ်ၸိုဝ်ႈၾၢႆႇ ဢၼ်ႁဝ်းၶႆႈႁဵတ်ႉၵၢၼ်ၸွမ်းလႄႈ ဢၼ်ထိသွင်သမ်ႉ ပဵၼ် mode မၼ်းယဝ်ႉ။

တႃႇတေၽုၺ်ႇ File ၼႆႉႁဝ်းတေမီးဝႆႉ 4 mode ၼႆႉ။

1) "**r**" (read) - ၸႂ်ႉတႃႇလူလိၵ်ႈ။ သင် file ဢၼ်တေလူ ဢမ်ႇမီးၼႆႉ မၼ်းတေၼႄပၼ်ၶိုၼ်း error ယဝ်ႉ။

2) "**a**" (append) - ၸႂ်ႉတႃႇတႅမ်ႈလိၵ်ႈ။ သင်ၼႂ်းမၼ်း မီးဝႆႉလိၵ်ႈၼႆ မၼ်းတေၽိူမ်ႉထႅမ်တႅမ်ႈၵႂႃႇပၼ်။ သင်ၾၢႆႇႁဝ်းဢမ်ႇမီး တေၶူင်ပၼ် ၾၢႆႇဢၼ်မႂ်ႇ။

3) "**w**" (write) - ၸႂ်ႉတႃႇတႅမ်ႈလိၵ်ႈ။ သင်ၼႂ်းၾၢႆႇ မီးဝႆႉလိၵ်ႈယဝ်ႉၼႆ မၼ်းတေ overwrite ပႅတ်ႈ။ သင်ၾၢႆႇဢမ်ႇမီး တေႁဵတ်ႉပၼ် ဢၼ်မႂ်ႇ။

4) "**x**" (create) - ၸႂ်ႉတႃႇၶူင်သၢင်ႈၾၢႆႇဢၼ်မႂ်ႇ။ သင်မီးဝႆႉၼႆၸိုင် တေဢွၵ်ႇၼႄပၼ် error ။
