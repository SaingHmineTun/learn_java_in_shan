
## 18) Set Type

Set ၼႆႉသမ်ႉ တေႁမ်ႉ မိူၼ်ၵၼ်တၢင်း List ယူႇ။ ၵူၺ်းၼႂ်းမၼ်းၼၼ်ႉ ႁဝ်းတေဢမ်ႇသႂ်ႇလႆႈ ၵႃႊၶၼ်မိူၼ်ၵၼ်လႄႈ မၼ်းဢမ်ႇၶပ်ႉၸွမ်း ပိူင်သင်ဝႆႉ။ ယဝ်ႉၵေႃႈ မၼ်းဢမ်ႇၸႂ်ႉ index လႄႈ key မိူၼ် list လႄႈ dictionary ၼႆၵေႃႈ ႁဝ်းတေဢဝ်ၵႃႊၶၼ် ၵမ်းဢၼ်ၵေႃႈ ဢမ်ႇလႆႈ။

Set ၼႆႉႁဝ်းၶူင်သၢင်ႈ ၸႂ်ႉတိုဝ်း တူင်ႇၶဝ်ႈသွႆး (curly bracket) ယဝ်ႉ။

```
thisset = {"abc", 34, True, 40, "male"}  
print(thisset)
```

**Add New Items into Set**

ၼႂ်း Set ၼႆႉ ၵႃႊၶၼ် ဢၼ်လႆႈသႂ်ႇဝႆႉၼၼ်ႉ ႁဝ်းတေမႄး update ၶိုၼ်း ဢမ်ႇလႆႈသေတႃႉ ဢဝ်တေသႂ်ႇထႅမ်လႆႈ၊ ထွၼ်ဢွၵ်ႇလႆႈ ၵႃႊၶၼ်မၼ်းယူႇ။ တႃႇတေ ထႅမ်ၵႃႊၶၼ်မႂ်ႇၼႆႉ ႁဝ်းၸႂ်ႉတိုဝ်း add() method ယဝ်ႉ။

```
thisset = {"apple", "banana", "cherry"}  
thisset.add("orange")
```

သင်ၶႆႈထႅမ်သႂ်ႇဝႆႉ ၵႃႊၶၼ် set ထႅင်ႈဢၼ်ၼိုင်ႈၼႆၸိုင် ႁဝ်းၸႂ်ႉလႆႈ update() method ယဝ်ႉ။

```
thisset = {"apple", "banana", "cherry"}  
tropical = {"pineapple", "mango", "papaya"}  
thisset.update(tropical)
```

**Remove Set Items**

သင်ၶႆႈထွၼ်ဢွၵ်ႇ ၵႃႊၶၼ်ၵဝ်ႇ ဢၼ်မီးဝႆႉ ၼႂ်းမၼ်းၼႆၸိုင် ႁဝ်းႁွင်ႉၸႂ်ႉလႆႈ remove() ယဝ်ႉ။

```
thisset = {"apple", "banana", "cherry"}  
thisset.remove("banana")
```

လွင်ႈဢၼ်ၸႂ်ႉတိုဝ်း remove() ၼႆႉ သင်ၵႃႊၶၼ် ဢၼ်ႁဝ်းၶႆႈထွၼ် မၼ်းဢမ်ႇမီးၼႆၸိုင် မၼ်းတေဢွၵ်ႇၼႄပၼ်ႁဝ်း Error ယဝ်ႉ။ သင်ၼၼ် ႁဝ်းၶိုၼ်းၸႂ်ႉတိုဝ်း discard() method လႆႈယူႇ။ မၼ်းတေ ဢမ်ႇၼႄ error သင်ပၼ်ႁဝ်း။

```
thisset.discard("banana")
```

သင်ႁဝ်း ၶႆႈထွၼ်ဢွၵ်ႇ ၵႃႊၶၼ်သေဢၼ်ဢၼ်လႄႈ ဢမ်ႇၶဝ်ႈၸႂ်ဝႃႈ တေထွၼ်ဢၼ်လႂ်ၼႆၸိုင် ႁဝ်းၸႂ်ႉလႆႈ pop() method ယဝ်ႉ။ မၼ်းတေပၼ်ႁဝ်းၶိုၼ်း ၵႃႊၶၼ် ဢၼ်ႁဝ်း ထွၼ်ဢွၵ်ႇပႅတ်ႈဢိူဝ်ႈ။

```
x = thisset.pop()  
print(x)
```

သင်ၶႆႈထွၼ်ဢွၵ်ႇ ၵႃႊၶၼ်တၢင်းမူတ်း ဢၼ်မီးဝႆႉတီႈၼႂ်း set ၼႆၸိုင် ႁဝ်းၸႂ်ႉလႆႈ clear() method ယဝ်ႉ။

```
thisset.clear()
```
