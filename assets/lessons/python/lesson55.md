
### 7.2) Create or write to a file

**Create a new file**

တီႈတေႉ သင်ႁဝ်းၸႂ်ႉတိုဝ်း "a" လႄႈ "w" ၵေႃႈ မၼ်းတေၶူင်ပၼ် ၾၢႆႇဢၼ်ၼိုင်ႈယူႇ။ ၵူၺ်းသင်ၸိူဝ်ႉ ၶႆႈႁဵတ်ႉ ၾၢႆႇလၢႆလၢႆၵူၺ်းၼႆ ႁဝ်းၸႂ်ႉလႆႈ "x" (create) ယဝ်ႉ။

```
f = open("myfile.txt", "x")
```

**Write to a file**

တႃႇတေတႅမ်ႈလိၵ်ႈၼႂ်းၾၢႆႇၼႆ ႁဝ်းမီးဝႆႉယူႇ 2 လၢႆး။ "a" (append) ၼႆႉ မၼ်းတေၽိူမ်ႉထႅမ် တႅမ်ႈၵႂႃႇတီႈလိၵ်ႈ လိုၼ်းသုတ်ႉ ဢၼ်မီးဝႆႉလႄႈ "w" (write) ၼႆႉသမ်ႉ မၼ်းတေ overwrite ပႅတ်ႈ ၾၢႆႇတၢင်းဢၼ်ယဝ်ႉ။

သင်ၽုၺ်ႇ ၾၢႆႇယဝ်ႉၼႆ ႁဝ်းၸႂ်ႉတိုဝ်း write() method သေတီႈၼႂ်းမၼ်း တႅမ်ႈပၼ်ထႅဝ်လိၵ်ႈ ဢၼ်ႁဝ်းၶႆႈတႅမ်ႈသႂ်ႇၼႂ်း file ႁဝ်းၵေႃႈယဝ်ႉ။ သင်ၸိုဝ်ႈၾၢႆႇ ဢၼ်ႁဝ်းဢဝ်ဝႆႉၼၼ်ႉ ဢမ်ႇမီးၼႆၸိုင် မၼ်းတေတႅမ်ႈပၼ် file ၵႂႃႇဢၼ်ၼိုင်ႈယဝ်ႉ။

```
f = open("demofile2.txt", "a")  
f.write("Now the file has more content!")  
f.close()
```
