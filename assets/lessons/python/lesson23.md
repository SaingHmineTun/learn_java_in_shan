
## 23) Comparison Operators

ၼႆႉသမ်ႉ ႁဝ်းၸႂ်ႉတိုဝ်း တႃႇတႅၵ်ႊလိူင်း ၵႃႊၶၼ်ၵၼ်ယဝ်ႉ။ Operators ၼႆႉတေႉ မၼ်းတေ ပၼ်ႁဝ်းၶိုၼ်း boolean value ၵူၺ်းဢိူဝ်ႈ။

| Operator | Name | Example |
| --- | --- | --- |
| \== | Equal | x == y |
| !=  | Not equal | x != y |
| \>  | Greater than | x > y |
| <   | Less than | x < y |
| \>= | Greater than or equal to | x >= y |
| <=  | Less than or equal to | x <= y |

###### လၢႆးၸႂ်ႉတိုဝ်း 
```
x = 5
y = 10
print(x == y) # False
print(x != y) # True
print(x > y) # False
print(x < y) # True
print(x >= y) # False
print(x <= y) # True

y = 5
print(x >= y) # True
```
