## 9) Python Basic Data Types
Data Types ၼႆႉပွင်ႇဝႃႈပဵၼ် ၶေႃႈမုၼ်း ဢၼ်ႁဝ်းတေ ၸႂ်ႉႁဵတ်ႉၵၢၼ်လႆႈ၊ ဢမ်ႇၼမ် သိမ်းဝႆႉလႆႈၼႂ်း variable ၼႆယဝ်ႉ။ တီႈ python ၼႆႉ မၼ်းတေမီးဝႆႉ built-in data type ဢၼ်ၵိုၵ်းပႃးမႃးၸွမ်းမၼ်းဝႆႉ ၼင်ႇတီႈတႂ်ႈၼႆႉယူႇ။

| **Type Description** | **Type in Python** |
| --- | --- |
| Boolean Type | bool |
| Numeric Types | int, float, complex |
| Text Type | str |
| None Type: | NoneType |
| Sequence Types: | list, tuple |
| Mapping Type: | dict |
| Set Types: | set |
|     |     |

**Getting the data type**

သင်ၶႆႈႁူႉဝႃႈ variable ႁဝ်းၼႆႉ ပဵၼ် data type သင်ၼႆၸိုင် ၸႂ်ႉတိုဝ်းလႆႈ type() function ၼင်ႇတီႈတႂ်ႈၼႆႉသေ ၸွပ်ႇလႆႈယူႇ။ ၶူတ်ႉတီႈတႂ်ႈၼႆႆႉ ပေႃးႁဝ်းဢွၵ်ႇတူၺ်းၼႆ မၼ်းတေၼႄပၼ်ႁဝ်းဝႃႈ int ၼႆယဝ်ႉ။

```
num = 123  
print(type(num)) # <class 'int'>  
num = 'Python'  
print(type(num)) # <class 'str'> 
num = True  
print(type(num)) # <class 'bool'>
```
