
## 21) Arithmetic operators

ၼႆႉႁဝ်းၸႂ်ႉတိုဝ်း တႃႇႁဵတ်ႉၵၢၼ်ၸွမ်း မၢႆၼပ်ႉ မိူၼ်ၸိူင်ႉ မၢႆလေႃး၊ မၢႆထွၼ်ၸိူဝ်းၼႆႉယဝ်ႉ။ တီႈ python မၼ်းတေမီးဝႆႉ arithmetic operators (7) ဢၼ်ၼႆႉ။

| Operator | Name | Example |
| --- | --- | --- |
| +   | Addition | x + y |
| \-  | Subtraction | x - y |
| \*  | Multiplication | x \* y |
| /   | Division | x / y |
| %   | Modulus | x % y |
| \*\* | Exponentiation | x \*\* y |
| //  | Floor division | x // y |

#### Example

```
x = 5
y = 10
print(x + y) # 15
print(x - y) # -5
print(x * y) # 50
print(x / y) # 0.5
print(x // y) # 0
print(x % y) # 5
print(x ** 3) # 5^3 = 125
```
