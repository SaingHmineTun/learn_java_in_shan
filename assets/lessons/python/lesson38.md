
## Python Lambda

တီႈတေႉ မၼ်းၵေႃႈ ဢမ်ႇၸႂ်သင်။ မၼ်းၵေႃႈပဵၼ် function ဢၼ်ၼိုင်ႈၵူၺ်းၼင်ႇၵဝ်ႇ။ Anonymous function ၼႆၵေႃႈ ပိူၼ်ႈႁွင်ႉမၼ်းယဝ်ႉ။ Lambda ၼႆႉ မၼ်းတေႁပ်ႉၶၢမ်ႇလႆႈ arguments တၢင်းၼမ်သေတႃႉ မၼ်းတေလႆႈမီး expression ဢၼ်လဵဝ်ၵူၺ်း။ ၶူတ်ႉတီႈတႂ်ႈၼႆႉ မၼ်းယွၼ်းဝႆႉ argument (3) ဢၼ်သေ return ၶိုၼ်း ၵႃႊၶၼ် ဢၼ်လေႃးၵၼ်ဝႆႉယဝ်ႉ။

```
x = lambda a, b, c: a + b + c  
print(x(5, 6, 2))
```

သင်ဢမ်ႇၸႂ်ႉ lambda သေမႃးၸႂ်ႉ function ယူဝ်းယူဝ်းၵေႃႈ လႆႈၼင်ႇၵဝ်ႇ။ Lambda တီႈၼိူဝ်ၼႆႉလႄႈ function တီႈတႂ်ႈၼႆႉ မၼ်းတေ မိူၼ်ၵၼ်ဝႆႉၵူၺ်းဢိူဝ်ႈ။

```
def x (a, b, c):  
    return a + b + c  

print(x(1,2,3))
```
