
## Renaming a module

မိူဝ်ႈၽွင်းႁဝ်း import module သေဢၼ်ဢၼ်ၼၼ်ႉ သင်သမ်ႉ ႁဝ်းၶႆႈႁဵတ်ႉ shortcut တႃႇမၼ်းၼႆၸိုင် ႁဝ်းၸႂ်ႉၵိုၵ်းၸွမ်း as ၵေႃႈလႆႈယူႇ။ သင်ၼၼ် ႁဝ်းဢမ်ႇလူဝ်ႇႁွင်ႉၸွမ်း ၸိုဝ်ႈတဵမ်မၼ်းသေ ႁွင်ႉၸွမ်း shortcut မၼ်းၵေႃႈ လႆႈယဝ်ႉ။ ၶူတ်ႉတီႈတႂ်ႈၼႆႉ ၶႃႈလႆႈဢဝ် mymodule ၶႃႈပဵၼ် mx ဝႆႉလႄႈ မိူဝ်ႈၽွင်းၶႃႈတေ ႁွင်ႉၸႂ်ႉ variable ဢၼ်မီးဝႆႉ တီႈၼႂ်းမၼ်းၼၼ်ႉ ၸင်ႇႁွင်ႉၸႂ်ႉၸွမ်း mx ၵေႃႈလႆႈယဝ်ႉ။

```
import mymodule as mx  
a = mx.person1["age"]  
print(a)
```
