
## Class & Object

Python ၼႆႉပဵၼ် Object oriented programming ဢိူဝ်ႈ။ ၼႂ်းမၼ်းၼၼ်ႉ ပဵၼ်ဝႆႉ class ၵမ်ႈၼမ်ယဝ်ႉ။ class ၼႆႉႁဝ်းၸႂ်ႉတိုဝ်း တႃႇၶူင်သၢင်ႈ object လႄႈ သင်ယဝ်ႉ ႁဝ်းတေႁွင်ႉၸႂ်ႉလႆႈ properties လႄႈ methods ဢၼ်ယူႇဝႆႉ တီႈၼႂ်း class ၼၼ်ႉယဝ်ႉ။

#### Create a class

တႃႇၶူင်သၢင်ႈ class ဢၼ်ၼိုင်ႈၼႆႉ ႁဝ်းၸႂ်ႉတိုဝ်း class keyword သေၽၢႆႇၼင်ႇမၼ်းၼၼ်ႉ ၶိုၼ်းတႅမ်ႈၵိုၵ်း ၸိုဝ်ႈမၼ်းယဝ်ႉ။ သင်ၼၼ် ႁဝ်းၸႂ်ႉ full column (:) သေၵေႃႈ ၼႂ်းမၼ်းၼၼ်ႉ ႁဝ်းၶိုၼ်းတႅမ်ႈလႆႈ properties လႄႈ method ယဝ်ႉ။ Properties ၼႆႉ မၼ်းၵေႃႈပဵၼ် variable ၵူၺ်း။ သင်ႁဝ်းၶူင်သၢင်ႈ variable တီႈၼႂ်း class ၼႆ ႁဝ်းႁွင်ႉမၼ်းဝႃႈ properties ၼႆယဝ်ႉ။ ၶူတ်ႉတႂ်ႈၼႆႉ ပဵၼ်လွင်ႈ ဢၼ်ႁဝ်းၶူင်သၢင်ႈ class ဢၼ်ၼိုင်ႈသေ ၼႂ်းမၼ်းလႆႈသႂ်ႇဝႆႉ properties ဢၼ်ၼိုင်ႈ (x) ယဝ်ႉ။

```
class MyClass:  
    x = 5
```

#### Create objects from a class
သင်ႁဝ်းမီး class ၼိုင်ႈဢၼ်ၼႆ ပိူင်ၼိူဝ် class ၼၼ်ႉသေ ႁဝ်းတေၶူင်လႆႈ objects တၢင်းၼမ်ယဝ်ႉ။ သင်ႁဝ်းတေၶူင်သၢင်ႈ object ၼႆတေႉ တႅမ်ႈၸိုဝ်ႈ class သေၵေႃႈ ၶိုၼ်ႈသုတ်ႉၸွမ်း တူင်ႇမူၼ်း ၵေႃႈလႆႈယဝ်ႉ။ သင်ၼၼ် ႁဝ်းတေလႆႈမႃး object ဢၼ်ၼိုင်ႈသေ type မၼ်းတေပဵၼ်ဝႆႉ class ၼၼ်ႉဝႆႉယဝ်ႉ။

သင်ႁဝ်းလႆႈမႃး object ဢၼ်ၼိုင်ၼႆၸိုင် ႁဝ်းၸႂ်ႉမၼ်းသေၵေႃႈ တေႁွင်ႉၸႂ်ႉလႆႈ properties လႄႈ method ဢၼ်မီးဝႆႉ ၼႂ်းမၼ်းယဝ်ႉ။ လွင်ႈၼႆႉၵေႃႈ ႁဝ်ၸႂ်ႉၵိုၵ်ႉ dot (.) ၵူၺ်း။

ၶူတ်ႉတႂ်ႈၼႆႉ လႆႈၼႄဝႆႉ လၢႆးၶူင်သၢင်ႈ object ၸႂ်ႉတိုဝ်း MyClass သေ လၢႆးႁွင်ႉၸႂ်ႉ properties ဢၼ်မီးဝႆႉတီႈၼႂ်းမၼ်းၼၼ်ႉယဝ်ႉ။

```
p1 = MyClass()  
print(p1.x)
```
