## 7) Python Comments

ၼႂ်း Programming ၼႆႉ comments ၼႆႉပွင်ႇဝႃႈ ပဵၼ်လိၵ်ႈ ဢၼ်ႁဝ်းတႅမ်ႈဝႆႉ၊ တႃႇႁဝ်းတေလူ ၼႆယဝ်ႉ။ လိၵ်ႈၼႆႉ
Interpreter မၼ်းတေ ၶၢမ်ႈပူၼ်ႉၵႂႃႇယဝ်ႉ။ သင်ႁဝ်း ဢမ်ႇႁဵတ်ႉ comments သေတႅမ်ႈတူဝ်လိၵ်ႈၼႆၸိုင်
မၼ်းတေၼႄဝႆႉပၼ် ၽိတ်းယူႇယဝ်ႉ။

တီႈ Python ၼႆႉမၼ်းတေမီးဝႆႉ comments (2) လွင်ႈ

**1\. Single line comments**

ၼႆႉတေႉ သင်ႁဝ်းၸႂ်ႉတိုဝ်းမၼ်းၼႆၸိုင် တူဝ်လိၵ်ႈ ဢၼ်ယူႇၽၢႆႇလင်မၼ်း တေပဵၼ် comments မူတ်းၼႆဢိူဝ်ႈ။ တီႈ
python ႁဝ်းၸႂ်ႉ **#** တႃႇတႅမ်ႈ single line comments ဢိူဝ်ႈ။

```
# This is a comment
# written in
# more than just one line
 ```

**2\. Multi line comments**

ၼႆႉတေႉၵေႃႈ ႁဝ်းတေမီးဝႆႉ တူဝ်ၽုၺ်ႇ၊ တူဝ်ဢိုတ်း တီႈမၼ်းသေလႄႈ လိၵ်ႈတၢင်းမူတ်း ဢၼ်ႁဝ်းတႅမ်ႈဝႆႉ
ၼႂ်းမၼ်းၼၼ်ႉ python မၵ်းမၼ်ႈပဵၼ် comments ယဝ်ႉ။ ႁဝ်းၸႂ်ႉ Triple quotes (""" … """) သေတႅမ်ႈ
multi-line comments ဢိူဝ်ႈ။ တီႈတေႉ python ၼႆႉ ၸႂ်ႉတိုဝ်း multi-line string တႃႇႁဵတ်ႉ multi-line
comment ၵူၺ်းဢိူဝ်ႈ။

```
"""  
    This is a comment  
    written in  
    more than just one line  
"""
```
