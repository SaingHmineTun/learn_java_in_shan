
## 4) Running python as a python program

Python ၼႆႉပဵၼ် Interpreted Programming Language ဢိူဝ်ႈ။ ၵွပ်ႈၼႆလႄႈ သင်ႁဝ်းမီးဝႆႉ python file (.py) ၼႆၸိုင် ႁဝ်းၸႂ်ႉ python သေၵေႃႈ run လႆႈၵမ်းလဵဝ်ယဝ်ႉ။

တႃႇတေၶူင်သၢင်ႈ python program ဢၼ်ၼိုင်ႈၼႆႉၵေႃႈ ဢမ်ႇယၢပ်ႇ။ ဢွၼ်တၢင်းသုတ်ႉ ၶူင်သၢင်ႈ **helloworld.py** ဝႆႉၼိူဝ် Desktop သူလႄႈ။ သင်ယဝ်ႉၵေႃႈ ၶိုၼ်းၽုၺ်ႇမၼ်း ၸႂ်ႉတိုဝ်း Text Editor ​သေဢၼ်ဢၼ်ၼႆၵေႃႈ ၶိုၼ်းတႅမ်ႈပၼ် code တီႈတႂ်ႈၼႆႉလႄႈ။

```
print("Hello World")
```

သင်တႅမ်ႈယဝ်ႉ save ပၼ်မၼ်းသေ ၶိုၼ်းႁွင်ႉဢဝ် Command Prompt (ဢမ်ႇၼၼ်) Terminal သေ ၶိုၼ်း run ၸွမ်း command တီႈတႂ်ႈၼႆႉလႄႈ။

```
python helloworld.py
```
