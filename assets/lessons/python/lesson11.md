
## 11) Numeric Types

တီႈ Python ၼႆႉ မၼ်းတေမီးဝႆႉ Numeric Type ယူႇ (3) မဵဝ်း

**(1) int**

int ၼႆႉ တႃႇႁဝ်းတေသိမ်းဝႆႉ whole number ၼႆယဝ်ႉ။ မၼ်းတေႁပ်ႉပၼ်လႆႈ positive, negative လႄႈ မၼ်းဢမ်ႇမီး size limit မိူၼ်ၽႃႇသႃႇတၢင်ႇဢၼ်။

```
age = 20  
number = -5  
print(age + number) # 15
```

**(2) float**

float သမ်ႉႁဝ်းၸႂ်ႉတိုဝ်း တႃႇသိမ်းဝႆႉ decimal number ယဝ်ႉ။ ၼႂ်းမၼ်း ႁဝ်းတေသႂ်ႇလႆႈ တူဝ် e သေ မၼ်းမၢႆထိုင် 10 power ၼႆယဝ်ႉ။

```
x = 1.0  
y = 1.2345e2  
print(x + y) # 124.45
```

**(3) complex**

complex number သမ်ႉႁဝ်းၸႂ်ႉၸွမ်း j ယဝ်ႉ။ မၼ်းတၢင်တူဝ်ၼပ်ႉ ဢၼ်ႁဝ်းဢမ်ႇႁူႉလႄႈ ၵမ်ႈၼမ် တေၸႂ်ႉတိုဝ်းၵၼ်တီႈ Electrical Engineering, Physics လႄႈ Mathmatics ၼၼ်ႉယဝ်ႉ။

```
z = 2 + 3j  
print(z) # 2+3j
```
