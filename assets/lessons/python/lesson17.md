
## 17) Dictionary Type

Dictionary ၼႆႉသမ်ႉ ၶဝ်ၸႂ်ႉ key-value တႃႇသိမ်း ၵႃႊၶၼ်မၼ်းဢိူဝ်ႈ။ ၼႂ်းမၼ်းၼၼ်ႉ မၼ်းတေသိမ်းပၼ်ဝႆႉ ၵႃႊၶၼ်လႆႈသေ တေမူၼ်ႉမႄးမၼ်းၵေႃႈ ႁဝ်းတေလႆႈ။ ၵူၺ်းၼႂ်းမၼ်း ႁဝ်းတေ ဢမ်ႇလႆႈမီး Key မိူၼ်ၵၼ်။ တေႇတီႈ Python 3.7 မႃးၼႆ ၼႂ်းမၼ်းၼႆႉ တေၶပ်ႉပၼ်ၸွမ်း ဢၼ်ႁဝ်းသႂ်ႇပၼ်ၵႂႃႇယူႇ၊ ပႃးသျိၼ်းလူင်းတႂ်ႈၼႆႉ ၶဝ်ဢမ်ႇၶပ်ႉပၼ်။

တႃႇတေၶူင်သၢင်ႈ dictionary ဢၼ်ၼိုင်ႈၼႆ ႁဝ်းၸႂ်ႉတိုဝ်း တူင်ႇၶဝ်ႈသွႆး (curly bracket) ယဝ်ႉ။ ၼႂ်းမၼ်းၼၼ်ႉ key လႄႈ value ၶိုၼ်းဢဝ် full column သေၶၼ်ႈၵၼ်ယဝ်ႉ။

```
thisdict = {  
    "brand": "Ford",  
    "model": "Mustang",  
    "year": 1964  
}
```

**Getting Dictionary Item**

တႃႇတေဢဝ် ၵႃႊၶၼ်ၵမ်းတူဝ်တူဝ် တီႈၼႂ်းမၼ်းၼႆၸိုင် ၸႂ်ႉၸွမ်း တူင်ႇသီႇၸဵင်ႇ (squared bracket) သေၼႂ်းမၼ်း ၶိုၼ်းသႂ်ႇပၼ် key မၼ်းယဝ်ႉ။ သင်ဢမ်ႇၼၼ် ႁွင်ႉၸႂ်ႉ get() method ၼႆၵေႃႈလႆႈယူႇ။

```
x = thisdict["model"]
x = thisdict.get("model")
```

**Get Keys and Values**

သင်ၶႆႈလႆႈ key ဢၼ်မီးဝႆႉတီႈၼႂ်း Dictionary ႁဝ်းၼႆၸိုင် ႁဝ်းႁွင်ႉၸႂ်ႉ keys() method ယဝ်ႉ။ သင်သမ်ႉ ၶႆႈလႆႈ values မၼ်းၼႆၸိုင် ၶိုၼ်းႁွင်ႉၸႂ်ႉလႆႈ values() method ၼင်ႇၵဝ်ႇ။ သင်ၼၼ် ႁဝ်းတေလႆႈမႃး List ဢၼ်တေသိမ်းပၼ်ဝႆႉ keys ဢမ်ႇၼၼ် values ႁဝ်းယဝ်ႉ။

```
x = thisdict.keys()
x = thisdict.values()
```

**Change Items**

သင်ၶႆႈလၢႆႊ ၵႃႊၶၼ်မၼ်း ၵမ်းဢၼ်ၼႆၸိုင် ႁဝ်းၸႂ်ႉတိုဝ်း တူင်ႇသီႇၸဵင်ႇသေ ၼႂ်းမၼ်းႁဝ်းၶိုၼ်းသႂ်ႇပၼ် key လႄႈ မၢႆမိူၼ်သေၵေႃႈ လႆႈယဝ်ႉ။

``` 
thisdict["year"] = 2018
```

သင်ဢမ်ႇၼၼ်ၵေႃႈ ႁဝ်းၸႂ်ႉတိုဝ်း update() method ၵေႃႈလႆႈယူႇ။ ၼႂ်းမၼ်းၼၼ်ႉ ႁဝ်းၶိုၼ်းသႂ်ႇပၼ် dictionary ဢၼ်ႁဝ်းၶႆႈႁဵတ်ႉ update ၼၼ်ႉယဝ်ႉ။

```
thisdict.update({"year": 2020})
```

**Add Items into Dictionary**

ၼႆႉၵေႃႈ ႁဝ်းယင်းၸႂ်ႉလႆႈ တူင်ႇသီႇၸဵင်ႇဢိူဝ်ႈ။ key မၼ်းႁဝ်းသႂ်ႇဝႆႉ ဢၼ်ပႆႇမီးဝႆႉၼႂ်း dictionary ႁဝ်းၼႆၵေႃႈ မၼ်းတေၶူင်ပၼ်ႁဝ်း key မႂ်ႇၵႂႃႇၼိုင်ႈဢၼ်ယဝ်ႉ။

```
thisdict["color"] = "red"
```

ဢမ်ႇၼၼ်ၵေႃႈ ႁဝ်းၸႂ်ႉတိုဝ်း update() method ၵေႃႈလႆႈယူႇ။ သင် key ဢၼ်ႁဝ်းၶႆႈႁဵတ်ႉ update ၼႆႉ ဢမ်ႇမီးၼႂ်း dictionary ႁဝ်းၼႆ မၼ်းၶိုၼ်းတေ ၶူင်ပၼ်ႁဝ်း key မႂ်ႇယဝ်ႉ။

```
thisdict.update({"color": "red"})
```

**Remove Items from Dictionary**

တီႈၼႆႈ ႁဝ်းၸႂ်ႉတိုဝ်း pop() method ယဝ်ႉ။ ၼႂ်း parameter မၼ်းၶိုၼ်းသႂ်ႇပၼ် ၵႃႊၶၼ် key ဢၼ်ႁဝ်း ၶႆႈထွၼ်ဢွၵ်ႇယဝ်ႉ။

```
thisdict.pop("model")
```

သင်ႁဝ်းၶႆးထွၼ်ဢွၵ်ႇ ၵႃႊၶၼ်ဢၼ်လႆႈသႂ်ႇဝႆႉ လိုၼ်းသုတ်ႉ တီႈ dictionary ႁဝ်းၼႆၸိုင် ႁဝ်းၸႂ်ႉတိုဝ်း popitem() method လႆႈယဝ်ႉ။ တီႈၼႆႈၵေႃႈ သင်ပဵၼ် version 3.7 လူင်းတႂ်ႈၼႆၸိုင် မၼ်းတေထွၼ်ပၼ် ၵႃႊၶၼ် သေဢၼ်ဢၼ် ထွၼ်လူၺ်ႈ random ၵႂႃႇၵူၺ်း။

```
thisdict.popitem()
```

သင်ၶႆႈလၢင်ႉၸဵင်ႈၸဵင်ႈ dictionary ႁဝ်းၼႆၸိုင် ၸႂ်ႉတိုဝ်း clear() method လႆႈဢိူဝ်ႈ။

```
thisdict.clear()
```
