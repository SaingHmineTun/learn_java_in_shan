
## 16) Tuple

Tuple လႄႈ List ၼႆႉ တေႁမ်ႉမိူၼ်ၵၼ်ဝႆႉ ၵမ်ႈၼမ်ယူႇ။ ၵူၺ်းၵႃႊ Tuple ၼႆႉ သင်ႁဝ်းသႂ်ႇဝႆႉ ၵႃႊၶၼ်ယဝ်ႉၼႆ ႁဝ်းတေ မူၼ်ႉမႄး ၵႃႊၶၼ်မၼ်း ဢမ်ႇလႆႈယဝ်ႉ။ မၼ်းပဵၼ် Unchangeable list ယဝ်ႉ။ တႃႇတေၶူင်သၢင်ႈ tuple ၼိုင်ႈဢၼ်ၼႆႉ ႁဝ်းၸႂ်ႉတိုဝ်း comma ၼႆၵေႃႈလႆႈယဝ်ႉ။ ၵမ်ႈၼမ်တေႉ ႁဝ်းတေၸႂ်ႉၸွမ်း တူင်ႇမူၼ်း (Round bracket) ယဝ်ႉ။ ၼႂ်း tuple ၵေႃႈ ႁဝ်းတေသႂ်ႇလႆႈ ၵႃႊၶၼ်မိူၼ်ၵၼ် (Duplicate values) မိူၼ်ၼင်ႇ List ယူႇၼင်ႇၵဝ်ႇ။ ယဝ်ႉၼႂ်းမၼ်းၼႆႉ ႁဝ်းတေသႂ်ႇလႆႈ ၵႃႊၶၼ် data type ဢၼ်ဢမ်ႇမိူၼ်ၵၼ်ၵေႃႈလႆႈ။

```
tuple1 = ("abc", 34, True, 40, "male")
tuple1 = "abc", 34, True, 40, "male"
```

**Access Tuple Items**

သင်ႁဝ်း ၶႆႈလႆႈ ၵႃႊၶၼ် ဢၼ်မီးဝႆႉၼႂ်း tuple ၼႆၸိုင် မၼ်းၵေႃႈ တေမိူၼ် List ၼင်ႇၵဝ်ႇ။ ႁဝ်းတေၸႂ်ႉလႆႈ Positive indexing, Negative indexing ယဝ်ႉ။

```
thistuple = ("apple", "banana", "cherry")  
print(thistuple[1])
print(thistuple[-1])
```

**Update Tuple Items**

Tuple ၼႆႉမၼ်းပဵၼ် immutable သေဢမ်ႇၵႃး ပဵၼ်ပႃး unchangeable လႄႈ ယႃႇဝႃႈ တေထႅမ်သႂ်ႇ ၵႃႊၶၼ်မႂ်ႇ၊ ထွၼ်ဢွၵ်ႇၵႃႊၶၼ်ၵဝ်ႇ၊ တေႃႈၼင်ႇ ႁဝ်းတေ မူၼ်ႉမႄး ၵႃႊၶၼ် ဢၼ်မီးဝႆႉ ၼႂ်းမၼ်းၵေႃႈ ယင်းတေဢမ်ႇလႆႈ။ ဢၼ်ႁဝ်းတေ ႁဵတ်ႉလႆႈတေႉၵေႃႈ ဢဝ် tuple ပိၼ်ႇပဵၼ် list ၵူၺ်းယဝ်ႉ။

```
x = ("apple", "banana", "cherry")  
y = list(x)
```
