
## 15) List Type

ႁဝ်းၸႂ်ႉ List သေၵေႃႈ ၼႂ်းမၼ်းၼၼ်ႉ ႁဝ်းတေသိမ်းဝႆႉလႆႈ ၵႃႊၶၼ်တၢင်းၼမ်။ တီႈ python ၼႆႉ ႁဝ်းၸႂ်ႉ တူင်ႇသီႇၸဵင်ႇ (Squared bracket) တႃႇတေၶူင်သၢင်ႈ List ၼိုင်ႈဢၼ်ယဝ်ႉ။ ၼႂ်းမၼ်းၼၼ်ႉ ႁဝ်းၶႆႈသႂ်ႇဝႆႉ type သင်ၵေႃႈ လႆးၵွၼ်ႇဢိူဝ်ႈ။ ၵႃႊၶၼ် ဢၼ်ႁဝ်း သႂ်ႇဝႆႉၼႂ်း List ၼႆႉ မၼ်းတေမီးဝႆႉ insertion order ဢိူဝ်ႈ။ ယဝ်ႉၵေႃႈ ႁဝ်းတေသႂ်ႇဝႆႉလႆႈ ၵႃႊၶၼ် မိူၼ်ၵၼ်သေလႄႈ List ႁဝ်းၼႆႉ ဝၢႆးသေၶူင်ယဝ်ႉႉ ႁဝ်းၶိုၼ်း ထႅမ်မႂ်ႇ၊ ထွၼ်ၵဝ်ႇ ၵေႃႈလႆႈယဝ်ႉ။

```
list = ["abc", 34, True, 40, "male"]
```

**Access Items**

List ၼႆႉ မၼ်းၸႂ်ႉတိုဝ်း indexing လႄႈ သင်ႁဝ်းၶႆႈဢဝ် ၵႃႊၶၼ်ၵမ်းတူဝ် ဢၼ်မီးဝႆႉၼႂ်းမၼ်းၼႆၸိုင် ႁဝ်းၸႂ်ႉ indexing လႆႈဢိူဝ်ႈ။ Indexing ၼႆႉ ပဵၼ်ပိူင် ဢၼ်ဝႃႈ မၢႆတေႇ ပဵၼ်မၢႆသူၼ် ၼႆယဝ်ႉ။ ၵွပ်ႈၼႆ သင်ႁဝ်းၶူင်သၢင်ႈဝႆႉ List ဢၼ်ၼိုင်ႈသေ သင်သမ်ႉ ၶႆႈဢဝ် ၵႃႊၶၼ် ဢၼ်မီးဝႆႉ ဢွၼ်တၢင်းသုတ်ႉၼႆၸိုင် ႁဝ်းတေလႆႈတေႇတီႈ မၢႆသုၼ်ယဝ်ႉ။

```
thislist = ["apple", "banana", "cherry"]  
print(thislist[1]) # banana
```

တီႈ python ၼႆႉႁဝ်းၸႂ်ႉလႆႈ negative indexing သေဢိူဝ်ႈ။ မၼ်းတေလုၵ်ႉတီႈ တူဝ်လိုၼ်းသုတ်ႉသေ ပၼ်ႁဝ်း ၵႃႊၶၼ်ဢၼ်မီးဝႆႉ တီႈၼႂ်း List ႁဝ်းယဝ်ႉ။

```
print(thislist[-1]) # cherry
```

**Change Item Value**

သင်ႁဝ်း ၶႆႈလၢႆႊ ၵႃႊၶၼ် ဢၼ်မီးဝႆႉ တီႈၼႂ်း List ၼႆၸိုင် ႁဝ်းၸႂ်ႉ indexing လႄႈ assignment operator ၸွမ်းၵၼ်ယဝ်ႉ။ ၶူတ်ႉတီႈတႂ်ႈၼႆႉ မိူဝ်ႈတေႇ index 1 ၼႆႉ မၼ်းပဵၼ်ဝႆႉ banana သေတႃႉ သင်ႁဝ်း run ယဝ်ႉမူတ်းၼႆ မၼ်းတေပဵၼ်ၵႂႃႇ blackcurrant ယဝ်ႉ။

```
thislist = ["apple", "banana", "cherry"]  
thislist[1] = "blackcurrant"
```

**Add List Item**

သင်ႁဝ်း တေထႅမ်သႂ်ႇ ၵႃႊၶၼ်မႂ်ႇ တီႈၼႂ်း List ၼႆၸိုင် ႁဝ်းတေလႆႈၸႂ်ႉ append() method ဢၼ်မီးဝႆႉ တီႈၼႂ်း List ယဝ်ႉ။ သင်ၼၼ် မၼ်းတေသႂ်ႇပၼ် ၵႃႊၶၼ်မႂ်ႇ ၵႂႃႇတီႈ ႁွင်ႈလိုၼ်းသုတ်ႉ ယဝ်ႉဢိူဝ်ႈ။

```
thislist = ["apple", "banana", "cherry"]  
thislist.append("orange")
```

သင်ႁဝ်းသမ်ႉ ၶႆႈသႂ်ႇ ၵႃႊၶၼ်တီႈ index ​သေတီႈတီႈ ဢၼ်ဢမ်ႇၸႂ် လိုၼ်းသုတ်ႉပိူၼ်ႈၼႆၸိုင် ႁဝ်းၶိုၼ်း ၸႂ်ႉတိုဝ်းလႆႈ insert() method ဢိူဝ်ႈ။ ၼႂ်းမၼ်း မၼ်းတေမီးဝႆႉ parameter သွင်ဢၼ်လႄႈ ဢၼ်ၼိုင်ႈ ပဵၼ် index သေ ထႅင်ႈဢၼ်ၼိုင်ႈတေႉ ပဵၼ်ၵႃႊၶၼ်မၼ်းယဝ်ႉ။

```
thislist = ["apple", "banana", "cherry"]  
thislist.insert(1, "orange")
```

သင်ႁဝ်းသမ်ႉ ၶႆႈဢဝ် List သွင်ဢၼ် ႁူမ်ႈၵၼ်ၼႆၸိုင် ႁဝ်းၶိုၼ်းၸႂ်ႉလႆႈ extend() method ဢိူဝ်ႈ။

```
thislist = ["apple", "banana", "cherry"]  
tropical = ["mango", "pineapple", "papaya"]  
thislist.extend(tropical)
```

**Remove List Item**

သင်ႁဝ်း ၶႆႈထွၼ်ဢွၵ်ႇ ၵႃႊၶၼ်ၵဝ်ႇ ဢၼ်မီးဝႆႉ တီႈၼႂ်း List ႁဝ်းၼႆၸိုင် ႁဝ်းတေၸႂ်ႉလႆႈ 3 လၢႆးၼႆႉ။

**(1) ထွၼ်ဢွၵ်ႇလူၺ်ႈၵႃႊၶၼ်**

လွင်ႈၼႆႉ ႁဝ်းၸႂ်ႉတိုဝ်း remove() method ဢိူဝ်ႈ။ ၼႂ်းမၼ်း ႁဝ်းၶိုၼ်းသႂ်ႇပၼ် ၵႃႊၶၼ် ဢၼ်ႁဝ်းၶႆႈထွၼ်ယဝ်ႉ။ မၼ်းတေ ၵႂႃႇသွၵ်ႈႁႃသေ သင်ႁၼ်ၼႆၵေႃႈ တေထွၼ်ပၼ်ယဝ်ႉ။ ၶူတ်ႉတႂ်ႈၼႆႉ မၼ်းတေထွၼ်ဢွၵ်ႇပႅတ်ႈ banana ယဝ်ႉ။ ၵွပ်ႈၼႆ သင်ၸႅတ်ႈတူၺ်းၶိုၼ်း ၵႃႊၶၼ်ၼႂ်းမၼ်းၼႆ banana ၼႆႉ တေဢမ်ႇႁၼ်ယဝ်ႉ။

```
thislist = ["apple", "banana", "cherry"]  
thislist.remove("banana")
```

**(2) ထွၼ်ဢွၵ်ႇလူၺ်ႈ index**

လွင်ႈၼႆႉသမ်ႉ ႁဝ်းၵႂႃႇတီႈ index ဢၼ်လႆႈသႂ်ႇဝႆႉပၼ်သေၵေႃႈ ထွၼ်ဢွၵ်ႇပႅတ်ႈ ၵႃႊၶၼ်မၼ်းယဝ်ႉ။ တီႈၼႆႈ ႁဝ်းၸႂ်ႉတိုဝ်း pop() method သေ ၼႂ်းမၼ်းၼၼ်ႉ ႁဝ်းၶိုၼ်းသႂ်ႇပၼ် index မၼ်းယဝ်ႉဢိူူဝ်ႈ။

```
thislist = ["apple", "banana", "cherry"]  
thislist.pop(1)
```

**(3) ထွၼ်ဢွၵ်ႇ ၵႃႊၶၼ် တၢင်းမူတ်း**

ၼႆႉတေႉ ႁဝ်းၶိုၼ်းၸႂ်ႉ clear() method ယဝ်ႉ။ သင်ၼၼ် ၵႃႊၶၼ်တၢင်းမူတ်း ဢၼ်မီးဝႆႉ တီႈၼႂ်း List ၼၼ်ႉ မၼ်းတေဢဝ်ပႅတ်ႈမူတ်းယဝ်ႉ။

```
thislist = ["apple", "banana", "cherry"]  
thislist.clear()
```
