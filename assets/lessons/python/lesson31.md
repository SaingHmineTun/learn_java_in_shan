
## 31) For Loop Statements

For ၼႆႉၵေႃႈ ပဵၼ် Looping ဢၼ်ၼိုင်ႈၵူၺ်းၼင်ႇၵဝ်ႇ။ ႁဝ်းၸႂ်ႉမၼ်း တႃႇတေ loop ဢၼ်ပဵၼ် sequence data type မိူၼ်ၼင်ႇ **list, tuple, dictionary, set, string** ၸိူဝ်းၼႆႉယဝ်ႉ။ ၵွပ်ႈၼႆ မၼ်းတေ လၢၵ်ႇလၢႆးၵၼ်တင်း for loop ဢၼ်မီးဝႆႉတီႈ programming languages တၢင်ႇဢၼ်ယူႇ။ ၵူၺ်း မၼ်းမိူၼ်ၵၼ်တင်း foreach တီႈ C# လႄႈ enhanced for တီႈ Java ယဝ်ႉ။ ၶူတ်ႉတီႈတႂ်ႈၼႆႉ ပဵၼ်ၶူတ်ႉ ဢၼ်တေဢွၵ်ႇၼႄပၼ် ၵႃႊၶၼ် ၵူႊဢၼ် ဢၼ်မီးဝႆႉတီႈၼႂ်း fruits list ၼႆႉယဝ်ႉ။

```
fruits = ["apple", "banana", "cherry"]  
for x in fruits:  
    print(x)
```

ႁဝ်းၸႂ်ႉတိုဝ်း လၢႆးတီႈၼိူဝ်ၼႆႉသေ loop လႆႈ sequence ယဝ်ႉ။ ယဝ်ႉၵေႃႈ သင်ၶႆႈၸႂ်ႉတိုဝ်း **break**, **continue, else, pass** ၵေႃႈ ၸႂ်ႉလႆႈယူႇ။

**Then range() function**

Function ၼႆႉ မၼ်းတေႁပ်ႉၶၢမ်ႇဝႆႉ parameter သေ ၼႂ်းမၼ်းႁဝ်းတေသႂ်ႇလႆႈ တူဝ် int ယဝ်ႉ။ သင်သႂ်ႇဝႆႉ တူဝ်လဵဝ်ၼႆ ပဵၼ်တီႈၵိုတ်းမၼ်းယဝ်ႉ။ ၵူၺ်း မၼ်းပဵၼ် exclusive လႄႈ မၼ်းတေဢမ်ႇၵိုၵ်းပႃး တူဝ်မၼ်း။ မိူၼ်ၼင်ႇ သင်ႁဝ်းသႂ်ႇဝႆႉ 6 ၼႆ မၼ်းတေဢမ်ႇၵိုၵ်းပႃး မၢႆ 6။ ၶူတ်ႉတီႈတႂ်ႈၼႆႉ မၼ်းတေၼႄပၼ်ႁဝ်း (0,1,2,3,4,5) ၼႆယဝ်ႉ။

```
for x in range(6):  
    print(x)
```

သင်သႂ်ႇပၼ်ထႅင်ႈ parameter ဢၼ်ၼိုင်ႈၼႆ ပွင်ႇဝႃႈ မၼ်းတေ တေႇၵႂႃႇတီႈၼၼ်ႈၼႆယဝ်ႉ။ ၶူတ်ႉတီႈတႂ်ႈၼႆႉ မၼ်းတေ ဢမ်ႇတေႇတီ 0 သေ တေလုၵ်ႉတီႈ 2 သေ တေႇၵႂႃႇ။ ၵွပ်ႈၼႆ ႁဝ်းတေလႆႈႁၼ် (2,3,4,5) ၼႆယဝ်ႉ။

```
for x in range(2, 6):  
    print(x)
```
