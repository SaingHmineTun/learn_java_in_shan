
## **Java Basic 69**

#### **INTRODUCTION TO COLLECTIONS FRAMEWORK**

Link - <https://youtu.be/tMLSnvkNY6g>

**Framework** ဢၼ်ဝႃႈၼႆႉ ပဵၼ် folder ဢၼ်ၼိုင်ႈ ဢၼ်မီးဝႆႉ Java class ဢၼ်ပိူၼ်ႈႁဵတ်ႉဝႆႉ တႃႇႁဝ်းတေ ၸႂ်ႉတိုဝ်းလႆႈငၢႆႊၼႆယဝ်ႉ။

**Collections framework** ၼႆႉတေႉၵေႃႈ ပဵၼ် Framework ဢၼ်ၵိုၵ်းပႃးဝႆႉ code တႃႇၸႂ်ႉတိုဝ်းၸွမ်း Data structure & Algorithm ယဝ်ႉ။

**Data structure** ၼႆႉတေႉ ပဵၼ်လွင်ႈဢၼ် ႁဝ်းတေႁဵတ်ႉၵၢၼ်ၸွမ်း Data ယဝ်ႉ။ ထိုင်တေႃႈလဵဝ် ဢၼ်ႁဝ်း မီးဝႆႉ တႃႇႁဵတ်ႉၵၢၼ်ၸွမ်း Data တၢင်းၼမ်ၼႆႉတေႉ ပဵၼ် Array ၵူၺ်း။ ၵူၺ်းၵႃႊ မၼ်းမီး လွင်ႈယုပ်ႈယွမ်း ဢၼ်ဝႃႈ ပေႃးႁဝ်းၶူင်သၢင်ႈ Array ယဝ်ႉၼႆၸိုင် တၢင်းယႂ်ႇမၼ်းၼၼ်ႉ ႁဝ်းတေထႅမ်မႂ်ႇ၊ ဢမ်ႇၼၼ်ၵေႃႈ ထွၼ်ၵဝ်ႇ ၼႆႉဢမ်ႇလႆႈယဝ်ႉ။ ၵွပ်ႈၼႆ ပေႃးသမ်ႉ ႁဝ်းၶႆႈႁဵတ်ႉၵၢၼ်ၸွမ်း Data တၢင်းၼမ်သေ ဢမ်ႇႁူႉဝႃႈ Data ႁဝ်းၼႆႉ မီးၵႃႊႁိုဝ်ၼႆ ႁဝ်းတေလႆႈၸႂ်ႉတိုဝ်း Collections framework ယဝ်ႉ။

**Algorithm** ၼႆႉတေႉၵေႃႈ ပဵၼ်လွင်ႈၸီႉသင်ႇ ႁႂ်ႈႁဵတ်ႉၵၢၼ်သေဢၼ်ဢၼ် ႁႂ်ႈထိုင်တီႈၵူၺ်း။ မိူၼ်ၼင်ႇ သင်ႇၶႃႈၶႆႈသူင်ႇလိၵ်ႈ ၸူးတႆးၵေႃႉ ၼိူဝ် Messenger ၼႆ Algorithm မၼ်းတေပဵၼ်

- ၶဝ်ႈၼႂ်း Messenger
- ႁႃၸိုဝ်ႈတႆးၵေႃႉ ဢၼ်ႁဝ်းၶႆႈလၢတ်ႈတေႃႇ
- ၼဵၵ်ႉၶဝ်ႈၼႂ်း Messenger မၼ်း
- တႅမ်ႈလိၵ်ႈတီႈ ႁွင်ႈတႅမ်ႈလိၵ်ႈ
- ၼဵၵ်ႉပၼ် Send

Algorithm ဢၼ်ၸႂ်ႉတိုဝ်းၸွမ်း Data structure ၵမ်ႈၼၼ်တေႉ ပဵၼ် Sorting (Merge Sort, Bubble Sort, Insertion Sort) လႄႈ Searching (Binary search, Linear search) ယဝ်ႉ။

တီႈ Java ၼႆႉ Data structure တၢင်းမူတ်း ဢၼ်ႁဝ်းတေ ႁွင်ႉၸႂ်ႉလႆႈၼၼ်ႉ မီးဝႆႉတီႈ java.util ယဝ်ႉ။ ၼႂ်းၼၼ်ႉ တေမီးဝႆႉယူႇ interface ဢၼ်ၸႂ်ႉတိုဝ်း ၼမ်ဝႆႉ (3) လွင်ႈ။

- **List interface**
- **Map interface**
- **Set interface**
