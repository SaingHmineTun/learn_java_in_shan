## Lesson 10: Shan Text Styling

ပေႃးႁဝ်းၸႂ်ႉ CSS ပိူင်ၵဝ်ႇ (Default) ၼႆ မၼ်းတေႁဵတ်းႁႂ်ႈ တူဝ်လိၵ်ႈတႆးႁဝ်း "ၶတ်းၵၼ်" ဢမ်ႇၼၼ် "သွၼ်ႉၵၼ်" ယဝ်ႉ။ 
လွင်ႈၼႆႉ ႁဝ်းၸၢင်ႈၸႂ်ႉတိုဝ်း Property ၸိူဝ်းၼႆႉလႄႈ ၵႄႈလိတ်ႈယဝ်ႉ:

### 1. Line Height (တၢင်းသုင်ထႅဝ်လိၵ်ႈ)

ၼႆႉပဵၼ် Property ဢၼ်လမ်ႇလွင်ႈသုတ်း တွၼ်ႈတႃႇလိၵ်ႈတႆးယဝ်ႉ။ 
ပေႃးႁဝ်းဢမ်ႇပၼ် ဢွင်ႈပဝ်ႇတၢင်းၼိူဝ်/တၢင်းတႂ်ႈ ႁႂ်ႈၵုမ်ႇထူၼ်ႈၼႆ မၢႆသဵင်တၢင်းၼိူဝ် တေၵႂႃႇၸပ်ႉမၢႆသဵင်တၢင်းတႂ်ႈသေ တေဢမ်ႇလီတူၺ်း။

* **လၢႆးတႅမ်ႈ:**

```css
p {
  line-height: 1.8;
}

```

---

### 2. Letter Spacing (ဢွင်ႈလွၵ်ႇတူဝ်လိၵ်ႈ)

လိၵ်ႈတႆးႁဝ်းၼၼ်ႉ ပေႃးတူဝ်လိၵ်ႈ ၸပ်းၵၼ်ပူၼ်ႉတီႈ မၼ်းတေလူယၢပ်ႇ။ ႁဝ်းၸၢင်ႈပၼ် တၢင်းႁၢင်ႇၵၼ် ၼႂ်းၵႄႈတူဝ်လိၵ်ႈ ယဝ်ႉ။

* **လၢႆးတႅမ်ႈ:**

```css
p {
  letter-spacing: 0.05em;
}

```

---

### 3. Text Alignment (လွင်ႈၸတ်းထႅဝ်လိၵ်ႈ)

ႁဝ်းၸၢင်ႈမၵ်းမၼ်ႈ ႁႂ်ႈလိၵ်ႈႁဝ်း ယူႇၽၢႆႇသၢႆႉ/ၶႂႃ/ၵၢင် လႆႈၼင်ႇတီႈတႂ်ႈယဝ်ႉ။

* `text-align: left;` - တႄႇတၢင်းသၢႆႉ (ပဵၼ် default)။
* `text-align: center;` - ဝႆႉၵႄႈၵၢင်။
* `text-align: justify;` - ၸတ်းႁႂ်ႈသွင်ၶၢင်ႈၽဵင်ႇၵၼ်။

---

### 4. Text Decoration & Transform

* `text-decoration: none;` - ၸႂ်ႉတႃႇဢဝ် "သဵၼ်ႈတႂ်ႈ" လိၵ်ႈဢွၵ်ႇ (သူင်ၸႂ်ႉတီႈ Link)။
* `font-weight: bold;` - ႁဵတ်းႁႂ်ႈတူဝ်လိၵ်ႈၼႃ။

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **Line-height:** တွၼ်ႈတႃႇလိၵ်ႈတႆး လူဝ်ႇပၼ်ဝႆႉ 1.8 ၶိုၼ်ႈၼိူဝ် ႁႂ်ႈမၢႆသဵင်ဢမ်ႇၶတ်းၵၼ်။
* **Letter-spacing:** ၸွႆႈႁႂ်ႈလိၵ်ႈတႆး ဢမ်ႇၸပ်းၵၼ်ပူၼ်ႉတီႈ။
* **Text-align:** ၸတ်းဝၢင်းလိၵ်ႈ ႁႂ်ႈႁၢင်ႈလီ။
