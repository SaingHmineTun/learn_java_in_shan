## Lesson 10: Shan Text Styling

ပေႃးႁဝ်းၸႂ်ႉ CSS ပိူင်ၵဝ်ႇ (Default) ၼၼ်ႉ မၼ်းတေႁဵတ်းႁႂ်ႈ မၢႆသဵင်လိၵ်ႈတႆးႁဝ်း "ၶတ်းၵၼ်" ဢမ်ႇၼၼ် "ထိုင်ၵၼ်" ၵႂႃႇၶႃႈ။ ႁဝ်းမႃးၵႄႈလႅတ်းလူၺ်ႈ Property ၸိူဝ်းၼႆႉၶႃႈ:

### 1. Line Height (တၢင်းသုင်ထႅဝ်လိၵ်ႈ)

ၼႆႉပဵၼ် Property ဢၼ်လမ်ႇလွင်ႈသုတ်း တွၼ်ႈတႃႇလိၵ်ႈတႆးၶႃႈ။ ပေႃးႁဝ်းဢမ်ႇပၼ်ဢွင်ႈလွၵ်ႇတၢင်းၼိူဝ်/တၢင်းတႂ်ႈႁႂ်ႈၵုမ်ႇထူၼ်ႈၼၼ်ႉ မၢႆသဵင်တၢင်းၼိူဝ် တေၵႂႃႇတပ်ႉမၢႆသဵင်တၢင်းတႂ်ႈ ၶွင်ထႅဝ်လိၵ်ႈၼိူဝ်မၼ်းၶႃႈ။

* **လၢႆးတႅမ်ႈ:** ၸႂ်ႉတူဝ်ၼပ်ႉ 1.8 တေႃႇ 2.0 ၶႃႈ။

```css
p {
  line-height: 1.8; /* ပၼ်ဢွင်ႈလွၵ်ႇ 1.8 ပၼ်ႇ ၶွင်ၶၼၢတ်ႈလိၵ်ႈ */
}

```

---

### 2. Letter Spacing (ဢွင်ႈလွၵ်ႇတူဝ်လိၵ်ႈ)

လိၵ်ႈတႆးႁဝ်းၼၼ်ႉ ပေႃးတူဝ်လိၵ်ႈၸုၵ်းၸိတ်းၵၼ်ပူၼ်ႉတီႈ မၼ်းတေလူယၢပ်ႇၶႃႈ။ ႁဝ်းၸၢင်ႈပၼ်ဢွင်ႈလွၵ်ႇ ၼႂ်းၵႄႈတူဝ်လိၵ်ႈ ဢိတ်းဢီတ်းၶႃႈ။

* **လၢႆးတႅမ်ႈ:**

```css
p {
  letter-spacing: 0.05em; /* ပၼ်ဢွင်ႈလွၵ်ႇ ဢိတ်းဢီတ်း ႁႂ်ႈလူငၢႆႈ */
}

```

---

### 3. Text Alignment (လွင်ႈၸတ်းထႅဝ်လိၵ်ႈ)

ႁဝ်းၸၢင်ႈမၵ်းမၼ်ႈ လွင်ႈၸုၵ်းၶွင်လိၵ်ႈႁဝ်း ၼႂ်း "ႁပ်း" (Box) ၶႃႈ။

* `text-align: left;` - တႄႇတၢင်းသၢႆႉ (ပိူင်ၵဝ်ႇ)။
* `text-align: center;` - ဝႆႉၵႄႈၵၢင် (တွၼ်ႈတႃႇႁူဝ်လိၵ်ႈ)။
* `text-align: justify;` - ၸတ်းႁႂ်ႈသွင်ၶၢင်ႈၽဵင်ႇၵၼ် (တွၼ်ႈတႃႇလိၵ်ႈဢၼ်ၼမ်)။

---

### 4. Text Decoration & Transform

* `text-decoration: none;` - ၸႂ်ႉတႃႇဢဝ် "သဵၼ်ႈတႂ်ႈ" လိၵ်ႈဢွၵ်ႇ (သူင်ၸႂ်ႉတီႈ Link)။
* `font-weight: bold;` - ႁဵတ်းႁႂ်ႈတူဝ်လိၵ်ႈၼႃ။

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **Line-height:** တွၼ်ႈတႃႇလိၵ်ႈတႆး လူဝ်ႇပၼ်ဝႆႉ 1.8 ၶိုၼ်ႈၼိူဝ် ႁႂ်ႈမၢႆသဵင်ဢမ်ႇၶတ်းၵၼ်။
* **Letter-spacing:** ၸွႆႈႁႂ်ႈလိၵ်ႈတႆး ဢမ်ႇၸိတ်းၵၼ်ပူၼ်ႉတီႈ။
* **Text-align:** ၸတ်းဝၢင်းလိၵ်ႈ ႁႂ်ႈႁၢင်ႈလီ။
