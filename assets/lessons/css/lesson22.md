## Lesson 22: Media Query Syntax

Media Query ပဵၼ်ၵၢၼ်တႅမ်ႈ "ၶေႃႈမၵ်းမၼ်ႈ" (Condition) ၼႂ်း CSS။ မၼ်းတေၸူၵ်းတူၺ်း တၢင်းၵႂၢင်ႈၼႃႈจอ (Screen Width) သေ ၸင်ႇတေႁဵတ်းၵၢၼ်ၶႃႈ။

### 1. ပိူင်တႅမ်ႈပိုၼ်ႉထၢၼ် (Syntax)

ႁဝ်းတေၸႂ်ႉ `@media` တႄႇၼႃႈသေ သင်ႇဝႃႈ ၶႂ်ႈႁႂ်ႈ CSS ၼႂ်းဝူင်းလဵၵ်းၼၼ်ႉ ႁဵတ်းၵၢၼ်မိူဝ်ႈလႂ်။

**တူဝ်ယၢင်ႇ:**

```css
/* CSS ပၵ်းၵဝ်ႇ တႃႇ Computer */
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr; /* မီး 3 ထႅဝ်ၸုၵ်း */
}

/* @media: သင်ဝႃႈ ၼႃႈจอ ၵႅပ်ႈလိူဝ် 768px (မိူၼ်ၼင်ႇ Tablet/Phone) */
@media (max-width: 768px) {
  .container {
    grid-template-columns: 1fr; /* ႁႂ်ႈၵိုတ်း 1 ထႅဝ်ၸုၵ်းၵွၺ်း ႁႂ်ႈလူငၢႆႈ */
  }
}

```

---

### 2. ၶေႃႈၵႂၢမ်းလမ်ႇလွင်ႈ

* **max-width:** "ဢမ်ႇပၼ်ႁႂ်ႈၵႂၢင်ႈလိူဝ်..." (ၸႂ်ႉတႃႇမၵ်းမၼ်ႈ CSS တွၼ်ႈတႃႇ Screen ဢၼ်လဵၵ်ႉၵႂႃႇ)။
* **min-width:** "တႄႇတၢင်းၵႂၢင်ႈတီႈ..." (ၸႂ်ႉတႃႇမၵ်းမၼ်ႈ CSS တွၼ်ႈတႃႇ Screen ဢၼ်ယႂ်ႇၶိုၼ်ႈမႃး)။

---

### 3. တူဝ်ယၢင်ႇလွင်ႈၸႂ်ႉတိုဝ်း တႃႇလိၵ်ႈတႆး

ၼိူဝ် Computer ႁဝ်းၸၢင်ႈၸႂ်ႉ Font တူဝ်လဵၵ်ႉဢိတ်းၼိုင်ႈလႆႈ၊ ၵွၺ်းၵႃႈၼိူဝ် Phone ႁဝ်းလူဝ်ႇႁႂ်ႈမၼ်းယႂ်ႇ ႁႂ်ႈၵူၼ်းထဝ်ႈၶဝ်လူငၢႆႈၶႃႈ။

```css
/* ပိူင်ၵဝ်ႇ (Desktop) */
body {
  font-size: 18px;
}

/* တွၼ်ႈတႃႇ Mobile ( Screen ဢၼ်လဵၵ်ႉလိူဝ် 480px ) */
@media (max-width: 480px) {
  body {
    font-size: 22px; /* ႁဵတ်းႁႂ်ႈလိၵ်ႈတႆး ယႂ်ႇၶိုၼ်ႈမႃးၼိူဝ်ၾူၼ်း */
    background-color: #f9f9f9;
  }
}

```

---

### 4. တူဝ်ၸွႆႈထႅင်ႈ (And, Not, Only)

ႁဝ်းၸၢင်ႈလေႃးၶေႃႈမၵ်းမၼ်ႈလႆႈၼမ်လိူဝ် 1 ဢၼ်ၶႃႈ:

```css
/* တွၼ်ႈတႃႇ Tablet ဢၼ်မီးတၢင်းၵႂၢင်ႈၼႂ်ႈၵႄႈ 481px တေႃႇ 768px */
@media (min-width: 481px) and (max-width: 768px) {
  /* CSS တႃႇ Tablet ၵွၺ်း */
}

```

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **@media:** ပဵၼ်ၵၢၼ်တႄႇတႅမ်ႈ Responsive CSS။
* **Breakpoints:** ပဵၼ်တူဝ်ၼပ်ႉ (မိူၼ်ၼင်ႇ 768px) ဢၼ်ႁဝ်းလိူၵ်ႈလႅၵ်ႈလၢႆႈ Design။
* **Flexibility:** ၸွႆႈႁႂ်ႈဝႅပ်ႉသၢႆႉႁဝ်း "လႅတ်း" သေ လႅၵ်ႈလၢႆႈတူဝ်မၼ်းၸွမ်း Device ၵူႈပိူင်။
