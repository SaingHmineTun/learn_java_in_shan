
## Lesson 5: Box Model Core

ၼႂ်း CSS ၼၼ်ႉ ႁဝ်းတေလႆႈတူၺ်း Elements တင်းသဵင်ႈ (မိူၼ်ၼင်ႇ h1, p, div) မိူၼ်မၼ်းပဵၼ်ဝႆႉ **"(Box)"** ဢၼ်မီးသီႇၸဵင်ႇယဝ်ႉ။ 
ပေႃးႁဝ်းပွင်ႇၸႂ်လွင်ႈ "Box" ၼႆႉယဝ်ႉ ႁဝ်းတေၸၢင်ႈၸတ်းဝၢင်း Layout ႁဝ်းႁႂ်ႈၶႅမ်ႉလႅပ်ႈလႆႈၶႃႈ။

### 1. ပိူင်သၢင်ႈ 4 ၸၼ်ႉ (The 4 Layers)

Box ၼိုင်ႈဢၼ်ၼၼ်ႉ မီးဝႆႉ 4 ၸၼ်ႉ ၼင်ႇတီႈတႂ်ႈယဝ်ႉ:

1. **Content (ၼိူဝ်ႉလိၵ်ႈ):** ပဵၼ်တီႈလိၵ်ႈ ဢမ်ႇၼၼ် ႁၢင်ႈၾၢင် ဢၼ်ႁဝ်းသႂ်ႇဝႆႉ။
2. **Padding (ပႅတ်ႉတိၼ်ႇ):** ပဵၼ်ဢွင်ႈတီႈလွၵ်ႇ (Space) ဢၼ်ယူႇ **ၼႂ်း** Box။
3. **Border (ၶွပ်ႇ):** ပဵၼ်သဵၼ်ႈၶွပ်ႇ ဢၼ်ပၼ်ႇႁွပ်ႈ Padding ဝႆႉ။
4. **Margin (မႃးၵျိၼ်ႇ):** ပဵၼ်ဢွင်ႈတီႈလွၵ်ႇ ဢၼ်ယူႇ **ၼွၵ်ႈ** Box။

---

### 2. တူဝ်ယၢင်ႇလွင်ႈတႅမ်ႈ Code

```css
.box {
  width: 300px;
  padding: 20px;    /* ပၼ်ဢွင်ႈတီႈလွၵ်ႇၼႂ်း 20px */
  border: 5px solid red; /* ၶွပ်ႇသီလႅင် တၢင်းၼႃ 5px */
  margin: 50px;     /* ထႅၵ်ႇႁၢင်ႇတီႈပိူၼ်ႈ 50px */
}

```

---

### 3. လွင်ႈပႅၵ်ႇပိူင်ႈ Padding လႄႈ Margin

* **Padding:** ၸႂ်ႉမိူဝ်ႈၶႂ်ႈႁႂ်ႈ "ၼိူဝ်ႉလိၵ်ႈ" ႁဝ်း မီးဢွင်ႈတီႈႁၢင်ႇတီႈၶွပ်ႇ (Border) မၼ်း။
* **Margin:** ၸႂ်ႉမိူဝ်ႈၶႂ်ႈႁႂ်ႈ "box" ႁဝ်း ႁၢင်ႇၵႆတီႈ "box ပိူၼ်ႈ"။

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **Content** = တူဝ်လိၵ်ႈ။
* **Padding** = ဢွင်ႈလွၵ်ႇၼႂ်းၶွပ်ႇ။
* **Border** = သဵၼ်ႈၶွပ်ႇ။
* **Margin** = ဢွင်ႈလွၵ်ႇၼွၵ်ႈၶွပ်ႇ။
