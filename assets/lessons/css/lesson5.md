
## Lesson 5: Box Model Core

ၼႂ်း CSS ၼၼ်ႉ ႁဝ်းတေလႆႈတူၺ်း Elements တင်းသဵင်ႈ (မိူၼ်ၼင်ႇ h1, p, div) ဝႃႈ မၼ်းပဵၼ် **"ႁပ်း (Box)"** ဢၼ်မီးသီႇၸဵင်ႇၶႃႈ။ ပေႃးႁဝ်းပွင်ႇၸႂ်လွင်ႈ "ႁပ်း" ၼႆႉယဝ်ႉ ႁဝ်းတေၸၢင်ႈၸတ်းဝၢင်း Layout ႁဝ်းႁႂ်ႈၶႅမ်ႉလႅပ်ႈလႆႈၶႃႈ။

### 1. ပိူင်သၢင်ႈ 4 ၸၼ်ႉ (The 4 Layers)

ႁပ်း (Box) ၼိုင်ႈဢၼ်ၼၼ်ႉ မီးဝႆႉ 4 ၸၼ်ႉၶႃႈ:

1. **Content (ၼိူဝ်ႉလိၵ်ႈ):** ပဵၼ်တီႈလိၵ်ႈ ဢမ်ႇၼၼ် ႁၢင်ႈၾၢင်ႁဝ်းယူႇဝႆႉ။
2. **Padding (ပႅတ်ႉတိၼ်ႇ):** ပဵၼ်ဢွင်ႈတီႈလွၵ်ႇ (Space) ဢၼ်ယူႇ **ၼႂ်း** ႁပ်း (ယူႇၼႂ်းၵႄႈ လိၵ်ႈ လႄႈ ၶွပ်ႇ)။
3. **Border (ၶွပ်ႇ):** ပဵၼ်သဵၼ်ႈၶွပ်ႇ ဢၼ်ပၼ်ႇႁွပ်ႈ Padding ဝႆႉ။
4. **Margin (မႃးၵျိၼ်ႇ):** ပဵၼ်ဢွင်ႈတီႈလွၵ်ႇ ဢၼ်ယူႇ **ၼွၵ်ႈ** ႁပ်း (ပဵၼ်ဢၼ်ထႅၵ်ႇႁပ်းၼိုင်ႈဢၼ် တေႃႇ ႁပ်းထႅင်ႈၼိုင်ႈဢၼ်)။

---

### 2. တူဝ်ယၢင်ႇလွင်ႈတႅမ်ႈ Code

```css
.box {
  width: 300px;
  padding: 20px;    /* ပၼ်ဢွင်ႈတီႈလွၵ်ႇၼႂ်း 20px */
  border: 5px solid red; /* ၶွပ်ႇသီလႅင်ၼႃ 5px */
  margin: 50px;     /* ထႅၵ်ႇႁၢင်ႇတီႈပိူၼ်ႈ 50px */
}

```

> **ၶေႃႈၼႄႉၼမ်းတႃႇလိၵ်ႈတႆး:**
> လိၵ်ႈတႆးႁဝ်းၼၼ်ႉ မီးမၢႆသဵင် (Tone marks) လႄႈ သရႃႉ (Vowels) ဢၼ်ယူႇတၢင်းၼိူဝ်လႄႈတၢင်းတႂ်ႈၼမ်။ ပေႃးႁဝ်းသႂ်ႇ **Padding** ႁႂ်ႈမီးဢွင်ႈလွၵ်ႇၵုမ်ႇထူၼ်ႈၼၼ်ႉ မၼ်းတေႁဵတ်းႁႂ်ႈလိၵ်ႈတႆးႁဝ်း လူင်ႆႈ လႄႈ တူၺ်းလီၶႃႈ။

---

### 3. လွင်ႈပႅၵ်ႇပိူင်ႈ Padding လႄႈ Margin

* **Padding:** ၸႂ်ႉမိူဝ်ႈၶႂ်ႈႁႂ်ႈ "ၼိူဝ်ႉလိၵ်ႈ" ႁဝ်း မီးဢွင်ႈတီႈႁၢင်ႇတီႈၶွပ်ႇ (Border) မၼ်း။ (မိူၼ်ၼင်ႇ ႁဝ်းၼုင်ႈသိူဝ်ႈလူင်လူင် ႁႂ်ႈမၼ်းမီးဢွင်ႈလွၵ်ႇၼႂ်းၼၼ်ႉၶႃႈ)
* **Margin:** ၸႂ်ႉမိူဝ်ႈၶႂ်ႈႁႂ်ႈ "ႁပ်း" ႁဝ်း ႁၢင်ႇၵႆတီႈ "ႁပ်းပိူၼ်ႈ"။ (မိူၼ်ၼင်ႇ ႁဝ်းၸုၵ်းႁၢင်ႇၵၼ် တႃႇလူတ်းယွမ်းလွင်ႈသုၵ်ႉသၵ်ႉၼၼ်ႉၶႃႈ)

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **Content** = တူဝ်လိၵ်ႈ။
* **Padding** = ဢွင်ႈလွၵ်ႇၼႂ်းၶွပ်ႇ။
* **Border** = သဵၼ်ႈၶွပ်ႇ။
* **Margin** = ဢွင်ႈလွၵ်ႇၼွၵ်ႈၶွပ်ႇ။
