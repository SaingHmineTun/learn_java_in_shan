
## Lesson 2: Linking Methods

မိူဝ်ႈႁဝ်းတႅမ်ႈ CSS ယဝ်ႉယဝ်ႉၼၼ်ႉ ႁဝ်းတေလႆႈၵပ်းသိုပ်ႇ (Link) မၼ်းသႂ်ႇၼႂ်း HTML။ လွင်ႈၵပ်းသိုပ်ႇၼႆႉ မီးယူႇ 3 ပိူင်ၶႃႈ:

### 1. Inline Styles (တႅမ်ႈသႂ်ႇၼႂ်း Tag)

ပဵၼ်လွင်ႈတႅမ်ႈ CSS သႂ်ႇဝႆႉၼႂ်း HTML Tag ၵမ်းသိုဝ်ႈ။ ႁဝ်းၸႂ်ႉ Attribute ဢၼ်ႁွင်ႉဝႃႈ `style` ၼၼ်ႉၶႃႈ။

* **တူဝ်ယၢင်ႇ:**

```html
<p style="color: red;">လိၵ်ႈတႆးသီလႅင်</p>

```

> **မၢႆတွင်း:** ပိူင်ၼႆႉ ႁဝ်းဢမ်ႇသူႈၸႂ်ႉၶႃႈ၊ ပေႃးမီးလိၵ်ႈၼမ် တေႁဵတ်းႁႂ်ႈ Code ႁဝ်းသုၵ်ႉသၵ်ႉ။

---

### 2. Internal CSS (တႅမ်ႈသႂ်ႇၼႂ်း Head)

ပဵၼ်လွင်ႈတႅမ်ႈ CSS သႂ်ႇဝႆႉၼႂ်း Tag `<style>` ဢၼ်ယူႇၼႂ်းတွၼ်ႈ `<head>` ၶွင် HTML ၼၼ်ႉၶႃႈ။

* **တူဝ်ယၢင်ႇ:**

```html
<head>
  <style>
    p {
      color: blue;
    }
  </style>
</head>

```

> **မၢႆတွင်း:** ၶႅမ်ႉဝႆႉတွၼ်ႈတႃႇ Web Page ၼႃႈလဵဝ်။ ၵွၺ်းၵႃႈ ပေႃးဝႅပ်ႉသၢႆႉႁဝ်းမီးၼႃႈလိၵ်ႈၼမ် တေယၢပ်ႇတွၼ်ႈတႃႇမႄးၶႃႈ။

---

### 3. External CSS (တႅမ်ႈသႂ်ႇၾၢႆႇလွင်ႈလွင်ႈ)

ဢၼ်ၼႆႉပဵၼ် **ပိူင်ဢၼ်လီသုတ်း (Best Practice)** ၶႃႈ။ ႁဝ်းတႅမ်ႈ CSS ဝႆႉၼႂ်းၾၢႆႇ (File) ဢၼ်မီးသုတ်းသဵၼ်ႈဝႃႈ `.css` (တူဝ်ယၢင်ႇ: `style.css`) ယဝ်ႉၵွၺ်းဢဝ်မႃး Link သႂ်ႇၼႂ်း HTML ၶႃႈ။

* **ၶၼ်တွၼ်ႈတႅမ်ႈ:**

1. သၢင်ႈၾၢႆႇ `style.css` ယဝ်ႉတႅမ်ႈ Code CSS သႂ်ႇ။
2. ၼႂ်း HTML (တွၼ်ႈ `<head>`), သႂ်ႇ Tag `<link>` ၼင်ႇၼႆ:

```html
<head>
  <link rel="stylesheet" href="style.css">
</head>

```

---

### ၵွပ်ႈသင် External CSS သမ်ႉလီသုတ်း?

* **Reusable:** တႅမ်ႈ CSS ၾၢႆႇလဵဝ် ၸႂ်ႉလႆႈတင်းဝႅပ်ႉသၢႆႉ (မီး 10 ၼႃႈၵေႃႈၸႂ်ႉၾၢႆႇလဵဝ်ၵွၺ်း)။
* **Clean Code:** ႁဵတ်းႁႂ်ႈ HTML ႁဝ်းလူင်ႆႈ၊ ဢမ်ႇသုၵ်ႉသၵ်ႉ။
* **Easy to Update:** မိူဝ်ႈၶႂ်ႈလႅၵ်ႈလၢႆႈသီဝႅပ်ႉသၢႆႉ တႅမ်ႈလႅၵ်ႈလၢႆႈတီႈၾၢႆႇ CSS တီႈလဵဝ်ၵေႃႈ ယဝ်ႉတူဝ်ႈယဝ်ႉၶႃႈ။

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **Inline:** တႅမ်ႈသႂ်ႇတီႈ Tag (ဢမ်ႇသူႈၸႂ်ႉ)။
* **Internal:** တႅမ်ႈသႂ်ႇၼႂ်း `<style>` (ၸႂ်ႉတွၼ်ႈတႃႇၼႃႈလဵဝ်)။
* **External:** တႅမ်ႈသႂ်ႇၼႂ်းၾၢႆႇ `.css` လွင်ႈလွင်ႈ (**လီသုတ်း**)။
