## Lesson 25: CSS Transitions

Transition ပွင်ႇဝႃႈ "ၵၢၼ်ၶၢႆႉလႅၵ်ႈ" ၶႃႈ။ မၼ်းတေၸွႆႈႁႂ်ႈမိူဝ်ႈႁဝ်းဢဝ် Mouse ၵႂႃႇၸီႉ (Hover) တီႈပုမ်ႇ ဢမ်ႇၼၼ် တီႈလိၵ်ႈၼၼ်ႉ မၼ်းမီးလွင်ႈတူင်ႉၼိုင် ဢၼ်လီတူၺ်းၶႃႈ။

### 1. ပိုၼ်ႉထၢၼ် 4 ယိူင်ႈ (The 4 Properties)

တွၼ်ႈတႃႇတေႁဵတ်း Transition လႆႈၼၼ်ႉ ႁဝ်းလူဝ်ႇမၵ်းမၼ်ႈ 4 ယိူင်ႈၼႆႉၶႃႈ:

1. **transition-property:** ၶႂ်ႈႁႂ်ႈသင်လႅၵ်ႈလၢႆႈ? (မိူၼ်ၼင်ႇ `background-color`, `width`)
2. **transition-duration:** ၶႂ်ႈႁႂ်ႈယၢဝ်းၵေႇႁိုဝ်? (မိူၼ်ၼင်ႇ `0.3s`, `1s`)
3. **transition-timing-function:** ၶႂ်ႈႁႂ်ႈမၼ်းလႅၼ်ႈၸိူင်ႉႁိုဝ်? (`ease`, `linear`, `ease-in-out`)
4. **transition-delay:** ၶႂ်ႈႁႂ်ႈမၼ်းပႂ်ႉၵေႇႁိုဝ် ၸင်ႇတႄႇတူင်ႉၼိုင်?

---

### 2. လၢႆးတႅမ်ႈပွတ်း (Shorthand Syntax)

ႁဝ်းဢမ်ႇလူဝ်ႇတႅမ်ႈ 4 ထႅဝ်ၵေႃႈလႆႈၶႃႈ၊ ႁဝ်းၸၢင်ႈတႅမ်ႈလေႃးၵၼ်ၼႂ်းထႅဝ်လဵဝ်:

```css
.button {
  background-color: #AA0000;
  color: white;
  /* သင်ႇဝႃႈ: ႁႂ်ႈၵူႈယိူင်ႈ (all) လႅၵ်ႈလၢႆႈၼႂ်း 0.3 วินาที ပိူင် ease */
  transition: all 0.3s ease;
}

.button:hover {
  background-color: #FFD700; /* သီတေၵွႆႈၵွႆႈလႅၵ်ႈပဵၼ်သီလိူင် */
  transform: scale(1.1);      /* တူဝ်တေယႂ်ႇၶိုၼ်ႈဢိတ်းၼိုင်ႈ */
}

```

---

### 3. ပိူင် Timing Functions

* **ease:** တႄႇဝႆး၊ ၵႄႈၵၢင်ဝႆး၊ လိုၼ်းသုတ်းၵွႆႈ (ပိူင်ၼႆႉ ပိူၼ်ႈၸႂ်ႉၼမ်သုတ်း)။
* **linear:** ဝႆးၽဵင်ႇၵၼ် တႄႇတေႃႇသုတ်း။
* **ease-in:** တႄႇၵွႆႈၵွႆႈ သေၸင်ႇမႃးဝႆး။
* **ease-out:** တႄႇဝႆးဝႆး သေၸင်ႇမႃးၵွႆႈ (ၸႂ်ႉတွၼ်ႈတႃႇ Elements ဢၼ်တေမႃးၵိုတ်း)။

---

### 4. တူဝ်ယၢင်ႇတႃႇလိၵ်ႈတႆး

ႁဝ်းၸၢင်ႈၸႂ်ႉ Transition တွၼ်ႈတႃႇႁဵတ်းႁႂ်ႈ လိၵ်ႈ "လႅင်း" ၶိုၼ်ႈမႃး မိူဝ်ႈၵူၼ်းလူဢဝ် Mouse မႃးၸီႉ:

```css
.shan-text {
  color: #333;
  transition: color 0.5s ease-in-out;
}

.shan-text:hover {
  color: #AA0000; /* လိၵ်ႈတႆးတေၵွႆႈၵွႆႈပဵၼ်သီလႅင် ႁၢင်ႈလီလီ */
}

```

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **Transition:** ၸွႆႈႁႂ်ႈၵၢၼ်လႅၵ်ႈလၢႆႈ Property ၼုမ်ႇၼวลၶိုၼ်ႈ။
* **Duration:** 0.2s တေႃႇ 0.5s ပဵၼ်ၶၢဝ်းယၢမ်းဢၼ်လီသုတ်း (သင်ယၢဝ်းပူၼ်ႉ ၵူၼ်းၸႂ်ႉတေဢမ်ႇၸႂ်ယၢဝ်း)။
* **Hover:** Transition ၵမ်ႈၼမ် တေၸႂ်ႉလေႃးၵၼ်တင်း `:hover`။
