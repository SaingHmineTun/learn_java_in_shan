
## Lesson 4: The Cascade

**The Cascade** ဢၼ်ဝႃႈၼၼ်ႉ မၼ်းပဵၼ် "ပိူင်ထတ်းသိၼ်" ၶွင် CSS ၶႃႈ။ ပေႃးမီး Code CSS တင်းၼမ် သင်ႇဝႆႉပႅၵ်ႇပိူင်ႈၵၼ်ၼၼ်ႉ Browser တေၸႂ်ႉပၵ်းယိူင်ႈ 3 ယိူင်ႈၼႆႉ တေႃႇထတ်းသိၼ်ဝႃႈ တေဢဝ်ဢၼ်လႂ်မႃးၸႂ်ႉၶႃႈ။

### 1. Order of Appearance (လွင်ႈၸၼ်ႉထႅဝ်)

ပေႃး Selector မိူၼ်ၵၼ်၊ ၶေႃႈၵႂၢမ်းသင်ႇပႅၵ်ႇၵၼ် ဢၼ်ယူႇ **"တၢင်းတႂ်ႈသုတ်း"** တေပဵၼ်ဢၼ်ပႄႉ (Win) ၶႃႈ။

**တူဝ်ယၢင်ႇ:**

```css
p {
  color: blue;
}

p {
  color: red; /* ဢၼ်ၼႆႉတေပႄႉ ယွၼ်ႉမၼ်းမႃးၸွမ်းလင် */
}

```

> *ၼႂ်းဝႅပ်ႉသၢႆႉႁဝ်း တေၼႄပဵၼ် သီလႅင် ၶႃႈ။*

---

### 2. Specificity (လွင်ႈတႅတ်ႈတေႃး)

ပေႃး Selector ပႅၵ်ႇၵၼ် ဢၼ်လႂ် **"တႅတ်ႈတေႃး (Specific)"** လိူဝ်ၵေႃႈ ဢၼ်ၼၼ်ႉတေပႄႉၶႃႈ။ Browser တေၼပ်ႉဢၢၼ်ႇၸၼ်ႉမၼ်းၼင်ႇၼႆ:

1. **ID (#)** - ၵတ်ႉၶႅၼ်ႇသုတ်း (ပႄႉပိူၼ်ႈတင်းသဵင်ႈ)
2. **Class (.)** - ၵတ်ႉၶႅၼ်ႇပၢၼ်ၵາງ
3. **Element (tag)** - ဢူၼ်ႈဢွၼ်ႇသုတ်း

**တူဝ်ယၢင်ႇ:**

```css
#header { color: red; }    /* ID - ၵတ်ႉၶႅၼ်ႇ 100 */
.title { color: blue; }    /* Class - ၵတ်ႉၶႅၼ်ႇ 10 */
h1 { color: green; }       /* Tag - ၵတ်ႉၶႅၼ်ႇ 1 */

```

> *သင်ဝႃႈ h1 ၼၼ်ႉမီးတင်း ID လႄႈ Class၊ သီမၼ်းတေပဵၼ် **သီလႅင်** ယွၼ်ႉ ID မီးဢဵၼ်ႁႅင်းလူင် (Specificity သုင်) ၼၼ်ႉၶႃႈ။*

---

### 3. Inheritance (လွင်ႈပူၺ်ႈပၼ်)

Property ၵမ်ႈၽွင်ႈ (မိူၼ်ၼင်ႇ `color` ဢမ်ႇၼၼ် `font-family`) တေ "ပူၺ်ႈ" ၸိူဝ်းမႃးတီႈ "ပေႃႈမႄႈ" (Parent element) သေ သူင်ႇတေႃႇပၼ် "လုၵ်ႈလၢင်း" (Child element) မၼ်းၶႃႈ။

**တူဝ်ယၢင်ႇ:**

```css
body {
  color: darkgrey;
  font-family: 'PangLong';
}

```

> *ပေႃးႁဝ်းတႅမ်ႈၼင်ႇၼႆ၊ လိၵ်ႈတင်းၼႃႈဝႅပ်ႉ (တင်း p, h1, li) တေပဵၼ်သီလႅတ်း လႄႈ ၸႂ်ႉ Font PangLong တင်းသဵင်ႈ ယွၼ်ႉမၼ်းပူၺ်ႈ (Inherit) မႃးတီႈ `body` ၼၼ်ႉၶႃႈ။*

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **Cascade:** ပဵၼ်ပိူင်လုၵ်ႇလၢင်ႇ ဢၼ်တႅမ်ႈဝႆႉတၢင်းတႂ်ႈ တေပႄႉတၢင်းၼိူဝ်။
* **Specificity:** ID ပႄႉ Class၊ Class ပႄႉ Tag။
* **Inheritance:** Style ၵမ်ႈၽွင်ႈတေလုၵ်ႉတီႈ Parent ပူၺ်ႈလူင်းၵႂႃႇၸူး Child။
