
## Lesson 4: The Cascade

**The Cascade** ဢၼ်ဝႃႈၼၼ်ႉ မၼ်းပဵၼ် "ပိူင်ထတ်းသိၼ်" ၶွင် CSS ယဝ်ႉ။ 
ပေႃးမီး Code CSS တင်းၼမ် သႂ်ႇဝႆႉ ပႅၵ်ႇပိူင်ႈၵၼ်ၼၼ်ႉ Browser တေၸႂ်ႉပိူင် 3 ပိူင်ႈၼႆႉ တႃႇထတ်းသိၼ်ဝႃႈ တေဢဝ်ဢၼ်လႂ်မႃးၸႂ်ႉ ၼႆယဝ်ႉ။

### 1. Order of Appearance (လွင်ႈၸၼ်ႉထႅဝ်)

ပေႃး Selector မိူၼ်ၵၼ်၊ ၶေႃႈၵႂၢမ်းသင်ႇပႅၵ်ႇၵၼ် ဢၼ်ယူႇ **"တၢင်းတႂ်ႈသုတ်း"** တေပဵၼ်ဢၼ်ပႄႉ (Win) လႄႈ တေၸႂ်ႉတိုဝ်းမၼ်းယဝ်ႉ။

**တူဝ်ယၢင်ႇ:**

```css
p {
  color: blue;
}

p {
  color: red; /* ဢၼ်ၼႆႉတေပႄႉ ယွၼ်ႉမၼ်းမႃးၸွမ်းလင် */
}

```

> *ၼႂ်းဝႅပ်ႉသၢႆႉႁဝ်း တေၼႄပဵၼ် သီလႅင်။*

---

### 2. Specificity (လွင်ႈတႅတ်ႈတေႃး)

ပေႃး Selector ပႅၵ်ႇၵၼ် ဢၼ်လႂ် **"တႅတ်ႈတေႃး (Specific)"** လိူဝ်ၵေႃႈ ဢၼ်ၼၼ်ႉတေပႄႉယဝ်ႉ။ Browser တေၼပ်ႉဢၢၼ်ႇၸၼ်ႉမၼ်းၼင်ႇၼႆ:

1. **ID (#)** - ၵတ်ႉၶႅၼ်ႇသုတ်း (ပႄႉပိူၼ်ႈတင်းသဵင်ႈ)
2. **Class (.)** - ၵတ်ႉၶႅၼ်ႇပၢၼ်ၵາງ
3. **Element (tag)** - ဢူၼ်ႈဢွၼ်ႇသုတ်း

**တူဝ်ယၢင်ႇ:**

```css
#header { color: red; }   
.title { color: blue; }    
h1 { color: green; }       

```

> *သင်ဝႃႈ h1 ၼၼ်ႉမီးတင်း ID လႄႈ Class၊ သီမၼ်းတေပဵၼ် **သီလႅင်** ယွၼ်ႉ ID မီးဢဵၼ်ႁႅင်းလူင် (Specificity သုင်) လိူဝ်သေ class လႄႈ element ယဝ်ႉ။*

---

### 3. Inheritance (လွင်ႈပူၺ်ႈပၼ်)

Property ၵမ်ႈၽွင်ႈ (မိူၼ်ၼင်ႇ `color` ဢမ်ႇၼၼ် `font-family`) တေဢဝ်မႃးတီႈ "ပေႃႈမႄႈ" (Parent element) သေ သူင်ႇပၼ် "လုၵ်ႈလၢင်း" (Child element) ယဝ်ႉ။

**တူဝ်ယၢင်ႇ:**

```css
body {
  color: green;
  font-family: 'PangLong';
}

```

> *ပေႃးႁဝ်းတႅမ်ႈၼင်ႇၼႆ၊ လိၵ်ႈတင်းၼႃႈဝႅပ်ႉ (ဢမ်ႇဝႃး p, h1, li) တေပဵၼ်သီၶဵဝ် လႄႈ ၸႂ်ႉ Font PangLong တင်းသဵင်ႈ ယွၼ်ႉမၼ်း ဢဝ် (Inherit) မႃးတီႈ `body` ၼၼ်ႉယဝ်ႉ။*

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **Cascade:** ပဵၼ်ပိူင် ဢၼ်တႅမ်ႈဝႆႉတၢင်းတႂ်ႈ တေပႄႉတၢင်းၼိူဝ်။
* **Specificity:** ID ပႄႉ Class၊ Class ပႄႉ Tag။
* **Inheritance:** Style ၵမ်ႈၽွင်ႈ တေလုၵ်ႉတီႈ Parent လူင်းၵႂႃႇၸူး Child။
