## Lesson 8: Units of Measurement

ၼႂ်း CSS ၼၼ်ႉ ႁဝ်းၸၢင်ႈၸႅၵ်ႇ "တူဝ်ၼပ်ႉၶၼၢတ်ႈ" ဝႆႉပဵၼ် 2 ၸုမ်းလူင်ၶႃႈ:

### 1. Absolute Units

ပဵၼ် unit ဢၼ် "ၵိုတ်းတိုင်ႇ" ဝႆႉ၊ ဢမ်ႇလႅၵ်ႈလၢႆႈၸွမ်းသင်။

* **px (Pixels):** ပဵၼ်ဢၼ်ႁဝ်းၸႂ်ႉၼမ်သုတ်း။ 1px ၼႆႉ မၼ်းတေတႅတ်ႈတေႃးဝႆႉယူႇတိုင်ႇ။
* **လွင်ႈၸႂ်ႉတိုဝ်း:** ၸႂ်ႉတႃႇမၵ်းမၼ်ႈၶွပ်ႇ (Border) ဢမ်ႇၼၼ် တီႈလွၵ်ႇဢၼ်ႁဝ်းဢမ်ႇၶႂ်ႈႁႂ်ႈမၼ်းယွၼ်ႈ။

---

### 2. Relative Units

ပဵၼ် unit ဢၼ် "လႅၵ်ႈလၢႆႈ" ၸွမ်း Font ဢမ်ႇၼၼ် ၸွမ်းတၢင်းၵႂၢင်ႈ Screen။

* **% (Percentage):** ယွၼ်ႈၸွမ်းတၢင်းၵႂၢင်ႈၶွင် "ႁပ်းပေႃႈမႄႈ" (Parent Box) မၼ်း။
* **em:** ယွၼ်ႈၸွမ်းၶၼၢတ်ႈ Font ၶွင် Element ၼၼ်ႉ။
* **rem (Root em):** ယွၼ်ႈၸွမ်းၶၼၢတ်ႈ Font ၶွင် `<html>` (Root)။ ဢၼ်ၼႆႉ **လီသုတ်း** တႃႇၸႂ်ႉတႅမ်ႈ Font လိၵ်ႈတႆး ႁႂ်ႈမၼ်းလူင်ႆႈၵူႈတီႈ။
* **vw / vh (Viewport Width/Height):** ယွၼ်ႈၸွမ်းတၢင်းၵႂၢင်ႈ/တၢင်းသုင် ၶွင်ၼႃႈจอ (Screen) ၵမ်းသိုဝ်ႈ။

---

### 3. တူဝ်ယၢင်ႇလွင်ႈတႅမ်ႈ Code

```css
h1 {
  font-size: 2.5rem;   /* ယႂ်ႇလိူဝ် Font ပိုၼ်ႉထၢၼ် 2.5 ပၼ်ႇ */
}

.container {
  width: 90%;          /* ႁႂ်ႈၵႂၢင်ႈ 90% ၶွင်ၼႃႈ Screen (တေမီးဢွင်ႈလွၵ်ႇသွင်ၶၢင်ႈ) */
  padding: 20px;       /* ပၼ် Padding တႅတ်ႈတေႃးလူၺ်ႈ px */
}

```

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **px:** ၸႂ်ႉမိူဝ်ႈၶႂ်ႈလႆႈၶၼၢတ်ႈ "တႅတ်ႈတေႃး"။
* **rem:** ၸႂ်ႉမိူဝ်ႈမႄး "Font" ႁႂ်ႈမၼ်းလူင်ႆႈၵူႈ Screen။
* **% / vw:** ၸႂ်ႉမိူဝ်ႈမႄး "တၢင်းၵႂၢင်ႈႁပ်း" ႁႂ်ႈမၼ်းႁူတ်းၶဝ်ႈယိုတ်းဢွၵ်ႇလႆႈ။
