## Lesson 1: CSS Anatomy (ပိူင်သၢင်ႈ CSS)

ၼႂ်း HTML ၼၼ်ႉ ႁဝ်းၸႂ်ႉ "Tag" တႃႇၵေႃႇသၢင်ႈ Skeleton မၼ်းယဝ်ႉ။ 
တေႃႈလဵဝ် ႁဝ်းတေၸႂ်ႉ **CSS** တႃႇပၼ်သီ၊ တႃႇမႄးႁၢင်ႈၾၢင်မၼ်းၶႃႈ။

### 1. ပိူင်သၢင်ႈ CSS (Syntax)

တႃႇပွင်ႇၸႂ် CSS ၼၼ်ႉ ႁဝ်းတေလႆႈပွင်ႇၸႂ် 3 လွင်ႈၼႆႉယဝ်ႉ:

1. **Selector:** ႁဝ်းတေလိူၵ်ႈ Tag ဢၼ်လႂ်ၼႂ်း HTML မႃးမႄး? (တူဝ်ယၢင်ႇ: `h1`, `p`)
2. **Property:** ႁဝ်းတေမႄးလွင်ႈသင်? (သီ, တၢင်းယႂ်ႇတူဝ်လိၵ်ႈ, ပိုၼ်ႉလင်)
3. **Value:** တေပၼ်ၵႃႈၶၼ်မၼ်းၵႃႈႁိုင်? (သီလႅင်, 32px)

> **ပိူင်တႅမ်ႈ:**
> `Selector { Property: Value; }`

---

### 2. တူဝ်ယၢင်ႇတႄႉ (Real Example)

သင်ဝႃႈ ႁဝ်းၶႂ်ႈပၼ်သီ "လိူင်" တႃႇႁူဝ်လိၵ်ႈ (h1) ၼႂ်းဝႅပ်ႉႁဝ်းၼႆ:

```css
h1 {
  color: yellow;
  font-size: 40px;
}

```

* **h1** ၼႆႉပဵၼ် **Selector** (လိူၵ်ႈဢဝ်ႁူဝ်လိၵ်ႈ h1 မႃးမႄး)
* **color** ၼႆႉပဵၼ် **Property** (မႄးသီလိၵ်ႈ)
* **yellow** ၼႆႉပဵၼ် **Value** (သီလိူင်)
* **font-size** ၼႆႉပဵၼ် **Property** (မႄးတၢင်းယႂ်ႇတူဝ်လိၵ်ႈ)
* **40px** ၼႆႉပဵၼ် **Value** (ပၼ်တၢင်းယႂ်ႇမၼ်း 40 pixel)

---

### 3. ၶေႃႈၵႂၢမ်းလၢတ်ႈၼႂ်း CSS (Terminology)

တွၼ်ႈတႃႇတေတႅမ်ႈ CSS လီငၢမ်းၼႆႉ လူဝ်ႇတွင်းဝႆႉၶေႃႈၵႂၢမ်းၸိူဝ်းၼႆႉၶႃႈ:

* **Declaration Block:** ဢၼ်ယူႇၼႂ်းၵႄႈ ၶွပ်ႇၵၢင် `{ ... }` ၼၼ်ႉ ႁဝ်းႁွင်ႉဝႃႈ ပဵၼ်တီႈပၼ်ၵႂၢမ်းသင်ႇ။
* **Semicolon ( ; ):** တေလႆႈသႂ်ႇဝႆႉ တီႈသုတ်း သဵၼ်ႈၵႂၢမ်းသင်ႇ တႃႇၼႄဝႃႈ ၶေႃႈၵႂၢမ်းၼၼ်ႉ ယဝ်ႉတူဝ်ႈယဝ်ႉ။

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **Selector** = လိူၵ်ႈ "ၽႂ်"
* **Property** = မႄး "သင်"
* **Value** = မႄး "ၵႃႈႁိုင် / သီသင်"
