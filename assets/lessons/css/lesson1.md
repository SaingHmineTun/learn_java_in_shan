## Lesson 1: CSS Anatomy (ပိူင်သၢင်ႈ CSS)

ၼႂ်း HTML ၼၼ်ႉ ႁဝ်းၸႂ်ႉ "Tag" တေႃႇၵေႃႇသၢင်ႈ လုပ်ႇလႅင်း (Skeleton) မၼ်း။ တေႃႈလဵဝ် ႁဝ်းတေၸႂ်ႉ **CSS** တေႃႇပၼ်သီ၊ တေႃႇမႄးႁၢင်ႈၾၢင်မၼ်းၶႃႈ။

### 1. ပိူင်သၢင်ႈ CSS (Syntax)

ၵတ်ႉၶႅၼ်ႇ CSS ဢၼ်ဝႃႈၼၼ်ႉ ၵွၺ်းပိုင်ႈဢိင်ၼိူဝ် လွင်ႈပွင်ႇၸႂ် 3 ယိူင်ႈၼႆႉၵွၺ်း:

1. **Selector (ဢၼ်လိူၵ်ႈ):** ႁဝ်းတေလိူၵ်ႈ Tag ဢၼ်လႂ်ၼႂ်း HTML မႃးမႄး? (တူဝ်ယၢင်ႇ: `h1`, `p`)
2. **Property (ၵုင်ႇသမ်ႇပတ်ႉတီႇ):** ႁဝ်းတေမႄးလွင်ႈသင်? (သီ, ၶၼၢတ်ႈလိၵ်ႈ, ပိုၼ်ႉလင်)
3. **Value (ၵႃႈၶၼ်):** တေပၼ်ၵႃႈၶၼ်မၼ်းၵႃႈႁိုင်? (သီလႅင်, 32px)

> **ပိူင်တႅမ်ႈ:**
> `Selector { Property: Value; }`

---

### 2. တူဝ်ယၢင်ႇတႄႉ (Real Example)

သင်ဝႃႈ ႁဝ်းၶႂ်ႈပၼ်သီ "လိူင်" တေႃႇႁူဝ်လိၵ်ႈ (h1) ၼႂ်းဝႅပ်ႉႁဝ်းၼႆ:

```css
h1 {
  color: yellow;
  font-size: 40px;
}

```

* **h1** ၼႆႉပဵၼ် **Selector** (လိူၵ်ႈဢဝ်ႁူဝ်လိၵ်ႈ h1 မႃးမႄး)
* **color** ၼႆႉပဵၼ် **Property** (မႄးသီလိၵ်ႈ)
* **yellow** ၼႆႉပဵၼ် **Value** (သီလိူင်)
* **font-size** ၼႆႉပဵၼ် **Property** (မႄးၶၼၢတ်ႈလိၵ်ႈ)
* **40px** ၼႆႉပဵၼ် **Value** (ပၼ်သၢႆး 40 pixel)

---

### 3. ၶေႃႈၵႂၢမ်းလၢတ်ႈၼႂ်း CSS (Terminology)

တွၼ်ႈတႃႇတေတႅမ်ႈ CSS ႁႂ်ႈထုၵ်ႇလႅတ်းၼၼ်ႉ လူဝ်ႇတွင်းဝႆႉၶေႃႈၵႂၢမ်းၸိူဝ်းၼႆႉၶႃႈ:

* **Declaration Block:** ဢၼ်ယူႇၼႂ်းၵႄႈ ပိၵ်ႉပၵ်း `{ ... }` ၼၼ်ႉ ႁဝ်းႁွင်ႉဝႃႈ ပဵၼ်တီႈပၼ်ၵႂၢမ်းသင်ႇ။
* **Semicolon ( ; ):** တေလႆႈသႂ်ႇဝႆႉ တီႈသုတ်းသဵၼ်ႈၵႂၢမ်းသင်ႇ တႃႇၼႄဝႃႈ ၶေႃႈၵႂၢမ်းၼၼ်ႉ ယဝ်ႉတူဝ်ႈယဝ်ႉ။ (မိူၼ်ၼင်ႇ ႁဝ်းသႂ်ႇ လၢၵ်ႈၼႃႈ `-` တီႈသုတ်းလိၵ်ႈတႆးၼၼ်ႉၶႃႈ)

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **Selector** = လိူၵ်ႈ "ၽႂ်"
* **Property** = မႄး "သင်"
* **Value** = မႄး "ၵႃႈႁိုင် / သီသင်"
