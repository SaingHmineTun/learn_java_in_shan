## Lesson 18: Fractional Units (fr)

`fr` ၼႆႉ တႅၼ်းတၢင်ၶေႃႈၵႂၢမ်းဝႃႈ "Fraction" (ပွတ်း/တွၼ်ႈ) ၶႃႈ။ မၼ်းတေတူၺ်းဝႃႈ တီႈလွၵ်ႇ (Space) ဢၼ်ၵိုတ်းဝႆႉၼၼ်ႉ မီးၵေႇႁိုဝ် သေတေၸႅၵ်ႇပၼ် Elements ႁဝ်းၸွမ်းၼင်ႇႁဝ်းမၵ်းမၼ်ႈဝႆႉၼၼ်ႉၶႃႈ။

### 1. တေႃႇသူႈလွင်ႈမၢႆၼပ်ႉ (The Math Killer)

မိူဝ်ႈၵွၼ်ႇ ပေႃးႁဝ်းၶႂ်ႈလႆႈ 3 ထႅဝ်ၽဵင်ႇၵၼ် ႁဝ်းတေလႆႈတႅမ်ႈ `width: 33.33%`။ ပေႃးႁဝ်းသႂ်ႇ Padding ဢမ်ႇၼၼ် Gap ဢိတ်းၼိုင်ႈၵေႃႈ မၢႆၼပ်ႉႁဝ်းတေၽိတ်းသေ ဝႅပ်ႉတေတႅၵ်ႇၵမ်းသိုဝ်ႈ။

**လွင်ႈငၢႆႈၶွင် fr:**

```css
.container {
  display: grid;
  /* ၸႅၵ်ႇပဵၼ် 3 တွၼ်ႈၽဵင်ႇၵၼ် (မၼ်းတေၼပ်ႉပၼ်ႁင်းမၼ်း) */
  grid-template-columns: 1fr 1fr 1fr;
}

```

### 2. ၵၢၼ်ၸႅၵ်ႇတွၼ်ႈ ဢၼ်ဢမ်ႇၽဵင်ႇၵၼ်

ႁဝ်းၸၢင်ႈၸႂ်ႉ `fr` တႃႇမၵ်းမၼ်ႈဝႃႈ ၶႂ်ႈႁႂ်ႈထႅဝ်လႂ် ယႂ်ႇလိူဝ်ထႅဝ်လႂ်ၵေႃႈလႆႈၶႃႈ။

**တူဝ်ယၢင်ႇ:**

```css
.container {
  display: grid;
  /* ထႅဝ်တၢင်းၶႂႃ တေယႂ်ႇလိူဝ်ထႅဝ်တၢင်းသၢႆႉ 2 ပၼ်ႇ */
  grid-template-columns: 1fr 2fr;
}

```

---

### 3. ၵၢၼ်ၸႂ်ႉလေႃးၵၼ် (Mixing Units)

ႁဝ်းၸၢင်ႈၸႂ်ႉ `px` လႄႈ `fr` လေႃးၵၼ်လႆႈၶႃႈ။ ဢၼ်ၼႆႉၶႅမ်ႉသုတ်းတွၼ်ႈတႃႇႁဵတ်း Sidebar။

**တူဝ်ယၢင်ႇ:**

```css
.container {
  display: grid;
  /* Sidebar ပၵ်းဝႆႉ 200px၊ တီႈၵိုတ်းတင်းသဵင်ႈ ပၼ် Main Content */
  grid-template-columns: 200px 1fr;
}

```

### 4. ၵၢၼ်ၸႂ်ႉ repeat() တူဝ်ႈတူဝ်ႈ

```css
grid-template-columns: repeat(4, 1fr); /* လႆႈ 4 ထႅဝ် ဢၼ်ယိုတ်းလႆႈ ႁူတ်းလႆႈ */

```

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **fr:** ပဵၼ်တူဝ်ၼပ်ႉဢၼ် "ၸႅၵ်ႇတွၼ်ႈ" ၸွမ်းၼင်ႇတီႈလွၵ်ႇမီးဝႆႉ။
* **Flexibility:** မၼ်းတေၼပ်ႉပၼ်ႁင်းမၼ်း ပေႃးႁဝ်းသႂ်ႇ Gap ဢမ်ႇၼၼ် Border၊ ႁဵတ်းႁႂ်ႈဝႅပ်ႉဢမ်ႇတႅၵ်ႇ။
* **Best Practice:** ၸႂ်ႉ `fr` တွၼ်ႈတႃႇ Layout လုၵ်ႈလူင် ႁႂ်ႈမၼ်း Responsive လႆႈငၢႆႈၶႃႈ။
