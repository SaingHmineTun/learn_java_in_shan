## Lesson 20: Gap & Alignment

မိူဝ်ႈႁဝ်းၸၼ်လွၵ်း Grid ယဝ်ႉ၊ လွင်ႈလမ်ႇလွင်ႈထႅင်ႈပိူင်ၼိုင်ႈၵေႃႈပဵၼ် လွင်ႈပၼ် "ဢွင်ႈထႅဝ်" ၼႂ်းၵႄႈလွၵ်းၸိူဝ်းၼၼ်ႉၶႃႈ။

### 1. Gap (ဢွင်ႈလွၵ်ႇၼႂ်းၵႄႈ)

ၼႂ်းပိူင်ၵဝ်ႇၼၼ်ႉ ႁဝ်းလူဝ်ႇၸႂ်ႉ `margin` တႃႇႁဵတ်းႁႂ်ႈ Box ႁၢင်ႇၵၼ်၊ ၵွၺ်းၵႃႈၼႂ်း Grid ႁဝ်းၸၢင်ႈၸႂ်ႉ `gap` လႆႈငၢႆႈငၢႆႈၶႃႈ။

* **row-gap:** ပၼ်ဢွင်ႈလွၵ်ႇ တၢင်းၼွၼ်း (ၼိူဝ်/တႂ်ႈ)။
* **column-gap:** ပၼ်ဢွင်ႈလွၵ်ႇ တၢင်းၸုၵ်း (သၢႆႉ/ၶႂႃ)။
* **gap:** ပၼ်တင်းသွင်ၾၢႆႇ ၼႂ်း Code ထႅဝ်လဵဝ်။

**တူဝ်ယၢင်ႇ:**

```css
.container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px; /* ႁဵတ်းႁႂ်ႈၵူႈလွၵ်း ႁၢင်ႇၵၼ် 20px */
}

```

---

### 2. Grid Alignment (လွင်ႈၸတ်းဝၢင်း)

ႁဝ်းၸၢင်ႈၸတ်း Elements ၼႂ်း Grid လႆႈ 2 ၸၼ်ႉၶႃႈ:

#### A. Aligning the Whole Grid (ၸတ်းတင်းလွၵ်း)

ၸႂ်ႉမိူဝ်ႈ Grid ႁဝ်း ၵႅပ်ႈလိူဝ် Container (ပေႃႈမႄႈ) မၼ်း။

* `justify-content`: ၸတ်းတၢင်းၼွၼ်း (center, space-between, etc.)
* `align-content`: ၸတ်းတၢင်းၸုၵ်း

#### B. Aligning Items Inside Cells (ၸတ်းၼႂ်းလွၵ်းတႃ)

ၸႂ်ႉမိူဝ်ႈၶႂ်ႈႁႂ်ႈ Element ၼိုင်ႈဢၼ် ယူႇၵႄႈၵၢင်လွၵ်းမၼ်းတႅတ်ႈတေႃး။

* `justify-items`: ၸတ်းတၢင်းၼွၼ်း ၼႂ်းလွၵ်း။
* `align-items`: ၸတ်းတၢင်းၸုၵ်း ၼႂ်းလွၵ်း။

---

### 3. Place-items (လၢႆးတႅမ်ႈပွတ်း)

သင်ႁဝ်းၶႂ်ႈႁႂ်ႈၵူႈယိူင်ႈ ယူႇၵႄႈၵၢင် (Center) တင်းသွင်ၾၢႆႇ၊ ႁဝ်းၸၢင်ႈၸႂ်ႉ `place-items` လႆႈၶႃႈ။

**တူဝ်ယၢင်ႇ:**

```css
.container {
  display: grid;
  height: 200px;
  place-items: center; /* ႁဵတ်းႁႂ်ႈ "လုၵ်ႈ" တင်းသဵင်ႈ ယူႇၵႄႈၵၢင်တႅတ်ႈတေႃး */
}

```

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **Gap:** ၸွႆႈပၼ်ဢွင်ႈလွၵ်ႇ ၼႂ်းၵႄႈ Grid Items လႆႈငၢႆႈငၢႆႈ ဢမ်ႇလူဝ်ႇၸႂ်ႉ Margin။
* **Justify/Align Content:** ၸတ်းဝၢင်း Grid တင်းဝူင်း။
* **Justify/Align Items:** ၸတ်းဝၢင်း Elements ၼႂ်းလွၵ်းတႃတႅပ်ႉၽႂ်မၼ်း။
