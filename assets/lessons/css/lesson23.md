## Lesson 23: Mobile-First Design

**Mobile-First** ပွင်ႇဝႃႈ "တႅမ်ႈ Code တႃႇၾူၼ်းဢွၼ်တၢင်း" ၶႃႈ။ မိူဝ်ႈၵွၼ်ႇ ပိူၼ်ႈတႅမ်ႈတႃႇ Computer ယဝ်ႉ ၸင်ႇမႃးမႄးတႃႇၾူၼ်း (Desktop-First)၊ ၵွၺ်းၵႃႈ ဝၼ်းမိူဝ်ႈၼႆႉ ၵူၼ်းၸႂ်ႉၾူၼ်းၼမ်လိူဝ် Computer ယဝ်ႉလႄႈ ႁဝ်းလူဝ်ႇလႅၵ်ႈလၢႆးတႅမ်ႈၶႃႈ။

### 1. လၢႆးတႅမ်ႈ Code (The Logic)

ၼႂ်း Mobile-First ႁဝ်းတေတႅမ်ႈ CSS ပိုၼ်ႉထၢၼ် ႁႂ်ႈမၼ်းႁၢင်ႈလီၼိူဝ်ၾူၼ်း (Mobile) သေ၊ ၸင်ႇၸႂ်ႉ Media Query တႅၼ်းတၢင်ၵၢၼ် "ၶႂၢၵ်ႈ" (Expand) ႁႂ်ႈမၼ်းယႂ်ႇၶိုၼ်ႈ တွၼ်ႈတႃႇ Computer ၶႃႈ။

* **Property ဢၼ်ၸႂ်ႉ:** `min-width` (တႄႇတၢင်းၵႂၢင်ႈတီႈ...)

**တူဝ်ယၢင်ႇ:**

```css
/* 1. CSS ပိုၼ်ႉထၢၼ်: တႃႇ Mobile (Default) */
.content-box {
  width: 100%; /* ႁႂ်ႈတဵမ်ၼႃႈจอၾူၼ်း */
  padding: 10px;
}

/* 2. တွၼ်ႈတႃႇ Desktop: မိူဝ်ႈ Screen ၵႂၢင်ႈလိူဝ် 1024px */
@media (min-width: 1024px) {
  .content-box {
    width: 50%; /* ႁႂ်ႈၵိုတ်းၶိုင်ႈလဵဝ် မိူဝ်ႈယူႇၼိူဝ် Computer */
    margin: 0 auto;
  }
}

```

### 2. ၵွပ်ႈသင်လူဝ်ႇႁဵတ်း Mobile-First?

1. **လွင်ႈဝႆး (Performance):** ၾူၼ်းတေလူတ်ႇ (Load) CSS ဢၼ်ငၢႆႈငၢႆႈဢွၼ်တၢင်း၊ ႁဵတ်းႁႂ်ႈဝႅပ်ႉပိုတ်ႇဝႆး။
2. **လွင်ႈသုၵ်ႉယုင်ႈ (Simplicity):** ၵၢၼ်တႅမ်ႈ Layout တႃႇၾူၼ်းၼၼ်ႉငၢႆႈ (ပဵၼ်ထႅဝ်လဵဝ်လူင်းၵႂႃႇ)။ ပေႃးႁဝ်းတႄႇတီႈဢၼ်ငၢႆႈ၊ ၵၢၼ်တႅမ်ႈ Code ႁဝ်းတေဢမ်ႇယုင်ႈယၢပ်ႇ။
3. **User Experience:** ၸွႆႈႁႂ်ႈႁဝ်းဝူၼ်ႉထိုင် ၵူၼ်းၸႂ်ႉၾူၼ်းပဵၼ်ပိူင်လူင် (လိၵ်ႈတႆးတူဝ်ယႂ်ႇ၊ ပုမ်ႇတဵၵ်းငၢႆႈ)။

---

### 3. လၢႆးၸႂ်ႉတိုဝ်း တႃႇဝႅပ်ႉသၢႆႉတႆး

ပေႃးႁဝ်းတေႁဵတ်း ဝႅပ်ႉသၢႆႉႁူင်းႁဵၼ်းလိၵ်ႈတႆး:

* **Mobile (Default):** Menu ႁႂ်ႈပဵၼ်သဵၼ်ႈလူင်းၵႂႃႇ၊ လိၵ်ႈတႆးတူဝ်ယႂ်ႇ 22px။
* **Desktop (min-width: 1024px):** Menu ႁႂ်ႈၸုၵ်းပဵၼ်ထႅဝ်ၼွၼ်း (Flexbox)၊ လိၵ်ႈတႆးတူဝ်လဵၵ်ႉလူင်းဢိတ်းၼိုင်ႈ 18px။

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **Mobile-First:** တႅမ်ႈ Code ပိုၼ်ႉထၢၼ် တႃႇၾူၼ်း။
* **min-width:** ၸႂ်ႉ Media Query တႃႇမၵ်းမၼ်ႈ တွၼ်ႈတႃႇ Screen ဢၼ်ယႂ်ႇၶိုၼ်ႈ။
* **Benefit:** ႁဵတ်းႁႂ်ႈဝႅပ်ႉဝႆး လႄႈ Code ၸိုၼ်ႈသႂ်။
