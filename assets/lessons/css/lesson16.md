## Lesson 16: Flex Alignment

ၼႂ်း Lesson 15 ႁဝ်းႁူႉၸၵ်း `justify-content` ဢၼ်ၸတ်းဝၢင်းတၢင်းၼွၼ်း (Main Axis) ယဝ်ႉ။ 
ၼႂ်းတွၼ်ႈၼႆႉ ႁဝ်းတေမႃးသွၼ်လွင်ႈ ၸတ်းဝၢင်းတၢင်းၸုၵ်း (Cross Axis)။

### 1. Align Items (ၸတ်းဝၢင်းတၢင်းၸုၵ်း)

Property ၼႆႉ ၸႂ်ႉတွၼ်ႈတႃႇၸတ်း "လုၵ်ႈ" (Items) ၼႂ်း Flex Container ႁႂ်ႈမၼ်းယူႇတီႈလႂ် ၼႂ်းတၢင်းသုင်ၶွင်ထႅဝ်ၼၼ်ႉၶႃႈ။

* **flex-start:** ၸုၵ်းဝႆႉတၢင်းၼိူဝ်သုတ်း။
* **flex-end:** ၸုၵ်းဝႆႉတၢင်းတႂ်ႈသုတ်း။
* **center:** ၸုၵ်းဝႆႉၵႄႈၵၢင် (တၢင်းၸုၵ်း)။
* **stretch (default):** ယိုတ်းတူဝ်ႁႂ်ႈသုင်တဵမ် Container။

---

### 2. The "Holy Grail" (လၢႆးၸတ်းၵႄႈၵၢင်တႅတ်ႈတေႃး)

မိူဝ်ႈၵွၼ်ႇ ၵၢၼ်တေဢဝ် Box ဢၼ်ၼိုင်ႈ ဝႆႉၵႄႈၵၢင်ၼႃႈ တင်းၼွၼ်းတင်းၸုၵ်းၼၼ်ႉ ယၢပ်ႇၼႃႇ။ 
ၵွၺ်းၵႃႈ ပေႃးၸႂ်ႉ Flexbox ႁဝ်းတႅမ်ႈ Code 3 ထႅဝ်ၵွၺ်းၵေႃႈလႆႈယဝ်ႉ:

**တူဝ်ယၢင်ႇ Code:**

```css
.hero-section {
  display: flex;
  justify-content: center; /* ၸတ်းၵႄႈၵၢင် တၢင်းၼွၼ်း */
  align-items: center;     /* ၸတ်းၵႄႈၵၢင် တၢင်းၸုၵ်း */
  height: 100vh;           /* ပၼ်တၢင်းသုင်တဵမ်ၼႃႈจอ */
}

```

### 3. Flex Wrap (ၵၢၼ်ၶိုၼ်ႈထႅဝ်မႂ်ႇ)

ပေႃးႁဝ်းမီး "လုၵ်ႈ" ၼမ်ပူၼ်ႉတီႈ ပေႃးတဵမ်တၢင်းၵႂၢင်ႈၼႃႈ screen ၵႂႃႇၼႆၸိုင် ႁဝ်းၸၢင်ႈသင်ႇႁႂ်ႈမၼ်း "တူၵ်းထႅဝ်" မႃးယူႇၸၼ်ႉတႂ်ႈလႆႈၶႃႈ။

* **nowrap (default):** ၶပ်ႉၵၼ်ဝႆႉၼႂ်းထႅဝ်လဵဝ် (မၼ်းတေၸၢင်ႈလႅၼ်ႈဢွၵ်ႇၼႃႈ)။
* **wrap:** ပေႃးတဵမ်ယဝ်ႉ ႁႂ်ႈမၼ်းလူင်းပၼ်ထႅဝ်မႂ်ႇ။

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **Justify-content:** ၸတ်းဝၢင်းတၢင်းၼွၼ်း (Left, Right, Center)။
* **Align-items:** ၸတ်းဝၢင်းတၢင်းၸုၵ်း (Top, Bottom, Center)။
* **Flex-wrap:** ၸွႆႈႁႂ်ႈ Elements လူင်းပၼ်ထႅဝ်မႂ်ႇ မိူဝ်ႈတဵမ်ၼႃႈ (Responsive)။
