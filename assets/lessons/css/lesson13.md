## Lesson 13: Static vs. Relative

ၼႂ်း CSS ၼၼ်ႉ `position` ပဵၼ် Property ဢၼ်ၸႂ်ႉမၵ်းမၼ်ႈ "တီႈယူႇ" ၶွင် Element ယဝ်ႉ။ 
position property ဢၼ်ႁဝ်းၸႂ်ႉတိုဝ်းလႆႈ မီးယူႇ 2 လွင်ႈ ၼင်ႇတီႈတႂ်ႈယဝ်ႉ:

### 1. Static

ပဵၼ်ပိူင်ဢၼ် HTML Elements ၸႂ်ႉဝႆႉယဝ်ႉ။

* **ၾၢင်ႁၢင်ႈ:** မၼ်းတေၸုၵ်းဝႆႉၸွမ်းၼင်ႇ ထႅဝ်လိၵ်ႈ (Document Flow) မီးဝႆႉ။
* **လွင်ႈယိူင်ႉ:** ႁဝ်းဢမ်ႇၸၢင်ႈၸႂ်ႉ `top`, `bottom`, `left`, `right` တီႈမၼ်းလႆႈ။

---

### 2. Relative

ပဵၼ်ပိူင်ဢၼ်ႁဝ်း "ယိူင်ႉ" Element ၼၼ်ႉဢွၵ်ႇၵႂႃႇ တီႈၼိုင်ႈၸူးတီႈၼိုင်ႈ ပိုင်ႈဢိင်ၼိူဝ် "တီႈယူႇၵဝ်ႇ" မၼ်းၶႃႈ။

* **ၾၢင်ႁၢင်ႈ:** မိူဝ်ႈႁဝ်းယိူင်ႉမၼ်းၵႂႃႇယဝ်ႉ "တီႈယူႇၵဝ်ႇ" မၼ်းၼၼ်ႉ တေယဵၼ်းပဵၼ် "ဢွင်ႈလွၵ်ႇ" ဝႆႉယူႇ၊ မၼ်းတေဢမ်ႇၵႂႃႇယႃႉ Document Flow ၶွင်တူဝ်တၢင်ႇဢၼ်။
* **လွင်ႈၸႂ်ႉတိုဝ်း:** ၸႂ်ႉမိူဝ်ႈၶႂ်ႈ "ၼိတ်း" (Nudge) Element ႁႂ်ႈမၼ်းၶၢႆႉၵႂႃႇဢိတ်းဢီတ်း။

**တူဝ်ယၢင်ႇလွင်ႈတႅမ်ႈ Code:**

```css
.box-relative {
  position: relative;
  top: 20px;  /* ယိူင်ႉလူင်းတၢင်းတႂ်ႈ 20px လုၵ်ႉတီႈပိုၼ်ႉထၢၼ်ၵဝ်ႇမၼ်း */
  left: 50px; /* ယိူင်ႉၵႂႃႇတၢင်းၶႂႃ 50px */
}

```

---

### 3. လွင်ႈလမ်ႇလွင်ႈထႅင်ႈပိူင်ၼိုင်ႈ ၶွင် Relative

ၼွၵ်ႈလိူဝ်သေၵၢၼ် "ယိူင်ႉ" တူဝ်မၼ်းယဝ်ႉ၊ `position: relative;` ၼႆႉ ယင်းမီးၼႃႈတီႈပဵၼ် **"ႁိူၼ်း" (Home base)** တွၼ်ႈတႃႇလုၵ်ႈမၼ်း ဢၼ်တေၸႂ်ႉ `position: absolute` ယဝ်ႉ။ (ႁဝ်းတေသွၼ်လွင်ႈၼႆႉၼႂ်း Lesson 14 ၶႃႈ)။

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **Static:** တီႈယူႇပၵ်းသဝ်းၸွမ်းထႅဝ် HTML၊ ယိူင်ႉဢမ်ႇလႆႈ။
* **Relative:** တီႈယူႇဢၼ်ယိူင်ႉလႆႈ (top, left, etc.) လုၵ်ႉတီႈဢွင်ႈၵဝ်ႇမၼ်း၊ ၵွၺ်းၵႃႈ "ႁွႆးတူဝ်မၼ်း" တီႈၵဝ်ႇၼၼ်ႉ ယင်းမီးဝႆႉယူႇ။
