
## Lesson 3: Selectors & ID/Class

မိူဝ်ႈတွၼ်ႈသွၼ် 1 ၼၼ်ႉ ႁဝ်းလိူၵ်ႈၸႂ်ႉ Tag Selector (မိူၼ်ၼင်ႇ `h1`, `p`)။ ၵွၺ်းၵႃႈ သင်ဝႃႈၼႂ်းဝႅပ်ႉႁဝ်းမီး `<p>` တင်းၼမ်သေ ႁဝ်းၶႂ်ႈပၼ်သီပႅၵ်ႇပိူင်ႈၵၼ်ၼၼ်ႉ ႁဝ်းတေလႆႈၸႂ်ႉ **Class** လႄႈ **ID** ၶႃႈ။

### 1. Class Selector (ၸႂ်ႉၸွမ်းၵၼ်လႆႈၼမ်)

**Class** ၼႆႉ မိူၼ်ၼင်ႇ "ၶူဝ်းၼုင်ႈယူႇၼီႇၾွမ်း" ၶႃႈ။ ႁဝ်းၸၢင်ႈဢဝ် Class လဵဝ်ၵၼ်ၼၼ်ႉ ၵႂႃႇသႂ်ႇပၼ် HTML Elements တင်းၼမ်လႆႈ။

* **ပိူင်တႅမ်ႈၼႂ်း HTML:** ၸႂ်ႉ Attribute `class="နာမည်"`
* **ပိူင်တႅမ်ႈၼႂ်း CSS:** တေလႆႈတႄႇလူၺ်ႈ **မၢႆၸုၵ်း ( . )**

**တူဝ်ယၢင်ႇ:**

```html
<p class="highlight">လိၵ်ႈတႆးသီၶမ်း</p>
<p class="highlight">လိၵ်ႈတႆးသီၶမ်းထႅင်ႈပၼ်ၵေႃႈၼိုင်ႈ</p>

```

```css
/* CSS */
.highlight {
  color: gold;
  font-weight: bold;
}

```

---

### 2. ID Selector (ၸႂ်ႉလႆႈတီႈလဵဝ်ၵွၺ်း)

**ID** ၼႆႉ မိူၼ်ၼင်ႇ "မၢႆၾၢင်ႁူဝ်သိူဝ်" ဢမ်ႇၼၼ် "Passport" ၶႃႈ။ ၼႂ်းၼႃႈလိၵ်ႈဝႅပ်ႉၼိုင်ႈၼႃႈ ID ၼိုင်ႈဢၼ် လူဝ်ႇမီးတီႈလဵဝ်ၵွၺ်း (Unique)။

* **ပိူင်တႅမ်ႈၼႂ်း HTML:** ၸႂ်ႉ Attribute `id="နာမည်"`
* **ပိူင်တႅမ်ႈၼႂ်း CSS:** တေလႆႈတႄႇလူၺ်ႈ **မၢႆၶႅပ်း ( # )**

**တူဝ်ယၢင်ႇ:**

```html
<h1 id="main-title">ႁူဝ်လိၵ်ႈလူင်</h1>

```

```css
/* CSS */
#main-title {
  text-align: center;
  font-size: 50px;
}

```

---

### 3. လွင်ႈပႅၵ်ႇပိူင်ႈ Class လႄႈ ID (The Difference)

| လွင်ႈတူၺ်း | Class ( . ) | ID ( # ) |
| --- | --- | --- |
| **လွင်ႈၸႂ်ႉတိုဝ်း** | ၸႂ်ႉလႆႈတင်းၼမ် (Reusable) | ၸႂ်ႉလႆႈတီႈလဵဝ် (Unique) |
| **တူဝ်ယၢင်ႇ** | မိူၼ်ၼင်ႇ သီၶူဝ်းၼုင်ႈ | မိူၼ်ၼင်ႇ မၢႆၾၢင် |
| **ၼႂ်း CSS** | တႄႇလူၺ်ႈ `.` | တႄႇလူၺ်ႈ `#` |

---

### 4. Grouping Selectors (လိူၵ်ႈၵမ်းလဵဝ်တင်းၼမ်)

သင်ဝႃႈႁဝ်းၶႂ်ႈပၼ် Property မိူၼ်ၵၼ် တေႃႇ Tag တင်းၼမ် ႁဝ်းၸၢင်ႈၸႂ်ႉ **မၢႆၶၼ်ႇ ( , )** ၶႃႈ။

```css
h1, h2, p {
  font-family: 'NamKhone Web';
  color: #333;
}

```

*(Code ၼႆႉ တေႁဵတ်းႁႂ်ႈ h1, h2 လႄႈ p ပဵၼ် Font လဵဝ်ၵၼ် တင်းသဵင်ႈၶႃႈ)*

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **Tag Selector:** လိူၵ်ႈၸွမ်းၸိုဝ်ႈ Tag (h1, p, div)။
* **Class Selector ( . ):** ၸႂ်ႉတွၼ်ႈတႃႇၸုမ်း elements ဢၼ်ၶႂ်ႈပၼ် Style မိူၼ်ၵၼ်။
* **ID Selector ( # ):** ၸႂ်ႉတွၼ်ႈတႃႇ Element ဢၼ်ၶႂ်ႈႁဵတ်းႁႂ်ႈပႅၵ်ႇပိူင်ႈပိူၼ်ႈ တီႈလဵဝ်ၵွၺ်း။
