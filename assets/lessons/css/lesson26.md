## Lesson 26: Transforms (2D & 3D)

Property `transform` ၼႆႉ ၸွႆႈႁႂ်ႈႁဝ်းလႅၵ်ႈလၢႆႈ Visual Space ၶွင် Element လႆႈ ၵမ်းသိုဝ်ႈ လူၺ်ႈဢမ်ႇၵႂႃႇယႃႉ Layout ၶွင်တူဝ်တၢင်ႇဢၼ်ၶႃႈ။

### 1. 2D Transforms (ၵၢၼ်လႅၵ်ႈလၢႆႈ 2 ၾၢႆႇ)

မီးယူႇ 4 ယိူင်ႈ ဢၼ်ၸႂ်ႉၼမ်သုတ်း:

* **translate(x, y):** ၶၢႆႉ Element ၵႂႃႇၸွမ်းသဵၼ်ႈၼွၼ်း (x) ဢမ်ႇၼၼ် သဵၼ်ႈၸုၵ်း (y)။
* **rotate(angle):** ပၼ်ႇ Element ၸွမ်းၼင်ႇ Degree ဢၼ်ႁဝ်းပၼ် (တူဝ်ယၢင်ႇ: `45deg`)။
* **scale(x, y):** ႁဵတ်းႁႂ်ႈတူဝ်ယႂ်ႇၶိုၼ်ႈ ဢမ်ႇၼၼ် လဵၵ်ႉလူင်း (တူဝ်ယၢင်ႇ: `1.5` ၵေႃႈပဵၼ် 150%)။
* **skew(x, y):** ႁဵတ်းႁႂ်ႈ Element ၼၼ်ႉ "ၸူၼ်း" ၵႂႃႇၾၢႆႇၼိုင်ႈ။

---

### 2. 3D Transforms (ၵၢၼ်လႅၵ်ႈလၢႆႈ 3 ၾၢႆႇ)

တွၼ်ႈတႃႇတေႁဵတ်းႁႂ်ႈမၼ်းတူၺ်းမီး "တၢင်းလိုၵ်ႉ" (Depth)၊ ႁဝ်းၸၢင်ႈပၼ်ႇမၼ်းၸွမ်းသဵၼ်ႈ **Z-axis** လႆႈၶႃႈ။

**တူဝ်ယၢင်ႇ:**

```css
.card {
  /* ပၼ်ႇၸွမ်းသဵၼ်ႈ Y ႁႂ်ႈမိူၼ်ၼင်ႇ ၵၢၼ်ပိုတ်ႇပပ်ႉလိၵ်ႈ */
  transform: perspective(500px) rotateY(45deg);
}

```

---

### 3. ၵၢၼ်ၸႂ်ႉလေႃးၵၼ် (Combining Transforms)

ႁဝ်းၸၢင်ႈသႂ်ႇ Transform လၢႆလၢႆယိူင်ႈ ၼႂ်းထႅဝ်လဵဝ်ၵၼ်လႆႈၶႃႈ:

```css
.shan-icon:hover {
  /* ႁႂ်ႈမၼ်းယႂ်ႇၶိုၼ်ႈဢိတ်းၼိုင်ႈ လႄႈ ပၼ်ႇၵႂႃႇ 10 တီႇၵရီႇ */
  transform: scale(1.2) rotate(10deg);
  transition: transform 0.3s ease; /* သႂ်ႇ Transition ႁႂ်ႈမၼ်းၼုမ်ႇၼวล */
}

```

---

### 4. Transform Origin

ႁဝ်းၸၢင်ႈလိူၵ်ႈလႆႈဝႃႈ ၶႂ်ႈႁႂ်ႈမၼ်း "တႄႇပၼ်ႇ" တီႈလႂ်။

* **Default:** ပဵၼ် `center` (ၵႄႈၵၢင်)။
* **Options:** `top left`, `bottom right`, ဢမ်ႇၼၼ် မၵ်းမၼ်ႈပဵၼ် `%` ၵေႃႈလႆႈၶႃႈ။

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **Translate:** ၸႂ်ႉၶၢႆႉတီႈ (လီလိူဝ်ၵၢၼ်ၸႂ်ႉ top/left ၵွပ်ႈမၼ်းဝႆႉလိူဝ်)။
* **Rotate:** ၸႂ်ႉပၼ်ႇ Element။
* **Scale:** ၸႂ်ႉပၼ်ႇၶၼၢတ်ႈ (Zoom)။
* **Note:** ပေႃးၸႂ်ႉ Transform တေလူဝ်ႇသႂ်ႇ Transition ပႃး ၸင်ႇတေတူၺ်းႁၢင်ႈလီၶႃႈ။
