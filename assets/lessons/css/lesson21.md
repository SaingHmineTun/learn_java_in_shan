## Lesson 21: The Viewport Meta Tag

**Viewport** ၼႆႉ ပွင်ႇဝႃႈ "ဢွင်ႈတီႈလူလိၵ်ႈ" ၼိူဝ်ၼႃႈจอၶႃႈ။ ပေႃးႁဝ်းဢမ်ႇသင်ႇဝႆႉဝႃႈ ဝႅပ်ႉႁဝ်းၼႆႉပဵၼ်ဝႅပ်ႉ Mobile-friendly ၼႆ၊ Browser ၼိူဝ်ၾူၼ်းတေထၢင်ႇဝႃႈ ဝႅပ်ႉႁဝ်းၼႆႉပဵၼ်ဝႅပ်ႉ Desktop သေ မၼ်းတေသိုပ်ႇ "Zoom ဢွၵ်ႇ" ဝႆႉ ႁဵတ်းႁႂ်ႈလိၵ်ႈတႆးႁဝ်း တူဝ်လဵၵ်ႉၵွၺ်း ၸၼ်လူယၢပ်ႇၼႃႇၶႃႈ။

### 1. တူဝ် Code ဢၼ်လမ်ႇလွင်ႈ

ႁဝ်းလူဝ်ႇဢဝ် Code ထႅဝ်ၼႆႉ ၵႂႃႇသႂ်ႇဝႆႉၼႂ်းတွၼ်ႈ `<head>` ၶွင် HTML ၶႃႈ:

```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">

```

### 2. ပွင်ႇဝႃႈသင်?

* **width=device-width:** သင်ႇဝႃႈ ႁႂ်ႈတၢင်းၵႂၢင်ႈဝႅပ်ႉသၢႆႉ ၽဵင်ႇၵၼ်တင်း တၢင်းၵႂၢင်ႈၼႃႈจอၾူၼ်း ဢၼ်ၵူၼ်းၸႂ်ႉယူႇၼၼ်ႉ။
* **initial-scale=1.0:** သင်ႇဝႃႈ မိူဝ်ႈပိုတ်ႇဝႅပ်ႉမႃးၵမ်းแรก ႁႂ်ႈ Zoom ဝႆႉတီႈ 100% (ဢမ်ႇႁႂ်ႈမၼ်း Zoom ၶဝ်ႈ/ဢွၵ်ႇ ႁင်းမၼ်း)။

---

### 3. ပေႃးဢမ်ႇသႂ်ႇ တေပဵၼ်သင်?

သင်ႁဝ်းဢမ်ႇသႂ်ႇ Meta Tag ၼႆႉ:

* လိၵ်ႈတႆးႁဝ်း တေတူဝ်လဵၵ်ႉၼႃႇ တေႃႇပေႃးလူဢမ်ႇလႆႈ။
* ပုမ်ႇ (Buttons) ၸိူဝ်းၼၼ်ႉ တေလဵၵ်ႉသေ ဢဝ်မိုဝ်းတဵၵ်းယၢပ်ႇ။
* CSS Media Queries (Lesson 22) ဢၼ်ႁဝ်းတေတႅမ်ႈၼၼ်ႉ တေဢမ်ႇႁဵတ်းၵၢၼ် ၼိူဝ်ၾူၼ်းၵမ်ႈၼမ်ၶႃႈ။

---

### 4. တူဝ်ယၢင်ႇၼႂ်း HTML

```html
<!DOCTYPE html>
<html lang="shn">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>ဝႅပ်ႉသၢႆႉတႆး</title>
</head>
<body>
    <h1>မႂ်ႇသုင်ၶႃႈ</h1>
</body>
</html>

```

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **Viewport Tag:** ပဵၼ်ၵၢၼ်သင်ႇ Browser ဝႃႈ "ဝႅပ်ႉၼႆႉ ၸႂ်ႉၼိူဝ်ၾူၼ်းလႆႈယဝ်ႉ"။
* **Requirement:** မၼ်းပဵၼ် "ၶေႃႈလူဝ်ႇမီး" ဢွၼ်တၢင်းတေတႅမ်ႈ Responsive CSS။
* **Placement:** သႂ်ႇဝႆႉၼႂ်း `<head>` ၵူႈၼႃႈလိၵ်ႈ။
