## Lesson 11: Color Theory & Systems

ၼႂ်း CSS ၼၼ်ႉ ႁဝ်းၸၢင်ႈမၵ်းမၼ်ႈသီလႆႈလၢႆလၢႆပိူင်။ 
ဢၼ် Web Developer ၶဝ်ၸႂ်ႉၼမ်သုတ်းမီးယူႇ 3 ပိူင်ၼႆႉ:

### 1. Hex Codes (ပိူင်မၢႆလိၵ်ႈ 6 တူဝ်)

ပဵၼ်ပိူင်ဢၼ်ၸႂ်ႉ "မၢႆၶႅပ်း" (#) တႄႇၼႃႈ။ မၼ်းၸၢင်ႈမၵ်းမၼ်ႈလႆႈ သီ ဢၢၼ်ႇႁူဝ်လၢၼ်ႉမၼ်းယဝ်ႉ။

* **တူဝ်ယၢင်ႇ:** `#FF0000` (သီလႅင်), `#00FF00` (သီၶဵဝ်), `#0000FF` (သီသွမ်ႇ)။
* `#FFFFFF` ပဵၼ်သီၶၢဝ်၊ `#000000` ပဵၼ်သီလမ်။

### 2. RGB (Red, Green, Blue)

ပဵၼ်ၵၢၼ်လေႃးသီ ပိုၼ်ႉထၢၼ် 3 သီ။ တူဝ်ၼပ်ႉမၼ်းမီး 0 တေႃႇ 255။

* **တူဝ်ယၢင်ႇ:** `rgb(255, 0, 0)` (သီလႅင်)။
* **RGBA:** တူဝ် `a` (Alpha) ၼၼ်ႉ ၸႂ်ႉတွၼ်ႈတႃႇမၵ်းမၼ်ႈ "လွင်ႈသႂ်" (Opacity)။ တူဝ်ယၢင်ႇ `rgba(0,0,0, 0.5)` တေပဵၼ်သီလမ်ဢၼ်သႂ် 50% ၶႃႈ။

### 3. HSL (Hue, Saturation, Lightness)

ပိူင်ၼႆႉ ပွင်ႇၸႂ်ငၢႆႈလိူဝ်ပိူၼ်ႈ တွၼ်ႈတႃႇၵူၼ်းတႅမ်ႈဝႅပ်ႉၶႃႈ။

* **Hue:** သီသင်? (0-360)
* **Saturation:** သီၸိုတ်းႁႃႉ ဢမ်ႇၼၼ် သီလႅတ်းႁႃႉ? (%)
* **Lightness:** သီလႅင်းႁႃႉ ဢမ်ႇၼၼ် သီလပ်းႁႃႉ? (%)

---

### တူဝ်ယၢင်ႇလွင်ႈတႅမ်ႈ Code

```css
h1 {
  color: #AA0000; /* သီႁူဝ်လိၵ်ႈပဵၼ်သီလႅင်တႆး */
}

p {
  color: hsl(0, 0%, 20%); /* သီလိၵ်ႈလမ်လႅတ်းလႅတ်း ႁႂ်ႈလူငၢႆႈ */
}

div {
  background-color: rgba(255, 215, 0, 0.2); /* ပိုၼ်ႉလင်သီၶမ်းသႂ်ဢိတ်းဢိတ်း */
}

```

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **Hex:** ၸႂ်ႉတွၼ်ႈတႃႇမၵ်းမၼ်ႈသီတႅတ်ႈတေႃး။
* **RGBA:** ၸႂ်ႉမိူဝ်ႈၶႂ်ႈလႆႈသီဢၼ် "သႂ်" (Transparent)။
* **Contrast:** ပဵၼ်လွင်ႈလမ်ႇလွင်ႈသုတ်း တွၼ်ႈတႃႇႁႂ်ႈလိၵ်ႈတႆးလူငၢႆႈ။
