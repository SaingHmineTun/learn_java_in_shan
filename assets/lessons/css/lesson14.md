
## Lesson 14: Absolute & Fixed

မိူဝ်ႈႁဝ်းၸႂ်ႉ `Absolute` ဢမ်ႇၼၼ် `Fixed` ၼၼ်ႉ Element ၼၼ်ႉတေမိူၼ်ၼင်ႇ "မိၼ်" ဢွၵ်ႇၵႂႃႇၼိူဝ်ၼႃႈလိၵ်ႈသေ ဢမ်ႇၵိၼ်တီႈလွၵ်ႇ (Space) ၶွင်ၽႂ်ထႅင်ႈယဝ်ႉၶႃႈ။

### 1. Absolute (ပွႆႇလွတ်ႈၸွမ်း Parent)

`position: absolute;` ၼႆႉ တေႁဵတ်းႁႂ်ႈ Element ၼၼ်ႉ မိၼ်ဢွၵ်ႇၵႂႃႇသေ ၵႂႃႇၸုၵ်းဝႆႉတီႈလႂ်ၵေႃႈလႆႈ ၼႂ်း "ႁပ်းပေႃႈမႄႈ" (Parent) ဢၼ်မီး `position: relative` ဝႆႉၼၼ်ႉၶႃႈ။

* **ၾၢင်ႁၢင်ႈ:** မၼ်းတေဢမ်ႇၵိၼ်တီႈလွၵ်ႇၼိူဝ်ထႅဝ် HTML ထႅင်ႈ။ တူဝ်တၢင်ႇဢၼ်တေၶိုၼ်ႈမႃးတႅၼ်းတီႈမၼ်းၵမ်းသိုဝ်ႈ။
* **လွင်ႈၸႂ်ႉတိုဝ်း:** ၸႂ်ႉတွၼ်ႈတႃႇႁဵတ်း မၢႆ (Badge) ဢၼ်တပ်ႉဝႆႉၼိူဝ်ႁၢင်ႈၾၢင် ဢမ်ႇၼၼ် Icon ဢၼ်ယူႇၸဵင်ႇႁပ်း။

**တူဝ်ယၢင်ႇလွင်ႈတႅမ်ႈ Code:**

```css
.parent {
  position: relative; /* ပဵၼ်ႁိူၼ်း/ပိုၼ်ႉထၢၼ် */
  width: 300px;
  height: 200px;
}

.child-absolute {
  position: absolute;
  top: 0;
  right: 0; /* ႁႂ်ႈမၼ်းၵႂႃႇၸုၵ်းဝႆႉ တီႈၸဵင်ႇၶႂႃတၢင်းၼိူဝ်သုတ်း */
}

```

---

### 2. Fixed (ပၵ်းသဝ်းၸွမ်းၼႃႈ)

`position: fixed;` ၼႆႉ မၼ်းတေမိၼ်ဢွၵ်ႇၵႂႃႇသေ ၵႂႃႇ "ၸပ်း" ဝႆႉတီႈၼႃႈจอ (Browser Window) ၵမ်းသိုဝ်ႈ။ ႁဝ်းတေ Scroll လူင်းၵႂႃႇၵေႃႈယႃႇ မၼ်းတေယူႇတီႈၵဝ်ႇတိုင်ႇတိုင်ႇၶႃႈ။

* **ၾၢင်ႁၢင်ႈ:** မၼ်းပိုင်ႈဢိင်ၼိူဝ် ၼႃႈจอ (Viewport) ၵွၺ်း၊ မၼ်းဢမ်ႇပိုင်ႈဢိင်ၼိူဝ် Parent မၼ်း။
* **လွင်ႈၸႂ်ႉတိုဝ်း:** ၸႂ်ႉတွၼ်ႈတႃႇႁဵတ်း ထႅဝ် Menu (Navigation Bar) ဢၼ်ႁႂ်ႈမၼ်းၸပ်းဝႆႉတၢင်းၼိူဝ်တိုင်ႇ ဢမ်ႇၼၼ် ပုမ်ႇ Chat ဢၼ်ယူႇၸဵင်ႇတႂ်ႈ။

**တူဝ်ယၢင်ႇလွင်ႈတႅမ်ႈ Code:**

```css
.navbar {
  position: fixed;
  top: 0;
  width: 100%;
  background-color: #333;
}

```

---

### 3. လွင်ႈပႅၵ်ႇပိူင်ႈ Absolute လႄႈ Fixed

* **Absolute:** လႅၵ်ႈလၢႆႈတီႈယူႇၸွမ်း Parent (ပေႃး Scroll လူင်း မၼ်းတေလႅၼ်ႈၸွမ်းပိူၼ်ႈၵႂႃႇ)။
* **Fixed:** ၸပ်းဝႆႉတီႈၼႃႈจอ (ပေႃး Scroll လူင်း မၼ်းတေယူႇတီႈၵဝ်ႇ ဢမ်ႇၶၢႆႉၵႂႃႇလႂ်)။

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **Absolute:** မိၼ်ဢွၵ်ႇတီႈထႅဝ် HTML သေ ၸုၵ်းၸွမ်း Parent (Relative)။
* **Fixed:** မိၼ်ဢွၵ်ႇတီႈထႅဝ် HTML သေ ၸပ်းဝႆႉတီႈၼႃႈจอ (Viewport)။
