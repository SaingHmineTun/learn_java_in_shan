
## Lesson 15: Flexbox Basics

Flexbox (Flexible Box) ပဵၼ်ပိူင်ၸတ်းဝၢင်း Box ႁႂ်ႈမၼ်းၶၢႆႉလႆႈ၊ ယိုတ်းလႆႈ၊ လႄႈ ႁူတ်းလႆႈ ၸွမ်းၼင်ႇတၢင်းၵႂၢင်ႈယဝ်ႉ။

### 1. ပေႃႈမႄႈ လႄႈ လုၵ်ႈ (Container & Items)

တွၼ်ႈတႃႇတေၸႂ်ႉ Flexbox လႆႈၼၼ်ႉ ႁဝ်းလူဝ်ႇမီးသွင်ၸၼ်ႉ:

1. **Flex Container (ပေႃႈမႄႈ):** Box ဢၼ်တေယူႇ တင်းၼွၵ်ႈသုတ်ႉ။ ႁဝ်းၸႂ်ႉ `display: flex;` တႃႇၶူင်သၢင်ႈႁႂ်ႈပဵၼ် Flexbox ယဝ်ႉ။
2. **Flex Items (လုၵ်ႈ):** Box ဢၼ်ယူႇတၢင်းၼႂ်း။ မၼ်းတေ တူင်ႉၼိုင်ၸွမ်းၵမ်းသိုဝ်ႈ။

**တူဝ်ယၢင်ႇ Code:**

```css
.container {
  display: flex; /* ၼႆႉပဵၼ်ၵၢၼ်ပိုတ်ႇၸႂ်ႉ Flexbox */
}

```

---

### 2. Flex Direction (တၢင်းၵႂႃႇ)

မိူဝ်ႈႁဝ်းပိုတ်ႇ `display: flex;` ယဝ်ႉ၊ ႁဝ်းၸၢင်ႈလိူၵ်ႈလႆႈဝႃႈ ၶႂ်ႈႁႂ်ႈ "လုၵ်ႈ" မၼ်း ၶပ်းပၼ် "ထႅဝ်ၼွၼ်း" ဢမ်ႇၼၼ် "ထႅဝ်ၸုၵ်း" ၼႆယဝ်ႉ။

* **row (default):** တေၶပ်ႉပၼ်လူၺ်ႈ တၢင်းၼွၼ်း (သၢႆႉၵႂႃႇၶႂႃ)။
* **column:** တေၶပ်ႉပၼ်လူၺ်ႈ တၢင်းၸုၵ်း (ၼိူဝ်လူင်းတႂ်ႈ)။

---

### 3. Justify Content (ၸတ်းဝၢင်းတၢင်းၼွၼ်း)

ဢၼ်ၼႆႉ ၸႂ်ႉတွၼ်ႈတႃႇ ၶပ်ႉတူဝ်လုၵ်ႈ ယဝ်ႉ:

* **flex-start:** ၶပ်ႉပၼ်တၢင်းသၢႆႉသုတ်း။
* **flex-end:** ၶပ်ႉပၼ်တၢင်းၶႂႃသုတ်း။
* **center:** ၶပ်ႉဝႆႉၵႄႈၵၢင်။
* **space-between:** ၶပ်ႉႁၢင်ႇၵၼ် ႁႂ်ႈမီးဢွင်ႈပဝ်ႇ ၼႂ်းၵႄႈၽဵင်ႇၵၼ်။

---

### 4. တူဝ်ယၢင်ႇလွင်ႈၸႂ်ႉတိုဝ်း

သင်ႁဝ်းၶႂ်ႈႁဵတ်း Menu တွၼ်ႈတႃႇဝႅပ်ႉသၢႆႉတႆးႁဝ်း ႁႂ်ႈမၼ်းၸုၵ်းႁၢင်ႇၵၼ်ႁၢင်ႈလီလီၼႆၸိုင်:

```css
.menu-container {
  display: flex;
  flex-direction: row;
  justify-content: space-between; /* ႁႂ်ႈ Menu ႁၢင်ႇၵၼ်ၽဵင်ႇၵၼ် */
  background-color: #AA0000;
}

```

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **display: flex;** ပဵၼ်ၵၢၼ်တႄႇၸႂ်ႉ Flexbox တီႈ Parent။
* **Flex-direction:** မၵ်ႉမၼ်ႈပၼ်ဝႃး တေၶပ်ႉလူၺ်ႈ တၢင်းၼွၼ်း (row) ဢမ်ႇၼၼ် တၢင်းၸုၵ်း (column)။
* **Justify-content:** ၸတ်းဢွင်ႈပဝ်ႇ ၼႂ်းၵႄႈ Elements ႁႂ်ႈႁၢင်ႈလီ။
