
## Lesson 15: Flexbox Basics

Flexbox (Flexible Box) ပဵၼ်ပိူင်ၸတ်းဝၢင်း "ႁပ်း" (Box) ႁႂ်ႈမၼ်းၶၢႆႉလႆႈ၊ ယိုတ်းလႆႈ၊ လႄႈ ႁူတ်းလႆႈ ၸွမ်းၼင်ႇတၢင်းၵႂၢင်ႈၼႃႈจอၶႃႈ။

### 1. ပေႃႈမႄႈ လႄႈ လုၵ်ႈ (Container & Items)

တွၼ်ႈတႃႇတေၸႂ်ႉ Flexbox လႆႈၼၼ်ႉ ႁဝ်းလူဝ်ႇမီးသွၼ်ၸၼ်ႉ:

1. **Flex Container (ပေႃႈမႄႈ):** ႁပ်းဢၼ်ႁွပ်ႈဝႆႉ။ ႁဝ်းတေသင်ႇ `display: flex;` တီႈၼႆႈ။
2. **Flex Items (လုၵ်ႈ):** ႁပ်းဢၼ်ယူႇတၢင်းၼႂ်း။ မၼ်းတေတူင်ႉၼိုင်ၸွမ်းၵမ်းသိုဝ်ႈ။

**တူဝ်ယၢင်ႇ Code:**

```css
.container {
  display: flex; /* ၼႆႉပဵၼ်ၵၢၼ်ပိုတ်ႇၸႂ်ႉ Flexbox */
}

```

---

### 2. Flex Direction (တၢင်းၵႂႃႇ)

မိူဝ်ႈႁဝ်းပိုတ်ႇ `display: flex;` ယဝ်ႉ၊ ႁဝ်းၸၢင်ႈလိူၵ်ႈလႆႈဝႃႈ ၶႂ်ႈႁႂ်ႈ "လုၵ်ႈ" မၼ်း ၸုၵ်းပဵၼ်ထႅဝ်ၼွၼ်း ဢမ်ႇၼၼ် ထႅဝ်ၸုၵ်း။

* **row (ပိူင်ၵဝ်ႇ):** ၸုၵ်းเรียงၵၼ် တၢင်းၼွၼ်း (သၢႆႉၵႂႃႇၶႂႃ)။
* **column:** ၸုၵ်းเรียงၵၼ် တၢင်းၸုၵ်း (ၼိူဝ်လူင်းတႂ်ႈ)။

---

### 3. Justify Content (ၸတ်းဝၢင်းတၢင်းၼွၼ်း)

ဢၼ်ၼႆႉၸႂ်ႉတွၼ်ႈတႃႇ ၸတ်းဢွင်ႈလွၵ်ႇၼႂ်းၵႄႈ "လုၵ်ႈ" ႁဝ်းၶႃႈ:

* **flex-start:** ၵႂႃႇၸုၵ်းတၢင်းသၢႆႉသုတ်း။
* **flex-end:** ၵႂႃႇၸုၵ်းတၢင်းၶႂႃသုတ်း။
* **center:** ၸုၵ်းဝႆႉၵႄႈၵၢင်။
* **space-between:** ၸုၵ်းႁၢင်ႇၵၼ် ႁႂ်ႈမီးဢွင်ႈလွၵ်ႇၼႂ်းၵႄႈၽဵင်ႇၵၼ်။

---

### 4. တူဝ်ယၢင်ႇလွင်ႈၸႂ်ႉတိုဝ်း

သင်ႁဝ်းၶႂ်ႈႁဵတ်း Menu တွၼ်ႈတႃႇဝႅပ်ႉသၢႆႉတႆးႁဝ်း ႁႂ်ႈမၼ်းၸုၵ်းเรียงၵၼ်ႁၢင်ႈလီ:

```css
.menu-container {
  display: flex;
  flex-direction: row;
  justify-content: space-between; /* ႁႂ်ႈ Menu ႁၢင်ႇၵၼ်ၽဵင်ႇၵၼ် */
  background-color: #AA0000;
}

```

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **display: flex;** ပဵၼ်ၵၢၼ်တႄႇၸႂ်ႉ Flexbox တီႈ Parent။
* **Flex-direction:** တႅတ်ႈတေႃးဝႃႈ တေၵႂႃႇတၢင်းၼွၼ်း (row) ဢမ်ႇၼၼ် တၢင်းၸုၵ်း (column)။
* **Justify-content:** ၸတ်းဢွင်ႈလွၵ်ႇ ၼႂ်းၵႄႈ Elements ႁႂ်ႈႁၢင်ႈလီ။
