## Lesson 7: Display Property

ၼႂ်း CSS ၼၼ်ႉ `display` ပဵၼ် Property ဢၼ်လုမ်းလႃးလွင်ႈတူင်ႉၼိုင်ၶွင် "ႁပ်း" ၶႃႈ။ ပိူင်လူင်မၼ်းမီးယူႇ 3 ယိူင်ႈ ဢၼ်ႁဝ်းလူဝ်ႇပွင်ႇၸႂ်ဝႆႉ:

### 1. Block (ပိူင်ႁပ်းတဵမ်)

Elements ဢၼ်ပဵၼ် `block` ၼၼ်ႉ မၼ်းတေၸိူတ်ႇဢဝ် "တီႈလွၵ်ႇတင်းထႅဝ်" ၶႃႈ။ မၼ်းတေဢမ်ႇယွမ်းပၼ်ၽႂ်မႃးၸုၵ်းၶႅၼ်းၶၢင်ႈမၼ်း။

* **တူဝ်ယၢင်ႇ:** `<h1>`, `<p>`, `<div>`, `<ul>`
* **ၾၢင်ႁၢင်ႈ:** ၸၢင်ႈပၼ် Width လႄႈ Height လႆႈ။ မၼ်းတေတႄႇထႅဝ်မႂ်ႇ (New Line) ၵူႈပွၵ်ႈ။

---

### 2. Inline (ပိူင်ထႅဝ်လဵဝ်)

Elements ဢၼ်ပဵၼ် `inline` ၼၼ်ႉ မၼ်းတေၸုၵ်းဝႆႉ "ၸွမ်းၼင်ႇၼိူဝ်ႉလိၵ်ႈမီး" ၵွၺ်းၶႃႈ။ မၼ်းယွမ်းပၼ် Elements တၢင်ႇဢၼ် မႃးၸုၵ်းၶႅၼ်းၶၢင်ႈမၼ်းလႆႈ။

* **တူဝ်ယၢင်ႇ:** `<span>`, `<a>`, `<strong>`
* **ၾၢင်ႁၢင်ႈ:** **ဢမ်ႇၸၢင်ႈ** ပၼ် Width လႄႈ Height လႆႈ။ မၼ်းတေၸုၵ်းသိုပ်ႇၵၼ်ၵႂႃႇတိၵ်းတိၵ်း။

---

### 3. Inline-Block (ပိူင်လေႃးၵၼ်)

ဢၼ်ၼႆႉပဵၼ် "လုၵ်ႈလေႃး" ၶႃႈ။ မၼ်းမီးၵုင်ႇသမ်ႇပတ်ႉတီႇတင်း Block လႄႈ Inline လေႃးၵၼ်ဝႆႉ။

* **ၾၢင်ႁၢင်ႈ:** ၸၢင်ႈပၼ် Width လႄႈ Height လႆႈ (မိူၼ် Block)၊ ၵွၺ်းၵႃႈ မၼ်းယွမ်းၸုၵ်းသိုပ်ႇထႅဝ်လဵဝ်ၵၼ်လႆႈ (မိူၼ် Inline)။
* **လွင်ႈၸႂ်ႉတိုဝ်း:** သူင်ၸႂ်ႉတွၼ်ႈတႃႇႁဵတ်း ပုမ်ႇ (Buttons) ဢမ်ႇၼၼ် Navigation Menu ၶႃႈ။

---

### တူဝ်ယၢင်ႇလွင်ႈတႅမ်ႈ Code

```css
/* ႁဵတ်းႁႂ်ႈ Link (a) ဢၼ်ပဵၼ် Inline ပဵၼ်ၵႂႃႇမိူၼ် Button */
a {
  display: inline-block;
  width: 120px;
  padding: 10px;
  background-color: blue;
  color: white;
  text-align: center;
}

```

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **Block:** တဵမ်တင်းထႅဝ်၊ ၶိုၼ်ႈထႅဝ်မႂ်ႇတိုင်ႇ၊ ပၼ် Width လႆႈ။
* **Inline:** ယူႇၸွမ်းၼိူဝ်ႉလိၵ်ႈ၊ ဢမ်ႇၶိုၼ်ႈထႅဝ်မႂ်ႇ၊ ပၼ် Width ဢမ်ႇလႆႈ။
* **Inline-Block:** ယူႇထႅဝ်လဵဝ်ၵၼ်လႆႈ၊ ပၼ် Width လႆႈ (**ၶႅမ်ႉသုတ်းတႃႇႁဵတ်း Menu**)။
