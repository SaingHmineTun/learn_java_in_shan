## Lesson 7: Display Property

ၼႂ်း CSS ၼၼ်ႉ `display` ပဵၼ် Property ဢၼ်လုမ်းလႃး လွင်ႈတူင်ႉၼိုင်ၶွင် "Box" ယဝ်ႉ။ 
display (3) လွင်ႈ ဢၼ်ႁဝ်းလူဝ်ႇပွင်ႇၸႂ်ၼႆႉ ပဵၼ်ၼင်ႇတီႈတႂ်ႈယဝ်ႉ:

### 1. Block (ပိူင်ႁပ်းတဵမ်)

Elements ဢၼ်ပဵၼ် `block` ၼၼ်ႉ မၼ်းတေဢဝ် "တီႈယူႇတင်းထႅဝ်" ယဝ်ႉ။ မၼ်းတေဢမ်ႇယွမ်းပၼ်ၽႂ်မႃး မီးၼႂ်းမၼ်း။ သင်ၸႂ်ႉမၼ်းၼႆ တၢင်ႇလွင်ႈၼႆ့ တေလူင်းမီးတီႈတႂ်ႈမၼ်းယဝ်ႉ။

* **တူဝ်ယၢင်ႇ:** `<h1>`, `<p>`, `<div>`, `<ul>`
* **ၾၢင်ႁၢင်ႈ:** ၸၢင်ႈပၼ် Width လႄႈ Height လႆႈ။ မၼ်းတေတႄႇထႅဝ်မႂ်ႇ ၵူႈပွၵ်ႈ။

---

### 2. Inline (ပိူင်ထႅဝ်လဵဝ်)

Elements ဢၼ်ပဵၼ် `inline` ၼၼ်ႉ မၼ်းတေၸုၵ်းဝႆႉ "ၸွမ်းၼင်ႇၼိူဝ်ႉလိၵ်ႈမီး" ၵွၺ်း။ မၼ်းယွမ်းပၼ် Elements တၢင်ႇဢၼ် မႃးၸုၵ်းၶၢင်ႈမၼ်းလႆႈ။

* **တူဝ်ယၢင်ႇ:** `<span>`, `<a>`, `<strong>`
* **ၾၢင်ႁၢင်ႈ:** **ဢမ်ႇၸၢင်ႈ** ပၼ် Width လႄႈ Height လႆႈ။ မၼ်းတေ သိုပ်ႇၵၼ်ၵႂႃႇတိၵ်းတိၵ်း။

---

### 3. Inline-Block (ပိူင်လေႃးၵၼ်)

ဢၼ်ၼႆႉပဵၼ် "လုၵ်ႈၸၢပ်ႈ" ယဝ်ႉ။ မၼ်းပဵၼ်ဢၼ်ဢဝ် Block လႄႈ Inline လေႃးၵၼ်ဝႆႉယဝ်ႉ။

* **ၾၢင်ႁၢင်ႈ:** ၸၢင်ႈပၼ် Width လႄႈ Height လႆႈ (မိူၼ် Block)၊ ၵွၺ်းၵႃႈ မၼ်းတေၶပ်ႉပၼ် ထႅဝ်လဵဝ်ၵၼ်လႆႈ (မိူၼ် Inline)။
* **လွင်ႈၸႂ်ႉတိုဝ်း:** ၵမ်ႈၼမ် ၸႂ်ႉတိုဝ်း တွၼ်ႈတႃႇႁဵတ်း တုမ်ႇ (Buttons) ဢမ်ႇၼၼ် Navigation Menu ယဝ်ႉ။

---

### တူဝ်ယၢင်ႇလွင်ႈတႅမ်ႈ Code

```css
/* ႁဵတ်းႁႂ်ႈ Link (a) ဢၼ်ပဵၼ် Inline ပဵၼ်ၵႂႃႇမိူၼ် Button */
a {
  display: inline-block;
  width: 120px;
  padding: 10px;
  background-color: blue;
  color: white;
  text-align: center;
}

```

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **Block:** တဵမ်တင်းထႅဝ်၊ ၶိုၼ်ႈပၼ်ထႅဝ်မႂ်ႇ၊ ပၼ် Width လႆႈ။
* **Inline:** ယူႇၸွမ်းၼိူဝ်ႉလိၵ်ႈ၊ ဢမ်ႇၶိုၼ်ႈထႅဝ်မႂ်ႇ၊ ပၼ် Width ဢမ်ႇလႆႈ။
* **Inline-Block:** ယူႇထႅဝ်လဵဝ်ၵၼ်လႆႈ၊ ပၼ် Width လႆႈ (**ၶႅမ်ႉသုတ်းတႃႇႁဵတ်း Menu**)။
