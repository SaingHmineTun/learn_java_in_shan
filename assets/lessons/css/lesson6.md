## Lesson 6: Box Sizing

မိူဝ်ႈႁဝ်းတႅမ်ႈ CSS ၼၼ်ႉ ၸိူဝ်းပဵၼ် Padding လႄႈ Border မၼ်းၸၢင်ႈႁဵတ်းႁႂ်ႈ "ႁပ်း" ႁဝ်း ယႂ်ႇလိူဝ်သေ ၶၼၢတ်ႈ (Width) ဢၼ်ႁဝ်းပၼ်ဝႆႉၼၼ်ႉၶႃႈ။ လွင်ႈၼႆႉႁဵတ်းႁႂ်ႈ Layout ႁဝ်းၵႂႃႇယႃႉၵၼ်။

### 1. ပၼ်ႁႃၶွင် Content-box (ပိူင်ၵဝ်ႇ)

ပိူင်ၵဝ်ႇ (Default) ၼႆႉ ႁဝ်းႁွင်ႉဝႃႈ `content-box` ၶႃႈ။ ပေႃးႁဝ်းပၼ် Width ဝႆႉ 300px ယဝ်ႉ သမ်ႉၵႂႃႇသႂ်ႇ Padding 20px ၼႆ - ႁပ်းႁဝ်းတေပဵၼ်ၵႂႃႇ **340px** ၶႃႈယဝ်ႉ။

* **လၢႆးၵေပ်းသဵၼ်ႈ:** `Width + Padding + Border = Total Width`
* **ပၼ်ႁႃ:** မၼ်းႁဵတ်းႁႂ်ႈႁဝ်းၵေပ်းသဵၼ်ႈယၢပ်ႇ လႄႈ ဝႅပ်ႉသၢႆႉႁဝ်းတႅၵ်ႇငၢႆႈ။

---

### 2. ၶေႃႈၵႄႈလႅတ်း: Border-box (ပိူင်မႂ်ႇ)

တွၼ်ႈတႃႇၵႄႈပၼ်ႁႃၼႆႉ ႁဝ်းၸႂ်ႉ Property ဢၼ်ႁွင်ႉဝႃႈ `box-sizing: border-box;` ၶႃႈ။ ပေႃးႁဝ်းၸႂ်ႉပိူင်ၼႆႉ - Padding လႄႈ Border တေၶဝ်ႈၵႂႃႇယူႇ **ၼႂ်း** Width ဢၼ်ႁဝ်းပၼ်ဝႆႉၼၼ်ႉၵမ်းသိုဝ်ႈၶႃႈ။

* **တူဝ်ယၢင်ႇ:**

```css
.box {
  width: 300px;
  padding: 20px;
  border: 5px solid black;
  box-sizing: border-box; /* ၶၼၢတ်ႈႁပ်းတင်းသဵင်ႈ တေၵိုတ်း 300px ၵွၺ်း */
}

```

> **မၢႆတွင်း:** ပေႃးၸႂ်ႉ `border-box` ယဝ်ႉ ႁပ်းႁဝ်းတေဢမ်ႇယႂ်ႇလိူဝ်ၵဝ်ႇ၊ မၼ်းတေတႅတ်ႈတေႃးၸွမ်း Width ဢၼ်ႁဝ်းပၼ်ဝႆႉၼၼ်ႉၶႃႈ။

---

### 3. လၢႆးၸႂ်ႉတိုဝ်းႁႂ်ႈၵတ်ႉၶႅၼ်ႇ (Best Practice)

တွၼ်ႈတႃႇဢမ်ႇလႆႈတႅမ်ႈ `box-sizing` တီႈၵူႈ Element ၼၼ်ႉ၊ ၸိူဝ်းပဵၼ် Web Developer ၶဝ် သူင်ၸႂ်ႉ **Universal Selector (*)** တႃႇပၼ်ၵႂၢမ်းသင်ႇ တင်းၼႃႈဝႅပ်ႉၼင်ႇၼႆၶႃႈ:

```css
* {
  box-sizing: border-box;
}

```

*Code ၼႆႉ တေႁဵတ်းႁႂ်ႈ "ႁပ်း" တင်းသဵင်ႈၼႂ်းဝႅပ်ႉႁဝ်း ၵေပ်းသဵၼ်ႈငၢႆႈ လႄႈ ၸတ်း Layout ငၢႆႈၶႃႈယဝ်ႉ။*

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **Content-box:** Padding လႄႈ Border တေႁဵတ်းႁႂ်ႈႁပ်းယႂ်ႇၶိုၼ်ႈ (ၵေပ်းသဵၼ်ႈယၢပ်ႇ)။
* **Border-box:** Padding လႄႈ Border တေယူႇၼႂ်း Width (ၵေပ်းသဵၼ်ႈငၢႆႈ - **လီသုတ်း**)။
