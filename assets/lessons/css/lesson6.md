## Lesson 6: Box Sizing

မိူဝ်ႈႁဝ်းတႅမ်ႈ CSS ၼၼ်ႉ ၸိူဝ်းပဵၼ် Padding လႄႈ Border မၼ်းၸၢင်ႈႁဵတ်းႁႂ်ႈ "Box" ႁဝ်း ယႂ်ႇလိူဝ်သေ တင်းၵႂၢင်ႈ (Width) ဢၼ်ႁဝ်းပၼ်ဝႆႉၼၼ်ႉယဝ်ႉ။ 
လွင်ႈၼႆႉႁဵတ်းႁႂ်ႈ Layout ႁဝ်းၵႂႃႇယႃႉၵၼ်လႄႈ တူၺ်းဢမ်ႇႁၢင်ႈလီယဝ်ႉ။

### 1. ပၼ်ႁႃၶွင် Content-box (ပိူင်ၵဝ်ႇ)

ပိူင်ၵဝ်ႇ (Default) ၼႆႉ ႁဝ်းႁွင်ႉဝႃႈ `content-box` ၶႃႈ။ 
ပေႃးႁဝ်းပၼ် Width ဝႆႉ 300px ယဝ်ႉ သမ်ႉၵႂႃႇသႂ်ႇ Padding 20px ၼႆ Box ႁဝ်းတေပဵၼ်ၵႂႃႇ **340px** ယဝ်ႉ။

* **လၢႆးၼပ်ႉသွၼ်ႇ:** `Width + Padding + Border = Total Width`
* **ပၼ်ႁႃ:** မၼ်းႁဵတ်းႁႂ်ႈႁဝ်း ၼပ်ႉသွၼ်ႇယၢပ်ႇယဝ်ႉ။

---

### 2. ၶေႃႈၵႄႈလႅတ်း: Border-box (ပိူင်မႂ်ႇ)

တွၼ်ႈတႃႇၵႄႈပၼ်ႁႃၼႆႉ ႁဝ်းၸႂ်ႉ Property ဢၼ်ႁွင်ႉဝႃႈ `box-sizing: border-box;` ယဝ်ႉ။ 
ပေႃးႁဝ်းၸႂ်ႉပိူင်ၼႆႉ - Padding လႄႈ Border တေၶဝ်ႈၵႂႃႇယူႇ **ၼႂ်း** Width ဢၼ်ႁဝ်းပၼ်ဝႆႉၼၼ်ႉ ၵမ်းသိုဝ်ႈၶႃႈ။

* **တူဝ်ယၢင်ႇ:**

```css
.box {
  width: 300px;
  padding: 20px;
  border: 5px solid black;
  box-sizing: border-box; /* တင်းၵႂၢင်ႈ Box တင်းသဵင်ႈ တေၵိုတ်း 300px ၵွၺ်း */
}

```

> **မၢႆတွင်း:** ပေႃးၸႂ်ႉ `border-box` ယဝ်ႉ Box ႁဝ်းတေဢမ်ႇယႂ်ႇလိူဝ်ၵဝ်ႇ၊ မၼ်းတေတႅတ်ႈတေႃးၸွမ်း Width ဢၼ်ႁဝ်းပၼ်ဝႆႉၼၼ်ႉယဝ်ႉ။

---

### 3. လၢႆးၸႂ်ႉတိုဝ်းႁႂ်ႈၵတ်ႉၶႅၼ်ႇ (Best Practice)

တွၼ်ႈတႃႇဢမ်ႇလႆႈတႅမ်ႈ `box-sizing` တီႈ Element ၵူႈဢၼ်ၼၼ်ႉ၊ ၸိူဝ်းပဵၼ် Web Developer ၶဝ် သူင်ၸႂ်ႉ **Universal Selector (*)** တႃႇပၼ်ၵႂၢမ်းသင်ႇ တင်းၼႃႈဝႅပ်ႉၼင်ႇၼႆယဝ်ႉ:

```css
* {
  box-sizing: border-box;
}

```

*Code ၼႆႉ တေႁဵတ်းႁႂ်ႈ "Box" တင်းသဵင်ႈၼႂ်းဝႅပ်ႉႁဝ်း ၼပ်ႉသွၼ်ႇငၢႆႈ လႄႈ ၸတ်း Layout ငၢႆႈၶႃႈယဝ်ႉ။*

---

### ၶေႃႈၵိုတ်းယတ်း (Summary)

* **Content-box:** Padding လႄႈ Border တေႁဵတ်းႁႂ်ႈ Box ယႂ်ႇၶိုၼ်ႈ။
* **Border-box:** Padding လႄႈ Border တေယူႇၼႂ်း Width။
