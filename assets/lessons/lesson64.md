
## **Java Basic 64**

#### **MULTI-THREADED PROGRAMMING**

Link - <https://youtu.be/r7J9_4CTlD0>

Java ၼႆႉပဵၼ် multi-threaded programming ဢိူဝ်ႈ။ Thread ၼႆႉ မၼ်းမိူၼ် ၵူၼ်းႁဵတ်ႉၵၢၼ် ယဝ်ႉ။ ထိုင်တေႃႈလဵဝ် Java program ႁဝ်းၼႆႉ မၼ်းမီး Thread ဢၼ်လဵဝ်ၵူၺ်းလႄႈ မၼ်းပဵၼ် main thread ယဝ်ႉ။ ၵွပ်ႈၼႆ မၢင်ပွၵ်ႈ သင်ၸိူဝ်ႉ ႁဝ်းထူပ်းမႅၼ်ႈ infinite loop ၼႆ မၼ်းတေ ဢမ်ႇၸၢင်ႊလူင်းၵႂႃႇ တီႈတႂ်ႈမၼ်းယဝ်ႉ။

ဝႆႉဝႃႈ ႁဝ်းလူတ်ႇလူင်းဢဝ် Software သေဢၼ်ဢၼ် တီႈ Chrome ၼႆလႄႈ။ သင်ၸိူဝ်ႉ ဢမ်ႇၸႂ်ႉတိုဝ်း Thread ၼႆၸိုင် မိူဝ်ႈႁဝ်း တိုၵ်ႉလူတ်ႇလူင်း သေဢၼ်ဢၼ်ယူႇၼၼ်ႉ မၼ်းတေဢမ်ႇသိုပ်ႇ ႁဵတ်ႉၵၢၼ်သင်လႆႈယဝ်ႉ။ ႁဝ်းတေ ပၢၵ်ႊသၢႆႉပၢၵ်ႊၶႂႃၵေႃႈ ဢမ်ႇလႆႈ။ App ႁဝ်းၼႆႉ တေၵိုတ်း တေ freeze ဝႆႉယူႇယဝ်ႉ။ ၵွပ်ႈၼႆ Thread ၼႆႉၵေႃႈ ပဵၼ်ပိူင်ယႂ်ႇဢၼ်ၼိုင်ႈ တႃႇႁဵတ်ႉႁႂ်ႈ App ႁဝ်းႁဵတ်ႉၵၢၼ်ၽႂ်း ယူႇယဝ်ႉ။
