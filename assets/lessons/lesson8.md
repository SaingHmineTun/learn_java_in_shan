
## **Java Basic 8**

#### **UNARY OPERATORS**

Link - <https://youtu.be/f0W1zWVpuws>

Operator ဢၼ်မီး operand ဢၼ်လဵဝ်ၼႆႉ ႁဝ်းႁွင်ႉဝႃႈ Unary operator ၼႆယဝ်ႉ။ Unary Operator ၼႆႉ တေမီးဝႆႉ 5 မဵဝ်းၶႃႈ။

- _\+ (Positive)_
- _\- (Negative)_
- _++ (Increment)_
- _\-- (Decrement)_
- _! (NOT)_

###### လၢႆးၸႂ်ႉတိုဝ်း

```
int i = -5; // negative operator  
boolean b = !false; // true
int x = ++i; // -4
```

**Prefix & Postfix operator**

Increment လႄႈ Decrement operators သွင်ဢၼ်ၼႆႉ ယူႇတီႈ ဢၼ်ႁဝ်းဝႆႉဝၢင်း operator သေ မၼ်းတေမီးၵႂႃႇ prefix လႄႈ postfix ၼႆယဝ်ႉ။

**Prefix** ၼႆႉ ပဵၼ်ဢၼ် ႁဝ်းဢဝ် operator ၼႆႉ ဝႆႉဢွၼ်တၢင်း operand ယဝ်ႉ။ သင်ၼႆ မၼ်းတေ လေႃးပၼ် ၵႃႊၶၼ်ဢွၼ်တၢင်းယဝ်ႉ။ ၶူတ်ႉတီႈတႂ်ႈၼႆႉ ၵႃႊၶၼ် y ၼႆႉ မၼ်းပဵၼ် 6 ဢိူဝ်ႈ။

###### လၢႆးတႅမ်ႈ Prefix

```
int x = 5;
int y = ++x;
```

**Postfix** ၼႆႉသမ်ႉ ပဵၼ်လွင်ႈ ဢၼ်ႁဝ်းဢဝ် operator ၼႆႉ ဝႆႉၽၢႆႇလင် operand ဢိူဝ်ႈ။ ပေႃးၼၼ် မၼ်းတေ ဢဝ်ၵႃႊၶၼ်ဢွၼ်တၢင်းယဝ်ႉ ၸင်ႇသႂ်ႇၼႆလႄႈ ၶူတ်ႉတီႈတႂ်ႈၼႆႉ မၼ်းတေတိုၵ်ႉပဵၼ် 5 ဝႆႉၵူၺ်း။

###### လၢႆးတႅမ်ႈ Postfix

```
int x = 5;
int y = x++;
```
