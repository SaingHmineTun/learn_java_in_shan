
## **Java Basic 19**

#### **IF CONDITIONAL STATEMENT - PART 2**

Link - <https://youtu.be/K-V8Kr0o8NQ>

လိူဝ်သေ if လႄႈ else ႁဝ်းၸႂ်ႉတိုဝ်းလႆႈ else if ဢိူဝ်ႈ။ မၼ်းၵေႃႈ တေႁပ်ႉၶၢမ်ႇဝႆႉ boolean လႄႈ သင်ၸိူဝ်ႉ ၶူတ်ႉဢၼ်ၸႅတ်ႈဝႆႉ တီႈၼိူဝ်မၼ်း ဢမ်ႇႁဵတ်းၵၢၼ်၊ ၸင်ႇၶိုၼ်း ၶဝ်ႈၸႅတ်ႈၼႂ်းမၼ်းထႅင်ႈယဝ်ႉ။ တီႈ if statement ၼႆႉ if လႄႈ else တေလႆႈမီး ၼိုင်ႈထႅဝ်ၵူၺ်းသေတႃႉ else if တေႉ ႁဝ်းတႅမ်ႈ လၢႆထႅဝ်ၵေႃႈလႆႈ၊ ဢမ်ႇတႅမ်ႈသေထႅဝ်ၵေႃႈ လႆႈယဝ်ႉ။

###### လၢႆးၸႂ်ႉတိုဝ်း
```
int num = 6;
if (num > 0) {
    System.out.println("Positive Value");
} else if (num < 0) {
    System.out.println("Negative Value");
} else {
    System.out.println("Zero");
}
```
