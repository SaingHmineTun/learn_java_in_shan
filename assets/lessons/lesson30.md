
## **Java Basic 30**

#### **STRING TO INT**

Link - <https://youtu.be/4F_eT_Ns8D0>

String ဢၼ်ဝႃႈၼႆႉ တီႈၼႂ်းမၼ်းတေတႅမ်ႈလႆႈတၢင်းမူတ်း။ ၵွပ်ႈပိူဝ်ႈၼႆ ႁဝ်းၶႆႈတႅမ်ႈ "123" ၸိူဝ်းၼႆႉၵေႃႈ မၼ်းတေပဵၼ် String ဝႆႉၵူၺ်းဢိူဝ်ႈ။ တီႈၼႆႈ မၢင်ပွၵ်ႈမၢင်လႂ် ႁဝ်းသမ်ႉတေၶႆႈဢဝ် "123" (String) ၼႆႉ ပိၼ်ႇပဵၼ် 123 (int) သေ ဢဝ်မႃးႁဵတ်းၵၢၼ် ၵဵဝ်ႇလူၺ်ႈပၢႆးၼပ်ႉၼႆ မၼ်းတေယၢပ်ႇၸႂ်ဢိူဝ်ႈ။ ၵွပ်ႈပိူဝ်ႈဝႃႈ String ဢၼ်ဝႃႈၼၼ်ႉ မၼ်းၵႂႃႇလေႃးၸွမ်းၽႂ်ၵေႃႈလီ မၼ်းတေပဵၼ် String concat ၵႂႃႇၵူၺ်း။

တႃႇႁဝ်းတေဢဝ် String ၼႆႉပိၼ်ႇပဵၼ် int ၼႆၸိုင် ႁဝ်းတေလႆႈၸႂ်ႉ ဢၼ်ႁွင်ႉဝႃႈ Integer wrapper class ယဝ်ႉ။ လွင်ႈ wrapper class ၼႆႉ ၶႃႈလႆႈလၢတ်ႈဝႆႉတီႈ ၵၢၼ်သွၼ် 45 ယူႇ။

###### လၢႆးၸႂ်ႉတိုဝ်း

```
String numString = "123";
int numInt = Integer.parseInt(numString);
```
