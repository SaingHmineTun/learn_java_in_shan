# Lesson 78: Debugging with Browser DevTools

တႅၼ်းၵၢၼ်တေၸႂ်ႉ `console.log()` ၵူႈတီႈ (ဢၼ်ႁဵတ်းႁႂ်ႈၵူတ်ႉယုင်ႈ)၊ Browser ၵူႈတူဝ် (Chrome, Firefox, Edge) မီးၶိူင်ႈမိုဝ်းၶိုၵ်ႉတွၼ်း ဢၼ်ႁွင်ႉဝႃႈ **DevTools** မႃးပၼ်ယဝ်ႉၶႃႈ။

### 1. Breakpoints (ၵၢၼ်ၵိုတ်းၶိင်ႇ)

ၼႆႉပဵၼ် Tool ဢၼ်ၵတ်ႉၶႅၼ်ႇသုတ်း။ ၸဝ်ႈၵဝ်ႇၸၢင်ႈ "ၵိုတ်း" ၵူတ်ႉဝႆႉတီႈထႅဝ်လႂ်ၵေႃႈလႆႈ သေၵူတ်ႇထတ်းတူၺ်းၵႃႈၶၼ် Variable ၼႂ်းၶိင်ႇၼၼ်ႉတေႉတေႉၶႃႈ။

* **လၢႆးႁဵတ်း:** ပိုတ်ႇ DevTools (F12) -> သိုပ်ႇၵႂႃႇတီႈ Tab **Sources** -> ၼဵၵ်းထႅဝ်ၵူတ်ႉဢၼ်ၶႂ်ႈၵိုတ်း။

---

### 2. Stepping Through Code

မိူဝ်ႈၵူတ်ႉၵိုတ်းဝႆႉယဝ်ႉ (At Breakpoint)၊ ၸဝ်ႈၵဝ်ႇၸၢင်ႈ "ပႆ" ၵႂႃႇၸွမ်းၵူတ်ႉလႆႈၸၼ်ႉတေႃႇၸၼ်ႉ:

* **Step Over:** ၶၢမ်ႈၵႂႃႇထႅဝ်ၼႃႈ။
* **Step Into:** ၶဝ်ႈၵႂႃႇတူၺ်းၼႂ်း Function (သင်ထႅဝ်ၼၼ်ႉမီး Function)။
* **Step Out:** ဢွၵ်ႇမႃးတီႈ Function ဢၼ်တိုၵ်ႉယူႇၼၼ်ႉ။

---

### 3. The Watch & Scope Panels

ၼႂ်း Panel ၽၢႆႇၶၢင်ႈ၊ ၸဝ်ႈၵဝ်ႇတေႁၼ်:

* **Scope:** တေၼႄၵႃႈၶၼ် Variable တင်းမူတ်း ဢၼ်လႅၼ်ႈယူႇယၢမ်းလဵဝ် (Local, Global)။
* **Watch:** ၸဝ်ႈၵဝ်ႇၸၢင်ႈထႅမ် Variable ဢၼ်ၸဝ်ႈၵဝ်ႇ "သူင်ၸႂ်" ဝႆႉပဵၼ်ၶူး တႃႇတူၺ်းဝႃႈၵႃႈၶၼ်မၼ်းလႅၵ်ႈလၢႆႈၵႂႃႇႁိုဝ်။

---

### 4. Network Tab (Debugging API)

သင် App ၸဝ်ႈၵဝ်ႇၵွင်ႉၸူး API သေမၼ်းဢမ်ႇႁဵတ်းၵၢၼ်၊ တူၺ်းတီႈ **Network Tab** ၶႃႈ:

* မၼ်းတေၼႄ Request တင်းမူတ်း။
* တူၺ်း **Status Code** (200=OK, 404=Not Found, 500=Server Error)။
* တူၺ်း **Payload** (ၶေႃႈမုၼ်းဢၼ်သူင်ႇၵႂႃႇ) လႄႈ **Preview** (ၶေႃႈမုၼ်းဢၼ် Server သူင်ႇမႃး)။

---

### 5. Console Tricks

ၼွၵ်ႈလိူဝ် `console.log()`၊ ၸဝ်ႈၵဝ်ႇၸၢင်ႈၸႂ်ႉ:

* `console.table(data)`: တႃႇၼႄ Object ႁိုဝ် Array ႁႂ်ႈပဵၼ် "လွၵ်းတေႃး" (Table) လူငၢႆႈ။
* `console.error()`: တႃႇၼႄ Error ပဵၼ်သီလႅင်။
* `console.time("label")` ... `console.timeEnd("label")`: တႃႇတႅၵ်ႈတူၺ်းဝႃႈ ၵူတ်ႉထႅဝ်ၼၼ်ႉၵိၼ်ၶိင်ႇႁိုင်ၵႃႈႁိုဝ်။

---

### 📝 ၶေႃႈမုၼ်းတႃႇတွၼ်း (Key Takeaway)

* ၵၢၼ်ၸႂ်ႉ **Breakpoints** ၸွႆႈႁႂ်ႈႁဝ်းႁၼ် "ပၢႆးဝူၼ်ႉ" ၶွင်ၵူတ်ႉ ဢၼ်လႅၵ်ႈလၢႆႈၵႂႃႇၸွမ်းၶိင်ႇ (State changes) လႆႈလီလိူဝ် `console.log()`။
* ယႃႇလိုမ်း "ပိတ်း" (Deactivate) Breakpoints မိူဝ်ႈၵႄႈပၼ်ႁႃယဝ်ႉယဝ်ႉ။
* Debugging ဢမ်ႇၸႂ်ႈၵၢၼ်ဢၼ် "တူၵ်းသုမ်း"၊ မၼ်းပဵၼ်ၵၢၼ် "ႁဵၼ်းႁူႉ" ဝႃႈၵူတ်ႉႁဝ်းႁဵတ်းၵၢၼ်ႁိုဝ်တေႉတေႉၶႃႈ။
