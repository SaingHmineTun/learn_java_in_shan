# Lesson 17: Nullish Coalescing (??)

မိူဝ်ႈ Lesson 16 ႁဝ်းႁဵၼ်းဝႃႈ `||` တေႁႃၵႃႈၶၼ် **Truthy**။ ၵူၺ်းၵႃႈ မၼ်းမီးပၼ်ႁႃဝႃႈ သင်ႁဝ်းမီးၵႃႈၶၼ် `0` ႁိုဝ် လိၵ်ႈပဝ်ႇ `""` ၼၼ်ႉ `||` တေၼပ်ႉပဵၼ် **Falsy** သေ တေၵႂႃႇဢဝ်ၵႃႈၶၼ် Default မႃးၵမ်းလဵဝ်။

တေႉတေႉမၼ်း `0` ၵေႃႈယဝ်ႉ `""` ၵေႃႈယဝ်ႉ မၼ်းပဵၼ် "ၶေႃႈမုၼ်း" ဢၼ်ၸဝ်ႈၵဝ်ႇၶႂ်ႈလႆႈယူႇ၊ မၼ်းဢမ်ႇၸႂ်ႈလွင်ႈၽိတ်းပိူင်ႈသင်။ ၵွပ်ႈၼၼ် ႁဝ်းၸင်ႇလူဝ်ႇၸႂ်ႉ **`??`** ၶႃႈ။

### 1. `??` ႁဵတ်းၵၢၼ်ၸိူင်ႉၼိုင်?

`??` တေတူၺ်းၵူၺ်း 2 ယၢင်ႇ: **`null`** လႄႈ **`undefined`** ၵူၺ်းၶႃႈ။

* သင်ၵႃႈၶၼ်ၽၢႆႇၼႃႈပဵၼ် `null` ႁိုဝ် `undefined`၊ မၼ်းတေၵႂႃႇဢဝ်ၵႃႈၶၼ်ၽၢႆႇလင်။
* သင်ၵႃႈၶၼ်ၽၢႆႇၼႃႈပဵၼ် `0`, `""`, ႁိုဝ် `false`၊ မၼ်းတေ **"ဢဝ်ၵႃႈၶၼ်ၼၼ်ႉၵမ်းလဵဝ်"** (ဢမ်ႇၵႂႃႇဢဝ် Default)။

---

### 2. တူၺ်းလွင်ႈပႅၵ်ႇၵၼ် (Comparison)

```javascript
let count = 0;

// ၸႂ်ႉ OR (||) - ဢဝ် 0 ပဵၼ် Falsy
const total1 = count || 10; 
console.log(total1); // 10 (⚠️ ၽိတ်းယဝ်ႉ! ႁဝ်းၶႂ်ႈလႆႈ 0)

// ၸႂ်ႉ Nullish Coalescing (??) - ဢဝ် 0 ပဵၼ်ၵႃႈၶၼ်မၢၼ်ႇ
const total2 = count ?? 10;
console.log(total2); // 0 (✅ မၢၼ်ႇယဝ်ႉ!)

```

### 3. မိူဝ်ႈလႂ်တေၸႂ်ႉ `??`?

* ၸႂ်ႉမိူဝ်ႈၸဝ်ႈၵဝ်ႇၶႂ်ႈပၼ်ၵႃႈၶၼ် Default ၵူၺ်းၵႃႈ ၸဝ်ႈၵဝ်ႇယင်းတိုၵ်ႉၶႂ်ႈႁပ်ႉဢဝ် `0` ႁိုဝ် `""` ပဵၼ်ၵႃႈၶၼ်ဢၼ်မၢၼ်ႇယူႇၼၼ်ႉၶႃႈ။

```javascript
let username = "";
const displayName = username ?? "Anonymous";
console.log(displayName); // "" (ယွၼ်ႉ "" ဢမ်ႇၸႂ်ႈ null/undefined)

let age = null;
const userAge = age ?? 18;
console.log(userAge); // 18 (ယွၼ်ႉ age ပဵၼ် null)

```

---

### 📝 ၶေႃႈမုၼ်းတႃႇတွၼ်း (Key Takeaway)

* **`||`** ႁႃ **Truthy** (0 လႄႈ "" တေထုၵ်ႇၶၢမ်ႈပႅတ်ႈ)။
* **`??`** ႁႃၵႃႈၶၼ်ဢၼ်ဢမ်ႇၸႂ်ႈ **Nullish** (`null`, `undefined`)။
* ၼႂ်းၵၢၼ်တႅမ်ႈၵူတ်ႉတေႉ၊ `??` ပဵၼ်ဢၼ် "Safe" လိူဝ်လိူဝ် တႃႇၸတ်းၵၢၼ်ၶေႃႈမုၼ်းဢၼ်မႃးတီႈ User ႁိုဝ် API ၶႃႈ။
