# Lesson 5: Conditionals (ၵၢၼ်တႅပ်းတတ်းၸႂ် - If, Else If, Else)

ၼႂ်းၵၢၼ်တႅမ်ႈၵူတ်ႉ (Coding) ၼၼ်ႉ ႁဝ်းလူဝ်ႇသင်ႇႁႂ်ႈၶွမ်ႇပိဝ်ႇတႃႇ "လိူၵ်ႈႁဵတ်း" ၸွမ်းၼင်ႇသၢႆငၢႆ (Conditions) ဢၼ်ပဵၼ်ယူႇ။ ႁဝ်းၸႂ်ႉ `if` ပိူဝ်ႈတေတႅပ်းတတ်းၸႂ်ၶႃႈ။

### 1. If Statement (သင်ဝႃႈ...)

`if` ပဵၼ်ပိုၼ်ႉထၢၼ်ဢၼ်ငၢႆႈသုတ်း။ သင်ဝႃႈသၢႆငၢႆၼႂ်းဝူင်းလဵၵ်း `( )` ၼၼ်ႉ ပဵၼ် **True (တေႉ)**၊ မၼ်းတေႁဵတ်းၵၢၼ်ၼႂ်း `{ }` ယဝ်ႉ။

```javascript
let speed = 90;

if (speed > 80) {
  console.log("မႂ်းလႅၼ်ႈၽႂ်းၼႃႇ! လူတ်းယွမ်းဝႆးဝႆး။");
}

```

### 2. Else Statement (သင်ဢမ်ႇၸႂ်ႈၼႆၸိုင်...)

မိူဝ်ႈသၢႆငၢႆ `if` ၼၼ်ႉပဵၼ် **False (ဢမ်ႇတေႉ)**၊ ႁဝ်းၸႂ်ႉ `else` ပိူဝ်ႈတေသင်ႇႁႂ်ႈမၼ်းႁဵတ်း ထႅင်ႈလွင်ႈၼိုင်ႈ ဢၼ်ပႅၵ်ႇပိူင်ႈၵၼ်။

```javascript
let score = 45;

if (score >= 50) {
  console.log("မႂ်းဢွင်ႇယဝ်ႉ!");
} else {
  console.log("မႂ်းတူၵ်းယဝ်ႉ၊ ၶတ်းၸႂ်ထႅင်ႈၵမ်းၼိုင်ႈ။");
}

```

### 3. Else If (သင်ဝႃႈ... ထႅင်ႈလွင်ႈၼိုင်ႈ)

သင်ဝႃႈႁဝ်းမီးတၢင်းလိူၵ်ႈ လၢႆယၢင်ႇ (Multiple choices)၊ ႁဝ်းၸႂ်ႉ `else if` မႃးၵိုၵ်းၵၼ်လႆႈၼမ်ၼမ်ၶႃႈ။

```javascript
let money = 100;

if (money > 500) {
  console.log("ၵႂႃႇၵိၼ် Pizza လႆႈယူႇ။");
} else if (money >= 100) {
  console.log("ၵိၼ်ၶဝ်ႈသွႆးၵေႃႈ လႆႈယူႇၶႃႈ။");
} else {
  console.log("ၵိၼ်ၼႅမ်ႉတီႈႁိူၼ်းၵူၺ်းယဝ်ႉ။");
}

```

### 4. Comparison Operators (တူဝ်တႅၵ်ႈၼိူင်း)

ပိူဝ်ႈတေတႅၵ်ႈၼိူင်းၼႂ်း `if` လႆႈၼၼ်ႉ ႁဝ်းလူဝ်ႇႁူႉၸၵ်း တူဝ်တႅၵ်ႈၼိူင်းၸိူဝ်းၼႆႉၶႃႈ:

* **`===`** : မိူၼ်ၵၼ်တေႉတေႉ (Value + Type)
* **`!==`** : ဢမ်ႇမိူၼ်ၵၼ်
* **`>`** : ယႂ်ႇသေ
* **`<`** : လဵၵ်ႉသေ
* **`>=`** : ယႂ်ႇသေ ႁမ်ႉ မိူၼ်ၵၼ်
* **`<=`** : လဵၵ်ႉသေ ႁမ်ႉ မိူၼ်ၵၼ်

> **Ebook Note:** ၼႂ်း JavaScript ႁဝ်းထိုင်လီၸႂ်ႉ `===` (Strict Equality) တႅၼ်း `==` တႃႇသေႇ ပိူဝ်ႈတေႁႄႉၵၢင်ႈ Bugs ဢၼ်ဢမ်ႇလူဝ်ႇပဵၼ်။

### 5. Truthy လႄႈ Falsy (ၵႃႈၶၼ်ဢၼ်တေႉ လႄႈ ဢၼ်ဢမ်ႇတေႉ)

ၼႂ်း JavaScript၊ ၵႃႈၶၼ် (Value) ၵူႈယၢင်ႇ မီး "ႁႅင်းတေႉ/ဢမ်ႇတေႉ" ၼႂ်းတူဝ်မၼ်း။ သင်ႁဝ်းဢဝ်ၶေႃႈမုၼ်းၸိူဝ်းၼႆႉ သႂ်ႇၼႂ်း `if`၊ မၼ်းတေၼပ်ႉပဵၼ် **False** တူဝ်မၼ်းၵမ်းလဵဝ်:

* `false`
* `0`
* `""` (လိၵ်ႈလၢႆ)
* `null`
* `undefined`
* `NaN`

ၵႃႈၶၼ်တၢင်ႇၸိူဝ်း (လိူဝ်သေၸိူဝ်းၼိူဝ်ၼႆႉ) တေၼပ်ႉပဵၼ် **True** တင်းမူတ်းၶႃႈ။

---

### 📝 ၶေႃႈမုၼ်းတႃႇတွၼ်း (Key Takeaway for Lesson 5)

* **`if`**: ၸႂ်ႉတႃႇတႄႇၵူတ်ႇထတ်း သၢႆငၢႆ။
* **`else if`**: ၸႂ်ႉမိူဝ်ႈမီးတၢင်းလိူၵ်ႈ ထႅင်ႈလၢႆလၢႆယၢင်ႇ။
* **`else`**: ၸႂ်ႉတႃႇသၢႆငၢႆဢၼ်ၵိုတ်းဝႆႉ တင်းမူတ်း။
