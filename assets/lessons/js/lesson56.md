# Lesson 56: Callbacks & The Pyramid of Doom** ၶႃႈ။

### 1. ၵၢၼ်ၸႂ်ႉ Callback တႃႇ Asynchronous

မိူဝ်ႈႁဝ်းလူဝ်ႇႁဵတ်းၵၢၼ်ဢၼ်ၼိုင်ႈ "ပေႃးယဝ်ႉယဝ်ႉ ၸင်ႇၵွႆႈႁဵတ်းထႅင်ႈဢၼ်ၼိုင်ႈ" (မိူၼ်ၼင်ႇ: လူတ်ႇၶေႃႈမုၼ်း User -> ပေႃးလႆႈယဝ်ႉ ၸင်ႇၵႂႃႇလူတ်ႇ Post ၶွင် User ၼၼ်ႉ)၊ ႁဝ်းလႆႈသူင်ႇ Function ၶဝ်ႈၵႂႃႇၼႂ်း Function တၢင်ႇတူဝ်ၶႃႈ။

### 2. Pyramid of Doom (ႁိုဝ် Callback Hell)

ပၼ်ႁႃမၼ်းၵိူတ်ႇမိူဝ်ႈႁဝ်းမီးၵၢၼ်ဢၼ်လူဝ်ႇပႂ်ႉတေႃႇၵၼ် "ၼမ်ၼႃႇ"။ ၵူတ်ႉႁဝ်းတေတႄႇ "ယွၼ်ႇ" ၶဝ်ႈၵႂႃႇၽၢႆႇၶႂႃတိၵ်းတိၵ်း တေႃႇပေႃးမိူၼ်ႁၢင်ႈ **Pyramid** ႁိုဝ် **ၵၢတ်ႇၽီ (Hell)** ဢၼ်လူယၢပ်ႇ လႄႈ Debug ယၢပ်ႇသုတ်းၶႃႈ။

---

### 💻 တူဝ်ယၢင်ႇ (Code Example)

ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းလူဝ်ႇႁဵတ်းၵၢၼ် 3 ဢၼ်တေႃႇၵၼ်:

```javascript
// လၢႆးတႅမ်ႈဢၼ်ပဵၼ် Callback Hell
getData(function(user) {
  getPosts(user.id, function(posts) {
    getComments(posts[0].id, function(comments) {
      console.log("Comments: ", comments);
      // သင်မီးထႅင်ႈ မၼ်းတေယွၼ်ႇၶဝ်ႈၵႂႃႇတိၵ်းတိၵ်း...
    });
  });
});

```

#### ပၼ်ႁႃဢၼ်ၵိူတ်ႇၶိုၼ်ႈ:

1. **လူယၢပ်ႇ (Unreadable):** တႃႇတေတူၺ်းဝႃႈ ပူၵ်ႇ `{}` တူဝ်လႂ် ပိတ်းတီႈလႂ်ၼၼ်ႉ ယၢပ်ႇၼႃႇ။
2. **Handling Errors:** ႁဝ်းတေလႆႈတႅမ်ႈ `if (error)` ဝႆႉၼႂ်းၵူႈၸၼ်ႉ၊ ႁဵတ်းႁႂ်ႈၵူတ်ႉၼမ်လိူဝ်ၵဝ်ႇ။
3. **Hard to Maintain:** သင်ၶႂ်ႈလႅၵ်ႈလၢႆႈလမ်ႇတတ်းၵၢၼ်ႁဵတ်း (Order)၊ တေလႆႈတႅမ်ႈမႂ်ႇတင်းမူတ်း။

---

### 3. လၢႆးၵႄႈလႅတ်း

ယွၼ်ႉပၼ်ႁႃၼႆႉသေ JavaScript ၸင်ႇလႆႈထႅမ်ပၼ်ၶိူင်ႈမိုဝ်းမႂ်ႇမႃး တွၼ်ႈတႃႇႁႂ်ႈၵူတ်ႉႁဝ်း "သိုဝ်ႈ" (Flat) လိူဝ်ၵဝ်ႇ:

* **Promises** (ES6) - ဢၼ်ႁဝ်းတေႁဵၼ်းၼႂ်း Lesson ၼႃႈ။
* **Async/Await** (ES2017) - လၢႆးတႅမ်ႈဢၼ်ငၢႆႈသုတ်းၼႂ်းပၢၼ်ယၢမ်းလဵဝ်။

---

### 📝 ၶေႃႈမုၼ်းတႃႇတွၼ်း (Key Takeaway)

* **Callback** ပဵၼ်ပိုၼ်ႉထၢၼ် ၵူၺ်းၵႃႈ ဢမ်ႇလီၸႂ်ႉ "သွၼ်ႉ" ၵၼ်ၼမ်လိူဝ် 2-3 ၸၼ်ႉ။
* သင်ႁၼ်ၵူတ်ႉၸဝ်ႈၵဝ်ႇတႄႇပဵၼ်ႁၢင်ႈ **Pyramid**၊ ပွင်ႇဝႃႈယၢမ်းလဵဝ်ထုၵ်ႇလီလႅၵ်ႈၸႂ်ႉ **Promises** ယဝ်ႉၶႃႈ။
