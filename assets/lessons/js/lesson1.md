# Lesson 1: JavaScript ႁဵတ်းၵၢၼ်ၸိူင်ႉႁိုဝ်? (How JS Runs)

မိူဝ်ႈပႆႇတႅမ်ႈၵူတ်ႉ (Code) ၼၼ်ႉ ႁဝ်းလူဝ်ႇႁူႉဝႃႈ "မၼ်းပဵၼ်သင်?" လႄႈ "မၼ်းႁဵတ်းၵၢၼ်ၸိူင်ႉႁိုဝ်?" ၼႂ်းၶွမ်ႇပိဝ်ႇတႃႇႁဝ်း။

### 1. V8 Engine (ၸၵ်ႉလူင် V8)

JavaScript ၼႆႉ မၼ်းဢမ်ႇၸႂ်ႈ လိၵ်ႈဢၼ်ၶွမ်ႇပိဝ်ႇတႃႇ တေပွင်ႇၸႂ်လႆႈၵမ်းလဵဝ်။ မၼ်းလူဝ်ႇမီး "ၸၵ်ႉပိၼ်ႇလိၵ်ႈ" (Translator)။

* **V8 Engine** ၼႆႉ ပဵၼ်ၸၵ်ႉဢၼ် Google ႁဵတ်းဢွၵ်ႇဝႆႉ (မီးၼႂ်း Chrome)။
* မၼ်းတေလူလိၵ်ႈ JS ႁဝ်းယဝ်ႉ ပိၼ်ႇပဵၼ် **Machine Code** (လိၵ်ႈၸၵ်ႉ 0101) ၼင်ႇႁႂ်ႈ Computer ႁဵတ်းၵၢၼ်လႆႈၽႂ်းၽႂ်း။

### 2. Single-threaded (ၵၢၼ်ႁဵတ်းသၢင်ႈသဵၼ်ႈလဵဝ်)

JavaScript ၼႆႉ ပဵၼ် **Single-threaded** တူဝ်မၼ်းမီး "မိုဝ်း" ၶွင်ႉလဵဝ်ၵူၺ်း။

* မၢႆထိုင်ဝႃႈ မၼ်းႁဵတ်းၵၢၼ်လႆႈ **ၼိုင်ႈယၢင်ႇ တေႃႇ ၼိုင်ႈၶၢဝ်း** (One thing at a time)။
* မၼ်းတေဢမ်ႇႁဵတ်းၵၢၼ်သွင်ယၢင်ႇ ပိုင်ႈၵၼ်။ သင်ဝႃႈၵၢၼ်ဢၼ်ၼိုင်ႈ ပႆႇယဝ်ႉ၊ မၼ်းတေဢမ်ႇၵႂႃႇႁဵတ်းဢၼ်မႂ်ႇ။

### 3. The Call Stack (ၶူဝ်းၶႅပ်းၵၢၼ်)

တေႃးၼႆႉ မိူၼ်ၼင်ႇ "ၶႅပ်းပိၵ်ႉ" ဢၼ်ႁဝ်းဝၢႆႇဝႆႉ။

* မိူဝ်ႈႁဝ်းႁွင်ႉၸႂ်ႉ Function (ၵၢၼ်ႁဵတ်းသၢင်ႈ) ၼိုင်ႈဢၼ်၊ မၼ်းတေဢဝ်ၵႂႃႇဝႆႉၼိူဝ်သုတ်း (Top)။
* ပေႃးၵၢၼ်ၼၼ်ႉယဝ်ႉ၊ မၼ်းတေၸင်ႇထွၼ်ဢွၵ်ႇ (Pop)။
* လွၵ်းလၢႆးၼႆႉ ႁဝ်းႁွင်ႉဝႃႈ **LIFO** (Last In, First Out) - ဢၼ်ၶဝ်ႈလိုၼ်းသုတ်း တေဢွၵ်ႇၵႂႃႇဢွၼ်တၢင်းပိူၼ်ႈ။

### 4. The Event Loop (ဝူင်းၵၢၼ်ပၼ်ႇ)

ပေႃး JS ႁဵတ်းၵၢၼ်လႆႈ ၼိုင်ႈယၢင်ႇၵူၺ်း၊ ၵွပ်ႈသင် Website ႁဝ်းဢမ်ႇ "ၶိတ်း" (Freeze)?
ၼႆႉပဵၼ်ၵွပ်ႈမီး **Event Loop** ယဝ်ႉ။

* သင်ဝႃႈမီးၵၢၼ်ဢၼ်ၵိၼ်ၶၢဝ်းယၢမ်းႁိုင် (မိူၼ်ၼင်ႇလူတ်ႇႁူပ်ႉၾၢင် တေႃႇၼႅတ်ႉ)၊ JS တေသူင်ႇၵၢၼ်ၼၼ်ႉၵႂႃႇဝႆႉၼႂ်း "လုမ်းလင်" (Web APIs)။
* ပေႃးၵၢၼ်ၼၼ်ႉယဝ်ႉသေ JS တူဝ်တၢင်းၼႃႈ လွတ်ႈယဝ်ႉ၊ **Event Loop** တေၸင်ႇၵႂႃႇဢဝ်ၵၢၼ်ၼၼ်ႉ မႃးသႂ်ႇၶိုၼ်းၼႂ်း Call Stack ႁႂ်ႈႁဝ်းႁဵတ်းယဝ်ႉတူဝ်ႈၶႃႈ။

---

### 📝 ၶေႃႈမုၼ်းတႃႇတွၼ်း (Key Takeaway)

* **V8 Engine:** ၸၵ်ႉပိၼ်ႇလိၵ်ႈ JS။
* **Single-threaded:** ႁဵတ်းၵၢၼ်လႆႈ ၼိုင်ႈဢၼ်တေႃႇၼိုင်ႈၶၢဝ်း။
* **Call Stack:** တီႈဝႆႉၵၢၼ် ဢၼ်တေႁဵတ်းၸွမ်းလမ်ႇတပ်ႉ။
* **Event Loop:** ၵေႃႉၸတ်းၵၢၼ် ႁႂ်ႈ Program ႁဝ်းဢမ်ႇၶိတ်း။
