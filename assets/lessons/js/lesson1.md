# Lesson 01: How JS Runs & The V8 Engine

JavaScript ၼႆႉ ပဵၼ်ၽႃႇသႃႇၶေႃတ်ႉ (Programming Language) ဢၼ်ႁွင်ႉဝႃႈ **High-level Language**။ မၼ်းပွင်ႇဝႃႈ တူဝ်လိၵ်ႈဢၼ်ႁဝ်းတႅမ်ႈၼၼ်ႉ မၼ်း "ငၢႆးမိူၼ်လိၵ်ႈၵူၼ်း" လႄႈ လူယႃႇငၢႆႈ၊ ၵူၺ်းၵႃႈ Computer ဢမ်ႇပွင်ႇၸႂ်မၼ်းၵမ်းလဵဝ်ၶႃႈ။ Computer ပွင်ႇၸႂ်ၵူၺ်း **Machine Code (0 လႄႈ 1)** ၵူၺ်း။

ၵွပ်ႈၼၼ် မၼ်းၸင်ႇလူဝ်ႇမီး "ၸၢၵ်ႈပိၼ်ႇၽႃႇသႃႇ" ဢၼ်ႁွင်ႉဝႃႈ **Engine** မႃးၸွႆႈၶႃႈ။

### 1. V8 Engine ပဵၼ်သင်?

V8 ပဵၼ် JavaScript Engine ဢၼ်ၵတ်ႉၶႅၼ်ႇသုတ်း ဢၼ် Google သၢင်ႈဝႆႉ (ၸႂ်ႉၼႂ်း Chrome လႄႈ Node.js)။ မၼ်းမီးၼႃႈၵၢၼ်ဢဝ် JavaScript ဢၼ်ႁဝ်းတႅမ်ႈၼၼ်ႉ ပိၼ်ႇပဵၼ် **Machine Code** ႁႂ်ႈ Computer ႁဵတ်းၵၢၼ်လႆႈၽႂ်းသုတ်းၶႃႈ။

### 2. ၶၵ်ႉတွၼ်ႈၵၢၼ်ႁဵတ်းၵၢၼ် (Execution Process)

မိူဝ်ႈႁဝ်း Start ၵူတ်ႉ JS၊ V8 Engine တေႁဵတ်းၵၢၼ် 3 ၶၵ်ႉတွၼ်ႈလူင်ၶႃႈ:

1. **Parsing (ၵၢၼ်လူလႄႈၸႅၵ်ႇ):** Engine တေလူၵူတ်ႉႁဝ်းသေ လႅၵ်ႈပဵၼ် **AST (Abstract Syntax Tree)**။ မၼ်းပဵၼ် "ၽႅၼ်ႇတီႈ" ဢၼ်ၸွႆႈႁႂ်ႈ Engine ပွင်ႇၸႂ်ဝႃႈ ႁဝ်းတႅမ်ႈသင်ဝႆႉ။
2. **Compilation (ၵၢၼ်ပိၼ်ႇၽႃႇသႃႇ):** JS ၸႂ်ႉလၢႆး **JIT (Just-In-Time) Compilation**။ မၼ်းဢမ်ႇပိၼ်ႇၵူတ်ႉတင်းမူတ်းယဝ်ႉၸင်ႇ Run၊ မၼ်း "ပိၼ်ႇၵႂႃႇ Run ၵႂႃႇ" ၼႂ်းမိူဝ်ႈတိုၵ်ႉႁဵတ်းၵၢၼ်ၼၼ်ႉၵမ်းလဵဝ်ၶႃႈ။
3. **Optimization (ၵၢၼ်မႄးႁႂ်ႈၽႂ်း):** Engine တေတူၺ်းဝႃႈ ၵူတ်ႉတွၼ်ႈလႂ်ႁွင်ႉၸႂ်ႉၼမ်၊ မၼ်းတေၶိုၼ်းမႄးတွၼ်ႈၼၼ်ႉ ႁႂ်ႈမၼ်းလႅတ်းၽႂ်းလိူဝ်ၵဝ်ႇၶႃႈ။

---

### 3. JavaScript ပဵၼ် Single-Threaded

လွင်ႈၼႆႉလမ်ႇလွင်ႈၼႃႇၶႃႈ! JavaScript ႁဵတ်းၵၢၼ်လႆႈ **"သဵၼ်ႈလဵဝ်/ပွၵ်ႈလဵဝ်"** ၵူၺ်း။ မၼ်းမိူၼ်ၵူၼ်းဢၼ်မီးမိုဝ်းၶွၼ်းလဵဝ်၊ မၼ်းတေႁဵတ်းၵၢၼ် ဢၼ်ၼိုင်ႈယဝ်ႉ ၸင်ႇသိုပ်ႇၵႂႃႇထႅင်ႈဢၼ်ၼိုင်ႈ။ ႁဝ်းႁွင်ႉဝႃႈ **Single-threaded Thread** ၶႃႈ။

---

### 📝 ၶေႃႈမုၼ်းတႃႇတွၼ်း (Key Takeaway)

* **V8 Engine** ပဵၼ် "ဢွၵ်းဢေႃ" ဢၼ်ပိၼ်ႇ JS ႁႂ်ႈပဵၼ် Machine Code။
* **JIT Compilation** ႁဵတ်းႁႂ်ႈ JS ႁဵတ်းၵၢၼ်လႆႈၽႂ်း ၼႂ်းမိူဝ်ႈတိုၵ်ႉ Run ယူႇ။
* **Parser** ပဵၼ်တူဝ်ၵူတ်ႇထတ်းဝႃႈ ႁဝ်းတႅမ်ႈၵူတ်ႉ (Syntax) ၽိတ်းႁႃႉ ထုၵ်ႇႁႃႉ။
