# Lesson 6: The Switch Statement (လွၵ်းလၢႆးလိူၵ်ႈလၢႆလမ်ႇတပ်ႉ)

မိူဝ်ႈႁဝ်းမီး "သၢႆငၢႆ" (Conditions) တင်းၼမ် ဢၼ်လူဝ်ႇၵူတ်ႇထတ်းၼၼ်ႉ သင်ႁဝ်းၸႂ်ႉ `if...else if` ၼမ်လႅၼ်တ်း တေႁဵတ်းႁႂ်ႈၵူတ်ႉႁဝ်း လူယႃႇယၢပ်ႇ။ **Switch** တေၸွႆႈႁႂ်ႈၵူတ်ႉႁဝ်း တႅတ်ႈတေႃးသေ သႅၼ်းသႃႇဝႆႉၶႃႈ။

### 1. Structure ၶွင် Switch (လွၵ်းလၢႆးတႅမ်ႈ)

Switch တေတူၺ်း "ၵႃႈၶၼ်" (Value) ဢၼ်ၼိုင်ႈ ယဝ်ႉတႅၵ်ႈၼိူင်းၸွမ်း **`case`** ဢၼ်ႁဝ်းဝႆႉဝႆႉၼၼ်ႉၶႃႈ။

```javascript
let day = 1;

switch (day) {
  case 1:
    console.log("ဝၼ်းဢႃးတိတ်ႉ");
    break;
  case 2:
    console.log("ဝၼ်းၸၼ်");
    break;
  default:
    console.log("ဢမ်ႇမီးၼႂ်းသဵၼ်ႈမၢႆ");
}

```

### 2. Keywords ဢၼ်လူဝ်ႇႁူႉ (ၶေႃႈၵႂၢမ်းလၵ်း)

* **`switch (expression)`**: တီႈၼႆႈပဵၼ် "ၵႃႈၶၼ်" ဢၼ်ႁဝ်းတေဢဝ်မႃးတႅၵ်ႈ (မိူၼ်ၼင်ႇ တူဝ်ၼပ်ႉ ႁမ်ႉ လိၵ်ႈ)။
* **`case`**: ပဵၼ်လွင်ႈတႅၵ်ႈၼိူင်း။ သင်ဝႃႈမိူၼ်ၵၼ်၊ မၼ်းတေႁဵတ်းၵၢၼ်ၼႂ်းၼၼ်ႉ။
* **`break`**: **ၼႆႉလုၵ်ႈလၵ်းၶႃႈ!** မၼ်းပဵၼ်ၵၢၼ် "ၵိုတ်း"။ သင်ဢမ်ႇမီး `break`၊ မၼ်းတေႁဵတ်းၵၢၼ် Case တၢင်ႇဢၼ်ၸွမ်းၵႂႃႇတိၵ်းတိၵ်း (Fall-through)။
* **`default`**: မိူၼ်ၼင်ႇ `else` ယဝ်ႉ။ သင်ဝႃႈတႅၵ်ႈတူၺ်း Case လႂ်ၵေႃႈ ဢမ်ႇမိူၼ်၊ မၼ်းတေမႃးႁဵတ်းၵၢၼ်တီႈၼႆႈ။

---

### 3. Fall-through (ၵၢၼ်လႆလူင်း)

သင်ဝႃႈႁဝ်းလိုမ်းတႅမ်ႈ `break`၊ JavaScript တေဢမ်ႇၵိုတ်း။ မၼ်းတေႁဵတ်းၵၢၼ် Case တီႈတႂ်ႈမၼ်းၵႂႃႇမူတ်း တေႃႇပေႃးထိုင် `break` ဢၼ်ၼိုင်ႈ။ မၢင်ၵမ်းႁဝ်းၸႂ်ႉလၢႆးၼႆႉ ပိူဝ်ႈတေႁႂ်ႈလၢႆ Case ႁဵတ်းၵၢၼ်မိူၼ်ၵၼ်။

```javascript
let fruit = "apple";

switch (fruit) {
  case "apple":
  case "banana":
    console.log("ၼႆႉပဵၼ်မၢၵ်ႇမႆႉ။");
    break;
  default:
    console.log("ဢမ်ႇႁူႉၸၵ်းၶႃႈ။");
}

```

---

### 4. တႅၵ်ႈၼိူင်း `if...else` လႄႈ `switch`

| လွင်ႈတႅၵ်ႈၼိူင်း | `if...else` | `switch` |
| --- | --- | --- |
| **ၵၢၼ်ၸႂ်ႉတူဝ်ၼပ်ႉ** | ၸႂ်ႉလႆႈတင်း `>`, `<`, `==` | ၸႂ်ႉလႆႈတႃႇတႅၵ်ႈ "ၵႃႈၶၼ်" (Equality) ၵူၺ်း |
| **တၢင်းငၢႆႈ** | သုပ်ႉသေႃႉၼမ် သင်မီးလၢႆသၢႆငၢႆ | လူယႃႇငၢႆႈ (Clean) တႃႇတၢင်းလိူၵ်ႈၼမ်ၼမ် |

---

### 📝 ၶေႃႈမုၼ်းတႃႇတွၼ်း (Key Takeaway for Lesson 6)

* **Switch**: ၸႂ်ႉမိူဝ်ႈမီး Case တင်းၼမ် ဢၼ်လူဝ်ႇတႅၵ်ႈၵႃႈၶၼ် (Value)။
* **Break**: ယႃႇလိုမ်းတႅမ်ႈ `break` တႃႇသေႇ၊ သင်ဢမ်ႇၼၼ် ၵူတ်ႉတေႁဵတ်းၵၢၼ်ၵႂႃႇၸွမ်းၵၼ်မူတ်း။
* **Default**: ဝႆႉတႃႇသၢႆငၢႆ ဢၼ်ဢမ်ႇပႃးၼႂ်း Case ဢၼ်ႁဝ်းတႅမ်ႈဝႆႉ။
