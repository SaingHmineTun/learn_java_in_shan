# Lesson 9: Loops (ၵၢၼ်ပၼ်ႇႁဵတ်းသမ်ႉ - for, while, do...while)

**Loop** မၢႆထိုင် လွင်ႈသင်ႇႁႂ်ႈ Program ႁဝ်းႁဵတ်းၵၢၼ်ၼိုင်ႈယၢင်ႇ သမ်ႉၵၼ်လၢႆလၢႆပၼ်ႇ တေႃႇပေႃးထိုင်တီႈဢၼ်ႁဝ်းမၢႆဝႆႉ။

### 1. The `for` Loop (ပၼ်ႇၸွမ်းလမ်ႇတပ်ႉ)

ၼႆႉပဵၼ် Loop ဢၼ်ႁဝ်းၸႂ်ႉၼမ်သုတ်း။ ႁဝ်းၸႂ်ႉမၼ်းမိူဝ်ႈႁဝ်းႁူႉဝႃႈ "တေပၼ်ႇၵႂႃႇလၢႆပၼ်ႇ"။

**လွၵ်းလၢႆးတႅမ်ႈ:**

```javascript
for (initialization; condition; increment) {
  // ၵူတ်ႉဢၼ်တေႁဵတ်းသမ်ႉ
}

```

**တူဝ်ယၢင်ႇ (Example):**

```javascript
for (let i = 1; i <= 5; i++) {
  console.log(`ပၼ်ႇပွၵ်ႈၵမ်း ${i}`);
}

```

* **`let i = 1`**: တႄႇၼပ်ႉတီႈ 1။
* **`i <= 5`**: ပၼ်ႇတေႃႇပေႃးထိုင် 5 (သင်ပဵၼ် 6 မၼ်းတေၵိုတ်း)။
* **`i++`**: တိူဝ်းၵႃႈၶၼ်ၶိုၼ်ႈၼိုင်ႈဢၼ် ၵူႈပွၵ်ႈဢၼ်ပၼ်ႇယဝ်ႉ။

---

### 2. The `while` Loop (ပၼ်ႇတေႃႇပေႃး...)

ႁဝ်းၸႂ်ႉ `while` မိူဝ်ႈႁဝ်းဢမ်ႇႁူႉဝႃႈ တေလႆႈပၼ်ႇလၢႆပွၵ်ႈ၊ တေႃႉတေႉႁဝ်းႁူႉဝႃႈ "တေပၼ်ႇၵႂႃႇ တေႃႇပေႃးသၢႆငၢႆပဵၼ် False"။

```javascript
let energy = 3;

while (energy > 0) {
  console.log("တိုၵ်ႉႁဵတ်းၵၢၼ်လႆႈယူႇ...");
  energy--; // ယွမ်းႁႅင်းလူင်း 1 ၵူႈပွၵ်ႈ
}
console.log("ႁႅင်းသုတ်းယဝ်ႉ၊ ၵိုတ်းၵၢၼ်။");

```

> **သတိ (Caution):** သင်ဝႃႈသၢႆငၢႆ (Condition) ပဵၼ် True ဝႆႉတႃႇသေႇ၊ မၼ်းတေပဵၼ် **Infinite Loop** (ပၼ်ႇဢမ်ႇယဝ်ႉဢမ်ႇဝၢႆး) သေ ႁဵတ်းႁႂ်ႈ Browser ႁဝ်းၶိတ်း (Crash) လႆႈၶႃႈ။

---

### 3. The `do...while` Loop (ႁဵတ်းဢွၼ်တၢင်း ၸင်ႇၵူတ်ႇထတ်း)

မၼ်းမိူၼ် `while` ယူႇ၊ ၵူၺ်းၵႃႈ မၼ်းတေ **ႁဵတ်းၵၢၼ်ဢွၼ်တၢင်း ၼိုင်ႈပွၵ်ႈ** ၸင်ႇၵွႆႈၵႂႃႇၵူတ်ႇထတ်းသၢႆငၢႆ။ မၢႆထိုင်ဝႃႈ ၵူတ်ႉၼႂ်းၼႆႉ တေႁဵတ်းၵၢၼ်ဢယၢတ်းသုတ်း 1 ပွၵ်ႈ တႃႇသေႇ။

```javascript
let count = 10;

do {
  console.log("ၼႆႉတေဢွၵ်ႇမႃး 1 ပွၵ်ႈၵူၺ်း");
  count++;
} while (count < 5);

```

---

### 4. `break` လႄႈ `continue` (ၵိုတ်း လႄႈ ၶၢမ်ႈ)

* **`break`**: ၸႂ်ႉတႃႇ "တတ်း" Loop ႁႂ်ႈၵိုတ်းၵမ်းလဵဝ် (Exit the loop)။
* **`continue`**: ၸႂ်ႉတႃႇ "ၶၢမ်ႈ" ၵၢၼ်ပၼ်ႇပွၵ်ႈၼႆႉသေ ၵႂႃႇတႄႇပွၵ်ႈမႂ်ႇ (Skip current iteration)။

---

### 📝 ၶေႃႈမုၼ်းတႃႇတွၼ်း (Key Takeaway for Lesson 9)

* **`for`**: ၸႂ်ႉမိူဝ်ႈႁူႉလၢႆပွၵ်ႈ တႅတ်ႈတေႃး။
* **`while`**: ၸႂ်ႉမိူဝ်ႈလူဝ်ႇပၼ်ႇၸွမ်းသၢႆငၢႆ (Condition)။
* **`do...while`**: ႁဵတ်းၵၢၼ် 1 ပွၵ်ႈဢွၼ်တၢင်း ၸင်ႇတူၺ်းသၢႆငၢႆ။
* **`i++`**: ယႃႇလိုမ်းတိူဝ်းၵႃႈၶၼ်၊ သင်ဢမ်ႇၼၼ် မၼ်းတေပဵၼ် Infinite Loop ၶႃႈ။
