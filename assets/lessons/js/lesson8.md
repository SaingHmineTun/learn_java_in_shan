# Lesson 08: Arithmetic & Assignment Operators

### 1. Arithmetic Operators (ၶိူင်ႈမၢႆၼပ်ႉသွၼ်ႇ)

ၼႆႉပဵၼ်ၶိူင်ႈမၢႆပိုၼ်ႉထၢၼ် ဢၼ်ႁဝ်းၸႂ်ႉၼႂ်းပၢႆးၼပ်ႉ (Math) ၶႃႈ။

| Operator | ၵၢၼ်ႁဵတ်းၵၢၼ် | တူဝ်ယၢင်ႇ |
| --- | --- | --- |
| `+` | ပၵ်း (Addition) | `10 + 5 = 15` |
| `-` | လတ်း (Subtraction) | `10 - 5 = 5` |
| `*` | ၵုၼ်ႇ (Multiplication) | `10 * 5 = 50` |
| `/` | ၽႄ (Division) | `10 / 5 = 2` |
| `%` | မူဝ်ႇၵျူႇလူဝ်ႇ (Remainder) | `10 % 3 = 1` (ဢဝ်ၵိုတ်း) |
| `**` | ယုၵ်ႉၵမ်ႉ (Exponentiation) | `2 ** 3 = 8` (2 ယုၵ်ႉၵမ်ႉ 3) |

```javascript
let x = 10;
let y = 3;
console.log(x % y); // 1 (ယွၼ်ႉ 3 * 3 = 9, ၵိုတ်း 1)

```

### 2. Assignment Operators (ၶိူင်ႈမၢႆပၼ်ၵႃႈၶၼ်)

ႁဝ်းၸႂ်ႉ `=` တႃႇပၼ်ၵႃႈၶၼ်၊ ၵူၺ်းၵႃႈ JavaScript ယင်းမီး "လၢႆးတႅမ်ႈပွတ်း" (Shorthand) တႃႇမႄးၵႃႈၶၼ် Variable တူဝ်ၵဝ်ႇၶႃႈ။

| Operator | လၢႆးတႅမ်ႈတဵမ် | လၢႆးတႅမ်ႈပွတ်း |
| --- | --- | --- |
| `=` | `x = 10` | `x = 10` |
| `+=` | `x = x + 5` | `x += 5` |
| `-=` | `x = x - 5` | `x -= 5` |
| `*=` | `x = x * 5` | `x *= 5` |
| `/=` | `x = x / 5` | `x /= 5` |

```javascript
let score = 100;
score += 10; // score ယၢမ်းလဵဝ်ပဵၼ် 110
score -= 5;  // score ယၢမ်းလဵဝ်ပဵၼ် 105
console.log(score);

```

### 3. Increment & Decrement (ၵၢၼ်ထႅမ် လႄႈ ၵၢၼ်ယွမ်း)

မိူဝ်ႈႁဝ်းၶႂ်ႈထႅမ် ႁိုဝ် ယွမ်း ၵႃႈၶၼ် "ၼိုင်ႈ (1)" ၵူၺ်းၼၼ်ႉ ႁဝ်းၸၢင်ႈၸႂ်ႉ:

* **`++`**: ထႅမ် 1 (`x++` မိူၼ်ၼင်ႇ `x = x + 1`)
* **`--`**: ယွမ်း 1 (`x--` မိူၼ်ၼင်ႇ `x = x - 1`)

```javascript
let counter = 0;
counter++; 
console.log(counter); // 1

```

---

### 📝 ၶေႃႈမုၼ်းတႃႇတွၼ်း (Key Takeaway)

* **`%` (Modulo)** လမ်ႇလွင်ႈၼႃႇ တႃႇၵူတ်ႇထတ်းဝႃႈ တူဝ်ၼပ်ႉၼၼ်ႉ ပဵၼ်ၶူပ်း (Even) ႁိုဝ် ၶိူၼ်း (Odd)။
* **Assignment Operators (`+=`, `-=`)** ၸွႆႈႁႂ်ႈၵူတ်ႉႁဝ်း သႅၼ်ႈသႂ် လႄႈ တႅမ်ႈလႆႈၽႂ်း။
* ၾၢင်ႉလွင်ႈၸႂ်ႉ **`++`**၊ မၼ်းတေလႅၵ်ႈလၢႆႈၵႃႈၶၼ် Variable တူဝ်ၼၼ်ႉၵမ်းလဵဝ် (လူဝ်ႇၸႂ်ႉ `let`)။
