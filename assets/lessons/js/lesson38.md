
### 1. While Loop (ၵူတ်ႇထတ်းယဝ်ႉၸင်ႇလႅၼ်ႈ)

`while` loop တေၵူတ်ႇထတ်း Condition ဢွၼ်တၢင်း။ သင်ပဵၼ် `true` မၼ်းၸင်ႇတေလႅၼ်ႈၵူတ်ႉၼႂ်းပူၵ်ႇမၼ်းၶႃႈ။

**ပိူင်တႅမ်ႈ:**

```javascript
let i = 1;

while (i <= 5) {
  console.log("Count is: " + i);
  i++; // ⚠️ လမ်ႇလွင်ႈၼႃႇ: လူဝ်ႇလႅၵ်ႈလၢႆႈၵႃႈၶၼ် i ဢမ်ႇၼၼ်တေပဵၼ် Infinite Loop
}

```

* **မိူဝ်ႈလႂ်တေၸႂ်ႉ?** မိူဝ်ႈႁဝ်းဢမ်ႇႁူႉဝႃႈ လူဝ်ႇလႅၼ်ႈၵႃႈႁိုင် (မိူၼ်ၼင်ႇ တေႃႇၵူၼ်းလဵၼ်ႈၵဵမ်းတေၼဵၵ်းၼဵတ်ႇ "Stop")။

---

### 2. Do-While Loop (လႅၼ်ႈယဝ်ႉၸင်ႇၵူတ်ႇထတ်း)

`do-while` loop ၼႆႉ ပႅၵ်ႇပိူင်ႈၵၼ်ဢိတ်းၼိုင်ႈ။ မၼ်းတေ **"လႅၼ်ႈၵူတ်ႉဢွၼ်တၢင်း ၼိုင်ႈပွၵ်ႈ"** ယဝ်ႉၸင်ႇၵွႆႈၵႂႃႇၵူတ်ႇထတ်း Condition ၶႃႈ။

**ပိူင်တႅမ်ႈ:**

```javascript
let i = 10;

do {
  console.log("This will run at least once!");
  i++;
} while (i < 5);

// Result: "This will run at least once!" 
// တေလႅၼ်ႈ 1 ပွၵ်ႈ တင်းၼင်ႇ 10 ဢမ်ႇလႅၵ်ႈလိူဝ် 5 ၵေႃႈယဝ်ႉ

```

* **မိူဝ်ႈလႂ်တေၸႂ်ႉ?** မိူဝ်ႈႁဝ်းၶႂ်ႈႁႂ်ႈၵူတ်ႉၼၼ်ႉလႅၼ်ႈ "ဢမ်ႇယိူင်ႈ 1 ပွၵ်ႈ" (မိူၼ်ၼင်ႇ ၼႄ Menu ႁႂ်ႈၵူၼ်းၸႂ်ႉလိူၵ်ႈဢွၼ်တၢင်း)။

---

### 💡 တႅၵ်ႈၼိူဝ် (Comparison)

| Loop Type | ၵၢၼ်ၵူတ်ႇထတ်း (Check) | လွင်ႈလႅၼ်ႈသုတ်း (Min Runs) |
| --- | --- | --- |
| **`while`** | ၵူတ်ႇထတ်း "ဢွၼ်တၢင်း" (Pre-test) | 0 ပွၵ်ႈ (သင် condition ပဵၼ် false တႄႇတႅတ်ႉ) |
| **`do-while`** | ၵူတ်ႇထတ်း "ၽၢႆႇလင်" (Post-test) | 1 ပွၵ်ႈ တႃႇသေႇ |

---

### ⚠️ သတိ: Infinite Loops (Loop ဢမ်ႇသုတ်း)

သင်ၸဝ်ႈၵဝ်ႇလိုမ်းထႅမ်ၵႃႈၶၼ် Variable (i++) ႁိုဝ် သႂ်ႇ Condition ဢၼ်ပဵၼ် `true` တႃႇသေႇ၊ Loop ၼၼ်ႉတေလႅၼ်ႈဢမ်ႇၵိုတ်း သေႁဵတ်းႁႂ်ႈ Browser "Hang" ၵႂႃႇလႆႈၶႃႈ။

```javascript
// ❌ ဢမ်ႇလီႁဵတ်း!
while (true) {
  console.log("I will never stop!");
}

```

---

### 📝 ၶေႃႈမုၼ်းတႃႇတွၼ်း (Key Takeaway)

* ၸႂ်ႉ **`while`** မိူဝ်ႈလူဝ်ႇၵူတ်ႇထတ်းငဝ်းလၢႆးဢွၼ်တၢင်းတေႁဵတ်းၵၢၼ်။
* ၸႂ်ႉ **`do-while`** မိူဝ်ႈလူဝ်ႇႁႂ်ႈၵူတ်ႉလႅၼ်ႈဢွၼ်တၢင်း 1 ပွၵ်ႈ ဢမ်ႇဝႃႈ Condition တေပဵၼ်သင်ၵေႃႈယဝ်ႉ။
* တႃႇသေႇ၊ လူဝ်ႇမီး "လၢႆးဢွၵ်ႇ" (Exit Strategy) ၼႂ်း Loop တႃႇသေႇၶႃႈ။
