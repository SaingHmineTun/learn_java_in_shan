# Lesson 10: Comparison Operators & Strict Equality

Comparison Operators (ၶိူင်ႈမၢႆၵၢၼ်တႅၵ်ႈၼိူဝ်) ၸႂ်ႉတႃႇတူၺ်းဝႃႈ ၵႃႈၶၼ် (Value) ဢၼ်ၼိုင်ႈ လႄႈ ဢၼ်ၼိုင်ႈ မိူၼ်ၵၼ်ႁႃႉ၊ ယႂ်ႇလိူဝ်ႁႃႉ ႁိုဝ် ဢွၼ်ႇလိူဝ်ႁႃႉၶႃႈ။ ၵႃႈၶၼ်ဢၼ်ဢွၵ်ႇမႃး တေပဵၼ် **Boolean (`true` ႁိုဝ် `false`)** တႃႇသေႇၶႃႈ။

### 1. ၶိူင်ႈမၢႆတႅၵ်ႈၼိူဝ် ပိုၼ်ႉထၢၼ်

| Operator | ၵၢၼ်ႁဵတ်းၵၢၼ် | တူဝ်ယၢင်ႇ |
| --- | --- | --- |
| `>` | ယႂ်ႇလိူဝ် (Greater than) | `10 > 5` -> `true` |
| `<` | ဢွၼ်ႇလိူဝ် (Less than) | `10 < 5` -> `false` |
| `>=` | ယႂ်ႇလိူဝ် ႁိုဝ် မိူၼ်ၵၼ် | `10 >= 10` -> `true` |
| `<=` | ဢွၼ်ႇလိူဝ် ႁိုဝ် မိူၼ်ၵၼ် | `5 <= 8` -> `true` |

---

### 2. Equality (ၵၢၼ်တႅၵ်ႈလွင်ႈမိူၼ်ၵၼ်)

ၼႂ်း JavaScript မၼ်းမီးလၢႆးတႅၵ်ႈ "မိူၼ်ၵၼ်" ယူႇ 2 မဵဝ်း ဢၼ်လူဝ်ႇၾၢင်ႉၼႃႇၶႃႈ:

#### **A. Loose Equality (`==`)**

မၼ်းတေတႅၵ်ႈၵူၺ်း **"ၵႃႈၶၼ် (Value)"**။ သင်မဵဝ်းၶေႃႈမုၼ်း (Data Type) ဢမ်ႇမိူၼ်ၵၼ်၊ မၼ်းတေၸႂ်ႉ *Type Coercion* (ဢၼ်ႁဝ်းႁဵၼ်းမႃးၼႂ်း Lesson 9) သေပိၼ်ႇပၼ်ႁဝ်းၵမ်းလဵဝ်ၶႃႈ။

```javascript
console.log(5 == "5"); // true (ယွၼ်ႉ JS ပိၼ်ႇ "5" ပဵၼ် Number သေတႅၵ်ႈ)

```

#### **B. Strict Equality (`===`)**

မၼ်းတေတႅၵ်ႈတင်း **"ၵႃႈၶၼ် (Value)"** လႄႈ **"မဵဝ်းၶေႃႈမုၼ်း (Data Type)"** ၵမ်းလဵဝ်ၶႃႈ။ ၼႆႉပဵၼ်လၢႆးဢၼ် Pro ၶဝ်ၸႂ်ႉၵၼ်ၶႃႈ။

```javascript
console.log(5 === "5"); // false (ယွၼ်ႉ Number ဢမ်ႇမိူၼ် String)

```

---

### 3. Not Equal (ၵၢၼ်တႅၵ်ႈလွင်ႈဢမ်ႇမိူၼ်ၵၼ်)

* **`!=`** (Loose Not Equal): တူၺ်းၵႃႈၶၼ်ၵူၺ်း။
* **`!==`** (Strict Not Equal): တူၺ်းတင်းၵႃႈၶၼ် လႄႈ Type။

```javascript
console.log(10 != "10");  // false (ယွၼ်ႉမၼ်းဝႃႈ "ၵႃႈၶၼ်မၼ်းမိူၼ်ၵၼ်ယူႇၼႃႇ")
console.log(10 !== "10"); // true (ယွၼ်ႉမၼ်းဝႃႈ "Type မၼ်းဢမ်ႇမိူၼ်ၵၼ်တေႉ")

```

---

### 📝 ၶေႃႈမုၼ်းတႃႇတွၼ်း (Key Takeaway)

* **Always use `===` and `!==**`: ၼႆႉတေၸွႆႈႁႄႉၵင်ႈ Bug ဢၼ်ဢမ်ႇလူဝ်ႇပဵၼ် လႄႈ ႁဵတ်းႁႂ်ႈၵူတ်ႉႁဝ်းမီး Logic ဢၼ်ၸႅၼ်ႇလႅင်းၶႃႈ။
* ၵၢၼ်တႅၵ်ႈၼိူဝ် တေပၼ်ၵႃႈၶၼ်ပဵၼ် **Boolean** (`true`/`false`) တႃႇသေႇ၊ ဢၼ်ၼႆႉႁဝ်းတေဢဝ်ၵႂႃႇၸႂ်ႉၼႂ်း `if/else` ၶႃႈ။
