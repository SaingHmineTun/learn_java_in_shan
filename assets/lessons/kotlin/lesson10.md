# တွၼ်ႈသွၼ် 10: တၢင်းႁၼ်ထိုင်လွင်ႈ Null Safety (The Philosophy of Null Safety)

ၼႂ်းၵၢၼ်တႅမ်ႈၶူတ်ႉၼၼ်ႉ ပၼ်ႁႃဢၼ်ယႂ်ႇသုတ်း ဢၼ်ႁဵတ်းႁႂ်ႈ Software ၵၢင်ႉ ဢမ်ႇၼၼ် Program လူႉ (Crash) ၼၼ်ႉၵႆႉပဵၼ်မႃးတီႈ **`null`** ယဝ်ႉ။

### 1. `null` ဢၼ်ဝႃႈၼၼ်ႉ ပဵၼ်သင်?

`null` ၼႆႉ မီးတီႈပွင်ႇဝႃႈ **"ဢမ်ႇမီးသင်"** ဢမ်ႇၼၼ် **"လွၵ်းလၢႆႇ"** (Empty/Nothing)။

* တူဝ်ယၢင်ႇ: သင်ႁဝ်းမီး Variable ၸိုဝ်ႈဝႃႈ `name` သေတႅမ်ႈဝႃႈ `val name = null`၊ မၼ်းပွင်ႇဝႃႈ "ၸိုဝ်ႈၼႆႉ ပႆႇမီးၶေႃႈမုၼ်းသင်"။

### 2. ပၼ်ႁႃ "The Billion Dollar Mistake"

ၼႂ်းၽႃႇသႃႇ Java ဢမ်ႇၼၼ် C++၊ သင်ႁဝ်းဢဝ် Variable ဢၼ်ပဵၼ် `null` ၵႂႃႇၸႂ်ႉ (မိူၼ်ၼင်ႇ ၼပ်ႉတူဝ်လိၵ်ႈ)၊ Program ႁဝ်းတေလူႉၵမ်းလဵဝ်။ ၵူၼ်းသၢင်ႈ Java လၢတ်ႈဝႃႈ လွင်ႈသၢင်ႈ `null` မႃးၼႆႉ ပဵၼ်လွင်ႈၽိတ်းပိူၼ်ႈ ဢၼ်သုမ်းငိုၼ်းပဵၼ် ပၼ်ႇလၢၼ်ႉ (Billion Dollar Mistake) ယဝ်ႉ။

---

### 3. Kotlin ၵႄႉၶႆးပၼ်ႁႃၼႆႉၸိူဝ်ႉႁိူဝ်?

Kotlin မီးတၢင်းႁၼ်ထိုင် (Philosophy) ဝႃႈ **"ပၼ်ႁႃ Null လူဝ်ႇလႆႈၵႄႉၶႆး မိူဝ်ႈတႅမ်ႈၶူတ်ႉ (Compile-time)၊ ဢမ်ႇၵႂႃႇပွႆႇပဵၼ် မိူဝ်ႈၸႂ်ႉ Program (Runtime)"**။

* **Non-Nullable (ပိူင်ငဝ်ႈ):** ၼႂ်း Kotlin၊ Variable ၵူႈဢၼ် တေပဵၼ် `null` ဢမ်ႇလႆႈ တႃႇသေႇ။ သင်ႁဝ်းတႅမ်ႈ `val name: String = null` ၼႆ ၶွမ်ႇပိဝ်ႇတႃႇတေလၢတ်ႈဝႃႈ "ၽိတ်းယဝ်ႉ" ၵမ်းလဵဝ်။
* **Type System:** Kotlin ၸႅၵ်ႇ `String` (ဢၼ်မီးၶေႃႈမုၼ်းတႃႇသေႇ) လႄႈ `String?` (ဢၼ်ၸၢင်ႈပဵၼ် null) ဢွၵ်ႇၵၼ်ၸႅင်ႈၸႅင်း။

---

### 4. ၶေႃႈတွၼ်းသတိ

ၵွပ်ႈပိူဝ်ႈ Kotlin မီး Null Safety၊ ႁဝ်းတေလႆႈ:

1. ဝူၼ်ႉဢွၼ်တၢင်းဝႃႈ Variable ၼႆႉ ၸၢင်ႈပဵၼ် `null` လႆႈႁႃႉ?
2. သင်ၸၢင်ႈပဵၼ် `null` လႆႈ၊ ႁဝ်းတေလႆႈၵုမ်းထိင်းမၼ်း (Handle) ၸိူဝ်ႉႁိူဝ်?

**တူဝ်ယၢင်ႇ:**

```kotlin
var message: String = "မႂ်ႇသုင်"
// message = null  // ၼႆႉတေ Error တီႈ Compiler ၵမ်းလဵဝ် (မၼ်းၸွႆႈၵႄႉပၼ်ႁႃႁဝ်း)

```

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Null:** တီႈပွင်ႇဝႃႈ ဢမ်ႇမီးၶေႃႈမုၼ်းသင် (Value is missing)။
* **NullPointerException (NPE):** ပဵၼ် Error ဢၼ်ပဵၼ်မႃး မိူဝ်ႈႁဝ်းၸႂ်ႉၶေႃႈမုၼ်းဢၼ်ပဵၼ် null။
* **Compile-time:** မိူဝ်ႈႁဝ်းတႅမ်ႈၶူတ်ႉ လႄႈ လႅၵ်ႈလၢႆႈၶူတ်ႉ။
* **Runtime:** မိူဝ်ႈ Program ႁဵတ်းၵၢၼ်ယူႇ။
