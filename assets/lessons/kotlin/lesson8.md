# တွၼ်ႈသွၼ် 8: ၶိူင်ႈမၢႆၼပ်ႉသၢင်ႈ (Arithmetic, Assignment & Comparison Operators)

### 1. Arithmetic Operators (ၶိူင်ႈမၢႆၼပ်ႉတူဝ်ၼပ်ႉ)

ၸႂ်ႉတႃႇၼပ်ႉသၢင်ႈ တူဝ်ၼပ်ႉ (Math operations) ပိုၼ်ႉထၢၼ်:

* **`+`** (Plus): ၸူႉၵၼ် / ထႅမ်
* **`-`** (Minus): လူတ်း / ႁၵ်းဢွၵ်ႇ
* **`*`** (Times): တူပ်ႉ / ၵူၼ်ႇ
* **`/`** (Division): ၸႅၵ်ႇ / ၽႄ
* **`%`** (Modulus): ဢဝ်ၵွင်း (เศษ) ဢၼ်ၵူၼ်ႇဢမ်ႇသုတ်း

```kotlin
val a = 10
val b = 3

println(a + b) // 13
println(a % b) // 1 (ၵွပ်ႈဝႃႈ 10 ၽႄ 3 လႆႈ 3 သေ ၵွင်း 1)

```

### 2. Assignment Operators (ၶိူင်ႈမၢႆဢဝ်ၶေႃႈမုၼ်းသႂ်ႇ)

ၸႂ်ႉတႃႇဢဝ် "Value" သႂ်ႇပၼ် Variable ဢၼ်ႁဝ်းသၢင်ႈဝႆႉ။

* **`=`**: ဢဝ်ၶေႃႈမုၼ်းၽၢႆႇၶႂႃ သႂ်ႇပၼ် ၽၢႆႇသၢႆႉ။
* **`+=`**: ဢဝ်ၶေႃႈမုၼ်းမႂ်ႇ ထႅမ်သႂ်ႇ တူဝ်ၵဝ်ႇ (တူဝ်ယၢင်ႇ: `x += 5` မိူၼ်တင်း `x = x + 5`)။
* **`-=`**: ဢဝ်ၶေႃႈမုၼ်းမႂ်ႇ ႁၵ်းဢွၵ်ႇ တူဝ်ၵဝ်ႇ။

```kotlin
var score = 10
score += 5  // မိူဝ်ႈလဵဝ် score ပဵၼ် 15 ယဝ်ႉ

```

---

### 3. Comparison Operators (ၶိူင်ႈမၢႆတႅၵ်ႈၼိူဝ်း)

ၸႂ်ႉတႃႇတႅၵ်ႈၼိူဝ်းၵၼ်သေ ၽူၼ်မၼ်းတေဢွၵ်ႇမႃးပဵၼ် **Boolean** (`true` ဢမ်ႇၼၼ် `false`)။

* **`==`**: တႅတ်ႈတေႃးၵၼ် (Equal to)
* **`!=`**: ဢမ်ႇတႅတ်ႈတေႃးၵၼ် (Not equal to)
* **`>`** : ယႂ်ႇသေ (Greater than)
* **`<`** : ဢွၼ်ႇသေ (Less than)
* **`>=`**: ယႂ်ႇသေ ဢမ်ႇၼၼ် ၽဵင်ႇၵၼ်
* **`<=`**: ဢွၼ်ႇသေ ဢမ်ႇၼၼ် ၽဵင်ႇၵၼ်

```kotlin
val myAge = 20
val friendAge = 25

println(myAge == friendAge) // false
println(myAge < friendAge)  // true

```

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Operator:** ၶိူင်ႈမၢႆဢၼ်သင်ႇႁႂ်ႈႁဵတ်းၵၢၼ် (မိူၼ်ၼင်ႇ `+`, `-`)။
* **Operand:** တူဝ်ၼပ်ႉ ဢၼ်ထုၵ်ႇ Operator ၼပ်ႉသၢင်ႈ (မိူၼ်ၼင်ႇ `10` လႄႈ `5`)။
* **Modulus:** လၢႆးၵၢၼ်ႁႃၵွင်း (เศษ) ဝၢႆးၽႄတူဝ်ၼပ်ႉယဝ်ႉ။
