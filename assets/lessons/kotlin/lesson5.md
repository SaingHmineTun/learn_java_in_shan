# တွၼ်ႈသွၼ် 5: Type Inference လႄႈ ၵၢၼ်ပိုတ်ႇၼႄမဵဝ်းၶေႃႈမုၼ်း (Explicit Type Declarations)

မိူဝ်ႈႁဝ်းသၢင်ႈ Variable ၼၼ်ႉ Kotlin လူဝ်ႇႁူႉဝႃႈ ၶေႃႈမုၼ်းၼၼ်ႉ ပဵၼ်လိၵ်ႈ (String) ႁႃႉ? ပဵၼ်တူဝ်ၼပ်ႉ (Int) ႁႃႉ? ၼႆယဝ်ႉ။

### 1. Type Inference (ၵၢၼ်လၢမ်းႁူႉႁင်းၵူၺ်း)

Kotlin ၼႆႉ ပဵၼ်ၽႃႇသႃႇဢၼ်လႅတ်းၽႂ်း (Smart) ဝႆႉယဝ်ႉ။ သင်ႁဝ်းဢဝ်ၶေႃႈမုၼ်းသႂ်ႇပၼ်မၼ်းၵမ်းလဵဝ်၊ မၼ်းတေ **လၢမ်းႁူႉ (Infer)** ၵမ်းလဵဝ်ဝႃႈ မၼ်းပဵၼ် Type သင်။ ႁဝ်းဢမ်ႇလူဝ်ႇတႅမ်ႈလၢတ်ႈၼႄမၼ်းၵေႃႈ လႆႈယဝ်ႉ။

```kotlin
val city = "Taunggyi"  // Kotlin ႁူႉၵမ်းလဵဝ်ဝႃႈပဵၼ် String (လိၵ်ႈ)
val zipCode = 11101    // Kotlin ႁူႉၵမ်းလဵဝ်ဝႃႈပဵၼ် Int (တူဝ်ၼပ်ႉ)

```

---

### 2. Explicit Type Declarations (ၵၢၼ်တႅမ်ႈလၢတ်ႈၸႅင်ႈလႅင်း)

မၢင်ပွၵ်ႈမၢင်လႂ် ႁဝ်းတေၶႂ်ႈတႅမ်ႈလၢတ်ႈၼႄၸႅင်ႈလႅင်းဝႃႈ Variable ၼႆႉ တေလႆႈပဵၼ် Type ၼႆႉၵူၺ်းၼႃ။ ႁဝ်းၸၢင်ႈတႅမ်ႈလႆႈ လူၺ်ႈၵၢၼ်သႂ်ႇၶိူင်ႈမၢႆ **`:` (Colon)** ဝႆႉၽၢႆႇလင်ၸိုဝ်ႈ Variable ၼၼ်ႉၶႃႈ။

```kotlin
val country: String = "Shan State"
val population: Int = 6000000

```

**လၢႆးတႅမ်ႈ (Syntax):**

> `val` ၸိုဝ်ႈ : **Type** = **ၶေႃႈမုၼ်း**

---

### 3. ပႅၵ်ႇပၢင်ႇၵၼ်ၸိူဝ်ႉႁိူဝ်?

* **Type Inference:** တႅမ်ႈငၢႆႈ၊ ၶူတ်ႉတူၺ်းသႃႇ (Clean Code)။ ၵူၼ်းတႅမ်ႈ Kotlin ၵႆႉၸႂ်ႉလၢႆးၼႆႉ 90% ယဝ်ႉ။
* **Explicit Type:** ၸႂ်ႉမိူဝ်ႈႁဝ်းပႆႇသႂ်ႇၶေႃႈမုၼ်းၵမ်းလဵဝ် ဢမ်ႇၼၼ် မိူဝ်ႈႁဝ်းၶႂ်ႈႁႂ်ႈၶူတ်ႉႁဝ်း ၸႅင်ႈလႅင်းသုတ်းသုတ်း။

> **ၶေႃႈတွၼ်းသတိ:** ၼႂ်း Kotlin ၼၼ်ႉ သင်ႁဝ်းပိုတ်ႇ Variable ဝႆႉသေ ပႆႇသႂ်ႇၶေႃႈမုၼ်း (Initial value) ၼႆ ႁဝ်း **လူဝ်ႇတႅမ်ႈလၢတ်ႈ Type** မၼ်းၸႅင်ႈၸႅင်ႈလႅင်းလႅင်းယဝ်ႉ။

```kotlin
val myCar: String  // လူဝ်ႇတႅမ်ႈလၢတ်ႈ Type ၵွပ်ႈပႆႇမီးၶေႃႈမုၼ်း
myCar = "Toyota"

```

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Type Inference:** ၵၢၼ်ဢၼ် Compiler လၢမ်းႁူႉမဵဝ်းၶေႃႈမုၼ်းႁင်းၵူၺ်း။
* **Explicit Type:** ၵၢၼ်တႅမ်ႈလၢတ်ႈမဵဝ်းၶေႃႈမုၼ်း ႁႂ်ႈၸႅင်ႈလႅင်း။
* **Statically Typed:** ၽႃႇသႃႇဢၼ်လူဝ်ႇႁူႉ Type မၼ်းၸႅင်ႈလႅင်း မိူဝ်ႈတႅမ်ႈၶူတ်ႉ။
