# တွၼ်ႈသွၼ် 13: ၶိူင်ႈမၢႆဢဝ်ၶေႃႈမုၼ်းတႅၼ်းတၢင်း (The Elvis Operator - `?:`)

မိူဝ်ႈႁဝ်းၸႂ်ႉ Safe Call (`?.`) ယဝ်ႉၼၼ်ႉ သင်ၶေႃႈမုၼ်းပဵၼ် `null` မၼ်းတေဢွၵ်ႇပဵၼ် `null` ဝႆႉ။ ၵူၺ်းၵႃႈ မၢင်ပွၵ်ႈႁဝ်းဢမ်ႇၶႂ်ႈလႆႈ `null`၊ ႁဝ်းၶႂ်ႈလႆႈ **"ၶေႃႈမုၼ်းတႅၼ်းတၢင်း"** (Default Value) သင်သင်ၼိုင်ႈ။ တီႈၼႆႈယဝ်ႉ ႁဝ်းတေၸႂ်ႉ `?:`။

### 1. `?:` ႁဵတ်းၵၢၼ်ၸိူဝ်ႉႁိူဝ်?

ၶိူင်ႈမၢႆ **`?:`** ၼႆႉ မၼ်းတေတူၺ်းၶေႃႈမုၼ်းၽၢႆႇသၢႆႉ:

* **သင်ၽၢႆႇသၢႆႉ ဢမ်ႇပဵၼ် null:** မၼ်းတေဢဝ်ၶေႃႈမုၼ်း ၽၢႆႇသၢႆႉၼၼ်ႉၵမ်းလဵဝ်။
* **သင်ၽၢႆႇသၢႆႉ ပဵၼ် null:** မၼ်းတေဢဝ်ၶေႃႈမုၼ်း ၽၢႆႇၶႂႃ မႃးတႅၼ်းတၢင်းပၼ်။

**တူဝ်ယၢင်ႇ:**

```kotlin
val name: String? = null

// သင် name ပဵၼ် null၊ ႁႂ်ႈမၼ်းတႅမ်ႈဝႃႈ "Guest"
val displayName = name ?: "Guest"

println(displayName) // ၽူၼ်မၼ်းတေဢွၵ်ႇ: Guest

```

---

### 2. ၸႂ်ႉၵိုၵ်းၵၼ်တင်း Safe Call

ႁဝ်းၵႆႉၸႂ်ႉ `?.` လႄႈ `?:` ၵိုၵ်းၵၼ် တႃႇႁႃတၢင်းယၢဝ်းလိၵ်ႈ ဢမ်ႇၼၼ် ၶေႃႈမုၼ်းပိုင်ႈဢိင် ႁႂ်ႈမၼ်းဢွၵ်ႇမႃးပဵၼ်တူဝ်ၼပ်ႉတႃႇသေႇ။

```kotlin
val city: String? = null
val length = city?.length ?: 0

println(length) // သင် city ပဵၼ် null၊ length တေပဵၼ် 0 (ဢမ်ႇပဵၼ် null)

```

---

### 3. ၸႂ်ႉတႃႇထတ်းသိၼ်လႄႈ ပွႆႇ Error (return / throw)

ၼႂ်း Kotlin၊ ႁဝ်းၸၢင်ႈၸႂ်ႉ `return` ဢမ်ႇၼၼ် `throw` ဝႆႉၽၢႆႇၶႂႃၶွင် Elvis Operator လႆႈယဝ်ႉ။ ၼႆႉပဵၼ်လၢႆးတႅမ်ႈ ဢၼ်ၵတ်ႉၶႅၼ်ႇသုတ်း တႃႇၵႄႉၶႆး Null။

```kotlin
fun printName(name: String?) {
    // သင် name ပဵၼ် null၊ ႁႂ်ႈမၼ်းဢွၵ်ႇ (return) တီႈ Function ၵမ်းလဵဝ်
    val input = name ?: return
    println("ၸိုဝ်ႈပဵၼ်: $input")
}

```

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Elvis Operator:** ၶိူင်ႈမၢႆ `?:` ဢၼ်ဝႆႉၸႂ်ႉတႃႇသႂ်ႇ Default Value မိူဝ်ႈပဵၼ် null။
* **Default Value:** ၶေႃႈမုၼ်းဢၼ်ႁဝ်းတင်ႈဝႆႉ တႃႇၸႂ်ႉတႅၼ်းတၢင်း မိူဝ်ႈဢမ်ႇမီးၶေႃႈမုၼ်းတႄႉ။
* **Fallback:** လၢႆးၵၢၼ်ဢဝ်ၶေႃႈမုၼ်းထႅင်ႈပိူင်ၼိုင်ႈ မႃးၸႂ်ႉ မိူဝ်ႈပိူင်လူင်မၼ်းၸႂ်ႉဢမ်ႇလႆႈ။
