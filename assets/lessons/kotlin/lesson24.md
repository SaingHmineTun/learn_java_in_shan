# တွၼ်ႈသွၼ် 24: ၵၢၼ်ႁဵတ်း Loop ၸွမ်းငဝ်းလၢႆး (While and Do-While)

### 1. `while` Loop (ထတ်းဢွၼ်ႇတၢင်း ႁဵတ်းဝၢႆးလင်)

`while` ၼႆႉ မၼ်းတေထတ်းတူၺ်းငဝ်းလၢႆးၼႂ်းဝူင်းလႅမ် `( )` ဢွၼ်တၢင်း။ သင်ပဵၼ် `true` မၼ်းတေႁဵတ်းၵၢၼ်။ သင်ပဵၼ် `false` တႄႇငဝ်ႈမၼ်း၊ မၼ်းတေ **ဢမ်ႇႁဵတ်းသင်သေဢၼ်** ယဝ်ႉ။

```kotlin
var apples = 3

while (apples > 0) {
    println("ၵိၼ်မၢၵ်ႇၶူၼ်း: ၵိုတ်း $apples လုၵ်ႈ")
    apples-- // လူတ်းလူင်းၵမ်း 1 လုၵ်ႈ
}
// ၽူၼ်မၼ်း: ၵိၼ် 3 လုၵ်ႈ၊ 2 လုၵ်ႈ၊ 1 လုၵ်ႈ သေယဝ်ႉၵႂႃႇ

```

---

### 2. `do-while` Loop (ႁဵတ်းဢွၼ်တၢင်း ထတ်းဝၢႆးလင်)

`do-while` ၼႆႉ လၢၵ်ႇလၢႆးဝႆႉဢိတ်းၼိုင်ႈ။ မၼ်းတေ **ႁဵတ်းၵၢၼ်ဢွၼ်တၢင်း 1 ပွၵ်ႈတႃႇသေႇ** သေ ၸင်ႇတေမႃးထတ်းတူၺ်းငဝ်းလၢႆးၽၢႆႇလင်ယဝ်ႉ။

```kotlin
var battery = 0

do {
    println("ပိုတ်ႇၾူၼ်း... (တိုၵ်ႉႁဵတ်းၵၢၼ်ယူႇ)")
} while (battery > 0)

// ၽူၼ်မၼ်း: တေ Print ဢွၵ်ႇမႃး 1 ပွၵ်ႈ တင်းသႅင်ဝႃႈ battery ပဵၼ် 0 ၵေႃႈယဝ်ႉ

```

---

### 3. ပႅၵ်ႇပၢင်ႇၵၼ်ၸိူဝ်ႉႁိူဝ်?

| ပိူင် | **while** | **do-while** |
| --- | --- | --- |
| **ၵၢၼ်ထတ်း** | ထတ်းဢွၼ်တၢင်း (Check first) | ထတ်းဝၢႆးလင် (Check last) |
| **ႁဵတ်းၵၢၼ်** | ၸၢင်ႈဢမ်ႇလႆႈႁဵတ်းသင်သေပွၵ်ႈ (0 times) | တေလႆႈႁဵတ်းၵၢၼ် 1 ပွၵ်ႈတႃႇသေႇ (At least 1 time) |
| **ၵၢၼ်ၸႂ်ႉတိုဝ်း** | ၸႂ်ႉမိူဝ်ႈလူဝ်ႇမၼ်ႈၸႂ်ငဝ်းလၢႆး ဢွၼ်တၢင်းတေႁဵတ်း | ၸႂ်ႉမိူဝ်ႈလူဝ်ႇႁႂ်ႈမၼ်းႁဵတ်းၵၢၼ်ၵႂႃႇဢွၼ်တၢင်း ၸင်ႇၵွႆႈထတ်း |

---

### 4. ၶေႃႈတွၼ်းသတိ: Infinite Loop

သင်ႁဝ်းလိုမ်းလူတ်း (Decrease) ဢမ်ႇၼၼ် တိုး (Increase) တူဝ်ၼပ်ႉၼႂ်း `while`၊ ငဝ်းလၢႆးမၼ်းတေပဵၼ် `true` တႃႇသေႇ သေ ႁဵတ်းႁႂ်ႈ Program ႁဝ်းႁဵတ်းၵၢၼ်ဢမ်ႇၵိုတ်း (Infinite Loop) ယဝ်ႉ။

```kotlin
// ယႃႇႁဵတ်းၼႆ:
// while (true) { println("ၵဝ်တေဢမ်ႇၵိုတ်း!") }

```

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Condition:** ငဝ်းလၢႆး (True/False) ဢၼ်ၵုမ်းထိင်း Loop။
* **Infinite Loop:** ၵၢၼ်ႁဵတ်းၵၢၼ်တူပ်ႉၵၼ် ဢၼ်ဢမ်ႇမီးဝၼ်းသုတ်း။
* **Body:** တွၼ်ႈၶူတ်ႉဢၼ်ယူႇၼႂ်းပိၵ်ႇၵႃ `{ }` ဢၼ်တေလႆႈႁဵတ်းၵၢၼ်တူပ်ႉၵၼ်။
