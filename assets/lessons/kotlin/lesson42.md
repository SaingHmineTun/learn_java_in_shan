# တွၼ်ႈသွၼ် 42: Lazy Initialization (`by lazy`)

**Lazy Initialization** ပွင်ႇဝႃႈ "ၵၢၼ်တင်ႈၵႃႈ မိူဝ်ႈၶႆႉ"။ သင်ဝႃႈ Property ၼိုင်ႈဢၼ် လူဝ်ႇၸႂ်ႉ Memory ၼမ် ဢမ်ႇၼၼ် လူဝ်ႇၼပ်ႉသၢင်ႈႁိုင်၊ ႁဝ်းတေဢမ်ႇႁႂ်ႈမၼ်းႁဵတ်းၵၢၼ် တေႃႇပေႃးထိုင်ၶၢဝ်းယၢမ်း ဢၼ်ႁဝ်း "ႁွင်ႉၸႂ်ႉ" မၼ်းတႄႉတႄႉယဝ်ႉ။

### 1. လၢႆးတႅမ်ႈ (Syntax)

ႁဝ်းၸႂ်ႉ Keyword **`by lazy { }`** ၵိုၵ်းတင်း **`val`** ယဝ်ႉ။

```kotlin
val pi: Double by lazy {
    println("တိုၵ်ႉၼပ်ႉသၢင်ႈၵႃႈ pi ယူႇ...")
    3.14159
}

fun main() {
    println("တႄႇ Program...")
    // ၵမ်းၼႆႉ pi တေပႆႇလႆႈႁဵတ်းၵၢၼ်
    
    println(pi) // ႁွင်ႉၸႂ်ႉပွၵ်ႈ 1: ၶူတ်ႉၼႂ်း { } တေႁဵတ်းၵၢၼ်
    println(pi) // ႁွင်ႉၸႂ်ႉပွၵ်ႈ 2: မၼ်းတေဢဝ် Value ဢၼ်သိမ်းဝႆႉ (Cached) မႃးပၼ်ၵမ်းလဵဝ်
}

```

---

### 2. ပၵ်းပိူင်လမ်ႇလွင်ႈ

* **`val` Only:** `by lazy` ၸႂ်ႉလႆႈၵူၺ်းတင်း **`val`** (Read-only) ၵွပ်ႈမၼ်းတေၼပ်ႉသၢင်ႈ Value ပွၵ်ႈလဵဝ် သေသိမ်းဝႆႉတႃႇသေႇ။
* **Thread Safety:** တႄႇငဝ်ႈမၼ်း `by lazy` တေမီးလွင်ႈႁႄႉၵင်ႈ (Synchronized) ႁႂ်ႈမၼ်းၼပ်ႉသၢင်ႈလႆႈပွၵ်ႈလဵဝ် တင်းသႅင်ဝႃႈမီးလၢႆလၢႆ Thread မႃးႁွင်ႉၸႂ်ႉၵေႃႈယဝ်ႉ။

---

### 3. ပႅၵ်ႇပၢင်ႇၵၼ်တင်း `lateinit` ၸိူဝ်ႉႁိူဝ်?

| တွၼ်ႈတႅၵ်ႈ | **lateinit** | **by lazy** |
| --- | --- | --- |
| **Keyword** | `var` | `val` |
| **ၵၢၼ်သႂ်ႇ Value** | ႁဝ်းလူဝ်ႇသႂ်ႇပၼ်မၼ်း (Manual) | မၼ်းၼပ်ႉသၢင်ႈႁင်းၵူၺ်း (Automatic) |
| **Type** | ဢမ်ႇၸႂ်ႈ Primitive (Int, etc.) | ၸႂ်ႉလႆႈၵူႈ Type |
| **ၵၢၼ်ၸႂ်ႉတိုဝ်း** | ၸႂ်ႉမိူဝ်ႈပႂ်ႉ Framework Setup | ၸႂ်ႉမိူဝ်ႈလူဝ်ႇယႃႉ Memory/Performance |

---

### 4. ၵွပ်ႈသင်ၸင်ႇလီ?

* **Performance:** ၸွႆႈႁႂ်ႈ Program ပိုတ်ႇလႅတ်းၽႂ်း (Startup time) ၵွပ်ႈမၼ်းဢမ်ႇလႆႈသၢင်ႈၵူႈယၢင်ႇဝႆႉတႄႇငဝ်ႈ။
* **Resource Saving:** သင်ဝႃႈ Property ၼၼ်ႉ ၵူၼ်းၸႂ်ႉ (User) ဢမ်ႇပိုတ်ႇတူၺ်း၊ မၼ်းၵေႃႈတေဢမ်ႇသုမ်း Memory ပႅတ်ႈလၢႆလၢႆ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Lazy Initialization:** ၵၢၼ်သၢင်ႈ Value မိူဝ်ႈလူဝ်ႇၸႂ်ႉၵူၺ်း။
* **Delegation (`by`):** ၵၢၼ်ဢၢပ်ႈပုၼ်ႈၽွၼ်းပၼ်တူဝ်တၢင်ႇဢၼ် (Lazy) ႁဵတ်းၵၢၼ်တႅၼ်း။
* **Caching:** ၵၢၼ်သိမ်းၽူၼ်လႅတ်းဝႆႉ တႃႇၸႂ်ႉၶိုၼ်းငၢႆႈငၢႆႈ။
* **Thread Safety:** ၵၢၼ်ႁႄႉၵင်ႈပၼ်ႁႃ မိူဝ်ႈမီးၵၢၼ်ႁဵတ်းၵၢၼ်လၢႆလၢႆဢၼ် ၼႂ်းၶၢဝ်းယၢမ်းလဵဝ်ၵၼ်။
