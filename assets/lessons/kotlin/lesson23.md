# တွၼ်ႈသွၼ် 23: ၵၢၼ်ႁဵတ်းၵၢၼ်တူပ်ႉၵၼ် (For Loops)

လၢႆးတႅမ်ႈ `for` ၼႂ်း Kotlin ၼၼ်ႉ တူၺ်းငၢႆႈသေ Java ၼမ်ဝႆႉၶႃႈ။ ႁဝ်းၸႂ်ႉၶိူင်ႈမၢႆ **`in`** တႃႇလၢတ်ႈဝႃႈ "ႁႂ်ႈတူဝ်ၼႆႉ ဝႅၼ်ၶၢမ်ႈၵႂႃႇၼႂ်းၸုမ်းၼႆႉ"။

### 1. ၸႂ်ႉ `for` ၵိုၵ်းတင်း Ranges

ႁဝ်းၸၢင်ႈဢဝ် Ranges ဢၼ်ႁဝ်းႁဵၼ်းမႃးၼႂ်းတွၼ်ႈသွၼ် 22 ၼၼ်ႉ မႃးၸႂ်ႉလႆႈၵမ်းလဵဝ်ယဝ်ႉ။

```kotlin
// ၼပ်ႉ 1 တေႃႇ 5
for (i in 1..5) {
    println("တူဝ်ၼပ်ႉ: $i")
}

// ၼပ်ႉထွႆလင် 3 တေႃႇ 1
for (x in 3 downTo 1) {
    println("Countdown: $x")
}

```

---

### 2. ၸႂ်ႉ `for` ၵိုၵ်းတင်း Collections (List)

မိူဝ်ႈႁဝ်းမီး List ၶွင်ၶေႃႈမုၼ်း (မိူၼ်ၼင်ႇ ၸိုဝ်ႈၵူၼ်း)၊ ႁဝ်းၸၢင်ႈၸႂ်ႉ `for` တႃႇဢဝ်ဢွၵ်ႇမႃးၼႄလၢႆလၢႆတူဝ်လႆႈယဝ်ႉ။

```kotlin
val fruits = listOf("Apple", "Banana", "Cherry")

for (fruit in fruits) {
    println("ၵဝ်ၶႂ်ႈၵိၼ်: $fruit")
}

```

---

### 3. ၵၢၼ်ဢဝ်ပႃး Index (တႅမ်င်ႈ/တွၼ်ႈ)

မၢင်ပွၵ်ႈႁဝ်းဢမ်ႇၵူၺ်းၶႂ်ႈလႆႈ "ၶေႃႈမုၼ်း"၊ ႁဝ်းယင်းၶႂ်ႈႁူႉပႃးဝႃႈ မၼ်းယူႇတီႈ "တႅမ်င်ႈ" (Index) ဢၼ်လႂ်။ ႁဝ်းၸၢင်ႈၸႂ်ႉ `.withIndex()` လႆႈၶႃႈ။

```kotlin
val colors = listOf("Red", "Green", "Blue")

for ((index, color) in colors.withIndex()) {
    println("တႅမ်င်ႈ $index ပဵၼ်သီ $color")
}

```

---

### 4. ၵွပ်ႈသင် `for` ၼႂ်း Kotlin ၸင်ႇလီ?

* **ဢမ်ႇလူဝ်ႇ Index:** သင်ႁဝ်းဢမ်ႇလူဝ်ႇၼပ်ႉ 0 တေႃႇ length၊ ႁဝ်းဢမ်ႇလူဝ်ႇသၢင်ႈ Variable `i` မႃးၵုမ်းထိင်းမၼ်း (မိူၼ်ၼင်ႇ `for(int i=0; i<n; i++)`)။
* **Safe:** မၼ်းတေဢမ်ႇမီးပၼ်ႁႃ `IndexOutOfBounds` ငၢႆႈငၢႆႈ ၵွပ်ႈမၼ်းဝႅၼ်ၵႂႃႇၸွမ်းၼင်ႇၶေႃႈမုၼ်းမီးၵူၺ်း။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Iteration:** ၵၢၼ်ႁဵတ်းၵၢၼ်တူပ်ႉၵၼ်ၼိူဝ်ၸုမ်းၶေႃႈမုၼ်း။
* **Collection:** ၸုမ်းၶေႃႈမုၼ်း (မိူၼ်ၼင်ႇ List, Set, Map)။
* **Index:** တူဝ်ၼပ်ႉဢၼ်ၼႄတႅမ်င်ႈၶွင်ၶေႃႈမုၼ်း (တႄႇၼပ်ႉတီႈ 0)။
* **Iteratable:** သိင်ႇဢၼ်ၸၢင်ႈဢဝ်မႃးႁဵတ်း Loop လႆႈ။
