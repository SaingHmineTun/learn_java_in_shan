# တွၼ်ႈသွၼ် 3: ၵၢၼ်တႅမ်ႈ Program ဢွၼ်တၢင်းသုတ်း (The main function)

ၼႂ်းၽႃႇသႃႇ Kotlin ၼၼ်ႉ တႃႇတေႁႂ်ႈ Program ႁဝ်းတႄႇႁဵတ်းၵၢၼ်လႆႈၼၼ်ႉ မၼ်းလူဝ်ႇမီး **"ႁူဝ်ၼႃႈ"** ဢၼ်ၼိုင်ႈ။ ႁူဝ်ၼႃႈၼၼ်ႉ ႁဝ်းႁွင်ႉဝႃႈ **`main` function** ယဝ်ႉ။

### 1. တႅမ်ႈၶူတ်ႉ Hello World

ပိုတ်ႇ IntelliJ IDEA ၶိုၼ်ႈမႃး၊ သၢင်ႈ File မႂ်ႇ ဢၼ်ၸိုဝ်ႈဝႃႈ `Main.kt` သေ တႅမ်ႈၶူတ်ႉပႃႈတႂ်ႈၼႆႉလႅတ်းလႅတ်းၶႃႈ:

```kotlin
fun main() {
    println("Hello, Shan Developer!")
}

```

### 2. သပ်းလႅင်းၶေႃႈၵႂၢမ်းၼႂ်းၶူတ်ႉ

ႁဝ်းမႃးတူၺ်းၵၼ်ဝႃႈ ဢၼ်ႁဝ်းတႅမ်ႈၵႂႃႇၼၼ်ႉ မၼ်းမီးတီႈပွင်ႇဝႃႈသင်လႃႈ:

* **`fun`**: ၶေႃႈၼႆႉ ယေႃးမႃးတီႈ "Function" ယဝ်ႉ။ မၼ်းပဵၼ်ၵၢၼ်လၢတ်ႈၼႄဝႃႈ "ႁဝ်းတေတႄႇသၢင်ႈ လွၵ်းလၢႆးႁဵတ်းၵၢၼ်ဢၼ်ၼိုင်ႈယဝ်ႉၼႃ"။
* **`main`**: ၼႆႉပဵၼ်ၸိုဝ်ႈမၼ်း။ JVM တေမႃးႁႃၸိုဝ်ႈၼႆႉ ပဵၼ်ဢၼ်ဢွၼ်တၢင်းသုတ်း တႃႇတႄႇ Run Program။ သင်ဢမ်ႇမီး `main` ၼႆ Program တေႁဵတ်းၵၢၼ်ဢမ်ႇလႆႈ။
* **`()`**: ဝၢႆးၸိုဝ်ႈ Function တေလႆႈပႃးၶွပ်ႇၼႆႉတႃႇသေႇ (ဝႆႉတႃႇသႂ်ႇၶေႃႈမုၼ်းပိုင်ႈဢိင်)။
* **`{ ... }`**: လိၵ်ႈဢၼ်မီးၼႂ်းပိၵ်ႇၵႃၼႆႉ ႁဝ်းႁွင်ႉဝႃႈ **Function Body**။ မၼ်းပဵၼ်တီႈဢၼ်ႁဝ်းတေတႅမ်ႈ သင်ႇႁႂ်ႈၶွမ်ႇပိဝ်ႇတႃႇႁဵတ်းၵၢၼ်။
* **`println`**: ၼႆႉပဵၼ်ၵၢၼ်သင်ႇႁႂ်ႈ "ၼႄလိၵ်ႈဢွၵ်ႇမႃးတီႈၼႃႈၸေႃ (Console)" သေ လူင်းထႅဝ်မႂ်ႇ။

---

### 3. လၢႆး Run Program

မိူဝ်ႈတႅမ်ႈယဝ်ႉယဝ်ႉ:

1. တူၺ်းတီႈၼႃႈၸေႃ တီႈၶူတ်ႉႁဝ်းၼၼ်ႉ တေမီးႁၢင်ႈ **သၢမ်ၸဵမ်သီႇၶဵဝ် (Play Button)** ၼၼ်ႉၶႃႈ။
2. ၼဵၵ်းတီႈႁၢင်ႈၼၼ်ႉသေ လိူၵ်ႈ **Run 'MainKt'**။
3. တူၺ်းတီႈပႃႈတႂ်ႈ ၼႃႈၸေႃ (Console)၊ မၼ်းတေဢွၵ်ႇလိၵ်ႈမႃးဝႃႈ: `Hello, Shan Developer!`

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Function:** ၸုမ်ႇၶေႃႈသင်ႇ ဢၼ်ႁဝ်းႁွင်ႉၸႂ်ႉလႆႈ တႃႇႁဵတ်းၵၢၼ်သင်သင်ၼိုင်ႈ။
* **Main Function:** တီႈတႄႇႁဵတ်းၵၢၼ် (Entry Point) ၶွင် Program ၵူႈဢၼ်။
* **Console/Terminal:** ၼႃႈၸေႃလိၵ်ႈ ဢၼ်ၼႄၽူၼ်လႅတ်း (Output) ၶွင်ၶူတ်ႉႁဝ်း။
