# တွၼ်ႈသွၼ် 6: မဵဝ်းၶေႃႈမုၼ်းပိုၼ်ႉထၢၼ် (Numbers, Booleans & Char)

တႃႇတေတႅမ်ႈ Program ဢၼ်ၵတ်ႉၶႅၼ်ႇၼၼ်ႉ ႁဝ်းလူဝ်ႇလိူၵ်ႈၸႂ်ႉ "ၵွၵ်း" (Type) ဢၼ်ၵိုၵ်းၵၼ်တင်း "ၶေႃႈမုၼ်း" ႁဝ်းယဝ်ႉ။

### 1. Numbers (တူဝ်ၼပ်ႉ)

တူဝ်ၼပ်ႉၼႂ်း Kotlin ၸႅၵ်ႇပဵၼ် 2 ၸုမ်းလူင်:

**ၵ) တူဝ်ၼပ်ႉတဵမ် (Integers):**

* **Byte:** (8-bit) တူဝ်ၼပ်ႉဢွၼ်ႇ -128 တေႃႇ 127။
* **Short:** (16-bit) တူဝ်ၼပ်ႉပၢၼ်ၵາງ။
* **Int:** (32-bit) ဢၼ်ၵႆႉၸႂ်ႉၼမ်သုတ်း တႃႇတူဝ်ၼပ်ႉၵူႈလွင်ႈ။
* **Long:** (64-bit) တႃႇတူဝ်ၼပ်ႉဢၼ်ယႂ်ႇမၢၵ်ႈ (လူဝ်ႇသႂ်ႇ `L` ၽၢႆႇလင် တူဝ်ယၢင်ႇ: `100L`)။

**ၶ) တူဝ်ၼပ်ႉမီးၸုတ်း (Floating-point numbers):**

* **Float:** (32-bit) တူဝ်ၼပ်ႉမီးၸုတ်း (လူဝ်ႇသႂ်ႇ `f` ၽၢႆႇလင် တူဝ်ယၢင်ႇ: `10.5f`)။
* **Double:** (64-bit) တူဝ်ၼပ်ႉမီးၸုတ်း ဢၼ်မီးလွင်ႈတႅတ်ႈတေႃး (Precision) သုင်။

```kotlin
val myAge: Int = 27
val myHeight: Float = 1.75f
val myWeight: Double = 70.0

```
---

### 2. Booleans (ၶေႃႈမုၼ်းထတ်းသိၼ်)

Boolean ၼႆႉ မီးၶေႃႈမုၼ်းလႆႈ 2 ယၢင်ႇၵူၺ်း။ မၼ်းဝႆႉၸႂ်ႉတႃႇထတ်းသိၼ် လွင်ႈပဵၼ်တႄႉ ဢမ်ႇၼၼ် လွင်ႈပႅတ်ႉလႅၼ်။

* **true:** မၼ်ႈၸႂ် / ထုၵ်ႇတေႃး / ပဵၼ်တႄႉ။
* **false:** ဢမ်ႇမၼ်ႈၸႂ် / ၽိတ်း / ပႅတ်ႉလႅၼ်။

```kotlin
val isKotlinFun: Boolean = true
val isFishFlying: Boolean = false

```

---

### 3. Characters (တူဝ်လိၵ်ႈလဵဝ်)

**Char** ၼႆႉ ဝႆႉၸႂ်ႉတႃႇသိမ်း "တူဝ်လိၵ်ႈ" ဢၼ်လဵဝ်ၵူၺ်း။ တေလႆႈၸႂ်ႉၶွပ်ႇ **' '** (Single Quotes) ၵမ်းသိုဝ်ႈယဝ်ႉ။

```kotlin
val grade: Char = 'A'
val letter: Char = 'ၵ'

```

> **တွၼ်းသတိ:** `Char` ဢမ်ႇမိူၼ်တင်း `String` (လိၵ်ႈယၢဝ်း)။ String တေၸႂ်ႉ `" "` (Double Quotes) ယဝ်ႉ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Integer:** တူဝ်ၼပ်ႉတဵမ် (ဢမ်ႇမီးၸုတ်း)။
* **Floating-point:** တူဝ်ၼပ်ႉဢၼ်မီးၸုတ်း (Decimal)။
* **Boolean:** ၶေႃႈမုၼ်းဢၼ်မီးလွင်ႈ True ဢမ်ႇၼၼ် False။
* **Bit/Byte:** ပဵၼ်လွၵ်းလၢႆးတူင်ႇတႅၵ်ႇ တၢင်းယႂ်ႇၶေႃႈမုၼ်းၼႂ်း Memory။
