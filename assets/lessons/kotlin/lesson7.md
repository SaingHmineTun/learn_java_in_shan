# တွၼ်ႈသွၼ် 7: String Templates လႄႈ လိၵ်ႈလၢႆထႅဝ် (Multi-line Strings)

### 1. String Templates (ၵၢၼ်သႂ်ႇၶေႃႈမုၼ်းၼႂ်းလိၵ်ႈ)

မိူဝ်ႈႁဝ်းၶႂ်ႈဢဝ် တူဝ်လႅၵ်ႈလၢႆႈ (Variable) ၵႂႃႇၸႂ်ႉၼႂ်းလိၵ်ႈၼၼ်ႉ ႁဝ်းဢမ်ႇလူဝ်ႇၸႂ်ႉၶိူင်ႈမၢႆ `+` သေတႅမ်ႈယၢဝ်းယၢဝ်းယဝ်ႉ။ ႁဝ်းၸၢင်ႈၸႂ်ႉၶိူင်ႈမၢႆ **`$` (Dollar sign)** သႂ်ႇၽၢႆႇၼႃႈၸိုဝ်ႈ Variable ၼၼ်ႉလႆႈၵမ်းလဵဝ်ၶႃႈ။

```kotlin
val name = "Sai Sai"
val age = 25

// လၢႆးတႅမ်ႈၵဝ်ႇ: "Hello, " + name + "! Your age is " + age
// လၢႆးတႅမ်ႈ Kotlin (String Template):
println("Hello, $name! Your age is $age")

```

သင်ႁဝ်းၶႂ်ႈဢဝ် **Expression** (ၵၢၼ်ၼပ်ႉသၢင်ႈ) သႂ်ႇၼႂ်းလိၵ်ႈ ႁဝ်းၸၢင်ႈၸႂ်ႉ **`${ ... }`** ယဝ်ႉ။

```kotlin
val price = 100
println("တၢင်းသဵင်ႈသဵင်ႈပဵၼ်: ${price * 2} ၵျၢတ်ႉ") 
// မၼ်းတေဢွၵ်ႇမႃး: တၢင်းသဵင်ႈသဵင်ႈပဵၼ်: 200 ၵျၢတ်ႉ

```

### 2. Multi-line Strings (လိၵ်ႈလၢႆထႅဝ်)

သင်ႁဝ်းၶႂ်ႈတႅမ်ႈလိၵ်ႈဢၼ်ယၢဝ်းယၢဝ်း ဢမ်ႇၼၼ် လိၵ်ႈဢၼ်မီးလၢႆထႅဝ် (မိူၼ်ၼင်ႇ ႁူဝ်လိၵ်ႈ/ၶေႃႈၵႂၢမ်း) ႁဝ်းၸၢင်ႈၸႂ်ႉၶွပ်ႇ **`"""` (Triple Quotes)** ယဝ်ႉ။

```kotlin
val message = """
    မႂ်ႇသုင်ၶႃႈ၊
    ႁိၼ်းတွၼ်းၸႂ်ႉ Kotlin ၼႆႉ
    ငၢႆႈလႄႈ ၽႂ်းဝႆႉယဝ်ႉ။
""".trimIndent()

println(message)

```

* **`.trimIndent()`**: ၼႆႉပဵၼ်ၵၢၼ်တတ်းပႅတ်ႈ "လွၵ်းလၢႆႇ" (Spaces) ဢၼ်မီးၽၢႆႇၼႃႈ ႁႂ်ႈလိၵ်ႈႁဝ်းၶဝ်ႈထႅဝ်ၵၼ်လီလီယဝ်ႉ။

---

### 3. String Properties & Functions

ႁဝ်းၸၢင်ႈၸႂ်ႉ Function ပိုၼ်ႉထၢၼ် တႃႇၸတ်းၵၢၼ်လိၵ်ႈလႆႈၼင်ႇၼႆ:

* `.length`: ၼပ်ႉတူဝ်လိၵ်ႈတင်းหมด။
* `.uppercase()`: လႅၵ်ႈပဵၼ်တူဝ်လိၵ်ႈလူင် (ဢင်းၵိတ်ႉ)။
* `.lowercase()`: လႅၵ်ႈပဵၼ်တူဝ်လိၵ်ႈဢွၼ်ႇ (ဢင်းၵိတ်ႉ)။

```kotlin
val city = "Keng Tung"
println(city.length) // 9
println(city.uppercase()) // KENG TUNG

```

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **String Template:** ၵၢၼ်ဢဝ် Variable သႂ်ႇၼႂ်းလိၵ်ႈ လူၺ်ႈၸႂ်ႉ `$`.
* **Expression:** ၶူတ်ႉဢၼ်ၼပ်ႉသၢင်ႈယဝ်ႉသေ ဢွၵ်ႇမႃးပဵၼ် "Value" သင်သင်ၼိုင်ႈ။
* **Raw String:** လိၵ်ႈဢၼ်တႅမ်ႈၼႂ်း `"""` ဢၼ်ဢမ်ႇလႅၵ်ႈလၢႆႈႁၢင်ႈၽၢင်မၼ်း။
