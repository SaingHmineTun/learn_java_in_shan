# တွၼ်ႈသွၼ် 64: The Collection Hierarchy

ၼႂ်း Kotlin, Collections တင်းသႅင် ၸၼ်ႉသုင်သုတ်းမၼ်းတႄႇမႃးတီႈ `Iterable` သေ ၸႅၵ်ႇဢွၵ်ႇပဵၼ် 3 သၢႆလူင်ယဝ်ႉ။

### 1. List (Order and Duplicates)

**List** ပဵၼ်ၵၢၼ်သိမ်းၶေႃႈမုၼ်း ဢၼ်မၵ်းမၼ်ႈ **"လႅမ်ႇတပ်ႉ" (Order)**။

* **လွင်ႈပႅၵ်ႇပၢင်ႇ:** သိမ်းၶေႃႈမုၼ်းဢၼ် "သမ်ႉၵၼ်" (Duplicate) လႆႈ။
* **ၵၢၼ်ၶဝ်ႈၸႂ်ႉ:** ၸႂ်ႉ **Index** (0, 1, 2...) တႃႇၸီႉဢဝ်တူဝ်ဢၼ်ႁဝ်းၶႂ်ႈလႆႈ။
* **တူဝ်ယၢင်ႇ:** မိူၼ်ၼင်ႇ "သဵၼ်ႈၸိုဝ်ႈၵူၼ်းႁဵတ်းၵၢၼ်"။ ၸိုဝ်ႈသမ်ႉၵၼ်ၵေႃႈမီးလႆႈ၊ လႅမ်ႇတပ်ႉၵေႃႈလမ်ႇလွင်ႈ။

```kotlin
val list = listOf("Sai", "Kyaw", "Sai") 
println(list[0]) // Sai

```

---

### 2. Set (Uniqueness)

**Set** ပဵၼ်ၵၢၼ်သိမ်းၶေႃႈမုၼ်း ဢၼ်မၵ်းမၼ်ႈဝႃႈ **"ၵူႈတူဝ်တေလႆႈပႅၵ်ႇၵၼ်"**။

* **လွင်ႈပႅၵ်ႇပၢင်ႇ:** ဢမ်ႇပၼ်သိမ်းၶေႃႈမုၼ်းသမ်ႉၵၼ်။ သင်သႂ်ႇ "Sai" ၶဝ်ႈၵႂႃႇသွင်ပွၵ်ႈ၊ မၼ်းတေသိမ်းဝႆႉပၼ်ၵူၺ်းတူဝ်လဵဝ်။
* **လႅမ်ႇတပ်ႉ:** ၵႆႉလေႇဢမ်ႇမၵ်းမၼ်ႈလႅမ်ႇတပ်ႉ (Unordered)။
* **တူဝ်ယၢင်ႇ:** မိူၼ်ၼင်ႇ "မၢႆပတ်ႉသပိုတ်ႇ" (ID Card)။ တေလႆႈဢမ်ႇမီးၽႂ်သမ်ႉၽႂ်။

```kotlin
val set = setOf("Sai", "Kyaw", "Sai")
println(set) // [Sai, Kyaw] (Sai တူဝ်ထိုင်သွင်တေႁၢႆၵႂႃႇ)

```

### 3. Map (Key-Value Pairs)

**Map** ပဵၼ်ၵၢၼ်သိမ်းၶေႃႈမုၼ်း ဢၼ်မႃးပဵၼ် **"ၵူပ်ႉ" (Pairs)**။ မၼ်းဢမ်ႇၸႂ်ႈသၢႆၸိူဝ်ႉၶိူဝ်းလဵဝ်ၵၼ်တင်း `Collection` interface ၵမ်းလဵဝ် ၵူၺ်းၵႃႈမၼ်းယူႇၼႂ်းၸုမ်း Collections မိူၼ်ၵၼ်။

* **Key:** တူဝ်ၸီႉ (တေလႆႈဢမ်ႇသမ်ႉၵၼ် - Unique)။
* **Value:** ၶေႃႈမုၼ်း (သမ်ႉၵၼ်လႆႈ)။
* **တူဝ်ယၢင်ႇ:** မိူၼ်ၼင်ႇ "ပပ်ႉသပ်း" (Dictionary)။ ၶေႃႈၵႂၢမ်း (Key) တေလႆႈမီးဢၼ်လဵဝ်၊ ၵူၺ်းၵႃႈတီႈပွင်ႇ (Value) မၼ်းၸၢင်ႈမိူၼ်ၵၼ်လႆႈ။

```kotlin
val map = mapOf("TH" to "Thailand", "MM" to "Myanmar")
println(map["TH"]) // Thailand

```

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Index:** မၢႆၶပ်ႉ (0-based) ဢၼ်ၸႂ်ႉၸီႉဢဝ်ၶေႃႈမုၼ်းၼႂ်း List။
* **Unique:** ဢၼ်လဵဝ်၊ ဢမ်ႇမိူၼ်ၽႂ်။
* **Duplicate:** ဢၼ်သမ်ႉၵၼ်။
* **Key-Value Pair:** ၵူပ်ႉၶေႃႈမုၼ်း ဢၼ်မီးတူဝ်ၸီႉ လႄႈ ၵႃႈမၼ်း။
