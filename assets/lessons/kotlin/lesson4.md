# တွၼ်ႈသွၼ် 4: တူဝ်လႅၵ်ႈလၢႆႈ (Variables) — val vs var

ၼႂ်း Kotlin ၼၼ်ႉ ႁဝ်းမီးလၢႆးသၢင်ႈတူဝ်လႅၵ်ႈလၢႆႈ (Variable) ယူႇ 2 ပိူင်လူင်။ လွင်ႈပႅၵ်ႇပၢင်ႇမၼ်းၼၼ်ႉ ၸိၼႄဝႃႈ ၶေႃႈမုၼ်းၼႂ်းမၼ်းၼၼ်ႉ လႅၵ်ႈလၢႆႈလႆႈ ဢမ်ႇၼၼ် လႅၵ်ႈလၢႆႈဢမ်ႇလႆႈၼႆယဝ်ႉ။

### 1. val (Immutable - လႅၵ်ႈလၢႆႈဢမ်ႇလႆႈ)

`val` ၼႆႉ ယေႃးမႃးတီႈ "Value" ယဝ်ႉ။ မိူဝ်ႈႁဝ်းဢဝ်ၶေႃႈမုၼ်းသႂ်ႇပၼ်မၼ်းယဝ်ႉ ၼိုင်ႈပွၵ်ႈ၊ ႁဝ်းတေဢမ်ႇၸၢင်ႈလႅၵ်ႈလၢႆႈမၼ်းလႆႈထႅင်ႈ (Read-only)။

```kotlin
val name = "Sai Sai"
// name = "Kyaw Kyaw" // ၼႆႉတေ Error!! ၵွပ်ႈဝႃႈ val လႅၵ်ႈဢမ်ႇလႆႈ။

```

### 2. var (Mutable - လႅၵ်ႈလၢႆႈလႆႈ)

`var` ၼႆႉ ယေႃးမႃးတီႈ "Variable" ယဝ်ႉ။ မၼ်းမိူၼ်ၼင်ႇၵွၵ်း ဢၼ်ႁဝ်းထွၵ်ႇၶေႃႈမုၼ်းၵဝ်ႇပႅတ်ႈ သေ သႂ်ႇၶေႃႈမုၼ်းမႂ်ႇလႆႈတႃႇသေႇ။

```kotlin
var age = 20
age = 21 // ႁဵတ်းလႆႈ! ၵွပ်ႈပိူဝ်ႈ var ပဵၼ်တူဝ်ဢၼ်လႅၵ်ႈလၢႆႈလႆႈ။

```

---

### 3. လွၵ်းလၢႆးတႅမ်ႈ (Syntax)

လၢႆးတႅမ်ႈ Variable ၼႂ်း Kotlin တႄႉ ပဵၼ်ၼႆၶႃႈ:

> **Keyword** + **ၸိုဝ်ႈတူဝ်လႅၵ်ႈလၢႆႈ** = **ၶေႃႈမုၼ်း**

* **တူဝ်ယၢင်ႇ:** `val language = "Kotlin"`

### 4. ၵွပ်ႈသင် Kotlin ၸင်ႇမီးတင်း val လႄႈ var?

တႅမ်ႈၶူတ်ႉၼႆႉ လွင်ႈ **ႁူမ်ႇလူမ်ႈ (Safety)** ပဵၼ်ဢၼ်လုမ်းၼႃးသုတ်း။

* သင်ႁဝ်းႁူႉဝႃႈ ၶေႃႈမုၼ်းဢၼ်ၼၼ်ႉ တေဢမ်ႇလႅၵ်ႈလၢႆႈ (မိူၼ်ၼင်ႇ ဝၼ်းၵိူတ်ႇ၊ ၸိုဝ်ႈမိူင်း) — ႁဝ်းထုၵ်ႇလီၸႂ်ႉ `val`။
* သင်ၶေႃႈမုၼ်းၼၼ်ႉ တေလႆႈလႅၵ်ႈလၢႆႈယူႇတႃႇသေႇ (မိူၼ်ၼင်ႇ ၶႅၼ်းၵိမ်း/Score, ရႃႇသီႇဢုၵ်ႉတု) — ႁဝ်းၸင်ႇၸႂ်ႉ `var`။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Immutable:** ဢၼ်ၼိမ်ဝႆႉ၊ လႅၵ်ႈလၢႆႈဢမ်ႇလႆႈ။
* **Mutable:** ဢၼ်လႅၵ်ႈလၢႆႈလႆႈတႃႇသေႇ။
* **Assignment:** ၵၢၼ်ဢဝ်ၶေႃႈမုၼ်းသႂ်ႇပၼ် Variable (ၸႂ်ႉၶိူင်ႈမၢႆ `=`)။
