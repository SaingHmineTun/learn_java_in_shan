# တွၼ်ႈသွၼ် 21: ၵၢၼ်ၸႂ်ႉ 'when' ဢၼ်ဢမ်ႇမီး Argument (when without Arguments)

မိူဝ်ႈႁဝ်းၸႂ်ႉ `when` လူၺ်ႈဢမ်ႇသႂ်ႇသင်ၼႂ်းဝူင်းလႅမ် `when { ... }` ၼၼ်ႉ၊ ၵူႈထႅဝ် (Branch) ၼႂ်းမၼ်း တေလႆႈပဵၼ် **Boolean Expression** (ငဝ်းလၢႆးဢၼ်ပဵၼ် `true` ဢမ်ႇၼၼ် `false`) ယဝ်ႉ။

### 1. လွၵ်းလၢႆးတႅမ်ႈ

လၢႆးၼႆႉ လီသုတ်းတႃႇၸႂ်ႉ မိူဝ်ႈႁဝ်းမီးငဝ်းလၢႆးဢၼ် "ဢမ်ႇၵိုၵ်းၵၼ်" (Multiple complex conditions)။

```kotlin
val score = 85
val attendance = 95

when {
    score >= 90 && attendance >= 90 -> println("ၵတ်ႉၶႅၼ်ႇသုတ်း (Exceptional)")
    score >= 80 -> println("ၵတ်ႉၶႅၼ်ႇ (Great)")
    attendance < 50 -> println("လူဝ်ႇမႃးၶိုၼ်းႁဵၼ်းၼမ်ၼမ်")
    else -> println("တၢမ်းပိူင်")
}

```

---

### 2. ၵွပ်ႈသင်ၸင်ႇထုၵ်ႇလီၸႂ်ႉလၢႆးၼႆႉ?

* **တူၺ်းငၢႆႈ (Clean Code):** တႅၼ်းတၢင်းတေတႅမ်ႈ `if (...) { ... } else if (...) { ... }` ဢၼ်ယုင်ႈယၢင်ႇ၊ ႁဝ်းၸၢင်ႈၸႂ်ႉ `when` သေ ဝႆႉငဝ်းလၢႆးမၼ်းသိုပ်ႇၵၼ်လူၺ်ႈဢမ်ႇလူဝ်ႇပိၵ်ႇၵႃၼမ်ၼမ်ယဝ်ႉ။
* **လွတ်ႈလႅဝ်း (Flexible):** ႁဝ်းၸၢင်ႈထတ်းတူၺ်း Variable ၵူႈမဵဝ်း ၼႂ်း `when` ဢၼ်လဵဝ်ၵၼ်လႆႈ (တူဝ်ယၢင်ႇ: ထတ်းတင်း `score` လႄႈ `attendance` ၸွမ်းၵၼ်)။

---

### 3. ၸႂ်ႉပဵၼ် Expression (ဢဝ် Value)

မိူၼ်တင်း `when` ပိူင်ငဝ်ႈယဝ်ႉ၊ ႁဝ်းၸၢင်ႈဢဝ်မၼ်းသႂ်ႇၼႂ်း Variable လႆႈယဝ်ႉ။

```kotlin
val temp = 38

val weather = when {
    temp > 40 -> "လိူတ်ႇၼမ်"
    temp in 25..35 -> "မုၵ်ႉဢုၼ်ႇလီ"
    temp < 15 -> "ၵတ်း"
    else -> "တၢမ်းပိူင်"
}

println(weather) // ၽူၼ်မၼ်းတေဢွၵ်ႇ: မုၵ်ႉဢုၼ်ႇလီ

```

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Argument:** ၶေႃႈမုၼ်း ဢၼ်ႁဝ်းသူင်ႇပၼ် Function ဢမ်ႇၼၼ် Control Flow (မိူၼ်ၼင်ႇ Variable ၼႂ်း `when(x)`)။
* **Boolean Expression:** ၶူတ်ႉတွၼ်ႈဢၼ်ပၼ်ၽူၼ်လႅတ်းပဵၼ် `true` ဢမ်ႇၼၼ် `false` ၵူၺ်း။
* **Readability:** လွင်ႈငၢႆႈငၢႆႈ ၼႂ်းၵၢၼ်ဢၢၼ်ႇ လႄႈ ပွင်ႇၸႂ်ၶူတ်ႉ။
