# တွၼ်ႈသွၼ် 26: ၵၢၼ်တႅၵ်ႈတူၺ်းလွင်ႈမိူၼ်ၵၼ် (Structural vs Referential Equality)

မိူဝ်ႈႁဝ်းတႅၵ်ႈတူၺ်း Variable 2 ဢၼ်ၼၼ်ႉ ႁဝ်းလူဝ်ႇလႆႈႁူႉဝႃႈ ႁဝ်းတႅၵ်ႈတူၺ်း "ၶေႃႈမုၼ်းပႃႈၼႂ်း" ဢမ်ႇၼၼ် တႅၵ်ႈတူၺ်း "တီႈယူႇၼႂ်း Memory" ယဝ်ႉ။

### 1. Structural Equality (`==`)

ၶိူင်ႈမၢႆ **`==`** ၼႆႉ ဝႆႉၸႂ်ႉတႃႇထတ်းတူၺ်းဝႃႈ **"ၶေႃႈမုၼ်းပႃႈၼႂ်း မၼ်းမိူၼ်ၵၼ်ႁႃႉ?"** (What is inside?)

* ၼႂ်း Kotlin၊ ၵၢၼ်ၸႂ်ႉ `==` မၼ်းတေၵႂႃႇႁွင်ႉၸႂ်ႉ Function `.equals()` ႁင်းၵူၺ်းမၼ်းယဝ်ႉ။
* မၼ်းထတ်းတူၺ်း **Content/Value** မၼ်းၵူၺ်း။

```kotlin
val a = "Shan State"
val b = "Shan State"

println(a == b) // ၽူၼ်မၼ်း: true (ၵွပ်ႈလိၵ်ႈပႃႈၼႂ်း မိူၼ်ၵၼ်)

```

### 2. Referential Equality (`===`)

ၶိူင်ႈမၢႆ **`===`** ၼႆႉ ဝႆႉၸႂ်ႉတႃႇထတ်းတူၺ်းဝႃႈ Variable တင်း 2 ဢၼ်ၼၼ်ႉ **"ၸี้ၵႂႃႇတီႈ Object တူဝ်လဵဝ်ၵၼ် ၼႂ်း Memory ႁႃႉ?"** (Same object in memory?)

* မၼ်းထတ်းတူၺ်း **Address/Reference** မၼ်း။
* သင်ဝႃႈမၼ်းပဵၼ် Object ဢၼ်သၢင်ႈမႃးလၢႆပွၵ်ႈ (သိုပ်ႇၵၼ်လူၺ်ႈ Memory တၢင်ႇတီႈ) တင်းသႅင်ဝႃႈၶေႃႈမုၼ်းမိူၼ်ၵၼ်ၵေႃႈ `===` တေပဵၼ် `false` ယဝ်ႉ။

```kotlin
val p1 = Person("Sai")
val p2 = Person("Sai")

println(p1 == p2)  // true (သင်ပဵၼ် Data Class ဢၼ်မီး Content မိူၼ်ၵၼ်)
println(p1 === p2) // false (ၵွပ်ႈမၼ်းပဵၼ် Object ၵူၼ်းလုၵ်ႈၵၼ် ယူႇ Memory တၢင်ႇတီႈ)

```

---

### 3. ၵၢၼ်တႅၵ်ႈတူၺ်းလွင်ႈ "ဢမ်ႇမိူၼ်ၵၼ်"

* **`!=`** : ထတ်းတူၺ်းဝႃႈ ၶေႃႈမုၼ်းပႃႈၼႂ်း ဢမ်ႇမိူၼ်ၵၼ် (Structural Not Equal)။
* **`!==`** : ထတ်းတူၺ်းဝႃႈ ဢမ်ႇၸႂ်ႈ Object တူဝ်လဵဝ်ၵၼ် (Referential Not Equal)။

---

### 4. ၶေႃႈပႅၵ်ႇပၢင်ႇ ဢၼ်လူဝ်ႇမၢႆတွင်း

* ၼႂ်း **Java**: `==` ၸႂ်ႉတႃႇတႅၵ်ႈ Reference (မိူၼ် `===` ၼႂ်း Kotlin)။
* ၼႂ်း **Kotlin**: `==` ၸႂ်ႉတႃႇတႅၵ်ႈ Value (မိူၼ် `.equals()` ၼႂ်း Java)။
* **Primitive Types**: တႃႇတူဝ်ၼပ်ႉမိူၼ်ၼင်ႇ `Int`, `Float` ၼၼ်ႉ `==` လႄႈ `===` တေပဵၼ် `true` မိူၼ်ၵၼ် ၵွပ်ႈ Kotlin ၵုမ်းထိင်း Memory ႁင်းၵူၺ်းမၼ်းယဝ်ႉ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Structural Equality:** လွင်ႈမိူၼ်ၵၼ် ၸွမ်းၼင်ႇၶေႃႈမုၼ်းပႃႈၼႂ်း။
* **Referential Equality:** လွင်ႈမိူၼ်ၵၼ် ၸွမ်းၼင်ႇတီႈယူႇၼႂ်း Memory (Pointer)။
* **Memory Address:** တႅမ်င်ႈတီႈယူႇၶွင်ၶေႃႈမုၼ်း ၼႂ်းၶိူင်ႈ Computer။
* **Data Class:** ပဵၼ် Class ၶိုၵ်ႉတွၼ်းၼႂ်း Kotlin ဢၼ်ဝႆႉသိမ်းၶေႃႈမုၼ်းသေ တႅၵ်ႈ `==` လႆႈငၢႆႈငၢႆႈ။
