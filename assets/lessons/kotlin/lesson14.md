# တွၼ်ႈသွၼ် 14: ၶိူင်ႈမၢႆမၼ်ႈၸႂ်ဝႃႈဢမ်ႇပဵၼ် Null (The Not-Null Assertion - `!!`)

ၶိူင်ႈမၢႆ **`!!`** ၼႆႉ ပဵၼ်ၵၢၼ်လၢတ်ႈၼႄ Compiler ဝႃႈ: **"ၵဝ်မၼ်ႈၸႂ် 100% ဝႃႈ Variable ၼႆႉ ဢမ်ႇပဵၼ် null လႆႈတႄႉတႄႉ။ သင်မၼ်းပဵၼ် null ၵေႃႈ ပွႆႇႁႂ်ႈ Program လူႉၵႂႃႇလႆႈၵမ်းလဵဝ်ၶႃႈ!"**

### 1. `!!` ႁဵတ်းၵၢၼ်ၸိူဝ်ႉႁိူဝ်?

မိူဝ်ႈႁဝ်းၸႂ်ႉ `!!` သႂ်ႇၽၢႆႇလင် Variable ဢၼ်ပဵၼ် Nullable ၼၼ်ႉ:

* **သင်မၼ်းမီးၶေႃႈမုၼ်း:** မၼ်းတေႁဵတ်းၵၢၼ်ၵႂႃႇၸွမ်းၼင်ႇပၵ်းပိူင်။
* **သင်မၼ်းပဵၼ် null:** မၼ်းတေပွႆႇ Error ဢၼ်ႁွင်ႉဝႃႈ `NullPointerException` (NPE) မႃးသေ ႁဵတ်းႁႂ်ႈ Program ႁဝ်းၵၢင်ႉ/တၢႆ (Crash) ၵမ်းလဵဝ်။

**တူဝ်ယၢင်ႇ:**

```kotlin
val name: String? = null

// ၵၢၼ်ၸႂ်ႉ !! တီႈၶေႃႈမုၼ်းဢၼ်ပဵၼ် null
val length = name!!.length  // !! တေႁဵတ်းႁႂ်ႈ Program Crash ၵမ်းလဵဝ်!

```

---

### 2. မိူဝ်ႈလႂ်ႁဝ်းထုၵ်ႇလီၸႂ်ႉ `!!`?

ၵူၼ်းတႅမ်ႈ Kotlin ၵတ်ႉၶႅၼ်ႇၶဝ် တေဢမ်ႇၸႂ်ႉ `!!` ပေႃးဢမ်ႇလူဝ်ႇတႄႉတႄႉ။ ႁဝ်းတေၸႂ်ႉမၼ်းမိူဝ်ႈ:

1. ႁဝ်းမၼ်ႈၸႂ်တႄႉတႄႉဝႃႈ ၶေႃႈမုၼ်းၼၼ်ႉ ဢမ်ႇၸၢင်ႈပဵၼ် null (တူဝ်ယၢင်ႇ: ႁဝ်းႁူႉဝႃႈ ႁဝ်းတႅမ်ႈၶေႃႈမုၼ်းသႂ်ႇဝႆႉယဝ်ႉ ၼႂ်းထႅဝ်ပႃႈၼိူဝ်)။
2. ႁဝ်းၶႂ်ႈႁႂ်ႈ Program ႁဝ်းလူႉ/တၢႆ ၵမ်းလဵဝ် သင်ဝႃႈၶေႃႈမုၼ်းၼၼ်ႉ ဢမ်ႇမီး (တႃႇႁူႉဝႃႈပဵၼ်တီႈလႂ်)။

---

### 3. လွၵ်းလၢႆးဢၼ်ႁူမ်ႇလူမ်ႈသေ `!!`

တႅၼ်းတၢင်းတေၸႂ်ႉ `!!`၊ ႁဝ်းထုၵ်ႇလီၸႂ်ႉလၢႆးပႃႈတႂ်ႈၼႆႉ တေလီသေယဝ်ႉ:

* **Safe Call:** `name?.length`
* **Elvis Operator:** `name?.length ?: 0`
* **If check:** `if (name != null) { ... }`

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Assertion:** ၵၢၼ်မၼ်ႈၸႂ် ဢမ်ႇၼၼ် ၵၢၼ်လၢတ်ႈယိုၼ်းယၼ်ၸႅင်ႈလႅင်း။
* **NullPointerException (NPE):** ပဵၼ် Error ဢၼ်ပဵၼ်မႃး မိူဝ်ႈႁဝ်းၶတ်းၸႂ်ၸႂ်ႉ Property ၶွင် null။
* **Crash:** ၵၢၼ်ဢၼ် Software ႁၵ်း/တၢႆ သေ ၸႂ်ႉၵၢၼ်ဢမ်ႇလႆႈထႅင်ႈ။
