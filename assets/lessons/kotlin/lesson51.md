# တွၼ်ႈသွၼ် 51: Visibility Modifiers (ၵၢၼ်ၵုမ်းထိင်းၵၢၼ်ၶဝ်ႈၸႂ်ႉ)

ၼႂ်း Kotlin မီး Modifiers ယူႇ 4 တူဝ်လူင်။ သင်ႁဝ်းဢမ်ႇသႂ်ႇသင်၊ တႄႇငဝ်ႈမၼ်း (Default) တေပဵၼ် **`public`** တႃႇသေႇယဝ်ႉ။

### 1. `public` (ပိုတ်ႇႁႃၵူႈၵေႃႉ)

* **ပွင်ႇဝႃႈ:** ၶဝ်ႈၸႂ်ႉလႆႈၵူႈတီႈ၊ တင်းၼႂ်း Class လဵဝ်ၵၼ် လႄႈ ၼွၵ်ႈ Class/Package။
* **Default:** သင်ဢမ်ႇတႅမ်ႈသင်၊ မၼ်းပဵၼ် public ယဝ်ႉ။

### 2. `private` (လပ်ႉသုတ်း)

* **ပွင်ႇဝႃႈ:** ၶဝ်ႈၸႂ်ႉလႆႈၵူၺ်းၼႂ်း **Scope (ပိၵ်ႇၵႃ `{ }`)** ဢၼ်သၢင်ႈမၼ်းဝႆႉၼၼ်ႉၵူၺ်း။
* **ၵၢၼ်ၸႂ်ႉတိုဝ်း:** ၸႂ်ႉတႃႇသိမ်း Variables ဢမ်ႇၼၼ် Function ဢၼ်ႁဝ်းဢမ်ႇၶႂ်ႈႁႂ်ႈပိူၼ်ႈမႃးမႄးလႆႈ (Encapsulation)။

### 3. `protected` (ပၼ်လုၵ်ႈလၢၼ်ၵူၺ်း)

* **ပွင်ႇဝႃႈ:** မိူၼ်တင်း `private` ၵူၺ်းၵႃႈ ထႅမ်မႃးထႅင်ႈပိူင်ၼိုင်ႈပဵၼ် **Subclasses (Class တူဝ်လုၵ်ႈ)** ၸၢင်ႈၶဝ်ႈၸႂ်ႉလႆႈ။
* **ၶေႃႈတွၼ်းသတိ:** `protected` ၼႂ်း Kotlin တေၸႂ်ႉ **ဢမ်ႇလႆႈ** တီႈ Top-level (ၼွၵ်ႈ Class)။

### 4. `internal` (ၼႂ်း Module လဵဝ်ၵၼ်)

* **ပွင်ႇဝႃႈ:** ၶဝ်ႈၸႂ်ႉလႆႈၵူႈတီႈ ဢၼ်ယူႇၼႂ်း **Module (Project)** လဵဝ်ၵၼ်။ သင်ပဵၼ်ၵူၼ်းၼွၵ်ႈ ဢၼ် Import Library ႁဝ်းၵႂႃႇၸႂ်ႉ၊ မၼ်းတေဢမ်ႇႁၼ် `internal` ႁဝ်းယဝ်ႉ။
* **ၵၢၼ်ၸႂ်ႉတိုဝ်း:** ၸွႆႈလႆႈၼမ် မိူဝ်ႈတႅမ်ႈ Library တႃႇသိမ်း Logic ၼႂ်းဢၼ်ဢမ်ႇၶႂ်ႈပိုတ်ႇပဵၼ် Public API။

---

### တွၼ်ႈတႅၵ်ႈ Visibility (Comparison Table)

| Modifier | ၼႂ်း Class လဵဝ်ၵၼ် | ၼႂ်း Subclass (တူဝ်လုၵ်ႈ) | ၼႂ်း Module လဵဝ်ၵၼ် | ၼွၵ်ႈ Module |
| --- | --- | --- | --- | --- |
| **public** | ✅ | ✅ | ✅ | ✅ |
| **internal** | ✅ | ✅ | ✅ | ❌ |
| **protected** | ✅ | ✅ | ❌ | ❌ |
| **private** | ✅ | ❌ | ❌ | ❌ |

### 5. Visibility ၼႂ်း Constructor

ႁဝ်းၸၢင်ႈသႂ်ႇ Modifier ပၼ် Constructor ၵေႃႈလႆႈ (တေလႆႈသႂ်ႇ Keyword `constructor` ၸွမ်းတႃႇသေႇ)။

```kotlin
// ပိတ်းဝႆႉ ဢမ်ႇပၼ်ပိူၼ်ႈသၢင်ႈ Object လႆႈတီႈၼွၵ်ႈ
class Secret private constructor(val code: String) {
    companion object {
        fun create() = Secret("1234")
    }
}

```

---

### 6. ၵွပ်ႈသင်ၸင်ႇလီ?

* **Security:** ႁႄႉၵင်ႈဢမ်ႇႁႂ်ႈၵူၼ်းတၢင်ႇၵေႃႉ (ဢမ်ႇၼၼ် ၶူတ်ႉတွၼ်ႈတၢင်ႇဢၼ်) မႃးမႄးၶေႃႈမုၼ်းလမ်ႇလွင်ႈႁဝ်း။
* **Clarity:** ၸွႆႈႁႂ်ႈ Programmer ၵေႃႉတၢင်ႇႁူႉဝႃႈ Function လႂ်ပဵၼ်ဢၼ်ထုၵ်ႇလီႁွင်ႉၸႂ်ႉ (API) လႄႈ Function လႂ်ပဵၼ်တွၼ်ႈၼႂ်း (Internal)။
* **Maintenance:** မိူဝ်ႈႁဝ်းမႄးၶူတ်ႉတွၼ်ႈ `private`၊ ႁဝ်းဢမ်ႇလူဝ်ႇမႆႈၸႂ်ဝႃႈ မၼ်းတေၵႂႃႇယႃႉၶူတ်ႉတီႈၼွၵ်ႈ ၵွပ်ႈဢမ်ႇမီးၽႂ်ႁွင်ႉၸႂ်ႉမၼ်းလႆႈယဝ်ႉ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Visibility Modifier:** Keyword ဢၼ်မၵ်းမၼ်ႈၶွပ်ႇပိူင်ၵၢၼ်ၶဝ်ႈၸႂ်ႉ။
* **Encapsulation:** ၵၢၼ်ႁႄႉၵင်ႈၶေႃႈမုၼ်း ႁႂ်ႈယူႇပဵၼ်တီႈပဵၼ်တၢင်း။
* **Module:** ၸုမ်းၶူတ်ႉ (Project) ဢၼ် Compile ဢွၵ်ႇမႃးၸွမ်းၵၼ် (မိူၼ်ၼင်ႇ IntelliJ Module ဢမ်ႇၼၼ် Gradle Source Set)။
* **Top-level:** သိင်ႇဢၼ်တႅမ်ႈဝႆႉၼွၵ်ႈ Class (ယူႇၼႂ်း File ၵမ်းလဵဝ်)။
