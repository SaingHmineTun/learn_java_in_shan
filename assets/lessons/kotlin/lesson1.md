# တွၼ်ႈသွၼ် 1: တႄႇႁူႉၸၵ်း Kotlin လႄႈ လွၵ်းလၢႆးႁဵတ်းၵၢၼ် JVM

**(Introduction to Kotlin & JVM Architecture)**

### Kotlin ဢၼ်ဝႃႈၼၼ်ႉ ပဵၼ်သင်?

**Kotlin** ၼႆႉ ပဵၼ်ၽႃႇသႃႇတႅမ်ႈၶူတ်ႉ (Programming Language) ဢၼ်မႂ်ႇ လႄႈ ၶိုတ်းၵၢပ်ႈဝႆႉယဝ်ႉ။ ၶမ်ႇပၼီႇ **JetBrains** လႆႈတႄႇသၢင်ႈမႃး မိူဝ်ႈပီ 2011။ မိူဝ်ႈလဵဝ်တႄႉ တၢင်း Google လႆႈပိုတ်ႇၽၢဝ်ႇဝႃႈ မၼ်းပဵၼ် ၽႃႇသႃႇပိူင်လူင် (Primary Language) တႃႇတေသၢင်ႈ Android App ယဝ်ႉ။

**ၵွပ်ႈသင်ႁဝ်းတေၸႂ်ႉ Kotlin?**

* **တႅမ်ႈငၢႆႈ (Concise):** ဢမ်ႇလူဝ်ႇတႅမ်ႈလိၵ်ႈၼမ်ၼမ် ၵေႃႈ ႁဵတ်းၵၢၼ်လႆႈၼမ်။
* **ႁူမ်ႇလူမ်ႈ (Safe):** ၵႄႉၶႆးပၼ်ႁႃ ၶူတ်ႉၵၢင်ႉ (Null Pointer Exceptions) ဢၼ်ၵႆႉပဵၼ်ၼႂ်း Java။
* **ၸႂ်ႉႁူမ်ႈၵၼ်လႆႈ (Interoperable):** ၸႂ်ႉၵိုၵ်းၵၼ်တင်း Java ၼႂ်း Project လဵဝ်ၵၼ်လႆႈ 100%။

---

### လွၵ်းလၢႆးႁဵတ်းၵၢၼ် JVM (JVM Architecture)

Kotlin ၼႆႉ ပဵၼ်ၽႃႇသႃႇဢၼ်လူဝ်ႇလႅၵ်ႈလၢႆႈ (Compiled Language) ယဝ်ႉ။ ၶူတ်ႉဢၼ်ႁဝ်းတႅမ်ႈၼၼ်ႉ ၶွမ်ႇပိဝ်ႇတႃႇဢမ်ႇပွင်ႇၸႂ်ၵမ်းလဵဝ်။ မၼ်းလူဝ်ႇလတ်းၽၢၼ်ႇ လၢႆးၵၢၼ်ၼင်ႇပႃႈတႂ်ႈၼႆႉယဝ်ႉ။

1. **Source Code (.kt):** ၶူတ်ႉဢၼ်ႁဝ်းတႅမ်ႈ လူၺ်ႈလိၵ်ႈဢင်းၵိတ်ႉ။
2. **Kotlin Compiler:** တူဝ်လႅၵ်ႈလၢႆႈ ဢၼ်တေလႅၵ်ႈၶူတ်ႉႁဝ်း ႁႂ်ႈပဵၼ် **Bytecode**။
3. **Bytecode (.class):** ၶေႃႈမုၼ်းဢၼ်လႅၵ်ႈဝႆႉယဝ်ႉ ဢၼ်တေဢဝ်ၵႂႃႇႁဵတ်းၵၢၼ်ၼိူဝ် JVM။
4. **JVM (Java Virtual Machine):** ပဵၼ်ၸၢၵ်ႈၼိမိတ်ႉ ဢၼ်မီးၼႃႈတီႈ ဢဝ် Bytecode ၼၼ်ႉ ၵႂႃႇႁဵတ်းႁႂ်ႈမၼ်းပဵၼ် Software ဢၼ်ၸႂ်ႉလႆႈ ၼိူဝ် Windows, Mac, လႄႈ Linux။

> **သပ်းလႅင်းထႅမ်:** ၵွပ်ႈပိူဝ်ႈ Kotlin ႁဵတ်းၵၢၼ်ၼိူဝ် JVM ၼၼ်ႉသေ မၼ်းၸင်ႇမီးလွင်ႈၼိမ် (Stable) လႄႈ ၸႂ်ႉလႆႈၵူႈတီႈ (Portable) ယဝ်ႉ။

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Compile:** လွၵ်းလၢႆးလႅၵ်ႈလၢႆႈၶူတ်ႉ ႁႂ်ႈပဵၼ်ၽႃႇသႃႇၸၢၵ်ႈ။
* **Statically Typed:** တေလႆႈသႂ်ႇပၼ် မဵဝ်းၶေႃႈမုၼ်း (Data Type) မၼ်း ၸႅင်ႈၸႅင်ႈလႅင်းလႅင်း။
* **Open Source:** ပဵၼ်ၽႃႇသႃႇဢၼ်ပိုတ်ႇဝႆႉလၢႆလၢႆ ၵူႈၵေႃႉၶဝ်ႈၸွႆႈထႅမ်လႆႈ။
