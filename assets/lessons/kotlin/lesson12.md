# တွၼ်ႈသွၼ် 12: ၶိူင်ႈမၢႆႁွင်ႉၸႂ်ႉႁူမ်ႇလူမ်ႈ (The Safe Call Operator - `?.`)

မိူဝ်ႈႁဝ်းမီး Variable ဢၼ်မီးၶိူင်ႈမၢႆ `?` (Nullable) ဝႆႉၼၼ်ႉ ႁဝ်းဢမ်ႇၸၢင်ႈၸႂ်ႉမၼ်းၵမ်းလဵဝ်လႆႈ။ ၵွပ်ႈၼၼ် Kotlin ၸင်ႇပၼ်ၶိူင်ႈမၢႆ **`?.`** မႃးၸွႆႈယဝ်ႉ။

### 1. `?.` ႁဵတ်းၵၢၼ်ၸိူဝ်ႉႁိူဝ်?

ၶိူင်ႈမၢႆ **`?.`** ၼႆႉ မၼ်းတေထတ်းတူၺ်း (Check) ဢွၼ်တၢင်းဝႃႈ Variable ၼၼ်ႉ ပဵၼ် `null` ႁႃႉ?

* **သင်ဢမ်ႇပဵၼ် null:** မၼ်းတေႁဵတ်းၵၢၼ်ၸွမ်းၼင်ႇႁဝ်းသင်ႇ။
* **သင်ပဵၼ် null:** မၼ်းတေဢမ်ႇႁဵတ်းသင်သေ ပွႆႇပဵၼ် `null` ဝႆႉ၊ ၵူၺ်းၵႃႈ Program ႁဝ်းတေဢမ်ႇလူႉ (No Crash)။

**တူဝ်ယၢင်ႇ:**

```kotlin
val name: String? = null

// println(name.length)  // ၼႆႉတေ Error (Compile Error)
println(name?.length)    // ၼႆႉႁဵတ်းလႆႈ! ၽူၼ်မၼ်းတေဢွၵ်ႇမႃးဝႃႈ: null

```

---

### 2. ၵၢၼ်ၸႂ်ႉ `?.` လၢႆထႅဝ် (Chaining Safe Calls)

ႁဝ်းၸၢင်ႈၸႂ်ႉ `?.` တေႃႇၵၼ်ၵႂႃႇလၢႆလၢႆဢၼ်လႆႈ။ သင်ဝႃႈတူဝ်လႂ်တူဝ်ၼိုင်ႈၼႂ်းၼၼ်ႉ ပဵၼ် `null` ၵႂႃႇၵေႃႈ Program တေဢမ်ႇလူႉ၊ မၼ်းတေပွႆႇၽူၼ်မၼ်းဢွၵ်ႇမႃးပဵၼ် `null` ၵမ်းလဵဝ်ယဝ်ႉ။

```kotlin
// တူဝ်ယၢင်ႇ: ႁႃၸိုဝ်ႈဝဵင်း ၵူၼ်းယူႇၼႂ်းႁိူၼ်း
// person?.house?.address?.city

```

သင်ဝႃႈ `person` ဢမ်ႇမီး၊ ဢမ်ႇၼၼ် `house` ဢမ်ႇမီး၊ မၼ်းတေဢမ်ႇႁဵတ်းၵၢၼ်ၵႂႃႇထႅင်ႈသေ ဢွၵ်ႇပဵၼ် `null` ဝႆႉယဝ်ႉ။

### 3. ၸႂ်ႉ `?.let` တႃႇႁဵတ်းၵၢၼ်မိူဝ်ႈဢမ်ႇပဵၼ် Null

မၢင်ပွၵ်ႈႁဝ်းၶႂ်ႈႁႂ်ႈမၼ်းႁဵတ်းၵၢၼ်သင်သင်ၼိုင်ႈ "မိူဝ်ႈမၼ်းမီးၶေႃႈမုၼ်းၵူၺ်း"၊ ႁဝ်းၸၢင်ႈၸႂ်ႉ `?.let` လႆႈၶႃႈ။

```kotlin
val name: String? = "Shan"

name?.let {
    // တွၼ်ႈၼႆႉတေႁဵတ်းၵၢၼ် မိူဝ်ႈ name ဢမ်ႇပဵၼ် null ၵူၺ်း
    println("ၸိုဝ်ႈၼႆႉမီး $it တူဝ်လိၵ်ႈယဝ်ႉ။")
}

```

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Safe Call:** ၵၢၼ်ႁွင်ႉၸႂ်ႉ Property ဢမ်ႇၼၼ် Function လူၺ်ႈၵၢၼ်ထတ်း null ၵမ်းလဵဝ်။
* **Chaining:** ၵၢၼ်ဢဝ်ၶေႃႈသင်ႇမႃးသိုပ်ႇၵၼ်ပဵၼ်သၢႆ။
* **Runtime Crash:** ပဵၼ်မိူဝ်ႈ Program ႁဝ်းၵၢင်ႉ/တၢႆ ၼႂ်းမိုဝ်းၵူၼ်းၸႂ်ႉ။
