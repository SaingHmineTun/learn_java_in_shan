# တွၼ်ႈသွၼ် 20: ႁႅင်းဢူၼ်ၶွင် 'when' (The Power of 'when')

`when` ၸၢင်ႈၸႂ်ႉလႆႈတင်းပဵၼ် **Statement** (သင်ႇႁဵတ်းၵၢၼ်) လႄႈ **Expression** (ပၼ် Value) မိူၼ်ၼင်ႇ `if` ယဝ်ႉ။

### 1. ၵၢၼ်ၸႂ်ႉ 'when' ပိူင်ငဝ်ႈ

ႁဝ်းၸၢင်ႈဢဝ် Variable သႂ်ႇဝႆႉၼႂ်းဝူင်းလႅမ် `( )` သေ ထတ်းတူၺ်းငဝ်းလၢႆးမၼ်းလၢႆလၢႆပိူင်။

```kotlin
val day = 3

when (day) {
    1 -> println("ဝၼ်းၸၼ်")
    2 -> println("ဝၼ်းဢင်းၵၢၼ်း")
    3 -> println("ဝၼ်းပုၼ်ႉ")
    else -> println("ဝၼ်းတၢင်ႇၸိူဝ်း")
}

```

### 2. ၵၢၼ်ၸူႉငဝ်းလၢႆး (Combining Conditions)

သင်ဝႃႈလၢႆလၢႆငဝ်းလၢႆး ပၼ်ၽူၼ်လႅတ်းမိူၼ်ၵၼ် ႁဝ်းၸၢင်ႈၸႂ်ႉၶိူင်ႈမၢႆ **`,` (Comma)** မႃးၸူႉၵၼ်လႆႈယဝ်ႉ။

```kotlin
val dayType = when (day) {
    1, 2, 3, 4, 5 -> "ဝၼ်းႁဵတ်းၵၢၼ်"
    6, 7 -> "ဝၼ်းၵိုတ်းလိုဝ်ႈ"
    else -> "ဢမ်ႇမီးၼႂ်းပၵ်းယဝ်ႉ"
}

```

---

### 3. Advanced Pattern Matching (ၵၢၼ်ထတ်းတူၺ်းပိူင်လၢႆးလႅတ်းၽႂ်း)

`when` ၼႂ်း Kotlin ၵတ်ႉၶႅၼ်ႇၼမ် ၵွပ်ႈမၼ်းထတ်းတူၺ်းလႆႈပႃး **Ranges** လႄႈ **Types** ယဝ်ႉ။

* **ထတ်းတူၺ်း Range (ၼႂ်းၵႄႈ):** ၸႂ်ႉ `in`
* **ထတ်းတူၺ်း Type (မဵဝ်းၶေႃႈမုၼ်း):** ၸႂ်ႉ `is` (လႄႈမီး Smart Cast ပႃးၵမ်းလဵဝ်)

```kotlin
val x: Any = 15

when (x) {
    in 1..10 -> println("ယူႇၼႂ်းၵႄႈ 1 တေႃႇ 10")
    is String -> println("ၼႆႉပဵၼ်လိၵ်ႈ ဢၼ်မီးတၢင်းယၢဝ်း ${x.length}") // Smart Cast ပဵၼ် String ယဝ်ႉ
    is Int -> println("ၼႆႉပဵၼ်တူဝ်ၼပ်ႉ: $x")
    else -> println("ဢမ်ႇႁူႉမဵဝ်းမၼ်းၶႃႈ")
}

```

---

### 4. ၵွပ်ႈသင်ထုၵ်ႇလီၸႂ်ႉ 'when' တႅၼ်း 'if-else'?

* **တူၺ်းငၢႆႈ (Readability):** ၶူတ်ႉဢမ်ႇယုင်ႈယၢင်ႇ မိူၼ် `if-else if` ၼမ်ၼမ်။
* **လႅတ်းၽႂ်း (Versatile):** ၸၢင်ႈထတ်းတူၺ်းလႆႈၵူႈယၢင်ႇ (Value, Range, Type)။
* **Safe:** သင်ၸႂ်ႉပဵၼ် Expression, Kotlin တေတဵၵ်းၸႂ်ႉႁႂ်ႈႁဝ်းသႂ်ႇ `else` (ဢမ်ႇၼၼ် ထတ်းႁႂ်ႈၵူမ်ၵူႈ) တႃႇႁႂ်ႈ Program ဢမ်ႇၽိတ်း။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Pattern Matching:** လွၵ်းလၢႆးၵၢၼ်ထတ်းတူၺ်းၶေႃႈမုၼ်း ၸွမ်းၼင်ႇပိူင်လၢႆး (Patterns) ဢၼ်မၼ်းၵိုၵ်းၵၼ်။
* **Exhaustive:** ၵၢၼ်ထတ်းတူၺ်းႁႂ်ႈၵူမ်ၵူႈ ၵူႈငဝ်းလၢႆး ဢၼ်ၸၢင်ႈပဵၼ်လႆႈ။
* **Branch:** ၵိင်ႇၵၢၼ်ႉၶွင်ၶူတ်ႉ (တွၼ်ႈၶူတ်ႉၼႂ်း `when` ၼိုင်ႈဢၼ် ႁွင်ႉဝႃႈ ၼိုင်ႈ Branch)။
