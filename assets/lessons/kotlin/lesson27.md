# တွၼ်ႈသွၼ် 27: ၶိူင်ႈမၢႆပၢႆးဝူၼ်ႉ လႄႈ ၵၢၼ်ၼပ်ႉသၢင်ႈလႅတ်းၽႂ်း

### 1. Logical Operators (ၶိူင်ႈမၢႆပၢႆးဝူၼ်ႉ)

မီးယူႇ 3 ပိူင်လူင် ဢၼ်ႁဝ်းၸႂ်ႉတႃႇႁူမ်ႈငဝ်းလၢႆး `true` / `false` ယဝ်ႉ။

* **`&&` (AND):** တေပဵၼ် `true` မိူဝ်ႈ **တင်းသွၼ်ၽၢႆႇ** ပဵၼ် `true` မိူၼ်ၵၼ်ၵူၺ်း။
* **`||` (OR):** တေပဵၼ် `true` မိူဝ်ႈ **မီးၽၢႆႇလႂ်ၽၢႆႇၼိုင်ႈ** ပဵၼ် `true` ၵေႃႈလႆႈယဝ်ႉ။
* **`!` (NOT):** ဝႆႉလႅၵ်ႈလၢႆႈၽၢႆႇ (Invert)။ သင်ပဵၼ် `true` တေပဵၼ် `false`၊ သင်ပဵၼ် `false` တေပဵၼ် `true`။

```kotlin
val hasKip = true
val hasBaht = false

if (hasKip && hasBaht) {
    println("သိုဝ်ႉလႆႈၵူႈယၢင်ႇ")
}

if (hasKip || hasBaht) {
    println("သိုဝ်ႉလႆႈယူႇ")
}

```

### 2. Short-circuit Evaluation (ၵၢၼ်တတ်းတွၼ်ႈၼပ်ႉသၢင်ႈ)

ၼႆႉပဵၼ်လွၵ်းလၢႆးဢၼ် Kotlin (လႄႈ ၽႃႇသႃႇၼမ်ၼမ်) ၸႂ်ႉတႃႇယႃႉၶၢဝ်းယၢမ်း။ မၼ်းတေဢမ်ႇထတ်းတူၺ်းငဝ်းလၢႆးတင်းၼမ် သင်မၼ်းႁူႉၽူၼ်လႅတ်းယဝ်ႉ။

#### ၵ) တႃႇ `&&` (AND):

သင်ၽၢႆႇသၢႆႉပဵၼ် **`false`** ယဝ်ႉ၊ မၼ်းတေ **ဢမ်ႇထတ်း** ၽၢႆႇၶႂႃထႅင်ႈၵမ်းလဵဝ်။ ၵွပ်ႈဝႃႈ `false && (သင်ၵေႃႈယဝ်ႉ)` ၽူၼ်မၼ်းတေပဵၼ် `false` တႃႇသေႇ။

#### ၶ) တႃႇ `||` (OR):

သင်ၽၢႆႇသၢႆႉပဵၼ် **`true`** ယဝ်ႉ၊ မၼ်းတေ **ဢမ်ႇထတ်း** ၽၢႆႇၶႂႃထႅင်ႈ။ ၵွပ်ႈဝႃႈ `true || (သင်ၵေႃႈယဝ်ႉ)` ၽူၼ်မၼ်းတေပဵၼ် `true` တႃႇသေႇ။

### 3. တူဝ်ယၢင်ႇဢၼ်ၸႂ်ႉလႆႈတႄႉ (Null Safety)

ႁဝ်းၵႆႉၸႂ်ႉ Short-circuit တႃႇႁႄႉၵင်ႈ Null Pointer Error ယဝ်ႉ။

```kotlin
val name: String? = "Sai"

// သင် name ပဵၼ် null၊ မၼ်းတေဢမ်ႇၵႂႃႇထတ်း name.length လေႃႇ
if (name != null && name.length > 2) {
    println("ၸိုဝ်ႈတႅတ်ႈတေႃးယဝ်ႉ")
}

```

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Logical Operator:** ၶိူင်ႈမၢႆဢၼ်ၸႂ်ႉတႃႇႁူမ်ႈ Boolean Value (True/False)။
* **Short-circuit Evaluation:** ၵၢၼ်တတ်းတွၼ်ႈၼပ်ႉသၢင်ႈ မိူဝ်ႈႁူႉၽူၼ်လႅတ်းယဝ်ႉ တႃႇႁႂ်ႈ Program လႅတ်းၽႂ်း။
* **Operand:** ၶေႃႈမုၼ်း ဢၼ်ယူႇၽၢႆႇသၢႆႉ လႄႈ ၽၢႆႇၶႂႃ ၶွင်ၶိူင်ႈမၢႆ (Operators)။
