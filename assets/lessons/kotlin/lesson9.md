# တွၼ်ႈသွၼ် 9: ပိူင်တႅမ်ႈၶူတ်ႉ လႄႈ လွၵ်းလၢႆးႁၢင်ႈလီ (Kotlin Coding Conventions)

**Coding Conventions** ၼႆႉ မိူၼ်ၼင်ႇ "ပၵ်းပိူင်" ဢၼ်ၵူၼ်းတႅမ်ႈ Kotlin တင်းလုမ်ႈၾႃႉ တူၵ်းလူင်းၵၼ်ဝႆႉ တႃႇႁႂ်ႈၶူတ်ႉၵူႈၵေႃႉ မိူၼ်ၵၼ်ယဝ်ႉ။

### 1. လွၵ်းလၢႆးတင်ႈၸိုဝ်ႈ (Naming Conventions)

* **ၸိုဝ်ႈ Class:** တေလႆႈတႄႇလူၺ်ႈ **တူဝ်လိၵ်ႈလူင်** သေ တႅမ်ႈၵိုၵ်းၵၼ် (PascalCase)။
* *တူဝ်ယၢင်ႇ:* `MyFirstClass`, `UserData`.


* **ၸိုဝ်ႈ Function လႄႈ Variable:** တေလႆႈတႄႇလူၺ်ႈ **တူဝ်လိၵ်ႈဢွၼ်ႇ** သေ တူဝ်လိၵ်ႈထႅပ်ႈၼၼ်ႉ ႁႂ်ႈပဵၼ်တူဝ်လူင် (camelCase)။
* *တူဝ်ယၢင်ႇ:* `calculateTotal()`, `userName`, `isLoggedIn`.



---

### 2. ၵၢၼ်ၸႂ်ႉလွၵ်းလၢႆႇ (Formatting)

* **Indentation (ၵၢၼ်ၶยိၼ်ႉထႅဝ်):** ၵႆႉၸႂ်ႉလွၵ်းလၢႆႇ (Space) 4 လွၵ်း တႃႇၼိုင်ႈထႅဝ်။ (IntelliJ IDEA တေၸွႆႈၸတ်းပၼ်ႁင်းၵူၺ်း မိူဝ်ႈၼဵၵ်း Tab)။
* **ပိၵ်ႇၵႃ `{ }`:** တူဝ်ပိၵ်ႇၵႃပိုတ်ႇ `{` တေလႆႈဝႆႉတီႈသုတ်းထႅဝ် ဢၼ်ပဵၼ်ၶူတ်ႉၼၼ်ႉ၊ ဢမ်ႇလူဝ်ႇလူင်းထႅဝ်မႂ်ႇ။

```kotlin
// လၢႆးတႅမ်ႈဢၼ်ထုၵ်ႇတေႃး (Correct)
fun sayHello() {
    println("Hello")
}

// လၢႆးတႅမ်ႈဢၼ်ဢမ်ႇပဵၼ်ထၢင်ႇ (Avoid)
fun sayHello() 
{
    println("Hello")
}

```

### 3. ၵၢၼ်တႅမ်ႈၶေႃႈသပ်းလႅင်း (Commenting)

မၢင်ပွၵ်ႈ ႁဝ်းလူဝ်ႇတႅမ်ႈလိၵ်ႈဝႆႉ တႃႇသပ်းလႅင်းၼႄဝႃႈ ၶူတ်ႉတွၼ်ႈၼႆႉ ႁဵတ်းၵၢၼ်သင်။

* **Single-line comment:** ၸႂ်ႉ `//` တႃႇသပ်းလႅင်းၼိုင်ႈထႅဝ်။
* **Multi-line comment:** ၸႂ်ႉ `/* ... */` တႃႇသပ်းလႅင်းလၢႆထႅဝ်။

```kotlin
// ၼႆႉပဵၼ်ၵၢၼ်ၼပ်ႉဢႃႇယု
val age = 20 + 5 

/* ၶူတ်ႉတွၼ်ႈၼႆႉ 
   တေၸွႆႈၼပ်ႉသၢင်ႈ 
   ၶႅၼ်းၵိမ်းတင်းหมด
*/

```

### 4. လိူၵ်ႈၸႂ်ႉ val တႅၼ်း var

ပၵ်းပိူင် Kotlin လၢတ်ႈဝႃႈ: **"သင်ဢမ်ႇလူဝ်ႇလႅၵ်ႈလၢႆႈၶေႃႈမုၼ်းတႄႉတႄႉၼႆ ႁႂ်ႈၸႂ်ႉ `val` တႃႇသေႇ"**။ မၼ်းတေႁဵတ်းႁႂ်ႈၶူတ်ႉႁဝ်း ၼိမ်လႄႈ ႁူမ်ႇလူမ်ႈယဝ်ႉ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Convention:** ပၵ်းပိူင် ဢမ်ႇၼၼ် လွၵ်းလၢႆးဢၼ်ၵူၼ်းၼမ်ၸႂ်ႉတိုဝ်း။
* **CamelCase:** လၢႆးတင်ႈၸိုဝ်ႈ ဢၼ်တူဝ်ၼႃႈပဵၼ်တူဝ်ဢွၼ်ႇ တူဝ်လႅၵ်ႈတေႃႇၵႂႃႇပဵၼ်တူဝ်လူင် (မိူၼ်ၼင်ႇ ႁူဝ်ဢူဌ်)။
* **Indentation:** ၵၢၼ်ဝႆႉလွၵ်းလၢႆႇ ၽၢႆႇၼႃႈထႅဝ်ၶူတ်ႉ ႁႂ်ႈတူၺ်းငၢႆႈ။
