# တွၼ်ႈသွၼ် 2: ၵၢၼ်ၵေႃႇတင်ႈၶိူင်ႈမိုဝ်း (Environment Setup)

**(IntelliJ IDEA & Kotlin SDK)**

တႃႇတေတႅမ်ႈ Kotlin ၼၼ်ႉ ၶိူင်ႈမိုဝ်းဢၼ်လီသုတ်းလႄႈ ၵူၼ်းၸႂ်ႉၼမ်သုတ်းတႄႉ ပဵၼ် **IntelliJ IDEA** ယဝ်ႉ။ မၼ်းပဵၼ်ၵၢၼ်သၢင်ႈၶမ်ႇပၼီႇ JetBrains (တူဝ်ဢၼ်သၢင်ႈ Kotlin) ၼၼ်ႉယဝ်ႉ။

### 1. လူတ်ႇလူင်းလႄႈ တပ်ႉၵေႃႇ IntelliJ IDEA

1. ၵႂႃႇတီႈဝႅပ်ႉသၢႆႉ [JetBrains Website](https://www.jetbrains.com/idea/download/).
2. လိူၵ်ႈဢဝ်ပိူင် **Community Edition** (မၼ်းပဵၼ် Free Version ဢၼ်ႁဝ်းၸႂ်ႉလႆႈလၢႆလၢႆ)။
3. လူတ်ႇလူင်း (Download) သေ တပ်ႉၵေႃႇ (Install) ၸွမ်းၼင်ႇၶွမ်ႇပိဝ်ႇတႃႇတူၵ်းယိူၼ်းၼၼ်ႉၶႃႈ။

### 2. လူတ်ႇလူင်း Kotlin SDK (Software Development Kit)

မိူဝ်ႈႁဝ်းတပ်ႉၵေႃႇ IntelliJ IDEA ယဝ်ႉၼၼ်ႉ မၼ်းတေပႃး Kotlin လႄႈ Java SDK မႃးၸွမ်းၵမ်းလဵဝ်ယဝ်ႉ။ သင်ဝႃႈပႆႇမီးၵေႃႈ ၼႂ်း IntelliJ တေၸိၼႄႁႂ်ႈႁဝ်း Download ဢဝ်ငၢႆႈငၢႆႈယဝ်ႉ။

* **SDK ဢၼ်ဝႃႈၼၼ်ႉ:** ပဵၼ်ၸုမ်ႇၶိူင်ႈမိုဝ်း ဢၼ်တေၸွႆႈႁႂ်ႈၶွမ်ႇပိဝ်ႇတႃႇႁဝ်း ပွင်ႇၸႂ်လႄႈ ႁဵတ်းၵၢၼ်ၸွမ်းၶူတ်ႉ Kotlin လႆႈၼၼ်ႉယဝ်ႉ။

---

### 3. တႄႇသၢင်ႈ Project မႂ်ႇ (Create New Project)

မိူဝ်ႈပိုတ်ႇ IntelliJ IDEA မႃးယဝ်ႉၼၼ်ႉ ႁဵတ်းၸွမ်းၼင်ႇပႃႈတႂ်ႈၼႆႉၶႃႈ:

1. ၼဵၵ်းတီႈ **New Project**.
2. လိူၵ်ႈတီႈ **Kotlin** (ယူႇၽၢႆႇသၢႆႉ)။
3. သႂ်ႇၸိုဝ်ႈ Project (တူဝ်ယၢင်ႇ: `HelloWorld`)။
4. တီႈ **Build System** ၼၼ်ႉ လိူၵ်ႈဢဝ် **IntelliJ** (တႃႇတႄႇႁဵၼ်းမၼ်းတေငၢႆႈသုတ်း)။
5. ၼဵၵ်း **Create**.

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **IDE (Integrated Development Environment):** ပဵၼ် Software ဢၼ်ႁဝ်းၸႂ်ႉတႅမ်ႈၶူတ်ႉ (မိူၼ်ၼင်ႇ IntelliJ IDEA)။
* **SDK (Software Development Kit):** ၶိူင်ႈမိုဝ်းပိုင်ႈဢိင် တႃႇသၢင်ႈ Software။
* **Community Edition:** ပဵၼ် Version ဢၼ်ပၼ်ၸႂ်ႉလၢႆလၢႆ တႃႇၵူၼ်းတႅမ်ႈၶူတ်ႉၵူႈၵေႃႉ။
