# တွၼ်ႈသွၼ် 87: Introduction to Coroutines

### 1. ပၼ်ႁႃ: Synchronous Programming (ၵၢၼ်ႁဵတ်းၵၢၼ်ၸွမ်းလမ်ႇတပ်ႉ)

ၸွမ်းပိူင်ပိုၼ်ႉထၢၼ်၊ ၶူတ်ႉတေႁဵတ်းၵၢၼ်ထႅဝ်ၼိုင်ႈယဝ်ႉ ၸင်ႇၵႂႃႇထႅဝ်ၼိုင်ႈ။ သင်ထႅဝ်ၼိုင်ႈလူဝ်ႇ "ပႂ်ႉ" ၶေႃႈမုၼ်းတေႃႇ Internet ယၢဝ်းၼႃႇ၊ App တေ "ၶၢင်ႉ" (Freeze) သေ ၵူၼ်းၸႂ်ႉတေၵူတ်ႉသင်ၵေႃႈဢမ်ႇလႆႈယဝ်ႉ။

### 2. လၢႆးၵႄႈ: Asynchronous Programming

ပွင်ႇဝႃႈ "ၵၢၼ်ႁဵတ်းၵၢၼ်လၢႆယၢင်ႇႁူမ်ႈၵၼ် လူၺ်ႈဢမ်ႇလူဝ်ႇပႂ်ႉၵၼ်"။ မိူၼ်တင်းၸဝ်ႈၵဝ်ႇ "ႁုင်ၶဝ်ႈ" ဝႆႉယဝ်ႉ၊ ၼႂ်းၼၼ်ႉၸဝ်ႈၵဝ်ႇၵေႃႈ "ၵႂႃႇသုၵ်ႉဝၼ်" ပႂ်ႉ—ဢမ်ႇလူဝ်ႇၸုၼ်ႉပႂ်ႉတေႃႇၶဝ်ႈပေႃးသုၵ်း ၸင်ႇၵႂႃႇသုၵ်ႉဝၼ်ၼၼ်ႉယဝ်ႉ။

### 3. Coroutines ၵေႃႈပဵၼ်သင်?

**Coroutines** ပဵၼ် "Lightweight Threads" (Thread ဢၼ်မၢင်လႅတ်း)။ မၼ်းၸွႆႈႁႂ်ႈႁဝ်းတႅမ်ႈၶူတ်ႉဢၼ်ႁဵတ်းၵၢၼ်ယၢပ်ႇယၢပ်ႇ (Asynchronous) ႁႂ်ႈတူၺ်းငၢႆႈမိူၼ်ၶူတ်ႉပၵ်းပၵ်း (Synchronous) ယဝ်ႉ။

---

### 4. ၵွပ်ႈသင်ၸင်ႇၸႂ်ႉ Coroutines တႅၼ်း Threads?

* **Resource Efficient**: Thread ၼိုင်ႈတူဝ် ၵိၼ် Memory ၼမ် (မိူၼ်ၼင်ႇ 1MB)၊ သင်မီး Thread 1,000 တူဝ် App တေ Crash။ ၵူၺ်းၵႃႈ Coroutine ၼိုင်ႈတူဝ် ၵိၼ် Memory ဢေႇတႄႉတႄႉ၊ ၸဝ်ႈၵဝ်ႇသၢင်ႈမၼ်းပဵၼ် "သႅၼ်တူဝ်" ၵေႃႈ App တိုၵ်ႉဝႆးယူႇ။
* **Non-blocking**: မၼ်းၸၢင်ႈ "ၵတ်းယင်ႉ" (Suspend) ၵၢၼ်ႁဵတ်းၵၢၼ်ဝႆႉ လူၺ်ႈဢမ်ႇႁဵတ်းႁႂ်ႈ Thread တူဝ်ၼၼ်ႉၶၢင်ႉ။
* **Structured Concurrency**: ၸွႆႈၵုမ်းၵမ်လွင်ႈသၢႆၸႂ်ၶွင်ၵၢၼ်ႁဵတ်းၵၢၼ် ႁႂ်ႈမၼ်းပိတ်း (Cancel) ၵႂႃႇၸွမ်းၵၼ် မိူဝ်ႈဢမ်ႇလူဝ်ႇၸႂ်ႉယဝ်ႉ။

---

### 5. ၶေႃႈၵႂၢမ်းပိုၼ်ႉထၢၼ် (Basic Vocabulary)

* **Scope**: ဝူင်းသၢႆၸႂ်ၶွင် Coroutine (မိူၼ်ၼင်ႇ `GlobalScope` ဢမ်ႇၼၼ် `viewModelScope`)။
* **Job**: တူဝ်ၵုမ်းၵမ် Coroutine ဢၼ်ႁဝ်းၸၢင်ႈ "ပိတ်း (Cancel)" မၼ်းလႆႈ။
* **Dispatcher**: တူဝ်လိူၵ်ႈဝႃႈ တေႁႂ်ႈ Coroutine ၼႆႉ ၵႂႃႇႁဵတ်းၵၢၼ်တီႈ Thread လႂ် (မိူၼ်ၼင်ႇ Thread တႃႇ UI ဢမ်ႇၼၼ် Thread တႃႇ Internet)။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Asynchronous:** ၵၢၼ်ႁဵတ်းၵၢၼ်ဢၼ်ဢမ်ႇလူဝ်ႇပႂ်ႉၵၼ် (ႁဵတ်းလႆႈလၢႆယၢင်ႇႁူမ်ႈၵၼ်)။
* **Lightweight:** ဢၼ်မၢင်၊ ဢမ်ႇၵိၼ် Memory ၼမ်။
* **Blocking:** ၵၢၼ်ဢၼ် "ပိတ်းတၢင်း" ႁဵတ်းႁႂ်ႈၵၢၼ်တၢင်ႇဢၼ်ၵႂႃႇဢမ်ႇလႆႈ (မိူၼ်ၼင်ႇ Thread ၶၢင်ႉ)။
* **Suspending:** ၵၢၼ် "ၵတ်းယင်ႉ" ၸူဝ်ႈၶၢဝ်း (မိူၼ်ၼင်ႇ ပႂ်ႉၶေႃႈမုၼ်း) သေပၼ် Thread ၵႂႃႇႁဵတ်းၵၢၼ်တၢင်ႇဢၼ်လႆႈ။
