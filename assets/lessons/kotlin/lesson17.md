# တွၼ်ႈသွၼ် 17: မဵဝ်းၶေႃႈမုၼ်း Any လႄႈ Unit

### 1. `Any` Type (ပဵၼ်လႆႈၵူႈပိူင်)

ၼႂ်း Kotlin ၼၼ်ႉ **`Any`** ပဵၼ် "မႄႈ" ၶွင်ၵူႈ Type တင်းหมด (The root of the Kotlin class hierarchy)။ မၼ်းမိူၼ်တင်း `Object` ၼႂ်း Java ယဝ်ႉ။

* Variable ဢၼ်ပဵၼ် Type **`Any`** ၼၼ်ႉ ၸၢင်ႈႁပ်ႉသိမ်းၶေႃႈမုၼ်းလႆႈ **ၵူႈမဵဝ်း** (Strings, Numbers, Booleans)။

```kotlin
val x: Any = "Hello"
val y: Any = 100
val z: Any = true

```

> **ၶေႃႈတွၼ်းသတိ:** `Any` တေပႅၵ်ႇပၢင်ႇတင်း `Any?` ဢိတ်းၼိုင်ႈ။ `Any` တေလႆႈမီးၶေႃႈမုၼ်းတႃႇသေႇ၊ ၵူၺ်းၵႃႈ `Any?` တႄႉ ၸၢင်ႈပဵၼ်လႆႈတင်း ၶေႃႈမုၼ်း လႄႈ `null` ယဝ်ႉ။

---

### 2. `Unit` Type (ဢမ်ႇမီးၽူၼ်လႅတ်း)

**`Unit`** ၼႆႉ ၸၢင်ႈတႅၵ်ႈၼိူဝ်းလႆႈတင်း `void` ၼႂ်း Java သေတႃႉ မၼ်းမီးလွင်ႈပႅၵ်ႇပၢင်ႇၵၼ်ယူႇ။ မၼ်းဝႆႉၸႂ်ႉတႃႇလၢတ်ႈၼႄဝႃႈ Function ဢၼ်ၼႆႉ **"ဢမ်ႇပၼ်ၽူၼ်လႅတ်း (Return value) သင်မႃး"** ၼႆယဝ်ႉ။

* ၼႂ်း Kotlin၊ သင်ႁဝ်းတႅမ်ႈ Function ဢၼ်ဢမ်ႇမီး `return` ၼႆ မၼ်းတေပဵၼ် **`Unit`** ႁင်းၵူၺ်းမၼ်း။

```kotlin
fun sayHello(): Unit {
    println("မႂ်ႇသုင်ၶႃႈ")
    // ဢမ်ႇလူဝ်ႇတႅမ်ႈ return ၵွပ်ႈမၼ်းပဵၼ် Unit
}

// ႁဝ်းၵႆႉတႅမ်ႈၵေႃႈလႆႈ၊ ဢမ်ႇတႅမ်ႈၵေႃႈလႆႈ (Optional)
fun sayHi() {
    println("Hi!")
}

```

**ၵွပ်ႈသင်ႁွင်ႉဝႃႈ Unit?**
ၼႂ်းတၢင်းပၢႆးၼပ်ႉ (Mathematics)၊ **Unit** မီးတီႈပွင်ႇဝႃႈ "မီးဢၼ်လဵဝ်"။ ၼႂ်း Kotlin ၵေႃႈ `Unit` ပဵၼ် Object ဢၼ်မီးတူဝ်လဵဝ်ၵူၺ်း (Singleton instance)။

---

### 3. ပႅၵ်ႇပၢင်ႇၵၼ်ၸိူဝ်ႉႁိူဝ်?

| Feature | **Any** | **Unit** |
| --- | --- | --- |
| **တီႈပွင်ႇ** | ပဵၼ်လႆႈၵူႈ Type (Supertype) | တႅၼ်းတၢင်း "ဢမ်ႇမီး Return value" |
| **ၵၢၼ်ၸႂ်ႉတိုဝ်း** | ၸႂ်ႉမိူဝ်ႈဢမ်ႇႁူႉဝႃႈ ၶေႃႈမုၼ်းတေပဵၼ်သင် | ၸႂ်ႉပဵၼ် Type ၶွင် Function ဢၼ်သင်ႇႁဵတ်းၵၢၼ်ၵူၺ်း |
| **Value** | မီးလႆႈတင်းၼမ် (1, "A", true, ...) | မီးလႆႈ `Unit` တူဝ်လဵဝ်ၵူၺ်း |

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Hierarchy:** ပိူင်ထၼ်း (မိူၼ်ၼင်ႇ ၵိင်ႇၵၢၼ်ႉတူၼ်ႈမႅၵ်ႇ)။
* **Root:** ႁၢၵ်ႈ ဢမ်ႇၼၼ် တီႈတႄႇငဝ်ႈမၼ်း။
* **Singleton:** Object ဢၼ်မီးလႆႈတူဝ်လဵဝ်ၵူၺ်း ၼႂ်း Program တင်းหมด။
* **Void:** ၶေႃႈၵႂၢမ်းတၢင်ႇၽႃႇသႃႇ ဢၼ်မီးတီႈပွင်ႇဝႃႈ "လွၵ်းလၢႆႇ/ဢမ်ႇမီးသင်"။
