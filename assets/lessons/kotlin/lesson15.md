# တွၼ်ႈသွၼ် 15: ၵၢၼ်ထတ်းတူၺ်းမဵဝ်းၶေႃႈမုၼ်း (Smart Casts & The 'is' Check)

မိူဝ်ႈႁဝ်းမီး Variable ဢၼ်ႁဝ်းဢမ်ႇမၼ်ႈၸႂ်ဝႃႈ မၼ်းပဵၼ် String ႁႃႉ ဢမ်ႇၼၼ် ပဵၼ် Int ႁႃႉၼၼ်ႉ ႁဝ်းလူဝ်ႇလႆႈထတ်းတူၺ်းယဝ်ႉ။

### 1. ၶိူင်ႈမၢႆ `is` (The 'is' Check)

ၼႂ်း Kotlin ႁဝ်းၸႂ်ႉၶိူင်ႈမၢႆ **`is`** တႃႇထတ်းတူၺ်း မဵဝ်းၶေႃႈမုၼ်း (Type) ယဝ်ႉ။ မၼ်းမိူၼ်တင်းၵၢၼ်ထၢမ်ဝႃႈ "တူဝ်ၼႆႉ ပဵၼ်မဵဝ်းၼႆႉႁႃႉ?" ၼႆယဝ်ႉ။

```kotlin
val obj: Any = "Hello Shan State"

if (obj is String) {
    // ပႃႈၼႂ်းၼႆႉ Kotlin တေႁူႉၵမ်းလဵဝ်ဝႃႈ obj ပဵၼ် String
    println(obj.length) // ၼႆႉႁွင်ႉဝႃႈ Smart Cast
}

```

---

### 2. Smart Cast ဢၼ်ဝႃႈၼၼ်ႉ ပဵၼ်သင်?

ၼႂ်း Java ၼၼ်ႉ မိူဝ်ႈႁဝ်းထတ်းတူၺ်းယဝ်ႉ ႁဝ်းလူဝ်ႇလႆႈတႅမ်ႈလႅၵ်ႈ (Cast) မၼ်းထႅင်ႈၵမ်းၼိုင်ႈ။ ၵူၺ်းၵႃႈ **Kotlin Smart Cast** တႄႉ:

* မိူဝ်ႈႁဝ်းထတ်းတူၺ်းလူၺ်ႈ `is` ယဝ်ႉ။
* Kotlin Compiler တေလႅၵ်ႈမဵဝ်းၶေႃႈမုၼ်းပၼ် **ႁင်းၵူၺ်းမၼ်း (Automatically)** ၼႂ်းတွၼ်ႈၶူတ်ႉ (Block) ၼၼ်ႉယဝ်ႉ။
* ႁဝ်းၸၢင်ႈၸႂ်ႉ Function ၶွင် Type ၼၼ်ႉလႆႈၵမ်းလဵဝ်။

---

### 3. Smart Cast ၸႂ်ႉၵိုၵ်း Null Safety

လၢႆးၵၢၼ်ၼႆႉၵႆႉၸႂ်ႉတႃႇထတ်း `null` ၸိူဝ်ႉၼႆၶႃႈ:

```kotlin
val name: String? = "Sai Sai"

if (name != null) {
    // ဝၢႆးသေထတ်းဝႃႈ ဢမ်ႇပဵၼ် null ယဝ်ႉ
    // Kotlin တေ Smart Cast ပၼ် 'name' ႁႂ်ႈပဵၼ် Non-Nullable String
    println(name.length) // ဢမ်ႇလူဝ်ႇၸႂ်ႉ ?. ယဝ်ႉ
}

```

---

### 4. ၶိူင်ႈမၢႆ `!is` (Not Is)

သင်ႁဝ်းၶႂ်ႈထတ်းတူၺ်းဝႃႈ "ဢမ်ႇပဵၼ်" ႁဝ်းၸၢင်ႈၸႂ်ႉ **`!is`** လႆႈယဝ်ႉ။

```kotlin
if (obj !is String) {
    println("ၼႆႉဢမ်ႇၸႂ်ႈလိၵ်ႈၶႃႈ")
}

```

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Smart Cast:** ၵၢၼ်ဢၼ် Compiler လႅၵ်ႈမဵဝ်းၶေႃႈမုၼ်းပၼ်ႁင်းၵူၺ်း ဝၢႆးထတ်းတူၺ်းယဝ်ႉ။
* **Type Checking:** ၵၢၼ်ထတ်းတူၺ်းဝႃႈ Variable ၼၼ်ႉ ပဵၼ်မဵဝ်းၶေႃႈမုၼ်းသင်။
* **Any:** ပဵၼ်မဵဝ်းၶေႃႈမုၼ်းလူင် (Parent Type) ဢၼ်ႁပ်ႉလႆႈၶေႃႈမုၼ်းၵူႈပိူင်။
