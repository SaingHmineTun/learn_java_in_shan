# တွၼ်ႈသွၼ် 25: ၵၢၼ်ၵုမ်းထိင်း Loop (Break, Continue လႄႈ Labels)

### 1. `break` (ၵိုတ်းၵမ်းလဵဝ်)

ၸႂ်ႉမိူဝ်ႈႁဝ်းၶႂ်ႈႁႂ်ႈ Loop ၼၼ်ႉ **ၵိုတ်းၵမ်းလဵဝ်** သေ ဢွၵ်ႇၵႂႃႇႁဵတ်းၶူတ်ႉၽၢႆႇၼွၵ်ႈယဝ်ႉ။

```kotlin
for (i in 1..10) {
    if (i == 5) break // သင်ထိုင် 5 ႁႂ်ႈၵိုတ်းၵမ်းလဵဝ်
    println(i)
}
// ၽူၼ်မၼ်း: 1, 2, 3, 4 (ၵိုတ်းတီႈ 5)

```

### 2. `continue` (ဝႅၼ်ၶၢမ်ႈ)

ၸႂ်ႉမိူဝ်ႈႁဝ်းၶႂ်ႈ **ဝႅၼ်ၶၢမ်ႈ** တူဝ်ၼၼ်ႉသေ ၵႂႃႇတႄႇႁဵတ်းတူဝ်ၼႃႈတေႃႇၵႂႃႇ (ဢမ်ႇဢွၵ်ႇ Loop)။

```kotlin
for (i in 1..5) {
    if (i == 3) continue // သင်ပဵၼ် 3 ႁႂ်ႈဝႅၼ်ၶၢမ်ႈၵႂႃႇတူဝ် 4 ၵမ်းလဵဝ်
    println(i)
}
// ၽူၼ်မၼ်း: 1, 2, 4, 5 (ဢမ်ႇမီး 3)

```

---

### 3. Loop Labels (ၵၢၼ်ပၼ်ၸိုဝ်ႈ Loop)

မိူဝ်ႈႁဝ်းမီး Loop သွၼ်ႉၵၼ် (Nested Loops)၊ `break` ဢမ်ႇၼၼ် `continue` ယူႇယူႇၼၼ်ႉ မၼ်းတေႁဵတ်းၵၢၼ်ၵူၺ်းၼႂ်း **Loop ဢၼ်ယူႇၸိူၵ်ႈမၼ်းသုတ်း**။ သင်ႁဝ်းၶႂ်ႈၵုမ်းထိင်း Loop တူဝ်ၼွၵ်ႈ၊ ႁဝ်းလူဝ်ႇၸႂ်ႉ **Label** ယဝ်ႉ။

* **လၢႆးတႅမ်ႈ:** ၸႂ်ႉ `ၸိုဝ်ႈ@` ဝႆႉၽၢႆႇၼႃႈ Loop။

```kotlin
outerLoop@ for (i in 1..3) {
    for (j in 1..3) {
        if (i == 2 && j == 2) break@outerLoop // ႁႂ်ႈၵိုတ်းတင်း Loop ၼွၵ်ႈၵမ်းလဵဝ်
        println("i: $i, j: $j")
    }
}

```

---

### 4. ၵွပ်ႈသင် Label ၸင်ႇလမ်ႇလွင်ႈ?

* **ၵုမ်းထိင်းလႆႈတႅတ်ႈတေႃး:** ၸွႆႈႁႂ်ႈႁဝ်းသင်ႇၵိုတ်း Loop ဢၼ်ႁဝ်းၶႂ်ႈၵိုတ်းတႄႉတႄႉ (မိူၼ်ၼင်ႇ မိူဝ်ႈႁႃၶေႃႈမုၼ်းႁၼ်ယဝ်ႉ ၼႂ်း List သွၼ်ႉၵၼ်)။
* **ၶူတ်ႉတူၺ်းငၢႆႈ:** ႁဵတ်းႁႂ်ႈႁဝ်းဢမ်ႇလူဝ်ႇၸႂ်ႉ Variable ပိူင် Boolean (မိူၼ်ၼင်ႇ `found = true`) မႃးထတ်းတူၺ်းလၢႆလၢႆၸၼ်ႉ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Break:** ၵၢၼ်ဢွၵ်ႇတူင်ႇ (Loop) ၵမ်းလဵဝ်။
* **Continue:** ၵၢၼ်ဝႅၼ်ၶၢမ်ႈၵၢၼ်ႁဵတ်းၵၢၼ်တွၼ်ႈၼၼ်ႉ သေၵႂႃႇတႄႇတူဝ်ၼႃႈ။
* **Label:** ၵၢၼ်ပၼ်ၸိုဝ်ႈဝႆႉ တႃႇႁႂ်ႈႁဝ်းႁွင်ႉၸႂ်ႉလႆႈ (မိူၼ်ၼင်ႇ `loop@`)။
* **Nested Loop:** Loop ဢၼ်မီးဝႆႉၼႂ်း Loop ထႅင်ႈဢၼ်ၼိုင်ႈ။
