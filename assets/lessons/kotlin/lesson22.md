# တွၼ်ႈသွၼ် 22: ၶွပ်ႇတူဝ်ၼပ်ႉ လႄႈ ၵၢၼ်ၶยိၼ်ႉ (Ranges and Progressions)

**Range** ၼႆႉ ပဵၼ်ၵၢၼ်သၢင်ႈ "သၢႆတူဝ်ၼပ်ႉ" ဢၼ်မီးတီႈတႄႇ လႄႈ တီႈသုတ်းယဝ်ႉ။

### 1. ၶိူင်ႈမၢႆ `..` (Range To)

ၸႂ်ႉတႃႇသၢင်ႈတူဝ်ၼပ်ႉ တႄႇတီႈဢွၼ်ႇ တေႃႇ တီႈယႂ်ႇ (မၼ်းတေဢဝ်ပႃး တူဝ်သုတ်းသုတ်းမၼ်း)။

```kotlin
val myRange = 1..5  // မီး 1, 2, 3, 4, 5

```

### 2. `until` (ဢမ်ႇဢဝ်တူဝ်သုတ်း)

မိူၼ်တင်း `..` ၵူၺ်းၵႃႈ မၼ်းတေ **ဢမ်ႇဢဝ်** တူဝ်သုတ်းသုတ်းမၼ်းပႃး။

```kotlin
val myRange = 1 until 5 // မီး 1, 2, 3, 4 (ဢမ်ႇဢဝ် 5)

```

### 3. `downTo` (ၼပ်ႉထွႆလင်)

သင်ႁဝ်းၶႂ်ႈၼပ်ႉ တႄႇတီႈယႂ်ႇ လွင်ႈမႃး တီႈဢွၼ်ႇ။

```kotlin
val reverseRange = 5 downTo 1 // မီး 5, 4, 3, 2, 1

```

---

### 4. `step` (ၵၢၼ်ဝႅၼ်ၶၢမ်ႈ)

ပၵ်းပိူင် (Default) မၼ်းတေတိုးၵႂႃႇၵမ်း 1 တူဝ်။ သင်ႁဝ်းၶႂ်ႈႁႂ်ႈမၼ်းဝႅၼ်ၶၢမ်ႈ ၵမ်း 2 တူဝ် ဢမ်ႇၼၼ် ၼမ်သေၼၼ်ႉ ႁဝ်းၸၢင်ႈၸႂ်ႉ `step` ယဝ်ႉ။

```kotlin
val oddNumbers = 1..10 step 2 // မီး 1, 3, 5, 7, 9

```

---

### 5. ၵၢၼ်ထတ်းတူၺ်းၶေႃႈမုၼ်း (in check)

ႁဝ်းၸၢင်ႈၸႂ်ႉၶွပ်ႇတူဝ်ၼပ်ႉ ၵိုၵ်းၵၼ်တင်း `if` ဢမ်ႇၼၼ် `when` တႃႇထတ်းတူၺ်းဝႃႈ တူဝ်ၼပ်ႉၼၼ်ႉ ယူႇၼႂ်းၶွပ်ႇႁဝ်းႁႃႉ။

```kotlin
val age = 20
if (age in 13..19) {
    println("ပဵၼ်လုၵ်ႈဢွၼ်ႇသႅၼ်းပၢၼ် (Teenager)")
} else {
    println("ဢမ်ႇၸႂ်ႈသႅၼ်းပၢၼ်ယဝ်ႉ")
}

```

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Range:** ၶွပ်ႇတူဝ်ၼပ်ႉ ဢၼ်မီးတီႈတႄႇ လႄႈ တီႈသုတ်း။
* **Progression:** သၢႆတူဝ်ၼပ်ႉ ဢၼ်မီးပၵ်းပိူင်ၵၢၼ်တိုးၶိုၼ်း (မိူၼ်ၼင်ႇ Step)။
* **Inclusive:** ၵၢၼ်ဢဝ်ပႃး တူဝ်သုတ်းသုတ်း (မိူၼ်ၼင်ႇ `..`)။
* **Exclusive:** ၵၢၼ်ဢမ်ႇဢဝ်ပႃး တူဝ်သုတ်းသုတ်း (မိူၼ်ၼင်ႇ `until`)။
