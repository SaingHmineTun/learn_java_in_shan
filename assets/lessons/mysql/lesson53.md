# Lesson 53: "Stored Procedures: Reusable Code with Parameters"

**Stored Procedure** မၢႆထိုင် ၸုမ်းၶေႃႈၵႂၢမ်း SQL ဢၼ်ႁဝ်းတႅမ်ႈဝႆႉယဝ်ႉ သေသိမ်းဝႆႉၼႂ်း Database Server ၼႆၶႃႈ။ 
မိူဝ်ႈႁဝ်းၶႂ်ႈၸႂ်ႉ... ႁဝ်းၵူၺ်းလူဝ်ႇ "ႁွင်ႉ" (Call) ၸိုဝ်ႈမၼ်း ၵမ်းလဵဝ်ၵေႃႈ မၼ်းတေႁဵတ်းၵၢၼ်ပၼ်တင်းမူတ်းၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept) - တူဝ်ယင်ႇ "ၸၢၵ်ႈ ATM"

ဝူၼ်ႉတူၺ်း လၢႆးၵၢၼ်ထွၼ်ငိုၼ်းၼႂ်း Bank ၶႃႈ...

* **Normal SQL Query (ၵၢၼ်တႅမ်ႈမႂ်ႇၵူႈပွၵ်ႈ):**
  မိူၼ်ၼင်ႇ ၸဝ်ႈၵဝ်ႇလႅၼ်ႈၵႂႃႇတီႈ Counter သေ တႅမ်ႈလိၵ်ႈယွၼ်းထွၼ်ငိုၼ်း၊ သူင်ႇမၢႆၾၢင်၊ ပၼ်ပပ်ႉငိုၼ်း... ႁဵတ်းၼင်ႇၼႆၵူႈပွၵ်ႈ မိူဝ်ႈၸဝ်ႈၵဝ်ႇၶႂ်ႈလႆႈငိုၼ်းၶႃႈ။ မၼ်း "ထိူင်း" လႄႈ "ယုင်ႈယၢင်ႈ" ၵွပ်ႈၸဝ်ႈၵဝ်ႇ လႆႈလၢတ်ႈၼႄပိူၼ်ႈ ၵူႈပွၵ်ႈၶႃႈ။

* **Stored Procedure (ၵၢၼ်ၸႂ်ႉၸၢၵ်ႈ ATM):**
  ပဵၼ်ၵၢၼ်ဢၼ် Bank ၶဝ် "တႅမ်ႈပၵ်းၵၢၼ်" ဝႆႉယဝ်ႉတူဝ်ႈၼႂ်းၸၢၵ်ႈ ATM ၶႃႈ။
    1.  ၸဝ်ႈၵဝ်ႇ ဢမ်ႇလူဝ်ႇႁူႉဝႃႈ ၽၢႆႇလင်ၸၢၵ်ႈၼၼ်ႉ မၼ်းၵႂႃႇ SELECT ၶေႃႈမုၼ်းၸိူင်ႉႁိုဝ်။
    2.  ၸဝ်ႈၵဝ်ႇၵူၺ်းလူဝ်ႇသူင်ႇ **"Parameter"** (ၵႃႈၶၼ်) ၶဝ်ႈၵႂႃႇ 2 ဢၼ်: **မၢႆ Pin** လႄႈ **တၢင်းၼမ်ငိုၼ်း** ဢၼ်ၶႂ်ႈလႆႈ။
    3.  ၸၢၵ်ႈ (Procedure) တေႁဵတ်းၵၢၼ်: *ၵူတ်ႇထတ်း Pin -> ၵူတ်ႇထတ်းငိုၼ်းၼႂ်းပပ်ႉ -> တတ်းငိုၼ်း -> သူင်ႇငိုၼ်းဢွၵ်ႇမႃး* ႁင်းမၼ်းၵမ်းလဵဝ်ၶႃႈ။


---

### 2. ႁဵတ်းသင်လႄႈ လူဝ်ႇၸႂ်ႉ Stored Procedure?
1.  **Reusability:** တႅမ်ႈပွၵ်ႈလဵဝ် ၸႂ်ႉလႆႈလၢႆလၢႆတီႈ (မိူၼ်ၼင်ႇ တီႈၼႂ်း App, Website, ဢမ်ႇၼၼ် Admin Tool)။
2.  **Performance:** Database Server တေ "Compiled" (ႁဵတ်းႁႂ်ႈဝႆး) ဝႆႉယဝ်ႉတႃႇသေႇ၊ မၼ်းဝႆးလိူဝ်ၵၢၼ်သူင်ႇ Query ယၢဝ်းယၢဝ်းလုၵ်ႉတီႈ App ၵႂႃႇၶႃႈ။
3.  **Security:** ၸဝ်ႈၵဝ်ႇၸၢင်ႈပၼ်သုၼ်ႇ ႁႂ်ႈ User ႁွင်ႉၸႂ်ႉ Procedure ၵူၺ်း၊ လူၺ်ႈဢမ်ႇပၼ်ၶဝ် SELECT/UPDATE Table ဢၼ်တႄႉမၼ်း ႁင်းၶဝ်ၶႃႈ။

### 3. Syntax ၵၢၼ်သၢင်ႈ (MySQL)
မိူဝ်ႈတႅမ်ႈ Procedure ၼႂ်း MySQL ၼၼ်ႉ ႁဝ်းလူဝ်ႇလႅၵ်ႈ `DELIMITER` ဢွၼ်တၢင်း တွၼ်ႈတႃႇႁႂ်ႈမၼ်းႁူႉဝႃႈ Code ၼႂ်းတူင်ႇၼၼ်ႉ ပဵၼ်ဢၼ်လဵဝ်ၵၼ်ၶႃႈ။

```sql
DELIMITER //

CREATE PROCEDURE GetStudentByCity(IN city_name VARCHAR(50))
BEGIN
    SELECT * FROM students 
    WHERE city = city_name;
END //

DELIMITER ;
```

**လၢႆးႁွင်ႉၸႂ်ႉ (Call):**
```sql
CALL GetStudentByCity('Muse');
```

### 4. Parameters: IN, OUT, လႄႈ INOUT
Stored Procedure ၸၢင်ႈမီး "တၢင်းၶဝ်ႈ" လႄႈ "တၢင်းဢွၵ်ႇ" ၶႃႈ:
* **IN:** သူင်ႇၵႃႈၶၼ်ၶဝ်ႈၵႂႃႇ (မိူၼ်ၼင်ႇ ၸိုဝ်ႈဝဵင်း)။
* **OUT:** ဢဝ်ၵႃႈၶၼ်ဢွၵ်ႇမႃးပၼ် (မိူၼ်ၼင်ႇ ႁူမ်ႈငိုၼ်းတင်းမူတ်း)။
* **INOUT:** သူင်ႇၶဝ်ႈၵႂႃႇ သေလႅၵ်ႈလၢႆႈၵႃႈၶၼ်ယဝ်ႉ သူင်ႇဢွၵ်ႇမႃးၶိုၼ်း။

### 5. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ Procedures
* **Logic Storage:** ၸဝ်ႈၵဝ်ႇၸၢင်ႈသႂ်ႇ `IF-ELSE` ဢမ်ႇၼၼ် `WHILE` ၼႂ်း Procedure လႆႈ (ဢၼ်ႁဝ်းတေႁဵၼ်းၼႂ်း Lesson ၼႃႈ)။
* **Maintenance:** သင် Logic ၵၢၼ်ၼပ်ႉသွၼ်ႇလႅၵ်ႈလၢႆႈ... ၸဝ်ႈၵဝ်ႇၵူၺ်းလူဝ်ႇမႄးၼႂ်း Procedure တီႈလဵဝ်၊ ဢမ်ႇလူဝ်ႇလႅၵ်ႈ Code ၼႂ်း App ၸဝ်ႈၵဝ်ႇၵူႈတီႈၶႃႈ။
* **Testing:** ၸဝ်ႈၵဝ်ႇၸၢင်ႈၸၢမ်း Procedure ၼႂ်း Workbench ႁႂ်ႈမၼ်ႈၸႂ်ဢွၼ်တၢင်း ၸင်ႇဢဝ်ၵႂႃႇၸႂ်ႉၼႂ်း App ၶႃႈၼႃႈ။

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Stored Procedure:** ၸုမ်း Code SQL ဢၼ်သိမ်းဝႆႉၼႂ်း Database Server။
* **Delimiter:** တူဝ်မၵ်းမၼ်ႈ တွၼ်ႈတတ်း Code (ၵမ်ႈၼမ်ပဵၼ် `;` ၵူၺ်းၵႃႈၼႂ်း Procedure ႁဝ်းလႅၵ်ႈပဵၼ် `//`)။
* **Parameter:** ၵႃႈၶၼ်ဢၼ်သူင်ႇၶဝ်ႈၵႂႃႇ ၼႂ်း Procedure။
* **Pre-compiled:** ၵၢၼ်ဢၼ် Database ႁဵတ်းႁႂ်ႈ Code ဝႆးဝႆႉယဝ်ႉ ဢွၼ်တၢင်းၵၢၼ်ႁွင်ႉၸႂ်ႉ။
