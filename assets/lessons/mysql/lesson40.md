တူဝ်ယင်ႇမိူၼ်ၼင်ႇ ႁဝ်းမီး Table `Students` လႄႈ Table `Classes`။

ႁဝ်းတေလႆႈဢဝ်ၶဝ်မႃး "ၵွင်ႉ" (Link) ၵၼ်ၶိုၼ်းၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept) - လၢႆးၵွင်ႉ Table
သင်ၸဝ်ႈၵဝ်ႇၶႂ်ႈဢဝ် Table သွင်ဢၼ်မႃး "ၵွင်ႉ" (Link) ၸပ်းၵၼ်ၼႆ... ၸဝ်ႈၵဝ်ႇတေလႆႈမီး "ၸိူၵ်ႈ" ဢၼ်မတ်ႉၶဝ်ဝႆႉၶႃႈ။

* **Primary Key (PK): "မၢႆသေႃး"** (ၼႂ်း Table တူဝ်မႄႈ)
    * ပဵၼ်မၢႆၼပ်ႉ (ID) တႅတ်ႈတေႃး ဢၼ်မၵ်းမၼ်ႈဝႃႈ ၶေႃႈမုၼ်းၼၼ်ႉ ပဵၼ်ၶေႃႈမုၼ်းၵေႃႉလႂ်/ဢၼ်လႂ် တႅတ်ႈတေႃး။
    * *တူဝ်ယင်ႇ:* ၼႂ်း Table **`Customers`**... မၢႆ `customer_id` ပဵၼ် **PK** (မိူၼ်မၢႆၾၢင်ၸဝ်ႈၶွင်)။

* **Foreign Key (FK): "မၢႆသူင်ႇ"** (ၼႂ်း Table တူဝ်လုၵ်ႈ)
    * ပဵၼ်ၵၢၼ်ဢဝ် "မၢႆသေႃး" (PK) ၼၼ်ႉ ၵႂႃႇတႅမ်ႈမၢႆဝႆႉၼႂ်း Table တၢင်ႇဢၼ် တွၼ်ႈတႃႇသူင်ႇၸူး (Reference) ၵၼ်။
    * *တူဝ်ယင်ႇ:* ၼႂ်း Table **`Orders`**... ႁဝ်းဢဝ်မၢႆ `customer_id` မႃးတႅမ်ႈဝႆႉ တွၼ်ႈတႃႇႁူႉဝႃႈ "Order ၼႆႉ ပဵၼ်ၶေႃႈၵူၼ်းၵေႃႉလႂ်"။ မၢႆၼႆႉယဝ်ႉ ႁွင်ႉဝႃႈ **FK**။


* **JOIN: "ၵၢၼ်ႁူမ်ႈထႅဝ်"**
    * မိူဝ်ႈႁဝ်းၸႂ်ႉၶေႃႈၵႂၢမ်း `JOIN`... Database တေၵႂႃႇသွၵ်ႈတူၺ်းဝႃႈ "မၢႆသူင်ႇ" (FK) ၼႂ်း Table ၼိုင်ႈ တူဝ်ႇၵၼ်တင်း "မၢႆသေႃး" (PK) ၼႂ်းထႅင်ႈ Table ၼိုင်ႈႁႃႉ?
    * သင်တူဝ်ႇၵၼ်ယဝ်ႉ... မၼ်းတေဢဝ်ၶေႃႈမုၼ်းတင်းသွင်ၽၢႆႇ မႃးသုပ်းၸပ်းၵၼ် သေၼႄပဵၼ်ထႅဝ် (Row) ယၢဝ်းယၢဝ်းဢၼ်လဵဝ်ၵၼ် ႁႂ်ႈႁဝ်းလူငၢႆႈၶႃႈ။

### 2. ႁဵတ်းသင်လႄႈ လူဝ်ႇၸႂ်ႉ JOIN?
* **Data Retrieval:** တွၼ်ႈတႃႇဢဝ်ၶေႃႈမုၼ်း ဢၼ်ယူ ၵေႃႉတီႈတီႈ မႃးၼႄႁူမ်ႈၵၼ် ၼႂ်းၼႃႈ Report ဢၼ်လဵဝ်။
* **Relational Power:** မၼ်းၸွႆႈႁႂ်ႈႁဝ်းသိမ်းၶေႃႈမုၼ်းလႆႈ "သႅၼ်ႈသႂ်" (ဢမ်ႇသွၼ်ႉၵၼ်) ၵူၺ်းၵႃႈ ယင်းၸၢင်ႈလူ (Read) ႁူမ်ႈၵၼ်လႆႈငၢႆႈငၢႆႈၶႃႈ။


### 3. Syntax ပိုၼ်ႉထၢၼ် (JOIN Syntax)
မိူဝ်ႈႁဝ်းတႅမ်ႈ JOIN... ႁဝ်းလူဝ်ႇမၵ်းမၼ်ႈ 3 ပိူင်ၶႃႈ:
1.  **SELECT:** ၶႂ်ႈလႆႈ Column လႂ် လုၵ်ႉတီႈ Table လႂ်?
2.  **FROM:** Table တူဝ်ဢွၼ်တၢင်းသုတ်းပဵၼ်သင်?
3.  **ON:** Column လႂ် ဢၼ်မီးၵႃႈၶၼ်မိူၼ်ၵၼ် (PK = FK)?

```sql
SELECT students.name, classes.class_name
FROM students
JOIN classes ON students.class_id = classes.class_id;
```

### 4. ၵၢၼ်ၸႂ်ႉ Table Aliasing (ၸိုဝ်ႈတႅၼ်း)
မိူဝ်ႈၸိုဝ်ႈ Table ယၢဝ်းႁႅင်း... ႁဝ်းၸၢင်ႈပၼ်ၸိုဝ်ႈပွတ်း (Short Name) ႁႂ်ႈတႅမ်ႈ Code ငၢႆႈလိူဝ်ၵဝ်ႇၶႃႈ။

```sql
SELECT s.name, c.class_name
FROM students AS s
JOIN classes AS c ON s.class_id = c.class_id;
```
*(s = students, c = classes)*

### 5. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ Join Fundamentals
* **Common Column:** ၵၢၼ် Join တေႁဵတ်းၵၢၼ်လႆႈလီ မိူဝ်ႈတင်းသွင် Table မီး Column ဢၼ်မီးၶေႃႈမုၼ်း "ၸုမ်းလဵဝ်ၵၼ်" (မိူၼ်ၼင်ႇ ID) ၶႃႈ။
* **Ambiguous Columns:** သင်တင်းသွင် Table မီးၸိုဝ်ႈ Column မိူၼ်ၵၼ် (မိူၼ်ၼင်ႇ `name` တင်း `name`)... ၸဝ်ႈၵဝ်ႇတေလႆႈတႅမ်ႈ `students.name` တွၼ်ႈတႃႇလၢတ်ႈၼႄ DB ႁႂ်ႈၸႅင်ႈလႅင်းၶႃႈ။
* **Relationship Types:** ၸဝ်ႈၵဝ်ႇၸၢင်ႈ Join Table ၸိူဝ်းဢၼ်မီး Relationship ပဵၼ် 1:1, 1:N, ဢမ်ႇၼၼ် M:N လႆႈၵူႈပိူင်ၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Join:** ၵၢၼ်ႁူမ်ႈၶေႃႈမုၼ်း ဢၼ်မီးဝႆႉတီႈ Table လၢႆလၢႆဢၼ် ၸွမ်းၼင်ႇသဵၼ်ႈၵွင်ႉ (Relationship)။
* **Predicate (ON):** တီႈဢၼ်မၵ်းမၼ်ႈဝႃႈ တေ Join ၵၼ်တီႈလႂ်။
* **Aliasing:** ၵၢၼ်ပၼ်ၸိုဝ်ႈပွတ်းပွတ်း တႃႇ Table တွၼ်ႈတႃႇတႅမ်ႈ Query ငၢႆႈ။
* **Result Set:** ၽွၼ်းလႆႈ (Output) ဢၼ်ဢွၵ်ႇမႃး ဝၢႆးသေ Join ယဝ်ႉ။
