# Lesson 42: "Outer Joins I: Mastering LEFT JOIN Logic"

**LEFT JOIN** (ဢမ်ႇၼၼ် **LEFT OUTER JOIN**) မၢႆထိုင်ၵၢၼ်ဢဝ်ၶေႃႈမုၼ်းလုၵ်ႉတီႈ Table ၽၢႆႇသၢႆႉ (Left Table) မႃးၼႄ **တင်းမူတ်း** သေ 
သင်မီးၶေႃႈမုၼ်းၼႂ်း Table ၽၢႆႇၶႂႃ (Right Table) ဢၼ်ၵွင်ႉၵၢႆႇၵၼ်ၺႃး... 
မၼ်းတေဢဝ်မႃးၼႄၸွမ်း **NULL** ဝႆႉၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်း "ၵၢၼ်ၸႅၵ်ႇပၼ်ႁွင်ႈၼွၼ်း" ၶႃႈ...
* **Table A (Left):** သဵၼ်ႈမၢႆလုၵ်ႈႁဵၼ်းတင်းမူတ်း။
* **Table B (Right):** သဵၼ်ႈမၢႆႁွင်ႈၼွၼ်း။
* **LEFT JOIN:** တေၼႄၸိုဝ်ႈလုၵ်ႈႁဵၼ်း **ၵူႈၵေႃႉ**။ ၵေႃႉလႂ်မီးႁွင်ႈ... ၵေႃႈၼႄမၢႆႁွင်ႈၸွမ်း။ ၵေႃႉလႂ်ဢမ်ႇပႆႇမီးႁွင်ႈ... ၵေႃႈၼႄၸိုဝ်ႈမၼ်းဝႆႉသေ တွၼ်ႈမၢႆႁွင်ႈၼၼ်ႉ တေပဵၼ်ဝႆႉ **NULL** (ပဝ်ႇဝႆႉ) ၶႃႈ။



### 2. Syntax ၵၢၼ်တႅမ်ႈ LEFT JOIN
လၢႆးတႅမ်ႈမိူၼ် `INNER JOIN` ၵူၺ်းၶႃႈ၊ လႅၵ်ႈၶေႃႈၵႂၢမ်း Keyword မၼ်းပဵၼ် LEFT JOIN ၵႂႃႇၵူၺ်းၶႃႈ။

```sql
SELECT 
    s.student_name, 
    c.class_name
FROM students AS s        -- ၼႆႉပဵၼ် LEFT Table
LEFT JOIN classes AS c    -- ၼႆႉပဵၼ် RIGHT Table
    ON s.class_id = c.class_id;
```

### 3. တူဝ်ယင်ႇ: ၽွၼ်းလႆႈ (Result)
ဝူၼ်ႉတူၺ်းလုၵ်ႈႁဵၼ်းႁဝ်း:
1. **ၸၢႆးမၢဝ်း** (Class 101)
2. **ၸၢႆးသႅင်** (Class NULL - ဢမ်ႇပႆႇမီးႁွင်ႈ)

**မိူဝ်ႈၸႂ်ႉ LEFT JOIN:**
* **ၸၢႆးမၢဝ်း** -> ၵွင်ႉၺႃး Class 101 -> ၼႄ: `ၸၢႆးမၢဝ်း | Programming`
* **ၸၢႆးသႅင်** -> ဢမ်ႇမီး Class -> ၼႄ: `ၸၢႆးသႅင် | NULL`

> **Note:** သင်ပဵၼ် `INNER JOIN` ... **ၸၢႆးသႅင်** တေဢမ်ႇဢွၵ်ႇမႃးၼႄၶႃႈ။ ၵူၺ်းၵႃႈ **LEFT JOIN** တေဢဝ်မၼ်းဢွၵ်ႇမႃးၼႄတႃႇသေႇ ၵွပ်ႈမၼ်းယူႇၼႂ်း Table ၽၢႆႇသၢႆႉ (Students) ၶႃႈ။



### 4. ႁဵတ်းသင်လႄႈ လူဝ်ႇၸႂ်ႉ LEFT JOIN?
* **Finding Gaps:** တႃႇသွၵ်ႈႁႃ "ၶေႃႈမုၼ်းဢၼ်ႁၢႆလၢႆ" (မိူၼ်ၼင်ႇ ႁႃလုၵ်ႈႁဵၼ်း ဢၼ်ဢမ်ႇပႆႇလႆႈသိုဝ်ႉပပ်ႉ)။
* **Complete Reports:** တႃႇႁဵတ်းသဵၼ်ႈမၢႆ ဢၼ်ဢမ်ႇၶႂ်ႈႁႂ်ႈၸိုဝ်ႈၵူၼ်းႁၢႆ သင်ၶဝ်ဢမ်ႇပႆႇမီး သဵၼ်ႈၵွင်ႉၶႃႈ။

### 5. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ LEFT JOIN
* **Table Order Matters:** ၼႂ်း LEFT JOIN... ၵၢၼ်ဝႆႉ Table လႂ် ဝႆႉၽၢႆႇၼႃႈ (Left) ပဵၼ်လွင်ႈလွင်ႈယႂ်ႇသုတ်း။ သင်ၸဝ်ႈၵဝ်ႇပိၼ်ႇ `Classes` မႃးဝႆႉၽၢႆႇၼႃႈ... ၽွၼ်းလႆႈတေဢွၵ်ႇမႃးပႅၵ်ႇၵၼ်ၶႃႈၼႃႈ။
* **IS NULL:** ႁဝ်းၵႆႉၸႂ်ႉ `LEFT JOIN` ႁူမ်ႈတင်း `WHERE c.class_id IS NULL` တႃႇသွၵ်ႈႁႃ ၵူၼ်းဢၼ် "ဢမ်ႇမီးႁွင်ႈ" ၵူၺ်း ၼႆၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **LEFT JOIN:** ၵၢၼ်ႁူမ်ႈ Table ဢၼ်ဢဝ်ၶေႃႈမုၼ်းၽၢႆႇသၢႆႉပဵၼ်လူင်။
* **Outer Join:** ၸုမ်း JOIN ဢၼ်ပၼ်သုၼ်ႇႁႂ်ႈမီး NULL ၼႂ်းၽွၼ်းလႆႈ (ပႃးတင်း Left, Right, Full)။
* **Preserving Records:** ၵၢၼ်ဝႆႉၶေႃႈမုၼ်း ဢမ်ႇႁႂ်ႈႁၢႆပႅတ်ႈ မိူဝ်ႈဢမ်ႇမီးတူဝ်ၵွင်ႉ (Match)။
