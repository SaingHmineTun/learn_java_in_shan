# Lesson 3: "MySQL Ecosystem: Server Engine vs. Workbench GUI"

ၼႂ်းလွၵ်းၵၢၼ် MySQL ၼႆႉ မၼ်းမီးသွင်တွၼ်ႈလူင် ဢၼ်ႁဵတ်းၵၢၼ်ၸွမ်းၵၼ်ၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းၵႂႃႇလႅၵ်ႈငိုၼ်းၼႂ်း **"ယေးငိုၼ်း" (Bank)** ၼိုင်ႈဢၼ်ၶႃႈ...
* **MySQL Server:** ပဵၼ် "ႁွင်ႈသိမ်းငိုၼ်း" (The Vault) ဢၼ်ယူႇၽၢႆႇၼႂ်း၊ မၼ်းသိမ်းငိုၼ်းဝႆႉႁႂ်ႈလွတ်ႈၽေး။ ႁဝ်းၶဝ်ႈၵႂႃႇႁင်းၵူၺ်းဢမ်ႇလႆႈ။
* **MySQL Workbench:** ပဵၼ် "ၼႃႈတွၵ်ႇ" (The Teller/Counter) တီႈဢၼ်ႁဝ်းၵႂႃႇလၢတ်ႈတေႃႇ ဝႃႈၶႂ်ႈလႆႈသင်၊ သေၶဝ်တေၵႂႃႇဢဝ်ၼႂ်းႁွင်ႈသိမ်းငိုၼ်းမႃးပၼ်ႁဝ်းၶႃႈ။



### 2. MySQL Server (The Engine)
ၼႆႉပဵၼ် "တူဝ်ႁဵတ်းၵၢၼ်" (Background Process) ဢၼ်ႁဝ်းဢမ်ႇႁၼ်ၼႃႈတႃမၼ်းၶႃႈ။
* **Role:** မၼ်းမီးၼႃႈတီႈ သိမ်းၶေႃႈမုၼ်း၊ ၵုမ်းၵမ်းလွင်ႈလွတ်ႈၽေး (Security) လႄႈ ႁဵတ်းၵၢၼ်ၸွမ်း SQL ဢၼ်ႁဝ်းသူင်ႇၵႂႃႇ။
* **How it works:** မၼ်းတေ "တိုၼ်ႇ" ဝႆႉယူႇတႃႇသေႇ ၼႂ်း Background ၶွမ်းႁဝ်း (Service) တႃႇပႂ်ႉထၢမ်/တွပ်ႇ ၶေႃႈမုၼ်းၶႃႈ။

### 3. MySQL Workbench (The GUI)
ၼႆႉပဵၼ် "Software" ဢၼ်မီးၼႃႈတႃ (Visual Tool) ႁႂ်ႈႁဝ်းၸႂ်ႉငၢႆႈၶႃႈ။
* **GUI (Graphical User Interface):** ႁဝ်းတဵၵ်း (Click) လႆႈ၊ တႅမ်ႈ SQL လႆႈ၊ သေႁၼ်ၶေႃႈမုၼ်းပဵၼ် Table လႆႈငၢႆႈငၢႆႈ။
* **Role:** မၼ်းၸွႆႈႁဝ်း တႅမ်ႈ Query၊ သၢင်ႈ EER Diagram (ပိူင်သၢင်ႈ Database) လႄႈ ၸတ်းၵၢၼ် Server ၶႃႈ။
* **Alternatives:** လိူဝ်သေ Workbench ယဝ်ႉ ႁဝ်းယင်းၸႂ်ႉလႆႈ **phpMyAdmin**, **DBeaver**, ဢမ်ႇၼၼ် **Command Line (CMD)** ၵေႃႈလႆႈၶႃႈ။

### 4. ၵၢၼ်ၵွင်ႉၵၢႆႇၵၼ် (Connection)
မိူဝ်ႈႁဝ်းပိုတ်ႇ Workbench... ႁဝ်းလႆႈ "ၵွင်ႉ" (Connect) ၵႂႃႇၸူး Server ၶႃႈ။
* **Address:** ၵႆႉပဵၼ် `127.0.0.1` (Localhost) ဢၼ်ပဵၼ်ၶွမ်းႁဝ်းၵူၺ်း။
* **Port:** MySQL ၵႆႉၸႂ်ႉ **3306** ပဵၼ် "သဵၼ်ႈတၢင်း" ၵွင်ႉၵၢႆႇၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Database Engine:** တူဝ် Software ဢၼ်ႁဵတ်းၵၢၼ်သိမ်း လႄႈ သွၵ်ႈႁႃၶေႃႈမုၼ်းတႄႉတႄႉ။
* **Client:** Software ဢၼ်ၸႂ်ႉတႃႇၵွင်ႉၸူး Server (မိူၼ်ၼင်ႇ Workbench)။
* **Service:** ပရူဝ်ႇၵရႅမ်ႇ ဢၼ်ႁဵတ်းၵၢၼ်ယူႇ Background ၶွမ်း။
* **GUI:** ၼႃႈတႃ Software ဢၼ်မီးႁၢင်ႈၽၢင်၊ ပုမ်ႇတဵၵ်း ႁႂ်ႈၸႂ်ႉငၢႆႈ။
