# Lesson 5: "Database Hierarchy: Instances, Schemas, Tables, and Rows"

မိူဝ်ႈႁဝ်းပိုတ်ႇ MySQL Workbench မႃးၼၼ်ႉ ႁဝ်းတေႁၼ်မီးသဵၼ်ႈမၢႆလၢႆလၢႆဢၼ်ၶႃႈ။ တႃႇတေဢမ်ႇယုင်ႈၼၼ်ႉ ႁဝ်းလူဝ်ႇပွင်ႇၸႂ်ဝႃႈ "သင်ယူႇၼႂ်းသင်" ၼႆၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်း "ႁူင်းႁဵၼ်း" (School) ၼိုင်ႈဢၼ်ၶႃႈ...
* **Instance:** ပဵၼ် "တူဝ်ႁူင်းႁဵၼ်း" တင်းသဵင်ႈ (Building)။
* **Schema (Database):** ပဵၼ် "ႁွင်ႈၼိုင်ႈႁွင်ႈ" (Classroom) ဢၼ်မီးၼႂ်းႁူင်းႁဵၼ်း။
* **Table:** ပဵၼ် "ပပ်ႉသဵၼ်ႈမၢႆ" (Register Book) ဢၼ်မီးၼႂ်းႁွင်ႈၼၼ်ႉ။
* **Row (Record):** ပဵၼ် "ထႅဝ်လိၵ်ႈ" ၼိုင်ႈထႅဝ် ၼႂ်းပပ်ႉ (ၸိုဝ်ႈလုၵ်ႈႁဵၼ်းၵေႃႉၼိုင်ႈ)။
* **Column (Field):** ပဵၼ် "ႁူဝ်ၶေႃႈ" ၼိူဝ်ထႅဝ် (ၸိုဝ်ႈ၊ ဢႃႇယု၊ ဝဵင်း)။



### 2. Instance (တူဝ် Server)
ၼႆႉပဵၼ်ၸၼ်ႉသုင်သုတ်းၶႃႈ။ မၼ်းပဵၼ် MySQL Software ဢၼ် Install ဝႆႉၼႂ်းၶွမ်းႁဝ်း။
* Instance ၼိုင်ႈဢၼ် ၸၢင်ႈမီး **Schemas** (Database) လႆႈတင်းၼမ်ၶႃႈ။

### 3. Schema / Database
ၼႂ်း MySQL ၼႆႉ ၶေႃႈၵႂၢမ်း "Schema" လႄႈ "Database" ပဵၼ်ဢၼ်လဵဝ်ၵၼ်ၶႃႈ။
* မၼ်းပဵၼ် "ထူင်" ဢမ်ႇၼၼ် "တိူက်း" ဢၼ်ႁဝ်းၸႂ်ႉတႃႇၸႅၵ်ႇ Project ႁဝ်းၶႃႈ။
* တူဝ်ယင်ႇ: `mao_app_gallery`, `shan_proverbs_db`, `student_manager`။

### 4. Table
ၼႆႉပဵၼ်တီႈဢၼ် "ၶေႃႈမုၼ်းတႄႉတႄႉ" ယူႇဝႆႉၶႃႈ။
* Table ၼိုင်ႈဢၼ် လူဝ်ႇမီးၸိုဝ်ႈ ဢၼ်ၸႅင်ႈလႅင်း (မိူၼ်ၼင်ႇ `Users`, `Products`)။
* မၼ်းမီး **Columns** တႃႇမၵ်းမၼ်ႈ "ပိူင်ၶေႃႈမုၼ်း" (Data Type) ၶႃႈ။

### 5. Row လႄႈ Column
* **Column:** ပဵၼ် "Vertical" (ထႅဝ်တင်ႈ)။ မၼ်းမၵ်းမၼ်ႈဝႃႈ ၶေႃႈမုၼ်းၼၼ်ႉ တေပဵၼ် "လိၵ်ႈ" (String) ႁႃႉ ဢမ်ႇၼၼ် "တူဝ်ၼပ်ႉ" (Integer) ၶႃႈ။
* **Row:** ပဵၼ် "Horizontal" (ထႅဝ်ၼွၼ်း)။ မၼ်းပဵၼ်ၶေႃႈမုၼ်းတဵမ်ထူၼ်ႈ ၼိုင်ႈဢၼ် (Record) ၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Data Model:** ပိူင်သၢင်ႈတူဝ်ယင်ႇ ဢၼ်ၼႄလွင်ႈၵွင်ႉၵၢႆႇၵၼ် ၼႂ်း Hierarchy။
* **Metadata:** ၶေႃႈမုၼ်း ဢၼ်လၢတ်ႈၼႄလွင်ႈ "ပိူင်သၢင်ႈ" (မိူၼ်ၼင်ႇ ၸိုဝ်ႈ Table, ၸိုဝ်ႈ Column)။
* **Field:** ၸိုဝ်ႈထႅင်ႈဢၼ်ၼိုင်ႈ တွၼ်ႈတႃႇ Column။
* **Record:** ၸိုဝ်ႈထႅင်ႈဢၼ်ၼိုင်ႈ တွၼ်ႈတႃႇ Row။
