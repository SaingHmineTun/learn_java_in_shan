# Lesson 35: "Nullability: The IS NULL and IS NOT NULL Logic"

မိူဝ်ႈႁဝ်းသွၵ်ႈႁႃၶေႃႈမုၼ်း... ၵႆႉမီးလုၵ်း (Cells) ဢၼ်ပဝ်ႇဝႆႉ (မိူၼ်ၼင်ႇ လုၵ်ႈႁဵၼ်းဢၼ်ဢမ်ႇပႆႇမီးမၢႆၾူၼ်း)။ 
ၵၢၼ်ၸႂ်ႉၶိူင်ႈမၢႆ `=` တႃႇႁႃ NULL တေဢမ်ႇႁဵတ်းၵၢၼ်ၶႃႈ၊ ႁဝ်းလူဝ်ႇၸႂ်ႉၶေႃႈၵႂၢမ်းတႅတ်ႈတေႃးမၼ်းၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်း "ၵၢၼ်ၵူတ်ႇထတ်းသဵၼ်ႈမၢႆ" ၶႃႈ...
* **IS NULL:** ႁႃထႅဝ်ဢၼ် "ယင်းဢမ်ႇပႆႇလႆႈထႅမ်" ၶေႃႈမုၼ်းသႂ်ႇ။
* **IS NOT NULL:** ႁႃထႅဝ်ဢၼ် "မီးၶေႃႈမုၼ်းဝႆႉယဝ်ႉ" တႅတ်ႈတေႃးၶႃႈ။



### 2. ႁဵတ်းသင် `=` ၸႂ်ႉတင်း NULL ဢမ်ႇလႆႈ?
ၼႂ်း SQL... `NULL` မၢႆထိုင် "Unknown"။ သင်ႁဝ်းထၢမ်ဝႃႈ `Unknown = Unknown` ႁႃႉ? Database တေတွပ်ႇဝႃႈ "Unknown" (ဢမ်ႇႁူႉ) မိူၼ်ၵၼ်ၶႃႈ။ 
ၵွပ်ႈၼၼ် ႁဝ်းၸင်ႇလူဝ်ႇၸႂ်ႉ Operator ဢၼ်ပဵၼ်ၵၢၼ်ၸႅတ်ႈ "သိုင်ႇဝႆႉ" (State) တႅၼ်းၵၢၼ်တႅၵ်ႈ "ၵႃႈၶၼ်" (Value) ၶႃႈ။

```sql
-- ႁၢမ်ႈတႅမ်ႈၼင်ႇၼႆ (မၼ်းတေဢမ်ႇဢွၵ်ႇသင်မႃးၼႄ)
SELECT * FROM students WHERE phone = NULL; 

-- တေလႆႈတႅမ်ႈၼင်ႇၼႆ (Correct Syntax)
SELECT * FROM students WHERE phone IS NULL;
```

### 3. ၵၢၼ်ၸႂ်ႉ IS NOT NULL
ၸႂ်ႉမိူဝ်ႈႁဝ်းၶႂ်ႈလႆႈ ၶေႃႈမုၼ်းဢၼ် "တဵမ်ထူၼ်ႈ" ယဝ်ႉၼၼ်ႉၶႃႈ။

```sql
-- လိူၵ်ႈလုၵ်ႈႁဵၼ်း ဢၼ်မီး Email တႅတ်ႈတေႃးယဝ်ႉၵူၺ်း
SELECT * FROM students 
WHERE email IS NOT NULL;
```

### 4. ၵၢၼ်ၸႂ်ႉ NULL ႁူမ်ႈတင်း Logic တၢင်ႇပိူင်
ႁဝ်းၸၢင်ႈၸႂ်ႉ `IS NULL` ႁူမ်ႈတင်း `AND` ဢမ်ႇၼၼ် `OR` တႃႇမၵ်းမၼ်ႈ Filter ဢၼ်ယုင်ႈယၢင်ႈလႆႈၶႃႈ။

```sql
-- ႁႃလုၵ်ႈႁဵၼ်း ဢၼ်ယူႇဝဵင်း 'Muse' သေ ယင်းဢမ်ႇပႆႇမီး မၢႆၾူၼ်း
SELECT * FROM students 
WHERE city = 'Muse' AND phone IS NULL;
```

### 5. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ Nullability
* **NULL is not 0:** တွင်းဝႆႉတႃႇသေႇဝႃႈ `0` ပဵၼ်တူဝ်ၼပ်ႉ၊ `''` ပဵၼ်လိၵ်ႈပဝ်ႇ၊ ၵူၺ်းၵႃႈ `NULL` ပဵၼ်လွင်ႈဢၼ် "ဢမ်ႇမီးၶေႃႈမုၼ်း" ၶႃႈ။
* **Arithmetic with NULL:** သင်ၸဝ်ႈၵဝ်ႇဢဝ် `10 + NULL`... ၽွၼ်းလႆႈမၼ်းတေဢွၵ်ႇမႃးပဵၼ် `NULL` တႃႇသေႇၶႃႈ။
* **Sorting:** မိူဝ်ႈႁဝ်း `ORDER BY`... MySQL တေဢဝ် NULL ဝႆႉၽၢႆႇၼႃႈသုတ်း (Ascending) ၶႃႈၼႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **NULL:** ၵႃႈၶၼ်ဢၼ်ဢမ်ႇပႆႇမၵ်းမၼ်ႈဝႆႉ (Unknown/Missing)။
* **Predicate:** ၶေႃႈၵႂၢမ်းတႃႇၸႅတ်ႈ Condition (မိူၼ်ၼင်ႇ IS NULL)။
* **Three-valued Logic:** ပိူင် Logic ၼႂ်း SQL ဢၼ်မီး True, False, လႄႈ Unknown (NULL)။
* **Empty String:** လိၵ်ႈဢၼ်ဢမ်ႇမီးတူဝ်လိၵ်ႈသင် ( '' ) ၵူၺ်းၵႃႈ ဢမ်ႇၸႂ်ႈ NULL။
