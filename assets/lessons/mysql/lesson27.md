# Lesson 27: "Data Removal Logic: DROP vs. TRUNCATE vs. DELETE"

မိူဝ်ႈႁဝ်းၶႂ်ႈ "လၢင်ႉ" ၶေႃႈမုၼ်း ဢမ်ႇၼၼ် "ယႃႉ" Table ပႅတ်ႈၼၼ်ႉ... ႁဝ်းလူဝ်ႇလိူၵ်ႈၸႂ်ႉၶေႃႈၵႂၢမ်း ႁႂ်ႈမႅၼ်ႈၸွမ်း ၵၢၼ်ၸႂ်ႉတိုဝ်းၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်း "ပပ်ႉလိၵ်ႈ" ၼိုင်ႈပပ်ႉၶႃႈ...
* **DELETE:** မိူၼ်ၼင်ႇ "ထွၼ်ပႅတ်ႈထႅဝ်လိၵ်ႈ" - ယႃႇပၼ်ထႅဝ်လိၵ်ႈဢၼ်ႁဝ်းလိူၵ်ႈ၊ ၵူၺ်းၵႃႈ ပပ်ႉယင်းမီးယူႇ။
* **TRUNCATE:** မိူၼ်ၼင်ႇ "သိၵ်ႇပႅတ်ႈ ၵူႈၼႃႈ" - ႁဵတ်းႁႂ်ႈပပ်ႉပဵၼ်ၸေႈပဝ်ႇ တင်းမူတ်း၊ ၵူၺ်းၵႃႈ တူဝ်ပပ်ႉယင်းမီးယူႇ။
* **DROP:** မိူၼ်ၼင်ႇ "ဢဝ်ပပ်ႉၵႂႃႇၽဝ်ၾႆး" - ယႃႉပႅတ်ႈတင်းၶေႃႈမုၼ်း လႄႈ တင်းပိူင်သၢင်ႈ (Structure) မၼ်း တင်းမူတ်းၶႃႈ။



### 2. DELETE (ၵၢၼ်ယႃႉၸွမ်းထႅဝ်)
ၸႂ်ႉတွၼ်ႈတႃႇ ယႃႉၶေႃႈမုၼ်းမၢင်တွၼ်ႈ ၼႂ်း Table ၶႃႈ။
* **Command:** `DELETE FROM students WHERE id = 1;`
* **Logic:** မၼ်းတေတူၺ်းၸွမ်းပၵ်းၵၢၼ် `WHERE`။ သင်ဢမ်ႇသႂ်ႇ `WHERE`... မၼ်းတေယႃႉပႅတ်ႈ ၵူႈထႅဝ် (ၵူၺ်းၵႃႈ မၼ်းတေယႃႉၵႂႃႇ ၼိုင်ႈထႅဝ် ယဝ်ႉ ၼိုင်ႈထႅဝ် ၶႃႈ)။
* **Rollback:** ႁဝ်းၸၢင်ႈ "Undo" (ၶိုၼ်းဢဝ်မႃး) လႆႈ သင်ႁဝ်းၸႂ်ႉ Transaction ဝႆႉၶႃႈ။

### 3. TRUNCATE (ၵၢၼ်လၢင်ႉ Table)
ၸႂ်ႉတွၼ်ႈတႃႇ ယႃႉၶေႃႈမုၼ်း "တင်းမူတ်း" ၼႂ်း Table ႁႂ်ႈပဝ်ႇၵႂႃႇၵမ်းလဵဝ်ၶႃႈ။
* **Command:** `TRUNCATE TABLE students;`
* **Logic:** မၼ်းတေဢမ်ႇယႃႉၸွမ်းထႅဝ်၊ မၼ်းတေ "ယႃႉပႅတ်ႈ Table သေ သၢင်ႈမႂ်ႇၶိုၼ်း" (Drop and Recreate) ႁင်းမၼ်းၶႃႈ။
* **Effect:** မၢႆၼပ်ႉ **AUTO_INCREMENT** တေၶိုၼ်းတႄႇတီႈ **1** မႂ်ႇၶိုၼ်းၶႃႈ။
* **Speed:** ဝႆးလိူဝ် `DELETE` တႄႉတႄႉ ၵွပ်ႈမၼ်းဢမ်ႇတႅမ်ႈ Log ၼမ်ၶႃႈ။

### 4. DROP (ၵၢၼ်ယႃႉတင်းပိူင်)
ၸႂ်ႉတွၼ်ႈတႃႇ ယႃႉ Table ပႅတ်ႈတင်းသဵင်ႈ ပႃးတင်း ၶေႃႈမုၼ်းၼႂ်း Database ၶႃႈ။
* **Command:** `DROP TABLE students;`
* **Logic:** ၶေႃႈမုၼ်းၵေႃႈႁၢႆ၊ ပိူင်သၢင်ႈ (Structure) ၵေႃႈႁၢႆ၊ သဵၼ်ႈၵွင်ႉ (Index/Constraints) ၵေႃႈႁၢႆၶႃႈ။

### 5. ပၵ်းၵၢၼ်တႅၵ်ႈတူၺ်း (Summary)

| Feature | DELETE | TRUNCATE | DROP |
| :--- | :--- | :--- | :--- |
| **Command Type** | DML | DDL | DDL |
| **WHERE clause** | Support | No | No |
| **Speed** | Slow | Fast | Fast |
| **Auto-Increment** | Keeps current | Resets to 1 | Deleted |
| **Structure** | Stays | Stays | Removed |

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **DML (Data Manipulation Language):** ၶေႃႈၵႂၢမ်းၸတ်းၵၢၼ်ၶေႃႈမုၼ်း (မိူၼ်ၼင်ႇ DELETE)။
* **DDL (Data Definition Language):** ၶေႃႈၵႂၢမ်းၸတ်းၵၢၼ်ပိူင်သၢင်ႈ (မိူၼ်ၼင်ႇ DROP, TRUNCATE)။
* **Transaction:** ၵၢၼ်ႁဵတ်းသၢင်ႈ ဢၼ်ၸၢင်ႈ Commit (မၼ်ႈၵိုမ်း) ဢမ်ႇၼၼ် Rollback (ၶိုၼ်းဢဝ်မႃး) လႆႈ။
* **Logging:** ၵၢၼ်မၢႆတွင်း ႁွႆးၵၢၼ်ႁဵတ်းသၢင်ႈ ၼႂ်း Database။
