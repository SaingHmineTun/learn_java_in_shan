# Lesson 20: "Temporal Data II: DATETIME vs. TIMESTAMP and Timezones"

မိူဝ်ႈႁဝ်းသိမ်း "ဝၼ်းလႄႈမွင်း" ႁူမ်ႈၵၼ် (မိူၼ်ၼင်ႇ `2026-04-03 14:30:00`) ၼၼ်ႉ... ႁဝ်းလူဝ်ႇလိူၵ်ႈဝႃႈ တေဢဝ်တူဝ်ဢၼ် "မၼ်ႈၵိုမ်း" ဢမ်ႇၼၼ် တူဝ်ဢၼ် "လႅၵ်ႈလၢႆႈၸွမ်းၶၢဝ်းယၢမ်းၵူႈမိူင်း" ၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်း "ႁၢင်ႈထႅမ်မွင်း" 2 ပိူင်ၶႃႈ...
* **DATETIME:** မိူၼ်ၼင်ႇ "ပၵ်းယိၼ်ႇဢၼ်တႅမ်ႈဝႆႉၼိူဝ်ၸေႈ" - မၼ်းတေတႅမ်ႈဝႆႉမၢႆလၢႆၵေႃႈ ပဵၼ်မၢႆၼၼ်ႉတႃႇသေႇ၊ ဢမ်ႇလႅၵ်ႈလၢႆႈၸွမ်း တီႈယူႇႁဝ်း။
* **TIMESTAMP:** မိူၼ်ၼင်ႇ "မွင်း Digital" - မၼ်းတေမႄးပၼ် (Convert) ၸွမ်းၼင်ႇ TimeZone ဢၼ်ႁဝ်းယူႇဝႆႉၶႃႈ။



### 2. DATETIME (မၼ်ႈၵိုမ်း)
ၸႂ်ႉတွၼ်ႈတႃႇ ၶေႃႈမုၼ်းဢၼ်ႁဝ်းၶႂ်ႈႁႂ်ႈမၼ်း "မိူၼ်ၵဝ်ႇ" တႃႇသေႇၶႃႈ။
* **Range:** '1000-01-01' ထိုင် '9999-12-31' (ယၢဝ်းၼမ်လိူဝ်)။
* **TimeZone:** ဢမ်ႇၵွင်ႉၵၢႆႇၸွမ်း TimeZone။ သင်ၸဝ်ႈၵဝ်ႇသိမ်းမွင်း 10:00 AM... ၵူၼ်းယူႇမိူင်းထႆး ဢမ်ႇၼၼ် မိူင်းဢမေႇရိၵႃႉ ပိုတ်ႇတူၺ်းၵေႃႈ တေႁၼ်ပဵၼ် 10:00 AM မိူၼ်ၵၼ်ၶႃႈ။
* **Best for:** ဝၼ်းႁွင်ႉႁူပ်ႉထူပ်း (Appointments), ပိုၼ်းၵဝ်ႇၵဝ်ႇ (Historical data)။

### 3. TIMESTAMP (လႅၵ်ႈလၢႆႈၸွမ်း TimeZone)
ၸႂ်ႉတွၼ်ႈတႃႇ ၶေႃႈမုၼ်းဢၼ်လူဝ်ႇႁူႉဝႃႈ "မၼ်းပဵၼ်မိူဝ်ႈလႂ်တႄႉတႄႉ" ၼႂ်းၶၢဝ်းယၢမ်းလုမ်ႈၾႃႉ (UTC) ၶႃႈ။
* **Range:** '1970-01-01' ထိုင် '2038-01-19' (ဢေႇလိူဝ် DATETIME)။
* **TimeZone:** မိူဝ်ႈသိမ်း... MySQL တေပိၼ်ႇမၼ်းပဵၼ် UTC။ မိူဝ်ႈလူ (Read)... မၼ်းတေပိၼ်ႇၶိုၼ်းပဵၼ် ၶၢဝ်းၸွမ်းၼင်ႇ TimeZone ၶွမ်းႁဝ်းၶႃႈ။
* **Best for:** ဝၼ်းဢၼ် App ႁဝ်းသႂ်ႇၶေႃႈမုၼ်း (Created_at), ဝၼ်းဢၼ်မႄးၶေႃႈမုၼ်း (Updated_at)။

### 4. လွင်ႈပႅၵ်ႇပိူင်ႈ (Key Differences)

| Feature | DATETIME | TIMESTAMP |
| :--- | :--- | :--- |
| **Storage** | 8 bytes | 4 bytes |
| **Range** | 1000 - 9999 | 1970 - 2038 |
| **TimeZone** | No (Fixed) | Yes (Automatic) |
| **Usage** | Application Logic | Row Versioning/Logging |

### 5. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ Timezone Selection
* **The 2038 Limit:** တွင်းဝႆႉဝႃႈ **TIMESTAMP** တေၸႂ်ႉဢမ်ႇလႆႈယဝ်ႉ သင်ၽႅဝ်ပီႊ 2038 (Y2K38 problem)။ သင်ၸဝ်ႈၵဝ်ႇတႅမ်ႈ App ဢၼ်တေၸႂ်ႉယၢဝ်းယၢဝ်း... ၸႂ်ႉ **DATETIME** တေလီလိူဝ်ၶႃႈ။
* **Automatic Initialization:** ၼႂ်း MySQL... ႁဝ်းၸၢင်ႈတင်ႈႁႂ်ႈ TIMESTAMP မႄးမၢႆမွင်းပၼ်ႁင်းတူဝ် မိူဝ်ႈ Row ၼၼ်ႉလႅၵ်ႈလၢႆႈ (ON UPDATE CURRENT_TIMESTAMP) ၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **UTC (Coordinated Universal Time):** ၶၢဝ်းယၢမ်း Standard ဢၼ်လုမ်ႈၾႃႉၸႂ်ႉတႅၵ်ႈ။
* **TimeZone Offset:** တၢင်းပႅၵ်ႇပိူင်ႈၶၢဝ်းယၢမ်း (မိူၼ်ၼင်ႇ Myanmar ပဵၼ် UTC+6:30)။
* **Epoch Time:** ၶၢဝ်းယၢမ်းတႄႇၼပ်ႉ (1970-01-01) ဢၼ် TIMESTAMP ၸႂ်ႉ။
* **Y2K38 Problem:** ပၼ်ႁႃတူဝ်ၼပ်ႉ 32-bit ဢၼ်တေတဵမ် ၼႂ်းပီႊ 2038။
