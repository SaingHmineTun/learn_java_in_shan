# Lesson 13: "Normalization II: The 2nd and 3rd Normal Forms"

မိူဝ်ႈ Table ႁဝ်းပဵၼ် 1NF (Atomicity) ယဝ်ႉၼၼ်ႉ ႁဝ်းတေလႆႈသိုပ်ႇႁဵတ်းႁႂ်ႈမၼ်းပဵၼ် **2NF** လႄႈ **3NF** ၶႃႈ။ 
လွင်ႈၼႆႉ တေၸွႆႈႁႂ်ႈ Database ႁဝ်း ဢမ်ႇသိမ်းၶေႃႈမုၼ်းသွၼ်ႉၵၼ် (Redundancy) ၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်း "သဵၼ်ႈမၢႆၵၢၼ်သင်ႇသိုဝ်ႉ" (Orders) ၶႃႈ...
* **The Problem:** သင်ၼႂ်း Table `Orders` ၼၼ်ႉ ႁဝ်းသိမ်းတင်း "ၸိုဝ်ႈၵုၼ်ႇ"၊ "ၵႃႈၵုၼ်ႇ"၊ လႄႈ "ၸိုဝ်ႈၵူၼ်းၵႃႉ" ႁူမ်ႈၵၼ်ဝႆႉ... မိူဝ်ႈၵူၼ်းၵႃႉၵေႃႉၼိုင်ႈ သင်ႇသိုဝ်ႉ 10 ပွၵ်ႈ၊ ႁဝ်းလႆႈတႅမ်ႈၸိုဝ်ႈမၼ်းသွၼ်ႉၵၼ် 10 ထႅဝ်ၶႃႈ။
* **Normalization Rule:** ၶေႃႈမုၼ်းသင်ၵေႃႈယဝ်ႉ သင်ဢမ်ႇလၢတ်ႈၼႄလွင်ႈ **Primary Key** တင်းသဵင်ႈၼၼ်ႉ... ႁဝ်းလူဝ်ႇ "ၸႅၵ်ႇ" (Split) မၼ်းဢွၵ်ႇပဵၼ် Table မႂ်ႇၶႃႈ။



### 2. 2nd Normal Form (2NF) - ၵၢၼ်ၸႅၵ်ႇၸွမ်း Key
တႃႇတေပဵၼ် 2NF လႆႈၼၼ်ႉ Table ၼၼ်ႉလူဝ်ႇမီးပၵ်းၵၢၼ် 2 ပိူင်ၶႃႈ...
1.  **Must be in 1NF:** တေလႆႈပဵၼ် 1NF ယဝ်ႉတူဝ်ႈ။
2.  **Full Functional Dependency:** Column ၵူႈဢၼ် တေလႆႈၵွင်ႉၵၢႆႇတင်း **Primary Key** တင်းသဵင်ႈၶႃႈ။
    * *Example:* ၼႂ်း Table `Order_Items`... "ၸိုဝ်ႈၵုၼ်ႇ" မၼ်းၵွင်ႉၵၢႆႇတင်း `Product_ID` ၵူၺ်း၊ မၼ်းဢမ်ႇလႆႈၵွင်ႉၵၢႆႇတင်း `Order_ID`။ ၵွပ်ႈၼၼ် ႁဝ်းလူဝ်ႇၸႅၵ်ႇ "ၵုၼ်ႇ" ဢွၵ်ႇပဵၼ် Table `Products` ႁင်းမၼ်းၶႃႈ။

### 3. 3rd Normal Form (3NF) - ၵၢၼ်ၸႅၵ်ႇလွင်ႈၵွင်ႉၵၢႆႇၸၼ်ႉသွင်
တႃႇတေပဵၼ် 3NF လႆႈၼၼ်ႉ လူဝ်ႇမီးပၵ်းၵၢၼ် 2 ပိူင်ၶႃႈ...
1.  **Must be in 2NF:** တေလႆႈပဵၼ် 2NF ယဝ်ႉတူဝ်ႈ။
2.  **No Transitive Dependency:** Column ၼိုင်ႈဢၼ် တေဢမ်ႇလႆႈၵွင်ႉၵၢႆႇတင်း Column ထႅင်ႈဢၼ်ၼိုင်ႈ ဢၼ်ဢမ်ႇၸႂ်ႈ Primary Key ၶႃႈ။
    * *Example:* ၼႂ်း Table `Students`... "ဝဵင်း" (City) မၼ်းၵႆႉၵွင်ႉၸွမ်း "မၢႆလိၵ်ႈသူင်ႇ" (ZipCode)။ `ZipCode` သမ်ႉၵွင်ႉၸွမ်း `Student_ID`။ လွင်ႈၼႆႉ ႁွင်ႉဝႃႈ ၵွင်ႉၵၢႆႇၸၼ်ႉသွင် (Transitive)။ ႁဝ်းလူဝ်ႇၸႅၵ်ႇ `ZipCode` လႄႈ `City` ဢွၵ်ႇပဵၼ် Table `Locations` ၶႃႈ။



### 4. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ 2NF & 3NF
* **Split to Tables:** သင်ၸဝ်ႈၵဝ်ႇႁၼ် ၶေႃႈမုၼ်းဢၼ်တႅမ်ႈသွၼ်ႉၵၼ်လၢႆလၢႆပွၵ်ႈ (မိူၼ်ၼင်ႇ ၸိုဝ်ႈ Category)... ထုၵ်ႇလီၶူင်ဢွၵ်ႇပဵၼ် Table ထႅင်ႈဢၼ်ၼိုင်ႈ ယဝ်ႉၶႃႈ။
* **Foreign Keys:** မိူဝ်ႈၸႅၵ်ႇ Table ယဝ်ႉ... တွင်းဝႆႉဝႃႈ လူဝ်ႇၸႂ်ႉ **Foreign Key** တႃႇၵွင်ႉၶဝ်ၶိုၼ်းၶႃႈၼႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **2NF:** ၵၢၼ်ယႃႉလွင်ႈၵွင်ႉၵၢႆႇမၢင်တွၼ်ႈ (Partial Dependency) ၼႂ်း Composite Keys။
* **3NF:** ၵၢၼ်ယႃႉလွင်ႈၵွင်ႉၵၢႆႇၸၼ်ႉသွင် (Transitive Dependency)။
* **Redundancy:** ၵၢၼ်သိမ်းၶေႃႈမုၼ်းသွၼ်ႉၵၼ် ဢၼ်ဢမ်ႇလူဝ်ႇ။
* **Normalization Levels:** ၸၼ်ႉၵၢၼ်ၸတ်းပိူင် DB (ၵမ်ႈၼမ် ႁဝ်းႁဵတ်းထိုင် 3NF ၵေႃႈ တဵမ်ထူၼ်ႈယဝ်ႉၶႃႈ)။
