# Lesson 60: "Concurrency: Isolation Levels and Read Phenomena"

မိူဝ်ႈ User သွင်ၵေႃႉ (ဢမ်ႇၼၼ် လၢႆလၢႆၵေႃႉ) မႄးၶေႃႈမုၼ်းၼႂ်း Table လဵဝ်ၵၼ် ၼႂ်းၶၢဝ်းယၢမ်းလဵဝ်ၵၼ်ၼႆၸိုင်
မၼ်းၸၢင်ႈပဵၼ် "ပၼ်ႁႃၵၢၼ်လူ" (Read Phenomena) ဢၼ်လၢၵ်ႇလၢႆးၶႃႈ။

### 1. Read Phenomena (ပၼ်ႁႃၵၢၼ်လူၶေႃႈမုၼ်း)
1.  **Dirty Read:** လူႁၼ်ၶေႃႈမုၼ်းဢၼ် "ယုင်ႈဝႆႉ" (Uncommitted data) လုၵ်ႉတီႈ Transaction တၢင်ႇဢၼ်။ သင်မၼ်း Rollback ၶိုၼ်း... ၶေႃႈမုၼ်းဢၼ်ၸဝ်ႈၵဝ်ႇလူၼၼ်ႉ တေပဵၼ်ၶေႃႈမုၼ်း ဢၼ်ဢမ်ႇမႅၼ်ႈၶႃႈ။
2.  **Non-repeatable Read:** လူၶေႃႈမုၼ်းထႅဝ်လဵဝ်ၵၼ် သွင်ပွၵ်ႈ ၼႂ်း Transaction လဵဝ်ၵၼ် ၵူၺ်းၵႃႈ ၵႃႈၶၼ်မၼ်း "လႅၵ်ႈလၢႆႈ" ၵႂႃႇ ၵွပ်ႈမီးပိူၼ်ႈမႃး Commit ၵႄႈၵၢင်တၢင်းၶႃႈ။
3.  **Phantom Read:** လူႁၼ် "ထႅဝ်မႂ်ႇ" (New rows) ဢွၵ်ႇမႃး ၵွပ်ႈမီးပိူၼ်ႈမႃး INSERT ၶေႃႈမုၼ်းမႂ်ႇ ၶဝ်ႈၼႂ်း Range ဢၼ်ၸဝ်ႈၵဝ်ႇတိုၵ်ႉလူယူႇၶႃႈ။



### 2. Isolation Levels (ၸၼ်ႉၵၢၼ်ၸႅၵ်ႇ)
တွၼ်ႈတႃႇၵႄႈပၼ်ႁႃၸိူဝ်းၼၼ်ႉ... SQL မီးၸၼ်ႉၵၢၼ်ၸႅၵ်ႇ 4 ပိူင်ၶႃႈ:

| Isolation Level | Dirty Read | Non-repeatable | Phantom Read |
| :--- | :--- | :--- | :--- |
| **READ UNCOMMITTED** | ပဵၼ်လႆႈ | ပဵၼ်လႆႈ | ပဵၼ်လႆႈ |
| **READ COMMITTED** | ဢမ်ႇပဵၼ် | ပဵၼ်လႆႈ | ပဵၼ်လႆႈ |
| **REPEATABLE READ** | ဢမ်ႇပဵၼ် | ဢမ်ႇပဵၼ် | ပဵၼ်လႆႈ (MySQL ၵႄႈပၼ်ယဝ်ႉ) |
| **SERIALIZABLE** | ဢမ်ႇပဵၼ် | ဢမ်ႇပဵၼ် | ဢမ်ႇပဵၼ် |

* **REPEATABLE READ:** ပဵၼ် **Default** ႁင်း MySQL (InnoDB) ၶႃႈ။ မၼ်းမၼ်ႈၸႂ်ဝႃႈ သင်ၸဝ်ႈၵဝ်ႇလူပွၵ်ႈၼိုင်ႈယဝ်ႉ... လူထႅင်ႈပွၵ်ႈၵေႃႈ တေႁၼ်မိူၼ်ၵဝ်ႇတႃႇသေႇ။
* **SERIALIZABLE:** မၼ်ႈၵိုမ်းသုတ်း ၵူၺ်းၵႃႈ **ထိူင်းသုတ်း** ၵွပ်ႈမၼ်းတေ Lock Table တင်းမူတ်း ဢမ်ႇပၼ်ပိူၼ်ႈမႄးသင် တေႃႇၸဝ်ႈၵဝ်ႇ ႁဵတ်းၵၢၼ်ယဝ်ႉၶႃႈ။

### 3. ၵၢၼ်လႅၵ်ႈလၢႆႈ Isolation Level
ၸဝ်ႈၵဝ်ႇၸၢင်ႈမၵ်းမၼ်ႈလႆႈ ဢွၼ်တၢင်းတႄႇ Transaction ၶႃႈ:
```sql
SET TRANSACTION ISOLATION LEVEL READ COMMITTED;
START TRANSACTION;
```

### 4. ႁဵတ်းသင်လႄႈ လွင်ႈယႂ်ႇ?
* **Balance:** တႃႇႁႃၵႃႈၶၼ်ၵႄႈၵၢင် ၼႂ်းၵႄႈ "တၢင်းဝႆး" (Performance) လႄႈ "တၢင်းမၼ်ႈၵိုမ်း" (Consistency) ၶႃႈ။
* **Concurrency:** တွၼ်ႈတႃႇပၼ်သုၼ်ႇ ႁႂ်ႈ User လၢႆလၢႆၵေႃႉ ၸႂ်ႉ Database ႁူမ်ႈၵၼ်လႆႈ ၼႂ်းၶၢဝ်းယၢမ်းလဵဝ်ၵၼ် လူၺ်ႈဢမ်ႇမီးပၼ်ႁႃၶႃႈ။

### 5. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ Concurrency
* **Deadlock:** မိူဝ်ႈ Transaction A ပႂ်ႉ B သေ B သမ်ႉပႂ်ႉ A... မၼ်းတေ "ၶႅင်" ၵႂႃႇတင်းသွင်ၶႃႈ။ MySQL တေတတ်းပႅတ်ႈ Transaction ၼိုင်ႈဢၼ် ႁင်းမၼ်း တွၼ်ႈတႃႇၵႄႈပၼ်ႁႃၶႃႈ။
* **Locking:** Isolation Level ယႂ်ႇလိူဝ်... ၵၢၼ် Lock Table ၵေႃႈတေၼမ်လိူဝ် သေႁဵတ်းႁႂ်ႈပိူၼ်ႈ "ပႂ်ႉ" ႁိုင်လိူဝ်ၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Concurrency:** ၵၢၼ်ႁဵတ်းၵၢၼ်လၢႆလၢႆဢၼ် ၼႂ်းၶၢဝ်းယၢမ်းလဵဝ်ၵၼ်။
* **Isolation Level:** ၸၼ်ႉၵၢၼ်ၸႅၵ်ႇ Transaction ႁႂ်ႈလွတ်ႈၽေး။
* **Dirty Read:** ၵၢၼ်လူႁၼ်ၶေႃႈမုၼ်း ဢၼ်ဢမ်ႇပႆႇ Commit။
* **Deadlock:** လွင်ႈပဵၼ်မႃး ဢၼ် Transaction သွင်ဢၼ်ပႂ်ႉၵၼ် တေႃႇပေႃးၵိုတ်းလွင်ႈႁဵတ်းၵၢၼ်။
