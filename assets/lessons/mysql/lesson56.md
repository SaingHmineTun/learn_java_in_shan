# Lesson 56: "Triggers: Automating Audits and Data Sync"

**Trigger** မိူၼ်ၼင်ႇ "ပလိၵ်ႈ" ဢမ်ႇၼၼ် "ၸၢၵ်ႈပႂ်ႉယၢမ်း" ဢၼ်ႁဝ်းတင်ႈဝႆႉၼႂ်း Table ၶႃႈ။ 
မၼ်းတေဢမ်ႇမီးၽႂ် CALL မၼ်း၊ ၵူၺ်းၵႃႈ မၼ်းတေတူၺ်းဝႃႈ "မီးၽႂ်မႃးမႄးၶေႃႈမုၼ်း?" သင်မီး... မၼ်းတေႁဵတ်းၵၢၼ် ၸွမ်းၵမ်းလဵဝ်ၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်း "ၵၢၼ်ပိုတ်ႇၽၵ်းတူႁင်းမၼ်း" (Auto Door) ၶႃႈ...
* **Event:** မီးၵူၼ်းယၢင်ႈၶဝ်ႈမႃး (INSERT)။
* **Trigger:** ၸၢၵ်ႈပႂ်ႉၽၵ်းတူႁၼ် (Sensor) သေ ပိုတ်ႇၽၵ်းတူပၼ်ႁင်းမၼ်း ၵမ်းလဵဝ်ၶႃႈ။



### 2. ၶၢဝ်းယၢမ်းၵၢၼ်ႁဵတ်းၵၢၼ် (Timing)
ႁဝ်းၸၢင်ႈမၵ်းမၼ်ႈလႆႈဝႃႈ ၶႂ်ႈႁႂ်ႈ Trigger ႁဵတ်းၵၢၼ်မိူဝ်ႈလႂ်:
* **BEFORE:** ႁဵတ်းၵၢၼ် "ဢွၼ်တၢင်း" ၶေႃႈမုၼ်းတေထုၵ်ႇသိမ်း (ၵႆႉၸႂ်ႉတႃႇ Validation ဢမ်ႇၼၼ် မႄးၶေႃႈမုၼ်းႁႂ်ႈမၢၼ်ႇ)။
* **AFTER:** ႁဵတ်းၵၢၼ် "ဝၢႆးသေ" ၶေႃႈမုၼ်းသိမ်းယဝ်ႉတူဝ်ႈ (ၵႆႉၸႂ်ႉတႃႇ Audit Logs ဢမ်ႇၼၼ် Sync ၶေႃႈမုၼ်းၸူး Table တၢင်ႇဢၼ်)။

### 3. Syntax ၵၢၼ်သၢင်ႈ Trigger
ႁဝ်းတေလႆႈမၵ်းမၼ်ႈ **Timing**, **Event**, လႄႈ **Table** ၶႃႈ။

**တူဝ်ယင်ႇ:** ၵၢၼ်ႁဵတ်း **Audit Log** (မၢႆဝႆႉဝႃႈ မႄးငိုၼ်းၵႂႃႇ ၵႃႈႁိုဝ်)။
```sql
CREATE TRIGGER before_balance_update
BEFORE UPDATE ON accounts
FOR EACH ROW
BEGIN
    -- သိမ်းၶေႃႈမုၼ်းၵဝ်ႇ ဝႆႉၼႂ်း Table တၢင်ႇဢၼ် တႃႇၵူတ်ႇထတ်း
    INSERT INTO audit_logs(account_id, old_balance, new_balance, changed_at)
    VALUES (OLD.id, OLD.balance, NEW.balance, NOW());
END;
```
* **OLD:** မၢႆထိုင်ၵႃႈၶၼ် "ၵဝ်ႇ" ဢၼ်မီးဝႆႉယူႇယဝ်ႉ။
* **NEW:** မၢႆထိုင်ၵႃႈၶၼ် "မႂ်ႇ" ဢၼ်တေမႃးလႅၵ်ႈတႅၼ်း။

### 4. ႁဵတ်းသင်လႄႈ လူဝ်ႇၸႂ်ႉ Trigger?
1.  **Auditing (ၵၢၼ်ၵူတ်ႇထတ်း):** တႃႇႁူႉဝႃႈ ၽႂ်မႃးလႅၵ်ႈလၢႆႈၶေႃႈမုၼ်း မိူဝ်ႈလႂ်။
2.  **Data Synchronization:** မိူဝ်ႈ UPDATE Table ဢၼ်ၼိုင်ႈ ႁႂ်ႈမၼ်းၵႂႃႇႁဵတ်း UPDATE Table တၢင်ႇဢၼ်ၸွမ်းႁင်းမၼ်း (မိူၼ်ၼင်ႇ တတ်း Stock ၵမ်းလဵဝ် မိူဝ်ႈမီး Order)။
3.  **Complex Validation:** ၸႅတ်ႈတူၺ်းၶေႃႈမုၼ်း ဢၼ်ယုင်ႈယၢင်ႈ ဢၼ် `CHECK` constraint ႁဵတ်းဢမ်ႇလႆႈ။

### 5. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ Triggers
* **Performance:** Trigger ၼမ်လူၼ်ႉၵႂႃႇ ၸၢင်ႈႁဵတ်းႁႂ်ႈၵၢၼ် INSERT/UPDATE **ထိူင်းလိူဝ်ၵဝ်ႇ** ၵွပ်ႈမၼ်းလႆႈႁဵတ်းၵၢၼ်ၼမ်လိူဝ်ၵဝ်ႇၶႃႈ။
* **Hidden Logic:** Trigger ႁဵတ်းၵၢၼ် ၽၢႆႇလင်။ မၢင်ပွၵ်ႈ Developer ၸၢင်ႈဝူၼ်ႉၵႂၢင် (Confused) ဝႃႈ "ႁဵတ်းသင်ၶေႃႈမုၼ်းၼႆႉ လႅၵ်ႈလၢႆႈၵႂႃႇႁင်းမၼ်း?" သင်ၶဝ်ဢမ်ႇႁူႉဝႃႈမီး Trigger ဝႆႉၶႃႈ။
* **Debugging:** ၵၢၼ်ၵႄႈပၼ်ႁႃ (Debug) Trigger ယၢပ်ႇလိူဝ် Code ၶႃႈၼႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Trigger:** ၸုမ်း Code ဢၼ်ႁဵတ်းၵၢၼ်ႁင်းမၼ်း မိူဝ်ႈမီး Event ၼႂ်း Table။
* **Audit Trail:** ပိုၼ်းၵၢၼ်လႅၵ်ႈလၢႆႈၶေႃႈမုၼ်း (History logs)။
* **NEW/OLD Keywords:** တူဝ်တႅၼ်းၵႃႈၶၼ် မႂ်ႇ လႄႈ ၵဝ်ႇ ၼႂ်း Trigger။
* **Data Integrity:** ၵၢၼ်ဝႆႉၶေႃႈမုၼ်း ႁႂ်ႈမၢၼ်ႇမႅၼ်ႈ လႄႈ တႅတ်ႈတေႃးယူႇတႃႇသေႇ။
