# Lesson 2: "The SQL Language: DDL, DML, DQL, and DCL"

**`SQL (Structured Query Language)`** ၼႆႉ ပဵၼ်ၽႃႇသႃႇဢၼ်ႁဝ်းၸႂ်ႉ "လၢတ်ႈတေႃႇ" Database ၶႃႈ။ 
မၼ်းမိူၼ်ၼင်ႇ ႁဝ်းၸႂ်ႉလိၵ်ႈတႆး လၢတ်ႈတေႃႇၵၼ်ၼၼ်ႉယဝ်ႉၶႃႈ။ 
ၼႂ်း SQL ၼႆႉ ႁဝ်းၸႅၵ်ႇ ၵၢၼ်ၸႂ်ႉတိုဝ်းမၼ်းဝႆႉပဵၼ် 4 ၸုမ်းလူင်ၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းတေႁဵတ်း "ႁိူၼ်း" ၼိုင်ႈလင်...
* **DDL:** ပဵၼ်ၵၢၼ် "ပၵ်းသဝ်/ၵေႃႇၾႃ" (သၢင်ႈပိူင်ႁိူၼ်း)။
* **DML:** ပဵၼ်ၵၢၼ် "ဢဝ်ၶိူင်ႈၶူဝ်းၶဝ်ႈဝႆႉ" ၼႂ်းႁိူၼ်း (သႂ်ႇၶေႃႈမုၼ်း)။
* **DQL:** ပဵၼ်ၵၢၼ် "ၵႂႃႇထၢမ်တူၺ်း" ဝႃႈၶိူင်ႈၶူဝ်းမီးတီႈလႂ် (သွၵ်ႈႁႃၶေႃႈမုၼ်း)။
* **DCL:** ပဵၼ်ၵၢၼ် "ပၼ်သေႃးႁိူၼ်း" ဝႃႈၽႂ်ၶဝ်ႈလႆႈ/ဢမ်ႇလႆႈ (မၵ်းမၼ်ႈသုၼ်ႇၸႂ်ႉ)။

### 2. ၸႅၵ်ႇၸုမ်းၶေႃႈၵႂၢမ်း SQL (SQL Sub-languages)

#### **1. DDL (Data Definition Language)** - တႃႇသၢင်ႈ/မႄး ပိူင်သၢင်ႈ
ၸႂ်ႉတႃႇ "သၢင်ႈ" (Create) ဢမ်ႇၼၼ် "ထွၼ်" (Delete) Table ၶႃႈ။
* **CREATE:** သၢင်ႈ Table ဢမ်ႇၼၼ် Database။
* **ALTER:** မႄးလႅၵ်ႈလၢႆႈ ပိူင်သၢင်ႈ Table (မိူၼ်ၼင်ႇ ထႅမ် Column)။
* **DROP:** ထွၼ် Table ဢမ်ႇၼၼ် Database ပႅတ်ႈတင်းသဵင်ႈ။

#### **2. DML (Data Manipulation Language)** - တႃႇၸတ်းၵၢၼ် ၶေႃႈမုၼ်း
ၸႂ်ႉတႃႇ "သႂ်ႇ" ဢမ်ႇၼၼ် "မႄး" ၶေႃႈမုၼ်း ဢၼ်ယူႇၼႂ်း Table ၶႃႈ။
* **INSERT:** သႂ်ႇၶေႃႈမုၼ်းမႂ်ႇ။
* **UPDATE:** မႄးၶေႃႈမုၼ်းၵဝ်ႇ။
* **DELETE:** ထွၼ်ၶေႃႈမုၼ်း (ၵူၺ်းၵႃႈ Table တေတိုၵ်ႉမီးဝႆႉယူႇ)။

#### **3. DQL (Data Query Language)** - တႃႇသွၵ်ႈႁႃ ၶေႃႈမုၼ်း
ၼႆႉပဵၼ် ဢၼ်ႁဝ်း တေလႆႈၸႂ်ႉၼမ်လိူဝ်ပိူၼ်ႈၶႃႈ။
* **SELECT:** ၸႂ်ႉတႃႇထၢမ်ဢဝ် ၶေႃႈမုၼ်းဢွၵ်ႇမႃးၼႄ။

#### **4. DCL (Data Control Language)** - တႃႇၵုမ်းၵမ် သုၼ်ႇၸႂ်ႉ
ၸႂ်ႉတႃႇမၵ်းမၼ်ႈ လွင်ႈလွတ်ႈၽေး (Security) ၶႃႈ။
* **GRANT:** ပၼ်သုၼ်ႇၸႂ်ႉ (Permission)။
* **REVOKE:** ၶိုၼ်းဢဝ်သုၼ်ႇၸႂ်ႉပႅတ်ႈ။

### 3. တူဝ်ယင်ႇ Code: ၵၢၼ်ၸႂ်ႉ SQL ၵူႈပိူင်
```sql
-- [DDL] သၢင်ႈ Table မႂ်ႇ
CREATE TABLE Students (
    ID INT PRIMARY KEY,
    Name VARCHAR(100)
);

-- [DML] သႂ်ႇၶေႃႈမုၼ်းလုၵ်ႈႁဵၼ်း
INSERT INTO Students (ID, Name) VALUES (1, 'ၸၢႆးသႅင်');

-- [DQL] ထၢမ်တူၺ်း သဵၼ်ႈမၢႆလုၵ်ႈႁဵၼ်း
SELECT * FROM Students;

-- [DCL] ပၼ်သုၼ်ႇ 'User1' ႁႂ်ႈလူလႆႈ Table ၼႆႉ
GRANT SELECT ON Students TO 'User1';
```

### 4. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ SQL
* **Query:** ၶေႃႈၵႂၢမ်းဢၼ်ႁဝ်းသူင်ႇၵႂႃႇထၢမ် Database။
* **Statement:** ၶေႃႈၵႂၢမ်း SQL ဢၼ်တဵမ်ထူၼ်ႈ ၼိုင်ႈထႅဝ်။
* **Semi-colon (;):** ၵႆႉၸႂ်ႉတႃႇမၢႆတွင်းဝႃႈ ၶေႃႈၵႂၢမ်း SQL ၼၼ်ႉ သုတ်းယဝ်ႉ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **DDL:** ၵၢၼ်မၵ်းမၼ်ႈ ပိူင်သၢင်ႈၶေႃႈမုၼ်း။
* **DML:** ၵၢၼ်ၸတ်းၵၢၼ် ၶေႃႈမုၼ်းၼႂ်း Table။
* **DQL:** ၵၢၼ်သွၵ်ႈႁႃ ၶေႃႈမုၼ်း (Queries)။
* **DCL:** ၵၢၼ်ၵုမ်းၵမ်း သုၼ်ႇၶဝ်ႈၸႂ်ႉၶေႃႈမုၼ်း။
* **Declarative:** SQL ပဵၼ်ၽႃႇသႃႇဢၼ်ႁဝ်း "လၢတ်ႈဝႃႈ ၶႂ်ႈလႆႈသင်" (What we want)၊ ဢမ်ႇၸႂ်ႈလၢတ်ႈဝႃႈ "ႁဵတ်းႁိုဝ်လႆႈမႃး" (How to do)။
