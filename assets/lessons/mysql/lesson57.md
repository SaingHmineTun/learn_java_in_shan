# Lesson 57: "Events: Scheduling Automated Database Tasks"

**MySQL Event Scheduler** မၢႆထိုင် ၵၢၼ်မၵ်းမၼ်ႈ ႁႂ်ႈ Database Server ႁဵတ်းၵၢၼ် ၼႂ်းၶၢဝ်းယၢမ်း ဢၼ်မၵ်းမၼ်ႈဝႆႉ တႃႇသေႇၼႆၶႃႈ။
(မိူၼ်ၼင် 12:00 AM ၵူႈဝၼ်း) ၶႃႈ။
ဢေႃႈၶႃႈ... ပွင်ႇၸႂ်ယဝ်ႉၶႃႈ! တူဝ်ယင်ႇ "ၼႃႇလိၵ်ႇပႂ်ႉပၼ်ႇ" ၼၼ်ႉ မၢင်ပွၵ်ႈမၼ်းယင်းတိုၵ်ႉႁၼ်ႁၢင်ႈယၢပ်ႇယူႇ။ ႁဝ်းမႃးဝူၼ်ႉတူၺ်း တူဝ်ယင်ႇ **"ၵၢၼ်လၢင်ႉႁိူၼ်း"** ၼႆႉ တေၸႅၼ်ႇလႅင်းလိူဝ်ၶႃႈ။

---

### 1. ပိူင်သၢင်ႈ (The Concept) - တူဝ်ယင်ႇ "ၵၢၼ်လၢင်ႉႁိူၼ်း"

ဝူၼ်ႉတူၺ်းဝႃႈ ၸဝ်ႈၵဝ်ႇပဵၼ်ၸဝ်ႈၶွင်ႁိူၼ်းလူင်ဢၼ်ၼိုင်ႈသေ ၶႂ်ႈႁႂ်ႈႁိူၼ်းသႅၼ်ႈသႂ်ယူႇတႃႇသေႇၶႃႈ...

* **Trigger (ပႂ်ႉတူၺ်း "ၵၢၼ်ႁဵတ်း"):**
  မိူၼ်ၼင်ႇ ၸဝ်ႈၵဝ်ႇသင်ႇဝႆႉဝႃႈ **"ပေႃးမီးၵူၼ်း ႁႂ်ႈၼမ်ႉတူၵ်းသႂ်ႇလိၼ် (Event) ၵႂႃႇၸဵတ်ႉပႅတ်ႈ (Action) ၵမ်းလဵဝ်"** ၼႆၶႃႈ။
    * မၼ်းတေဢမ်ႇတူၺ်းၶၢဝ်းယၢမ်း၊ မၼ်းတူၺ်းဝႃႈ "မီးလွင်ႈတူၵ်းႁူင်ႈ" ႁႃႉ? ပေႃးမီးၵေႃႈ ႁဵတ်းၵၢၼ်ၵမ်းလဵဝ်ၶႃႈ။

* **Event (ပႂ်ႉတူၺ်း "ၶၢဝ်းယၢမ်း"):**
  မိူၼ်ၼင်ႇ ၸဝ်ႈၵဝ်ႇမၵ်းမၼ်ႈပၵ်းၵၢၼ်ဝႆႉဝႃႈ **"ၵူႈဝၼ်းသဝ် (Time)... ႁႂ်ႈမႃးလၢင်ႉႁိူၼ်းတင်းလင် (Action)"** ၶႃႈ။
    * မၼ်းဢမ်ႇတူၺ်းဝႃႈ ႁိူၼ်းတေသုၵ်ႉ ႁိုဝ် သႅၼ်ႈ။ ပေႃးထိုင် "ဝၼ်းသဝ်" မႃးၵေႃႈ... မၼ်းတေႁဵတ်းၵၢၼ်ၸွမ်းပၵ်းၵၢၼ် ဢၼ်တင်ႈဝႆႉၼၼ်ႉၵမ်းလဵဝ်ၶႃႈ။



### 2. ၵၢၼ်ပိုတ်ႇ Event Scheduler
ဢွၼ်တၢင်းတေၸႂ်ႉလႆႈ... ၸဝ်ႈၵဝ်ႇတေလႆႈမၼ်ႈၸႂ်ဝႃႈ Database Server ၸဝ်ႈၵဝ်ႇ ပိုတ်ႇ Scheduler ဝႆႉယဝ်ႉၶႃႈ။
```sql
SET GLOBAL event_scheduler = ON;
```

### 3. Syntax ၵၢၼ်သၢင်ႈ Event
ႁဝ်းၸၢင်ႈသၢင်ႈလႆႈ 2 ပိူင်ၶႃႈ:
* **One-time Event:** ႁဵတ်းၵၢၼ် "ပွၵ်ႈလဵဝ်" ယဝ်ႉၵေႃႈယဝ်ႉၵႂႃႇ။
* **Recurring Event:** ႁဵတ်းၵၢၼ် ၸွမ်းၶၢဝ်းယၢမ်း တႃႇသေႇ (မိူၼ်ၼင်ႇ ၵူႈလိူၼ်၊ ၵူႈဝၼ်း)။

**တူဝ်ယင်ႇ:** ၵၢၼ်လၢင်ႉ "ၶေႃႈမုၼ်းၵဝ်ႇ" (Cleanup) ၵူႈဝၼ်း။
```sql
CREATE EVENT daily_cleanup
ON SCHEDULE EVERY 1 DAY
STARTS '2026-04-05 00:00:00'
DO
    DELETE FROM logs WHERE log_date < NOW() - INTERVAL 30 DAY;
```
* **ON SCHEDULE:** မၵ်းမၼ်ႈဝႃႈ ၶႂ်ႈႁႂ်ႈႁဵတ်းၵၢၼ် မိူဝ်ႈလႂ်/ထိုင်လႂ်။
* **DO:** ၶေႃႈၵႂၢမ်း SQL ဢၼ်ၶႂ်ႈႁႂ်ႈမၼ်းႁဵတ်းၵၢၼ် (ၸႂ်ႉ `CALL` Stored Procedure ၵေႃႈလႆႈၶႃႈ)။

### 4. ႁဵတ်းသင်လႄႈ လူဝ်ႇၸႂ်ႉ Event?
1.  **Maintenance (ၵၢၼ်လုမ်းလႃး):** လၢင်ႉၶေႃႈမုၼ်းၸူဝ်ႈၶၢဝ်း (Temporary data) ဢမ်ႇၼၼ် Logs ဢၼ်ၵဝ်ႇႁႅင်း။
2.  **Report Generation:** ႁႂ်ႈမၼ်းၼပ်ႉသွၼ်ႇႁူမ်ႈငိုၼ်းၶၢႆ ၵူႈၶမ်ႈ သေသိမ်းဝႆႉၼႂ်း Table Report ႁင်းမၼ်း။
3.  **Data Sync:** သူင်ႇၶေႃႈမုၼ်းၵႂႃႇၸူး Server တၢင်ႇဢၼ် ၸွမ်းၶၢဝ်းယၢမ်းဢၼ်တႅတ်ႈတေႃး။

### 5. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ Events
* **Server Dependence:** သင် Database Server ပိၵ်ႉဝႆႉ... Event တေဢမ်ႇႁဵတ်းၵၢၼ်ၶႃႈၼႃႈ။
* **Overlapping:** သင် Event ၸဝ်ႈၵဝ်ႇႁဵတ်းၵၢၼ်ယၢဝ်းလိူဝ် ၶၢဝ်းယၢမ်းဢၼ်ၸဝ်ႈၵဝ်ႇတင်ႈဝႆႉ (မိူၼ်ၼင်ႇ ႁဵတ်းၵၢၼ် 2 မိၼတ်ႉ ၵူၺ်းၵႃႈ တင်ႈဝႆႉႁႂ်ႈတိုၼ်ႇ ၵူႈ 1 မိၼတ်ႉ)... မၼ်းၸၢင်ႈႁဵတ်းႁႂ်ႈ Server ၼၵ်းႁႅင်းၶႃႈ။
* **Privileges:** ၸဝ်ႈၵဝ်ႇလူဝ်ႇမီး Permission **`EVENT`** ၸင်ႇတေၸၢင်ႈသၢင်ႈ လႄႈ မႄးမၼ်းလႆႈၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Event Scheduler:** ၶိူင်ႈမိုဝ်းၸတ်းၵၢၼ် ၶၢဝ်းယၢမ်းၼႂ်း MySQL။
* **Recurring Event:** ၵၢၼ်ႁဵတ်ၵၢၼ် ႁဵတ်းယဝ်ႉႁဵတ်းထႅင်ႈ (မိူၼ်ၼင်ႇ Every Week)။
* **Interval:** ၶၢဝ်းယၢမ်းၵႄႈၵၢင် (မိူၼ်ၼင်ႇ 1 DAY, 1 HOUR)။
* **Cleanup Task:** ၵၢၼ်လၢင်ႉၶေႃႈမုၼ်းဢၼ်ဢမ်ႇလူဝ်ႇၸႂ်ႉယဝ်ႉ။
