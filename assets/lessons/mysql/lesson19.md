# Lesson 19: "Temporal Data I: DATE, TIME, and YEAR Types"

မိူဝ်ႈႁဝ်းသိမ်း "ဝၼ်းၵိူတ်ႇ"၊ "ၶၢဝ်းယၢမ်းၶဝ်ႈႁဵၼ်း" ဢမ်ႇၼၼ် "ပီႊဢၼ်ဢွၵ်ႇပပ်ႉလိၵ်ႈ" ၼၼ်ႉ... ႁဝ်းလူဝ်ႇလိူၵ်ႈၸႂ်ႉ Data Type တွၼ်ႈတႃႇယွမ်းတီႈယူႇ (Storage) လႄႈ ႁႂ်ႈသွၵ်ႈႁႃငၢႆႈၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်း "ပၵ်းယဵမ်ႈဝၼ်း" လႄႈ "ၶၢဝ်းယၢမ်း" ၶႃႈ...
* **DATE:** သိမ်း "ဝၼ်း-လိူၼ်-ပီႊ" ၵူၺ်း (ဢမ်ႇပႃးၸူဝ်ႈမွင်း)။
* **TIME:** သိမ်း "မွင်း-မိၼိတ်ႉ-သႅၵ်ႉၶၼ်ႉ" ၵူၺ်း (ဢမ်ႇပႃးဝၼ်း)။
* **YEAR:** သိမ်း "ပီႊ" ၵူၺ်း (4 တူဝ်ၼပ်ႉ)။



### 2. DATE (ဝၼ်း-လိူၼ်-ပီႊ)
ၸႂ်ႉတွၼ်ႈတႃႇ ၶေႃႈမုၼ်းဢၼ်လူဝ်ႇ "ဝၼ်း" ၵူၺ်းၶႃႈ။
* **Format:** `YYYY-MM-DD` (တူဝ်ယင်ႇ: `2026-04-03`)။
* **Range:** '1000-01-01' ထိုင် '9999-12-31'။
* **Best for:** ဝၼ်းၵိူတ်ႇ (Birthdate), ဝၼ်းၶဝ်ႈႁဵၼ်း (Enrollment Date), ဝၼ်းပွႆး (Event Date)။

### 3. TIME (ၶၢဝ်းယၢမ်း)
ၸႂ်ႉတွၼ်ႈတႃႇ သိမ်းၶၢဝ်းယၢမ်း ဢမ်ႇၼၼ် "ၶၢဝ်းတၢင်း" (Duration) ၶႃႈ။
* **Format:** `HH:MM:SS` (တူဝ်ယင်ႇ: `14:30:00`)။
* **Range:** '-838:59:59' ထိုင် '838:59:59' (မၼ်းသိမ်းလႆႈတင်း ၶၢဝ်းယၢမ်းၼႂ်းဝၼ်း လႄႈ ၶၢဝ်းတၢင်းဢၼ်ယၢဝ်းလိူဝ် 24 မွင်းၶႃႈ)။
* **Best for:** ၶၢဝ်းယၢမ်းတႄႇႁဵၼ်း (Start Time), ၶၢဝ်းတၢင်းႁဵတ်းၵၢၼ် (Duration/Interval)။

### 4. YEAR (ပီႊ)
ၸႂ်ႉတွၼ်ႈတႃႇ သိမ်း "ပီႊ" ဢၼ်မီး တူဝ်ၼပ်ႉ 4 တူဝ် ႁႂ်ႈယွမ်းတီႈယူႇသုတ်း (ၸႂ်ႉ 1 byte ၵူၺ်း) ၶႃႈ။
* **Format:** `YYYY` (တူဝ်ယင်ႇ: `2026`)။
* **Range:** 1901 ထိုင် 2155။
* **Best for:** ပီႊဢွၵ်ႇၵႃး (Car Model Year), ပီႊဢွၵ်ႇပပ်ႉလိၵ်ႈ (Release Year)။

### 5. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ Temporal Selection
* **Standard Format:** ၼႂ်း MySQL... ႁဝ်း**တေလႆႈတႅမ်ႈ** ဝၼ်းပဵၼ် `YYYY-MM-DD` တႃႇသေႇၶႃႈ။ ႁဝ်းဢမ်ႇၸၢင်ႈတႅမ်ႈ `DD-MM-YYYY` ၵမ်းလဵဝ် ၼႂ်း Query ၶႃႈ (ၵူၺ်းၵႃႈ ႁဝ်းၸၢင်ႈမႄးႁၢင်ႈမၼ်း မိူဝ်ႈ `SELECT` ဢွၵ်ႇမႃးၼႄၶႃႈ)။
* **Zero Dates:** သင်ၶေႃႈမုၼ်းၽိတ်း... MySQL ၸၢင်ႈသိမ်းပဵၼ် `0000-00-00` (သင်ဢမ်ႇလႆႈတင်ႈ Strict Mode ဝႆႉၶႃႈ)။

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Temporal Data:** ၶေႃႈမုၼ်းဢၼ်ၵွင်ႉၵၢႆႇတင်း ၶၢဝ်းယၢမ်း။
* **ISO 8601:** ပိူင် Standard ၵၢၼ်တႅမ်ႈဝၼ်း (YYYY-MM-DD) ဢၼ် MySQL ၸႂ်ႉ။
* **Literal:** ၵႃႈၶၼ်ဢၼ်ႁဝ်းတႅမ်ႈသႂ်ႇၼႂ်း Query (မိူၼ်ၼင်ႇ '2026-01-01')။
* **Interval:** ၶၢဝ်းတၢင်း (ၶၢဝ်းယၢမ်းၼႂ်းၵႄႈ ဝၼ်းသွင်ဢၼ်)။
