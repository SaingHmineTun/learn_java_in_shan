# Lesson 47: "Grouping Logic: The GROUP BY Clause"

**GROUP BY** မၢႆထိုင်ၵၢၼ်ဢဝ်ထႅဝ် (Rows) ဢၼ်မီးၶေႃႈမုၼ်းမိူၼ်ၵၼ် မႃးႁူမ်ႈၵၼ်ပဵၼ် "ၸုမ်း" (Groups) သေၸင်ႇၼပ်ႉသွၼ်ႇ (Aggregate) ၼႂ်းၸုမ်းၼၼ်ႉၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်း "ၵၢၼ်ၸႅၵ်ႇမၢၵ်ႇမႆႉ" ၶႃႈ...
* ၸဝ်ႈၵဝ်ႇမီး မၢၵ်ႇၵဵင်၊ မၢၵ်ႇၸွၵ်း၊ မၢၵ်ႇၵႃႇ ႁူမ်ႈၵၼ်ဝႆႉၼႂ်းသႃႉ ၼႆၶႃႈ။
* **GROUP BY:** ၸွႆႈၸႅၵ်ႇ "မၢၵ်ႇၵဵင်" ပဵၼ်ၵွင်ၼိုင်ႈ၊ "မၢၵ်ႇၸွၵ်း" ဝႆႉပဵၼ်ၵွင်ၼိုင်ႈ။
* **Aggregate:** ၸဝ်ႈၵဝ်ႇၸင်ႇ "ၼပ်ႉ" (Count) ဝႃႈ ၼႂ်းၵွင်မၢၵ်ႇၵဵင်ၼႆ မီးလၢႆလုၵ်ႈၼႆၶႃႈ။



### 2. Syntax ၵၢၼ်တႅမ်ႈ GROUP BY
ႁဝ်းတေလႆႈဝႆႉ `GROUP BY` ၽၢႆႇလင် `FROM` ဢမ်ႇၼၼ် `WHERE` ၶႃႈ။

```sql
SELECT city, COUNT(*) AS total_students
FROM students
GROUP BY city;
```
* **SELECT city:** ၼႄၸိုဝ်ႈဝဵင်း။
* **COUNT(*):** ၼပ်ႉတၢင်းၼမ်လုၵ်ႈႁဵၼ်း ဢၼ်မီးဝႆႉ **ၼႂ်းဝဵင်းၼၼ်ႉ**။
* **GROUP BY city:** ၸႅၵ်ႇဢွၵ်ႇၸွမ်းၼင်ႇ "ၸိုဝ်ႈဝဵင်း"။

### 3. GROUP BY ႁူမ်ႈတင်း JOIN
ၼႆႉပဵၼ်လၢႆး ဢၼ်ႁဝ်းလႆႈၸႂ်ႉၼမ်လိူဝ်ပိူၼ်ႈ တွၼ်ႈတႃႇႁဵတ်း Report ၶႃႈ။

```sql
-- ႁႃတၢင်းၼမ်လုၵ်ႈႁဵၼ်း ၸႅၵ်ႇၸွမ်း "ၸိုဝ်ႈႁွင်ႈႁဵၼ်း"
SELECT c.class_name, COUNT(s.student_id) AS student_count
FROM classes c
LEFT JOIN students s ON c.class_id = s.class_id
GROUP BY c.class_name;
```

### 4. ၵၢၼ်ၸႅၵ်ႇလၢႆၸၼ်ႉ (Multiple Columns)
ႁဝ်းၸၢင်ႈၸႅၵ်ႇၸုမ်း လၢႆလၢႆ Column ႁူမ်ႈၵၼ်ၵေႃႈလႆႈၶႃႈ။

```sql
-- ႁႃႁူမ်ႈငိုၼ်းၶၢႆ ၸႅၵ်ႇၸွမ်း "ဝဵင်း" လႄႈ "သီၵုၼ်ႇ"
SELECT city, color, SUM(price)
FROM sales
GROUP BY city, color;
```

### 5. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ Grouping
* **SELECT Rule:** တွင်းဝႆႉတႃႇသေႇဝႃႈ Column ဢၼ်ၸဝ်ႈၵဝ်ႇ SELECT မႃးၼႄ (ဢၼ်ဢမ်ႇၸႂ်ႈ Aggregate Function) ၼၼ်ႉ... **တေလႆႈမီးယူႇၼႂ်း GROUP BY** တႃႇသေႇၶႃႈ။ သင်ဢမ်ႇၼၼ် MySQL တေဢမ်ႇႁူႉဝႃႈ တေဢဝ်ၶေႃႈမုၼ်းထႅဝ်လႂ်မႃးၼႄၶႃႈ။
* **WHERE vs GROUP BY:** ႁဝ်းၸႂ်ႉ `WHERE` တွၼ်ႈတႃႇတတ်း (Filter) ထႅဝ်ဢွၼ်တၢင်းတေၸႅၵ်ႇၸုမ်း။ (တူဝ်ယင်ႇ: ၸႅၵ်ႇၸုမ်းလုၵ်ႈႁဵၼ်း 'ၵူၺ်းၵေႃႉဢၼ်ဢႃႇယု > 18')။
* **Sorting:** ၸဝ်ႈၵဝ်ႇၸၢင်ႈၸႂ်ႉ `ORDER BY` ၽၢႆႇလင် `GROUP BY` တႃႇၸတ်းလႅၼ်ႇၸုမ်းၸိူဝ်းၼၼ်ႉ (မိူၼ်ၼင်ႇ ဢဝ်ဝဵင်းဢၼ်မီးလုၵ်ႈႁဵၼ်းၼမ်သုတ်း ဝႆႉၼိူဝ်သုတ်း) ၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Grouping:** ၵၢၼ်ၸႅၵ်ႇၶေႃႈမုၼ်းပဵၼ်ၸုမ်း ၸွမ်းၼင်ႇၵႃႈၶၼ်ဢၼ်မိူၼ်ၵၼ်။
* **Aggregate Function:** Function ဢၼ်ၸႂ်ႉၼပ်ႉသွၼ်ႇၼႂ်းၸုမ်း (SUM, COUNT, AVG, etc.)။
