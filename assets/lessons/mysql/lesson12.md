# Lesson 12: "Normalization I: The 1st Normal Form (Atomicity)"

**Normalization** ပဵၼ်ၵၢၼ် "ၵေႃႇသၢင်ႈ" ပိူင်သၢင်ႈ Table ႁဝ်း ႁႂ်ႈယွမ်း လွင်ႈသိမ်းၶေႃႈမုၼ်းသွၼ်ႉၵၼ် (Redundancy) လႄႈ 
ႁႂ်ႈၵႄႈလိတ်ႈပၼ်ႁႃ ၶေႃႈမုၼ်းၽိတ်းၶႃႈ။ တွၼ်ႈဢွၼ်တၢင်းသုတ်း ႁဝ်းလူဝ်ႇႁဵတ်းႁႂ်ႈမၼ်းပဵၼ် **1st Normal Form (1NF)** ၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်း "သဵၼ်ႈမၢႆၵုၼ်ႇ" ၼႂ်း Excel ၼိုင်ႈထႅဝ်ၶႃႈ...
* **Un-normalized:** ၼႂ်း Column `Colors` ၼၼ်ႉ ႁဝ်းတႅမ်ႈဝႆႉဝႃႈ "Red, Blue, Green" ႁူမ်ႈၵၼ်ဝႆႉၼႂ်းလုၵ်း (Cell) လဵဝ်ၵူၺ်း။
* **The Problem:** သင်ႁဝ်းၶႂ်ႈသွၵ်ႈႁႃ "ၵုၼ်ႇသီလႅင်" (Red) ၵူၺ်းၼႆ Database တေသွၵ်ႈႁႃယၢပ်ႇ လႄႈ တေႁဵတ်းၵၢၼ် ထိူင်းၶႃႈ။
* **1NF Rule:** ၶေႃႈမုၼ်းၼႂ်းၵူႈလုၵ်း (Cell) တေလႆႈပဵၼ် **Atomicity** (မီးၵႃႈၶၼ်ဢၼ်လဵဝ်ၵူၺ်း)။



### 2. ပၵ်းၵၢၼ် 3 ပိူင် တွၼ်ႈတႃႇ 1NF
တႃႇတေမၵ်းမၼ်ႈဝႃႈ Table ႁဝ်းပဵၼ် 1NF ယဝ်ႉႁႃႉၼၼ်ႉ လူဝ်ႇတူၺ်းၼင်ႇၼႆၶႃႈ...
1.  **Atomic Values:** ၶေႃႈမုၼ်းၼႂ်း Column ၼိုင်ႈဢၼ် တေဢမ်ႇလႆႈမီးလၢႆလၢႆၵႃႈၶၼ် (No multi-valued attributes)။
2.  **Unique Column Names:** ၸိုဝ်ႈ Column တေဢမ်ႇလႆႈမိူၼ်ၵၼ်။
3.  **Primary Key:** Table ၼၼ်ႉ တေလႆႈမီး Primary Key တွၼ်ႈတႃႇၸႅၵ်ႇ Row ႁႂ်ႈပႅၵ်ႇပိူၼ်ႈၶႃႈ။

### 3. တူဝ်ယင်ႇ ၵၢၼ်မႄး 1NF
ဝူၼ်ႉတူၺ်း Table **`Student_Hobbies`** ၶႃႈ...

**Before (Not 1NF):**

| Student_ID | Name | Hobbies |
| :--- | :--- | :--- |
| 1 | ၸၢႆးသႅင် | Reading, Coding, Football |

**After (1NF):**

| Student_ID | Name | Hobby |
| :--- | :--- | :--- |
| 1 | ၸၢႆးသႅင် | Reading |
| 1 | ၸၢႆးသႅင် | Coding |
| 1 | ၸၢႆးသႅင် | Football |

*မိူဝ်ႈႁဝ်းၸႅၵ်ႇဢွၵ်ႇပဵၼ် ထႅဝ်ၽႂ်ထႅဝ်မၼ်းၼႆ... ႁဝ်းတေ `SELECT` သွၵ်ႈႁႃၶေႃႈမုၼ်းလႆႈငၢႆႈ လႄႈ ဝႆးလိူဝ်ၶႃႈ။*

### 4. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ 1NF
* **Atomicity:** မၢႆထိုင်ဝႃႈ "ၸႅၵ်ႇဢမ်ႇလႆႈထႅင်ႈယဝ်ႉ" (Indivisible)။ မိူၼ်ၼင်ႇ ၸိုဝ်ႈတူဝ်လဵဝ်၊ မၢႆၾူၼ်းတူဝ်လဵဝ်။
* **Repeating Groups:** ႁဝ်းတေဢမ်ႇၸႂ်ႉ Column မိူၼ်ၼင်ႇ `Hobby1`, `Hobby2`, `Hobby3` ၸိူဝ်းၼႆႉၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Normalization:** ၵၢၼ်ၸတ်းပိူင် Table ႁႂ်ႈမၢၼ်ႇမႅၼ်ႈ။
* **1NF (1st Normal Form):** ပိူင်ၸၼ်ႉဢွၼ်တၢင်းသုတ်း ဢၼ်မၵ်းမၼ်ႈလွင်ႈ Atomicity။
* **Atomicity:** ၵႃႈၶၼ်ၶေႃႈမုၼ်းဢၼ်လဵၵ်ႉသုတ်း သေဢမ်ႇၸၢင်ႈၸႅၵ်ႇလႆႈထႅင်ႈ။
* **Multi-valued Attribute:** Attribute ဢၼ်မီးၶေႃႈမုၼ်းလၢႆလၢႆဢၼ် ႁူမ်ႈၵၼ်ဝႆႉ။
