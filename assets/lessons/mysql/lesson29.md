# Lesson 29: "Data Modification: Secure UPDATE and DELETE Operations"

မိူဝ်ႈႁဝ်းသႂ်ႇ (Insert) ၶေႃႈမုၼ်းယဝ်ႉ... 
ႁဝ်းတေ ၵႆႉမီးၵၢၼ်လႅၵ်ႈလၢႆႈ (မိူၼ်ၼင်ႇ ၵူၼ်းၸႂ်ႉ App လၢႆႊၸိုဝ်ႈ ဢမ်ႇၼၼ် ထွၼ် Post) ၶႃႈ။
သင်ၼၼ် ႁဝ်းၸႂ်ႉလႆႈ `UPDATE` လႄႈ `DELETE` တႃႇၸတ်းၵၢၼ်ၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်း "ၵၢၼ်မႄးလိၵ်ႈၼႂ်းပပ်ႉ" ၶႃႈ...
* **UPDATE:** လိူၵ်ႈထႅဝ်လိၵ်ႈသေ ထွၼ်ဢွၵ်ႇ ၶေႃႈၵႂၢမ်းၵဝ်ႇသေ တႅမ်ႈသႂ်ႇၶိုၼ်းၶေႃႈၵႂၢမ်းမႂ်ႇ။
* **DELETE:** ဢဝ်လိၵ်ႈထႅဝ်ၼၼ်ႉ ထွၼ်ဢွၵ်ႇပႅတ်ႈ။
* **The WHERE Clause:** ၼႆႉပဵၼ် "သေႃး" ဢၼ်လွင်ႈယႂ်ႇသုတ်း။ မၼ်းမၵ်းမၼ်ႈဝႃႈ ႁဝ်းတေမႄး/ယႃႉ တီႈထႅဝ် (Row) လႂ်တႅတ်ႈတႅတ်ႈၶႃႈ။



### 2. Secure UPDATE (ၵၢၼ်မႄးၶေႃႈမုၼ်း ႁႂ်ႈလွတ်ႈၽေး)
သင်ၸဝ်ႈၵဝ်ႇတႅမ်ႈ `UPDATE` လူၺ်ႈဢမ်ႇမီး `WHERE`... ၶေႃႈမုၼ်းၼႂ်း Column ၼၼ်ႉ တေလႅၵ်ႈပဵၼ်ဢၼ်လဵဝ်ၵၼ် တင်းမူတ်း (Every row) ၶႃႈၼႃႈ!

```sql
-- လၢႆးတႅမ်ႈ ဢၼ်မၢၼ်ႇမႅၼ်ႈ (တေလႆႈၸႂ်ႉ Primary Key ၼႂ်း WHERE)
UPDATE students 
SET email = 'saimao.new@email.com', 
    age = 21
WHERE student_id = 101; 

-- ႁၢမ်ႈတႅမ်ႈၼင်ႇၼႆ (သင်ဢမ်ႇၶႂ်ႈႁႂ်ႈၵူၼ်းၵူႈၵေႃႉ ၸိုဝ်ႈမိူၼ်ၵၼ်)
-- UPDATE students SET first_name = 'Sai Mao'; 
```

### 3. Secure DELETE (ၵၢၼ်ယႃႉၶေႃႈမုၼ်း ႁႂ်ႈမၼ်ႈၵိုမ်း)
မိူၼ်ၵၼ်တင်း UPDATE ၶႃႈ... `DELETE` ဢၼ်ဢမ်ႇမီး `WHERE` တေယႃႉၶေႃႈမုၼ်းပႅတ်ႈ တင်း Table ၶႃႈ။

```sql
-- ယႃႉၵူၺ်း ၶေႃႈမုၼ်းဢၼ်ႁဝ်းလူဝ်ႇ
DELETE FROM students 
WHERE student_id = 105;

-- ယႃႉၸွမ်း Condition (မိူၼ်ၼင်ႇ ယႃႉလုၵ်ႈႁဵၼ်း ဢၼ်ဢမ်ႇမႃးႁဵၼ်း 5 ပီႊယဝ်ႉ)
DELETE FROM students 
WHERE last_login < '2021-01-01';
```

### 4. Professional Tip: Safe Updates Mode
ၼႂ်း MySQL Workbench မီးပိူင်ၼိုင်ႈ ႁွင်ႉဝႃႈ **Safe Updates**။ 
မၼ်းတေႁၢမ်ႈ ဢမ်ႇပၼ်ၸဝ်ႈၵဝ်ႇ `UPDATE` ဢမ်ႇၼၼ် `DELETE` သင်ဝႃႈ ၸဝ်ႈၵဝ်ႇဢမ်ႇလႆႈၸႂ်ႉ Primary Key ၼႂ်း `WHERE` ၶႃႈ။

* **Check before Commit:** ဢွၼ်တၢင်းတေ `UPDATE/DELETE` တႄႉတႄႉ... ၸၢမ်းၸႂ်ႉ `SELECT` တူၺ်းဢွၼ်တၢင်း ႁႂ်ႈႁူႉဝႃႈ ၸဝ်ႈၵဝ်ႇလိူၵ်ႈၺႃး ထႅဝ် (Row) ဢၼ်မၢၼ်ႇမႅၼ်ႈယဝ်ႉႁႃႉ ၼႆၶႃႈၼႃႈ။

### 5. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ Data Modification
* **Always use WHERE:** တႃႇသေႇ! တႃႇသေႇ! တႃႇသေႇ! လူဝ်ႇမီး WHERE တႃႇသေႇၶႃႈ။
* **LIMIT:** ႁဝ်းၸၢင်ႈၸႂ်ႉ `LIMIT 1` ၽၢႆႇလင် `UPDATE/DELETE` တႃႇႁႂ်ႈမၼ်းႁဵတ်းၵၢၼ် 1 ထႅဝ်ၵူၺ်း တွၼ်ႈတႃႇ ယႃႇႁႂ်ႈပဵၼ် Error ၶႃႈ။
* **Soft Delete (Option):** Developer ၵမ်ႈၼမ် ဢမ်ႇၵႆႉ `DELETE` ၶေႃႈမုၼ်းတႄႉတႄႉ။ ၶဝ်ၵႆႉၸႂ်ႉ Column `is_deleted` (0/1) သေၸႂ်ႉ `UPDATE` တႅၼ်း တႃႇသိမ်းပိုၼ်းဝႆႉၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **SET:** ၶေႃႈၵႂၢမ်းၸႂ်ႉ တႃႇမၵ်းမၼ်ႈ ၵႃႈၶၼ်မႂ်ႇ ၼႂ်း UPDATE။
* **WHERE Clause:** တီႈမၵ်းမၼ်ႈ ငဝ်းလၢႆး (Condition) တွၼ်ႈတႃႇလိူၵ်ႈ Row။
* **Safe Updates:** ပိူင်ႁၢမ်ႈ Error ၼႂ်း MySQL ဢၼ်ၸႅတ်ႈလွင်ႈၸႂ်ႉ Key ၼႂ်း Query။
* **Soft Delete:** ၵၢၼ်မၢႆဝႆႉဝႃႈ "ယႃႉယဝ်ႉ" ၵူၺ်းၵႃႈ ၶေႃႈမုၼ်းယင်းမီးယူႇၼႂ်း DB တႄႉတႄႉ။
