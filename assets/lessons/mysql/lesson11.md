# Lesson 11: "One-to-One (1:1): Use Cases for Table Splitting"

လွင်ႈၵွင်ႉၵၢႆႇ **One-to-One (1:1)** မၢႆထိုင်ဝႃႈ ၶေႃႈမုၼ်းၼႂ်း Table A ၼိုင်ႈထႅဝ် ၵွင်ႉလႆႈတင်း ၶေႃႈမုၼ်းၼႂ်း Table B ၼိုင်ႈထႅဝ်ၵူၺ်းၶႃႈ။ 
ၵႆႉမီးလွင်ႈထၢမ်ဝႃႈ "သင်မၼ်းၵွင်ႉၵၼ် 1 တေႃႇ 1 ၼႆ... ႁဵတ်းသင်ႁဝ်းဢမ်ႇဢဝ်မႃးဝႆႉၼႂ်း Table လဵဝ်ၵၼ်ပႅတ်ႈ?" ၼႆၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်း "ၵူၼ်း" (Person) တင်း "မၢႆၾၢင်" (NRC/Passport) ၶႃႈ...
* **ၵူၼ်း 1 ၵေႃႉ:** မီးလႆႈ "မၢႆၾၢင်" ဢၼ်လဵဝ်ၵူၺ်း။
* **မၢႆၾၢင် 1 ဢၼ်:** ၵေႃႈပဵၼ်ၶေႃႈမုၼ်းၶွင် "ၵူၼ်း" ၵေႃႉလဵဝ်ၵူၺ်း။
* **လွင်ႈၵွင်ႉၵၢႆႇ:** ၼႆႉႁွင်ႉဝႃႈ One-to-One (1:1) ၶႃႈ။



### 2. Table Splitting (ၵၢၼ်ၸႅၵ်ႇ Table)
ႁဝ်းၸႅၵ်ႇ Table ဢွၵ်ႇပဵၼ်သွင်ဢၼ် တင်းဢၼ်မၼ်းၵွင်ႉၵၼ် 1:1 ၼၼ်ႉ ၵွပ်ႈလွင်ႈတၢင်းၸိူဝ်းၼႆႉၶႃႈ...
* **Security (လွင်ႈလွတ်ႈၽေး):** ႁဝ်းၸၢင်ႈၸႅၵ်ႇ "ၶေႃႈမုၼ်းယူႇၵူႈဝၼ်း" (Public Data) ဝႆႉၼႂ်း Table ၼိုင်ႈဢၼ်သေ ဢဝ် "ၶေႃႈမုၼ်းလပ်ႉ" (Private/Sensitive Data) မိူၼ်ၼင်ႇ Password ဢမ်ႇၼၼ် Salary ဝႆႉထႅင်ႈၼႂ်း Table ၼိုင်ႈဢၼ်ၶႃႈ။
* **Performance (လွင်ႈဝႆး):** သင် Table ၼိုင်ႈဢၼ် မီး Column ၼမ်ႁႅင်းၵႂႃႇ (မိူၼ်ၼင်ႇ 100 Columns)... ႁဝ်းၸႅၵ်ႇဢွၵ်ႇပဵၼ် Table လဵၵ်ႉလဵၵ်ႉ တေႁဵတ်းႁႂ်ႈ Database ႁဝ်းႁဵတ်းၵၢၼ်ဝႆးလိူဝ်ၶႃႈ။
* **Optional Data:** သင်ၶေႃႈမုၼ်းမၢင်ဢၼ် ဢမ်ႇၸႂ်ႈ ဢၼ်ၵူႈၵေႃႉတေလႆႈမီး (မိူၼ်ၼင်ႇ "ပိုၼ်းၵၢၼ်ႁဵၼ်း" လုၵ်ႈႁဵၼ်း)... ႁဝ်းၸႅၵ်ႇဝႆႉထႅင်ႈ Table ဢၼ်ၼိုင်ႈၼႆႉ တေၸွႆႈယွမ်းပၼ် တီႈယူႇပဝ်ႇ (NULL Values) ၼႂ်း Table လူင်ၶႃႈ။

### 3. တူဝ်ယင်ႇ ၼႂ်းၵၢၼ်တႅမ်ႈ App
ဝူၼ်ႉတူၺ်း System **"TMK Student Manager"** ၶႃႈ...
* **Table Students:** သိမ်းၸိုဝ်ႈ၊ ဢႃႇယု၊ ၸၼ်ႉႁဵၼ်း (ၶေႃႈမုၼ်းဢၼ်ၵႆႉၸႂ်ႉ)။
* **Table Student_Details:** သိမ်းပိုၼ်းပၢႆးယူႇလီ၊ ၸိုဝ်ႈပေႃႈမႄႈ (ၶေႃႈမုၼ်းဢၼ်ဢမ်ႇၵႆႉၸႂ်ႉ)။
* **How to link:** ႁဝ်းဢဝ် `student_id` မႃးပဵၼ် **Primary Key** လႄႈ **Foreign Key** ၼႂ်း Table `Student_Details` ၶႃႈ။



### 4. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ 1:1 Relationship
* **Shared Primary Key:** ၵမ်ႈၼမ် ႁဝ်းၵႆႉၸႂ်ႉ ID ဢၼ်လဵဝ်ၵၼ် တင်းသွင် Table ၶႃႈ။
* **Unique FK:** သင်ႁဝ်းၸႂ်ႉ Foreign Key ၼႂ်း Table B... ႁဝ်းလူဝ်ႇတင်ႈဝႆႉပဵၼ် **`UNIQUE`** တႃႇႁၢမ်ႈဢမ်ႇႁႂ်ႈ ID ၼိုင်ႈဢၼ် ၵႂႃႇၵွင်ႉလႆႈထႅင်ႈလၢႆလၢႆဢၼ်ၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **1:1 (One-to-One):** ၵၢၼ်ၵွင်ႉၶေႃႈမုၼ်း 1 ဢၼ် ၸူး 1 ဢၼ်ၵူၺ်း။
* **Table Splitting:** ၵၢၼ်ၸႅၵ်ႇ Table လူင် ဢွၵ်ႇပဵၼ် Table လဵၵ်ႉလၢႆဢၼ်။
* **Vertical Partitioning:** ၸိုဝ်ႈႁွင်ႉထႅင်ႈဢၼ်ၼိုင်ႈ တွၼ်ႈတႃႇ Table Splitting (ၸႅၵ်ႇၸွမ်းထႅဝ်တင်ႈ)။
* **Sensitive Data:** ၶေႃႈမုၼ်းလမ်ႇလွင်ႈ ဢၼ်လူဝ်ႇ လွင်ႈလွတ်ႈၽေးသုင်။
