# Lesson 39: "String Functions: CONCAT, SUBSTRING, and REPLACE"

MySQL မီး Functions တင်းၼမ်တႃႇၸတ်းၵၢၼ်လိၵ်ႈ၊ ၵူၺ်းၵႃႈ 3 တူဝ်ၼႆႉ ပဵၼ်ဢၼ်ၸဝ်ႈၵဝ်ႇတေလႆႈၸႂ်ႉ တႃႇသေႇၼႂ်းၵၢၼ်ႁဵတ်း Project ၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်း "ၵၢၼ်မႄးလိၵ်ႈၼႂ်း Word" ၶႃႈ...
* **CONCAT:** ဢဝ်လိၵ်ႈသွင်တွၼ်ႈ (ဢမ်ႇၼၼ် လၢႆလၢႆတွၼ်ႈ) မႃး "ၸပ်း" (Connect) ၵၼ်။
* **SUBSTRING:** "တတ်း" (Cut) ဢဝ်တွၼ်ႈဢၼ်ႁဝ်းလူဝ်ႇ လုၵ်ႉၼႂ်းလိၵ်ႈယၢဝ်းယၢဝ်း။
* **REPLACE:** "လႅၵ်ႈ" (Swap) ၶေႃႈၵႂၢမ်းၵဝ်ႇ ႁႂ်ႈပဵၼ်ၶေႃႈၵႂၢမ်းမႂ်ႇ။



### 2. CONCAT (ၵၢၼ်တွၵ်ႇလိၵ်ႈ)
ၸႂ်ႉမိူဝ်ႈႁဝ်းၶႂ်ႈၼႄ "ၸိုဝ်ႈတဵမ်" (Full Name) ဢမ်ႇၼၼ် ႁဵတ်းလိၵ်ႈတွၼ်ႈၼိုင်ႈ ဢွၵ်ႇမႃးၼႄၼႂ်း UI ၶႃႈ။

```sql
-- ဢဝ်ၸိုဝ်ႈ လႄႈ ၼၢမ်သၵုၼ် မႃးႁူမ်ႈၵၼ် (သႂ်ႇပႃး Space ၵႄႈမၼ်း)
SELECT CONCAT(first_name, ' ', last_name) AS full_name 
FROM students;

-- တူဝ်ယင်ႇ: 'ၸၢႆး' + ' ' + 'သႅင်' => 'ၸၢႆး သႅင်'
```

### 3. SUBSTRING (ၵၢၼ်တတ်းလိၵ်ႈ)
ၸႂ်ႉမိူဝ်ႈႁဝ်းၶႂ်ႈၼႄ "လိၵ်ႈပွတ်း" (Preview/Summary) ဢမ်ႇၼၼ် တတ်းဢဝ်မၢႆၾူၼ်း မၢင်တွၼ်ႈၶႃႈ။
* **Syntax:** `SUBSTRING(string, start, length)`

```sql
-- တတ်းဢဝ်လိၵ်ႈ တႄႇတီႈတူဝ် 1 ၵႂႃႇ 10 တူဝ် (တႃႇၼႄ Preview)
SELECT SUBSTRING(description, 1, 10) AS short_desc 
FROM products;

-- တူဝ်ယင်ႇ: 'Apple iPhone 15 Pro' => 'Apple iPho'
```

### 4. REPLACE (ၵၢၼ်လႅၵ်ႈလိၵ်ႈ)
ၸႂ်ႉမိူဝ်ႈႁဝ်းလူဝ်ႇမႄးၶေႃႈမုၼ်း လၢႆလၢႆထႅဝ်ၵမ်းလဵဝ် (မိူၼ်ၼင်ႇ လႅၵ်ႈၸိုဝ်ႈဝဵင်း ဢမ်ႇၼၼ် လႅၵ်ႈ Link)။

```sql
-- လႅၵ်ႈ 'Muse' ႁႂ်ႈပဵၼ် 'Mu-se' တင်းမူတ်း
SELECT REPLACE(city, 'Muse', 'Mu-se') AS updated_city 
FROM students;

-- ၸႂ်ႉ UPDATE တႃႇမႄးတႄႉတႄႉၵေႃႈလႆႈ
UPDATE users SET website = REPLACE(website, 'http://', 'https://');
```

### 5. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ String Functions
* **1-Based Indexing:** တွင်းဝႆႉတႃႇသေႇဝႃႈ ၼႂ်း MySQL... တူဝ်လိၵ်ႈတူဝ်ဢွၼ်တၢင်းသုတ်း တႄႇၼပ်ႉတီႈ **1** ၶႃႈ (ဢမ်ႇၸႂ်ႈ 0 မိူၼ်ၼင်ႇၼႂ်း Dart ဢမ်ႇၼၼ် Java)။
* **CONCAT with NULL:** သင်မီး Column ဢၼ်ၼိုင်ႈပဵၼ် `NULL`... ၽွၼ်းလႆႈ `CONCAT` တေပဵၼ် **NULL** တင်းမူတ်းၶႃႈ။ (လူဝ်ႇၸႂ်ႉ `CONCAT_WS()` တႃႇႁၢမ်ႈပၼ်ႁႃၼႆႉ)။
* **Case Sensitivity:** `REPLACE` တေၸႅတ်ႈတူဝ်လိၵ်ႈ ႁႂ်ႈမိူၼ်ၵၼ်တႅတ်ႈတေႃး (Case-sensitive) ၸိူင်ႉၼင်ႇၸဝ်ႈၵဝ်ႇတႅမ်ႈဝႆႉၼၼ်ႉၶႃႈၼႃႈ။

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Concatenation:** ၵၢၼ်ဢဝ်လိၵ်ႈမႃးၸပ်းၵၼ်။
* **Substring:** တွၼ်ႈလိၵ်ႈဢၼ်တတ်းဢွၵ်ႇမႃး လုၵ်ႉတီႈလိၵ်ႈလူင် (String)။
* **Pattern Replacement:** ၵၢၼ်သွၵ်ႈႁႃႁွႆးလိၵ်ႈ သေလႅၵ်ႈလၢႆႈပဵၼ်ဢၼ်မႂ်ႇ။
* **Index:** မၢႆၼပ်ႉတီႈယူႇ တူဝ်လိၵ်ႈ (ၼႂ်း MySQL တႄႇတီႈ 1)။
