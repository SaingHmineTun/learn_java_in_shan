# Lesson 46: "Aggregate Functions: SUM, AVG, MIN, MAX, and COUNT"

**Aggregate Functions** မၢႆထိုင်ၵၢၼ်ဢဝ်ၶေႃႈမုၼ်းလၢႆလၢႆထႅဝ် (Multiple Rows) မႃးၼပ်ႉသွၼ်ႇႁူမ်ႈၵၼ် သေဢွၵ်ႇမႃးပဵၼ် **ၵႃႈၶၼ်ဢၼ်လဵဝ်** (Single Value) ၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်းလွင်ႈ "ၵၢၼ်ၼပ်ႉငိုၼ်းၼႂ်းသဵၼ်ႈမၢႆ" ၶႃႈ...
* ၼႂ်းသဵၼ်ႈမၢႆ တေမီးငိုၼ်းလၢႆလၢႆထႅဝ် (Rows)။
* ၸဝ်ႈၵဝ်ႇဢဝ်မႃး "ႁူမ်ႈၵၼ်" (Aggregate) တွၼ်ႈတႃႇႁူႉဝႃႈ မီးငိုၼ်းၵႃႈႁိုဝ်။



### 2. တူဝ်ယင်ႇၵၢၼ်ၸႂ်ႉ Functions တင်း 5 ဢၼ်

* **COUNT():** ၼပ်ႉတူၺ်းဝႃႈ မီးလၢႆထႅဝ် (How many?)
    ```sql
    SELECT COUNT(*) FROM students; -- ၼပ်ႉလုၵ်ႈႁဵၼ်းတင်းမူတ်း
    ```
* **SUM():** ႁူမ်ႈၵႃႈၶၼ်တူဝ်ၼပ်ႉ (Total)
    ```sql
    SELECT SUM(price) FROM orders; -- ႁူမ်ႈငိုၼ်းၵၢၼ်ၶၢႆတင်းမူတ်း
    ```
* **AVG():** ႁႃၵႃႈၶၼ်ၵႄႈၵၢင် (Average)
    ```sql
    SELECT AVG(age) FROM students; -- ႁႃဢႃႇယုၵႄႈၵၢင်လုၵ်ႈႁဵၼ်း
    ```
* **MIN() & MAX():** ႁႃၵႃႈၶၼ်ဢေႇသုတ်း လႄႈ ၼမ်သုတ်း
    ```sql
    SELECT MIN(price), MAX(price) FROM products; -- ႁႃၵုၼ်ႇဢၼ်ထုၵ်ႇသုတ်း လႄႈ ပႅင်းသုတ်း
    ```

### 3. ၵၢၼ်ၸႂ်ႉ DISTINCT ၼႂ်း Aggregate
မၢင်ပွၵ်ႈႁဝ်းဢမ်ႇၶႂ်ႈၼပ်ႉၶေႃႈမုၼ်းဢၼ် "သွၼ်ႉၵၼ်" ၶႃႈ။
```sql
SELECT COUNT(DISTINCT city) FROM students;
-- မၼ်းတေၼပ်ႉဝႃႈ မီးလုၵ်ႈႁဵၼ်းမႃးတီႈ "ၵႃႈလၢႆဝဵင်း" (ဢမ်ႇၼပ်ႉဝဵင်းဢၼ်သမ်ႉၵၼ်)
```

### 4. Aggregate Functions တင်း NULL
တွၼ်ႈၼႆႉလွင်ႈယႂ်ႇတႄႉတႄႉၶႃႈ:
* **COUNT(*):** ၼပ်ႉၵူႈထႅဝ် (ပႃးတင်း NULL)။
* **SUM, AVG, MIN, MAX, COUNT(column):** တေ **"ၶၢမ်ႈ" (Ignore)** ၵႃႈၶၼ် NULL ပႅတ်ႈတႃႇသေႇၶႃႈ။


### 5. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ Aggregate Functions
* **Alias is Better:** မိူဝ်ႈၸႂ်ႉ Functions ၸိူဝ်းၼႆႉ... ၽွၼ်းလႆႈတေဢွၵ်ႇၸိုဝ်ႈမႃးဝႃႈ `COUNT(*)`။ ႁဝ်းထုၵ်ႇလီၸႂ်ႉ **`AS`** တႃႇပၼ်ၸိုဝ်ႈမႂ်ႇ (မိူၼ်ၼင်ႇ `AS total_students`) ႁႂ်ႈ App ႁဝ်းလူငၢႆႈၶႃႈ။
* **Combine with WHERE:** ၸဝ်ႈၵဝ်ႇၸၢင်ႈၸႂ်ႉ `WHERE` တွၼ်ႈတႃႇလိူၵ်ႈၸုမ်းဢၼ်တေမႃးၼပ်ႉသွၼ်ႇ (မိူၼ်ၼင်ႇ ႁူမ်ႈငိုၼ်းၶၢႆ တွၼ်ႈတႃႇ 'ဝၼ်းမိူဝ်ႈၼႆႉ' ၵူၺ်း)။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Aggregate Function:** Function ဢၼ်ဢဝ်ၶေႃႈမုၼ်းလၢႆထႅဝ် မႃးႁဵတ်းပဵၼ်ၵႃႈၶၼ်လဵဝ်။
* **Total/Sum:** ၵႃႈၶၼ်ႁူမ်ႈတင်းမူတ်း။
* **Average:** ၵႃႈၶၼ်ၵႄႈၵၢင် (Mean)။
* **Distinct:** ၵၢၼ်လိူၵ်ႈဢဝ်ၵူၺ်း ၶေႃႈမုၼ်းဢၼ်ဢမ်ႇမိူၼ်ၵၼ်။
