# Lesson 65: "Indexing I: B-Trees and Primary/Unique Indexes"

**Index** မၢႆထိုင် "သဵၼ်ႈမၢႆၼႃႈလိၵ်ႈ" ၼႂ်းပပ်ႉ။ သင်ၸဝ်ႈၵဝ်ႇ ၶႂ်ႈႁႃၶေႃႈၵႂၢမ်းၼိုင်ႈဢၼ် ၼႂ်းပပ်ႉ ဢၼ်မီး 500 ၼႃႈလိၵ်ႈၼၼ်ႉ
ၸဝ်ႈၵဝ်ႇတေဢမ်ႇပိုတ်ႇလူၵူႈၼႃႈ၊ ၵူၺ်းၵႃႈ တေၵႂႃႇတူၺ်းတီႈ **Index** ၽၢႆႇလင် သေပိုတ်ႇၵႂႃႇၸူးၼႃႈလိၵ်ႈၼၼ်ႉၵမ်းလဵဝ်ၶႃႈ။

### 1. B-Tree Structure (ပိူင်သၢင်ႈတူၼ်ႈမႆႉ)
MySQL (InnoDB) ၸႂ်ႉပိူင်သၢင်ႈဢၼ်ႁွင်ႉဝႃႈ **B-Tree** (Balanced Tree) တႃႇသိမ်း Index ၶႃႈ။
* မၼ်းတေၸႅၵ်ႇၶေႃႈမုၼ်းပဵၼ်ၸၼ်ႉ (Nodes)။
* မိူဝ်ႈသွၵ်ႈႁႃ... မၼ်းတေတႅၵ်ႈၵႃႈၶၼ် "ၼမ်လိူဝ်" ႁိုဝ် "ဢေႇလိူဝ်" သေတူၵ်းလူင်းၵႂႃႇၸူးၵိင်ႇမႆႉဢၼ်မၢၼ်ႇမႅၼ်ႈ။
* ၵၢၼ်ႁဵတ်းၼင်ႇၼႆ တေၸွႆႈႁႃ ၶေႃႈမုၼ်းလႆႈဝႆးႁႅင်းၶႃႈ (Logarithmic Time)။



### 2. Primary Index (သေႃးတႅတ်ႈတေႃး)
မိူဝ်ႈၸဝ်ႈၵဝ်ႇမၵ်းမၼ်ႈ **PRIMARY KEY**... MySQL တေၸွႆႈသၢင်ႈ Index ႁင်းမၼ်းၵမ်းလဵဝ်။
* **Clustered Index:** ၼႂ်း InnoDB... Primary Key ပဵၼ် Clustered Index။ ပွင်ႇဝႃႈ "ၶေႃႈမုၼ်းတႄႉတႄႉ" (Actual Data Rows) တေထုၵ်ႇၸတ်းလႅၼ်ႇဝႆႉၸွမ်းမၢႆ ID ၼၼ်ႉၵမ်းလဵဝ်ၶႃႈ။
* ၼႆႉပဵၼ်လွင်ႈတၢင်း ဢၼ်ႁဵတ်းႁႂ်ႈ `WHERE id = 101` ဝႆးသုတ်းၶႃႈ။

### 3. Unique Index (သေႃးဢၼ်ဢမ်ႇမိူၼ်ပိူၼ်ႈ)
**UNIQUE Index** မိူၼ်တင်း Primary Key ၵူၺ်းၵႃႈ:
* မၼ်းပၼ်ႁႂ်ႈမီးၵႃႈၶၼ် **NULL** လႆႈ (သင်ဢမ်ႇပႆႇသႂ်ႇ)။
* မၼ်းႁႄႉၵင်ႈဢမ်ႇႁႂ်ႈမီးၶေႃႈမုၼ်း "သွၼ်ႉၵၼ်" (Duplicates) ၼႂ်း Column ၼၼ်ႉ (မိူၼ်ၼင်ႇ Email ဢမ်ႇၼၼ် Username)။

### 4. ႁဵတ်းသင်လႄႈ လူဝ်ႇၸႂ်ႉ Index?
1.  **Speed:** ႁႃၶေႃႈမုၼ်းလႆႈဝႆးလိူဝ်ၵဝ်ႇ 100-1000 ပုၼ်ႈ ၼႂ်း Table ဢၼ်ယႂ်ႇ။
2.  **Sorting:** ၸွႆႈႁႂ်ႈ `ORDER BY` ႁဵတ်းၵၢၼ်ဝႆး ၵွပ်ႈၶေႃႈမုၼ်းၼႂ်း Index ထုၵ်ႇၸတ်းလႅၼ်ႇ ဝႆႉယဝ်ႉတူဝ်ႈၶႃႈ။
3.  **Uniqueness:** မၼ်ႈၸႂ်ဝႃႈ ၶေႃႈမုၼ်းဢၼ်လွင်ႈယႂ်ႇ တေဢမ်ႇသွၼ်ႉၵၼ်။

### 5. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ Indexing
* **Index Overhead:** Index ၸွႆႈႁႂ်ႈ SELECT ဝႆးသေတႃႉ မၼ်းတေႁဵတ်းႁႂ်ႈ **INSERT, UPDATE, DELETE ထိူင်းၵႂႃႇ** ဢိတ်းဢိတ်းၶႃႈ။ ၵွပ်ႈ MySQL လႆႈၵႂႃႇမႄးသဵၼ်ႈမၢႆ (Index) ၸွမ်းၵူႈပွၵ်ႈၶႃႈ။
* **Storage:** Index ၸႂ်ႉတီႈယူႇ (Disk Space) ၼမ်လိူဝ် Table ၶႃႈ။
* **Rule of Thumb:** ယႃႇသႂ်ႇ Index ၵူႈ Column! သႂ်ႇတီႈ Column ဢၼ်ၸဝ်ႈၵဝ်ႇၸႂ်ႉၼႂ်း **`WHERE`**, **`JOIN`**, လႄႈ **`ORDER BY`** ၼမ်လိူဝ်ပိူၼ်ႈ ၼၼ်ႉၶႃႈ။

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **B-Tree:** ပိူင်သၢင်ႈတူၼ်ႈမႆႉ ဢၼ်ၸွႆႈႁႃၶေႃႈမုၼ်းလႆႈဝႆး။
* **Clustered Index:** Index ဢၼ်ၸတ်းလႅၼ်ႇ ၶေႃႈမုၼ်းတႄႉၸွမ်းမၼ်း (Table ၼိုင်ႈဢၼ် မီးလႆႈမၼ်း ၼိုင်ႈဢၼ်ၵူၺ်းၶႃႈ)။
* **Non-Clustered Index:** Index ဢၼ်သိမ်းဝႆႉ တၢင်ႇတီႈသေ ၸႅၵ်ႇၼႄဝႃႈ ၶေႃႈမုၼ်းယူႇတီႈလႂ် (Secondary Index)။
* **Index Seek:** ၵၢၼ်ပိုတ်ႇၵႂႃႇၸူးၶေႃႈမုၼ်းၵမ်းလဵဝ် (ဝႆး)။
