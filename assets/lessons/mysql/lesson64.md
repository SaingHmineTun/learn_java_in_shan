# Lesson 64: "The Execution Plan: Understanding EXPLAIN"

မိူဝ်ႈၸဝ်ႈၵဝ်ႇသူင်ႇ Query ၵႂႃႇၸူး MySQL... မၼ်းတေမီး "ၵူၼ်းဝူၼ်ႉ" (Optimizer) ဢၼ်တေလိူၵ်ႈသဵၼ်ႈတၢင်း 
ဢၼ်ဝႆးသုတ်းတွၼ်ႈတႃႇႁႃၶေႃႈမုၼ်းၶႃႈ။ **`EXPLAIN`** ပဵၼ် Command ဢၼ်တေၼႄႁဝ်းဝႃႈ "MySQL ဝူၼ်ႉၸိူင်ႉႁိုဝ်?" ဢွၼ်တၢင်းမၼ်းတေႁဵတ်းၵၢၼ်ၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်း "ၵၢၼ်ၸႂ်ႉ Google Maps" ၶႃႈ...
* **SQL Query:** ပဵၼ်ၵၢၼ်မၵ်းမၼ်ႈဝႃႈ "ၶႂ်ႈၵႂႃႇတီႈလႂ်"။
* **EXPLAIN:** ပဵၼ် "သဵၼ်ႈတၢင်း (Route)" ဢၼ် Google Maps ဝၢင်းၽႅၼ်ဝႆႉပၼ်ၸဝ်ႈၵဝ်ႇ။ မၼ်းတေၼႄဝႃႈ တေၵႂႃႇသဵၼ်ႈတၢင်းလႂ်၊ တေမီးၵူၼ်းၼမ်ႁႃႉ (Traffic)၊ လႄႈ တေႁိုင်ၵႃႈႁိုဝ်ၶႃႈ။



### 2. လၢႆးၸႂ်ႉ EXPLAIN
ၸဝ်ႈၵဝ်ႇၵူၺ်းလူဝ်ႇတႅမ်ႈၶေႃႈၵႂၢမ်း `EXPLAIN` ဝႆႉၽၢႆႇၼႃႈ Query ၸဝ်ႈၵဝ်ႇၵူၺ်းၶႃႈ:

```sql
EXPLAIN SELECT * FROM students WHERE student_id = 101;
```

### 3. Column ဢၼ်လွင်ႈယႂ်ႇၼႂ်း EXPLAIN Result
မိူဝ်ႈ SELECT ဢွၵ်ႇမႃး... ၸဝ်ႈၵဝ်ႇတေႁၼ် Table ဢၼ်မီး Column လၢႆဢၼ်။ ဢၼ်လူဝ်ႇတူၺ်းလီလီမီး:

* **type (ၸၼ်ႉၵၢၼ်သွၵ်ႈ):** ၼႆႉပဵၼ်ဢၼ်လွင်ႈယႂ်ႇသုတ်း။
    * `const`: ဝႆးသုတ်း (ႁၼ် 1 Row တႅတ်ႈတေႃး)။
    * `ref` / `range`: လီယူႇ (ၸႂ်ႉ Index တႃႇသွၵ်ႈ)။
    * `ALL`: **ထိူင်းသုတ်း** (Full Table Scan) - MySQL လႆႈလူၵူႈထႅဝ်ၼႂ်း Table တေႃႇပေႃးယဝ်ႉ။
* **rows (တၢင်းၼမ်ထႅဝ်):** MySQL ဝူၼ်ႉဝႃႈ မၼ်းတေလႆႈလူ (Scan) လၢႆထႅဝ် ၸင်ႇတေႁၼ်ၶေႃႈမုၼ်း။ ယိူင်းဢၢၼ်းႁဝ်းၵေႃႈပဵၼ် "ႁႂ်ႈ Number ၼႆႉ ဢေႇသုတ်း" ၶႃႈ။
* **key (သေႃး):** မၼ်းၼႄဝႃႈ MySQL ၸႂ်ႉ **Index** တူဝ်လႂ်ယူႇ။ သင်ပဵၼ် `NULL`... ပွင်ႇဝႃႈမၼ်းဢမ်ႇၸႂ်ႉ Index သေဢၼ်ၶႃႈ။
* **Extra:** ၶေႃႈမုၼ်းထႅမ် (မိူၼ်ၼင်ႇ `Using filesort` ဢမ်ႇၼၼ် `Using temporary` - ၸိူဝ်းၼႆႉၶႃႈ)။

### 4. ႁဵတ်းသင်လႄႈ လူဝ်ႇၸႂ်ႉ EXPLAIN?
1.  **Debugging Performance:** တွၼ်ႈတႃႇႁူႉဝႃႈ ႁဵတ်းသင် Query ၼႆႉထိူင်းႁႅင်း။
2.  **Indexing Decision:** တႃႇတႅပ်းတတ်းဝႃႈ ထုၵ်ႇလီသႂ်ႇ Index တီႈ Column လႂ်ထႅင်ႈ။
3.  **Cost Analysis:** ၼပ်ႉသွၼ်ႇတူၺ်းဝႃႈ Query လႂ် ၵိၼ် "Server" ၼမ်ႁႅင်း ၼႆၶႃႈ။

### 5. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ Optimization
* **Avoid 'ALL':** သင်ႁၼ် `type = ALL` ၼႂ်း Table ဢၼ်မီးၶေႃႈမုၼ်းၼမ်... ၸဝ်ႈၵဝ်ႇလူဝ်ႇသႂ်ႇ **Index** ၵမ်းလဵဝ်ၶႃႈ။
* **Filter Early:** တူၺ်းဝႃႈ MySQL တတ်းၶေႃႈမုၼ်း (Filter) လႆႈဝႆးၵႃႈႁိုဝ်။
* **Explain Analyze (MySQL 8.0+):** ၸၢင်ႈၸႂ်ႉ `EXPLAIN ANALYZE` တႃႇႁႂ်ႈ MySQL ႁဵတ်းၵၢၼ်တႄႉ သေၼႄၶၢဝ်းယၢမ်းတႄႉ (Actual Time) ဢွၵ်ႇမႃးပႃးၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Execution Plan:** ၽႅၼ်ၵၢၼ်ႁဵတ်းၵၢၼ် ၶွင် Database Optimizer။
* **Full Table Scan:** ၵၢၼ်လူၶေႃႈမုၼ်းၵူႈထႅဝ် (ထိူင်း)။
* **Index Scan:** ၵၢၼ်လူၶေႃႈမုၼ်းၸွမ်းသဵၼ်ႈမၢႆ (ဝႆး)။
* **Cost-based Optimizer:** လွင်ႈဝၢင်းၽႅၼ် ဢၼ်လိူၵ်ႈသဵၼ်ႈတၢင်း ဢၼ် "သဵင်ႈႁႅင်း" ဢေႇသုတ်း။
