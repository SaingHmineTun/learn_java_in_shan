# Lesson 36: "Sorting: Advanced ORDER BY (Multi-column and Custom)"

ၼႂ်း Lesson ၼႆႉ ႁဝ်းတေမႃးႁဵၼ်း လၢႆးၶပ်ႉလၢႆလၢႆ Column ႁူမ်ႈၵၼ် လႄႈ လၢႆးၶပ်ႉၸွမ်း "ၵၢင်ၸႂ်" ႁဝ်းၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်း "ၵၢၼ်ၶပ်းထႅဝ်လုၵ်ႈႁဵၼ်း" ၶႃႈ...
* **ASC (Ascending):** တႄႇတီႈဢေႇ ၵႂႃႇၸူးၼမ် (A-Z, 1-100)။ ၼႆႉပဵၼ် Default ၶႃႈ။
* **DESC (Descending):** တႄႇတီႈၼမ် လုင်းမႃးဢေႇ (Z-A, 100-1)။
* **Multi-column:** မိူၼ်ၼင်ႇ "ၶပ်ႉၸွမ်းဝဵင်းဢွၼ်တၢင်း၊ သင်ဝဵင်းမိူၼ်ၵၼ် ၸင်ႇၶပ်ႉၸွမ်းၸိုဝ်ႈ" ၶႃႈ။



### 2. Basic Sorting (ၵၢၼ်ၸတ်းပိုၼ်ႉထၢၼ်)
ႁဝ်းတေလႆႈဝႆႉ `ORDER BY` ၽၢႆႇလင်သုတ်း (လိူဝ်သေ LIMIT) ၶႃႈ။

```sql
-- ၸတ်းၵႃႈၶၼ်ၵုၼ်ႇ တီႈၼမ်သုတ်း လုင်းမႃးဢေႇ
SELECT * FROM products 
ORDER BY price DESC;

-- ၸတ်းၸိုဝ်ႈလုၵ်ႈႁဵၼ်းၸွမ်းတူဝ်လိၵ်ႈ (A-Z)
SELECT * FROM students 
ORDER BY first_name ASC; -- ဢမ်ႇသႂ်ႇ ASC ၵေႃႈမိူၼ်ၵၼ်ၶႃႈ
```

### 3. Multi-column Sorting (ၵၢၼ်ၸတ်းလၢႆၸၼ်ႉ)
ၸႂ်ႉမိူဝ်ႈႁဝ်းလူဝ်ႇမီး "ပိူင်ၶပ်ႉမၢႆသွင်" (Secondary Sort) မိူဝ်ႈၶေႃႈမုၼ်း တွၼ်ႈဢွၼ်တၢင်းမၼ်း မိူၼ်ၵၼ်ၶႃႈ။

```sql
-- ၸတ်းၸွမ်းဝဵင်း (A-Z)၊ သင်ယူႇဝဵင်းလဵဝ်ၵၼ် ႁႂ်ႈၸတ်းၸွမ်းဢႃႇယု (ၼမ်ၵႂႃႇဢေႇ)
SELECT * FROM students 
ORDER BY city ASC, age DESC;
```

### 4. Custom Sorting (Using FIELD)
မၢင်ပွၵ်ႈ ႁဝ်းဢမ်ႇၶႂ်ႈၶပ်းၸွမ်းတူဝ်လိၵ်ႈ ဢမ်ႇၼၼ် တူဝ်ၼပ်ႉ။ 
တူဝ်ယင်ႇမိူၼ်ၼင်ႇ "ၶႂ်ႈႁႂ်ႈ Status 'Pending' မႃးဢွၼ်တၢင်း၊ 
'Shipped' မႃးမၢႆသွင်၊ လႄႈ 
'Delivered' တီႈသုတ်း" ၼႆၶႃႈ။

```sql
-- ၸတ်းၸွမ်းသဵၼ်ႈမၢႆ ဢၼ်ႁဝ်းလိူၵ်ႈဝႆႉ
SELECT * FROM orders 
ORDER BY FIELD(status, 'Pending', 'Shipped', 'Delivered');
```

### 5. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ Sorting
* **Column Position:** ႁဝ်းၸၢင်ႈၸႂ်ႉမၢႆၼပ်ႉ Column တႅၼ်းၸိုဝ်ႈမၼ်းလႆႈ (မိူၼ်ၼင်ႇ `ORDER BY 2`) ၵူၺ်းၵႃႈ **ႁၢမ်ႈၸႂ်ႉ** ၶႃႈ၊ ၵွပ်ႈဝႃႈ သင် Table structure လႅၵ်ႈလၢႆႈမႃးၼႆ Query ၸဝ်ႈၵဝ်ႇတေၸၢင်ႈၽိတ်းၶႃႈ။
* **NULLs in Sorting:** ၼႂ်း MySQL... `NULL` တေထုၵ်ႇၼပ်ႉပဵၼ်ၵႃႈၶၼ်ဢေႇသုတ်း။ သင်ၸတ်း `ASC`... NULL တေယူႇၼိူဝ်သုတ်း၊ သင် `DESC`... NULL တေယူႇတႂ်ႈသုတ်းၶႃႈ။
* **Performance:** ၵၢၼ်ၸတ်းလႅၼ်ႇၶေႃႈမုၼ်းၼမ်ၼမ် တေႁဵတ်းႁႂ်ႈ Server ႁဵတ်းၵၢၼ်ၼၵ်း။ သင်ၸဝ်ႈၵဝ်ႇ `ORDER BY` ၸွမ်း Column ဢၼ်မီး **Index**... မၼ်းတေဝႆးလိူဝ်တႄႉတႄႉၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **ASC (Ascending):** ၸတ်းတီႈဢေႇ ၶိုၼ်ႈၵႂႃႇၼမ် (Default)။
* **DESC (Descending):** ၸတ်းတီႈၼမ် လူင်းမႃးဢေႇ။
* **Secondary Sort:** Column တီႈသွင် ဢၼ်တေၸတ်းလႅၼ်ႇ မိူဝ်ႈ Column တီႈၼိုင်ႈသမ်ႉၵၼ်။
* **FIELD():** Function ၼႂ်း MySQL တႃႇၸတ်းလႅၼ်ႇၸွမ်းသဵၼ်ႈမၢႆ ဢၼ်မၵ်းမၼ်ႈဝႆႉ။
