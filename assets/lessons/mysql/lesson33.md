# Lesson 33: "Pattern Matching I: The LIKE Operator and Wildcards"

မိူဝ်ႈႁဝ်းၸႂ်ႉ `=` (Equal) ... ၶေႃႈမုၼ်းတေလႆႈ မိူၼ်လီလီၶႃႈ။ ၵူၺ်းၵႃႈ **`LIKE`** သမ်ႉ ၸွႆႈႁႂ်ႈႁဝ်းသွၵ်ႈႁႃၸွမ်း (Patterns) 
လႆႈလူၺ်ႈ ၵၢၼ်ၸႂ်ႉ **Wildcards** (တူဝ်တႅၼ်း) 2 ပိူင်ၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်း "ၵၢၼ်သွၵ်ႈႁႃ Contacts ၼႂ်းၾူၼ်း" ၶႃႈ...
* **`%` (Percent):** တႅၼ်းတူဝ်လိၵ်ႈ **လၢႆတူဝ်ၵေႃႈလႆႈ** (0 တူဝ် ၶိုၼ်ႈၼိူဝ်)။
* **`_` (Underscore):** တႅၼ်းတူဝ်လိၵ်ႈ **1 တူဝ်** ၵူၺ်း။



### 2. ၵၢၼ်ၸႂ်ႉ `%` (The Multi-character Wildcard)
ၼႆႉပဵၼ်တူဝ်ဢၼ်ႁဝ်းၸႂ်ႉၼမ်လိူဝ်ပိူၼ်ႈ တွၼ်ႈတႃႇႁဵတ်း Search box ၶႃႈ။

* **`'ၸၢႆး%'`** : တႄႇလူၺ်ႈ "ၸၢႆး" (မိူၼ်ၼင်ႇ: ၸၢႆးသႅင်, ၸၢႆးၶမ်း)။
* **`'%သႅင်'`** : သုတ်းလူၺ်ႈ "သႅင်" (မိူၼ်ၼင်ႇ: ၼၢင်းသႅင်, ၸၢႆးသႅင်)။
* **`'%မ%'`** : မီးတူဝ် "မ" ယူႇတီႈလႂ်ၵေႃႈလႆႈ (မိူၼ်ၼင်ႇ: ၸၢႆးၶမ်း, ၼၢင်းၼုမ်ႇ)။

```sql
-- ႁႃလုၵ်ႈႁဵၼ်း ဢၼ်ၸိုဝ်ႈတႄႇလူၺ်ႈ 'ၸၢႆး'
SELECT * FROM students 
WHERE first_name LIKE 'ၸၢႆး%';

-- ႁႃ Email ဢၼ်ပဵၼ် '@gmail.com'
SELECT * FROM users 
WHERE email LIKE '%@gmail.com';
```

### 3. ၵၢၼ်ၸႂ်ႉ `_` (The Single-character Wildcard)
ၸႂ်ႉမိူဝ်ႈႁဝ်းႁူႉတၢင်းယၢဝ်းလိၵ်ႈ တႅတ်ႈတေႃး ၵူၺ်းၵႃႈဢမ်ႇႁူႉမၢင်တူဝ်ၶႃႈ။

* **`'ၸ_ႆး'`** : ႁႃလိၵ်ႈ 3 တူဝ် ဢၼ်တႄႇလူၺ်ႈ "ၸ" သေ သုတ်းလူၺ်ႈ "ႆး" (မိူၼ်ၼင်ႇ: ၸၢႆး)။
* **`'H_t'`** : (မိူၼ်ၼင်ႇ: Hot, Hat, Hit)။

```sql
-- ႁႃ ID ဢၼ်မီး 5 တူဝ် သေတႄႇလူၺ်ႈ 'A'
SELECT * FROM products 
WHERE product_code LIKE 'A____'; 
```

### 4. ၵၢၼ်ၸႂ်ႉ NOT LIKE
ႁဝ်းၸၢင်ႈလိူၵ်ႈဢဝ် ၶေႃႈမုၼ်းဢၼ် "ဢမ်ႇမိူၼ်" ၸွမ်း Pattern ၼၼ်ႉၵေႃႈလႆႈၶႃႈ။

```sql
-- ႁႃလုၵ်ႈႁဵၼ်း ဢၼ်ၸိုဝ်ႈ 'ဢမ်ႇတႄႇလူၺ်ႈ' ၸၢႆး
SELECT * FROM students 
WHERE first_name NOT LIKE 'ၸၢႆး%';
```

### 5. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ LIKE Operator
* **Case Sensitivity:** ၼႂ်း MySQL... `LIKE` ၵမ်ႈၼမ် တေဢမ်ႇၸႅၵ်ႇ တူဝ်ယႂ်ႇ တူဝ်လဵၵ်ႉ (Case-insensitive) ၶႃႈ။
* **Performance:** တွင်းဝႆႉဝႃႈ ၵၢၼ်ၸႂ်ႉ `%` ဝႆႉၽၢႆႇၼႃႈ (မိူၼ်ၼင်ႇ `'%သႅင်'`) တေႁဵတ်းႁႂ်ႈ Database **ထိူင်းလိူဝ်** ၵွပ်ႈမၼ်းဢမ်ႇၸၢင်ႈၸႂ်ႉ Index လႆႈၶႃႈ။
* **Escape Character:** သင်ၸဝ်ႈၵဝ်ႇၶႂ်ႈႁႃတူဝ် `%` ဢမ်ႇၼၼ် `_` တႄႉတႄႉၼၼ်ႉ... လူဝ်ႇၸႂ်ႉ `\` (Backslash) ဝႆႉၽၢႆႇၼႃႈမၼ်း (မိူၼ်ၼင်ႇ `LIKE '50\%'`) ၶႃႈၼႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Pattern Matching:** ၵၢၼ်သွၵ်ႈႁႃၶေႃႈမုၼ်း ၸွမ်းၼင်ႇ Pattern ဢၼ်မၵ်းမၼ်ႈဝႆႉ။
* **Wildcard:** တူဝ်လိၵ်ႈဢၼ်ၸႂ်ႉတႅၼ်း တူဝ်လိၵ်ႈတၢင်ႇၸိူဝ်း (%, _ )။
* **Escape:** ၵၢၼ်မၵ်းမၼ်ႈဝႃႈ တူဝ်လိၵ်ႈၼၼ်ႉ ပဵၼ်တူဝ်လိၵ်ႈတႄႉတႄႉ ဢမ်ႇၸႂ်ႈ Wildcard။
