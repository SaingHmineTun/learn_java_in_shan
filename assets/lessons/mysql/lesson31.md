# Lesson 31: "Basic Filtering: Comparison and Logical Operators"

ၼႂ်း Lesson ၼႆႉ ႁဝ်းတေၸႂ်ႉ **`WHERE`** Clause ႁူမ်ႈတင်းၶိူင်ႈမၢႆၼပ်ႉသွၼ်ႇ လႄႈ ၶိူင်ႈမၢႆလွၵ်ႉၵျိတ်ႉ (Logic) ၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်း "ၵၢၼ်သွၵ်ႈႁႃၶူဝ်းၼႂ်း App Shopping" ၶႃႈ...
* **Comparison:** မိူၼ်ၼင်ႇ "ၵႃႈၶၼ် ယွမ်းလိူဝ် 500" (Price < 500)။
* **Logical:** မိူၼ်ၼင်ႇ "သီလႅင် **လႄႈ** Size L" (Color = 'Red' **AND** Size = 'L')။



### 2. Comparison Operators (ၶိူင်ႈမၢႆတႅၵ်ႈတူၺ်း)
ၸႂ်ႉတႃႇတႅၵ်ႈတူၺ်း ၵႃႈၶၼ်ၼႂ်း Column တင်း ၵႃႈၶၼ်ဢၼ်ႁဝ်းပၼ်ဝႆႉၶႃႈ။

* **`=`** : မိူၼ်ႇၵၼ် (Equal)
* **`!=`** ဢမ်ႇၼၼ် **`<>`** : ဢမ်ႇမိူၼ်ၵၼ် (Not Equal)
* **`>`** , **`<`** : ယႂ်ႇလိူဝ်, ယွမ်းလိူဝ်
* **`>=`** , **`<=`** : ယႂ်ႇလိူဝ်ဢမ်ႇၼၼ်မိူၼ်, ယွမ်းလိူဝ်ဢမ်ႇၼၼ်မိူၼ်

```sql
-- လိူၵ်ႈလုၵ်ႈႁဵၼ်း ဢၼ်ဢႃႇယု ယႂ်ႇလိူဝ် 20
SELECT * FROM students 
WHERE age > 20;

-- လိူၵ်ႈၵုၼ်ႇ ဢၼ်ၵႃႈၶၼ် ဢမ်ႇၸႂ်ႈ 1000
SELECT * FROM products 
WHERE price <> 1000;
```

### 3. Logical Operators (ၶိူင်ႈမၢႆလွတ်ႇၵျိၵ်ႉ)
ၸႂ်ႉမိူဝ်ႈႁဝ်းလူဝ်ႇၸႅတ်ႈ "ငဝ်းလၢႆးလၢႆလၢႆဢၼ်" ႁူမ်ႈၵၼ်ၶႃႈ။

* **`AND`** : တေလႆႈမၢၼ်ႇမႅၼ်ႈ **တင်းသွင်ၽၢႆႇ** (True + True)။
* **`OR`** : မၢၼ်ႇမႅၼ်ႈ **ၽၢႆႇၼိုင်ႈၵေႃးလႆႈ** (True + False = True)။
* **`NOT`** : ပိၼ်ႈၵႃႈၶၼ် (True ပဵၼ် False)။

```sql
-- လုၵ်ႈႁဵၼ်း ဢၼ်ယူႇဝဵင်း 'Muse' လႄႈ ဢႃႇယု ဢၼ်မီးဝႆႉ 18
SELECT * FROM students 
WHERE city = 'Muse' AND age = 18;

-- ၵုၼ်ႇ ဢၼ်ပဵၼ်သီ 'Red' ဢမ်ႇၼၼ် သီ 'Blue'
SELECT * FROM products 
WHERE color = 'Red' OR color = 'Blue';
```

### 4. ၵၢၼ်ၸႂ်ႉ ၶွပ်ႇမူၼ်း (Parentheses `()`)
မိူဝ်ႈၸဝ်ႈၵဝ်ႇၸႂ်ႉ `AND` လႄႈ `OR` ႁူမ်ႈၵၼ်... လူဝ်ႇသႂ်ႇၶွပ်ႇမူၼ်း `()` တႃႇမၵ်းမၼ်ႈဝႃႈ ႁႂ်ႈမၼ်းၼပ်ႉသွၼ်ႇတွၼ်ႈလႂ်ဢွၼ်တၢင်းၶႃႈ။

```sql
-- လိူၵ်ႈၵုၼ်ႇ ဢၼ်ပဵၼ် Categories 1 ဢမ်ႇၼၼ် 2 သေ တေလႆႈမီး Stock ၼမ်လိူဝ် 0
SELECT * FROM products 
WHERE (category_id = 1 OR category_id = 2) AND stock > 0;
```

### 5. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ Basic Filtering
* **String Sensitivity:** ၼႂ်း MySQL ၵမ်ႈၼမ်... ၵၢၼ်တႅၵ်ႈလိၵ်ႈ `'muse'` တင်း `'MUSE'` တေမိူၼ်ၵၼ် (Case-insensitive) ၶႃႈ။ ၵူၺ်းၵႃႈ လူဝ်ႇၸႅတ်ႈတူၺ်း Collation ၸဝ်ႈၵဝ်ႇထႅင်ႈပွၵ်ႈၼိုင်ႈၶႃႈၼႃႈ။
* **Performance:** ၵၢၼ်ၸႂ်ႉ `WHERE` ၸွမ်း Column ဢၼ်မီး **Index** (မိူၼ်ၼင်ႇ PK) တေႁဵတ်းႁႂ်ႈ Query ဝႆးလိူဝ်ပိူၼ်ႈၶႃႈ။
* **NULL Check:** တွင်းဝႆႉဝႃႈ ႁဝ်းဢမ်ႇၸၢင်ႈၸႂ်ႉ `= NULL` လႆႈၶႃႈ (ႁဝ်းတေလဵပ်ႈႁဵၼ်းလွင်ႈၼႆႉ ၼႂ်း Lesson 35 ၶႃႈၼႃႈ)။

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Filtering:** ၵၢၼ်လိူၵ်ႈဢဝ်ၶေႃႈမုၼ်းမၢင်တွၼ်ႈ ၸွမ်းၼင်ႇ ငဝ်းလၢႆး ဢၼ်မီးဝႆႉ။
* **Comparison Operator:** ၶိူင်ႈမၢႆၸႂ်ႉတႅၵ်ႈတူၺ်းၵႃႈၶၼ်။
* **Logical Operator:** ၶိူင်ႈမၢႆၸႂ်ႉႁူမ်ႈ ငဝ်းလၢႆး လၢႆလၢႆဢၼ် (AND, OR, NOT)။
* **Condition:** ငဝ်းလၢႆးၵၢၼ်ႁဵတ်း ဢၼ်မၵ်းမၼ်ႈဝႆႉ ၽၢႆႇလင် WHERE။
