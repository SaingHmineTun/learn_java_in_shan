# Lesson 1: "Foundations: From Flat Files to Relational Databases"

**`Database`** ၼႆႉ ပဵၼ်တီႈသိမ်းၶေႃႈမုၼ်း (Data) ႁႂ်ႈမီးပိူင်သၢင်ႈ လႄႈ ႁႂ်ႈႁဝ်းၶိုၼ်းဢဝ်မႃးၸႂ်ႉလႆႈငၢႆႈငၢႆႈၶႃႈ။ 
မိူဝ်ႈၵွၼ်ႇ ႁဝ်းၵႆႉသိမ်းၶေႃႈမုၼ်းၼႂ်းၾၢႆႇ (File) မိူၼ်ၼင်ႇ Excel ဢမ်ႇၼၼ် Notepad ဢၼ်ႁဝ်းႁွင်ႉဝႃႈ **Flat Files** ၼၼ်ႉၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်းဝႃႈ သင်ႁဝ်းသိမ်းသဵၼ်ႈမၢႆလုၵ်ႈႁဵၼ်း ၼႂ်း Excel ၽႅၼ်ႇလဵဝ်ၵူၺ်း...
* သင်လုၵ်ႈႁဵၼ်းၵေႃႉၼိုင်ႈ ၶိုၼ်းလႅၵ်ႈလၢႆႈ "တီႈယူႇ" (Address)၊ ႁဝ်းလႆႈၵႂႃႇလႅၵ်ႈလၢႆႈၵူႈထႅဝ် (Row) ဢၼ်မီးၸိုဝ်ႈမၼ်း။
* သင်ႁဝ်းလိုမ်းလႅၵ်ႈလၢႆႈပၼ်ဢၼ်ၼိုင်ႈ... ၶေႃႈမုၼ်းႁဝ်းတေ "ၽိတ်းပိူင်ႈ" (Data Inconsistency) ၵႂႃႇၶႃႈ။

**Relational Database (RDBMS)** တေမႃးၵႄႈလိတ်ႈလွင်ႈၼႆႉ ၵွပ်ႈမၼ်းၸႂ်ႉ **"Table"** လၢႆလၢႆဢၼ် မႃးၵွင်ႉၵၢႆႇၵၼ်ၶႃႈ။

### 2. တူဝ်ယင်ႇ Comparison: Excel vs. MySQL
ၼႂ်း Database ႁဝ်းတေၸႅၵ်ႇၶေႃႈမုၼ်းဝႆႉပဵၼ် Table ႁင်းၽႂ်ႁင်းမၼ်းဝႆႉၶႃႈ။

**Example: School System**
* **Table Students:** သိမ်းၸိုဝ်ႈ၊ ဝၼ်းၵိူတ်ႇ၊ ဢီးမႄးလ် (Email)။
* **Table Courses:** သိမ်းၸိုဝ်ႈပၢႆးလိၵ်ႈ၊ ၵႃႈႁဵၼ်း။
* **Relationship:** ႁဝ်းဢဝ် "Student_ID" ၵႂႃႇၵွင်ႉဝႆႉၼႂ်း Table Courses တႃႇတေႁူႉဝႃႈ ၽႂ်ႁဵၼ်းသင်ၶႃႈ။

### 3. ၽွၼ်းလီၵၢၼ်ၸႂ်ႉ Relational Database
ႁဝ်းၸႂ်ႉ MySQL တႅၼ်း Excel ၵွပ်ႈပိူဝ်ႈ မၼ်းမီး 3 ပိူင်လူင်ၼႆႉၶႃႈ။
1.  **Data Integrity:** ၶေႃႈမုၼ်းတေမၢၼ်ႇမႅၼ်ႈယူႇတႃႇသေႇ၊ ဢမ်ႇမီးၶေႃႈမုၼ်းၵိုတ်းၶၢင်း။
2.  **Scalability:** သိမ်းၶေႃႈမုၼ်းလႆႈပဵၼ်လၢၼ်ႉလၢၼ်ႉ (Millions of rows) သေ သွၵ်ႈႁႃလႆႈဝႆး။
3.  **Security:** ႁဝ်းမၵ်းမၼ်ႈလႆႈဝႃႈ "ၽႂ်တေလူလႆႈ"၊ "ၽႂ်တေမႄးလႆႈ" ၶေႃႈမုၼ်းၼၼ်ႉၶႃႈ။

### 4. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ Database
* **Redundancy:** ၵၢၼ်သိမ်းၶေႃႈမုၼ်းသွၼ်ႉၵၼ်လၢႆတီႈ (လွင်ႈၼႆႉ Database တေၸွႆႈယွမ်းပၼ်)။
* **Relationship:** ၵၢၼ်ၵွင်ႉၵၢႆႇၵၼ် ၼႂ်းၵႄႈ Table တင်း Table ၸႂ်ႉ Keys မႃးၵွင်ႉၵၼ်ၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Flat Files:** ၾၢႆႇၶေႃႈမုၼ်းဢၼ်ဢမ်ႇမီးၵၢၼ်ၵွင်ႉၵၢႆႇၵၼ် (မိူၼ်ၼင်ႇ .txt, .csv)။
* **RDBMS:** ပိူင်ၸတ်းၵၢၼ် Database ဢၼ်မီးၵၢၼ်ၵွင်ႉၵၢႆႇၵၼ် (မိူၼ်ၼင်ႇ MySQL)။
* **Data Inconsistency:** ၶေႃႈမုၼ်းဢၼ်ၽိတ်းပိူၼ်ႈ ၵွပ်ႈၵၢၼ်မႄးၶေႃႈမုၼ်းဢမ်ႇတူဝ်ႈထိုင်။
* **Persistence:** ၵၢၼ်သိမ်းၶေႃႈမုၼ်းဝႆႉႁႂ်ႈမၼ်ႈၵိုမ်း ဢမ်ႇႁၢႆလၢႆၵႂႃႇ မိူဝ်ႈဢိုတ်းၶွမ်း။
