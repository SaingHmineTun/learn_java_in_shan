# Lesson 58: "Transactions: Atomicity and the ACID Properties"

**Transaction** မၢႆထိုင် ၸုမ်းၵၢၼ်ႁဵတ်းၵၢၼ် SQL လၢႆလၢႆဢၼ် ဢၼ်ႁဝ်းၼပ်ႉပဵၼ် "ၵၢၼ်ဢၼ်လဵဝ်ၵၼ်" (Single Unit of Work) ၶႃႈ။ 
မၼ်းတေလႆႈ **ယဝ်ႉတူဝ်ႈတင်းမူတ်း** ဢမ်ႇၼၼ် **ဢမ်ႇယဝ်ႉသေဢၼ်** ၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်း "ၵၢၼ်သူင်ႇငိုၼ်းလုၵ်ႉတီႈ App" ၶႃႈ...
1.  **Step 1:** တတ်းငိုၼ်းတီႈ ပပ်ႉၸဝ်ႈၵဝ်ႇ ($100)။
2.  **Step 2:** ထႅမ်ငိုၼ်းၶဝ်ႈၸူး ပပ်ႉဢူၺ်းၵေႃႉ ($100)။

သင် Step 1 ယဝ်ႉသေ Internet ၶၢတ်ႇၵႂႃႇသေ Step 2 ဢမ်ႇယဝ်ႉ... 
ငိုၼ်းၸဝ်ႈၵဝ်ႇတေႁၢႆလၢႆၵႂႃႇလၢႆလၢႆၶႃႈ။ 
**Transaction** ၸွႆႈႁႂ်ႈ သင် Step 2 ဢမ်ႇယဝ်ႉၼႆ Step 1 ၵေႃႈတေပိၼ်ႇၶိုၼ်း (Undo) ႁင်းမၼ်း တွၼ်ႈတႃႇႁႂ်ႈငိုၼ်းဢမ်ႇႁၢႆၶႃႈ။



### 2. ACID Properties (ပၵ်းၵၢၼ်မၼ်ႈၵိုမ်း 4 ပိူင်)
တွၼ်ႈတႃႇတေပဵၼ် Transaction ဢၼ်လီ... မၼ်းတေလႆႈမီး ACID ၶႃႈ:

1.  **Atomicity (ဢၼ်လဵဝ်ၵၼ်):** "All or Nothing" - တေလႆႈယဝ်ႉတင်းမူတ်း ဢမ်ႇၼၼ် ဢမ်ႇယဝ်ႉသေဢၼ်။
2.  **Consistency (မၢၼ်ႇမႅၼ်ႈ):** ၶေႃႈမုၼ်းတေလႆႈမၢၼ်ႇမႅၼ်ႈၸွမ်း ပၵ်းၵၢၼ် Database (Rules/Constraints) တႃႇသေႇ။
3.  **Isolation (ၸႅၵ်ႇၵၼ်):** Transaction သွင်ဢၼ် ဢၼ်ႁဵတ်းၵၢၼ်ၸွမ်းၵၼ် တေလႆႈဢမ်ႇယုင်ႈယၢင်ႈၵၼ်။
4.  **Durability (မၼ်ႈၵိုမ်း):** မိူဝ်ႈယဝ်ႉတူဝ်ႈ (Commit) ယဝ်ႉ... ဢမ်ႇဝႃႈၾႆးတေမွတ်ႇ ႁိုဝ် Server တေတူၵ်းၵေႃႈယဝ်ႉ ၶေႃႈမုၼ်းၼၼ်ႉ တေလႆႈယူႇဝႆႉတႃႇသေႇၶႃႈ။



### 3. Engine ဢၼ်ၸႂ်ႉ Transactions
ၼႂ်း MySQL... Storage Engine ၵူႈဢၼ် ဢမ်ႇၸႂ်ႉလႆႈ Transactions ၶႃႈ:
* **InnoDB:** ၸႂ်ႉလႆႈ (Default Engine ဢၼ်ႁဝ်းၸႂ်ႉၼမ်လိူဝ်ပိူၼ်ႈ)။
* **MyISAM:** ၸႂ်ႉဢမ်ႇလႆႈ (မၼ်းဢမ်ႇမီး ACID)။

### 4. ႁဵတ်းသင်လႄႈ Transactions လွင်ႈယႂ်ႇ?
* **Data Safety:** ႁႄႉၵင်ႈၶေႃႈမုၼ်း ဢမ်ႇႁႂ်ႈႁၢႆလၢႆ မိူဝ်ႈမီး Error ၼႂ်းၵၢင်တၢင်း။
* **Accuracy:** တွၼ်ႈတႃႇမၼ်ႈၸႂ်ဝႃႈ ၶေႃႈမုၼ်းဢၼ်ၵွင်ႉၵၢႆႇၵၼ် (မိူၼ်ၼင်ႇ Stock လႄႈ Order) မိူၼ်ၵၼ်တႃႇသေႇ။
* **Error Recovery:** ၸွႆႈႁႂ်ႈ Database ပိၼ်ႇၶိုၼ်းၸူး ၵႃႈၶၼ်ဢၼ်မၢၼ်ႇမႅၼ်ႈ ဝႆးဝႆးၶႃႈ။

### 5. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ Transactions
* **Auto-commit:** ၼႂ်း MySQL ၼၼ်ႉ မၼ်းတေ Auto-commit ပၼ် Command ၵူႈဢၼ်ၶႃႈ။ ႁဝ်းတေလႆႈၸႂ်ႉ `START TRANSACTION` တွၼ်ႈတႃႇတႄႇပၵ်းၵၢၼ် ACID ၶႃႈ။
* **Performance:** ၵၢၼ်ပိုတ်ႇ Transaction ယၢဝ်းႁႅင်း (Long-running) တေႁဵတ်းႁႂ်ႈ Table ထုၵ်ႇ "Lock" ဝႆႉ သေပိူၼ်ႈၸႂ်ႉဢမ်ႇလႆႈၶႃႈၼႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Transaction:** ၸုမ်းၵၢၼ်ႁဵတ်းၵၢၼ် လၢႆလၢႆဢၼ် ဢၼ်ၼပ်ႉပဵၼ်ၼိုင်ႈဢၼ်။
* **ACID:** ပၵ်းၵၢၼ် 4 ပိူင် တႃႇ Data Integrity (Atomicity, Consistency, Isolation, Durability)။
* **Atomic:** ဢၼ်လဵဝ်ၵၼ် ဢၼ်ၸႅၵ်ႇဢမ်ႇလႆႈယဝ်ႉ။
* **Rollback:** ၵၢၼ်ပိၼ်ႇၶေႃႈမုၼ်းၶိုၼ်း မိူဝ်ႈမီး Error။
