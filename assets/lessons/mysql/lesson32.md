# Lesson 32: "Range and Set Filtering: Using BETWEEN and IN"

မိူဝ်ႈႁဝ်းလူဝ်ႇလိူၵ်ႈၶေႃႈမုၼ်းဢၼ်ယူႇၼႂ်း "ၵႄႈ" (Range) ဢမ်ႇၼၼ် ယူႇၼႂ်း "ၸုမ်း" (Set) ဢၼ်ႁဝ်းလိူၵ်ႈဝႆႉၼၼ်ႉ... 
ႁဝ်းၸႂ်ႉ **`BETWEEN`** လႄႈ **`IN`** ၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်း "ၵၢၼ်လိူၵ်ႈသိုဝ်ႉၶူဝ်း" ၶႃႈ...
* **BETWEEN:** မိူၼ်ၼင်ႇ "ႁႃၶူဝ်းဢၼ်ၵႃႈၶၼ် ယူႇၼႂ်းၵႄႈ 100 ထိုင် 500" (Range)။
* **IN:** မိူၼ်ၼင်ႇ "လိူၵ်ႈဢဝ် ဢၼ်မီး သီလႅင်၊ သီၶဵဝ်၊ လႄႈ သီလိူင်" ၵူၺ်း (Set/List)။



### 2. BETWEEN (လိူၵ်ႈၼႂ်းၵႄႈ)
ၸႂ်ႉတႃႇလိူၵ်ႈၶေႃႈမုၼ်း ဢၼ်ယူႇၼႂ်းၵႄႈၵႃႈၶၼ်သွင်ဢၼ်။ တွင်းဝႆႉဝႃႈ မၼ်းပဵၼ် **Inclusive** (ပႃးတင်းၵႃႈၶၼ် တႄႇ လႄႈ သုတ်း ၼၼ်ႉၶႃႈ)။

```sql
-- လၢႆးတႅမ်ႈ (Lesson 31)
SELECT * FROM products WHERE price >= 100 AND price <= 500;

-- လၢႆးတႅမ်ႈ Professional (Using BETWEEN)
SELECT * FROM products 
WHERE price BETWEEN 100 AND 500;

-- ၸႂ်ႉတင်း ဝၼ်းထီႉ (Dates) ၵေႃႈလႆႈ
SELECT * FROM orders 
WHERE order_date BETWEEN '2026-01-01' AND '2026-03-31';
```

### 3. IN (လိူၵ်ႈၼႂ်းသဵၼ်ႈမၢႆ)
ၸႂ်ႉတႃႇလိူၵ်ႈၶေႃႈမုၼ်း ဢၼ်မၢၼ်ႇမႅၼ်ႈၸွမ်း ၵႃႈၶၼ်ဢၼ်မီးၼႂ်း ၶွပ်ႇမူၼ်း `()` ၶႃႈ။ မၼ်းၸွႆႈႁႂ်ႈႁဝ်းဢမ်ႇလူဝ်ႇတႅမ်ႈ `OR` လၢႆလၢႆၵမ်းၶႃႈ။

```sql
-- လၢႆးတႅမ်ႈ (Using OR)
SELECT * FROM students WHERE city = 'Muse' OR city = 'Namhkam' OR city = 'Kutkai';

-- လၢႆးတႅမ်ႈ Professional (Using IN)
SELECT * FROM students 
WHERE city IN ('Muse', 'Namhkam', 'Kutkai');
```

### 4. ၵၢၼ်ၸႂ်ႉ NOT ႁူမ်ႈၸွမ်း
ႁဝ်းၸၢင်ႈၸႂ်ႉ `NOT` ဝႆႉၽၢႆႇၼႃႈ တႃႇလိူၵ်ႈဢဝ် "ဢၼ်ဢမ်ႇၸႂ်ႈ" ၶႃႈ။

* **`NOT BETWEEN`**: လိူၵ်ႈဢဝ်ၵႃႈၶၼ်ဢၼ်ယူႇ "ၼွၵ်ႈၵႄႈ"။
* **`NOT IN`**: လိူၵ်ႈဢဝ်ၵႃႈၶၼ်ဢၼ် "ဢမ်ႇမီးၼႂ်းသဵၼ်ႈမၢႆ"။

```sql
-- လုၵ်ႈႁဵၼ်း ဢၼ်ဢႃႇယု ဢမ်ႇယူႇၼႂ်းၵႄႈ 18-25
SELECT * FROM students WHERE age NOT BETWEEN 18 AND 25;

-- ၵုၼ်ႇ ဢၼ်ဢမ်ႇၸႂ်ႈ Categories 1, 2, ဢမ်ႇၼၼ် 3
SELECT * FROM products WHERE category_id NOT IN (1, 2, 3);
```

### 5. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ Range & Set
* **Readability:** ၵၢၼ်ၸႂ်ႉ `BETWEEN` လႄႈ `IN` ႁဵတ်းႁႂ်ႈ Code ၸဝ်ႈၵဝ်ႇ လူငၢႆႈလိူဝ်ၵဝ်ႇတႄႉတႄႉၶႃႈ။
* **Performance:** ၼႂ်း MySQL... ၵၢၼ်ၸႂ်ႉ `IN` ၵႆႉဝႆးလိူဝ်ၵၢၼ်ၸႂ်ႉ `OR` လၢႆလၢႆဢၼ် ၵွပ်ႈ DB တေၸတ်းလႅၼ်ႇသဵၼ်ႈမၢႆၼၼ်ႉသေ သွၵ်ႈႁႃဝႆးလိူဝ်ၶႃႈ။
* **NULL values:** တွင်းဝႆႉဝႃႈ `IN` တေဢမ်ႇလိူၵ်ႈၺႃး `NULL` ၶႃႈ (သင်ၶႂ်ႈလႆႈ NULL လူဝ်ႇၸႂ်ႉ `IS NULL` ႁင်းမၼ်းၶႃႈ)။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Range Filtering:** ၵၢၼ်လိူၵ်ႈၶေႃႈမုၼ်းၼႂ်းၵႄႈ (BETWEEN)။
* **Set Filtering:** ၵၢၼ်လိူၵ်ႈၶေႃႈမုၼ်းၼႂ်းၸုမ်း/သဵၼ်ႈမၢႆ (IN)။
* **Inclusive:** ၵၢၼ်ၼပ်ႉပႃးတင်း ၵႃႈၶၼ်တီႈသုတ်းသွင်ၽၢႆႇ။
* **Subquery (Advance):** ၵၢၼ်ၸႂ်ႉ `IN` ႁူမ်ႈတင်း `SELECT` ထႅင်ႈဢၼ်ၼိုင်ႈ (ႁဝ်းတေႁဵၼ်းလိုၼ်းလုၼ်းၶႃႈ)။
