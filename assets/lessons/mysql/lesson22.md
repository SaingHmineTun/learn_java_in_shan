# Lesson 22: "The CHECK Constraint: Business Rules at the DB Level"

မိူဝ်ႈႁဝ်းတႅမ်ႈ App ၼၼ်ႉ ၵႆႉမီးပၵ်းၵၢၼ် (Business Rules) ဢၼ်ႁဝ်းလူဝ်ႇၸႅတ်ႈ ၼမ်လိူဝ် NOT NULL ဢမ်ႇၼၼ် UNIQUE ၶႃႈ။ 
မိူၼ်ၼင်ႇ "ဢႃႇယု လုၵ်ႈႁဵၼ်း တေလႆႈပဵၼ် 18 ၶိုၼ်းၼိူဝ်ၵူၺ်း" ဢမ်ႇၼၼ် "ၵႃႈၶၼ်ၵုၼ်ႇ တေဢမ်ႇလႆႈပဵၼ် Negative Value (-)" ၼၼ်ႉၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်း "ၵၢၼ်ၸႅတ်ႈၶေႃႈမုၼ်း" (Validation) ၶႃႈ...
* **The Rule:** ႁဝ်းမၵ်းမၼ်ႈဝႃႈ ၶေႃႈမုၼ်းဢၼ်တေၶဝ်ႈမႃးၼၼ်ႉ တေလႆႈပဵၼ် "True" (ၸႂ်ႈ) ၸွမ်းၼင်ႇ ငဝ်းလၢႆး (Condition) ဢၼ်ႁဝ်းတႅမ်ႈဝႆႉၵူၺ်း။
* **The Check:** သင်ၵူၼ်းၸႂ်ႉ App သႂ်ႇၶေႃႈမုၼ်း ဢၼ်ၽိတ်းတင်းပၵ်းၵၢၼ်... Database တေႁၢမ်ႈဝႆႉသေ ဢမ်ႇပၼ်သိမ်းၶႃႈ။



### 2. လၢႆးၸႂ်ႉ CHECK Constraint
ႁဝ်းၸၢင်ႈတႅမ်ႈ `CHECK` သႂ်ႇၸွမ်း ၼႂ်းတွၼ်ႈသၢင်ႈ Table ၶႃႈ။

```sql
CREATE TABLE students (
    student_id INT PRIMARY KEY AUTO_INCREMENT,
    name VARCHAR(100) NOT NULL,
    age INT,
    -- ပၵ်းၵၢၼ်: ဢႃႇယု တေလႆႈယႂ်ႇလိူဝ် ဢမ်ႇၼၼ် တူဝ်ႇတင်း 18
    CONSTRAINT chk_age CHECK (age >= 18)
);
```

### 3. တူဝ်ယင်ႇ လၢႆးၸႅတ်ႈလၢႆလၢႆပိူင်
ႁဝ်းၸၢင်ႈၸႂ်ႉ `CHECK` တႃႇၸႅတ်ႈၶေႃႈမုၼ်း ၸိူဝ်းၼႆႉၶႃႈ...

* **ၸႅတ်ႈတူဝ်ၼပ်ႉ (Numeric Range):** `CHECK (price > 0)` (ၵႃႈၶၼ်တေလႆႈယႂ်ႇလိူဝ် 0)
* **ၸႅတ်ႈသဵၼ်ႈမၢႆလိၵ်ႈ (String list):** `CHECK (gender IN ('Male', 'Female', 'Other'))`
* **ၸႅတ်ႈလၢႆလၢႆ Column:** `CHECK (sale_price < original_price)` (ၵႃႈၶၼ်ၶၢႆ တေလႆႈယွမ်းလိူဝ် ၵႃႈၶၼ်တႄႇ)

### 4. ႁဵတ်းသင်လႄႈ လူဝ်ႇၸႂ်ႉ CHECK ၼႂ်း Database?
ၵႆႉမီး ၵူၼ်းထၢမ်မႃးဝႃႈ "ႁဝ်းၸႅတ်ႈ (Validate) ၼႂ်း Flutter ဢမ်ႇၼၼ် Java ၵေႃႈယဝ်ႉယဝ်ႉဢမ်ႇၸႂ်ႈႁႃႉ?"
* **The Answer:** ၵၢၼ်ၸႅတ်ႈၼႂ်း App (Client-side) မၼ်းၸွႆႈပၼ် User Experience လီၵူၺ်း။ ၵူၺ်းၵႃႈ တွၼ်ႈတႃႇ "လွင်ႈလွတ်ႈၽေးၶေႃႈမုၼ်း" (Data Safety) တႄႉတႄႉၼၼ်ႉ... ႁဝ်းလူဝ်ႇၸႅတ်ႈၼႂ်း Database ၸၼ်ႉလိုၼ်းသုတ်းထႅင်ႈပွၵ်ႈၼိုင်ႈ ၵွပ်ႈဝႃႈ ၸၢင်ႈမီးၵူၼ်းလၵ်ႉသူင်ႇၶေႃႈမုၼ်း ၶဝ်ႈ Database ႁဝ်း လူၺ်ႈဢမ်ႇၽၢၼ်ႇ App ၵေႃႈမီးၶႃႈ။

### 5. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ CHECK Constraint
* **MySQL 8.0:** တွင်းဝႆႉဝႃႈ `CHECK` ၼႆႉ တႄႇႁဵတ်းၵၢၼ်တႄႉတႄႉ ၼႂ်း **MySQL Version 8.0.16** ၶိုၼ်ႈၼိူဝ်ၶႃႈ။ (Version ၵဝ်ႇၵဝ်ႇၼၼ်ႉ တႅမ်ႈလႆႈယူႇ ၵူၺ်းၵႃႈ DB တေဢမ်ႇၸႅတ်ႈပၼ်ၶႃႈ)။
* **Naming:** ထုၵ်ႇလီပၼ်ၸိုဝ်ႈ Constraint (မိူၼ်ၼင်ႇ `chk_price`) တႃႇတေငၢႆႈမိူဝ်ႈႁဝ်းလူဝ်ႇမႄးၶႃႈ။

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **CHECK Constraint:** ပၵ်းၵၢၼ်ၸႅတ်ႈၶေႃႈမုၼ်း ၸွမ်းၼင်ႇငဝ်းလၢႆး။
* **Business Rules:** ပၵ်းၵၢၼ်ၼႂ်းၵၢၼ်ႁဵတ်းသၢင်ႈ ဢၼ်မၵ်းမၼ်ႈဝႆႉ ၼႂ်း System။
* **Validation:** ၵၢၼ်ၸႅတ်ႈတူၺ်း လွင်ႈမၢၼ်ႇမႅၼ်ႈ ၶေႃႈမုၼ်း။
* **In-operator:** ၶေႃႈၵႂၢမ်းၸႂ်ႉတႃႇတူၺ်းဝႃႈ ၶေႃႈမုၼ်းၼၼ်ႉ မီးၼႂ်းသဵၼ်ႈမၢႆဢၼ်ႁဝ်းလိူၵ်ႈဝႆႉႁႃႉ။
