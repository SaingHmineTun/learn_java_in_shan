# Lesson 34: "Pattern Matching II: Introduction to REGEXP (Regex)"

ၼႂ်း MySQL... **`REGEXP`** ပဵၼ်ၶိူင်ႈမိုဝ်းဢၼ်မီး "ဢဵၼ်ႁႅင်း" ၼမ်တႄႉတႄႉ။ 
မၼ်းၸႂ်ႉတူဝ်မၵ်းမၼ်ႈ (Special Characters) တႃႇႁႃလိၵ်ႈ ၸွမ်းၼင်ႇ Logic ဢၼ်ႁဝ်းလူဝ်ႇၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်း "ၵၢၼ်ၸႅတ်ႈမၢႆၾူၼ်း" ၶႃႈ...
* **LIKE:** ႁႃလႆႈၵူၺ်းဝႃႈ "တႄႇလူၺ်ႈ 09 ႁႃႉ?"
* **REGEXP:** ႁႃလႆႈပႃးဝႃႈ "တႄႇလူၺ်ႈ 09 သေ တေလႆႈမီးတူဝ်ၼပ်ႉၸွမ်းလင် 7 ထိုင် 9 တူဝ်ႁႃႉ?"



### 2. တူဝ်မၵ်းမၼ်ႈ (Regex Symbols) ဢၼ်ၵႆႉၸႂ်ႉ
ၼႆႉပဵၼ်တူဝ်ပိုၼ်ႉထၢၼ် ဢၼ်ၸဝ်ႈၵဝ်ႇလူဝ်ႇတွင်းဝႆႉၶႃႈ...

* **`^` (Caret):** တႄႇလူၺ်ႈ (Beginning of string)
* **`$` (Dollar):** သုတ်းလူၺ်ႈ (End of string)
* **`|` (Pipe):** ဢမ်ႇၼၼ် (OR)
* **`[abc]`:** မီးတူဝ်လိၵ်ႈ ၼႂ်းၶွပ်ႇ ဢၼ်လႆႈၵေႃႈလႆႈ (Any character inside brackets)
* **`[a-z]`:** မီးတူဝ်လိၵ်ႈ ယူႇၼႂ်းၵႄႈ a ထိုင် z (Range)
* **`.` (Dot):** တႅၼ်းတူဝ်လိၵ်ႈ 1 တူဝ် (Any single character)

### 3. တူဝ်ယင်ႇၵၢၼ်ၸႂ်ႉ REGEXP

**ၵၢၼ်ႁႃၸိုဝ်ႈ ဢၼ်တႄႇလူၺ်ႈ 'Sai' ဢမ်ႇၼၼ် 'Nang':**
```sql
SELECT * FROM students 
WHERE first_name REGEXP '^Sai|^Nang';
```

**ၵၢၼ်ႁႃၸိုဝ်ႈ ဢၼ်မီးတူဝ် 'a', 'b', ဢမ်ႇၼၼ် 'c' ယူႇၼႂ်းမၼ်း:**
```sql
SELECT * FROM students 
WHERE first_name REGEXP '[abc]';
```

**ၵၢၼ်ႁႃမၢႆၾူၼ်း ဢၼ်သုတ်းလူၺ်ႈ 5 ဢမ်ႇၼၼ် 7:**
```sql
SELECT * FROM users 
WHERE phone REGEXP '[57]$';
```

### 4. ၵၢၼ်ၸႂ်ႉ Regex တွၼ်ႈတႃႇ Range
ႁဝ်းၸၢင်ႈမၵ်းမၼ်ႈလႆႈဝႃႈ ႁဝ်းၶႂ်ႈလႆႈတူဝ်ၼပ်ႉ ဢမ်ႇၼၼ် တူဝ်လိၵ်ႈၵူၺ်းၶႃႈ။

```sql
-- ႁႃ Product Code ဢၼ်တႄႇလူၺ်ႈ တူဝ်လိၵ်ႈ A ထိုင် E
SELECT * FROM products 
WHERE product_code REGEXP '^[A-E]';
```

### 5. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ REGEXP
* **Case Sensitivity:** ၼႂ်း MySQL... `REGEXP` တေမိူၼ် `LIKE` ၶႃႈ၊ မၼ်းဢမ်ႇၸႅၵ်ႇတူဝ်ယႂ်ႇတူဝ်လဵၵ်ႉ (Case-insensitive) သင်ၸဝ်ႈၵဝ်ႇဢမ်ႇၸႂ်ႉ `REGEXP BINARY` ၶႃႈ။
* **Performance:** `REGEXP` **ထိူင်းလိူဝ်** `LIKE` တႄႉတႄႉ ၵွပ်ႈမၼ်းလူဝ်ႇၸႂ်ႉ Logic ၼမ်တႃႇသွၵ်ႈႁႃ။ ၵွပ်ႈၼၼ်... သင်ႁႃလႆႈလူၺ်ႈ `LIKE` ၼႆ ၸႂ်ႉ `LIKE` တေလီလိူဝ်ၶႃႈ။
* **Full String Match:** တွင်းဝႆႉဝႃႈ `REGEXP` တေႁႃ "တွၼ်ႈလႂ်ၵေႃႈလႆႈ" ၼႂ်းလိၵ်ႈ (မိူၼ်ၸႂ်ႉ `%...%`) သင်ၸဝ်ႈၵဝ်ႇဢမ်ႇၸႂ်ႉ `^` လႄႈ `$` တႃႇမၵ်းမၼ်ႈၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **REGEXP (Regular Expression):** ပိူင်တႅမ်ႈ Logic တႃႇသွၵ်ႈႁႃႁွႆးလိၵ်ႈ။
* **Wildcard:** တူဝ်တႅၼ်းၼႂ်း Regex ဢၼ်မီးတီႈပွင်ႇပႅၵ်ႇပိူင်ႈၵၼ်။
* **Caret (^):** မၢႆထိုင် တီႈတႄႇမၼ်း။
* **Pipe (|):** မၢႆထိုင် "ဢမ်ႇၼၼ်" (Logical OR)။
