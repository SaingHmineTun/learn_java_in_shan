# Lesson 38: "Math Functions: ABS, ROUND, CEIL, and FLOOR"

ၼႂ်း MySQL မီး Built-in Functions တႃႇၸွႆႈၼပ်ႉသွၼ်ႇတူဝ်ၼပ်ႉလႆႈဝႆးလိူဝ်ၵၢၼ်တႅမ်ႈ Code ၼႂ်း App ၶႃႈ။ ႁဝ်းတေမႃးတူၺ်း 4 တူဝ် ဢၼ်ၸႂ်ႉၼမ်လိူဝ်ပိူၼ်ႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်း "ၵၢၼ်ၼပ်ႉသွၼ်ႇၵႃႈၶၼ်ၵုၼ်ႇ" ၶႃႈ...
* **ABS:** ဢဝ်ၵႃႈၶၼ် "ပွတ်းၼိူဝ်" (Positive) တႃႇသေႇ၊ ဢမ်ႇဢဝ်ၶိူင်ႈပွတ်းတႂ်ႈ (- )။
* **ROUND:** ၵၢၼ် "တတ်းၸုၵ်ႉ" ၸွမ်းၼင်ႇပၵ်းၵၢၼ် (သင် 0.5 ၶိုၼ်ႈၼိူဝ် ၼပ်ႉၶိုၼ်ႈ၊ တႂ်ႈ 0.5 ၼပ်ႉလူင်း)။
* **CEIL:** ၼပ်ႉၶိုၼ်ႈ "ၸူးတူဝ်ၼပ်ႉတဵမ် ဢၼ်ၸမ်သုတ်း" တႃႇသေႇ (Ceiling)။
* **FLOOR:** ၼပ်ႉလူင်း "ၸူးတူဝ်ၼပ်ႉတဵမ် ဢၼ်ၸမ်သုတ်း" တႃႇသေႇ (Floor)။



### 2. ABS (Absolute Value)
ၸႂ်ႉမိူဝ်ႈႁဝ်းလူဝ်ႇလႆႈ "ၵႆ - ၸမ်" ဢမ်ႇၼၼ် "တၢင်းပႅၵ်ႇပိူင်ႈ" ဢၼ်ဢမ်ႇၶႂ်ႈလႆႈတူဝ်ၼပ်ႉပွတ်းတႂ်ႈ (-) ၶႃႈ။

```sql
-- ၽွၼ်းလႆႈတေပဵၼ် 50 တႃႇသေႇ
SELECT ABS(-50) AS positive_value;

-- ႁႃတၢင်းပႅၵ်ႇပိူင်ႈၵႃႈၶၼ် (ဢမ်ႇလႅၵ်ႈဝႃႈ ဢၼ်လႂ်ယႂ်ႇလိူဝ်)
SELECT ABS(actual_price - budget_price) AS price_diff FROM orders;
```

### 3. ROUND (ၵၢၼ်တတ်းၸုၵ်ႉ ၸွမ်းပၵ်းၵၢၼ်)
ၸႂ်ႉတႃႇတတ်းတူဝ်ၼပ်ႉၸုၵ်ႉ (Decimal) ႁႂ်ႈၵိုတ်းၸွမ်းတၢင်းၼမ် ဢၼ်ႁဝ်းလူဝ်ႇၶႃႈ။

```sql
-- ၸုၵ်ႉ 2 တူဝ် (ၽွၼ်းလႆႈ: 123.46 - ၵွပ်ႈ 5 ၼပ်ႉၶိုၼ်ႈ)
SELECT ROUND(123.456, 2);

-- တူဝ်ၼပ်ႉထူၼ်ႈ (ၽွၼ်းလႆႈ: 123)
SELECT ROUND(123.456, 0);
```

### 4. CEIL လႄႈ FLOOR (ၼပ်ႉၶိုၼ်ႈ လႄႈ ၼပ်ႉလူင်း)
သွင်တူဝ်ၼႆႉ မၼ်းတေဢမ်ႇတူၺ်းဝႃႈ ပဵၼ် 0.1 ဢမ်ႇၼၼ် 0.9 ၶႃႈ၊ မၼ်းတေတႅတ်ႈတေႃးၸွမ်းႁူဝ်မၼ်းၵမ်းလဵဝ်။

* **CEIL (Ceiling):** ၵႆႉၸႂ်ႉတႃႇၼပ်ႉသွၼ်ႇ "တၢင်းၼမ်ၼႃႈ" (Total Pages) ၼႂ်း App ၸဝ်ႈၵဝ်ႇ။
* **FLOOR:** ၵႆႉၸႂ်ႉတႃႇၼပ်ႉ "ဢႃႇယု" (Age) လုၵ်ႉတီႈဝၼ်းၵိူတ်ႇ။

```sql
-- ၽွၼ်းလႆႈ: 11 (ၼပ်ႉၶိုၼ်ႈတႃႇသေႇ)
SELECT CEIL(10.1);

-- ၽွၼ်းလႆႈ: 10 (ၼပ်ႉလူင်းတႃႇသေႇ)
SELECT FLOOR(10.9);
```

### 5. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ Math Functions
* **Data Types:** သင်ၸဝ်ႈၵဝ်ႇၸႂ်ႉ `ROUND/CEIL/FLOOR` ၸွမ်းတူဝ်ၼပ်ႉထူၼ်ႈ (Integer) ယူႇယဝ်ႉၼၼ်ႉ... ၽွၼ်းလႆႈတေဢမ်ႇလႅၵ်ႈလၢႆႈသင်ၶႃႈ။
* **Nesting:** ႁဝ်းၸၢင်ႈၸႂ်ႉ Function ၸိူဝ်းၼႆႉ "သွၼ်ႉ" (Nest) ၵၼ်လႆႈ (မိူၼ်ၼင်ႇ `ROUND(ABS(col), 2)`) ၶႃႈ။
* **Precision:** တွင်းဝႆႉဝႃႈ `ROUND` ၸၢင်ႈပႃး Error ဢိတ်းဢွတ်းသင်ၸႂ်ႉတင်း `FLOAT` ၵွပ်ႈၼၼ် ထုၵ်ႇလီၸႂ်ႉတင်း `DECIMAL` ၶႃႈၼႃႈ။

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Absolute Value:** ၵႃႈၶၼ်ဢၼ်ဢမ်ႇမီးၶိူင်ႈမၢႆ (Positive only)။
* **Rounding:** ၵၢၼ်တတ်းတူဝ်ၼပ်ႉၸုၵ်ႉ ႁႂ်ႈၸမ်သုတ်း။
* **Ceiling:** ၵၢၼ်ၼပ်ႉၶိုၼ်ႈၸူးတူဝ်ၼပ်ႉထူၼ်ႈ ဢၼ်ယႂ်ႇလိူဝ်မၼ်း။
* **Floor:** ၵၢၼ်ၼပ်ႉလူင်းၸူးတူဝ်ၼပ်ႉထူၼ်ႈ ဢၼ်ယွမ်းလိူဝ်မၼ်း။
