# Lesson 4: "Connection Architecture: Ports, Hosts, and Socket Basics"

မိူဝ်ႈႁဝ်းတႅမ်ႈ App ၼႂ်း Flutter ဢမ်ႇၼၼ် JavaFX သေ ၶႂ်ႈဢဝ်ၶေႃႈမုၼ်းၵႂႃႇသိမ်းၼႂ်း MySQL ၼၼ်ႉ... 
ႁဝ်းလူဝ်ႇမီး "သဵၼ်ႈတၢင်း" တႃႇလၢတ်ႈတေႃႇၵၼ်ၶႃႈ။ လွင်ႈၼႆႉ ႁဝ်းႁွင်ႉဝႃႈ **Connection Architecture** ၼႆၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းၶႂ်ႈသူင်ႇ "လိၵ်ႈ" ၵႂႃႇၸူး ၵူၼ်းၵေႃႉၼိုင်ႈ ဢၼ်ယူႇၼႂ်း ႁူင်းႁဵၼ်းလူင်ၼိုင်ႈဢၼ်ၶႃႈ...
* **Host:** ပဵၼ် "တီႈယူႇႁူင်းႁဵၼ်း" (Address)။
* **Port:** ပဵၼ် "မၢႆၵၢၼ်း/မၢႆလုၵ်း" (Room Number) တီႈဢၼ်ၵေႃႉၼၼ်ႉယူႇဝႆႉ။
* **Protocol/Socket:** ပဵၼ် "ၽႃႇသႃႇလိၵ်ႈ" ဢၼ်ႁဝ်းတႅမ်ႈ သေသူင်ႇၵႂႃႇၸူးၵၼ်ၶႃႈ။



### 2. Host (တီႈယူႇ Server)
Host ၼႆႉ ပဵၼ်တူဝ်မၵ်းမၼ်ႈဝႃႈ MySQL Server ႁဝ်း မီးဝႆႉတီႈၶွမ်းလႂ်ၶႃႈ။
* **Localhost (127.0.0.1):** သင်ႁဝ်းတႅမ်ႈ App လႄႈ MySQL ဝႆႉၼႂ်းၶွမ်းတူဝ်လဵဝ်ၵၼ်... ႁဝ်းၸႂ်ႉ `localhost` ၶႃႈ။
* **Remote Host (IP Address):** သင် Database ႁဝ်းမီးဝႆႉတီႈ Cloud ဢမ်ႇၼၼ် ၶွမ်းထႅင်ႈတူဝ်ၼိုင်ႈ... ႁဝ်းလႆႈၸႂ်ႉ IP Address (မိူၼ်ၼင်ႇ `192.168.1.5`) ၶႃႈ။

### 3. Port (သဵၼ်ႈတၢင်းၶဝ်ႈ)
ၼႂ်းၶွမ်းၼိုင်ႈတူဝ် မီးသဵၼ်ႈတၢင်းၶဝ်ႈ (Port) လၢႆလၢႆဢၼ်ၶႃႈ။
* **MySQL Default Port:** ပဵၼ် **`3306`** ၶႃႈ။
* **Why Port?** ၶွမ်းႁဝ်းလူဝ်ႇႁူႉဝႃႈ ၶေႃႈမုၼ်းဢၼ်သူင်ႇမႃးၼၼ်ႉ တေလႆႈသူင်ႇပၼ် Software တူဝ်လႂ် (MySQL ႁႃႉ? Web Server ႁႃႉ?) ၼႆၶႃႈ။

### 4. Socket လႄႈ TCP/IP
ၼႆႉပဵၼ် "လၢႆးၵွင်ႉၵၢႆႇ" တႄႉတႄႉမၼ်းၶႃႈ။
* **TCP/IP:** ပဵၼ်လၢႆးၵွင်ႉၵၢႆႇ ၸွမ်းၼင်ႇ Network ဢၼ်ႁဝ်းၸႂ်ႉၵူႈဝၼ်း။
* **Unix Socket:** ပဵၼ်လၢႆးၵွင်ႉၵၢႆႇ ဢၼ်ဝႆးလိူဝ် (Fastest) ၵူၺ်းၵႃႈ ၸႂ်ႉလႆႈမိူဝ်ႈ App လႄႈ DB ယူႇၼႂ်းၶွမ်းတူဝ်လဵဝ်ၵၼ် (ၼႂ်း Linux/macOS) ၵူၺ်းၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **IP Address:** မၢႆတီႈယူႇၶွမ်း ၼႂ်း Internet/Network။
* **3306:** Port ဢၼ်ပဵၼ် Standard တွၼ်ႈတႃႇ MySQL။
* **Connection String:** ၶေႃႈၵႂၢမ်းဢၼ်မီး Host, Port, User, Password တႃႇ App ၸႂ်ႉၵွင်ႉၸူး DB။
* **Endpoint:** တီႈသုတ်းသဵၼ်ႈတၢင်းၵွင်ႉၵၢႆႇ (Host + Port)။
