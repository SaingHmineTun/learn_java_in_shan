# Lesson 41: "The INNER JOIN: Extracting Matching Records"

မိူဝ်ႈႁဝ်းၸႂ်ႉ **INNER JOIN**... Database တေတူၺ်း Table သွင်ဢၼ်သေ 
လိူၵ်ႈဢဝ် ထႅဝ် (Rows) ဢၼ်မီးၵႃႈၶၼ် **PK** လႄႈ **FK** မိူၼ်ၵၼ်တႅတ်ႈတေႃး ၵူၺ်းၶႃႈ။
ထႅဝ်ဢၼ် ဢမ်ႇမိူၼ်ၵၼ်ၼၼ်ႉ (မိူၼ်ၼင်ႇ လုၵ်ႈႁဵၼ်း ဢၼ်ဢမ်ႇပႆႇမီးႁွင်ႈႁဵၼ်း)... မၼ်းတေဢမ်ႇဢဝ်ဢွၵ်ႇမႃးၼႄၶႃႈ။

ယေႃးၶႃႈ! ၼင်ႇႁႂ်ႈပွင်ႇၸႂ်ငၢႆႈလိူဝ်ၵဝ်ႇၼၼ်ႉ ႁဝ်းမႃးဝူၼ်ႉတူၺ်းတူဝ်ယင်ႇ **"ပွႆးၵိၼ်ဝၼ်းၵိူတ်ႇ"** ၶႃႈၼႃႈ။

---

### 1. ပိူင်သၢင်ႈ (The Concept) - တူဝ်ယင်ႇငၢႆႈငၢႆႈ
ဝူၼ်ႉတူၺ်းဝႃႈ ၸဝ်ႈၵဝ်ႇမီးသဵၼ်ႈမၢႆ (2) ဢၼ်ၶႃႈ:
* **Table A (သဵၼ်ႈမၢႆၶႅၵ်ႇ):** မီးၸိုဝ်ႈၵူၼ်းဢၼ်ၸဝ်ႈၵဝ်ႇ "ႁွင်ႉ" (Invite) မႃးတင်းမူတ်း။
* **Table B (သဵၼ်ႈမၢႆၵူၼ်းမႃး):** မီးၸိုဝ်ႈၵူၼ်းဢၼ် "မႃးၽႅဝ်" တီႈပွႆးတႄႉတႄႉ။

**INNER JOIN** ပဵၼ်ၵၢၼ်သွၵ်ႈႁႃ **"ၵူၼ်းဢၼ်မီးၸိုဝ်ႈ ၼႂ်းတင်း သွင်သဵၼ်ႈမၢႆ"** ၶႃႈ။
* သင်ၸဝ်ႈၵဝ်ႇ "ႁွင်ႉ" (Invite) ဝႆႉ ၵူၺ်းၵႃႈ "မၼ်းဢမ်ႇမႃး" (No match) -> **ဢမ်ႇဢွၵ်ႇၼႄ**။
* သင် "မၼ်းမႃး" (Show up) ၵူၺ်းၵႃႈ ၸဝ်ႈၵဝ်ႇ "ဢမ်ႇလႆႈႁွင်ႉ" (No invite) -> **ဢမ်ႇဢွၵ်ႇၼႄ**။
* **INNER JOIN** တေၼႄ: **"ၵူၼ်းဢၼ်ၸဝ်ႈၵဝ်ႇႁွင်ႉဝႆႉ လႄႈ မၼ်းၵေႃႈမႃးၽႅဝ်တႄႉတႄႉ"** ၵူၺ်းၼၼ်ႉၶႃႈ။

---


### 2. Syntax ၵၢၼ်တႅမ်ႈ INNER JOIN
ႁဝ်းတေလႆႈမၵ်းမၼ်ႈ Table ဢၼ်တေ Join လႄႈ ပၵ်းၵၢၼ် `ON` (PK = FK) ၶႃႈ။

```sql
SELECT 
    s.student_name, 
    c.class_name
FROM students AS s
INNER JOIN classes AS c 
    ON s.class_id = c.class_id;
```
* **FROM:** Table ဢွၼ်တၢင်း (Left Table)
* **INNER JOIN:** Table ဢၼ်တေမႃးႁူမ်ႈၸွမ်း (Right Table)
* **ON:** ပိူင် ဢၼ်ၸွႆႈၵွင်ႉ ID ၶဝ်သွင်ၽၢႆႇ

### 3. တူဝ်ယင်ႇ: ၵၢၼ်ႁူမ်ႈ Table တႄႉတႄႉ

ဝူၼ်ႉတူၺ်း Table ႁဝ်းမီးၼင်ႇၼႆၶႃႈ:
* **Students:** `[ID: 1, Name: 'Mao', Class_ID: 10]`, `[ID: 2, Name: 'Sang', Class_ID: NULL]`
* **Classes:** `[ID: 10, Name: 'Computer']`, `[ID: 11, Name: 'English']`

**ၽွၼ်းလႆႈ (Result):**
မၼ်းတေဢွၵ်ႇမႃးၵူၺ်း `Mao - Computer` ၶႃႈ။
* ၵွပ်ႈသင်? ၵွပ်ႈဝႃႈ 'Sang' ဢမ်ႇမီး Class_ID (NULL) သေ 'English' ၵေႃႈ ဢမ်ႇပႆႇမီးလုၵ်ႈႁဵၼ်းၵွင်ႉၸူးၶႃႈ။ **INNER JOIN တေဢဝ်ၵူၺ်း ဢၼ်ၵွင်ႉၵၼ်ၺႃး (Match) ၶႃႈ။**

### 4. ႁဵတ်းသင်လႄႈ လူဝ်ႇၸႂ်ႉ INNER JOIN?
* **Data Accuracy:** တႃႇႁႂ်ႈမၼ်ႈၸႂ်ဝႃႈ ၶေႃႈမုၼ်းဢၼ်ၼႄၼၼ်ႉ မီးသဵၼ်ႈၵွင်ႉတဵမ်ထူၼ်ႈ တင်းသွင်ၽၢႆႇ။
* **Default Join:** ၵမ်ႈၼမ် သင်ႁဝ်းတႅမ်ႈ `JOIN` လၢႆလၢႆ (ဢမ်ႇသႂ်ႇ INNER)... MySQL တေၼပ်ႉပဵၼ် `INNER JOIN` ႁင်းမၼ်းၶႃႈ။


### 5. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ INNER JOIN
* **Filter Before Join:** ႁဝ်းၸၢင်ႈသႂ်ႇ `WHERE` ၽၢႆႇလင် JOIN တႃႇလိူၵ်ႈၶေႃႈမုၼ်းထႅင်ႈပွၵ်ႈၼိုင်ႈၶႃႈ။
* **Ambiguous Columns:** သင် Table သွင်ဢၼ် မီး Column ၸိုဝ်ႈမိူၼ်ၵၼ် (မိူၼ်ၼင်ႇ `name` တင်း `name`)... တေလႆႈၸႂ်ႉ `s.name` လႄႈ `c.name` တႃႇၸႅၵ်ႇၶႃႈၼႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **INNER JOIN:** ၵၢၼ်ႁူမ်ႈ Table ဢၼ်ဢဝ် ၶေႃႈမုၼ်းဢၼ်မၢၼ်ႇမႅၼ်ႈၸွမ်းၵၼ် တင်းသွင်ၽၢႆႇ။
* **Intersection:** တွၼ်ႈဢၼ်ၶေႃႈမုၼ်း သွင် Table မႃးၺႃးၵၼ်။
* **Match:** ၵၢၼ်ဢၼ်ၵႃႈၶၼ် PK လႄႈ FK မိူၼ်ၵၼ်။
