# Lesson 18: "Numeric Data: INT, DECIMAL, and Floating Point Logic"

မိူဝ်ႈႁဝ်းသိမ်း "ဢႃႇယု"၊ "ၵႃႈၶၼ်ၵုၼ်ႇ" ဢမ်ႇၼၼ် "မၢႆၼပ်ႉလုၵ်ႈႁဵၼ်း" ၼၼ်ႉ... ႁဝ်းတေလႆႈလိူၵ်ႈၸႂ်ႉ Data Type ဢၼ်သိမ်းပၼ်လႆႈ တူဝ်ၼပ်ႉယဝ်ႉၶႃႈ။

### 1. ပိူင်သၢင်ႈ (The Concept)
ႁဝ်းမႃးတူၺ်း "ၶိူင်ႈၼပ်ႉ" 3 ပိူင်ၶႃႈ...
* **INT (Integer):** မိူၼ်ၼင်ႇ "ၵၢၼ်ၼပ်ႉႁူဝ်ၵူၼ်း" (1, 2, 3...) - တူဝ်ၼပ်ႉဢၼ်ဢမ်ႇမီး "တူဝ်ၸုတ်ႇ" (Decimal)။
* **DECIMAL:** မိူၼ်ၼင်ႇ "ၶိူင်ႈၼပ်ႉငိုၼ်း" (Fixed-point) - တူဝ်ၼပ်ႉ ဢၼ်မၢၼ်ႇမႅၼ်ႈတႅတ်ႈတေႃးသုတ်း။
* **FLOAT/DOUBLE:** မိူၼ်ၼင်ႇ "ၶိူင်ႈတႅၵ်ႈပၢႆးသၢႆႊ" (Floating-point) - တူဝ်ၼပ်ႉဢၼ်ယႂ်ႇၼမ်လိူဝ်ပိူၼ်ႈ ၵူၺ်းၵႃႈ ၸၢင်ႈမီးလွင်ႈၶၢတ်ႇဢိတ်းဢွတ်း။



### 2. Integers (တူဝ်ၼပ်ႉတဵမ်)
ၸႂ်ႉတွၼ်ႈတႃႇ ၶေႃႈမုၼ်းဢၼ်ဢမ်ႇမီး တူဝ်ၸုတ်ႇ (Decimal) ၶႃႈ။
* **TINYINT:** သိမ်းလႆႈ -128 ထိုင် 127 (Best for: Age, Status 0/1)။
* **INT:** သိမ်းလႆႈထိုင် 2 ပီႇလီႇယႅၼ်ႇ (Best for: Primary Keys, Counts)။
* **BIGINT:** သိမ်းလႆႈတူဝ်ၼပ်ႉဢၼ်ယႂ်ႇၼမ်လိူဝ် (Best for: Global IDs)။
* **UNSIGNED:** သင်ႁဝ်းမၵ်းမၼ်ႈ `UNSIGNED`... မၼ်းတေသိမ်းလႆႈ 0 ၶိုၼ်းၼိူဝ် သေမၼ်းတေႁဵတ်းႁႂ်ႈ သိမ်းတူဝ်ၼပ်ႉလႆႈ ၼမ်လိူဝ်ၵဝ်ႇ 2 ပုၼ်ႈၶႃႈ။

### 3. DECIMAL (တူဝ်ၼပ်ႉတႅတ်ႈတေႃး)
ၼႆႉပဵၼ် တူဝ်ဢၼ်လွင်ႈယႂ်ႇသုတ်း တွၼ်ႈတႃႇ "ငိုၼ်း" ၶႃႈ။
* **Syntax:** `DECIMAL(P, D)` -> `P` ပဵၼ်တူဝ်ၼပ်ႉတင်းသဵင်ႈ၊ `D` ပဵၼ်တူဝ်ၼပ်ႉၽၢႆႇလင်ၸုၵ်ႉ။
* **Example:** `DECIMAL(10, 2)` မၢႆထိုင်ဝႃႈ သိမ်းလႆႈတူဝ်ၼပ်ႉတင်းမူတ်း 10 တူဝ် သေ မီးၽၢႆႇလင်ၸုတ်ႇလႆႈ 2 တူဝ် (မိူၼ်ၼင်ႇ `99999999.99`)။
* **Best for:** ငိုၼ်း (Money/Currency), ၵႃႈၶၼ်ၵုၼ်ႇ (Prices)။

### 4. FLOAT လႄႈ DOUBLE (Floating-point)
ၸႂ်ႉတွၼ်ႈတႃႇ ၶေႃႈမုၼ်းပၢႆးသၢႆႊ ဢၼ်တူဝ်ၼပ်ႉယၢဝ်းၼမ်လိူဝ်ပိူၼ်ႈၶႃႈ။
* **Warning:** တူဝ်ၼပ်ႉၸိူဝ်းၼႆႉ ၸၢင်ႈမီးလွင်ႈ "Rounding Errors" (ၶေႃႈမုၼ်းၸၢင်ႈၶၢတ်ႇဢိတ်းဢွတ်း)။ ၵွပ်ႈၼၼ် **ႁၢမ်ႈၸႂ်ႉ** တႃႇသိမ်းငိုၼ်းၶႃႈၼႃႈ။
* **Best for:** ပၢႆးသၢႆႊ (Scientific data), လွင်ႈတႅၵ်ႈၼမ်ႉၼၵ်း (Weight)။



### 5. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ Numeric Selection
* **Money Rule:** တႃႇသိမ်းငိုၼ်း... ၸႂ်ႉ **DECIMAL** တႃႇသေႇၶႃႈ။
* **ID Rule:** တႃႇ Primary Key... ၸႂ်ႉ **INT UNSIGNED** ဢမ်ႇၼၼ် **BIGINT UNSIGNED** တႃႇသေႇၶႃႈ။
* **Boolean:** ၼႂ်း MySQL ဢမ်ႇမီး `bool` တႄႉတႄႉ... ႁဝ်းၵႆႉၸႂ်ႉ **TINYINT(1)** (0 ပဵၼ် False, 1 ပဵၼ် True) ၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Precision:** တၢင်းတႅတ်ႈတေႃး (တူဝ်ၼပ်ႉတင်းသဵင်ႈ ၼႂ်း DECIMAL)။
* **Scale:** တူဝ်ၼပ်ႉၽၢႆႇလင်ၸုတ်ႇ (ၼႂ်း DECIMAL)။
* **Unsigned:** တူဝ်ၼပ်ႉဢၼ်ဢမ်ႇမီး "Negative Value" (-) သေတႄႇတီႈ 0 ၵႂႃႇ။
* **Floating Point:** တူဝ်ၼပ်ႉဢၼ်မီး "ၸုတ်ႇ" ဢၼ်ၸၢင်ႈၶၢႆႉလႆႈ (ၸၢင်ႈမီးလွင်ႈၶၢတ်ႇဢိတ်းဢွတ်း)။
