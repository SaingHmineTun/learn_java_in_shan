# Lesson 55: "Iterative Logic: WHILE and REPEAT Loops in SQL"

ၵၢၼ်ႁဵတ်း Loop ၼႂ်း SQL မၼ်းမိူၼ်တင်းၵၢၼ်တႅမ်ႈ Code ၼႂ်း Dart ဢမ်ႇၼၼ် Java ၶႃႈ။ 
မၼ်းတေႁဵတ်းၵၢၼ်သွၼ်ႉၵၼ် ၵႂႃႇတိၵ်းတိၵ်း တေႃႇပေႃးထိုင် "ငဝ်းလၢႆး" (Condition) ဢၼ်ႁဝ်းမၵ်းမၼ်ႈဝႆႉၶႃႈ။

### 1. WHILE Loop (ႁဵတ်းၵၢၼ်မိူဝ်ႈမၢၼ်ႇမႅၼ်ႈ)
**`WHILE`** တေၵူတ်ႇထတ်းငဝ်းလၢႆး "ဢွၼ်တၢင်း" တေႁဵတ်းၵၢၼ်ၶႃႈ။ 
သင်မၢၼ်ႇမႅၼ်ႈ (True) မၼ်းတေႁဵတ်း၊ သင်ဢမ်ႇမၢၼ်ႇမႅၼ်ႈ (False) မၼ်းတေၶၢမ်ႈၵႂႃႇၵမ်းလဵဝ်ၶႃႈ။



**တူဝ်ယင်ႇ:** ၼပ်ႉ 1 ထိုင် 5 သေ INSERT ၶဝ်ႈ Table ၸူဝ်ႈၶၢဝ်း။
```sql
DECLARE counter INT DEFAULT 1;

WHILE counter <= 5 DO
    INSERT INTO test_log(msg) VALUES (CONCAT('ပွၵ်ႈၵမ်း - ', counter));
    SET counter = counter + 1;
END WHILE;
```

### 2. REPEAT Loop (ႁဵတ်းၵၢၼ်တေႃႇပေႃးမၢၼ်ႇမႅၼ်ႈ)
**`REPEAT`** တေႁဵတ်းၵၢၼ် "ဢွၼ်တၢင်း" ၼိုင်ႈပွၵ်ႈ သေၸင်ႇၵူတ်ႇထတ်း ငဝ်းလၢႆးၽၢႆႇလင်ၶႃႈ။ 
မၼ်းတေႁဵတ်းၵၢၼ်သွၼ်ႉၵၼ် "တေႃႇပေႃး" (Until) ငဝ်းလၢႆးၼၼ်ႉပဵၼ် True ၶႃႈ။



**တူဝ်ယင်ႇ:**
```sql
DECLARE counter INT DEFAULT 1;

REPEAT
    INSERT INTO test_log(msg) VALUES (CONCAT('Repeat - ', counter));
    SET counter = counter + 1;
UNTIL counter > 5
END REPEAT;
```

### 3. တၢင်းပႅၵ်ႇပိူင်ႈ (WHILE vs REPEAT)
* **WHILE:** ၵူတ်ႇထတ်းဢွၼ်တၢင်း (Check first)။ သင်ငဝ်းလၢႆးၽိတ်းဝႆႉမိူဝ်ႈတႄႇမၼ်း... မၼ်းတေဢမ်ႇႁဵတ်းၵၢၼ်သေပွၵ်ႈၶႃႈ။
* **REPEAT:** ႁဵတ်းၵၢၼ်ဢွၼ်တၢင်း (Execute first)။ မၼ်းတေႁဵတ်းၵၢၼ် တီႈဢေႇသုတ်ႉ "ၼိုင်ႈပွၵ်ႈ" ဢမ်ႇဝႃႈငဝ်းလၢႆး ပဵၼ်သင်ၵေႃႈယဝ်ႉ။

### 4. ၵၢၼ်ၸႂ်ႉ LEAVE (Break) လႄႈ ITERATE (Continue)
* **LEAVE:** ၸႂ်ႉတွၼ်ႈတႃႇ "ဢွၵ်ႇ" (Exit) လုၵ်ႉၼႂ်း Loop ၵမ်းလဵဝ် (မိူၼ် `break`)။
* **ITERATE:** ၸႂ်ႉတွၼ်ႈတႃႇ "ၶၢမ်ႈ" ၵႂႃႇႁဵတ်းပွၵ်ႈၼႃႈၵမ်းလဵဝ် (မိူၼ် `continue`)။

### 5. ပၵ်းၵၢၼ်တွင်း တွၼ်ႈတႃႇ Looping
* **Infinite Loop:** ၾၢင်ႉတူၺ်းလီလီ တွၼ်ႈတႃႇ `SET counter = counter + 1;`။ သင်ၸဝ်ႈၵဝ်ႇလိုမ်းထႅမ်ၵႃႈၶၼ်... Loop တေႁဵတ်းၵၢၼ်ဢမ်ႇၵိုတ်း သေႁဵတ်းႁႂ်ႈ Database "ၶႅင်" (Hang) ၵႂႃႇလႆႈၶႃႈ။
* **Performance:** ၵၢၼ်ၸႂ်ႉ Loop ၼႂ်း SQL ၼမ်လူၼ်ႉၵႂႃႇ ၸၢင်ႈႁဵတ်းႁႂ်ႈ Server ထိူင်းၶႃႈ။ သင်ၸၢင်ႈၸႂ်ႉ `INSERT INTO ... SELECT` တႅၼ်းလႆႈ... ၼၼ်ႉတေဝႆးလိူဝ်ၶႃႈ။
* **Labels:** ႁဝ်းၸၢင်ႈပၼ်ၸိုဝ်ႈ (Label) တႃႇ Loop (မိူၼ်ၼင်ႇ `my_loop: WHILE ...`) တွၼ်ႈတႃႇၸႂ်ႉတင်း `LEAVE` ႁႂ်ႈငၢႆႈၶႃႈ။

---

### ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း

* **Iteration:** လွင်ႈႁဵတ်းၵၢၼ်သွၼ်ႉၶိုၼ်း ၼိုင်ႈပွၵ်ႈယဝ်ႉၼိုင်ႈပွၵ်ႈ။
* **Condition:** ငဝ်းလၢႆး ဢၼ်မၵ်းမၼ်ႈဝႃႈ တေသိုပ်ႇႁဵတ်း ႁိုဝ် တေၵိုတ်းႁိုဝ်။
* **Infinite Loop:** လွင်ႈႁဵတ်းၵၢၼ် ဢမ်ႇမီးဝၼ်းသုတ်း (ပဵၼ်လွင်ႈၽိတ်းပိူင်လူင်)။
* **Pre-test Loop:** Loop ဢၼ်ၵူတ်ႇထတ်း ငဝ်းလၢႆးဢွၼ်တၢင်း (WHILE)။
