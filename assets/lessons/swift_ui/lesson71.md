

## **Module 9 | Lesson 71: Preparing Metadata & ASO**

### **1. Metadata ပဵၼ်သင်?**

Metadata ပဵၼ်ၶေႃႈမုၼ်းတင်းမူတ်း ဢၼ်သပ်းလႅင်းၼႄလွင်ႈ App ၸဝ်ႈၵဝ်ႇၼိူဝ် Store ယဝ်ႉ။

* **App Name:** ၸိုဝ်ႈ App ဢၼ်တွင်းလႆႈငၢႆႈ (ယႃႇပေပူၼ်ႉ 30 တူဝ်လိၵ်ႈ)။
* **Subtitle:** ၶေႃႈၵႂၢမ်းပွတ်းဢၼ်ၼိုင်ႈ ဢၼ်သပ်းလႅင်းဝႃႈ App ၼႆႉႁဵတ်းသင်လႆႈ။
* **Description:** ၶေႃႈၵႂၢမ်းသပ်းလႅင်းယၢဝ်းယၢဝ်း တႃႇၼႄ Feature တင်းမူတ်း။
* **Keywords:** ၶေႃႈၵႂၢမ်းဢၼ် User ၵႆႉၸႂ်ႉ Search (မိူၼ်ၼင်ႇ: Task, Planner, Shan, TMK)။

### **2. Screenshots & Previews (ႁၢင်ႈ လႄႈ ဝီႇတီႇဢူဝ်ႇ)**

ၼႆႉပဵၼ်တွၼ်ႈဢၼ်လွင်ႈယႂ်ႇသုတ်း တႃႇႁႂ်ႈ User တႅပ်းတတ်းၸႂ် Download ယဝ်ႉ။

* **Sizes:** ၸဝ်ႈၵဝ်ႇလူဝ်ႇႁႅၼ်းႁၢင်ႈတႃႇ iPhone 6.7" (iPhone 15/16 Pro Max) လႄႈ 5.5" (iPhone 8 Plus) ပဵၼ်ပိူင်လူင်ယဝ်ႉ။
* **Design:** ယႃႇပေသူင်ႇႁၢင်ႈ Screenshot လၢႆလၢႆၵူၺ်း။ ထႅမ်လိၵ်ႈသပ်းလႅင်း (Captions) သႂ်ႇၼိူဝ်ႁၢင်ႈ ႁႂ်ႈတူၺ်းငၢႆႈလႄႈ။
* **App Preview:** ဝီႇတီႇဢူဝ်ႇပွတ်းပွတ်း (30 Sec) ဢၼ်ၼႄလၢႆးၸႂ်ႉ App ႁဝ်း။

### **3. ASO (App Store Optimization)**

ASO ပဵၼ်မိူၼ် SEO (Search Engine Optimization) ယဝ်ႉ။ မၼ်းပဵၼ်လၢႆးႁဵတ်းႁႂ်ႈ App ႁဝ်းယူႇၼိူဝ်သုတ်း မိူဝ်ႈ User Search ႁႃၶေႃႈၵႂၢမ်း ဢၼ်တီႈဢႅပ်းႁဝ်း မီးဝႆႉၼၼ်ႉယဝ်ႉ။

* **App Title:** သႂ်ႇ Keyword ဢၼ်လွင်ႈယႂ်ႇဝႆႉၼႂ်းၸိုဝ်ႈ (မိူၼ်ၼင်ႇ: *TMK Task Manager - Shan Planner*)။
* **Ratings & Reviews:** ၶတ်းၸႂ်ႁႂ်ႈလႆႈ Review 5 လၢဝ်ၼမ်ၼမ်။ မၼ်းတေၸွႆႈႁႂ်ႈ App ၸဝ်ႈၵဝ်ႇၶိုၼ်ႈယူႇၼိူဝ်သုတ်း။
* **Localization:** သင်ၸဝ်ႈၵဝ်ႇသႂ်ႇ Metadata ပဵၼ် "လိၵ်ႈတႆး" ၼႆ မိူဝ်ႈၵူၼ်းတႆး Search ႁႃလိၵ်ႈတႆးၼၼ်ႉ App ၸဝ်ႈၵဝ်ႇတေဢွၵ်ႇမႃးၵမ်းလဵဝ်ယဝ်ႉ။

### **4. Checklist: မၢႆတွင်းဢၼ်လူဝ်ႇႁႅၼ်း**

| Item | Requirement (Shan)                                                    |
| --- |-----------------------------------------------------------------------|
| **App Icon** | 1024x1024 px (PNG)                                                    |
| **Screenshots** | 6.7" Display (1290 x 2796 px)                                         |
| **Support URL** | Website ဢၼ် User ၸၢင်ႈတိတ်းတေႃႇၸဝ်ႈၵဝ်ႇလႆႈ (မိူၼ်ၼင်ႇ Facebook Page)။ |

### **Tip: Localization Metadata**

ၼႂ်း App Store Connect ၼၼ်ႉ ၸဝ်ႈၵဝ်ႇၸၢင်ႈထႅမ် "Language" လၢႆလၢႆဢၼ်ယဝ်ႉ။ 
သႂ်ႇထႅမ်ပၼ် **Shan** လႄႈ **Burmese** သေ သႂ်ႇၸိုဝ်ႈ/ၶေႃႈသပ်းလႅင်း ပဵၼ်လိၵ်ႈတႆး/လိၵ်ႈမၢၼ်ႈလႄႈ။
မၼ်းတေၸွႆႈႁႂ်ႈ App ၸဝ်ႈၵဝ်ႇ "Stand out" လိူဝ် App တၢင်ႇဢၼ်ၶႃႈ။
