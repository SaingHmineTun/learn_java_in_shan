## 10: The Event Delegation Model

ၼႂ်းလွင်ႈႁဵၼ်းပူၼ်ႉမႃးၸိူဝ်းၼၼ်ႉ ႁဝ်းလႆႈႁူႉလၢႆးၶူင်သၢင်ႈ UI ဢၼ်ႁၢင်ႈလီမႃးယဝ်ႉၶႃႈ။ 
ၵူၺ်းၵႃႈ ပေႃးႁဝ်းၼဵၵ်း `JButton` ဢမ်ႇၼၼ် ပေႃႉလိၵ်ႈသႂ်ႇၼႂ်း `JTextField` ၼၼ်ႉ၊ 
Program ႁဝ်းဢမ်ႇႁဵတ်းသင်သေဢိတ်း။ ပိူဝ်ႈသင်လႄႈပဵၼ်ၼၼ်?

ယွၼ်ႉဝႃႈ ၵၢၼ်ၶူင်သၢင်ႈ GUI ၼႆႉ မၼ်းပဵၼ်ပိူင် **Event-Driven Programming** (လွင်ႈႁဵတ်းၵၢၼ်ၸွမ်း လွင်ႈတူင်ႉၼိုင် ၶွင် User) ၶႃႈ။ 
တွၼ်ႈတႃႇႁႂ်ႈ Component ႁဝ်းႁဵတ်းၵၢၼ်လႆႈ ႁဝ်းတေလႆႈပွင်ႇၸႂ် ပိူင်ငဝ်ႈၶၢမ်ႇမၼ်း ဢၼ်ႁွင်ႉဝႃႈ **The Event Delegation Model** ၼၼ်ႉဢွၼ်တၢင်းၶႃႈ။

### The Event Delegation Model ပဵၼ်သင်?

**Event** ပွင်ႇဝႃႈ "လွင်ႈတူင်ႉၼိုင်" (ပိူင်ယၢင်ႇ: ၼဵၵ်း Mouse, ပေႃႉ Keyboard)။ **Delegation** ပွင်ႇဝႃႈ "လွင်ႈဢၢပ်ႈၵၢၼ်ပၼ်"။

လွင်ႈပွင်ႇၸႂ်ငၢႆႈငၢႆႈတႄႉ - `JButton` ၼႆႉ မၼ်းဢမ်ႇႁူႉဝႃႈ ပေႃးပိူၼ်ႈၼဵၵ်းမၼ်းယဝ်ႉ မၼ်းတေလႆႈႁဵတ်းသင်ၶႃႈ။ 
လွင်ႈႁဵတ်းၵၢၼ်မၼ်းၼၼ်ႉ မၼ်း "ဢၢပ်ႈၵၢၼ်ပၼ် (Delegate)" ၸူး Object ထႅင်ႈဢၼ်ၼိုင်ႈ ဢၼ်ႁွင်ႉဝႃႈ "Listener" ၼၼ်ႉ ႁႂ်ႈပဵၼ်ၵူၼ်းႁဵတ်းပၼ်ၵႂႃႇၶႃႈ။

### လွင်ႈလမ်ႇလွင်ႈ 3 ယိူင်ႈ ၼႂ်း Event Delegation Model

တွၼ်ႈတႃႇႁႂ်ႈ Action ၼိုင်ႈဢၼ် ပဵၼ်မႃးလႆႈ တေလႆႈမီး 3 လွင်ႈၼႆႉၶႃႈ:

1. **Event Source (ငဝ်ႈငႃႇ လွင်ႈတူင်ႉၼိုင်):**
* ပဵၼ် UI Component ဢၼ် User ၶဝ်မႃးတူင်ႉၼိုင်ၸွမ်း။ ပိူင်ယၢင်ႇ မၼ်းၸၢင်ႈပဵၼ် `JButton`, `JTextField` ဢမ်ႇၼၼ် `JCheckBox` ၶႃႈ။


2. **Event Object (ၶေႃႈမုၼ်း လွင်ႈတူင်ႉၼိုင်):**
* မိူဝ်ႈ User တူင်ႉၼိုင်ယဝ်ႉၵမ်းလဵဝ်၊ Java တေၶူင်သၢင်ႈ Object လပ်ႉလပ်ႉဢၼ်ၼိုင်ႈ ဢၼ်သိမ်းဝႆႉ ၶေႃႈမုၼ်းတင်းသဵင်ႈဝႃႈ ၽႂ်ၼဵၵ်း၊ ၼဵၵ်းမိူဝ်ႈလႂ်၊ ၼဵၵ်းတီႈလႂ်။
* ပိူင်ယၢင်ႇ: `ActionEvent` (တွၼ်ႈတႃႇလွင်ႈၼဵၵ်း), `MouseEvent` (တွၼ်ႈတႃႇလွင်ႈၶၢႆႉ Mouse)။


3. **Event Listener (ၽူႈပႂ်ႉထွမ်ႇ လႄႈ ၽူႈႁဵတ်းၵၢၼ်):**
* ၼႆႉပဵၼ် Code ဢၼ်ႁဝ်းတေလႆႈတႅမ်ႈၶႃႈ! မၼ်းပဵၼ် Object ဢၼ်ပႂ်ႉထွမ်ႇဝႃႈ "မီးလွင်ႈတူင်ႉၼိုင် (Event) ပဵၼ်မႃးယဝ်ႉႁိုဝ်?"။ ပေႃးပဵၼ်မႃးယဝ်ႉ မၼ်းတေဢဝ် Event Object ၼၼ်ႉသေ ႁဵတ်းၵၢၼ် (Run code) ဢၼ်ႁဝ်းတႅမ်ႈဝႆႉၵမ်းလဵဝ်ၶႃႈ။
* ပိူင်ယၢင်ႇ: `ActionListener`, `MouseListener`။

---

### လွင်ႈဢဝ်တင်း 3 ယိူင်ႈ မႃးၵိုၵ်းၸပ်းၵၼ် (Wiring them together)

တွၼ်ႈတႃႇႁႂ်ႈ "Event Source" လႄႈ "Event Listener" ႁူႉၸၵ်းၵၼ်လႆႈၼၼ်ႉ ႁဝ်းတေလႆႈၸႂ်ႉ Method ဢၼ်တႄႇတွၼ်ႈတင်းၶေႃႈၵႂၢမ်း `add...Listener` ၶႃႈ။

ပိူင်ယၢင်ႇမၼ်း:

```java
// 1. Event Source (Button)
JButton myButton = new JButton("Click Me");

// 2. ၶူင်သၢင်ႈ Event Listener (ႁဝ်းတေလႆႈတႅမ်ႈ Code တီႈၼႆႈ)
// 3. ဢဝ် Listener ၵႂႃႇၸပ်းသႂ်ႇတီႈ Button (Wiring)
myButton.addActionListener( ... Listener Code ... );

```
