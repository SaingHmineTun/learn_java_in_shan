## **Lesson 7: The Activity Lifecycle (သၢႆၸႂ် လႄႈ ၵၢၼ်တူင်ႉၼိုင် ၶွင် App)**

### **1. The Journey of an App (ၵၢၼ်ၶၢႆႉတၢင်း ၶွင် App)**
Activity ၼိုင်ႈဢၼ် ဢမ်ႇလႆႈပိုတ်ႇဝႆႉတႃႇသေႇၶႃႈ။ မၼ်းမီးလၢႆးႁဵတ်းၵၢၼ် (States) ၼင်ႇၼႆၶႃႈ:
* **Foreground (ၼႃႈၸေႃး):** မိူဝ်ႈၵူၼ်းၸႂ်ႉ တိုၵ်ႉတူၺ်း လႄႈ တူင်ႉၼိုင်ၸွမ်း App ယူႇ။
* **Background (ၽၢႆႇလင်):** မိူဝ်ႈၵူၼ်းၸႂ်ႉ တဵၵ်းတုမ်ႇ Home ဢမ်ႇၼၼ် မီးၵူၼ်းႁွင်ႉၾူၼ်းမႃးသေ App ႁဝ်းၵႂႃႇပႅတ်ႉဝႆႉ ၽၢႆႇလင်ၶႃႈ။
* **Destroyed (တၢႆ):** မိူဝ်ႈၵူၼ်းၸႂ်ႉ ပိၵ်ႉ App ဢမ်ႇၼၼ် System သိမ်းဢဝ် RAM ၶိုၼ်းၶႃႈ။



---

### **2. Overriding Functions (ၵၢၼ်ၵုမ်းထိင်း Lifecycle)**
Android တေႁွင်ႉၸႂ်ႉ Function ပိူင်လူင်ၸိူဝ်းၼႆႉ မိူဝ်ႈ App လႅၵ်ႈလၢႆႈ State မၼ်းၶႃႈ:

* **`onCreate()`**: မၼ်းတေႁဵတ်းၵၢၼ် ပွၵ်ႈၼိုင်ႈၵူၺ်း မိူဝ်ႈ App တႄႇပိုတ်ႇမႃး။ ႁဝ်းၸႂ်ႉတႃႇ Setup UI (View Binding) လႄႈ တင်ႈ Logic တႄႇႁူဝ်မၼ်းၶႃႈ။
* **`onStart()` / `onResume()`**: App တႄႇၼႄၼႃႈၸေႃး လႄႈ ၵူၼ်းၸႂ်ႉတႄႇတူင်ႉၼိုင်လႆႈၶႃႈ။
* **`onPause()` / `onStop()`**: မိူဝ်ႈ App ၵႂႃႇယူႇၽၢႆႇလင် (Background)။ တီႈၼႆႈ ႁဝ်းထုၵ်ႇလီ "ၵိုတ်း" ၵၢၼ်ႁဵတ်းၵၢၼ် ဢၼ်ၸႂ်ႉၾႆးၾႃႉၼမ် မိူၼ်ၼင်ႇ ၵၢၼ်လဵၼ်ႈ Video ၼႆၶႃႈ။
* **`onDestroy()`**: မိူဝ်ႈ App ပိၵ်ႉၶႃႈ။

### **3. The "Rotation Problem" (ပၼ်ႁႃ မိူဝ်ႈထွၵ်ႇၼႃႈၸေႃး)**
လွင်ႈၼႆႉ ႁဵတ်းႁႂ်ႈလုၵ်ႈႁဵၼ်းမႂ်ႇမႂ်ႇ ယုင်ႈတႄႉတႄႉၶႃႈ:
> **မိူဝ်ႈႁဝ်းထွၵ်ႇၼႃႈၸေႃး (Rotate Phone) ၼၼ်ႉ Android တေ "သုၵ်ႉ" (Destroy) Activity ဢၼ်ၵဝ်ႇပႅတ်ႉ ယဝ်ႉ "သၢင်ႈ" (Create) Activity ဢၼ်မႂ်ႇၶိုၼ်းၵမ်းလဵဝ်ၶႃႈ!**

* **ၽူၼ်ယွၼ်ႈ:** သင်ႁဝ်းမီးတူဝ်ၼပ်ႉ `counter = 10` ဝႆႉ၊ ပေႃးႁဝ်းထွၵ်ႇၼႃႈၸေႃးၵႂႃႇ၊ `onCreate()` တေႁဵတ်းၵၢၼ်မႂ်ႇ ၶိုၼ်းထႅင်ႈပွၵ်ႈလႄႈ တေႁဵတ်းႁႂ်ႈ `counter` ႁဝ်း လႅၵ်ႈပဵၼ် `0` မိူၼ်ၵဝ်ႇၶႃႈ။
* **လၢႆးၵႄႈ:** ၼႂ်း Lesson ၼႆႉ ႁဝ်းတေၼႄပၼ် "ပၼ်ႁႃ" မၼ်းၵွၼ်ႇၶႃႈ။ ယဝ်ႉၸင်ႇတေၵႂႃႇသွၼ်လၢႆးၵႄႈလူၺ်ႈ **ViewModel** ၼႂ်း Module 5 ၶႃႈ။

### **4. Example: Monitoring the Lifecycle (ၵၢၼ်ၸၢမ်းတူၺ်း)**
ႁဝ်းၸၢမ်းတႅမ်ႈ Code ၼႆႉ သႂ်ႇၼႂ်း `MainActivity.kt` ယဝ်ႉ တူၺ်းၼႂ်း **Logcat** ၶႃႈ:

```kotlin
override fun onStart() {
    super.onStart()
    Log.d("LIFECYCLE", "onStart: App တႄႇၼႄၼႃႈၸေႃးယဝ်ႉ")
}

override fun onResume() {
    super.onResume()
    Log.d("LIFECYCLE", "onResume: ၵူၼ်းၸႂ်ႉ တႄႇတူင်ႉၼိုင်လႆႈယဝ်ႉ")
}

override fun onPause() {
    super.onPause()
    Log.d("LIFECYCLE", "onPause: App ၵႂႃႇယူႇၽၢႆႇလင်ၸူဝ်ႈၶၢဝ်း")
}
```

### **Advice for Students (ၶေႃႈသႅပ်ႇပၼ်လုၵ်ႈႁဵၼ်း):**
"ၸၢမ်းပိုတ်ႇ App တူၺ်းၶႃႈ။ သင်ယဝ်ႉ တဵၵ်း Home ၶႃႈ။ ယဝ်ႉၶိုၼ်းပိုတ်ႇ App ၶိုၼ်းၶႃႈ။
ယဝ်ႉၸၢမ်းထွၵ်ႇၼႃႈၸေႃးတူၺ်းၶႃႈ။ မိူဝ်ႈႁဵတ်း ၸိူဝ်းၼႆႉၼႆ တူၺ်းတီႈ **Logcat** တႃႇသေႇၶႃႈ။
ႁဝ်းတေႁၼ်ဝႃႈ Android ႁွင်ႉၸႂ်ႉ Function ၸိူဝ်းၼႆႉ ၸွမ်းၼင်ႇငဝ်းလၢႆးမၼ်း ၼႆၶႃႈ"။
