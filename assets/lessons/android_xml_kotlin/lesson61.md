

## **Lesson 61: Security & ProGuard**

### **1. Code Obfuscation (ၵၢၼ်ႁဵတ်းႁႂ်ႈ Code ယုင်ႈ)**

မိူဝ်ႈႁဝ်းတႅမ်ႈ Code ၼၼ်ႉ ႁဝ်းၸႂ်ႉၸိုဝ်ႈ Variable ဢၼ်ဢၢၼ်ႇငၢႆႈ မိူၼ်ၼင်ႇ `val userPassword` ဢမ်ႇၼၼ် `fun loginToServer()` ၼႆယဝ်ႉ။
သင်ပိူၼ်ႈဢဝ် App ႁဝ်းၵႂႃႇ Decompile (ပိုတ်ႇတူၺ်း Code)ၼႆၸိုင် ၶဝ်တေႁူႉၸွမ်းလႆႈမူတ်းယဝ်ႉ။

* **R8 / ProGuard:** မၼ်းတေပိၼ်ႇၸိုဝ်ႈ Class လႄႈ Function ႁဝ်းတင်းမူတ်း ႁႂ်ႈပဵၼ် `a`, `b`, `c`... ဢၼ်ၵူၼ်းဢၢၼ်ႇဢမ်ႇပွင်ႇၸႂ်ယဝ်ႉ။
* **Shrinking:** မၼ်းတေယႃႉပႅတ်ႈ Code ဢၼ်ႁဝ်းဢမ်ႇၸႂ်ႉ (Unused code) သေ ႁဵတ်းႁႂ်ႈသႅၼ်း App (APK size) လဵၵ်ႉလူင်းလိူဝ်ၵဝ်ႇယဝ်ႉ။

---

### **2. Signing the App (ၵၢၼ်လူင်းလၢႆးမိုဝ်း)**

Android တေဢမ်ႇပၼ်ႁႂ်ႈႁဝ်း တင်ႈ (Install) App ဢၼ်ဢမ်ႇမီးၵၢၼ်လူင်းလၢႆးမိုဝ်း (Signed) ယဝ်ႉ။

* **Debug APK:** ပဵၼ်ဢၼ် Android Studio သၢင်ႈပၼ်ႁင်းၵွႆး တႃႇၸၢမ်း (Test) ၼႂ်းၶိူင်ႈႁဝ်းၵူၺ်း။ မၼ်းဢမ်ႇမီးလွင်ႈလွတ်ႈၽေး လႄႈ သူင်ႇၶဝ်ႈ Google Play Store ဢမ်ႇလႆႈ။
* **Release APK:** ပဵၼ်ဢၼ်ႁဝ်းလူင်းလၢႆးမိုဝ်း တူဝ်ၵဝ်ႇႁဝ်းဝႆႉယဝ်ႉ။ မၼ်းပဵၼ်တူဝ်တႄႉ ဢၼ်တေသူင်ႇပၼ်ၵူၼ်းၸႂ်ႉယဝ်ႉ။

---

### **3. Keystores (ပပ်ႉသိမ်းသေႃးလပ်ႉ)**

တွၼ်ႈတႃႇသၢင်ႈ Release APK ၼၼ်ႉ ႁဝ်းလူဝ်ႇမီးၾၢႆႇဢၼ်ႁွင်ႉဝႃႈ **`.jks` (Java Key Store)** ၼႆယဝ်ႉ။

* **ၼႃႈတီႈ:** မၼ်းမိူၼ်ၼင်ႇ "သေႃး" ဢၼ်မၢႆတွင်းဝႃႈ App ၼႆႉပဵၼ်ၶွင်ၸဝ်ႈၵဝ်ႇတႄႉတႄႉ။
* **လွင်ႈလမ်ႇလွင်ႈ:** သင်ၸဝ်ႈၵဝ်ႇ "ႁၢႆ" ၾၢႆႇ `.jks` ၼႆႉၸိုင် ၸဝ်ႈၵဝ်ႇတေ **ဢမ်ႇၸၢင်ႈ Update** App တူဝ်ၼၼ်ႉ ၼိူဝ် Play Store လႆႈထႅင်ႈသေပွၵ်ႈယဝ်ႉ (လူဝ်ႇလႆႈသူင်ႇ App မႂ်ႇ ၸိုဝ်ႈမႂ်ႇၵူၺ်း)။ ၵွပ်ႈၼၼ် ၾၢႆႇဢၼ်ၼႆႉ ႁဝ်းလူဝ်ႇလႆႈ Backup ဝႆႉလီလီယဝ်ႉ။

---

### **Advice for Students**

'ယႃႇပေပွႆႇ Password ၶွင် Keystore ၸဝ်ႈၵဝ်ႇဝႆႉၼိူဝ် GitHub ၼေႃ!' 
ၾၢႆႇ `.jks` လႄႈ Password မၼ်းၼၼ်ႉ လမ်ႇလွင်ႈလိူဝ်ပိူၼ်ႈယဝ်ႉ။ 
သင်ပိူၼ်ႈလၵ်းလႆႈၵႂႃႇၸိုင် ၶဝ်ၸၢင်ႈသၢင်ႈ App ဢၼ်မီး Virus သေ ဢဝ်မႃးတႅၼ်းတီႈ App ၸဝ်ႈၵဝ်ႇလႆႈငၢႆႈငၢႆႈယဝ်ႉ။ 
ၵွပ်ႈၼၼ် ႁႂ်ႈသိမ်းဝႆႉၼႂ်းသုၼ်ႇတူဝ်လီလီၼႃၶႃႈ!
