## **Lesson 13: Images and ScaleTypes (ၵၢၼ်ၸတ်ႉဝၢင်းႁၢင်ႈ လႄႈ ScaleTypes)**

### **1. ImageView Deep Dive (ၵၢၼ်ပွင်ႇၸႂ် ScaleType လၢႆလၢႆပိူင်)**
မိူဝ်ႈႁဝ်းဢဝ်ႁၢင်ႈသႂ်ႇၼႂ်း `ImageView` ၼၼ်ႉ သင်ႁၢင်ႈႁဝ်း ယႂ်ႇလိူဝ် ဢမ်ႇၼၼ် လဵၵ်ႉလိူဝ် လွၵ်း (Frame) ဢၼ်ႁဝ်းတင်ႈဝႆႉၼႆ 
ႁဝ်းလူဝ်ႇၸႂ်ႉ **`android:scaleType`** တႃႇမၼ်းဝႃႈ တေႁႂ်ႈမၼ်းၼႄႁၢင်ႈၸိူင်ႉႁိုဝ်ၶႃႈ:

* **`centerCrop`**: မၼ်းတေၶႂၢၵ်ႈႁၢင်ႈ ႁႂ်ႈတဵမ်လွၵ်း ImageView ၶႃႈ။ သင်ႁၢင်ႈဢမ်ႇမိူၼ်ၵၼ်တင်းလွၵ်းၼႆ မၼ်းတေ "တတ်း" (Crop) ၽၢႆႇၶၢင်ႈပႅတ်ႈၶႃႈ။ (လၢႆးၼႆႉ ၵႆႉၸႂ်ႉတႃႇႁဵတ်း Profile Picture ဢမ်ႇၼၼ် ႁၢင်ႈ Background ၶႃႈ)။
* **`fitCenter`**: မၼ်းတေၸွႆးမႄးႁၢင်ႈ ႁႂ်ႈႁၼ်တင်းႁၢင်ႈ လႄႈ ဝၢင်းဝႆႉတီႈၵၢင်လွၵ်းၶႃႈ။ (လၢႆးၼႆႉ ႁၢင်ႈတေဢမ်ႇထုၵ်ႇတတ်းပႅတ်ႉ၊ ၵူၺ်းၵႃႈ မၼ်းၸၢင်ႈမီးတီႈပဝ်ႇ သီလမ်/သီၶၢဝ် ၽၢႆႇၶၢင်ႈၶႃႈ)။
* **`centerInside`**: မိူၼ်ၼင်ႇ `fitCenter` ၼင်ႇၵဝ်ႇ၊ ၵူၺ်းၵႃႈ သင်ႁၢင်ႈလဵၵ်ႉလိူဝ်လွၵ်း မၼ်းတေဢမ်ႇၶႂၢၵ်ႈပၼ်ၶႃႈ။

### **2. Aspect Ratio (ၵၢၼ်ၵုမ်းထိင်း သတ်ႉတႅၼ်းႁၢင်ႈ)**
ပၼ်ႁႃဢၼ်ၵႆႉထူပ်းၵေႃႈပဵၼ် "ႁၢင်ႈယိုတ်ႈ" (Stretched Images) ယဝ်ႉၶႃႈ။ 
တႃႇတေၵႄႈလွင်ႈၼႆႉ ႁဝ်းထုၵ်ႇလီၸႂ်ႉ **`ConstraintLayout`** ႁူမ်ႈတင်း **`layout_constraintDimensionRatio`** ၶႃႈ။

* ႁဝ်းၸၢင်ႈတင်ႈဝႆႉဝႃႈ ႁႂ်ႈတၢင်းၵႂၢင်ႈ လႄႈ တၢင်းသုင် မီးသတ်ႉတႅၼ်း **4:3** ဢမ်ႇၼၼ် **1:1** (လွၵ်းသီႇၸဵင်ႇပဵင်း)။
* လၢႆးၼႆႉ တေႁဵတ်းႁႂ်ႈ ဢမ်ႇဝႃႈၾူၼ်းၼႃႈၸေႃးယႂ်ႇၵေႃႈယဝ်ႉ၊ ႁၢင်ႈႁဝ်းတေမီးသတ်ႉတႅၼ်းမိူၼ်ၵဝ်ႇတႃႇသေႇၶႃႈ။

#### **တူဝ်ယၢင်ႇ Code XML (ႁၢင်ႈပဵၼ်လွၵ်းသီႇၸဵင်ႇပဵင်း 1:1):**
```xml
<ImageView
    android:layout_width="0dp"
    android:layout_height="0dp"
    android:scaleType="centerCrop"
    android:src="@drawable/my_photo"
    app:layout_constraintDimensionRatio="1:1"
    app:layout_constraintStart_toStartOf="parent"
    app:layout_constraintEnd_toEndOf="parent"
    app:layout_constraintTop_toTopOf="parent" />
```

### **3. Placeholder Graphics (ၵၢၼ်ၸႂ်ႉ tools:src)**
မိူဝ်ႈႁဝ်းတိုၵ်ႉဢွၵ်ႇပိူင် (Design) ယူႇၼၼ်ႉ ၵႆႉမီးငဝ်းလၢႆး ဢၼ်ႁဝ်းဢမ်ႇပႆႇမီးႁၢင်ႈတႄႉ ဢမ်ႇၼၼ် ႁၢင်ႈၼၼ်ႉတေမႃးတီႈ Internet ၶႃႈ။
* ႁဝ်းၸၢင်ႈၸႂ်ႉ **`tools:src="@drawable/sample_image"`** တႅၼ်းတၢင် `android:src` ၶႃႈ။
* **လွင်ႈလီ:** ႁဝ်းတေႁၼ်ႁၢင်ႈၼႂ်း **Preview (Design View)** ၵူၺ်း။ မိူဝ်ႈႁဝ်း Run App တႄႉတႄႉၼၼ်ႉ ႁၢင်ႈၼႆႉတေဢမ်ႇပႃးၵႂႃႇၸွမ်းၶႃႈ။ ၼႆႉတေ ႁဵတ်းႁႂ်ႈ App ႁဝ်းဢမ်ႇၼၵ်းၶႃႈ။

### **Advice for Students (ၶေႃႈသႅပ်ႇပၼ်လုၵ်ႈႁဵၼ်း):**
'ယႃႇလိုမ်းၸၢမ်းလႅၵ်ႈ `scaleType` တူၺ်းၵူႈဢၼ်ၶႃႈ။' `centerCrop` လမ်ႇလွင်ႈတႃႇႁဵတ်း UI ႁႂ်ႈတူၺ်းတဵမ်ထူၼ်ႈ၊ ယဝ်ႉၵေႃႈ ၸႂ်ႉတိုဝ်း `tools:` တႃႇၸွႆးၼႂ်းၵၢၼ် Design ႁႂ်ႈမၼ်းငၢႆႈလိူဝ်ၵဝ်ႇၶႃႈ။
