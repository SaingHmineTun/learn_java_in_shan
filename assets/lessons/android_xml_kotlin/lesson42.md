

## **Lesson 42: How the Internet Works for Apps**

### **1. HTTP Basics (ပိုၼ်ႉထၢၼ်ၵၢၼ်သူင်ႇၶေႃႈမုၼ်း)**

မိူဝ်ႈ App ႁဝ်း ၶႂ်ႈလႆႈၶေႃႈမုၼ်းတီႈ Internet ၼၼ်ႉ မၼ်းတေသူင်ႇ "ၶေႃႈသင်ႇ" ဢၼ်ႁွင်ႉဝႃႈ **HTTP Request** ၵႂႃႇၸူး Server ယဝ်ႉ။
ဢၼ်ႁဝ်းၵႆႉၸႂ်ႉလိူဝ်ပိူၼ်ႈ မီး 2 လွင်ႈ ၼင်ႇၼႆယဝ်ႉ:

* **GET (ဢဝ်မႃး):** ၸႂ်ႉမိူဝ်ႈႁဝ်းၶႂ်ႈ "ယွၼ်းဢဝ်" ၶေႃႈမုၼ်းမႃးၼႄ (မိူၼ်ၼင်ႇ ယွၼ်းတူၺ်းသဵၼ်ႈၸိုဝ်ႈ ဢမ်ႇၼၼ် ၵႃႊၶၼ်သတွၵ်ႉ)။
* **POST (သူင်ႇၵႂႃႇ):** ၸႂ်ႉမိူဝ်ႈႁဝ်းၶႂ်ႈ "သူင်ႇ" ၶေႃႈမုၼ်းၵႂႃႇ (မိူၼ်ၼင်ႇ သူင်ႇၸိုဝ်ႈ Password တႃႇ Login ဢမ်ႇၼၼ် တႅမ်ႈ Post မႂ်ႇၼႂ်း Facebook)။

---

### **2. JSON 101 (ၵႂၢမ်းလၢတ်ႈၶွင် App လႄႈ Server)**

Server တေဢမ်ႇသူင်ႇ Code Kotlin ဢမ်ႇၼၼ် ၼႃႈ XML မႃးပၼ်ႁဝ်းၶႃႈ။ 
မၼ်းတေသူင်ႇ ၶေႃႈမူၼ်းပဵၼ်သႅၼ်း **JSON** (JavaScript Object Notation) မႃးၵူၺ်း။
JSON ၼႆႉမၼ်းဢၢၼ်ႇငၢႆႈ လႄႈ မၼ်းလႅၼ်ႈဝႆး ၵွပ်ႈၼၼ် ၵူႈ App တင်းမူတ်းၸင်ႇၸႂ်ႉမၼ်းယဝ်ႉ။

**တူဝ်ယၢင်ႇ JSON ၶွင် Recipe (ၵပ်းၶဝ်ႈ):**

```json
{
  "recipe_name": "Shan Noodles",
  "spicy_level": 5,
  "ingredients": ["Noodles", "Pork", "Tomato", "Peanut"]
}

```

* **{ }** ပွင်ႇဝႃႈ ပဵၼ် Object
* **[ ]** ပွင်ႇဝႃႈ ပဵၼ် List

### **3. API Keys (သေႃးလပ်ႉ တႃႇပိုတ်ႇၶေႃႈမူၼ်း)**

API မၢင်ဢၼ် ႁဝ်းဢမ်ႇၸၢင်ႈၵႂႃႇႁွင်ႉၸႂ်ႉလႆႈလၢႆလၢႆၶႃႈ။ ႁဝ်းလူဝ်ႇမီး **API Key** ဢၼ်မိူၼ်ၼင်ႇ "Password" တႃႇလၢတ်ႈၼႄ Server ဝႃႈပဵၼ် App ႁဝ်းၶႃႈၼႃ ၼႆယဝ်ႉ။

* **လွင်ႈလွတ်ႈၽေး:** ႁဝ်းဢမ်ႇထုၵ်ႇလီတႅမ်ႈ API Key ဝႆႉၼႂ်း Code ႁဝ်းသိုဝ်ႈသိုဝ်ႈ (မိူၼ်ၼင်ႇ `val key = "12345"`)။
* **လၢႆးသိမ်း:** ႁဝ်းတေၸႂ်ႉၾၢႆႇ `local.properties` ဢမ်ႇၼၼ် **BuildConfig** တႃႇသိမ်းသေ "သိူင်ႇ" (Hide) မၼ်းဝႆႉ ဢမ်ႇႁႂ်ႈၵူၼ်းတၢင်ႇၵေႃႉႁၼ် မိူဝ်ႈႁဝ်းသူင်ႇ Code ၶိုၼ်ႈ GitHub ၶႃႈ။

---

### **Advice for Students**

'ဝူၼ်ႉတူၺ်းဝႃႈ API မိူၼ်ၼင်ႇ ၵူၼ်းႁပ်ႉၶႅၵ်ႇ (Waiter) ၼႂ်းႁၢၼ်ႉၶဝ်ႈၶႃႈ။' 
ႁဝ်း (App) မီးၼႃႈတီႈသင်ႇၶဝ်ႈ၊ ဝေႇတႃႇ (API) သမ်ႉတေၵႂႃႇဢဝ်ၶဝ်ႈတီႈၼႂ်း သွင်ႇၾႆးသေ (Server) မႃးသူင်ႇပၼ်ႁဝ်း။ 
သင်ႁဝ်းသင်ႇၽိတ်း ပိူင် (Syntax), ဝေႇတႃႇၵေႃႈတေ ဢမ်ႇဢဝ်ၶဝ်ႈမႃးပၼ်ႁဝ်းလႆႈၶႃႈ!
