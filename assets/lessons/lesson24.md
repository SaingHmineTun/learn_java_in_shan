
## **Java Basic 24**

#### **FOR LOOP**

Link - <https://youtu.be/ZRHrp0Ui5L8>

for loop ၼႆႉ ပဵၼ် looping ဢၼ်ၸႂ်ႉတိုဝ်းၼမ်သေပိူၼ်ႈၵေႃႈလၢတ်ႈလႆႈယူႇ။ တႃႇတေတႅမ်ႈ definite loop ၼႆ တေလႆႈၸွမ်းပိူင် 3 ပိူင်ၼႆ ၶႃႈယၢမ်ႈလၢတ်ႈဝႆႉတီႈ while loop ယဝ်ႉ။ ပေႃးၸႂ်ႉ while loop ၼႆ ႁဝ်းတေလႆႈတႅမ်ႈ ၵမ်းထႅဝ်ထႅဝ် တႃႇၸွမ်းပိူင်သၢမ်ပိူင်ၼၼ်ႉသေ တီႈ for loop ၼႆႉသမ်ႉ ႁဝ်းတႅမ်ႈ ထႅဝ်လဵဝ်ၵူၺ်း။ ၼႂ်းတူင်ႇမူၼ်း for ၼႆႉ ႁဝ်းတေသႂ်ႇလႆႈ expression သၢမ်ဢၼ်လႄႈ ၶဝ်ပဵၼ် ပိူင်သၢမ်ပိူင်ၼၼ်ႉယဝ်ႉ။ ၼႂ်းမၼ်းၼၼ်ႉ ဢၼ်လႄႈဢၼ် ၶိုၼ်းၸႂ်ႉတိုဝ်း ; (semi-column) တႅမ်ႈယဝ်ႉ။

###### လၢႆးၸႂ်ႉတိုဝ်း
```
for (int i = 0; i < 10; i ++) {
    System.out.println(i);
}
```

**break keyword**

break ၼႆႉ သင်ႁဝ်းၸႂ်ႉတိုဝ်းၼႂ်း looping ၼႆၸိုင် ပွင်ႇဝႃႈ မၼ်းတေ ဢွၵ်ႇၵႂႃႇတီႈ loop ၼႆယဝ်ႉ။ code ဢၼ်တႅမ်ႈဝႆႉၼႆႉ မိူဝ်ႈတေႇမၼ်းတေႁဵတ်ႉၵၢၼ် 100 တေႃႇထိုင် 0 သေတႃႉ ၵွပ်ႈပိူဝ်ႈမႃး break ဝႆႉတီႈ 20 ၼႆလႄႈ မၼ်းတေႁဵတ်ႉၵၢၼ် 100 ထိုင် 25 ၵူၺ်းယဝ်ႉ။

###### လၢႆးၸႂ်ႉတိုဝ်း

```
for (int a = 100; a >= 0; a-=5) {
    if (a == 20) break;
    System.out.println(a);
}
```
