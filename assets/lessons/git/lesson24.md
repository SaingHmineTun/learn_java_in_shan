## **Module 3: Remote Connectivity**
### **Lesson 24: Downloading Code (ၵၢၼ်လူတ်ႇဢဝ် Code လူၺ်ႈ git pull)**

မိူဝ်ႈႁဝ်းႁဵတ်းၵၢၼ်ၸွမ်းၵူၼ်းၼမ် ဢမ်ႇၼၼ် ႁဝ်းမႄး Code ဝႆႉတီႈ Computer ထႅင်ႈၶိူင်ႈၼိုင်ႈ (မိူၼ်ၼင်ႇ တီႈႁိူၼ်း လႄႈ တီႈႁူင်းႁဵၼ်း) ၼၼ်ႉ Code ၼိူဝ် GitHub တေ "မႂ်ႇ" သေ Code ၼႂ်းၶိူင်ႈႁဝ်းယဝ်ႉ။ 
သင်ႁဝ်း ၶႂ်ႈလႆႈ Code မႂ်ႇမႂ်ႇ ဢၼ်မီးဝႆႉၼိူဝ် Github ၼၼ်ႉ ႁဝ်းတေလႆႈၸႂ်ႉ Command **`git pull`** ၶႃႈ။

#### **1. git pull ၼႆႉပဵၼ်သင်?**
* မၼ်းပဵၼ်ၵၢၼ် "ထၢမ်" GitHub ဝႃႈ **"မီးသင်မႂ်ႇႁႃႉ?"**။
* သင်ဝႃႈမီး Commit မႂ်ႇၼိူဝ် GitHub ၼၼ်ႉ Git တေလူတ်ႇဢဝ် (Download) မႃး ယဝ်ႉဢဝ်မႃး **Merge** သႂ်ႇၼႂ်း Code ၼႂ်းၶိူင်ႈႁဝ်းၵမ်းလဵဝ်ၶႃႈယဝ်ႉ။

#### **2. လၢႆးၸႂ်ႉ (Usage)**
တႅမ်ႈ Command ငၢႆႈငၢႆႈၼႂ်း Terminal:
`git pull origin main`

* **`origin`**: ၸိုဝ်ႈ Remote (GitHub)။
* **`main`**: ၸိုဝ်ႈ Branch ဢၼ်ႁဝ်းၶႂ်ႈလူတ်ႇဢဝ်မႃး။



#### **3. မိူဝ်ႈလႂ်ႁဝ်းတေၸႂ်ႉ `git pull`?**
* **ၵူႈဝၼ်း:** မိူဝ်ႈပႆႇတႅမ်ႈ Code ၼၼ်ႉ ထုၵ်ႇလီ `pull` ဢွၼ်တၢင်းၵူႈပွၵ်ႈ ၼင်ႇႁႂ်ႈ Code ႁဝ်းမႂ်ႇယူႇတႃႇသေႇ။
* **မိူဝ်ႈပႆႇ Push:** သင်ႁဝ်းၶႂ်ႈသူင်ႇ Code ၶိုၼ်ႈၵႂႃႇ၊ ၵူၺ်းၵႃႈၼိူဝ် GitHub သမ်ႉမီး Code ဢၼ်မႂ်ႇသေႁဝ်း ဝႆႉယူႇၼၼ်ႉ Git တေဢမ်ႇပၼ်ႁဝ်း Push။ ႁဝ်းတေလႆႈ `pull` မႃးမႄး (Merge) ဢွၼ်တၢင်း ၸင်ႇတေ Push လႆႈၶႃႈ။

#### **4. မၼ်းတေဢွၵ်ႇသင်မႃး?**
မိူဝ်ႈ `pull` ယဝ်ႉယဝ်ႉ မၼ်းတေတႅမ်ႈၼႄဝႃႈ:
`Updating a1b2c3d..e5f6g7h`
`Fast-forward`
`main.dart | 10 +++++++++-`
`1 file changed, 9 insertions(+), 1 deletion(-)`
*ၼႆႉပွင်ႇဝႃႈ Code ၼႂ်းၶိူင်ႈႁဝ်း မႂ်ႇမိူၼ်ၼင်ႇၼိူဝ် GitHub ယဝ်ႉၶႃႈ!*

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
Command **`git pull`** ၼႆႉ ပဵၼ်ၵၢၼ် **"Update"**။ မၼ်းၸွႆႈႁႂ်ႈ Developer ၵူႈၵူၼ်း မီး Code ဢၼ်မိူၼ်ၵၼ် လႄႈ ႁဵတ်းႁႂ်ႈႁဝ်းႁဵတ်းၵၢၼ်ၸွမ်းၵၼ်လႆႈ လူၺ်ႈဢမ်ႇမီးပၼ်ႁႃၶႃႈဢေႃႈ။
