## **Module 1: The Local Repository**
### **Lesson 2: Setup (Installing Git & Identity)**

မိူဝ်ႈႁဝ်းႁူႉလွင်ႈ Version Control ယဝ်ႉၼၼ်ႉ တႃႇတေၸႂ်ႉလႆႈ တၵ်းတေလႆႈသႂ်ႇ Software မၼ်း လႄႈ မိတ်းမၢႆၸိုဝ်ႈ (Identity) ႁဝ်းဝႆႉၼႂ်း Computer ဢွၼ်တၢင်းၶႃႈ။

#### **1. Installing Git (လၢႆးသႂ်ႇ Git ဝႆႉၼႂ်းၶိူင်ႈ)**
တႃႇတေ Download ဢဝ် Git ၼၼ်ႉ ႁဝ်းလႆႈၵႂႃႇတီႈ Website မၼ်းၶႃႈ။
* **Download:** ၵႂႃႇတီႈ [git-scm.com](https://git-scm.com/) ယဝ်ႉလိူၵ်ႈဢဝ်တႃႇ Windows, Mac ဢမ်ႇၼၼ် Linux။
* **Verify (ၵူတ်ႇထတ်း):** မိူဝ်ႈသႂ်ႇယဝ်ႉယဝ်ႉၼၼ်ႉ ပိုတ်ႇ **Terminal** (ဢမ်ႇၼၼ် Command Prompt) ယဝ်ႉတႅမ်ႈ Command ၼႆႉတူၺ်းၶႃႈ:
  `git --version`
  *(သင်မၼ်းဢွၵ်ႇမႃးဝႃႈ `git version 2.x.x` ၼႆ ၼၼ်ႉမၼ်းလႅပ်ႈၵၼ် ၸႂ်ႉလႆႈယဝ်ႉၶႃႈ)*



#### **2. Setting Up Your Identity (တႅမ်ႈမၢႆၸိုဝ်ႈတူဝ်ၵဝ်ႇ)**
ၵွပ်ႈသင်ႁဝ်းလူဝ်ႇ တႅမ်ႈမၢႆၸိုဝ်ႈႁဝ်း? 
ၵွပ်ႈဝႃႈ Git ၼႆႉ မၼ်းတေမၢႆဝႆႉပၼ် လွင်ႈလႅၵ်ႈလၢႆႈၵူႈလွင်ႈယဝ်ႉ။ 
သင်ႁဝ်းႁဵတ်းၵၢၼ်ၸွမ်း ၵူၼ်းလၢႆလၢႆၵေႃႉၼႆ တႃႇႁဝ်းတေလႆႈႁူႉဝႃႈ **"ၽႂ်ပဵၼ်ၵူၼ်းမႄး Code ၼႆႉ?"** ၼႆၶႃႈဢေႃႈ။

ၸႂ်ႉ Command ၼႂ်း Terminal ၼင်ႇၼႆၶႃႈ:
* **တႅမ်ႈၸိုဝ်ႈ:** `git config --global user.name "Sai Mao"`
* **တႅမ်ႈ Email:** `git config --global user.email "saimao@example.com"`

*(**မၢႆတွင်း:** ၸိုဝ်ႈလႄႈ Email ၼႆႉ တေၸပ်းၵႂႃႇၸွမ်းၵူႈ Commit ဢၼ်ႁဝ်းႁဵတ်းၼၼ်ႉၶႃႈဢေႃႈ)*

#### **3. ၵူတ်ႇထတ်းတူၺ်းၶိုၼ်း (Checking Config)**
တႃႇတေတူၺ်းဝႃႈ ႁဝ်းသႂ်ႇၸိုဝ်ႈလႄႈ Email ၽိတ်းႁႃႉ ထုၵ်ႇႁႃႉၼၼ်ႉ တႅမ်ႈဝႃႈ:
`git config --list`



#### **ႁူဝ်ႁုပ်ႈ (Summary)**
ၵၢၼ် Setup ၼႆႉ ပဵၼ်ၵၢၼ်ႁဵတ်း **"ဝႂ်ၶႅပ်း (ID Card)"** တႃႇတႅမ်ႈ Program ယဝ်ႉၶႃႈ။ 
မိူဝ်ႈမီး Software လႄႈ မီးၸိုဝ်ႈယဝ်ႉယဝ်ႉၼၼ်ႉ Computer ႁဝ်းၵေႃႈ တေတူဝ်တၼ်းဝႆႉတႃႇ "မၢႆတွင်း" လွင်ႈတႅမ်ႈ Code ႁဝ်းယဝ်ႉ။
