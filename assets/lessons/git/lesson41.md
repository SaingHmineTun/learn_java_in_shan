## **Module 5: Troubleshooting & Data Integrity**
### **Lesson 41: Public Safety (ၵၢၼ်ယႃႉပႅတ်ႈလွင်ႈၽိတ်းလူၺ်ႈ git revert)**

**`git revert`** ၼႆႉ မၼ်းဢမ်ႇၸႂ်ႈၵၢၼ် "ယႃႉပႅတ်ႈ (Delete)" ပိုၼ်း၊ 
ၵူၺ်းၵႃႈ မၼ်းပဵၼ်ၵၢၼ် **"သၢင်ႈ Commit မႂ်ႇ"** ဢၼ်တေမႃးႁဵတ်းၵၢၼ် "သၢၼ်ၶတ်း (Opposite)" တင်း Commit ဢၼ်ႁဝ်းဢမ်ႇၶႂ်ႈလႆႈၼၼ်ႉၶႃႈယဝ်ႉ။

#### **1. Reset vs Revert (ပႅၵ်ႇၵၼ်ႁိုဝ်?)**
* **Reset:** "ယႃႇလႅဝ် (Delete)" ပိုၼ်းပႅတ်ႈ။ (မိူၼ်ၼင်ႇႁဝ်းဢမ်ႇပႆႇ တႅမ်ႈမၢႆတွင်းၼၼ်ႉယဝ်ႉ)
* **Revert:** "တႅမ်ႈထႅမ်" ပိုၼ်းမႂ်ႇ တႃႇၵႂႃႇယႃႉ Code ဢၼ်ၵဝ်ႇ။ (မိူၼ်ၼင်ႇႁဝ်းတႅမ်ႈမၢႆတွင်းဝႃႈ "တွၼ်ႈၼိူဝ်ၼၼ်ႉ ႁဵတ်းၽိတ်းယဝ်ႉၶႃႈ၊ တွၼ်ႈၼႆႉမႄးပၼ်ယဝ်ႉ")



#### **2. မိူဝ်ႈလႂ်တေၸႂ်ႉ `revert`?**
* မိူဝ်ႈႁဝ်းသူင်ႇ Code ၶိုၼ်ႈ **GitHub** ၵႂႃႇယဝ်ႉ။
* သေၵေႃႉၵေႃႉ သမ်ႉပေႃး လူတ်ႇဢဝ် Code ႁဝ်းၵႂႃႇၸႂ်ႉယဝ်ႉလႄ ႁၼ်တီႈၽိတ်း။
* ၵွပ်ႈ `revert` တေဢမ်ႇယႃႉပႅတ်ႈပိုၼ်း (History)၊ မၼ်းတေထႅမ် Commit မႂ်ႇၵႂႃႇၽၢႆႇၼႃႈၵူၺ်း တႃႇႁႂ်ႈတၢင်ႇၵေႃႉ ဢမ်ႇထူပ်းပၼ်ႁႃ Timeline ယုင်ႈယၢင်ႈၶႃႈဢေႃႈ။

#### **3. လၢႆးၸႂ်ႉ (Usage)**
ႁဝ်းတေလႆႈႁူႉ Commit ID ဢၼ်ႁဝ်းၶႂ်ႈယႃႉပႅတ်ႈၼၼ်ႉဢွၼ်တၢင်း (ၸႂ်ႉ `git log --oneline` တူၺ်းလႆႈၶႃႈ)။
`git revert a1b2c3d`

* **မၼ်းတေႁဵတ်းသင်:** Git တေၵႂႃႇတူၺ်းဝႃႈ Commit `a1b2c3d` ၼၼ်ႉ မၼ်း "ထႅမ်" သင်? မၼ်း "ယႃႉ" သင်? ယဝ်ႉမၼ်းတေ ႁဵတ်းလွင်ႈသၼ်ၶတ်း (Doing the opposite) ပၼ်သေ သၢင်ႈပဵၼ် Commit မႂ်ႇပၼ်ၵမ်းလဵဝ်ၶႃႈ။
* လွင်ႈသၼ်ၶတ်း တီႈၼႆႈပွင်ႇဝႃႈ သင်ႁဝ်း "ထႅမ်" ၼႆ မၼ်းတေယႃႉလႅဝ်။ သင်ႁဝ်း "ယႃႉ" ၼႆ မၼ်းတေထႅမ်သႂ်ႇပၼ် ၼႆၶႃႈ။ 

#### **4. ၵၢၼ်တွပ်ႇ Message**
မိူဝ်ႈတႅမ်ႈ `revert` ယဝ်ႉ Git တေပိုတ်ႇ Editor (မိူၼ်ၼင်ႇ Vim) ႁႂ်ႈႁဝ်းတႅမ်ႈ Message။ ၵမ်ႈၼမ် မၼ်းတေတႅမ်ႈဝႆႉပၼ်ယဝ်ႉလႄႈ:
`Revert "Add buggy feature"`
ႁဝ်းလူဝ်ႇသိမ်း (Save) သေဢွၵ်ႇမႃးၵေႃႈယဝ်ႉယဝ်ႉၶႃႈ။

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
**`git revert`** မၼ်းၸွႆႈႁႂ်ႈႁဝ်း မႄးလွင်ႈၽိတ်းလႆႈ လူၺ်ႈဢမ်ႇတုမ်ႉတိူဝ်ႉပိုၼ်းၶွင်တၢင်ႇၵေႃႉ။ သင်ဝႃႈ Code ၼၼ်ႉ "တၢင်ႇ" ၶိုၼ်ႈ Cloud ၵႂႃႇယဝ်ႉ တၵ်းတေလႆႈၸႂ်ႉ `revert` ၵူၺ်း ၸင်ႇတေၵတ်းယဵၼ်ၶႃႈဢေႃႈ။
