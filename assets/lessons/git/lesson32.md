## **Module 4: Distributed Collaboration**
### **Lesson 32: Creating Your First PR (ၵၢၼ်သူင်ႇ PR ဢွၼ်တၢင်းသုတ်း)**

တွၼ်ႈတႃႇတေသူင်ႇ PR လႆႈလီငၢမ်းၼၼ်ႉ ႁဝ်းတေလႆႈႁဵတ်းၸွမ်း **Contribution Flow** ၼင်ႇၼႆၶႃႈ:

#### **1. ၶၼ်ႉတွၼ်ႈၵၢၼ်ႁဵတ်းသၢင်ႈ (The Flow)**
1.  **Fork & Clone:** (ႁဝ်းႁဵတ်းယဝ်ႉယဝ်ႉၼႂ်း Lesson 29-30)။
2.  **Create a Branch:** **"လွင်ႈၼႆႉလႅၵ်ႈလၢႆႈသုတ်း!"** ႁဝ်းတေဢမ်ႇထုၵ်ႇလီမႄး Code ၼႂ်း `main` တူဝ်ၵဝ်ႇသေသူင်ႇ PR။ ႁဝ်းတေလႆႈသၢင်ႈ Branch မႂ်ႇတႃႇ Feature ၼၼ်ႉ:
    `git checkout -b feature-shan-translation`
3.  **Make Changes:** တႅမ်ႈ Code မႄးပၼ်ႁႃ ဢမ်ႇၼၼ် ထႅမ် Feature ဢၼ်ႁဝ်းၶႂ်ႈလႆႈၼၼ်ႉ။
4.  **Commit & Push:**
    `git add .`
    `git commit -m "ထႅမ်လိၵ်ႈတႆးၼႂ်းၼႃႈ Login"`
    `git push origin feature-shan-translation`

#### **2. သၢင်ႈ PR ၼိူဝ် GitHub**
မိူဝ်ႈႁဝ်း Push Branch မႂ်ႇၼၼ်ႉယဝ်ႉယဝ်ႉ:
1.  ၵႂႃႇတီႈ GitHub Repo ၶွင် **တူဝ်ၵဝ်ႇ (Your Fork)**။
2.  ႁဝ်းတေႁၼ် ပႅၼ်ႇၼႃႇ (Banner) သီလိူင် ၼႄမႃးဝႃႈ **"Compare & pull request"**။ ၼဵၵ်းပဵၼ်မၼ်းသေၵမ်းၶႃႈ။
3.  **Review the Diff:** GitHub တေၼႄပၼ်ဝႃႈ Code ဢၼ်ႁဝ်းမႄးၼၼ်ႉ မၼ်းတေၵႂႃႇႁူမ်ႈၸွမ်း `main` ၶွင် Project လူင်ၼၼ်ႉထုၵ်ႇႁႃႉ?
4.  **Write Title & Description:**
    * **Title:** တႅမ်ႈႁူဝ်ၶေႃႈဢၼ်ပွင်ႇၸႂ်ငၢႆႈ (မိူၼ်ၼင်ႇ: Add Shan Language Support)။
    * **Description:** တႅမ်ႈၼႄဝႃႈ ႁဝ်းမႄးသင်ၵႂႃႇ? မၼ်းၸွႆႈသင်လႆႈ? (ၸၢင်ႈတႅမ်ႈ "Fixes #12" သင်ဝႃႈမၼ်းမႄး Issue မၢႆ 12)။
5.  **Create Pull Request:** ၼဵၵ်းပဵၼ်သေၵမ်း!



#### **3. မၼ်းတေ သိုပ်ႇပဵၼ်သင်?**
မိူဝ်ႈၼဵၵ်း Create ယဝ်ႉယဝ်ႉ:
* Code ႁဝ်းတေၵႂႃႇ "ၼႄ" ဝႆႉၼႂ်း Tab **Pull Requests** ၶွင် Project လူင်ၼၼ်ႉ။
* ၸဝ်ႈၶွင် Project (Maintainer) တေလႆႈလိၵ်ႈမၢႆ (Notification) ဝႃႈ မီးၵူၼ်းမႃးၸွႆႈတႅမ်ႈ Code ယဝ်ႉ။
* ၶဝ်တေမႃး "ၵူတ်ႇထတ်း (Review)" Code ႁဝ်းၶႃႈယဝ်ႉ။

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
ၵၢၼ်သၢင်ႈ **Pull Request** ၼႆႉ မိူၼ်ၼင်ႇႁဝ်း "ယိုၼ်ႈၵၢၼ်ႁိူၼ်း" ပၼ်ၶူးသွၼ်။ မၼ်းပဵၼ်လၢႆးႁဵတ်းသၢင်ႈဢၼ် "ၵတ်းယဵၼ်" လႄႈ "မီးပိူင်"။
