## **Module 1: The Local Repository**
### **Lesson 4: git init (တႄႇပိုတ်ႇတွၼ်ႈမၢႆတွင်းမႂ်ႇ)**

မိူဝ်ႈႁဝ်းမီး Folder ဢၼ်တေတႅမ်ႈ Code (Java, Python, ဢမ်ႇၼၼ် Dart) ယဝ်ႉၼၼ်ႉ တႃႇတေႁႂ်ႈ Git တႄႇႁဵတ်းၵၢၼ်လႆႈ ႁဝ်းတေလႆႈၸႂ်ႉ Command ၼႆႉၶႃႈ။

#### **1. git init ဢၼ်ဝႃႈၼၼ်ႉပဵၼ်သင်?**
* **init** ၼႆႉ လုၵ်ႉတီႈၶေႃႈၵႂၢမ်း **Initialize** ဢၼ်ပွင်ႇဝႃႈ "တႄႇတင်ႈ" ၼၼ်ႉၶႃႈ။
* မၼ်းပဵၼ်ၵၢၼ်သၢင်ႈ Folder လပ်ႉ `.git` (ဢၼ်ႁဝ်းႁဵၼ်းၵႂႃႇၼႂ်း Lesson 3 ၼၼ်ႉ) ဝႆႉၼႂ်း Project ႁဝ်း။
* သင်ႁဝ်းဢမ်ႇၸႂ်ႉ `git init` ၼႆ Git တေဢမ်ႇမၢႆတွင်းပၼ်သင်ၼႂ်း Folder ၼၼ်ႉၶႃႈ။

#### **2. လၢႆးၸႂ်ႉ (Step-by-Step)**
1. ပိုတ်ႇ **Terminal** ဢမ်ႇၼၼ် **Command Prompt**။
2. ၶဝ်ႈၵႂႃႇၼႂ်း Folder Project ႁဝ်း (ၸႂ်ႉ Command `cd` ၼင်ႇၼႆ):
   `cd path/to/your/project`
3. တႅမ်ႈ Command တႃႇတႄႇ Git:
   `git init`



#### **3. မၼ်းတေဢွၵ်ႇသင်မႃး?**
မိူဝ်ႈၼဵၵ်း Enter ယဝ်ႉ မၼ်းတေဢွၵ်ႇလိၵ်ႈမႃးဝႃႈ:
`Initialized empty Git repository in C:/Users/SaiMao/MyProject/.git/`
*ၼႆႉပွင်ႇဝႃႈ Git ႁႅၼ်းဝႆႉတႃႇတေတူၺ်း (Track) လွင်ႈလႅၵ်ႈလၢႆႈတႃႇႁဝ်းယဝ်ႉၶႃႈ။*

#### **4. မိူဝ်ႈလႂ်ႁဝ်းတေၸႂ်ႉ `git init`?**
* ႁဝ်းတေၸႂ်ႉမၼ်း **"ပွၵ်ႈလဵဝ်ၵူၺ်း"** တႃႇ Project ၼိုင်ႈဢၼ်ၶႃႈ။
* ဢမ်ႇလူဝ်ႇၸႂ်ႉမၼ်းၵူႈဝၼ်းၶႃႈ။ မိူဝ်ႈတႄႇသၢင်ႈ Project မႂ်ႇၵူၺ်း ႁဝ်းၸင်ႇတေၸႂ်ႉၶႃႈ။

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
Command **`git init`** ၼႆႉ ပဵၼ်ၵၢၼ်ပၼ် "သၢႆၸႂ်" တႃႇ Project ႁဝ်း 
ႁႂ်ႈမၼ်းမီးလွင်ႈမၢႆတွင်း (Memory)။ 
တေႃႉတႄႉၼဵၵ်း Enter ၵမ်းလဵဝ်ၵူၺ်းၵေႃႈ Git တေတႄႇတူၺ်းပၼ် Code ႁဝ်းၵူႈလိၵ်ႈ ၵူႈထႅဝ်ယဝ်ႉၶႃႈ။
