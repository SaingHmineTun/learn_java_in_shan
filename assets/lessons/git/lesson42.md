## **Module 5: Troubleshooting & Data Integrity**
### **Lesson 42: Amending the Last Commit (ၵၢၼ်မႄး Commit လူၺ်ႈ --amend)**

ပၼ်ႁႃဢၼ်ႁဝ်းၵႆႉထူပ်းပဵၼ်:
1. Commit ယဝ်ႉသေ ႁၼ်ဝႃႈ **"တႅမ်ႈလိၵ်ႈ Commit Message ၽိတ်း"**။
2. Commit ယဝ်ႉသေ ပွင်ႇၸႂ်ဝႃႈ **"လိုမ်းသႂ်ႇ File ဢၼ်ၼိုင်ႈ"** ၶဝ်ႈၵႂႃႇၸွမ်း။
3. တိုၵ်ႉ Commit ယဝ်ႉၵမ်းလဵဝ် သမ်ႉႁၼ် Bug ဢိတ်းၼိုင်ႈသေ ၶႂ်ႈမႄးပႅတ်ႈ လူၺ်ႈဢမ်ႇၶႂ်ႈသၢင်ႈ Commit မႂ်ႇထႅင်ႈ။

Command **`git commit --amend`** တေမႃးၸွႆႈ "ယႃႉဢၼ်ၵဝ်ႇ သေ သႂ်ႇတႅၼ်းတၢင်" ပၼ်ၶႃႈယဝ်ႉ။

#### **1. မႄးလိၵ်ႈ Commit Message (Fixing the Message)**
သင်ဝႃႈႁဝ်းတႅမ်ႈ Message ၽိတ်းသေ ၶႂ်ႈလႅၵ်ႈမႂ်ႇ:
* **Command:** `git commit --amend -m "လိၵ်ႈမႂ်ႇမႂ်ႇ"`
* **Result:** Git တေယႃႉ Commit ဢၼ်သုတ်းသုတ်းၼၼ်ႉပႅတ်ႈ သေ သၢင်ႈဢၼ်မႂ်ႇ ဢၼ်မီးလိၵ်ႈမႂ်ႇၼႆႉ တႅၼ်းတၢင်ပၼ်ၶႃႈယဝ်ႉ။

#### **2. ထႅမ် File ဢၼ်လိုမ်းဝႆႉ (Adding a Forgotten File)**
သင်ဝႃႈႁဝ်း Commit ၵႂႃႇယဝ်ႉ သမ်ႉပွင်ႇၸႂ်ဝႃႈ လိုမ်းသႂ်ႇ `image.png` ၶဝ်ႈၵႂႃႇၸွမ်း:
1.  Stage File ဢၼ်လိုမ်းၼၼ်ႉဢွၼ်တၢင်း: `git add image.png`
2.  တႅမ်ႈ Command: `git commit --amend --no-edit`
    * *(`--no-edit` ပွင်ႇဝႃႈ "ဢဝ် Message ဢၼ်ၵဝ်ႇၼၼ်ႉၵူၺ်း ဢမ်ႇလူဝ်ႇမႄး" ၼႆၶႃႈ)*



#### **3. လၢႆးႁဵတ်းၵၢၼ်မၼ်း (How it Works)**
ႁဝ်းတေလႆႈပွင်ႇၸႂ်ဝႃႈ Git **ဢမ်ႇလႆႈ "မႄး (Edit)"** Commit ဢၼ်ၵဝ်ႇၶႃႈ။ မၼ်းပဵၼ်ၵၢၼ် **"သၢင်ႈ Commit မႂ်ႇ (New Commit)"** သေ ဢဝ်မႃးဝႆႉတႅၼ်းတၢင် တီႈဢၼ်ၵဝ်ႇၼၼ်ႉၶႃႈဢေႃႈ။ 
ၵွပ်ႈၼၼ် Commit ID (SHA) မၼ်းတေလႅၵ်ႈလၢႆႈၵႂႃႇၶႃႈ။

#### **4. ပိူင်လွင်ႈယႂ်ႇ (The Golden Rule)**
* **"ႁၢမ်ႈၸႂ်ႉ `--amend` တႃႇ Commit ဢၼ် `push` ၶိုၼ်ႈ GitHub ယဝ်ႉ"**။
* ၵွပ်ႈသင်? ၵွပ်ႈဝႃႈမၼ်းပဵၼ်ၵၢၼ် "တႅမ်ႈပိုၼ်းမႂ်ႇ"။ သင်ႁဝ်း Push ၵႂႃႇယဝ်ႉသေ မႃးမႄး `--amend` ၼႂ်းၶိူင်ႈ၊ ပိုၼ်းၼႂ်းၶိူင်ႈ လႄႈ ၼိူဝ် GitHub တေဢမ်ႇၶဝ်ႈၵၼ်ယဝ်ႉ သေ တေႁဵတ်းႁႂ်ႈ ၵူၼ်းတၢင်ႇၵေႃႉ ဝူၼ်ႉၵႂၢင်မူတ်းၶႃႈ။
* **Rule:** ၸႂ်ႉ `--amend` တႃႇ Commit ဢၼ်တိုၵ်ႉမီးၼႂ်းၶိူင်ႈ (Local) ႁဝ်းၵူၺ်းၶႃႈၼႃ။

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
**`git commit --amend`** ပဵၼ်ၵၢၼ် "ႁဵတ်းႁႂ်ႈပိုၼ်း (History) သႅၼ်ႈသႂ်"။ မၼ်းၸွႆႈႁႂ်ႈႁဝ်း ဢမ်ႇလူဝ်ႇမီး Commit ဢၼ်ယုင်ႈယၢင်ႈၼမ် မိူၼ်ၼင်ႇ "Fix typo", "Actually fix typo" ၸိူဝ်းၼၼ်ႉၶႃႈဢေႃႈ။
