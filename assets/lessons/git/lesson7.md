## **Module 1: The Local Repository**
### **Lesson 7: git commit (မၢႆတွင်းဝႆႉပဵၼ်ပိုၼ်း)**

မိူဝ်ႈႁဝ်း `git add` သေပဵၼ်ဝႆႉ "သီၶဵဝ်" (Staged) ယဝ်ႉၼၼ်ႉ 
ၵမ်းၼႆႉပဵၼ်ၶၢဝ်းယၢမ်း တႃႇတေ **Lock** လွင်ႈလႅၵ်ႈလၢႆႈၼၼ်ႉ ႁႂ်ႈမၼ်းပဵၼ် "ပိုၼ်း" (History) ဢၼ်ဢမ်ႇႁၢႆလႆႈယဝ်ႉၶႃႈ။

#### **1. git commit ဢၼ်ဝႃႈၼၼ်ႉပဵၼ်သင်?**
* မၼ်းမိူၼ်ၼင်ႇႁဝ်း **"ထႆႇၶႅပ်းႁၢင်ႈ (Snapshot)"** ဝႆႉၼႂ်းၶၢဝ်းယၢမ်းၼၼ်ႉ။
* မၼ်းတေမၢႆတွင်းဝႆႉပၼ်ဝႃႈ: **Code ပဵၼ်သင်ယူႇ?** **ၽႂ်ပဵၼ်ၵူၼ်းမႄး?** လႄႈ **မႄးၵႂႃႇမိူဝ်ႈလႂ်?**
* Commit ၵူႈဢၼ် တေမီးမၢႆ ID (Hash) ဢၼ်ဢမ်ႇမိူၼ်ၵၼ် ၼင်ႇႁႂ်ႈႁဝ်းၶိုၼ်းၵႂႃႇႁႃလႆႈငၢႆႈငၢႆႈ။

#### **2. လၢႆးၸႂ်ႉ (Usage)**
သင်ႁဝ်း တေႁဵတ်း Commit ၼႆ ႁဝ်းတိုၼ်းတေလႆႈတႅမ်ႈ **"လိၵ်ႈတွၼ်ႈမၢႆ (Message)"** ၸွမ်းၵူႈပွၵ်ႈ ၼင်ႇႁႂ်ႈႁဝ်းႁူႉဝႃႈ ႁဝ်းမႄးသင်ၵႂႃႇ:
`git commit -m "Add my first project file"`

* `-m` ၼႆႉ လုၵ်ႉတီႈၶေႃႈၵႂၢမ်း **Message** ၶႃႈဢေႃႈ။



#### **3. လၢႆးတႅမ်ႈ Message ႁႂ်ႈပိူၼ်ႈ လူပွင်ႇငၢႆႈ**
တႃႇတႅမ်ႈ Code တႃႇ **Shan Community** ႁဝ်းၼၼ်ႉ ထုၵ်ႇလီတႅမ်ႈႁႂ်ႈပွင်ႇၸႂ်ငၢႆႈ:
* ✅ `Add login screen logic`
* ✅ `Fix: Resolve null pointer error in main`
* ❌ `Update code` (ဢမ်ႇႁူႉဝႃႈမႄးသင်)
* ❌ `Fixed everything` (ဢမ်ႇၸႅင်ႈလႅင်း)

#### **4. ငဝ်းလၢႆးဝႆး (The Shortcut)**
သင်ဝႃႈႁဝ်းမီး File ဢၼ် Git ႁူႉၸိုဝ်ႈဝႆႉယဝ်ႉ (Modified) သေ 
ဢမ်ႇၶႂ်ႈတႅမ်ႈ `add` ၵူႈပွၵ်ႈ တႃႇတေႁဵတ်း `commit` ၼၼ်ႉ ႁဝ်းၸၢင်ႈႁူမ်ႈၵၼ်လႆႈ:
`git commit -am "Fix typo in README"`
*(`-a` ၼႆႉ ပွင်ႇဝႃႈ **Add All** ၶႃႈဢေႃႈ)*

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
Command **`git commit`** ၼႆႉ ပဵၼ်ၵၢၼ် **"သိမ်းပိုၼ်း"**။ မၼ်းပဵၼ်ၵၢၼ်ႁဵတ်းႁႂ်ႈ Code ႁဝ်းမီး "တွၼ်ႈ" (Versions)။ 
သင်ဝႃႈဝၼ်းၽုၵ်ႈ ႁဝ်းတႅမ်ႈ Code ၽိတ်းၵႂႃႇ ႁဝ်းၸၢင်ႈၶိုၼ်းၵႂႃႇၸူး "Commit" ဢၼ်လီယူႇၼၼ်ႉ လႆႈၵူႈၶၢဝ်းယၢမ်းၶႃႈဢေႃႈ။
