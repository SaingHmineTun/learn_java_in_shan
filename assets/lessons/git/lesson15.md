## **Module 2: Branching & The Safety Net**
### **Lesson 15: Managing Multiple Stashes (ၵုမ်းထိင်းႁွင်ႈလပ်ႉလၢႆလၢႆဢၼ်)**

မိူဝ်ႈႁဝ်းၸႂ်ႉ `git stash` လၢႆလၢႆပွၵ်ႈၼၼ်ႉ Git တေသိမ်းဝႆႉပဵၼ် **Stack** (မိူၼ်ၼင်ႇႁဝ်းဝႆႉ ဝၢၼ်ႇၵိၼ်ၶဝ်ႈ ၶပ်ႉၵၼ်ဝႆႉၼၼ်ႉၶႃႈ)။ 
ဢၼ်ႁဝ်း `stash` မႂ်ႇသုတ်း တေယူႇဝႆႉၽၢႆႇၼႃႈသုတ်း (Top) ၵူႈၶၢဝ်းယၢမ်းၶႃႈ။

#### **1. လၢႆးတူၺ်းသဵၼ်ႈမၢႆ Stash (List)**
တႃႇတေတူၺ်းဝႃႈ ႁဝ်းမီးသင်ၵဵပ်းဝႆႉၽႅဝ်လႂ်ယဝ်ႉၼၼ်ႉ တႅမ်ႈ Command:
`git stash list`

* မၼ်းတေၼႄပၼ်ၼင်ႇၼႆ:
  `stash@{0}: WIP on main: a1b2c3d Add login UI`
  `stash@{1}: WIP on feature-fonts: e5f6g7h Fix font size`
* **stash@{0}** ၼႆႉ ပဵၼ်ဢၼ်ႁဝ်းၵဵပ်းဝႆႉ **"မႂ်ႇသုတ်း"** ၶႃႈယဝ်ႉ။



#### **2. လၢႆးၵဵပ်းသိမ်းသေ တႅမ်ႈလိၵ်ႈတွၼ်ႈမၢႆ (Save with Message)**
သင်ႁဝ်း `git stash` ၵူၺ်း Git တေတႅမ်ႈမၢႆဝႆႉပၼ်ၵူၺ်းဝႃႈ "WIP" (Work In Progress)။ တႃႇႁႂ်ႈႁဝ်းတွၼ်ႈမၢႆလႆႈငၢႆႈ ႁဝ်းထုၵ်ႇလီတႅမ်ႈ Message ၸွမ်း:
`git stash save "မႄးလွင်ႈ Font တႆး တိုၵ်ႉၵိုတ်းဝႆႉ"`
*(ၼႂ်း Git ပၢၼ်မႂ်ႇ ၸၢင်ႈၸႂ်ႉ `git stash push -m "message"` ၵေႃႈလႆႈၶႃႈ)*

#### **3. လၢႆးဢဝ်ၶိုၼ်းမႃး (Apply vs Pop)**
ႁဝ်းမီးလၢႆးဢဝ်ဢွၵ်ႇမႃးၸႂ်ႉ 2 လၢႆးၶႃႈ:
* **`git stash pop`:** ဢဝ်ဢၼ်မႂ်ႇသုတ်း (`stash@{0}`) ဢွၵ်ႇမႃးယဝ်ႉ **ယႃႉပႅတ်ႈ** ၼႂ်းၵွင်လပ်ႉၵမ်းလဵဝ်။
* **`git stash apply`:** ဢဝ် Code ၶိုၼ်းမႃးၸႂ်ႉ ၵူၺ်းၵႃႈ **တိုၵ်ႉသိမ်းဝႆႉ** ၼႂ်းၵွင်လပ်ႉမိူၼ်ၵဝ်ႇ (တႃႇၸၢင်ႈဢဝ်ၵႂႃႇၸႂ်ႉတီႈ Branch တၢင်ႇဢၼ်ထႅင်ႈၵေႃႈလႆႈ)။

#### **4. လၢႆးလိူၵ်ႈဢဝ်ဢၼ်ႁဝ်းၶႂ်ႈလႆႈ**
သင်ႁဝ်းၶႂ်ႈလႆႈဢၼ်ၵဝ်ႇ (မိူၼ်ၼင်ႇ `stash@{1}`) ႁဝ်းၸၢင်ႈတႅမ်ႈ ID မၼ်းသႂ်ႇၸွမ်း:
`git stash pop stash@{1}`

#### **5. လၢႆးယႃႉပႅတ်ႈ (Drop & Clear)**
* **ယႃႉပႅတ်ႈဢၼ်လဵဝ်:** `git stash drop stash@{0}`
* **ယႃႉပႅတ်ႈတင်ႈမူတ်း (Clean all):** `git stash clear`

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
ၵၢၼ်ၵုမ်းထိင်း **Multiple Stashes** ၼႆႉ ၸွႆႈႁႂ်ႈႁဝ်း ၸၢင်ႈ "ၵိုတ်းၶၢဝ်ႇ" ၵၢၼ်တႅမ်ႈ Code လၢႆလၢႆယၢင်ႇဝႆႉလႆႈ။ 
မၼ်းမိူၼ်ၼင်ႇႁဝ်းမီး **"ႁွင်ႈသိမ်းၶူဝ်း"** ဢၼ်မီးမၢႆၸိုဝ်ႈဝႆႉ ႁဵတ်းႁႂ်ႈၵၢၼ်လႅၵ်ႈ Branch ၵႂႃႇမႃးႁဝ်း မီးပိူင်မီးၽၢင် လႄႈ ဢမ်ႇယုင်ႈယၢင်ႈၼၼ်ႉၶႃႈဢေႃႈ။
