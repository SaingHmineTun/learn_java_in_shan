## **Module 4: Distributed Collaboration**
### **Lesson 30: Forking (သၢင်ႈ Copy ဝႆႉၼႂ်း Account တူဝ်ၵဝ်ႇ)**

**Forking** ၼႆႉ ဢမ်ႇၸႂ်ႈ Command ၼႂ်း Git ၵူၺ်းၵႃႈ မၼ်းပဵၼ် Feature ၶွင် GitHub ဢၼ်ၸွႆႈႁႂ်ႈႁဝ်း "ပွမ်" (Copy) Project တၢင်ႇၵေႃႉ မႃးဝႆႉၼႂ်း Account ႁဝ်းႁင်းၵူၺ်းၶႃႈ။

#### **1. Forking ႁဵတ်းၵၢၼ်ၸိူင်ႉႁိုဝ်?**
* မိူဝ်ႈႁဝ်းၼဵၵ်း **"Fork"** တီႈ GitHub ၼၼ်ႉ GitHub တေဢဝ် Repository တင်ႈမူတ်းၶွင်ၶဝ် မႃးသၢင်ႈပဵၼ် Repo မႂ်ႇၼႂ်း Account ႁဝ်း။
* **မိူၼ်ၼင်ႇ:** `github.com/OtherPerson/project` -> `github.com/SaiMao/project`
* ၵမ်းၼႆႉ ႁဝ်းတေပဵၼ် **"ၸဝ်ႈၶွင်"** ၶွင် Repo ဢၼ်မႂ်ႇၼၼ်ႉယဝ်ႉ။ ႁဝ်းၸၢင်ႈမႄး၊ ၸၢင်ႈ Push၊ ၸၢင်ႈယႃႉ လႆႈၸွမ်းၸႂ် ၵွပ်ႈမၼ်းယူႇၼႂ်း "ႁိူၼ်း" ႁဝ်းယဝ်ႉၼၼ်ႉၶႃႈဢေႃႈ။



#### **2. တၢင်းလီၶွင်ၵၢၼ် Fork**
1. **Safety (ၵတ်းယဵၼ်):** ႁဝ်းၸၢင်ႈမႄး Code ၶဝ်လႆႈ ၵူၺ်းၵႃႈ မၼ်းတေဢမ်ႇၵႂႃႇတုမ်ႉတိူဝ်ႉ Code တႄႉတႄႉၶွင်ၶဝ် (Original)။
2. **Contribution:** ပဵၼ်တွၼ်ႈ ဢွၼ်တၢင်းသုတ်း တႃႇတေသူင်ႇ **Pull Request** ၵႂႃႇၸွႆႈတႅမ်ႈ Code ၸွမ်းၶဝ်ၶိုၼ်းၼၼ်ႉၶႃႈ။
3. **Personalization:** ၸၢင်ႈဢဝ် App ၶဝ်မႃးမႄးပဵၼ် "လၢႆးတႆး" ႁဝ်းႁင်းယဝ်ႉၶႃႈ (မိူၼ်ၼင်ႇ ဢဝ် Typing Tutor ၶွင်ပိူၼ်ႈ မႃးမႄးပဵၼ် တႃႇလိၵ်ႈတႆး)။

#### **3. လၢႆးႁဵတ်း (Step-by-Step)**
1.  ၵႂႃႇတီႈ GitHub Repo ဢၼ်ႁဝ်းၶႂ်ႈလႆႈ။
2.  ၼဵၵ်းပဵၼ် **"Fork"** (ယူႇတီႈၸဵင်ႇၶူၼ်ႉသုတ်းၽၢႆႇၶႂႃ)။
3.  လိူၵ်ႈ "Owner" (ပဵၼ် Account ႁဝ်း) ယဝ်ႉၼဵၵ်း **"Create fork"**။
4.  မိူဝ်ႈယဝ်ႉယဝ်ႉ ႁဝ်းတေလႆႈ URL မႂ်ႇ ဢၼ်ပဵၼ် `github.com/SaiMao/...`။
5.  ၵမ်းၼႆႉ ႁဝ်းၸင်ႇၸႂ်ႉ `git clone <URL-ႁဝ်း>` မႃးဝႆႉၼႂ်းၶိူင်ႈ ၼင်ႇႁႂ်ႈႁဝ်း `push` လႆႈၼၼ်ႉၶႃႈဢေႃႈ။



#### **4. Fork vs Clone (ၶိုၼ်းၵမ်းၼိုင်ႈ)**
* **Clone:** လုၵ်ႉၼိူဝ် Cloud ဢဝ်မႃးဝႆႉၼႂ်း Computer (တႃႇႁႂ်ႈမႄး Code လႆႈ)။
* **Fork:** လုၵ်ႉၼိူဝ် Cloud တၢင်ႇဢၼ် မႃးဝႆႉၼိူဝ် Cloud ႁဝ်း (တႃႇႁႂ်ႈမီးသုၼ်ႇ ပဵၼ်ၸဝ်ႈၶွင် လႄႈ Push လႆႈ)။

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
**Forking** ၼႆႉ ပဵၼ်ၵၢၼ် "ၵေႃႇသၢင်ႈသဵၼ်ႈတၢင်း" တူဝ်ၵဝ်ႇ။ မၼ်းၸွႆႈႁႂ်ႈႁဝ်း ၶဝ်ႈၸွႆႈထႅမ်ၼႂ်းလူၵ်ႈ Open Source လႆႈ လူၺ်ႈဢမ်ႇလူဝ်ႇလႆႈၶႂၢင်ႉ (Permission) တီႈၸဝ်ႈၶွင်ၶဝ် ၵွပ်ႈႁဝ်းမႄးၼႂ်း Copy ႁဝ်းႁင်းၵူၺ်းၼၼ်ႉၶႃႈဢေႃႈ။
