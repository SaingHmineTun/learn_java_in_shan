## **Module 1: The Local Repository**
### **Lesson 9: Reviewing History (ၵူတ်ႇထတ်းပိုၼ်းလူၺ်ႈ git log)**

မိူဝ်ႈႁဝ်းတႅမ်ႈ Code ယဝ်ႉ၊ Add ယဝ်ႉ၊ Commit ယဝ်ႉလၢႆလၢႆပွၵ်ႈၼၼ်ႉ တႃႇတေၶိုၼ်းတူၺ်း 
"သဵၼ်ႈတၢင်း" ဢၼ်ႁဝ်းပႆမႃးၼၼ်ႉ ႁဝ်းတေလႆႈၸႂ်ႉ Command **`git log`** ၶႃႈယဝ်ႉ။

#### **1. git log ဢၼ်ဝႃႈၼၼ်ႉပဵၼ်သင်?**
* မၼ်းပဵၼ်ၵၢၼ်ၼႄ "ပိုၼ်းၵၢၼ် Save" (Commit History) တင်ႈမူတ်းၼႂ်း Project ႁဝ်း။
* မၼ်းတေၼႄပၼ်:
    * **Commit Hash:** မၢႆ ID ၶွင် Commit (မိူၼ်ၼင်ႇ `a1b2c3d...`)။
    * **Author:** ၽႂ်ပဵၼ်ၵူၼ်း Commit (ၸိုဝ်ႈလႄႈ Email ဢၼ်ႁဝ်း Setup ဝႆႉၼၼ်ႉ)။
    * **Date:** ဝၼ်းထိလႄႈၶၢဝ်းယၢမ်း။
    * **Message:** လိၵ်ႈတွၼ်ႈမၢႆ ဢၼ်ႁဝ်းတႅမ်ႈဝႆႉၼၼ်ႉ။

#### **2. git log --oneline (လၢႆးတူၺ်းငၢႆႈ)**
သင်ဝႃႈႁဝ်းမီး Commit ၼမ်ၼမ် (မိူၼ်ၼင်ႇ 100 ပွၵ်ႈ) `git log` ၼႆႉ တေၼႄပၼ် ၼမ်ႁႅင်းလႄႈ တူၺ်းယၢပ်ႇ။ ၵွပ်ႈၼၼ် ႁဝ်းၸင်ႇၸႂ်ႉ:
`git log --oneline`

* မၼ်းတေႁဵတ်းႁႂ်ႈ 1 Commit ပဵၼ်လိၵ်ႈ 1 ထႅဝ်ၵူၺ်း။
* မၼ်းတေၼႄ **Hash** (7) တူဝ်ၵူၺ်း လႄႈ **Commit Message** ယဝ်ႉ။
* မၼ်းႁဵတ်းႁႂ်ႈႁဝ်းႁၼ် "သဵၼ်ႈတၢင်း" Project ႁဝ်းလႆႈၸႅင်ႈလႅင်းလီၶႃႈ။



#### **3. လၢႆးဢွၵ်ႇ (How to Exit)**
မိူဝ်ႈပိုတ်ႇ `git log` ယဝ်ႉ သင်မၼ်းယၢဝ်းၼမ် မၼ်းတေၵိုတ်းဝႆႉသေ မီးမၢႆၸမ်ႈ `:` ယူႇၽၢႆႇတႂ်ႈ။
* ၼဵၵ်း **`q`** (Quit) တႃႇဢွၵ်ႇၶိုၼ်းမႃးၸူး Terminal ႁဝ်းၶႃႈ။

#### **4. Command ဢၼ်လီႁူႉဝႆႉထႅင်ႈ**
* **`git log -n 3`:** ၼႄပၼ် Commit (3) ဢၼ် ဢၼ်မႂ်ႇသုတ်ႉ။
* **`git log --author="Sai Mao"`:** ၼႄပၼ် ပိုၼ်းၵၢၼ် Commit ၶွင်ၵူၼ်းၸိုဝ်ႈဝႃႈ "Sai Mao"။

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
Command **`git log --oneline`** ၼႆႉ မိူၼ်ၼင်ႇ **"သဵၼ်ႈမၢႆၵၢၼ်တႅမ်ႈ Code"** ၶွင်ႁဝ်းၶႃႈယဝ်ႉ။ 
မၼ်းၸွႆႈႁႂ်ႈႁဝ်းႁူႉဝႃႈ ႁဝ်းမႃးၽႅဝ်လႂ်ယဝ်ႉ၊ မႄးသင်ၵႂႃႇယဝ်ႉလႄႈ မၢႆ Hash မၼ်းပဵၼ်သင် ၼင်ႇႁႂ်ႈႁဝ်းၸၢင်ႈ "ၶိုၼ်းၵႂႃႇႁႃ" လႆႈငၢႆႈငၢႆႈၼၼ်ႉၶႃႈဢေႃႈ။
