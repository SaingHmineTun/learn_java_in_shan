## **Module 1: The Local Repository**
### **Lesson 8: The .gitignore File (လိူၵ်ႈပႅတ်ႈ File ဢၼ်ဢမ်ႇလူဝ်ႇ)**

ၼႂ်း Project ႁဝ်းၼၼ်ႉ မၼ်းၵႆႉမီး File ဢၼ်ႁဝ်းဢမ်ႇၶႂ်ႈႁႂ်ႈ Git မၢႆတွင်း 
(မိူၼ်ၼင်ႇ File ဢၼ် Computer သၢင်ႈပၼ်ႁင်းၵူၺ်း ဢမ်ႇၼၼ် Password လပ်ႉၶေႃႈမုၼ်းႁဝ်း)။

#### **1. .gitignore ဢၼ်ဝႃႈၼၼ်ႉပဵၼ်သင်?**
* မၼ်းပဵၼ် File လိၵ်ႈ (Text file) ဢၼ်ႁဝ်းသၢင်ႈဝႆႉၼႂ်း Folder တီႈ Project ႁဝ်းယဝ်ႉ။
* ၸိုဝ်ႈမၼ်းတၵ်းတေလႆႈပဵၼ် **`.gitignore`** (တၵ်းတေလႆႈမီးၸမ်ႈ `.` ယူႇၽၢႆႇၼႃႈ)။
* File ဢမ်ႇၼၼ် Folder ဢၼ်ႁဝ်းတႅမ်ႈဝႆႉၼႂ်း `.gitignore` ၼၼ်ႉ Git တေ "မင်" (Ignore) ပႅတ်ႈ၊ မၼ်းတေဢမ်ႇၼႄၼႂ်း `git status` ၶႃႈယဝ်ႉ။

#### **2. တႃႇသင်ႁဝ်းသမ်ႉလူဝ်ႇ "Ignore" (လိူၵ်ႈပႅတ်ႈ)?**
1. **Security (လွင်ႈၵတ်းယဵၼ်):** ဢမ်ႇၶႂ်ႈႁႂ်ႈ API Keys ဢမ်ႇၼၼ် Password ႁဝ်း တူၵ်းၵႂႃႇၼႂ်း GitHub။
2. **Cleanliness (လွင်ႈသႅၼ်ႈသႂ်):** ဢမ်ႇလူဝ်ႇမၢႆတွင်း File ၸိူဝ်းဢၼ်ပဵၼ် `build/`, `bin/`, ဢမ်ႇၼၼ် `node_modules/` ၵွပ်ႈမၼ်းပဵၼ် File ဢၼ်မီးၼမ်သုတ်းသေ ဢမ်ႇၸႂ်ႈ Code ဢၼ်ႁဝ်းတႅမ်ႈ။
3. **Speed (လွင်ႈဝႆး):** ႁဵတ်းႁႂ်ႈ Git ႁဵတ်းၵၢၼ်ဝႆး ၵွပ်ႈမၼ်းဢမ်ႇလူဝ်ႇတူၺ်း File ဢၼ်ဢမ်ႇလူဝ်ႇၸိူဝ်းၼၼ်ႉၶႃႈ။



#### **3. လၢႆးတႅမ်ႈၼႂ်း .gitignore**
ႁဝ်းၸၢင်ႈတႅမ်ႈလၢႆးငၢႆႈငၢႆႈၼင်ႇၼႆၶႃႈ:
* `secrets.txt` (လိူၵ်ႈပႅတ်ႈ File ၼႆႉဢၼ်လဵဝ်)
* `*.log` (လိူၵ်ႈပႅတ်ႈၵူႈ File ဢၼ်မီးၸိုဝ်ႈ သုတ်ႉဝႆႉလူၺ်ႈ .log)
* `build/` (လိူၵ်ႈပႅတ်ႈ Folder `build` တင်ႈမူတ်း)
* `.env` (လိူၵ်ႈပႅတ်ႈ File သိမ်းၶေႃႈမုၼ်းလပ်ႉ)

#### **4. လၢႆးသၢင်ႈ (How to create)**
ၼႂ်း Terminal တႅမ်ႈ Command ၼႆႉၵေႃႈလႆႈၶႃႈ:
`touch .gitignore`
*(ယဝ်ႉယဝ်ႉၵေႃႈ ပိုတ်ႇမၼ်းလူၺ်ႈ Notepad ဢမ်ႇၼၼ် IDE ႁဝ်းသေ တႅမ်ႈသဵၼ်ႈမၢႆ File ဢၼ်ႁဝ်းၶႂ်ႈလိူၵ်ႈပႅတ်ႈၼၼ်ႉသႂ်ႇၶႃႈ)*

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
File **`.gitignore`** ၼႆႉ မိူၼ်ၼင်ႇ **"တူဝ်ထတ်း (Filter)"** ၶွင် Project ႁဝ်းၶႃႈယဝ်ႉ။ 
မၼ်းၸွႆႈႁႂ်ႈ Git ႁဝ်း "သႅၼ်ႈသႂ်" သေ မၢႆတွင်းပၼ် Code ဢၼ်မီးၵႃႈၶၼ်တႃႇ Project ႁဝ်းၼၼ်ႉၶႃႈဢေႃႈ။
