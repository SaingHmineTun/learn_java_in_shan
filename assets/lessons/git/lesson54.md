## **Module 6: Android Studio Integration**
### **Lesson 54: Managing Pull Requests (ၵၢၼ်ၵုမ်းထိင်း Pull Request ၼႂ်း IDE)**

**Pull Request (PR)** ၼႆႉ ပဵၼ်ၵၢၼ်ဢၼ်ႁဝ်း တႅမ်ႈ Code ယဝ်ႉသေ ယွၼ်းႁႂ်ႈ ၵူတ်ႇထတ်း (Review) သေ ဢဝ်ၶဝ်ႈၵႂႃႇၼႂ်း Branch `main` ၼၼ်ႉၶႃႈယဝ်ႉ။

#### **1. လၢႆးပိုတ်ႇ Tab Pull Requests**
1.  တူၺ်းတီႈ Tab **"Pull Requests"** ဢၼ်ယူႇၽၢႆႇသၢႆႉ (Vertical tab) ၸွမ်းၵၼ်တင်း Commit Window (Lesson 47)။
2.  မိူဝ်ႈၼဵၵ်းပိုတ်ႇမႃးယဝ်ႉ ႁဝ်းတေႁၼ်သဵၼ်ႈမၢႆ PR တင်းမူတ်း ဢၼ်လုၵ်ႈႁဵၼ်းၶဝ်သူင်ႇဝႆႉၼိူဝ် GitHub ၶႃႈ။



#### **2. ၵၢၼ်ၵူတ်ႇထတ်း Code (Reviewing)**
မိူဝ်ႈႁဝ်း ၼဵၵ်းလိူၵ်ႈ PR ဢၼ်ၼိုင်ႈ:
* **Timeline:** တေႁၼ်ဝႃႈ "ၽႂ်တႅမ်ႈသင်ဝႆႉ?" လႄႈ "မီးၵူၼ်း Comment ဝႆႉႁႃႉ?"။
* **Files:** တေႁၼ် File ၵူႈဢၼ် ဢၼ်ၶဝ်မႄးဝႆႉ။
* **Diff:** ႁဝ်းၸၢင်ႈၼဵၵ်းတူၺ်း Code ပႅၵ်ႇၵၼ် (Side-by-side) သေ တႅမ်ႈ **Comment** သင်ႇၵႂႃႇၵမ်းလဵဝ် တီႈထႅဝ်လိၵ်ႈၼၼ်ႉလႆႈၶႃႈ။ (မိူၼ်ၼင်ႇ: "တွၼ်ႈၼႆႉ တႅမ်ႈၽိတ်းဝႆႉဢိတ်းၼိုင်ႈၼႃ")။

#### **3. Merge ဢမ်ႇၼၼ် Close PR**
မိူဝ်ႈႁဝ်း ၵူတ်ႇထတ်း Code ယဝ်ႉယဝ်ႉသေ ႁၼ်ဝႃႈမၼ်းလီယဝ်ႉ:
* **Merge:** ၼဵၵ်း "Merge" ၼႂ်း Android Studio ၵမ်းလဵဝ် တႃႇဢဝ် Code ၼၼ်ႉၶဝ်ႈၵႂႃႇၼႂ်း `main`။
* **Decline:** သင် Code ၼၼ်ႉၸႂ်ႉဢမ်ႇလႆႈ၊ ၸၢင်ႈ "Close PR" ပႅတ်ႈလႆႈၶႃႈ။



#### **4. Checkout PR Branch**
ဢၼ်ၶိုၵ်ႉသုတ်းပဵၼ်: ႁဝ်းၸၢင်ႈၼဵၵ်း **"Checkout"** တီႈ PR ၼၼ်ႉ။ Android Studio တေလူတ်ႇဢဝ် Code ၶွင်ၵေႃႉၼၼ်ႉ မႃးဝႆႉၼႂ်းၶိူင်ႈႁဝ်း တႃႇႁႂ်ႈႁဝ်း **Run App** တူၺ်းလႆႈတႄႉတႄႉ ဝႃႈမၼ်းမီး Bug ႁႃႉ? ဢွၼ်တၢင်း ႁဝ်းပႆႇ Merge ၶႃႈဢေႃႈ။

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
**Pull Request Management** ၼႂ်း IDE ႁဵတ်းႁႂ်ႈႁဝ်း ၵုမ်းထိင်း Project လႆႈဝႆးသုတ်း။ မၼ်းၸွႆႈႁႂ်ႈ ၵၢၼ်မႄး Code ၸွမ်းၵၼ် (Collaboration) ပဵၼ်လွင်ႈဢၼ်ငၢႆႈ ၶႃႈယဝ်ႉ။
