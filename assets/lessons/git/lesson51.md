## **Module 6: Android Studio Integration**
### **Lesson 51: Stashing vs Shelving (ၵၢၼ်သိမ်း Code ဝႆႉၼိုင်ႈၵႅပ်း)**

တင်းသွင်ယၢင်ႇၼႆႉ ႁဵတ်းၵၢၼ်မိူၼ်ၵၼ်: **"ဢဝ် Code ဢၼ်တိုၵ်ႉမႄးဝႆႉၼၼ်ႉ သိမ်းဝႆႉတီႈ ႁွင်ႈသိူင်ႇသိမ်း သေ ႁဵတ်းႁႂ်ႈ Project ႁဝ်းသႅၼ်ႈသႂ် (Clean) မိူၼ်ၵဝ်ႇ"** တႃႇႁႂ်ႈႁဝ်း လႅၵ်ႈ Branch လႆႈငၢႆႈငၢႆႈၶႃႈ။

#### **1. Git Stash (ပိူင်ၵၢၼ် Git တႄႉတႄႉ)**
* **မၼ်းပဵၼ်သင်:** မၼ်းပဵၼ် Feature ၶွင် Git။ သင်ႁဝ်း Stash ဝႆႉၼႂ်းၶိူင်ႈၼႆႉသေ သမ်ႉၵႂႃႇပိုတ်ႇ Project ၼႆႉၼႂ်း Terminal ဢမ်ႇၼၼ် VS Code၊ ႁဝ်းတိုၵ်ႉတေႁၼ် Code ၼၼ်ႉယူႇၶႃႈ။
* **လၢႆးၸႂ်ႉ:** ၼဵၵ်းမိုဝ်းၶႂႃတီႈ Project > **Git > Uncommitted Changes > Stash Changes...**
* **လၢႆးဢဝ်ၶိုၼ်း:** လိူၵ်ႈ **Unstash Changes...** သေလိူၵ်ႈဢဝ်ဢၼ်ႁဝ်းသိမ်းဝႆႉၼၼ်ႉၶႃႈ။

#### **2. Shelf (မေႃၸွႆႈၶွင် Android Studio)**
* **မၼ်းပဵၼ်သင်:** မၼ်းပဵၼ် Feature ၶွင် Android Studio ၵူၺ်း။ မၼ်းတေဢမ်ႇၵႂႃႇယုင်ႈယၢင်ႈတင်း Git Database။
* **တၢင်းလီ:**
    * မၼ်းတူၺ်းငၢႆႈ (Visual)။
    * ႁဝ်းၸၢင်ႈလိူၵ်ႈသိမ်း "သေထႅဝ်ထႅဝ် (Specific lines)" လႆႈငၢႆႈငၢႆႈ။
    * မၼ်းတေယူႇၼႂ်း Tab **"Commit" > "Shelf"** ၶႃႈ။
* **လၢႆးၸႂ်ႉ:** ၼႂ်း Commit Window (Alt + 0)၊ ၼဵၵ်းမိုဝ်းၶႂႃတီႈ File သေလိူၵ်ႈ **"Shelf Changes"**။



#### **3. တေလိူၵ်ႈၸႂ်ႉဢၼ်လႂ်? (Which one to use?)**

| Feature | Git Stash | IDE Shelf                                            |
| :--- | :--- |:-----------------------------------------------------|
| **Compatibility** | ၸႂ်ႉလႆႈၵူႈတီႈ (Terminal, IDE) | ၸႂ်ႉလႆႈၼႂ်း Android Studio ၵူၺ်း                     |
| **Safety** | သိမ်းဝႆႉၼႂ်း Git | သိမ်းဝႆႉၼႂ်း Config ၶွင် IDE                         |
| **Visibility** | တူၺ်းယၢပ်ႇဢိတ်းၼိုင်ႈ | တူၺ်းငၢႆႈ (မီး Tab ၼႄပၼ်)                            |
| **Recommendation** | **ၸႂ်ႉတႃႇၵၢၼ်လႅၵ်ႈ Branch ဝႆးဝႆး** | **ၸႂ်ႉတႃႇသိမ်း Code ဢၼ် "ၸၢမ်းတႅမ်ႈ" ဝႆႉႁိုင်ႁိုင်** |

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
**Stash** လႄႈ **Shelf** မိူၼ်ၼင်ႇႁဝ်းမီး "ၶွင်ႈထွတ်ႇ (Drawer)" တႃႇဢဝ် Code ဢၼ်တိုၵ်ႉတႅမ်ႈယူႇၼၼ်ႉ သႂ်ႇဝႆႉၵမ်းၼိုင်ႈ။ 
မၼ်းၸွႆႈႁႂ်ႈႁဝ်း ဢမ်ႇလူဝ်ႇ "Commit ဢၼ်လူႉလႅဝ်" တႃႇတေလႅၵ်ႈ Branch ၵႂႃႇမႄး Bug တၢင်ႇတီႈၼၼ်ႉၶႃႈဢေႃႈ။
