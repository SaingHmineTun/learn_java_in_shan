## **Module 2: Branching & The Safety Net**
### **Lesson 13: The Basic Merge (ၵၢၼ်ႁူမ်ႈ Code လၢႆး Fast-Forward)**

မိူဝ်ႈႁဝ်းတႅမ်ႈ Code ၼႂ်း Branch (မိူၼ်ၼင်ႇ `feature-fonts`) ယဝ်ႉယဝ်ႉသေ ၶႂ်ႈဢဝ်မႃးႁူမ်ႈၼႂ်း `main` ၼၼ်ႉ ႁဝ်းတေလႆႈၸႂ်ႉ Command **`merge`** ၶႃႈ။

#### **1. Fast-Forward Merge ဢၼ်ဝႃႈၼၼ်ႉပဵၼ်သင်?**
* မၼ်းပဵၼ်လၢႆး Merge ဢၼ် **"ငၢႆႈသုတ်း"**။
* မၼ်းတေၸႂ်ႉလႆႈ မိူဝ်ႈ Branch `main` ႁဝ်း **ဢမ်ႇပႆႇမီးလွင်ႈလႅၵ်ႈလၢႆႈသင်** ဝၢႆးသေ Branch မႂ်ႇမႂ်ႇ ၽႄဢွၵ်ႇၵႂႃႇၼၼ်ႉၶႃႈ။
* Git တေတူၺ်းဝႃႈ "ဢႃႈ... တီႈ main ၵေႃႈဢမ်ႇမီးသင်မႂ်ႇ၊ တီႈ feature သမ်ႉမီးတွၼ်ႈမႂ်ႇ၊ ၵမ်းၼႆႉ ၵူၺ်းၵႃႈ ႁဝ်းတေဢဝ် `main` ၵႂႃႇၸူးတီႈလဵဝ်ၵၼ်တင်း `feature` ၵူၺ်းၵေႃႈယဝ်ႉယဝ်ႉ" ၼႆၶႃႈဢေႃႈ။



#### **2. လၢႆးၸႂ်ႉ (Step-by-Step)**
တႃႇတေ Merge Code ၶဝ်ႈၸူး `main` ၼၼ်ႉ ႁဝ်းတေလႆႈႁဵတ်းၼင်ႇၼႆၶႃႈ:
1. လႅၵ်ႈၶိုၼ်းၵႂႃႇယူႇတီႈ Branch **"တူဝ်ႁပ်ႉ"** (မိူၼ်ၼင်ႇ `main`):
   `git checkout main`
2. ၸႂ်ႉ Command Merge ဢဝ် Branch **"တူဝ်သူင်ႇ"** ၶဝ်ႈမႃး:
   `git merge feature-fonts`

#### **3. မၼ်းတေဢွၵ်ႇသင်မႃး?**
မိူဝ်ႈၼဵၵ်း Enter ယဝ်ႉ မၼ်းတေတႅမ်ႈၼႄဝႃႈ:
`Updating a1b2c3d..e5f6g7h`
`Fast-forward`
`1 file changed, 5 insertions(+)`
*ၼႆႉပွင်ႇဝႃႈ Code မႃးႁူမ်ႈၵၼ်ယဝ်ႉယဝ်ႉၶႃႈ!*

#### **4. ၵၢၼ်သႅၼ်ႈသႂ် (Cleanup)**
မိူဝ်ႈ Merge ယဝ်ႉယဝ်ႉၼၼ်ႉ Branch ဢၼ်ႁဝ်းၸႂ်ႉတႅမ်ႈ Feature ၼၼ်ႉ ဢမ်ႇလူဝ်ႇၸႂ်ႉမၼ်းထႅင်ႈယဝ်ႉ။ ႁဝ်းၸၢင်ႈ Delete မၼ်းပႅတ်ႈလႆႈ ၼင်ႇႁႂ်ႈ Repo ႁဝ်းသႅၼ်ႈသႂ်:
`git branch -d feature-fonts`

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
**Fast-Forward Merge** ၼႆႉ မိူၼ်ၼင်ႇႁဝ်း **"လႅၼ်ႈဝႆး"** ၵႂႃႇၸူးတွၼ်ႈမႂ်ႇ။ မၼ်းပဵၼ်လၢႆးႁူမ်ႈ Code ဢၼ်ငၢႆႈ လႄႈ ဢမ်ႇမီးပၼ်ႁႃ (Conflict) သင် ၵွပ်ႈဝႃႈသဵၼ်ႈတၢင်းသွင်သဵၼ်ႈၼၼ်ႉ မၼ်းတိုၵ်ႉပဵၼ်သဵၼ်ႈတၢင်းလဵဝ်ၵၼ်ယူႇၼၼ်ႉၶႃႈဢေႃႈ။
