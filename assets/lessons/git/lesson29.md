## **Module 4: Distributed Collaboration**
### **Lesson 29: Cloning (ၵၢၼ်လူတ်ႇဢဝ်ပိုၼ်း Project တင်ႈမူတ်း)**

မိူဝ်ႈႁဝ်းႁၼ် Project တီႈ GitHub ဢၼ်ႁဝ်းၶႂ်ႈလႆႈ (မိူၼ်ၼင်ႇ Project ပိူၼ်ႈ ဢမ်ႇၼၼ် Project တူဝ်ႇၵဝ်ႇ ဢၼ်ႁဝ်းၶႂ်ႈလူတ်ႇမႃးသႂ်ႇၼႂ်း Computer မႂ်ႇ) ႁဝ်းတေလႆႈၸႂ်ႉ Command **`git clone`** ၶႃႈ။

#### **1. git clone ၼႆႉႁဵတ်းသင်ပၼ်လႆႈ?**
* မၼ်းတေ **သၢင်ႈ Folder မႂ်ႇ** ပၼ်ၼႂ်း Computer ႁဝ်း။
* မၼ်းတေ **လူတ်ႇဢဝ် File** တင်ႈမူတ်းမႃးပၼ်။
* ဢၼ်ၶိုၵ်ႉသုတ်းပဵၼ် မၼ်းတေ **လူတ်ႇဢဝ် "ပိုၼ်း (Full History)"** ဢၼ်ပဵၼ် `.git` ၼၼ်ႉမႃးပၼ်ၸွမ်းမူတ်းမူတ်း။ ႁဝ်းတေၸၢင်ႈတူၺ်း `git log` ၶွင်ၶဝ်လႆႈၵမ်းလဵဝ်ၶႃႈယဝ်ႉ။
* မၼ်းတေ **Setup Remote (origin)** ပၼ်ႁင်းၵူၺ်း၊ ႁဝ်းဢမ်ႇလူဝ်ႇတႅမ်ႈ `remote add` ထႅင်ႈယဝ်ႉ။



#### **2. လၢႆးၸႂ်ႉ (Usage)**
ၵႂႃႇတီႈ GitHub Repo ဢၼ်ႁဝ်းၶႂ်ႈလႆႈၼၼ်ႉ၊ ၼဵၵ်းတီႈ **"Code"** (သီၶဵဝ်) သေ Copy ဢဝ် URL မၼ်းမႃး။ ယဝ်ႉတႅမ်ႈၼႂ်း Terminal:

`git clone https://github.com/SaiMao/tmk-typing-tutor.git`

* သင်ႁဝ်းၶႂ်ႈလႅၵ်ႈၸိုဝ်ႈ Folder မၼ်းၵမ်းလဵဝ်ၵေႃႈ တႅမ်ႈၸိုဝ်ႈမႂ်ႇသႂ်ႇၽၢႆႇလင်လႆႈၶႃႈ:
  `git clone <url> my-new-folder-name`

#### **3. Clone vs Download ZIP**
လုၵ်ႈႁဵၼ်းမၢင်ၵေႃႉ ၵႆႉၼဵၵ်း "Download ZIP" တီႈ GitHub။ မၼ်းပႅၵ်ႇၵၼ်ၼင်ႇၼႆၶႃႈ:
* **Download ZIP:** လႆႈမႃး File Code မၼ်းလၢႆလၢႆၵူၺ်း၊ ဢမ်ႇမီးပိုၼ်း (No `.git`)၊ ဢမ်ႇၸၢင်ႈ `push` ဢမ်ႇၼၼ် `pull` လႆႈ။
* **Git Clone:** လႆႈမႃးတင်း Code လႄႈ တင်းပိုၼ်း (Full Git Repository)၊ ႁႅၼ်းဝႆႉတႃႇတေတႅမ်ႈ Code ၸွမ်းၵၼ် (Collaborate) လႆႈၵမ်းလဵဝ်။

#### **4. Shallow Clone (တႃႇ Project လူင်လူင်)**
သင် Project ၼၼ်ႉ မၼ်းလူင်ႁႅင်းၼႃႇ (မိူၼ်ၼင်ႇ Flutter SDK) သေ ႁဝ်းဢမ်ႇၶႂ်ႈလႆႈပိုၼ်းမၼ်းၵူႈ ၵွပ်ႈၵူဝ်မၼ်းၼၵ်းၶိူင်ႈၼႆ ႁဝ်းၸၢင်ႈဢဝ်တွၼ်ႈ "မႂ်ႇသုတ်း" ၼၼ်ႉလႆႈလူၺ်ႈ:
`git clone --depth 1 <url>`

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
Command **`git clone`** ၼႆႉ ပဵၼ်ၵၢၼ် **"တႄႇယၢင်ႈတိၼ်ဢွၼ်တၢင်း"** တႃႇတေၶဝ်ႈၸွႆႈတႅမ်ႈ Code ၸွမ်းၵူၼ်းၼမ်ယဝ်ႉၶႃႈ။ မိူဝ်ႈႁဝ်း Clone ယဝ်ႉယဝ်ႉ ႁဝ်းတေမီး Project ၼၼ်ႉဝႆႉၼႂ်းမိုဝ်းႁဝ်း တႃႇတေၸၢမ်း တႃႇတေႁဵၼ်း လႄႈ တႃႇတေမႄးပၼ်လႆႈယဝ်ႉၶႃႈ။
