## **Module 4: Distributed Collaboration**
### **Lesson 36: Contributing Etiquette and Issue Tracking (ၾိင်ႈထုင်ႉလႄႈၵၢၼ်တူၺ်း Issue)**

မိူဝ်ႈႁဝ်းၶဝ်ႈၵႂႃႇၼႂ်း Project ၶွင်ပိူၼ်ႈၼၼ်ႉ ႁဝ်းပဵၼ်မိူၼ်ၼင်ႇ "ၶႅၵ်ႇ" ဢၼ်ၵႂႃႇၸွႆႈႁဵၼ်း ၵၢၼ်ၼႂ်းႁိူၼ်းပိူၼ်ႈယဝ်ႉ။ 
ႁဝ်းတေလႆႈႁူႉၸွမ်းလၢႆးၵၢၼ်ၼင်ႇၼႆၶႃႈ:

#### **1. Issue Tracking (လွင်ႈတူၺ်းပၼ်ႁႃ)**
ဢွၼ်တၢင်း ႁဝ်းပႆႇတႅမ်ႈ Code သေသူင်ႇ PR ၼၼ်ႉ ႁဝ်းတေလႆႈၵႂႃႇတူၺ်းတီႈ Tab **"Issues"** ၼႂ်း GitHub ဢွၼ်တၢင်းၶႃႈ။
* **Issues** ပဵၼ်တီႈဢၼ်ၵူၼ်းၸႂ်ႉ App ၶဝ်မႃးတႅမ်ႈၼႄဝႃႈ "မီး Bug (Error)" ဢမ်ႇၼၼ် "ၶႂ်ႈလႆႈ Feature မႂ်ႇ"။
* ႁဝ်းထုၵ်ႇလီလိူၵ်ႈဢဝ် Issue ဢၼ်မီးဝႆႉယူႇယဝ်ႉသေ တႅမ်ႈ Comment ဝႃႈ "ၶႃႈၶႂ်ႈၸွႆႈမႄးတွၼ်ႈၼႆႉၶႃႈ (I'd like to work on this)"။ တႃႇႁႂ်ႈႁဝ်း ဢမ်ႇတႅမ်ႈ Code သုမ်းလၢႆ လႄႈ ဢမ်ႇႁဵတ်းၵၢၼ်သွၼ်ႉ ၵၼ်တင်းပိူၼ်ႈၶႃႈ။



#### **2. Read the CONTRIBUTING.md**
Project လူင်လူင်ၵႆႉမီး File ၸိုဝ်ႈဝႃႈ `CONTRIBUTING.md`။
* မၼ်းတေၸိၼႄဝႃႈ: ၶဝ်ၶႂ်ႈႁႂ်ႈႁဝ်းတႅမ်ႈ Code လၢႆးလႂ်? တင်ႈၸိုဝ်ႈ Branch ႁိုဝ်? လႄႈ တႅမ်ႈ Message Commit ၼင်ႇႁိုဝ်?
* **Tip:** ၵၢၼ်ႁဵတ်းၸွမ်းၾိင်ႈထုင်းၶဝ် ႁဵတ်းႁႂ်ႈ PR ႁဝ်းလႆႈႁပ်ႉ (Merge) ဝႆးဝႆးၶႃႈဢေႃႈ။

#### **3. Contributing Etiquette (ၾိင်ႈထုင်းဢၼ်လီ)**
1. **Be Polite (ယဵၼ်ႇငႄႈ):** တႅမ်ႈလိၵ်ႈတွပ်ႇၵၼ်ႁႂ်ႈမီးလွင်ႈၼပ်ႉယမ်။
2. **Small PRs:** ၵၢၼ်သူင်ႇ PR ဢၼ်မႄးတွၼ်ႈလဵဝ် ၵႆႉလႆႈႁပ်ႉဝႆးသေ PR ဢၼ်မႄး Code တင်းၼမ် ၶႃႈ။
3. **Draft PR:** သင်ႁဝ်းတိုၵ်ႉတႅမ်ႈ Code ယူႇသေ ၶႂ်ႈၼႄပၼ်ပိူၼ်ႈတူၺ်းဢွၼ်တၢင်း ႁဝ်းၸၢင်ႈပိုတ်ႇပဵၼ် **"Draft Pull Request"** လႆႈၶႃႈ။

#### **4. Good First Issues (တႃႇလုၵ်ႈႁဵၼ်းမႂ်ႇ)**
သင်ႁဝ်း ၶႂ်ႈၸၢမ်းတႅမ်ႈ Code ၸွႆႈပိူၼ်ႈပွၵ်ႈဢွၼ်တၢင်းသုတ်းၼႆၸိုင် ႁႃတူၺ်း Label ဢၼ်ၸိုဝ်ႈဝႃႈ **`good first issue`** ဢမ်ႇၼၼ် **`help wanted`** ၶႃႈ။ 
ၼႆႉပဵၼ်ပၼ်ႁႃဢၼ်ငၢႆႈငၢႆႈ ဢၼ်ၶဝ်ႁၢင်ႈႁႅၼ်းဝႆႉတႃႇ Beginner ၶႃႈယဝ်ႉ။



#### **ႁူဝ်ႁုပ်ႈ (Summary)**
ၵၢၼ်ႁဵတ်းသၢင်ႈၸွမ်းၵၼ်ၼိူဝ် GitHub ၼႆႉ ဢမ်ႇၸႂ်ႈၵူၺ်းဝႃႈတႅမ်ႈ Code၊ မၼ်းပဵၼ်ၵၢၼ်သၢင်ႈ **Community** (တူင်ႇဝူင်းၵူၼ်း) ဢၼ်ၵႆႉၸွႆႈထႅမ် ၵၼ်ၼၼ်ႉၶႃႈဢေႃႈ။
