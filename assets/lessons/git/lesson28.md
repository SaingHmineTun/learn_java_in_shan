## **Module 4: Distributed Collaboration**
### **Lesson 28: The World of Open Source and Forking (တူင်ႇဝူင်း Open Source)**

မိူဝ်ႈႁဝ်းပႆႇတႅမ်ႈ Code ၸွမ်းၵၼ် ႁဝ်းတေလႆႈပွင်ႇၸႂ် "ၾိင်ႈထုင်း" ၶွင်ၵၢၼ်ႁဵတ်းသၢင်ႈ ၸွမ်းၵၼ်ဢွၼ်တၢင်းၶႃႈ။

#### **1. Open Source ၼႆႉပဵၼ်သင်?**
* **Open Source** ပွင်ႇဝႃႈ "Code ဢၼ်ပိုတ်ႇပွင့်ဝႆႉ"။ ၽႂ်ၵေႃႈယဝ်ႉ ၸၢင်ႈၵႂႃႇဢၢၼ်ႇ၊ ၸၢင်ႈၵႂႃႇ Download လႄႈ ၸၢင်ႈၵႂႃႇၸွႆႈမႄး (Contribute) လႆႈၶႃႈ။
* **မိူၼ်ၼင်ႇ:** Flutter, Android, ဢမ်ႇၼၼ် Project တႃႇ Shan Community ႁဝ်း ၸိူဝ်းၼႆႉၵႆႉပဵၼ် Open Source ၶႃႈယဝ်ႉ။

#### **2. Forking vs Cloning (ပႅၵ်ႇၵၼ်ႁိုဝ်?)**
ၼႂ်း Lesson ၼႆႉ ႁဝ်းတေမႃးပွင်ႇၸႂ် "လၢႆးၵၢၼ်" မၼ်းဢွၼ်တၢင်း:
* **Forking:** မိူၼ်ၼင်ႇႁဝ်း "ပွမ် (Copy)" Project တၢင်ႇၵေႃႉ ၵႂႃႇဝႆႉၼႂ်း **GitHub Account** ႁဝ်းၶႃႈ။ (Cloud to Cloud)
* **Cloning:** မိူၼ်ၼင်ႇႁဝ်း "လူတ်ႇ (Download)" Code တီႈ GitHub (ၶွင်ႁဝ်း ဢမ်ႇၼၼ် ၶွင်တၢင်ႇၵေႃႉ) မႃးဝႆႉၼႂ်း **Computer** ႁဝ်းၶႃႈ။ (Cloud to Local)



#### **3. ၵၢၼ်ႁဵတ်းသၢင်ႈၸွမ်းၵၼ် (The Workflow)**
မိူဝ်ႈႁဝ်းၶႂ်ႈၸွႆႈတႅမ်ႈ Code ၼႂ်း Project ပိူၼ်ႈၼႆ ႁဝ်းတေလႆႈႁဵတ်းၸွမ်း 3 တွၼ်ႈလူင်ၼႆႉၶႃႈ:
1. **Fork:** ဢဝ် Project ၶဝ်မႃးဝႆႉၼႂ်း Account ႁဝ်းဢွၼ်တၢင်း (ၼင်ႇႁိုဝ် ႁဝ်းတေမီးသုၼ်ႇမႄးလႆႈ)။
2. **Clone:** ဢဝ်ဢၼ်ႁဝ်း Fork ဝႆႉၼၼ်ႉ လူတ်ႇမႃးၼႂ်းၶိူင်ႈႁဝ်း တႃႇတေတႅမ်ႈ Code။
3. **Pull Request (PR):** မိူဝ်ႈတႅမ်ႈယဝ်ႉယဝ်ႉ သူင်ႇလွင်ႈလႅၵ်ႈလၢႆႈ ၶိုၼ်းၵႂႃႇပၼ် ၸဝ်ႈၶွင် Project လူင်ၼၼ်ႉ ႁႂ်ႈၶဝ်ၵူတ်ႇထတ်း လႄႈ ႁူမ်ႈ (Merge) ဢဝ်ၶႃႈ။

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
**Module 4** ၼႆႉ တေၼႄပၼ်ဝႃႈ ႁဝ်းဢမ်ႇလူဝ်ႇႁဵတ်း ၵေႃႉလဵဝ်။ ႁဝ်းၸၢင်ႈၸွႆႈၵၼ်သၢင်ႈ App တႃႇၸိူဝ်ႉၶိူဝ်းႁဝ်း ႁႂ်ႈၶိုၼ်ႈယႂ်ႇလႆႈ လူၺ်ႈၵၢၼ်ၸႂ်ႉ **Fork, Clone, လႄႈ Pull Request** ၼႆႉၶႃႈဢေႃႈ။
