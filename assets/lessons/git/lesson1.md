## **Module 1: The Local Repository**
### **Lesson 1: What is Version Control? (ပၼ်ႁႃ 'Final_v2')**

ၼႂ်းတွၼ်ႈသွၼ်ၼႆႉ ႁဝ်းတေမႃးဢုပ်ႇၵၼ်လွင်ႈဝႃႈ **Version Control** ဢၼ်ဝႃႈၼၼ်ႉ မၼ်းပဵၼ်သင်? တေႃႉတႄႉလူဝ်ႇမၼ်းၼႃး?

#### **1. ပၼ်ႁႃ "Final_v2" ဢၼ်ၵူၺ်းထူပ်းယူႇၵူႈဝၼ်း**
မိူဝ်ႈႁဝ်းတႅမ်ႈ Application ဢမ်ႇၼၼ် တႅမ်ႈလိၵ်ႈယဝ်ႉ မိူဝ်ႈႁဝ်းၶႂ်ႈမႄးထႅင်ႈ ႁဝ်းၵႆႉၵႆႉႁဵတ်းၼႆ:
* `my_project.dart`
* `my_project_final.dart`
* `my_project_final_v2.dart`
* `my_project_final_really_fixed.dart`

လၢႆးႁဵတ်းၼႆႉ မၼ်းႁဵတ်းႁႂ်ႈ Folder ႁဝ်းယုင်ႈယၢင်ႈ၊ ဢမ်ႇႁူႉဝႃႈ File ဢၼ်လႂ်ပဵၼ်ဢၼ်မႂ်ႇသုတ်း၊ သမ်ႉဢမ်ႇၸၢင်ႈၶိုၼ်းၵႂႃႇတူၺ်းလႆႈဝႃႈ "မိူဝ်ႈဝႃးၼၼ်ႉ ႁဝ်းမႄးသင်ၵႂႃႇ?"။ ၼႆႉယဝ်ႉပဵၼ်ဢၼ်ႁဝ်းႁွင်ႉဝႃႈ **"Final_v2 Problem"**။

#### **2. Version Control ဢၼ်ဝႃႈၼၼ်ႉပဵၼ်သင်?**
**Version Control** (VCS) ၼႆႉ မၼ်းမိူၼ်ၼင်ႇ **"ၶိူင်ႈၸၢၵ်ႈယၢမ်း (Time Machine)"** တႃႇ Code ႁဝ်းၶႃႈယဝ်ႉ။
* မၼ်းတေတူၺ်းပၼ်လွင်ႈလႅၵ်ႈလၢႆႈ (Track changes) ၼႂ်း File ႁဝ်းၵူႈဢၼ်။
* ႁဝ်းၸၢင်ႈ "Save" ဝႆႉပဵၼ်တွၼ်ႈ ပဵၼ်တွၼ်ႈ (Snapshots)။
* သင်ဝႃႈႁဝ်းတႅမ်ႈ Code ၽိတ်းၵႂႃႇၵေႃႈ ႁဝ်းၸၢင်ႈၶိုၼ်း "ၵႂႃႇၸူးၶၢဝ်းယၢမ်းၵဝ်ႇ" မိူဝ်ႈ Code တိုၵ်ႉလီယူႇၼၼ်ႉလႆႈငၢႆႈငၢႆႈ။



#### **3. တႃႇသင်ႁဝ်းသမ်ႉလူဝ်ႇၸႂ်ႉ Git?**
1.  **Safety (လွင်ႈၵတ်းယဵၼ်):** ဢမ်ႇလူဝ်ႇၵူဝ် Code ႁၢႆ ဢမ်ႇၼၼ် မႄးၽိတ်းယဝ်ႉ ၶိုၼ်းႁဵတ်းလီဢမ်ႇလႆႈ။
2.  **Collaboration (ၵၢၼ်ႁဵတ်းသၢင်ႈၸွမ်းၵၼ်):** မိူဝ်ႈႁဝ်းတႅမ်ႈ Application တႃႇ **Shan Community** ႁဝ်းၼၼ်ႉ သင်မီးၵူၼ်းတႅမ်ႈၸွမ်းၵၼ်လၢႆၵူၼ်း Git တေၸွႆႈႁႂ်ႈ Code ႁဝ်းဢမ်ႇတူၵ်းၶၢင်ႈၵၼ်။
3.  **History (ပိုၼ်းၵၢၼ်တႅမ်ႈ):** ႁဝ်းၸၢင်ႈၶိုၼ်းတူၺ်းလႆႈဝႃႈ "ၽႂ်ပဵၼ်ၵူၼ်းမႄး Code ၼႆႉ?" လႄႈ "မႄးၵႂႃႇမိူဝ်ႈလႂ်?"။

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
Version Control ၼႆႉ ဢမ်ႇၸႂ်ႈၵူၺ်းဝႃႈ ပဵၼ်ၵၢၼ် Save File၊ မၼ်းပဵၼ်ၵၢၼ် **"ၵုမ်းထိင်းပိုၼ်းလႄႈလွင်ႈလႅၵ်ႈလၢႆႈ"** ၶွင်ၵၢၼ်တႅမ်ႈ Program ႁဝ်းၼၼ်ႉၶႃႈဢေႃႈ။
