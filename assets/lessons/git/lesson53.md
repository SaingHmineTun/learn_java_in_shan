## **Module 6: Android Studio Integration**
### **Lesson 53: Remote Icons (ၵၢၼ်ၸႂ်ႉမၢႆၸမ်ႈတႃႇ Pull, Push, & Fetch)**

တီႈၸဵင်ႇၶႂႃၽၢႆႇၼိူဝ် (Top-Right) ၶွင် Android Studio၊ ႁဝ်းတေႁၼ်မၢႆၸမ်ႈသီသွမ်ႇ (Blue) လႄႈ သီၶဵဝ် (Green) ဢၼ်ၸွႆႈၵုမ်းထိင်း Git ၶႃႈ။

#### **1. The Update Project Icon (သုမ်ႇမၢႆလႅၼ်းလူင်း - Blue Arrow ⬇️)**
* **Command:** `git pull`
* **လွင်ႈႁဵတ်းၵၢၼ်:** မိူဝ်ႈႁဝ်းၼဵၵ်းတီႈၼႆႉ၊ Android Studio တေၵႂႃႇလူတ်ႇဢဝ် Code ဢၼ်မႂ်ႇသုတ်းတီႈ GitHub မႃးသႂ်ႇၼႂ်းၶိူင်ႈႁဝ်း။
* **Options:** မၼ်းတေထၢမ်ဝႃႈ ႁဝ်းၶႂ်ႈ "Merge" ဢမ်ႇၼၼ် "Rebase"? (ၵႆႉလိူၵ်ႈၵၼ် **Merge** ၵေႃႈယဝ်ႉယဝ်ႉၶႃႈ)။

#### **2. The Commit Icon (မၢႆၸႅတ်ႉ - Green Check ✅)**
* **Command:** `git commit`
* **လွင်ႈႁဵတ်းၵၢၼ်:** မၼ်းတေပိုတ်ႇ **Commit Tool Window** (ဢၼ်ႁဝ်းႁဵၼ်းၼႂ်း Lesson 47) ၼၼ်ႉပၼ်ၵမ်းလဵဝ် ၼင်ႇႁႂ်ႈႁဝ်းတႅမ်ႈ Message သေသိမ်း Code ၶႃႈ။

#### **3. The Push Icon (တုမ်ႇမၢႆလႅၼ်းၶိုၼ်ႈ - Green Arrow ⬆️)**
* **Command:** `git push`
* **လွင်ႈႁဵတ်းၵၢၼ်:** မိူဝ်ႈႁဝ်း Commit ယဝ်ႉယဝ်ႉ၊ ၼဵၵ်းတီႈၼႆႉတႃႇသူင်ႇ Code ၶိုၼ်ႈ GitHub ၵမ်းလဵဝ်ၶႃႈ။



#### **4. Fetch (ၵၢၼ်ၵႂႃႇတူၺ်း... ၵူၺ်းၵႃႈ ဢမ်ႇပႆႇလူတ်ႇ)**
* **Command:** `git fetch`
* **လၢႆးၸႂ်ႉ:** ၵႂႃႇတီႈ Menu **Git > Fetch**။
* **ၵၢၼ်ႁဵတ်းၵၢၼ်:** Git တေၵႂႃႇတူၺ်းတီႈ GitHub ဝႃႈ "မီးပိူၼ်ႈမႄးသင်ဝႆႉ?"။ မၼ်းတေၼႄပၼ်သဵၼ်ႈမၢႆမႂ်ႇၼႂ်း **Git Log** (Lesson 48) ၵူၺ်း၊ ၵူၺ်းၵႃႈ မၼ်းတေဢမ်ႇပႆႇလႅၵ်ႈလၢႆႈ Code ၼႂ်း File ႁဝ်းၶႃႈ။ မၼ်းလီတႃႇတူၺ်းဢွၼ်တၢင်း မိူဝ်ႈပႆႇ Pull ၶႃႈဢေႃႈ။

#### **5. Remote Branch Status (မၢႆၸမ်ႈသီႇသႅၼ်ႇ)**
သင်ႁဝ်းႁၼ် **မၢႆလႅၼ်းသီသွမ်ႇ (Blue Arrow)** ၼႄဝႆႉယူႇၽၢႆႇၼႃႈၸိုဝ်ႈ Branch ၼႂ်း Git Log၊ မၼ်းပွင်ႇဝႃႈ: *"မီး Code မႂ်ႇယူႇၼိူဝ် GitHub ဢၼ်ႁဝ်း ဢမ်ႇပႆႇလႆႈ Pull ဢဝ်မႃးၶႃႈ"* ၼႆၶႃႈယဝ်ႉ။

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
**Remote Icons** ၸိူဝ်းၼႆႉ ႁဵတ်းႁႂ်ႈႁဝ်း ဢမ်ႇလူဝ်ႇတွင်းၸိုဝ်ႈ Command ယၢပ်ႇယၢပ်ႇ။ တေႃႉတႄႉၼဵၵ်း **Check** တႃႇ Commit၊ ၼဵၵ်း **Up Arrow** တႃႇ Push ၵေႃႈ ၵၢၼ်ယဝ်ႉယဝ်ႉၶႃႈ!
