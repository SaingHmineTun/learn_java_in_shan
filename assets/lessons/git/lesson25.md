## **Module 3: Remote Connectivity**
### **Lesson 25: Fetching vs Pulling (ၵၢၼ်ၵူတ်ႇထတ်း လႄႈ ၵၢၼ်ႁပ်ႉဢဝ်)**

ၼႂ်း Git ၼၼ်ႉ ၵၢၼ်လူတ်ႇဢဝ် Code လုၵ်ႉတီႈ GitHub မႃးမီး 2 တွၼ်ႈၼႆႉၶႃႈ:

#### **1. git fetch (တူၺ်းဢွၼ်တၢင်း)**
* **Fetch** ပွင်ႇဝႃႈ "ၵႂႃႇဢဝ်မႃး"။ မၼ်းတေၵႂႃႇလူတ်ႇဢဝ်လွင်ႈလႅၵ်ႈလၢႆႈမႂ်ႇမႂ်ႇ တီႈ GitHub မႃးဝႆႉၼႂ်းၶိူင်ႈႁဝ်း ၵူၺ်းၵႃႈ **မၼ်းတိုၵ်ႉဢမ်ႇပႆႇ Merge** သႂ်ႇၼႂ်း Code ဢၼ်ႁဝ်းတိုၵ်ႉတႅမ်ႈယူႇၼၼ်ႉ။
* မၼ်းမိူၼ်ၼင်ႇႁဝ်း "ဢဝ်လိၵ်ႈၽိုၼ်ၶၢဝ်ႇမႃးဝႆႉၼိူဝ်ၽိူၼ် (Table) ၵူၺ်းၵႃႈ ဢမ်ႇပႆႇပိုတ်ႇဢၢၼ်ႇ"။
* **တၢင်းလီ:** ႁဝ်းၸၢင်ႈတူၺ်းလႆႈဝႃႈ "မီးသင်လႅၵ်ႈလၢႆႈၵႂႃႇၽႅဝ်လႂ်ယဝ်ႉ?" လူၺ်ႈဢမ်ႇႁဵတ်းႁႂ်ႈ Code ၼႂ်းၶိူင်ႈႁဝ်းယုင်ႈယၢင်ႈ။

#### **2. git pull (ႁပ်ႉဢဝ်ၵမ်းလဵဝ်)**
* **Pull** ၼႆႉ ပွင်ႇဝႃႈ **Fetch + Merge** ၶႃႈယဝ်ႉ။
* မၼ်းတေလူတ်ႇဢဝ် Code မႃးယဝ်ႉ ဢဝ်မႃးႁူမ်ႈ (Merge) သႂ်ႇၼႂ်း Branch ဢၼ်ႁဝ်းတိုၵ်ႉယူႇၼၼ်ႉၵမ်းလဵဝ်။
* မၼ်းမိူၼ်ၼင်ႇႁဝ်း "ဢဝ်လိၵ်ႈၽိုၼ်ၶၢဝ်ႇမႃးယဝ်ႉ ပိုတ်ႇဢၢၼ်ႇသေ မႄးလႅၵ်ႈလၢႆႈတူဝ်ႁဝ်းၵမ်းလဵဝ်" ၼၼ်ႉၶႃႈဢေႃႈ။



#### **3. လၢႆးၸႂ်ႉ (Usage & Workflow)**
သင်ႁဝ်း ၶႂ်ႈႁဵတ်းၵၢၼ်လၢႆး "ၵတ်းယဵၼ် (Safety First)" ၼၼ်ႉ ထုၵ်ႇလီႁဵတ်းၼင်ႇၼႆၶႃႈ:
1. **Fetch:** `git fetch origin`
   *(ၵႂႃႇဢဝ်ၶေႃႈမုၼ်းမႂ်ႇမႃး ၵူၺ်းၵႃႈ ဢမ်ႇပႆႇမႄး Code ႁဝ်း)*
2. **Check:** `git status` ဢမ်ႇၼၼ် `git log main..origin/main`
   *(တူၺ်းဝႃႈ မီးသင်ပႅၵ်ႇပိူင်ႈၵၼ် ၸိူင်ႉႁိုဝ်?)*
3. **Merge/Pull:** သင်ဝႃႈၵူတ်ႇထတ်းယဝ်ႉယဝ်ႉသေ ပွင်ႇၸႂ်လီယဝ်ႉ ၸင်ႇတႅမ်ႈ:
   `git pull origin main`

#### **4. ပွင်ႇၸႂ်လွင်ႈ `origin/main`**
မိူဝ်ႈႁဝ်း `fetch` ယဝ်ႉ ႁဝ်းတေႁၼ် Branch ဢၼ်ၼိုင်ႈ ႁွင်ႉဝႃႈ **`origin/main`**။ ၼႆႉပဵၼ် "ၵိင်ႇ (Remote Tracking Branch)" ဢၼ်မၢႆတွင်းဝႆႉဝႃႈ "ၼိူဝ် GitHub ၼၼ်ႉ Code မၼ်းၽႅဝ်လႂ်ယဝ်ႉ" ၼႆၶႃႈဢေႃႈ။

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
**`git fetch`** ပဵၼ်လွင်ႈၵူတ်ႇထတ်း (Review)၊ **`git pull`** ပဵၼ်လွင်ႈႁပ်ႉဢဝ် (Update)။ ၵၢၼ်ၸႂ်ႉ `fetch` ဢွၼ်တၢင်း မိူဝ်ႈပႆႇႁဵတ်း `pull` ၼၼ်ႉ ၸွႆႈႁႂ်ႈ Developer ၶဝ် ဢမ်ႇတူၵ်းၸႂ် မိူဝ်ႈမီး Code မႂ်ႇမႃးႁူမ်ႈၸွမ်း Code တူဝ်ၵဝ်ႇ လူၺ်ႈဢမ်ႇႁူႉတူဝ်ၼၼ်ႉၶႃႈဢေႃႈ။
