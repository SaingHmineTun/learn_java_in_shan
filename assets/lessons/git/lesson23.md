    ## **Module 3: Remote Connectivity**
### **Lesson 23: Uploading Code (ၵၢၼ်သူင်ႇ Code ၶိုၼ်ႈၵႂႃႇလူၺ်ႈ git push)**

တွၼ်ႈတႃႇတေဢဝ် Commit ဢၼ်မီးၼႂ်း Computer ႁဝ်း သူင်ႇၶိုၼ်ႈၵႂႃႇဝႆႉၼိူဝ် GitHub ၼၼ်ႉ ႁဝ်းတေလႆႈၸႂ်ႉ Command **`git push`** ၶႃႈ။

#### **1. git push -u origin main ဢၼ်ဝႃႈၼၼ်ႉပဵၼ်သင်?**
Command ၼႆႉ မၼ်းမီးပွင်ႇၸႅင်ႈလၢႆတွၼ်ႈၶႃႈ:
* **`git push`**: ၸႂ်ႉသူင်ႇလွင်ႈလႅၵ်ႈလၢႆႈ (Commits) ၵႂႃႇၸူး Remote။
* **`origin`**: ၸိုဝ်ႈလပ်ႉၶွင် GitHub Address ဢၼ်ႁဝ်း Setup ဝႆႉၼႂ်း Lesson 21။
* **`main`**: ၸိုဝ်ႈ Branch ဢၼ်ႁဝ်းၶႂ်ႈသူင်ႇၶိုၼ်ႈၵႂႃႇ။
* **`-u` (Upstream)**: ၼႆႉပဵၼ် "ၵၢၼ်မၢႆတွင်း" ဝႃႈ ၵမ်းၼႆႉၵႂႃႇ ႁႂ်ႈ Branch `main` ၼႂ်းၶိူင်ႈႁဝ်း ၵွင်ႉၸူး `main` တီႈ `origin` (GitHub) ၼၼ်ႉဝႆႉတႃႇသေႇၶႃႈ။



#### **2. တၢင်းလီၶွင် `-u` (Upstream)**
* ႁဝ်းၸႂ်ႉမၼ်း **"ပွၵ်ႈလဵဝ်ၵူၺ်း"** မိူဝ်ႈတႄႇသူင်ႇ Branch ၼၼ်ႉဢွၼ်တၢင်းသုတ်း။
* မိူဝ်ႈႁဝ်းမၢႆဝႆႉယဝ်ႉယဝ်ႉ ၼႂ်းပွၵ်ႈၼႃႈ ႁဝ်းတေႃႉတႄႉတႅမ်ႈ `git push` တူဝ်လဵဝ်ၵေႃႈယဝ်ႉယဝ်ႉ၊ Git တေႁူႉႁင်းၵူၺ်းဝႃႈ တေလႆႈသူင်ႇၵႂႃႇတီႈလႂ်ၶႃႈဢေႃႈ။

#### **3. လၢႆးၸႂ်ႉ (Usage)**
ပိုတ်ႇ Terminal ယဝ်ႉတႅမ်ႈ:
`git push -u origin main`

* သင်ဝႃႈ GitHub ထၢမ် Username လႄႈ Password (Token) ၼၼ်ႉ ႁႂ်ႈသႂ်ႇပၼ် ၼင်ႇႁဝ်းႁဵၼ်းၵႂႃႇၼႂ်း Lesson 22 ၼၼ်ႉၶႃႈၼႃ။

#### **4. မၼ်းတေဢွၵ်ႇသင်မႃး?**
မိူဝ်ႈသူင်ႇယဝ်ႉယဝ်ႉ Git တေတႅမ်ႈၼႄဝႃႈ:
`Enumerating objects: 5, done.`
`Writing objects: 100% (5/5), 450 bytes | 450.00 KiB/s, done.`
`To https://github.com/SaiMao/my-project.git`
` * [new branch]      main -> main`
`branch 'main' set up to track 'origin/main'.`



#### **ႁူဝ်ႁုပ်ႈ (Summary)**
Command **`git push`** ၼႆႉ ပဵၼ်ၵၢၼ် **"Publish"** Code ႁဝ်းၶႃႈယဝ်ႉ။ 
မိူဝ်ႈသူင်ႇယဝ်ႉယဝ်ႉ ႁဝ်းၸၢင်ႈၵႂႃႇပိုတ်ႇတူၺ်းၼိူဝ် Website GitHub သေ တေႁၼ် Code ႁဝ်းယူႇဝႆႉတီႈၼၼ်ႉ လႄႈ ၸၢင်ႈ Share Link မၼ်းပၼ်တၢင်ႇၵေႃႉယဝ်ႉၶႃႈ။
