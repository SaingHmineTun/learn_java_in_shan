## **Module 6: Android Studio Integration**
### **Lesson 47: The Visual Commit Tool Window (Alt + 0)**

ၼႂ်း Android Studio ၼၼ်ႉ ႁဝ်းဢမ်ႇလူဝ်ႇတႅမ်ႈ `git status` တႃႇတူၺ်း File ဢၼ်လႅၵ်ႈလၢႆႈယဝ်ႉ။ 
ႁဝ်းမီး **Commit Tool Window** ဢၼ်ၼႄပၼ်ၵူႈယၢင်ႇ လႄႈ ၸွႆႈႁႂ်ႈႁဝ်း Commit လႆႈဝႆးဝႆးၶႃႈ။

#### **1. လၢႆးပိုတ်ႇ (How to Open)**
* **Shortcut:** ၼဵၵ်း **`Alt + 0`** (Windows/Linux) ဢမ်ႇၼၼ် **`Cmd + 0`** (Mac)။
* **Mouse:** ၼဵၵ်းတီႈ Tab ဢၼ်တႅမ်ႈဝႃႈ **"Commit"** ယူႇၽၢႆႇသၢႆႉ (Vertical tab) ၶွင်ၼႃႈၸေႃး Android Studio ၶႃႈ။

#### **2. တွၼ်ႈလၢႆးႁဵတ်းၵၢၼ် (Key Features)**
မိူဝ်ႈပိုတ်ႇမႃးယဝ်ႉ ႁဝ်းတေႁၼ်တွၼ်ႈလူင် 3 ဢၼ်ၼႆႉၶႃႈ:
1.  **Changes List:** မၼ်းတေၼႄ File ၵူႈဢၼ် ဢၼ်ႁဝ်းမႄးဝႆႉ (Modified) ဢမ်ႇၼၼ် ထႅမ်ဝႆႉမႂ်ႇ (New)။
2.  **Commit Message Box:** တီႈတႅမ်ႈလိၵ်ႈ Commit Message (ဢမ်ႇလူဝ်ႇတႅမ်ႈ `-m` ထႅင်ႈယဝ်ႉ)။
3.  **Diff Preview (ဢၼ်ၶိုၵ်ႉသုတ်း):** သင်ႁဝ်း ၼဵၵ်းသွင်ပွၵ်ႈ (Double click) တီႈ File ၼၼ်ႉ မၼ်းတေပိုတ်ႇၼႃႈၸေႃး ၼႄပၼ်ဝႃႈ Code ဢၼ်ၵဝ်ႇ လႄႈ ဢၼ်မႂ်ႇ မၼ်းပႅၵ်ႇၵၼ်ၼင်ႇႁိုဝ် (Side-by-side) ၶႃႈယဝ်ႉ။



#### **3. လၢႆး Commit & Push (One Click)**
ၼႂ်း Window ၼႆႉ ႁဝ်းၸၢင်ႈလိူၵ်ႈလႆႈဝႃႈ:
* **Commit:** တႃႇသိမ်းဝႆႉၼႂ်းၶိူင်ႈ (Local)။
* **Commit and Push:** တႃႇသိမ်းဝႆႉ လႄႈ သူင်ႇၶိုၼ်ႈ GitHub (Origin) ၵမ်းလဵဝ်။

#### **4. Before Commit Options**
Android Studio မီး "မေႃၸွႆႈ" ဢၼ်တေၵူတ်ႇထတ်း Code ႁဝ်း ဢွၼ်တၢင်းမိူဝ်ႈပႆႇႁဵတ်း Commit:
* **Check TODO:** မၼ်းတေၼႄပၼ် သင်ႁဝ်းတႅမ်ႈ `// TODO` ဝႆႉသေ လိုမ်းမႄး။
* **Optimize Imports:** ယႃႉပႅတ်ႈ `import` ဢၼ်ဢမ်ႇၸႂ်ႉ ႁင်းၵူၺ်း။
* **Reformat Code:** မႄးပၼ်လွင်ႈႁၢင်ႈလိၵ်ႈ Code ႁႂ်ႈသႅၼ်ႈသႂ် (Indentation) ပၼ်ၵမ်းလဵဝ်။

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
**Commit Tool Window** ၼႆႉ တေႁဵတ်းႁႂ်ႈႁဝ်း မီးသၢႆတႃ "X-ray" ဢၼ်တူၺ်း Code ႁဝ်းလႆႈၵူႈတွၼ်ႈ။ 
မၼ်းၸွႆႈႁႂ်ႈႁဝ်း ဢမ်ႇ Commit ၽိတ်း လႄႈ ႁဵတ်းႁႂ်ႈ Code ႁဝ်းမီး Quality ဢၼ်သုင်သုတ်းၼၼ်ႉၶႃႈဢေႃႈ။
