## **Module 3: Remote Connectivity**
### **Lesson 19: What is a Remote? (GitHub vs Git)**

လုၵ်ႈႁဵၼ်းၵႆႉၵႆႉ "ဝူၼ်ႉၵႂၢင်" (Confused) ၵၼ်ဝႃႈ **Git** လႄႈ **GitHub** ၼႆႉ ပဵၼ်ဢၼ်လဵဝ်ၵၼ်ႁႃႉ? ၼႆၶႃႈ။ 
ႁဝ်းမႃးၸႅၵ်ႇၽႄတူၺ်းငၢႆႈငၢႆႈၼင်ႇၼႆ:

#### **1. Git (The Tool)**
* ပဵၼ် **Software** ဢၼ်ယူႇၼႂ်း Computer ႁဝ်း။
* မီးၼႃႈၵၢၼ်: မၢႆတွင်းပိုၼ်း (Commit), သၢင်ႈ Branch, လႄႈ Merge Code။
* မိူၼ်ၼင်ႇ **"ၵူၼ်းတႅမ်ႈမၢႆတွင်း"** ဢၼ်ယူႇၸွမ်းႁဝ်းၵူႈတီႈ။

#### **2. GitHub (The Home)**
* ပဵၼ် **Website / Cloud Service** ဢၼ်ယူႇၼိူဝ် Internet။
* မီးၼႃႈၵၢၼ်: ပဵၼ် "ႁိူၼ်း" ဢမ်ႇၼၼ် "တီႈၵိုတ်း" တႃႇသိမ်း Project ဢၼ် Git မၢႆတွင်းဝႆႉၼၼ်ႉ။
* မၼ်းႁဵတ်းႁႂ်ႈႁဝ်း "Share" Code ႁဝ်းပၼ် ၵူၼ်းတၢင်ႇၵေႃႉ လႆႈငၢႆႈငၢႆႈယဝ်ႉ။
*

#### **3. Remote ဢၼ်ဝႃႈၼၼ်ႉပဵၼ်သင်?**
* **Remote** ပွင်ႇဝႃႈ "တီႈၵႆ"။ ၼႂ်းလွင်ႈ Git ၼႆႉ မၼ်းပွင်ႇဝႃႈ ပဵၼ် Repository ဢၼ်ဢမ်ႇၸႂ်ႈၼႂ်း Computer ႁဝ်း (ၵႆႉၵႆႉတႄႉ ပဵၼ်ၼိူဝ် GitHub)။
* ႁဝ်းၸၢင်ႈ "သူင်ႇ" (Push) Code ၵႂႃႇၸူး Remote လႄႈ "လူတ်ႇဢဝ်" (Pull) Code လုၵ်ႉတီႈ Remote မႃးၸူးၶိူင်ႈႁဝ်းလႆႈ။

#### **4. တၢင်းလီၶွင်ၵၢၼ်ၸႂ်ႉ Remote (GitHub)**
1. **Backup:** Code ႁဝ်းတေဢမ်ႇႁၢႆ သင် Computer ႁဝ်းလူႉ ၵွပ်ႈမၼ်းမီးဝႆႉၼိူဝ် Cloud ယဝ်ႉ။
2. **Collaboration:** ၸွႆႈႁႂ်ႈ Developer တၢင်ႇၵူၼ်း ၶဝ်ႈမႃးၸွႆႈတႅမ်ႈ Code ၸွမ်းႁဝ်းလႆႈ။
3. **Portfolio:** ၸၢင်ႈၼႄၵၢၼ်ႁဵတ်းသၢင်ႈ (Project) ႁဝ်း ပၼ်ၵူၼ်းတၢင်းလူၵ်ႈႁၼ်လႆႈ (မိူၼ်ၼင်ႇ Project တႃႇ Shan Community ႁဝ်းၼၼ်ႉၶႃႈ)။

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
**Git** ပဵၼ်တူဝ်ႁဵတ်းၵၢၼ် (Engine)၊ **GitHub** ပဵၼ်တီႈၵိုတ်း (Garage)။ ၵၢၼ်မီး **Remote** ႁဵတ်းႁႂ်ႈ Code ႁဝ်းမီးသုၼ်ႇလႆႈ "ၶိုၼ်ႈယႂ်ႇ" ၵႂႃႇၸူးတၢင်းၼွၵ်ႈ လႄႈ ဢမ်ႇၵိုတ်းဝႆႉၼႂ်း Computer ႁဝ်းၵူၺ်းၶႃႈဢေႃႈ။
