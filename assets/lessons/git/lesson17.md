## **Module 2: Branching & The Safety Net**
### **Lesson 17: Comparing Branches (ၵူတ်ႇထတ်းလွင်ႈပႅၵ်ႇပိူင်ႈ ၼႂ်းၵႄႈ Branch)**

တွၼ်ႈတႃႇတေတူၺ်းလွင်ႈလႅၵ်ႈလၢႆႈ ၼႂ်းၵႄႈသဵၼ်ႈတၢင်းသွင်သဵၼ်ႈ (Two branches) ၼၼ်ႉ ႁဝ်းတေလႆႈၸႂ်ႉ Command **`git diff`** ၶႃႈယဝ်ႉ။

#### **1. git diff ဢၼ်ဝႃႈၼၼ်ႉပဵၼ်သင်?**
* မၼ်းပဵၼ်ၵၢၼ် "ၼိူင်းၵၼ်" (Compare) ၼႂ်းၵႄႈ Commit ဢမ်ႇၼၼ် Branch။
* မၼ်းတေၼႄပၼ်:
    * **ထႅဝ်လိၵ်ႈဢၼ်ထႅမ်သႂ်ႇမႂ်ႇ:** တေပဵၼ်သီၶဵဝ် (ဢမ်ႇၼၼ် မီးမၢႆ `+`)။
    * **ထႅဝ်လိၵ်ႈဢၼ်ယႃႉပႅတ်ႈ:** တေပဵၼ်သီလႅင် (ဢမ်ႇၼၼ် မီးမၢႆ `-`)။

#### **2. လၢႆးၸႂ်ႉ (Usage)**
တႃႇတေတူၺ်းလွင်ႈပႅၵ်ႇပၵ်ႇ ၼႂ်းၵႄႈ Branch `main` လႄႈ `feature` ႁဝ်းတႅမ်ႈၼင်ႇၼႆ:
`git diff main..feature-branch`

* **မၢႆတွင်း:** မၢႆၸမ်ႈသွင်ဢၼ် `..` ၼၼ်ႉ ပွင်ႇဝႃႈ "တႄႇတူၺ်းတီႈ `main` ၵႂႃႇၸူး `feature`"။
* မၼ်းတေၼႄပၼ်ၵူႈလွင်ႈ ဢၼ် `feature` မီး ၵူၺ်းၵႃႈ `main` ဢမ်ႇပႆႇမီးၼၼ်ႉၶႃႈဢေႃႈ။



#### **3. ၵူတ်ႇထတ်း "ၸိုဝ်ႈ File" (Stat) ၵူၺ်း**
သင်ဝႃႈႁဝ်းမႄး Code ၼမ်ၼမ်သေ ဢမ်ႇၶႂ်ႈတူၺ်းၵူႈထႅဝ်လိၵ်ႈ ၵူၺ်းၶႂ်ႈႁူႉဝႃႈ File လႂ်လႅၵ်ႈလၢႆႈၵႂႃႇၽႅဝ်လႂ်ယဝ်ႉၼႆ ၸႂ်ႉ:
`git diff main..feature-branch --stat`

* မၼ်းတေၼႄသဵၼ်ႈမၢႆ File လႄႈ ၼႄဝႃႈ "မီးလွင်ႈထႅမ်သႂ်ႇ (Insertions) လၢႆထႅဝ်၊ ယႃႉပႅတ်ႈ (Deletions) လၢႆထႅဝ်" ၼႆၶႃႈဢေႃႈ။

#### **4. တၢင်းလီၶွင်ၵၢၼ် Compare**
1. **Quality Control (ၵုမ်းထိင်းၼမ်ႉၸႂ်):** ႁဵတ်းႁႂ်ႈႁဝ်းႁၼ် Error ဢွၼ်တၢင်း မိူဝ်ႈပႆႇ Merge။
2. **Self-Review:** မိူၼ်ၼင်ႇႁဝ်းၵူတ်ႇထတ်း ၵၢၼ်ႁဵတ်းသၢင်ႈတူဝ်ႁဝ်းၶိုၼ်း။
3. **Communication:** သင်ႁဝ်းႁဵတ်းၵၢၼ်ၸွမ်းၵၼ်လၢႆလၢႆၵေႃႉ ႁဝ်းၸၢင်ႈၼႄလႆႈဝႃႈ "ၼႆႉယဝ်ႉ ပဵၼ် Code ဢၼ်ၶႃႈမႄးဝႆႉ" ၼႆၶႃႈ။

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
Command **`git diff`** ၼႆႉ ၸွႆႈႁႂ်ႈႁဝ်းႁၼ် လွင်ႈလႅၵ်ႈလၢႆႈ ၼႂ်းၵႄႈ Branch ယဝ်ႉၶႃႈ။
ၵၢၼ်တူၺ်း `diff` ဢွၼ်တၢင်းမိူဝ်ႈပႆႇ Merge ၼၼ်ႉ ပဵၼ်လွင်ႈဢၼ် Developer ၶဝ် တၵ်းတေလႆႈႁဵတ်း ၵူႈပွၵ်ႈၶႃႈဢေႃႈ။
