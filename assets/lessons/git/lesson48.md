## **Module 6: Android Studio Integration**
### **Lesson 48: Navigating the Visual Git Log Tab (ၵၢၼ်တူၺ်းပိုၼ်း Git လူၺ်ႈႁၢင်ႈၽၢင်)**

ႁဝ်းၸၢင်ႈပိုတ်ႇတူၺ်း **Git Log** လႆႈလူၺ်ႈ ၼဵၵ်းတီႈ Tab **"Git"** ဢၼ်ယူႇၽၢႆႇတႂ်ႈသုတ်း (Bottom) ၶွင် Android Studio ၶႃႈ။

#### **1. တွၼ်ႈလၢႆးႁဵတ်းၵၢၼ် (Log Tab Components)**
မိူဝ်ႈပိုတ်ႇမႃးယဝ်ႉ ႁဝ်းတေႁၼ် 3 တွၼ်ႈလူင်ၼႆႉၶႃႈ:
1.  **The Graph (သဵၼ်ႈတၢင်း):** မၼ်းတေၼႄ Commit ၵူႈဢၼ် ပဵၼ်သဵၼ်ႈလၢၵ်ႈၸူးၵၼ်။ သင်မီးလၢႆ Branch မၼ်းတေၼႄပဵၼ်လၢႆသီ ႁႂ်ႈႁဝ်းႁူႉဝႃႈ Branch လႂ်လုၵ်ႉတီႈလႂ်မႃးၶႃႈ။
2.  **Commit List:** ၼႄၸိုဝ်ႈ Commit, ၵူၼ်းတႅမ်ႈ (Author), လႄႈ ဝၼ်းယၢမ်း။
3.  **Selection Details (ၽၢႆႇၶႂႃ):** မိူဝ်ႈႁဝ်းၼဵၵ်းလိူၵ်ႈ Commit ဢၼ်ၼိုင်ႈ၊ မၼ်းတေၼႄပၼ်ဝႃႈ Commit ၼၼ်ႉ မႄး File လႂ်ၵႂႃႇလၢႆလၢႆ?



#### **2. လၢႆးၸႂ်ႉ Filter (ၵၢၼ်သွၵ်ႈႁႃ)**
သင် Project ႁဝ်းလူင်ၼႅၼ်ႇ၊ ႁဝ်းၸၢင်ႈ "ၸႅတ်ႈ (Filter)" တူၺ်းလႆႈငၢႆႈငၢႆႈ:
* **Branch:** တူၺ်းပိုၼ်း Branch ဢၼ်ႁဝ်းလိူၵ်ႈၵူၺ်း။
* **User:** တူၺ်းပိုၼ်း ဢၼ်ၵူၼ်းႁဵတ်း ၵေႃႉၼိုင်ႈတႅမ်ႈဝႆႉၵူၺ်း။
* **Date:** တူၺ်း ဢၼ်ႁဝ်းမႄးဝႆႉ မိူဝ်ႈဝၼ်းထိၼၼ်ႉၵူၺ်း။
* **Path:** တူၺ်း လွင်ႈလႅၵ်ႈလၢႆႈၼႂ်း Folder `lib/` ဢမ်ႇၼၼ် `assets/` ၵူၺ်း။

#### **3. Right-Click Magic (မိုဝ်းၶႂႃၼဵၵ်း)**
ႁဝ်းၸၢင်ႈၼဵၵ်းမိုဝ်းၶႂႃ တီႈ Commit ဢၼ်လႂ်ၵေႃႈယဝ်ႉ သေ ႁဵတ်းၵၢၼ်ၸိူဝ်းၼႆႉလႆႈၵမ်းလဵဝ်:
* **Checkout Revision:** ထွႆလင်ၵႂႃႇတူၺ်း Code မိူဝ်ႈတွၼ်ႈၼၼ်ႉ။
* **New Branch from Selected:** သၢင်ႈ Branch မႂ်ႇ လုၵ်ႉတီႈ Commit ၼၼ်ႉ။
* **Reset Current Branch to Here:** ၸႂ်ႉ `git reset` (Lesson 40) လူၺ်ႈမိုဝ်းၼဵၵ်းၵူၺ်း။
* **Revert Commit:** ၸႂ်ႉ `git revert` (Lesson 41) တႃႇယႃႉလွင်ႈၽိတ်း။

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
**Git Log Tab** ၼႆႉ မိူၼ်ၼင်ႇ "ၵႅမ်းပိုၼ်း (Archive)" ၶွင် Project။ 
မၼ်းႁဵတ်းႁႂ်ႈႁဝ်း ႁၼ် "ႁၢင်ႈၽၢင်လူင် (Big Picture)" ၶွင်ၵၢၼ်ႁဵတ်းသၢင်ႈၸွမ်းၵၼ် လႄႈ ၸွႆႈႁႂ်ႈႁဝ်းႁႃလွင်ႈၽိတ်းလႆႈ ဝႆးသုတ်းၶႃႈဢေႃႈ။
