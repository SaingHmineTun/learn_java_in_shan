## **Module 2: Branching & The Safety Net**
### **Lesson 11: Creating and Listing Branches (သၢင်ႈလႄႈတူၺ်းသဵၼ်ႈမၢႆ Branch)**

မိူဝ်ႈႁဝ်းတိုၵ်ႉ `git init` မႂ်ႇမႂ်ႇၼၼ်ႉ Git တေမီးပၼ် Branch ဢၼ်ၼိုင်ႈ ႁွင်ႉဝႃႈ **`main`** (ဢမ်ႇၼၼ် `master`) ဢၼ်ပဵၼ် သဵၼ်ႈတၢင်းလူင်ၼၼ်ႉၶႃႈ။

#### **1. လၢႆးတူၺ်း Branch တင်ႈမူတ်း**
ဢွၼ်တၢင်းတေၸႂ်ႉ တၵ်းတေလႆႈႁူႉဝႃႈ တွၼ်ႈၼႆႉႁဝ်းမီး Branch သင်ဝႆႉၽႅဝ်လႂ်ယဝ်ႉၼႆ:
`git branch`

* မၼ်းတေၼႄသဵၼ်ႈမၢႆ Branch တင်ႈမူတ်း။
* Branch ဢၼ်မီးမၢႆၸမ်ႈ **`*`** ယူႇၽၢႆႇၼႃႈၼၼ်ႉ ပွင်ႇဝႃႈ **HEAD** တိုၵ်ႉၸိဝႆႉ (ႁဝ်းတိုၵ်ႉယူႇဝႆႉ) တီႈၼၼ်ႉၶႃႈ။

#### **2. လၢႆးသၢင်ႈ Branch မႂ်ႇ**
သင်ႁဝ်းၶႂ်ႈတႄႇတႅမ်ႈ Feature မႂ်ႇ (မိူၼ်ၼင်ႇ "Shan Fonts Support") ႁဝ်းတေလႆႈသၢင်ႈ Branch မႂ်ႇၼင်ႇၼႆ:
`git branch feature-fonts`

* **မၢႆတွင်း:** Command ၼႆႉ မၼ်းတေ **"သၢင်ႈ"** ၵူၺ်း၊ မၼ်းတိုၵ်ႉဢမ်ႇပႆႇ **"လႅၵ်ႈ"** ၵႂႃႇၸူး Branch မႂ်ႇၼၼ်ႉၶႃႈ။ (ႁဝ်းတိုၵ်ႉတေယူႇဝႆႉတီႈ `main` မိူၼ်ၵဝ်ႇ)။



#### **3. လၢႆးတင်ႈၸိုဝ်ႈ Branch ႁႂ်ႈလူငၢႆႈ**
တႃႇတေတႅမ်ႈ App တႃႇ **Shan Community** ႁဝ်းၼၼ်ႉ ထုၵ်ႇလီတင်ႈၸိုဝ်ႈႁႂ်ႈပွင်ႇၸႂ်ငၢႆႈ:
* `feature/login-ui` (တႃႇသၢင်ႈၼႃႈတႃ Login)
* `bugfix/font-error` (တႃႇမႄး Error လွင်ႈလိၵ်ႈတႆး)
* `hotfix/crash-main` (တႃႇမႄးလွင်ႈ App လူႉဝႆးဝႆး)

#### **4. လၢႆးမႄးၸိုဝ်ႈ လႄႈ လၢႆး Delete**
* **Rename (မႄးၸိုဝ်ႈ):** `git branch -m <new-name>` (မႄႈၸိုဝ်ႈပၼ် branch ဢၼ်ႁဝ်းတိုၵ်ႉယူႇဝႆႉ)
* **Delete (ယႃႉပႅတ်ႈ):** `git branch -d <name>`
  *(မၢႆတွင်း: ႁဝ်းတေဢမ်ႇၸၢင်ႈ Delete Branch ဢၼ်ႁဝ်းတိုၵ်ႉယူႇဝႆႉၼၼ်ႉလႆႈၶႃႈ)*

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
Command **`git branch`** ၼႆႉ မိူၼ်ၼင်ႇ **"သဵၼ်ႈမၢႆတၢင်းၽႄ"** ၶွင်ႁဝ်းၶႃႈယဝ်ႉ။ 
မၼ်းၸွႆႈႁႂ်ႈႁဝ်းႁူႉဝႃႈ ႁဝ်းမီးသုၼ်ႇ ၵႂႃႇတၢင်းလႂ်လႆႈၼႆၶႃႈ။ 
ၵၢၼ်သၢင်ႈ Branch ၼမ်ၼမ် ဢမ်ႇၸႂ်ႈလွင်ႈၽိတ်း၊ မၼ်းၸွႆႈႁႂ်ႈၵၢၼ်တႅမ်ႈ Code ႁဝ်းမီးပိူင်မီးၽၢင်ၶႃႈဢေႃႈ။
