## **Module 6: Android Studio Integration**
### **Lesson 52: The Power of 'Annotate' (ၵၢၼ်တူၺ်းပိုၼ်း တူဝ်ၵူၼ်းတႅမ်ႈ Code)**

**Annotate** ၼႆႉ မၼ်းတေၼႄပၼ် "ၸိုဝ်ႈၵူၼ်းတႅမ်ႈ" လႄႈ "ဝၼ်းယၢမ်း" ဝႆႉယူႇၽၢႆႇၼႃႈ ထႅဝ်လိၵ်ႈ Code ၵူႈထႅဝ်ၶႃႈ။

#### **1. လၢႆးပိုတ်ႇၸႂ်ႉ (How to Enable)**
1. ပိုတ်ႇ File Code ဢၼ်ႁဝ်းၶႂ်ႈတူၺ်း။
2. ၼဵၵ်းမိုဝ်းၶႂႃ (Right-click) တီႈ **"သဵၼ်ႈမၢႆမၢႆထႅဝ်" (Gutter / Line number bar)** ဢၼ်ယူႇၽၢႆႇသၢႆႉသုတ်းၶွင် Code။
3. လိူၵ်ႈ **"Annotate with Git Blame"**။



#### **2. တၢင်းလီၶွင် Annotate**
* **Who & When:** ႁူႉလႆႈၵမ်းလဵဝ်ဝႃႈ ထႅဝ်ၼႆႉ ၵေႃႉလႂ် တႅမ်ႈဝႆႉမိူဝ်ႈလႂ်။
* **The "Why" (Commit Message):** သင်ႁဝ်းဢဝ် Mouse ၵႂႃႇဝႆႉ (Hover) တီႈၸိုဝ်ႈၵူၼ်းတႅမ်ႈၼၼ်ႉ၊ Android Studio တေၼႄပၼ် **Commit Message** ဢၼ်ၵွင်ႉၸူးထႅဝ်ၼၼ်ႉ။ မၼ်းတေၸွႆႈႁႂ်ႈႁဝ်းပွင်ႇၸႂ်ဝႃႈ "ၶဝ်တႅမ်ႈၼင်ႇၼႆ ၵွပ်ႈၶဝ်ၶႂ်ႈမႄး Bug တွၼ်ႈၼၼ်ႉၼႆၶႃႈၼႃ"။
* **Jump to History:** သင်ႁဝ်းၼဵၵ်းတီႈ Annotate ၼၼ်ႉ၊ မၼ်းတေပိုတ်ႇ Tab **Git Log** သေ ၸီႉပၼ်တီႈ Commit ၼၼ်ႉၵမ်းလဵဝ်ၶႃႈ။

#### **3. Visual Diff (တူၺ်းလွင်ႈလႅၵ်ႈလၢႆႈ)**
ႁဝ်းၸၢင်ႈၼဵၵ်းမိုဝ်းၶႂႃတီႈ Annotate သေလိူၵ်ႈ **"Show Diff"**။ မၼ်းတေၼႄပၼ်ဝႃႈ ဢွၼ်တၢင်း ပႆႇမီးထႅဝ်ၼႆႉ Code ၵဝ်ႇမၼ်းပဵၼ်ၼင်ႇႁိုဝ်? လႄႈ ၵေႃႉၼႆႉမႃးလႅၵ်ႈလၢႆႈသင်ၵႂႃႇ?

#### **4. လၢႆးပိၵ်ႉ (How to Turn Off)**
ၼဵၵ်းမိုဝ်းၶႂႃတီႈ Gutter ထႅင်ႈပွၵ်ႈၼိုင်ႈ သေ ၼဵၵ်းတီႈ **"Close Annotations"** ၵေႃႈယဝ်ႉယဝ်ႉၶႃႈ။

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
**Annotate** ၼႆႉ ဢမ်ႇၸႂ်ႈၵၢၼ် "တိပ်ႇထတ်း" ပိူၼ်ႈ (Blame)၊ ၵူၺ်းၵႃႈ မၼ်းပဵၼ်ၵၢၼ် "ပွင်ႇၸႂ်" ပိုၼ်းၶွင် Code။ မၼ်းၸွႆႈႁႂ်ႈႁဝ်း ၵုမ်းထိင်း Project လူင် ဢၼ်မီးၵူၼ်းတႅမ်ႈလၢႆလၢႆၵေႃႉ ၸွႆႈၵၼ်တႅမ်ႈၼၼ်ႉ လႆႈငၢႆႈသုတ်းၶႃႈဢေႃႈ။
