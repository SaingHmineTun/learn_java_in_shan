## **Module 1: The Local Repository**
### **Lesson 3: The .git Directory (ပွင်ႇၸႂ်လွင်ႈ ၶေႃႈမုၼ်းလပ်ႉ)**

မိူဝ်ႈႁဝ်းတႄႇၸႂ်ႉ Git ၼႂ်း Project ႁဝ်းၼၼ်ႉ မၼ်းတေမီး Folder လပ်ႉဢၼ်ၼိုင်ႈ ၸိုဝ်ႈဝႃႈ **.git** ဢွၵ်ႇမႃးၶႃႈ။

#### **1. .git Directory ဢၼ်ဝႃႈၼၼ်ႉပဵၼ်သင်?**
* မၼ်းပဵၼ် Folder ဢၼ် **Git** ၸႂ်ႉတႃႇသိမ်းဝႆႉ "ပိုၼ်း (History)" ၶွင် Project ႁဝ်း။
* သင်ဝႃႈႁဝ်း "Delete" ပႅတ်ႈ Folder ၼႆႉၼႆ **ပိုၼ်းၵၢၼ်တႅမ်ႈ Code ႁဝ်းၵူႈလွင်ႈ တေႁၢႆၵႂႃႇၵမ်းလဵဝ်** (ၵူၺ်းၵႃႈ File Code ႁဝ်းတႄႉ တိုၵ်ႉတေမီးယူႇ)။
* မၼ်းပဵၼ် **"မၢႆတွင်း (Memory)"** ၶွင် Git ယဝ်ႉၶႃႈ။

#### **2. လၢႆးတူၺ်း Folder လပ်ႉ (Hidden Folder)**
ၵွပ်ႈပဵၼ် Folder ဢၼ်မီးမၢႆၸမ်ႈ (`.`) ဝႆႉတီႈၼႃႈမၼ်း ၼႂ်း Windows ဢမ်ႇၼၼ် Mac တေဢမ်ႇႁၼ်မၼ်းငၢႆႈငၢႆႈ။
* **Command Line:** တႅမ်ႈ `ls -a` (တႃႇ Mac/Linux) ဢမ်ႇၼၼ် `dir /a` (တႃႇ Windows)။
* **File Explorer:** တေလႆႈၵႂႃႇပိုတ်ႇ "Show hidden files" ၼႂ်း Settings ဢွၼ်တၢင်းၶႃႈ။



#### **3. တၢင်းၼႂ်း .git မၼ်းမီးသင်ဝႆႉ?**
ႁဝ်းဢမ်ႇလူဝ်ႇမႄးသင်ၼႂ်းၼႆႉ ၵူၺ်းၵႃႈ ထုၵ်ႇလီႁူႉဝႆႉဝႃႈမၼ်းမီး:
* **config:** သိမ်းဝႆႉလွင်ႈ Settings ၶွင် Project ၼႆႉ (မိူၼ်ၼင်ႇ Remote URL)။
* **objects:** ၼႆႉပဵၼ် "ႁွင်ႈ" ဢၼ်သိမ်းဝႆႉ Code ႁဝ်း တႃႈ Version ၵူႈဢၼ်။
* **refs:** သိမ်းဝႆႉလွင်ႈ **Branches** (မိူၼ်ၼင်ႇ main, feature)။
* **HEAD:** ပဵၼ်တူဝ်မၢႆဝႆႉဝႃႈ "တွၼ်ႈၼႆႉ ႁဝ်းတိုၵ်ႉမီးဝႆႉတီႈ Version လႂ်?"။

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
Folder **.git** ၼႆႉ ပဵၼ် "ႁွင်ႈၶေႃႈမုၼ်း (Database)" ဢၼ်လပ်ႉဝႆႉ။ 
မၼ်းမၢႆတွင်းပၼ်ၵူႈလွင်ႈ ဢၼ်ႁဝ်းလႅၵ်ႈလၢႆႈၼႂ်း Code ႁဝ်း။ 
သင်မီး Folder ၼႆႉယူႇ ႁဝ်းၸၢင်ႈၶိုၼ်းၵႂႃႇတူၺ်း "ပိုၼ်းၵဝ်ႇ" လႆႈၵူႈၶၢဝ်းယၢမ်းၶႃႈဢေႃႈ။
