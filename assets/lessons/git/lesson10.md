## **Module 2: Branching & The Safety Net**
### **Lesson 10: The Concept of Branching (လူၵ်ႈၵိုင်ႇၵၼ် - Parallel Universes)**

မိူဝ်ႈႁဝ်းတႅမ်ႈ Application တႃႇ **Shan Community** ႁဝ်းၼၼ်ႉ ၵႆႉမီးငဝ်းလၢႆးၼင်ႇၼႆ:
* ႁဝ်းမီး Code ဢၼ်ၸႂ်ႉလႆႈလီယူႇ (Stable)။
* ႁဝ်းၶႂ်ႈထႅမ်သႂ်ႇ Feature မႂ်ႇ (မိူၼ်ၼင်ႇ "Dark Mode")။
* ၵူၺ်းၵႃႈ ႁဝ်းဢမ်ႇၶႂ်ႈမႄးၼႂ်း Code ဢၼ်လီယူႇၼၼ်ႉ ၵွပ်ႈၵူဝ်မၼ်းပဵၼ် Error သေ ႁဵတ်းႁႂ်ႈ App ႁဝ်းၸႂ်ႉဢမ်ႇလႆႈ။

ၼႆႉယဝ်ႉ ပဵၼ်တီႈဢၼ် **Branching** တေမႃးၸွႆႈၶႃႈ။

#### **1. Branch ဢၼ်ဝႃႈၼၼ်ႉပဵၼ်သင်?**
* **Branch** (ၵိင်ႇမႆႉ) ၼႆႉ မိူၼ်ၼင်ႇႁဝ်း "ၽႅၵ်ႇ" တၢင်းဢွၵ်ႇၵႂႃႇပဵၼ်သဵၼ်ႈမႂ်ႇ။
* ႁဝ်းၸၢင်ႈတႅမ်ႈ Code ၼႂ်း Branch မႂ်ႇ ၼင်ႇၵၢင်ၸႂ် လူၺ်ႈဢမ်ႇၵႂႃႇၵဝ်း Branch ဢၼ်ပဵၼ် `main` ၶႃႈ။



#### **2. တၢင်းလီၶွင်ၵၢၼ်ၸႂ်ႉ Branch**
1. **Experiment (ၵၢၼ်ၸၢမ်း):** ၸၢင်ႈၸၢမ်းတႅမ်ႈ Code ယုင်ႈယၢင်ႈတူၺ်းလႆႈ။ သင်ဝႃႈမၼ်းလူႉၵေႃႈ တေႃႉတႄႉ Delete Branch ၼၼ်ႉပႅတ်ႈၵူၺ်း၊ Code တီႈ `main` တိုၵ်ႉတေလီယူႇမိူၼ်ၵဝ်ႇ။
2. **Teamwork (ၵၢၼ်ႁဵတ်းသၢင်ႈ ၵူၼ်းၼမ်):** ၵူၼ်းၵေႃႉၼိုင်ႈတႅမ်ႈလွင်ႈ "Login", ထႅင်ႈၵေႃႉၼိုင်ႈတႅမ်ႈလွင်ႈ "Profile" ၼႂ်း Branch ၽႂ်မၼ်း။
3. **Safety (လွင်ႈၵတ်းယဵၼ်):** Code တီႈ Branch `main` တၵ်းတေလႆႈပဵၼ် Code ဢၼ် "သႅၼ်ႈသႂ်" လႄႈ "ၸႂ်ႉလႆႈ" ၵူႈၶၢဝ်းယၢမ်း။

#### **3. HEAD ဢၼ်ဝႃႈၼၼ်ႉပဵၼ်သင်?**
* ၼႂ်း Git မၼ်းတေမီးတူဝ်ၸိၼႄဢၼ်ၼိုင်ႈ ႁွင်ႉဝႃႈ **HEAD**။
* မၼ်းမိူၼ်ၼင်ႇ **"တူဝ်မၢႆဝႃႈ တွၼ်ႈၼႆႉႁဝ်းတိုၵ်ႉမီးဝႆႉတီႈ Branch လႂ်?"** ၼႆၶႃႈဢေႃႈ။ မိူဝ်ႈႁဝ်းလႅၵ်ႈ Branch ၵႂႃႇမႃး HEAD ၵေႃႈတေၵႂႃႇၸွမ်းၶႃႈ။



#### **ႁူဝ်ႁုပ်ႈ (Summary)**
**Branching** ၼႆႉ ဢမ်ႇၸႂ်ႈၵူၺ်းဝႃႈ ပဵၼ်ၵၢၼ်တႅမ်ႈ Code လၢႆလၢႆသဵၼ်ႈ၊ မၼ်းပဵၼ် **"လွင်ႈၵတ်းယဵၼ်ၸႂ် (Safety Net)"** တႃႇ Developer ၶႃႈယဝ်ႉ။ မၼ်းႁဵတ်းႁႂ်ႈႁဝ်း မီးသုၼ်ႇ "ၽိတ်း" လႆႈ ၼႂ်းၵၢၼ်တႅမ်ႈ Code လူၺ်ႈဢမ်ႇလူဝ်ႇၵူဝ်သင်ယဝ်ႉၶႃႈ။
