## **Module 2: Branching & The Safety Net**
### **Lesson 18: Deleting and Renaming Branches Safely (ယႃႉပႅတ်ႈလႄႈမႄးၸိုဝ်ႈ Branch ႁႂ်ႈၵတ်းယဵၼ်)**

ၼႂ်းၵၢၼ်တႅမ်ႈ Code တႄႉတႄႉၼၼ်ႉ ႁဝ်းၸၢင်ႈသၢင်ႈ Branch ၼမ်ၼမ်။ 
သင်ႁဝ်းဢမ်ႇယႃႉပႅတ်ႈ Branch ဢၼ်ၸႂ်ႉယဝ်ႉၼၼ်ႉၼႆ မၼ်းတေႁဵတ်းႁႂ်ႈႁဝ်း "ဝူၼ်ႉၵႂၢင်" (Confused) ဝႃႈဢၼ်လႂ်ပဵၼ်ဢၼ်လႂ်ၶႃႈဢေႃႈ။

#### **1. Renaming a Branch (မႄးၸိုဝ်ႈ Branch)**
သင်ႁဝ်းပၼ်ၸိုဝ်ႈၽိတ်း ဢမ်ႇၼၼ် ၶႂ်ႈမႄးၸိုဝ်ႈႁႂ်ႈမၼ်းၸႅင်ႈလႅင်းမႃးထႅင်ႈ။
* **မႄးၸိုဝ်ႈ Branch ဢၼ်ႁဝ်းတိုၵ်ႉယူႇဝႆႉ (Current Branch):**
  `git branch -m feature-new-name`
* **မႄးၸိုဝ်ႈ Branch တၢင်ႇဢၼ် (Any Branch):**
  `git branch -m old-name new-name`

#### **2. Deleting a Branch Safely (လၢႆးယႃႉပႅတ်ႈ ဢၼ်ၵတ်းယဵၼ်)**
Git မီးလၢႆးယႃႉပႅတ်ႈ 2 လၢႆး ဢၼ်ပႅၵ်ႇၵၼ်ၶႃႈ:

* **လၢႆးၵတ်းယဵၼ် (`-d`):**
  `git branch -d feature-login`
    * Git တေယႃႉပၼ် **branch** ဝၢႆးသေႁဝ်း Merge Code ၼႂ်း Branch ၼၼ်ႉ ၶဝ်ႈၸူး `main` ယဝ်ႉတူဝ်ႈယဝ်ႉၵူၺ်း။ သင်ႁဝ်းဢမ်ႇပႆႇ Merge မၼ်းတေႁၢမ်ႈႁဝ်း ၵွပ်ႈမၼ်းၵူဝ် Code ႁဝ်းႁၢႆၶႃႈ။

* **လၢႆးၸႂ်ႁၢႆႉ (`-D`):**
  `git branch -D feature-failed-experiment`
    * ၼႆႉပဵၼ်ၵၢၼ် **"Force Delete"**။ မၼ်းတေယႃႉပႅတ်ႈၵမ်းလဵဝ် လူၺ်ႈဢမ်ႇတူၺ်းဝႃႈ Merge ယဝ်ႉႁႃႉ ဢမ်ႇပႆႇယဝ်ႉ။ ႁဝ်းတေၸႂ်ႉမၼ်း မိူဝ်ႈႁဝ်း တႅမ်ႈ Code ယဝ်ႉသေ ဢမ်ႇၶႂ်ႈဢဝ်သင် ၼႂ်းၼၼ်ႉယဝ်ႉ ၼႆၶႃႈဢေႃႈ။



#### **3. မၢႆတွင်းဢၼ်ထုၵ်ႇလီၾၢင်ႉဝႆႉ (Caution)**
* ႁဝ်းဢမ်ႇၸၢင်ႈယႃႉပႅတ်ႈ Branch ဢၼ်ႁဝ်း **"တိုၵ်ႉယူႇဝႆႉ"** (Current Branch)။ သင်ၶႂ်ႈယႃႉ `feature-A` ႁဝ်းတေလႆႈ `checkout` ၵႂႃႇယူႇတီႈ `main` ဢွၼ်တၢင်းၶႃႈ။
* မိူဝ်ႈယႃႉပႅတ်ႈယဝ်ႉၼၼ်ႉ တူၺ်းၶိုၼ်း `git branch` ႁႂ်ႈႁူႉဝႃႈ မၼ်းႁၢႆၵႂႃႇယဝ်ႉတႄႉၼႆၶႃႈ။

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
ၵၢၼ် **Cleanup** ၼႆႉ ပဵၼ်ႁူဝ်ၸႂ်ၶွင် Developer ဢၼ်မီးပိူင်မီးၽၢင်ၶႃႈယဝ်ႉ။ 
ၵၢၼ်မႄးၸိုဝ်ႈႁႂ်ႈထုၵ်ႇ လႄႈ ၵၢၼ်ယႃႉပႅတ်ႈ Branch ဢၼ်ဢမ်ႇလူဝ်ႇၸိူဝ်းၼၼ်ႉ ႁဵတ်းႁႂ်ႈ Project ႁဝ်းသႅၼ်ႈသႂ် လႄႈ 
ႁဵတ်းႁႂ်ႈၵူၼ်းၼႂ်း Team ပွင်ႇၸႂ်ၵၼ်ငၢႆႈငၢႆႈၶႃႈဢေႃႈ။
