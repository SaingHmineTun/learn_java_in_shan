## **Module 5: Troubleshooting & Data Integrity**
### **Lesson 38: Resolving Conflicts Manually (လၢႆးမႄး Conflict ႁင်းၶေႃ)**

မိူဝ်ႈ Git ဢမ်ႇၸၢင်ႈ Merge ပၼ်ႁင်း၊ မၼ်းတေ "ၵိုတ်း" ဝႆႉသေ ပႂ်ႉႁႂ်းႁဝ်းမႃးမႄးၶႃႈ။ လၢႆးမႄးမၼ်းမီး 4 တွၼ်ႈငၢႆႈငၢႆႈၼင်ႇၼႆၶႃႈ:

#### **1. ႁႃထႅဝ်လိၵ်ႈဢၼ်မီးပၼ်ႁႃ (Identify)**
ပိုတ်ႇ File ဢၼ် Git တႅမ်ႈဝႃႈမီး Conflict ၼၼ်ႉသေ ႁႃမၢႆၸမ်ႈ (Markers) ၸိူဝ်းၼႆႉ:
* `<<<<<<< HEAD`
* `=======`
* `>>>>>>> [branch-name]`



#### **2. တႅပ်းတတ်းသဵၼ်ႈၸႂ် တေလိူၵ်ႈဢၼ်လႂ် (The Decision)**
ႁဝ်းတေလႆႈလိူၵ်ႈဢဝ် 1 လၢႆး ၼႂ်း 3 လၢႆးၼႆႉၶႃႈ:
* **Keep Yours:** ဢဝ် Code ႁဝ်း (HEAD) ဝႆႉ၊ ယႃႉပႅတ်ႈ Code ဢၼ်မႃးမႂ်ႇ။
* **Keep Theirs:** ဢဝ် Code ဢၼ်မႃးမႂ်ႇဝႆႉ၊ ယႃႉပႅတ်ႈ Code ႁဝ်း (HEAD)။
* **Keep Both / Hybrid:** ဢဝ် Code တင်းသွင်ဢၼ်မႃး "ၵွင်ႉ" ၸူးၵၼ် ႁႂ်ႈမၼ်းႁဵတ်းၵၢၼ်လႆႈတင်းသွင်ယၢင်ႇ။

#### **3. သႅၼ်ႈသႂ် File (Clean Up)**
မိူဝ်ႈလိူၵ်ႈ Code ဢၼ်ၶႂ်ႈလႆႈယဝ်ႉယဝ်ႉ:
* **"တၵ်းတေလႆႈယႃႉပႅတ်ႈ"** မၢႆၸမ်ႈ Git (`<<<<`, `====`, `>>>>`) တင်ႈမူတ်း။ သင်ၵိုတ်းဝႆႉသေဢၼ်ဢၼ် Code ႁဝ်းတေပဵၼ် Error ၵမ်းလဵဝ်ၶႃႈ။
* မႄး Code ႁႂ်ႈမၼ်းတႅတ်ႈတေႃးသေ သိမ်း (Save) File ၼၼ်ႉၶႃႈ။

#### **4. ပိတ်းပၢင် Merge (Finalize)**
မိူဝ်ႈမႄးယဝ်ႉယဝ်ႉ Git တိုၵ်ႉပႂ်ႉႁဝ်းယူႇၶႃႈ။ ႁဝ်းတေလႆႈ "မၢႆတွင်း" ဝႃႈမႄးယဝ်ႉယဝ်ႉလူၺ်ႈ:
1.  `git add [file-name]` (တႃႇမၢႆဝႃႈ Conflict ၼႆႉမႄးယဝ်ႉယဝ်ႉ)
2.  `git commit -m "Fix merge conflict in main.dart"` (တႃႇပိၵ်ႉၵၢၼ် Merge)

#### **သင်ႁဝ်းမႄးၽိတ်းသေ ၶႂ်ႈတႄႇမႂ်ႇ (Abort)?**
သင် Code ႁဝ်း ယုင်ႈယၢင်ႈ ႁႅင်းၼႃႇယဝ်ႉလႄႈ ၶႂ်ႈ "ယႃႉပႅတ်ႈ" ၵၢၼ် Merge ၼႆႉပႅတ်ႈ ၼင်ႇႁႂ်ႈမၼ်းၶိုၼ်းမိူၼ်ၵဝ်ႇ (ဢွၼ်တၢင်းပႆႇ Merge) ၼၼ်ႉ တႅမ်ႈ:
`git merge --abort`

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
**Resolving Manually** ၼႆႉ ပဵၼ်ၵၢၼ် "တႅပ်းတတ်းၸႂ်"။ Git တေဢမ်ႇလိူၵ်ႈပၼ်ႁဝ်း ၵွပ်ႈမၼ်းၵူဝ်မၼ်းလိူၵ်ႈၽိတ်း။ ၵၢၼ်မႄးႁင်းၶေႃ တေၸွႆႈႁႂ်ႈႁဝ်းမီးသုၼ်ႇၵုမ်းထိင်း Code ႁဝ်း 100% ၼၼ်ႉၶႃႈဢေႃႈ။
