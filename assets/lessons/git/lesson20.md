## **Module 3: Remote Connectivity**
### **Lesson 20: Creating a Remote Repository on GitHub (သၢင်ႈ Repo မႂ်ႇၼိူဝ် Cloud)**

ဢွၼ်တၢင်း မိူဝ်ႈပႆႇဢဝ် Code ၼႂ်းၶိူင်ႈႁဝ်း (Local) သူင်ႇၶိုၼ်ႈၵႂႃႇလႆႈၼၼ်ႉ ႁဝ်းတေလႆႈမီး Repository ဢၼ်ပဝ်ႇ (Empty) ဝႆႉၼိူဝ် GitHub ဢွၼ်တၢင်းၶႃႈ။

#### **1. Step-by-Step (လၢႆးသၢင်ႈ)**
1.  **Log in:** ၶဝ်ႈၵႂႃႇတီႈ [github.com](https://github.com/)။
2.  **New Repo:** ၼဵၵ်းမၢႆပၵ်း **`+`** တီႈၸဵင်ႇၶူၼ်ႉသုတ်းၽၢႆႇၶႂႃ ယဝ်ႉလိူၵ်ႈဢဝ် **"New repository"**။
3.  **Repository Name:** တင်ႈၸိုဝ်ႈ Project ႁဝ်း (မိူၼ်ၼင်ႇ `shan-proverbs-app` ဢမ်ႇၼၼ် `tmk-typing-tutor`)။
4.  **Public vs Private:**
    * **Public:** ၵူၼ်းတင်းလူၵ်ႈ တေႁၼ်လႆႈ (လီတႃႇ Project တႃႇၸွႆႈထႅမ်ၸိူဝ်ႉၶိူဝ်း)။
    * **Private:** တူဝ်ၸဝ်ႈၵဝ်ႇၵူၺ်း ႁၼ်လႆႈ (လီတႃႇၵၢၼ်လပ်ႉ ဢမ်ႇၼၼ် ၵၢၼ်ၵႃႉၶၢႆ)။
5.  **Initialize this repository:** တွၼ်ႈၼႆႉ **"ဢမ်ႇလူဝ်ႇ"** ၼဵၵ်းလိူၵ်ႈသင် (Don't check README, .gitignore, or license) ၵွပ်ႈဝႃႈႁဝ်းမီး Code ၼႂ်းၶိူင်ႈႁဝ်းယဝ်ႉယဝ်ႉ။
6.  **Create:** ၼဵၵ်းပဵၼ် **"Create repository"**။



#### **2. Quick Setup Page**
မိူဝ်ႈသၢင်ႈယဝ်ႉယဝ်ႉ GitHub တေၼႄၼႃႈလိၵ်ႈဢၼ်မီး Command လၢႆလၢႆဢၼ်။ 
ဢၼ်ႁဝ်းလူဝ်ႇတွင်းမၢႆဝႆႉသုတ်းသုတ်းပဵၼ် **Repository URL** ၶႃႈယဝ်ႉ။
* မၼ်းတေမိူၼ်ၼင်ႇ: `https://github.com/SaiMao/my-project.git`
* ၼႆႉယဝ်ႉပဵၼ် "တီႈယူႇ (Address)" ဢၼ်ႁဝ်းတေၸႂ်ႉ ၵွင်ႉၸူးၶိူင်ႈႁဝ်းၶႃႈဢေႃႈ။

#### **3. HTTPS vs SSH**
GitHub တေမီးပၼ်လၢႆးၵွင်ႉ 2 ယၢင်ႇ:
* **HTTPS:** ငၢႆႈသုတ်း (ၸႂ်ႉ Username/Password ဢမ်ႇၼၼ် Token)။
* **SSH:** ၵတ်းယဵၼ်သုတ်း (ၸႂ်ႉ Key လပ်ႉ ဢမ်ႇလူဝ်ႇတႅမ်ႈ Password ၵူႈပွၵ်ႈ)။
  *(တွၼ်ႈတႃႇလုၵ်ႈႁဵၼ်းမႂ်ႇ ထုၵ်ႇလီတႄႇၸႂ်ႉ HTTPS ဢွၼ်တၢင်းၶႃႈ)*

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
ၵၢၼ်သၢင်ႈ **Remote Repository** ၼႆႉ မိူၼ်ၼင်ႇႁဝ်း "သိုဝ်ႉလိၼ်" ဝႆႉၼိူဝ် Cloud။ မၼ်းပဵၼ် Repo ဢၼ်ပဝ်ႇဝႆႉယူႇ ဢၼ်တိုၵ်ႉမုင်ႈမွင်းပႂ်ႉႁပ်ႉ Code တီႈ Computer ႁဝ်း ႁႂ်ႈသူင်ႇၶိုၼ်ႈၵႂႃႇၼၼ်ႉၶႃႈဢေႃႈ။
