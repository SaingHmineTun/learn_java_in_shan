## **Module 6: Android Studio Integration**
### **Lesson 46: Connecting GitHub Account (ၵၢၼ်ၵွင်ႉ Account GitHub ၸူး Android Studio)**

ဢွၼ်တၢင်းသုတ်း ႁဝ်းတေလႆႈႁႂ်ႈ Android Studio ႁူႉၸၵ်းတင်း GitHub Account ႁဝ်းဢွၼ်တၢင်း ၼင်ႇႁႂ်ႈမၼ်း Push ဢမ်ႇၼၼ် Pull Code လႆႈလူၺ်ႈဢမ်ႇလူဝ်ႇသႂ်ႇ Password ၵူႈပွၵ်ႈၶႃႈ။

#### **1. လၢႆးၵွင်ႉ (Step-by-Step Settings)**
1.  ပိုတ်ႇ Android Studio သေ ၵႂႃႇတီႈ **Settings** (တႃႇ Windows: `File > Settings` / တႃႇ Mac: `Android Studio > Settings`)။
2.  ၵႂႃႇတီႈ **Version Control > GitHub**။
3.  ၼဵၵ်းတီႈမၢႆလေႃး **(+)** သေလိူၵ်ႈ **"Log In via GitHub..."**။
4.  မၼ်းတေပိုတ်ႇ Browser ႁႂ်ႈႁဝ်းၼဵၵ်း **"Authorize"** ပၼ်ၶႂၢင်ႉ။
5.  မိူဝ်ႈယဝ်ႉယဝ်ႉ ႁဝ်းတေႁၼ် ၶႅပ်းႁၢင်ႈ (Profile Picture) လႄႈ ၸိုဝ်ႈ Account GitHub ႁဝ်း ၼႄမႃးၼႂ်း Android Studio ၶႃႈယဝ်ႉ။



#### **2. တၢင်းလီၶွင်ၵၢၼ်ၵွင်ႉဝႆႉ**
* **Share Project:** ႁဝ်းၸၢင်ႈဢဝ် Project မႂ်ႇ ၶိုၼ်ႈ GitHub လႆႈၵမ်းလဵဝ်လူၺ်ႈ `VCS > Share Project on GitHub` (ဢမ်ႇလူဝ်ႇသၢင်ႈ Repo တီႈ Web ဢွၼ်တၢင်း)။
* **Clone Easily:** ၸၢင်ႈ Clone Project တီႈ GitHub ႁဝ်းမႃးလႆႈငၢႆႈငၢႆႈ။
* **PR Support:** ၸၢင်ႈဢၢၼ်ႇ လႄႈ မႄး **Pull Request** လႆႈၼႂ်း Android Studio ၵမ်းလဵဝ် (Lesson 54)။

#### **3. Token (သင်ဝႃႈ Browser Login ဢမ်ႇလႆႈ)**
သင်ဝႃႈလၢႆး Log In ယူဝ်းယူဝ်းဢမ်ႇလႆႈ၊ ႁဝ်းၸၢင်ႈၸႂ်ႉ **Personal Access Token (PAT)** တီႈ GitHub Settings မႃးသႂ်ႇလႆႈၶႃႈ။ (ၵူၺ်းၵႃႈ လၢႆး Browser ၼၼ်ႉ ငၢႆႈသေပိူၼ်ႈၶႃႈယဝ်ႉ)။

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
ၵၢၼ်ၵွင်ႉ **GitHub Account** ၸူး **Android Studio** ပဵၼ်ၵၢၼ် "ပိုတ်ႇၽၵ်းတူ" တႃႇႁႂ်ႈ IDE ႁဝ်း ႁဵတ်းၵၢၼ်ၸွမ်း Git လႆႈမူတ်းမူတ်း။ မၼ်းတေႁဵတ်းႁႂ်ႈႁဝ်း ဢမ်ႇလူဝ်ႇၵႂႃႇယုင်ႈယၢင်ႈၼိူဝ် Website GitHub ၼမ်ၼႅၼ်ႇၼၼ်ႉၶႃႈဢေႃႈ။
