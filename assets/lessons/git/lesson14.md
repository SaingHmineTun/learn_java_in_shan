## **Module 2: Branching & The Safety Net**
### **Lesson 14: Understanding the 'Safety Net' (ၵၢၼ်ၸႂ်ႉ git stash)**

မိူဝ်ႈႁဝ်းတႅမ်ႈ Code ၼႂ်း Branch ဢၼ်ၼိုင်ႈယူႇ (မိူၼ်ၼင်ႇ `feature-ui`) သေ ႁဝ်းတိုၵ်ႉမႄးဝႆႉယုင်ႈယၢင်ႈ၊ ဢမ်ႇပႆႇယဝ်ႉတူဝ်ႈ တႃႇတေ Commit လႆႈ။ 
ၵမ်းၼၼ်ႉ မီးၵၢၼ်ဝႆး (Emergency) တီႈ `main` ဢၼ်ႁဝ်းတေလႆႈၵႂႃႇမႄးၵမ်းလဵဝ်။

#### **1. ပၼ်ႁႃ (The Problem)**
* သင်ႁဝ်းတိုၵ်ႉမီး Code ဢၼ်မႄးဝႆႉ (Modified) သေ ႁဝ်း `git checkout main` ၼႆ Git တေႁၢမ်ႈႁဝ်း ၵွပ်ႈမၼ်းၵူဝ် Code ဢၼ်ႁဝ်းတိုၵ်ႉမႄးယူႇၼၼ်ႉႁၢႆၵႂႃႇ။
* ႁဝ်းၵေႃႈ ဢမ်ႇၶႂ်ႈ "Commit" ၵွပ်ႈ Code ႁဝ်းတိုၵ်ႉလူႉဝႆႉယူႇ၊ ဢမ်ႇပႆႇယဝ်ႉ။

#### **2. git stash ၼႆႉပဵၼ်သင်?**
* **Stash** ၼႆႉ ပွင်ႇဝႃႈ "သိမ်းဝႆႉတီႈလပ်ႉ"။
* မၼ်းတေဢဝ် Code ဢၼ်ႁဝ်းတိုၵ်ႉမႄးဝႆႉယူႇၼၼ်ႉ "ၵဵပ်း" သႂ်ႇဝႆႉၼႂ်းႁွင်ႈလပ်ႉ ႁႂ်ႈ Folder ႁဝ်းၵိုတ်းပဵၼ် Code ဢၼ်သႅၼ်ႈသႂ် (Clean) မိူၼ်ၵဝ်ႇ။
* မိူဝ်ႈႁဝ်းမႄးၵၢၼ်ဝႆးယဝ်ႉယဝ်ႉသေ ၶိုၼ်းမႃးၸူး Branch ၵဝ်ႇၼၼ်ႉ ႁဝ်းၸၢင်ႈ "ဢဝ်ဢွၵ်ႇ" (Pop) ၼႂ်းႁွင်ႈလပ်ႉၼၼ်ႉသေ မႃးသိုပ်ႇတႅမ်ႈလႆႈၶႃႈယဝ်ႉ။



#### **3. လၢႆးၸႂ်ႉ (Usage)**
* **လၢႆးၵဵပ်းသိမ်း:**
  `git stash`
  *(ယဝ်ႉယဝ်ႉၵေႃႈ တႅမ်ႈ `git status` တူၺ်း၊ ႁဝ်းတေႁၼ်ဝႃႈ Folder ႁဝ်းသႅၼ်ႈသႂ် ဢမ်ႇမီးသင်မႄးဝႆႉယဝ်ႉ)*

* **လၢႆးဢဝ်ၶိုၼ်းမႃး (ဢဝ်ဢွၵ်ႇ):**
  `git stash pop`
  *(မၼ်းတေဢဝ် Code ဢၼ်ႁဝ်းၵဵပ်းဝႆႉၼၼ်ႉ ၶိုၼ်းမႃးသႂ်ႇၼႂ်း File ႁဝ်း ၼင်ႇႁႂ်ႈႁဝ်းတႅမ်ႈသိုပ်ႇလႆႈၶႃႈ)*

#### **4. မၢႆတွင်း (Note)**
* `git stash` ၼႆႉ မၼ်းတေၵဵပ်းပၼ် File ဢၼ် Git "ႁူႉၸိုဝ်ႈ" ၵူၺ်း (Modified)။ သင်ပဵၼ် File ဢၼ်မႃးမႂ်ႇ (Untracked) ႁဝ်းတေလႆႈၸႂ်ႉ:
  `git stash -u`
  *(`-u` ပွင်ႇဝႃႈ **Untracked** ၶႃႈဢေႃႈ)*

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
**`git stash`** ၼႆႉ ပဵၼ်မိူၼ်ၼင်ႇ **"သႅၼ်ႈသႂ်ဝႆး"**။ မၼ်းၸွႆႈႁႂ်ႈႁဝ်း လႅၵ်ႈ Branch ၵႂႃႇမႃးလႆႈငၢႆႈငၢႆႈ လူၺ်ႈဢမ်ႇလူဝ်ႇ Commit Code ဢၼ်တိုၵ်ႉမႄး ဢမ်ႇပႆႇယဝ်ႉၼၼ်ႉၶႃႈဢေႃႈ။
