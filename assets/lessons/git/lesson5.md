## **Module 1: The Local Repository**
### **Lesson 5: git status (ၵူတ်ႇထတ်းငဝ်းလၢႆးၶွင် File)**

မိူဝ်ႈႁဝ်းႁဵတ်း `git init` ယဝ်ႉယဝ်ႉၼၼ်ႉ ႁဝ်းတေလႆႈႁူႉဝႃႈ Git တိုၵ်ႉတူၺ်း (Track) File ႁဝ်းယူႇႁႃႉ? 
ဢမ်ႇၼၼ် မီး File ဢၼ်လႂ်မႃးမႂ်ႇႁႃႉ? ၼႆၶႃႈ။

#### **1. git status ဢၼ်ဝႃႈၼၼ်ႉပဵၼ်သင်?**
* မၼ်းပဵၼ်ၵၢၼ်ထၢမ် Git ဝႃႈ **"တွၼ်ႈၼႆႉ ငဝ်းလၢႆး (Status) ပဵၼ်သင်ယူႇ?"**
* မၼ်းတေတႅမ်ႈၼႄဝႃႈ:
    1. ႁဝ်းမီးဝႆႉတီႈ Branch လႂ်? (မိူၼ်ၼင်ႇ `main`)
    2. မီး File ဢၼ်လႂ် "မႄးဝႆႉ" (Modified)?
    3. မီး File ဢၼ်လႂ် "မႃးမႂ်ႇ" (Untracked)?
    4. မီးသင်တႃႇတေ "Save" (Commit) ႁႃႉ?

#### **2. လၢႆးၸႂ်ႉ (Usage)**
တႅမ်ႈ Command ငၢႆႈငၢႆႈၼႂ်း Terminal:
`git status`

#### **3. ပွင်ႇၸႂ်လွင်ႈ "သီ (Colors)" ၼႂ်း Status**
မိူဝ်ႈႁဝ်းတႅမ်ႈ `git status` ယဝ်ႉ Git တေၼႄသီမႃးပၼ် (သင်ပဵၼ်ၼႂ်း Terminal ဢၼ်ၼႄပၼ်လႆႈ):
* **သီလႅင် (Untracked/Modified):** ပွင်ႇဝႃႈ Git ႁၼ်လွင်ႈလႅၵ်ႈလၢႆႈယဝ်ႉ ၵူၺ်းၵႃႈ ႁဝ်းတိုၵ်ႉဢမ်ႇပႆႇလႆႈ "ႁႅၼ်းဝႆႉ (Stage)" တႃႇတေ Save။
* **သီၶဵဝ် (Staged):** ပွင်ႇဝႃႈ File ၼၼ်ႉ ႁႅၼ်းဝႆႉ တႃႇတေ "Save" (Commit) ၼႂ်းပွၵ်ႈၼႆႉယဝ်ႉ။



#### **4. ၶေႃႈၵႂၢမ်းဢၼ်လီမၢႆတွင်း**
* **Untracked files:** File ဢၼ်မႃးမႂ်ႇသေ Git တိုၵ်ႉဢမ်ႇပႆႇႁူႉလွင်ႈမၼ်း။
* **Changes not staged for commit:** File ဢၼ်မႄးဝႆႉ ၵူၺ်းၵႃႈ တိုၵ်ႉဢမ်ႇပႆႇလႆႈထႅမ်သႂ်ႇၼႂ်း "သဵၼ်ႈမၢႆတေ Save"။
* **nothing to commit, working tree clean:** ပွင်ႇဝႃႈ ၵူႈလွင်ႈ Save ယဝ်ႉယဝ်ႉ၊ ဢမ်ႇမီးသင်လႅၵ်ႈလၢႆႈမႂ်ႇ။

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
Command **`git status`** ၼႆႉ မိူၼ်ၼင်ႇ **"တႃ (Eyes)"** ၶွင်ႁဝ်းၶႃႈယဝ်ႉ။ 
ဢွၼ်တၢင်းတေႁဵတ်းသင်ၵေႃႈယဝ်ႉ ထုၵ်ႇလီတႅမ်ႈ `git status` တူၺ်းၵူႈပွၵ်ႈ ၼင်ႇႁႂ်ႈႁဝ်းႁူႉဝႃႈ တွၼ်ႈၼႆႉ Code ႁဝ်းမီးငဝ်းလၢႆးပဵၼ်သင်ယူႇၼၼ်ႉၶႃႈဢေႃႈ။
