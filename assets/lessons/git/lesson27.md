## **Module 3: Remote Connectivity**
### **Lesson 27: Remote Troubleshooting (မႄးပၼ်ႁႃလွင်ႈပိုၼ်းဢမ်ႇၶဝ်ႈၵၼ်)**

လုၵ်ႈႁဵၼ်းတေ ၵႆႉၵႆႉထူပ်းပၼ်ႁႃ မိူဝ်ႈၶႂ်ႈသူင်ႇ (Push) Code ၶိုၼ်ႈၵႂႃႇ ၵူၺ်းၵႃႈ GitHub သမ်ႉတွပ်ႇမႃးဝႃႈ **"Rejected"** (ဢမ်ႇႁပ်ႉ)။ 
ၼႆႉပွင်ႇဝႃႈ ပိုၼ်း Code ၼိူဝ် Cloud မၼ်းမႂ်ႇသေ ဢမ်ႇၼၼ် မၼ်းပႅၵ်ႇၵၼ်တင်း ၼႂ်းၶိူင်ႈႁဝ်းယဝ်ႉၶႃႈ။

#### **1. ပၼ်ႁႃ "Updates were rejected"**
သင်ႁဝ်း ႁၼ်လိၵ်ႈတွပ်ႇၼင်ႇၼႆ:
`error: failed to push some refs to 'https://github.com/...'`
`hint: Updates were rejected because the remote contains work that you do`
`hint: not have locally. This is usually caused by another repository pushing`
* **လွင်ႈတၢင်း:** မီးၵူၼ်းတၢင်ႇၵေႃႉ (ဢမ်ႇၼၼ် တူဝ်ႁဝ်းၼင်ႇၵဝ်ႇ တႅမ်ႈတီႈၶိူင်ႈတၢင်ႇဢၼ်) သူင်ႇ Code ၶိုၼ်ႈၵႂႃႇဝႆႉယဝ်ႉ။
* **လၢႆးမႄး:** တေလႆႈ **Pull** ဢဝ်လွင်ႈလႅၵ်ႈလၢႆႈၼၼ်ႉ မႃးဝႆႉၼႂ်းၶိူင်ႈႁဝ်းဢွၼ်တၢင်း:
  `git pull origin main`
  *(ယဝ်ႉယဝ်ႉၵေႃႈ ၸင်ႇၶိုၼ်း `push` မႂ်ႇထႅင်ႈပွၵ်ႈၼိုင်ႈၶႃႈ)*



#### **2. ပၼ်ႁႃ "fatal: refusing to merge unrelated histories"**
ၼႆႉၵႆႉပဵၼ် မိူဝ်ႈႁဝ်းသၢင်ႈ Repo ၼႂ်းၶိူင်ႈပွၵ်ႈၼိုင်ႈ၊ ၼိူဝ် GitHub သမ်ႉထႅင်ႈပွၵ်ႈၼိုင်ႈ သေ မႃးၵွင်ႉၸူးၵၼ်ၼႆ Git တေထၢမ်ဝႃႈ "သွင်ဢၼ်ၼႆႉ မၼ်းဢမ်ႇၸႂ်ႈၸိူဝ်ႉလဵဝ်ၵၼ်၊ ပိုၼ်းမၼ်းဢမ်ႇၵွင်ႉၵၼ်"။
* **လၢႆးမႄး:** ႁဝ်းတေလႆႈ "ၸိ" ပၼ် Git ဝႃႈ ႁႂ်ႈမၼ်းႁူမ်ႈၵၼ်လႆႈယူႇ:
  `git pull origin main --allow-unrelated-histories`

#### **3. ၵၢၼ်ၸႂ်ႉ Force Push (`--force`)!**
သင်ဝႃႈႁဝ်းၶႂ်ႈ "ယႃႉပႅတ်ႈ" Code ၼိူဝ် GitHub ႁႂ်ႈမၼ်းမိူၼ်ၼႂ်းၶိူင်ႈႁဝ်း (လူၺ်ႈဢမ်ႇတူၺ်းဝႃႈ ၼိူဝ် GitHub မီးသင်ဝႆႉ) ၼၼ်ႉ ႁဝ်းၸၢင်ႈၸႂ်ႉ:
`git push origin main --force`
* **⚠️ ႁၢမ်ႈ:** ႁၢမ်ႈၸႂ်ႉ Command ၼႆႉ သင်ဝႃႈႁဝ်းႁဵတ်းၵၢၼ်ၸွမ်းၵူၼ်းၼမ် ၵွပ်ႈမၼ်းတေယႃႉပႅတ်ႈ Code ၶွင်ပိူၼ်ႈမူတ်းၶႃႈ။ ထုၵ်ႇလီၸႂ်ႉ မိူဝ်ႈႁဝ်းႁဵတ်းၵၢၼ် ၵေႃႉလဵဝ်ၵူၺ်းၶႃႈၼႃ။

#### **4. ၵူတ်ႇထတ်း Remote URL**
သင်ဝႃႈ Push ဢမ်ႇလႆႈသေ မၼ်းဝႃႈ "Remote not found" ၼႆ ထုၵ်ႇလီၵူတ်ႇထတ်း Address မၼ်း:
`git remote -v`
*(သင် Address ၽိတ်း ၸႂ်ႉ `git remote set-url origin <url-မႂ်ႇ>` ၶႃႈ)*

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
ၵၢၼ်မႄးပၼ်ႁႃ **Remote** ၼႆႉ ႁူဝ်ၸႂ်မၼ်းမီးယူႇတီႈ **"ၵူတ်ႇထတ်းဢွၼ်တၢင်း၊ ၸင်ႇမႄး"**။ ၵၢၼ်ၸႂ်ႉ `git pull` တႃႇႁူမ်ႈပိုၼ်း ၵႆႉပဵၼ်လၢႆးမႄးဢၼ်လီသုတ်း လႄႈ ၵတ်းယဵၼ်သုတ်း တႃႇ Project ႁဝ်းၶႃႈဢေႃႈ။
