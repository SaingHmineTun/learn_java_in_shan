## **Module 6: Android Studio Integration**
### **Lesson 49: Visual Branching and Switching (ၵၢၼ်လႅၵ်ႈလၢႆႈ Branch လူၺ်ႈမိုဝ်းၼဵၵ်း)**

ႁဝ်းၸၢင်ႈတူၺ်း လႄႈ ၵုမ်းထိင်း Branch တင်ႈမူတ်းလႆႈ တီႈ **Bottom-Right Corner** ၶွင် Android Studio (တီႈဢၼ်တႅမ်ႈဝႆႉဝႃႈ `Git: main` ဢမ်ႇၼၼ် ၸိုဝ်ႈ Branch ဢၼ်ႁဝ်းယူႇဝႆႉၼၼ်ႉၶႃႈ)။

#### **1. လၢႆးပိုတ်ႇ Branch Menu**
ၼဵၵ်းတီႈၸိုဝ်ႈ Branch ဢၼ်ယူႇၸဵင်ႇၶႂႃၽၢႆႇတႂ်ႈၼၼ်ႉ၊ မၼ်းတေပိုတ်ႇ Popup Window ဢၼ်ၼႄပၼ်:
* **Local Branches:** Branch ၸိူဝ်းဢၼ်မီးၼႂ်း Computer ႁဝ်း။
* **Remote Branches:** Branch ၸိူဝ်းဢၼ်မီးၼိူဝ် GitHub (Origin/Upstream)။



#### **2. လၢႆးလႅၵ်ႈ Branch (Switching / Checkout)**
သင်ႁဝ်း ၶႂ်ႈလုၵ်ႉတီႈ `main` သေ ၵႂႃႇတႅမ်ႈ Code ၼႂ်း `feature-shan-font`:
1.  ၼဵၵ်းပိုတ်ႇ Branch Menu။
2.  လိူၵ်ႈ Branch ဢၼ်ၶႂ်ႈၵႂႃႇ။
3.  ၼဵၵ်းလိူၵ်ႈ **"Checkout"**။
* **Magic:** Android Studio တေလႅၵ်ႈ File Code ၵူႈယၢင်ႇၼႂ်း Project ႁဝ်း ႁႂ်ႈပဵၼ်ၶွင် Branch ၼၼ်ႉပၼ်ၵမ်းလဵဝ်ၶႃႈ။

#### **3. သၢင်ႈ Branch မႂ်ႇ (New Branch)**
ၼဵၵ်းတီႈ **"New Branch"** ဢၼ်ယူႇၽၢႆႇၼိူဝ်သုတ်းၶွင် Menu ၼၼ်ႉ:
* တႅမ်ႈၸိုဝ်ႈ Branch မႂ်ႇ။
* ၼဵၵ်းလိူၵ်ႈ **"Create"** (သင်ၶႂ်ႈလႅၵ်ႈၵႂႃႇၵမ်းလဵဝ် မၢႆပၼ် Checkbox "Checkout branch" ဝႆႉၶႃႈၼႃ)။

#### **4. လၢႆး Merge လႄႈ လၢႆး Delete**
ႁဝ်းၸၢင်ႈၼဵၵ်းတီႈ Branch ဢၼ်လႂ်ၵေႃႈယဝ်ႉ သေ ႁဵတ်းၵၢၼ်ၸိူဝ်းၼႆႉ:
* **Merge into Current:** ဢဝ် Code လုၵ်ႉတီႈ Branch ၼၼ်ႉ မႃးႁူမ်ႈသႂ်ႇၼႂ်း Branch ဢၼ်ႁဝ်းယူႇဝႆႉ (မိူၼ်ၼင်ႇ `git merge`)။
* **Rename:** လႅၵ်ႈလၢႆႈၸိုဝ်ႈ Branch။
* **Delete:** ယႃႉပႅတ်ႈ Branch ဢၼ်ဢမ်ႇၸႂ်ႉယဝ်ႉ (သင်ဝႃႈမီး Code ဢၼ်ဢမ်ႇပႆႇ Merge၊ Android Studio တေတႅမ်ႈလိၵ်ႈၵိုတ်း (Warn) ႁဝ်းဢွၼ်တၢင်းၶႃႈ)။

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
**Bottom-Right Menu** ၼႆႉ ပဵၼ် "ၵႅမ်းၵုမ်းထိင်း (Control Center)" တႃႇ Branch ၶွင်ႁဝ်း။ မၼ်းၸွႆႈႁႂ်ႈႁဝ်း လႅၵ်ႈလၢႆႈၵၢၼ်တႅမ်ႈ Feature လၢႆလၢႆယၢင်ႇလႆႈဝႆးဝႆး လူၺ်ႈဢမ်ႇလူဝ်ႇတူၵ်းၸႂ်ၵူဝ်တႅမ်ႈ Command ၽိတ်းၶႃႈဢေႃႈ။
