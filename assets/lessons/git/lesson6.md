## **Module 1: The Local Repository**
### **Lesson 6: git add (ႁႅၼ်းဝႆႉတႃႇတေ Save)**

မိူဝ်ႈႁဝ်းတႅမ်ႈ `git status` ယဝ်ႉ သင်ႁၼ် File ႁဝ်းပဵၼ် "သီလႅင်" ယူႇၼၼ်ႉ ပွင်ႇဝႃႈ Git တိုၵ်ႉႁၼ်မၼ်း ပဵၼ်လွင်ႈလႅၵ်ႈလၢႆႈၵူၺ်း၊ 
မၼ်းတိုၵ်ႉဢမ်ႇပႆႇႁူႉဝႃႈ "ႁဝ်းတေဢဝ် File ၼႆႉ Save ၸွမ်းႁႃႉ?" ၼႆၶႃႈ။

#### **1. Staging Area ဢၼ်ဝႃႈၼၼ်ႉပဵၼ်သင်?**
* မၼ်းမိူၼ်ၼင်ႇ မိူဝ်ႈႁဝ်းႁၢင်ႈႁႅၼ်းၶူဝ်း မိူဝ်ႈပႆႇဢဝ်သႂ်ႇလေႃႉ တႃႇသူင်ႇဢွၵ်ႇၵႂႃႇယဝ်ႉ။
* ႁဝ်းၸၢင်ႈလိူၵ်ႈလႆႈဝႃႈ တေဢဝ် File လႂ်ၵႂႃႇ ဢမ်ႇၼၼ် တေဢဝ်ၵႂႃႇ တင်းမူတ်းႁ။
* မိူဝ်ႈႁဝ်းၸႂ်ႉ `git add` ယဝ်ႉ File ႁဝ်းတေလႅၵ်ႈလၢႆႈပဵၼ် **"သီၶဵဝ်"** (Staged) ၼႂ်း `git status` ၶႃႈယဝ်ႉ။



#### **2. လၢႆးၸႂ်ႉ (Usage)**
ႁဝ်းၸၢင်ႈၸႂ်ႉလႆႈ လၢႆလၢႆးၶႃႈ:
* **လိူၵ်ႈဢဝ် File လဵဝ်:**
  `git add main.dart`
* **လိူၵ်ႈဢဝ်လၢႆလၢႆ File ၸွမ်းၵၼ်:**
  `git add file1.dart file2.dart`
* **လိူၵ်ႈဢဝ်တင်းမူတ်း (Shortcut):**
  `git add .`
  *(မၢႆၸမ်ႈ `.` ၼႆႉ ပွင်ႇဝႃႈ ဢဝ်ၵူႈလွင်ႈ ဢၼ်မီးလွင်ႈလႅၵ်ႈလၢႆႈ ၼႂ်း Folder ၼႆႉ တင်ႈမူတ်းၶႃႈဢေႃႈ)*

#### **3. မိူဝ်ႈလႂ်ႁဝ်းတေၸႂ်ႉ `git add`?**
* သင်ႁဝ်းတႅမ်ႈ Code ယဝ်ႉသေ ၶႂ်ႈႁႂ်ႈ Git "မၢႆတွင်း" ဝႆႉၼၼ်ႉ ႁဝ်းတေလႆႈ `add` မၼ်းဢွၼ်တၢင်းၶႃႈ။

#### **4. လၢႆးဢဝ်ဢွၵ်ႇ (Unstage)**
သင်ဝႃႈႁဝ်း `add` ၽိတ်းၵႂႃႇသေ ၶႂ်းၶိုၼ်းဢဝ်မၼ်းဢွၵ်ႇၼႂ်း Staging Area ၼႆ ၸႂ်ႉ Command:
`git restore --staged <file_name>`
`git rm --cached <file_name>`

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
Command **`git add`** ၼႆႉ ပဵၼ်ၵၢၼ် **"ႁႅၼ်းၶူဝ်း"**။ မၼ်းပဵၼ်ၶၢပ်ႈၵၢင် (Middle step) ဢၼ်လႅၵ်ႈလၢႆႈ Code ႁဝ်း ႁႂ်ႈပဵၼ်သဵၼ်ႈမၢႆ ဢၼ်ႁႅၼ်းဝႆႉတႃႇတေ "Save" (Commit) ၼၼ်ႉၶႃႈဢေႃႈ။
