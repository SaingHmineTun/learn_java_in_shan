## **Module 5: Troubleshooting & Data Integrity**
### **Lesson 43: Recovering the 'Lost' (ၵၢၼ်ဢဝ်ၶိုၼ်း Code ဢၼ်ႁၢႆလူၺ်ႈ git reflog)**

ပၼ်ႁႃဢၼ်လီၵူဝ်သုတ်း တႃႇ Developer ပဵၼ်:
* ၸႂ်ႉ `git reset --hard` ၽိတ်းသေ Commit မႂ်ႇမႂ်ႇႁၢႆၵႂႃႇမူတ်း။
* "Delete" ပႅတ်ႈ Branch ဢၼ်တိုၵ်ႉတႅမ်ႈ Code ဝႆႉၼၼ်ႉ။
* မႄး `--amend` ၽိတ်းသေ ၶႂ်ႈလႆႈ Commit ဢၼ်ၵဝ်ႇ ဢၼ်ႁဝ်းယႃႉပႅတ်ႈၼၼ်ႉၶိုၼ်း။

**`git reflog`** ၼႆႉ ပဵၼ်တူဝ်ၸွႆႈ ၵွပ်ႈမၼ်းမၢႆတွင်းဝႆႉဝႃႈ **"HEAD (သဵၼ်ႈၸီႉ)"** ၶွင်ႁဝ်း ယၢမ်ႈၵႂႃႇၵိုတ်းတီႈလႂ်မႃးလၢႆလၢႆ ၼႂ်း 30-90 ဝၼ်းပူၼ်ႉမႃးၶႃႈ။

#### **1. လၢႆးတူၺ်းပိုၼ်းလပ်ႉ (Viewing the Reflog)**
တႅမ်ႈ Command:
`git reflog`

ႁဝ်းတေႁၼ် သဵၼ်ႈမၢႆဢၼ်မီး `HEAD@{0}`, `HEAD@{1}` လႄႈ Commit ID (SHA) ၸိူဝ်းၼၼ်ႉၶႃႈ။
* **`HEAD@{0}`**: ပဵၼ်တီႈဢၼ်ႁဝ်းယူႇဝႆႉ "မိူဝ်ႈလဵဝ်"။
* **`HEAD@{1}`**: ပဵၼ်တီႈဢၼ်ႁဝ်းယူႇဝႆႉ "ဢွၼ်တၢင်းၼႆႉ 1 ၸၼ်ႉ"။



#### **2. လၢႆးဢဝ်ၶိုၼ်း (The Recovery)**
သင်ဝႃႈႁဝ်း Reset ၽိတ်းသေ ၶႂ်ႈၶိုၼ်းၵႂႃႇတီႈ Commit ဢၼ်ႁၢႆၵႂႃႇၼၼ်ႉ:
1.  ႁႃတူၺ်းၼႂ်း `reflog` ဝႃႈ Commit ဢၼ်ႁဝ်းၶႂ်ႈလႆႈၼၼ်ႉ မီး ID (SHA) သင်? (မိူၼ်ၼင်ႇ `a1b2c3d`)။
2.  ယဝ်ႉၵေႃႈ တႅမ်ႈ Command တႃႇ "ၶိုၼ်းၵႂႃႇ" တီႈၼၼ်ႉ:
    `git reset --hard a1b2c3d`

* **Magic:** Code ဢၼ်ႁဝ်း ဝူၼ်ႉဝႃႈႁၢႆၵႂႃႇၼၼ်ႉ တေၶိုၼ်းမႃးၵမ်းလဵဝ်ၶႃႈ!

#### **3. ဢဝ် Branch ဢၼ် Delete ၵႂႃႇၼၼ်ႉ ၶိုၼ်းမႃး**
သင်ဝႃႈႁဝ်း Delete Branch ၽိတ်း:
1.  ႁႃတူၺ်း Commit ID ၶွင် Branch ၼၼ်ႉၼႂ်း `reflog`။
2.  သၢင်ႈ Branch မႂ်ႇ လုၵ်ႉတီႈ ID ၼၼ်ႉ:
    `git branch branch-name a1b2c3d`

#### **မၢႆတွင်း (Important Notes)**
* **Local Only:** `reflog` မီးၼႂ်း Computer ႁဝ်းၵူၺ်းၶႃႈ။ မၼ်းဢမ်ႇသူင်ႇၶိုၼ်ႈ GitHub ၵွပ်ႈၼၼ် ႁဝ်းဢမ်ႇၸၢင်ႈတူၺ်း `reflog` ပိူၼ်ႈလႆႈၶႃႈ။
* **Not Permanent:** Git တေ "ၽဵဝ်ႈလၢင်ႉ (Clean up)" `reflog` ၵူႈ 30-90 ဝၼ်းၶႃႈ။

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
**`git reflog`** ပဵၼ် "သၢႆၸႂ်" တွၼ်ႈသွၼ်ၼႆႉ တေႁဵတ်းႁႂ်ႈလုၵ်ႈႁဵၼ်းၶဝ် ဢမ်ႇလူဝ်ႇႁွင်ႉႁႆႈ မိူဝ်ႈႁဵတ်း Code ႁၢႆ။ မၼ်းပဵၼ်တူဝ်ၸွႆႈ ဢၼ်ႁဵတ်းႁႂ်ႈ Git ပဵၼ် Tool ဢၼ်ၵတ်းယဵၼ်သုတ်း တႃႇသၢင်ႈ Project လူင်ၼၼ်ႉၶႃႈဢေႃႈ။
