## **Module 2: Branching & The Safety Net**
### **Lesson 16: Advanced Branching (Three-Way Merge)**

မိူဝ်ႈႁဝ်းႁဵၼ်းၵႂႃႇၼႂ်း Lesson 13 ၼၼ်ႉ ပဵၼ်လွင်ႈ Fast-Forward ဢၼ် `main` ဢမ်ႇပႆႇလႅၵ်ႈလၢႆႈသင်။ ၵူၺ်းၵႃႈ ၼႂ်းၵၢၼ်တႅမ်ႈ Code တႄႉတႄႉၼၼ်ႉ:
* ႁဝ်းၽႄ Branch ဢွၵ်ႇၵႂႃႇမႄး "Feature A"။
* ၼႂ်းၶၢဝ်းယၢမ်းၼၼ်ႉ ၵူၼ်းထႅင်ႈၵေႃႉၼိုင်ႈ (ဢမ်ႇၼၼ် တူဝ်ႁဝ်းၼၼ်ႉၼင်ႇၵဝ်ႇ) သမ်ႉမႃးမႄး Code တီႈ `main` ယဝ်ႉ Commit ဝႆႉ။
* ၵမ်းၼႆႉ သဵၼ်ႈတၢင်းသွင်သဵၼ်ႈၼၼ်ႉ မၼ်းပဵၼ်ဝႆႉ တၢင်းၽႂ်တၢင်းမၼ်းဝႆႉယဝ်ႉၶႃႈ။ 

#### **1. Three-Way Merge ဢၼ်ဝႃႈၼၼ်ႉပဵၼ်သင်?**
ၵွပ်ႈဝႃႈသဵၼ်ႈတၢင်းသွင်သဵၼ်ႈၼၼ်ႉ မၼ်းဢမ်ႇၸႂ်ႉသဵၼ်ႈတၢင်းလဵဝ်ၵၼ်ယဝ်ႉ Git ၸင်ႇတေလႆႈၸႂ်ႉ **3 တွၼ်ႈ (Snapshots)** တႃႇမႃးႁူမ်ႈၵၼ်:
1. **Common Ancestor:** တွၼ်ႈ Commit သုတ်းသုတ်း ဢၼ်သဵၼ်ႈတၢင်းသွင်သဵၼ်ႈၼၼ်ႉ တိုၵ်ႉယူႇၸွမ်းၵၼ် (The "Parent").
2. **Branch A (main):** တွၼ်ႈမႂ်ႇသုတ်း ၶွင် Branch လူင်။
3. **Branch B (feature):** တွၼ်ႈမႂ်ႇသုတ်း ၶွင် Branch ဢၼ်ႁဝ်းၽႄဢွၵ်ႇၵႂႃႇ။



#### **2. Merge Commit**
ဢၼ်ပႅၵ်ႇပိူင်ႈၵၼ်တင်း Fast-Forward ၼၼ်ႉပဵၼ်:
* Git တေၸွႆႈသၢင်ႈ **"Commit မႂ်ႇ"** ဢၼ်ၼိုင်ႈပဵၼ်ဢၼ်ၶိုၵ်ႉတွၼ်း ႁွင်ႉဝႃႈ **Merge Commit**။
* Commit ၼႆႉ မၼ်းတေမီး "ပေႃႈမႄႇ (Parents)" သွင်တူဝ် ၵွပ်ႈမၼ်းလုၵ်ႉတီႈ သဵၼ်ႈတၢင်းသွင်သဵၼ်ႈ မႃးႁူမ်ႈၵၼ်ၼၼ်ႉၶႃႈ။

#### **3. လၢႆးၸႂ်ႉ (Usage)**
Command မၼ်းတႄႉ မိူၼ်ၵဝ်ႇၶႃႈ:
1. `git checkout main`
2. `git merge feature-branch`

မိူဝ်ႈၼဵၵ်း Enter ယဝ်ႉ Git တေပိုတ်ႇ **Text Editor** (မိူၼ်ၼင်ႇ Vim ဢမ်ႇၼၼ် Notepad) မႃးပၼ် ႁႂ်ႈႁဝ်းတႅမ်ႈ Message တႃႇ Merge Commit ၼၼ်ႉ။ 
ၵမ်ႈၼမ်တေႉ ႁဝ်းတေႃႉတႄႉ Save ဢဝ်ၸွမ်း Message ဢၼ် Git တႅမ်ႈဝႆႉပၼ်ၼၼ်ႉၵေႃႈလႆႈယဝ်ႉၶႃႈ။

#### **4. ပၼ်ႁႃ ဢၼ်တေၸၢင်ႈထူပ်း (Merge Conflict)**
သင်ဝႃႈ Branch တင်းသွင်သဵၼ်ႈၼၼ်ႉ မႃးမႄး **"ထႅဝ်လိၵ်ႈလဵဝ်ၵၼ်"** ၼႂ်း File လဵဝ်ၵၼ်ၼႆ Git တေဢမ်ႇႁူႉဝႃႈ တေလိူၵ်ႈဢဝ်ဢၼ်လႂ်။ ၼႆႉႁဝ်းႁွင်ႉဝႃႈ **Merge Conflict** (ဢၼ်ႁဝ်းတေၵႂႃႇႁဵၼ်းၼႂ်း Module 5 ၶႃႈၼႃ)။ 
ၵူၺ်းၵႃႈ သင်မႄး File တၢင်ႇဢၼ် ဢမ်ႇၼၼ် ထႅဝ်လိၵ်ႈဢၼ်ပႅၵ်ႇၵၼ်ၼႆ Git တေၸွႆႈႁူမ်ႈပၼ်ႁင်းၵူၺ်း (Automatically) ၶႃႈယဝ်ႉ။

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
**Three-Way Merge** ၼႆႉ ပဵၼ်ၵၢၼ်ႁူမ်ႈ Code ဢၼ်မီး "ပိုၼ်း" ပႅၵ်ႇၵၼ်။ 
မၼ်းၸွႆႈႁႂ်ႈႁဝ်း ၸၢင်ႈႁူမ်ႈၵၢၼ်တႅမ်ႈ Code ဢၼ်ယုင်ႈယၢင်ႈလႆႈ လူၺ်ႈမၼ်းတေသၢင်ႈ Merge Commit ဝႆႉပဵၼ်လၵ်းထၢၼ်ဝႃႈ "တွၼ်ႈၼႆႉ သဵၼ်ႈတၢင်းသွင်သဵၼ်ႈ မႃးထူပ်းၵၼ်ယဝ်ႉယဝ်ႉ" ၼႆၶႃႈဢေႃႈ။
