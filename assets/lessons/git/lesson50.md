## **Module 6: Android Studio Integration**
### **Lesson 50: The 3-Way Merge (ၵၢၼ်မႄး Conflict လူၺ်ႈႁၢင်ႈၽၢင် 3 ၼႃႈၸေႃး)**

မိူဝ်ႈႁဝ်း `merge` ဢမ်ႇၼၼ် `pull` Code မႃးသေ မၼ်းပဵၼ် Conflict၊ Android Studio တေပိုတ်ႇ Popup တႅမ်ႈဝႃႈ **"Conflicts"**။ ႁႂ်ႈၼဵၵ်းလိူၵ်ႈ File ၼၼ်ႉ သေ ၼဵၵ်း **"Merge..."** ၶႃႈ။

#### **1. Anatomy ၶွင် 3-Way Merge Window**
မိူဝ်ႈပိုတ်ႇမႃးယဝ်ႉ ႁဝ်းတေႁၼ် ၼႃႈၸေႃး ၸႅၵ်ႇပဵၼ် 3 တွၼ်ႈၼင်ႇၼႆၶႃႈ:
1.  **Left (Your Version / HEAD):** Code ဢၼ်မီးၼႂ်းၶိူင်ႈႁဝ်း (Branch ႁဝ်း)။
2.  **Right (Their Version / Incoming):** Code ဢၼ်ႁဝ်းလူတ်ႇမႃး (Branch ပိူၼ်ႈ)။
3.  **Middle (Result):** ၼႆႉပဵၼ် "တီႈၵေႃႇသၢင်ႈ"။ Code ဢၼ်ႁဝ်းလိူၵ်ႈ တေမႃးမီးၼႂ်းၼႆႉ သေ မၼ်းတေပဵၼ် Code ဢၼ်တႄႉတႄႉ မိူဝ်ႈႁဝ်းမႄးယဝ်ႉၶႃႈ။



#### **2. လၢႆးလိူၵ်ႈ Code (The Magic Buttons)**
ႁဝ်းတေႁၼ်မၢႆၸမ်ႈ **`>>`** ဢမ်ႇၼၼ် **`<<`** (သီလိူင်/သီလႅင်) ယူႇဝႆႉ:
* **Accept Left (`>>`):** ၼဵၵ်းတႃႇဢဝ် Code ႁဝ်းသႂ်ႇၼႂ်းတွၼ်ႈၵႄႈၵၢင် (Middle)။
* **Accept Right (`<<`):** ၼဵၵ်းတႃႇဢဝ် Code ပိူၼ်ႈသႂ်ႇၼႂ်းတွၼ်ႈၵႄႈၵၢင် (Middle)။
* **Ignore (`X`):** ယႃႉပႅတ်ႈလွင်ႈလႅၵ်ႈလၢႆႈၼၼ်ႉ (ဢမ်ႇဢဝ်တင်းသွင်ဢၼ်)။

#### **3. Resolve All Non-Conflicting (မေႃၸွႆႈဝႆး)**
သင်ဝႃႈၼႂ်း File ၼၼ်ႉ မၼ်းမီးလၢႆတွၼ်ႈ၊ ႁဝ်းၸၢင်ႈၼဵၵ်း **"Apply All Non-Conflicting Changes"** (မၢႆၸမ်ႈ Magic Wand)။ 
Android Studio တေမႄးပၼ်ၵမ်းလဵဝ် တွၼ်ႈဢၼ် Git မၼ်ႈၸႂ်ဝႃႈ မႄးလႆႈ ႁင်းၵူၺ်း၊ သေ တေၵိုတ်းဝႆႉပၼ် တွၼ်ႈဢၼ် "ယုင်ႈယၢင်ႈတႄႉတႄႉ" ႁႂ်ႈႁဝ်းတႅပ်းတတ်းၸႂ်ၶႃႈ။

#### **4. ၵၢၼ်ပိၵ်ႉပၢင် (Finish)**
မိူဝ်ႈႁဝ်း လိူၵ်ႈ Code ၸိူဝ်းဢၼ်သီလႅင်/လိူင် ၼၼ်ႉယဝ်ႉတူဝ်ႈမူတ်းယဝ်ႉ:
1. တူၺ်းၼႂ်းတွၼ်ႈၵႄႈၵၢင် (Middle) ဝႃႈ Code မၼ်းထုၵ်ႇတူင်ႇယဝ်ႉႁႃႉ?
2. ၼဵၵ်း **"Apply"** ဢၼ်ယူႇၸဵင်ႇၶႂႃၽၢႆႇတႂ်ႈ။
* **Magic:** Android Studio တေႁဵတ်းၵၢၼ် `git add` လႄႈ `git commit` ပၼ်ႁင်းၵူၺ်း ႁႂ်ႈယဝ်ႉတူဝ်ႈၵမ်းလဵဝ်ၶႃႈ!

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
**3-Way Merge Tool** ၼႆႉ ပဵၼ် Tool ဢၼ်ႁဵတ်းႁႂ်ႈ Conflict ဢမ်ႇၼႃႈၵူဝ်ထႅင်ႈယဝ်ႉ။ 
မၼ်းၸွႆႈႁႂ်ႈႁဝ်း ႁၼ် Code တင်းသွင်ၽၢႆႇ လႄႈ တွၼ်ႈဢၼ်တေဢွၵ်ႇမႃး (Result) ၼၼ်ႉ ၸႅင်ႈၸႅင်ႈလႅင်းလႅင်း ၼႂ်းၼႃႈၸေႃးလဵဝ် ၶႃႈဢေႃႈ။
