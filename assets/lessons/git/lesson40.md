## **Module 5: Troubleshooting & Data Integrity**
### **Lesson 40: Moving the Timeline (ၵၢၼ်လႅၵ်ႈလၢႆႈပိုၼ်းလူၺ်ႈ git reset)**

**`git reset`** ၼႆႉ မၼ်းတေ "တိုတ်ႉ" ဢဝ်သၢႆ Timeline ၶွင် Branch ႁဝ်း 
ႁႂ်ႈၶိုၼ်းၵႂႃႇၵိုတ်းတီႈ Commit ဢၼ်ႁဝ်းလိူၵ်ႈဝႆႉၼၼ်ႉၶႃႈ။ 
မၼ်းမီး 2 လၢႆး ဢၼ်ပႅၵ်ႇၵၼ် ၼင်ႇတီႈတႂ်ႈၶႃႈ:

#### **1. git reset --soft (လၢႆးၵတ်းယဵၼ်)**
လၢႆးၼႆႉ မၼ်းတေယႃႉပႅတ်ႈ Commit ၼၼ်ႉပႅတ်ႈ ၵူၺ်းၵႃႈ **"Code ဢၼ်ႁဝ်းတႅမ်ႈဝႆႉၼၼ်ႉ တိုၵ်ႉၵိုတ်းဝႆႉယူႇ"**။
* **မိူဝ်ႈလႂ်တေၸႂ်ႉ:** မိူဝ်ႈႁဝ်း Commit ၽိတ်းသေ ၶႂ်ႈမႄး Code ဢိတ်းၼိုင်ႈ ယဝ်ႉၸင်ႇ Commit မႂ်ႇ။
* **လၢႆးၸႂ်ႉ:** `git reset --soft HEAD~1`
  *(HEAD~1 ပွင်ႇဝႃႈ ထွႆလင်ၵႂႃႇ 1 Commit ၶႃႈ)*
* **မၼ်းတေပဵၼ်သင်:** Commit ၼၼ်ႉႁၢႆၵႂႃႇ၊ ၵူၺ်းၵႃႈ Code တေပဵၼ် "Staged (သီၶဵဝ်)" ဝႆႉယူႇ ႁႅၼ်းဝႆႉတႃႇ Commit မႂ်ႇၶႃႈယဝ်ႉ။

#### **2. git reset --hard (လၢႆးၸႂ်ႁၢႆႉ / Dangerous)**
လၢႆးၼႆႉ မၼ်းတေယႃႉပႅတ်ႈ Commit လႄႈ **"ယႃႉပႅတ်ႈတင်း Code ဢၼ်ႁဝ်းတႅမ်ႈဝႆႉ"** ၼၼ်ႉပႅတ်ႈမူတ်းမူတ်း။
* **မိူဝ်ႈလႂ်တေၸႂ်ႉ:** မိူဝ်ႈႁဝ်းႁဵတ်း Code လူႉလႅဝ်သုတ်းသုတ်းသေ ဢမ်ႇၶႂ်ႈလႆႈသင်ယဝ်ႉ ၶႂ်ႈႁႂ်ႈမၼ်းၶိုၼ်းမူတ်းသႅၼ်ႈမိူၼ်ၵဝ်ႇ။
* **လၢႆးၸႂ်ႉ:** `git reset --hard HEAD~1`
* **⚠️ သတိ:** Code ဢၼ်တႅမ်ႈဝႆႉၼၼ်ႉ တေႁၢႆၵႂႃႇ "တႄႉတႄႉ" တေဢမ်ႇၸၢင်ႈဢဝ်ၶိုၼ်းလႆႈငၢႆႈငၢႆႈယဝ်ႉၶႃႈၼႃ။



#### **3. ၵၢၼ် Reset ၵႂႃႇၸူး Commit ID**
သင်ႁဝ်း ၶႂ်ႈထွႆလင်ၵႂႃႇၵႆၵႆ ႁဝ်းၸၢင်ႈဢဝ် Commit ID (SHA) မႃးသႂ်ႇလႆႈၶႃႈ:
`git reset --hard a1b2c3d`

#### **မၢႆတွင်းဢၼ်တေလႆႈၾၢင်ႉ (Caution)**
* **"ႁၢမ်ႈ Reset"** ဢၼ်ႁဝ်း **Push** ၶိုၼ်ႈ GitHub ၵႂႃႇယဝ်ႉ။ ၵွပ်ႈမၼ်းတေႁဵတ်းႁႂ်ႈ Timeline ၶွင်တၢင်ႇၵေႃႉ ယုင်ႈယၢင်ႈမူတ်း။
* ၸႂ်ႉ `reset` တႃႇ Code ဢၼ်မီးၼႂ်းၶိူင်ႈ (Local) ႁဝ်းၵူၺ်း ၸင်ႇတေၵတ်းယဵၼ်ၶႃႈ။

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
**`git reset`** ပဵၼ်ၵၢၼ် "ယႃႉ (Delete)" ပိုၼ်း။ သင်ၶႂ်ႈမႄးၵူၺ်းတေႉ ၸႂ်ႉ `soft` လႆႈယူႇ၊ 
သင်ၶႂ်ႈယႃႉပႅတ်ႈမူတ်းမူတ်း ၸင်ႇၸႂ်ႉ `hard` ၶႃႈဢေႃႈ။
