## **Module 5: Troubleshooting & Data Integrity**
### **Lesson 39: Undoing Local Changes with git restore (ၵၢၼ်ယႃႉပႅတ်ႈ Code ဢၼ်မႄးဝႆႉ လူၺ်ႈ git restore)**

သင်ႁဝ်းၸၢမ်းတႅမ်ႈ Code ၵႂႃႇသေ မၼ်းသမ်ႉလူႉပႅတ်ႈ ဢမ်ႇၼၼ် ႁဝ်းဢမ်ႇၶႂ်ႈလႆႈလွင်ႈလႅၵ်ႈလၢႆႈၼၼ်ႉယဝ်ႉၼႆ 
Git မီး Command ဢၼ်ၸိုဝ်ႈဝႃႈ **`git restore`** တႃႇၸွႆႈၶႃႈ။

#### **1. ယႃႉပႅတ်ႈ Code ဢၼ်တိုၵ်ႉတႅမ်ႈယူႇ (Working Directory)**
သင်ႁဝ်းမႄး File `main.dart` သေ သမ်ႉၶႂ်ႈႁႂ်ႈမၼ်းၶိုၼ်းမိူၼ်ၵဝ်ႇ (မိူၼ်ၼင်ႇ Commit လိုၼ်းသုတ်ႉ) ၼၼ်ႉ တႅမ်ႈ:
`git restore main.dart`

* **မၼ်းတေႁဵတ်းသင်:** လွင်ႈလႅၵ်ႈလၢႆႈဢၼ်ႁဝ်းတႅမ်ႈဝႆႉၼၼ်ႉ တေႁၢႆၵႂႃႇၵမ်းလဵဝ် သေ Code တေၶိုၼ်းပဵၼ် ဢၼ်ၵဝ်ႇ ဢၼ်ႁဝ်း Commit ဝႆႉၼၼ်ႉၶႃႈ။
* **⚠️ သတိ:** လၢႆးၼႆႉ ႁဝ်းတေဢမ်ႇၸၢင်ႈ "Undo" ၶိုၼ်းလႆႈယဝ်ႉၶႃႈ။ Code ဢၼ်ႁဝ်းယႃႉပႅတ်ႈၼၼ်ႉ တေႁၢႆၵႂႃႇတႄႉတႄႉယဝ်ႉ။



#### **2. ဢဝ် File ဢွၵ်ႇၼႂ်း "သႅင်ႇ (Staging Area)"**
သင်ႁဝ်းတႅမ်ႈ `git add .` သေ သမ်ႉဝူၼ်ႉၺႃးဝႃႈ "ပႆႇၶႂ်ႈ Commit တွၼ်ႈၼႆႉၼႃ" ၼႆ ႁဝ်းၸၢင်ႈဢဝ်မၼ်းဢွၵ်ႇၼႂ်း Staging ၶိုၼ်းလႆႈလူၺ်ႈ:
`git restore --staged main.dart`

* **မၼ်းတေႁဵတ်းသင်:** File ၼၼ်ႉ တေဢမ်ႇဢဝ် Code မၼ်းႁၢႆ၊ ၵူၺ်းၵႃႈ မၼ်းတေလုၵ်ႉတီႈ "သီၶဵဝ် (Staged)" ၶိုၼ်းပဵၼ် "သီလႅင် (Unstaged)" တႃႇႁႂ်ႈႁဝ်းမႄးထႅင်ႈ ဢွၼ်တၢင်း ပႆႇႁဵတ်း Commit ၼၼ်ႉၶႃႈ။

#### **3. ယႃႉပႅတ်ႈလွင်ႈလႅၵ်ႈလၢႆႈ တင်ႈမူတ်း Folder**
သင်ႁဝ်းမႄးဝႆႉၼမ်ၼႅၼ်ႇသေ ၶႂ်ႈႁႂ်ႈတင်း Project ၶိုၼ်းမိူၼ်ၵဝ်ႇမူတ်းမူတ်း:
`git restore .`

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
**`git restore`** ၼႆႉ ပဵၼ်တူဝ်ၸွႆႈ "ပၼ်ႁႅင်းၸႂ်" တႃႇ Developer ၶႃႈယဝ်ႉ။ 
မၼ်းႁဵတ်းႁႂ်ႈႁဝ်း ဢမ်ႇၵူဝ်ၵၢၼ် "ၸၢမ်း (Experiment)"။ 
သင်ၸၢမ်းယဝ်ႉမၼ်းလူႉ ႁဝ်းၵေႃႈတေႃႉတႄႉ `restore` မၼ်းၶိုၼ်းၵေႃႈယဝ်ႉယဝ်ႉၶႃႈ။
