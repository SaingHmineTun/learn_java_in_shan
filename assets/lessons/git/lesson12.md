## **Module 2: Branching & The Safety Net**
### **Lesson 12: Switching Contexts (လႅၵ်ႈၵႂႃႇမႃး ၼႂ်း Branch)**

မိူဝ်ႈႁဝ်းသၢင်ႈ Branch ဝႆႉယဝ်ႉၼၼ်ႉ တႃႇတေတႄႇတႅမ်ႈ Code ၼႂ်းၼၼ်ႉ ႁဝ်းတေလႆႈ "လႅၵ်ႈ" (Switch) ၵႂႃႇယူႇတီႈ Branch ၼၼ်ႉဢွၼ်တၢင်းၶႃႈ။

#### **1. git checkout ဢၼ်ဝႃႈၼၼ်ႉပဵၼ်သင်?**
* မၼ်းပဵၼ်ၵၢၼ်လႅၵ်ႈ **HEAD** (တူဝ်ၸိ) ႁႂ်ႈၵႂႃႇၸိဝႆႉတီႈ Branch ဢၼ်ႁဝ်းၶႂ်ႈယူႇ။
* မိူဝ်ႈႁဝ်း `checkout` ယဝ်ႉ File ၼႂ်း Folder Project ႁဝ်း တေလႅၵ်ႈလၢႆႈၵႂႃႇၸွမ်း Version ၶွင် Branch ၼၼ်ႉၵမ်းလဵဝ်။

#### **2. လၢႆးၸႂ်ႉ (Standard Commands)**
ၼႂ်း Git မၼ်းမီး Command ဢၼ်ၸႂ်ႉလႆႈမိူၼ်ၵၼ် 2 ဢၼ်ၶႃႈ:
* **လၢႆးၵဝ်ႇ:** `git checkout feature-fonts`
* **လၢႆးမႂ်ႇ (Dart 3.0 ပၢၼ်မႂ်ႇ):** `git switch feature-fonts`
  *(Git မႂ်ႇမႂ်ႇၼႆႉ မၼ်းၸႂ်ႉ `switch` ၼင်ႇႁႂ်ႈႁဝ်းဢမ်ႇယုင်ႈယၢင်ႈၸွမ်း `checkout` ဢၼ်မီးလၢႆၵၢၼ်ၼၼ်ႉၶႃႈဢေႃႈ)*



#### **3. လၢႆးဝႆး (The Shortcut)**
သင်ႁဝ်းၶႂ်ႈ "သၢင်ႈ" လႄႈ "လႅၵ်ႈ" ၵႂႃႇၵမ်းလဵဝ် လူၺ်ႈဢမ်ႇလူဝ်ႇတႅမ်ႈ 2 ပွၵ်ႈ:
* **Checkout:** `git checkout -b feature-login`
* **Switch:** `git switch -c feature-login`
  *(`-b` ဢမ်ႇၼၼ် `-c` ပွင်ႇဝႃႈ **Create** ၶႃႈဢေႃႈ)*

#### **4. မၢႆတွင်း ဢၼ်ၸၢင်ႈပဵၼ်ၽေး (Caution)**
ဢွၼ်တၢင်း ပႆႇလႅၵ်ႈ Branch ၵႂႃႇမႃးၼၼ်ႉ:
* ထုၵ်ႇလီ **Commit** ဢမ်ႇၼၼ် **Stash** (Lesson 14) Code ႁဝ်းဝႆႉဢွၼ်တၢင်း။
* သင်ႁဝ်းတိုၵ်ႉမီး Code ဢၼ်မႄးဝႆႉသေ ဢမ်ႇပႆႇလႆႈ Save ၼၼ်ႉ Git ဢမ်ႇၵႆႉပၼ်ႁဝ်းလႅၵ်ႈ Branch ၵွပ်ႈမၼ်းၵူဝ် Code ႁဝ်းႁၢႆၶႃႈဢေႃႈ။



#### **ႁူဝ်ႁုပ်ႈ (Summary)**
Command **`git checkout`** ဢမ်ႇၼၼ် **`git switch`** ၼႆႉ ပဵၼ်ၵၢၼ် **"လႅၵ်ႈၼႃႈၵၢၼ်"**။ 
မၼ်းၸွႆႈႁႂ်ႈႁဝ်း လႅၵ်ႈၵႂႃႇမႄး Feature ၼိုင်ႈဢၼ် ယဝ်ႉၼိုင်ႈဢၼ်လႆႈဝႆးဝႆး လူၺ်ႈဢမ်ႇႁဵတ်းႁႂ်ႈ Code ႁဝ်းယုင်ႈယၢင်ႈၵၼ်ၼၼ်ႉၶႃႈဢေႃႈ။
