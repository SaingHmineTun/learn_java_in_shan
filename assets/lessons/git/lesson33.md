## **Module 4: Distributed Collaboration**
### **Lesson 33: Code Review (ၵၢၼ်ၵူတ်ႇထတ်း လႄႈ ၵၢၼ်မႄးၸွမ်း)**

**Code Review** ၼႆႉ မၼ်းပဵၼ်ၵၢၼ် "ၸွႆႈၵၼ်ႁႂ်ႈ Code လီမႃး" ၶႃႈဢေႃႈ။

#### **1. Handling Feedback (လၢႆးႁပ်ႉဢဝ်ၶေႃႈၼႄၼမ်ႉ)**
မိူဝ်ႈၸဝ်ႈၶွင် Project ၶဝ်တႅမ်ႈ Comment သႂ်ႇၼႂ်း PR ႁဝ်းၼၼ်ႉ:
* **ပွင်ႇၸႂ်:** သင်ၶဝ်ဝႃႈ "တွၼ်ႈၼႆႉ လႅၵ်ႈၸိုဝ်ႈ Variable ႁႂ်ႈၸႅင်ႈလႅင်းထႅင်ႈဢိတ်းလႆႈႁႃႉ?" ၼႆ ႁဝ်းဢမ်ႇလူဝ်ႇတူၵ်းၸႂ်။
* **တွပ်ႇၶိုၼ်း:** ႁဝ်းၸၢင်ႈတႅမ်ႈ Comment တွပ်ႇၶဝ်ၼႂ်း GitHub လႆႈၵမ်းလဵဝ် (မိူၼ်ၼင်ႇ "လႆႈယူႇၶႃႈ၊ တေမႄးပၼ်ၵမ်းလဵဝ်ၶႃႈ")။



#### **2. Making Changes (လၢႆးမႄး Code ၼႂ်း PR ဢၼ်တိုၵ်ႉပိုတ်ႇဝႆႉ)**
"သင်တေမႄး Code ၼႆ တေလႆႈပိတ်း PR ၵဝ်ႇသေ သၢင်ႈ PR မႂ်ႇထႅင်ႈႁႃႉ?" **"ဢမ်ႇလူဝ်ႇၶႃႈ!"**
1.  မႄး Code ၼႂ်း Computer ႁဝ်း (ၼႂ်း Branch လဵဝ်ၵၼ်တင်း ဢၼ်ႁဝ်းသူင်ႇ PR ၼၼ်ႉ)။
2.  `git add` လႄႈ `git commit` မိူၼ်ၵဝ်ႇ။
3.  **`git push origin <branch-name>`** ၶိုၼ်းထႅင်ႈပွၵ်ႈၼိုင်ႈ။
* **Magic:** မိူဝ်ႈႁဝ်း Push ၶိုၼ်းၼၼ်ႉ GitHub တေဢဝ် Commit မႂ်ႇၼၼ်ႉ သႂ်ႇပၼ်ၼႂ်း PR ဢၼ်တိုၵ်ႉပိုတ်ႇဝႆႉၼၼ်ႉ ႁင်းၵူၺ်း (Automatically) ၶႃႈယဝ်ႉ!

#### **3. Addressing Conversations (ၵၢၼ်ပိတ်းပၼ်ႁႃ)**
* မိူဝ်ႈႁဝ်းမႄးၸွမ်း ဢၼ်ၶဝ်ၼႄ ယဝ်ႉယဝ်ႉၼၼ်ႉ ႁဝ်းၸၢင်ႈၼဵၵ်း **"Resolve conversation"** တီႈ GitHub။
* မၼ်းတေၼႄပၼ် Maintainer ၶဝ်ဝႃႈ "တွၼ်ႈၼႆႉ ႁဝ်းမႄးယဝ်ႉယဝ်ႉၶႃႈၼႃ" ၼႆၶႃႈဢေႃႈ။

#### **4. The Final Approval**
မိူဝ်ႈၶဝ်ပွင်ႇၸႂ်လီယဝ်ႉ:
* ၶဝ်တေၼဵၵ်း **"Approve"**။
* ယဝ်ႉၵေႃႈ ၶဝ်တေၼဵၵ်း **"Merge pull request"**။
* **Congratulations!** Code ႁဝ်းၶဝ်ႈၵႂႃႇ ၼႂ်း Project လူင်ၼၼ်ႉယဝ်ႉၶႃႈ!



#### **ႁူဝ်ႁုပ်ႈ (Summary)**
**Code Review** ၼႆႉ ပဵၼ်ၵၢၼ်ႁဵၼ်းႁူႉဢၼ်လီသုတ်း။ 
မၼ်းႁဵတ်းႁႂ်ႈလုၵ်ႈႁဵၼ်းၶဝ် လႆႈႁူႉၸွမ်းလၢႆးတႅမ်ႈ Code ၶွင် Professional လႄႈ ၸွႆႈႁႂ်ႈႁဝ်းမီးၸႂ် "ႁူမ်ႈၵၼ် (Teamwork)" ၼၼ်ႉၶႃႈဢေႃႈ။
