## **Module 4: Distributed Collaboration**
### **Lesson 31: The Pull Request (PR) Concept (ၵၢၼ်ယွၼ်းႁူမ်ႈ Code)**

**Pull Request** (ၵႆႉႁွင်ႉဝႃႈ **PR**) ၼႆႉ ဢမ်ႇၸႂ်ႈ Command ၼႂ်း Git ၵူၺ်းၵႃႈ မၼ်းပဵၼ်ပိူင်ၵၢၼ် (Workflow) ၼိူဝ် GitHub ၶႃႈ။

#### **1. Pull Request ပွင်ႇဝႃႈသင်?**
* မၼ်းပွင်ႇဝႃႈ: "ၶႅၼ်းတေႃႈ **Pull** (လူတ်ႇဢဝ်) Code ဢၼ်ၶႃႈမႄးဝႆႉၼႆႉ ၵႂႃႇ **Merge** (ႁူမ်ႈ) သႂ်ႇၼႂ်း Project သူၸဝ်ႈပၼ်သေၵမ်းၶႃႈ" ၼႆၶႃႈဢေႃႈ။
* မၼ်းမိူၼ်ၼင်ႇႁဝ်း "ယိုၼ်ႈလိၵ်ႈယွၼ်းၶႂၢင်ႉ" ၼင်ႇႁႂ်ႈၸဝ်ႈၶွင် Project ၶဝ်ၵူတ်ႇထတ်း Code ႁဝ်းဢွၼ်တၢင်းၼၼ်ႉၶႃႈယဝ်ႉ။



#### **2. တၢင်းလီၶွင်ၵၢၼ်ၸႂ်ႉ PR**
1. **Code Review:** ၸဝ်ႈၶွင် Project တေၸၢင်ႈတူၺ်းလႆႈဝႃႈ ႁဝ်းမႄးသင်ၵႂႃႇ၊ မၼ်းမီး Error သင်? ဢွၼ်တၢင်း မိူဝ်ႈၶဝ်ပႆႇႁပ်ႉဢဝ်။
2. **Discussion:** ႁဝ်းလႄႈ ၸဝ်ႈၶွင် Project ၸၢင်ႈ "ဢုပ်ႇၵၼ်" ၼႂ်း PR ၼၼ်ႉလႆႈ (မိူၼ်ၼင်ႇ "တွၼ်ႈၼႆႉ မႄးၼင်ႇၼႆ တေလီသေၶႃႈဢေႃႈ")။
3. **Quality:** ၸွႆႈၵႅတ်ႇၶႄ ဢမ်ႇႁႂ်ႈ Code ဢၼ်လူႉၼၼ်ႉ ၶဝ်ႈၵႂႃႇၼႂ်း Project လူင်လႆႈငၢႆႈငၢႆႈ။

#### **3. ပိူင်ၵၢၼ် (The Logic)**
မိူဝ်ႈႁဝ်းသၢင်ႈ PR ၼၼ်ႉ GitHub တေၼႄပၼ်:
* **Base Repository:** Project လူင် (Original) ဢၼ်ႁဝ်းၶႂ်ႈသူင်ႇ Code ၶဝ်ႈၵႂႃႇ။
* **Head Repository:** Project ဢၼ်ႁဝ်း Fork မႃး (Copy ႁဝ်း) ဢၼ်မီး Code မႂ်ႇမႂ်ႇၼၼ်ႉ။



#### **4. ၸၢင်ႈပဵၼ်သင်လႆႈၽႅဝ်လႂ်?**
မိူဝ်ႈႁဝ်းသူင်ႇ PR ယဝ်ႉယဝ်ႉ:
* **Merged:** ၶဝ်ႁပ်ႉဢဝ် Code ႁဝ်းယဝ်ႉ။ (ၶႅမ်ႉလီငၢမ်း!)
* **Changes Requested:** ၶဝ်ၶႂ်ႈႁႂ်ႈႁဝ်းမႄးထႅင်ႈဢိတ်းၼိုင်ႈ ဢွၼ်တၢင်းပႆႇ Merge။
* **Closed:** ၶဝ်ဢမ်ႇႁပ်ႉဢဝ် Code ႁဝ်း (ၵွပ်ႈဝႃႈမၼ်းဢမ်ႇထုၵ်ႇ ဢမ်ႇၼၼ် မၼ်းမီးပၼ်ႁႃ)။

#### **ႁူဝ်ႁုပ်ႈ (Summary)**
**Pull Request** ၼႆႉ ပဵၼ်ၵၢၼ် "လႅၵ်ႈလၢႆႈတၢင်းႁၼ်ထိုင်" ၼႂ်းၵႄႈ Developer ၶႃႈယဝ်ႉ။ 
မၼ်းပဵၼ်လၢႆးႁဵတ်းသၢင်ႈဢၼ် Professional ၶဝ်ၸႂ်ႉ ၸႃႇႁႂ်ႈ App ၶဝ် ၶိုၵ်ႉတွၼ်း သုင်သုတ်းၼၼ်ႉၶႃႈဢေႃႈ။
