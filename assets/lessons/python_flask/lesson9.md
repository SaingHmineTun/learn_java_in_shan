

## **Module 2: Routing and Request Handling**

### **Lesson 9: HTTP Methods (လွင်ႈၸတ်းၵၢၼ် လွၵ်းလၢႆးသူင်ႇၶေႃႈမုၼ်း HTTP)**

#### **1. HTTP Methods ၼႆႉ ပဵၼ်သင်?**

တီႈၼိူဝ် Web ၼၼ်ႉ မိူဝ်ႈ Browser (ၶိူင်ႈပိုတ်ႇ Web) လႄႈ Server ၵပ်းသိုပ်ႇၵၼ်၊ ၶဝ်ၸႂ်ႉ "လွၵ်းလၢႆး" (Methods) လၢႆလၢႆမဵဝ်း တွၼ်ႈတႃႇမွၵ်ႇလၢတ်ႈဝႃႈ ၶဝ်ၶႂ်ႈႁဵတ်းသင်ၶႃႈ။ ဢၼ်ၵူၼ်းၸႂ်ႉၼမ်သေပိူၼ်ႈ မီး 4 မဵဝ်း -

* **GET:** တွၼ်ႈတႃႇ **"ယွၼ်းဢဝ်"** ၶေႃႈမုၼ်း (တူဝ်ယၢင်ႇ - ပိုတ်ႇၼႃႈ Website သေလူဢၢၼ်ႇ)။
* **POST:** တွၼ်ႈတႃႇ **"သူင်ႇ"** ၶေႃႈမုၼ်းလပ်ႉလပ်ႉ ဢမ်ႇၼၼ် ၶေႃႈမုၼ်းယႂ်ႇယႂ်ႇ (တူဝ်ယၢင်ႇ - သႂ်ႇ Username/Password သေ Login ဢမ်ႇၼၼ် ၼဵၵ်း Submit ၾွမ် / Form)။
* **PUT:** တွၼ်ႈတႃႇ **"မူၼ်ႉမႄး"** ဢမ်ႇၼၼ် လႅၵ်ႈလၢႆႈ ၶေႃႈမုၼ်း ဢၼ်မီးဝႆႉယဝ်ႉ။
* **DELETE:** တွၼ်ႈတႃႇ **"မွတ်ႇ"** ပႅတ်ႈ ၶေႃႈမုၼ်း။

#### **2. လွင်ႈၸႂ်ႉ `methods` ၼႂ်း `@app.route()**`

ၼႂ်း Flask ၼၼ်ႉ ပေႃးႁဝ်းဢမ်ႇတႅမ်ႈမၵ်းမၼ်ႈသင်ၼႆ၊ Flask တေမၵ်းမၼ်ႈဝႃႈ Route ၼၼ်ႉ ႁပ်ႉဢဝ် **GET** လွၵ်းလၢႆးလဵဝ်ၵူၺ်းၶႃႈ။
သင်ဝႃႈႁဝ်းၶႂ်ႈႁပ်ႉဢဝ် လွၵ်းလၢႆးတၢင်ႇမဵဝ်း (မိူၼ်ၼင်ႇ POST တွၼ်ႈတႃႇႁပ်ႉၶေႃႈမုၼ်းတီႈ Form) ၼႆ၊ ႁဝ်းလူဝ်ႇလႆႈမၵ်းမၼ်ႈသႂ်ႇၼႂ်း Code လိူဝ်မႃးၶႃႈ။

**တူဝ်ယၢင်ႇ Code တႃႇႁပ်ႉဢဝ် GET လႄႈ POST:**

```python
from flask import Flask, request

app = Flask(__name__)

@app.route('/login', methods=['GET', 'POST'])
def login():
    # ၵူတ်ႇတူၺ်းဝႃႈ Method မၼ်း ပဵၼ် POST ႁိုဝ် GET
    if request.method == 'POST':
        # ပေႃးပဵၼ် POST မၼ်းတေႁဵတ်းၵၢၼ် ၼႂ်းထႅဝ်ၼႆႉ
        return "တိုၵ်ႉသူင်ႇၶေႃႈမုၼ်း Login (POST Request) ၶဝ်ႈမႃးၶႃႈ!"
    else:
        # ပေႃးပဵၼ် GET ဢမ်ႇၼၼ် တၢင်ႇမဵဝ်း မၼ်းတေႁဵတ်းၵၢၼ် ၼႂ်းထႅဝ်ၼႆႉ
        return "ၼႆႉပဵၼ် ၼႃႈလိၵ်ႈ Login တွၼ်ႈတႃႇသႂ်ႇၶေႃႈမုၼ်း (GET Request) ၶႃႈ!"

if __name__ == '__main__':
    app.run(debug=True)

```

#### **3. လွင်ႈသပ်းလႅင်း Code (Code Explanation)**

* **`from flask import request`:** ႁဝ်းလူဝ်ႇလႆႈႁွင်ႉ `request` Object ၶွင် Flask မႃးၸႂ်ႉၶႃႈ။ မၼ်းပဵၼ် Object ဢၼ်သိမ်းဝႆႉ ၶေႃႈမုၼ်းၵူႈမဵဝ်းမဵဝ်း ဢၼ် Browser သူင်ႇၶဝ်ႈမႃးၸူး Server ႁဝ်း။
* **`methods=['GET', 'POST']`:** ပၼ်သုၼ်ႇႁႂ်ႈ Route ၼႆႉ ႁပ်ႉလႆႈတင်း 2 မဵဝ်း။ ပေႃးဢမ်ႇသႂ်ႇလွၵ်းၼႆႉသေ သမ်ႉမီးၵူၼ်းသူင်ႇ POST ၶဝ်ႈမႃးၼႆ Flask တေဢွၵ်ႇ Error ဢၼ်မီးမၢႆဝႃႈ **405 Method Not Allowed** (လွၵ်းလၢႆးၼႆႉ ဢမ်ႇပၼ်သုၼ်ႇၸႂ်ႉ) ၵမ်းသိုဝ်ႈၶႃႈ။
* **`request.method`:** ၸွႆႈၵူတ်ႇထတ်းပၼ်ဝႃႈ ယၢမ်းလဵဝ် ၵူၼ်းဢၼ်ၶဝ်ႈမႃးၼၼ်ႉ ၶဝ်ၸႂ်ႉ Method မဵဝ်းလႂ်သေ ၶဝ်ႈမႃး။ ႁဝ်းၸင်ႇၸႂ်ႉ `if/else` သေ ၸႅၵ်ႇလွင်ႈႁဵတ်းၵၢၼ်မၼ်း ႁႂ်ႈမၢၼ်ႇမႅၼ်ႈၶႃႈဢေႃႈ။
