

## **Module 3: Templates and Static Files**

### **Lesson 14: Jinja2 Control Structures (လွင်ႈၸႂ်ႉ Logic လႄႈ Control Structures ၼႂ်း Jinja2)**

တီႈၼႂ်း Lesson 13 ၼၼ်ႉ ႁဝ်းလႆႈႁဵၼ်းမႃး လွင်ႈၸႂ်ႉ `{{ }}` တွၼ်ႈတႃႇၼႄ ၵႃႈၶၼ် (Variable) လၢႆလၢႆၶႃႈ။
တီႈၼႂ်း Lesson 14 ၼႆႉ ႁဝ်းတေမႃးႁဵၼ်းႁူႉ လွၵ်းလၢႆးၸႂ်ႉ **Logic** တီႈၼႂ်း HTML ၵမ်းသိုဝ်ႈ ၸႂ်ႉတိုဝ်း **`{% %}`** (မၢႆၶွပ်ႇငွၵ်း ဢၼ်ပႃး မၢႆပႃႇသႅၼ်ႉ) ၶႃႈဢေႃႈ။

#### **1. လွင်ႈထတ်းသိၼ် (The `{% if %}` Statement)**

မၢင်ပွၵ်ႈမၢင်လႂ် ႁဝ်းၶႂ်ႈၼႄ HTML ဢၼ်ၼိုင်ႈ မိူဝ်ႈငဝ်းလၢႆးမၼ်းမၢၼ်ႇမႅၼ်ႈၵူၺ်း (တူဝ်ယၢင်ႇ - ၼႄ "Logout" တွၼ်ႈတႃႇ ၵူၼ်းဢၼ် Login ယဝ်ႉၵူၺ်း)။
တွၼ်ႈတႃႇလွင်ႈၼႆႉ ႁဝ်းၸႂ်ႉ `{% if %}` လႄႈ `{% else %}` ၶႃႈ။ **ဢၼ်လွင်ႈယႂ်ႇသေပိူၼ်ႈတႄႉ ယႃႇပေလိုမ်း ပိၵ်ႉမၼ်းၶိုၼ်းၸွမ်း `{% endif %}` ၶႃႈၼႃႈ။**

**တူဝ်ယၢင်ႇ ၽၢႆႇ Python (`app.py`):**

```python
@app.route('/')
def home():
    # ၸၢမ်းၵူတ်ႇတူၺ်းဝႃႈ User Login ယဝ်ႉႁိုဝ်ပႆႇ
    user_logged_in = True
    return render_template('index.html', is_logged_in=user_logged_in)

```

**တူဝ်ယၢင်ႇ ၽၢႆႇ HTML (`index.html`):**

```html
<div>
    {% if is_logged_in %}
        <!-- ပေႃး is_logged_in ပဵၼ် True တေၼႄပၼ် ဢၼ်ၼႆႉ -->
        <h1>မႂ်ႇသုင်ၶႃႈ! ၸဝ်ႈၵဝ်ႇ ၶဝ်ႈလၢၼ်ႉယဝ်ႉ။</h1>
        <a href="/logout">ဢွၵ်ႇလၢၼ်ႉ (Logout)</a>
    {% else %}
        <!-- သင်ဢမ်ႇၸႂ်ႈ တေၼႄပၼ် ဢၼ်ၼႆႉ -->
        <h1>ၶႅၼ်းတေႃႈ Login ၶဝ်ႈဢွၼ်တၢင်းၶႃႈ။</h1>
        <a href="/login">ၶဝ်ႈလၢၼ်ႉ (Login)</a>
    {% endif %}
</div>

```

#### **2. လွင်ႈပၼ်ႇတူင်ႇ (The `{% for %}` Loop)**

သင်ဝႃႈ ႁဝ်းလႆႈၶေႃႈမုၼ်းမႃးတီႈ Database ပဵၼ်သဵၼ်ႈမၢႆ (List) ယၢဝ်းယၢဝ်း (တူဝ်ယၢင်ႇ - သဵၼ်ႈမၢႆ ၸိုဝ်ႈၵူၼ်း 10 ၵေႃႉ) သေလႄႈ 
ၶႂ်ႈၼႄၼႂ်း HTML ၼႆ၊ ႁဝ်းဢမ်ႇတၢပ်ႈလူဝ်ႇ တႅမ်ႈ Code HTML 10 ထႅဝ်ၶႃႈ။ ႁဝ်းၸႂ်ႉ `{% for %}` တွၼ်ႈတႃႇပၼ်ႇ ႁႂ်ႈမၼ်းသၢင်ႈပၼ်ႁင်းၵူၺ်း လႆႈၶႃႈ။ 
**ယႃႇပေလိုမ်း ပိၵ်ႉမၼ်းၶိုၼ်းၸွမ်း `{% endfor %}` ၶႃႈ။**

**တူဝ်ယၢင်ႇ ၽၢႆႇ Python (`app.py`):**

```python
@app.route('/users')
def users():
    # သၢင်ႈ List ၶေႃႈမုၼ်း ဢၼ်မီးၸိုဝ်ႈ 3 ၵေႃႉ
    user_list = ['ၸၢႆးလႅင်း', 'ၼၢင်းၶမ်း', 'ၸၢႆးဢွင်ႇ']
    return render_template('users.html', users=user_list)

```

**တူဝ်ယၢင်ႇ ၽၢႆႇ HTML (`users.html`):**

```html
<h1>သဵၼ်ႈမၢႆ ၵူၼ်းၸႂ်ႉတိုဝ်း:</h1>
<ul>
    <!-- ဢဝ်ၶေႃႈမုၼ်းၼႂ်း users မႃးသႂ်ႇၼႂ်း user ၵမ်းလွၵ်းၵမ်းလွၵ်း -->
    {% for user in users %}
        <li>ၸိုဝ်ႈ: {{ user }}</li>
    {% endfor %}
</ul>

```

**ၽွၼ်းလႆႈ (Result):** Jinja2 တေပၼ်ႇလွပ်ႈသေ သၢင်ႈပၼ် `<li>` 3 ဢၼ် ဢၼ်မီးၸိုဝ်ႈ 3 ၵေႃႉၼၼ်ႉ ႁင်းၵူၺ်းမၼ်း ၵမ်းသိုဝ်ႈၶႃႈ။
