

## **Module 1: Introduction & Environment Setup**

### **Lesson 4: Running Python as a Python Program (လွင်ႈ Run Python ႁႂ်ႈပဵၼ် Program ဢၼ်ၼိုင်ႈ)**

တီႈၼႂ်း Lesson 1.3 ဢၼ်ပူၼ်ႉမႃးၼၼ်ႉ ႁဝ်းလႆႈတႅမ်ႈ Code `app.run(debug=True)` ဝႆႉတီႈတႂ်ႈသုတ်းသေ ႁဝ်းလႆႈ Run ၾၢႆႇၼၼ်ႉ ၸႂ်ႉ Command ဝႃႈ `python app.py` ၶႃႈၼေႃ။ 
လွၵ်းလၢႆးၼႆႉ မၼ်းႁွင်ႉဝႃႈ **"Running Python as a Python program"** (လွင်ႈ Run ႁႂ်ႈပဵၼ် Program ၶွင် Python ၵမ်းသိုဝ်ႈ) ၶႃႈ။

ႁဝ်းမႃးၸႅၵ်ႇတူၺ်း လွင်ႈလိုၵ်ႉလမ်ႇမၼ်း ဢၼ်မီးၼႂ်း Code ႁဝ်းၵၼ်ၶႃႈ။

#### **1. ၶေႃႈၵႂၢမ်း `if __name__ == '__main__':` ၼႆႉ ပွင်ႇဝႃႈသင်?**

ပေႃးႁဝ်းတူၺ်းၼႂ်း Code Flask ႁဝ်းၼၼ်ႉ ႁဝ်းတေလႆႈႁၼ် Code ထႅဝ်ၼႆႉ -

```python
if __name__ == '__main__':
    app.run(debug=True)

```

ၼႆႉပဵၼ် Code ဢၼ်လွင်ႈယႂ်ႇတႄႉတႄႉ တွၼ်ႈတႃႇ Python ၶႃႈ။ လွင်ႈသပ်းလႅင်းမၼ်း မီးၼင်ႇၼႆၶႃႈ -

* **`__name__`** ၼႆႉ ပဵၼ် Variable လပ်ႉလပ်ႉ (Hidden Variable) ဢၼ် Python သၢင်ႈပၼ် ၵူႈၾၢႆႇၾၢႆႇ ႁင်းၵူၺ်းမၼ်းၶႃႈ။
* ပေႃးႁဝ်း Run ၾၢႆႇၼၼ်ႉ ၵမ်းသိုဝ်ႈၼႂ်း Terminal (တူဝ်ယၢင်ႇ - `python app.py`)၊ Python တေမၵ်းမၼ်ႈ ၵႃႈၶၼ် (Value) ၶွင် `__name__` ၼၼ်ႉ ႁႂ်ႈပဵၼ် **`'__main__'`** ၶႃႈဢေႃႈ။
* ၵူၺ်းၵႃႈ သင်ဝႃႈ ၾၢႆႇ `app.py` ၼႆႉ ထုၵ်ႇ **Import** ၵႂႃႇၸႂ်ႉၼႂ်း ၾၢႆႇတၢင်ႇဢၼ် (တူဝ်ယၢင်ႇ - `import app`) ၼႆ၊ ၵႃႈၶၼ် `__name__` ၼၼ်ႉ မၼ်းတေဢမ်ႇပဵၼ် `'__main__'` ယဝ်ႉ။ မၼ်းတေပဵၼ် ၸိုဝ်ႈၾၢႆႇမၼ်း ဢၼ်ပဵၼ် `'app'` ၼၼ်ႉၶႃႈ။

**လွင်ႈလႆႈၸႂ်ႉမၼ်း:** ၶေႃႈၵႂၢမ်းၼႆႉ မၼ်းႁႄႉၵင်ႈပၼ် ႁႂ်ႈ Server (`app.run()`) ႁဵတ်းၵၢၼ် မိူဝ်ႈႁဝ်း Run ၾၢႆႇၼႆႉ **ၵမ်းသိုဝ်ႈ** ၵူၺ်း။ ပေႃးၾၢႆႇၼႆႉ ထုၵ်ႇ Import ၵႂႃႇတီႈတၢင်ႇဢၼ်၊ Server မၼ်းတေဢမ်ႇပိုတ်ႇႁင်းၵူၺ်းမၼ်းၶႃႈ။

---

#### **2. လွင်ႈပႅၵ်ႇပိူင်ႈ `python app.py` လႄႈ ၶေႃႈသင်ႇ `flask run**`

ၼႂ်း Flask ၼႆႉ တွၼ်ႈတႃႇတေ Run Server မၼ်းမီးလွၵ်းလၢႆး 2 ဢၼ် ဢၼ်ၵူၼ်းၸႂ်ႉၼမ်ၶႃႈ။

**လွၵ်းလၢႆး 1: Run မိူၼ် Python Program (ဢၼ်ႁဝ်းတိုၵ်ႉႁဵတ်းယူႇ)**

* ၸႂ်ႉ Command: `python app.py`
* မၼ်းတေႁဵတ်းၵၢၼ်ၸွမ်း Code `app.run()` ဢၼ်မီးၼႂ်းၾၢႆႇႁဝ်း။
* လွၵ်းလၢႆးၼႆႉ မၼ်းငၢႆႈ လႄႈ လီတွၼ်ႈတႃႇ ၵူၼ်းတႄႇႁဵၼ်းမႂ်ႇၶႃႈ။

**လွၵ်းလၢႆး 2: ၸႂ်ႉ Command ၶွင် Flask (Flask CLI)**
Flask မၼ်းမီး Command ႁင်းမၼ်း တွၼ်ႈတႃႇ Run Server ၶႃႈ။ ပေႃးၸႂ်ႉလွၵ်းလၢႆးၼႆႉ ႁဝ်းဢမ်ႇတၢပ်ႈလူဝ်ႇတႅမ်ႈ `if __name__ == '__main__':` ၵေႃႈလႆႈၶႃႈ။

* ၵႂႃႇတီႈ Terminal သေ တႅမ်ႈဝႃႈ:
```bash
flask --app app run --debug

```


* `--app app` ပွင်ႇဝႃႈ သင်ႇႁႂ်ႈ Flask ၵႂႃႇသွၵ်ႈႁႃ Application တီႈၼႂ်းၾၢႆႇ `app.py`။
* `--debug` ပွင်ႇဝႃႈ ပိုတ်ႇ Debug Mode (ႁႂ်ႈမၼ်း Refresh ႁင်းၵူၺ်း မိူဝ်ႈႁဝ်းမႄး Code)။
