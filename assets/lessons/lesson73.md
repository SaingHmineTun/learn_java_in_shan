
## **Java Basic 73**

#### **STACK**

Link - <https://youtu.be/Z6-3fSRErOY>

မၼ်းႁဵတ်ႉ extends ဝႆႉ Vector ၼႆလႄႈ ၼႂ်းမၼ်း တေၸႂ်ႉလႆႈ method ဢၼ်မီးဝႆႉၼႂ်း Vector တၢင်းသဵင်ႈယဝ်ႉ။ ယဝ်ႉၵေႃႈ မၼ်းၵေႃႈပဵၼ် Sync ၼင်ႇၵဝ်ႇ။ Stack ၼႆႉတေႉ မၼ်းတေမိူၼ် ဢၼ်ႁဝ်း ၶပ်ႉဝၢၼ်ႇၼၼ်ႉဢိူဝ်ႈ။ ဝၢၼ်ႇၼႆႉ ပေႃးလၢင်ႉယဝ်ႉ ႁဝ်းၵူၺ်း ဢဝ်ဢၼ်လၢင်ႉယဝ်ႉၼႆႉ ၶပ်ႉၶိုၼ်ႈၵႂႃႇ တီႈၼိူဝ်သုတ်ႉၵူၺ်း။ သင်ႁဝ်းတေဢဝ် ဝၢၼ်ႇတႃႇၸႂ်ႉၼႆၸိုင် ႁဝ်းၶိုၼ်းဢဝ် ဝၢၼ်ဢၼ် မီးဝႆႉတီႈၼိူဝ်သုတ်ႉၵူၺ်း။ ၼႆႉမၼ်းပဵၼ် ပိူင် LIFO (Last in, First out) ယဝ်ႉ။

methods ၵမ်ႈၽွင်ႈ ဢၼ်တေၸႂ်ႉတိုဝ်းလႆႈ တီႈၼႂ်းမၼ်းၼၼ်ႉတေႉ

- **push(E e)** - ဢဝ် element သႂ်ႇတီႈၼိူဝ် Stack
- **peek()** - ဢဝ် Data ဢၼ်မီးဝႆႉ တီႈၼိူဝ် Stack
- **pop()** - ဢဝ်ယဝ်ႉထွၼ် Data ဢၼ်မီးဝႆႉ တီႈၼိူဝ် Stack
- **search(E e)** - ႁႃသေ တွပ်ႇပၼ်ၶိုၼ်းဝႃႈ Element ၼႆႉ ယူႇတီႈလႂ်၊ တီႈၼႆႈ ဢမ်ႇၸႂ်ႉတိုဝ်း index လႄႈ မၼ်းတွပ်ႇၶိုၼ်းဝႃႈ element ဢၼ်ႁဝ်းႁႃယူႇၼၼ်ႉ ပေႃးလုၵ်ႉတီႈၼိူဝ် stack ၼႆ မၼ်းတေမီးဝႆႉ တီႈမၢႆလႂ်ၼႆယဝ်ႉ။ ပေႃးဢမ်ႇႁၼ်ၼႆသမ်ႉ ပၼ်ၶိုၼ်း -1 ယဝ်ႉ။

###### လၢႆးၶူင်သၢင်ႈ

```
Stack<String> list = new Stack<>();
```

###### လၢႆးသႂ်ႇၵႃႊၶၼ်

```
list.addElement("Sai");
list.add("Mao");
list.push("From");
```

###### လၢႆးဢဝ်ၵႃႊၶၼ်

```
System.out.println("Peek : " + list.peek());
```

###### လၢႆးဢဝ်ပႅတ်ႈၵႃႊၶၼ်

```
System.out.println("Pop : " + list.pop());
```

###### လၢႆးႁႃၵႃႊၶၼ်

```
System.out.println(list.search("MAO"));
```
