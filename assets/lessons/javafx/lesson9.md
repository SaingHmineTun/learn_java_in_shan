## **Lesson 9: Scene Builder Installation & Setup**

မိူဝ်ႈႁဝ်းတေ Design ၶၢပ်ႈ (Template) ID Card ႁဝ်းႁႂ်ႈႁၢင်ႈလီၼၼ်ႉ ႁဝ်းတေလႆႈၸႂ်ႉ **Scene Builder** ၶႃႈ။ 
မၼ်းပဵၼ်ၶိူင်ႈၵမ်ႉၸွႆႈ ဢၼ်တေႁဵတ်းႁႂ်ႈႁဝ်း "ႁၼ်ႁၢင်ႈ App" ၵမ်းလဵဝ် (Visual Tool) မိူဝ်ႈႁဝ်းတိုၵ်ႉတႅမ်ႈ Code ယူႇၼၼ်ႉၶႃႈ။

### **1. ၵၢၼ်လူတ်ႇ Scene Builder (Download)**
Software ၼႆႉ ပဵၼ်ၶွင် ၵေႃပွင်ၵၢၼ် **Gluon** ဢၼ်ပၼ်ၸႂ်ႉလၢႆ (Free) ၶႃႈ။
* **Link:** [gluonhq.com/products/scene-builder/](https://gluonhq.com/products/scene-builder/)
* လိူၵ်ႈလူတ်ႇဢဝ်ၸွမ်းၼင်ႇ OS Computer ႁဝ်း (Windows, macOS, ဢမ်ႇၼၼ် Linux) ယဝ်ႉၵေႃႈ Install ႁႂ်ႈယဝ်ႉတူဝ်ႈၶႃႈ။



### **2. ၵၢၼ်သႂ်ႇ (Setup) ၼႂ်း IntelliJ IDEA**
တွၼ်ႈတႃႇတေပိုတ်ႇ Scene Builder လႆႈငၢႆႈၼႂ်း IDE ၼၼ်ႉ ႁဝ်းတေလႆႈသႂ်ႇ Path မၼ်းဝႆႉၶႃႈ။
1.  ပိုတ်ႇ **IntelliJ** ယဝ်ႉၵႂႃႇတီႈ **Settings** (File > Settings)။
2.  ၵႂႃႇတီႈ **Languages & Frameworks** > **JavaFX**။
3.  ၼႂ်းၼၼ်ႉ တူၺ်းတီႈ **Path to Scene Builder** ယဝ်ႉလိူၵ်ႈဢဝ်ၾၢႆႇ `SceneBuilder.exe` (တႃႇ Windows) ဢၼ်မီးၼႂ်း Folder ဢၼ်ႁဝ်း Install ဝႆႉၼၼ်ႉၶႃႈ။



### **3. How to Open your ID Card UI**
မိူဝ်ႈႁဝ်းမီးၾၢႆႇ `.fxml` ၼႂ်း Project ယဝ်ႉ:
* Right-click ၼိူဝ်ၾၢႆႇ FXML ၼၼ်ႉ။
* လိူၵ်ႈဢဝ် **"Open In Scene Builder"**။
* IntelliJ တေပိုတ်ႇ Software Scene Builder မႃးပၼ်ႁဝ်း တွၼ်ႈတႃႇ Design ၶၢပ်ႈ ID Card လႆႈၵမ်းလဵဝ်ၶႃႈ။

### **4. Introduction to FXML Structure**
မိူဝ်ႈႁဝ်းၸႂ်ႉ Scene Builder ၼၼ်ႉ JavaFX တေဢမ်ႇသၢင်ႈ UI လူၺ်ႈ Java Code ၵမ်းသိုဝ်ႈ၊ မၼ်းတေသၢင်ႈၾၢႆႇဢၼ်ႁွင်ႉဝႃႈ **.fxml** ၶႃႈ။
* **FXML ၼႆႉပဵၼ်သင်?** မၼ်းမိူၼ်ၼင်ႇ "ၽႅၼ်ႇၵေႃႇသၢင်ႈ" (Blueprint) ၶွင် ID Card ႁဝ်းၶႃႈ။ မၼ်းတေတႅမ်ႈဝႆႉဝႃႈ ႁၢင်ႈလုၵ်ႈႁဵၼ်းယူႇၽၢႆႇသၢႆႉ၊ ၸိုဝ်ႈယူႇၽၢႆႇၶႂႃ၊ သီ Card ပဵၼ်သီသင်။
* **ၸွႆႈသင်?** မၼ်းၸွႆႈႁႂ်ႈႁဝ်းၸႅၵ်ႇၽႄ ၵၢၼ် Design (FXML) လႄႈ ၵၢၼ်တႅမ်ႈ Logic (Java) ဢွၵ်ႇသေ ႁႂ်ႈတူၺ်းငၢႆႈ လႄႈ ၸတ်းၵၢၼ်လႆႈလီတိူဝ်းၶႃႈ။

* **FXML:** ပဵၼ် "ၽႅၼ်ႇၵေႃႇသၢင်ႈ" (Blueprint) ၶွင် ID Card UI။
