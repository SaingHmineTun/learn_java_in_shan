## Lesson 33: The 'Optional' Concept: Why Swift Doesn't Allow Nulls

ၼႂ်းလိၵ်ႈ Programming တၢင်ႇမဵဝ်း (မိူၼ်ၼင်ႇ Java ဢမ်ႇၼၼ် C#)၊ Variable ၸၢင်ႈပဵၼ် **`null`** (ဝၢင်ႇဝႆႉ/ဢမ်ႇမီးၶေႃႈมูล) လႆႈၵူႈတူဝ်။ ပေႃးႁဝ်းလိုမ်းၵူတ်ႇထတ်းသေ ၵႂႃႇၸႂ်ႉ Variable ဢၼ်ပဵၼ် `null` ၼၼ်ႉ App တေ Crash ၵမ်းလဵဝ်။ ဢၼ်ၼႆႉပဵၼ်ပၼ်ႁႃလူင် ဢၼ်ၵူၼ်းတႅမ်ႈ Code ၵူဝ်ပိူၼ်ႈသုတ်းယဝ်ႉ။

Swift ၸင်ႇၵႄႈလႅတ်ႇပၼ်ႁႃၼႆႉလူၺ်ႈ **Optionals** ၶႃႈ။

### 1. Swift ဢမ်ႇပၼ်မီး Null (By Default)
ၼႂ်း Swift, Variable ထမ်ႇမတႃႇ **တေလႆႈမီးၵႃႈၶၼ်ၵူႈပွၵ်ႈ**။ မၼ်းဢမ်ႇပၼ်ႁဝ်းသႂ်ႇ `nil` (ၶေႃႈၵႂၢမ်း Swift ဢၼ်ၸႂ်ႉတႅၼ်း null) ၶဝ်ႈၵႂႃႇၼႂ်း Variable ဢၼ်ဢမ်ႇၸႂ်ႉ Optional ၶႃႈ။

```swift
var name: String = "Sai Mao"
// name = nil // Error: String ဢမ်ႇပၼ်ပဵၼ် nil လႆႈ
```

### 2. Optional မၢႆထိုင်သင်?
**Optional** မိူၼ်ၼင်ႇ "ၵွၵ်း" (Box) ဢၼ်ႁဝ်းႁေႃႇဝႆႉယဝ်ႉ။
*   ၼႂ်းၵွၵ်းၼၼ်ႉ ၸၢင်ႈမီး **"ၶေႃႈมูล"** ယူႇ။
*   ဢမ်ႇၼၼ် ၼႂ်းၵွၵ်းၼၼ်ႉ ၸၢင်ႈ **"ဝၢင်ႇဝႆႉ"** (`nil`) ၵေႃႈလႆႈ။

ႁဝ်းၸႂ်ႉမၢႆ **`?`** တွၼ်ႈတႃႇမၵ်းမၼ်ႈဝႃႈ Variable ၼႆႉပဵၼ် Optional ၶႃႈ။



```swift
var nickname: String? = "Mao" // မီးၶေႃႈมูล "Mao" ယူႇၼႂ်းၵွၵ်း
nickname = nil                // ယၢမ်းလဵဝ် ၵွၵ်းဝၢင်ႇဝႆႉယဝ်ႉ (မၢၼ်ႇမႅၼ်ႈ၊ ဢမ်ႇ Error)
```

---

### 3. ၵွပ်ႈသင်လၢႆးၼႆႉသမ်ႉလီ?
1.  **Safety (လွင်ႈလွတ်ႈၽေး):** Swift တေบังคับ ႁႂ်ႈႁဝ်း "ၵူတ်ႇထတ်း" (Unwrap) ၵွၵ်းၼၼ်ႉဢွၼ်တၢင်း ၸင်ႇတေပၼ်ၸႂ်ႉၶေႃႈมูลၽၢႆႇၼႂ်း။ မၼ်းႁဵတ်းႁႂ်ႈႁဝ်းဢမ်ႇလိုမ်းၵူတ်ႇထတ်း `nil` ယဝ်ႉ။
2.  **Clarity (လွင်ႈၸႅၵ်ႇလႅင်း):** ပေႃးႁဝ်းလူ Code၊ ႁဝ်းတေႁူႉလႆႈၵမ်းလဵဝ်ဝႃႈ Variable တူဝ်လႂ် ၸၢင်ႈပဵၼ် `nil` လႆႈ လႄႈ တူဝ်လႂ်ဢၼ်မီးၶေႃႈมูลသေႇသေႇ (Always has a value)။

---

### 4. ၵၢၼ်ၸႂ်ႉတိုဝ်းတႄႉ (Best Practice)
> **"တွၼ်ႈတႃႇ ၸၢႆးမၢဝ်း ႁဵတ်း Keyboard ၼၼ်ႉ: Variable တွၼ်ႈတႃႇၵဵပ်း 'တူဝ်လိၵ်ႈဢၼ် User တိုၵ်ႉၼိပ်ႉ' ၼၼ်ႉ လူဝ်ႇပဵၼ် Optional (`String?`)။ ၵွပ်ႈဝႃႈမၢင်ပွၵ်ႈ User ပႆႇလႆႈၼိပ်ႉသင် (ပဵၼ် `nil`)။ ပေႃးႁဝ်းၸႂ်ႉ Optional၊ Swift တေၸွႆးႁႄႉၵင်ႈဢမ်ႇႁႂ်ႈ App Keyboard ႁဝ်း Crash မိူဝ်ႈႁဝ်းၵႂႃႇႁွင်ႉၸႂ်ႉၶေႃႈมูล ဢၼ်ဢမ်ႇပႆႇမီးၼၼ်ႉယဝ်ႉ။"**
