## Lesson 58: Working with Templates
**(လွၵ်းလၢႆးမၵ်းမၼ်ႈ တီႈဝၢင်းတူဝ်လိၵ်ႈ ၼိူဝ်ႁၢင်ႈ Template)**

### 1. ပိူင်ဝၢင်း Coordinate (X, Y)
ၼႂ်းလွၵ်းပၢႆးၼပ်ႉ (Graphics) ၼၼ်ႉ ႁဝ်းတေၸႂ်ႉ **X** လႄႈ **Y** တႃႇမၵ်းမၼ်ႈတီႈယူႇၶႃႈ:
* **X (Horizontal):** တႄႇၼပ်ႉတီႈ ၽၢႆႇသၢႆႉ (0) ၵႂႃႇၽၢႆႇၶႂႃ။
* **Y (Vertical):** တႄႇၼပ်ႉတီႈ ၽၢႆႇၼိူဝ် (0) လူင်းမႃးၽၢႆႇတႂ်ႈ။



တႃႇတေႁႂ်ႈ "ၸိုဝ်ႈလုၵ်ႈႁဵၼ်း" ယူႇၵၢင်ဝႂ်မၵ်းမၼ်ႈတႄႉတႄႉၼၼ်ႉ ႁဝ်းလူဝ်ႇႁူႉတၢင်းၵႂၢင်ႈ (Width) ၶွင်ႁၢင်ႈ ၸင်ႇတေၼပ်ႉသွၼ်ႇလႆႈၶႃႈယဝ်ႉ။

---

### 2. လွၵ်းလၢႆး Positioning (The Logic)
ၵၢၼ်ဝၢင်းလိၵ်ႈၼိူဝ် Template မီး 3 ၸၼ်ႉၼႆႉၶႃႈ:
1.  **Find Center:** ဢဝ်တၢင်းၵႂၢင်ႈႁၢင်ႈ (Image Width) မႃးၸႅၵ်ႇၵၼ်တင်း 2။
2.  **Anchor Point:** မၵ်းမၼ်ႈဝႃႈ ႁႂ်ႈတူဝ်လိၵ်ႈ ဢၼ်ယူႇဝႆႉၼႂ်းၵၢင် (Center) ၼၼ်ႉ မႃးတူႇၸွမ်း Coordinate ဢၼ်ႁဝ်းပၼ်ဝႆႉ။
3.  **Offset:** ၸၼ်လိၵ်ႈၶိုၼ်းၼိူဝ်/လူင်းတႂ်ႈ (Y-axis) ႁႂ်ႈမႅၼ်ႈၸွမ်းထႅဝ် ဢၼ်ဝႆႉပဝ်ႇဝႆႉၼၼ်ႉၶႃႈ။

---

### 3. တူဝ်ယၢင်ႇ Code (Concept of Placement)
ၼႂ်း Lesson ၼႆႉ ႁဝ်းတေတူၺ်း Logic မၼ်းၵွၼ်ႇၶႃႈၼႃ (Lesson 59 ၸင်ႇတေၸႂ်ႉ Library PIL တႃႇ Generate တူဝ်တႄႉၶႃႈ)။

```python
# ဝူၼ်ႉတူၺ်းဝႃႈ ႁၢင်ႈ Certificate ႁဝ်းမီး Width = 2000 px, Height = 1414 px
img_width = 2000
img_height = 1414

# တီႈယူႇ ဢၼ်ႁဝ်းၶႂ်ႈဝၢင်းၸိုဝ်ႈ (Center of the image)
center_x = img_width / 2  # = 1000
center_y = 700            # ဝၢင်းဝႆႉၵႄႈၵၢင် ၼပ်ႉလူင်းမႃးတီႈၼိူဝ်

print(f"Positioning Text at: ({center_x}, {center_y})")

# Logic: သင်ႁဝ်းဝၢင်းလိၵ်ႈလူၺ်ႈဢမ်ႇၸႂ်ႉ 'anchor', 
# လိၵ်ႈတေတႄႇတႅမ်ႈတီႈ 1000 သေ ယၢဝ်းၵႂႃႇၽၢႆႇၶႂႃ (တေဢမ်ႇယူႇၵၢင်)။
# ၵွပ်ႈၼၼ် ႁဝ်းလူဝ်ႇသင်ႇ 'anchor=mm' (Middle-Middle) ၼႂ်း Pillow ၶႃႈ။
```

### 4. ၶေႃႈပိုတ်ႇၸႅင်ႈ (Breakdown)
* **Pixels (px):** ၼႂ်းႁၢင်ႈ Template ႁဝ်းဢမ်ႇၼပ်ႉပဵၼ် Inches ဢမ်ႇၼၼ် cm၊ ႁဝ်းၼပ်ႉပဵၼ် **Pixels** ၶႃႈယဝ်ႉ။
* **Visual Balance:** တီႈၵၢၼ်ဝၢင်းလိၵ်ႈၼႆႉ ဢၼ်လွင်ႈယႂ်ႇသုတ်းပဵၼ် "လွင်ႈမိူၼ်ၵၼ်" (Symmetry) ၶႃႈ။ သင်ၸိုဝ်ႈလုၵ်ႈႁဵၼ်း ယၢဝ်းဢမ်ႇၼၼ် ပွတ်းၸိူင်ႉႁိုဝ်ၵေႃႈ ႁႂ်ႈမၼ်းယူႇၵၢင်တႃႇသေႇၶႃႈ။
* **Dynamic Text:** ယွၼ်ႉၸိုဝ်ႈလုၵ်ႈႁဵၼ်းၵူႈၵေႃႉ တၢင်းယၢဝ်းဢမ်ႇမိူၼ်းၵၼ်လႄႈ ႁဝ်းဢမ်ႇၸၢင်ႈမၵ်းမၼ်ႈ X ဢၼ်တႄႇတႅမ်ႈ (Start Point) လႆႈ။ ႁဝ်းလူဝ်ႇမၵ်းမၼ်ႈ **Center Point** ၵူၺ်းၶႃႈ။
