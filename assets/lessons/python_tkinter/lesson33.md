## Lesson 33: Introduction to APIs
**(လွၵ်းလၢႆးၵပ်းသိုပ်ႇတင်း Web လူၺ်ႈၵၢၼ်ၸႂ်ႉ API)**

### 1. API ပဵၼ်သင်? (What is an API?)
**API** တီႈယၢဝ်းပွင်ႇဝႃႈ **Application Programming Interface** ၶႃႈ။ ဝူၼ်ႉတူၺ်းဝႃႈ ႁဝ်းၵႂႃႇၼင်ႈၼႂ်းသႅင်ႇၶဝ်ႈမုၼ်း (Restaurant) ၼိုင်ႈတီႈၼႆ:
* **ၸဝ်ႈၵဝ်ႇ (The Client/Software):** ၵူၼ်းသင်ႇၵိၼ် ၶဝ်ႈမုၼ်း (Data)။
* **ၵူၼ်းႁပ်ႉၶႅၵ်ႇ (The API/Waiter):** ၵူၼ်းဢၼ်ဢဝ်ၶေႃႈသင်ႇၸဝ်ႈၵဝ်ႇ ၵႂႃႇပၼ်တီႈၼႂ်းသိုင်ႇၾႆး။
* **သိုင်ႇၾႆး (The Server):** တီႈဢၼ်မီးၶေႃႈမုၼ်း (Data) တင်းၼမ်သေ ႁဵတ်းၵၢၼ်ပၼ်ၸဝ်ႈၵဝ်ႇ။

မိူဝ်ႈၵူၼ်းတူမ်ႈႁုင် တီႈၼႂ်းသိုင်ႇၾႆး ႁဵတ်းၶဝ်ႈမုၼ်း (Data) ယဝ်ႉယဝ်ႉၼၼ်ႉ ၵူၼ်းႁပ်ႉၶႅၵ်ႇ (API) တေဢဝ်မႃးသူင်ႇပၼ်တီႈ ဢွင်ႈၸဝ်ႈၵဝ်ႇၶိုၼ်းၶႃႈယဝ်ႉ။



---

### 2. ပဵၼ်သင် Software ႁဝ်းလူဝ်ႇၸႂ်ႉ API?
ၼႂ်း Project **Muse Live Weather Dashboard** ၼၼ်ႉ ႁဝ်းဢမ်ႇၸၢင်ႈတႅမ်ႈ Code ႁႂ်ႈ Python ႁူႉၾိင်ႈၾႃႉႁင်းမၼ်းလႆႈၶႃႈ။
* ႁဝ်းလူဝ်ႇ API တႃႇၵႂႃႇထၢမ်တီႈ **OpenWeatherMap** (Server လူင်)။
* မၼ်းတေသူင်ႇၶေႃႈမုၼ်း **Temperature (လွင်ႈမႆႈ)** လႄႈ **Humidity (လွင်ႈပိူဝ်ႉယမ်း)** ဢၼ်ပဵၼ်ဝႆႉတေႃးလဵဝ်ႈ (Live Data) ၼၼ်ႉမႃးပၼ်ႁဝ်းၶႃႈ။

---

### 3. ပိူင်ဝၢင်း API Request (The Workflow)
မိူဝ်ႈႁဝ်းသူင်ႇ Request ၵႂႃႇၼၼ်ႉ ႁဝ်းလူဝ်ႇမီး 3 မဵဝ်းၶႃႈ:
1.  **URL (Address):** တီႈယူႇၶွင် Server (တူဝ်ယၢင်ႇ: `api.openweathermap.org/...`)
2.  **Parameters (Details):** ႁဝ်းလူဝ်ႇၶေႃႈမုၼ်းမိူင်းလႂ်? (City = Muse)
3.  **API Key (Password):** မၢႆလပ်ႉဢၼ် Server ပၼ်ႁဝ်းဝႆႉ တႃႇၼႄဝႃႈ ႁဝ်းပဵၼ်ၵူၼ်းၸႂ်ႉတိုဝ်း ဢၼ်မႅၼ်ႈၸွမ်းပိူင်ၶႃႈ။

---

### 4. လွင်ႈတေလႆႈမၢႆတွင်း (Key Points)
* **API Response:** ၶေႃႈမုၼ်းဢၼ် Server သူင်ႇၶိုၼ်းမႃးၼၼ်ႉ ၵမ်ႈၼမ်တေပဵၼ်လိၵ်ႈပိူင် **JSON** (ဢၼ်ႁဝ်းတေႁဵၼ်းၼႂ်း Lesson 34)။
* **Real-time:** ၶေႃႈမုၼ်းဢၼ်လႆႈမႃး ပဵၼ်ၶေႃႈမုၼ်း ဢၼ်လႅၵ်ႈလၢႆႈယူႇၵူႈမိူဝ်ႈၵူႈယၢမ်းၶႃႈယဝ်ႉ။
