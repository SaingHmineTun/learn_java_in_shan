
## **Java Basic 23**

#### **DO - WHILE LOOP**

Link - <https://youtu.be/vPLehgSFeY0>

မၼ်းတေ မိူၼ်ၵၼ်တင်း while loop ဝႆႉတၢင်းၼမ်။ တီႈဢမ်ႇမိူၼ်ၵၼ် ၼၼ်ႉတေႉ do while loop ၼႆႉ မၼ်းတေၸႂ်ႉတိုဝ်း keyword 2 ဢၼ်သေႁဵတ်ႉၵၢၼ် (do, while) ၊ မၼ်းဢဝ် body မၼ်းၼၼ်ႉ ယူႇဢွၼ်တၢင်း while statement လႄႈ ပေႃးၸႂ်ႉတိုဝ်းမၼ်းၼႆ မၼ်းပၼ်လႆႈၵႂၢမ်းမၼ်ႈဝႃႈ ၵဝ်တေႁဵတ်ႉၵၢၼ် တီႈဢေႇသုတ်ႉ 1 ပွၵ်ႈ ပဵၼ်ႁိုဝ်မႃးၵေႃႈယဝ်ႉၼႆယူႇ။

###### လၢႆးၸႂ်ႉတိုဝ်း

```
char c = 'a';
do {
    System.out.println(c);
    c ++;
} while (c <= 'z');
```
