## Lesson 21: Terminal vs. Shell (Command Prompt & PowerShell)

သင်ႁဝ်းတေ တႄႇတႅမ်ႈၶေႃႈသင်ႇ၊ ႁဝ်းလူဝ်ႇလႆႈႁူႉၸၵ်း "ၶိူင်ႈမိုဝ်း" ဢၼ်ႁဝ်းတေၸႂ်ႉၼၼ်ႉၵွၼ်ႇ။ 
ၼႂ်း Windows၊ ႁဝ်းမီး "ႁွင်ႈတႅမ်ႈလိၵ်ႈ" 2 မဵဝ်းယဝ်ႉ။



---

#### 1. What is a Terminal? (Terminal ပဵၼ်သင်?)

**Terminal** ၼႆႉ ပဵၼ် "ပၢင်ႇ" ဢမ်ႇၼၼ် "ႁွင်ႈ" ဢၼ်ႁဝ်းၸႂ်ႉတွၼ်ႈတႃႇတႅမ်ႈလိၵ်ႈသေ လၢတ်ႈတေႃႇၶွမ်ႇပိဝ်ႇတႃႇယဝ်ႉ။ 
မၼ်းပဵၼ် Window သီလမ် ဢၼ်မီးတူဝ်လိၵ်ႈသီၶၢဝ် ဢၼ်ပႂ်ႉႁပ်ႉၶေႃႈသင်ႇႁဝ်းၼၼ်ႉယဝ်ႉ။

---

#### 2. Command Prompt vs. PowerShell

ၼႂ်း Windows ၸဝ်ႈၵဝ်ႇတေႁၼ် 2 မဵဝ်း:

* **Command Prompt (cmd.exe):** ၼႆႉပဵၼ်ဢၼ်ၵဝ်ႇ (Legacy)။ မၼ်းငၢႆႈ၊ ဝႆး၊ လႄႈ ၸႂ်ႉတွၼ်ႈတႃႇၵၢၼ်ပိုၼ်ႉထၢၼ် မိူၼ်ၼင်ႇၵူတ်ႇထတ်း Internet ဢမ်ႇၼၼ် ပိုတ်ႇ Program။
* **PowerShell:** ၼႆႉပဵၼ်ဢၼ်မႂ်ႇ လႄႈ မီးဢႃႇၼႃႇၼမ်လိူဝ်။ မၼ်းၸၢင်ႈသင်ႇၵၢၼ်ယၢပ်ႇယၢပ်ႇလႆႈ။ Programmer ၶဝ်ၵမ်ႈၼမ်တေၸႂ်ႉ PowerShell ယဝ်ႉ။

---

#### 3. How to Open? (လၢႆးပိုတ်ႇၸႂ်ႉ)

ၸဝ်ႈၵဝ်ႇႁဵတ်းၸွမ်းလၢႆးၼႆႉလႆႈၵူႈၶိူင်ႈ:
1.  ၼဵၵ်းပၼ်ၼိူဝ် **Windows** (Start) တီႈ Keyboard။
2.  တႅမ်ႈလိၵ်ႈႁႃဝႃႈ `cmd` (တွၼ်ႈတႃႇ Command Prompt) ဢမ်ႇၼၼ် `powershell` (တွၼ်ႈတႃႇ PowerShell)။
3.  ၼဵၵ်း **Enter**။

---

#### 4. Understanding the "Prompt"

မိူဝ်ႈၸဝ်ႈၵဝ်ႇပိုတ်ႇမႃးယဝ်ႉ၊ ၸဝ်ႈၵဝ်ႇတေႁၼ်လိၵ်ႈမိူၼ်ၼင်ႇၼႆ:
`C:\Users\YourName>`

ဢၼ်ၼႆႉႁွင်ႉဝႃႈ **Prompt**။ မၼ်းတိုၵ်ႉလၢတ်ႈၼႄၸဝ်ႈၵဝ်ႇဝႃႈ:
* **C:** - ၸဝ်ႈၵဝ်ႇတိုၵ်ႉယူႇၼႂ်း Hard Drive လုၵ်ႈလႂ်။
* **\Users\YourName** - ၸဝ်ႈၵဝ်ႇတိုၵ်ႉယူႇၼႂ်း Folder လႂ် (တီႈယူႇၸဝ်ႈၵဝ်ႇယၢမ်းလဵဝ်)။
* **>** - မၼ်းတိုၵ်ႉပႂ်ႉထွမ်ႇၶေႃႈသင်ႇ (Command) ၸဝ်ႈၵဝ်ႇယူႇယဝ်ႉ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ **Terminal** ၼႆႉ ပဵၼ် **"ႁူင်းၾူၼ်း"** (Telephone Booth) ဢၼ်ၼိုင်ႈ:
1.  ၸဝ်ႈၵဝ်ႇၶဝ်ႈၵႂႃႇၼႂ်းႁူင်းၾူၼ်း (ပိုတ်ႇ Terminal)။
2.  ၸဝ်ႈၵဝ်ႇတေလႆႈ "ၼဵၵ်းမၢႆၾူၼ်း" (တႅမ်ႈ Command) တွၼ်ႈတႃႇလၢတ်ႈတေႃႇၶွမ်ႇပိဝ်ႇတႃႇ။
3.  သင်ၸဝ်ႈၵဝ်ႇၼဵၵ်းမၢႆၾူၼ်းၽိတ်း၊ မၼ်းတေလၢတ်ႈဝႃႈ "မၢႆၾူၼ်းၼႆႉဢမ်ႇမီး" (Command not recognized)။
4.  သင်မၢၼ်ႇမႅၼ်ႈ၊ ၶွမ်ႇပိဝ်ႇတႃႇတေႁဵတ်းၸွမ်းၼင်ႇ ၸဝ်ႈၵဝ်ႇသင်ႇယဝ်ႉ။


---

**ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း**

* **Terminal:** "ႁွင်ႈ" ဢမ်ႇၼၼ် Window ဢၼ်ႁဝ်းၸႂ်ႉတႅမ်ႈလိၵ်ႈၶဝ်ႈၵႂႃႇ (Command Prompt တီႈ Windows၊ Terminal တီႈ Mac/Linux)။
* **Shell:** "ၵူၼ်းပွင်ႇၵႂၢမ်း" ဢၼ်ယူႇၼႂ်း Terminal သေ ဢဝ်ၶေႃႈၵႂၢမ်းႁဝ်း ၵႂႃႇသင်ႇ Kernel ႁႂ်ႈႁဵတ်းၵၢၼ်။
* **Prompt:** တူဝ်လိၵ်ႈတီႈႁူဝ်ထႅဝ် (မိူၼ်ၼင်ႇ `C:\>`) ဢၼ်ၼႄဝႃႈ ၶွမ်ႇပိဝ်ႇတႃႇ "တိုၵ်ႉပႂ်ႉ" ၶေႃႈသင်ႇႁဝ်းယူႇ။
