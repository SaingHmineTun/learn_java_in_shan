## Lesson 16: Processes & Threads (How the OS Multitasks)

မိူဝ်ႈၸဝ်ႈၵဝ်ႇၸႂ်ႉၶွမ်ႇပိဝ်ႇတႃႇ၊ ၸဝ်ႈၵဝ်ႇၸၢင်ႈ "ပိုတ်ႇၵႂၢမ်း" ထွမ်ႇၵႂႃႇ၊ တိုၵ်ႉ "တႅမ်ႈလိၵ်ႈ" ၵႂႃႇ၊ လႄႈ "လူ Website" ၵႂႃႇၸွမ်းၵမ်းလဵဝ်ၵၼ်လႆႈ။ OS ၸတ်းၵၢၼ်လွင်ႈၼႆႉၸိူင်ႉႁိုဝ်? ႁဝ်းမႃးတူၺ်းလွင်ႈ **Processes** လႄႈ **Threads** ယဝ်ႉ။



---

#### 1. What is a Process? (Process ပဵၼ်သင်?)

**Process** ၼႆႉ ပဵၼ် Program ဢၼ် "တိုၵ်ႉလႅၼ်ႈယူႇ" (Running program) ၼၼ်ႉယဝ်ႉ။
* မိူဝ်ႈ Program ၼိုင်ႈဢၼ်ယူႇၼႂ်း Storage (HDD/SSD)၊ ႁဝ်းႁွင်ႉမၼ်းဝႃႈ Program။
* ၵူၺ်းၵႃႈ မိူဝ်ႈ OS ထုတ်ႇဢဝ်မၼ်းမႃးဝၢင်းၼႂ်း RAM သေ သင်ႇႁႂ်ႈ CPU ႁဵတ်းၵၢၼ်ပၼ်ယဝ်ႉၼၼ်ႉ၊ မၼ်းတေပဵၼ် **Process** ယဝ်ႉ။
* **Isolation:** Process ၼိုင်ႈဢၼ် တေမီး "ၶွပ်ႇပႅၼ်" (Boundary) ႁင်းၵူၺ်းမၼ်း။ မၼ်းတေဢမ်ႇၵႂႃႇယုင်ႈတူဝ်လိၵ်ႈ ဢမ်ႇၼၼ် ၶေႃႈမုၼ်းၼႂ်း Process တၢင်ႇဢၼ်လႆႈ။

---

#### 2. What is a Thread? (Thread ပဵၼ်သင်?)

**Thread** ၼႆႉ ပဵၼ် "ၵူၼ်းႁဵတ်းၵၢၼ်" လဵၵ်ႉလဵၵ်ႉ ဢၼ်မီးဝႆႉၼႂ်း Process ၼိုင်ႈဢၼ်ယဝ်ႉ။
* Process ၼိုင်ႈဢၼ် ၸၢင်ႈမီး Thread လၢႆလၢႆဢၼ် ႁဵတ်းၵၢၼ်ၸွမ်းၵၼ်။
* **တူဝ်ယၢင်ႇ:** ၼႂ်း Program တႅမ်ႈလိၵ်ႈ (Word Processor) ၼိုင်ႈဢၼ်:
    * **Thread 1:** ၼႄတူဝ်လိၵ်ႈ ဢၼ်ၸဝ်ႈၵဝ်ႇတိုၵ်ႉတႅမ်ႈ။
    * **Thread 2:** ၵူတ်ႇထတ်းပၼ်ဝႃႈ ၸဝ်ႈၵဝ်ႇတႅမ်ႈလိၵ်ႈၽိတ်း (Spell Check) ႁႃႉ?
    * **Thread 3:** ၸွႆႈ Save လိၵ်ႈဝႆႉပၼ် တႃႇသေႇ (Auto-save)။



---

#### 3. Scheduling (ၵၢၼ်ၸႅၵ်ႇယၢမ်းပၼ် CPU)

CPU ႁဝ်းၼႆႉ မၼ်းဝႆးမၢၵ်ႇၼႃႇ။ OS မီးၼႃႈၵၢၼ်ႁဵတ်း **Scheduling**—မၼ်းတေလႅၵ်ႈလၢႆႈပၼ် Program ၵူႈဢၼ် ႁႂ်ႈလႆႈၸႂ်ႉ CPU ၵမ်းဢိတ်းၵမ်းဢိတ်း ဝႆးမၢၵ်ႇၼႃႇ တေႃႇပေႃးၵူၼ်းႁဝ်း ဝူၼ်ႉဝႃႈမၼ်း "လႅၼ်ႈၸွမ်းၵၼ်" (Simultaneously) ယူႇၼၼ်ႉယဝ်ႉ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ **OS** ပဵၼ် **"ၸဝ်ႈလၢၼ်ႉတၢင်းၵိၼ်"**:
1.  **Process:** ပဵၼ် "တႅပ်းၶဝ်ႈ" (Table) ၼိုင်ႈဝိုင်း။ ၶႅၵ်ႇဝိုင်းၼိုင်ႈ တေဢမ်ႇၵႂႃႇၵိၼ်ၶဝ်ႈၼႂ်းပၢၼ်ႇ ၶႅၵ်ႇထႅင်ႈဝိုင်းၼိုင်ႈ။
2.  **Thread:** ပဵၼ် "ၵူၼ်းၼႂ်းဝိုင်း" (People at the table)။ ၶဝ်ၵိၼ်ၶဝ်ႈသွႆးၸွမ်းၵၼ်၊ လၢတ်ႈၵႂၢမ်းၸွမ်းၵၼ်၊ လႄႈ ၸႂ်ႉၶႅပ်းၸေႈ (Napkin) ႁူမ်ႈၵၼ်ၼိူဝ်တႅပ်းလဵဝ်ၵၼ်ယဝ်ႉ။
3.  **Scheduling:** ပဵၼ် "ၵူၼ်းပၼ်ႇၶဝ်ႈ" ဢၼ်ၵႂႃႇသူင်ႇတၢင်းၵိၼ် ပၼ်ၵူႈဝိုင်း ဝႆးဝႆး ႁႂ်ႈၵူႈၵေႃႉလႆႈၵိၼ်ၸွမ်းၵၼ်ၼၼ်ႉယဝ်ႉ။
