## Lesson 7: Binary (How Computers Count with 1s and 0s)

ၼႂ်းၵၢၼ်တႅမ်ႈ "Program" ၼၼ်ႉ ၸဝ်ႈၵဝ်ႇလူဝ်ႇလႆႈႁူႉဝႃႈ ၶွမ်ႇပိဝ်ႇတႃႇၼႆႉ မၼ်းဢမ်ႇပွင်ႇၸႂ်မၢႆၼပ်ႉ 0 တေႃႇ 9 မိူၼ်ၵူၼ်းႁဝ်း။ မၼ်းပွင်ႇၸႂ် **0** လႄႈ **1** ၵူၺ်း။ ႁဝ်းႁွင်ႉ လွၵ်းၵၢၼ်ၼပ်ႉဢၼ်ၼႆႉဝႃႈ **Binary** (မၢႆၼပ်ႉသွင်တူဝ်) ယဝ်ႉ။



---

## ## 1. Why only 0 and 1? (ၵွပ်ႈသင်မီးၵူၺ်း 0 လႄႈ 1?)

မိူၼ်ၼင်ႇႁဝ်းလဵပ်ႈႁဵၼ်းမႃးၼႂ်း Lesson 6 ၼၼ်ႉယဝ်ႉ၊ CPU မီးသဝိတ်ႉၾႆးၾႃႉ (Transistors) ပဵၼ်လၢၼ်ႉလၢၼ်ႉဢၼ်။

* **0** မၢႆထိုင် **Off** (ပိၵ်ႉၾႆး / ဢမ်ႇမီးၾႆး)
* **1** မၢႆထိုင် **On** (ပိုတ်ႇၾႆး / မီးၾႆး)

ၶွမ်ႇပိဝ်ႇတႃႇဢဝ် "သဝိတ်ႉ" ၸိူဝ်းၼႆႉမႃးလဵၼ်ႈႁူမ်ႈၵၼ်သေ ၼပ်ႉဢၢၼ်ႇပဵၼ်မၢႆၼပ်ႉလူင်မႃးယဝ်ႉ။ ၶွမ်ႇပိဝ်ႇတႃႇ ဢမ်ႇႁူႉဝႃႈ "2" ၼႆႉပဵၼ်သင်၊ မၼ်းႁူႉဝႃႈ "မီးၾႆး ဢမ်ႇၼၼ် ဢမ်ႇမီးၾႆး" ၼႆၵူၺ်း။

---

## ## 2. How to Count in Binary (လၢႆးၼပ်ႉ Binary)

ၼႂ်းလွၵ်းၵၢၼ်ၵူၼ်း (Decimal)၊ ႁဝ်းမီးမၢႆၼပ်ႉ 10 တူဝ် (0-9)။ မိူဝ်ႈတဵမ် 9 ယဝ်ႉ ႁဝ်းၸင်ႇထႅမ်ထႅဝ်မႂ်ႇပဵၼ် 10။
ၵူၺ်းၵႃႈၼႂ်း **Binary**၊ မၼ်းမီး 0 လႄႈ 1 ၵူၺ်း။ ၵွပ်ႈၼၼ် မၼ်းတေလႅၵ်ႈလၢႆႈၵႂႃႇၼင်ႇၼႆ:

| မၢႆလႅပ်ႈၵူၼ်း (Decimal) | မၢႆလႅပ်ႈၶွမ်ႇ (Binary) | တီႈပွင်ႇ |
| :--- | :--- | :--- |
| 0 | **0** | ပိၵ်ႉ (Off) |
| 1 | **1** | ပိုတ်ႇ (On) |
| 2 | **10** | ပိုတ်ႇ၊ ပိၵ်ႉ |
| 3 | **11** | ပိုတ်ႇ၊ ပိုတ်ႇ |
| 4 | **100** | ပိုတ်ႇ၊ ပိၵ်ႉ၊ ပိၵ်ႉ |

> **Programmer Tip:** ၸဝ်ႈၵဝ်ႇဢမ်ႇလူဝ်ႇတွင်းလွၵ်းၵၢၼ်ၼပ်ႉၼႆႉ တင်းသဵင်ႈၵေႃႈလႆႈ။ ၵူၺ်းၵႃႈ လူဝ်ႇပွင်ႇၸႂ်ဝႃႈ **"မၢႆၼပ်ႉၵူႈဢၼ်"** ဢၼ်ၸဝ်ႈၵဝ်ႇတႅမ်ႈၼႂ်း Code ၼၼ်ႉ၊ ၶွမ်ႇပိဝ်ႇတႃႇတေလႅၵ်ႈပဵၼ် 1 လႄႈ 0 တင်းသဵင်ႈယဝ်ႉ။

## ## 3. Bits: The Smallest Piece of Data (Bit ပဵၼ်သင်?)

မၢႆၼပ်ႉ ဢၼ်ၵူၺ်းတေမီးဝႆႉ (0 ဢမ်ႇၼၼ် 1) ၼၼ်ႉ ႁဝ်းႁွင်ႉဝႃႈ **Bit** ယဝ်ႉ။ မၼ်းပဵၼ်ၶေႃႈမုၼ်း ဢၼ်လဵၵ်ႉသုတ်း ၼႂ်းလူၵ်ႈၶွမ်ႇပိဝ်ႇတႃႇယဝ်ႉ။

* **Example:** သင်ဝႃႈၸဝ်ႈၵဝ်ႇမီး Binary `1011` ၼႆ၊ မၼ်းမၢႆထိုင်ဝႃႈ ၸဝ်ႈၵဝ်ႇမီးၶေႃႈမုၼ်း **4 Bits** ယဝ်ႉ။

---

### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇမီး **"ၾႆးသႅင်"** (Flashlight) 3 တူဝ်:

1.  သင်ပိုတ်ႇဝႆႉတင်း 3 တူဝ် (**111**), ၼႂ်း Binary မၼ်းတေမီးၵႃႈၶၼ်ပဵၼ် **7**။
2.  သင်ပိၵ်ႉဝႆႉတင်း 3 တူဝ် (**000**), မၼ်းမၢႆထိုင် **0**။
3.  လွၵ်းၵၢၼ်ၼႆႉယဝ်ႉ ဢၼ်ၶွမ်ႇပိဝ်ႇတႃႇၸႂ်ႉသေ ဝူၼ်ႉသွၼ်ႇပၼ်ႁဝ်း။
