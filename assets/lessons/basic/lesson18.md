## Lesson 18: User Accounts & Permissions (Security and Privacy)

ၼႂ်း Lesson ၼႆႉ ႁဝ်းတေမႃးတူၺ်းလွင်ႈ OS ႁႄႉၵင်ႈပၼ် "လွင်ႈလပ်ႉ" (Privacy) လႄႈ "လွင်ႈၵတ်းယဵၼ်" (Security) ၸိူင်ႉႁိုဝ်။ OS မီးၼႃႈၵၢၼ်မၵ်းမၼ်ႈဝႃႈ ၽႂ်ၸၢင်ႈႁဵတ်းသင်လႆႈ လႄႈ ၽႂ်ၸၢင်ႈတူၺ်း File လႂ်လႆႈၼၼ်ႉယဝ်ႉ။



---

#### 1. User Accounts (သဵၼ်ႈမၢႆၵူၼ်းၸႂ်ႉ)

OS ၸတ်းၵၢၼ်လွင်ႈၵူၼ်းၸႂ်ႉ လူၺ်ႈၵၢၼ်ၸႅၵ်ႇ "Account"။
* **Administrator (Admin):** ပဵၼ် "ၸဝ်ႈၶွင်" ဢၼ်မီးဢႃႇၼႃႇသုတ်း။ ၸၢင်ႈလူ၊ တႅမ်ႈ၊ ယႃႉပႅတ်ႈ File လႆႈၵူႈဢၼ် လႄႈ ၸၢင်ႈသႂ်ႇ Program မႂ်ႇမႂ်ႇလႆႈ။
* **Standard User:** ပဵၼ် "ၵူၼ်းၸႂ်ႉၵူႈၼ်" ဢၼ်ၸၢင်ႈၸႂ်ႉ Program လႆႈ ၵူၺ်းၵႃႈ ဢမ်ႇၸၢင်ႈၵႂႃႇလႅၵ်ႈလၢႆႈ "ၾိင်ႈၵၢၼ်" (Settings) ၶွင် OS လႆႈ။

---

#### 2. Permissions (သႅတ်ႉဢႃႇၼႃႇ)

ၼႂ်း File System၊ File ၵူႈဢၼ်လႄႈ Folder ၵူႈဢၼ် မီး "သႅတ်ႉ" (Permissions) ဢၼ်မၵ်းမၼ်ႈဝႆႉ 3 မဵဝ်း:

1.  **Read (R):** တူၺ်းလႆႈ (ပိၼ်ႇတူၺ်းလႆႈ ၵူၺ်းၵႃႈ လႅၵ်ႈလၢႆႈဢမ်ႇလႆႈ)။
2.  **Write (W):** တႅမ်ႈလႆႈ/လႅၵ်ႈလၢႆႈလႆႈ (ၸၢင်ႈ Edit ၶေႃႈမုၼ်း လႄႈ Save လႆႈ)။
3.  **Execute (X):** ပိုတ်ႇလႅၼ်ႈလႆႈ (ၸႂ်ႉတွၼ်ႈတႃႇ Program ဢမ်ႇၼၼ် Script)။

> **Programmer Tip:** မိူဝ်ႈၸဝ်ႈၵဝ်ႇတႅမ်ႈ Code သေ ၶတ်းၸႂ်တေ "Save" File ၼၼ်ႉ၊ သင် OS ႁၼ်ဝႃႈၸဝ်ႈၵဝ်ႇဢမ်ႇမီး **Write Permission**၊ မၼ်းတေႁႄႉဝႆႉသေ ၼႄလိၵ်ႈဝႃႈ "Access Denied" ယဝ်ႉ။

---

#### 3. Why it Matters? (ၵွပ်ႈသင်မၼ်းလမ်ႇလွတ်ႈ?)

သင်ဢမ်ႇမီး User Permissions:
* Virus တေၸၢင်ႈၵႂႃႇယႃႉပႅတ်ႈ File System ၸဝ်ႈၵဝ်ႇလႆႈငၢႆႈငၢႆႈ။
* ၵူၼ်းတၢင်ႇၵေႃႉ ဢၼ်ၸႂ်ႉၶွမ်ႇပိဝ်ႇတႃႇလဵဝ်ၵၼ် တေၸၢင်ႈမႃးပိုတ်ႇတူၺ်း "ၶႅပ်းႁၢင်ႈလွင်ႈလပ်ႉ" ၸဝ်ႈၵဝ်ႇလႆႈ။
* OS ၸဝ်ႈၵဝ်ႇ ၸၢင်ႈလႅၼ်ႈဢမ်ႇမၼ်ႈၼိမ် ယွၼ်ႉမီးၵူၼ်းၵႂႃႇလႅၵ်ႈလၢႆႈ File ပိုၼ်ႉထၢၼ်မၼ်း။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ **OS** ပဵၼ် **"ႁိူၼ်းလူင်"** ဢၼ်မီးႁွင်ႈၼမ်ၼမ်:
1.  **User Account:** ပဵၼ် "ၵူၼ်းယူႇၼႂ်းႁိူၼ်း"။ ၵူႈၵေႃႉမီး "ၵူၵ်း" (Key) ႁင်းၵူၺ်း။
2.  **Admin:** ပဵၼ် "ၸဝ်ႈႁိူၼ်း" ဢၼ်မီး **Master Key** ပိုတ်ႇလႆႈၵူႈႁွင်ႈ။
3.  **Permissions:**
    * ႁွင်ႈၼွၼ်းၸဝ်ႈၵဝ်ႇ: ၸဝ်ႈၵဝ်ႇမီး **Read & Write** (ၼွၼ်းလႆႈ၊ သိမ်းၶူဝ်းလႆႈ)။
    * ႁွင်ႈၼွၼ်းပိူၼ်ႈ: ၸဝ်ႈၵဝ်ႇ **ဢမ်ႇမီး Permission** (ၶဝ်ႈဢမ်ႇလႆႈ)။
    * ပပ်ႉသဵၼ်ႈမၢႆႁိူၼ်း: ၸဝ်ႈၵဝ်ႇမီး **Read Only** (တူၺ်းလႆႈ ၵူၺ်းၵႃႈ ၵႂႃႇၶိတ်းၶႅၼ်းလႅၵ်ႈလၢႆႈဢမ်ႇလႆႈ)။
