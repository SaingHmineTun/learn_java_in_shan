## Lesson 22: Navigation (Moving Through Folders)

မိူဝ်ႈႁဝ်းၸႂ်ႉ Mouse၊ ႁဝ်းၼဵၵ်း Double-click သေ ၶဝ်ႈၼႂ်း Folder။ ၵူၺ်းၵႃႈ ၼႂ်း CLI၊ ႁဝ်းတေလႆႈၸႂ်ႉ "ၶေႃႈလိၵ်ႈ" တွၼ်ႈတႃႇ "ပႆ" ၵႂႃႇမႃးၼႂ်းၶွမ်ႇပိဝ်ႇတႃႇႁဝ်းယဝ်ႉ။



---

#### 1. dir (Directory - တူၺ်းဝႃႈမီးသင်ဝႆႉ?)

ၶေႃႈသင်ႇဢွၼ်တၢင်းသုတ်း ဢၼ်ၸဝ်ႈၵဝ်ႇလူဝ်ႇမေႃၵေႃႈ ပဵၼ် `dir` ယဝ်ႉ။
* **ၸႂ်ႉတွၼ်ႈတႃႇ:** တူၺ်းသဵၼ်ႈမၢႆ File လႄႈ Folder တင်းသဵင်ႈ ဢၼ်မီးယူႇၼႂ်းတီႈဢၼ်ၸဝ်ႈၵဝ်ႇတိုၵ်ႉယူႇၼၼ်ႉ။
* **လၢႆးတႅမ်ႈ:** တႅမ်ႈ `dir` သေ ၼဵၵ်း **Enter**။

---

#### 2. cd (Change Directory - ၶၢႆႉတီႈယူႇ)

သင်ၸဝ်ႈၵဝ်ႇႁၼ် Folder ၼိုင်ႈဢၼ် (တူဝ်ယၢင်ႇ: `Documents`) သေ လူဝ်ႇၶဝ်ႈၵႂႃႇၼႂ်းမၼ်း၊ ႁဝ်းၸႂ်ႉ `cd`။

* **လၢႆးၶဝ်ႈ:** `cd Documents` (မၼ်းတေဢွၼ်ၸဝ်ႈၵဝ်ႇၶဝ်ႈၵႂႃႇၼႂ်း Folder Documents)။
* **လၢႆးဢွၵ်ႇ:** `cd ..` (ၼဵၵ်းၸႅတ်ႉသွင်ဢၼ် တွၼ်ႈတႃႇ "ထွၼ်လင်" ဢွၵ်ႇမႃးၸၼ်ႉၼိုင်ႈ)။



---

#### 3. cls (Clear Screen - ပတ်းၵွတ်ႇၸေႃး)

မိူဝ်ႈၸဝ်ႈၵဝ်ႇတႅမ်ႈလိၵ်ႈၼမ်ပွတ်းသေ ၸေႃးလမ်ၼၼ်ႉ ယုင်ႈၵႂႃႇတင်းသဵင်ႈယဝ်ႉၼႆ၊ ၸႂ်ႉ `cls` သေ ပတ်းၵွတ်ႇႁႂ်ႈမၼ်းၵဵင်ႈမိူၼ်မႂ်ႇလႆႈယဝ်ႉ။

---

#### 4. The Tab Key Trick (လွၵ်းလၢႆးဝႆး)

ၸဝ်ႈၵဝ်ႇဢမ်ႇလူဝ်ႇတႅမ်ႈၸိုဝ်ႈ Folder တင်းသဵင်ႈၵေႃႈလႆႈ။
* တူဝ်ယၢင်ႇ: သင်ၸဝ်ႈၵဝ်ႇတႅမ်ႈ `cd Doc` သေ ၼဵၵ်းပုမ်ႇ **Tab** တီႈ Keyboard၊ CLI တေတႅမ်ႈလိၵ်ႈဢၼ်ၵူဝ်ႇ `Documents` ပၼ်ၸဝ်ႈၵဝ်ႇႁင်းၵူၺ်း (Auto-complete) ယဝ်ႉ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉယူႇၼႂ်း **"ႁိူၼ်း"**:
1.  **dir:** မိူၼ်ၼင်ႇၸဝ်ႈၵဝ်ႇ မူင်ၼိုင်တူၺ်းၼႂ်းႁွင်ႈဝႃႈ မီးၽၵ်းတူလႂ်ၽတ်းဝႆႉလၢႆဢၼ်။
2.  **cd Documents:** မိူၼ်ၼင်ႇၸဝ်ႈၵဝ်ႇ ယၢင်ႈတိၼ်ၶဝ်ႈၵႂႃႇၼႂ်း "ႁွင်ႈၼွၼ်း"။
3.  **cd ..:** မိူၼ်ၼင်ႇၸဝ်ႈၵဝ်ႇ ယၢင်ႈတိၼ်ဢွၵ်ႇမႃးတီႈ "ႁွင်ႈၼင်ႇလဵၼ်ႈ" ၶိုၼ်းယဝ်ႉ။
