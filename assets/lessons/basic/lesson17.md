## Lesson 17: Memory Management (Protecting the RAM)

ၼႂ်း Lesson ၼႆႉ ႁဝ်းတေမႃးတူၺ်းလွင်ႈ OS ၸတ်းၵၢၼ် RAM ၸိူင်ႉႁိုဝ်။ RAM ၼႆႉမီးတီႈၵႅပ်ႈ၊ ၵွပ်ႈၼၼ် OS တေလႆႈပဵၼ် "ၵူၼ်းပႂ်ႉယၢမ်း" ဢၼ်ၵတ်ႉၶႅၼ်ႇ တွၼ်ႈတႃႇႁႂ်ႈ Program ၵူႈဢၼ်လႅၼ်ႈလႆႈလီလီယဝ်ႉ။



---

#### 1. Space Allocation (ၵၢၼ်ပၼ်တီႈယူႇ)

မိူဝ်ႈၸဝ်ႈၵဝ်ႇပိုတ်ႇ Program မႂ်ႇၼိုင်ႈဢၼ်၊ OS မီးၼႃႈၵၢၼ်:
* **Find Space:** ႁႃတူၺ်းဝႃႈ ၼႂ်း RAM မီးတီႈပဝ်ႇတီႈလႂ်?
* **Assign:** ပၼ်လွၵ်း RAM ၼၼ်ႉၸူး Program ၼၼ်ႉ။
* **Protect:** ႁႄႉၵင်ႈဝႆႉ ဢမ်ႇႁႂ်ႈ Program တၢင်ႇဢၼ် မႃးတႅမ်ႈၶေႃႈမုၼ်း သႂ်ႇၼႂ်းလွၵ်း RAM ဢၼ်မၼ်းပၼ်ဝႆႉၼၼ်ႉ။ သင်ဝႃႈ Program A ၵႂႃႇယုင်ႈ RAM ၶေႃႈမုၼ်း Program B၊ ၶွမ်ႇပိဝ်ႇတႃႇတေ "လူႉ" (Crash) ၵမ်းလဵဝ်ယဝ်ႉ။

---

#### 2. Virtual Memory (RAM ပလွတ်ႉ)

သင်ဝႃႈ RAM ၸဝ်ႈၵဝ်ႇ "တဵမ်" (Full) ၵႂႃႇလေႃ? OS မီးလၢႆးၵႄႈလိတ်ႈဢၼ်ၼိုင်ႈ ဢၼ်ႁွင်ႉဝႃႈ **Virtual Memory**။
* မၼ်းတေၸႂ်ႉ "တီႈပဝ်ႇ" ၼႂ်း **Storage (HDD/SSD)** သေ ႁဵတ်းပဵၼ် RAM ပလွတ်ႉ။
* မၼ်းတေထုတ်ႇဢဝ် Program ဢၼ်ၸဝ်ႈၵဝ်ႇ "တိုၵ်ႉဝၢင်းဝႆႉ" (ဢမ်ႇတိုၵ်ႉၸႂ်ႉ) ၼၼ်ႉ ၵႂႃႇသိမ်းဝႆႉၼႂ်း Storage ၵွၼ်ႇ၊ ယဝ်ႉၵေႃႈ ဢဝ်လွၵ်း RAM ၼၼ်ႉ မႃးပၼ် Program ဢၼ်ၸဝ်ႈၵဝ်ႇတိုၵ်ႉၸႂ်ႉယူႇယၢမ်းလဵဝ်ယဝ်ႉ။

---

#### 3. Garbage Collection (ၵၢၼ်ၵဵပ်းယုၵ်းယၢၵ်း)

မိူဝ်ႈၸဝ်ႈၵဝ်ႇ "ပိၵ်ႉ" (Close) Program ၼိုင်ႈဢၼ်၊ OS တေလႆႈႁဵတ်းၼႃႈၵၢၼ် "ပတ်းၽဵဝ်ႈ" ယဝ်ႉ။
* မၼ်းတေၵႂႃႇ "ယႃႉပႅတ်ႈ" (Free up) ၶေႃႈမုၼ်းၸိူဝ်းဢၼ်ယူႇၼႂ်း RAM ၼၼ်ႉ။
* လွင်ႈၼႆႉ ႁဵတ်းႁႂ်ႈ RAM မီးတီႈပဝ်ႇ တွၼ်ႈတႃႇ Program တၢင်ႇဢၼ်တေမႃးၸႂ်ႉထႅင်ႈယဝ်ႉ။



---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ **RAM** ၼႆႉ ပဵၼ် **"ၽိူၼ်ႁဵတ်းၵၢၼ်"** (Work Desk) လဵၵ်ႉလဵၵ်ႉၼိုင်ႈဢၼ်:
1.  **OS:** ပဵၼ် "ၵူၼ်းၸတ်းၵၢၼ်" ဢၼ်မႃးဝၢင်းပပ်ႉ လႄႈ ၶိူင်ႈတႅမ်ႈလိၵ်ႈပၼ်ၸဝ်ႈၵဝ်ႇ။
2.  သင်ၸဝ်ႈၵဝ်ႇ မီးၵၢၼ်ၼမ်ႁႅင်းလႄႈ သင်ဝႃး ၽိူၼ်ႁဵတ်းၵၢၼ်တေတဵမ်။ **Virtual Memory** ၵေႃႈမိူၼ်ၼင်ႇ ၵူၼ်းၸတ်းၵၢၼ်ၼၼ်ႉ ဢဝ်ပပ်ႉဢၼ်ၸဝ်ႈၵဝ်ႇဢမ်ႇတိုၵ်ႉလူၼၼ်ႉ ၵႂႃႇဝၢင်းဝႆႉၼိူဝ် **"သဵင်ႇ" (Storage)** တွၼ်ႈတႃႇႁႂ်ႈမီးတီႈၵွၼ်ႇယဝ်ႉ။
3.  မိူဝ်ႈၸဝ်ႈၵဝ်ႇႁဵတ်းၵၢၼ်ယဝ်ႉ၊ မၼ်းတေမႃးဢဝ်ယုၵ်းယၢၵ်းၸိူဝ်းၼၼ်ႉ ၵႂႃႇထိုင်ႈပႅတ်ႈ (**Garbage Collection**) တွၼ်ႈတႃႇႁႂ်ႈမီး တီႈယူႇပဝ်ႇယဝ်ႉ။

---

**ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း**

* **Virtual Memory:** ၵၢၼ်ဢဝ်တီႈယူႇၼႂ်း Hard Drive မႃးၸႂ်ႉတၢင် RAM မိူဝ်ႈ RAM တဵမ်။
* **Memory Leak:** ပၼ်ႁႃ ဢၼ် App ၼိုင်ႈဢၼ် ဢဝ် RAM ဝႆႉသေ ဢမ်ႇပွႆႇၶိုၼ်း၊ ႁဵတ်းႁႂ်ႈၶွမ်ႇပိဝ်ႇတႃႇၸဵဝ်းတဵမ်။
* **Swapping:** ၵၢၼ်လႅၵ်ႈလၢႆႈၶေႃႈမုၼ်း ၼႂ်းၵႄႈ RAM လႄႈ Hard Drive တွၼ်ႈတႃႇၸတ်းၵၢၼ် တီႈယူႇ။
