## Lesson 12: Files and Formats (How Data is Packaged)

တေႃႇထိုင်ယၢမ်းလဵဝ် ႁဝ်းႁူႉယဝ်ႉဝႃႈ ၶွမ်ႇပိဝ်ႇတႃႇမီး Bits, Bytes, Strings, လႄႈ Pixels။ ၵူၺ်းၵႃႈ ၶေႃႈမုၼ်းၸိူဝ်းၼႆႉ မၼ်းတေဢမ်ႇၸၢင်ႈယူႇၶေႃၶေႃ။ မၼ်းလူဝ်ႇလႆႈမီး "ထူင်" တွၼ်ႈတႃႇသိမ်းဝႆႉ ဢၼ်ႁဝ်းႁွင်ႉဝႃႈ **File** (ၽၢႆႇ) ၼၼ်ႉယဝ်ႉ။



---

#### 1. What is a File? (File ပဵၼ်သင်?)

**File** ၼႆႉ ပဵၼ်ၸုမ်းၶေႃႈမုၼ်း (Collection of data) ဢၼ်မီးၸိုဝ်ႈသေ သိမ်းဝႆႉၼႂ်း **Storage** (Hard Drive/SSD) ယဝ်ႉ။ ၶွမ်ႇပိဝ်ႇတႃႇတေတူၺ်း "ႁၢင်ႈၾၢင်" (Format) မၼ်းသေ ၸင်ႇတေႁူႉဝႃႈ လူဝ်ႇပိုတ်ႇမၼ်းၸိူင်ႉႁိုဝ်။

---

#### 2. File Extensions (မၢႆသိုပ်ႇၸိုဝ်ႈၽၢႆႇ)

တွၼ်ႈတႃႇ ႁႂ်ႈၶွမ်ႇပိဝ်ႇတႃႇႁူႉဝႃႈ File ဢၼ်ၼႆႉ ပဵၼ်လိၵ်ႈ၊ ပဵၼ်ႁၢင်ႈ၊ ဢမ်ႇၼၼ် ပဵၼ်ၵႂၢမ်း၊ ႁဝ်းၸႂ်ႉ **File Extension** ဢၼ်မီးဝႆႉတီႈလင် "ၸိုဝ်ႈၽၢႆႇ" ၼၼ်ႉယဝ်ႉ။

* **.txt / .docx:** သိမ်းၶေႃႈမုၼ်းပဵၼ် **Text** (လိၵ်ႈ)။
* **.jpg / .png:** သိမ်းၶေႃႈမုၼ်းပဵၼ် **Pixels** (ႁၢင်ႈ)။
* **.mp3:** သိမ်းၶေႃႈမုၼ်းပဵၼ် **Sound waves** (သဵင်)။
* **.py:** ၼႆႉပဵၼ် File တွၼ်ႈတႃႇသိမ်း **Python Code** (ၶေႃႈသင်ႇၶွမ်ႇပိဝ်ႇတႃႇ)။

---

#### 3. Metadata (ၶေႃႈမုၼ်း လွင်ႈၶေႃႈမုၼ်း)

ၼႂ်း File ၵူႈဢၼ်၊ မၼ်းဢမ်ႇၵူၺ်းမီးၶေႃႈမုၼ်းဢၼ်ႁဝ်းတႅမ်ႈဝႆႉၵူၺ်း။ မၼ်းမီးထႅင်ႈၶေႃႈမုၼ်းလွင်ႈ "ပိုၼ်း" ၽၢႆႇဢၼ်ၼၼ်ႉ ဢၼ်ႁွင်ႉဝႃႈ **Metadata**။

* **တူဝ်ယၢင်ႇ:**
    * File ၼႆႉ တႄႇၵေႃႇသၢင်ႈမိူဝ်ႈဝၼ်းထိလၢႆ? (Created Date)
    * ၸိုဝ်ႈၵူၼ်းတႅမ်ႈမၼ်းပဵၼ်ၽႂ်? (Author)
    * တၢင်းယႂ်ႇမၼ်းမီးလၢႆ KB? (File Size)

---

#### 4. Binary Files vs. Text Files

တွၼ်ႈတႃႇ Programmer ၼိုင်ႈၵေႃႉ၊ ႁဝ်းၸႅၵ်ႇ File ပဵၼ် 2 မဵဝ်းလူင်:

1. **Text Files:** မၼ်းမီးတူဝ်လိၵ်ႈ ဢၼ်ၵူၼ်းႁဝ်းလူလႆႈငၢႆႈငၢႆႈ (မိူၼ်ၼင်ႇ Code ႁဝ်း)။
2. **Binary Files:** မၼ်းသိမ်းဝႆႉ 1 လႄႈ 0 ဢၼ်လၢၵ်ႇလၢႆး။ သင်ႁဝ်းဢဝ် Notepad ပိုတ်ႇတူၺ်း၊ ႁဝ်းတေႁၼ်မၼ်းပဵၼ်လိၵ်ႈ "ယုင်ႈယၢင်ႈ" ဢၼ်ၵူၼ်းႁဝ်းလူဢမ်ႇဢွၵ်ႇ (မိူၼ်ၼင်ႇ File ငဝ်းတူင်ႉ ဢမ်ႇၼၼ် Game)။

> **Programmer Tip:** မိူဝ်ႈၸဝ်ႈၵဝ်ႇတႅမ်ႈ Code၊ ၸဝ်ႈၵဝ်ႇတေလႆႈပွင်ႇၸႂ်လွင်ႈ "လၢႆးပိုတ်ႇလႄႈလၢႆးပိၵ်ႉ" (Open/Close) File။ သင်ၸဝ်ႈၵဝ်ႇပိုတ်ႇ File ဝႆႉသေ ဢမ်ႇပိၵ်ႉၶိုၼ်း၊ ၶေႃႈမုၼ်းၸိူဝ်းၼၼ်ႉ ၸၢင်ႈလူႉၵႂႃႇလႆႈယဝ်ႉ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ **File** ၼႆႉ ပဵၼ် **"ပပ်ႉလိၵ်ႈ"** ၼိုင်ႈပပ်ႉ:
1. **File Name:** ၸိုဝ်ႈပပ်ႉ။
2. **Extension (.txt):** မဵဝ်းပပ်ႉ (ပပ်ႉလိၵ်ႈ၊ ပပ်ႉတူၺ်းႁၢင်ႈ)။
3. **Data:** လိၵ်ႈဢၼ်မီးတၢင်းၼႂ်းပပ်ႉ။
4. **Metadata:** ဝၼ်းထိဢၼ်ဢိတ်ႇဢွၵ်ႇပပ်ႉၼၼ်ႉ။
