## Lesson 36: DNS (The Phonebook of the Internet)

ၼႂ်း Lesson ပူၼ်ႉမႃးၼၼ်ႉ၊ ႁဝ်းႁူႉယဝ်ႉဝႃႈ ၶွမ်ႇပိဝ်ႇတႃႇၸႂ်ႉ **IP Address** (မၢႆလိၵ်ႈ) တွၼ်ႈတႃႇႁႃၵၼ်။ 
ၵူၺ်းၵႃႈ ၵူၼ်းႁဝ်းၼႆႉ တွင်းၸိုဝ်ႈမၢႆၼမ်ၼမ် ဢမ်ႇလႆႈ။ ၵွပ်ႈၼၼ် ႁဝ်းၸင်ႇလူဝ်ႇ **DNS** ယဝ်ႉ။



---

#### 1. What is DNS? (DNS ပဵၼ်သင်?)

**DNS** (Domain Name System) ပဵၼ်သၢႆၵၢၼ်ဢၼ် "ပိၼ်ႇ" (Translate) ၸိုဝ်ႈ Website ႁႂ်ႈပဵၼ်မၢႆ IP Address ဢၼ်ၶွမ်ႇပိဝ်ႇတႃႇပွင်ႇၸႂ်ယဝ်ႉ။
* **Domain Name:** ပဵၼ်ၸိုဝ်ႈမိူၼ်ၼင်ႇ `google.com`, `facebook.com`, `wikipedia.org`။
* **IP Address:** ပဵၼ်မၢႆမိူၼ်ၼင်ႇ `142.250.190.46`။

---

#### 2. How it works? (မၼ်းႁဵတ်းၵၢၼ်ၸိူင်ႉႁိုဝ်?)

မိူဝ်ႈၸဝ်ႈၵဝ်ႇတႅမ်ႈ `google.com` သႂ်ႇၼႂ်း Browser:
1.  Browser ၸဝ်ႈၵဝ်ႇတေၵႂႃႇထၢမ် **DNS Server** ဝႃႈ: "ၸိုဝ်ႈ `google.com` ၼႆႉ မၢႆ IP မၼ်းပဵၼ်သင်?"
2.  DNS Server တေၵူတ်ႇထတ်းၼႂ်းသဵၼ်ႈမၢႆမၼ်း သေတွပ်ႇဝႃႈ: "မၢႆမၼ်းပဵၼ် `142.250.190.46` ဢေႃႈ!"
3.  Browser ၸဝ်ႈၵဝ်ႇ ၸင်ႇတေၸႂ်ႉမၢႆၼၼ်ႉသေ ၵပ်းသိုပ်ႇၵႂႃႇတီႈ Server ၶွင် Google ယဝ်ႉ။

---

#### 3. Why is it important?

* **Easy to Remember:** ၵူၼ်းႁဝ်းၼႆႉ တွင်းၸိုဝ်ႈ `facebook.com` ငၢႆႈလိူဝ်သေ တွင်းမၢႆ IP မၼ်းယဝ်ႉ။
* **Flexibility:** သင် Google လႅၵ်ႈမၢႆ IP မႂ်ႇ၊ ၶဝ်ၵူၺ်းလူဝ်ႇၵႂႃႇပိၼ်ႇၼႂ်း DNS သေ ႁဝ်းၵေႃႈတိုၵ်ႉၸၢင်ႈၸႂ်ႉၸိုဝ်ႈ `google.com` မိူၼ်ၵဝ်ႇလႆႈယူႇ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉၸႂ်ႉ **"ၾူၼ်း"** ၸဝ်ႈၵဝ်ႇ:
1.  **Domain Name:** ပဵၼ် **"ၸိုဝ်ႈၵူၼ်း"** (Name) ၼႂ်းသဵၼ်ႈမၢႆၾူၼ်းၸဝ်ႈၵဝ်ႇ။ တူဝ်ယၢင်ႇ: "ၸၢႆးသႅင်"။
2.  **IP Address:** ပဵၼ် **"မၢႆၾူၼ်း"** (Phone Number) မၼ်း။ တူဝ်ယၢင်ႇ: `09-12345678`။
3.  **DNS:** ပဵၼ် **"သဵၼ်ႈမၢႆၵပ်းသိုပ်ႇ"** (Contact List) ၼႂ်းၾူၼ်း။
    * ၸဝ်ႈၵဝ်ႇဢမ်ႇလူဝ်ႇတွင်း မၢႆၾူၼ်းပိူၼ်ႈ ၵူႈၵေႃႉ။
    * ၸဝ်ႈၵဝ်ႇၵူၺ်းလူဝ်ႇလိူၵ်ႈ "ၸိုဝ်ႈ" သေၼဵၵ်း၊ ၾူၼ်းၸဝ်ႈၵဝ်ႇ တေၵႂႃႇႁႃမၢႆမၼ်းသေ ၵပ်းသိုပ်ႇပၼ်ႁင်းၵူၺ်းယဝ်ႉ။
