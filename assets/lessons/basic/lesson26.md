## Lesson 26: Permissions (Running as Administrator)

မိူဝ်ႈၸဝ်ႈၵဝ်ႇတႅမ်ႈၶေႃႈသင်ႇ (Command) ၼႂ်း CLI၊ မၢင်ပွၵ်ႈၸဝ်ႈၵဝ်ႇ တေႁၼ်လိၵ်ႈလၢတ်ႈဝႃႈ **"Access Denied"** (ဢမ်ႇပၼ်ၶဝ်ႈ)။ 
ၼႆႉပဵၼ်ယွၼ်ႉ Windows တိုၵ်ႉႁႄႉၵင်ႈဝႆႉ ဢမ်ႇႁႂ်ႈၸဝ်ႈၵဝ်ႇၵႂႃႇလႅၵ်ႈလၢႆႈ File ဢၼ်လမ်ႇလွင်ႈၼၼ်ႉယဝ်ႉ။

---

#### 1. What is Administrator Mode?

ၼႂ်း Windows၊ မၼ်းမီးၸၼ်ႉဢႃႇၼႃႇ 2 မဵဝ်း:
1.  **Standard User:** ၸၢင်ႈၸႂ်ႉ Program ၵူႈလွင်ႈ၊ ၵူၺ်းၵႃႈ ဢမ်ႇၸၢင်ႈၵႂႃႇယႃႉ File ပိုၼ်ႉထၢၼ်ၶွင် OS။
2.  **Administrator (Admin):** ပဵၼ် "ၸဝ်ႈဢႃႇၼႃႇ" ဢၼ်ၸၢင်ႈႁဵတ်းလႆႈၵူႈလွင်ႈ၊ တႄႇတီႈၵၢၼ်လူပ်းပႅတ်ႈ File System တေႃႇထိုင်ၵၢၼ်သႂ်ႇ Program မႂ်ႇမႂ်ႇ။



---

#### 2. How to "Run as Administrator"

သင်ၸဝ်ႈၵဝ်ႇလူဝ်ႇၸႂ်ႉၶေႃႈသင်ႇ ဢၼ်လူဝ်ႇဢႃႇၼႃႇသုင် (တူဝ်ယၢင်ႇ: ၵၢၼ်လႅၵ်ႈလၢႆႈ Settings ၶွင် Network)၊ ၸဝ်ႈၵဝ်ႇတေလႆႈပိုတ်ႇ CLI လူၺ်ႈလၢႆးၼႆႉ:

1.  ၼဵၵ်းပုမ်ႇ **Windows** သေတႅမ်ႈ `cmd`။
2.  ယႃႇပႆႇၼဵၵ်း Enter၊ ၼဵၵ်း **Right-click** (ၼဵၵ်းၶႅၼ်းၶႂႃ) တီႈၼိူဝ် Command Prompt။
3.  လိူၵ်ႈဢဝ် **"Run as Administrator"**။
4.  ၸေႃးလမ်ၼၼ်ႉတေပိုတ်ႇမႃး၊ တီႈၼိူဝ် Window ၼၼ်ႉ တေမီးလိၵ်ႈတႅမ်ႈဝႆႉဝႃႈ `Administrator: Command Prompt` ယဝ်ႉ။

---

#### 3. Why is this Dangerous? (ၵွပ်ႈသင်မၼ်းမီးၽေးၶဵၼ်ၼမ်?)

မိူဝ်ႈၸဝ်ႈၵဝ်ႇယူႇၼႂ်း Admin Mode၊ ၸဝ်ႈၵဝ်ႇမီး "ၶႅပ်းၸေႈ" ဢၼ်ၸၢင်ႈယႃႉပႅတ်ႈ "ၶူဝ်" (Bridge) ၼႂ်းၶွမ်ႇပိဝ်ႇတႃႇလႆႈၵူႈဢၼ်။
* သင်ၸဝ်ႈၵဝ်ႇတႅမ်ႈ `del` ၽိတ်းတီႈၼႂ်း Admin Mode၊ ၸဝ်ႈၵဝ်ႇၸၢင်ႈယႃႉပႅတ်ႈ Windows တင်းသဵင်ႈ ႁႂ်ႈမၼ်းပိုတ်ႇဢမ်ႇလႆႈထႅင်ႈၵေႃႈ ၸၢင်ႊပဵၼ်လႆႈယဝ်ႉ။
* **Programmer Rules:** ၸႂ်ႉ Admin Mode မိူဝ်ႈလူဝ်ႇတႄႉတႄႉၵူၺ်း။ သင်ၵၢၼ်ၼၼ်ႉ ႁဵတ်းလႆႈၼႂ်း Standard Mode၊ ႁႂ်ႈၸႂ်ႉ Standard Mode တႃႇသေႇ တွၼ်ႈတႃႇလွင်ႈၵတ်းယဵၼ် (Security) ယဝ်ႉ။



---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉယူႇၼႂ်း **"ႁူင်းႁဵၼ်း"**:
1.  **Standard User:** မိူၼ်ၼင်ႇ "လုၵ်ႈႁဵၼ်း"။ ၸဝ်ႈၵဝ်ႇၶဝ်ႈႁွင်ႈႁဵၼ်းလႆႈ၊ တႅမ်ႈလိၵ်ႈလႆႈ၊ ၵူၺ်းၵႃႈ ၸဝ်ႈၵဝ်ႇဢမ်ႇၸၢင်ႈၵႂႃႇ "ယႃႉပႅတ်ႈ" ၽႃႁွင်ႈႁဵၼ်း လႄႈ ဢမ်ႇၸၢင်ႈ "ပိုတ်ႇၽၵ်းတူလူင်" ႁူင်းႁဵၼ်းယၢမ်းၵၢင်ၶိုၼ်းလႆႈ။
2.  **Administrator:** မိူၼ်ၼင်ႇ "ၵူၼ်းပႂ်ႉႁူင်းႁဵၼ်း" (Janitor/Principal) ဢၼ်မီး "သေႃး" (Key) ၵူႈဢၼ်။ မၼ်းၸၢင်ႈပိုတ်ႇလႆႈၵူႈႁွင်ႈ၊ လႅၵ်ႈလၢႆႈလႆႈၵူႈလွင်ႈ။ ၵူၺ်းၵႃႈ သင်မၼ်းၸႂ်ႉၽိတ်း၊ မၼ်းၸၢင်ႈႁဵတ်းႁႂ်ႈ ႁူင်းႁဵၼ်းလူႉယဝ်ႉ။

---

#### 🏁 Module 4 Checkpoint
Module 4 ယဝ်ႉလီငၢမ်းယဝ်ႉ! တေႃႈလဵဝ်ၸဝ်ႈၵဝ်ႇပဵၼ် "ၵူၼ်းၸႂ်ႉ CLI" ယဝ်ႉ:
* **Terminal/Shell:** ႁူႉၸၵ်း Command Prompt လႄႈ PowerShell.
* **Navigation:** ပႆလႆႈလူၺ်ႈ `dir` လႄႈ `cd`.
* **Operations:** သၢင်ႈလႄႈယႃႉလႆႈလူၺ်ႈ `mkdir`, `echo`, `del`.
* **Paths:** ပွင်ႇၸႂ်လွင်ႈ Absolute လႄႈ Relative Path.
* **Networking:** ၵူတ်ႇထတ်း Internet လႆႈလူၺ်ႈ `ping`, `ipconfig`.
* **Permissions:** ႁူႉလၢႆးပိုတ်ႇ Admin Mode တွၼ်ႈတႃႇၵၢၼ်ယၢပ်ႇယၢပ်ႇ.
