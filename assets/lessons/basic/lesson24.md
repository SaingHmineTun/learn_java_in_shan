## Lesson 24: Paths (Absolute vs. Relative)

မိူဝ်ႈႁဝ်းၸႂ်ႉ CLI၊ ႁဝ်းလူဝ်ႇလႆႈ "လၢတ်ႈ" တႅတ်ႈတႅတ်းဝႃႈ File ၼၼ်ႉယူႇတီႈလႂ်။ 
လၢႆးလၢတ်ႈၼၼ်ႉ ႁဝ်းႁွင်ႉဝႃႈ **Path** (သဵၼ်ႈတၢင်း) ယဝ်ႉ။ File Path ၼႆႉ မၼ်းမီး 2 လၢႆး ဢၼ်ၸဝ်ႈၵဝ်ႇလူဝ်ႇလႆႈႁူႉယဝ်ႉ:

---

#### 1. Absolute Path (သဵၼ်ႈတၢင်းတဵမ်)

**Absolute Path** ပဵၼ်ၵၢၼ်လၢတ်ႈသဵၼ်ႈတၢင်း တႄႇတီႈ "ငိူၼ်ႈ" (Root) မၼ်းမႃးၵမ်းလဵဝ်။ ၼႂ်း Windows၊ မၼ်းတေတႄႇလူၺ်ႈ `C:\` ဢမ်ႇၼၼ် `D:\` တႃႇသေႇယဝ်ႉ။

* **တူဝ်ယၢင်ႇ:** `C:\Users\Sai\Documents\homework.txt`
* **လွင်ႈလီ:** ၸဝ်ႈၵဝ်ႇတေယူႇတီႈလႂ်ၵေႃႈယႃႇ၊ သင်ၸဝ်ႈၵဝ်ႇတႅမ်ႈ Absolute Path၊ CLI တေႁႃ File ၼၼ်ႉ ႁၼ်လႆႈၵမ်းလဵဝ်။ မၼ်းမိူၼ်ၼင်ႇ "ႁပ်ႉၸိုဝ်ႈတီႈယူႇ (GPS Address)" တဵမ်တဵမ်ၼၼ်ႉယဝ်ႉ။



---

#### 2. Relative Path (သဵၼ်ႈတၢင်းၵပ်းၵၢႆႇ)

**Relative Path** ပဵၼ်ၵၢၼ်လၢတ်ႈသဵၼ်ႈတၢင်း မၵ်းမၼ်ႈဢဝ် "တီႈဢၼ်ၸဝ်ႈၵဝ်ႇတိုၵ်ႉယူႇ" ပဵၼ်ၵၢင်မၼ်း။

* **တူဝ်ယၢင်ႇ:** သင်တေႃႈလဵဝ် ၸဝ်ႈၵဝ်ႇတိုၵ်ႉယူႇၼႂ်း Folder `Sai` သေ လူဝ်ႇၶဝ်ႈ `Documents`၊ ၸဝ်ႈၵဝ်ႇတႅမ်ႈၵူၺ်းဝႃႈ `cd Documents` (ဢမ်ႇလူဝ်ႇတႄႇလူၺ်ႈ `C:\`)။
* **လွင်ႈလီ:** မၼ်းတႅမ်ႈငၢႆႈ လႄႈ ဝႆး။ ၵူၺ်းၵႃႈ သင်ၸဝ်ႈၵဝ်ႇယူႇၽိတ်းတီႈ၊ CLI တေႁႃဢမ်ႇႁၼ်။

---

#### 3. The Special Symbols (ၶေႃႈလိၵ်ႈလပ်ႉ)

ၼႂ်း Relative Path၊ မီးတူဝ်လိၵ်ႈ 2 ဢၼ် ဢၼ်ၸဝ်ႈၵဝ်ႇတေလႆႈၸႂ်ႉတႃႇသေႇ:

1.  **`.` (ၸႅတ်ႉဢၼ်လဵဝ်):** မၵ်းမၼ်ႈဝႃႈ "Folder ဢၼ်တိုၵ်ႉယူႇယၢမ်းလဵဝ်"။
2.  **`..` (ၸႅတ်ႉသွင်ဢၼ်):** မၵ်းမၼ်ႈဝႃႈ "Folder ဢၼ်ယူႇၸၼ်ႉၼိူဝ်" (Parent Folder)။

* **တူဝ်ယၢင်ႇ:** `cd ..\Music` (မၵ်းမၼ်ႈဝႃႈ: ထွၼ်လင်ဢွၵ်ႇမႃးၸၼ်ႉၼိုင်ႈၵွၼ်ႇ၊ ယဝ်ႉၸင်ႇၶဝ်ႈၵႂႃႇၼႂ်း Folder Music)။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉ "ႁႃႁွင်ႈၼွၼ်း" ၼႂ်းႁိူၼ်းလင်ၼိုင်ႈ:

1.  **Absolute Path:** မိူၼ်ၼင်ႇၸဝ်ႈၵဝ်ႇလၢတ်ႈဝႃႈ: *"မိူင်းတႆး၊ ဝဵင်းတွင်ႇၵျီး၊ ပွၵ်ႉ 1၊ ႁိူၼ်းမၢႆ 5၊ ၸၼ်ႉသွင်၊ ႁွင်ႈသၢႆႉမိုဝ်း"*။ (ၽႂ်မႃးၵေႃႈ ႁႃႁၼ်ၵမ်းလဵဝ်)။
2.  **Relative Path:** သင်ၸဝ်ႈၵဝ်ႇတိုၵ်ႉယူႇၼႂ်း "ႁွင်ႈၼင်လဵၼ်ႈ" ယဝ်ႉ၊ ၸဝ်ႈၵဝ်ႇၵူၺ်းလူဝ်ႇလၢတ်ႈဝႃႈ: *"ၶိုၼ်ႈၵႂႃႇၸၼ်ႉၼိူဝ်၊ သေၶဝ်ႈႁွင်ႈသၢႆႉမိုဝ်း"*။

သင်ၸဝ်ႈၵဝ်ႇတိုၵ်ႉယူႇ "ၼႂ်းႁိူၼ်း" ဝႆႉယူႇယဝ်ႉၼႆ၊ ၸႂ်ႉတိုဝ်း Relative Path ၼႆႉ ငၢႆႈလိူဝ်ယဝ်ႉ။
