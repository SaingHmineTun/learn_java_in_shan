## Lesson 15: File Systems (How the Manager Organizes Storage)

မိူၼ်ၼင်ႇႁဝ်းႁဵၼ်းမႃးၼႂ်း Module 2၊ "Files" ၼႆႉ ပဵၼ်ထူင်သိမ်းၶေႃႈမုၼ်း။ ၵမ်းၼႆႉ OS တေမႃးၸတ်းၵၢၼ်ဝႃႈ တေဝၢင်းထူင်ၸိူဝ်းၼၼ်ႉဝႆႉတီႈလႂ် ၼႂ်း Storage ႁႂ်ႈႁဝ်းႁႃၶိုၼ်းလႆႈငၢႆႈငၢႆႈယဝ်ႉ။



---

#### 1. What is a File System? (File System ပဵၼ်သင်?)

**File System** ပဵၼ် "လၢႆးၸတ်းၵၢၼ်" ဢၼ် OS ၸႂ်ႉတွၼ်ႈတႃႇ သိမ်း (Save) လႄႈ ႁႃ (Retrieve) ၶေႃႈမုၼ်းၼႂ်း Storage (HDD/SSD) ယဝ်ႉ။ သင်ဢမ်ႇမီး File System၊ ၶေႃႈမုၼ်းတင်းသဵင်ႈ တေၵႂႃႇလေႃးၵၼ်သေ ႁဝ်းတေဢမ်ႇၸၢင်ႈႁူႉဝႃႈ File ၼိုင်ႈဢၼ် တႄႇတီႈလႂ် ယဝ်ႉတီႈလႂ်ယဝ်ႉ။

---

#### 2. Folders and Paths (သဵၼ်ႈတၢင်းၶေႃႈမုၼ်း)

တွၼ်ႈတႃႇႁႂ်ႈၶေႃႈမုၼ်းမီးလွၵ်းမီးလၢႆး၊ File System ၸႂ်ႉလၢႆးၶပ်ႉယၢႆးၼင်ႇၼႆ:

* **Folders (Directories):** မိူၼ်ၼင်ႇ "လဵၼ်း" (Drawers) ဢၼ်ႁဝ်းဢဝ် File လၢႆလၢႆဢၼ်မႃးၸုမ်ႉၵၼ်ဝႆႉ။
* **Paths:** ပဵၼ် "သဵၼ်ႈတၢင်း" တွၼ်ႈတႃႇၵႂႃႇၸူး File ၼိုင်ႈဢၼ်။
    * **Windows Path:** `C:\Users\Sai\Music\song.mp3`
    * **Mac/Linux Path:** `/Users/Sai/Music/song.mp3`

---

#### 3. Common File Systems (မဵဝ်း File System ဢၼ်ၸႂ်ႉၼမ်)

OS ၵူႈဢၼ် မီးလၢႆး "ၶပ်ႉပပ်ႉ" ၼိူဝ်သႅင်ႇဢမ်ႇမိူၼ်ၵၼ်:

* **NTFS:** ၸႂ်ႉၼႂ်း Windows (မၼ်းၵႅၼ်ႇၶႅင် လႄႈ သိမ်း File ယႂ်ႇလႆႈ)။
* **APFS:** ၸႂ်ႉၼႂ်း Apple Mac (ဝႆးမၢၵ်ႇၼႃႇ တွၼ်ႈတႃႇ SSD)။
* **FAT32:** ပဵၼ်လၢႆးၵဝ်ႇ ဢၼ်ၸႂ်ႉလႆႈၵူႈ OS။ ၵမ်ၼမ်ၸႂ်ႉၼႂ်း USB Flash Drive ယွၼ်ႉမၼ်းၵပ်းသိုပ်ႇလႆႈတင်း Windows လႄႈ Mac ယဝ်ႉ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ **Storage** ၼႆႉ ပဵၼ် **"ႁွင်ႈတူၺ်းလိၵ်ႈ"** (Library) လူင်ၼိုင်ႈလင်:
1.  **File:** ပဵၼ် "ပပ်ႉလိၵ်ႈ" ၼိုင်ႈပပ်ႉ။
2.  **Folder:** ပဵၼ် "လွၵ်းၶပ်ႉပပ်ႉ" (Shelf) ၼိုင်ႈဢၼ်။
3.  **File System:** ပဵၼ် "လၢႆးၸတ်းၵၢၼ်" ဢၼ်မၵ်းမၼ်ႈဝႃႈ ပပ်ႉလိၵ်ႈလွင်ႈၵၢၼ်တႅမ်ႈ Code တေလႆႈဝႆႉတီႈလွၵ်း A၊ ပပ်ႉလိၵ်ႈလွင်ႈတၢင်းၵိၼ် တေလႆႈဝႆႉတီႈလွၵ်း B။

သင်ၸဝ်ႈၵဝ်ႇလူဝ်ႇႁႃပပ်ႉ "Python", ၸဝ်ႈၵဝ်ႇတေၵႂႃႇၸွမ်း **Path**: `Library -> Coding Shelf -> Python Book` ယဝ်ႉ။
