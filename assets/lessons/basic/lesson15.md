## Lesson 15: File Systems (How the Manager Organizes Storage)

မိူၼ်ၼင်ႇႁဝ်းႁဵၼ်းမႃးၼႂ်း Module 2၊ "Files" ၼႆႉ ပဵၼ်ထူင်သိမ်းၶေႃႈမုၼ်း။ 
ၵမ်းၼႆႉ OS တေမႃးၸတ်းၵၢၼ်ဝႃႈ တေဝၢင်းထူင်ၸိူဝ်းၼၼ်ႉဝႆႉတီႈလႂ် ၼႂ်း Storage ႁႂ်ႈႁဝ်းႁႃၶိုၼ်းလႆႈငၢႆႈငၢႆႈယဝ်ႉ။



---

#### 1. What is a File System? (File System ပဵၼ်သင်?)

**File System** ပဵၼ် "လၢႆးၸတ်းၵၢၼ်" ဢၼ် OS ၸႂ်ႉတွၼ်ႈတႃႇ သိမ်း (Save) လႄႈ ႁႃ (Retrieve) ၶေႃႈမုၼ်းၼႂ်း Storage (HDD/SSD) ယဝ်ႉ။ 
သင်ဢမ်ႇမီး File System၊ ၶေႃႈမုၼ်းတင်းသဵင်ႈ တေၵႂႃႇလေႃးၵၼ်သေ ႁဝ်းတေဢမ်ႇၸၢင်ႈႁူႉဝႃႈ File ၼိုင်ႈဢၼ် တႄႇတီႈလႂ် ယဝ်ႉတီႈလႂ်ယဝ်ႉ။

---

#### 2. Folders and Paths (သဵၼ်ႈတၢင်းၶေႃႈမုၼ်း)

တွၼ်ႈတႃႇႁႂ်ႈၶေႃႈမုၼ်းမီးလွၵ်းမီးလၢႆး၊ File System ၸႂ်ႉလၢႆးၶပ်ႉၼင်ႇၼႆ:

* **Folders (Directories):** မိူၼ်ၼင်ႇ "ၶွင်ႈထွတ်ႇ/ၶွင်ႈၸၼ်" (Drawers) ဢၼ်ႁဝ်းဢဝ် File လၢႆလၢႆဢၼ်မႃးၸုမ်ႉၵၼ်ဝႆႉ။
* **Paths:** ပဵၼ် "သဵၼ်ႈတၢင်း" တွၼ်ႈတႃႇၵႂႃႇၸူး File ၼိုင်ႈဢၼ်။
    * **Windows Path:** `C:\Users\Sai\Music\song.mp3`
    * **Mac/Linux Path:** `/Users/Sai/Music/song.mp3`

---

#### 3. Common File Systems (မဵဝ်း File System ဢၼ်ၸႂ်ႉၼမ်)

OS ၵူႈဢၼ် မီးလၢႆး "ၶပ်ႉပပ်ႉ" ဢမ်ႇမိူၼ်ၵၼ်:

* **NTFS:** ၸႂ်ႉၼႂ်း Windows (မၼ်းၵႅၼ်ႇၶႅင် လႄႈ သိမ်း File ယႂ်ႇလႆႈ)။
* **APFS:** ၸႂ်ႉၼႂ်း Apple Mac (ႁဵတ်းၵၢၼ်ဝႆး တွၼ်ႈတႃႇ SSD)။
* **FAT32:** ပဵၼ်လၢႆးၵဝ်ႇ ဢၼ်ၸႂ်ႉလႆႈၵူႈ OS။ ၵမ်ၼမ်ၸႂ်ႉၼႂ်း USB Flash Drive ယွၼ်ႉမၼ်းၵပ်းသိုပ်ႇလႆႈတင်း Windows လႄႈ Mac ယဝ်ႉ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ **Storage** ၼႆႉ ပဵၼ် **"ႁွင်ႈတူၺ်းလိၵ်ႈ"** (Library) လူင်ၼိုင်ႈလင်:
1.  **File:** ပဵၼ် "ပပ်ႉလိၵ်ႈ" ၼိုင်ႈပပ်ႉ။
2.  **Folder:** ပဵၼ် "လွၵ်းၶပ်ႉပပ်ႉ" (Shelf) ၼိုင်ႈဢၼ်။
3.  **File System:** ပဵၼ် "လၢႆးၸတ်းၵၢၼ်" ဢၼ်မၵ်းမၼ်ႈဝႃႈ ပပ်ႉလိၵ်ႈလွင်ႈၵၢၼ်တႅမ်ႈ Code တေလႆႈဝႆႉတီႈလွၵ်း A၊ ပပ်ႉလိၵ်ႈလွင်ႈတၢင်းၵိၼ် တေလႆႈဝႆႉတီႈလွၵ်း B။

---

**ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း**

* **File System:** လွၵ်းလၢႆးၸတ်းၵၢၼ် ဝႆႉဝၢင်း လႄႈ တႅမ်ႈၸိုဝ်ႈ File ၼႂ်း Hard Drive (တူဝ်ယၢင်ႇ: NTFS, FAT32)။
* **Root Directory:** ႁူဝ်ၶူဝ်ႉ (Folder) ဢၼ်သုင်သုတ်း ဢၼ်ၵဵပ်းဝႆႉ Folder တၢင်ႇဢၼ် (မိူၼ်ၼင်ႇ C:\)။
* **Path:** "သဵၼ်ႈတၢင်း" ဢၼ်လၢတ်ႈၼႄဝႃႈ File ၼိုင်ႈဢၼ် ဝႆႉဝၢင်းဝႆႉတီႈလႂ်။
