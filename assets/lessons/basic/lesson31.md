## Lesson 31: How Software is Built (The Look and the Brain)

မိူဝ်ႈႁဝ်းႁၼ် Software ဢမ်ႇၼၼ် App ၼိုင်ႈဢၼ်၊ ႁဝ်းၵမ်ႈၼမ်ႁၼ် "ၼႃႈတႃ" မၼ်းၵူၺ်း။ 
ၵူၺ်းၵႃႈ ၼႂ်းၵၢၼ်သၢင်ႈ Software၊ မၼ်းမီး 2 တွၼ်ႈလူင် ဢၼ်ႁဵတ်းၵၢၼ်ၸွမ်းၵၼ်ယဝ်ႉ။



---

#### 1. The Front-end (The Look - ၼႃႈတႃ)

**Front-end** ပဵၼ်တွၼ်ႈဢၼ် "ၸဝ်ႈၵဝ်ႇႁၼ်" လႄႈ "ၸဝ်ႈၵဝ်ႇၼဵၵ်းလႆႈ" တီႈၼႃႈၸေႃးယဝ်ႉ။

* **Role:** မၼ်းမီးၼႃႈၵၢၼ် ႁဵတ်းႁႂ်ႈ Interface ၼၼ်ႉ တူၺ်းလီ လႄႈ ၸႂ်ႉငၢႆႈ။
* **Components:** တုမ်ႇ (**Buttons**), သီ (**Colors**), တူဝ်လိၵ်ႈ (**Fonts**), လႄႈ ႁၢင်ႈ (**Images** )။
* **Analogy:** မိူၼ်ၼင်ႇ "ၼႃႈတႃ" လႄႈ "ၶူဝ်းၼုင်ႈ" ၶွင်ၵူၼ်းၵေႃႉၼိုင်ႈ။

---

#### 2. The Back-end (The Brain - ဢူၵ်းဢေႃ)

**Back-end** ပဵၼ်တွၼ်ႈဢၼ် "ၸဝ်ႈၵဝ်ႇဢမ်ႇႁၼ်"။ မၼ်းယူႇဝႆႉတၢင်းလင်သေ ႁဵတ်းၵၢၼ်ယၢပ်ႇယၢပ်ႇၸိူဝ်းၼၼ်ႉယဝ်ႉ။

* **Role:** မၼ်းမီးၼႃႈၵၢၼ် "ဝူၼ်ႉ" (**Logic**) လႄႈ "ၵဵပ်းၶေႃႈမုၼ်း" (**Data storage**)။
* **Action:** မိူဝ်ႈၸဝ်ႈၵဝ်ႇၼဵၵ်းတုမ်ႇ "Login"၊ Back-end တေၵႂႃႇၵူတ်ႇထတ်းတူၺ်းဝႃႈ Password ၸဝ်ႈၵဝ်ႇ မၢၼ်ႇႁႃႉ ဢမ်ႇမၢၼ်ႇ။
* **Analogy:** မိူၼ်ၼင်ႇ "ဢူၵ်းဢေႃ" လႄႈ "ပွတ်ႇ/ၸႂ်" ဢၼ်ယူႇတၢင်းၼႂ်းသေ ႁဵတ်းႁႂ်ႈတူဝ်ၶိူင်ႈလႅၼ်ႈလႆႈ။

---

#### 3. How they talk to each other?

Software ဢၼ်လီၼၼ်ႉ Front-end လႄႈ Back-end လူဝ်ႇလႆႈလၢတ်ႈတေႃႇၵၼ် တႃႇသေႇ။

1.  **Front-end:** "ၵူၼ်းၸႂ်ႉတိုဝ်း ၼဵၵ်းတုမ်ႇ 'သူင်ႇလိၵ်ႈ' ယဝ်ႉဢေႃႈ!"
2.  **Back-end:** "ႁပ်ႉႁူႉယဝ်ႉ! ၶႃႈတေဢဝ်လိၵ်ႈၼၼ်ႉ ၵႂႃႇၵဵပ်းဝႆႉၼႂ်း Database သေ သူင်ႇပၼ်ပိူၼ်ႈၵမ်းလဵဝ်။"
3.  **Front-end:** "သူင်ႇယဝ်ႉယဝ်ႉၼႆ ၶႃႈတေၼႄလိၵ်ႈသီၶဵဝ်ဝႃႈ 'Success' ၼႆၼႃႈ!"

---

**ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း**

* **Frontend:** တွၼ်ႈ Software ဢၼ်ၵူၼ်းၸႂ်ႉတိုဝ်းတူၺ်းႁၼ် လႄႈ ၼဵၵ်းလႆႈ (ပဵၼ်ႁၢင်ႈၽၢင်မၼ်း)။
* **Backend:** တွၼ်ႈ "ဢူၵ်းဢေႃ" ဢၼ်ယူႇၽၢႆႇလင်သေ ၸတ်းၵၢၼ်ၶေႃႈမုၼ်း လႄႈ လွၵ်းလၢႆးဝူၼ်ႉ။
* **Bug:** လွင်ႈၽိတ်းၽႅၼ်ႇၼႂ်းၶူတ်ႉ ဢၼ်ႁဵတ်းႁႂ်ႈ Software ႁဵတ်းၵၢၼ်ဢမ်ႇမၢၼ်ႇမႅၼ်ႈ။
