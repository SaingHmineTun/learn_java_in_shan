## Lesson 23: File Operations (Creating and Deleting)

ၼႂ်း Lesson ၼႆႉ ႁဝ်းတေမႃးႁဵၼ်းလၢႆး "သၢင်ႈ" (Create) လႄႈ "ယႃႉ" (Delete) File တင်း Folder တင်းသဵင်ႈ လူၺ်ႈဢမ်ႇၸႂ်ႉ Mouse သေဢိတ်းယဝ်ႉ။

---

#### 1. mkdir (Make Directory - သၢင်ႈ Folder မႂ်ႇ)

သင်ၸဝ်ႈၵဝ်ႇလူဝ်ႇသၢင်ႈ Folder မႂ်ႇၼိုင်ႈဢၼ်၊ ႁဝ်းၸႂ်ႉ `mkdir` (Make Directory)။

* **လၢႆးတႅမ်ႈ:** `mkdir MyNewFolder`
* **ၽွၼ်းလႆႈ:** မၼ်းတေၵူတ်ႇ Folder မႂ်ႇဢၼ်ၸိုဝ်ႈဝႃႈ "MyNewFolder" ပၼ်ၸဝ်ႈၵဝ်ႇၵမ်းလဵဝ်။

---

#### 2. Creating a File (သၢင်ႈ File တူဝ်လိၵ်ႈ)

ၼႂ်း Windows Command Prompt၊ လၢႆးငၢႆႈသုတ်းတွၼ်ႈတႃႇသၢင်ႈ File တူဝ်လိၵ်ႈ ၼိုင်ႈဢၼ်ၵေႃႈ ပဵၼ်ၵၢၼ်ၸႂ်ႉ `type` ဢမ်ႇၼၼ် `echo` ယဝ်ႉ။

* **လၢႆးတႅမ်ႈ:** `echo Hello World > note.txt`
* **ၽွၼ်းလႆႈ:** မၼ်းတေတႅမ်ႈလိၵ်ႈ "Hello World" သႂ်ႇၼႂ်း File ဢၼ်ၸိုဝ်ႈဝႃႈ `note.txt`။ သင် File ၼၼ်ႉဢမ်ႇပႆႇမီး၊ မၼ်းတေတႄႇသၢင်ႈပၼ်မႂ်ႇၵမ်းလဵဝ်ယဝ်ႉ။



---

#### 3. del & rmdir (ၵၢၼ်ယႃႉပႅတ်ႈ)

မိူဝ်ႈႁဝ်းမေႃသၢင်ႈယဝ်ႉ၊ ႁဝ်းၵေႃႈလူဝ်ႇမေႃယႃႉ (Delete) ၶိုၼ်း။
> **ၾၢင်ႉ (Warning):** ၵၢၼ်ယႃႉၼႂ်း CLI ၼႆႉ မၼ်းတေဢမ်ႇၵႂႃႇယူႇၼႂ်း Recycle Bin (တီႈၵွင်ယုၵ်းယၢၵ်း)။ မၼ်းတေ **ႁၢႆလၢႆၵႂႃႇၵမ်းလဵဝ်**၊ ၵွပ်ႈၼၼ် လူဝ်ႇလႆႈၾၢင်ႉတႄႉတႄႉယဝ်ႉ။

* **del:** ၸႂ်ႉတွၼ်ႈတႃႇယႃႉ **File**။
    * တူဝ်ယၢင်ႇ: `del note.txt`
* **rmdir:** ၸႂ်ႉတွၼ်ႈတႃႇယႃႉ **Folder**။
    * တူဝ်ယၢင်ႇ: `rmdir MyNewFolder` (Folder ၼၼ်ႉလူဝ်ႇလႆႈပဝ်ႇဝႆႉ ၸင်ႇတေယႃႉလႆႈ)။

---

#### 4. ren (Rename - ပိၼ်ႇၸိုဝ်ႈ)

သင်ၸဝ်ႈၵဝ်ႇလူဝ်ႇလႅၵ်ႈလၢႆႈၸိုဝ်ႈ File:

* **လၢႆးတႅမ်ႈ:** `ren note.txt lesson1.txt`
* **ၽွၼ်းလႆႈ:** File `note.txt` ၼၼ်ႉတေပိၼ်ႇၸိုဝ်ႈပဵၼ် `lesson1.txt` သေယူႇဝႆႉတီႈၵဝ်ႇယဝ်ႉ။



---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇပဵၼ် **"ၸဝ်ႈသူၼ်"**:
1.  **mkdir:** မိူၼ်ၼင်ႇၸဝ်ႈၵဝ်ႇ ၶုတ်း "လုၵ်း" တွၼ်ႈတႃႇတေၽုၵ်ႇ တူၼ်ႈမႂ်ႇ။
2.  **echo > note.txt:** မိူၼ်ၼင်ႇၸဝ်ႈၵဝ်ႇ ဢဝ် "ၽၼ်းမၼ်း" (Data) သႂ်ႇၼႂ်းလုၵ်းၼၼ်ႉ။
3.  **del:** မိူၼ်ၼင်ႇၸဝ်ႈၵဝ်ႇ ထွၼ်တူၼ်ႈဢၼ်ဢမ်ႇလူဝ်ႇယဝ်ႉၼၼ်ႉ ဢွၵ်ႇပႅတ်ႈ။
4.  **ren:** မိူၼ်ၼင်ႇၸဝ်ႈၵဝ်ႇ ပိၼ်ႇ "ပၢႆႉၸိုဝ်ႈ" တီႈၼႃႈတူၼ်ႈမၢၵ်ႇၸဝ်ႈၵဝ်ႇယဝ်ႉ။
