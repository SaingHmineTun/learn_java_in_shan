## Lesson 1: The Cycle (Input, Process, Output)

ၼႂ်းတွၼ်ႈၵၢၼ်တႅမ်ႈ "Program" ၼၼ်ႉ လူဝ်ႇလႆႈပွင်ႇၸႂ်ပိုၼ်းၵၢၼ်ပိုၼ်ႉထၢၼ် "Computer" ဢၼ်ၼႆႉၵွၼ်ႇဢေႃႈ။ ၶွမ်ႇပိဝ်ႇတႃႇၼႆႉ ပဵၼ်ၸၢၵ်ႈဢၼ်ႁပ်ႉဢဝ်ၶေႃႈမုၼ်း၊ မႃးဝူၼ်ႉသၢင်ႈ၊ ယဝ်ႉၵေႃႈ ပၼ်ၽွၼ်းလႅပ်ႈၶိုၼ်းၼၼ်ႉယဝ်ႉ။



---

#### 1. Input (ၵၢၼ်ဢဝ်ၶဝ်ႈ)
**Input** ပဵၼ်တွၼ်ႈတႃႇႁပ်ႉဢဝ် "ၶေႃႈမုၼ်း" (Data) ဢမ်ႇၼၼ် "ၶေႃႈသင်ႇ" (Instructions) တီႈၵူၼ်းႁဝ်းသေ သႂ်ႇၵႂႃႇၼႂ်းၶွမ်ႇပိဝ်ႇတႃႇ။

* **တူဝ်ယၢင်ႇ:** * **Keyboard:** တွၼ်ႈတႃႇတႅမ်ႈလိၵ်ႈ။
    * **Mouse:** တွၼ်ႈတႃႇၼဵၵ်းလိူၵ်ႈ။
    * **Microphone:** တွၼ်ႈတႃႇသႂ်ႇသဵင်။
* **တွၼ်ႈတႃႇ Programmer:** မိူဝ်ႈၸဝ်ႈၵဝ်ႇတႅမ်ႈ Code ၼၼ်ႉၵေႃႈ ပဵၼ်ၵၢၼ်သႂ်ႇ Input ၼင်ႇၵဝ်ႇ။

---

#### 2. Process (ၵၢၼ်ႁဵတ်းသၢင်ႈ)
မိူဝ်ႈၶွမ်ႇပိဝ်ႇတႃႇလႆႈႁပ်ႉဢဝ် Input ယဝ်ႉ၊ မၼ်းတေသူင်ႇၵႂႃႇတီႈ "မၢၵ်ႇႁူဝ်ၸႂ်" (**CPU**) တွၼ်ႈတႃႇဝူၼ်ႉသွၼ်ႇ။

* **တူဝ်ယၢင်ႇ:** မိူဝ်ႈၸဝ်ႈၵဝ်ႇၼဵၵ်း `2 + 2` တီႈ Keyboard၊ ၶွမ်ႇပိဝ်ႇတႃႇတေ "ဝူၼ်ႉ" သေ ဢဝ်မႃးႁူမ်ႈၵၼ်ႁႂ်ႈပဵၼ် `4`။
* **တွၼ်ႈတႃႇ Programmer:** ၼႆႉပဵၼ်တွၼ်ႈဢၼ် Code ႁဝ်းႁဵတ်းၵၢၼ်တႄႉတႄႉၼၼ်ႉယဝ်ႉ။

---

#### 3. Output (ၵၢၼ်ဢဝ်ဢွၵ်ႇ)
မိူဝ်ႈဝူၼ်ႉသွၼ်ႇယဝ်ႉတူဝ်ႈယဝ်ႉ၊ ၶွမ်ႇပိဝ်ႇတႃႇတေၼႄၽွၼ်းလႅပ်ႈ (**Result**) မၼ်းဢွၵ်ႇမႃးပၼ်ႁဝ်း။

* **တူဝ်ယၢင်ႇ:** * **Monitor:** ၼႃႈၸေႃးၼႄလိၵ်ႈ ဢမ်ႇၼၼ် ႁၢင်ႈ။
    * **Speaker:** သဵင်ဢွၵ်ႇမႃး။
    * **Printer:** ဢိတ်ႇလိၵ်ႈဢွၵ်ႇမႃးၼိူဝ်ၸေႈ။
* **တွၼ်ႈတႃႇ Programmer:** ၼႆႉပဵၼ်ၽွၼ်းလႅပ်ႈ ဢၼ်လုၵ်ႉတီႈ Code ႁဝ်းသေ ဢွၵ်ႇမႃးၼၼ်ႉယဝ်ႉ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)
ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉၸႂ်ႉ **Calculator** (ၸၢၵ်ႈတွပ်ႇလႅပ်ႈ):
1.  **Input:** ၸဝ်ႈၵဝ်ႇၼဵၵ်းမၢႆလႅပ်ႈ `10 x 10`.
2.  **Process:** ၶွမ်ႇပိဝ်ႇတႃႇဢဝ် `10` မႃးၵုၼ်ႇတင်း `10`.
3.  **Output:** ၼႃႈၸေႃးၼႄမၢႆတွပ်ႇ `100`.
