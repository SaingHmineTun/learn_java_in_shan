## Lesson 2: The CPU (The Brain)

**CPU (Central Processing Unit)** ၼႆႉ ပဵၼ် "မၢၵ်ႇႁူဝ်ၸႂ်" ဢမ်ႇၼၼ် "ဢူၵ်းဢေႃ" ၶွမ်ႇပိဝ်ႇတႃႇယဝ်ႉ။ မၼ်းပဵၼ်ပိၵ်ႉ (Chip) လဵၵ်ႉလဵၵ်ႉဢၼ်ၼိုင်ႈ ဢၼ်မီးၼႃႈၵၢၼ်ႁဵတ်းၸွမ်း "ၶေႃႈသင်ႇ" (Code) ဢၼ်ႁဝ်းတႅမ်ႈၼၼ်ႉၵူႈထႅဝ်ယဝ်ႉ။



---

#### 1. What is a CPU? (CPU ဢၼ်ဝႃႈၼၼ်ႉ ပဵၼ်သင်?)

ၼႂ်းတူဝ်ၶွမ်ႇပိဝ်ႇတႃႇၼၼ်ႉ CPU ပဵၼ်တွၼ်ႈဢၼ်လမ်ႇလွတ်ႈသုတ်း။ သင်ဝႃႈဢမ်ႇမီး CPU ၸိုင် ၶွမ်ႇပိဝ်ႇတႃႇတေဢမ်ႇၸၢင်ႈဝူၼ်ႉသင်လႆႈ။

* **The Engine:** CPU မိူၼ်ၼင်ႇ "ၸၢၵ်ႈလူတ်ႉၶိူင်ႈ"။ မၼ်းပဵၼ်တူဝ်ဢၼ်ပၼ်ႁႅင်းသေ ႁဵတ်းႁႂ်ႈ Program ႁဝ်းလႅၼ်ႈလႆႈ။
* **The Follower:** လူဝ်ႇလႆႈတွင်းဝႆႉဝႃႈ CPU ၼႆႉ မၼ်းဢမ်ႇၸိုဝ်ႈဝႃႈ "ၵတ်ႉၶႅၼ်ႇ" (Smart) လူင်လၢင်။ မၼ်း মাথႃးပဵၼ် ၸၢၵ်ႈဢၼ် "ထွမ်ႇၵႂၢမ်း" တႄႉတႄႉ။ သင်ႁဝ်းသင်ႇမၼ်းႁဵတ်းၽိတ်း မၼ်းၵေႃႈတေႁဵတ်းၽိတ်းၸွမ်းၵမ်းလဵဝ်ယဝ်ႉ။

---

#### 2. Speed: Gigahertz (GHz) (လွင်ႈဝႆး CPU)

မိူဝ်ႈႁဝ်းသိုဝ်ႉၶွမ်ႇပိဝ်ႇတႃႇ ႁဝ်းတေလႆႈႁၼ်လိၵ်ႈတႅမ်ႈဝႆႉဝႃႈ `3.2 GHz` ဢမ်ႇၼၼ် `2.4 GHz`။ ဢၼ်ၼၼ်ႉ မၼ်းၼႄပၼ် လွင်ႈဝႆး CPU ယဝ်ႉ။

* **Giga:** မၢႆထိုင် "ပၼ်ႇ" (Billion / 1,000,000,000)။
* **Hertz:** မၢႆထိုင် "ပွၵ်ႈ" (Cycles) ၼႂ်းၼိုင်ႈသိၵ်ႈ (Second)။
* **ၼႂ်းတွၼ်ႈ Programmer:** သင်ဝႃႈ CPU ၸဝ်ႈၵဝ်ႇဝႆး၊ Code ဢၼ်ၸဝ်ႈၵဝ်ႇတႅမ်ႈၼၼ်ႉၵေႃႈ တေလႅၼ်ႈယဝ်ႉတူဝ်ႈဝႆးၼၼ်ႉယဝ်ႉ။

---

#### 3. Cores (မၢၵ်ႇႁူဝ်ၸႂ်လၢႆဢၼ်)

မိူဝ်ႈၵွၼ်ႇ CPU တေမီး "ဢူၵ်းဢေႃ" ဢၼ်လဵဝ်ၵူၺ်း။ တေႃႈလဵဝ် CPU ၼိုင်ႈဢၼ် မီး Cores (မၢၵ်ႇႁူဝ်ၸႂ်လဵၵ်ႉ) လၢႆလၢႆဢၼ် မိူၼ်ၼင်ႇ Dual-core (2), Quad-core (4), ဢမ်ႇၼၼ် Octa-core (8)။

* **Multi-tasking:** ၼႆႉမိူၼ်ၼင်ႇမီး "ၵူၼ်းႁဵတ်းၵၢၼ်" လၢႆၵေႃႉၼႂ်းၶွမ်ႇပိဝ်ႇတႃႇဢၼ်လဵဝ်။ ၵေႃႉၼိုင်ႈၼဵၵ်းၵႂၢမ်းပၼ်၊ ၵေႃႉၼိုင်ႈတႅမ်ႈလိၵ်ႈပၼ်၊ ၵေႃႉၼိုင်ႈပိုတ်ႇ Website ပၼ်ၼၼ်ႉယဝ်ႉ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ CPU ၼႆႉ ပဵၼ် **"ၵူၼ်းႁုင်ၶဝ်ႈ"** (Chef) ၵေႃႉၼိုင်ႈ။
1.  ၸဝ်ႈၵဝ်ႇပဵၼ် **Programmer** ဢၼ်တႅမ်ႈ "ပပ်ႉသင်ႇႁုင်ၶဝ်ႈ" (Recipe/Code)။
2.  **CPU** တေလူပပ်ႉၼၼ်ႉသေ ႁဵတ်းၸွမ်းၵမ်းလဵဝ်။
3.  သင်ၸဝ်ႈၵဝ်ႇတႅမ်ႈဝႆႉဝႃႈ "သႂ်ႇၵိူဝ် 5 သိၼ်ႇ"၊ CPU ၵေႃႈတေသႂ်ႇ 5 သိၼ်ႇတႄႉတႄႉ (မၼ်းတေဢမ်ႇထဵင်ဝႃႈ ၼမ်ၼႃႇဢေႃႈ ၼႆသင်)။
