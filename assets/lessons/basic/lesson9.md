## Lesson 9: Text and Strings (How Computers Read Your Name)

တေႃႇထိုင်ယၢမ်းလဵဝ် ႁဝ်းႁူႉယဝ်ႉဝႃႈ ၶွမ်ႇပိဝ်ႇတႃႇပွင်ႇၸႂ်မၢႆၼပ်ႉ (Numbers) ၵူၺ်း။ ၵမ်းၼႆႉ လိၵ်ႈဢၼ်ႁဝ်းတႅမ်ႈယူႇၼႆႉသမ်ႉ မၼ်းပဵၼ်မႃးၸိူင်ႉႁိုဝ်?



---

#### 1. The Code Table (ပပ်ႉပိၼ်ႇလိၵ်ႈ)

ယွၼ်ႉပိူဝ်ႈၶွမ်ႇပိဝ်ႇတႃႇ ဢမ်ႇလူလႆႈတူဝ်လိၵ်ႈ 'A' ဢမ်ႇၼၼ် 'B'၊ ႁဝ်းၸင်ႇလႆႈၵေႃႇသၢင်ႈ "ပပ်ႉပိၼ်ႇလိၵ်ႈ" (Encoding Table) မႃးယဝ်ႉ။ ႁဝ်းမၵ်းမၼ်ႈဝႃႈ မၢႆၼပ်ႉဢၼ်ၼႆႉ တေတႅၼ်းတူဝ်လိၵ်ႈဢၼ်ၼၼ်ႉ ၼႆယဝ်ႉ။

* **ASCII:** ပဵၼ်ပပ်ႉပိၼ်ႇလိၵ်ႈပၢၼ်ၵဝ်ႇ။ တူဝ်ယၢင်ႇ:
    * မၢႆလႅပ်ႈ **65** = တူဝ်လိၵ်ႈ **A**
    * မၢႆလႅပ်ႈ **66** = တူဝ်လိၵ်ႈ **B**
* **Unicode:** ပဵၼ်ပပ်ႉပိၼ်ႇလိၵ်ႈပၢၼ်မႂ်ႇ။ မၼ်းသိမ်းလႆႈ လိၵ်ႈဢိင်းၵလဵတ်ႈ၊ လိၵ်ႈတႆး၊ လိၵ်ႈမၢၼ်ႈ၊ လႄႈ Emoji (😀, 🚀) တင်းသဵင်ႈယဝ်ႉ။

---

#### 2. What is a String? (String ဢၼ်ဝႃႈၼၼ်ႉ ပဵၼ်သင်?)

တွၼ်ႈတႃႇ Programmer ၼိုင်ႈၵေႃႉ၊ ႁဝ်းဢမ်ႇႁွင်ႉ "လိၵ်ႈ" ဝႃႈ Text၊ ႁဝ်းႁွင်ႉမၼ်းဝႃႈ **String** ယဝ်ႉ။

* **String** မၢႆထိုင် "တူဝ်လိၵ်ႈဢၼ်မႃးသိုပ်ႇၵၼ်ဝႆႉ" ယဝ်ႉ။
* **Example:** သင်ဝႃႈၸဝ်ႈၵဝ်ႇတႅမ်ႈဝႃႈ `"Shan"`, ၶွမ်ႇပိဝ်ႇတႃႇတေႁၼ်မၼ်းပဵၼ် တူဝ်လိၵ်ႈ 4 တူဝ် (S, h, a, n) ဢၼ်မႃးမတ်ႉၵၼ်ဝႆႉၼၼ်ႉယဝ်ႉ။

---

#### 3. Why Programmers Use Quotes? (ၵွပ်ႈသင်တေလႆႈၸႂ်ႉ " ")

မိူဝ်ႈၸဝ်ႈၵဝ်ႇ ၶႂ်ႈတႅမ်ႈတူဝ်လိၵ်ႈ (Text) တီႈၼႂ်း Code၊ ၸဝ်ႈၵဝ်ႇ တေလႆႈသႂ်ႇၶွပ်ႇၼူၵ်ႉ (Quotes) တႃႇသေႇယဝ်ႉ။
* **"Hello"** = ၼႆႉပဵၼ် **String** (လိၵ်ႈ)။
* **100** = ၼႆႉပဵၼ် **Number** (မၢႆၼပ်ႉ)။

သင်ၸဝ်ႈၵဝ်ႇဢမ်ႇသႂ်ႇ Quotes၊ ၶွမ်ႇပိဝ်ႇတႃႇတေထၢင်ႇဝႃႈ လိၵ်ႈၼၼ်ႉပဵၼ် "ၶေႃႈသင်ႇ" (Command) သေ မၼ်းတေဝူၼ်ႉသၢင်ႈၽိတ်းယဝ်ႉ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇသူင်ႇၶေႃႈၵႂၢမ်း **"Hi"** ၵႂႃႇၸူးပိူၼ်ၼႆလႄႈ:
1.  **H** = Binary `01001000` (မၢႆၼပ်ႉ 72)
2.  **i** = Binary `01101001` (မၢႆၼပ်ႉ 105)
3.  ၵွပ်ႈၼႆ ၶွမ်ႇပိဝ်ႇတႃႇ တေသူင်ႇ `01001000 01101001` ၵႂႃႇၸူးၵူၼ်းထႅင်ႈၵေႃႉၼိုင်ႈယဝ်ႉ။
4.  ၶွမ်ႇပိဝ်ႇတႃႇတၢင်ႇလုၵ်ႈ သမ်ႉၶိုၼ်း တေလူပပ်ႉပိၼ်ႇလိၵ်ႈ (Unicode) သေ ၼႄပၼ်ၶိုၼ်းဝႃႈ **"Hi"** ၼႆယဝ်ႉ။

---

**ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း**

* **Character:** တူဝ်လိၵ်ႈၼိုင်ႈတူဝ်, မၢႆၼပ်ႉၼိုင်ႈတူဝ် ဢမ်ႇၼၼ် သင်ႇၶေႇ (Symbol) ၼိုင်ႈဢၼ်။
* **Unicode:** လွၵ်းလၢႆးမၵ်းမၼ်ႈတူဝ်လိၵ်ႈ ဢၼ်ၸႂ်ႉလႆႈၵူႈၽႃႇသႃႇၼႂ်းလူၵ်ႈ (ပႃးတင်း Emoji)။
* **Concatenation:** ၵၢၼ်ဢဝ် String သွင်ဢၼ်မႃးသိုပ်ႇၵၼ် (မိူၼ်ၼင်ႇ "Hello" + "World")။
