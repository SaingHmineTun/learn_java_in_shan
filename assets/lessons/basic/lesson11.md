## Lesson 11: Colors and Pixels (How Images Work)

တေႃႇထိုင်ယၢမ်းလဵဝ် ႁဝ်းႁူႉယဝ်ႉဝႃႈ ၶွမ်ႇပိဝ်ႇတႃႇၸႂ်ႉ Binary တႃႇသိမ်းမၢႆၼပ်ႉ လႄႈ တူဝ်လိၵ်ႈ။ ၵမ်းၼႆႉ ႁဝ်းတေမႃးတူၺ်းဝႃႈ ၶွမ်ႇပိဝ်ႇတႃႇဢဝ် 1 လႄႈ 0 မႃးလႅၵ်ႈပဵၼ် "ႁၢင်ႈသီ" (Images) လႆႈၸိူင်ႉႁိုဝ်။



---

#### 1. What is a Pixel? (Pixel ပဵၼ်သင်?)

ႁၢင်ႈၵူႈႁၢင်ႈ ဢၼ်ၸဝ်ႈၵဝ်ႇႁၼ်ၼိူဝ်ၼႃႈၸေႃး (Monitor) ၼၼ်ႉ မၼ်းၵေႃႇသၢင်ႈဝႆႉလူၺ်ႈ "မၢၵ်ႇႁိၼ်သီ" လဵၵ်ႉလဵၵ်ႉ ဢၢၼ်ႇႁူဝ်လၢၼ်ႉမၼ်းယဝ်ႉ။ ႁဝ်းႁွင်ႉမၢၵ်ႇႁိၼ်သီလဵၵ်ႉလဵၵ်ႉၼၼ်ႉဝႃႈ **Pixel** ယဝ်ႉ။

* **Picture Element:** ၸိုဝ်ႈ Pixel ၼႆႉ လုၵ်ႉတီႈၶေႃႈၵႂၢမ်းသွင်ဢၼ် "Pix" (Picture) လႄႈ "el" (Element) မႃးလေႃးၵၼ်။
* **Resolution:** သင်ဝႃႈၼႃႈၸေႃးၸဝ်ႈၵဝ်ႇမီး Pixel ၼမ် (မိူၼ်ၼင်ႇ 4K)၊ ႁၢင်ႈၼၼ်ႉၵေႃႈတေၸႅင်ႈလႅင်းလီ တေႉတေႉယဝ်ႉ။

---

#### 2. The RGB Model (သီပိုၼ်ႉထၢၼ် 3 သီ)

ၶွမ်ႇပိဝ်ႇတႃႇ ဢမ်ႇမီးသီၵူႈသီ၊ မၼ်းၵူၺ်းမီး သီ 3 မဵဝ်း ၼႆႉၵူၺ်း။ ႁဝ်းႁွင်ႉမၼ်းသႃႈ **RGB** ၼႆယဝ်ႉ:
1. **R**ed (သီလႅင်)
2. **G**reen (သီၶဵဝ်)
3. **B**lue (သီၾႃႉ)

မိူဝ်ႈၶွမ်ႇပိဝ်ႇတႃႇဢဝ် 3 သီၼႆႉ မႃးလေႃးၵၼ်လူၺ်ႈ "တၢင်းၶႅၼ်ႇ" (Intensity) ဢၼ်ဢမ်ႇမိူၼ်ၵၼ်၊ မၼ်းၸၢင်ႈၵေႃႇသၢင်ႈပဵၼ်မႃးလႆႈ သီတၢင်းၼမ်လၢႆယဝ်ႉ။



---

#### 3. Binary to Color (လုၵ်ႉတီႈ Binary ပဵၼ်မႃးသီ)

တွၼ်ႈတႃႇ Pixel ၼိုင်ႈဢၼ်၊ ၶွမ်ႇပိဝ်ႇတႃႇတေၸႂ်ႉ Binary (Bytes) တွၼ်ႈတႃႇမၵ်းမၼ်ႈဝႃႈ တေလႆႈပိုတ်ႇ သီလႅင်၊ သီၶဵဝ်၊ လႄႈ သီသွမ်ႇ ၸိူင်ႉႁိုဝ်ၼႆယဝ်ႉ။

* **တူဝ်ယၢင်ႇ:**
    * `(255, 0, 0)` = သီလႅင်ၸႅင်ႈ (Red)
    * `(0, 255, 0)` = သီၶဵဝ်ၸႅင်ႈ (Green)
    * `(0, 0, 255)` = သီသွမ်ႇၾႃႉ (Blue)
    * `(0, 0, 0)` = သီလမ် (ပိၵ်ႉၾႆးတင်းသဵင်ႈ)
    * `(255, 255, 255)` = သီၶၢဝ် (ပိုတ်ႇၾႆးတင်းသဵင်ႈ)

> **Programmer Tip:** မိူဝ်ႈၸဝ်ႈၵဝ်ႇတႅမ်ႈ Code တွၼ်ႈတႃႇႁဵတ်း Website ဢမ်ႇၼၼ် Game၊ ၸဝ်ႈၵဝ်ႇတေလႆႈၸႂ်ႉ "မၢႆသီ" (Color Codes) ၸိူဝ်းၼႆႉတႃႇသေႇ။ လွင်ႈပွင်ႇၸႂ် RGB တေၸွႆႈႁႂ်ႈၸဝ်ႈၵဝ်ႇ လိူၵ်ႈသီလႆႈတႅတ်ႈၼႅတ်ႈယဝ်ႉ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉတူၺ်း **"ႁၢင်ႈ"** သီလိူင် ၼႆလႄႈ:
1.  ၶွမ်ႇပိဝ်ႇတႃႇတေတူၺ်း တီႈယူႇ (Coordinates) ၶွင် Pixel ၵူႈဢၼ်။
2.  မၼ်းတေလူ Binary ဝႃႈ Pixel ၼႆႉလူဝ်ႇသီသင်။
3.  တွၼ်ႈတႃႇသီလိူင်၊ မၼ်းတေပိုတ်ႇသီလႅင် (Red) လႄႈ သီၶဵဝ် (Green) မႃးလေႃးၵၼ်သေ ၼႄပၼ်ၸဝ်ႈၵဝ်ႇယဝ်ႉ။
