## Lesson 28: Windows & Icons (The 'Desktop' Metaphor)

မိူဝ်ႈၸဝ်ႈၵဝ်ႇပိုတ်ႇၶွမ်ႇပိဝ်ႇတႃႇ၊ ၸေႃးဢၼ်ၸဝ်ႈၵဝ်ႇႁၼ်ၼၼ်ႉ ႁဝ်းႁွင်ႉဝႃႈ **Desktop** (ၼႃႈတႅပ်း)။ ၵွပ်ႈသင်ၶဝ်ဢမ်ႇႁွင်ႉဝႃႈ "Screen" လၢႆလၢႆလေႃ? ယွၼ်ႉလူဝ်ႇႁဵတ်းႁႂ်ႈႁဝ်း ပွင်ႇၸႂ်ငၢႆႈၼၼ်ႉယဝ်ႉ။



---

#### 1. What is a Metaphor? (လွင်ႈၵပ်းၵၢႆႇ)

**Metaphor** ပဵၼ်ၵၢၼ်ဢဝ် "လွင်ႈတၢင်းဢၼ်ႁဝ်းႁူႉၸၵ်းၼႂ်းလူၵ်ႈတူဝ်တႅတ်ႈ" မႃးၸႂ်ႉတႅၼ်းတၢင်ႇ "လွင်ႈတၢင်းၼႂ်းၶွမ်ႇပိဝ်ႇတႃႇ"။
* **Folders:** ၼႂ်းၶွမ်ႇပိဝ်ႇတႃႇ၊ မၼ်းပဵၼ်ၵူၺ်း "လွၵ်းၵဵပ်းၶေႃႈမုၼ်း"။ ၵူၺ်းၵႃႈ ၶဝ်ၸႂ်ႉႁၢင်ႈ **Folder** (ပပ်ႉသိမ်းလိၵ်ႈ) ႁႂ်ႈႁဝ်းႁူႉဝႃႈ မၼ်းမီးဝႆႉတွၼ်ႈတႃႇ "သႂ်ႇ File" ယဝ်ႉ။
* **Desktop:** ၶဝ်ဝူၼ်ႉဝႃႈ ၸေႃးၶွမ်ႇပိဝ်ႇတႃႇၸဝ်ႈၵဝ်ႇ မိူၼ်ၼင်ႇ "တႅပ်းႁဵတ်းၵၢၼ်"။ ၸဝ်ႈၵဝ်ႇဝၢင်းပပ်ႉ (Files) ဝႆႉၼိူဝ်တႅပ်း၊ ယဝ်ႉၵေႃႈ ဢဝ်ယုၵ်းယၢၵ်းသႂ်ႇၼႂ်း ထိုင်ယုၵ်းယၢၵ်း (Trash bin)။

---

#### 2. Windows (ပူင်ႇၾႃႉ)

**Windows** ပဵၼ်လွၵ်းသီႇၸဵင်ႇ ဢၼ်ပိုတ်ႇဢွၵ်ႇမႃး မိူဝ်ႈႁဝ်းၸႂ်ႉ Program ၼိုင်ႈဢၼ်။
* မၼ်းႁဵတ်းႁႂ်ႈႁဝ်း "တူၺ်း" (View) ၶဝ်ႈၵႂႃႇၼႂ်း Program ၼၼ်ႉလႆႈ။
* ၸဝ်ႈၵဝ်ႇၸၢင်ႈပိုတ်ႇလၢႆလၢႆ Window ၼိူဝ်ၸေႃးလဵဝ်ၵၼ်၊ မိူၼ်ၼင်ႇၸဝ်ႈၵဝ်ႇ ဝၢင်းၸေႈလၢႆလၢႆၽႅၼ်ႇ ၼိူဝ်တႅပ်းၼၼ်ႉယဝ်ႉ။



---

#### 3. Icons (ႁၢင်ႈတႅၼ်း)

**Icon** ပဵၼ်ႁၢင်ႈလဵၵ်ႉလဵၵ်ႉ ဢၼ်လၢတ်ႈၼႄဝႃႈ Program ဢၼ်ၼၼ်ႉ ႁဵတ်းသင်လႆႈ။
* ႁၢင်ႈ **"မၢၵ်ႇမူၼ်းလူၵ်ႈ"** (Globe) = Browser တွၼ်ႈတႃႇၵႂႃႇ Internet။
* ႁၢင်ႈ **"ၸၢၵ်ႈထၢႆႇႁၢင်ႈ"** (Camera) = Program တွၼ်ႈတႃႇထၢႆႇႁၢင်ႈ။
* ႁၢင်ႈၸိူဝ်းၼႆႉ ၸွႆႈႁႂ်ႈႁဝ်းႁူႉဝႃႈ မၼ်းပဵၼ်သင် လူၺ်ႈဢမ်ႇလူဝ်ႇဢၢၼ်ႇလိၵ်ႈ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉယူႇၼႂ်း **"ႁွင်ႈႁဵတ်းၵၢၼ်"**:
1.  **The Screen:** ပဵၼ် "တႅပ်း" (Desktop) ၸဝ်ႈၵဝ်ႇ။
2.  **Files:** ပဵၼ် "ၸေႈလိၵ်ႈ" ဢၼ်ၸဝ်ႈၵဝ်ႇတႅမ်ႈဝႆႉ။
3.  **Folders:** ပဵၼ် "ပပ်ႉသိမ်းၸေႈ" ဢၼ်ၸဝ်ႈၵဝ်ႇဢဝ်ၸေႈၸိူဝ်းၼၼ်ႉ သႂ်ႇဝႆႉႁႂ်ႈမၼ်းမီးလွၵ်းလၢႆး။
4.  **The Mouse:** ပဵၼ် "မိုဝ်း" ၸဝ်ႈၵဝ်ႇ ဢၼ်ၵႂႃႇယိပ်း၊ ၵႂႃႇလၢၵ်ႈ၊ ၵႂႃႇပိုတ်ႇပပ်ႉၸိူဝ်းၼၼ်ႉယဝ်ႉ။

ၵၢၼ်ၸႂ်ႉ "Desktop Metaphor" ၼႆႉ ႁဵတ်းႁႂ်ႈၵူၼ်းႁဝ်း ဢမ်ႇလူဝ်ႇလႆႈႁဵၼ်းလွင်ႈ Code ယၢပ်ႇယၢပ်ႇၵေႃႈ ၸႂ်ႉၶွမ်ႇပိဝ်ႇတႃႇလႆႈငၢႆႈငၢႆႈယဝ်ႉ။
