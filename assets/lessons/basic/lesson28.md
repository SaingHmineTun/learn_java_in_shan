## Lesson 28: Windows & Icons (The 'Desktop' Metaphor)

မိူဝ်ႈၸဝ်ႈၵဝ်ႇပိုတ်ႇၶွမ်ႇပိဝ်ႇတႃႇ၊ ၸေႃးဢၼ်ၸဝ်ႈၵဝ်ႇႁၼ်ၼၼ်ႉ ႁဝ်းႁွင်ႉဝႃႈ **Desktop** (ၼႃႈတႅပ်း) ၼႆယဝ်ႉ။



---

#### 1. What is a Metaphor?

**Metaphor** ပဵၼ်ၵၢၼ်ဢဝ် "လွင်ႈတၢင်းဢၼ်ႁဝ်းႁူႉၸၵ်း" မႃးၸႂ်ႉတႅၼ်းတၢင်ႇ "လွင်ႈတၢင်းၼႂ်းၶွမ်ႇပိဝ်ႇတႃႇ"။
* **Folders:** ၼႂ်းၶွမ်ႇပိဝ်ႇတႃႇ၊ မၼ်းပဵၼ် "လွၵ်းၵဵပ်းႁွမ်ၸူႉၶေႃႈမုၼ်း" ၵူၺ်း။ ၵူၺ်းၵႃႈ ၶဝ်ၸႂ်ႉႁၢင်ႈ **Folder** (ပပ်ႉသိမ်းလိၵ်ႈ) ႁႂ်ႈႁဝ်းႁူႉဝႃႈ မၼ်းမီးဝႆႉတွၼ်ႈတႃႇ "သႂ်ႇ File" ယဝ်ႉ။
* **Desktop:** ၶဝ်ဝူၼ်ႉဝႃႈ ၸေႃးၶွမ်ႇပိဝ်ႇတႃႇၸဝ်ႈၵဝ်ႇ မိူၼ်ၼင်ႇ "ၽိူၼ်ႁဵတ်းၵၢၼ်"။ ၸဝ်ႈၵဝ်ႇဝၢင်းပပ်ႉ (Files) ဝႆႉၼိူဝ်ၽိူၼ် လႆႈယဝ်ႉ။

---

#### 2. Windows

**Windows** ပဵၼ်လွၵ်းသီႇၸဵင်ႇ ဢၼ်ပိုတ်ႇဢွၵ်ႇမႃး မိူဝ်ႈႁဝ်းၸႂ်ႉ Program ၼိုင်ႈဢၼ်။
* မၼ်းႁဵတ်းႁႂ်ႈႁဝ်း "တူၺ်း" (View) ၶဝ်ႈၵႂႃႇၼႂ်း Program ၼၼ်ႉလႆႈ။
* ၸဝ်ႈၵဝ်ႇၸၢင်ႈပိုတ်ႇ Window လၢႆလၢႆဢၼ် ၼိူဝ်ၸေႃး ဢၼ်လဵဝ်ယဝ်ႉ။



---

#### 3. Icons

**Icon** ပဵၼ်ႁၢင်ႈလဵၵ်ႉလဵၵ်ႉ ဢၼ်လၢတ်ႈၼႄဝႃႈ Program ဢၼ်ၼၼ်ႉ ႁဵတ်းသင်လႆႈ။
* ႁၢင်ႈ **"မၢၵ်ႇမူၼ်းလူၵ်ႈ"** (Globe) = Browser တွၼ်ႈတႃႇၵႂႃႇ Internet။
* ႁၢင်ႈ **"ၸၢၵ်ႈထၢႆႇႁၢင်ႈ"** (Camera) = Program တွၼ်ႈတႃႇထၢႆႇႁၢင်ႈ။
* ႁၢင်ႈၸိူဝ်းၼႆႉ ၸွႆႈႁႂ်ႈႁဝ်းႁူႉဝႃႈ မၼ်းပဵၼ်သင် လူၺ်ႈဢမ်ႇလူဝ်ႇဢၢၼ်ႇလိၵ်ႈ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉယူႇၼႂ်း **"ႁွင်ႈႁဵတ်းၵၢၼ်"**:
1.  **The Screen:** ပဵၼ် "ၽိူၼ်" (Desktop) ၸဝ်ႈၵဝ်ႇ။
2.  **Files:** ပဵၼ် "ၸေႈလိၵ်ႈ" ဢၼ်ၸဝ်ႈၵဝ်ႇတႅမ်ႈဝႆႉ။
3.  **Folders:** ပဵၼ် "ပပ်ႉသိမ်းၸေႈ" ဢၼ်ၸဝ်ႈၵဝ်ႇဢဝ်ၸေႈၸိူဝ်းၼၼ်ႉ သႂ်ႇဝႆႉ။
4.  **The Mouse:** ပဵၼ် "မိုဝ်း" ၸဝ်ႈၵဝ်ႇ ဢၼ်ၵႂႃႇယိပ်း၊ ၵႂႃႇလၢၵ်ႈ၊ ၵႂႃႇပိုတ်ႇပပ်ႉၸိူဝ်းၼၼ်ႉယဝ်ႉ။

ၵၢၼ်ၸႂ်ႉ "Desktop Metaphor" ၼႆႉ ႁဵတ်းႁႂ်ႈၵူၼ်းႁဝ်း ဢမ်ႇလူဝ်ႇလႆႈႁဵၼ်းလွင်ႈ Code ယၢပ်ႇယၢပ်ႇၵေႃႈ ၸႂ်ႉၶွမ်ႇပိဝ်ႇတႃႇလႆႈငၢႆႈငၢႆႈယဝ်ႉ။
