## Lesson 8: Bits and Bytes (Measuring Digital Information)

မိူဝ်ႈႁဝ်းႁူႉယဝ်ႉဝႃႈ ၶွမ်ႇပိဝ်ႇတႃႇၼပ်ႉဢၢၼ်ႇလူၺ်ႈ 1 လႄႈ 0 (Binary)၊ တေႃႈလဵဝ်ႁဝ်းလူဝ်ႇလႆႈႁူႉၸွမ်းထႅင်ႈဝႃႈ ၶဝ်ဢဝ်မၢႆၼပ်ႉၸိူဝ်းၼၼ်ႉ မႃးႁူမ်ႈၵၼ်ႁဵတ်းႁႂ်ႈပဵၼ် "ၶေႃႈမုၼ်း" ဢၼ်ယႂ်ႇမႃးၸိူင်ႉႁိုဝ်။



---

#### 1. What is a Byte? (Byte ဢၼ်ဝႃႈၼၼ်ႉ ပဵၼ်သင်?)

ၼႂ်း Lesson 7 ႁဝ်းႁူႉယဝ်ႉဝႃႈ မၢႆၼပ်ႉ ဢၼ်ၵူၺ်းမီး (0 ဢမ်ႇၼၼ် 1) ၼၼ်ႉႁွင်ႉဝႃႈ **Bit**။ ၵူၺ်းၵႃႈ Bit ဢၼ်လဵဝ်ၼၼ်ႉ မၼ်းလဵၵ်ႉၼႃႇလႄႈ သိမ်းၶေႃႈမုၼ်းသင်ၵေႃႈ ဢမ်ႇပေႃးလႆႈ။ ၵွပ်ႈၼၼ် ၶွမ်ႇပိဝ်ႇတႃႇၸင်ႇဢဝ်ၶဝ်မႃးႁူမ်ႈၵၼ် ၼင်ႇတီႈတႂ်ႈယဝ်ႉ။

* **8 Bits = 1 Byte**
* **Byte** ပဵၼ် "မူႇၸုမ်း" ပိုၼ်ႉထၢၼ် ဢၼ်ၶွမ်ႇပိဝ်ႇတႃႇၸႂ်ႉတွၼ်ႈတႃႇသိမ်း "လိၵ်ႈၼိုင်ႈတူဝ်" (မိူၼ်ၼင်ႇ တူဝ်လိၵ်ႈ 'A')။

---

#### 2. Scaling Up (လၢႆးတႅၵ်ႈတၢင်းယႂ်ႇ ၶေႃႈမုၼ်း)

မိူဝ်ႈၶေႃႈမုၼ်းႁဝ်းၼမ်မႃး၊ ႁဝ်းတေၸႂ်ႉၸိုဝ်ႈႁွင်ႉဢၼ်ယႂ်ႇမႃးၸိူင်ႉၼႆ:

| တၢင်းယႂ်ႇ (Size) | တီႈပွင်ႇ (Meaning) | တူဝ်ယၢင်ႇ (Example) |
| :--- | :--- | :--- |
| **Kilobyte (KB)** | မွၵ်ႈ 1,000 Bytes | လိၵ်ႈဢၼ်တႅမ်ႈဝႆႉ ၼိုင်ႈၼႃႈ |
| **Megabyte (MB)** | မွၵ်ႈ 1,000 KB | ၵႂၢမ်း 1 ပုဒ် ဢမ်ႇၼၼ် ႁၢင်ႈ 1 ႁၢင်ႈ |
| **Gigabyte (GB)** | မွၵ်ႈ 1,000 MB | တူၺ်းငဝ်းတေႃႇ (Movie) 1 ႁူဝ် |
| **Terabyte (TB)** | မွၵ်ႈ 1,000 GB | ပႅတ်ႉသိမ်းၶေႃႈမုၼ်း (Hard Drive) ယႂ်ႇယႂ်ႇ 1 ဢၼ် |

> **Programmer Tip:** မိူဝ်ႈၸဝ်ႈၵဝ်ႇတႅမ်ႈ Code၊ ၸဝ်ႈၵဝ်ႇတေလႆႈဝူၼ်ႉၸွမ်းဝႃႈ Program ၸဝ်ႈၵဝ်ႇၸႂ်ႉ RAM လၢႆ MB? ဢမ်ႇၼၼ် File ဢၼ်ၸဝ်ႈၵဝ်ႇသိမ်းၼၼ်ႉ မီးဝႆႉ လၢႆ KB? လွင်ႈပွင်ႇၸႂ်တၢင်းယႂ်ႇၸိူဝ်းၼႆႉ တေၸွႆႈႁႂ်ႈၸဝ်ႈၵဝ်ႇတႅမ်ႈ Code လႆႈၶႅမ်ႉလိူဝ်မႃးယဝ်ႉ။

---

#### 3. Why the number 8? (ၵွပ်ႈသင်တေလႆႈပဵၼ် 8 Bits?)

ၵွပ်ႈသင်ၶွမ်ႇပိဝ်ႇတႃႇ ဢမ်ႇဢဝ် 10 Bits ပဵၼ် 1 Byte လေႃ?
ၼႆႉၵေႃႈယွၼ်ႉပိူဝ်ႈ Binary ၼၼ်ႉယဝ်ႉ။ ၶွမ်ႇပိဝ်ႇတႃႇမၵ်းမၼ်ႈဝႃႈ လွၵ်းၵၢၼ် 8-bit ($2^8 = 256$) ၼၼ်ႉ မၼ်းပေႃးတွၼ်ႈတႃႇတေသိမ်းလႆႈ တူဝ်လိၵ်ႈ၊ မၢႆၼပ်ႉ၊ လႄႈ မၢႆၶိုၵ်ႉ (Symbols) တင်းသဵင်ႈ ဢၼ်ႁဝ်းၸႂ်ႉယူႇၼိူဝ် Keyboard ယဝ်ႉ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇမီး **"ၶႆႇၵႆႇ"**:
1.  **Bit:** ပဵၼ် "ၶႆႇၵႆႇ ၼိုင်ႈလုၵ်ႈ" (0 ဢမ်ႇၼၼ် 1)။
2.  **Byte:** ပဵၼ် "ၶႆႇၵႆႇ ၼိုင်ႈႁေႃႇ" (ဢၼ်မီး 8 လုၵ်ႈ)။
3.  **Kilobyte:** ပဵၼ် "ၶႆႇၵႆႇ ၼိုင်ႈထူင်လူင်" (ဢၼ်မီး 1,000 ႁေႃႇ)။
