## Lesson 20: The CLI (Why Text is More Powerful than Icons)

တေႃႈလဵဝ် ႁဝ်းၵမ်ႈၼမ်ၸႂ်ႉတိုဝ်း Mouse သေ ၼဵၵ်း Icon ၵူၺ်း။ 
ဢၼ်ၼၼ်ႉ ႁဝ်းႁွင်ႉဝႃႈ **GUI** (Graphical User Interface) ယဝ်ႉ။ 
ၵူၺ်းၵႃႈ ၼႂ်းတွၼ်ႈၼႆႉ ႁဝ်းတေမႃးႁဵၼ်းလွင်ႈ **CLI** (Command Line Interface) ဢၼ်ပဵၼ် လၢႆးတႅမ်ႈလိၵ်ႈ တႃႇသင်ႇၵၢၼ်ယဝ်ႉ။



---

#### 1. Why use the Command Line? (ၵွပ်ႈသင်ၸႂ်ႉ CLI?)

ပဵၼ်သင် Programmer ၵမ်ႈၼမ် သမ်ႉသူင်ၸႂ်ႉ "LI" ဢၼ်မီးတူဝ်လိၵ်ႈၵူၺ်းၼၼ်ႉလေႃ?

* **Speed (လွင်ႈဝႆး):** သင်ၸဝ်ႈၵဝ်ႇလူဝ်ႇသၢင်ႈ Folder 100 ဢၼ်၊ သင်ၸႂ်ႉ Mouse ၸဝ်ႈၵဝ်ႇတေလႆႈၶူင် New Folder ပဵၼ် 100 ပွၵ်ႈ။ ၵူၺ်းၵႃႈ သင်ၸႂ်ႉတိုဝ်း CLI ၼႆၸိုင် ၸဝ်ႈၵဝ်ႇတႅမ်ႈလိၵ်ႈထႅဝ်လဵဝ်ၵေႃႈ ယဝ်ႉတူဝ်ႈယဝ်ႉ။
* **Direct Control:** ၸဝ်ႈၵဝ်ႇသင်ႇၵၢၼ်တေႃႇ Operating System ၶႅမ်ႉတိူဝ်းလိူဝ် GUI။
* **Universal Tool:** ၶွမ်ႇပိဝ်ႇတႃႇၵူႈလုၵ်ႈ မီး CLI ဝႆႉယူႇယဝ်ႉ။ သင် Mouse လူႉၵေႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉၸၢင်ႈၸႂ်ႉ CLI သေ ၵုမ်းထိင်းၶွမ်ႇပိဝ်ႇတႃႇလႆႈယူႇ။

---

#### 2. How it works? (မၼ်းႁဵတ်းၵၢၼ်ၸိူင်ႉႁိုဝ်?)

ၼႂ်း Module 3 ႁဝ်းႁဵၼ်းမႃးဝႃႈ **Shell** ပဵၼ် "ၼႃႈတႃ" OS။
* CLI ၼႆႉၵေႃႈ ပဵၼ် Shell မဵဝ်းၼိုင်ႈယဝ်ႉ။
* မၼ်းတေပႂ်ႉထွမ်ႇ "ၶေႃႈလိၵ်ႈ" (Command) ၸဝ်ႈၵဝ်ႇ။
* မိူဝ်ႈၸဝ်ႈၵဝ်ႇတႅမ်ႈယဝ်ႉ ၼဵၵ်း **Enter**၊ Shell တေဢဝ်ၶေႃႈလိၵ်ႈၼၼ်ႉ ၵႂႃႇလၢတ်ႈၼႄ **Kernel** ႁႂ်ႈႁဵတ်းၵၢၼ်ပၼ်ယဝ်ႉ။



---

#### 3. CLI in Windows (CLI ၼႂ်း Windows)

တွၼ်ႈတႃႇၵူၼ်းၸႂ်ႉ Windows၊ ႁဝ်းမီး CLI 2 မဵဝ်းဝႆႉ:
1.  **Command Prompt (cmd):** ပဵၼ်ဢၼ်ၵဝ်ႇ ဢၼ်ငၢႆႈသုတ်း။
2.  **PowerShell:** ပဵၼ်ဢၼ်မႂ်ႇ ဢၼ်မီးဢႃႇၼႃႇၼမ်လႄႈ Programmer ၶဝ်သူင်ၸႂ်ႉၼမ်။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇၵႂႃႇလၢၼ်ႉတၢင်းၵိၼ် **Fast Food**:
1.  **GUI (Mouse):** မိူၼ်ၼင်ႇၸဝ်ႈၵဝ်ႇၵႂႃႇၼဵၵ်း "ႁၢင်ႈတၢင်းၵိၼ်" တီႈၸေႃးယဝ်ႉ။ ၸဝ်ႈၵဝ်ႇလိူၵ်ႈလႆႈ ၸွမ်းၼင်ႇႁၢင်ႈဢၼ်မၼ်းၼႄဝႆႉၵူၺ်း။
2.  **CLI (Text):** မိူၼ်ၼင်ႇၸဝ်ႈၵဝ်ႇ လၢတ်ႈတေႃႇၵူၼ်းႁပ်ႉသင်ႇ (Waiter) ၵမ်းသိုဝ်ႈယဝ်ႉ။ ၸဝ်ႈၵဝ်ႇသင်ႇလႆႈ တႅတ်ႈၼႅတ်ႈလိူဝ်ယဝ်ႉ။ မိူၼ်ၼင်ႇ "ဢဝ် Burger ဢၼ်ဢမ်ႇသႂ်ႇၽၵ်းမီႇ၊ သႂ်ႇသျိတ်ႉၼမ်ၼမ်"။

CLI ပၼ် "ဢႃႇၼႃႇ" ၸဝ်ႈၵဝ်ႇ သင်ႇၵၢၼ်လႆႈၵူႈလွင်ႈ လိူဝ် GUI ယဝ်ႉ။
