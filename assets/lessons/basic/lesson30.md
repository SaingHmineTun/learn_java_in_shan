## Lesson 30: Software vs. Apps (What’s the difference?)

ႁဝ်းၸႂ်ႉၶေႃႈၵႂၢမ်းဝႃႈ "Software", "Program", လႄႈ "Apps" ယူႇတႃႇသေႇ။ 
တီႈတႅတ်ႈၼႅတ်ႈမၼ်း၊ ၶဝ်တင်းသဵင်ႈပဵၼ် "ၶေႃႈသင်ႇ" ဢၼ်သင်ႇႁႂ်ႈၶွမ်ႇပိဝ်ႇတႃႇႁဵတ်းၵၢၼ်ၵူၺ်း။ 
ၵူၺ်းၵႃႈ လၢႆးၸႂ်ႉတိုဝ်းၶဝ် ပႅၵ်ႇပိူင်ႈၵၼ်ယူႇဢိတ်းဢီတ်းယဝ်ႉ။

---

#### 1. Software & Programs (တွၼ်ႈတႃႇၶွမ်ႇပိဝ်ႇတႃႇလူင်)

မိူဝ်ႈႁဝ်းယူႇၼိူဝ် Laptop ဢမ်ႇၼၼ် Desktop၊ ႁဝ်းၵမ်ႈၼမ်ႁွင်ႉဝႃႈ **Software** ဢမ်ႇၼၼ် **Program**။
* **Complexity:** မၼ်းၵမ်ၼမ် ပဵၼ်ဢၼ်လူင်၊ ဢၼ်ယၢပ်ႇ၊ လႄႈ မီးၶိူင်ႈမိုဝ်းၼမ်။
* **Examples:** Microsoft Word, Photoshop, ဢမ်ႇၼၼ် Video Games။
* **Installation:** ၵမ်ၼမ်လူဝ်ႇလႆႈ Install (သႂ်ႇၼႂ်းၶိူင်ႈ) ႁဝ်းယဝ်ႉ။



---

#### 2. Apps (Application - တွၼ်ႈတႃႇမိုဝ်းထိုဝ်)

ၶေႃႈၵႂၢမ်းဝႃႈ **"App"** ၼႆႉ လႆႈၸိုဝ်ႈသဵင်မႃးမိူဝ်ႈမီး Smartphone (iPhone/Android) ယဝ်ႉ။
* **Single Purpose:** App ၵမ်ႈၼမ်တေႁဵတ်းၵၢၼ် "လွင်ႈလဵဝ်" ၵူၺ်း။ တူဝ်ယၢင်ႇ: App ထႆႇႁၢင်ႈ၊ App သူင်ႇလိၵ်ႈ။
* **Fast & Light:** မၼ်း "မဝ်" လႄႈ ပိုတ်ႇလႆႈဝႆး။
* **Touch-First:** မၼ်းထုၵ်ႇသၢင်ႈဝႆႉ တွၼ်ႈတႃႇၸႂ်ႉ "ၼိဝ်ႉမိုဝ်း" ၼဵၵ်း၊ ဢမ်ႇၸႂ်း Mouse။

---

#### 3. Web Apps (ဢၼ်လႅၼ်ႈၼိူဝ် Browser)

ၼႆႉပဵၼ် Software မဵဝ်းမႂ်ႇ ဢၼ်ၸဝ်ႈၵဝ်ႇဢမ်ႇလူဝ်ႇ Install သႂ်ႇၼႂ်းၶိူင်ႈ။ ၸဝ်ႈၵဝ်ႇပိုတ်ႇ Browser သေ ၸႂ်ႉလႆႈၵမ်းသိုဝ်ႈ။
* **Examples:** Facebook, Gmail, YouTube.
* **Benefit:** ၸဝ်ႈၵဝ်ႇၸႂ်ႉတီႈလႂ်ၵေႃႈလႆႈ၊ တီႈ Computer ၵေႃႈလႆႈ၊ တီႈ Phone ၵေႃႈလႆႈ၊ ၶေႃႈမုၼ်းၸဝ်ႈၵဝ်ႇ တေမိူၼ်ၵၼ်တင်းသဵင်ႈ။


---

**ၶေႃႈၵႂၢမ်း Technical ဢၼ်လူဝ်ႇမၢႆတွင်း**

* **Application Software:** Program ဢၼ်သၢင်ႈမႃးႁႂ်ႈၵူၼ်းၸႂ်ႉတိုဝ်း ႁဵတ်းၵၢၼ်သင်မဵဝ်းမဵဝ်း မိူၼ်ၼင်ႇတႅမ်ႈလိၵ်ႈ ဢမ်ႇၼၼ် လဵၼ်ႈၵိမ်း။
* **System Software:** Program ဢၼ်ယူႇၽၢႆႇလင်သေ ၵုမ်းထိူင်း Hardware ႁႂ်ႈၸၢၵ်ႈႁဵတ်းၵၢၼ်လႆႈ (မိူၼ်ၼင်ႇ Windows)။
* **SaaS (Software as a Service):** Software ဢၼ်ႁဝ်းၸႂ်ႉၼိူဝ် Internet လူၺ်ႈဢမ်ႇလူဝ်ႇသႂ်ႇဝႆႉၼႂ်းၸၢၵ်ႈ (မိူၼ်ၼင်ႇ Google Docs)။
