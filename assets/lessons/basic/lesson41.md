## Lesson 41: HTTP/HTTPS (The Rules of the Web)

မိူဝ်ႈႁဝ်းၶဝ်ႈ Website ၼိုင်ႈဢၼ်၊ ႁဝ်းတေႁၼ်တူဝ်လိၵ်ႈ **http://** ဢမ်ႇၼၼ် **https://** ဝႆႉတၢင်းၼႃႈၸိုဝ်ႈမၼ်း။ ၼႆႉပဵၼ် "ၵတိ" (**Protocols**) ဢမ်ႇၼၼ် ပဵၼ်လၢႆးၵၢၼ် ဢၼ်ၶွမ်ႇပိဝ်ႇတႃႇသွင်လုၵ်ႈ လၢတ်ႈတေႃႇၵၼ်ယဝ်ႉ။

[Image showing the difference between HTTP and HTTPS: HTTP is an open envelope, HTTPS is a locked briefcase]

---

#### 1. What is HTTP?

**HTTP** (**HyperText Transfer Protocol**) ပဵၼ်လၢႆးၵၢၼ်ပိုၼ်ႉထၢၼ် ဢၼ်ၸႂ်ႉသူင်ႇၶေႃႈမုၼ်း (လိၵ်ႈ၊ ႁၢင်ႈ) ၼႂ်း Web။
* **The Problem:** ၶေႃႈမုၼ်း ဢၼ်သူင်ႇၸွမ်း HTTP ၼၼ်ႉ မၼ်းပဵၼ် **"လိၵ်ႈလၢႆ"** (Plain Text)။
* သင်ၸဝ်ႈၵဝ်ႇတႅမ်ႈ Password သႂ်ႇၼႂ်း Website ဢၼ်ပဵၼ် HTTP၊ ၵူၼ်းႁၢႆႉ (Hackers) ၸၢင်ႈ "လွမ်တူၺ်း" သေ ႁၼ် Password ၸဝ်ႈၵဝ်ႇလႆႈငၢႆႈငၢႆႈယဝ်ႉ။

---

#### 2. What is HTTPS? (The Secure Way)

**HTTPS** (တူဝ် **S** မၵ်းမၼ်ႈဝႃႈ **Secure**) ပဵၼ် HTTP ဢၼ်သႂ်ႇဝႆႉပႃး **Encryption** (လွင်ႈသိမ်းလပ်ႉ)။
* မိူဝ်ႈၸဝ်ႈၵဝ်ႇသူင်ႇၶေႃႈမုၼ်း၊ မၼ်းတေ "သႂ်ႇၵူၵ်း" ဝႆႉပၼ်။ ၵူၼ်းႁၢႆႉ ၸၢင်ႈႁၼ်ဝႃႈမီးၶေႃႈမုၼ်းလႅၼ်ႈၵႂႃႇယူႇ၊ ၵူၺ်းၵႃႈ ၶဝ်တေဢမ်ႇႁူႉဝႃႈ မၼ်းပဵၼ် Password သင်။
* **The Lock Icon:** ၼႂ်း Browser ၸဝ်ႈၵဝ်ႇ၊ သင်မီးႁၢင်ႈ **"သေႃး"** (**Padlock**) ဝႆႉတၢင်းၼႃႈၸိုဝ်ႈ Website၊ မၼ်းမၵ်းမၼ်ႈဝႃႈ Website ၼၼ်ႉပဵၼ် HTTPS ယဝ်ႉ။



---

#### 3. Why it matters?

* **Privacy:** ဢမ်ႇႁႂ်ႈပိူၼ်ႈလွမ်တူၺ်း ၶေႃႈမုၼ်းလပ်ႉၸဝ်ႈၵဝ်ႇ။
* **Trust:** Website ဢၼ်မီး HTTPS ၼၼ်ႉ ပဵၼ် Website ဢၼ်မီးလွင်ႈၵတ်းယဵၼ် လႄႈ ထုၵ်ႇလီယုမ်ႇယမ်။
* **Security:** ၸွႆႈၵႅတ်ႇၶႄ ၵၢၼ်လႅၼ်ဢဝ်ၶေႃႈမုၼ်း (**Phishing**)။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇသူင်ႇ **"လိၵ်ႈ"** ၸူးပိူၼ်ႈ:
1.  **HTTP:** မိူၼ်ၼင်ႇၸဝ်ႈၵဝ်ႇတႅမ်ႈလိၵ်ႈ သႂ်ႇၼိူဝ် **"ပူတ်ႉသ်ၵၢတ်ႉ"** (**Postcard**)။ ၵူၼ်းသူင်ႇလိၵ်ႈ၊ ၵူၼ်းယူႇတၢင်းႁိမ်း တေၸၢင်ႈဢၢၼ်ႇလိၵ်ႈၸဝ်ႈၵဝ်ႇလႆႈ ၵူႈၵေႃႉ။
2.  **HTTPS:** မိူၼ်ၼင်ႇၸဝ်ႈၵဝ်ႇတႅမ်ႈလိၵ်ႈ သႂ်ႇၼႂ်း **"သွင်လိၵ်ႈ"** (**Envelope**) သေ ပိတ်းၵาว ဝႆႉလီလီ။ ၵူၼ်းသူင်ႇလိၵ်ႈ တေႁူႉဝႃႈ ၸဝ်ႈၵဝ်ႇသူင်ႇလိၵ်ႈယူႇ၊ ၵူၺ်းၵႃႈ ၶဝ်တေဢမ်ႇၸၢင်ႈဢၢၼ်ႇ လွင်ႈတၢင်းၼႂ်းမၼ်းလႆႈယဝ်ႉ။
