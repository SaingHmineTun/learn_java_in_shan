## Lesson 4: Storage (Hard Drives and SSDs)

ၼႂ်း Lesson 3 ႁဝ်းႁူႉယဝ်ႉဝႃႈ RAM ပဵၼ်တီႈသိမ်းၶေႃႈမုၼ်း "ၸူဝ်ႈၶၢဝ်း"။ ၵမ်းၼႆႉ ႁဝ်းတေမႃးတူၺ်း **Storage**—ဢၼ်ပဵၼ်တီႈသိမ်းၶေႃႈမုၼ်း "တႃႇသေႇ" ဝႆႉၼၼ်ႉယဝ်ႉ။



---

#### 1. What is Storage? (Storage ပဵၼ်သင်?)

**Storage** (ႁွင်ႉဝႃႈ Hard Drive ဢမ်ႇၼၼ် SSD) ၼႆႉ မိူၼ်ၼင်ႇ "ႁွင်ႈသိမ်းၶူဝ်း" (Pantry) ယဝ်ႉ။
* **Non-Volatile:** မၼ်းပႅၵ်ႇပိူင်ႈတင်း RAM ယွၼ်ႉဝႃႈ သင်ၸဝ်ႈၵဝ်ႇပိၵ်ႉၾႆးၵႂႃႇၵေႃႈ ၶေႃႈမုၼ်းၸိူဝ်းၼၼ်ႉ **တေဢမ်ႇႁၢႆ**။
* **Examples:** ငဝ်းတူင်ႉ၊ ၶႅပ်းႁၢင်ႈ၊ Game၊ လႄႈ File ၸိူဝ်းဢၼ်ၸဝ်ႈၵဝ်ႇ Save ဝႆႉတင်းသဵင်ႈ တေယူႇတီႈၼႆႈယဝ်ႉ။

---

#### 2. HDD vs. SSD (လၢႆးသိမ်းသွင်မဵဝ်း)

တေႃႈလဵဝ် မီးၶိူင်ႈသိမ်းၶေႃႈမုၼ်း 2 မဵဝ်း ဢၼ်ၵူၼ်းၸႂ်ႉၼမ်:

| မဵဝ်းမၼ်း | လၢႆးႁဵတ်းၵၢၼ် (How it works)                    | လွင်ႈဝႆး (Speed) |
| :--- |:------------------------------------------------|:-----------------|
| **HDD** (Hard Disk Drive) | ၸႂ်ႉ Disk ဢၼ်ပၼ်ႇ မိူဝ်ႈၽွင်းႁဵတ်းၵၢၼ်။         | ထိူင်း (Slow)    |
| **SSD** (Solid State Drive) | ၸႂ်ႉ "ၶျိပ်ႉ" (Chips) ဢၼ်ဢမ်ႇမီးတူဝ်တူင်ႉၼိုင်။ | ဝႆး (Very Fast)  |

> **Programmer Tip:** မိူဝ်ႈၸဝ်ႈၵဝ်ႇ "ပိုတ်ႇ" Program ၼိုင်ႈဢၼ်၊ ၶွမ်ႇပိဝ်ႇတႃႇတေၵႂႃႇထုတ်ႇဢဝ် Code တီႈ **Storage** သေ သူင်ႇၵႂႃႇဝၢင်းဝႆႉၼႂ်း **RAM** တွၼ်ႈတႃႇႁႂ်ႈ CPU ႁဵတ်းၵၢၼ်ယဝ်ႉ။

---

#### 3. Why the difference matters?

ၵွပ်ႈသင်ႁဝ်းဢမ်ႇၸႂ်ႉ Storage တႅၼ်း RAM တင်းသဵင်ႈလေႃ?
1. **Speed:** RAM ဝႆးလိူဝ် Storage မိူဝ်ႈၽွင်းႁဵတ်းၵၢၼ်တင်းၼမ်။ သင် CPU တေလႆႈၵႂႃႇထုတ်ႇဢဝ်ၶေႃႈမုၼ်းတီႈ Storage ၵူႈပွၵ်ႈၼႆ ၶွမ်ႇပိဝ်ႇတႃႇတေ "ႁဵတ်းၵၢၼ်ထိူင်း" (Lag) တႄႉတႄႉယဝ်ႉ။
2. **Cost:** RAM ၼၼ့် ၵႃႊၶၼ်မၼ်း သုင်လိူဝ် Storage တၢင်းၼမ်။ ၵွပ်ႈၼၼ် ႁဝ်းၸင်ႇမီး RAM ဢိတ်းၼိုင်ႈ (တွၼ်ႈတႃႇဝႆး) လႄႈ Storage ၼမ်ၼမ် (တွၼ်ႈတႃႇသိမ်းၶူဝ်း) ယဝ်ႉ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇပဵၼ် **ၵူၼ်းႁုင်ၶဝ်ႈ**:
* **RAM** ပဵၼ် "တီႈႁုင်ၶဝ်ႈ" တီႈၸဝ်ႈၵဝ်ႇဝၢင်းၽၵ်းသေ ၶတ်းၸႂ်တူမ်ႈႁုင်ဝႆႉယူႇ။
* **Storage** ပဵၼ် "တူႉယဵၼ်" (Refrigerator) ဢၼ်မီးတၢင်းလင် တႃႇတေသိမ်းဝႆႉ ၽၵ်းၵမ်ႈၽွင်ႈ။

မိူဝ်ႈၸဝ်ႈၵဝ်ႇလူဝ်ႇႁုင်၊ ၸဝ်ႈၵဝ်ႇၵႂႃႇဢဝ်ၽၵ်းတီႈ **တူႉယဵၼ် (Storage)** မႃးဝၢင်းၼိူဝ် **တီႈႁဵတ်းၵၢၼ် (RAM)** ယဝ်ႉၵေႃႈ ၸင်ႇတေ **ႁုင် (CPU)** ယဝ်ႉ။
