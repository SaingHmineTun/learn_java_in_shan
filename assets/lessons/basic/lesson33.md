## Lesson 33: Accessibility (Software for Everyone)

ၼႂ်း Lesson သုတ်းယွတ်ႈၶွင် Module 5 ၼႆႉ ႁဝ်းတေမႃးႁဵၼ်းလွင်ႈ တၢင်းလမ်ႇလွင်ႈၶွင် "ၵၢၼ်ႁဵတ်းႁႂ်ႈၵူႈၵေႃႉၸႂ်ႉလႆႈ" (**Accessibility**) ၼႆယဝ်ႉ။ 
Interface ဢၼ်လီၼၼ်ႉ ဢမ်ႇၸႂ်းဝႃႈတေတူၺ်းႁၢင်ႈလီ (Beautiful) ၵူၺ်း၊ မၼ်းတေလႆႈပဵၼ်ဢၼ် ၵူၼ်းၵူႈၵေႃႉၸႂ်ႉလႆႈယဝ်ႉ။



---

#### 1. What is Accessibility? (A11y)

**Accessibility** မၵ်းမၼ်ႈဝႃႈ Software ၼၼ်ႉ လူဝ်ႇလႆႈပဵၼ်ဢၼ်ၵူၼ်းၵူႈၵေႃႉ ၸႂ်ႉလႆႈငၢႆႈငၢႆႈ၊ ဢမ်ႇဝႃႈၶဝ်တေမီး ပၼ်ႁႃတၢင်းသၢႆတႃ၊ တၢင်းႁူ ဢမ်ႇၼၼ် တၢင်းၵၢၼ်တူင်ႉၼိုင်။

> **Note:** Programmer ၶဝ်ၵမ်ႈၼမ်တႅမ်ႈဝႃႈ **A11y** (ယွၼ်ႉမီးတူဝ်လိၵ်ႈ 11 တူဝ် ၼႂ်းၵႄႈ A လႄႈ Y)။

---

#### 2. Common Tools (ၶိူင်ႈမိုဝ်းၸွႆႈထႅမ်)

တွၼ်ႈတႃႇၸွႆႈၵူၼ်းၵူႈၵေႃႉ ၸႂ်ႉတိုဝ်းလႆႈ ၶိူင်ႈႁဝ်းၼႆ မီးလွၵ်းလၢႆးလၢႆမဵဝ်းဝႆႉ:

* **Screen Readers:** Software ဢၼ် "ဢၢၼ်ႇလိၵ်ႈ" ဢွၵ်ႇသဵင်ပၼ် ၵူႊလွင်ႈလွင်ႈ ဢၼ်မီးဝႆႉ တီႈၼိူဝ်ၸေႃး (တွၼ်ႈတႃႇၵူၼ်းဢမ်ႇႁၼ်တႃ)။
* **High Contrast:** ၵၢၼ်ပိၼ်ႇသီၸေႃး ႁႂ်ႈႁၼ်ၸႅင်ႊလႅင်းလီ (တွၼ်ႈတႃႇၵူၼ်းတႃဝဝ်း)။
* **Voice Control:** ၵၢၼ်ၸႂ်ႉ "သဵင်" သင်ႇၵၢၼ် တႅၼ်းတၢင်ႇၵၢၼ်ၼဵၵ်း Mouse (တွၼ်ႈတႃႇၵူၼ်းမိုဝ်းဢမ်ႇလီ)။
* **Subtitles:** လိၵ်ႈဢၼ်ဢွၵ်ႇမႃးၸွမ်းဝီႇတီႇဢူဝ်ႇ (တွၼ်ႈတႃႇၵူၼ်းႁူဢမ်ႇလီ)။



---

#### 3. Why Designers Care? (ၵွပ်ႈသင်ႁဝ်းလူဝ်ႇလႆႈသႂ်ႇၸႂ်?)

မိူဝ်ႈႁဝ်းသၢင်ႈ Interface ႁဝ်းလူဝ်ႇဝူၼ်ႉပႃး:
* **Text Size:** တူဝ်လိၵ်ႈလဵၵ်ႉႁႅင်းဝႆႉႁႃႉ? ၵူၼ်းထဝ်ႈတေလူလႆႈယူႇႁႃႉ?
* **Color Contrast:** သီႁိူင်ႊႁႅင်းဝႆႉႁႃႉ?
* **Buttons:** တုမ်ႇၼၼ်ႉလဵၵ်ႉႁႅင်းဝႆႉႁႃႉ? ၼဵၵ်းယၢပ်ႇယူႇႁႃႉ?

---

## 🏁 Module 5 Checkpoint

Module 5 ယဝ်ႉတူဝ်လီငၢမ်းယဝ်ႉ! တေႃႈလဵဝ်ၸဝ်ႈၵဝ်ႇႁူႉယဝ်ႉဝႃႈ:
* **GUI:** ပဵၼ်ၵၢၼ်ၸႂ်ႉႁၢင်ႈတႅၼ်းတၢင်ႇ Code။
* **Desktop Metaphor:** ၸေႃးၶွမ်ႇပိဝ်ႇတႃႇ မိူၼ်ၼင်ႇၽိူၼ်ႁဵတ်းၵၢၼ်။
* **Browser:** ပဵၼ်ၽၵ်းတူလူင် ဢၼ်ဢွၼ်ၶဝ်ႈၵႂႃႇၼႂ်း Internet။
* **Software vs Apps:** လွင်ႈပႅၵ်ႇပိူင်ႈၵၼ်ၶွင် Program ၸိူဝ်းၼၼ်ႉ။
* **Accessibility:** ၵၢၼ်သၢင်ႈ Interface တွၼ်ႈတႃႇၵူၼ်းၵူႈၵေႃႉ။
