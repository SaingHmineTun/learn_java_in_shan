## Lesson 33: Accessibility (Software for Everyone)

ၼႂ်း Lesson သုတ်းယွတ်ႈၶွင် Module 5 ၼႆႉ ႁဝ်းတေမႃးႁဵၼ်းလွင်ႈ တၢင်းလမ်ႇလွတ်ႈၶွင် "ၵၢၼ်ႁဵတ်းႁႂ်ႈၵူႈၵေႃႉၸႂ်ႉလႆႈ" (**Accessibility** )။ Interface ဢၼ်လီၼၼ်ႉ ဢမ်ႇၸႂ်းဝႃႈတေလႆႈသႅၼ်ႇ (Beautiful) ၵူၺ်း၊ မၼ်းတေလႆႈပဵၼ်ဢၼ် ၵူၼ်းၵူႈၵေႃႉၸႂ်ႉလႆႈယဝ်ႉ။



---

#### 1. What is Accessibility? (A11y)

**Accessibility** (ဢႅတ်ႉသႅတ်ႉသ်-သီႇ-ပီႇ-လီႇ-တီႇ) မၵ်းမၼ်ႈဝႃႈ Software ၼၼ်ႉ လူဝ်ႇလႆႈပဵၼ်ဢၼ်ၵူၼ်းၵူႈၵေႃႉ ၸႂ်ႉလႆႈငၢႆႈငၢႆႈ၊ ဢမ်ႇဝႃႈၶဝ်တေမီး ပၼ်ႁႃတၢင်းသၢႆတႃ၊ တၢင်းႁူ ဢမ်ႇၼၼ် တၢင်းၵၢၼ်တူင်ႉၼိုင်။

> **Note:** Programmer ၶဝ်ၵမ်ၼမ်တႅမ်ႈယေႃႈဝႃႈ **A11y** (ယွၼ်ႉမီးတူဝ်လိၵ်ႈ 11 တူဝ် ၼႂ်းၵႄႈ A လႄႈ Y)။

---

#### 2. Common Tools (ၶိူင်ႈမိုဝ်းၸွႆႈထႅမ်)

တွၼ်ႈတႃႇၸွႆႈၵူၼ်းၵူႈၵေႃႉ၊ Interface မီးလွၵ်းလၢႆးလၢႆမဵဝ်း:

* **Screen Readers:** Software ဢၼ် "ဢၢၼ်ႇလိၵ်ႈ" တီႈၼိူဝ်ၸေႃးပၼ် သဵင်လၢတ်ႈ (တွၼ်ႈတႃႇၵူၼ်းတႃဢမ်ႇႁၼ်)။
* **High Contrast:** ၵၢၼ်ပိၼ်ႇသီၸေႃး ႁႂ်ႈႁၼ်ၸႅၵ်ႇလႅင်းလီ (တွၼ်ႈတႃႇၵူၼ်းတႃဝၢႈ)။
* **Voice Control:** ၵၢၼ်ၸႂ်ႉ "သဵင်" သင်ႇၵၢၼ် တႅၼ်းတၢင်ႇၵၢၼ်ၼဵၵ်း Mouse (တွၼ်ႈတႃႇၵူၼ်းမိုဝ်းဢမ်ႇလီ)။
* **Subtitles:** လိၵ်ႈဢၼ်ဢွၵ်ႇမႃးၸွမ်းဝီႇတီႇဢူဝ်ႇ (တွၼ်ႈတႃႇၵူၼ်းႁူႇဢမ်ႇလီ)။



---

#### 3. Why Designers Care? (ၵွပ်ႈသင်ႁဝ်းလူဝ်ႇလႆႈသႂ်ႇၸႂ်?)

မိူဝ်ႈႁဝ်းသၢင်ႈ Interface ႁဝ်းလူဝ်ႇဝူၼ်ႉပႃး:
* **Text Size:** တူဝ်လိၵ်ႈလဵၵ်ႉပွတ်းႁႃႉ? ၵူၼ်းထဝ်ႈတေလူလႆႈယူႇႁႃႉ?
* **Color Contrast:** သီလိူၼ်းပွတ်းႁႃႉ? သီလိူၼ်းလႄႈသီၶၢဝ် ၸႅၵ်ႇၵၼ်ႁၼ်ၸႅတ်ႈယူႇႁႃႉ?
* **Buttons:** ပုမ်ႇၼၼ်ႉလဵၵ်ႉပွတ်းႁႃႉ? ၼဵၵ်းယၢပ်ႇယူႇႁႃႉ?

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉသၢင်ႈ **"ႁူင်းငၢၼ်း"** (Building) ၼိုင်ႈလင်:
1.  **GUI:** ပဵၼ် "ၽၵ်းတူ" လႄႈ "ၸေႃး" ဢၼ်ၵူၼ်းၵူႈၵေႃႉႁၼ်။
2.  **Accessibility:** ပဵၼ်ၵၢၼ်သၢင်ႈ **"တၢင်းလၢတ်ႈ" (Ramp)** တွၼ်ႈတႃႇၵူၼ်းၸႂ်ႉလူတ်ႉဝီႇၶျႄး (**Wheelchair**) လႄႈ **"ၸၼ်ၵႆႇ" (Elevator)**။
    * သင်ႁိူၼ်းၸဝ်ႈၵဝ်ႇမီးၵူၺ်း ၶူဝ်လၢႆလၢႆ (Stairs)၊ ၵူၼ်းၵေႃႉဢၼ်ပႆဢမ်ႇလႆႈၼၼ်ႉ တေၶဝ်ႈႁိူၼ်းၸဝ်ႈၵဝ်ႇဢမ်ႇလႆႈ။
    * Software ၵေႃႈမိူၼ်ၵၼ်၊ သင်မၼ်းဢမ်ႇမီး Accessibility၊ မၼ်းတေပိၵ်ႉလွၵ်းတၢင်း ဢမ်ႇပၼ်ၵူၼ်းမၢင်ၵေႃႉၸႂ်ႉလႆႈယဝ်ႉ။

---

## 🏁 Module 5 Checkpoint

ယဝ်ႉတူဝ်ႈယဝ်ႉ Module 5! တေႃႈလဵဝ်ၸဝ်ႈၵဝ်ႇႁူႉယဝ်ႉဝႃႈ:
* **GUI:** ပဵၼ်ၵၢၼ်ၸႂ်ႉႁၢင်ႈတႅၼ်းတၢင်ႇ Code။
* **Desktop Metaphor:** ၸေႃးၶွမ်ႇပိဝ်ႇတႃႇ မိူၼ်ၼင်ႇတႅပ်းႁဵတ်းၵၢၼ်။
* **Browser:** ပဵၼ်ၽၵ်းတူလူင် ဢၼ်ဢွၼ်ၶဝ်ႈၵႂႃႇၼႂ်း Internet။
* **Software vs Apps:** လွင်ႈပႅၵ်ႇပိူင်ႈၵၼ်ၶွင် Program ၸိူဝ်းၼၼ်ႉ။
* **Accessibility:** ၵၢၼ်သၢင်ႈ Interface တွၼ်ႈတႃႇၵူၼ်းၵူႈၵေႃႉ။
