## Lesson 3: RAM (The Workspace)

**RAM (Random Access Memory)** ၼႆႉ ပဵၼ် "တီႈႁဵတ်းၵၢၼ်" ၸူဝ်ႈၶၢဝ်းၶွမ်ႇပိဝ်ႇတႃႇယဝ်ႉ။ မၼ်းမီးၼႃႈၵၢၼ်သိမ်းဝႆႉ ၶေႃႈမုၼ်း ဢမ်ႇၼၼ် Program ၸိူဝ်းဢၼ်တိုၵ်ႉ "ပိုတ်ႇၸႂ်ႉဝႆႉ" ယူႇၼၼ်ႉယဝ်ႉ။



---

#### 1. What is RAM? (RAM ဢၼ်ဝႃႈၼၼ်ႉ ပဵၼ်သင်?)

သင်ဝႃႈ CPU ပဵၼ်ၵူၼ်းဝူၼ်ႉၵေႃႈ၊ RAM သမ်ႉပဵၼ် "တႅပ်း" (Desk) တွၼ်ႈတႃႇဝၢင်းၶူဝ်းသေ ႁဵတ်းၵၢၼ်ၼၼ်ႉယဝ်ႉ။

* **Short-term Memory:** RAM ၼႆႉ မၼ်းတွင်းလႆႈ "ၸူဝ်ႈၶၢဝ်း" ၵူၺ်း။ မၼ်းသိမ်းဝႆႉၶေႃႈမုၼ်း တွၼ်ႈတႃႇႁႂ်ႈ CPU ဢဝ်ၵႂႃႇၸႂ်ႉလႆႈဝႆးဝႆး။
* **Fast Access:** RAM ဝႆးလိူဝ်သေ Storage (Hard Drive) တေႉတေႉ။ ၵွပ်ႈၼၼ် ၶွမ်ႇပိဝ်ႇတႃႇၸင်ႇဢဝ် Program ဢၼ်ႁဝ်းတိုၵ်ႉၸႂ်ႉယူႇၼၼ်ႉ မႃးဝႆႉၼႂ်း RAM ယဝ်ႉ။

---

#### 2. Volatile Memory (မၼ်းႁၢႆလႆႈ)

ဢၼ်ၼႆႉ ပဵၼ်ဢၼ်လမ်ႇလွတ်ႈသုတ်း ဢၼ် Programmer တေလႆႈႁူႉဝႆႉ။

* **Power Dependent:** RAM လူဝ်ႇလႆႈမီး "ၾႆးၾႃႉ" ၸင်ႇတေတွင်းၵႂၢမ်းလႆႈ။
* **The Reset:** သင်ဝႃႈၾႆးမတ်ႉ ဢမ်ႇၼၼ် ပိၵ်ႉၶွမ်ႇပိဝ်ႇတႃႇၵႂႃႇ၊ ၶေႃႈမုၼ်းတင်းသဵင်ႈ ဢၼ်မီးၼႂ်း RAM ၼၼ်ႉ တေ "ႁၢႆၵႂႃႇ" (Wipe clean) တင်းသဵင်ႈယဝ်ႉ။

---

#### 3. Why do we need more RAM? (ၵွပ်ႈသင်ႁဝ်းလူဝ်ႇ RAM ၼမ်?)

မိူဝ်ႈႁဝ်းပိုတ်ႇ Program ၼမ်ၼမ် (မိူၼ်ၼင်ႇ ပိုတ်ႇ Facebook, YouTube, လႄႈ တႅမ်ႈ Code ၵမ်းလဵဝ်ၵၼ်)၊ RAM ႁဝ်းတေတဵမ်မႃး။

* **The Desk Analogy:** သင်ဝႃႈ "တႅပ်း" ႁဵတ်းၵၢၼ်ႁဝ်းလဵၵ်း၊ ႁဝ်းဝၢင်းပပ်ႉလၢႆလၢႆပပ်ႉၵေႃႈ တေတဵမ်ယဝ်ႉ။ မိူဝ်ႈတႅပ်းတဵမ်၊ ႁဝ်းတေႁဵတ်းၵၢၼ်ယၢပ်ႇသေ ၸၵ်ႉၸႃႉမႃး။ ၶွမ်ႇပိဝ်ႇတႃႇၵေႃႈ မိူၼ်ၵၼ်ယဝ်ႉ၊ သင် RAM တဵမ် မၼ်းတေ "ဢၢိူၵ်ႈ" (Lag) ယဝ်ႉ။

---

### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇတိုၵ်ႉ **"တႅမ်ႈလိၵ်ႈ"** ယူႇၼႂ်း Microsoft Word။
1.  လိၵ်ႈဢၼ်ၸဝ်ႈၵဝ်ႇတိုၵ်ႉတႅမ်ႈယူႇ၊ လႄႈ တိုၵ်ႉႁၼ်ယူႇၼိူဝ်ၼႃႈၸေႃးၼၼ်ႉ မၼ်းမီးဝႆႉၼႂ်း **RAM**။
2.  သင်ဝႃႈၾႆးမတ်ႉၵႂႃႇ မိူဝ်ႈၸဝ်ႈၵဝ်ႇ "ဢမ်ႇပႆႇလႆႈ Save"၊ လိၵ်ႈၸိူဝ်းၼၼ်ႉတေႁၢႆၵႂႃႇတင်းသဵင်ႈ။
3.  ၵွပ်ႈသင်? ၵွပ်ႈဝႃႈ လိၵ်ႈၼၼ်ႉမၼ်းတိုၵ်ႉမီးၼႂ်း RAM (Workspace) ၵူၺ်း၊ မၼ်းပႆႇလႆႈၵႂႃႇသိမ်းဝႆႉၼႂ်း **Storage** ၼၼ်ႉယဝ်ႉ။
