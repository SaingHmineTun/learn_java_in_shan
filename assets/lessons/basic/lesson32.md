## Lesson 32: Update & Version (Why does software change?)

ၸဝ်ႈၵဝ်ႇတေလႆႈႁၼ် Smartphone ဢမ်ႇၼၼ် Laptop ၸဝ်ႈၵဝ်ႇ ထၢမ်ယူႇတႃႇသေႇဝႃႈ **"လူဝ်ႇ Update Software ယဝ်ႉ"**။ 
ၵွပ်ႈသင် Software ၼၼ်ႉ ဢမ်ႇႁဵတ်းႁႂ်ႈမၼ်းယဝ်ႉတူဝ်ႈၵႂႃႇ ၼိုင်ႈပွၵ်ႈလဵဝ်လေႃ?

---

#### 1. Why do we need Updates? (ၵွပ်ႈသင်လူဝ်ႇ Update?)

Software ၼႆႉ မၼ်းဢမ်ႇမိူၼ်ၼင်ႇ "ပပ်ႉလိၵ်ႈ" ဢၼ်တႅမ်ႈယဝ်ႉၵေႃႈ ယဝ်ႉယဝ်ႉ။ 
မၼ်းမိူၼ်ၼင်ႇ "တူၼ်ႈမႆႉ" ဢၼ်လူဝ်ႇလႆႈလုမ်းလႃးတႃႇသေႇယဝ်ႉ။

* **Fixing Bugs:** မၢင်ပွၵ်ႈ Software မီး "တီႈၽိတ်း" (**Bugs**) ဢၼ်ႁဵတ်းႁႂ်ႈမၼ်းၵိုတ်း ဢမ်ႇၼၼ် ပိၵ်းႁင်းၵူၺ်း။ Programmer ၶဝ်တႅမ်ႈ Code မႂ်ႇသေ မႃး "မေး" (**Fix**) မၼ်းယဝ်ႉ။
* **Security:** ၵူၼ်းႁၢႆႉ (**Hackers**) တေႃႉတႄႉႁႃလၢႆးၶဝ်ႈယႃႇၼႂ်း Software ယူႇတႃႇသေႇ။ ၵၢၼ် Update ပဵၼ်ၵၢၼ် "ၸႅတ်ႈထတ်းလွင်ႈႁူမ်ႈလူမ်ႈ" ႁႂ်ႈလီတိူဝ်းလိူဝ်ၵဝ်ႇယဝ်ႉ။
* **New Features:** ၵၢၼ်သႂ်ႇ လွင်ႈႁဵတ်းသၢင်ႈမႂ်ႇ၊ ႁၢင်ႈမႂ်ႇ ဢမ်ႇၼၼ် လၢႆးၸႂ်ႉဢၼ်ဝႆးလိူဝ်ၵဝ်ႇ။



---

#### 2. Understanding Version Numbers (မၢႆဝႃႇသျိၼ်ႇ)

ၸဝ်ႈၵဝ်ႇတေလႆႈႁၼ်မၢႆမိူၼ်ၼင်ႇ **v1.0.2** ဢမ်ႇၼၼ် **v2.0**။ မၢႆၸိူဝ်းၼႆႉ လၢတ်ႈၼႄႁဝ်းဝႃႈ Software ၼၼ်ႉ ပိၼ်ႇလႅၵ်ႈမႃးၼမ်တၢၼ်ႇႁိုဝ်ယဝ်ႉ။

* **Major Version (v1, v2):** ပဵၼ်ၵၢၼ်လႅၵ်ႈလၢႆႈ "လူင်"။ တူဝ်ယၢင်ႇ: ပိၼ်ႇၼႃႈတႃမႂ်ႇတင်းသဵင်ႈ။
* **Minor Version (v1.1, v1.2):** ပဵၼ်ၵၢၼ်သႂ်ႇ "လွင်ႈတၢင်းမႂ်ႇ" ဢိတ်းဢီတ်း။
* **Patch (v1.1.1):** ပဵၼ်ၵၢၼ် "မေးတီႈၽိတ်း" (**Bug fix**) လဵၵ်ႉလဵၵ်ႉၵူၺ်း။



---

#### 3. Beta Software (Software ပုၼ်ႈတႃႇၵူတ်ႇထတ်း)

မၢင်ပွၵ်ႈ ၸဝ်ႈၵဝ်ႇတေႁၼ်ၶေႃႈၵႂၢမ်းဝႃႈ **"Beta"**။
* ၼႆႉမၵ်းမၼ်ႈဝႃႈ Software ၼၼ်ႉ "ပႆႇယဝ်ႉတူဝ်ႈလီ"။
* ၶဝ်ပိုတ်ႇပၼ် ႁႂ်ႈၵူၼ်းၵမ်ၼမ်ၸႂ်ႉတူၺ်း တွၼ်ႈတႃႇႁႂ်ႈၸွႆႈႁႃ "တီႈၽိတ်း" (**Bugs**) ယဝ်ႉ။

---

#### 💡 တူဝ်ယၢင်ႇ (Example)

ဝူၼ်ႉဝႃႈ ၸဝ်ႈၵဝ်ႇမီး **"လူတ်ႉၵႃး"** ၼိုင်ႈလမ်း:
1.  **Bug Fix:** မိူၼ်ၼင်ႇၸဝ်ႈၵဝ်ႇ ၵႂႃႇမေး "ၾႆးတႃလူတ်ႉ" ဢၼ်ဢမ်ႇလႅင်း။
2.  **Security Update:** မိူၼ်ၼင်ႇၸဝ်ႈၵဝ်ႇ ၵႂႃႇလႅၵ်ႈ "ၽၵ်းတူ" ႁႂ်ႈမၼ်းၶႅမ်ႉလိူဝ်ၵဝ်ႇ။
3.  **New Feature:** မိူၼ်ၼင်ႇၸဝ်ႈၵဝ်ႇ ၵႂႃႇသႂ်ႇ "ၸၢၵ်ႈၵတ်း" (**Aircon**) မႂ်ႇ သႂ်ႇၼႂ်းလူတ်ႉ။
4.  **Version 2.0:** မိူၼ်ၼင်ႇၸဝ်ႈၵဝ်ႇ လႅၵ်ႈ "ၶိူင်ႈလူတ်ႉ" (**Engine**) မႂ်ႇတင်းလမ်း လႄႈ ပိၼ်ႇသီမႂ်ႇတင်းလမ်းယဝ်ႉ။
